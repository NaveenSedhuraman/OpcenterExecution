Definieren von Versandzielen

Ein Versandziel ist die Fabrik oder der Kunde, an den der Container versandt wird. Das Versandziel enthält auch die Route, die verwendet wird, wenn die Versand-Transaktion Remote-Transaktionen an einen Zielstandort schickt.

Wenn Sie Versandziele definieren

Das Versandziel enthält folgende optionale Modellierungsdefinitionen:

• Kunde
• Fabrik und Orte, die für die Fabrik definiert wurden.
• Standort

Das Versandziel ist ein optionales Feld bei folgendem:

• Operation-Modellierungsdefinition
• Versandzielgruppe Modellierungsdefinition
• Transaktion "Remoteentgegennahme"

Das Versandziel ist ein erforderliches Feld in Transaktionen vom Typ "Lagertransfer" und "Versenden".

Vorgehensweise zum Definieren eines Versandziels

Gehen Sie folgendermaßen vor, um ein Versandziel zu definieren:

1. Öffnen Sie die Seite Versandziel. Die Seite "Versandziel“ erscheint in der Modellierungsseite.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für das Ziel in das Feld Versandziel an.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
