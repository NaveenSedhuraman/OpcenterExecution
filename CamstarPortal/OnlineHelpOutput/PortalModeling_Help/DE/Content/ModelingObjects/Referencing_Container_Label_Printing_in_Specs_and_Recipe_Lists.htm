Referenzieren von Containeretikettendruck in Spezifikationen und Rezeptlisten

Zum Implementieren des Etikettendrucks, nachdem Sie ihn definiert haben, müssen Sie die entsprechenden revisionierten Objekte des Typs Spezifikation oder Rezeptliste ändern und eine Etikettentransaktionszuordnung mit folgenden Angaben einfügen:

• Die Containertransaktionen, die Etiketten drucken sollen
• Die zu verwendenden Drucketikettendefinitionen
• Die Anzahl der zu druckenden Etiketten

Beim Referenzieren des Containeretikettendrucks

Dieses Thema deckt nur die grundlegenden Schritte ab, mit denen Drucketikettendefinitionen einer Spezifikation zugewiesen werden. Detaillierte Informationen zu den Feldern der Seite "Spez." finden Sie unter "Definieren von Spezifikationen".

Im Feld "Transaktionstyp" in der Tabelle "Etiketten-Transaktionszuordnung" werden alle Transaktionen für eine Spezifikation angezeigt.

Wichtig: Lesen Sie "Etikettenvorlagen und -Tags", bevor Sie Instanzen des Objekts "Spez." ändern.

Es ist nicht erforderlich, den Etikettendruck in einer Spezifikation zu referenzieren, wenn Sie den Etikettendruck in der Rezeptliste referenzieren.

Für den Zielmaterialcontainer steht das Feld "Materialcontainer" im Service. Der Etiketten-Tag-Ausdruck muss MaterialContainer.Name referenzieren, wenn Sie den Namen des Zielmaterialcontainerfelds in der Drucketikettendefinition referenzieren möchten.

Dieser Ausdruck kann verwendet werden, wenn die Drucketikettendefinition von "Rezeptaufgabe ausführen" und anderen Services verwendet werden soll:

IsFieldDefined(MaterialContainer)?MaterialContainer.Name:Container.Name

Etiketten-Transaktionszuordnung

Die Tabelle "Etiketten-Transaktionszuordnung" wird in den Modellierungsobjekten "Rezeptliste" und "Spez." angezeigt.

Vorgehensweise zum Hinzufügen einer Drucketikettendefinition zu einer Spezifikation

Gehen Sie folgendermaßen vor, um die Etikettendruckanforderungen für eine Spezifikation festzulegen:

1. Öffnen Sie die Seite Spez. Die Seite "Spez." wird innerhalb der Seite "Modellierung" angezeigt.
2. Wählen Sie eine vorhandene Instanz von Spez. aus. Die Anwendung aktualisiert die Seite "Spez." und zeigt die Werte der Instanz an.
3. Klicken Sie in der Tabelle "Etiketten-Transaktionszuordnung" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
4. Wählen Sie die entsprechende Transaktion, für die Containeretiketten gedruckt werden sollen, im Feld Transaktionstyp aus.
5. Wählen Sie die Drucketikettendefinition aus, die der Transaktion zugewiesen werden soll.
6. Geben Sie die Anzahl der für diese Transaktion zu druckenden Etiketten in das Feld Etikettenzahl ein. Dies kann eine Zahl oder ein Ausdruck sein, z. B. Container.Qty.
7. Wiederholen Sie die Schritte 3 bis 6, um Transaktionen zusätzliche Etiketten zuzuordnen.
8. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.

Vorgehensweise zum Hinzufügen einer Drucketikettendefinition zu einer Rezeptliste

Gehen Sie folgendermaßen vor, um einer Rezeptliste eine Drucketikettendefinition hinzuzufügen:

1. Öffnen Sie die Seite Rezeptliste. Die Seite "Rezeptliste" wird innerhalb der Seite "Modellierung" angezeigt.
2. Wählen Sie eine vorhandene Instanz einer Rezeptliste aus. Die Anwendung aktualisiert die Seite "Rezeptliste" und zeigt die Werte der Instanz an.
3. Klicken Sie in der Tabelle "Etiketten-Transaktionszuordnung" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
4. Wählen Sie die entsprechende Transaktion, für die Containeretiketten gedruckt werden sollen, im Feld Transaktionstyp aus.
5. Wählen Sie die Drucketikettendefinition aus, die der Transaktion zugewiesen werden soll.
6. Geben Sie die Anzahl der für diese Transaktion zu druckenden Etiketten in das Feld Etikettenzahl ein. Dies kann eine Zahl oder ein Ausdruck sein, z. B. Container.Qty.
7. Wiederholen Sie die Schritte 3 bis 6, um Transaktionen zusätzliche Etiketten zuzuordnen.
8. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
