Definitionen des physischen Modells

Das physische Modell umfasst die physischen Komponenten Ihrer Produktionsanlage, insbesondere Unternehmen, Fabrik, Ort, Organisation, Mitarbeiter, Rollen und Ressourcen. Weitere Objekte sind Ressourcengruppe, Einrichtung, Dokumentsatz, Dokument, Dokumentviewer, Rezept und Arbeitsbereich.

Mit Ausnahme von Dokumenten, die im Dokumentsatz enthalten sein müssen, können Sie diese Objekte in jeder beliebigen Reihenfolge definieren.

Beim Erstellen von Modellierungsdefinitionen

Wenn Sie Modellierungsdefinition erstellen, dürfen Sie keine unzulässigen Zeichen im Objektnamen verwenden. Z. B. darf kein Apostroph verwendet werden. Weitere Informationen finden Sie unten im entsprechenden produktspezifischen Systemdesigner-Benutzerhandbuch:
