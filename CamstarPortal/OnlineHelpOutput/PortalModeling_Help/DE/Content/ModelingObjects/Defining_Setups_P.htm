Definieren von Einrichtungen

Eine Einrichtung ist die physische Maschinenkonfiguration für einen bestimmten Prozess. Sie kann einen Dokumentsatz und den voraussichtlichen Zeitaufwand für die Ressourceneinrichtung beinhalten. Eine Einrichtung ist die Konfiguration einer Ressource, die für die Bearbeitung eines bestimmten Materialtyps benötigt wird. (Meist ist dies ein bestimmtes Produkt oder ein Container, der eine bestimmte Spezifikation referenziert.)

Die Definition einer Einrichtung enthält in der Regel nur Elemente wie Hardware-Konfigurationen, deren Änderung einige Zeit erfordert. Rezepte hingegen können eine Vielzahl anderer Elemente umfassen, wie etwa Prozesseinstellungen, die sofort geändert werden können.

Die Standardeinrichtung von Siemens enthält Elemente, wie den betreffenden Dokumentensatz, in dem die Einrichtungskonfiguration beschrieben ist, eine Schätzung des voraussichtlichen Zeitaufwands für die Konfiguration der Einrichtung usw. Die Definition von Einrichtungen dient häufig dazu, Planungs- oder Erledigungsinformationen bereitzustellen, mit denen Container oder Aufträge so bearbeitet werden können, dass keine längeren Umrüstzeiten zwischen den Elementen entstehen.

Beim Definieren einer Einrichtung

"Einrichtung" enthält die optionale Modellierungsdefinition "Dokumentsatz".

"Einrichtung" ist ein optionales Feld:

• In der Definition einer Spezifikation
• In den Transaktionen "Ressourceneinrichtung" und "Ressourcendurchsatz"

Felddefinitionen der Seite "Einrichtung"

In dieser Tabelle sind die Felder definiert, die nur auf der Seite "Einrichtung" zu finden sind.

Siehe auch "Gemeinsame Felder auf Modellierungsseiten". Dieser Abschnitt enthält Informationen zu den Feldern, die alle Modellierungsobjekte aufweisen.

Feld

Definition

Typ

Allgemein

Änderungsauftrag

Änderungsauftrag, der dieser Revision zugewiesen wurde. Sie können maximal 30 Zeichen eingeben.

Optional

Dokumentsatz

Der Name des Dokumentsatzes, der der Einrichtung zugeordnet ist. Zu den Einrichtungsdokumenten zählen in der Regel Ausrüstungszeichnungen, Teilelisten, Einrichtungsanweisungen und Rezepte.

Optional

Standardeinrichtungszeit

Die vergangene Gesamtzeit (in Tagen, Stunden, Minuten), die standardmäßig benötigt wird, um die mit der Einrichtung verbundenen Aktivitäten durchzuführen. Der Wert ist der Standardwert für Berechnungen der Zykluszeit.

Optional

Standardeinrichtungsarbeitszeit

Die gesamte Arbeitszeit (in Tagen, Stunden, Minuten), die standardmäßig benötigt wird, um die mit der Einrichtung verbundenen Aktivitäten durchzuführen. Der Wert unterscheidet sich vom Wert für die vergangene Zeit im Feld "Standardeinrichtungszeit", wenn für die Einrichtung mehrere Personen benötigt werden.

Optional

Vorgehensweise zum Definieren einer Einrichtung

Gehen Sie folgendermaßen vor, um eine Einrichtung zu definieren:

1. Öffnen Sie die Seite Einrichtung. Die Seite "Einrichtung" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für diese Einrichtung in das Feld Einrichtung ein.
4. Geben Sie die Revision dieses Objekts in das Feld Revision ein.
5. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
6. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
