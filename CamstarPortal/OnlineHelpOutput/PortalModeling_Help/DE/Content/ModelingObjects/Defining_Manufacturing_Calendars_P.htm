Definieren von Produktionskalendern

Ein Produktionskalender wird verwendet, um den Zeitstempel einer Transaktion in ein bestimmtes Fertigungsdatum und eine bestimmte Schicht zu übersetzen.

Beim Definieren eines Produktionskalenders

Siemens bietet zwei Möglichkeiten zum Hinzufügen von Schichten zu einem Produktionskalender. Sie können eine Excel-Datei mit Schichtinformationen importieren oder jede Schicht manuell hinzufügen. Sie müssen jede Schicht für jeden Tag eingeben, wenn Sie Schichtinformationen manuell eingeben.

Wenn Sie in der Anwendung neue Schichten zu einem Kalender hinzufügen möchten, müssen Sie den Kalender exportieren, die Schichten manuell hinzufügen und den Kalender wieder importieren.

Die Anwendung erfordert nicht, dass Sie das Start- und Enddatum für eine Schicht eingeben. Allerdings müssen Sie beide Datumsangaben eingeben, wenn Sie ein Datum eingeben. Wenn Sie z. B. den Anfangszeitpunkt (Datum und Uhrzeit) für eine Schicht eingeben, müssen Sie auch den Endzeitpunkt für die Schicht eingeben.

Die fiskalischen Geschäftsspalten in der Tabelle "Kalenderschicht" ermöglichen es Ihnen, Ihr Geschäftsjahr für die Berichterstattung einzugeben. Wenn Ihr Produktionskalender z. B. von April bis März läuft, Ihr Schichtkalender jedoch am 2. Februar 2013 anfängt, würden Sie folgende Geschäftszahlen für Einträge für den 2. Januar eingeben:

• Geschäftsjahr = 2012
• Geschäftsquartal = 4
• Geschäftsmonat = 10
• Geschäftswoche = 40

"Produktionskalender" enthält die optionalen Modellierungsdefinitionen "Schicht" und "Team".

"Produktionskalender" ist ein optionales Feld in den Modellierungsdefinitionen "Fabrik" und "Arbeitsbereich".

Beim Importieren eines Schichtkalenders

Die Anwendung importiert Excel-Dateien nur im *.xlsx-Format. Excel-Dateien früherer Versionen werden nicht unterstützt.

Siemens stellt einen Musterproduktionskalender bereit, den Sie an Ihre Anforderungen anpassen können. Die Datei "ManufacturingCalendar.xlsx" enthält ein Arbeitsblatt mit Anleitungen, die beschreiben, wie das Kalenderarbeitsblatt angepasst werden kann. Standardmäßig befindet sich die Datei "ManufacturingCalendar.xlsx" im folgenden Verzeichnis:

C:\Programme (x86)\Camstar\Camstar Portal

Sie müssen die Schichten und Teams, die in der Excel-Datei angegeben sind, definieren, bevor Sie den Import starten. Die Anwendung lässt Sie den Produktionskalender nicht speichern, wenn die Schichten und Teams vorher nicht in der Modellierung definiert wurden.

Beim Definieren eines Produktionskalenders für OEE
Hinweis: Die nachstehend beschriebene Funktionalität ist nur verfügbar, wenn der Industry Solutions-Arbeitsbereich installiert ist. Weitere Informationen finden Sie unter "Industry Solutions-Arbeitsbereich".

Im Feld "Ungeplante Zeit" in der Tabelle "Kalenderschicht" können Sie die Zeit eingeben, für die Ressourcen während einer Schicht ungeplant sind. Ungeplante Zeit darf die Dauer einer Schicht nicht überschreiten.

So wird ein Produktionskalender definiert

Gehen Sie folgendermaßen vor, um einen Produktionskalender zu definieren:

1. Öffnen Sie die Seite Produktionskalender. Die Seite "Produktionskalender" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen in das Feld Produktionskalender ein.
4. Weitere Informationen zum Laden von Schichtinformationen finden Sie in den folgenden Handlungsanweisungen:
• "Vorgehensweise zum manuellen Hinzufügen von Schichten"
• "So werden Schichtinformationen aus einer Excel-Datei geladen"
5. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
6. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum manuellen Hinzufügen von Schichten

Gehen Sie folgendermaßen vor, um Schichten manuell hinzuzufügen:

1. Führen Sie die Schritte unter "So wird ein Produktionskalender definiert" aus.

Oder

Wählen Sie eine vorhandene Instanz eines Produktionskalenders aus.

2. Klicken Sie in der Tabelle "Kalenderschicht" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
3. Wählen Sie eine Schicht und ein Team aus, wenn Sie Schichtinformationen aus Excel importieren.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Wiederholen Sie die Schritte 2 bis 4 für jede Schicht an jedem Tag, an dem Ihr Standort in Betrieb ist.
6. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.

So werden Schichtinformationen aus einer Excel-Datei geladen

Gehen Sie folgendermaßen vor, um Schichtinformationen aus einer Excel-Datei zu laden:

1. Führen Sie die Schritte unter "So wird ein Produktionskalender definiert" aus.

Oder

Wählen Sie eine vorhandene Instanz eines Produktionskalenders aus.

2. Klicken Sie auf das Symbol Datei hochladen, um zu der Excel-Datei zu navigieren.
3. Klicken Sie auf Schicht hinzufügen. Die Schichtinformationen werden in den Produktionskalender importiert.
4. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
