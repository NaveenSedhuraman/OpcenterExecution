Definieren von Ressourcenfamilien

Eine Ressourcenfamilie bietet eine Möglichkeit, Ressourcen nach ihren Attributen und Ihren Geschäftsanforderungen zu kategorisieren. Sie können eine Ressourcenfamilie als Filter für verschiedene Fertigungstransaktionen verwenden.

Die Behälterfamilie ist eine Unterklasse der Ressourcenfamilie.

Weitere Informationen zu diesem Modellierungsobjekt finden Sie unter "Definieren von Behälterfamilien".

Beim Definieren einer Ressourcenfamilie

Ressourcenfamilien sind z. B.:

• Zentrifugen
• Tester
• Handler

"Ressourcenfamilie" ist ein optionales Feld des Modellierungsobjekts "Ressource".

"Ressourcenfamilie" enthält die optionale Modellierungsdefinition "UI-Einstellungen".

Beim Definieren einer Ressourcenfamilie für OEE
Hinweis: Die nachstehend beschriebene Funktionalität ist nur verfügbar, wenn der Industry Solutions-Arbeitsbereich installiert ist. Weitere Informationen finden Sie unter "Industry Solutions-Arbeitsbereich".

Über das Feld "Ideale Zykluszeit" in der Tabelle "Ideale Zykluszeiten" können Sie die ideale Zykluszeit für Ressourcen in derselben Ressourcenfamilie definieren. Beim Definieren einer idealen Zykluszeit gelten folgende Regeln:

• Die ideale Zykluszeit, die auf der Seite "Ressource" definiert wurde, hat Vorrang vor der idealen Zykluszeit, die auf der Seite "Ressourcenfamilie" definiert wurde.
• Sie müssen für jede ideale Zykluszeit, die in der Tabelle "Ideale Zykluszeiten" definiert wird, ein Produkt oder eine Produktfamilie angeben.
• Sie können ein Produkt oder eine Produktfamilie nur einmal in der Tabelle "Ideale Zykluszeiten" angeben. Das Produkt oder die Produktfamilie muss für jede ideale Zykluszeit eindeutig sein.
• Die ideale Zykluszeit, die für ein Produkt definiert wird, hat Vorrang vor der idealen Zykluszeit, die für eine Produktfamilie definiert ist.

Beim Zuweisen von OEE-Einstellungen für die Ressourcenfamilie
Hinweis: Die nachstehend beschriebene Funktionalität ist nur verfügbar, wenn der Industry Solutions-Arbeitsbereich installiert ist. Weitere Informationen finden Sie unter "Industry Solutions-Arbeitsbereich".

Die folgenden Regeln gelten beim Zuweisen von OEE-Einstellungen:

• Die OEE-Einstellungen, die dem Modellierungsobjekt "Ressource" zugewiesen wurden, haben Vorrang vor den OEE-Einstellungen, die dem Modellierungsobjekt "Ressourcenfamilie" zugewiesen wurden.
• Die OEE-Einstellungen, die dem Modellierungsobjekt "Ressourcenfamilie" zugewiesen wurden, haben Vorrang vor den OEE-Einstellungen, die dem Modellierungsobjekt "Fabrik" zugewiesen wurden.

Weitere Informationen zum Zuweisen von OEE-Einstellungen für die Ressource finden Sie unter "Beim Zuweisen von OEE-Einstellungen für die Ressource". Weitere Informationen zum Zuweisen von OEE-Einstellungen für die Fabrik finden Sie unter "Zuweisen von OEE-Einstellungen für die Fabrik".

Beim Definieren von Zeitplänen für Ausfallzeit für eine Ressourcenfamilie
Hinweis: Die nachstehend beschriebene Funktionalität ist nur verfügbar, wenn der Industry Solutions-Arbeitsbereich installiert ist. Weitere Informationen finden Sie unter "Industry Solutions-Arbeitsbereich".

In der Tabelle "Zeitplan für Ausfallzeit" (Downtime Schedule) können Sie Zeitpläne für Ausfallzeit für die ausgewählte Ressourcenfamilie definieren. Die Anwendung füllt die Tabelle "Beschlossener Downtime-Zeitplan" auf der Seite "Ressource" mit Informationen aus der Tabelle "Zeitplan für Ausfallzeit" (Downtime Schedule).

Diese Regel gilt beim Definieren von Zeitplänen für Ausfallzeit für eine Ressourcenfamilie:

• Der auf der Seite "Ressource" definierte Zeitplan für Ausfallzeit hat Vorrang vor dem auf der Seite "Ressourcenfamilie" definierten Zeitplan für Ausfallzeit. Informationen zum Definieren von Zeitplänen für Ausfallzeit für eine Ressource finden Sie unter "Beim Definieren von Zeitplänen für Ausfallzeit für eine Ressource".

Vorgehensweise zum Definieren einer Ressourcenfamilie

Gehen Sie folgendermaßen vor, um eine Ressourcenfamilie zu definieren:

1. Öffnen Sie die Seite Ressourcenfamilie. Die Seite "Ressourcenfamilie" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen in das Feld Ressourcenfamilie ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Unter "Gemeinsame Felder auf Modellierungsseiten" finden Sie Informationen zu den optionalen Feldern.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
