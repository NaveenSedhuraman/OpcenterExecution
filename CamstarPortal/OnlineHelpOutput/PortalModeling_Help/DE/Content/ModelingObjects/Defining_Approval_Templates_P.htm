Definieren von Genehmigungsvorlagen

Ein Genehmigungsvorlage gibt die Benutzer an, die eine bestimmte Qualitätsdatensatzlösung genehmigen müssen, und gibt eine Liste von möglichen Entscheidungen für diese Benutzer an. Indem Sie einer Lösung eine Genehmigungsvorlage zuweisen, stellen Sie sicher, dass den entsprechenden Benutzern zum Zeitpunkt der Genehmigung die richtige Entscheidungsliste angezeigt wird.

Wenn Sie Genehmigungsvorlagen für eine Lösung von Qualitätsdatensätzen zuweisen, weisen Sie einer Organisation die Vorlagen zu. Die Anwendung weist die Genehmigungsvorlage anschließend dem Qualitätsdatensatz zu.

Weitere Informationen zum Zuweisen von Genehmigungsvorlagen zu einer Organisation finden Sie unter "Definieren einer Organisation".

Beim Definieren einer Genehmigungsvorlage

Bedenken Sie Folgendes, wenn Sie eine Genehmigungsvorlage definieren:

• Die Anwendung erfordert, dass Sie eine Genehmigungsentscheidungsliste (in der Modellierung definiert) auswählen.
Hinweis: Sie müssen die Seite "Genehmigungsentscheidungsliste" in der Modellierung verwenden, um Genehmigungsentscheidungslisten hinzuzufügen oder zu ändern.
• Sie können bei der Zuweisung der Genehmigungsvorlage weitere Genehmiger angeben, wenn Sie einen Qualitätsdatensatz lösen.
• Sie können allgemeine Anweisungen für alle Genehmiger und spezielle Anweisungen für einen einzelnen Genehmiger definieren.
• Sie können die Änderungsoptionen definieren, um die Änderungen zu steuern, die ein Eigentümer im Genehmigungsvorgang an einer Genehmigerzeile vornehmen darf. Sie können dem Eigentümer z. B. die Möglichkeit geben, Genehmiger zu ändern.
• Sie können Ersetzungsoptionen definieren, um zu bestimmen, ob Benutzer, die dieselbe Rolle haben wie der Genehmiger, die Genehmigung vornehmen dürfen.
Hinweis: Alle Genehmigungen für eine Ebene müssen abgeschlossen sein, bevor die Anwendung Genehmigungen an die nächste Ebene weiterleitet.

Beim Definieren von Rollenersatzoptionen

Sie müssen folgende Aufgaben ausführen, wenn es Ersatzbenutzern möglich sein soll, eine Genehmigung anstelle des angegebenen Genehmigers zu erteilen:

• Fügen Sie die Ersatzgenehmiger (Mitarbeiter) und deren Organisationen dem Modellierungsobjekt "Rolle" des angegebenen Genehmigers hinzu. Ersatzgenehmiger müssen zur selben Rolle gehören wie der angegebene Genehmiger. Weitere Informationen zur Aktualisierung des Modellierungsobjekts "Rolle" finden Sie unter "Definieren von Rollen".
• Wählen Sie "Rolle" als Ersetzungsoption für den angegebenen Genehmiger aus, wenn Sie die Genehmigungsvorlage definieren. Wenn Sie eine Rolle auswählen, kann ein Stellvertreter (Ersatz) die Genehmigung anstelle des angegebenen Genehmigers vornehmen.
• Wenn Sie der Genehmigungsvorlage Ersatzgenehmiger hinzufügen, müssen Sie diese zur selben Ebene hinzufügen, zu der Sie den angegebenen Genehmiger hinzugefügt haben.

Felddefinitionen der Seite "Genehmigungsvorlage"

Diese Tabelle definiert die Felder, die nur auf der Seite "Genehmigungsvorlage" und im Popup "Genehmiger hinzufügen" zu finden sind.

Siehe auch "Gemeinsame Felder auf Modellierungsseiten". Dieser Abschnitt enthält Informationen zu den Feldern, die alle Modellierungsobjekte aufweisen.

Feld

Definition

Typ

Allgemein

Genehmigungsentscheidungsliste

Genehmigungsentscheidungsliste, die dieser Genehmigungsvorlage zugewiesen ist.

Erforderlich

Anweisungen für alle Genehmiger

Anweisungen, die für alle Genehmiger gelten, die dieser Vorlage zugewiesen sind (bis zu 255 Zeichen).

Optional

Tabelle "Genehmiger"

Tabelle mit allen Genehmigungsrollen und entsprechenden Genehmigern für diese Vorlage. Die Felder in dieser Tabelle werden in den Popups "Hinzufügen" und "Bearbeiten" angezeigt.

Optional

Level

Das Genehmigungslevel, das dem Genehmiger zugewiesen ist. Es können nur numerische Zeichen eingegeben werden.

Das Level definiert die Weiterleitungsreihenfolge für eine Genehmigung. Die Anwendung kann die Genehmigung der Reihe nach, parallel oder in einer Kombination aus beiden an Genehmiger weiterleiten.

Erforderlich

Rolle

Vorgesehene Rolle des Genehmigers

Erforderlich

Name

Name des Genehmigers.

Hinweis: Wenn Sie hier keinen Genehmiger angeben, erfordert die Anwendung, dass der Eigentümer des Genehmigungsprozesses den Genehmiger vor der Weiterleitung angibt.

Erforderlich

Dauer

Eine Zahl, die die Gesamtzeit angibt, die für die Genehmigung zulässig ist.

Optional

Zeitraum

Einheiten, die verwendet werden, um die für die Genehmigung zulässige Zeit zu messen.

Optional

Eingabe erforderlich

Kontrollkästchen, das angibt, ob auf diesem Level eine Genehmigung erforderlich ist. Der Eigentümer kann diese Zeile für diesen Genehmiger während des Genehmigungsprozesses nicht ändern.

Optional

Änderungsoption

Option, die die Änderungen angibt, die ein Benutzer an einer Genehmigerzeile im Verlauf des Genehmigungsprozesses vornehmen kann:

• "Rolle" und "Name" (die Standardwerte) geben an, dass der Eigentümer die Rolle und den Namen des Genehmigers ändern kann.
• "Name" gibt an, dass der Eigentümer nur den Genehmigernamen ändern kann.
• "Kein" gibt an, dass der Eigentümer weder die Rolle noch den Namen des Genehmigers ändern kann. Wenn Sie "Kein" auswählen, stellt die Anwendung das Feld "Ersetzungsoption" auf "Kein" ein und macht das Feld zu einem schreibgeschützten Feld.

Erforderlich

Ersetzungsoption

Eine Option, die angibt, ob neben dem Benutzer, der in der Spalte "Name" angegeben ist, auch andere Benutzer die Genehmigung vornehmen können:

• "Rolle" bedeutet, dass jeder Benutzer in derselben Rolle die Genehmigung vornehmen kann.
• "Kein" gibt an, dass nur der angegebene Benutzer die Genehmigung vornehmen kann.

Erforderlich

Anweisungen für Genehmiger

Spezielle Anweisungen für diesen Genehmiger (bis zu 255 Zeichen).

Optional

Popup "Genehmiger hinzufügen"

Der Titel des Popups, das zum Hinzufügen oder Bearbeiten von Genehmigern in der Tabelle "Genehmiger" auf der Seite "Genehmigungsvorlage" verwendet wird, ändert sich dynamisch in Abhängigkeit von der Aktion, die Sie gerade ausführen. Durch Klicken auf "Neue Zeile hinzufügen" in der Tabelle "Genehmiger" wird das Popup "Hinzufügen" angezeigt. Wenn Sie einen vorhandenen Genehmiger in der Tabelle "Genehmiger" auswählen und auf "Ausgewählte Zeile bearbeiten" klicken, wird das Popup "Bearbeiten" angezeigt.

Vorgehensweise zum Definieren einer Genehmigungsvorlage

Gehen Sie folgendermaßen vor, um ein Genehmigungsvorlage zu definieren:

1. Öffnen Sie die Seite Genehmigungsvorlage. Die Seite "Genehmigungsvorlage" wird innerhalb der Seite "Modellierung" angezeigt.
2. Geben Sie einen Namen für diese Genehmigungsvorlage in das Feld Genehmigungsvorlagename ein.
3. Wählen Sie eine Genehmigungsentscheidungsliste aus der Liste aus.
4. Fügen Sie mindestens einen Genehmiger zur Tabelle "Genehmiger" hinzu, indem Sie die Schritte in "Vorgehensweise zum Hinzufügen von Genehmigern zu einer Genehmigungsvorlage" ausführen.
5. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
6. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Hinzufügen von Genehmigern zu einer Genehmigungsvorlage

Gehen Sie folgendermaßen vor, um Genehmiger zu einer Genehmigungsvorlage hinzuzufügen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren einer Genehmigungsvorlage" aus.

Oder

Wählen Sie eine vorhandene Instanz einer Genehmigungsvorlage aus.

2. Klicken Sie auf Neue Zeile hinzufügen in der Tabelle "Genehmiger". Das Popup "Hinzufügen" wird angezeigt.
3. Geben Sie ein numerisches Genehmigungslevel in das Feld Level ein.
4. Wählen Sie eine Rolle aus der Liste aus.
5. Wählen Sie einen Namen in der Liste aus.
6. Wählen Sie eine Änderungsoption in der Liste aus.
7. Wählen Sie eine Ersetzungsoption in der Liste aus.
8. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
9. Klicken Sie auf OK. Die Anwendung schließt das Popup "Genehmiger" und zeigt die Genehmigerinformationen in der Tabelle "Genehmiger" an.
10. Wiederholen Sie die Schritte 2 bis 9, um weitere Genehmiger hinzuzufügen.
11. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
