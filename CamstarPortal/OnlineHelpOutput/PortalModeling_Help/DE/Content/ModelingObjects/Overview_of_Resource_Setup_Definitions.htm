Beim Einrichten der Ressource

Wenn das Ressourcenstatusmodell eingerichtet ist, können Sie auf der Seite "Ressourceneinrichtung" einen Ressourcenstatuscode auswählen. Dadurch wird die Auswahl an Ressourcenstatusursachen eingrenzt und die Verfügbarkeit wird standardmäßig gesetzt.

Informationen zur Transaktion "Ressourceneinrichtung" finden Sie im Opcenter Execution Medical Device and Diagnostics Shop Floor-Benutzerhandbuch und im Opcenter Execution Core Shop Floor-Benutzerhandbuch.
