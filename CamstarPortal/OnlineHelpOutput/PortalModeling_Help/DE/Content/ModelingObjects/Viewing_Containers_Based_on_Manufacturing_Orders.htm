Anzeigen von Containern auf Grundlage von Produktionsaufträgen

Sie können die für einen Produktionsauftrag gestarteten Container verfolgen, indem Sie sich die Instanz in der Anwendung anzeigen lassen.

Im Feld "Container" wird eine schreibgeschützte Namensliste der für diesen Auftrag gestarteten Container angezeigt. Mengen und sonstige Statusinformationen werden nicht bereitgestellt. Diese Ansicht ist in der Modellierung verfügbar, wird aber dem Benutzer in der Fertigung nicht angezeigt.

Vorgehensweise zum Anzeigen eines Containers

Gehen Sie folgendermaßen vor, um einen Container auf der Grundlage eines Produktionsauftrags anzuzeigen:

1. Klicken Sie auf der Registerkarte "Modellierung" auf Produktionsauftrag. Die Registerkarte "Produktionsaufträge" wird angezeigt.
2. Wählen Sie eine Instanz aus der Liste Produktionsauftrag aus. Die Seite "Produktionsauftrag" wird angezeigt.
3. Gehen Sie zum Abschnitt Details.
4. Zeigen Sie die Einträge in der Tabelle "Container" an.
