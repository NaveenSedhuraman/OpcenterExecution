Definieren von Checklistenvorlagen

Das Modellierungsobjekt "Checklistenvorlage" ermöglicht es Ihnen, eine Checkliste von Elementen (Fragen oder Aufgaben) zu definieren, die Sie folgenden Objekten zuweisen können:

• Organisation
• Einteilungsspez.

Eine Checkliste stellt sicher, dass der Benutzer erforderliche Schritte bei der Verarbeitung eines Ereignisses ausführt. Eine Checklistenfrage oder ein Checklistenelement kann erforderlich oder optional sein, je nachdem, wie die Vorlage definiert wurde. Weitere Informationen zum Zuweisen von Checklistenvorlagen zu einer Einteilungsspezifikation finden Sie unter "Definieren von Einteilungsspezifikationen".

Beim Definieren von Checklistenvorlagen

Eine Checklistenvorlage enthält die optionale Modellierungsdefinition "Antwortsatz".

Die Tabelle "Checklistenvorlagenkonfiguration" wird verwendet, um der Checklistenvorlage Elemente hinzuzufügen. Sie müssen einen Antworteintrag für jedes Checklistenelement angeben.

Mindestens ein Element muss in der Tabelle "Checklistenvorlagenkonfiguration" definiert sein, bevor Sie die Checklistenvorlage speichern können.

Die folgenden Elemente sind gültige Antworteinträge, die Sie für ein Checklistenelement angeben können:

• Ein Antworteingabe-Steuerelement (z. B. ein Kontrollkästchen oder eine Auswahlliste), das einem Antwortsatz zugewiesen ist (falls zutreffend)
• Ein erforderliches Feld für Kommentare
• Ein Antworteingabe-Steuerelement, das einem Antwortsatz zugewiesen ist (falls zutreffend), und ein optionales oder erforderliches Feld für Kommentare

Diese Liste beschreibt die Einstellungen in der Konfigurationstabelle für bestimmte Checklistentypen. Wenn Sie ein Checklistenelement definieren möchten:

• das im Format einer einfachen To-Do-Liste (ein einzelnes Kontrollkästchen oder Optionsfeld für das Element) vorliegt, wählen Sie als Antworteingabe-Steuerelement entweder "Kontrollkästchen" oder "Optionsfeld" aus und belassen Sie den Antwortsatz unausgefüllt.
• für das Kommentare die einzige benötigte Antwort sind, wählen Sie "Kommentare erforderlich" in der Spalte "Kommentareintrag" aus. Belassen Sie alle Antwortspalten für das Checklistenelement unausgefüllt.

Darüber hinaus führt die Anwendung entsprechende Überprüfungen aus und Sie müssen:

• Ein Antworteingabe-Steuerelement angeben, wenn Sie einen Antwortsatz angeben.
• Der Auswahlliste einen Antwortsatz zuordnen, wenn Sie "Auswahlliste" als Antworteingabe-Steuerelement auswählen.
• Mindestens ein Antworteingabe-Steuerelement auswählen, wenn das Kontrollkästchen "Antworteintrag erforderlich" in der Tabelle "Checklistenvorlagenkonfiguration" aktiviert ist. Die Anwendung fordert Sie auf, ein Antworteingabe-Steuerelement auszuwählen.

Sie können den Nach-oben- und den Nach-unten-Pfeil verwenden, um die Reihenfolge der Checklistenelemente festzulegen, wenn Sie Checklistenelemente zur Tabelle "Checklistenvorlagenkonfiguration" hinzufügen.

Festlegen des Layouts für Checklistenelemente

Sie können das Layout der einzelnen Checklistenelemente festlegen, wenn Sie eine Checklistenvorlage konfigurieren. Die Option "Antwortlayout" ermöglicht es Ihnen, die Position des Antwortsatzes relativ zum Checklistenelement festzulegen. Die folgenden Optionen sind verfügbar:

• Unterhalb des Checklistenelements - vertikal (Standardeinstellung)

Die Anwendung zeigt die Antworten in einem dreispaltigen Format unter dem Checklistenelement an. Gibt es mehr als drei Antworten in einem Antwortsatz sind, werden die Antworten auf die nächste Zeile umbrochen.

• Unterhalb des Checklistenelements - horizontal

Die Anwendung zeigt die Antworten unter der Checkliste in einer einzelnen Spalte an.

• Rechts vom Checklistenelement

Die Anwendung verringert die Breite des Anzeigebereichs für das Checklistenelement, bricht den Text ggf. um und zeigt den Antwortsatz rechts vom Element an. Gibt es mehr Antworten, als in eine Zeile passen, werden die Antworten auf die nächste Zeile des Anzeigebereichs für den Antwortsatz umbrochen. (Die Anzahl von Antworten pro Zeile ist konfigurierbar.)

• Links vom Checklistenelement

Diese Option ist nur verfügbar, wenn Sie keinen Antwortsatz auswählen. Daher wird nur ein einziges Antworteingabe-Steuerelement links vom Checklistenelement angezeigt.

Hinweis: Wenn Sie keinen definierten Antwortsatz haben und "Antwortlayout" auf "Links vom Checklistenelement" festgelegt ist, legt die Anwendung den Wert von "Antwortlayout" auf den Standardwert (Unterhalb des Checklistenelements - vertikal) fest, wenn Sie einen Antwortsatz auswählen.

Wenn Sie angeben, dass ein Textfeld für Kommentare zu einem Prüflistenelement gehören soll, wird das Textfeld für Kommentare immer unter den Antworten angezeigt.

Vorgehensweise zum Definieren einer Checklistenvorlage

Gehen Sie folgendermaßen vor, um eine Checklistenvorlage zu definieren:

1. Öffnen Sie die Seite Checklistenvorlage. Die Seite "Checklistenvorlage" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für diese Checklistenvorlage in das Feld Checklistenvorlage ein.
4. Geben Sie die Revision dieses Objekts in das Feld Revision ein.
5. Fügen Sie Checklistenelemente hinzu, indem Sie die Schritte unter "Vorgehensweise zum Hinzufügen von Checklistenelementen zu einer Checklistenvorlage" ausführen.
6. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
7. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Hinzufügen von Checklistenelementen zu einer Checklistenvorlage

Gehen Sie folgendermaßen vor, um Checklistenelemente zu einer Checklistenvorlage hinzuzufügen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren einer Checklistenvorlage" aus.

Oder

Wählen Sie eine vorhandene Instanz einer Checklistenvorlage aus.

2. Navigieren Sie zur Tabelle "Checklistenvorlagenkonfiguration" und klicken Sie auf Neue Zeile hinzufügen. Das Popup "Hinzufügen" wird angezeigt.
3. Geben Sie eine Checklistenfrage oder -aufgabe in das Feld Checklistenelement ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Klicken Sie auf OK. Die Anwendung schließt das Popup "Hinzufügen" und zeigt das Checklistenelement in der Tabelle "Checklistenvorlagenkonfiguration" an.
6. Wiederholen Sie die Schritte 2 bis 5 für jedes weitere Checklistenelement, das Sie hinzufügen möchten.
7. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
