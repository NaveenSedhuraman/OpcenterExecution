Einrichten von Behältern

Verwenden Sie die Transaktion "Ressourceneinrichtung", um Änderungen an der Einrichtung eines Behälters aufzuzeichnen oder die Verfügbarkeit eines Behälters anzugeben. Die Verfügbarkeit eines Behälters bestimmt, ob ein Mitarbeiter diesen verwenden kann, um Fertigungstransaktionen durchzuführen.

Informationen zur Ressourceneinrichtung und Verfügbarkeit finden Sie im Opcenter Execution Medical Device and Diagnostics Shop Floor-Benutzerhandbuch und im Opcenter Execution Core Shop Floor-Benutzerhandbuch. Verwenden Sie die Schritte auf dieser Seite, um den Behälter einzurichten.

Beim Definieren der Behältereinrichtung

Die Einrichtungsinformationen enthalten einen Verweis auf eine bestimmte Behältereinrichtungsdefinition und den Behälter, der bearbeitet wird.

Die Verfügbarkeitsinformationen enthalten die Verfügbarkeit des Behälters ("Verfügbar" oder "Nicht verfügbar"), den Status (z. B. "Schmutzig", "Sauber", "Schmutzig aber brauchbar") und einen Grund (z. B. "Schmutzig wegen Säureverwendung"). Die Anwendung verwendet die Verfügbarkeitsinformationen, um eine Historie der Behälternutzung und der Verfügbarkeitsdetails zu erstellen.

Wenn ein Behälter z. B. nicht verfügbar ist, können Sie die Transaktion "Ressourceneinrichtung" ausführen, um die Verfügbarkeit in "Nicht verfügbar", den Status in "Schmutzig" und den Grund in "Schmutzig wegen Säureverwendung" zu ändern. Wenn der Behälter gesäubert und wieder verfügbar ist, können Sie die Transaktion erneut ausführen, um die Verfügbarkeit auf "Verfügbar" und den Status auf "Sauber" zu setzen. Sie verfügen jetzt über eine Aufzeichnung, aus der hervorgeht, wann, aus welchem Grund und wie lange der Behälter nicht verfügbar war.

So wird eine Behälterressource eingerichtet

Gehen Sie folgendermaßen vor, um einen Behälter einzurichten:

1. Öffnen Sie die Seite Ressourceneinrichtung über das Menü "Ressource". Die Seite "Ressourceneinrichtung" wird angezeigt.
2. Wählen Sie die entsprechende Behältergruppe aus der Liste Ressourcengruppe aus, um die Anzahl von Ressourcen zu beschränken, die im Feld Ressource angezeigt werden.
3. Wählen Sie im Feld Ressource den Behälter aus, den Sie einrichten möchten.
4. Wählen Sie im Feld Ressourcenstatuscode einen Ressourcenstatuscode aus.
5. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
6. Klicken Sie auf die Schaltfläche Einrichtung, um die Daten, die Sie für die Einrichtung eingegeben haben, zu sammeln.
7. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
