Referenzieren des Etikettendrucks für Produktionsereignisse in der Fabrik

Sie müssen das Fabrikobjekt folgendermaßen ändern, um die Etikettendruckfunktion für Abweichungen einzurichten:

• Legen Sie die Druckwarteschlange in den Produktionsereignistransaktionen fest, die Etiketten für die Fabrik drucken. (Dies ist die der Transaktion zugewiesene Fabrik, welche standardmäßig die Fabrik des Mitarbeiters ist.)
• Legen Sie eine Etikettentransaktionszuordnung für Produktionsereignisse fest, aus der Folgendes hervorgeht:
• Die Produktionsereignistransaktionen, die Etiketten drucken sollen
• Die zu verwendenden Drucketikettendefinitionen
• Die Anzahl der zu druckenden Etiketten (kann ein Ausdruck sein, z. B. GetListCount(QualityObject.EventData.EventLogs).
Hinweis: Produktionsereignisse werden in der Anwendung Portal Shop Floor aufgezeichnet und verwaltet. Weitere Informationen zum Etikettendruck für Produktionsereignistransaktionen in Portal Shop Floor finden Sie im Portal Shop Floor-Benutzerhandbuch.

Lesen Sie "Etikettenvorlagen und -Tags", bevor Sie Instanzen des Fabrikobjekts ändern.

Vorgehensweise zum Hinzufügen einer Druckwarteschlange und einer Drucketikettendefinition für Produktionsereignisse zu einer Fabrik

Gehen Sie folgendermaßen vor, um Ihre Etikettendruckanforderungen für Produktionsereignisse für eine Fabrik festzulegen:

1. Öffnen Sie die Seite Fabrik. Die Seite "Fabrik" wird innerhalb der Seite "Modellierung" angezeigt.
2. Wählen Sie eine vorhandene Fabrik aus. Die Anwendung zeigt die Informationen dieser Fabrik an.
3. Wählen Sie eine Druckwarteschlange aus, wenn Sie nur einen Drucker zum Druck aller Produktionsereignisetiketten der Fabrik verwenden möchten.
4. Erweitern Sie den Abschnitt Transaktionen.
5. Klicken Sie in der Tabelle "Zuordnung für Produktionsereignisetiketten" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
6. Wählen Sie einen Produktionsereignis-Transaktionstyp aus, für den Etiketten gedruckt werden sollen.
7. Wählen Sie eine Drucketikettendefinition aus, die Sie dem Produktionsereignis-Transaktionstyp zuweisen möchten.
8. Geben Sie die Anzahl der für diese Transaktion zu druckenden Etiketten in das Feld Etikettenanzahl ein.
9. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
