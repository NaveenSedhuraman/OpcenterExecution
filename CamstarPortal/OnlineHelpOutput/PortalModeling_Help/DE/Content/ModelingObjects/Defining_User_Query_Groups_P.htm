Definieren von Benutzerabfragegruppen

Eine Benutzerabfragegruppe ist ein Satz von Benutzerabfragen. Eine Benutzerabfragegruppe kann verwendet werden, um die Anzahl von Benutzerabfragen entsprechend Ihren geschäftlichen Anforderungen einzuschränken. Sie können z. B. eine Benutzerabfragegruppe definieren, die nur die Benutzerabfragen enthält, die für eine Containersuche in der Portal Shop Floor-Anwendung verwendet werden. Eine Benutzerabfragegruppe kann individuelle Benutzerabfragen und andere, vordefinierte Benutzerabfragegruppen enthalten.

Beim Definieren einer Benutzerabfragegruppe

Sie können den betreffenden Tabellen mehrere Einträge oder Gruppen hinzufügen.

Sie müssen eine oder mehrere Benutzerabfragen definieren, damit Sie eine Benutzerabfragegruppe definieren können.

Die Schaltfläche "Aufgelöste Einträge" ermöglicht es Ihnen, eine Liste aller Werte anzuzeigen, die in der Eintragsliste für diese Gruppe und alle darin enthaltenen Gruppen angegeben sind.

Vorgehensweise zum Definieren einer Benutzerabfragegruppe

Gehen Sie folgendermaßen vor, um eine Benutzerabfragegruppe zu definieren:

1. Öffnen Sie die Seite Benutzerabfragegruppe. Die Seite "Benutzerabfragegruppe" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für diese Gruppe in das Feld Benutzerabfragegruppe ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Hinzufügen von Einträgen zu einer Benutzerabfragegruppe

Gehen Sie folgendermaßen vor, um der Benutzerabfragegruppe einen Eintrag hinzuzufügen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren einer Benutzerabfragegruppe" aus.

Oder

Wählen Sie eine vorhandene Instanz einer Benutzerabfragegruppe aus.

2. Fügen Sie der Gruppe wie folgt Einträge hinzu:
a. Klicken Sie unten in der Tabelle "Einträge" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
b. Klicken Sie in die neue Zeile, um eine Liste von Einträgen anzuzeigen, und wählen Sie einen Eintrag aus, den Sie der Gruppe hinzufügen möchten. Der ausgewählte Eintrag wird der Tabelle hinzugefügt.
c. Wiederholen Sie die Schritte a und b, um der Tabelle einen weiteren Eintrag hinzuzufügen.
3. Fügen Sie der Gruppe wie folgt eine Untergruppe hinzu:
a. Klicken Sie unten in der Tabelle "Gruppen" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
b. Klicken Sie in die neue Zeile, um eine Liste zuvor definierter Gruppen anzuzeigen. Wählen Sie dann eine Gruppe aus, die Sie der Gruppe (als Untergruppe) hinzufügen möchten. Die Anwendung fügt die ausgewählte Gruppe der Tabelle hinzu.
c. Wiederholen Sie die Schritte a und b, um der Tabelle eine weitere Gruppe hinzuzufügen.
4. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung mit dem Hinweis an, dass das Modellierungsobjekt hochgeladen wurde.
