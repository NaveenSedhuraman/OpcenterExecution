Containerverarbeitung auf Grundlage von Erledigungsregeln

Siemens stellt fünf verschiedenartige Erledigungsregeln bereit. Eine einzelne Erledigungsregeldefinition kann einen oder mehrere dieser Regeltypen umfassen. Wenn Sie eine Erledigungsregel definieren, geben Sie die Erledigungsregeltypen an und ordnen jeden Typ anschließend einer von Siemens oder von Ihnen erstellten Abfrage zu. Die Abfrage bestimmt die Reihenfolge, in der die Elemente dem Mitarbeiter in der Fertigung angezeigt werden.

Erledigungsregeltypen und Abfragen

Diese Tabelle enthält eine Kurzreferenz zu den Erledigungstypen und den Namen der damit assoziierten Siemens-Abfragen, die Sie den Erledigungstypen zuordnen können. Diese Abfragen sind die Standard-Designer-Abfragen, die auf der Seite "Erledigungsregel" verwendet werden, wenn Sie "Designer-Abfrage" als Abfragetyp verwenden. Sie benötigen diese Informationen beim Definieren von Erledigungsregeln.

Containermengen auf Grundlage von Produktionsaufträgen

Möglicherweise geben Ihre Geschäftsregeln vor, dass Container auf Grundlage von Produktionsaufträgen gestartet werden sollen (Erledigungstyp: Auftrag). In diesem Fall wird dem Benutzer in der Fertigung eine Liste von Produktionsaufträgen angezeigt. Wenn "Erledigung durchsetzen" auf "Wahr" festgelegt ist, müssen die Container in der Reihenfolge gestartet werden, in der sie in der Liste angezeigt werden. Sonst können Sie die Container in beliebiger Reihenfolge starten. Weitere Informationen zur Einstellung "Erledigung durchsetzen" finden Sie unter "Definieren von Erledigungsregeln".

Der Produktionsauftrag ganz oben in der Liste basiert auf der implementierten Abfrage. Wenn Sie Standard-Abfrage verwenden, ist der Produktionsauftrag ganz oben in der Liste der Auftrag mit dem frühesten geplanten Startdatum, gefolgt vom Auftrag mit der höchsten Priorität.

Wenn Sie die Standard-Abfrage verwenden, verfolgen Opcenter EX MDD und Opcenter EX CR die Mengen von Containern auf Grundlage der im Produktionsauftrag angegebenen Menge nach. Wenn die Mengen für den Produktionsauftrag erreicht sind, wird der Auftrag aus der Erledigungsliste entfernt. In der Zwischenzeit ermöglichen es Opcenter EX MDD und Opcenter EX CR dem Benutzer, in der Fertigung die Mengen zu verfolgen, die bereits im Produktionsauftrag verarbeitet wurden.

Vorrangregeln

Sie können Erledigungsregeln für Operationen, Arbeitsbereiche und Fabriken definieren. Die Opcenter EX MDD- und Opcenter EX CR-Logik erzwingt Erledigungsregeln, die auf den folgenden Vorrangregeln basieren:

• Erledigungsregeln, die für Operationen definiert sind, haben Vorrang vor jenen für Arbeitsbereiche.
• Sind keine Erledigungsregeln für Operationen definiert, so haben diejenigen für Arbeitsbereiche Vorrang vor jenen für Fabriken.
