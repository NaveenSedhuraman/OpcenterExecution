Definieren von Behälterstatusgründen

Ein Behälterstatusgrund bietet eine zusätzliche Detailebene zur Beschreibung des Grundes für den aktuellen Status des Behälters. So kann z. B. Sterilisierungsprozess fehlgeschlagen den Grund für den Behälterstatus Kontaminiert beschreiben. Die standardisierten Codes für Gründe können Behältern zugewiesen werden und Bediener in die Lage versetzen, Entscheidungen in der Fertigung anhand besserer Informationen zu treffen.

Siehe auch "Gemeinsame Felder auf Modellierungsseiten". Dieser Abschnitt enthält Informationen zu den Feldern, die alle Modellierungsobjekte aufweisen.

Vorgehensweise zum Definieren eines Behälterstatusgrunds

Gehen Sie folgendermaßen vor, um einen Behälterstatusgrund zu definieren:

1. Öffnen Sie die Seite Behälterstatusgrund. Die Seite "Behälterstatusgrund" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für den Statusgrund in das Feld Behälterstatusgrund ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Unter "Gemeinsame Felder auf Modellierungsseiten" finden Sie Informationen zu den optionalen Feldern.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
