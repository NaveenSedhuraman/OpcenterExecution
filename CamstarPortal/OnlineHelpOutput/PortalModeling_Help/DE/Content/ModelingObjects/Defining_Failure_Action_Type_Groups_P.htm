Definieren von Fehleraktionstypgruppen

Das Modellierungsobjekt "Fehleraktionstypgruppe" ermöglicht es Ihnen, ähnliche Fehleraktionstypen zu gruppieren. Fehleraktionstypgruppen vereinfachen die Auswahl von Fehleraktionstypen für eine Ereignisaufzeichnung, indem sie die verfügbaren Aktionstypen einengen.

Beim Definieren einer Fehleraktionstypgruppe

Sie können den betreffenden Tabellen mehrere Einträge oder Gruppen hinzufügen.

Die Schaltfläche "Aufgelöste Einträge" ermöglicht es Ihnen, eine Liste aller Werte anzuzeigen, die in der Eintragsliste für diese Gruppe und alle darin enthaltenen Gruppen angegeben sind.

"Fehleraktionstypgruppe" enthält die optionale Modellierungsdefinition "Fehleraktionstyp".

Vorgehensweise zum Definieren einer Fehleraktionstypgruppe

Gehen Sie folgendermaßen vor, um eine Fehleraktionstypgruppe zu definieren:

1. Öffnen Sie die Seite Fehleraktionstypgruppe. Die Seite "Fehleraktionstypgruppe" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für diese Fehleraktionstypgruppe in das Feld Fehleraktionstypgruppe ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Hinzufügen von Einträgen zu einer Fehleraktionstypgruppe

Gehen Sie folgendermaßen vor, um einer Fehleraktionstypgruppe Einträge hinzuzufügen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren einer Fehleraktionstypgruppe" aus.

Oder

Wählen Sie eine bereits vorhandene Instanz einer Fehleraktionstypgruppe aus.

2. Klicken Sie auf Neue Zeile hinzufügen in der Tabelle "Einträge". Eine neue Zeile wird angezeigt.
3. Klicken Sie auf die leere Zeile und wählen Sie einen Fehleraktionstyp aus der Liste verfügbarer Fehleraktionstypen aus.
4. Wiederholen Sie die Schritte 2 bis 3, um dieser Gruppe weitere Fehleraktionstypen hinzuzufügen.
5. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.

Vorgehensweise zum Hinzufügen von Gruppen zu einer Fehleraktionstypgruppe

Gehen Sie folgendermaßen vor, um einer Fehleraktionstypgruppe weitere Fehleraktionstypgruppen hinzuzufügen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren einer Fehleraktionstypgruppe" aus.

Oder

Wählen Sie eine bereits vorhandene Instanz einer Fehleraktionstypgruppe aus.

2. Klicken Sie in der Tabelle "Gruppen" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
3. Klicken Sie auf die leere Zeile und wählen Sie eine Fehleraktionstypgruppe aus der Liste von verfügbaren Gruppen aus.
4. Wiederholen Sie die Schritte 2 bis 3, um dieser Gruppe weitere Fehleraktionstypgruppen hinzuzufügen.
5. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
