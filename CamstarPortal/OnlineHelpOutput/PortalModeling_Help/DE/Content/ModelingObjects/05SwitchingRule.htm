Definieren von Umschaltregeln

Die Seite "Umschaltregel" ermöglicht es Ihnen, die Regeln zu definieren, anhand derer entschieden wird, ob ein Stichprobenplan von einer Prüfebene auf eine andere bewegt wird. Umschaltregeln ermöglichen den Wechsel eines Stichprobenplans auf einen reduzierten oder strengeren Inspektionsplan, je nach Anzahl der bestandenen oder durchgefallenen Container in einer Reihe von aufeinanderfolgenden Containern. In einem Stichprobenplan können Sie Umschaltregeln definieren, entweder für den gesamten Plan oder für jeden Stichprobentest. Eine Umschaltregel wird auf Grundlage von X bestandenen oder durchgefallenen Containern auf Y Container oder auf Grundlage eines Ausdrucks angewandt.

Umschaltregeln gelten nicht für Lose.

Beim Definieren von Umschaltregeln

Die folgenden Regeln gelten für das Modellieren einer Umschaltregel:

• Die Felder "Anzahl bestandener (oder durchgefallener) Container", "Anzahl in letztem Containersatz" und "Bestanden/Durchgefallen" sind erforderlich, wenn das Feld "Ausdruck" nicht ausgefüllt ist.
• Das Feld "Ausdruck" ist erforderlich, wenn die Felder "Anzahl bestandener (oder durchgefallener) Container", "Anzahl in letztem Containersatz" und "Bestanden/Durchgefallen" nicht ausgefüllt sind.

Wenn das Umschaltregel-Kriterium erfüllt wird, kann auf Grundlage der Umschaltregel ein Inspektionsplanwechsel automatisch durchgeführt werden oder manuelles Eingreifen erfordern, nachdem eine E-Mail-Benachrichtigung an einen Entscheidungsträger gesendet wurde.

Ähnlich wie die Stichprobenpläne können auch Umschaltregeln eine Maximaldauer haben. Als Teil des Stichprobenplans kann für die Umschaltregel eine Maximaldauer definiert werden. Daher werden bei der Auswertung von Umschaltregeln nur Container einbezogen, die innerhalb des maximalen Dauer-Zeitfensters bearbeitet wurden.

"Umschaltregel" enthält die folgenden, optionalen Modellierungsdefinitionen:

• E-Mail-Nachricht
• E-Mail-Verteiler

"Umschaltregel" enthält die erforderliche Modellierungsdefinition "Prüfebene".

"Umschaltregel" ist ein optionales Feld in der Definition des Stichprobenplans.

Definition

Von Prüfebene

Liste aller Prüfebenen, aus denen Sie die Ausgangsprüfebene für diese Umschaltregel auswählen können. Diese Umschaltregel kann angewandt werden, wenn die aktuelle Prüfebene diesem Wert entspricht.

Zu Prüfebene

Liste aller Prüfebenen, aus denen Sie die resultierende Prüfebene für diese Umschaltregel auswählen können. Dies ist die neue Prüfebene, wenn die Kriterien der Umschaltregel erfüllt werden.

Erforderlich

Anzahl bestandener (oder durchgefallener) Container

Anzahl der Container, die die Prüfung bestanden haben oder durchgefallen sind, in Abhängigkeit vom Wert im Feld "Bestanden/Durchgefallen". Z. B. haben 3 der letzten 4 Container die Prüfung bestanden (oder sind durchgefallen), wenn "3" (dieses Feld) der X-Wert im X-von-Y-Ausdruck ist. Dieses Feld muss eine Integer-Zahl sein.

Hinweis: Es müssen entweder dieses Feld, "Anzahl in letztem Containersatz" und "Bestanden/Durchgefallen" ausgefüllt werden, oder es darf nur das Feld "Ausdruck" ausgefüllt werden.

Optional

Anzahl in letztem Containersatz

Anzahl der Container im letzten Set, die verwendet werden, um die Bestanden/Durchgefallen-Rate zu bestimmen, je nach Wert im Feld "Bestanden/Durchgefallen". Z. B. haben 3 der letzten 4 Container die Prüfung bestanden (oder sind durchgefallen), wenn "4" (dieses Feld) der Y-Wert im X-von-Y-Ausdruck ist. Dieses Feld muss eine Integer-Zahl sein.

Hinweis: Es müssen entweder dieses Feld, "Anzahl bestandener (oder durchgefallener) Container" und "Bestanden/Durchgefallen" ausgefüllt werden, oder es darf nur das Feld "Ausdruck" ausgefüllt werden.

Optional

Bestanden/Durchgefallen

Kontrollkästchen, das angibt, ob das Feld "Anzahl bestandener (oder durchgefallener) Container" ein Zähler für erfolgreiche oder fehlgeschlagene Stichprobentests ist.

Hinweis: Es müssen entweder dieses Feld, "Anzahl bestandener (oder durchgefallener) Container" und "Anzahl in letztem Containersatz" ausgefüllt werden, oder es darf nur das Feld "Ausdruck" ausgefüllt werden.

Optional

Ausdruck

Einheitlicher Ausdruck, der anstelle der Felder Anzahl bestandener (oder durchgefallener) Container und Anzahl in letztem Containersatz verwendet werden kann, um zu bestimmen, ob die Umschaltregel ausgelöst wird. Der Ausdruck muss in einem booleschen Wert (wahr/falsch) resultieren. Weitere Informationen finden Sie im Designer-Benutzerhandbuch.

Hinweis: Es müssen entweder die Felder "Anzahl bestandener (oder durchgefallener) Container", "Anzahl in letztem Containersatz" und "Bestanden/Durchgefallen" ausgefüllt werden, oder es darf nur das Feld "Ausdruck" ausgefüllt werden.

Optional

Automatisches Umschalten erlauben

Kontrollkästchen, das angibt, ob die Prüfebene automatisch auf die "Zu Prüfebene" aktualisiert wird, wenn die Kriterien der Umschaltregel erfüllt werden.

Optional

Benachrichtigung notwendig

Kontrollkästchen, das angibt, ob eine E-Mail-Benachrichtigung gesendet wird, wenn die Kriterien der Umschaltregel erfüllt werden.

Optional

E-Mail-Verteiler

E-Mail-Verteilerliste für diese Umschaltregel, falls sie sich von der Standardliste im Abschnitt "In Verarbeitung" unterscheidet.

Optional

Vorgehensweise zum Definieren einer Umschaltregel

Gehen Sie folgendermaßen vor, um eine Umschaltregel zu definieren:

1. Öffnen Sie die Seite Umschaltregel. Die Seite "Umschaltregel" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie den Namen der Umschaltregel in das Feld Umschaltregel ein.
4. Geben Sie die Revision dieser Umschaltregel in das Feld Revision ein.
5. Führen Sie die Schritte unter "Vorgehensweise zum Definieren von Umschaltregeldetails" aus.
6. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
7. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Definieren von Umschaltregeldetails

Gehen Sie folgendermaßen vor, um Umschaltregeldetails zu definieren:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren einer Umschaltregel" durch.
2. Erweitern Sie den Abschnitt Details.
3. Klicken Sie in der Tabelle "Umschaltregeldetails" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
4. Wählen Sie die Prüfebene aus der Liste im Feld Von Prüfebene aus, die die Ausgangsebene für diese Umschaltregel sein soll.
5. Wählen Sie die Prüfebene, zu der gewechselt werden soll, wenn die Kriterien der Umschaltregel erfüllt werden, aus der Liste im Feld Zu Prüfebene aus.
6. Definieren Sie die Bedingungen der Umschaltregel, indem Sie die Komponenten oder einen Ausdruck folgendermaßen definieren.

Die Komponenten
a. Geben Sie in das Feld Anzahl bestandener (oder durchgefallener) Container den X-Wert des Ausdrucks Wenn X von Y Containern bestehen/durchfallen, wird die Umschaltregel aktiviert ein. "Bestanden" bzw. "Durchgefallen" wird in Schritt c angegeben.
b. Geben Sie in das Feld Anzahl in letztem Containersatz den Y-Wert des Ausdrucks Wenn X von Y Containern bestehen/durchfallen, wird die Umschaltregel aktiviert ein. "Bestanden" bzw. "Durchgefallen" wird in Schritt c angegeben.
c. Wählen Sie aus der Liste im Feld Bestanden/Durchgefallen die Option Bestanden oder Durchgefallen aus, je nachdem, ob der Zähler in den Schritten a und b erfolgreiche oder fehlgeschlagene Stichprobentests zählt.
d. Gehen Sie zu Schritt 7.

Einen Ausdruck
a. Geben Sie in das Feld Ausdruck den einheitlichen Ausdruck ein, der die vorhergehenden Schritte ersetzen kann, um zu bestimmen, ob die Umschaltregel ausgelöst wird.
b. Gehen Sie zu Schritt 7.
7. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
8. Wiederholen Sie die Schritte 3 bis 7, um weitere Umschaltregeldetails zu definieren.
9. Klicken Sie auf OK. Das Popup "Hinzufügen" wird geschlossen und die Anwendung zeigt die Seite "Umschaltregel" an.
10. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
