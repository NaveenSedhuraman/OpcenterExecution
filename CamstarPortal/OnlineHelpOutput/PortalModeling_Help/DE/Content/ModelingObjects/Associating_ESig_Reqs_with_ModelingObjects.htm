Zuweisen von Anforderungen zur elektronischen Unterschrift zu Modellierungsobjekten

Sie müssen einem Modellierungsobjekt eine Anforderung zur elektronischen Unterschrift zuweisen, um eine Authentifizierung mittels elektronischer Unterschrift für eine Fertigungs- oder Qualitätstransaktion anzufordern. Diese Seiten für Modellierungsobjekte enthalten eine "Elektr. Unterschrift: Transaktionszuordnung", die es Ihnen ermöglicht, die Anforderung zur elektronischen Unterschrift sowie die Transaktion auszuwählen, für die Unterschriften erfasst werden müssen:

• Fabrik – Eine beliebige Division, Abteilung oder Gruppe, die in ihrer Buchhaltung und in ihrem Berichtswesen eigenständig ist. Ein Unternehmen kann eine oder mehrere Fabriken enthalten.
• Spez. – Definiert die Aktivitäten, die an einem Schritt durchgeführt werden, und wird von einem Schritt in einem Arbeitsplan referenziert.
• Organisation – Eine Wirtschaftseinheit, die für Verarbeitungs- und Berichtszwecke abgetrennt wurde. Eine Organisation ist weder einem Unternehmen noch einer Fabrik zugeordnet.

Die Seiten "Fabrik" und "Spez." enthalten eine "Elektr. Unterschrift: Transaktionszuordnung", die es Ihnen ermöglicht, Anforderungen zur elektronischen Unterschrift für Fertigungstransaktionen auszuwählen. Die Seite "Organisation" enthält dagegen eine "Elektr. Unterschrift: Transaktionszuordnung", die es Ihnen ermöglicht, Anforderungen zur elektronischen Unterschrift für Ereignisaufzeichnungstransaktionen auszuwählen.

Wenn am Modellierungsobjekt Fertigungs- oder Ereignisdatensatztransaktionen durchgeführt werden, wird die angegebene "Elektr. Unterschrift: Transaktionszuordnung"-Definition verwendet, um zu ermitteln, ob eine elektronische Unterschrift für diese Transaktion erforderlich ist. Wenn Sie z. B. möchten, dass der Vorarbeiter mit einer elektronischen Unterschrift bestätigt, dass Schachteln korrekt verpackt und versiegelt wurden, bevor sie aus dem Verpackungsschritt abgemeldet werden, würden Sie der Abmeldetransaktion für die Verpackungsspezifikation eine Anforderung zur elektronischen Unterschrift zuweisen.

Hinweis: Elektronische Unterschriften können allerdings nicht für die Rückgängigmachung von Containertransaktionen erfasst werden, da für diesen Dienst kein eigener Verlauf aufgezeichnet wird.

In den vorhergehenden Beispielen wurde eine Anforderung zur elektronischen Unterschrift zum Überprüfen von Schachteln erstellt, damit Schachtelkontrolleure bestätigen können, dass die Schachteln korrekt verpackt und versiegelt wurden. Damit das Popup für elektronische Unterschriften während des Fertigungsdienstes angezeigt werden kann, sollten Sie der Verpackungsspezifikation diese "Elektr. Unterschrift: Transaktionszuordnung" zuweisen:

Transaktions-ID: Abmelden (Move Std)

Elektr. Unterschrift: Anforderung: Schachtelüberprüfung

Diese Zuweisung führt dazu, dass ein Popup für elektronische Unterschriften angezeigt wird, wenn im Verpackungsschritt die Abmeldetransaktion initiiert wird. Die Seite erfordert, dass eine Person, die der Verifizierungsrollengruppe zugeordnet ist, eine Unterschrift leistet, um zu bestätigen, dass die Schachtel korrekt verpackt und versiegelt wurde.

Beim Zuweisen von Anforderungen zur elektronischen Unterschrift zu Modellierungsobjekten

Bevor Sie die Modellierungsobjekte definieren, denen Sie Anforderungen zur elektronischen Unterschrift zuweisen möchten, stellen Sie sicher, dass Sie Ihr grundlegendes Informationsmodell definiert haben.

Für Spezifikationen umfasst dies:

• Produkte und deren Produktfamilien
• Operationen
• Containerlevels

Wenn Sie eine Anforderung zur elektronischen Unterschrift erstellen, müssen Sie diese auf der Seite "Spez." einer entsprechenden Operation zuordnen.

Organisationen und Fabriken erfordern keine weiteren Modellierungsobjekte, enthalten jedoch optionale Modellierungsobjekte. Für Organisationen umfassen sie:

• Rollen
• Mitarbeiter
• Einteilungsspezifikationen
• Fehlermodusgruppen
• Nummerierungsregeln
• UI-Einstellungen
• SMTP-Transporte

Für Fabriken umfassen diese:

• Unternehmen
• Produktionskalender
• Erledigungsregel
• Trainingsanforderungsgruppe
• Druckwarteschlange
• Nummerierungsregel

Dieser Abschnitt enthält die grundlegenden Schritte zum Einrichten von elektronischen Unterschriften für Modellierungsobjekte. Weitere Informationen finden Sie unter den Themen zum Definieren der oben aufgeführten Modellierungsobjekte.

Beim Festlegen von Anforderungen zur elektronischen Unterschrift für Ereignisaufzeichnungstransaktionen

Wenn Sie die Ereignisaufzeichnungstransaktionen festlegen, für die Sie elektronische Unterschriften erfassen möchten, müssen Sie nur diesen Qualitätsdienst zur "Elektr. Unterschrift: Transaktionszuordnung" des Modellierungsobjekts der Organisation hinzufügen. Wenn Sie z. B. eine elektronische Unterschrift erfassen möchten, wenn ein Benutzer die Genehmigung für eine Qualitätsdatensatzlösung abbricht, müssen Sie den Qualitätsservice "Genehmigungsabbruchformular" zur "Elektr. Unterschrift: Transaktionszuordnung" hinzufügen.

Ereigniserstellung ist die Ausnahme. Wenn Sie möchten, dass bei Ereignisaufzeichnungs-Seitenfolgen ebenfalls elektronische Unterschriften eingegeben werden müssen, müssen Sie nicht nur den "Ereignis erstellen"-Dienst in der "Elektr. Unterschrift: Transaktionszuordnung", sondern auch den "Aktualisierungsereignis"-Dienst hinzufügen.

Felder für elektronische Unterschriften auf Modellierungsobjektseiten

Verwenden Sie die "Elektr. Unterschrift: Transaktionszuordnung"-Tabellen auf den Modellierungsobjektseiten, um einem Modellierungsobjekt eine Anforderung zur elektronischen Unterschrift zuzuweisen.

Vorgehensweise zum Verknüpfen einer Anforderung zur elektronischen Unterschrift mit einer Fabrik oder Spezifikation

Gehen Sie folgendermaßen vor, um einer Fabrik oder Spezifikation eine Anforderung zur elektronischen Unterschrift zuzuordnen:

1. Öffnen Sie die Seite Fabrik oder Spez. Die entsprechende Seite wird innerhalb der Seite "Modellierung" angezeigt.
2. Wählen Sie eine vorhandene Instanz einer Fabrik oder Spezifikation aus. Die Seite wird aktualisiert und zeigt die Informationen für die ausgewählte Fabrik oder Spezifikation an.
3. Erweitern Sie den Abschnitt Transaktionen.
4. Klicken Sie in der Tabelle "Elektr. Unterschrift: Transaktionszuordnung" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
5. Wählen Sie über das Feld Transaktion die Transaktion aus, für die Sie elektronische Unterschriften aktivieren möchten.

Oder

Klicken Sie auf das Kontrollkästchen Alle Transaktionen, wenn Sie die Funktionalität von elektronischen Unterschriften für alle Transaktionen für diese Spezifikation aktivieren möchten.

6. Wählen Sie die Anforderung zur elektronischen Unterschrift im Feld Elektr. Unterschrift: Anforderung aus.
7. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.

Vorgehensweise zum Verknüpfen einer Anforderung zur elektronischen Unterschrift mit einer Organisation

Gehen Sie folgendermaßen vor, um einer Organisation eine Anforderung zur elektronischen Unterschrift zuzuweisen:

1. Öffnen Sie die Seite Organisation. Die Seite "Organisation" wird innerhalb der Seite "Modellierung" angezeigt.
2. Wählen Sie eine Organisationsinstanz aus. Die Seite wird aktualisiert und zeigt Informationen zur ausgewählten Organisation an.
3. Möchten Sie eine Anforderung zur elektronischen Unterschrift für alle Ereignisaufzeichnungstransaktionen oder nur für ausgewählte Ereignisaufzeichnungstransaktionen festlegen?

Sie möchten eine Anforderung zur elektr. Unterschrift festlegen für . . .

Dann . . .

Alle Ereignisaufzeichnungstransaktionen
a. Klicken Sie auf das Kontrollkästchen Elektr. Unterschrift auf alle Transaktionen anwenden.
b. Gehen Sie zu Schritt 4.

Bestimmte Ereignisaufzeichnungstransaktionen
a. Klicken Sie in der Tabelle "Elektr. Unterschrift: Transaktionszuordnung" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
b. Wählen Sie eine Transaktion aus der Liste Qualitätsservice aus.
c. Wiederholen Sie die Schritte a-b, um zusätzliche Qualitätsservices hinzuzufügen.
d. Gehen Sie zu Schritt 4.
4. Aktivieren Sie das Kontrollkästchen Benutzer-ID-Eingabe erforderlich, wenn Sie möchten, dass sowohl das Kennwort als auch der Benutzername des Mitarbeiters für die Verifizierung der elektronischen Unterschrift erforderlich sind. (Lassen Sie das Kontrollkästchen leer, wenn nur das Passwort erforderlich sein soll.)
5. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
