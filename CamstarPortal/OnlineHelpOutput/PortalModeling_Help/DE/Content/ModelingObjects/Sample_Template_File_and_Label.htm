Arbeiten mit einer Binärdatei

Wenn Ihre Druckersoftware für Barcode-Etiketten, z. B. BarTender®, eine Binärdatei erzeugt, können Sie eine Vorlagenausgabedatei im ZPL-Format (Zebra Print Language) erstellen.

Die Software ermöglicht die Verfolgung einer Ausgabedatei über den druckerspezifischen Treiber. Dazu werden die Eigenschaften des Zebra-Druckers so geändert, dass der Druckercode über eine Protokollierungsoption aufgezeichnet wird. Das Ergebnis ist eine .prn-Datei, die bei Verwendung des Zebra-Druckertreibers das erforderliche ZPL-Dateiformat liefert.

Sehen Sie in den Anleitungen des Herstellers nach, um genauere Informationen zur Barcode-Druckersoftware zu finden.
