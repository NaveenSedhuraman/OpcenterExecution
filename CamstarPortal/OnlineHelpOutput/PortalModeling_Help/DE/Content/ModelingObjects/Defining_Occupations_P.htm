Definieren von Tätigkeiten

Das Modellierungsobjekt "Tätigkeit" wird verwendet, um eine bestimmte Tätigkeit – z. B. die eines Inspektors – zu definieren, die bei der Eingabe einer Reklamation verwendet werden soll.

Hinweis: Die Implementierung von Opcenter EX MDD oder Opcenter EX CR verwendet dieses Objekt nicht. Das Objekt steht Ihnen jedoch standardmäßig in Ihrer benutzerdefinierten Implementierung zur Verfügung.

Siehe auch "Gemeinsame Felder auf Modellierungsseiten". Dieser Abschnitt enthält Informationen zu den Feldern, die alle Modellierungsobjekte aufweisen.

Vorgehensweise zum Definieren einer Tätigkeit

Gehen Sie folgendermaßen vor, um eine Tätigkeit zu definieren:

1. Öffnen Sie die Seite Tätigkeit. Die Seite "Tätigkeit" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie den Namen der Tätigkeit in das Feld Tätigkeit ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Unter "Gemeinsame Felder auf Modellierungsseiten" finden Sie Informationen zu den optionalen Feldern.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
