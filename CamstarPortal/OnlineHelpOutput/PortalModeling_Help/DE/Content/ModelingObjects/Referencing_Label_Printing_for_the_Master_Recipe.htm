Referenzieren des Etikettendrucks im Stammrezept

Das revisionierte Datenobjekt "Stammrezept" enthält ein Drucketikettendefinitionsfeld. In diesem Feld können Sie eine Standard-Drucketikettendefinition für beliebige Zielmaterialien festlegen, die für das Stammrezept gestartet werden. Das Feld liefert eine Liste aller Drucketikettendefinitionen, die in der Modellierung erstellt wurden.

Beim Referenzieren des Etikettendrucks im Stammrezept

Sie müssen ein neues Stammrezept erstellen oder ein bereits vorhandenes auswählen, damit Sie eine Drucketikettendefinition hinzufügen können.

Vorgehensweise zum Referenzieren einer Drucketikettendefinition im Stammrezept

Gehen Sie folgendermaßen vor, um eine Drucketikettendefinition im Stammrezept zu referenzieren:

1. Öffnen Sie die Seite Stammrezept. Die Seite "Stammrezept" wird innerhalb der Seite "Modellierung" angezeigt.
2. Wählen Sie ein vorhandenes Stammrezept aus.

Oder

Erstellen Sie ein neues Stammrezept, indem Sie die Schritte unter "Vorgehensweise zum Definieren eines Stammrezepts" ausführen.

3. Erweitern Sie den Abschnitt Details.
4. Wählen Sie eine Drucketikettendefinition aus der Liste aus.
5. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
