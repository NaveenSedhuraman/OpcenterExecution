Definieren von E-Mail-Verteilern

Das Modellierungsobjekt "E-Mail-Verteiler" wird verwendet, um die Empfänger von E-Mail-Benachrichtigungen anzugeben.

Sie können Empfänger aus einer oder mehreren der folgenden Kategorien konfigurieren:

• Externe Empfänger
• Rollenempfänger
• Mitarbeiter-Empfänger

Beim Definieren von E-Mail-Verteilern

Mitarbeiter-Empfänger werden aus der Modellierungsdefinition "Mitarbeiter" abgerufen. Für einen Mitarbeiter muss eine E-Mail-Adresse im Mitarbeiter-Objekt definiert sein, bevor dieser Mitarbeiter in der Liste "Mitarbeiter-Empfänger" angezeigt wird.

Rollenempfänger werden aus der Modellierungsdefinition "Rolle" abgerufen.

Die Anwendung überprüft externe Empfänger nicht.

"E-Mail-Verteiler" ist ein optionales Feld in der Organisationsdefinition.

Vorgehensweise zum Definieren eines E-Mail-Verteilers

Gehen Sie folgendermaßen vor, um einen E-Mail-Verteiler zu definieren:

1. Öffnen Sie die Seite E-Mail-Verteiler. Die Seite "E-Mail-Verteiler" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für diese E-Mail-Verteilerdefinition in das Feld E-Mail-Verteiler ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Geben Sie eine E-Mail-Adresse in dieses Feld ein, wenn Sie einen externen Empfänger definieren möchten.
6. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Hinzufügen von Empfängern zu einem E-Mail-Verteiler

Gehen Sie folgendermaßen vor, um Empfänger einem E-Mail-Verteiler hinzuzufügen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren eines E-Mail-Verteilers" aus.

Oder

Wählen Sie eine vorhandene Instanz eines E-Mail-Verteilers aus.

2. Klicken Sie auf Neue Zeile hinzufügen in einer der folgenden Tabellen:
• Externe Empfänger
• Rollenempfänger
• Mitarbeiter-Empfänger
3. Möchten Sie einen externen Empfänger, einen Rollenempfänger oder einen Mitarbeiter-Empfänger hinzufügen?

Wenn Sie Folgendes hinzufügen möchten . . .

Dann . . .

Einen externen Empfänger.
a. Geben Sie eine E-Mail-Adresse in die leere Zeile ein.
b. Gehen Sie zu Schritt 4.

Einen Mitarbeiter-Empfänger.
a. Wählen Sie einen Mitarbeiter-Empfänger aus der Liste aus.
b. Gehen Sie zu Schritt 4.

Einen Rollenempfänger.
a. Wählen Sie einen Rollenempfänger aus der Liste aus.
b. Gehen Sie zu Schritt 4.
4. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
