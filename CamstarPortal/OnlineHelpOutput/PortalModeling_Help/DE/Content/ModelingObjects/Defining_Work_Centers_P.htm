Definieren von Arbeitsbereichen

Ein Arbeitsbereich ist ein physischer Bereich oder eine logische Gruppierung, in der ähnliche Arbeiten durchgeführt werden, wie z. B. Verpacken oder Qualitätskontrollen. In jeder Operationsdefinition werden Operationen Arbeitsbereichen zugewiesen. Ein Arbeitsbereich kann, muss aber keine Operationen beinhalten.

Der Arbeitsbereich ist die Komponente, über die die Ablaufplanung realisiert wird. Eine Ablaufplanung kann entweder mitarbeiter-orientiert oder ressourcen-orientiert erfolgen. Ein Arbeitsbereich kann mehrere Operationen umfassen, wenn sich die Operationen Mitarbeiter, Ressourcen oder beides teilen.

Wenn z. B. eine Operation mit der Bezeichnung Mahlen und eine Operation mit der Bezeichnung Polieren die gleichen Ressourcen verwenden, sind sie deshalb für Planungszwecke demselben Arbeitsbereich zugeordnet. Es kann auch eine Operation pro Arbeitsbereich geben.

Beim Definieren eines Arbeitsbereichs

Der Arbeitsbereich enthält die optionalen Modellierungsdefinitionen "Erledigungsregel", "Produktionskalender", "Ressourcengruppe" und "Trainingsanforderungsgruppe".

"Arbeitsbereich" ist ein optionales Feld in der Definition eines Mitarbeiters und einer Operation.

Der Portal-Benutzer kann einen Arbeitsbereich als "Erprobungsbereich" definieren, wenn er ein Ereignis erstellt.

Hinweis: Ein Arbeitsbereich kann zugehörige Umlaufbestandsnachrichten haben. Weitere Informationen finden Sie unter "Definieren von Umlaufbestandsnachrichten" sowie im Opcenter Execution Medical Device and Diagnostics Shop Floor-Benutzerhandbuch und im Opcenter Execution Core Shop Floor-Benutzerhandbuch.

Felddefinitionen der Seite "Arbeitsbereich"

In dieser Tabelle sind die Felder definiert, die nur auf der Seite "Arbeitsbereich" zu finden sind.

Siehe auch "Gemeinsame Felder auf Modellierungsseiten". Dieser Abschnitt enthält Informationen zu den Feldern, die alle Modellierungsobjekte aufweisen.

Feld

Definition

Typ

Allgemein

Produktionskalender

Kalender, der verwendet wird, um den Zeitstempel des Ausführungszeitpunkts einer Transaktion in ein bestimmtes Fertigungsdatum und eine bestimmte Schicht zu übersetzen.

Optional

Ressourcengruppe

Die allgemeine Gruppe, zu der die Ressource gehört.

Optional

Erledigungsregel

Liste von Erledigungsregelinstanzen. Verwenden Sie dieses Feld, um eine bestimmte Erledigungsregel für den Arbeitsbereich zu referenzieren.

Optional

Trainingsanforderungsgruppe

Die Trainingsgruppe, die benötigt wird, damit ein diesem Arbeitsbereich zugeordneter Benutzer autorisiert wird, eine bestimmte Aufgabe auszuführen.

Optional

Normale Wartezeit

Die Dauer (in Stunden), die eine Arbeit normalerweise in einem Arbeitsbereich auf den Beginn der Bearbeitung wartet.

Optional

Kurze Wartezeit

Die Mindestdauer (in Stunden), die eine Arbeit in diesem Arbeitsbereich auf den Beginn der Bearbeitung wartet.

Optional

Vorgehensweise zum Definieren eines Arbeitsbereichs

Gehen Sie folgendermaßen vor, um einen Arbeitsbereich zu definieren:

1. Öffnen Sie die Seite Arbeitsbereich. Die Seite "Arbeitsbereich" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie den Namen dieses Arbeitsbereichs in das Feld Arbeitsbereich ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
