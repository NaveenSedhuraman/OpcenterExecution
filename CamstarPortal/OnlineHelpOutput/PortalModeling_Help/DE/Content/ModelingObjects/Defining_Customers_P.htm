Definieren von Kunden

Ein Kunde ist ein Objekt, dem Waren und Dienste geliefert werden. In der Regel ist der Kunde eine andere Firma, kann jedoch auch eine andere Fabrik oder Abteilung innerhalb des Unternehmens sein. Produktdefinitionen können einen bestimmten Kunden bestimmen, für den das Produkt hergestellt wird.

Beim Definieren eines Kunden

"Kunde" ist ein optionales Feld in der Definition von "Produkt" und "Versandziel" sowie in den Transaktionen "Start" und "Containerwartung".

Hinweis: Eine Kundendefinition kann zugehörige Umlaufbestandsnachrichten haben, wenn sie mit einem Feld aus einer Containerdefinition verbunden ist. Weitere Informationen finden Sie unter "Definieren von Umlaufbestandsnachrichten" sowie im Opcenter Execution Medical Device and Diagnostics Shop Floor-Benutzerhandbuch und im Opcenter Execution Core Shop Floor-Benutzerhandbuch.

Vorgehensweise zum Definieren eines Kunden

Gehen Sie folgendermaßen vor, um einen Kunden zu definieren:

1. Öffnen Sie die Seite Kunde. Die Kunden-Seite wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Kundennamen in das Feld Kunde ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Hinzufügen von Kundenkontakten

Gehen Sie folgendermaßen vor, um einem Kunden einen Kontakt hinzuzufügen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren eines Kunden" aus.

Oder

Wählen Sie eine vorhandene Kunden-Instanz aus.

2. Erweitern Sie den Abschnitt Kontakte.
3. Klicken Sie in der Tabelle "Kundenkontakte" auf Neue Zeile hinzufügen. Das Popup "Kontaktdetail" wird angezeigt.
4. Geben Sie den vollen Namen des Kundenkontakts in das Feld Voller Name ein.
5. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
6. Klicken Sie auf OK. Der Kontakt wird in der Tabelle "Kundenkontakte" angezeigt.
7. Wiederholen Sie die Schritte 3 bis 6, um weitere Kontakte hinzuzufügen.
8. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
