Definieren von Abfragen

Der erste Schritt in der Modellierung der Erledigungsverwaltung ist die Festlegung der Abfragen, auf denen Sie Ihre Regeln aufbauen. (Beachten Sie, dass die Abfrage die Reihenfolge bestimmt, in der die Elemente dem Benutzer in der Fertigung angezeigt werden.) Als Grundlage für die Regel können Sie eine Benutzerabfrage oder eine Designer-Abfrage verwenden.

Beim Definieren einer Benutzerabfrage

Wenn Sie eine Benutzerabfrage definieren, müssen Sie eine SQL-Anweisung in das Feld "Abfragetext" eingeben. Die Anwendung benötigt eine SQL-Anweisung, um Informationen aus der Datenbank abzurufen. Wahlweise können Sie in der Tabelle "Benutzerabfrageparameter" Parameter festlegen, um die Suchergebnisse einzugrenzen. Die Anwendung fordert den Benutzer bei der Ausführung der Abfrage auf, die Parameterinformationen einzugeben.

Hinweis: Nachdem eine Benutzerabfrage definiert wurde, können Sie diese sofort in der Produktion ausführen, ohne die Transaktionsdatenbank zu aktualisieren. Weitere Informationen finden Sie unter "Definieren von Benutzerabfragen".

Designer-Abfragen

Darüber hinaus können Sie benutzerdefinierte Abfragen über Designer erstellen und verwalten. Informationen zum Erstellen von benutzerdefinierten Abfragen finden Sie im Opcenter Execution Medical Device und Diagnostics Designer-Benutzerhandbuch und im Opcenter Execution Core Designer-Benutzerhandbuch.
