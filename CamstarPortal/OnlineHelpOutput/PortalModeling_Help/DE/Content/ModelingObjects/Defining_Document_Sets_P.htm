Definieren von Dokumentsätzen

Ein Dokumentsatz ist eine Sammlung von einem oder mehreren Dokumenten. Auf Dokumentsätze wird von anderen Modellierungsobjekten aus verwiesen; sie enthalten ausführlichere Informationen wie Zeichnungen, gescannte Bilder, Verfahren und Rezepte. Z. B. kann aus einer Einrichtungsdefinition auf einen Dokumentsatz verwiesen werden, der eine technische Zeichnung und Anweisungen zur Einrichtung enthält.

Jede Instanz eines Dokuments (CDO) enthält einen Identifikator für das externe Dokument (etwa einen Dateinamen oder eine URL) und einen Verweis auf das Programm, das zur Anzeige des Dokuments benutzt wird.

Beim Definieren eines Dokumentsatzes

Sie müssen mindestens ein Dokument zu einem Dokumentsatz hinzufügen. Die Anwendung hindert Sie daran, einen Dokumentsatz zu speichern, falls Sie keine Dokumente hinzugefügt haben.

Wählen Sie ein Dokument in der Tabelle "Dokumente" aus und klicken Sie auf die Schaltfläche "Dokument anzeigen", um das ausgewählte Dokument anzuzeigen. Die Anwendung aktiviert die Schaltfläche "Dokument anzeigen", wenn Sie eine Datei auswählen.

Die Anzeige des Dokuments stellt sicher, dass Sie das korrekte Dokument zum Dokumentsatz hinzufügen. Sie sollten Folgendes überprüfen, wenn das Dokument nicht im Browser angezeigt wird:

• Die korrekte Definition der URL für das Dokument-Objekt.
Wichtig: Standardmäßig werden HTTP-URLs von der Anwendung blockiert. Weitere Informationen zum Aktivieren von unsicheren Inhalten an einem sicheren Standort finden Sie in den Browsereinstellungen.
• Sie müssen für den Typ des Dokuments, das Sie öffnen möchten, die entsprechende Software auf Ihrem Computer oder Server installiert haben.
• EWI-Dokumente werden in einem separaten Popup geöffnet, in dem Sie aufgefordert werden, Benutzernamen und Kennwort für Teamcenter einzugeben. Sie müssen Zugriff auf Teamcenter haben, um EWI-Dateien anzuzeigen. Weitere Informationen zum Definieren von EWI-Dokumenten finden Sie unter "Definieren von Dokumenten".

Beachten Sie Folgendes:

• "Dokumentsatz" enthält die erforderliche Modellierungsdefinition "Dokument", was auch "Rezepte" einschließt.
• "Dokumentsatz" ist ein optionales Feld bei der Definition einer Einrichtung und einer Spezifikation.
• Das Popup "Dokumente" wird angezeigt, wenn Sie auf "Neue Zeile hinzufügen" in der Tabelle "Dokumente" klicken.

So wird ein Dokumentsatz definiert

Gehen Sie folgendermaßen vor, um einen Dokumentsatz zu definieren:

1. Öffnen Sie die Seite Dokumentsatz. Die Seite "Dokumentsatz" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für den Dokumentsatz in das Feld Dokumentsatz ein.
4. Klicken Sie in der Tabelle "Dokumente" auf Neue Zeile hinzufügen. Das Popup "Dokumente" wird angezeigt.
5. Geben Sie den Namen des Dokumenteintrags in das Feld Name für Dokumenteneintrag ein.
6. Wählen Sie ein Dokument oder ein Rezept aus dem Feld Dokument aus.
7. Klicken Sie auf OK. Das Popup "Dokumente" wird geschlossen und die angegebenen Informationen werden in der Tabelle "Dokumente" angezeigt.
8. Wiederholen Sie die Schritte 4 bis 7, um weitere Dokumente hinzuzufügen.
9. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
10. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
