Erledigungsregeln – Einführung

Der Erledigungsprozess für Container enthält folgende Aktivitäten, die einer vorab festgelegten Reihenfolge ausgeführt werden:

• Starten der Container in einer vorab auf Grundlage der Produktionsaufträge festgelegten Reihenfolge
• Anmelden und Abmelden von Containern an einer Operation

Sie können in Ihrem Modell Erledigungsregeln festlegen, um die Erledigungslisten zu definieren, die dem Benutzer in der Fertigung angezeigt werden. Dieser orientiert sich in der Folge an diesen Listen, wenn er Container startet oder durch die Fabrik leitet.

Mit Erledigungsregeln werden folgende Aufgaben ausgeführt:

• Sie zeigen eine Liste durchzuführender Arbeiten an
• Optional erzwingen sie die Reihenfolge, in der die Arbeiten durchgeführt werden

Sie müssen diese Modellierungsdefinition erstellen oder ändern, um diesen Prozess zu definieren:

• Erledigungsregeln
• Produktionsaufträge
• Operationen
• Arbeitsbereiche
• Fabriken
