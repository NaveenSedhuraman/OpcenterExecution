Referenzieren von Druckwarteschlangen in Ressourcen, Operationen und Fabriken

Sie ändern die Spezifikationen so, dass sie die Drucketikettendefinitionen referenzieren, und ändern außerdem das Ressourcen-, Operations- oder Fabrikobjekt so, dass es die Druckwarteschlangendefinition referenziert.

Die Druckwarteschlangendefinition wird zur Laufzeit auf folgende Art und Weise abgerufen:

Wenn der Container oder die Transaktion . . .

wird die Druckwarteschlange . . .

einer Ressource zugewiesen ist,

aus der Ressourcendefinition abgerufen.

Ausnahme: In der Transaktion "Start" wird die Ressource aus den Startdetails referenziert.

keiner Ressource zugewiesen ist,

aus der Operationsdefinition abgerufen.

Lesen Sie "Etikettenvorlagen und -Tags", bevor Sie die Instanzen der Ressource oder des Operationsobjekts ändern.

Vorgehensweise zum Referenzieren einer Druckwarteschlangendefinition in einer Ressource

Gehen Sie folgendermaßen vor, um eine Druckwarteschlange in einer Ressource zu referenzieren:

1. Öffnen Sie die Seite Ressource. Die Seite "Ressource" wird innerhalb der Seite "Modellierung" angezeigt.
2. Wählen Sie die Ressource aus, die Sie ändern möchten. Die Anwendung zeigt die Informationen dieser Ressource an.
3. Wählen Sie die Druckwarteschlangendefinition aus der Liste Druckwarteschlange aus.
4. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.

Vorgehensweise zum Referenzieren einer Druckwarteschlangendefinition in einer Operation

Gehen Sie folgendermaßen vor, um eine Druckwarteschlangendefinition in einer Operation zu referenzieren:

1. Öffnen Sie die Seite Operation. Die Seite "Operation" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf die Operation, die Sie ändern möchten. Die Anwendung zeigt die Informationen zu dieser Operation an.
3. Erweitern Sie den Abschnitt Details.
4. Wählen Sie die Druckwarteschlangendefinition aus der Liste Druckwarteschlange aus.
5. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
