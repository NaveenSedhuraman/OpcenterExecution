Definieren von AQL-Levels

Die Seite "AQL-Level" ermöglicht es Ihnen, zulässige Qualitäts-Level zu definieren, indem Sie einem Stichprobengrößencode eine Stichprobengröße und eine Ablehnungsanzahl zuweisen.

Beim Definieren von AQL-Levels

Diese Regeln gelten für das Modellieren von AQL-Leveln:

• Die Seite "AQL-Level" muss mindestens ein Element in der Tabelle "Stichprobengrößendetails" enthalten.
• Die Stichprobengrößencodes für das AQL-Level müssen zu den Stichprobengrößencodes passen, die in den Prüfebenen-Instanzen angegeben sind.

"AQL-Level" ist ein optionales Feld in der Definition des Stichprobenplans.

Vorgehensweise zum Definieren eines AQL-Levels

Gehen Sie folgendermaßen vor, um ein AQL-Level zu definieren:

1. Öffnen Sie die Seite AQL-Level. Die Seite "AQL-Level" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für das AQL-Level in das Feld AQL-Level ein.
4. Klicken Sie in der Tabelle "Stichprobengrößendetails" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
5. Geben Sie einen Stichprobengrößencode in das Feld Stichprobengrößencode in die Zeile ein, die Sie soeben angelegt haben.
6. Geben Sie eine entsprechende Stichprobengröße in das Feld Stichprobengröße ein.
7. Geben Sie die maximale Anzahl von Defekten, die in einer Stichprobe zulässig sind, in das Feld Ablehnungsnummer ein.
8. Wiederholen Sie die Schritte 4 bis 7 für jede Stichprobengröße.
9. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
10. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
