Automatische Abläufe und Prozessberechnungen

Automatische Abläufe und Prozessberechnungen ermöglichen es, Gruppen von Produktionsaufgaben anzuzeigen und nachzuverfolgen. Ein automatischer Ablauf ist ein revisionierbares Objekt, das es Ihnen ermöglicht, eine Sammlung von Aufgaben einer Spezifikation (Spez.) zuzuweisen. Eine Aufgabe ist eine bestimmte Arbeitseinheit, die an einem Container für eine Spezifikation ausgeführt werden muss. Prozessberechnungsaufgaben ermöglichen es Ihnen, benutzerdefinierte Konstanten und detaillierte Berechnungsausdrücke in die Definition von Aufgaben einzufügen.

Ein automatischer Ablauf stellt für jede Aufgabe eine elektronische Aufzeichnung zur Verfügung. Die Möglichkeit, einer Spezifikation automatische Abläufe zuzuweisen, ermöglicht Ihnen die Ausführung mehrerer Arbeitsschritte im Rahmen der Spezifikation. Dabei ist es nicht erforderlich, jede Aufgabe in einer separaten Spezifikation abzubilden, was eine wesentlich größere Zahl von Abmeldetransaktionen nach sich ziehen würde.

Wenn Sie Ihr Modell für automatische Abläufe definieren, gruppieren Sie Aufgaben logisch in Aufgabenlisten ein. Sie legen auch fest, ob die einzelnen Aufgaben erforderlich oder optional sind. Jeder Ihrer automatischen Abläufe kann eine oder mehrere Aufgabenlisten enthalten.
