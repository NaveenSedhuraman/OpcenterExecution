Definieren von Stichprobendatenpunkten

Mit dem Stichprobendatenpunkt-Objekt können Sie die Datenpunkte definieren, die als Teil eines Stichprobentests erforderlich sind. Die Datenpunktdefinition beinhaltet den Datentyp und die Grenzen. Das Stichprobendatenpunkt-Objekt wird nur bei einem gemessenen Stichprobentest verwendet.

Beim Definieren von Stichprobendatenpunkten

Die folgende Regel gilt beim Modellieren von Stichprobendatenpunkten:

• Erforderlich ist entweder das Feld "Oberes Limit" oder das Feld "Unteres Limit".

"Stichprobendatenpunkt" enthält die optionale Modellierungsdefinition "Maßeinheit".

"Stichprobendatenpunkt" ist ein optionales Feld in der Definition des Stichprobentests.

Felddefinitionen der Seite "Stichprobendatenpunkt"

Diese Tabelle definiert die Felder, die nur auf der Seite "Stichprobendatenpunkt" zu finden sind.

Siehe auch "Gemeinsame Felder auf Modellierungsseiten". Dieser Abschnitt enthält Informationen zu den Feldern, die alle Modellierungsobjekte aufweisen.

Definition

Typ

Details

Datentyp

Standardliste, die den Datenpunkttyp definiert. Beispieltypen sind unter anderem "Dezimal", "Festkomma" und "Gleitkomma".

Maßeinheit

Liste aller Maßeinheitscodes. Wenn Sie einen Maßeinheitscode auswählen, wird diese Maßeinheit auf der Transaktionsseite "Stichprobendaten sammeln" angezeigt.

Kontrollkästchen, das angibt, ob die Limits auf der Transaktionsseite "Stichprobendaten sammeln" angezeigt werden.

Optional

Unteres Limit

Minimaler zulässiger Wert für einen Datenpunkt. Ein Datenpunkt unter dieser Grenze führt zu einem Fehler.

Hinweis: Es muss zumindest dieses Feld oder das Feld "Oberes Limit" definiert sein.

Optional

Oberes Limit

Maximaler zulässiger Wert für einen Datenpunkt. Ein Datenpunkt über dieser Grenze führt zu einem Fehler.

Hinweis: Es muss zumindest dieses Feld oder das Feld "Unteres Limit" definiert sein.

Optional

Standardwert

Wenn Sie einen Standarddatenpunktwert eingeben, wird der Wert auf der Transaktionsseite "Stichprobendaten sammeln" im Voraus gefüllt.

Optional

Vorgehensweise zum Definieren eines Stichprobendatenpunkts

Gehen Sie folgendermaßen vor, um einen Stichprobendatenpunkt zu definieren:

1. Öffnen Sie die Seite Stichprobendatenpunkt. Die Seite "Stichprobendatenpunkt" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie den Namen der Datenpunktinstanz in das Feld Stichprobendatenpunkt ein.
4. Wählen Sie einen Datentyp in der Liste aus.
5. Geben Sie einen Wert in das Feld Oberes Limit oder Unteres Limit oder in beide Felder ein, um die oberen oder unteren Werte festzulegen, bei denen ein Fehler gemeldet wird.
6. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
7. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
