Definieren von Fehleraktionstypen

Ein Fehleraktionstyp definiert eine bestimmte Aktion (z. B. "Korrigierend"), die als Reaktion auf eine bestimmte Ursache ausgeführt werden kann. Aktionstypen werden Qualitätsdatensätzen zugeordnet.

Beim Definieren eines Fehleraktionstyps

"Fehleraktionstyp" ist ein optionales Feld in der Modellierungsdefinition "Fehleraktionstypgruppe".

Benutzer können einen oder mehrere Fehleraktionstypen allgemeinen und Produktionsereignisdatensätzen zuordnen.

Siehe auch "Gemeinsame Felder auf Modellierungsseiten". Dieser Abschnitt enthält Informationen zu den Feldern, die alle Modellierungsobjekte aufweisen.

Vorgehensweise zum Definieren eines Fehleraktionstyps

Gehen Sie folgendermaßen vor, um einen Fehleraktionstyp zu definieren:

1. Öffnen Sie die Seite Fehleraktionstyp. Die Seite "Fehleraktionstyp" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für diesen Fehleraktionstyp in das Feld Fehleraktionstyp ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Unter "Gemeinsame Felder auf Modellierungsseiten" finden Sie Informationen zu den optionalen Feldern.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
