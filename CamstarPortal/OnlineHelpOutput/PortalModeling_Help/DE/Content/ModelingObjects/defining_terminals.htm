Definieren von Terminals

Ein Terminal ist ein Fertigungscomputer, der durch eine IP-Adresse eindeutig identifiziert wird. Auf der Seite "Terminal" können Sie ein Terminal mit Einstellungen der terminalbasierten Linienzuordnung definieren und konfigurieren. Einstellungen der terminalbasierten Linienzuordnung überschreiben die Einstellungen der mitarbeiterbasierten Linienzuordnung, während ein Mitarbeiter am konfigurierten Terminal angemeldet ist.

Ein Mitarbeiter, der sich am Terminal anmeldet, kann keine der Einstellungen der terminalbasierten Linienzuordnung ändern, es sei denn, die Fabrik ist so konfiguriert, dass die Option "Überschreiben der terminalbasierten Linienzuordnung erlauben" ausgewählt ist. Mit dieser Option vorgenommene Änderungen gelten nur für die Dauer der aktuellen Sitzung des Mitarbeiters am Terminal. Weitere Informationen finden Sie unter "Definieren einer Fabrik".

Vorgehensweise zum Definieren eines Terminals

Gehen Sie folgendermaßen vor, um ein Terminal zu definieren:

1. Öffnen Sie die Seite Terminal. Die Seite "Terminal" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie in das Feld Terminal einen Namen für das Terminal ein.
4. Geben Sie die IP-Adresse des Terminals ein.
5. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
6. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
