Einführung in Geschäftsregeln

In Geschäftsregeln werden zu erfüllende Bedingungen und eine oder mehrere Aktionen definiert, die durchgeführt werden sollen, falls diese Bedingung zutrifft. Sie können Geschäftsregeln an mehreren Punkten im Qualitätsmanagementprozess zuordnen, um zu steuern, wann bestimmte Aktionen automatisch ausgeführt werden.

Die folgenden Themen enthalten Informationen zum Definieren der Modellierungsobjekte, die für die Umsetzung von Geschäftsregeln erforderlich sind:

• Definieren von Geschäftsregelhandlern
• Definieren von Geschäftsregeln
• Definieren von geplanten Geschäftsregeln
