Einrichten einer Arbeitsplatz- oder Arbeitszellenressource

Mit der Transaktion "Ressourceneinrichtung" können Sie Änderungen an einem Arbeitsplatz oder einer Arbeitszelle aufzeichnen, um deren Verfügbarkeit anzuzeigen. Die Verfügbarkeit einer Ressource bestimmt, ob Sie diese zur Ausführung einer Fertigungstransaktion verwenden können.

Informationen zur Ressourceneinrichtung und Verfügbarkeit finden Sie im Opcenter Execution Medical Device and Diagnostics Shop Floor-Benutzerhandbuch und im Opcenter Execution Core Shop Floor-Benutzerhandbuch. Verwenden Sie die Schritte auf dieser Seite, um einen Arbeitsplatz oder eine Arbeitszelle einzurichten.

Beim Einrichten einer Ressource

Wenn Sie einen Arbeitsplatz oder eine Arbeitszelle als Ressource einrichten, gewährleisten Sie, dass im Verfügbarkeitsfeld "Verfügbar" ausgewählt ist. Obwohl das Verfügbarkeitsfeld optional ist, müssen Sie "Verfügbar" festlegen, damit die Ressource in den entsprechenden Wertelisten für die Ressourcenauswahl angezeigt wird. Wenn Sie nicht "Verfügbar" festlegen, wird die Ressource u. U. nicht als gültige Auswahlmöglichkeit angezeigt, obwohl sie in der Ressourcengruppe definiert ist.

Vorgehensweise zum Einrichten einer Ressource

Gehen Sie folgendermaßen vor, um einen Arbeitsplatz oder eine Arbeitszelle als Ressource einzurichten:

1. Öffnen Sie die Seite Ressourceneinrichtung. Die Seite "Ressourceneinrichtung" wird angezeigt.
2. Wählen Sie die entsprechende Ressourcengruppe aus, um die Anzahl von Ressourcen einzuschränken, die im Feld Ressource angezeigt werden.
3. Wählen Sie im Feld Ressource den Arbeitsplatz oder die Arbeitszelle aus, die Sie einrichten möchten.
4. Wählen Sie einen Ressourcenstatuscode aus, dessen Verfügbarkeit "Verfügbar" lautet. Die Werte werden aktualisiert und im Feld "Verfügbarkeit" angezeigt.
5. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie im Portal Shop Floor-Benutzerhandbuch.
6. Klicken Sie auf Einrichtung. Die Anwendung zeigt eine Erfolgsmeldung an.
