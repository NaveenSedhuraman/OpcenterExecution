Definieren von Ressourcenstatuscodes

Ein Ressourcenstatuscode beschreibt den Status, der einer Ressource zunächst bei der Einrichtung und später bei jeder Statusänderung zugewiesen werden kann. Mögliche Beispiele sind "Verfügbar", "Im Einsatz" und "Nicht verfügbar".

Beim Definieren eines Ressourcenstatuscodes

"Ressourcenstatuscode" ist ein optionales Feld in den Modellierungsdefinitionen "Ressourcenstatuscodegruppe" und "Ressourcenstatusmodell".

"Ressourcenstatuscode" ist ein erforderliches Feld der Transaktion "Ressourceneinrichtung".

Beim Definieren eines Ressourcenstatuscodes für OEE
Hinweis: Die nachstehend beschriebene Funktionalität ist nur verfügbar, wenn der Industry Solutions-Arbeitsbereich installiert ist. Weitere Informationen finden Sie unter "Industry Solutions-Arbeitsbereich".

Zum Ausführen von GAE-Berechnungen muss die Anwendung die Ressourcenstatuscodedaten für jede Schicht zusammenfassen. Die Anwendung fasst die Daten am Ende jeder Schicht mithilfe einer gespeicherten Prozedur zusammen, um die Zeit zu berechnen, die die Ressource jeden Status aufgewiesen hat.

Im Feld "GAE-Verlustkategorie" auf der Seite "Ressourcenstatuscode" können Sie den Typ des Verlusts für jede geplante Ausfallzeit angeben. Operanden schließen ein:

• Verfügbarkeit - Gibt an, dass die Zeit in diesem Ressourcenstatuscode als Verfügbarkeitsverlust für eine Ressource zählt. Die Anwendung schließt die Zeit in die Berechnung der geplanten Ausfallzeit ein.
• Leistung - Gibt an, dass die Zeit in diesem Ressourcenstatuscode als Leistungsverlust für eine Ressource zählt.
• Zeitplan - Gibt an, dass die Zeit in diesem Ressourcenstatuscode als Zeitplanverlust für eine Ressource zählt. Die Anwendung schließt die Zeit in die Berechnung der ungeplanten Ausfallzeit ein.

Vorgehensweise zum Definieren eines Ressourcenstatuscodes

Gehen Sie folgendermaßen vor, um einen Ressourcenstatuscode zu definieren:

1. Öffnen Sie die Seite Ressourcenstatuscode. Die Seite "Ressourcenstatuscode" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für den Statuscode in das Feld Ressourcenstatuscode ein.
4. Wählen Sie die Verfügbarkeit aus, die mit dem Statuscode verknüpft ist.
5. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
6. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
