Definieren von Erledigungsregeln

Erledigungsregeln definieren die Erledigungslisten, die dem Benutzer in der Fertigung angezeigt werden. Dieser orientiert sich in der Folge an diesen Listen, wenn er Container startet oder durch die Fabrik leitet.

Mit Erledigungsregeln werden folgende Aufgaben ausgeführt:

• Anzeigen einer Liste auszuführender Arbeit
• Optionales Erzwingen der Reihenfolge, in der diese Arbeit ausgeführt werden

Felddefinitionen der Seite "Erledigungsregel"

Diese Tabelle definiert die Felder, die nur auf der Seite "Erledigungsregel" zu finden sind.

Siehe auch "Gemeinsame Felder auf Modellierungsseiten". Dieser Abschnitt enthält Informationen zu den Feldern, die alle Modellierungsobjekte aufweisen.

Abfragetyp

Liste der Abfragetypen auf Grundlage, wo die Abfrage definiert wurde:

• "Designer" bedeutet, dass es eine von Siemens über den Designer bereitgestellte Abfrage ist.
• "Benutzer" bedeutet, dass die benutzerdefinierte Abfrage in der Modellierung erstellt wurde.

"Abfragetyp" ist ein schreibgeschütztes Feld, wenn Sie eine vorhandene Instanz einer Erledigungsregel öffnen.

Tabelle "Erledigungsdetails"

Tabelle mit den Transaktionen, die von der Erledigungsregel betroffen sind, und den Details für jede Transaktion.

Optional

Erledigungstyp

Typ der Containertransaktion, die von dieser Erledigungsregel betroffen ist:

• Containeranmeldung – der Erledigungstyp für Anmelden-Transaktionen.
• Containerabmeldung – der Erledigungstyp für Abmelden-Transaktionen (MoveStd und MoveNonStd).
• Automatischer Ablauf – der Erledigungstyp für die Sortierung von Containern auf der Seite "Automatischer Ablauf".
• Auftrag – der Erledigungstyp für Start-Transaktionen auf Grundlage von Produktionsaufträgen.
• Batchreihenfolge – der Erledigungstyp für "Batch starten"-Transaktionen.

Optional

Erledigung durchsetzen

Kontrollkästchen, das angibt, ob die Anwendung die Durchsetzung der Regel für die Fertigungsservices erzwingt.

Beispiel für Dienst starten:

Wenn diese Option ausgewählt ist, kann der Benutzer in der Fertigung Container nur auf Grundlage des Produktionsauftrags am Anfang der Liste erstellen.

Wenn diese Option nicht ausgewählt ist, werden die Aufträge in der Erledigungsliste weiterhin auf Grundlage der Priorität sortiert, Sie können aber Container in beliebiger Reihenfolge in der Liste starten.

Optional

Abfragename

Liste der Abfragen, mit denen die Erledigungsliste gefüllt wird. Die Elemente in der Liste basieren auf dem Wert für "Abfragetyp".

Wenn der Abfragetyp gleich "Designer-Abfrage" ist, zeigt die Liste von Siemens bereitgestellte Abfragenamen an.

Wenn der Abfragetyp gleich "Benutzerabfrage" ist, zeigt die Liste vom Benutzer erstellte Abfragenamen an.

Optional

Vorgehensweise zum Definieren einer Erledigungsregel

Gehen Sie folgendermaßen vor, um eine Erledigungsregel zu definieren:

1. Öffnen Sie die Seite Erledigungsregel. Die Seite "Erledigungsregel" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen in das Feld Erledigungsregel ein.
4. Wählen Sie folgendermaßen einen Wert aus der Liste Abfragetyp aus:
• Wählen Sie Designer-Abfrage aus, wenn Sie die im Designer definierte Standardabfrage verwenden möchten, um die Erledigungsliste zu füllen.
• Wählen Sie Benutzerabfrage aus, wenn Sie eine in der Modellierung definierte Abfrage verwenden möchten, um die Erledigungsliste zu füllen.
5. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
6. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.

So werden Erledigungsdetails hinzugefügt

Gehen Sie folgendermaßen vor, um Erledigungsdetails hinzuzufügen:

1. Führen Sie die Schritte unter "Definieren von Erledigungsregeln" aus.

Oder

Wählen Sie eine vorhandene Instanz einer Erledigungsregel aus.

2. Klicken Sie auf die Schaltfläche Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
3. Wählen Sie einen der folgenden Einträge im Feld Erledigungstyp aus:
• Containeranmeldung, wenn Sie eine Erledigungsliste von Containern für Anmelden-Transaktionen anzeigen möchten.
• Containerabmeldung, wenn Sie eine Erledigungsliste von Containern für Abmelden-Transaktionen anzeigen möchten.
• Automatischer Ablauf, wenn Sie eine Erledigungsliste für Container auf der Seite "Automatischer Ablauf" anzeigen möchten.
• Auftrag, wenn Sie eine Erledigungsliste von Produktionsaufträgen für Start-Transaktionen anzeigen möchten.
• Batchreihenfolge, wenn Sie eine Erledigungsliste von Produktionsaufträgen für "Batch starten"-Transaktionen anzeigen möchten.
4. Aktivieren Sie das Kontrollkästchen Erledigung durchsetzen, wenn Sie Erledigungsregeln durchsetzen möchten. Wenn dieses Kontrollkästchen aktiviert ist, muss der Fertigungsdienst für das erste Element in der Erledigungsliste ausgeführt werden.
5. Wählen Sie die Abfrage aus der Liste Abfragename aus. Diese Abfrage wird verwendet, um die Erledigungsliste zu füllen.
6. Wiederholen Sie die Schritte 2 bis 5, um entsprechend den Anforderungen dieser Erledigungsregel weitere Erledigungstypen hinzuzufügen.
7. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
