Definieren von Behälterstatuscodes

Sie können standardisierte Behälterstatuscodes erstellen, die in Fertigungstransaktionen verwendet und zugewiesen werden können. Die Verwendung von standardisierten Statuscodes ermöglicht es den Bedienern, Entscheidungen in der Fertigung anhand besserer Informationen zu treffen.

Beim Definieren von Behälterstatuscodes

Behälterstatuscodes müssen eindeutig sein. Es folgen Beispiele für Behälterstatuscodes:

• Benutzt
• Sauber
• Benutzt, aber verfügbar

"Behälterstatuscode" ist ein optionales Feld in der Modellierungsdefinition "Ressourcenstatuscodegruppe". Behälterstatuscodes werden von der Modellierungsdefinition "Ressourcenstatus" referenziert, allerdings nur dann, wenn Sie eine Ressourcenstatuscodegruppe angeben, die Behälterstatuscodes enthält.

"Behälterstatuscode" enthält die optionale Modellierungsdefinition "Ressourcenstatusursachengruppe".

Vorgehensweise zum Definieren eines Behälterstatuscodes

Gehen Sie folgendermaßen vor, um einen Behälterstatuscode zu definieren:

1. Öffnen Sie die Seite Behälterstatuscode. Die Seite "Behälterstatuscode" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für den Behälterstatuscode in das Feld Behälterstatuscode ein.
4. Wählen Sie die Verfügbarkeit aus, die diesem Statuscode zugewiesen werden soll.
5. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
6. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
