Definieren von Ressourcenstatuscodegruppen

Eine Ressourcenstatuscodegruppe enthält eine logische Gruppierung von ähnlichen Ressourcenstatuscodes. Die Statuscodes In Reparatur und Geplante Wartung können z. B. in einer Statuscodegruppe namens Reparatur zusammengefasst werden.

Beim Definieren einer Ressourcenstatuscodegruppe

Das Feld "Anfängliche Statuscodegruppe" muss eine Ressourcenstatuscodegruppe enthalten, wenn Sie das Ressourcenstatusmodell definieren.

Sie können den betreffenden Tabellen mehrere Einträge oder Gruppen hinzufügen.

Die Schaltfläche "Aufgelöste Einträge" ermöglicht es Ihnen, eine Liste aller Werte anzuzeigen, die in der Eintragsliste für diese Gruppe und alle darin enthaltenen Gruppen angegeben sind.

Vorgehensweise zum Definieren einer Ressourcenstatuscodegruppe

Gehen Sie folgendermaßen vor, um eine Ressourcenstatuscodegruppe zu definieren:

1. Öffnen Sie die Seite Ressourcenstatuscodegruppe. Die Seite "Ressourcenstatuscodegruppe" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen in das Feld Ressourcenstatuscodegruppe ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Fügen Sie der Gruppe wie folgt Einträge hinzu:
a. Klicken Sie unten in der Tabelle "Einträge" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
b. Klicken Sie in die neue Zeile, um eine Liste von Einträgen anzuzeigen, und wählen Sie einen Eintrag aus, den Sie der Gruppe hinzufügen möchten. Der ausgewählte Eintrag wird der Tabelle hinzugefügt.
c. Wiederholen Sie die Schritte a und b, um der Tabelle einen weiteren Eintrag hinzuzufügen.
6. Fügen Sie der Gruppe wie folgt eine Untergruppe hinzu:
a. Klicken Sie unten in der Tabelle "Gruppen" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
b. Klicken Sie in die neue Zeile, um eine Liste zuvor definierter Gruppen anzuzeigen. Wählen Sie dann eine Gruppe aus, die Sie der Gruppe (als Untergruppe) hinzufügen möchten. Die ausgewählte Gruppe wird der Tabelle hinzugefügt.
c. Wiederholen Sie die Schritte a und b, um der Tabelle eine weitere Gruppe hinzuzufügen.
7. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
