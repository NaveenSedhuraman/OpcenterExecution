Definieren von SPC-Chartgruppen

Eine SPC-Chartgruppe wird verwendet, um eine Sammlung von SPC-Charts und/oder SPC-Chartgruppen festzulegen, je nachdem, ob ein einzelnes Chart oder eine Chartgruppe angezeigt werden soll. Sie werden gleichzeitig verarbeitet. Dieses Objekt ist eine Named Object Group.

Sie können eine SPC-Chartgruppe auf der Seite "Datensammlungsdefinition" oder "Benutzerdatensammlungsdefinition" festlegen, wenn Sie die traditionelle Datensammlungsmethode verwenden. Für eine aufgabenbasierte Datensammlung können Sie eine Datensammlungsdefinition angeben, wenn Sie eine Aufgabe mithilfe des Modellierungsobjekts "Aufgabenliste" festlegen.

Beim Definieren einer SPC-Chartgruppe

"SPC-Chartgruppe" ist ein optionales Feld, wenn Sie Datensammlung und Benutzerdatensammlung definieren. Eine SPC-Chartgruppe kann an eine Datensammlungsdefinition angehängt werden, die wiederum an eine Spezifikation angehängt werden kann.

Die SPC-Chartgruppe enthält die optionale Modellierungsdefinition "SPC-Chart".

Die Schaltfläche "Aufgelöste Einträge" ermöglicht es Ihnen, eine Liste aller Werte anzuzeigen, die in der Eintragsliste für diese Gruppe und alle darin enthaltenen Gruppen angegeben sind.

Informationen zum Einrichten und Konfigurieren von SPC finden Sie im Opcenter Execution Core Statistical Process Control-Referenzhandbuch.

Vorgehensweise zum Definieren einer SPC-Chartgruppe

Gehen Sie folgendermaßen vor, um eine SPC-Chartgruppe zu definieren:

1. Öffnen Sie die Seite SPC-Chartgruppe. Die Seite "SPC-Chartgruppe" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für die neue Chartgruppe in das Feld SPC-Chartgruppe ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Hinzufügen von Einträgen zu einer SPC-Chartgruppe

Gehen Sie folgendermaßen vor, um einer SPC-Chartgruppe Einträge hinzuzufügen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren einer SPC-Chartgruppe" aus.

Oder

Wählen Sie eine bereits vorhandene SPC-Chartgruppeninstanz aus.

2. Klicken Sie in der Tabelle "Einträge" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
3. Klicken Sie in die neue Zeile, um eine Liste von SPC-Charts anzuzeigen, und wählen Sie einen Chart aus, den Sie der Gruppe hinzufügen möchten. Die Anwendung fügt den ausgewählten Eintrag der Tabelle hinzu.
4. Wiederholen Sie die Schritte 2 und 3, um der Tabelle einen weiteren Eintrag hinzuzufügen.
5. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.

Vorgehensweise zum Hinzufügen von Gruppen zu einer SPC-Chartgruppe

Gehen Sie folgendermaßen vor, um einer SPC-Chartgruppe Einträge hinzuzufügen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren einer SPC-Chartgruppe" aus.

Oder

Wählen Sie eine bereits vorhandene SPC-Chartgruppeninstanz aus.

2. Klicken Sie in der Tabelle "Gruppen" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
3. Klicken Sie in die neue Zeile, um eine Liste zuvor definierter Gruppen anzuzeigen. Wählen Sie dann eine Gruppe aus, die Sie der Gruppe (als Untergruppe) hinzufügen möchten. Die Anwendung fügt die ausgewählte Gruppe der Tabelle hinzu.
4. Wiederholen Sie die Schritte 2 und 3, um der Tabelle eine weitere Gruppe hinzuzufügen.
5. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
