Kopieren und Löschen von Umlaufbestandsnachrichten

Manchmal möchten Sie eventuell eine Kopie einer Umlaufbestandsnachricht erstellen. Opcenter EX MDD und Opcenter EX CR bieten die Möglichkeit, die Umlaufbestandsnachricht unter einem neuen oder unter demselben Schlüsseltyp wie das Original zu kopieren.

Vorgehensweise zum Kopieren einer Umlaufbestandsnachricht

Gehen Sie folgendermaßen vor, um eine Umlaufbestandsnachricht zu kopieren:

1. Öffnen Sie eine Instanz eines Modellierungsobjekts, das Umlaufbestandsnachrichten unterstützt. Die Modellierungsobjektseite wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Umlaufbestandsnachrichten. Die Seite "Umlaufbestandsnachrichten" wird angezeigt.
3. Wählen Sie eine vorhandene Umlaufbestandsnachricht im Bereich Umlaufbestandsnachrichtentyp aus. Die Schaltfläche "Kopieren" wird angezeigt.
4. Klicken Sie auf Kopieren. Die Felder für "Neue Umlaufbestandsnachricht erzeugen" werden angezeigt.
5. Wählen Sie einen Schlüssel aus der Liste "Umlaufbestandsnachrichtentyp" aus (standardmäßig ist hier "Alle Schlüssel" ausgewählt). Ein zusätzliches Feld wird angezeigt, wenn Sie "Operationsschlüssel" oder "Etikettenschlüssel" auswählen.
6. Haben Sie "Etikettenschlüssel" oder "Operationsschlüssel" ausgewählt?

Wenn Sie Folgendes ausgewählt haben . . .

Dann . . .

Etikettenschlüssel
a. Geben Sie eine Beschriftung in das Feld "Zu findendes Etikett" ein.
b. Gehen Sie zu Schritt 7.

Operationsschlüssel
a. Wählen Sie eine Operation aus der Liste "Zu findende Operation" aus.
b. Gehen Sie zu Schritt 7.

Alle Schlüssel

Gehen Sie zu Schritt 7.
7. Klicken Sie auf Fortsetzen. Die Felder für "Umlaufbestandsnachrichten" werden angezeigt.
8. Ändern Sie die vorhandene Umlaufbestandsnachricht.

Oder

Geben Sie eine neue Umlaufbestandsnachricht in das Feld Nachrichtentext ein.

9. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
10. Klicken Sie auf Nachricht speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass die Umlaufbestandsnachricht aktualisiert wurde.

Vorgehensweise zum Löschen einer Umlaufbestandsnachricht

Gehen Sie folgendermaßen vor, um eine Umlaufbestandsnachricht zu löschen:

1. Öffnen Sie eine Instanz eines Modellierungsobjekts, das Umlaufbestandsnachrichten unterstützt. Die Modellierungsobjektseite wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Umlaufbestandsnachrichten. Die Seite "Umlaufbestandsnachrichten" wird angezeigt.
3. Wählen Sie eine vorhandene Umlaufbestandsnachricht im Bereich Umlaufbestandsnachrichtentyp aus. Die Schaltfläche "Löschen" wird angezeigt.
4. Klicken Sie auf Löschen. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass die Umlaufbestandsnachricht gelöscht wurde.
