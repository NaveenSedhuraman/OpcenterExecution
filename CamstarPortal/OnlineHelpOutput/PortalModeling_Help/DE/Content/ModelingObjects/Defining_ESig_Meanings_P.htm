Definieren von Bedeutungen für elektronische Unterschriften

Eine Bedeutung für eine elektronische Unterschrift stellt den Zweck und die Verantwortlichkeit einer Unterschrift dar. Sie ermöglicht es Ihnen festzulegen, welcher Prozess überprüft wird, wenn eine elektronische Unterschrift angewandt wird.

Die Seite "Elektr. Unterschrift: Bedeutung" ermöglicht es Ihnen, eine Bedeutung für jede erforderliche Unterschrift zu definieren. Wenn Sie z. B. möchten, dass ein Verpackungsvorarbeiter mit der Rolle "Schachtelüberprüfer" eine elektronische Unterschrift leistet, um zu bestätigen, dass die Schachteln korrekt befüllt wurden, können Sie die Bedeutung "Schachtel überprüft" für die elektronische Unterschrift erstellen. Wenn der Schachtelüberprüfer eine Unterschrift leistet, bedeutet das, dass die Aufgabe "Schachtel überprüfen" ausgeführt wurde.

Beim Definieren einer Bedeutung für eine elektronische Unterschrift

"Elektr. Unterschrift: Bedeutung" ist ein Pflichtfeld in der Modellierungsdefinition "Elektr. Unterschrift: Anforderung".

Siehe auch "Gemeinsame Felder auf Modellierungsseiten". Dieser Abschnitt enthält Informationen zu den Feldern, die alle Modellierungsobjekte aufweisen.

Vorgehensweise zum Definieren einer Bedeutung für eine elektronische Unterschrift

Gehen Sie folgendermaßen vor, um eine Bedeutung für eine elektronische Unterschrift zu definieren:

1. Öffnen Sie die Seite Elektr. Unterschrift: Bedeutung. Die Seite "Elektr. Unterschrift: Bedeutung" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie den Namen der Bedeutung in das Feld Elektr. Unterschrift: Bedeutung ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Unter "Gemeinsame Felder auf Modellierungsseiten" finden Sie Informationen zu den optionalen Feldern.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
