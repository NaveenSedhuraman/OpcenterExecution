Definieren von automatischen Abläufen

Ein automatischer Ablauf ist ein revisionierbares Objekt, das es Ihnen ermöglicht, einer Spezifikation (Spez.) eine Sammlung von Aufgaben zuzuweisen.

Beim Definieren von automatischen Abläufen

Gewährleisten Sie, dass Sie eine oder mehrere Aufgabenlisten definiert haben, bevor Sie einen automatischen Ablauf definieren. Die Anwendung erfordert, dass Sie einem automatischen Ablauf mindestens eine Aufgabenliste zuordnen.

Der Objektname darf keine ungültigen Zeichen enthalten. Z. B. darf kein Apostroph verwendet werden.

Hinweis: Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics Designer-Benutzerhandbuch oder im Opcenter Execution Core Designer-Benutzerhandbuch.

"Automatischer Ablauf" ist ein optionales Feld in den Definitionen "Bedingte Prozessanweisung" und "Spez.".

Felddefinitionen der Seite "Automatischer Ablauf"

Siehe auch "Gemeinsame Felder auf Modellierungsseiten". Dieser Abschnitt enthält Informationen zu den Feldern, die alle Modellierungsobjekte aufweisen.

Einstellung, die angibt, ob die Anwendung die Aufgabenlisten in einer bestimmten Reihenfolge (Sequenziell) oder in beliebiger Reihenfolge (Nicht sequenziell) ausführt. Der Standardwert ist "Sequenziell".

Vorgehensweise zum Definieren eines automatischer Ablaufs

Gehen Sie folgendermaßen vor, um einen automatischen Ablauf zu definieren:

1. Öffnen Sie die Seite Automatischer Ablauf. Die Seite "Automatischer Ablauf" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für den automatischen Ablauf in das Feld Automatischer Ablauf ein.
4. Geben Sie die Revision dieses automatischen Ablaufs in das Feld Revision ein.
5. Wählen Sie einen Ausführungsmodus in der Liste aus.
6. Fügen Sie mithilfe des Verfahrens "So wird eine Aufgabenliste zu einem automatischen Ablauf hinzugefügt" eine oder mehrere Aufgabenlisten zur Tabelle "Aufgabenlisten" hinzu.
7. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
8. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.

So wird eine Aufgabenliste zu einem automatischen Ablauf hinzugefügt

Gehen Sie folgendermaßen vor, um eine Aufgabenliste zu einem automatischen Ablauf hinzuzufügen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren eines automatischer Ablaufs" aus.

Oder

Wählen Sie eine vorhandene Instanz eines automatischen Ablaufs aus.

2. Klicken Sie in der Tabelle "Aufgabenlisten" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
3. Klicken Sie in das Feld, um eine Liste von Aufgabenlisten anzuzeigen.
4. Wählen Sie die Aufgabenliste aus, die diesem automatischen Ablauf zugeordnet werden soll.
5. Wiederholen Sie die Schritte 2 bis 4, um weitere Aufgabenlisten hinzuzufügen.
6. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
