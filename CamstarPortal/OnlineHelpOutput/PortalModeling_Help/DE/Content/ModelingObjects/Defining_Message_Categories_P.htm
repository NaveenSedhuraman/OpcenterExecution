Definieren von Nachrichtenkategorien

Eine Nachrichtenkategorie ist ein Begriff, der verwendet wird, um Nachrichtengruppen zu identifizieren, die in Concierge und in der Nachrichtenzentrale angezeigt werden. Nachrichtenkategorien können als Suchparameter auf der Seite "Nachrichtenzentrale" verwendet werden.

Beim Definieren von Nachrichtenkategorien

Die folgenden Nachrichtenkategorien werden standardmäßig bereitgestellt:

• Meine Aufgaben
• Meine Genehmigungen
• Meine offenen Punkte
• Meine Warnhinweise

Sie müssen eine Nachrichtenkategorie einem oder mehreren Benachrichtigungstypen im Modellierungsobjekt "Portalnachrichtenkategorie" zuweisen, bevor die Anwendung die Nachrichtenkategorie anzeigt. Weitere Informationen finden Sie unter "Definieren von Portalnachrichtenkategorien".

"Nachrichtenkategorie" ist ein erforderliches Feld in der Modellierungsdefinition "Portalnachrichtenkategorie".

Vorgehensweise zum Definieren einer Nachrichtenkategorie

Gehen Sie folgendermaßen vor, um eine Nachrichtenkategorie zu definieren:

1. Öffnen Sie die Seite Nachrichtenkategorie. Die Seite "Nachrichtenkategorie" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für die Nachrichtenkategorie in das Feld Nachrichtenkategorie ein.
4. Geben Sie einen Wert in das Feld Nachrichtenkategoriewert ein.
5. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
6. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
