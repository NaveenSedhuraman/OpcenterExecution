Definieren von Trainingsplänen

Ein Trainingsplan besteht aus einer oder mehreren Trainingsanforderungen und Unter-Trainingsplänen. Kursleiter und Vorgesetzte bestimmen und erstellen gemeinsam einen geeigneten Trainingsplan für einen Mitarbeiter, weisen den Plan dem entsprechenden Mitarbeiter zu und ändern den Plan anschließend nach Bedarf. Pläne werden geändert, wenn neue Anforderungen und neue Unter-Trainingspläne hinzugefügt werden. Der Trainingsplan wird vom Mitarbeiter referenziert.

Hinweis: Die Verantwortlichen für die Verwaltung von Trainingsplänen entfernen die hinzugefügten Trainingsanforderungen eines Plans vermutlich schon deshalb nicht, weil diese vom Plan referenzierten Anforderungen an Gültigkeitsdaten gebunden sind, die überprüft werden, wenn ein Benutzer eine Aktion durchführen möchte, die ein Training voraussetzt.

Beim Definieren von Trainingsplänen

Ein Trainingsplan enthält die folgenden optionalen Modellierungsdefinitionen:

• Trainingsplan als Unterplan
• Trainingsanforderung

Die Anwendung erfordert, dass Sie diese Objekte definieren, um den Trainingsplan zu speichern, wenn Sie der Tabelle "Trainingsplandetail" ein entsprechendes Trainingsdetail hinzufügen. Wenn Sie z. B. eine Trainingsanforderungszeile in die Tabelle einfügen, müssen Sie zunächst eine vorhandene Trainingsanforderung zuordnen, bevor Sie den Trainingsplan speichern können.

Inwieweit Sie die Spaltenwerte in der Tabelle "Trainingsplandetail" bearbeiten können, hängt vom Trainingsdetailtyp der betreffenden Zeile ab. Die Spalte "Trainingsplan" ist nur für Details zu Unter-Trainingsplänen aktiviert und die Spalte "Trainingsanforderung" nur für Trainingsanforderungs-Details.

Hinweis: Sie können über mehrere Details verfügen, welche Trainingsanforderungen und Unter-Trainingsanforderungspläne kombinieren.

Vorgehensweise zum Definieren eines Trainingsplans

Gehen Sie folgendermaßen vor, um einen Trainingsplan zu definieren:

1. Öffnen Sie die Seite Trainingsplan. Die Seite "Trainingsplan" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen in das Feld Trainingsplan ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die besagt, dass der Trainingsplan hinzugefügt wurde.

Vorgehensweise zum Hinzufügen eines Unter-Trainingsplans

Gehen Sie folgendermaßen vor, um einen Unter-Trainingsplan einem Trainingsplan hinzuzufügen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren eines Trainingsplans" aus.
2. Wählen Sie Unter-Trainingsplan im Feld Trainingsdetailtyp aus.
3. Klicken Sie in eine leere Zeile der Tabelle "Trainingsdetailtyp".

Oder

Klicken Sie in der Tabelle "Trainingsplandetail" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.

4. Wählen Sie einen Trainingsplan aus dem Menü Trainingsplan in der Tabelle "Trainingsplandetail" aus.
5. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
6. Klicken Sie außerhalb der Tabelle, um den Trainingsplan hinzuzufügen.
7. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.

Vorgehensweise zum Hinzufügen einer Trainingsanforderung

Gehen Sie folgendermaßen vor, um eine Trainingsanforderung einem Trainingsplan hinzuzufügen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren eines Trainingsplans" aus.
2. Wählen Sie Trainingsanforderung im Feld Trainingsdetailtyp aus.
3. Klicken Sie in eine leere Zeile der Tabelle "Trainingsdetailtyp".

Oder

Klicken Sie in der Tabelle "Trainingsplandetail" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.

4. Wählen Sie eine Trainingsanforderung oder deren Revision aus dem Menü Trainingsanforderung in der Tabelle "Trainingsplandetail" aus.
5. Aktivieren Sie das Kontrollkästchen Ist Standardrevision, falls dies die Standardrevision ist.
6. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
7. Klicken Sie außerhalb der Tabelle, um den Trainingsplan hinzuzufügen.
8. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
