Definieren von Datensammlungsdefinitionen

Eine Datensammlungsdefinition referenziert eine parametrische Datendefinition, die einen Satz von Datensammlungsparametern enthält. Die Datensammlungsparameter werden Benutzern während der Bearbeitung von Transaktionen angezeigt. Die Datensammlungsdefinition und die Transaktion, mit der sie genutzt wird, sind in der Spezifikationsdefinition angegeben, die dem Schritt zugeordnet ist, in dem die Daten erfasst werden.

Serverempfehlungen zur Datensammlung

Für Implementierungen mit hohen Transaktionsvolumina sollten Sie in Erwägung ziehen, einen Server für die Datensammlung außerhalb eines Clusters einzurichten, ähnlich wie in dieser Abbildung.

Bevor Sie irgendeine Instanz einer Datensammlungsdefinition erstellen, müssen Sie mithilfe des Datensammlungs-Assistenten (empfohlen) oder des Designers parametrische Datendefinitionen definieren. Der Name der parametrischen Datendefinition wird anschließend im "Datensammlungstyp"-Feld der Datensammlungsdefinition referenziert.

Beim Definieren einer Datensammlungsdefinition

Wenn Sie eine Datensammlungsdefinition für eine Portal-Fertigungsseite definieren, müssen Sie einen Datensammlungstyp angeben und ein Webpart auswählen. Der Wert, den Sie im Feld "Datensammlungstyp" auswählen, sollte dem ausgewählten Webpart entsprechen.

Beachten Sie Folgendes:

• "Datensammlungsdefinition" enthält die optionale Modellierungsdefinition "SPC-Chartgruppe".
• "Datensammlungsdefinition" ist ein optionales Feld in der Modellierungsdefinition "Spez.".
Hinweis: Eine Datensammlungsdefinition kann zugehörige Umlaufbestandsnachrichten haben, wenn sie mit einem Feld aus einer Containerdefinition verknüpft ist. Weitere Informationen finden Sie unter "Definieren von Umlaufbestandsnachrichten" sowie im Opcenter Execution Medical Device and Diagnostics Portal Shop Floor-Benutzerhandbuch und im Opcenter Execution Core Shop Floor-Benutzerhandbuch.

So wird eine Datensammlung definiert

Gehen Sie folgendermaßen vor, um eine Datensammlungsdefinition zu definieren:

1. Öffnen Sie die Seite Datensammlungsdefinition. Die Seite "Datensammlungsdefinition" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie in das Feld Datensammlungsdefinition den Namen dieser Definition ein.
4. Geben Sie die Revision oder Version dieses Objekts in das Feld Revision ein.
5. Aktivieren Sie das Kontrollkästchen Standardrevision, falls dies die Standardrevision ist.
6. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
7. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
