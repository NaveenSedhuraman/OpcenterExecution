Fertigungsservices und Erledigung von Containern

Für die folgenden Transaktionen (Services) werden Erledigungslisten bereitgestellt:

• Start
• Anmelden (MoveIn)
• Abmelden - standardmäßig (MoveStd)

Menüs für Benutzer in der Fertigung

Siemens stellt Seiten zur Verfügung, über die den Benutzern in der Fertigung Erledigungslisten angezeigt werden, wenn sie die oben aufgeführten Services ausführen. Um diese Seiten zu verwenden, müssen Sie für die Benutzer in der Fertigung Menüs definieren, die auf die in der folgenden Tabelle aufgeführten Dateien verweisen.

Die folgenden Seiten für Fertigungsservices unterstützen Erledigungsregeln.

Zu verwendende Datei

Service

Beschreibung

OrderDispatchVP.xml

Start

Zeigt eine Tabelle mit Produktionsaufträgen an, die die Grundlage zum Starten von Containern ist.

OperationalViewVP.xml und
OperationalViewScanVP.xml

MoveIn

MoveOut (inkl. MoveStd)

Erlaubt es Ihnen, die folgende Transaktion in einem Kombinationsformular auszuführen:

• Anmelden an eines Containers aus der Tabelle "In Warteschlange". Die Tabelle ist vorhanden, wenn Anmelden für die Operation erforderlich ist.
• "Abmelden - standardmäßig" für einen Container aus der Tabelle "In Bearbeitung" (Container werden von der Operation abgemeldet).

MoveStdVP.xml

MoveStd

Erlaubt es Ihnen, die Transaktion "Abmelden - standardmäßig" auszuführen, indem Sie einen Container aus der Tabelle auswählen.

Informationen zum Ausführen von Services finden Sie im Opcenter Execution Medical Device and Diagnostics Shop Floor-Benutzerhandbuch und im Opcenter Execution Core Shop Floor-Benutzerhandbuch.
