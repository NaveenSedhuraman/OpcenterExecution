Definieren von Stücklisten (BOMs) und ERP-Stücklisten

Zu fertigende Produkte werden mithilfe einer nummerierten und wohldefinierten Liste von Rohmaterialien und Unterbaugruppen hergestellt. Diese werden als Materiallisten bezeichnet. Ein Stücklisten-CDO ermöglicht eine Sammlung von Materiallisten. Es gibt zwei Typen von Listenobjekten:

• Stückliste (Bill of Materials, BOM) – Definiert die Materialien (Komponenten und Mengen), die benötigt werden, um ein bestimmtes Produkt (Endartikel oder Unterbaugruppe) herzustellen. BOMs werden von einer Produktdefinition referenziert.
• ERP-Stückliste (Enterprise Resource Planning): Verweist auf Schritte in einer ERP-Route, anstatt auf Schritte in einem Opcenter EX MDD- oder Opcenter EX CR-Arbeitsplan zu verweisen. Es handelt sich hierbei um eine Verbindung zwischen Opcenter EX MDD- oder Opcenter EX CR- und ERP-Systemen.

Beim Definieren einer Stückliste

Eine Stückliste enthält die optionalen Modellierungsdefinitionen "Stücklistentyp", "Produkt" und "Spez."

Stückliste ist ein optionales Feld in der Definition eines Produkts.

Hinweis: Eine Stückliste kann zugehörige Umlaufbestandsnachrichten haben. Weitere Informationen finden Sie unter "Definieren von Umlaufbestandsnachrichten" und im Portal Shop Floor-Benutzerhandbuch.

Die für den Produktionsauftrag oder die Stückliste definierte Materialliste hat Vorrang vor der Materialliste, die für die ERP-Stückliste definiert ist. Weitere Informationen finden Sie unter "Definieren zulässiger Komponentenmengen in der ERP-Stückliste" und "Definieren zulässiger Komponentenmengen im Produktionsauftrag".

Feld "Phantomstückliste"

Das Feld "Phantomstückliste" im Popup für Stücklistendetails ermöglicht es Ihnen, auf eine Phantomstückliste in Ihrer Materialliste zu verweisen. Eine Phantomstückliste ist eine logische Gruppierung von Materialien, die als einzelnes Element auf einer Materialliste aufgeführt ist. Z. B. kann ein Produkt eine bestimmte Komponente enthalten, die ebenfalls von Ihrem Unternehmen hergestellt wird. Diese Komponente hat eine eigene Stückliste. Dennoch möchten Sie die Komponente im Kontext Ihres Hauptprodukts als einzelnes Element auf der Stückliste des Produkts aufführen. In diesem Fall würden Sie mit einer Phantomstückliste auf das Element verweisen.

Wenn Sie eine Phantomstückliste referenzieren, hat das auch Auswirkungen auf die Methode zum Abrufen von Anforderungen. Wenn eine Phantomstückliste keine Spezifikationsdefinition hat, führt das dazu, dass die Logik für Abrufen von Anforderungen die Komponenten auf Grundlage der ihnen zugewiesenen Spezifikationen analysiert.

Vorgehensweise zum Definieren einer Stückliste

Gehen Sie folgendermaßen vor, um eine Stückliste zu definieren:

1. Öffnen Sie die Seite Stückliste. Die Seite "Stückliste" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie den Namen dieser Stückliste in das Feld Stückliste ein.
4. Geben Sie die Revision dieser Stückliste in das Feld Revision ein.
5. Fügen Sie Elemente zur Tabelle "Materialliste" hinzu, indem Sie die Schritte unter "Vorgehensweise zum Hinzufügen von Elementen zur Stücklistenmaterialliste" ausführen.
6. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
7. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Hinzufügen von Elementen zur Stücklistenmaterialliste

Gehen Sie folgendermaßen vor, um ein Element zur Stücklistenmaterialliste hinzuzufügen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren einer Stückliste" aus.

Oder

Wählen Sie eine vorhandene Stücklisteninstanz aus.

2. Expandieren Sie den Abschnitt Materialien.
3. Klicken Sie auf Neue Zeile hinzufügen in der Tabelle Materialliste. Das Popup "Hinzufügen" wird angezeigt.
4. Wählen Sie ein Produkt aus.
5. Wählen Sie eine Verbrauchssteuerung aus.
6. Geben Sie einen Wert in das Feld Erforderliche Menge ein.
7. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
8. Klicken Sie auf OK. Das Popup "Hinzufügen" wird geschlossen und das Materiallistenelement wird in der Tabelle "Materialliste" auf der Seite "Stückliste" angezeigt.
9. Wiederholen Sie die Schritte 3 bis 8, um weitere Elemente hinzuzufügen.
10. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.

Minderverbr. erlauben

Hinweis: Das Kontrollkästchen "Materialverbrauch prüfen" wird für Instanzen des Modellierungsobjekts "Spez." aktiviert oder deaktiviert. Weitere Informationen finden Sie unter Definieren von Spezifikationen.

Optional

Mehrverbrauch erlauben

Kontrollkästchen, das angibt, ob der Schritt "Materialverbrauch prüfen" übersprungen wird, falls der tatsächliche Verbrauch des Produkts, das Sie zur Materialliste hinzufügen, über dem angegebenen Wert für den zugewiesenen Schritt liegt. Diese Option ist standardmäßig aktiviert.

Hinweis: Das Kontrollkästchen "Materialverbrauch prüfen" wird für Instanzen des Modellierungsobjekts "Spez." aktiviert oder deaktiviert. Weitere Informationen finden Sie unter Definieren von Spezifikationen.

Gültig bis

Datum, bis zu dem das Komponentenprodukt gültig ist. Falls dieses Feld leer gelassen wird, bleibt das Komponentenprodukt auf unbestimmte Zeit gültig. Das Datum muss nach dem Datum im Feld "Gültig ab" liegen.

Vorgehensweise zum Definieren einer ERP-Stückliste

Gehen Sie folgendermaßen vor, um eine ERP-Stückliste zu definieren:

1. Öffnen Sie die Seite ERP-Stückliste. Die Seite "ERP-Stückliste" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie den Namen dieser ERP-Stückliste in das Feld ERP-Stückliste ein.
4. Geben Sie die Revision dieser ERP-Stückliste in das Feld Revision ein.
5. Fügen Sie Elemente zur Tabelle "Materialliste" hinzu, indem Sie die Schritte unter "Vorgehensweise zum Hinzufügen von Elementen zur ERP-Stücklistenmaterialliste" ausführen.
6. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
7. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Hinzufügen von Elementen zur ERP-Stücklistenmaterialliste

Gehen Sie folgendermaßen vor, um ein Element zur ERP-Stücklistenmaterialliste hinzuzufügen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren einer ERP-Stückliste" aus.

Oder

Wählen Sie eine vorhandene ERP-Stücklisteninstanz aus.

2. Expandieren Sie den Abschnitt Materialien.
3. Klicken Sie auf Neue Zeile hinzufügen in der Tabelle "Materialliste". Das Popup "Hinzufügen" wird angezeigt.
4. Wählen Sie ein Produkt aus.
5. Wählen Sie eine Verbrauchssteuerung aus.
6. Geben Sie einen Wert in das Feld Erforderliche Menge ein.
7. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
8. Klicken Sie auf OK. Das Popup "Hinzufügen" wird geschlossen und das Materiallistenelement wird in der Tabelle "Materialliste" auf der Seite "ERP-Stückliste" angezeigt.
9. Wiederholen Sie die Schritte 3 bis 8, um weitere Elemente hinzuzufügen.
10. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.

Vorgehensweise zum Verwenden der Felder "Minderverbr. erlauben" und "Mehrverbrauch erlauben"

Sie können die Materialverbrauchsfelder "Minderverbr erlauben" und "Mehrverbrauch erlauben" aktivieren oder deaktivieren, wenn Sie die Materialliste für eine ERP-Stückliste festlegen.

Gehen Sie folgendermaßen vor, um die Prüfungen für ein Produkt auf der ERP-Stücklistenmaterialliste zu aktivieren oder zu deaktivieren:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren einer ERP-Stückliste" aus.

Oder

Wählen Sie eine vorhandene ERP-Stücklisteninstanz aus.

2. Expandieren Sie den Abschnitt Materialien.
3. Klicken Sie auf Neue Zeile hinzufügen in der Tabelle "Materialliste". Das Popup "Hinzufügen" wird angezeigt.
4. Wählen Sie ein Produkt aus.
5. Wählen Sie eine Verbrauchssteuerung aus.
6. Geben Sie einen Wert in das Feld Erforderliche Menge ein.
7. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
8. Aktivieren Sie das Kontrollkästchen Minderverbr. erlauben oder Mehrverbrauch erlauben oder beide.
9. Klicken Sie auf OK. Das Popup "Hinzufügen" wird geschlossen und das Materiallistenelement wird in der Tabelle "Materialliste" auf der Seite "ERP-Stückliste" angezeigt.
10. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
