Definieren von E-Mail-Nachrichten

Das Modellierungsobjekt "E-Mail-Nachricht" wird verwendet, um die Nachrichten für E-Mail-Benachrichtigungen festzulegen. Nach dem Speichern ist die Definition im Modellierungsobjekt "Organisation" verfügbar.

Beim Definieren einer E-Mail-Nachricht

Die gleiche E-Mail-Nachricht kann häufig wiederverwendet werden.

"E-Mail-Nachricht" ist ein optionales Feld, das Sie bei der Definition einer Organisation angeben können. Es ist für die Definition eines Benachrichtigungsereignisses erforderlich.

Vorgehensweise zum Definieren einer E-Mail-Nachricht

Gehen Sie folgendermaßen vor, um eine E-Mail-Nachricht zu definieren:

1. Öffnen Sie die Seite E-Mail-Nachricht. Die Seite "E-Mail-Nachricht" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für diese E-Mail-Nachrichtendefinition in das Feld E-Mail-Nachricht ein.
4. Geben Sie die E-Mail-Adresse des Absenders in das Feld Absender ein.
5. Geben Sie einen Betreff in das Feld Betreff ein.
6. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
7. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
