Definieren von Lieferanten

Ein Lieferant ist ein Anbieter von Waren oder Dienstleistungen wie etwa:

• Komponenten und Baugruppen, die zur Herstellung von Produkten verwendet werden
• Produkte, die zum Wiederverkauf erworben werden
• Nicht-Produktionsmaterialien
• Dienstleistungen wie etwa die Wartung von Ressourcen und Gebäuden

Lieferantendefinitionen können auch eine Liste von Produkten (Artikeln) enthalten, die der Lieferant liefert. Durch die Referenzierung von Lieferantenartikeln aus den Definitionen erworbener Produkte können Sie angeben, von welchem Lieferanten ein Produkt gekauft wird.

Beim Definieren eines Lieferanten

"Lieferant" ist ein optionales Feld in der Definition von Produkten.

So wird ein Lieferant definiert

Gehen Sie folgendermaßen vor, um einen Lieferanten zu definieren:

1. Öffnen Sie die Seite Lieferant. Die Seite "Lieferant" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie den Namen des Lieferanten in das Feld Lieferant ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Hinzufügen von Lieferantenartikeln

Gehen Sie folgendermaßen vor, um Lieferantenartikel hinzuzufügen:

1. Führen Sie die Schritte unter "So wird ein Lieferant definiert" aus.

Oder

Wählen Sie eine vorhandene Lieferanteninstanz aus.

2. Klicken Sie in der Tabelle "Lieferantenartikel" auf Neue Zeile hinzufügen, um einen vom Lieferanten bereitgestellten Artikel hinzuzufügen. Eine neue Zeile wird angezeigt.
3. Geben Sie den Namen des vom Lieferanten gelieferten Elements in das Feld Lieferantenartikelname ein.
4. Geben Sie nach Bedarf Werte für die folgenden optionalen Informationen ein:
• Mindestbestellmenge
• Vorlaufzeit
• Kosten
5. Wiederholen Sie nach Bedarf die Schritte 2 bis 4, um einen weiteren Lieferantenartikel hinzuzufügen.
6. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
