Definieren von Stammrezepten

Ein Stammrezept auf Grundlage des S88-Rezeptmodells ist ein revisionierbares Objekt, das alle Informationen enthält, die zur Herstellung eines Materialbatches benötigt werden. Ein Stammrezept ist eine Sammlung von einem oder mehreren Abläufen (Rezeptlisten), die Sie einem Batch bei dessen Start zuweisen.

Vor dem Definieren von Stammrezepten

Bevor Sie ein Stammrezept definieren, müssen Sie eines oder mehrere der folgenden Elemente definieren:

• Rezeptlisten
• Spezifikationen

Möglicherweise (je nach Ihren Anforderungen) müssen Sie zusätzlich die folgenden Objekte definieren:

• Produkt
• Maßeinheit
• Arbeitsplan

Beim Definieren von Stammrezepten

Die Anwendung erfordert, dass Sie mindestens eine Rezeptliste angeben, wenn Sie ein Stammrezept erstellen.

Wenn Sie dem Stammrezept eine Rezeptliste hinzufügen, erfordert die Anwendung, dass Sie die Rezeptliste zu einer Spezifikation zuweisen.

Sie können dem Stammrezept dieselbe Rezeptliste mehrmals hinzufügen, aber Sie müssen die Liste jedes Mal einer anderen Spezifikation zuweisen. Es ist nicht möglich, ein Stammrezept mit mehreren Kopien derselben Rezeptliste für dieselbe Spezifikation zu speichern.

Hinweis: Achten Sie darauf, Nenn- (nominale) und Grenzwerte für das Stammrezept anzugeben. Die Anwendung benötigt keinen dieser Werte, um ein Stammrezept zu speichern, aber sie benötigt sie, wenn ein Batch gestartet wird.

Ausführungsreihenfolge einer Rezeptliste

Wenn Sie einem Stammrezept eine Rezeptliste hinzufügen, bestimmt die Sequenznummer die Ausführungsreihenfolge und die Untersequenznummer die Anzeigereihenfolge. Wenn Sie mehrere Rezeptlisten derselben Spezifikation zuordnen, erfordert die Anwendung eindeutige Sequenz- und Untersequenzkombinationen, damit die richtige Anzeigereihenfolge sichergestellt werden kann.

Hinweis: Rezeptlisten in einer Spezifikation können gleichzeitig mit demselben Sequenzwert ausgeführt werden.

Felder für Rezeptmenge

Beim Starten eines Batches werden die folgenden Werte verwendet, um die Menge zu überprüfen:

• Rezeptnominalmenge: Die gewünschte Menge des Produkts, das mit diesem Rezept hergestellt werden soll.
• Rezeptmenge: untere Grenze: Die untere Grenze für die Menge des Produkts, das mit diesem Rezept hergestellt werden soll.
• Rezeptmenge: obere Grenze: Die obere Grenze für die Menge des Produkts, das mit diesem Rezept hergestellt werden soll.

Der Wert, den Sie in eines dieser Felder eingegeben, kann ein statischer Text oder ein Ausdruck sein. Wenn Sie einen Ausdruck eingeben, berechnet die Anwendung die Menge zur Laufzeit. Sie können maximal 255 Zeichen eingeben.

Vorgehensweise zum Definieren eines Stammrezepts

Gehen Sie folgendermaßen vor, um ein Stammrezept zu definieren:

1. Öffnen Sie die Seite Stammrezept. Die Seite "Stammrezept" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für das Rezept in das Feld Stammrezept ein.
4. Geben Sie die Revision dieses Objekts in das Feld Revision ein.
5. Wählen Sie Sequenziell oder Nicht sequenziell aus der Liste Ausführungsmodus aus.
6. Fügen Sie mindestens ein Rezept mit dem Verfahren "Vorgehensweise zum Hinzufügen einer Rezeptliste zu einem Stammrezept" hinzu.
7. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
8. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Hinzufügen einer Rezeptliste zu einem Stammrezept

Gehen Sie folgendermaßen vor, um einem Stammrezept eine Rezeptliste hinzuzufügen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren eines Stammrezepts" aus.

Oder

Wählen Sie eine vorhandene Stammrezept-Instanz aus.

2. Erweitern Sie den Abschnitt Rezeptprozedur.
3. Klicken Sie in der Tabelle "Rezeptprozeduren" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
4. Wählen Sie eine Rezeptliste im Feld Rezeptliste aus.
5. Wählen Sie eine Spezifikation im Feld Spez. aus.
6. Geben Sie eine Sequenznummer für die Rezeptliste in das Feld Reihenfolge ein.
7. Geben Sie eine Untersequenznummer für die Rezeptliste in das Feld Untersequenz ein.
8. Wiederholen Sie die Schritte 3 bis 7, um weitere Rezeptlisten hinzuzufügen.
9. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
