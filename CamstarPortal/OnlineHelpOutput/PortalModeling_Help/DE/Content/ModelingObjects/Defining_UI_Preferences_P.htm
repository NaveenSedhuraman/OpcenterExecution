Definieren von UI-Einstellungen (Benutzeroberflächeneinstellungen)

Mithilfe von UI-Einstellungen können Sie die Informationen festlegen, die auf der Registerkarte "Status" der Seite "Produktionsprotokoll" für einen Container, Behälter oder Ereignisdatensatz angezeigt werden sollen. Sie können damit auch die Informationen festlegen, die für einen Ressourcendatensatz auf der Registerkarte "Status" der Seite "Ressourcenprotokoll" angezeigt werden sollen. Die UI-Einstellungen bestimmen auch die Reihenfolge, in der die Anwendung Informationen auf der Registerkarte "Status" anzeigt.

Beim Definieren einer UI-Einstellung

Der Objektname darf keine ungültigen Zeichen enthalten. Z. B. darf kein Apostroph verwendet werden. Informationen zu gültigen Zeichen finden Sie im Opcenter Execution Medical Device and Diagnostics Designer-Benutzerhandbuch und im Opcenter Execution Core Designer-Benutzerhandbuch.

Beim Angeben der anzuzeigenden Felder

Verwenden Sie für Feldnamen die folgenden Formate:

• Container.FieldName, wenn Sie die anzuzeigenden Felder für Container- und Behälterdatensätze angeben.
• QualityObject.FieldName, wenn Sie die anzuzeigenden Felder für Ereignisdatensätze angeben
• Resource.FieldName, wenn Sie die anzuzeigenden Felder für Ressourcendatensätze angeben

Zuweisen von UI-Einstellungen (Benutzeroberflächeneinstellungen) zu Datensatztypen

"UI-Einstellungen" ist ein optionales Feld in den folgenden Modellierungsobjekten:

• Containerlevel
• Organisation
• Ressourcentyp
• Ressourcenfamilie

Die Zuweisung von UI-Einstellungen zu den Datensatztypen ist Voraussetzung dafür, dass auf der Registerkarte "Status" der Seite "Produktionsprotokoll" bzw. "Ressourcenprotokoll" Informationen zum ausgewählten Datensatz angezeigt werden. Sie können die UI-Einstellung für Container- und Behälterdatensätze im Feld "Konfiguration der Benutzeroberflächenanzeige" des Modellierungsobjekts "Containerlevel" zuweisen. Sie haben auch die Möglichkeit, jedem definierten Containerlevel eine andere UI-Einstellung zuzuweisen. Weitere Informationen zu Containerlevels finden Sie unter "Definieren von Containerlevels".

Die UI-Einstellung für Ereignisdatensätze wird in der UI-Einstellungenzuordnung des Modellierungsobjekts "Organisation" zugewiesen. Sie können jedem Datensatztyp eine andere UI-Einstellung zuweisen. Weitere Informationen zur Zuweisung von Benutzeroberflächeneinstellungen finden Sie unter "Definieren einer Organisation".

Darüber hinaus müssen Sie die Organisation als primäre Organisation des Mitarbeiters, der das Ereignis aufzeichnet, festlegen. Die Registerkarte "Status" wird nur dann für Ereignisdatensätze angezeigt, wenn eine UI-Einstellung (Benutzeroberflächeneinstellung) für die Organisation angegeben wurde und diese Organisation als primäre Organisation des Mitarbeiters, der das Ereignis aufzeichnet, festgelegt wurde. Weitere Informationen zum Festlegen einer primären Organisation finden Sie unter "Definieren von Mitarbeitern".

Die UI-Einstellungen für Ressourcendatensätze werden im Feld "UI-Einstellungen" des Modellierungsobjekts "Ressourcentyp" und/oder "Ressourcenfamilie" zugewiesen. "Ressourcentyp" und "Ressourcenfamilie" sind optionale Felder im Modellierungsobjekt für Ressourcen und können eine UI-Einstellung angeben. Die Anwendung verwendet zuerst die für den Ressourcentyp angegebene UI-Einstellung. Falls für den Ressourcentyp keine UI-Einstellung definiert ist, wird die für die Ressourcenfamilie definierte UI-Einstellung verwendet. Sie können jeder definierten Ressource eine andere UI-Einstellung zuweisen. Weitere Informationen finden Sie unter "Definieren von Ressourcentypen" und "Definieren von Ressourcenfamilien".

Vorgehensweise zum Definieren einer UI-Einstellung (Benutzeroberflächeneinstellung)

Gehen Sie folgendermaßen vor, um eine UI-Einstellung zu definieren:

1. Öffnen Sie die Seite UI-Einstellungen. Die Seite "UI-Einstellungen" werden innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für die UI-Einstellung in das Feld UI-Einstellungen ein.
4. Klicken Sie in der Tabelle "Anzuzeigende Felder" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
5. Geben Sie den Namen des Feld, das angezeigt werden soll, in die Spalte UI-Einst.: Felddefinitionsname ein.
6. Wiederholen Sie die Schritte 4 und 5, um zusätzliche Felder anzugeben.
7. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
8. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.

So wird die für Container- und Behälterdatensätze zu verwendende UI-Einstellung angegeben

Gehen Sie folgendermaßen vor, um die für Container- und Behälterdatensätze zu verwendende UI-Einstellung anzugeben:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren eines Containerlevels" aus.

Oder

Wählen Sie eine vorhandene Instanz eines Containerlevels aus.

2. Geben Sie eine UI-Einstellung in das Feld Konfiguration der Benutzeroberflächenanzeige ein oder wählen Sie eine UI-Einstellung aus.
3. Wiederholen Sie diese Schritte für jedes Containerlevel.
4. Klicken Sie auf Speichern. Die Anwendung speichert die Änderungen und verwendet von nun an diese UI-Einstellungen, wenn auf der Registerkarte "Status" der Seite "Produktionsprotokoll" Informationen für Container- oder Behälterdatensatz angezeigt werden.

Vorgehensweise zum Ändern der für Ereignisdatensätze zu verwendenden UI-Einstellung

Gehen Sie folgendermaßen vor, um die für Ereignisdatensätze zu verwendende UI-Einstellung anzugeben:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren einer Organisation" aus.

Oder

Wählen Sie eine vorhandene Organisationsinstanz aus.

2. Klicken Sie in der Tabelle "UI-Einstellungenzuordnung" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
3. Geben Sie den entsprechenden Datensatztyp ein oder wählen Sie ihn aus.
4. Geben Sie die UI-Einstellung ein, die mit dem Datensatztyp verknüpft werden soll, oder wählen Sie sie aus.
5. Klicken Sie auf Speichern. Die Anwendung speichert die Änderungen und verwendet von nun an diese UI-Einstellung, wenn auf der Registerkarte "Status" der Seite "Produktionsprotokoll" Informationen für den angegebenen Qualitätsdatensatz angezeigt werden.
