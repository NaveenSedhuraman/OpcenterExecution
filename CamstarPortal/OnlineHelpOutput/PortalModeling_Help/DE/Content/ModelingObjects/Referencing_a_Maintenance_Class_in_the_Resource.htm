Referenzieren einer Wartungsklasse in der Ressource

Ressourcen, die die gleichen Wartungsanforderung haben, gehören zu einer Wartungsklasse. Wartungsklassen gelten auch für Behälter (ein weiterer Ressourcentyp).

Weitere Informationen zu Ressourcen finden Sie unter "Definitionen des physischen Modells".

Vorgehensweise zum Ändern einer Ressource, sodass sie eine Wartungsklasse referenziert

Gehen Sie folgendermaßen vor, um die Wartungsklasse anzugeben, zu der die Ressource gehört:

1. Öffnen Sie die Seite Ressource. Die Seite "Ressource" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf die Ressource, die Sie ändern möchten. Die Seite "Ressource" wird für die ausgewählte Ressource angezeigt.
3. Wählen Sie die entsprechende Wartungsklasse für diese Ressource im Feld Wartungsklasse aus.
4. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu dieser Seite finden Sie unter "Felddefinitionen der Seite "Ressource"".
5. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
