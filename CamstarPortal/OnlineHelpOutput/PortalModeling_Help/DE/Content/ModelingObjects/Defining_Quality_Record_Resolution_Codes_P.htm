Definieren von Auflösungscodes für Qualitätsdatensätze

Der Auflösungscode eines Qualitätsdatensatzes gibt an, aus welchem Grund ein Qualitätsdatensatz aufgelöst (geschlossen) wurde. Die Anwendung verlangt zur Auflösung des Qualitätsdatensatzes einen Auflösungscode für Qualitätsdatensätze.

Beim Definieren von Auflösungscodes für Qualitätsdatensätze

Auflösungscodes sind auf der Seite "Qualitätsobjektauflösung" erforderlich, wenn Sie einen Qualitätsdatensatz auflösen.

Siehe auch "Gemeinsame Felder auf Modellierungsseiten". Dieser Abschnitt enthält Informationen zu den Feldern, die alle Modellierungsobjekte aufweisen.

Vorgehensweise zum Definieren eines Qualitätsdatensatzauflösungscodes

Gehen Sie folgendermaßen vor, um einen Qualitätsdatensatzauflösungscode zu definieren:

1. Öffnen Sie die Seite Qualitätsdatensatzauflösungscode. Die Seite "Qualitätsdatensatzauflösungscode" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen zur Festlegung des Auflösungscodes in das Feld Qualitätsdatensatzauflösungscode ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Unter "Gemeinsame Felder auf Modellierungsseiten" finden Sie Informationen zu den optionalen Feldern.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
