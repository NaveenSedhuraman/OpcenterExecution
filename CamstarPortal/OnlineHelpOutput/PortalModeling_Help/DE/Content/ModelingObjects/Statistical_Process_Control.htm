Statistische Prozesskontrolle

Qualitätsdaten sind eine wertvolle Quelle von Produktionsinformationen. Die Spezifikationen in Opcenter EX MDD und Opcenter EX CR gewährleisten, dass Umlaufbestandseinheiten, die nicht den Mindestanforderungen entsprechen, nicht versandt werden können, aber das ist nur der Anfang. Hersteller können hochwertige Produkte ohne Abstriche bei der Produktivität produzieren. Opcenter Execution Core Statistical Process Control (SPC) ist ein leistungsfähiges und flexibles Modul, mit dem Hersteller Probleme lösen können, bevor diese zu einer Verringerung von Ertrag oder Durchsatz führen. In diesem Kapitel werden die Modellierungsobjekte, die zum Erstellen von SPC-Objekten verwendet werden, beschrieben.

Weitere Informationen finden Sie im Opcenter Execution Core Statistical Process Control-Referenzhandbuch.
