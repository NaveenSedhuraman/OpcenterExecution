Definieren von Maßeinheiten

Das Modellierungsobjekt "Maßeinheit" bietet die Möglichkeit, Maßeinheiten zu definieren, die den Mengenangaben eines Containers zugeordnet werden können.

Beim Definieren von Maßeinheiten

Die Namen der einzelnen Maßeinheiten müssen eindeutig sein. Beispiele für Maßeinheiten sind:

• Each
• kg
• ml

"Maßeinheit" ist ein optionales Feld in den folgenden Modellierungsobjekten:

• Produkt
• Produktfamilie
• Produktionsauftrag
• Verkaufsauftrag

"Maßeinheit" ist ein optionales Feld in folgenden Transaktionen:

• Containerwartung
• Ressourcendurchsatz
• Start
• Starten - zwei Level

Beim Definieren von Konvertierungsinformationen für Maßeinheiten

Sie können Mengenkonvertierungsinformationen für eine Maßeinheit definieren, wenn Batch Processing installiert ist. Die Tabelle "Maßeinheitenkonvertierung" ist nur zusammen mit Batch Processing verfügbar. In der Tabelle können Sie folgende Informationen konfigurieren:

• Einen Faktor für die Konvertierung der betreffenden Maßeinheit in eine andere.
• Eine Regel für die Rundung von konvertierten Mengen.
• Die Anzahl der erlaubten Dezimalstellen für die berechnete konvertierte Menge. Der Maximalwert ist 4.

Die folgenden Regeln gelten beim Definieren von Umrechnungsinformationen für Maßeinheiten:

• Damit eine Rundung stattfindet müssen die Felder "Rundungsregel" und "Dezimalstellen" ausgefüllt werden.
• Konvertierungsinformationen, die für das Produkt definiert sind, haben Vorrang vor Konvertierungsinformationen, die für die Produktfamilie definiert sind.
• Konvertierungsinformationen, die für die Produktfamilie definiert sind, haben Vorrang vor Konvertierungsinformationen, die für die Maßeinheit definiert sind.

Weitere Informationen finden Sie unter "Definieren der Maßeinheitenkonvertierungsinformationen für das Produkt" und "Definieren der Maßeinheitenkonvertierungsinformationen für die Produktfamilie".

Vorgehensweise zum Definieren einer Maßeinheit

Gehen Sie folgendermaßen vor, um eine Maßeinheit zu definieren:

1. Öffnen Sie die Seite Maßeinheit. Die Seite "Maßeinheit" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie im Feld Maßeinheit einen Namen für die Maßeinheit ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
