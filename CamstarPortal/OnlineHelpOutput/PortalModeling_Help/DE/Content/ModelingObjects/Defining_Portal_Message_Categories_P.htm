Definieren von Portalnachrichtenkategorien

Siemens stellt eine einzelne Instanz des Modellierungsobjekts "Portalnachrichtenkategorie" für die Portalinstallation bereit. Mit dem Modellierungsobjekt "Portalnachrichtenkategorie" können Sie folgende Anpassungen vornehmen:

• Die in Concierge und Nachrichtenzentrale angezeigten Nachrichtenkategorien
• Die in den Nachrichtenkategorien angezeigten Benachrichtigungstypen
• Die den Nachrichtenkategorien zugewiesenen Symbole
Hinweis: Die Symbolzuweisung ist nur verfügbar, wenn Sie das Thema "Classic" verwenden.
• Fälligkeitsdatum-Indikatoren für Qualitätselemente
Hinweis: Informationen zu Concierge und Nachrichtenzentrale finden Sie unter:
• Opcenter Execution Medical Device and Diagnostics Event Management-Benutzerhandbuch oder Opcenter Execution Core Event Management-Benutzerhandbuch
• Opcenter Execution Medical Device and Diagnostics Nonconformance-Benutzerhandbuch oder Opcenter Execution Core Nonconformance Management-Benutzerhandbuch
• Onlinehilfe, die über die Seiten für Qualitätsdatensätze verfügbar ist

Es kann nur eine Liste von Nachrichtenkategorien im Portal geben. Sie können diese Modellierungsobjekt-Instanz weder löschen noch kopieren.

Beim Bearbeiten der Portalnachrichtenkategorie-Instanz

Die Portalnachrichtenkategorie enthält das erforderliche Modellierungsobjekt "Nachrichtenkategorie". Siemens stellt vier Nachrichtenkategorien bereit, die dieser Instanz standardmäßig zugewiesen sind. Sie können jedoch weitere Nachrichtenkategorien definieren und hinzufügen. Weitere Informationen zum Definieren von Nachrichtenkategorien finden Sie unter "Definieren von Nachrichtenkategorien".

Wichtig: Fügen Sie Concierge maximal sechs Nachrichtenkategorien hinzu, um Leistungseinbußen zu verhindern.

Die Benachrichtigungstypen, die Sie einer Nachrichtenkategorie zuweisen können, werden zusammen mit der Anwendung installiert und können nicht geändert werden. Sie können einer Nachrichtenkategorie einen oder mehrere Benachrichtigungstypen zuweisen. Nur wenn Sie einer Nachrichtenkategorie mindestens einen Benachrichtigungstyp zuweisen, wird sie in der Nachrichtenzentrale oder in Concierge angezeigt.

Definieren des kleinsten und des größten Bereichs für Gelb

Die Felder "Gelb: minimaler Bereich" und "Gelb: maximaler Bereich" auf der Seite "Portalnachrichtenkategorie" bestimmen, wann die Fälligkeitsdatum-Indikatoren (grüner Kreis, orangefarbener Kreis und roter Kreis) neben den Einträgen für "Concierge" und "Nachrichtenzentrale" angezeigt werden. Die nachstehenden Informationen können von Nutzen sein, wenn Sie den kleinsten und den größten Bereich für die Fälligkeitsdatum-Indikatoren festlegen.

Regeln der Felder für Fälligkeitsdatum-Bereichsindikatoren

Es gibt folgende Regeln für die Festlegung der Fälligkeitsdatum-Bereichsindikatoren:

• Die Werte in den Feldern "Kleinster Bereich für Gelb" und "Größter Bereich für Gelb" müssen ganze Zahlen sein.
• Der Wert im Feld "Kleinster Bereich für Gelb (in Tagen)" muss kleiner sein als der Wert im Feld "Größter Bereich für Gelb (in Tagen)".
• "Kleinster Bereich für Gelb" gibt an, nach wie vielen Tagen ab dem Fälligkeitsdatum der orangefarbene Indikator angezeigt wird. Der Standardwert ist 0. Wenn die Datumsdifferenz für die Aktivität kleiner ist als diese Anzahl von Tagen, wird der rote Kreis angezeigt.
• "Größter Bereich für Gelb" ist die maximale Anzahl von Tagen ab dem Fälligkeitsdatum, an denen der orangefarbene Indikator angezeigt wird. Der Standardwert ist 2. Wenn die Datumsdifferenz für die Aktivität größer ist als diese Anzahl von Tagen, wird der rote Kreis angezeigt.
• Rote Indikatoren werden erst nach dem Fälligkeitsdatum angezeigt.

Vorgehensweise zum Berechnen der Prioritätsindikatoren

Hier wird erklärt, wie die Prioritätsindikatoren berechnet werden. Unter der Annahme, dass:

• Kleinster Bereich für Gelb = n
• Größter Bereich für Gelb = m
• m => n

Gilt Folgendes:

• Rot: (Fälligkeitsdatum - heute) < n
• Orange: n <= (Fälligkeitsdatum – heute) <= m
• Grün: m < (Fälligkeitsdatum - heute)

Vorgehensweise zum Auswählen der Portalnachrichtenkategorie

Gehen Sie folgendermaßen vor, um die Portalnachrichtenkategorie auszuwählen:

1. Öffnen Sie die Seite Portalnachrichtenkategorie. Die Seite "Portalnachrichtenkategorie" wird innerhalb der Seite "Modellierung" angezeigt.
2. Wählen Sie die Portalnachrichtenkategorie-Instanz aus. Die Felder und Werte für die Instanz werden angezeigt.

Vorgehensweise zum Bearbeiten einer Nachrichtenkategorie

Gehen Sie folgendermaßen vor, um eine Nachrichtenkategorie in der Liste "Portalnachrichtenkategorie" zu bearbeiten:

1. Führen Sie die Schritte unter "Vorgehensweise zum Auswählen der Portalnachrichtenkategorie" aus.

Oder

Wählen Sie eine vorhandene Portalnachrichtenkategorie-Instanz aus.

2. Klicken Sie auf Ausgewählte Zeile bearbeiten, um die Eigenschaften einer vorhandene Nachrichtenkategorie zu bearbeiten. Das Popup "Bearbeiten" wird angezeigt.
3. Führen Sie einen oder mehrere der folgenden Schritte aus, um die Einstellungen der Nachrichtenkategorie zu ändern:
• Wählen Sie eine Kategorie aus der Liste Nachrichtenkategorie aus, um die vorhandene Kategorie zu ändern.
• Aktivieren bzw. deaktivieren Sie das Kontrollkästchen In Nachrichtenzentrale anzeigen oder In Concierge anzeigen.
• Klicken Sie in der Tabelle "Benachrichtigungstypen" auf Neue Zeile hinzufügen, um eine Benachrichtigung auszuwählen, die der Nachrichtenkategorie zugeordnet werden soll. Eine neue Zeile wird angezeigt.
• Wählen Sie eine Zeile aus und klicken Sie in der Tabelle "Benachrichtigungstypen" auf Ausgewählte Zeile löschen, um einen Benachrichtigungstyp aus der Nachrichtenkategorie zu entfernen.
4. Führen Sie einen der folgenden Schritte aus: 

Klicken Sie auf OK. Das Popup für die Definition der Nachrichtenkategorie wird geschlossen.

Oder

Klicken Sie ohne zu speichern auf Schließen. Das Popup "Bearbeiten" wird geschlossen.

5. Klicken Sie auf Speichern. Die Anwendung speichert die Änderungen und zeigt eine Nachricht an, die besagt, dass die Portalnachrichtenkategorie aktualisiert wurde.

Vorgehensweise zum Hinzufügen einer Kategorie zur Tabelle "Definition der Nachrichtkategorie"

Gehen Sie folgendermaßen vor, um der Tabelle "Definition der Nachrichtkategorie" eine neue Kategorie hinzuzufügen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Auswählen der Portalnachrichtenkategorie" aus.

Oder

Wählen Sie eine vorhandene Portalnachrichtenkategorie-Instanz aus.

2. Klicken Sie auf Neue Zeile hinzufügen in der Tabelle "Definition der Nachrichtkategorie". Das Popup "Hinzufügen" wird angezeigt.
3. Wählen Sie eine Nachrichtenkategorie in der Liste aus.
4. Aktivieren Sie das Kontrollkästchen In Nachrichtenzentrale anzeigen, wenn diese Nachrichtenkategorie in der Nachrichtenzentrale angezeigt werden soll.
5. Aktivieren Sie das Kontrollkästchen In Concierge anzeigen, wenn diese Nachrichtenkategorie in der Anzeige des Concierge angezeigt werden soll.
6. Klicken Sie auf Neue Zeile hinzufügen in der Tabelle "Benachrichtigungstypen" und wählen Sie einen Benachrichtigungstyp aus der Liste Benachrichtigungstypen aus.
7. Wiederholen Sie Schritt 7, um weitere Benachrichtigungstypen zuzuweisen.
8. Klicken Sie auf OK. Das Popup "Hinzufügen" wird geschlossen und die neue Nachrichtenkategorie wird in der Tabelle "Definition der Nachrichtkategorie" angezeigt.
9. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.

Vorgehensweise zum Löschen einer Nachrichtenkategoriedefinition aus der Liste

Gehen Sie folgendermaßen vor, um eine Nachrichtenkategorie aus der Liste zu löschen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Auswählen der Portalnachrichtenkategorie" aus.

Oder

Wählen Sie eine vorhandene Portalnachrichtenkategorie-Instanz aus.

2. Wählen Sie in der Tabelle "Definition der Nachrichtkategorie" eine Zeile aus.
3. Klicken Sie auf Ausgewählte Zeile löschen in der Tabellenfußzeile, um die ausgewählte Nachrichtenkategorie zu entfernen.
4. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.

Vorgehensweise zum Ändern der Sortierreihenfolge für Concierge und Nachrichtenzentrale

Gehen Sie folgendermaßen vor, um die Sortierreihenfolge in Concierge und Nachrichtenzentrale zu ändern:

1. Führen Sie die Schritte unter "Vorgehensweise zum Auswählen der Portalnachrichtenkategorie" aus.

Oder

Wählen Sie eine vorhandene Portalnachrichtenkategorie-Instanz aus.

2. Wählen Sie in der Tabelle "Definition der Nachrichtkategorie" eine Zeile aus.
3. Verschieben Sie die ausgewählte Zeile mit den entsprechenden Pfeilen in der Tabellenfußzeile an die richtige Stelle in der Liste.
4. Wiederholen Sie diese Schrittfolge, bis die Kategorien in der richtigen Reihenfolge angezeigt werden.
5. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.

Vorgehensweise zum Definieren von Bereichen für Fälligkeitsdatums-Indikatoren

Gehen Sie folgendermaßen vor, um die Bereiche für die Fälligkeitsdatum-Indikatoren festzulegen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Auswählen der Portalnachrichtenkategorie" aus.

Oder

Wählen Sie eine vorhandene Portalnachrichtenkategorie-Instanz aus.

2. Erweitern Sie den Abschnitt Fälligkeitsdatumsbereich.
3. Geben Sie eine Zahl in das Feld Kleinster Bereich für Gelb (in Tagen) ein.
4. Geben Sie eine Zahl in das Feld Größter Bereich für Gelb (in Tagen) ein.
5. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
