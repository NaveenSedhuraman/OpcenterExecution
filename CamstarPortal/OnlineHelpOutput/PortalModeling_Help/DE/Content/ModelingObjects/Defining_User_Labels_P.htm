Definieren von Benutzerbeschriftungen

Eine Benutzerbeschriftung ist ein Text auf der Benutzeroberfläche der Anwendung. Beschriftungen können Feldnamen, Systemnachrichten usw. sein. Siemens stellt Standard-Systembeschriftungen zur Verfügung. Diese Beschriftungen werden von Anwendungen in Opcenter EX MDD und Opcenter EX CR verwendet.

Wie bei den meisten Konfigurationen der Anwendung können Sie die Systembeschriftungen ändern, ohne die ursprünglichen Definitionen zu ändern. Sie können auch eigene Beschriftungen (Benutzerbeschriftungen) als Instanzdaten erstellen und dies sogar ganz bequem über das Portal.

Sie können entweder den Beschriftungsnamen oder die Beschriftungs-ID an die Services oder den XML-Client der Anwendung weitergeben.

Die Anwendung enthält eine Unterklasse des Modellierungsobjekts "Benutzerbeschriftung", nämlich "Nachrichtenkategorie". Weitere Informationen zum Modellierungsobjekt "Benutzerbeschriftung" finden Sie unter "Definieren von Nachrichtenkategorien".

Beim Übersetzen von System- und Benutzerbeschriftungen

Sie können "Wörterbuch"-Modellierungsobjekte definieren, um Übersetzungen besser zu organisieren, wenn Sie System- und Benutzerbeschriftungen übersetzen möchten. Wörterbücher sind eine Sammlung von benutzerdefinierten Beschriftungen, die Mitarbeitern zugewiesen sind.

Wenn sich ein Mitarbeiter, dem ein Wörterbuch zugewiesen ist, beim Portal anmeldet, werden auf der Benutzeroberfläche des Mitarbeiters die entsprechenden Beschriftungen aus dem Wörterbuch angezeigt. Die Standardbeschriftungen werden angezeigt, wenn einem Mitarbeiter kein Wörterbuch zugewiesen ist.

Hinweis: Weitere Informationen zum Modellierungsobjekt "Wörterbuch" finden Sie unter "Definieren von Wörterbüchern".

Sprach- und fachspezifische Beschriftungen

Sie können Beschriftungen erstellen, die in eine dieser Gruppen fallen:

• Sprachspezifische Beschriftungen, die zu einem bestimmten Vokabular gehören.

In dieser Gruppe können Sie sprachspezifische Beschriftungen erzeugen und auch Siemens Beschriftungen aufrufen und übersetzen.

• Terminologiespezifische Beschriftungen, die fachspezifisch sind.

Was in der Anwendung als Container bezeichnet wird, heißt möglicherweise in Ihrer Fabrik Los oder Behälter. Der Mitarbeiter heißt in einer anderen Umgebung u. U. Produktionsmitarbeiter. Terminologiespezifische Beschriftungen sind z. B. für Großunternehmen mit mehreren Geschäftsbereichen hilfreich, bei denen bestimmte Fachbegriffe nicht geschäftsbereichsübergreifend verwendet werden können.

Nachdem Sie die erforderlichen Beschriftungen definiert haben, können Sie diese Wörterbüchern hinzufügen. Weitere Informationen finden Sie unter "Definieren von Wörterbüchern".

Vorgehensweise zum Definieren einer Benutzerbeschriftung

Gehen Sie folgendermaßen vor, um eine Benutzerbeschriftung zu definieren:

1. Öffnen Sie die Seite Benutzerbeschriftung. Die Seite "Benutzerbeschriftung" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für die Benutzerbeschriftung in das Feld Benutzerbeschriftung ein.
4. Geben Sie den Beschriftungswert ein, der auf der Benutzeroberfläche angezeigt werden soll.
5. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
6. Klicken Sie auf Speichern. Eine Beschriftungs-ID für die Beschriftung wird erzeugt.
