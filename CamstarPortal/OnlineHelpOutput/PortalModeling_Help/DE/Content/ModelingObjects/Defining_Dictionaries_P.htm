Definieren von Wörterbüchern

Mit Opcenter EX MDD und Opcenter EX CR können Sie benutzerdefinierte oder übersetzte Benutzer- und Systembeschriftungen für die Anwendung erstellen. Beschriftungen in der Anwendung sind Nachrichten und Text, z. B. angezeigte Feldüberschriften, Fehlermeldungen, Beendigungsnachrichten für Transaktionen, Zeichenkettenauswahlwerte usw. Benutzerdefinierte und übersetzte Beschriftungen werden mithilfe von Wörterbüchern organisiert und Mitarbeitern zugewiesen.

Sie können benutzerdefinierte oder übersetzte Beschriftungen in zwei Wörterbuchtypen gruppieren:

• Sprachwörterbuch – speichert übersetzte Beschriftungen für bestimmte Sprachen.
• Terminologiewörterbuch – speichert benutzerdefinierte Beschriftungen für branchenspezifische Sprachen. Beispiel: Opcenter EX MDD und Opcenter EX CR verwenden das Wort "Container", um eine separate Sammlung von Produkten darzustellen, die in einer Fabrik hergestellt wurden. Außerdem verwenden sie "Mitarbeiter", um eine Person darzustellen. Unter Umständen möchten Sie für Ihr Unternehmen andere Wörter verwenden, z. B. Behälter anstelle von Container und Kollege anstelle von Mitarbeiter.

Die Anwendung zeigt Beschriftungen basierend auf den verbundenen Wörterbüchern des angemeldeten Mitarbeiters an. Die Standardbeschriftungen und benutzerdefinierten Beschriftungen werden angezeigt, wenn einem Mitarbeiter kein Wörterbuch zugewiesen ist. Weitere Informationen zum Zuweisen von Wörterbüchern zu einem Mitarbeiter finden Sie unter "Definieren von Mitarbeitern".

Die Anwendung erlaubt es Ihnen, einem Mitarbeiter sowohl Sprach- als auch Terminologiewörterbücher zuzuweisen. Wenn Beschriftungen in beiden Wörterbüchern vorhanden sind, haben Beschriftungen im Terminologiewörterbuch Priorität vor den Beschriftungen im Sprachwörterbuch.

Wörterbuchbeschriftungen werden in Anwendungsserver- und Portal-Anwendungscaches zwischengespeichert. Opcenter EX MDD- und Opcenter EX CR-Wörterbuchbeschriftungen werden in der Datenbank gespeichert und im IIS-Cache des Portals zwischengespeichert. Sie müssen den IIS-Anwendungspool neu starten, um den Cache zu aktualisieren. Der Wörterbuchbeschriftungscache wird aktualisiert, wenn eine Wörterbuchmodellierungsinstanz aktualisiert wird.

Beim Speichern einer Wörterbuchmodellierungsinstanz wird der Beschriftungscache standardmäßig automatisch aktualisiert, wenn der Speichervorgang Änderungen umfasst.

Sie geben diese Wörterbuchtypen oder -kategorien in der Mitarbeiterdefinition an, um einem Mitarbeiter ein Wörterbuch zuzuweisen.

Wahlweise können Sie Wörterbuchbeschriftungen mithilfe von benutzerdefinierten CSV-Dateien in Management Studio laden. Informationen zum Laden von benutzerdefinierten Wörterbüchern finden Sie in der Opcenter Execution Medical Device and Diagnostics-Anleitung zur Systemadministration und in der Opcenter Execution Core-Anleitung zur Systemadministration.

Beim Definieren eines Wörterbuchs

Ein Wörterbuch enthält die optionale Modellierungsdefinition "Benutzerbeschriftung".

"Wörterbuch" ist ein optionales Feld in der Definition von "Mitarbeiter".

Über die Schaltfläche "Suchen" können Sie Suchergebnisse anzeigen, die entsprechend den Informationen gefiltert sind, die Sie in die Filter-Felder eingegeben haben.

Hinweis: Sie müssen die Instanz speichern, bevor Sie eine Suche ausführen.

Vorgehensweise zum Definieren eines Wörterbuchs

Gehen Sie folgendermaßen vor, um ein Wörterbuch zu definieren:

1. Öffnen Sie die Seite Wörterbuch. Die Seite "Wörterbuch" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für das Wörterbuch in das Feld Wörterbuch ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.

So wird eine Beschriftung im Wörterbuch geändert

Gehen Sie folgendermaßen vor, um eine Beschriftung im Wörterbuch zu bearbeiten:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren eines Wörterbuchs" aus.

Oder

Wählen Sie eine vorhandene Instanz eines Wörterbuchs aus.

2. Bearbeiten Sie Beschriftungen in diesem Wörterbuch wie folgt:

Sie möchten . . .

Dann . . .

Bearbeiten von Opcenter EX MDD- oder Opcenter EX CR-Beschriftungen
a. Wählen Sie einen Kategoriefilter, der mit dem Präfix csi beginnt, in der Liste "Kategoriefilter" aus, um Ihre Suche auf Opcenter EX MDD- oder Opcenter EX CR-Beschriftungen einzuschränken. Sie können auch einen Teil des Namens einer Benutzerbeschriftung in das Feld "Beschriftungsnamefilter" oder einen Teil eines Wertes in das Feld "Beschriftungswertfilter" eingeben.
b. Gehen Sie zu Schritt 3.

Bearbeiten Sie die Benutzerbeschriftungen, die Sie erstellt haben
a. Geben Sie den Namen der Benutzerbeschriftung in "Beschriftungsnamefilter" oder den Beschriftungswert in "Beschriftungswertfilter" ein, um die Suche auf Benutzerbeschriftungen einzuengen, die den eingegebenen Text enthalten. Weitere Informationen finden Sie unter "Vorgehensweise zum Definieren einer Benutzerbeschriftung".
b. Gehen Sie zu Schritt 3.
3. Klicken Sie auf Suchen. Die Tabelle "Suchergebnisse" zeigt die Beschriftungen an, nach denen Sie gesucht haben.
4. Wählen Sie eine Beschriftung aus der Tabelle aus und klicken Sie auf Ausgewählte Zeile bearbeiten. Das Popup "Bearbeiten" wird angezeigt.
5. Geben Sie die Übersetzung für die Beschriftung in das Feld Beschriftungstext ein.
6. Klicken Sie auf OK. Das Popup "Bearbeiten" wird geschlossen und der neue Beschriftungstext wird in der Tabelle "Suchergebnisse" angezeigt.
7. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
