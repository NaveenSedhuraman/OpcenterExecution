Definieren von Behälterfamilien

Eine Behälterfamilie bietet Ihnen die Möglichkeit, Behälter nach deren Attributen und entsprechend Ihren Geschäftsanforderungen zu kategorisieren. Sie können die Behälterfamilie anschließend benutzen, um Wartungszeitpläne festzulegen, oder Sie verwenden sie als Filterkriterium für verschiedene Fertigungstransaktionen. Jeder Behälter kann eine Behälterfamilie haben, etwa Fass, Kessel, Tablett und so weiter.

Beim Definieren einer Behälterfamilie

Es folgen einige Beispiele von Behälterfamilien:

• 100-l-Fass
• 200-l-Fass
• Glasschale
• Metalltablett

"Behälterfamilie" ist ein optionales Feld im Modellierungsobjekt "Behälter".

Siehe auch "Gemeinsame Felder auf Modellierungsseiten". Dieser Abschnitt enthält Informationen zu den Feldern, die alle Modellierungsobjekte aufweisen.

Vorgehensweise zum Definieren einer Behälterfamilie

Gehen Sie folgendermaßen vor, um eine Behälterfamilie zu definieren:

1. Öffnen Sie die Seite Behälterfamilie. Die Seite "Behälterfamilie" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für die Behälterfamilie in das Feld Behälterfamilie ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Unter "Gemeinsame Felder auf Modellierungsseiten" finden Sie Informationen zu den optionalen Feldern.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
