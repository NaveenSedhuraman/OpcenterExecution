Definieren von Antwortsätzen

Das Modellierungsobjekt "Antwortsatz" ermöglicht es Ihnen, verschiedene Sätze von Antworten zu konfigurieren, die Benutzern angezeigt werden, wenn diese ein Checklistenelement abarbeiten oder eine Checklistenfragen beantworten. Ein Antwortsatz kann z. B. Antworten enthalten wie "Ja", "Nein" oder "Ausstehend". Der Satz kann einem Checklistenelement zugewiesen werden, wie "Hat das Produkt alle Qualitätssicherungs-Tests bestanden?"

Beim Definieren eines Antwortsatzes

Sie legen einen Antwortsatz fest, wenn Sie ein Checklistenelement bei der Definition der Checklistenvorlage hinzufügen. Weitere Informationen zur Definition von Checklistenvorlagen finden Sie unter "Definieren von Checklistenvorlagen".

Siemens empfiehlt maximal fünf Antworten pro Antwortsatz.

"Antwortsatz" ist ein optionales Feld in der Definition einer Checklistenvorlage.

Vorgehensweise zum Definieren eines Antwortsatzes

Gehen Sie folgendermaßen vor, um einen Antwortsatz zu definieren:

1. Öffnen Sie die Seite Antwortsatz. Die Seite "Antwortsatz" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen in das Feld Antwortsatz ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Hinzufügen von Antworten zu einem Antwortsatz

Gehen Sie folgendermaßen vor, um einem Antwortsatz Antworten hinzuzufügen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren eines Antwortsatzes" aus.

Oder

Wählen Sie eine vorhandene Antwortsatzinstanz aus.

2. Klicken Sie auf Neue Zeile hinzufügen in der Tabelle "Antwortsatz". Eine neue Zeile wird der Tabelle hinzugefügt.
3. Geben Sie eine Zahl in die Spalte Nummer ein.
4. Geben Sie eine Antwort in die Spalte Antwortbeschriftung ein.
5. Wiederholen Sie die Schritte 2 bis 4, um zusätzliche Antworten hinzuzufügen.
6. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
