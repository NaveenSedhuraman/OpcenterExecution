Festlegen von Drucketikettendefinitionen

Die Drucketikettendefinition ist das Objekt, das die Netzwerkadresse aller vordefinierten Etikettenvorlage festlegt. Jede Drucketikettendefinitions-Instanz enthält:

• Die Netzwerkadresse der Etikettenvorlage
• Die Etiketten-Tags (in der Etikettenvorlage referenzierte Variablen)

Beim Festlegen der Drucketikettendefinitionen

Beachten Sie Folgendes, wenn Sie Drucketikettendefinition festlegen:

• Der Objektname darf keine ungültigen Zeichen enthalten. Z. B. darf kein Apostroph verwendet werden.
Hinweis: Informationen zu gültigen Zeichen und reservierten Wörtern finden Sie im Opcenter Execution Medical Device and Diagnostics Designer-Benutzerhandbuch und im Opcenter Execution Core Designer-Benutzerhandbuch.
• "Drucketikettendefinition" ist ein optionales Feld in der Definition einer Spezifikation
• Für Drucketikettendefinitionen muss mindestens ein Etiketten-Tag in der Tabelle "Etiketten-Tag" definiert sein.

Vorgehensweise zum Definieren einer Drucketikettendefinition

Gehen Sie folgendermaßen vor, um eine Drucketikettendefinition festzulegen:

1. Öffnen Sie die Seite Drucketikettendefinition. Die Seite "Drucketikettendefinition" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für diese Drucketikettendefinition in das Feld Drucketikettendefinition ein.
4. Geben Sie einen Revisionsnamen in das Feld Revision ein.
5. Erweitern Sie den Abschnitt Details.
6. Geben Sie die Netzwerkadresse der Etikettenvorlagendatei in das Feld Etikettenvorlage ein.
7. Geben Sie einen Wert in das Feld Anfangstrennzeichen ein.
8. Geben Sie einen Wert in das Feld Endetrennzeichen ein.
9. Führen Sie die Schritte unter "Vorgehensweise zum Hinzufügen von Tag-Informationen zur Drucketikettendefinition" aus.

Vorgehensweise zum Hinzufügen von Tag-Informationen zur Drucketikettendefinition

Gehen Sie folgendermaßen vor, um Tag-Informationen zur Drucketikettendefinition hinzuzufügen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren einer Drucketikettendefinition".
2. Klicken Sie in der Tabelle "Etiketten-Tag" auf Neue Zeile hinzufügen, um die Tag-Informationen dieser Drucketikettendefinition hinzuzufügen. Eine neue Zeile wird angezeigt.
3. Geben Sie einen Namen für das Etiketten-Tag in das Feld Etiketten-Tag-Name ein.
4. Geben Sie einen Ausdruck in das Feld Ausdruck ein.
5. Wiederholen Sie die Schritte 2 bis 4, um zusätzliche Etiketten-Tags hinzuzufügen.
6. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
7. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
