Zuweisen einer Ressourcengruppe und eines automatischen Ablaufs zu einer Spezifikation

Eine Spezifikation (Spez.) definiert die Aktivitäten, die in einem Schritt ausgeführt werden, und wird von einem Schritt in einem Arbeitsplan referenziert. Mehrere Schritte in einem Arbeitsplan können dieselbe Spezifikation verwenden.

Hinweis: Spezifikationen referenzieren viele andere Modellierungsobjekte, einschließlich Operation und Einrichtung. Spezifikationen enthalten auch detaillierte Ablaufplanungs- und Verarbeitungsparameterinformationen.

Optional können Sie einer Spezifikation eine Ressourcengruppe und einen automatischen Ablauf zuweisen. Indem Sie einer Spezifikation eine Ressourcengruppe zuweisen, legen Sie fest, welche Ressourcen zur Auswahl verfügbar sind, wenn ein Container bei einer Spezifikation abgemeldet wird. Indem Sie einer Spezifikation einen automatischen Ablauf zuweisen, legen Sie die Aufgaben fest, die abgeschlossen werden müssen, bevor der Container bei der Spezifikation abgemeldet werden kann.

Eine Spezifikation kann immer nur jeweils eine Ressourcengruppe und einen automatischen Ablauf referenzieren. Allerdings kann ein automatischer Ablauf eine Serie von Aufgabenlisten referenzieren, die mehrere Aufgaben enthalten.

Weitere Informationen zu Spezifikationen finden Sie unter "Definieren von Spezifikationen".

Vorgehensweise zum Zuweisen einer Ressourcengruppe zu einer Spezifikation

Gehen Sie folgendermaßen vor, um einer Spezifikation eine Ressourcengruppe zuzuweisen:

1. Öffnen Sie die Seite Spez. Die Seite "Spez." wird innerhalb der Seite "Modellierung" angezeigt.
2. Führen Sie die Schritte unter "Vorgehensweise zum Definieren einer Spezifikation " aus.

Oder

Wählen Sie eine vorhandene Spezifikationsinstanz aus. Die Seite "Spez." zeigt Informationen zur ausgewählten Spezifikation an.

3. Wählen Sie einen Arbeitsplatz oder eine Arbeitszellengruppe aus dem Feld Ressourcengruppe aus.
4. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.

Vorgehensweise zum Zuweisen eines automatischen Ablaufs zu einer Spezifikation

Gehen Sie folgendermaßen vor, um einer Spezifikation einen automatischen Ablauf zuzuweisen:

1. Öffnen Sie die Seite Spez. Die Seite "Spez." wird innerhalb der Seite "Modellierung" angezeigt.
2. Führen Sie die Schritte unter "Vorgehensweise zum Definieren einer Spezifikation " aus.

Oder

Wählen Sie eine vorhandene Spezifikation aus. Die Seite "Spez." zeigt Informationen zur ausgewählten Spezifikation an.

3. Wählen Sie einen automatischen Ablauf oder eine bestimmte Revision eines automatischen Ablaufs aus dem Feld Automatischer Ablauf aus.
4. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
