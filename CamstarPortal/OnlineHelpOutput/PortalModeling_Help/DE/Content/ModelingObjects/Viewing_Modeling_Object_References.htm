Anzeigen von referenzierten Modellierungsobjekt-Instanzen

Sie können alle Modellierungsobjekt-Instanzen anzeigen, die durch die ausgewählte Instanz referenziert werden. Sie können auch die Modellierungsobjekt-Instanzen anzeigen, die die Instanz referenzieren, die auf der Seite "Modellierung" ausgewählt wurde.

Hinweis: In der Anwendung werden keine Instanzen des Modellierungsobjekts "Filter-Tags" angezeigt (auch dann nicht, wenn es mit der ausgewählten Instanz verknüpft ist). Bei den Filter-Tags handelt es sich um eine Liste mit Kommas als Trennzeichen, die nicht mit dem Objekt als Referenz verknüpft ist.

Beim Anzeigen referenzierter Instanzen

Die Anzeige "Hierarchie" zeigt die ausgewählte Modellierungsobjekt-Instanz und alle Instanzen an, die sie referenziert. Durch Klicken auf die Schaltfläche "Hierarchie" auf der Seite der Modellierungsobjekt-Instanz wird das schreibgeschützte Popup "Hierarchie" angezeigt. Im Popup "Hierarchie" wird eine eingerückte Strukturansicht der Instanzen angezeigt, die durch die ausgewählte Objektinstanz referenziert werden. Die ausgewählte Instanz wird in schwarzer Fettschrift angezeigt. Die referenzierten Instanzen werden in normaler schwarzer Schrift angezeigt.

Die Ansicht "Hierarchie" enthält auch indirekte Referenzen. Die Anwendung zeigt z. B. die Spezifikationen an, die durch den Arbeitsplan referenziert werden, der durch die ausgewählte Produktinstanz referenziert wird.

Hinweis: In der Anwendung kann eine Zeitüberschreitung auftreten, wenn Sie versuchen, ein Modellierungsprotokoll mit einer großen Anzahl von Datensätzen anzuzeigen. Sie können die Anzahl an Millisekunden in der Timeout-Einstellung in der Datei "XMLClientConfig.xml" erhöhen. Die Datei "XMLClientConfig.xml" befindet sich standardmäßig in C:\Programme (x86)\Camstar\InSite Utility.

Beim Anzeigen referenzierter Instanzen

In der Ansicht "Wo genutzt" werden die Modellierungsobjekt-Instanzen angezeigt, die die ausgewählte Instanz direkt referenzieren. Durch Klicken auf die Schaltfläche "Wo genutzt" auf der Seite der Modellierungsobjekt-Instanz wird das schreibgeschützte Popup "Wo genutzt" angezeigt. Im Popup "Wo genutzt" werden alle Modellierungsobjekte und Modellierungsobjekt-Instanzen angezeigt, die auf die aktuelle Instanz verweisen.

Das Popup "Wo genutzt" zeigt keine Instanzen an, die die Instanz referenzieren, die die ausgewählte Instanz referenziert. Die Anwendung zeigt z. B. nicht das Produkt an, das die Arbeitspläne referenziert, die die ausgewählte Spezifikationsinstanz referenzieren. Die Anwendung zeigt nur die Stücklisten und Arbeitspläne an, die die ausgewählte Spezifikationsinstanz referenzieren.

Vorgehensweise zum Anzeigen von Instanzen, die von der ausgewählten Instanz referenziert werden

Gehen Sie folgendermaßen vor, um Instanzen anzuzeigen, die die ausgewählte Instanz referenzieren:

1. Wählen Sie ein Modellierungsobjekt aus.
2. Wählen Sie eine Instanz aus.
3. Klicken Sie auf Hierarchie. Das Popup "Hierarchie" wird angezeigt.
4. Klicken Sie auf Schließen, um das Popup zu schließen.

Vorgehensweise zum Anzeigen von Instanzen, die die ausgewählte Instanz referenzieren

Gehen Sie folgendermaßen vor, um Instanzen anzuzeigen, die die ausgewählte Instanz referenzieren:

1. Wählen Sie Modellierung im Menü Modellierung aus. Die Seite "Modellierung" wird angezeigt.
2. Wählen Sie ein Modellierungsobjekt aus. Die Seite des Objekts wird auf der Seite "Modellierung" angezeigt.
3. Wählen Sie eine vorhandene Instanz aus. Die Instanz und die Instanzinformationen werden angezeigt.
4. Klicken Sie auf Wo genutzt. Das Popup "Wo genutzt" wird angezeigt.
5. Klicken Sie auf Schließen, um das Popup zu schließen.
