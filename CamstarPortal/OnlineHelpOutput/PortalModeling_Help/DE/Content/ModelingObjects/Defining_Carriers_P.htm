Definieren von Behältern

Ein Behälter ist eine physische Einheit, etwa ein Gefäß, ein Bottich, ein Tablett, ein Fass oder ein Ausrüstungsteil, das verwendet wird, um die Materialien für Ihre Produktionsprozesse aufzubewahren. Das bekannteste Beispiel für einen Behälter ist ein Tablett. Behälter, die Materialien enthalten, werden durch die Fertigung bewegt.

Beim Definieren von Behältern

Sie können einen Container einem Behälter zuweisen. Transaktionen werden stets am Container ausgeführt, nicht am Behälter. Sie können die Containerdetails anzeigen und Transaktionen für den Container ausführen, indem Sie den Namen des Behälters angeben.

"Behälter" ist ein optionales Feld in der Modellierungsdefinition "Behältergruppe" Sie können einen Behälter zu einer Behältergruppe zuordnen, um eine logische Gruppierung bereitzustellen. Behältergruppen werden verwendet, um Auswahllisten gültiger Behälter zur Verfügung zu stellen und Benutzereingaben zu überprüfen.

Ein Behälter enthält die folgenden, optionalen Modellierungsdefinitionen: Fabrik, Behälterfamilie, Lieferant, Ressourcentyp und Ressourcenstatusmodell.

Geben Sie jedem Behälter einen eindeutigen Namen. Die Eindeutigkeit von Behältern erstreckt sich sowohl auf Ressourcen als auch auf Behälter. Wenn Sie z. B. eine Ressource namens "Tablett-0015543" erstellen, können Sie keinen Behälter namens "Tablett-0015543" mehr erstellen.

Beispiel eines Behälters: ein Bediener (Operator) stellt ein Medikament in der Fabrik her und teilt die 100 kg der ersten Zutat auf fünf Tabletts auf, auf denen nun jeweils 20 kg liegen. Die Tabletts in diesem Beispiel sind die Behälter.

Optional

Containerstatus

Gibt den Status an, den ein Container haben muss, um in den Behälter geladen zu werden. Zu den Standardstatus gehören:

• Beliebig: keine Statusbeschränkung
• Fehlgeschlagen: beschränkt auf Container mit dem Status "Fehlgeschlagen"
• Gut: beschränkt auf Container mit dem Status "Bestanden" oder "Gut"

Vorgehensweise zum Definieren eines Behälters

Gehen Sie folgendermaßen vor, um einen Behälter zu definieren:

1. Öffnen Sie die Seite Behälter. Die Seite "Behälter" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für den Behälter in das Feld Behälter ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
