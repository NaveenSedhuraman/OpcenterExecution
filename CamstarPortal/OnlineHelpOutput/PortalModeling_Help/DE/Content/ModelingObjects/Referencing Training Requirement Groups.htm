Referenzieren von Trainingsanforderungsgruppen

In diesem Thema wird erläutert, wie Trainingsanforderungsgruppen unterschiedlichen Modellierungsobjekten zugewiesen werden.

Beim Referenzieren von Trainingsanforderungsgruppen

Als Voraussetzung für die Referenzierung einer Trainingsanforderungsgruppe müssen Sie ein oder mehrere dieser Elemente definieren:

• Unternehmen
• Fabrik
• Produktfamilie
• Produkt
• Arbeitsbereich
• Operation
• Ressource
• Spezifikation
• Aufgabe

Diese Objekte können nur Trainingsanforderungsgruppen referenzieren und keine einzelnen Trainingsanforderungen. Auch wenn ein Modellierungsobjekt nur eine Trainingsanforderung hat, müssen Sie für diese Anforderung eine Gruppe erstellen, damit Sie sie dem Modellierungsobjekt zuweisen können.

Vorgehensweise zum Referenzieren einer Trainingsanforderungsgruppe

Gehen Sie folgendermaßen vor, um in einem Modellierungsobjekt eine Trainingsanforderungsgruppe zu referenzieren:

1. Wählen Sie auf der Registerkarte "Modellierung" ein erlaubtes Modellierungsobjekt aus. Die Definitionsseite wird für das ausgewählte Objekt angezeigt.
2. Erweitern Sie den Abschnitt In Verarbeitung.
3. Wählen Sie aus der Liste im Feld Trainingsanforderungsgruppe eine Gruppe aus.
4. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
