Definieren von Dokumentviewern

Ein Dokumentviewer gibt eine Anwendung an, mit der Dokumente angezeigt werden können, die in Opcenter EX MDD oder Opcenter EX CR definiert sind. Eine Dokumentviewerdefinition enthält den Speicherort der Anwendung und eine Befehlszeilen-Zeichenkette, die das Programm zum Anzeigen des Dokuments aufruft. Z. B. können Sie Dokumentviewer definieren, die Word, Excel oder das Portable Document Format (PDF) verwenden, um verschiedene Dokumente anzuzeigen.

Hinweis: Die Standardimplementierung von Opcenter EX MDD oder Opcenter EX CR verwendet dieses Objekt nicht. Das Objekt steht Ihnen jedoch in Ihrer benutzerdefinierten Implementierung zur Verfügung.

Felddefinitionen der Seite "Dokumentviewer"

Diese Tabelle definiert die Felder, die nur auf der Seite "Dokumentviewer" zu finden sind.

Siehe auch "Gemeinsame Felder auf Modellierungsseiten". Dieser Abschnitt enthält Informationen zu den Feldern, die alle Modellierungsobjekte aufweisen.

Feld

Definition

Typ

Allgemein

Aktuelles Verzeichnis

Verzeichnis, aus dem die Vieweranwendung gestartet wird.

Optional

Befehlszeile

Zeichenfolge mit dem Namen der zu startenden Anwendung und sämtlichen benötigten Argumenten. Die Zeichenkette enthält einen Bezeichner, der durch einen Dokumentbezeichner ersetzt wird, wenn die Befehlszeichenkette zur Laufzeit zusammengesetzt wird.

Z. B. ruft die Befehlszeichenkette "AcroRd32.exe%1" die Anwendung Acrobat® Reader® auf und ersetzt "%1" durch einen Dokumentbezeichner.

Optional

Vorgehensweise zum Definieren eines Dokumentviewers

Gehen Sie folgendermaßen vor, um einen Dokumentviewer zu definieren:

1. Öffnen Sie die Seite Dokumentviewer. Die Seite "Dokumentviewer" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie den Namen des Dokumentviewers in das Feld Dokumentviewer ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
