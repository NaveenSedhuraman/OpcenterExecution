Definieren von Benutzerkonstanten

Werte von Benutzerkonstanten können bei Bedarf in Berechnungsausdrücken verwendet werden. Dazu müssen sie allerdings zuerst als Benutzerkonstanten definiert werden. Mithilfe von Benutzerkonstanten können Sie konstante Werte erstellen und verwalten, die sich nicht (wie "Pi") oder nur selten ändern. Unter Umständen gibt es einige für Ihre Geschäftsprozesse typische Konstanten, die bei unterschiedlichen Berechnungen verwendet werden.

Beim Definieren von Benutzerkonstanten

Benutzerkonstantenausdrücke müssen mithilfe von einheitlichen Ausdrücken von Opcenter EX MDD oder Opcenter EX CR definiert werden. Informationen zu Konstantenwerten und Ausdrücken finden Sie Anhang für einheitliche Ausdrücke im Opcenter Execution Medical Device und Diagnostics Designer-Benutzerhandbuch und im Opcenter Execution Core Designer-Benutzerhandbuch.

Wenn Sie Literale für Benutzerkonstanten eingeben, schließen Sie den Eintrag in doppelte Anführungszeichen (") ein.

Die Benutzerkonstantenfunktion unterstützt die folgenden Operatoren:

• Addition
• Subtraktion
• Multiplikation
• Division
• Summe (mehrere Parameter)
• Bereich (mehrere Parameter)
• Unäres Minus/Plus-Operator
• Anwendungsfunktionen
• Alle auf Anwendungszeitstempeln basierenden Berechnungen

Vorgehensweise zum Definieren einer Benutzerkonstante

Gehen Sie folgendermaßen vor, um eine Benutzerkonstante zu definieren:

1. Öffnen Sie die Seite Benutzerkonstante. Die Seite "Benutzerkonstante" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für diese Instanz in das Feld Benutzerkonstante ein.
4. Wählen Sie einen Datentyp im Feld aus.
5. Geben Sie in das Feld Wert einen Wert ein, der dem Datentyp zugewiesen werden soll.
6. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
7. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
