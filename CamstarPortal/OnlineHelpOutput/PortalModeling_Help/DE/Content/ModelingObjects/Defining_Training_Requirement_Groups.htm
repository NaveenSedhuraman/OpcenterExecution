Definieren von Trainingsanforderungsgruppen

Eine Trainingsanforderungsgruppe ist eine Sammlung einzelner Trainingsanforderungen und Trainingsanforderungsgruppen (eine Gruppe von Untergruppen). Gruppieren von Trainingsanforderungen:

• Bietet eine bequeme Lösung zur Verknüpfung verschiedener Anforderungen mit einer bestimmten Modellierungseinheit, wie einer Operation oder einer Fabrik.
• Ermöglicht es, dass Modellierungsobjekte mehrere zusammenhängende Anforderungen zusammen statt jeweils nur eine Anforderung referenzieren.

Beim Definieren einer Trainingsanforderungsgruppe

Alle Trainingsanforderungen müssen zu einer Gruppe gehören, auch wenn die Gruppe nur eine Trainingsanforderungen umfasst. Dies ist erforderlich, weil Modellierungsobjekte eine Trainingsanforderungsgruppe und nicht einzelne Trainingsanforderungsdefinitionen referenzieren.

Überprüfen Sie, ob Sie eine oder mehrere Trainingsanforderungen definiert haben, bevor Sie ihre Trainingsanforderungsgruppen definieren.

Beachten Sie Folgendes:

• Trainingsanforderungsgruppen werden referenziert von:
• Unternehmen
• Fabrik
• Produkt
• Arbeitsbereich
• Operation
• Ressource
• Spezifikation
• Aufgabe
• Trainingsanforderungsgruppen referenzieren:
• Trainingsanforderung
• Trainingsanforderungsgruppe

Vorgehensweise zum Erstellen einer Trainingsanforderungsgruppen-Definition

Gehen Sie folgendermaßen vor, um eine Trainingsanforderungsgruppen-Definition zu definieren:

1. Öffnen Sie die Seite Trainingsanforderungsgruppe. Die Seite "Trainingsanforderungsgruppe" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie den Namen in das Feld Trainingsanforderungsgruppe ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Hinzufügen einer Trainingsanforderung

Gehen Sie folgendermaßen vor, um eine Trainingsanforderung hinzuzufügen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Erstellen einer Trainingsanforderungsgruppen-Definition" aus.
2. Klicken Sie in ein leeres Feld in der Tabelle "Einträge". 

Oder

Klicken Sie auf Neue Zeile hinzufügen in der Tabelle "Einträge". Eine neue Zeile wird angezeigt.

3. Wählen Sie aus dem Menü in der neuen Zeile eine Trainingsanforderung oder deren Revision aus.
4. Aktivieren Sie das Kontrollkästchen "Ist Standardrevision", falls dies die Standardrevision ist.
5. Klicken Sie außerhalb der Tabelle, um die Trainingsanforderung hinzuzufügen. Die Trainingsanforderung wird der Tabelle "Einträge" hinzugefügt.
6. Wiederholen Sie die Schritte 2 bis 5, um weitere Trainingsanforderungen hinzufügen.
7. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.

Vorgehensweise zum Hinzufügen einer Trainingsanforderungsgruppe

Gehen Sie folgendermaßen vor, um eine Trainingsanforderungsgruppe hinzuzufügen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Erstellen einer Trainingsanforderungsgruppen-Definition" aus.
2. Klicken Sie in ein leeres Feld in der Tabelle "Gruppen". 

Oder

Klicken Sie in der Tabelle "Gruppen" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.

3. Wählen Sie aus dem Menü in der neuen Zeile eine Gruppe oder deren Revision aus.
4. Aktivieren Sie das Kontrollkästchen "Ist Standardrevision", falls dies die Standardrevision ist.
5. Klicken Sie außerhalb der Tabelle, um die Gruppe hinzuzufügen. Die Gruppe wird der Tabelle "Gruppen" hinzugefügt.
6. Wiederholen Sie die Schritte 2 bis 5, um weitere Gruppen hinzuzufügen.
7. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
