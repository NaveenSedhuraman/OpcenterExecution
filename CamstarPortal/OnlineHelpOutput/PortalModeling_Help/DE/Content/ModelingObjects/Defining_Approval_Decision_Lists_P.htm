Definieren von Genehmigungsentscheidungslisten

Genehmigungsentscheidungslisten werden benutzt, um die Entscheidungen zu definieren, die den entsprechenden Genehmigern angezeigt werden, wenn die Anwendung eine Genehmigung für die Lösung eines Qualitätsdatensatzes verlangt. Jede Entscheidung wird einem vordefinierten Entscheidungstyp zugeordnet (z. B. "Genehmigt" oder "Abgelehnt").

Die Anwendung bestimmt auf Grundlage der Genehmigungsvorlage, die der Lösung oder dem Übergang zugeordnet ist, die zu verwendende Genehmigungsentscheidungsliste. Sie geben beim Definieren einer Genehmigungsvorlage eine Genehmigungsentscheidungsliste an. Weitere Informationen zum Definieren von Genehmigungsvorlagen finden Sie unter "Definieren von Genehmigungsvorlagen".

Beispiele für Genehmigungsentscheidungen

Es folgen zwei Beispiele für Genehmigungsentscheidungen:

• Ablehnen mit Kommentaren
• Genehmigen ohne Kommentare

Beim Definieren von Genehmigungsentscheidungslisten

Beachten Sie Folgendes, wenn Sie Genehmigungsentscheidungslisten definieren:

• Sie müssen mindestens eine Entscheidung für jede Genehmigungsentscheidungsliste definieren.
• Wahrscheinlich möchten Sie unterschiedliche Genehmigungsentscheidungslisten für jeden Genehmigungstyp definieren. Z. B. können Sie eine Entscheidungsliste speziell für Lösungen von Qualitätsdatensätzen definieren.
• "Genehmigungsentscheidungsliste" ist ein erforderliches Feld in der Definition einer Genehmigungsvorlage.

Vorgehensweise zum Definieren einer Genehmigungsentscheidungsliste

Gehen Sie folgendermaßen vor, um eine Genehmigungsentscheidungsliste zu definieren:

1. Öffnen Sie die Seite Genehmigungsentscheidungsliste. Die Seite "Genehmigungsentscheidungsliste" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für diese Genehmigungsentscheidungsliste in das Feld Genehmigungsentscheidungsliste ein.
4. Klicken Sie in der Tabelle "Entscheidungen" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
5. Geben Sie den Entscheidungsnamen in die Spalte Entscheidungsname ein.
6. Wählen Sie einen Genehmigungsstatus aus der Liste Genehmigungsstatus aus.
7. Wiederholen Sie die Schritte 4 bis 6, um eine weitere Entscheidung hinzuzufügen.
8. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
9. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
