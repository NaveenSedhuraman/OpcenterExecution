Definieren von Paketerstellungsvorlagen

Eine Paketerstellungsvorlage enthält die Details, den Arbeitsplan zur Änderungsverwaltung und die Bereitstellungsdetails für ein Paket. Außerdem enthält sie Angaben zu den einem Paket zugewiesenen Genehmigungs- und Projektmitarbeitervorlagen. Auf der Seite "Paketerstellungsvorlage" können Sie Vorlagen definieren, die auf der Seite "Paket erstellen" verwendet werden können. Durch Auswählen einer Vorlage wird die Seite "Paket erstellen" mit den vordefinierten Paketdetails und Bereitstellungsdetails und den entsprechenden Vorlagen für Genehmiger und Zusammenarbeit ausgefüllt. Die ausgewählte Vorlage wird anschließend zum Ausfüllen der Seite "Paket aktualisieren" verwendet. Die Registerkarten "Genehmiger" und "Projektmitarbeiter" werden dabei anhand der angegebenen Genehmigungs- und Projektmitarbeitervorlagen ausgefüllt. Wenn Sie ein Paket erstellen, ist es nicht erforderlich, jedes Mal alle Informationen manuell einzugeben.

Informationen zum Aktivieren der Zusammenarbeit und zum Erstellen eines Pakets finden Sie im Opcenter Execution Medical Device and Diagnostics Change Management-Benutzerhandbuch und im Opcenter Execution Core Change Management-Benutzerhandbuch.

Beim Definieren von Paketerstellungsvorlagen

Beachten Sie Folgendes:

• Die Genehmigungsvorlage wird aktiviert, wenn Sie einen Arbeitsplan zur Änderungsverwaltung öffnen, für den in der Änderungsverwaltungsspezifikation im ersten Schritt des Arbeitsplans zuzuweisende Genehmigungen konfiguriert sind. Weitere Informationen zum Konfigurieren von Arbeitsplänen finden Sie unter "Definieren von Arbeitsplänen zur Änderungsverwaltung".
• Die Projektmitarbeitervorlage wird nur aktiviert, wenn Sie "Eigentümer und Projektmitarbeiter" als Einstellung für "Inhaltsprojektmitarbeiter" festlegen.

Genehmigungsvorlage

Optional

Inhaltsprojektmitarbeiter

Wert, der angibt, ob nur der Eigentümer oder der Eigentümer und die Projektmitarbeiter dem Paket Inhalte zuweisen dürfen:

• Nur Eigentümer (der Standardwert) gibt an, dass nur der Benutzer mit Eigentümerberechtigungen dem Paket Inhalte zuweisen darf.
• Eigentümer und Projektmitarbeiter gibt an, dass der Benutzer mit Eigentümerberechtigungen und vom Eigentümer bestimmte Projektmitarbeiter dem Paket Inhalte zuweisen dürfen. Wenn Sie diese Option aktivieren, wird die Projektmitarbeitervorlage aktiviert.

Optional

Projektmitarbeitervorlage

Hinweis: Projektmitarbeitervorlagen werden in der Modellierung definiert.

Optional

Vorgehensweise zum Definieren einer Paketerstellungsvorlage

Gehen Sie folgendermaßen vor, um eine Paketerstellungsvorlage zu definieren:

1. Öffnen Sie die Seite Paketerstellungsvorlage. Die Seite "Paketerstellungsvorlage" wird innerhalb der Seite "Modellierung" angezeigt.
2. Geben Sie einen Namen für diese Paketerstellungsvorlage in das Feld Paketerstellungsvorlage ein.
3. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
4. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
