Definieren von Ressourcentypen

Der Ressourcentyp bestimmt einen Ausrüstungstyp, wie etwa Gewöhnliche Wartungsausrüstung, die mit einem Ressourcenstatusmodell verknüpft werden kann.

Beim Definieren eines Ressourcentyps

"Ressourcentyp" ist ein optionales Feld in der Modellierungsdefinition für Ressourcen Modellierungsdefinition.

"Ressourcentyp" enthält die optionalen Modellierungsdefinitionen "Ressourcenstatusmodell" und "UI-Einstellungen".

Felddefinitionen der Seite "Ressourcentyp"

In dieser Tabelle sind die Felder definiert, die nur auf der Seite "Ressourcentyp" zu finden sind.

Siehe auch "Gemeinsame Felder auf Modellierungsseiten". Dieser Abschnitt enthält Informationen zu den Feldern, die alle Modellierungsobjekte aufweisen.

Feld

Definition

Typ

Allgemein

Ressourcenstatusmodell

Das Ressourcenstatusmodell, das diesem Ressourcentyp zugeordnet ist.

Optional

UI-Einstellungen

UI-Einstellungen, die diesem Ressourcentyp zugeordnet werden sollen. UI-Einstellungen legen die Informationen fest, die für eine Ressource auf der Registerkarte "Status" auf der Seite "Ressourcenprotokoll" angezeigt werden sollen. UI-Einstellungen werden in der Modellierung definiert.

Optional

Vorgehensweise zum Definieren eines Ressourcentyps

Gehen Sie folgendermaßen vor, um einen Ressourcentyp zu definieren:

1. Öffnen Sie die Seite Ressourcentyp. Die Seite "Ressourcentyp" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen in das Feld Ressourcentyp ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
