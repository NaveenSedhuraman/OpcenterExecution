Definieren von Umlaufbestandsnachrichten

Es werden zwei Verfahren benötigt, um Umlaufbestandsnachrichten zu erstellen:

• Erstellen der Umlaufbestandsnachricht
• Konfigurieren der Kriterien zur Bewertung von Umlaufbestandsnachrichten. Weitere Informationen finden Sie unter "Angeben von auszuwertenden Umlaufbestandsnachrichten".

Sie müssen beide Verfahren in beliebiger Reihenfolge durchführen. Wenn ein Container mit den Attributen gestartet und bearbeitet wird, die Umlaufbestandsnachrichten zugewiesen sind, und wenn der Container den Verarbeitungsschritt erreicht, für den eine Umlaufbestandsnachricht definiert wurde, werden alle Nachrichten, die den Kriterien entsprechen, abgerufen.

Sie können die Umlaufbestandsnachricht auch so konfigurieren, dass eine E-Mail an einen anderen Empfänger als den Bediener gesendet wird (z. B. einen Techniker oder Vorarbeiter), wenn ein Container einen bestimmten Schritt im Produktionsprozess erreicht hat.

Übersicht über die Erstellung von Umlaufbestandsnachrichten

Sie müssen folgende Verfahren in der angegebenen Reihenfolge durchführen, um eine Umlaufbestandsnachricht zu erstellen:

1. Öffnen Sie die Seite Umlaufbestandsnachricht.
2. Starten Sie die neue Umlaufbestandsnachricht.
3. Geben Sie die Umlaufbestandsnachrichtendetails ein.

Die Schaltfläche ist nur für Definitionen aktiviert, die es Ihnen ermöglichen, Umlaufbestandsnachrichten hinzuzufügen, und nur nachdem diese Definition gespeichert wurden.

Anweisungen zum Ausführen dieser Aufgaben folgen.

Vorgehensweise zum Definieren von Umlaufbestandsnachrichten

Gehen Sie folgendermaßen vor, um Umlaufbestandsnachrichten zu definieren:

1. Öffnen Sie eine Instanz eines Modellierungsobjekts, das Umlaufbestandsnachrichten unterstützt. Die Modellierungsobjektseite wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Umlaufbestandsnachrichten. Die Seite "Umlaufbestandsnachrichten" wird angezeigt.
3. Klicken Sie auf Neu. Das Feld "Umlaufbestandsnachrichtentyp" wird angezeigt.
4. Wählen Sie einen Schlüssel aus der Liste Umlaufbestandsnachrichtentyp aus (standardmäßig ist hier "Alle Schlüssel" ausgewählt). Ein zusätzliches Feld wird angezeigt, wenn Sie "Operationsschlüssel" oder "Etikettenschlüssel" auswählen.
5. Haben Sie "Etikettenschlüssel" oder "Operationsschlüssel" ausgewählt?

Wenn Sie Folgendes ausgewählt haben . . .

Dann . . .

Etikettenschlüssel
a. Geben Sie eine Beschriftung in das Feld "Zu findendes Etikett" ein.
b. Gehen Sie zu Schritt 6.

Operationsschlüssel
a. Wählen Sie eine Operation aus der Liste "Zu findende Operation" aus.
b. Gehen Sie zu Schritt 6.

Alle Schlüssel

Gehen Sie zu Schritt 6.
6. Klicken Sie auf Fortsetzen. Die Felder für "Umlaufbestandsnachrichten" werden angezeigt.
7. Geben Sie eine Umlaufbestandsnachricht in das Feld Nachrichtentext ein.
8. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
9. Klicken Sie auf Nachricht speichern. Die Anwendung speichert die Umlaufbestandsnachricht und zeigt eine Erfolgsmeldung an.

So werden Benachrichtigungsziele zu Umlaufbestandsnachrichten hinzugefügt

Gehen Sie folgendermaßen vor, um die Definition von Umlaufbestandsnachrichtendetails abzuschließen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren von Umlaufbestandsnachrichten" aus.

Oder

Wählen Sie eine vorhandene Instanz einer Umlaufbestandsnachricht aus.

2. Erweitern Sie den Abschnitt Benachrichtigung.
3. Klicken Sie in der Tabelle "Benachrichtigungsziele" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
4. Wählen Sie eine E-Mail-Benachrichtigung aus der Liste Name aus.
5. Wiederholen Sie die Schritte 3 bis 4, um zusätzliche Benachrichtigungsziele hinzuzufügen.
6. Klicken Sie auf Nachricht speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass die Umlaufbestandsnachricht aktualisiert wurde.
