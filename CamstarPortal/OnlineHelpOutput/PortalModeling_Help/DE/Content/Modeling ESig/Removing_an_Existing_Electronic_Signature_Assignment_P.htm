Entfernen einer vorhandenen Zuweisung zur elektronischen Unterschrift

Sie können eine Anforderung zur elektronischen Unterschrift, die einem Service zugewiesen ist, entfernen. Verwenden Sie die Seite "Elektr. Unterschrift für Modellierung", um die Zuweisung zu entfernen. Lesen Sie "Ändern einer vorhandenen Zuweisung zur elektronischen Unterschrift", wenn Sie die Zuweisung ändern möchten, statt sie zu entfernen.

Beim Entfernen einer vorhandenen Zuweisung zur elektronischen Unterschrift

Wenn Sie die Zuweisung auf "Kein" setzen, wird eine Überschreibung erzeugt, die angibt, dass für den Service auch dann keine Unterschrift erforderlich ist, wenn dem Eltern-Service eine Unterschrift zugewiesen ist. Alternativ können Sie die Zuweisung eines Service entfernen, indem Sie für die Zuweisung "Erben" festlegen. "Erben" besagt, dass der betreffende Service nicht über eine direkte Zuweisung verfügt, auf Grundlage der Zuweisung eines Eltern-Service aber dennoch eine Unterschrift erforderlich sein kann.

Vorgehensweise zum Entfernen einer vorhandenen Zuweisung für elektronische Unterschriften

Gehen Sie folgendermaßen vor, um eine vorhandene Zuweisung zur elektronischen Unterschrift zu entfernen:

1. Öffnen Sie die Seite Elektr. Unterschrift für Modellierung. Die Seite "Elektr. Unterschrift für Modellierung" wird angezeigt.
2. Wählen Sie den Modellierungsobjekt-Service mit der Zuweisung zur elektronischen Unterschrift, die Sie entfernen möchten, aus der Liste im Abschnitt "Services" aus.
3. Wählen Sie die Anforderung zur elektronischen Unterschrift, die die Zuweisung enthält, aus der entsprechenden Aktionsliste (Erstellen, Aktualisieren, Löschen oder Sperren/Entsperren) im Abschnitt "Zuweisung einer Unterschriftsanforderung" aus.
4. Wählen Sie Kein aus der Liste aus, um Zuweisungen zu dem Service zu überschreiben.

Oder

Wählen Sie Erben aus der Liste aus, wenn die Anforderungszuweisung vom Eltern-Service geerbt werden soll.

5. Klicken Sie auf Aktualisieren. Die Anforderung zur elektronischen Unterschrift ist dem Service nicht mehr zugewiesen.
