Anzeigen des Protokolls für eine elektronische Unterschrift

Details zur elektronischen Unterschrift werden im Rahmen des Modellierungsprotokolls für das Modellierungsobjekt angezeigt. Die Aktion, die der Zuweisung einer Anforderung zur elektronischen Unterschrift zugeordnet ist, wird in der Aktionsspalte des Protokolls aufgeführt.

Weitere Informationen zum Modellierungsprotokoll finden Sie unter "Modellierungsprotokoll".

Felddefinitionen des Popups "Modellierungsprotokoll"

Definitionen der Felder im Popup finden Sie unter "Felddefinitionen des Popups "Modellierungsprotokoll"".

Vorgehensweise zum Anzeigen des Popups "Modellierungsprotokoll" für eine elektronische Unterschrift

Gehen Sie folgendermaßen vor, um das Popup "Modellierungsprotokoll" für eine elektronische Unterschrift anzuzeigen:

1. Öffnen Sie ein Modellierungsobjekt, das eine Anforderung zur elektronischen Unterschrift enthält. Die Modellierungsobjektseite wird mit einer Liste vorhandener Instanzen dieses Objekts angezeigt.
2. Wählen Sie die vorhandene Modellierungsobjektinstanz aus, deren Modellierungsprotokoll angezeigt werden soll. Die Seite für diese Modellierungsobjekt-Instanz wird angezeigt.
3. Klicken Sie auf Protokoll. Das Popup "Modellierungsprotokoll" wird in der reduzierten Ansicht angezeigt.
4. Möchten Sie die Ergebnisse auf einen bestimmten Datumsbereich einschränken?
