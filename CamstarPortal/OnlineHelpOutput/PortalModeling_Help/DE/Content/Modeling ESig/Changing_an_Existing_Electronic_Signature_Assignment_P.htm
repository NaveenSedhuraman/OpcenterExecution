Ändern einer vorhandenen Zuweisung zur elektronischen Unterschrift

Sie können die einem Dienst (Service) zugeordnete Anforderung zur elektronischen Unterschrift ändern. Verwenden Sie die Seite "Elektr. Unterschrift für Modellierung", um die Änderungen in der Zuweisung vorzunehmen. Sie können die Zuweisung einer Anforderung zur elektronischen Unterschrift auch vollständig entfernen. Weitere Informationen finden Sie unter "Entfernen einer vorhandenen Zuweisung zur elektronischen Unterschrift".

Vorgehensweise zum Ändern einer vorhandenen Zuweisung für elektronische Unterschriften

Gehen Sie folgendermaßen vor, um eine vorhandene Zuweisung zur elektronischen Unterschrift zu ändern:

1. Öffnen Sie die Seite Elektr. Unterschrift für Modellierung. Die Seite "Elektr. Unterschrift für Modellierung" wird angezeigt.
2. Wählen Sie den Modellierungsobjekt-Dienst mit der Anforderung zur elektronischen Unterschrift, die sie ändern möchten, aus der Liste im Abschnitt "Services" aus.
3. Wählen Sie eine Anforderung zur elektronischen Unterschrift aus der entsprechenden Aktionsliste Erstellen, Aktualisieren, Löschen oder Sperren/Entsperren in der Tabelle "Zuweisung einer Unterschriftsanforderung" aus, um eine vorhandene Anforderung für den Dienst zu ändern.
4. Klicken Sie auf Aktualisieren. Die neue Anforderung zur elektronischen Unterschrift wird dem Dienst (Service) zugewiesen.
