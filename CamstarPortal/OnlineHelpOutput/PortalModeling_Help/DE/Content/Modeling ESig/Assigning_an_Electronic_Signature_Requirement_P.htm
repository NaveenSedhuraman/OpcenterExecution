Zuweisen einer Anforderung zur elektronischen Unterschrift

Die Seite "Elektr. Unterschrift für Modellierung", die über das Menü "Modellierung" verfügbar ist, ermöglicht es Ihnen, vier verschiedenen Aktionen eines Modellierungsobjekt-Dienstes eine Anforderung zur elektronischen Unterschrift zuzuweisen.

Aktion

Erfordert eine elektronische Unterschrift, wenn Sie . . .

Erstellen

Eine neue Instanz eines Modellierungsobjekts erstellen.

Aktualisieren

Eine vorhandene Instanz eines Modellierungsobjekts aktualisieren.

Löschen

Eine vorhandene Instanz eines Modellierungsobjekts löschen.

Sperren/Entsperren

Sperrt oder entsperrt eine vorhandene Instanz eines Modellierungsobjekts.

Wenn Sie eine Anforderung zur elektronischen Unterschrift zuweisen, wendet die Anwendung die Zuweisung auf den ausgewählten Dienst an und zeigt die Zuweisungen im Abschnitt "Aufgelöste Unterschriftsanforderung" an.

Beim Zuweisen einer Anforderung zur elektronischen Unterschrift

Bevor Sie eine Anforderung zur elektronischen Unterschrift zuweisen können, müssen Sie bereits über "Elektr. Unterschrift: Anforderung" in der Modellierung elektronische Unterschriften erstellt haben. Weitere Informationen zum Definieren von elektronischen Unterschriften finden Sie unter "Definieren von Anforderungen zur elektronischen Unterschrift".

Eine elektronische Unterschrift kann keinem Service zugewiesen werden, der verwendet wird, um die Beziehung zwischen elektronischer Unterschrift und Wartungsservice zu verwalten (z. B. MaintenanceESigReq). Sie können den Zugriff zum Ausführen dieses Wartungsservices auch über die in der Anwendung verfügbare rollenbasierte Zugriffssteuerungssicherheit einschränken.

Vorgehensweise zum Zuweisen einer Anforderung zur elektronischen Unterschrift

Gehen Sie folgendermaßen vor, um eine Anforderung zur elektronischen Unterschrift zuzuweisen:

1. Öffnen Sie die Seite Elektr. Unterschrift für Modellierung. Die Seite "Elektr. Unterschrift für Modellierung" wird angezeigt.
2. Wählen Sie den Modellierungsobjekt-Dienst aus der Liste in der Tabelle "Services" aus.
3. Wählen Sie eine Anforderung zur elektronischen Unterschrift aus der entsprechenden Aktionsliste Erstellen, Aktualisieren, Löschen oder Sperren/Entsperren in der Tabelle "Zuweisung einer Unterschriftsanforderung" aus, um dem Service (Dienst) eine Anforderung zuzuweisen.
4. Klicken Sie auf Aktualisieren. Die Anforderung zur elektronischen Unterschrift wird dem Dienst zugewiesen.
