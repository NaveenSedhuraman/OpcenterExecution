Zuordnen von Mitarbeitern zu Rollengruppen für elektronische Unterschrift

Sie müssen Ihre Mitarbeiter zu Rollengruppen für elektronische Unterschrift zuordnen, damit sie elektronische Unterschriften leisten können. Wenn Sie einen Mitarbeiter der entsprechenden Rollengruppe für elektronische Unterschrift zuordnen, kann dieser alle Transaktionen elektronisch unterschreiben, die der Rolle zugeordnet sind.

Muss z. B. George Jackson gewährleisten, dass Schachteln ordentlich verpackt und versiegelt wurden, können Sie ihm die Rollengruppe für elektronische Unterschrift "Schachtelüberprüfer" zuordnen. Wird angenommen, dass die Rollengruppe "Schachtelüberprüfer" die beiden Rollen "Schachtelzukleber" und "Schachtelprüfer" umfasst, hat George die entsprechende Berechtigung, Unterschriften für beide Forderungen zu leisten.

Wenn Sie Mitarbeiter einer elektronischen Rollengruppen zuweisen

Sie können jedem Mitarbeiter auf der Seite "Mitarbeiter" eine elektronische Unterschrift zuweisen, nachdem die Rollengruppe für elektronische Unterschrift durch die "Elektr. Unterschrift: Rollengruppe" definiert wurde.

Felddefinitionen der Seite "Mitarbeiter"

Weitere Informationen zur Seite "Mitarbeiter" sowie dazu, wie Mitarbeiter definiert werden, finden Sie unter "Definieren von Mitarbeitern".

Vorgehensweise zum Verknüpfen eines Mitarbeiters mit einer Rollengruppe für elektronische Unterschrift

Gehen Sie folgendermaßen vor, um einen Mitarbeiter einer Rollengruppe für elektronische Unterschrift zuzuordnen:

1. Öffnen Sie die Seite Mitarbeiter. Die Seite "Mitarbeiter" wird auf der Seite "Modellierung" angezeigt.
2. Klicken Sie auf eine vorhandene Mitarbeiterdefinition.

Oder

Klicken Sie auf Neu, um eine neue Mitarbeiterdefinition zu erstellen. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.

3. Geben Sie den Namen des Mitarbeiters in das Feld Name ein, wenn Sie einen neuen Mitarbeiter hinzufügen.
4. Wählen Sie in der Liste Elektr. Unterschrift: Rollengruppe eine Rollengruppe aus.
5. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
