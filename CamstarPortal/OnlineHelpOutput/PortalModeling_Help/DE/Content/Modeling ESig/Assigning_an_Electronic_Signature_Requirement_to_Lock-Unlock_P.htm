Zuweisen einer Anforderung zur elektronischen Unterschrift zu "Sperren/Entsperren"

Opcenter EX MDD und Opcenter EX CR ermöglichen es Ihnen, dem Sperren/Entsperren-Ereignis über die Seite "Elektr. Unterschrift für Modellierung" eine Anforderung zur elektronischen Unterschrift zuzuweisen. Dadurch wird gewährleistet, dass eine oder mehrere Unterschriften erforderlich sind, bevor ein revisioniertes Objekt gesperrt oder entsperrt werden kann. Ein revisioniertes Objekt kann auch eine Sperren/Entsperren-Zuordnung von einem Elternteil erben ("Erben") oder kann die Zuordnung des Elternteils überschreiben ("Kein").

Vorgehensweise zum Zuweisen einer Anforderung zur elektronischen Unterschrift zum Sperren/Entsperren

Gehen Sie folgendermaßen vor, um eine vorhandene Anforderung zur elektronischen Unterschrift zu sperren:

1. Öffnen Sie die Seite Elektr. Unterschrift für Modellierung. Die Seite "Elektr. Unterschrift für Modellierung" wird angezeigt.
2. Wählen Sie einen Dienst für revisionierte Objekte in der Liste im Abschnitt "Services" aus.
3. Wählen Sie eine Anforderung zur elektronischen Unterschrift aus der Sperren/Entsperren-Liste im Abschnitt "Zuweisung einer Unterschriftsanforderung" aus, um sie den Sperren- und Entsperren-Ereignissen zuzuweisen.
4. Klicken Sie auf Aktualisieren. Die neue Anforderung zur elektronischen Unterschrift wird dem Dienst zugewiesen und eine Unterschrift ist nun erforderlich, bevor das revisionierte Objekt gesperrt oder entsperrt werden kann.
