Definieren von Rollengruppen für elektronische Unterschriften

Eine Rollengruppe für elektronische Unterschriften ist eine Gruppe ähnlicher oder verwandter Rollen, die einem Mitarbeiter zugewiesen werden kann, damit dieser mehrere Rollen für elektronische Unterschriften erhält. Eine Rollengruppe besteht aus einzelnen Rolleneinträgen und vorher festgelegten Rollengruppen.

Wenn z. B. Mary Smith Unterschriften leisten muss, um zu bestätigen, dass Kartons gefüllt und verschlossen sind, können Sie die Rollengruppe "Kartonoperationen " erstellen und dieser Gruppe die Rollen "Kartonbefüller" und "Kartonverschließer" zuweisen. Danach können Sie Mary Smith der Rollengruppe "Kartonoperationen" zuweisen.

Beim Definieren von Rollengruppen für elektronische Unterschriften

"Elektr. Unterschrift: Rollengruppe" ist eine optionale Modellierungsdefinition für "Mitarbeiter".

"Elektr. Unterschrift: Rollengruppe" enthält die optionale Modellierungsdefinition "Rolle".

Die Schaltfläche "Aufgelöste Einträge" ermöglicht es Ihnen, eine Liste aller Werte anzuzeigen, die in der Eintragsliste für diese Gruppe und alle darin enthaltenen Gruppen angegeben sind.

Vorgehensweise zum Definieren einer Rollengruppe für elektronische Unterschriften

Gehen Sie folgendermaßen vor, um eine Rollengruppe für elektronische Unterschriften zu definieren:

1. Öffnen Sie die Seite Elektr. Unterschrift: Rollengruppe. Die Seite "Elektr. Unterschrift: Rollengruppe" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie den Namen der Gruppe in das Feld Elektr. Unterschrift: Rollengruppe ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Hinzufügen von Einträgen zu einer Rollengruppe für elektronische Unterschriften

Gehen Sie folgendermaßen vor, um Einträge zu einer Rollengruppe für elektronische Unterschriften hinzuzufügen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren einer Rollengruppe für elektronische Unterschriften" aus.

Oder

Wählen Sie eine vorhandene Instanz einer Rollengruppe für elektronische Unterschriften aus.

2. Klicken Sie in der Tabelle "Rollen" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
3. Wählen Sie eine Rolle aus der Liste aus.
4. Wiederholen Sie die Schritte 2 bis 3, um der Tabelle weitere Rollen hinzuzufügen.
5. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.

Vorgehensweise zum Hinzufügen von Gruppen zu einer Rollengruppe für elektronische Unterschriften

Gehen Sie folgendermaßen vor, um einer Rollengruppe für elektronische Unterschriften weitere Gruppen hinzuzufügen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Definieren einer Rollengruppe für elektronische Unterschriften" aus.

Oder

Wählen Sie eine vorhandene Instanz einer Rollengruppe für elektronische Unterschriften aus.

2. Klicken Sie in der Tabelle "Gruppen" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
3. Wählen Sie in der Liste eine Gruppe aus, die sie (als Untergruppe) hinzufügen möchten.
4. Wiederholen Sie die Schritte 2 bis 3, um der Tabelle weitere Gruppen hinzuzufügen.
5. Klicken Sie auf Speichern. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass das Modellierungsobjekt aktualisiert wurde.
