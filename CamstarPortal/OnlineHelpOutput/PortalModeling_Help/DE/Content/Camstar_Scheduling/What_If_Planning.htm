Was-wäre-wenn-Planung

Opcenter Execution Core Scheduling umfasst Funktionen für eine Was-wäre-wenn-Planung, um zukünftige oder Was-wäre-wenn-Ablaufplanungsszenarien basierend auf einem geplanten Produktmix zu analysieren. In diesen Situationen müssen dennoch einige oder alle Produktionsaufträge erstellt werden. Die Anwendung sendet Produkte und deren zugehörige ERP-Routen von Opcenter EX MDD oder Opcenter EX CR an Preactor. Anhand dieser Daten kann der Benutzer hypothetische Aufträge in Preactor erstellen, um Was-wäre-wenn-Zeitplanungsszenarien zu analysieren.

Sie müssen Opcenter Execution Core Scheduling aktivieren, um die Was-wäre-wenn-Planung verwenden zu können. Sie müssen eine ERP-Route definieren. Sie müssen ebenfalls ein Produkt definieren und die ERP-Route dem Produkt zuweisen. Weitere Informationen finden Sie unter "Aktivieren von Opcenter Execution Core Scheduling auf der Seite "Fabrik"". Informationen zum Definieren von ERP-Routen finden Sie unter "Beim Definieren von ERP-Routen für die Verwendung mit Opcenter Execution Core Scheduling". Informationen zum Definieren von Produkten finden Sie unter "Beim Definieren von Produkten für die Verwendung mit Opcenter Execution Core Scheduling".
