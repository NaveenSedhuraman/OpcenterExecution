Definieren von Auftragsstatus

Ein Auftragsstatus gibt den Status eines Produktionsauftrags an. Mit dem Objekt "Auftragsstatus" können Sie den Status "Offen" oder "Geschlossen" zuweisen.

Beim Definieren von Auftragsstatus

"Auftragsstatus" ist ein optionales Feld in der Modellierungsdefinition "Produktionsauftrag". Sie müssen der Instanz eines Produktionsauftrags allerdings einen Auftragsstatus mit dem Status "Offen" zuweisen, damit eine Verwendung mit Opcenter Execution Core Scheduling möglich ist. Weitere Informationen finden Sie im Opcenter Execution Core Scheduling-Benutzerhandbuch.

Das Kontrollkästchen "Ist bestätigt" ist verfügbar, wenn Opcenter Execution Electronics installiert ist. Es ermöglicht Ihnen, den Auftragsstatus für die Produktion zu bestätigen. Weitere Informationen finden Sie im Opcenter Execution Electronics-Benutzerhandbuch.

Einzelheiten zu den Feldern, die alle Modellierungsobjekte gemeinsam haben, finden Sie unter "Gemeinsame Felder auf Modellierungsseiten“.

Beim Definieren von Auftragsstatus für die Verwendung mit Opcenter Execution Semiconductor
Hinweis: Die nachstehend beschriebene Funktionalität ist nur verfügbar, wenn Opcenter Execution Semiconductor installiert ist.

Opcenter Execution Semiconductor erfordert Folgendes:

• Ein Arbeitsauftrag muss jedem mit Preactor geplanten Los zugewiesen sein. Sie können Arbeitsaufträge zu Losen zuweisen, indem Sie die Seite "Los starten" oder "Los erstellen" verwenden oder vorläufige Planungsdaten auf einer der Seiten zur Ablaufplanung eingeben.
• Der Arbeitsauftrag, dem ein Auftragsstatus zuzuweisen ist. Sie müssen den Auftragsstatus als "Offen" und mit aktiviertem Kontrollkästchen "Ist verfügbar" konfigurieren.

Vorgehensweise zum Definieren eines Auftragsstatus

Gehen Sie folgendermaßen vor, um einen Auftragsstatus zu definieren:

1. Öffnen Sie die Seite Auftragsstatus. Die Seite "Auftragsstatus" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie im Feld Auftragsstatus einen Namen für diesen Auftragsstatus ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
