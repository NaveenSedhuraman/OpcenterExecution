Revisionsverwaltung

Ein revisioniertes Objekt wird für Objekte verwendet, die einen Namen und eine Revision haben. Mithilfe von Revisionen können Sie mehrere Versionen derselben Modellierungsdefinition verwalten, wobei eine Revision als Standardrevision festgelegt wird. Standardmäßig wird die neueste Revision als Standardrevision festgelegt, bis Sie eine andere Revision angeben. Auch andere Revisionen können aktiv und verwendbar sein, aber die Standardrevision ist die Revision, die derzeit in der Modellierung und in der Fertigung als Standard festgelegt ist.

Standardrevision

Die Funktion "Standardrevision" reduziert den Wartungsaufwand am Modell und gewährleistet die Verwendung korrekter Revisionen in der Fertigung.

Wenn Sie z. B. einen Arbeitsplan anlegen, erstellen Sie Schritte, indem Sie Spezifikationen (oder andere Arbeitspläne) referenzieren. Wenn Sie eine Spezifikation referenzieren, müssen Sie entweder eine bestimmte Revision oder die Standardrevision auswählen. Wenn Sie:

• Auf eine bestimmte Revision verweisen, verwendet der Arbeitsplan immer diese Revision, auch wenn neue Revisionen erstellt werden.
• Auf eine Standardrevision verweisen, verwendet der Arbeitsplan immer die Standardrevision unabhängig von der tatsächlichen Revisionsnummer.

Der Vorteil liegt darin, dass Sie die Standardrevision referenzieren können und so nicht jedes Mal die Arbeitspläne, die diese Spezifikation verwenden, überarbeiten müssen, wenn eine neue Revision der Spezifikation zur Verwendung zugelassen wird.

Nur eine Revision kann als Standardrevision festgelegt werden, aber Sie können diese Zuweisung jederzeit ändern und eine andere Revision als Standardrevision festlegen.

Beim Definieren einer neuen Revision

Sie können eine neue Revision einer vorhandenen Instanz eines revisionierten Objekts definieren, indem Sie eine von zwei Schaltflächen verwenden:

• Mit der Schaltfläche "Neue Revision" wird eine Kopie der Revision erstellt, in der alle Felder bis auf den Namen leer sind. Verwenden Sie "Neue Revision", wenn die meisten Attribute für die neue Revision eindeutig sind.
• Wenn Sie auf die Schaltfläche "Revision kopieren" klicken, wird ein Duplikat des revisionierten Objekts mit allen Informationen der Originalrevision außer der Revisionsnummer erstellt.

Die erste Instanz eines erstellten revisionierten Objekts ist standardmäßig die Standardrevision. Jede Revision, die anschließend definiert wird, ist standardmäßig die neue Standardrevision.

Revisionierte Objekte in der Liste "Instanzauswahl"

Die Liste "Instanzauswahl" wird in der erweiterten Ansicht angezeigt. So können Sie die Instanzen und die mit ihnen verbundenen Revisionen sehen.

Durch Klicken auf den Namen eines revisionierten Objekts wird die Wartungsseite für die Standardrevision dieser Instanz angezeigt. Durch Klicken auf den Namen einer bestimmten Revision wird die Wartungsseite für diese Revision angezeigt.

Die Standardrevision ist durch ein Häkchen im Feld hinter dem Revisionsnamen markiert.

Vorgehensweise zum Definieren einer neuen Revision

Gehen Sie folgendermaßen vor, um eine neue Revision zu definieren:

1. Klicken Sie auf die Seite, die das revisionierte Objekt enthält, für das Sie eine neue Revision definieren möchten. Die Liste "Instanzauswahl" wird angezeigt.
2. Klicken Sie auf den Instanznamen. Eine Seite mit der ausgewählten Instanz wird angezeigt.
3. Klicken Sie auf Neue Revision. Die Seite für die neue Revision wird angezeigt.
4. Geben Sie die erforderlichen Informationen ein, z. B. Operation, Produkttyp usw.
5. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
6. Klicken Sie auf Speichern. Die Anwendung aktualisiert die Revision und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Löschen einer Instanz oder Revision

Gehen Sie folgendermaßen vor, um eine Instanz oder eine Revision zu löschen:

1. Klicken Sie auf die Seite, die das revisionierte Objekt enthält, für das Sie eine Instanz oder Revision löschen möchten. Die Liste "Instanzauswahl" wird angezeigt.
2. Klicken Sie auf eine bestimmte Instanz oder Revision.
3. Führen Sie einen der folgenden Schritte aus:

Wenn Sie Folgendes löschen möchten . . .

Dann . . .

Alle Revisionen
a. Wählen Sie die Standardrevision des Modellierungsobjekts aus. Die Schaltfläche "Alles löschen" wird angezeigt.
b. Klicken Sie auf Alles löschen. Das Dialogfeld "Löschen" wird angezeigt.
c. Gehen Sie zu Schritt 4.

Eine bestimmte Revision
a. Wählen Sie eine bestimmte Revision des Modellierungsobjekts aus. Die Schaltfläche "Revision löschen" wird angezeigt.
b. Klicken Sie auf Revision löschen. Das Dialogfeld "Löschen" wird angezeigt.
c. Gehen Sie zu Schritt 4.
4. Klicken Sie im Dialogfeld "Löschen" auf Löschen. Die ausgewählte Revision oder alle Revisionen werden gelöscht, je nachdem, welche Option Sie in Schritt 3 ausgewählt haben.
