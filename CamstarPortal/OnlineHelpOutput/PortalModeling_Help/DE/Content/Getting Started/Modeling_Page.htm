Seite "Modellierung"

Über die Seite "Modellierung" können Sie auf alle Modellierungsobjekte zugreifen. Allerdings wird eventuell nur eine Untermenge der verfügbaren Modellierungsseiten angezeigt, je nachdem, welche Berechtigungseinstellungen Ihre Mitarbeiterdefinition aufweist. Diese Seite ermöglicht es Ihnen, ein Modellierungsobjekt auszuwählen und die Seite dieses Objekts auf der Seite "Modellierung" zu öffnen.

Diese Abbildung zeigt ein Beispiel der Seite "Modellierung".

Option "Änderungsmanagement-Speicherung"

Wenn Sie die Änderungsmanagement-Speicherung verwenden und mindestens ein Änderungsverwaltungspaket erstellt haben, wird auf der Seite "Modellierung" die zusätzliche Option "Änderungsmanagement-Speicherung" angezeigt. Mithilfe dieser Option können Sie Modellierungsobjekt-Instanzen erstellen und aktualisieren und sie vorhandenen, aktiven Paketen zuweisen. Weitere Informationen finden Sie im produktspezifischen Handbuch zur Änderungsverwaltung:

• Opcenter Execution Core Change Management-Benutzerhandbuch
• Opcenter Execution Medical Device and Diagnostics Change Management-Benutzerhandbuch

Suchen eines Modellierungsobjekts

Geben Sie die ersten Buchstaben oder den gesamten Namen des Modellierungsobjekts in das Feld "Filter" ein, um nach einem bestimmten Modellierungsobjekt zu suchen. Weitere Informationen zur Verwendung der Filterfunktion finden Sie unter "Filtern der Instanzliste".

Öffnen einer Modellierungsseite

Sie können über die Seite "Modellierung" auf Modellierungsobjekte zugreifen. Weitere Informationen zum Auswählen einer Modellierungsseite und zum Zugreifen auf diese Seite finden Sie unter "Zugreifen auf Modellierungsseiten".

Diese Tabelle definiert die Schaltflächen, die auf der Seite einer Modellierungsobjektinstanz angezeigt werden.

Schaltfläche

Funktion der Schaltfläche . . .

Anzeigen von leeren Feldern, damit Sie eine neue Instanz des ausgewählten Modellierungsobjekts definieren können.

Erstellen einer Kopie der Revision, in der alle Felder bis auf das Namensfeld leer sind. Verwenden Sie "Neue Revision", wenn die meisten Attribute für die neue Revision eindeutig sind.

Hinweis: Diese Schaltfläche wird nur für eine ausgewählte Instanz eines revisionierten Objekts angezeigt.

Anzeigen des Popups "Kopieren" für die aktuell ausgewählte Instanz eines revisionierten Objekts oder Named Data Objects. Die Anwendung fordert Sie auf, einen Namen für die Zielinstanz einzugeben (sowohl für revisionierte Objekte wie auch für Named Data Objects). Für revisionierte Objekte müssen Sie zusätzlich einen Revisionsnamen eingeben.

Anzeigen des Popups "Kopieren" für die derzeit ausgewählte Instanz des revisionierten Objekts. Wenn Sie auf diese Schaltfläche klicken, wird ein Duplikat des revisionierten Objekts mit allen Informationen der Originalrevision, mit Ausnahme des Revisionswert, erstellt. Die Anwendung fordert Sie auf, eine neue Revision für die Modellierungsobjekt-Instanz festzulegen.

Hinweis: Diese Schaltfläche wird nur für eine ausgewählte Instanz eines revisionierten Objekts angezeigt.

Anzeigen des Popups "Umlaufbestandsnachrichten" für die derzeit ausgewählte Objektinstanz. Über diese Schaltfläche können Sie eine neue Umlaufbestandsnachricht eingeben oder vorhandene einsehen.

Hinweis: Opcenter EX MDD und Opcenter EX CR unterstützen Umlaufbestandsnachrichten nur für Fertigungstransaktionen. Die Schaltfläche "Nachrichten" wird nicht angezeigt, wenn Sie ein Modellierungsobjekt für Änderungsverwaltung auswählen.

Anzeigen des Popups "Löschen" für die derzeit ausgewählte Named Data Object-Instanz. Über diese Schaltfläche können Sie das ausgewählte Named Data Object löschen.

Hinweis: Diese Schaltfläche wird nur für eine ausgewählte Instanz eines Named Data Objects angezeigt.

Anzeigen des Popups "Löschen" für die derzeit ausgewählte revisionierte Objektinstanz. Über diese Schaltfläche können Sie das ausgewählte Modellierungsobjekt und alle zugehörigen Revisionen löschen.

Hinweis: Diese Schaltfläche wird nur für eine ausgewählte Standardrevision eines revisionierten Objekts angezeigt.

Anzeigen des Popups "Löschen" für die derzeit ausgewählte Revision. Über diese Schaltfläche können Sie die ausgewählte Revision löschen.

Hinweis: Diese Schaltfläche wird nur angezeigt, wenn Sie eine Revision einer revisionierten Objektinstanz auswählen, die nicht die Standardrevision ist.

Anzeigen des Popups "Modellierungsprotokoll" für die ausgewählte Instanz. Weitere Informationen finden Sie unter "Modellierungsprotokoll".

Anzeigen des Popups "Hierarchie" für die ausgewählte Instanz. Informationen finden Sie unter "Vorgehensweise zum Anzeigen von Instanzen, die von der ausgewählten Instanz referenziert werden".

Anzeigen des Popups "Wo genutzt" für die ausgewählte Instanz. Informationen finden Sie unter "Vorgehensweise zum Anzeigen von Instanzen, die die ausgewählte Instanz referenzieren".

Anzeigen des Popups "Paketzuordnungen verwalten" für die derzeit ausgewählte Objektinstanz. Diese Schaltfläche wird nur angezeigt, wenn Sie die Änderungsverwaltung verwenden, mindestens eine Instanz des Modellierungsobjekts gespeichert haben, das Sie einem Änderungsverwaltungspaket zuweisen möchten, und über die entsprechende Rolle und die erforderlichen Berechtigungen verfügen.

Weitere Informationen finden Sie im Opcenter Execution Core Change Management-Benutzerhandbuch und im Opcenter Execution Medical Device and Diagnostics Change Management-Benutzerhandbuch.

Gemeinsame Felder auf Modellierungsseiten

Modellierungsobjektseiten enthalten Felder, die anderen Modellierungsobjekten zur Verfügung stehen. Diese Felder werden im Kopf der Seite und ganz unten auf der Seite angezeigt.

In dieser Tabelle sind die Felder definiert, die auf den Modellierungsseiten angezeigt werden.
