Portal Studio

Modellierungsseiten werden in Portal Studio definiert. Portal Studio ist ein eigenständiges Entwicklungs- und Konfigurationswerkzeug, das die Einrichtungs- und Konfigurationszeit der Erstinstallation verkürzt und den Bedarf an benutzerdefiniertem Code erheblich verringert.

Portal Studio hat gegenüber herkömmlichen Entwicklungswerkzeugen zahlreiche Vorteile:

• Es werden keine zusätzlichen Installationen oder Steuerelemente benötigt.
• Sie benötigen lediglich einen Browser und eine URL für eine installierte Umgebung.
• Portal Studio verfügt über eine integrierte Quellcodeverwaltung.
• Änderungen müssen nicht kompiliert werden.
• Tests können während des Betriebs durchgeführt werden.
• Änderungen am Objektmodell erfordern lediglich eine einfache Aktualisierung des Browserfensters, nachdem die Windows Communication Foundation-Dienste (WCF-Dienste) generiert wurden.
• Sämtliche Komponenten der Benutzeroberfläche können in eine einzelne, selbstbeschreibende Flatfile exportiert oder aus dieser importiert werden.
• Grundlegende Seitenvererbung ist integriert.
• Upgrades werden automatisch auf die Systemebene angewandt.
