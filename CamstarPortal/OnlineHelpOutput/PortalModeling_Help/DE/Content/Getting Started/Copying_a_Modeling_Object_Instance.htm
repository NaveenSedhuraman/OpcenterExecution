Kopieren einer Modellierungsobjekt-Instanz

Die Anwendung zeigt das Popup "Kopieren" oder "Revision kopieren" an, wenn Sie auf die entsprechenden Schaltflächen auf der Symbolleiste für eine ausgewählte Modellierungsinstanz klicken. Wenn Sie im Popup "Kopieren" oder "Revision kopieren" auf die Schaltfläche "Kopieren" klicken oder die Eingabetaste drücken, wird die neue Instanz oder Revision der Instanz gespeichert, und das Popup wird geschlossen. Die Anwendung zeigt eine Erfolgsmeldung an, die angibt, dass die neue Modellierungsobjekt-Instanz erstellt wurde.

Wenn Sie eine Kopie erstellen, wird die ausgewählte Instanz mit allen ursprünglichen Informationen außer dem Instanznamen (und dem Revisionsnamen bei revisionierten Objekten) dupliziert. Die Anwendung erfordert, dass Sie vor dem Speichern einer Named Data Object-Instanz einen neuen Instanznamen und vor dem Speichern eines revisionierten Objekts einen neuen Instanznamen und Revisionswert eingeben.

Wenn Sie eine Kopie einer Revision erstellen, wird ein Duplikat des revisionierten Objekts mit allen Informationen der Originalrevision erstellt, ausgenommen der Revisionswert. Die Anwendung verlangt von Ihnen, dass Sie einen neuen Wert für die Revision eingeben, bevor Sie eine neue Revision der ausgewählten Instanz speichern.

Hinweis: Die Schaltfläche "Revision kopieren" wird nur für revisionierte Objekte angezeigt.

Vorgehensweise zum Kopieren einer revisionierten Objektinstanz

Gehen Sie folgendermaßen vor, um eine revisionierte Objektinstanz zu kopieren:

1. Wählen Sie Modellierung im Menü Modellierung aus. Die Seite "Modellierung" wird angezeigt.
2. Klicken Sie auf ein revisioniertes Objekt Die Seite des revisionierten Objekts wird innerhalb der Seite "Modellierung" angezeigt.
3. Wählen Sie eine vorhandene Instanz des revisionierten Objekts aus. Die Instanz wird angezeigt.
4. Klicken Sie auf Kopieren. Das Popup "Kopieren" wird angezeigt.
5. Geben Sie einen Namen für die Modellierungsinstanz in das Feld Zielinstanzname ein.
6. Geben Sie eine Revision in das Feld Neuer Revisionsname ein.
7. Klicken Sie auf Kopieren oder drücken Sie Eingabe auf der Tastatur.
8. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Kopieren einer Revision einer revisionierten Objektinstanz

Gehen Sie folgendermaßen vor, um die Revision einer revisionierten Objektinstanz zu kopieren:

1. Wählen Sie Modellierung im Menü Modellierung aus. Die Seite "Modellierung" wird angezeigt.
2. Klicken Sie auf ein revisioniertes Objekt Die Seite des revisionierten Objekts wird innerhalb der Seite "Modellierung" angezeigt.
3. Wählen Sie eine vorhandene Instanz des revisionierten Objekts aus. Die Instanz wird angezeigt.
4. Klicken Sie auf Revision kopieren. Das Popup "Revision kopieren" wird angezeigt.
5. Geben Sie eine Revision in das Feld Neuer Revisionsname ein.
6. Klicken Sie auf Kopieren oder drücken Sie Eingabe auf der Tastatur.
7. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Kopieren einer Modellierungsinstanz eines Named Data Object

Gehen Sie folgendermaßen vor, um die Modellierungsinstanz eines Named Data Object zu kopieren:

1. Wählen Sie Modellierung im Menü Modellierung aus. Die Seite "Modellierung" wird angezeigt.
2. Klicken Sie auf ein Named Data Object. Die Seite des Named Data Object wird innerhalb der Seite "Modellierung" angezeigt.
3. Wählen Sie eine bereits vorhandene Instanz des Named Data Object aus. Die Instanz wird angezeigt.
4. Klicken Sie auf Kopieren. Das Popup "Kopieren" wird angezeigt.
5. Geben Sie einen Namen für die Modellierungsinstanz in das Feld Zielinstanzname ein.
6. Klicken Sie auf Kopieren oder drücken Sie Eingabe auf der Tastatur.
7. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
