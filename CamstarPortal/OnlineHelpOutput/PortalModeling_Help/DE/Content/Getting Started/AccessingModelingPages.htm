Zugreifen auf Modellierungsseiten

Alle Modellierungsobjektseiten sind von der Hauptmodellierungsseite aus zugänglich, die im Menü "Modellierung" aufgerufen werden kann. Die Anwendung öffnet die Modellierungsobjektseite, die Sie ausgewählt haben, innerhalb der Seite "Modellierung". Eine umfassende Beschreibung der Funktionen auf der Seite "Modellierung" finden Sie unter "Seite "Modellierung"".

Vorgehensweise zum Zugreifen auf eine Modellierungsseite

Gehen Sie folgendermaßen vor, um auf eine Seite "Modellierung" zuzugreifen:

1. Anmelden im Portal.
Hinweis: Melden Sie sich als CamstarAdmin, InSiteAdmin oder Administrator an, wenn dies Ihre erste Anmeldung nach der Installation ist.
2. Wählen Sie Modellierung im Menü Modellierung aus. Die Seite "Modellierung" wird angezeigt.
3. Klicken Sie auf den Alle Wartung-Pfeil, um eine Liste der Modellierungsobjekte unter dieser Gruppe anzuzeigen.

Oder

Klicken Sie auf Alle anzeigen (alphabetisch) in der Liste Objekte anzeigen nach, um eine Liste von Modellierungsobjekten in alphabetischer Reihenfolge anzuzeigen. Die Anwendung zeigt eine Liste aller Modellierungsobjekte in alphabetischer Reihenfolge an.

4. Wählen Sie ein Modellierungsobjekt aus. Die Seite des Objekts wird auf der Seite "Modellierung" angezeigt.
