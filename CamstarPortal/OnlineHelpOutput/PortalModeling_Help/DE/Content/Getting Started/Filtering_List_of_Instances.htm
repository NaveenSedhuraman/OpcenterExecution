Filtern der Instanzliste

Über das Filterfeld kann eine große Anzahl von Modellierungsinstanzen eingeschränkt werden.

Geben Sie einen Teil einer Zeichenkette ein und klicken Sie auf "Aktualisieren", um eine Liste aller Instanzen anzuzeigen, deren Namen mit dieser Teilzeichenkette beginnen. Wenn Sie bei leerem Filterfeld auf "Aktualisieren" klicken, sehen Sie wieder alle Instanzen.

Listen zurückgegebener Instanzen werden alphabetisch in der bevorzugten Sprache sortiert, sofern die betreffende Sprache vom Portal unterstützt wird. Listen zurückgegebener Instanzen werden standardmäßig alphabetisch in Englisch sortiert. Unterhalb des Filters wird ein Listen-Steuerelement angezeigt, wenn die Anzahl der definierten Instanzen größer als 100 ist.

Opcenter Execution Core und Opcenter Execution MDD unterstützen mehrere Datenbanken. Diese Datenbanken unterscheiden sich in ihrem Standardansatz zur Sortierung von Listen, die Akzentzeichen enthalten. Informationen zur linguistischen Sortierung finden Sie unten in der entsprechenden produktspezifischen Anleitung zur Systemadministration:
