Definieren von CIO-Einstellungen

Das Objekt "CIO-Einstellungen" enthält Konfigurationseinstellungen für ausgehende Nachrichten. Diese Einstellungen umfassen die URL für das Opcenter Connect MOM-Client-Gateway und das Timeout für das Senden von Nachrichten an die URL.

Das Objekt "CIO-Einstellungen" enthält auch Konfigurationseinstellungen für den Opcenter Connect MOM-Konfigurationsdienst. Diese Einstellungen sind:

• user name
• Kennwort (Password)
• URL

Weitere Informationen zu Opcenter Connect MOM finden Sie im Opcenter Connect MOM-Benutzerhandbuch.

Der CIO-Installationsprozess erstellt automatisch eine Instanz von CIO-Einstellungen mit dem Namen ciosettings, wobei die Standardwerte in den Feldern "Client-Gateway-URL" und "Client-Gateway-Post-Timeout" ausgefüllt werden.

Vorgehensweise zum manuellen Definieren der CIO-Einstellungen

Gehen Sie folgendermaßen vor, um CIO-Einstellungen manuell zu definieren:

1. Öffnen Sie die Seite CIO-Einstellungen. Die Seite "CIO-Einstellungen" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie ciosettings in das Feld CIO-Einstellungen ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
