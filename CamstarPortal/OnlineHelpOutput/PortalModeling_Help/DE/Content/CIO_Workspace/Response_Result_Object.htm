CIO-Antwortergebnis - Beispiel

Dies ist ein Beispiel für ein CIO-Antwortergebnis.

 

<?xml version="1.0" encoding="UTF-8"?>

<CIOSynchronousResponse Version="" Timestamp="" noNamespaceSchemaLocation="">

<CIOMessage>

<MessageID>#GUID</MessageID>

</CIOMessage>

<ResultSet>

<Object>

<Name DataType="String">Move Std Remote Validation</Name>

<Expression DataType="String">CLF::Response.ResponseValue</Expression>

<TargetExpression DataType="String">Container.ContainerComments</TargetExpression>

<ResponseValue DataType="String">Updated by Syncresponse #ContainerName</ResponseValue>

<ValidateExpression DataType="String"><![CDATA[Container.Qty>50]]></ValidateExpression>

</Object>

</ResultSet>

</CIOSynchronousResponse>

 

Wenn dieses Format verwendet wird, gelten folgende Konzepte:

• "ValidateExpression" wird verwendet, um den jeweiligen Ausdruck auszuwerten. Die Transaktion wird beendet, wenn die Auswertung des Ausdrucks "False" ergibt.
• "Expression" wird verwendet, um einen Ausdruck zu erstellen, der "TargetExpression" füllt. Der Antwortwert kann im Ausdruck verwendet werden.
