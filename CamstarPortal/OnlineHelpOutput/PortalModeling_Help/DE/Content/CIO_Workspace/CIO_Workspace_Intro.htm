CIO-Arbeitsbereich

Hinweis: Die nachstehend beschriebene Funktionalität ist nur verfügbar, wenn CIO installiert ist. Informationen zum Installieren von CIO finden Sie in der Opcenter Execution Medical Device and Diagnostics-Anleitung zur Systemadministration und in der Opcenter Execution Core-Anleitung zur Systemadministration.

CIO bietet Funktionen für ausgehende Nachrichten von Opcenter Execution Medical Device and Diagnostics oder Opcenter Execution Core an die externen Systeme. CIO arbeitet mit Opcenter Connect MOM zusammen, das Funktionen für eingehende Nachrichten bietet. Dieser Anhang enthält Informationen zu Modellierungsobjekten für die CIO-Funktionalität. Weitere Informationen zu Opcenter Connect MOM finden Sie im Opcenter Connect MOM-Benutzerhandbuch.

CIO-Funktionalität wird in den folgenden Themen behandelt:

• Definieren von CIO-Einstellungen
• Erstellen von Definitionen für ausgehende CIO-Nachrichten
• Definieren von CIO-Vorlagen
• Definieren von CIO-Listenbearbeitern
• Definieren von CIO-Benutzerabfragen
• CIO-Antwortergebnis - Beispiel
