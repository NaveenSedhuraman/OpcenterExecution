Definieren von CIO-Vorlagen

Mit dem Objekt "CIO-Vorlage" können Sie eine Vorlage für ausgehende CIO-Nachrichten erstellen. Die Struktur einer ausgehenden Nachricht wird durch eine Vorlage für eine ausgehende Nachricht definiert, die verschiedene Formate (z. B. XML, Flat File oder einzelne Textvariable) aufweisen kann.

Beim Definieren einer CIO-Vorlage

Das Feld "Vorlage" muss einen statischen Inhalt aufweisen, der sowohl für Opcenter Connect MOM-Zuordnungen als auch als inhaltlicher Rahmen für ausgehende Nachrichten und Textvariablen verwendet werden kann. Hierbei kann es sich entweder um ein XML-Dokument oder anderen Freitext handeln.

Klicken Sie auf die Schaltfläche "Format XML", um den Vorlagentext als XML zu formatieren.

Definition der Felder auf der Seite "CIO-Vorlage"

Diese Tabelle definiert die Felder, die nur auf der Seite "CIO-Vorlage" zu finden sind.

Siehe auch "Gemeinsame Felder auf Modellierungsseiten". Dieser Abschnitt enthält Informationen zu den Feldern, die alle Modellierungsobjekte aufweisen.

Feld

Definition

Typ

Allgemein

Vorlage

Aufgelöster Wert der Vorlage und alle Textvariablen in der Vorlage. Beispiel: "#CIOUserQueryResult" kann eine Textvariable mit dem Feldausdruck "Transaction::__CDOID.CIOUserQueryResult" sein. Das Feld wird durch Klicken auf "Format XML" ausgefüllt.

Hinweis: CDATA (<![CDATA[ ) wird in CIO-Vorlagen nicht unterstützt.

Optional

Vorgehensweise zum Definieren einer CIO-Vorlage

Gehen Sie folgendermaßen vor, um eine CIO-Vorlage zu definieren:

1. Öffnen Sie die Seite CIO-Vorlage. Die Seite "CIO-Vorlage" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie in das Feld CIO-Vorlage einen Namen für die Vorlage ein.
4. Geben Sie in das Feld Revision die Revisionsbezeichnung ein.
5. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
6. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
