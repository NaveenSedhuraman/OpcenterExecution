Definieren von CIO-Abfrageportalseiten

[US30047 V8 RH]

Mit dem Objekt "CIO-Abfrageportalseite" können Sie eine Sequenz von Abfragen so definieren, dass Ergebnisse an die CIO-Abfrageportalseitenabfrage zurückgegeben werden. Informationen zum Definieren von Benutzerabfragen finden Sie im Portal Modeling-Benutzerhandbuch und unter "Definieren von CIO-Benutzerabfragen".

Beim Definieren von CIO-Abfrageportalseiten

Damit Sie Felder im Objekt "CIO-Abfrageportalseite" definieren können, müssen Sie zunächst über die Seite "CIO-Benutzerabfrage" CIO-Benutzerabfragen definieren.

In der Tabelle für die Abfrageportalseitenabfragen können Sie maximal vier Abfragen definieren. Die Anwendung ordnet den Abfragen, die in der Tabelle für die Abfrageportalseitenabfragen aufgeführt sind, automatisch eine Sequenzreihenfolge zu.

Die erste in der Tabelle "Abfrageportalseitenabfragen" aufgeführte Abfrage muss eine Abfrage des Zusammenfassungstyps sein. Diese Abfrage erfordert keine Parameter. Ein Beispiel für eine Abfrage des Zusammenfassungstyps ist eine Abfrage zum Anzeigen der Zeilen für die CIO-Protokolleinträge des aktuellen Tages.

Die zweite Abfrage muss Parameter aufweisen, die denselben Namen verwenden wie die Zeilen in der ersten Abfrage. Die Anwendung füllt die Ergebnisse der zweiten Abfrage auf der Basis der Zeilenauswahl in der ersten Abfrage. Die Anwendung füllt die Ergebnisse der dritten Abfrage auf der Basis der Zeilenauswahl in der zweiten Abfrage. Die Anwendung trägt die vierte Abfrage abhängig von der Zeilenauswahl in der dritten Abfrage ein. Weitere Informationen zum Durchführen von CIO-Abfrageportalseitenabfragen finden Sie unter "Durchführen einer CIO-Abfrageportalseitenabfrage".

Vorgehensweise zum Definieren einer CIO-Abfrageportalseite

Gehen Sie folgendermaßen vor, um eine CIO-Abfrageportalseite zu definieren:

1. Öffnen Sie das Objekt CIO-Abfrageportalseite. Das Objekt "CIO-Abfrageportalseite" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie in das Feld CIO-Abfrageportalseite einen Namen für die Abfrageportalseite ein.
4. Klicken Sie in der Tabelle "Abfrageportalseitenabfragen" auf Neue Zeile hinzufügen. Eine neue Zeile wird angezeigt.
5. Geben Sie im Feld Abfragename einen Namen für die erste Abfrage ein.
6. Wählen Sie eine Benutzerabfrage in der Liste aus.
7. Wiederholen Sie die Schritte 4 bis 6, um Abfragen hinzuzufügen.
8. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
9. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
