Wiegen von Komponenten - Übersicht

Hinweis: Die nachstehend beschriebene Funktionalität ist nur verfügbar, wenn Batch Processing installiert ist. Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics Batch Processing-Benutzerhandbuch und im Opcenter Execution Core Batch Processing-Benutzerhandbuch.

Die Rezeptanweisungen für die Verarbeitung eines Batches werden unter Verwendung der vorhandenen Funktionalität für automatische Abläufe erstellt. Rezeptschritte können zusätzlich zu einer Wiegeaufgabe beliebige andere Aufgabentypen enthalten, die für die Batchproduktion erforderlich sind. Sie können eine Verbrauchswiegeaufgabe mit dem Modellierungsobjekt "Aufgabenliste" erstellen, um Wiegevorgang und Verbrauch eines Komponentenprodukts für einen Batch-Container zu definieren. Die Verbrauchswiegeaufgabe bietet die Möglichkeit, den Punkt im Prozess zu definieren, an dem die Komponente gewogen werden muss. Die Verbrauchswiegeaufgabe bietet auch die Möglichkeit, die Regeln zum Wiegen der Komponente zu definieren. Informationen zum Erstellen einer Verbrauchswiegeaufgabe finden Sie unter "Beim Definieren von Verbrauchswiegeaufgaben".
