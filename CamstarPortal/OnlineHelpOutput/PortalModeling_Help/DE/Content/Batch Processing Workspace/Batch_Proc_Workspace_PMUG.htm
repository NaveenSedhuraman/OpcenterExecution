Batch Processing

Hinweis: Die nachstehend beschriebene Funktionalität ist nur verfügbar, wenn Batch Processing installiert ist. Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics Batch Processing-Benutzerhandbuch und im Opcenter Execution Core Batch Processing-Benutzerhandbuch.

Batch Processing ermöglicht die Durchführung von Funktionen in Bezug auf:

• Waagenverwaltung
• Rezeptverwaltung
• Maßeinheitenkonvertierung
• Wiegen von Komponenten

Hersteller können diese Funktionen verwenden, um die Produktion von kleinen chemischen Batches zu steuern und zu verfolgen, die als Komponenten in ihren Geräten verwendet werden. Beispiele dafür sind Überzüge für Blutzuckerstreifen und Arzneirezepturen für transdermale Pflaster.

Sie müssen eine Materialliste definieren, um Batch Processing verwenden zu können. Beim Definieren der Materialliste können Sie die erforderliche Menge sowie die minimale und maximale Menge für die einzelnen gewogenen Komponenten definieren. Materiallisten für das fertige Produkt können mit der Tabelle "Materialliste" der Modellierungsobjekte "Stückliste", "ERP-Stückliste" oder "Produktionsauftrag" definiert werden. Weitere Informationen finden Sie unter "Definieren zulässiger Komponentenmengen in der Stückliste", "Definieren zulässiger Komponentenmengen in der ERP-Stückliste" und "Definieren zulässiger Komponentenmengen im Produktionsauftrag".
