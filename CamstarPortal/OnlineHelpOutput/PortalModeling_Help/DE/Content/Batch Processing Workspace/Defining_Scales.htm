Definieren von Waagen

Eine Waage ist eine Ressource, die Einrichtungsinformationen für eine Waage bereitstellt, die für die Verarbeitung von Batches verwendet wird. Diese Informationen umfassen:

• Arbeitsbereich
• Genauigkeit
• Informationen zum Lieferanten der Waage
• Maßeinheit

Sie können als Teil einer Waagengruppe Spezifikationen, Produkte und Produktfamilien zuweisen. Waagengruppen werden verwendet, um logische Gruppierungen von Waagen zu erstellen, damit die Benutzer die gültigen Waagen in einer Liste auswählen können. Weitere Informationen finden Sie unter "Definieren von Waagengruppen".

Sie können Status und Zustand einer Waage auf der Seite "Ressourceneinrichtung" verwalten. Informationen zur Seite "Ressourceneinrichtung" finden Sie im Opcenter Execution Medical Device and Diagnostics Shop Floor-Benutzerhandbuch und im Opcenter Execution Core Shop Floor-Benutzerhandbuch.

Beim Definieren von Waagen

Sie können folgende Angaben machen:

• In den Feldern "Minimale Kapazität" und "Maximale Kapazität" können Sie den Arbeitsbereich für das Gewicht angeben, das die Waage messen kann.
• In den Feldern "Ziffern insgesamt (Genauigkeit)" und "Dezimalstellen (Waage)" können Sie die Genauigkeit der Messungen mit der Waage angeben.
• Im Feld "Ablesbarkeit" können Sie das kleinste Inkrement der Waagenanzeige angeben.

Beim Integrieren von Waagen

Die Funktionen von Opcenter Connect MOM und CIO werden zur Integration von Waagen verwendet. Bei der Batchverarbeitung kann eine beliebige Waage verwendet werden. Weitere Informationen zu Opcenter Connect MOM finden Sie im Opcenter Connect MOM-Benutzerhandbuch. Informationen zu CIO-Modellierungsobjekten finden Sie unter "CIO-Arbeitsbereich".

Vorgehensweise zum Definieren einer Waage

Gehen Sie folgendermaßen vor, um eine Waage zu definieren:

1. Öffnen Sie die Seite Waage. Die Seite "Waage" wird in der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie in das Feld Waage einen Namen für diese Waage ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
