Definieren von Waagenfamilien

Eine Waagenfamilie bietet eine Möglichkeit, Waagen nach ihren Attributen und Ihren Geschäftsanforderungen entsprechend zu kategorisieren. Sie können Waagenfamilien verwenden, um gemeinsame Attribute für alle Waagen zu definieren, die auf die Waagenfamilie verweisen.

"Waagenfamilie" ist ein optionales Feld im Modellierungsobjekt "Waage".

Felddefinitionen der Seite "Waagenfamilie"

Diese Tabelle definiert die Felder, die nur auf der Seite "Waagenfamilie" vorhanden sind.

Siehe auch "Gemeinsame Felder auf Modellierungsseiten". Dieser Abschnitt enthält Informationen zu den Feldern, die alle Modellierungsobjekte aufweisen.

Vorgehensweise zum Definieren einer Waagenfamilie

Gehen Sie folgendermaßen vor, um eine Waagenfamilie zu definieren:

1. Öffnen Sie die Seite Waagenfamilie. Die Seite "Waagenfamilie" wird in der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie in das Feld Waagenfamilie einen Namen für diese Waagenfamilie ein.
4. Erweitern Sie den Abschnitt Details.
5. Geben Sie die maximale Kapazität für die Waagenfamilie in das Feld Maximumkapazität ein.
6. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
7. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
