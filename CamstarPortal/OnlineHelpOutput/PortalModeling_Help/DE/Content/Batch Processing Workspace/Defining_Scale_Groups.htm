Definieren von Waagengruppen

Eine Waagengruppe ist eine Unterklasse einer Ressourcengruppe, die eine Gruppe von Waagen darstellt und Verweise auf einzelne Waagen oder andere Waagengruppen enthalten kann. Die Gruppierung von Waagen bietet die Möglichkeit, dass Modellierungsobjekte mehrere zusammenhängende Anforderungen zusammen statt jeweils nur eine Anforderung referenzieren. Sie können Waagengruppen auswählen, für die Wartungsmaßnahmen geplant werden sollen.

Sie können Waagengruppen Spezifikationen, Produkten, Produktfamilien oder Verbrauchswiegeaufgaben zuweisen, um die erlaubten Waagen zum Wiegen von Komponenten festzulegen. Die der Verbrauchswiegeaufgabe zugewiesene Waagengruppe hat Vorrang vor allen anderen zugewiesenen Waagengruppen, wenn mindestens einem der aufgeführten Objekte eine Waagengruppe zugewiesen ist. Die dem Produkt zugewiesene Waagengruppe hat Vorrang vor Waagengruppen für Produktfamilien und Spezifikationen. Die der Produktfamilie zugewiesene Waagengruppe hat Vorrang vor Waagengruppen für Spezifikationen. Wenn einem dieser Objekte keine Waagengruppe zugewiesen ist, können Sie eine beliebige verfügbare Waage zum Wiegen auswählen.

Informationen zum Zuweisen einer Waagengruppe für die Spezifikation finden Sie unter "Zuweisen einer Waagengruppe für die Spezifikation". Informationen zum Zuweisen einer Waagengruppe für das Produkt finden Sie unter "Zuweisen einer Waagengruppe zu einem Komponentenprodukt". Informationen zum Zuweisen einer Waagengruppe für die Produktfamilie finden Sie unter "Zuweisen einer Waagengruppe für die Produktfamilie". Informationen zum Zuweisen einer Waagengruppe für die Verbrauchswiegeaufgabe finden Sie unter "Beim Definieren von Verbrauchswiegeaufgaben".

Beim Definieren von Waagengruppen

Sie können mehrere Waagen und Waagengruppen zur Waagengruppe hinzufügen.

Wenn Sie auf die Schaltfläche "Aufgelöste Einträge" klicken, wird eine Liste der in der Gruppe enthaltenen Waagen angezeigt, die einzeln oder als Gruppe hinzugefügt wurden.

Vorgehensweise zum Definieren einer Waagengruppe

Gehen Sie folgendermaßen vor, um eine Waagengruppe zu definieren:

1. Öffnen Sie die Seite Waagengruppe. Die Seite "Waagengruppe" wird in der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie in das Feld Waagengruppe einen Namen für diese Waagengruppe ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
