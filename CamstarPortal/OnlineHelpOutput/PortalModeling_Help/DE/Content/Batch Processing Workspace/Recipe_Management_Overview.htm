Rezeptverwaltung - Übersicht

Hinweis: Die nachstehend beschriebene Funktionalität ist nur verfügbar, wenn Batch Processing installiert ist. Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics Batch Processing-Benutzerhandbuch und im Opcenter Execution Core Batch Processing-Benutzerhandbuch.

Bisweilen muss die in der Stückliste definierte Menge für Rezepte angepasst werden. Zu den Gründen für solche Anpassungen zählen u a. Änderung der exakten Menge von Komponenten, die vorher hinzugefügt wurde, Unterschiede bei der Wirksamkeit von Rohmaterialien oder unterschiedliche Batchgrößen. Mithilfe von Batch Processing-Funktionen können Sie die Mengen der Materialliste anpassen, indem Sie Ausdrücke zum Berechnen der Zielmenge sowie der erforderlichen minimalen und maximalen Menge während der Rezeptverarbeitung angeben. Sie können diese Ausdrücke im Produkt oder im Element für die Verbrauchswiegeaufgabe des Rezepts definieren.

Mengen aufgrund von für das Komponentenprodukt definierten Ausdrücken haben Vorrang vor den in der Materialliste definierten Mengen. Mengen aufgrund von für die Verbrauchswiegeaufgabe definierten Ausdrücken haben Vorrang vor den Mengen aufgrund von Ausdrücken mit für das Produkt definierten Ausdrücken. Weitere Informationen zum Definieren von Mengenausdrücken für das Produkt finden Sie unter "Definieren von Ausdrücken zum Anpassen der Komponentenmengen für das Produkt". Informationen zum Definieren von Mengenausdrücken für die Verbrauchswiegeaufgabe finden Sie unter "Beim Definieren von Verbrauchswiegeaufgaben".

Die Funktionalität zur Batchverarbeitung versetzt Sie auch in die Lage, mittlere Batches zu definieren, die bearbeitet werden müssen, bevor sie zum Hauptbatch hinzugefügt werden. Sie können Ausdrücke im Modellierungsobjekt "Aufgabenliste" definieren, um die Menge des mittleren Batches zu berechnen. Informationen zum Definieren mittlerer Batches finden Sie unter Beim Definieren eines mittleren Batches".
