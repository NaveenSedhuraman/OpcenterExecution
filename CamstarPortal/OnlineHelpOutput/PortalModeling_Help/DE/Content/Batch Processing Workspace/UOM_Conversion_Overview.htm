Maßeinheitenkonvertierung - Übersicht

Hinweis: Die nachstehend beschriebene Funktionalität ist nur verfügbar, wenn Batch Processing installiert ist. Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics Batch Processing-Benutzerhandbuch und im Opcenter Execution Core Batch Processing-Benutzerhandbuch.

Die für eine Komponente angegebene Maßeinheit in der Materialliste kann von der für den Batch angegebenen Maßeinheit abweichen. Mithilfe der Batch Processing-Funktionalität können Sie die richtige Menge berechnen. Dazu wird die Verbrauchsmenge in der Maßeinheit der Komponente über Konversionsfaktoren in die Maßeinheit für den Batch umgerechnet. Sie können diese Konversionsfaktoren in der Tabelle "Maßeinheitenkonvertierung" für die Modellierungsobjekte "Produkt", "Produktfamilie" und "Maßeinheit" definieren. Weitere Informationen zum Definieren von Konversionsfaktoren im Modellierungsobjekt "Produkt" finden Sie unter "Definieren der Maßeinheitenkonvertierungsinformationen für das Produkt". Weitere Informationen zum Definieren von Konversionsfaktoren im Modellierungsobjekt "Produktfamilie" finden Sie unter "Definieren der Maßeinheitenkonvertierungsinformationen für die Produktfamilie". Weitere Informationen zum Definieren von Konversionsfaktoren im Modellierungsobjekt "Maßeinheit" finden Sie unter "Beim Definieren von Konvertierungsinformationen für Maßeinheiten".

Konversionsfaktoren, die für das Produkt definiert sind, haben Vorrang vor den Konversionsfaktoren, die für die Produktfamilie definiert sind. Konversionsfaktoren, die für die Produktfamilie definiert sind, haben Vorrang vor den Konversionsfaktoren, die für die Maßeinheit definiert sind.
