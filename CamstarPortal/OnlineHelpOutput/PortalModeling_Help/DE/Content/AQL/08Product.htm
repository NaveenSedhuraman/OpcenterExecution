Zuweisen von Stichprobenplänen zu Produkten

Die Seite "Produkt" wird verwendet, um einem Produkt einen Stichprobenplan zuzuweisen. Die beiden Felder, die speziell für AQL-Stichproben gelten, "Stichprobenplan" und "Alle Container überprüfen", weisen einen Stichprobenplan zu und geben an, ob alle Container stichprobenartig überprüft werden müssen.

Beim Definieren des Produktobjekts für AQL-Stichproben

Wenn Sie ein Produktobjekt für AQL-Stichproben modellieren, gilt die folgende Regel:

• Wenn das optionale Feld "Alle Container überprüfen" aktiviert ist, muss das Feld "Stichprobenplan" ausgefüllt werden.

Felddefinitionen der Seite "Produkt"

Diese Tabelle definiert die erforderlichen Felder für AQL-Stichproben auf der Seite "Produkt".

Feld

Definition

Typ

Stichprobenplan

Liste aller definierten Stichprobenpläne. Die Stichprobenpläne definieren die Prüfkriterien des Produkts, wie z. B. den benötigten Stichprobentest, die Anzahl der Stichproben, die Stichprobenrate, den Ort der Stichprobennahme und die zulässigen Ablehnungen für einen Container dieses Produkts. Wählen Sie den Stichprobenplan aus, den Sie dem Produkt zuweisen möchten.

Hinweis: Dieses Feld ist erforderlich, wenn das Kontrollkästchen "Alle Container überprüfen" aktiviert ist.

Optional

Alle Container überprüfen

Kontrollkästchen, das angibt, ob die Stichprobenrate auf 1 festgelegt wird, sodass jeder Container dieses Produkts überprüft wird.

Diese Option wird in der Regel verwendet, wenn eine Produktion Qualitätsprobleme hat, um Verluste zu minimieren, oder um zu überprüfen, ob alles in Ordnung ist, wenn ein neues Produkt in Produktion geht.

Hinweis: Wenn dieses Kontrollkästchen aktiviert ist, wird ein Stichprobenplan benötigt.

Optional

Weitere Informationen zur Seite "Produkt" finden Sie unter "Definieren von Produkten".

Vorgehensweise zum Zuweisen eines Stichprobenplans zu einem Produkt

Gehen Sie folgendermaßen vor, um einen Stichprobenplan einem Produkt zuzuweisen:

1. Öffnen Sie die Seite Produkt. Die Seite "Produkt" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf das Produkt, dem ein Stichprobenplan zugewiesen werden soll. Die Instanz und die Instanzinformationen werden angezeigt.
3. Erweitern Sie den Abschnitt In Verarbeitung.
4. Wählen Sie den Stichprobenplan aus der Liste im Feld Stichprobenplan aus, den Sie dem Produkt zuweisen möchten.
5. Aktivieren Sie wahlweise das Kontrollkästchen "Alle Container überprüfen", um die Stichprobenrate auf 1 zu stellen, was bedeutet, dass jeder Container dieses Produkts überprüft wird.
6. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
