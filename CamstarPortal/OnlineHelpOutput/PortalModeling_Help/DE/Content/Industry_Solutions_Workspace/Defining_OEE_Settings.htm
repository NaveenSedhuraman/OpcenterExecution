Definieren von OEE-Einstellungen

Das Feld "Dauer für Echtzeit-OEE" (Real Time OEE Duration) im Modellierungsobjekt "OEE-Einstellungen" (OEE Settings) ermöglicht es Ihnen, die Zeitspanne festzulegen, für die die Anwendung Echtzeit-OEE berechnet. Operanden schließen ein:

• Tag (Day)
• Schicht (Shift)
• Arbeit

Sie können den folgenden Modellierungsobjekten OEE-Einstellungen zuweisen:

• Ressource – Weitere Informationen zum Zuweisen von OEE-Einstellungen zu einer Ressource finden Sie unter "Beim Zuweisen von OEE-Einstellungen für die Ressource".
• Ressourcenfamilie – Weitere Informationen zum Zuweisen von OEE-Einstellungen zu einer Ressourcenfamilie finden Sie unter "Beim Zuweisen von OEE-Einstellungen für die Ressourcenfamilie".
• Fabrik – Weitere Informationen zum Zuweisen von OEE-Einstellungen für die Fabrik finden Sie unter "Zuweisen von OEE-Einstellungen für die Fabrik".

Beim Definieren von OEE-Einstellungen

Beim Definieren von OEE-Einstellungen gelten die folgenden Regeln:

• Die OEE-Einstellungen, die auf der Seite "Ressource" zugewiesen werden, haben Vorrang vor den OEE-Einstellungen, die auf der Seite "Ressourcenfamilie" zugewiesen werden.
• Die OEE-Einstellungen, die auf der Seite "Ressourcenfamilie" zugewiesen werden, haben Vorrang vor den OEE-Einstellungen, die auf der Seite "Fabrik" zugewiesen werden.

Informationen zum Zuweisen von OEE-Einstellungen zu einer Ressource finden Sie unter "Beim Zuweisen von OEE-Einstellungen für die Ressource". Weitere Informationen zum Zuweisen von OEE-Einstellungen zu einer Ressourcenfamilie finden Sie unter "Beim Zuweisen von OEE-Einstellungen für die Ressourcenfamilie". Informationen zum Zuweisen von OEE-Einstellungen zu einer Fabrik finden Sie unter "Zuweisen von OEE-Einstellungen für die Fabrik".

Vorgehensweise zum Definieren von OEE-Einstellungen

Gehen Sie folgendermaßen vor, um OEE-Einstellungen zu definieren:

1. Öffnen Sie die Seite OEE-Einstellungen. Die Seite "OEE-Einstellungen" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie im Feld OEE-Einstellungen (OEE Settings) einen Namen für die OEE-Einstellungen ein.
4. Wählen Sie in der Liste eine Dauer für Echtzeit-OEE aus.
5. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
6. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
