Anzeigen des Protokolls für ein revisioniertes Objekt

Ein revisioniertes Objekt (RO) ist ein Objekt, das einen eindeutigen Namen und einen Revisionsstand hat. Ein Beispiel für ein RO wäre "Produkt".

Beim Anzeigen von revisionierten Objekten

Die Felder im Abschnitt "Objektschlüsselwerte" sind dynamisch und hängen von dem RO, das Sie ausgewählt haben, ab. Wenn Sie z. B. im Feld "CDO-Typ" den Eintrag "Produkt" auswählen, zeigt die Anwendung ein Produktfeld und ein Revisionsfeld an.

Die Anwendung erfordert, dass Sie eine bestimmte Revision der RO-Instanz auswählen, die angezeigt werden soll. Sie müssen die Objektinstanz erweitern und anschließend die Revision auswählen. Es ist nicht möglich, die Instanz auszuwählen und die Revision anschließend manuell einzugeben.

Nachdem Sie die Revision einer Instanz ausgewählt haben, aktiviert die Anwendung die Schaltfläche "Protokoll anzeigen". Durch Klicken auf die Schaltfläche "Protokoll anzeigen" wird das Popup "Modellierungsprotokoll" für die ausgewählte Instanz angezeigt.

Felddefinitionen der Seite "Modellierungsprotokoll" für revisionierte Objekte.

In dieser Tabelle sind die Felder auf der Seite "Modellierungsprotokoll" bei Auswahl eines RO als CDO-Typ definiert.

Feld

Definition

Typ

CDO-Typ

Enthält eine Liste der zur Auswahl stehenden Modellierungsobjekte.

Erforderlich

<CDO-Typ>

Instanz des revisionierten Objekts, deren Protokoll angezeigt werden soll.

Hinweis: Ihre Auswahl im Feld "CDO-Typ" bestimmt die Beschriftung, die diesem Feld zugewiesen wird. Wenn Sie z. B. im Feld "CDO-Typ" den Eintrag "Produkt" auswählen, lautet die Beschriftung für dieses Feld "Produkt".

Erforderlich

Revision

Revision für die ausgewählte Instanz.

Erforderlich

Standardrevision

Kontrollkästchen, mit dem Sie eine Instanz des revisionierten Objekts auswählen können, die als Standardrevision verwendet werden soll.

Optional

Vorgehensweise zum Anzeigen eines Modellierungsprotokolls für ein revisioniertes Objekt

Gehen Sie folgendermaßen vor, um das Protokoll für ein revisioniertes Objekt anzuzeigen:

1. Öffnen Sie die Seite Protokoll. Die Seite "Protokoll" wird angezeigt.
2. Wählen Sie aus dem Feld CDO-Typ ein revisioniertes Objekt aus. Die Felder "Objektname" und "Objekt-Revision" werden unter "Objektschlüsselwerte" angezeigt.
3. Wählen Sie die Instanz und die Revision der Instanz aus, deren Protokoll angezeigt werden soll. Die Anwendung aktiviert die Schaltfläche "Protokoll anzeigen".
4. Klicken Sie auf Protokoll anzeigen. Das Popup "Modellierungsprotokoll" wird angezeigt.
5. Klicken Sie auf einen der Pfeile, um das Protokoll zu erweitern und weitere Details anzuzeigen.
