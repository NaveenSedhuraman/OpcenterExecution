Anzeigen des Popups "Modellierungsprotokoll"

In der Modellierung besteht die Möglichkeit, die Protokolle vorhandener Modellierungsobjekte anzuzeigen. Sie können von zwei Orten aus auf das Popup "Protokoll" zugreifen:

• Von der Modellierungsseite einer vorhandenen Instanz eines Objekts aus
• Von der Seite "Protokoll" aus, die im Menü "Modellierung" aufgerufen werden kann

Zugreifen auf das Popup "Protokoll" ausgehend von einer Modellierungsseite

Modellierungsseiten enthalten die Schaltfläche "Protokoll", wenn Sie eine vorhandene Instanz eines Objekts, z. B. einer Fabrik, auswählen. Die Schaltfläche "Protokoll" ist auf der Modellierungsobjektseite für eine neue Instanz nicht verfügbar.

Wenn Sie z. B. auf der Seite "Fabrik" auf "Neu" klicken, zeigt die Anwendung leere Felder an, damit Sie eine neue Fabrikinstanz hinzufügen können. Auf dieser Seite ist die Schaltfläche "Protokoll" nicht vorhanden, da es für dieses Objekt noch keine Protokollhistorie gibt. Allerdings wird für die Instanz eine Protokollhistorie erstellt, wenn Sie das neue Objekt speichern. Darüber hinaus wird die Schaltfläche "Protokoll" auf der Seite angezeigt. Dies gilt auch, wenn Sie eine neue Instanz erstellen, indem Sie eine vorhandene Instanz kopieren.

Wenn Sie auf die Schaltfläche "Protokoll" klicken, wird das Protokoll des Objekts im Popup "Modellierungsprotokoll" angezeigt.

Zugreifen auf das Popup "Protokoll" ausgehend von der Seite "Protokoll"

Sie können von Seite "Protokoll" aus auf das Popup "Protokoll" für ein Modellierungsobjekt zugreifen. Wählen Sie aus dem Menü "Modellierung" das Element "Protokoll" aus, um die Seite "Protokoll" anzuzeigen. Auf der Seite "Protokoll" können Sie auf Grundlage von Kriterien, die für den CDO-Typ spezifisch sind, nach revisionierten Objekten, Named Data Objects und Unterobjekten suchen. Anschließend können Sie durch Klicken auf die Schaltfläche "Protokoll anzeigen" das Popup "Modellierungsprotokoll" für die Instanz anzeigen.

Weitere Informationen finden Sie unter "Verwenden der Seite "Protokoll"".

Beim Anzeigen des Popups "Modellierungsprotokoll"

Im Popup "Modellierungsprotokoll" wird zunächst eine reduzierte, oder zusammengefasste, Ansicht der Protokollinformationen angezeigt.

Die reduzierte Ansicht des Protokolls umfasst keine Informationen zu Unterobjekten oder Listen. Sie müssen die erweiterte Ansicht des Protokolls verwenden, um die Detailinformationen zu Unterobjekten und Listen einzusehen. Klicken Sie auf das Pfeilsymbol, um ein Postenelement zu erweitern und seine Details, Unterobjekte und Listen anzuzeigen.

Die Anwendung zeigt die neueste Änderung am Anfang und die älteste Änderung am Ende der Liste an.

Wenn Sie die Änderungsverwaltung verwenden, können Sie im Feld "Paketname" die Ergebnisse auf Instanzen beschränken, die einem bestimmten Änderungspaket zugewiesen sind. Wenn keine Ergebnisse für das angegebene Paket verfügbar sind, zeigt die Anwendung eine Meldung mit dem Hinweis an, dass keine Ergebnisse den Suchkriterien entsprechen.

Hinweis: Damit Sie eine Instanz für den zu aktualisierenden Paketnamen hinzufügen oder aktualisieren können, muss ein Änderungspaket vorhanden sein. Wenn die Änderung vor der Erstellung des Pakets erfolgt, wird das Protokoll des Modellierungsobjekts nicht mit den Namen des Pakets aktualisiert.

In den Feldern "Rückgabe von Ergebnissen zwischen" können Sie den Datumsbereich der angezeigten Ergebnisse ändern. Sie können entweder Daten aus dem Kalender auswählen oder die Daten manuell eingeben. Fall keine Ergebnisse innerhalb des angegebenen Bereichs liegen, meldet die Anwendung, dass es keine Ergebnisse gibt, die Ihren Suchkriterien entsprechen.

Hinweis: Die Anzahl der Zeilen, die standardmäßig angezeigt werden, ist eine Einstellung, die in Portal Studio konfiguriert werden kann.
Wichtig: In der Anwendung kann eine Zeitüberschreitung auftreten, wenn Sie versuchen, ein Modellierungsprotokoll mit einer großen Anzahl von Datensätzen anzuzeigen. Sie können die Anzahl an Millisekunden in der Timeout-Einstellung in der Datei "XMLClientConfig.xml" erhöhen. Die Datei "XMLClientConfig.xml" befindet sich standardmäßig in C:\Programme (x86)\Camstar\InSite Utility.

Beim Drucken des Modellierungsprotokolls

Sie können das Modellierungsprotokoll vom Popup aus drucken, indem Sie auf der Tastatur Strg+P drücken. Das Protokoll wird genauso gedruckt, wie es angezeigt wird. Sie müssen die gewünschten Abschnitte erweitern, bevor Sie das Modellierungsprotokoll drucken.

Vorgehensweise zum Anzeigen des Popups "Protokoll" ausgehend von einer Modellierungsseite

Gehen Sie folgendermaßen vor, um das Popup "Protokoll" von einer Modellierungsseite aus anzuzeigen:

1. Öffnen Sie ein Modellierungsobjekt, z. B. "Fabrik". Die Seite des ausgewählten Modellierungsobjekts wird innerhalb der Seite "Modellierung" angezeigt.
2. Wählen Sie eine vorhandene Modellierungsobjektinstanz aus, deren Protokoll Sie anzeigen möchten. Die Anwendung zeigt die Informationen für diese Instanz an.
3. Klicken Sie auf Protokoll. Das Popup "Modellierungsprotokoll" wird in der reduzierten Ansicht angezeigt.
4. Möchten Sie die Ergebnisse auf ein bestimmtes Änderungspaket einschränken?

Antwort . . .

Dann . . .

Ja
a. Geben Sie den Paketnamen in das Feld Paketname ein.
b. Klicken Sie auf Suchen. Die Anwendung gibt alle Ergebnisse zurück, die mit dem jeweiligen Paket verknüpft sind. Wenn keine Ergebnisse mit dem Paket verknüpft sind, zeigt die Anwendung eine Fehlermeldung an.
c. Gehen Sie zu Schritt 5.

Nein

Gehen Sie zu Schritt 5.
5. Möchten Sie die Ergebnisse auf einen bestimmten Datumsbereich einschränken?

Antwort . . .

Dann . . .

Ja
a. Geben Sie das Startdatum in das erste Feld Rückgabe von Ergebnissen zwischen ein oder wählen Sie ein Datum aus dem Kalender aus.
b. Geben Sie das Enddatum in das zweite Feld Rückgabe von Ergebnissen zwischen ein oder wählen Sie ein Datum aus dem Kalender aus.
c. Klicken Sie auf Suchen. Die Anwendung gibt alle Ergebnisse zurück, die im angegebenen Datumsbereich liegen. Falls keine Ergebnisse in diesem Datumsbereich liegen, zeigt die Anwendung eine Fehlermeldung an.

Gehen Sie zu Schritt 6.
6. Klicken Sie auf einen der Pfeile, um das Protokoll zu erweitern und weitere Details anzuzeigen.
7. Klicken Sie auf Schließen, um das Popup "Modellierungsprotokoll" zu schließen.

Vorgehensweise zum Drucken eines Modellierungsprotokolls

Gehen Sie folgendermaßen vor, um ein Protokoll zu drucken:

1. Führen Sie die Schritte unter "Vorgehensweise zum Anzeigen des Popups "Protokoll" ausgehend von einer Modellierungsseite" aus.
2. Drücken Sie Strg+P. Das Popup "Drucken" wird angezeigt.
3. Wählen Sie den richtigen Drucker und die entsprechenden Einstellungen aus.
4. Klicken Sie auf OK. Die Anwendung druckt die Seite "Protokoll" und das Popup "Modellierungsprotokoll" genauso, wie diese angezeigt werden.
