Anzeigen des Protokolls für ein Unterobjekt eines Modellierungsobjekts

Ein Unterobjekt ist ein Objekt, dessen Eigentümer ein anderes Objekt ist. Ein Objekt vom Typ Unterobjekt gehört ausschließlich zu einem Eltern-Objekt.

Beim Anzeigen von Unterobjekten von Modellierungsobjekten

Die Felder im Abschnitt "Objektschlüsselwerte" sind dynamisch und hängen von dem ausgewählten Unterobjekt und davon ab, ob für das Unterobjekt mehr als ein Eltern-CDO-Typ verfügbar ist. Die Anwendung füllt das Feld "Eltern-CDO-Typ" aus und zeigt das Objektfeld "Unterobjekt" automatisch an, wenn es für das ausgewählte Unterobjekt nur einen Eltern-CDO-Typ gibt. Wenn mehrere Eltern-CDO-Typen verfügbar sind, müssen Sie einen Eltern-CDO-Typ auswählen, damit die Anwendung das Objektfeld "Unterobjekt" anzeigt.

Wenn Sie ein Unterobjekt auswählen, wird eine Tabelle mit den Instanzen dieses Unterobjekts angezeigt. Durch Klicken auf die Schaltfläche "Protokoll anzeigen" wird das Popup "Modellierungsprotokoll" für die ausgewählte Instanz angezeigt. Nachdem Sie die Revision einer Instanz ausgewählt haben, aktiviert die Anwendung die Schaltfläche "Protokoll anzeigen".

Felddefinitionen der Seite "Modellierungsprotokoll" für ein Unterobjekt

In dieser Tabelle sind die Felder auf der Seite "Protokoll" bei Auswahl eines Unterobjekts als CDO-Typ definiert.

Feld

Definition

Typ

CDO-Typ

"CDO-Typ" stellt eine Liste der auswählbaren Modellierungsobjekte zur Verfügung.

Erforderlich

Eltern-CDO-Typ

Zeigt Eltern-CDO-Typen an, die für das ausgewählte Unterobjekt verfügbar sind. Die Anwendung füllt dieses Feld automatisch aus, wenn nur ein Eltern-CDO-Typ verfügbar ist.

Erforderlich

Objektinstanz-ID

Unterobjektinstanz des angegebenen Eltern-CDO-Typs. Die Anwendung zeigt ein Feld an, das alle Instanzen enthält, die für das ausgewählte Unterobjekt verfügbar sind.

Erforderlich

Vorgehensweise zum Anzeigen eines Modellierungsprotokolls für ein Unterobjekt

Gehen Sie folgendermaßen vor, um ein Protokoll für ein Unterobjekt anzuzeigen:

1. Öffnen Sie die Seite Protokoll. Die Seite "Protokoll" wird angezeigt.
2. Wählen Sie im Feld CDO-Typ ein Unterobjekt aus. Die Anwendung zeigt das Feld "Eltern-CDO-Typ" an. Die Anwendung zeigt auch das Objektfeld "Unterobjekt" an, wenn das Feld "Eltern-CDO-Typ" automatisch ausgefüllt wurde.
3. Wählen Sie den Eltern-CDO-Typ aus, falls erforderlich. Das Feld "Objektinstanz-ID" wird angezeigt.
4. Wählen Sie das Unterobjekt aus, dessen definierte Instanzen angezeigt werden sollen. Die Anwendung zeigt eine Tabelle mit allen Instanzen, die dem ausgewählten Objekt zugewiesen sind, an.
5. Wählen Sie ein Element aus der Tabelle aus.
6. Klicken Sie auf Protokoll anzeigen. Das Popup "Protokoll" wird angezeigt.
7. Klicken Sie auf einen der Pfeile, um das Protokoll zu erweitern und weitere Details anzuzeigen.
