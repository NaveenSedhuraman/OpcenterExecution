Anzeigen des Protokolls für ein Named Data Object

Ein Named Data Object (NDO) ist ein Objekt, das durch einen eindeutigen Namen identifiziert wird. Es handelt sich dabei um statische Objekte ohne Revisionsstände, z. B. Fabrik oder Mitarbeiter.

Beim Anzeigen von Named Data Objects

Das Feld im Abschnitt "Objektschlüsselwerte" ist dynamisch und hängt vom ausgewählten NDO ab. Wenn Sie z. B. im Feld "CDO-Typ" den Eintrag "Mitarbeiter" auswählen, zeigt die Anwendung ein Mitarbeiterfeld an.

Nachdem Sie eine Instanz ausgewählt haben, aktiviert die Anwendung die Schaltfläche "Protokoll anzeigen". Durch Klicken auf die Schaltfläche "Protokoll anzeigen" wird das Popup "Modellierungsprotokoll" für die ausgewählte Instanz angezeigt.

Felddefinitionen der Seite "Modellierungsprotokoll" für Named Data Objects

In dieser Tabelle sind die Felder der Seite "Protokoll" für NDOs definiert.

Feld

Definition

Typ

CDO-Typ

Enthält eine Liste der zur Auswahl stehenden Modellierungsobjekte.

Erforderlich

<CDO-Typ>

Named Data Object-Instanz, deren Protokoll Sie anzeigen möchten.

Hinweis: Ihre Auswahl im Feld "CDO-Typ" bestimmt die Beschriftung, die diesem Feld zugewiesen wird. Wenn Sie z. B. im Feld "CDO-Typ" den Eintrag "Mitarbeiter" auswählen, lautet die Beschriftung für dieses Feld "Mitarbeiter".

Vorgehensweise zum Anzeigen eines Modellierungsprotokolls für ein Named Data Object

Gehen Sie folgendermaßen vor, um das Protokoll für ein Named Data Object anzuzeigen:

1. Öffnen Sie die Seite Protokoll. Die Seite "Protokoll" wird angezeigt.
2. Wählen Sie aus dem Feld CDO-Typ ein Named Data Object aus. Das Feld "Objektname" wird unter "Objektschlüsselwerte" angezeigt.
3. Wählen Sie die Instanz aus, deren Protokoll Sie anzeigen möchten. Die Anwendung aktiviert die Schaltfläche "Protokoll anzeigen".
4. Klicken Sie auf Protokoll anzeigen. Das Popup "Modellierungsprotokoll" wird angezeigt.
5. Klicken Sie auf einen der Pfeile, um das Protokoll zu erweitern und weitere Details anzuzeigen.
