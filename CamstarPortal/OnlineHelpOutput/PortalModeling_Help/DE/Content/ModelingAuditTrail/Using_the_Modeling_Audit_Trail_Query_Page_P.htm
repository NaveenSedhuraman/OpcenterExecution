Verwenden der Seite "Protokoll"

Auf der Seite "Protokoll" können Sie das Protokoll für vorhandene und gelöschte Modellierungsobjekte anzeigen. Das Protokoll für eine Objektinstanz bleibt auch erhalten, nachdem das Objekt gelöscht wurde. Daher können Sie das Protokoll eines Objekts auch dann noch anzeigen, wenn dieses bereits gelöscht wurde. Auf der Seite "Protokoll" können Sie die folgenden Objekttypen anzeigen:

• Named Data Object
• Revisioniertes Objekt
• Unterobjekt

Beim Verwenden der Seite "Modellierungsprotokoll"

Klicken Sie in das Feld "CDO-Typ" und erweitern Sie "Named Data Object", "Revisioniertes Objekt" oder "Unterobjekt", um das entsprechende Modellierungsobjekt auszuwählen. Die Felder auf der Seite ändern sich, je nachdem, welchen CDO-Typ Sie ausgewählt haben.

Weitere Informationen zum Anzeigen von Protokollen für die verschiedenen CDO-Typen auf der Seite "Protokoll" finden Sie in folgenden Themen:

• "Anzeigen des Protokolls für ein Named Data Object"
• "Anzeigen des Protokolls für ein revisioniertes Objekt"
• "Anzeigen des Protokolls für ein Unterobjekt eines Modellierungsobjekts"

Die Anwendung aktiviert die Schaltfläche "Protokoll anzeigen", nachdem Sie einen CDO-Typ und ein Modellierungsobjekt ausgewählt haben. Wenn Sie auf diese Schaltfläche klicken, wird das Protokoll für das Modellierungsobjekt, das Sie ausgewählt haben, angezeigt.

Hinweis: Legen Sie Ihre bevorzugte Sprache fest, bevor Sie Modellierungsobjektinstanzen definieren, wenn die Anwendung alle Aktualisierungen im Modellierungsprotokoll in der bevorzugten Sprache anzeigen soll.

Objekttypen im Modellierungsprotokoll

Der Objekttyp, den Sie im Feld "CDO-Typ" auf der Seite "Protokoll" ausgewählt haben, bestimmt, welche anderen Felder auf der Seite angezeigt werden.

In dieser Tabelle sind die verschiedenen Felder definiert, die je nach dem ausgewählten Objekttyp angezeigt werden.

Wenn Sie diesen Typ auswählen . . .

Wird dieses Feld angezeigt

Named Data Object

Name

Dieses Feld entspricht dem NDO, das Sie ausgewählt haben. Wenn Sie z. B. einen Mitarbeiter als Objekttypnamen ausgewählt haben, wird das Objektnamensfeld "Mitarbeiter" angezeigt.

Revisioniertes Objekt

Objektname, Revision

Diese Felder entsprechen dem RO, das Sie ausgewählt haben. Wenn Sie z. B. ein Produkt als Objekttypnamen ausgewählt haben, wird das Objektnamensfeld "Produkt" angezeigt. Das Objektrevisionsfeld wird ebenfalls dem Produkt zugewiesen.

Unterobjekt

Eltern-CDO-Typ und Objektname

Diese Felder entsprechen dem Unterobjekt, das Sie ausgewählt haben. Wenn Sie z. B. "Trainingseintrag" als Objekttypname ausgewählt haben, wird das CDO-Feld "Eltern-Typ" angezeigt. Wenn Sie ein Eltern-CDO auswählen, z. B. Mitarbeiter, so entspricht das Feld "Objektname", das angezeigt wird, dem ausgewählten Elterntyp. In diesem Beispiel wird der Objektname "Mitarbeiter" angezeigt.
