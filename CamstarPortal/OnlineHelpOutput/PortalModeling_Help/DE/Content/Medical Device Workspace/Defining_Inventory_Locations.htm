Definieren von Lagerorten

Das Modellierungsobjekt "Lagerort" bietet die Möglichkeit, einen Aufbewahrungsort für Container, Lose oder Produkte zu erstellen. Informationen zum Zuweisen von Containern, Losen und Produkten zu einem Lagerort finden Sie im Opcenter Execution Medical Device and Diagnostics Shop Floor-Benutzerhandbuch.

Vorgehensweise zum Definieren eines Lagerorts

Gehen Sie folgendermaßen vor, um einen Lagerort zu definieren:

1. Öffnen Sie die Seite Lagerort. Die Seite "Lagerort" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie einen Namen für den Lagerort in das Feld Lagerort ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Unter "Gemeinsame Felder auf Modellierungsseiten" finden Sie Informationen zu den optionalen Feldern.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
