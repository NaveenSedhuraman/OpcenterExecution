Definieren von Gerätekennungen

Sie können Gerätekennungen mit dem Modellierungsobjekt für Gerätekennungen definieren. Dann können Sie diese Gerätekennungen auf der Startseite oder auf der Seite "Auftragserledigung" im Feld "Gerätekennung" direkt einem Container zuweisen. Sie können auch die Seite "Containerwartung" verwenden, um die Gerätekennung des Containers nach dem Start des Containers zuzuweisen. Informationen zum direkten Zuweisen von Gerätekennungen zu Containern finden Sie im Opcenter Execution Medical Device and Diagnostics Shop Floor-Benutzerhandbuch und im Opcenter Execution Core Shop Floor-Benutzerhandbuch.

Sie können einem Container eine Gerätekennung auf Grundlage von Informationen zuweisen, die im Modellierungsobjekt "Produkt" definiert sind. Weitere Informationen zum Definieren von Gerätekennungen für ein Produkt finden Sie unter "Definieren einer UDI auf der Seite "Produkt"".

Vorgehensweise zum Definieren einer Gerätekennung

Gehen Sie folgendermaßen vor, um eine Gerätekennung zu definieren:

1. Öffnen Sie die Seite Gerätekennung. Die Seite "Gerätekennung" wird innerhalb der Seite "Modellierung" angezeigt.
2. Klicken Sie auf Neu. Es werden leere Felder zum Definieren einer neuen Instanz angezeigt.
3. Geben Sie die Gerätekennung in das Feld Gerätekennung ein.
4. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in Tabelle mit den Felddefinition.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an.
