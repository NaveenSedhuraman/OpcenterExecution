Assigning Sampling Plans to Products

The Product page is used for assigning a sampling plan to a product. The two fields that are specific to AQL Sampling, Sampling Plan and Inspect All Containers, assign a sampling plan and indicate if all containers are to be sample tested.

When Defining the Product Object for AQL Sampling

This rule applies when modeling a Product object for AQL Sampling:

• The Sampling Plan field is required if the optional Inspect All Containers field is selected.

Product Page Field Definitions

This table defines the fields relevant to AQL Sampling on the Product page.

Field

Definition

Type

Sampling Plan

List of all defined sampling plans. Sampling plans define the inspection criteria for the product, such as the sample test required, number of samples, sample rate, where to sample, and the allowed rejects for a container of this product. Select the sampling plan to assign to the product.

Note: This field is required if the Inspect All Containers check box is selected.

Optional

Inspect All Containers

Check box indicating the sample rate is set to 1, which indicates that every container of this product is to be inspected.

This option is usually engaged when a production run is having quality issues to minimize losses or when a new product is going into production to verify that everything is set up properly.

Note: A sampling plan is required if this check box is selected.

Optional

Refer to "Defining Products" for information on the Product page.

How to Assign a Sampling Plan to a Product

Follow these steps to assign a Sampling Plan to a Product:

1. Open the Product page. The Product page appears within the Modeling page.
2. Click the product to be assigned a sampling plan. The instance and instance information appears.
3. Expand the Processing section.
4. Select the sampling plan to be assigned to the product from the list in the Sampling Plan field.
5. Optionally, select the Inspect All Containers check box to change the sample rate to 1, which indicates that every container of this product is to be inspected.
6. Click Save. The application saves the modeling object and displays a success message.
