Defining CIO Settings

The CIO Settings object contains configuration settings for outbound messaging. These settings include the URL for the Opcenter Connect MOM Client Gateway and the timeout for posting messages to the URL.

The CIO Settings object also contains configuration settings for the Opcenter Connect MOM Configuration service. These settings are:

• User name
• Password
• URL

Refer to the Opcenter Connect MOM User Guide for information on Opcenter Connect MOM.

The CIO installation process automatically creates a CIO Settings instance called ciosettings with default values populated in the Client Gateway URL and Client Gateway Post Timeout fields.

How to Define CIO Settings Manually

Follow these steps to define CIO Settings manually:

1. Open the CIO Settings page. The CIO Settings page appears within the Modeling page.
2. Click New. Blank fields appear for you to define a new instance.
3. Enter ciosettings in the CIO Settings field.
4. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
5. Click Save. The application saves the modeling object and displays a success message.
