Defining CIO Templates

The CIO Template object allows you to create the template for CIO outbound messaging. The structure of an outbound message is defined by an outbound message template, which has various formats, such as XML, Flat File, or a single Text Variable.

When Defining a CIO Template

The Template field must contain static content that can be used in Opcenter Connect MOM Mapping or as a content framework for outbound messages and text variables. This content may be an XML document or other literal text.

Click the Format XML button to format the template text as XML.

How to Define a CIO Template

Follow these steps to define a CIO Template:

1. Open the CIO Template page. The CIO Template page appears within the Modeling page.
2. Click New. Blank fields appear for you to define a new instance.
3. Enter the name for the template in the CIO Template field.
4. Enter the revision designation in the Revision field.
5. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
6. Click Save. The application saves the modeling object and displays a success message.
