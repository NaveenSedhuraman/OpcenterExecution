Modeling Audit Trail

Modeling Audit Trail provides a complete history (audit trail) of all changes to a modeling object. This helps companies track changes to modeling objects. Audit trails also help regulated industries show compliance with quality requirements.

An audit trail provides a sequence of audit records that result from activity on an object such as creating, modifying, or deleting the object. It is intended to provide an exact record of all changes made to an object during all edit sessions.

When a new object is created, the audit trail records all information for the new object. When an instance has been created, the audit trail provides details on what attributes of the instance were changed, who made the change, and when the change was made.

When a change is made to a revisioned base object, the change record is written to the audit trail for the revision.
