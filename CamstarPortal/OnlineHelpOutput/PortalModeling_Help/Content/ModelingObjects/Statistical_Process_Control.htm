Statistical Process Control

Quality data is an invaluable source of production information. Specifications in Opcenter EX MDD and Opcenter EX CR ensure that work in process (WIP) units that do not meet minimum criteria cannot be shipped, but this is only the beginning. Manufacturers can also produce high quality products without sacrificing productivity. Opcenter Execution Core Inline Statistical Process Control (Inline SPC) and Opcenter Execution Core Statistical Process Control (Legacy SPC) are powerful and flexible modules that enables manufacturers to solve problems before they cause a reduction in yield or throughput.

Opcenter Execution Core Inline Statistical Process Control (Inline SPC)

Inline SPC provides the same functionality as Legacy SPC using the Siemens SPC Common Component functionality for data calculation and visualization.

Refer to the Opcenter Execution Core Inline Statistical Process Control Reference Guide for information.
