Physical Model Definitions

The physical model represents the physical components of your manufacturing facility, primarily the Enterprise, Factory, Location, Organization, Employee, Role, and Resource. Other objects are Resource Group, Setup, Document Set, Document, Document Viewer, Recipe, and Work Center.

With the exception of Document, which is required in Document Set, you can define these objects in any sequence.

When Creating Modeling Definitions

When creating modeling definitions, you cannot use invalid characters in the object name. For example, you cannot use an apostrophe. Refer to the appropriate product-specific system Designer user guide below for information:

• Opcenter Execution Core Designer User Guide
• Opcenter Execution Medical Device and Diagnostics Designer User Guide
