Defining SPC Chart Groups

SPC Chart Group is used to define a collection of SPC charts, SPC chart groups, or both, depending on whether a single chart or a group of charts should be displayed. They are processed concurrently. This object is a named object group.

You can specify an SPC chart group on the Data Collection Def or User Data Collection Def pages if using the traditional data collection methodology. For task-based data collection, you can specify a data collection definition when defining a task using the Task List modeling object.

When Defining an SPC Chart Group

SPC Chart Group is an optional field when you are defining Data Collection and User Data Collection. An SPC chart group can be attached to a data collection definition and the data collection definition can be attached to a spec.

SPC Chart Group contains the optional Modeling definition, SPC Chart.

The Resolved Entries button enables you to display a list of all of the values specified in the entries list for this group and all of the nested groups.

Note: Legacy SPC and Inline SPC Charts cannot be included in the same Chart Group.

Refer to the Opcenter Execution Core Inline Statistical Process Control Reference Guide or the Opcenter Execution Core Statistical Process Control Reference Guide for information on setting up and configuring SPC.

How to Define an SPC Chart Group

Follow these steps to define an SPC Chart Group:

1. Open the SPC Chart Group. The SPC Chart Group page appears within the Modeling page.
2. Click New. Blank fields appear for you to define a new instance.
3. Enter a name for the new chart group in the SPC Chart Group field.
4. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
5. Click Save. The application saves the modeling object and displays a success message.

How to Add Entries to an SPC Chart Group

Follow these steps to add entries to an SPC chart group:

1. Perform the "How to Define an SPC Chart Group" procedure.

Or

Select an existing SPC Chart Group instance.

2. Click Add new row on the Entries grid. A new row appears.
3. Click in the new row to display a list of SPC charts, and then select one to add to the group. The application adds the selected entry to the grid.
4. Repeat steps 2-3 to add another entry to the grid.
5. Click Save. The application displays a success message indicating the modeling object was updated.

How to Add Groups to an SPC Chart Group

Follow these steps to add entries to an SPC chart group:

1. Perform the "How to Define an SPC Chart Group" procedure.

Or

Select an existing SPC Chart Group instance.

2. Click Add new row on the Groups grid. A new row appears.
3. Click in the new row to display a list of previously defined groups, and then select one to add (as a subgroup) to the group. The application adds the selected group to the grid.
4. Repeat steps 2–3 to add another group to the grid.
5. Click Save. The application displays a success message indicating the modeling object was updated.
