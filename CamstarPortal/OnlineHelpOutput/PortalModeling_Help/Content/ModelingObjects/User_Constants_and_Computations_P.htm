User Constants and Computations

The application provides the ability to create and store computational expressions and map those expression variables to a user data collection data point. These computations can then be used while executing an electronic procedure task, and allow the shop floor operator to view the result of the computation. This allows manufacturing operators to use predefined computations during manufacturing steps, and perform calculations as per the computation. The result of the calculation returned is based on the constants, parameters, and operator input provided to the computation expression by you.
