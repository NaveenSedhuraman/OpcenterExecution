Defining Resources and Resource Setup

Resources are part of the physical components in your manufacturing facility. They are the machines, equipment, and people involved in your manufacturing processes. Resources and several other resource-related objects are defined in Modeling.

As part of resource definition, you set up the system to handle resource states, also called resource setup. This component defines the states that a resource can enter and the transitions allowed from one resource state to another as its status changes. For example, a machine that is Idle can go to Down or Running; a machine that is Down, can go to Repair or Qualification, or back online (Up).
