Setting Up a Workstation or Work Cell Resource

Use the Resource Setup transaction to record changes to a workstation or work cell setup to indicate their availability. A resource’s availability determines whether you can use it to perform shop floor transactions.

Refer to the Opcenter Execution Medical Device and Diagnostics Shop Floor User Guide or the Opcenter Execution Core Shop Floor User Guide for information on resource setup and availability. Use the procedures on this page to simply set up a workstation or work cell.

When Setting Up a Resource

When setting up a workstation or work cell as a resource, be sure to select Up from the Availability field. Although the Availability field is optional, setting the availability to Up ensures the resource will appear in relevant resource selection values lists. If you do not set the availability to Up, the resource may not appear as a valid selection, even if the resource is defined in the resource group.

How to Set Up a Resource

Follow these steps to set up a Workstation or Work Cell as a Resource:

1. Open the Resource Setup page. The Resource Setup page appears.
2. Select the appropriate Resource Group to limit the list of resources displayed in the Resource field.
3. Select the workstation or work cell to set up from the Resource field.
4. Select a Resource Status Code for which the availability is Up. The values are updated and displayed in the Availability field.
5. Enter optional information according to your business requirements. Refer to the Portal Shop Floor User Guide for information on the optional fields.
6. Click Setup. The application displays a success message.
