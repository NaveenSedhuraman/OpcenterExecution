Defining Vendors

A Vendor is a provider of goods and services such as:

• Components and subassemblies used to manufacture products
• Products purchased for resale
• Nonproduction materials
• Services such as resource and building maintenance

Vendor definitions can also contain a list of the products (items) that the vendor provides. You can reference vendor items from purchased-product definitions to indicate the vendor from which product is procured.

When Defining a Vendor

Vendor is an optional field in the definition of Products.

How to Define a Vendor

Follow these steps to define a Vendor:

1. Open the Vendor page. The Vendor page appears within the Modeling page.
2. Click New. Blank fields appear for you to define a new instance.
3. Enter the name of the vendor in the Vendor field.
4. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
5. Click Save. The application saves the modeling object and displays a success message.

How to Add Vendor Items

Follow these steps to add Vendor items:

1. Perform the "How to Define a Vendor" procedure.

Or

Select an existing Vendor instance.

2. Click Add new row on the Vendor Items grid to add an item provided by the vendor. A new row appears.
3. Enter the name of the item provided by the vendor in the Vendor Item Name field.
4. Enter values for the following optional information if needed:
• Minimum Order Quantity
• Lead Time
• Cost
5. Repeat steps 2-4 to add another item if needed.
6. Click Save. The application displays a success message indicating the modeling object was updated.
