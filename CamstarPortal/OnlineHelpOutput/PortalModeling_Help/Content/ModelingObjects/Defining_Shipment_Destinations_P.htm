Defining Shipment Destinations

A Shipment Destination is the factory or customer to which the container is shipped. Shipment destination also contains the routing information used when the ship transaction is sending remote transactions to a destination site.

When Defining a Shipment Destination

Shipment Destination contains the optional modeling definitions:

• Customer
• Factory, and Locations defined for the factory
• Site

Shipment Destination is an optional field in the following:

• Operation modeling definition
• Shipment Destination Group modeling definition
• RemoteReceive transaction

Shipment Destination is a required field in the InventoryTransfer and Ship transactions.

How to Define a Shipment Destination

Follow these steps to define a Shipment Destination:

1. Open the Shipment Destination. The Shipment Destination page appears within the Modeling page.
2. Click New. Blank fields appear for you to define a new instance.
3. Enter a name for the destination in the Shipment Destination field.
4. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
5. Click Save. The application saves the modeling object and displays a success message.
