Viewing Containers Based on Manufacturing Orders

You can keep track of containers that have been started against a manufacturing order by viewing the instance in the application.

The Containers field displays a read-only list of container names started against this order. Quantities and other status information are not provided. This view is available in Modeling but does not display to the shop floor user.

How to View a Container

Follow these steps to view a container based on manufacturing order:

1. Click Mfg Order on the Modeling tab. The Manufacturing Order tab appears.
2. Select an instance from the Mfg Order list. The Mfg Order page appears.
3. Go to the Details section.
4. View the entries in the Containers grid.
