Modeling Objects Introduction

Modeling objects are available from the Modeling menu in the Portal. You can view modeling objects either alphabetically or by groups. The alphabetical view allows you to view all modeling objects in alphabetical order. Currently, the only group available is the All Maint group, which also contains all modeling objects.

The Portal also allows you to filter modeling objects to limit the number of objects displayed. For example, you can filter on the word Group to see only modeling objects that are named object groups or revisioned object groups.
