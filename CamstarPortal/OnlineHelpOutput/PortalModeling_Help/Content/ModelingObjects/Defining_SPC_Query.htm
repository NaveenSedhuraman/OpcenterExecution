Defining SPC Queries

The SPC Query object is used to model a query to plot SPC related to your process. You can copy existing queries from existing Legacy SPC macros to create a SPC Query. You can also create new SPC Queries to use with Inline SPC Solution.

When Defining New SPC Queries

Writing SPC queries requires that you be familiar with the Opcenter Execution Core SPC Object Model and SQL. You define new SPC queries using the SPC Query page.

SPC basic queries have been developed with numerous dynamic parameters to minimize the need to create new SPC queries.

How to Define an SPC Query

Follow these steps to define an SPC Query:

1. Open the SPC Query page. The SPC Query page appears within the Modeling page.
2. Click New. Blank fields appear for you to define a new instance.
3. Enter the name of the SPC Query in the SPC Query field.
4. Enter the text of the SQL query statement in the Query Text field.
5. Perform the following to add parameters to the Parameters grid:
a. Click Add new row.The application adds a blank row to the grid.
b. Enter the name for the SPC query parameter in the User Query Parameter Name field.
c. Enter the data type of the SPC query parameter in the Data Type field.
d. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
6. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
7. Click Save. The application saves the modeling object and displays a success message.

How to Test the Query

Follow these steps to test an SPC query definition:

1. Perform the steps in "How to Define an SPC Query."

Or

Select an existing SPC Query instance.

2. Click Save And Test Query to save and test the query. The application opens the Online Query Setup Test pop-up.
3. Enter any parameters required by the query.
4. Click Test Query. Any results matching the query and parameters entered are displayed in the Query Results grid.
5. Click Close. The Online Query Setup Test pop-up closes.
