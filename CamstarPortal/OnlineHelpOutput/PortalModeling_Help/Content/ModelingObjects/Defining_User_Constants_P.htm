Defining User Constants

User Constant values can be used in computation expressions if needed. They must first be defined as user constants. User constants allow you to create and maintain constant values which do not change like "pi," or which rarely change. You may have some constants specific to your business process which are used across multiple computations.

When Defining User Constants

User constant expressions must be defined using Opcenter EX MDD or Opcenter EX CR unified expressions. Refer to the unified expressions appendix in the Opcenter Execution Medical Device and Diagnostics Designer User Guide or the Opcenter Execution Core Designer User Guide for information on constant values and expressions.

When entering literals for user constants, enclose the entry in double quotes (“).

The User Constants function supports the following operators:

• Addition
• Subtraction
• Multiplication
• Division
• Sum (multiple parameters)
• Range (multiple parameters)
• Unary Minus/Plus Operator
• Application Functions
• All application timestamp-based calculations

How to Define a User Constant

Follow these steps to define a User Constant:

1. Open the User Constant page. The User Constant page appears within the Modeling page.
2. Click New. Blank fields appear for you to define a new instance.
3. Enter a name for this instance in the User Constant field.
4. Select a Data Type from the field.
5. Enter a value to assign to the data type in the Value field.
6. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
7. Click Save. The application saves the modeling object and displays a success message.
