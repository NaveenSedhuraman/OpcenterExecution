Execution Model Definitions

The Execution component of the Information Model tracks the status and history of products and resources on the shop floor and provides a real-time view of production activity.

Opcenter EX MDD and Opcenter EX CR deliver comprehensive production management, yield, resource utilization, and equipment status information. With this information, not only managers and operators, but also key players in the supply chain have the visibility to make decisions and improve production cost and efficiency. Opcenter EX MDD and Opcenter EX CR can operate in stand-alone mode or integrated with other Factory, ERP, or Supply Chain systems. Containers are created (started) and then processing is tracked on the shop floor. Opcenter EX MDD and Opcenter EX CR track containers and resources throughout the production process.

Execution includes these key components for which you will create some modeling definitions:

• Tracking and control
• Real-time WIP status and genealogy
• Real-time machine status and history
