Defining Units of Measure

The UOM modeling object enables you to define units of measure that can be associated with a container's quantity.

When Defining Units of Measure

The names for each unit of measure must be unique. Examples of units of measure include:

• Each
• kg
• ml

UOM is an optional field in these modeling objects:

• Product
• Product Family
• Mfg Order
• Sales Order

UOM is an optional field in these transactions:

• Container Maintenance
• Resource Thruput
• Start
• Start - Two Level

When Defining Conversion Information for Units of Measure

You can define quantity conversion information for a unit of measure when Batch Processing is installed. The UOM Conversion grid is available only with Batch Processing. The grid allow you to define:

• A factor to convert that unit of measure to another.
• A rule for a converted quantity to be rounded.
• The number of decimal places allowed for the calculated conversion quantity. The maximum number is 4.

These rules apply when defining conversion information for units of measure:

• Both the Rounding Rule and Decimal Places fields must be populated for rounding to occur.
• Conversion information defined on the product takes precedence over conversion information defined on the product family.
• Conversion information defined on the product family takes precedence over conversion information defined on the UOM.

Refer to "Defining UOM Conversion Information on the Product" and "Defining UOM Conversion Information on the Product Family" for information.

How to Define a Unit of Measure

Follow these steps to define a Unit of Measure:

1. Open the UOM page. The UOM page appears within the Modeling page.
2. Click New. Blank fields appear for you to define a new instance.
3. Enter a name for the unit of measure in the UOM field.
4. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
5. Click Save. The application saves the modeling object and displays a success message.
