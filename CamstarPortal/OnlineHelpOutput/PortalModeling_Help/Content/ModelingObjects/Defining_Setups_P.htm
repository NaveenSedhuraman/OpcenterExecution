Defining Setups

A Setup is the physical machine configuration for a particular process. It can include a document set and the estimated resource setup time. It is a configuration of a resource needed to process a certain type of material (typically a particular product or a container referencing a particular spec).

Normally the definition of a setup includes only items as hardware configurations that take significant time to change, while a recipe can include many other items such as process settings that may be changed instantly.

Standard Siemens-provided setup attributes include items such as the applicable set of documents describing the setup configuration, an estimate of time needed to configure the setup, and so forth. Setups are often defined to help provide scheduling or dispatching information on which containers or orders may be processed without significant changeover time between the items.

When Defining a Setup

Setup contains the optional modeling definition Document Set.

Setup is an optional field:

• In the definition of Spec.
• In the transactions Resource Setup and Resource Thruput.

Setup Page Field Definitions

This table defines the fields unique to the Setup page.

Refer to "Common Fields on Modeling Pages" for information on the fields common to all modeling objects.

Field

Definition

Type

General

Engineering Change Order

Engineering change order assigned to this revision. You can enter a maximum of 30 characters.

Optional

Document Set

Name of the document set associated with the setup. Typical setup documents include equipment drawings, parts lists, setup instructions, and recipes.

Optional

Standard Setup Time

Standard total elapsed time (in days, hours, and minutes) required to complete the activities associated with the setup. The value is the standard used for cycle time calculations.

Optional

Standard Setup Labor Time

Standard total labor time (in days, hours, and minutes) required to complete the activities associated with the setup. The value will be different from the elapsed time value in the Standard Setup Time field if the setup requires more than one person.

Optional

How to Define a Setup

Follow these steps to define a Setup:

1. Open the Setup page. The Setup page appears within the Modeling page.
2. Click New. Blank fields appear for you to define a new instance.
3. Enter a name for this setup in the Setup field.
4. Enter the revision of this object in the Revision field.
5. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
6. Click Save. The application saves the modeling object and displays a success message.
