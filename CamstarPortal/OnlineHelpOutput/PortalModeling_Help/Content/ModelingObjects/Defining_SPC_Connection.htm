Defining SPC Connections

The SPC Connection object is used to establish a 64-bit ODBC database connection. This is required to retrieve chart plotting data for Inline SPC.

Use the appropriate database driver. For example, SQL Server should have an ODBC driver called SQL Server and Oracle should have an ODBC driver similar to Oracle In OraDb11g_home1.

Note: You must know the connection values to configure this object.

How to Define an SPC Connection

Follow these steps to define an SPC Connection:

1. Open the SPC Connection page. The SPC Connection page appears within the Modeling page.
2. Click New. Blank fields appear for you to define a new instance.
3. Enter a name in the SPC Connection field.
4. Enter the ODBC connection string information in the following fields:
• Data Source Name
• Username
• Password
5. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
6. Click Save. The application saves the modeling object and displays a success message.
