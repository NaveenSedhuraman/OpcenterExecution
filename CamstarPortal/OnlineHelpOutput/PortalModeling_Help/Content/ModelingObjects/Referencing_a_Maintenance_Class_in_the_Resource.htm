Referencing a Maintenance Class in the Resource

Resources that have the same maintenance requirement belong to a Maintenance Class. Maintenance classes also apply to carriers, which are another type of resource.

Refer to "Physical Model Definitions" for information on resources, or carriers.

How to Modify a Resource to Reference the Maintenance Class

Follow these steps to specify the Maintenance Class to which the resource belongs:

1. Open the Resource page. The Resource page appears within the Modeling page.
2. Click the resource that you want to modify. The Resource page appears for the selected resource.
3. Select the appropriate Maintenance Class for this resource in the Maintenance Class field.
4. Enter optional information according to your business requirements. Refer to the "Resource Page Field Definitions" for information on this page.
5. Click Save. The application displays a success message indicating the modeling object was updated.
