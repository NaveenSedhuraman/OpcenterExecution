Defining AQL Levels

The AQL Level page enables you to define acceptable quality levels by associating a sample size and reject count to a sample size code.

When Defining AQL Levels

These rules apply when modeling the AQL Level:

• The AQL Level page must include at least one item in the Sample Size Details grid.
• The AQL Level sample size codes need to match the sample size codes identified on the inspection level instances.

AQL Level is an optional field in the definition of Sampling Plan.

How to Define an AQL Level

Follow these steps to define an AQL Level:

1. Open the AQL Level page. The AQL Level page appears within the Modeling page.
2. Click New. Blank fields appear for you to define a new instance.
3. Enter a name for the AQL level in the AQL Level field.
4. Click Add new row in the Sample Size Details grid. A new row appears.
5. Enter a sample size code in the Sample Size Code field in the row you just created.
6. Enter a corresponding sample size in the Sample Size field.
7. Enter the maximum number of defects allowed in a sample in the Rejection Number field.
8. Repeat steps 4-7 for each sample size.
9. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
10. Click Save. The application saves the modeling object and displays a success message.
