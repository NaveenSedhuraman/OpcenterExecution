"What If" Planning

Opcenter Execution Core Scheduling provides "What If" Planning functionality to analyze future, or "what if," scheduling scenarios based on a planned product mix. Some or all of the manufacturing orders have yet to be created in these situations. The application sends products and their associated ERP routes from Opcenter EX MDD or Opcenter EX CR to Preactor. This data enables the user to create hypothetical orders in Preactor to analyze "what if" scheduling scenarios.

You must enable Opcenter Execution Core Scheduling to use "what if" planning. You must define an ERP route. You must also define a product and assign the ERP route to the product. Refer to "Enabling Opcenter Execution Core Scheduling on the Factory Page" for information. Refer to "When Defining ERP Routes for Use with Opcenter Execution Core Scheduling" for information on defining ERP routes. Refer to "When Defining Products for Use with Opcenter Execution Core Scheduling" for information on defining products.
