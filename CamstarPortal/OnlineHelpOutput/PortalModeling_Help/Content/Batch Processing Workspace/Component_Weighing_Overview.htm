Component Weighing Overview

Note: The functionality described below is available only when Batch Processing is installed. Refer to the Opcenter Execution Medical Device and Diagnostics Batch Processing User Guide or the Opcenter Execution Core Batch Processing User Guide for information.

The recipe instructions for processing a batch are created using existing electronic procedures functionality. Recipe steps may include any other task types required to produce the batch in addition to a weighing task. You can create a weigh issue task using the Task List modeling object to define the weighing and issuing of a component product to a batch container. The weigh issue task allows you to define the point in the process the component is to be weighed. The weigh issue task also allows you to define the rules for weighing the component. Refer to "When Defining Weigh Issue Tasks" for information on creating a weigh issue task.
