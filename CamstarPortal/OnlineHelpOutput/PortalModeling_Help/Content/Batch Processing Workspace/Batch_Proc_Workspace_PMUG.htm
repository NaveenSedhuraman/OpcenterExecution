Batch Processing

Note: The functionality described below is available only when Batch Processing is installed. Refer to the Opcenter Execution Medical Device and Diagnostics Batch Processing User Guide or the Opcenter Execution Core Batch Processing User Guide for information.

Batch Processing provides a mechanism to deliver features associated with:

• Scale management
• Recipe management
• UOM conversion
• Component weighing

Manufacturers can use these features to control and track the production of small chemical batches used as components of their devices. Examples include coatings for blood glucose strips and drug formulations for transdermal patches.

You must define a material list to use batch processing. You can define the required quantity and the minimum and maximum quantities for each component that is weighed when defining the material list. Material lists can be defined for the finished product using the Material List grid on the BOM, ERP BOM, or Mfg Order modeling objects. Refer to "Defining Allowed Component Quantities on the BOM," "Defining Allowed Component Quantities on the ERP BOM," and "Defining Allowed Component Quantities on the Manufacturing Order" for information.
