Overall Equipment Effectiveness Overview

Note: The functionality described below is available only when the Industry Solutions workspace is installed. Refer to the Industry Solutions User Guide for information.

Overall Equipment Effectiveness (OEE) evaluates how effectively a manufacturing resource is performing. OEE represents the true productive time for a resource relative to its actual capacity. An OEE of 100% means that the resource is producing only good-quality output with no downtime as fast as that resource can produce. Manufacturers can use OEE data to improve operations and optimize the capacity of available resources. OEE consists of these elements:

• Availability
• Performance
• Quality

The formula for calculating OEE is the product of these elements:

OEE = Availability * Performance * Quality

During processing, the application stores the data necessary to calculate OEE in the database in tables. These tables facilitate OEE calculations and provide the ability to analyze each element of OEE.
