Changing an Existing Electronic Signature Assignment

You can change which electronic signature requirement has been assigned to a service. Use the Modeling ESig page to make the change in assignment. You can also completely remove an electronic signature requirement assignment. Refer to “Removing an Existing Electronic Signature Assignment” for information.

How to Change an Existing Electronic Signature Assignment

Follow these steps to change an existing electronic signature assignment:

1. Open the Modeling ESig page. The Modeling ESig page appears.
2. Select the modeling object service with the electronic signature requirement that you want to change from the list in the Services section.
3. Select an electronic signature requirement from the appropriate action list Create, Update, Delete, or Lock/Unlock in the Signature Requirement Assignment grid to change an existing requirement for the service.
4. Click Update. The new electronic signature requirement is assigned to the service.
