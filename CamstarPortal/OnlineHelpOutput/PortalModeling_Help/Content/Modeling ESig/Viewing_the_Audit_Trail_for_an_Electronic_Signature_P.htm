Viewing the Audit Trail for an Electronic Signature

Electronic signature details are displayed as part of the Modeling audit trail for the modeling object. The action associated with the electronic signature requirement assignment is listed in the Action column in the audit trail.

Refer to "Modeling Audit Trail" for information on the Modeling audit trail.

Modeling Audit Trail Pop-Up Field Definitions

Refer to "Modeling Audit Trail Pop-Up Field Definitions" for definitions of the fields on the pop-up.

How to View the Modeling Audit Trail Pop-Up for an Electronic Signature

Follow these steps to view the Modeling Audit Trail pop-up for an electronic signature:

1. Open a modeling object that contains an electronic signature requirement. The modeling object page appears with a list of existing instances of that object.
2. Select the existing modeling object instance for which you want to view the modeling audit trail. The page for that modeling object instance appears.
3. Click Audit Trail. The Modeling Audit Trail pop-up appears in the collapsed view.
