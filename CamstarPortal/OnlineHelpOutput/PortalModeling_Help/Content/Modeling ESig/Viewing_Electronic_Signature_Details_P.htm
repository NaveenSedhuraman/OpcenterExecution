Viewing Electronic Signature Details

The Modeling ESig page enables you to view the details of an electronic signature assignment. Select a modeling object service in the Services section to display any electronic signature assignment details for Create, Update, Delete, and Lock/Unlock in the Resolved Signature Requirements section. Configurable data object names are also displayed.

When Viewing Electronic Signature Details

The Lock/Unlock fields are only available for revisioned object services.

How to View Electronic Signature Details

Follow these steps to view electronic signature details:

1. Open the Modeling ESig page. The Modeling ESig page appears.
2. Select the modeling object service whose details you want to view from the list in the Services section. ESignature assignment details for Create, Update, Delete, and Lock/Unlock appear in the Resolved Signature Requirements section.
