Modeling Electronic Signature Information

Modeling ESignature enables an Opcenter EX MDD or Opcenter EX CR user, such as a modeler, to manage ESignature requirement assignments. Modeling ESignature enables you to:

• Assign a new electronic signature requirement
• View an existing electronic signature requirement assignment
• Change an existing electronic signature requirement assignment
• Remove an existing electronic signature requirement assignment
• Assign an electronic signature requirement to Lock/Unlock

Modeling electronic signatures is only available when you purchase the separately licensed Electronic Signatures module.

When Using Electronic Signatures with Modeling Services

Only one electronic signature requirement can be assigned to a maintenance service at one time.

Modeling ESignatures provides two options for the signature requirement assignment in addition to the list of electronic signature requirements:

• Inherit, the default assignment option. Inherit enables the selected service to inherit the requirement assignment of an ancestor.
• None, prevents the application from assigning any requirement to the service. This option is also used to remove the existing assignment.

Use the Update button to update the electronic signature requirement assigned to the service.

Modeling ESig Page

The Modeling ESig page enables you to update electronic signature requirements for modeling objects. All services are listed in a hierarchical fashion, allowing you to create or update an electronic signature requirement for a parent or a child. By default, the application applies the changes to the service’s children by inheritance. However, this can be overridden.

The Lock/Unlock fields are only available for revisioned object services. Assigning an electronic signature requirement to Lock/Unlock requires one or more electronic signatures whenever a modeler tries to lock or unlock a revisioned object.

The Resolved Signature Requirement fields display what requirement was assigned and where it was assigned.
