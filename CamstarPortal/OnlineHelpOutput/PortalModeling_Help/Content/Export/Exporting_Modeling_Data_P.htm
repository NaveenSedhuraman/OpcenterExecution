Exporting and Importing Modeling Data

The Portal in Opcenter EX MDD and Opcenter EX CR provides the ability to export and import modeling object data. You can export modeling object data from a source Opcenter EX MDD or Opcenter EX CR system and import that data into a counterpart destination system. For example, you can promote one or all of the Factory instances from a development system to a production system. The process by which the exported modeling data package from the export environment is delivered to the import environment is the responsibility of your organization.

The application displays a message indicating the success or failure of the export or import process on the last page of the pageflow, the Execution Console page. You can view more detailed summary and error information for the process by launching the Execution Summary page from the Execution Console page.
