Viewing the Status of the Export or Import

The Execution Summary page appears when you click Finish on the Execution Console page after performing an export or import of modeling object instance data. The page displays summary and error information for the export or import process. Refer to "Exporting and Importing Modeling Data" for information on exporting and importing.

The Success Summary tab appears on top by default. Click the Error Summary tab to view any errors that occurred. The Error Summary tab is blank if no errors occurred.

Note: Both the Success Summary and Error Summary tabs will be blank if you click the Finish button on the Execution Console page before the application completes the export or import process.
