Accessing Modeling Pages

All modeling object pages are accessed from the main Modeling page, which is accessed from the Modeling menu. The application opens the modeling object page you select within the Modeling page itself. Refer to "Modeling Page" for a complete description of the functionality available on the Modeling page.

How to Access a Modeling Page

Follow these steps to access a Modeling page:

1. Log on to the Portal.
Note: Log in as CamstarAdmin, InSiteAdmin, or Administrator if this is the initial login after installation.
2. Select Modeling from the Modeling menu. The Modeling page appears.
3. Click the All Maint arrow to see a list of modeling objects under that group.

Or

Select View All (Alphabetical) from the View Objects by list to display a list of modeling objects in alphabetical order. The application displays a list of all modeling objects in alphabetical order.

4. Select a modeling object. The object's page appears within the Modeling page.
