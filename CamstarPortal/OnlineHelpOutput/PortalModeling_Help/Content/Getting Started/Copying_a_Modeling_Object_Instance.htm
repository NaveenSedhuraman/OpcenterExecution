Copying a Modeling Object Instance

The application displays the Copy or Copy Rev pop-up when you click the Copy and Copy Rev buttons on the toolbar for a selected Modeling instance. Clicking the Copy button on the Copy or Copy Rev pop-up, or pressing Enter on the keyboard, saves the new instance or revision of an instance and closes the Copy or Copy Rev pop-up. The application displays a success message indicating the new modeling object instance was created.

Performing a Copy creates a copy of the selected instance with all of the original information except the instance name (and revision name for revisioned objects). The application requires you to enter a new instance name before saving a named data object instance and a new instance name and revision value before saving a revisioned object.

Performing a Copy Rev creates a copy of the revisioned object with all the information from the original revision except the revision value. The application requires you to enter a new revision value before saving a new revision of the selected instance.

Note: The Copy Rev button appears for revisioned objects only.

How to Copy a Revisioned Object Instance

Follow these steps to copy a revisioned object instance:

1. Select Modeling from the Modeling menu. The Modeling page appears.
2. Click a revisioned object. The revisioned object's page appears within the Modeling page.
3. Select an existing instance of the revisioned object. The instance appears.
4. Click Copy. The Copy pop-up appears.
5. Enter a name for the Modeling instance in the Target Instance Name field.
6. Enter a revision in the New Revision Name field.
7. Click Copy or press Enter on the keyboard.
8. Click Save. The application saves the modeling object and displays a success message.
