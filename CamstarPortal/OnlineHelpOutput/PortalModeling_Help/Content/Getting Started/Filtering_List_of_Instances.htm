Filtering the List of Instances

The Filter field can be used to narrow down a large number of modeling instances.

Enter a fragment of a string and click Refresh to display a list of instances with names beginning with that string fragment. Click Refresh with the Filter field blank to return all instances.

Lists of returned instances are sorted alphabetically in the preferred language, if that language is supported by the Portal. Lists of returned instances are sorted alphabetically in English by default. A list control appears below the filter if the number of defined instances is greater than 100.

Opcenter Execution Core and Opcenter Execution MDD support multiple databases. These databases differ in their default approach to sorting lists that contain accent marks. Refer to the appropriate product-specific system administration guide below for information on linguistic sorting:

• Opcenter Execution Core System Administration Guide
• Opcenter Execution Medical Device and Diagnostics System Administration Guide
