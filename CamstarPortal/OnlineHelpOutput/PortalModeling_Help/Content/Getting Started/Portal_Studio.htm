Portal Studio

Modeling pages are defined in Portal Studio. Portal Studio is a self-contained development and configuration tool used that reduces initial installation setup and configuration time, and greatly reduces the need for custom code.

Portal Studio offers several benefits over traditional development tools:

• No additional installations or controls are required.
• All you need is a browser and a URL for an installed environment.
• Portal Studio has a built-in source control.
• Changes do not need to be compiled.
• You can test on-the-fly.
• Object model changes only require a simple refresh after the Windows Communication Foundation (WCF) services are generated.
• Any UI component can be exported to, or imported from, a single self-describing flat file.
• Basic page inheritance is included.
• Upgrades are automatically applied to the system level.
