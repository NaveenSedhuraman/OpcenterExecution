查看建模审核追踪弹出窗口

建模提供了查看现有建模对象中的审核跟踪的功能。可以从两个位置访问“审核追踪”弹出窗口：

• 对象的现有实例的建模页
• 从“建模”菜单访问的“审核追踪”页

从建模页访问审核追踪弹出窗口

选择对象（例如“工厂”）的现有实例时，“建模”页会显示一个“审核追踪”按钮。在新实例的建模对象页上没有“审核追踪”按钮。

例如，如果在“工厂”页上单击“新建”，应用程序将显示空白字段，以便添加新的工厂实例。此页面没有“审核追踪”按钮，因为此对象还没有审核历史。但是，保存新对象时，将为该实例创建审核历史，页面上也会显示“审核追踪”按钮。这同样也适用于您通过复制一个现有实例而创建的新实例。

单击“审核追踪”按钮将在“建模审核追踪”弹出窗口中显示对象的审核追踪。

从审核追踪页访问审核追踪弹出窗口

可以从“审核追踪”页访问建模对象的“审核追踪”弹出窗口。从“建模”菜单中选择“审核跟踪”可查看“审核追踪”页。在“审核追踪”页，可以根据特定于 CDO 类型的准则搜索已修订对象、已命名数据对象和子实体。然后，可单击“查看审核”按钮来显示该实例的“建模审核追踪”弹出窗口。

有关信息，请参考“使用审核追踪页”。

在查看建模审核追踪弹出窗口时

“建模审核追踪”弹出窗口最初提供审核追踪信息的折叠或汇总视图。

审核跟踪的折叠视图不提供有关分项或列表的信息。您必须使用审核跟踪的展开视图才可以向下延伸到分项和列表细节。单击箭头图标可以展开行项，查看其详细信息、子实体和列表。

应用程序将最近的更改显示在列表顶部，将最旧的更改显示在列表底部。

如果使用的是 Change Management，则可使用“包名称”字段将结果限制为指派给特定更改包的实例。如果指定的包没有结果，应用程序将显示一条消息，表示没有任何结果符合搜索准则。

Note: 必须先存在更改包，才能为要更新的包名称添加或更新的实例。如果在创建包之前进行了更改，建模对象审核追踪则不会更新包名称。

可使用“返回以下范围的结果：”字段更改所显示结果的日期范围。可从日历中选择日期或手动输入日期。如果指定范围内没有结果，应用程序将显示一条消息，表示没有任何结果符合搜索准则。

Note: 在 Portal Studio 中，可以配置默认情况下显示的行数设置。
Important: 当尝试查看包含大量记录的建模审核追踪时，应用程序可能会超时。可在 XMLClientConfig 文件中的超时设置中增加毫秒数。默认情况下，XMLClientConfig.xml 文件位于 C:\Program Files (x86)\Camstar\InSite Utility。

在打印建模审核追踪时

通过在键盘上按 Ctrl+P，可以从弹出窗口打印建模审核追踪。审核追踪将完全按照显示的样式进行打印。在打印建模审核追踪之前，必须展开要查看的部分。

如何从建模页查看审核追踪弹出窗口

按照以下步骤从“建模”页查看“审核追踪”弹出窗口：

1. 打开一个建模对象，例如“工厂”。随即“建模”页中将显示所选建模对象页。
2. 选择要查看其审核跟踪的现有建模对象实例。应用程序显示该实例的信息。
3. 单击审核追踪。将显示“建模审核追踪”弹出窗口的折叠视图。
4. 是否要将结果限制在特定的更改包？

如果 ...

操作步骤 ...

是
a. 在包名称字段中输入包名称。
b. 单击搜索。应用程序返回与特定包关联的所有结果。如果没有与包关联的结果，应用程序将显示一条错误消息。
c. 转到步骤 5。

否

转到步骤 5。
5. 是否要将结果限制在特定的日期范围内？

如果 ...

操作步骤 ...

是
a. 在第一个返回以下范围的结果: 字段中输入开始日期，或者从日历中选择该日期。
b. 在第二个返回以下范围的结果: 字段中输入结束日期，或者从日历中选择该日期。
c. 单击搜索。应用程序返回指定日期范围内的所有结果。如果此范围内没有结果，应用程序将显示一条错误消息。
d. 转到步骤 6。

否

转到步骤 6。
6. 单击任意箭头，展开审核追踪并查看更多详细信息。
7. 单击关闭，关闭“建模审核追踪”弹出窗口。

如何打印建模审核跟踪

按照以下步骤打印审核追踪：

1. 执行“如何从建模页查看审核追踪弹出窗口”流程。
2. 按 Ctrl-P。将显示“打印”弹出窗口。
3. 选择适当的打印机和设置。
4. 单击确定。 应用程序完全按照显示的样式打印“审核追踪”页和“建模审核追踪”弹出窗口。
