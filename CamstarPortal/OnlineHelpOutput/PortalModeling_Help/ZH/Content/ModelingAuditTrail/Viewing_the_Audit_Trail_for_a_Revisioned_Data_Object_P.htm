查看已修订对象的审核追踪

已修订对象 (RO) 是具有唯一名称和修订版本的对象。例如，产品便是 RO。

在查看已修订对象时

“对象关键字值”部分中的字段是动态的，根据所选择的 RO 而变化。例如，如果从“CDO 类型”字段中选择“产品”，应用程序将显示“产品”字段和“修订版本”字段。

应用程序将要求选择要查看的 RO 实例的特定修订版本。因此，必须展开对象实例，然后选择修订版本。不能选择实例，然后手动输入修订版本。

在选择实例的修订版本之后，应用程序将激活“查看审核”按钮。单击“查看审核”按钮将显示所选实例的“建模审核追踪”弹出窗口。

如何查看已修订对象的建模审核追踪

按照以下步骤查看已修订对象的审核追踪：

1. 打开审核追踪页。将显示“审核追踪”页。
2. 从 CDO 类型字段中选择一个已修订对象。对象名称和对象修订字段出现在“对象关键值”(Object Key Values) 下。
3. 选择要查看其审核追踪的实例以及该实例的修订版本。应用程序激活“查看审核”按钮。
4. 单击查看审核。将显示“建模审核追踪”弹出窗口。
5. 单击任意箭头，展开审核追踪并查看更多详细信息。
