查看命名数据对象的审核跟踪

已命名数据对象 (NDO) 是一个由唯一名称标识的对象。该对象代表了没有修订版本的静态实体，例如“工厂”或“员工”。

在查看已命名数据对象时

“对象关键字值”部分中的字段是动态的，根据所选择的 NDO 而变化。例如，如果从“CDO 类型”字段中选择“员工”，应用程序将显示“员工”字段。

在选择实例之后，应用程序将激活“查看审核”按钮。单击“查看审核”按钮将显示所选实例的“建模审核追踪”弹出窗口。

如何查看命名数据对象的建模审核跟踪

按照以下步骤查看已命名数据对象的审核追踪：

1. 打开审核追踪页。将显示“审核追踪”页。
2. 从 CDO 类型字段中选择一个已命名数据对象。对象名称字段出现在“对象关键值”(Object Key Values) 下。
3. 选择要查看其审核追踪的实例。应用程序激活“查看审核”按钮。
4. 单击查看审核。将显示“建模审核追踪”弹出窗口。
5. 单击任意箭头，展开审核追踪并查看更多详细信息。
