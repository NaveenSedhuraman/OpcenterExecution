选择建模对象实例

“实例选择”页是手动数据导出和导入页面流中的第二页。在此页面中可以执行下列某项操作：

• 在执行手动导出时选择要导出的建模对象实例
• 执行手动导入时指定要导入的 XML 文件，以及指定要导入的特定建模对象实例

在选择要导出的建模对象实例时

应用程序显示了为所选建模对象定义的所有实例。可以选中标题行的“全选”复选框以自动选择所有显示的实例，也可以选中各个实例对应的复选框。

单击“指派”按钮可将所选实例添加到“对象类型名称”表格并更新“对象和实例计数”表格。表格中的条目按照建模对象名称的字母顺序显示。要查看为导出指派的特定实例，必须单击建模对象行中的“展开”图标。

“对象和实例计数”表格表示以下内容：

• 可供选择的建模对象数
• 可供选择的建模对象实例总数
• 为导出指派的建模对象数
• 为导出指派的建模对象实例总数
• 所选建模对象引用的建模对象数
• 为导出指派的引用建模对象实例总数

示例:

选择指派容器级别建模对象的批、支架和托盘实例进行导出。单击“指派”按钮将在“对象类型名称”表格中显示容器级别，并更新“对象和实例计数”表格，表示一个“已选”建模对象和三个实例。可以在“对象类型名称”表格中展开“容器级别”行，查看批、支架和托盘实例。

在选择要导入的建模对象实例时

应用程序将显示选定要导入的文件中包含的所有实例。可以删除不想导入的实例。单击“对象类型名称”表格上的“删除所选行”按钮将删除所选实例。选中标题行中的“全选”复选框以自动选择所有显示的实例，也可以选中各个实例对应的复选框。要查看各个对象的实例，必须展开建模对象。

“对象和实例计数”表格表示：

• 可供导入的建模对象数
• 可供导入的建模对象实例总数
• 为导入指派的建模对象数
• 为导入指派的建模对象实例总数

“浏览”按钮可打开“文件上传”对话框。文件位置默认为桌面。Siemens 建议将导出的文件复制到执行导入时易于访问的位置。

如何选择要导出的建模对象实例

按照以下步骤选择要导出的建模对象实例：

1. 完成“如何选择数据传输类型”过程以访问“实例选择”页。
2. 在可用对象中选择建模对象。应用程序会显示为该建模对象定义的所有实例。
3. 单击可用实例中的全选复选框，选择所有实例。

或

选中每个实例旁边的复选框。

4. 单击指派。应用程序将对象及其所选实例添加到“对象类型名称”表格并更新“对象和实例计数”。
Note: 单击“对象类型名称”表格中的展开图标可查看为该建模对象指派的所有实例。
5. 重复步骤 2-4，指定其他实例。
6. 单击下一步移动到页面流中的下一页。将显示“导出/导入”的“添加引用”页。

如何选择要导入的建模对象实例

按照以下步骤选择要导入的建模对象实例：

1. 完成“如何选择数据传输类型”过程以访问“实例选择”页。
2. 单击文件上传图标，然后使用显示的弹出窗口来选择要导入的 XML 文件。应用程序会在“对象类型名称”表格中显示所选文件中包含的建模对象实例，在“选择要包含的内容”字段中显示所选文件的名称。
Note: 单击“对象类型名称”表格中的展开图标可查看为该建模对象指派的所有实例。
3. 展开对象并选择不想导入的所有实例，然后单击“对象类型名称”表格上的删除所选行按钮。应用程序会从该表格中移除这些实例。
4. 单击下一步移动到页面流中的下一页。将显示“导出/导入”的“执行控制台”页。
