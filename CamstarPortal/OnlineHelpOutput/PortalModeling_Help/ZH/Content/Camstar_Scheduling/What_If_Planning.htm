“假设”计划

Opcenter Execution Core Scheduling 提供“假设”计划功能以分析将来的情况，或根据计划产品组合提供“假设”计划方案。在这些情况下仍需要创建部分或全部制造订单。应用程序会将产品及其关联的 ERP 路线从 Opcenter EX MDD 或 Opcenter EX CR 发送到 Preactor。通过此数据，用户可以在 Preactor 中创建假想订单以分析“假设”计划方案。

必须启用 Opcenter Execution Core Scheduling 才能使用“假设”计划。必须定义 ERP 路线。还必须定义产品并将 ERP 路线指派给产品。有关信息，请参考 “在工厂页上启用 Opcenter Execution Core Scheduling”。有关定义 ERP 路线的信息，请参考“定义用于 Opcenter Execution Core Scheduling 的 ERP 路线”。有关定义产品的信息，请参考“定义用于 Opcenter Execution Core Scheduling 的产品”。
