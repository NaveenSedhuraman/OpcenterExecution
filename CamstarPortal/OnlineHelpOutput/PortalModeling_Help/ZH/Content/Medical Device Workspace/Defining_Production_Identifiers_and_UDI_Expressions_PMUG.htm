定义生产标识和 UDI 表达式

在从“开始”页或“订单调度”页启动容器时，可以通过手动输入信息，直接在容器上定义生产标识和唯一设备标识。启动容器后可以使用“容器维护”页更新这些字段。有关将生产标识和 UDI 直接指派给容器的信息，请参考《Opcenter Execution Medical Device and Diagnostics Shop Floor 用户指南》或《Opcenter Execution Core Shop Floor 用户指南》。

可以使用 UDI 表达式对象为 UDI 和 PI 创建表达式。当产品上配置有这些表达式时，这些表达式用于在运行时对容器自动创建和指派 UDI 和 PI。这些表达式可以使用可从“移动”或“移入”事务引用的任何值。必须为实施中使用的每个 PI 和 UDI 表达式创建不同的表达式。

例如，在“变量表达式”字段中使用此变量表达式可以创建和指派包含设备标识、制造日期和容器名称的 UDI：

 

((Container.DeviceIdentifier!=Null? "(01)" + Container.DeviceIdentifier.Name:Null) + (DateConvert

(Container.ManufacturingDate,"yyMMdd")!=Null? "(11)" + DateConvert

(Container.ManufacturingDate,"yyMMdd"):Null) + (Container.Name!=Null? "(21)" + Container.Name:Null))

如何定义 UDI 表达式

按照以下步骤定义 UDI 表达式：

1. 打开 UDI 表达式页。随即“建模”页中将显示“UDI 表达式”页。
2. 单击新建。 显示用于定义新实例的空白字段。
3. 在UDI 表达式字段中输入 UDI 表达式的名称。
4. 在变量表达式字段中输入变量表达式。
5. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
6. 单击保存。 应用程序保存建模对象并显示成功消息。
