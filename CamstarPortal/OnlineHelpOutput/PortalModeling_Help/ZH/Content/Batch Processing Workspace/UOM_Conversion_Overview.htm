度量单位转换概述

Note: 仅当安装 Batch Processing 时下述功能才可用。有关信息，请参考《Opcenter Execution Medical Device and Diagnostics Batch Processing 用户指南》或《Opcenter Execution Core Batch Processing 用户指南》。

为物料列表中的组件指定的度量单位可能与为批次指定的度量单位不同。利用 Batch Processing 功能，可以通过提供转换系数，将分发数量从组件的度量单位转换为批次的度量单位，从而确保分发正确的数量。可以在“产品”、“产品系列”和“度量单位”建模对象上使用“度量单位转换”表格来定义这些转换系数。有关在“产品”建模对象上定义转换系数的信息，请参考“定义产品的 UOM 转换信息”。有关在“产品系列”建模对象上定义转换系数的信息，请参考“定义产品系列的 UOM 转换信息”。有关在“度量单位”建模对象上定义转换系数的信息，请参考“在定义度量单位的转换信息时”。

在产品上定义的转换系数优先于在产品系列上定义的转换系数。在产品系列上定义的转换系数优先于在度量单位上定义的转换系数。
