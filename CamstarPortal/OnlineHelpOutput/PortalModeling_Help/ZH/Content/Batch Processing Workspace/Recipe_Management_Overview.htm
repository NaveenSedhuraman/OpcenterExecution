配方管理概述

Note: 仅当安装 Batch Processing 时下述功能才可用。有关信息，请参考《Opcenter Execution Medical Device and Diagnostics Batch Processing 用户指南》或《Opcenter Execution Core Batch Processing 用户指南》。

有时，配方需要调整物料清单中定义的数量。调整的原因包括先前添加的组件的确切数量变化、原物料的效力差异或不同的批次大小。可以使用批次处理功能调整物料列表数量，通过提供表达式来计算配方处理过程中的目标数量和所需的最小和最大数量。可以在产品或配方的发出权重任务项上定义这些表达式。

从组件产品上所定义的表达式中得出的数量优先于物料列表上定义的数量。从称重分发任务上定义的表达式得出的数量优先于从产品上定义的表达式得出的数量。有关在产品上定义数量表达式的信息，请参考“定义表达式以调整产品上的组件数量”。有关在称重分发任务上定义数量表达式的信息，请参考“在定义称重分发任务时”。

您还可以使用批处理功能定义在添加到主批次之前必须处理的中间批次。可以在“任务列表”建模对象上定义表达式，以计算中间批次的数量。有关定义中间批次的信息，请参考“在定义中间批次时”。
