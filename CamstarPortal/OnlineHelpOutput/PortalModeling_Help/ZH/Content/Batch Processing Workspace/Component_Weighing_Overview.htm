组件称重概述

Note: 仅当安装 Batch Processing 时下述功能才可用。有关信息，请参考《Opcenter Execution Medical Device and Diagnostics Batch Processing 用户指南》或《Opcenter Execution Core Batch Processing 用户指南》。

处理批次的配置方式说明是使用现有的电子流程功能创建的。配置方式步骤可能包括生产批次所需的任何其他任务类型（除称重任务外）。您可以使用“任务列表”建模对象来创建称重发布任务，以定义组件产品的称重及其到批次容器的分发。通过称重发布任务，可定义在处理过程中要对组件进行称重的点。称重发布任务还可定义组件的称重规则。有关创建称重分发任务的信息，请参考“在定义称重分发任务时”。
