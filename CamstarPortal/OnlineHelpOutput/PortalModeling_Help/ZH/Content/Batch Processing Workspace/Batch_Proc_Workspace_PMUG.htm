Batch Processing

Note: 仅当安装 Batch Processing 时下述功能才可用。有关信息，请参考《Opcenter Execution Medical Device and Diagnostics Batch Processing 用户指南》或《Opcenter Execution Core Batch Processing 用户指南》。

Batch Processing 提供了一种机制，以交付与以下内容关联的功能：

• 测重器管理
• 配方管理
• UOM 转换
• 组件称重

制造商可以使用这些功能来控制和追踪用作设备组件的小批次化学制品的生产。示例包括血糖试纸的涂层，以及透皮贴剂的药物配方。

您必须定义物料列表以使用批处理。在定义物料列表时，您可以定义被称重的每个组件的所需数量及最小和最大数量。可使用“物料清单”、“ERP 物料清单”或“制造订单”建模对象上的“物料列表”表格为成品定义物料列表。有关信息，请参考“定义物料清单上允许的组件数量”、“定义 ERP 物料清单上允许的组件数量”和“定义制造订单上允许的组件数量”。
