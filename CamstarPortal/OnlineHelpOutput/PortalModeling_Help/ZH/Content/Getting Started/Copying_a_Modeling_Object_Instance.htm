复制建模对象实例

单击选定建模实例的工具栏上的“复制”和“复制修订版本”按钮时，应用程序显示“复制”或“复制修订版本”弹出窗口。通过单击“复制”或“复制修订版本”弹出窗口或按键盘上的 Enter 键，可以保存新实例或实例修订版本并关闭“复制”或“复制修订版本”弹出窗口。应用程序显示成功消息，指出新建模对象实例已创建。

执行“复制”将创建所选实例的副本，其中包含除实例名称（以及已修订对象的修订版本名称）以外的所有原始信息。应用程序要求在保存已命名数据对象实例之前输入新实例名称，在保存已修订对象之前输入新实例名称和修订值。

执行“复制修订版本”将创建已修订对象的副本，此对象除修订值以外的所有信息均来自原修订。应用程序要求在保存所选实例的新修订版本之前输入新的修订版本值。

Note: 只有已修订对象才会显示“复制修订版本”按钮。

如何复制已修订对象实例

按照以下步骤复制已修订对象实例：

1. 从建模菜单中选择建模。 将显示“建模”页。
2. 单击已修订对象。随即“建模”页中将显示已修订对象的页。
3. 选择已修订对象的现有实例。将显示该实例。
4. 单击复制。将显示“复制”弹出窗口。
5. 在目标实例名称字段中输入“建模”实例的名称。
6. 在新建修订版本名称字段中输入修订版本。
7. 单击复制或按键盘上的 Enter 键。
8. 单击保存。 应用程序保存建模对象并显示成功消息。

如何复制已修订对象实例的修订版本

按照以下步骤复制已修订对象实例的修订版本：

1. 从建模菜单中选择建模。 将显示“建模”页。
2. 单击已修订对象。随即“建模”页中将显示已修订对象的页。
3. 选择已修订对象的现有实例。将显示该实例。
4. 单击复制修订版本。将显示“复制修订版本”弹出窗口。
5. 在新建修订版本名称字段中输入修订版本。
6. 单击复制或按键盘上的 Enter 键。
7. 单击保存。 应用程序保存建模对象并显示成功消息。

如何复制已命名数据对象建模实例

按照以下步骤复制已命名数据对象的“建模”实例：

1. 从建模菜单中选择建模。 将显示“建模”页。
2. 单击已命名数据对象。随即“建模”页中将显示已命名数据对象的页。
3. 选择已命名数据对象的现有实例。将显示该实例。
4. 单击复制。将显示“复制”弹出窗口。
5. 在目标实例名称字段中输入“建模”实例的名称。
6. 单击复制或按键盘上的 Enter 键。
7. 单击保存。 应用程序保存建模对象并显示成功消息。
