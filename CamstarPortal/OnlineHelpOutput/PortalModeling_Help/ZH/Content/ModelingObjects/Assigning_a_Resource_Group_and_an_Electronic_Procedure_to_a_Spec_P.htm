将资源组和电子流程指派给规范

规范定义在工步执行的活动，由工作流程内的工步引用。许多工作流程工步可以使用相同的规范。

Note: 规范引用许多其他建模组件，其中包括“作业”和“设置”。工艺也包括详细的排程以及处理参数信息。

或者，可以将资源组和电子流程指派给规范。将资源组指派给规范可确定将容器移出规范时可供选择的资源。将电子流程指派给规范可确定在将容器移出规范之前必须完成的任务。

一个规范一次只能引用一个资源组和一个电子流程。但是，电子流程可以引用一系列包括多个任务的任务列表。

有关规范的信息，请参考“定义工艺”。

如何将资源组指派给规范

按照以下步骤将“资源组”指派给规范：

1. 打开规范页。随即“建模”页中将显示“规范”页。
2. 执行“如何定义规范”步骤。

或

选择现有规范实例。“规范”页将显示所选规范的信息。

3. 从资源组字段中选择工作站或工作单元组。
4. 单击保存。 应用程序显示成功消息，指出建模对象已更新。

如何将电子流程指派给规范

按照以下步骤将电子流程指派给规范：

1. 打开规范页。随即“建模”页中将显示“规范”页。
2. 执行“如何定义规范”步骤。

或

选择一个现有工艺。“规范”页将显示所选规范的信息。

3. 从电子流程字段中，选择电子流程或电子流程的特定修订版本。
4. 单击保存。 应用程序显示成功消息，指出建模对象已更新。
