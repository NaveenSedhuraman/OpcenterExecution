定义 SPC 图表组

SPC 图表组用于定义 SPC 图表、SPC 图表组或这两者的集合，具体取决于应显示的是单个图表还是一组图表。它们可以同时进行处理。此对象是已命名对象组。

如果使用传统的数据收集方法，则可以在“数据收集定义”或“用户数据收集定义”页上指定 SPC 图表组。对于基于任务的数据收集，可以在使用“任务列表”建模对象定义任务时指定数据收集定义。

在定义 SPC 图表组时

定义数据收集和用户数据收集时，SPC 图表组是可选字段。SPC 图表组可以附加到数据收集定义，而数据收集定义则可以附加到规范。

SPC 图表组包含可选的建模定义和 SPC 图表。

使用“解决条目”按钮可以显示在此组和所有嵌套组的条目列表中指定的所有值的列表。

有关设置和配置 SPC 的信息，请参考《Opcenter Execution Core Statistical Process Control 参考指南》。

如何定义 SPC 图表组

按照以下步骤定义 SPC 图表组：

1. 打开 SPC 图表组。随即“建模”页中将显示“SPC 图表组”页。
2. 单击新建。 显示用于定义新实例的空白字段。
3. 在 SPC 图表组字段中输入新图表组的名称。
4. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
5. 单击保存。 应用程序保存建模对象并显示成功消息。

如何将条目添加到 SPC 图表组

按照以下步骤将条目添加到 SPC 图表组：

1. 执行“如何定义 SPC 图表组”过程。

或

选择现有的 SPC 图表组实例。

2. 单击“条目”表格上的添加新行。 将显示新行。
3. 在新行中单击以显示 SPC 图表列表，然后选择一个图表以添加到组中。应用程序会将选定条目添加到表格中。
4. 重复步骤 2-3 以将另一个条目添加到表格中。
5. 单击保存。 应用程序显示成功消息，指出建模对象已更新。

如何将组添加到 SPC 图表组

按照以下步骤将条目添加到 SPC 图表组：

1. 执行“如何定义 SPC 图表组”过程。

或

选择现有的 SPC 图表组实例。

2. 单击“组”表格中的添加新行。 将显示新行。
3. 在新行中单击以显示先前定义的组的列表，然后选择一个（作为子组）添加到组中。应用程序将选定的组添加到表格中。
4. 重复步骤 2–3 以将另一个组添加到表格中。
5. 单击保存。 应用程序显示成功消息，指出建模对象已更新。
