统计流程控制

质量数据是生产信息的宝贵来源。Opcenter EX MDD 和 Opcenter EX CR 中的规范可以确保未能达到最低准则的在制品 (WIP) 单元不能装运，但这仅仅是开始。制造商还可以在不牺牲生产率的情况下生产高质量的产品。Opcenter Execution Core Statistical Process Control (SPC) 是一个强大且灵活的模块，支持制造商在问题导致产量减少或生产能力降低之前就解决它们。本节介绍用于创建 SPC 对象的建模对象。

有关信息，请参考《Opcenter Execution Core Statistical Process Control 参考指南》。
