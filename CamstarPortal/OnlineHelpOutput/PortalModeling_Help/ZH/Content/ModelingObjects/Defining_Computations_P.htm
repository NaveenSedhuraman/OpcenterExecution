定义计算

通过计算建模对象，可以创建和维护用于电子流程任务的过程计算定义。定义这些计算时，可以使用建模中定义的用户常量。

在定义计算时

在定义计算表达式时使用 Opcenter EX MDD 或 Opcenter EX CR 统一表达式语法。表达式最多可包含 4000 个字符，可能包括：

• 变量；例如，(?InQty)
• 用户定义的常量；例如，Constant::Pi
Note: 有关统一表达式语法的信息，请参考《Opcenter Execution Medical Device and Diagnostics Designer 用户指南》或《Opcenter Execution Core Designer 用户指南》。

如果计算表达式包含变量，则当按 Tab 键使焦点移出“计算表达式”字段中或保存计算时，应用程序将变量添加到计算变量表中。每个变量的默认数据类型为“整数”。表达式中的变量必须与表中的变量匹配。如果在表达式中删除或修改变量，则必须手动删除或修改表中的变量。同样，如果在表中修改或删除变量，则必须手动更新表达式。

在对容器执行计算时，可以指示应用程序将计算结果映射到容器。应用程序将计算结果存储为用户定义的属性。可以在“容器属性维护”页上查看、编辑和删除用户定义的属性。有关“容器属性维护”页的信息，请参考《Opcenter Execution Medical Device and Diagnostics Shop Floor 用户指南》或《Opcenter Execution Core Shop Floor 用户指南》。

在定义限制时

定义计算时，可以为计算结果设置上限和/或下限。可以使用表达式来定义上限、下限或全部两个限制。使用表达式时，在运行时确定限制。

Note: 输入表达式时，不会验证语法是否正确。有关 Opcenter EX MDD 或 Opcenter EX CR 统一表达式语法的信息，请参考《Opcenter Execution Medical Device and Diagnostics Designer 用户指南》或《Opcenter Execution Core Designer 用户指南》。

即使用户输入的值超出限制，应用程序也始终接受用户输入的任何值。“显示限制”和“允许替代限制”复选框确定是否在任务运行时向用户显示限制、当值超出限制时向用户显示的颜色，以及电子流程计算任务是完成还是失败。

如何定义计算

按照以下步骤定义计算：

1. 打开计算页。随即“建模”页中将显示“计算”页。
2. 单击新建。 显示用于定义新实例的空白字段。
3. 在计算字段中输入计算的名称。
4. 在结果名称字段中输入计算值的名称。
5. 从列表中选择结果数据类型。
6. 在计算表达式字段中输入用于计算结果的公式或表达式。
7. 单击保存或按 Tab 键使焦点移出计算表达式字段。表达式中的任何变量都添加到“计算规范参数”表格，并被指派了默认数据类型“整数”。
8. 在“计算机参数规范”表格中选择一行以启用“数据类型”字段。
9. 如果需要，为每个变量选择其他数据类型。
10. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
11. 单击保存。 应用程序保存建模对象并显示成功消息。
