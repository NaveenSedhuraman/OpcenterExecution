查看建模对象实例引用

可以查看所选实例引用的所有建模对象实例。还可以查看引用在“建模”页上选择的实例的建模对象实例。

Note: 即使有“过滤器标记”建模对象实例连接到所选实例，应用程序也不会显示这些实例。“过滤器标记”是一个逗号分隔的列表，不会作为引用附加到对象。

在查看引用的实例时

“层次结构”视图显示所选建模对象实例及其引用的所有实例。单击建模对象实例页上的“层次结构”按钮将显示只读的“层次结构”弹出窗口。“层次结构”弹出窗口显示一个缩进的树视图，其中包含所选对象实例引用的实例。所选实例以加粗黑色字体显示。所引用的实例以普通黑色字体显示。

“层次结构”视图还包括间接引用。例如，所选产品实例也引用了某工作流程，应用程序会显示此工作流程所引用的规范。

Note: 当尝试查看包含大量记录的建模审核追踪时，应用程序可能会超时。可在 XMLClientConfig 文件中的超时设置中增加毫秒数。默认情况下，XMLClientConfig.xml 文件位于 C:\Program Files (x86)\Camstar\InSite Utility。

在查看引用实例时

“何处使用”视图显示直接引用所选实例的建模对象实例。单击建模对象实例页上的“何处使用”按钮将显示只读的“何处使用”弹出窗口。“何处使用”弹出窗口显示所有建模对象以及引用所选实例的建模对象实例。

如果实例引用的实例已引用了所选实例，“何处使用”弹出窗口则不显示此实例。例如，工作流程已引用了所选的规范实例，应用程序不会显示引用这些工作流程的产品。应用程序只显示引用所选规范实例的物料清单和工作流程。

如何查看由选定实例引用的实例

按照以下步骤查看引用所选实例的实例：

1. 选择一个建模对象。
2. 选择一个实例。
3. 单击层次结构。将显示“层次结构”弹出窗口。
4. 单击关闭，退出弹出窗口。

如何查看引用选定实例的实例

按照以下步骤查看引用所选实例的实例：

1. 从“建模”菜单中选择“建模”。 将显示“建模”页。
2. 选择一个建模对象。随即“建模”页中将显示该对象的页面。
3. 选择一个现有实例。 将显示实例和实例信息。
4. 单击何处使用。将显示“何处使用”弹出窗口。
5. 单击关闭，退出弹出窗口。
