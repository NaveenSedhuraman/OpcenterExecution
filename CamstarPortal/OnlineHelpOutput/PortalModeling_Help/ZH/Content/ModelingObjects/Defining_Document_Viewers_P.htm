定义文档查看器

文档查看器指定用于查看在 Opcenter EX MDD 或 Opcenter EX CR 中定义的文档的应用程序。文档查看器定义包含应用程序的位置以及启动程序以显示文档的命令行字符串。例如，可以定义使用 Word、Excel 或 Portable Document Format (PDF) 的文档查看器，以查看各种文档。

Note: Opcenter EX MDD 或 Opcenter EX CR 的默认实施不使用此对象；但是，该对象可用于您的自定义实施。

如何定义文档查看器

按照以下步骤定义文档查看器：

1. 打开文档查看器页。随即“建模”页中将显示“文档查看器”页。
2. 单击新建。 显示用于定义新实例的空白字段。
3. 在文档查看器字段中输入文档查看器的名称。
4. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
5. 单击保存。 应用程序保存建模对象并显示成功消息。
