定义文档

文档实例包含查看外部文档所需的信息。每个定义都包含文档的文件名和修订版本。

可以浏览并上传任何类型的文件，还可以将该文件与文档对象相关联以及为该文件输入任何版本号。如果附加到对象实例的文件已保存到数据库中，则该文件的名称会显示在“存储的文件名”字段中。

还可以将文件下载并保存到最初作为对象实例的一部分上传的数据库。

大多数建模实体都包含对文档的引用。在这种情况下，文档用于提供扩展的（非结构化）信息，例如产品图样、扫描图像、现有的相关制造规范或操作流程。外部文档的每个文档实例 (CDO) 都包含一个标识，例如文件名。

Note: 配方 是文档的子类。有关配方建模对象的信息，请参考“定义配方”。

必要条件

计算机或服务器上必须具有适用于要尝试打开的文档类型的软件。

HTTP 浏览仅限于 Portal 服务器的虚拟目录，因此文档必须位于诸如 Camstar Portal 文件夹或其子文件夹之一等目录中。（不能使用诸如 C:\My Documents 等路径来定义文档的位置。）

用于 URL 浏览的默认路径基于在登录到会话时使用的 URL。因此，可能需要编辑文档的 URL。例如，如果使用 https://localhost/CamstarPortal/default.htm#/login 登录，则在“查看模式”字段中选择一个值后，将无法在“查找文件”字段中看到完全限定的域名。应该使用完全限定的域名登录，或者编辑 URL 以使用完全限定的域名，从而确保其他域中的用户可以访问该文档。

默认浏览器安全设置禁止打开不安全网站上的文档。必须更改设置才能打开不安全的内容。

Note: 在浏览器中更改此设置时，您会从应用程序中注销。

在定义文档时

定义文档时，以下规则适用：

• 每个文档实例在“查找文件”字段中都包含一个文件位置，定义外部 Web 地址的超链接时除外。
• 在“查看模式”字段中选择的值决定应用程序是将文件上传到数据库，还是上传文件的链接或外部 Web 地址的链接。选择“本地文件”并单击“文件上传”图标，以导航至指定的文件并将其上传到数据库。
• “存储的文件名”字段是一个只读字段，单击“保存”后，此字段将显示已上传的本地文件的名称。链接到文件（“查看模式”为“HTTP 文件”）时，“存储的文件名”字段保持为空。
• 文件的下载时间/上传时间因文件大小而异。
• 使用附件文档功能添加为可重用文档的文档在“文档”页上可见。

“文档”是：

• “文档集”定义中的必需字段。
• “规范”（配方文件中）以及“WIP 消息”定义中的可选字段。
Note: 如果文档定义与容器定义中的字段关联，则该文档定义可以具有相关的 WIP 消息。有关信息，请参考《Opcenter Execution Medical Device and Diagnostics Shop Floor 用户指南》或《Opcenter Execution Core Shop Floor 用户指南》。

在访问外部存储库中的文件时

可以从门户访问存储在外部存储库中的文档。如果外部存储库要求用户名和密码与用于访问门户的用户名和密码不同，则可在员工建模对象上设置用户名和密码。必须将“身份验证类型”设为“基本”才能使用此功能。

• 查看模式决定应用程序是将文件上传到数据库，还是上传文件的链接或外部 Web 地址的链接。选择“HTTP 文件”可链接到外部存储库中的指定文件。
• 仅当“查看模式”设为“HTTP 文件”时，应用程序才会显示“身份验证类型”选择列表。
• 必须在文档建模对象上将“身份验证类型”设为“基本”，才能使用在员工建模对象上设置的用户名字段和密码字段。如果选择“基本”，应用程序会将在员工建模对象上输入的用户名和密码传递至外部文档存储库以用于身份验证。
• 在员工建模对象上输入的用户名和密码必须与访问存储库所需的用户名和密码匹配。
• 当“身份验证类型”设为“基本”时，如果未在员工建模对象上指定用户名和密码，则应用程序会将用户的凭据传递至存储库。
链接到电子工作指示文件

可以链接到 EWI 文件，以使具有 Teamcenter 访问权限的用户可以在单独的弹出窗口中查看 EWI 信息。

• 查看模式决定应用程序是将文件上传到数据库，还是上传文件的链接或外部 Web 地址的链接。选择“URL”可链接到 EWI 文件。
• 仅当“查看模式”设为“URL”时，应用程序才会显示“URL”字段。为“查看模式”选择“URL”时，还会隐藏“身份验证类型”字段、“查找文件”字段、“存储的文件名”字段和“文件版本”字段。
• 可以在单独的弹出窗口中查看 EWI 文件。显示 EWI 文件之前，弹出窗口会提示输入 Teamcenter 用户名和密码。Teamcenter 会话将保持打开状态，直至浏览器关闭。
• 定义 EWI 文件的链接时，“URL”字段不接受包含任何用于配置工作指示（例如产品视图、最终装配、文档查看器设置或任何登录凭据）的参数的 URL。

如何定义文档

按照以下步骤定义文档：

1. 打开文档页。随即“建模”页中将显示“文档”页。
2. 单击新建。 显示用于定义新实例的空白字段。
3. 在文档字段中输入文档的名称。
4. 在修订版本字段中输入此文档的修订版本。
5. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
6. 单击保存。 应用程序将保存建模对象、启用“查看文档”按钮并显示成功消息。

如何将文件添加到文档定义

按照以下步骤将文件添加到文档定义：

1. 执行“如何定义文档”过程。
2. 要将文件上传到数据库，还是要上传文件的超链接或者上传外部 Web 地址或 EWI 文件的超链接？

如果您想完成以下操作 ...

操作步骤 ...

将文件上传到数据库
a. 从查看模式字段中选择本地文件（如果未选择）。将显示“浏览”按钮。
b. 单击“查找文件”字段旁边的文件上传图标。将显示“文件上传”弹出窗口。
c. 导航至文件并加以选择，然后单击打开。“文件上传”弹出窗口将关闭，并且文件名会显示在“查找文件”字段中。
d. 单击保存。 应用程序将显示成功消息，并且文件名会显示在“存储的文件名”字段中。

文件的超链接
a. 从查看模式字段中选择 HTTP 文件。“查找文件”字段中会显示一个箭头。
b. 从身份验证类型字段中选择基本或无。
c. 从查找文件字段中选择文件。文件的超链接会显示在“查找文件”字段中。
d. 单击保存。 应用程序显示成功消息，指出建模对象已更新。

外部 Web 地址或 EWI 文件的超链接。
a. 从查看模式字段中选择 URL。将显示“URL”字段。
b. 在“URL”字段中输入外部 Web 地址或 EWI 文件的超链接。
c. 单击保存。 应用程序显示成功消息，指出建模对象已更新。
