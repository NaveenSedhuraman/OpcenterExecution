定义用户界面首选项

通过用户界面 (UI) 首选项，可以在“制造审核追踪”页的“状态”选项卡中指定要为容器、物料箱或事件记录显示的信息。还可用于在“资源审核追踪”页的“状态”选项卡中指定要为资源记录显示的信息。用户界面首选项还可确定应用程序在“状态”选项卡上显示信息的顺序。

在定义用户界面首选项时

不可在对象名中使用无效字符。例如，不可使用撇号。有关有效字符的信息，请参考《Opcenter Execution Medical Device and Diagnostics Designer 用户指南》或《Opcenter Execution Core Designer 用户指南》。

在指定要显示的字段时

对字段名称使用这些格式：

• Container.FieldName（为容器和物料箱记录指定要显示的字段时）
• QualityObject.FieldName（为事件记录指定要显示的字段时）
• Resource.FieldName（为资源记录指定要显示的字段时）

将用户界面首选项与记录类型相关联

用户界面首选项是以下建模对象上的可选字段：

• 容器级别
• 组织
• 资源类型
• 资源系列

必须先将用户界面首选项与记录类型相关联，才能在“制造审核追踪”或“资源审核追踪”页的“状态”选项卡中显示任意信息。可在“容器级别”建模对象的“用户界面显示配置”字段中关联容器和物料箱记录的用户界面首选项。可以将不同的用户界面首选项与各个定义的容器级别相关联。有关容器级别的信息，请参考“定义容器级别”。

在组织建模对象上的用户界面首选项映射中，关联事件记录的用户界面首选项。可以为各个事件记录类型关联不同的用户界面首选项。有关关联用户界面首选项的信息，请参考“定义组织”。

此外，必须将组织设为记录事件的员工的主要组织。仅当在组织上指定了用户界面首选项且该组织已设为事件记录员工的主要组织时，才会显示事件记录的“状态”选项卡。有关指定主要组织的信息，请参考“定义员工”。

在“资源类型”或“资源系列”建模对象（或两者）的用户界面首选项字段中，关联资源记录的用户界面首选项。“资源类型”和“资源系列”是资源建模对象上的可选字段，两者都可以指定用户界面首选项。应用程序首先使用“资源类型”指定的用户界面首选项。如果未通过“资源类型”定义，则使用通过“资源系列”定义的用户界面首选项。可以将不同的用户界面首选项与各个已定义的资源相关联。有关信息，请参考“定义资源类型”和“定义资源系列”。

如何定义用户界面首选项

按照以下步骤定义用户界面首选项：

1. 打开用户界面首选项页。随即“建模”页中将显示“用户界面首选项”页。
2. 单击新建。 显示用于定义新实例的空白字段。
3. 在用户界面首选项字段中输入用户界面首选项的名称。
4. 单击“要显示的字段”表格中的添加新行。 将显示新行。
5. 在用户界面首选项字段定义名称列中输入要显示的字段名称。
6. 重复步骤 4-5 以指定其他字段。
7. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
8. 单击保存。 应用程序保存建模对象并显示成功消息。

如何指定用于容器和物料箱记录的用户界面首选项

按照以下步骤指定用于容器和物料箱记录的用户界面首选项：

1. 执行“如何定义容器级别”过程。

或

选择现有容器级别实例。

2. 在用户界面显示配置字段中输入或选择用户界面首选项。
3. 对每个容器级别重复此过程。
4. 单击保存。 应用程序将保存更改，并在“制造审核追踪”页的“状态”选项卡上显示指定容器或物料箱记录的信息时使用此用户界面首选项。

如何指定用于事件记录的用户界面首选项

按照以下步骤指定用于事件记录的用户界面首选项：

1. 执行“如何定义组织”过程。

或

选择现有组织实例。

2. 在“用户界面首选项映射”表格中单击添加新行。 将显示新行。
3. 输入或选择适当的记录类型。
4. 输入或选择要与记录类型关联的用户界面首选项。
5. 单击保存。 应用程序将保存更改，并在“制造审核追踪”页的“状态”选项卡上显示指定质量记录的信息时使用此用户界面首选项。
