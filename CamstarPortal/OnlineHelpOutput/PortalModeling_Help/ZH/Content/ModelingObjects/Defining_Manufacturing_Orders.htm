定义制造订单

制造订单是制造产品的请求。制造订单通常源于完成一笔销售订单或库存需要，并且常是 MES 和 ERP 系统之间的链接。

制造订单可包含一系列产品容器。容器与制造订单的可选关系是在容器的数据记录中建立并维护。制造订单利用这些信息向 ERP 系统提供 WIP 状态及 WIP 更新。

此外，可以为制造订单指派自定义属性和动态属性，应用程序会在启动容器时指派这些用户定义的属性。在容器本身中存储信息后，您和其他用户不必在容器的历史中搜索信息。

了解用户定义的属性

有关用户定义的属性的信息，请参考“了解用户定义的属性”。

有关在容器启动时指派用户定义的属性的信息，请参考“当容器启动时指派用户定义的属性”。

在定义制造订单时

制造订单包含以下可选建模定义：

• ERP 物料清单
• 文档
• 订单状态
• 订单类型
• 优先级
• 产品
• 路线工步
• 容器编号规则
• UOM

在大部分“容器”事务中，“制造订单”是可选字段。

在“优先级”和“计划开始日期”字段中输入的内容决定“订单调度”事务中制造订单的排序顺序。

在附加文档时

可以在“制造订单”建模页中将文档附加到制造订单，然后从制造订单拆离文档或查看文档。必须先保存制造订单实例，然后才能将文档附加到该制造订单。

附件为可选；但是，应用程序要求在附加文件时先完成某些字段再启动容器。

附件的最大大小为 30 MB。

Note: 可以将“制造订单”建模页上提供的文档附件功能添加到任何建模对象。可以通过 Portal Studio 执行此操作，方法是将文档附件 Web 部件包含到建模页上。有关信息，请参考《Opcenter Execution Medical Device and Diagnostics Portal Studio 用户指南》或《Opcenter Execution Core Portal Studio 用户指南》。

在查看附件时

如果是远程访问文件，则在执行其他步骤之前，浏览器可能会阻止下载文件。例如，在单击 Internet Explorer 信息栏中的“下载文件”之前，Internet Explorer 可能会阻止下载文件。

附加现有文档时，必须将文档文件附加到所选文档实例，否则单击弹出窗口上的“查看文档”时会收到错误消息。

还必须安装了正确的软件才能查看文件。例如，如果文件为 Excel 电子表格，则必须安装 Microsoft Excel 才能查看该文件。

添加需要修订版本的文档时，应用程序会自动创建新的文档实例。

定义制造订单上允许的组件数量
Note: 仅当安装 Batch Processing 时下述功能才可用。有关信息，请参考 "Batch Processing"。

使用“物料列表”表格，可以定义所需的组件数量以及最小和最大允许数量。也可以使用物料清单或 ERP 物料清单上的“物料列表”表格定义组件数量。为制造订单定义的物料列表优先于为物料清单或 ERP 物料清单定义的物料列表。有关信息，请参考“定义物料清单上允许的组件数量”和“定义 ERP 物料清单上允许的组件数量”。

使用“最小允许量”和“最大允许量”字段来定义允许的组件数量。在单击“添加新行”按钮向“物料列表”表格中添加物料时，这些字段将显示在弹出窗口中。

定义用于 Opcenter Execution Core Scheduling 的制造订单
Note: 仅当安装有 Opcenter Execution Core Scheduling 时，下述功能才可用。有关信息，请参考 "Opcenter Execution Core Scheduling"。

Opcenter Execution Core Scheduling 使用制造订单实例中提供的信息来计划订单。当使用 Opcenter Execution Core Scheduling 时指派给制造订单的 ERP 路线必须与下列字段之一匹配：

• 指派给产品工作流程的 ERP 路线
• 指派给产品系列工作流程的 ERP 路线

使用“制造订单”页上的“订单状态”字段将订单状态指派给制造订单。只能在 Opcenter Execution Core Scheduling 中计划打开的制造订单。

发布用于 Opcenter Execution Core Scheduling 的制造订单
Note: 仅当安装有 Opcenter Execution Core Scheduling 时，下述功能才可用。有关信息，请参考 "Opcenter Execution Core Scheduling"。

发布制造订单时，会触发接口将订单数据从 Opcenter EX MDD 或 Opcenter EX CR 发送到 Preactor。当保存新订单时会发布制造订单，或者当满足以下条件时会更新现有订单：

• 制造订单具有产品、数量和订单状态。
• 制造订单的产品或产品系列具有已指派的工作流程。
• 订单状态具有“打开”状态。

制造订单在发布时存放在 Preactor 暂存表中。可以通过修改并保存制造订单，对暂存订单进行更新。可以通过将订单状态更改为关闭并保存制造订单，将订单从暂存表中移除。订单在发布时被不断添加到暂存表中。将订单导入 Preactor 会将 Preactor 中的当前订单列表替换为暂存表中的当前订单。

分发控制

分发控制类型的列表。选择适当的分发控制类型：

• 仅显示：操作员能看见组件信息，但不记录与实际组件使用相关的任何信息。
• 分发容器（批次）：在 Opcenter EX MDD 或 Opcenter EX CR 中，组件物料在批量级别作为容器进行追踪。一个组件批次可以提供所需的部分或全部数量，最大达到组件批次的当前数量。
• 分发容器（序列）：在 Opcenter EX MDD 或 Opcenter EX CR 中，组件物料在序列级别作为容器进行追踪。每个必需的数量单位都代表一个批次。

如何定义制造订单

按照以下步骤定义制造订单：

1. 打开制造订单页。“制造订单”页显示在“建模”页中。
2. 单击新建。 显示用于定义新实例的空白字段。
3. 在制造订单字段中输入此任务列表的名称。
4. 要添加物料列表，请参考“如何将物料列表添加到制造订单”。
5. 要添加用户定义的属性，请参考“如何将用户定义的属性添加到容器级别”。
6. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
7. 单击保存。 应用程序保存建模对象并显示成功消息。
8. 要附加文档，请参考以下文档附件过程之一：
• “如何将不会重用的新文档附加到制造订单”
• “如何将会重用的新文档附加到制造订单”
• “如何将现有文档附加到制造订单”
9. 要拆离文档，请参考“如何从制造订单拆离文档”。
10. 要查看附加的文档，请参考“如何查看附加到制造订单的文档”。

如何将物料列表添加到制造订单

按照以下步骤将物料列表添加到制造订单：

1. 执行“如何定义制造订单”中所述的过程。

或

选择现有制造订单实例。

2. 展开物料部分。
3. 单击“物料列表”表格上的添加新行。将显示“添加”弹出窗口。
4. 从列表中选择产品。
5. 从列表中选择分发控制。
6. 在要求的数量字段中输入值。
7. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
8. 单击确定。 “添加”弹出窗口将关闭，并且物料列表项会显示在“制造订单”页上的“物料列表”表格中。
9. 单击保存。 应用程序显示成功消息，指出建模对象已更新。

如何将用户定义的属性添加到制造订单

按照以下步骤将用户定义的属性添加到制造订单：

1. 执行“如何定义制造订单”中所述的过程。

或

选择现有制造订单实例。

2. 展开用户属性部分。
3. 单击“用户属性列表”表格上的添加新行。 将显示新行。
4. 在属性名称字段中输入属性的名称。
5. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
6. 选中是表达式复选框（仅当为“类型”选择了字符串时）。
7. 重复步骤 3-6 添加其他属性。
8. 单击保存。 应用程序显示成功消息，指出建模对象已更新。

如何将不会重用的新文档附加到制造订单

按照以下步骤将不会重用的新文档附加到制造订单：

1. 执行“如何定义制造订单”中所述的过程。

或

选择现有制造订单实例。

2. 如果需要，展开附加的文档部分。
3. 单击“附加的文档详细信息”表格上的附加。
4. 选择附加不再重新使用的新文档（不需要修订版本）。将显示“文档路径”、“文档名称”和“文档描述”字段。
5. 单击文件上传图标以选择文件。
6. 在文档名称字段中输入文档名称。
7. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
8. 单击附加。应用程序将显示一条关于已成功附加文档的消息。

如何将会重用的新文档附加到制造订单

按照以下步骤将会重用的新文档附加到制造订单：

1. 执行“如何定义制造订单”中所述的过程。

或

选择现有制造订单实例。

2. 如果需要，展开附加的文档部分。
3. 单击“附加的文档详细信息”表格上的附加。将显示“附加文档”弹出窗口。
4. 选择附加要重新使用的新文档（必须指定修订版本）。将显示“文档路径”、“文档名称”、“修订版本”、“记录修订版本”和“文档描述”字段。
5. 单击文件上传图标以选择文件。
6. 在文档名称字段中输入文档名称。
7. 在修订版本字段中输入此文档的修订版本。
8. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
9. 单击附加。将创建一个文档实例，并且应用程序会显示一条消息，指明已成功附加该文档。

如何从制造订单拆离文档

按照以下步骤从制造订单拆离文档：

1. 打开制造订单页。“制造订单”页显示在“建模”页中。
2. 选择现有制造订单实例。
3. 如果需要，展开附加的文档部分。
4. 在“附加的文档详细信息”表格中选择一个项。
5. 单击拆离。将显示“拆离文档”弹出窗口，其中填充了“实例名称”和“附加的文档”字段。
6. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
7. 单击拆离。应用程序会显示一条消息，表示文档已成功拆离。
Note: 拆离文档时，不会删除在添加新的已修订文档时创建的文档实例。

如何查看附加到制造订单的文档

按照以下步骤查看附加到制造订单的文档：

1. 打开制造订单页。“制造订单”页显示在“建模”页中。
2. 选择现有制造订单实例。
3. 在“附加的文档详细信息”表格中选择一个项。
4. 单击查看。 应用程序会提示打开或保存该文件，或者它会自动下载该文件。（Internet 浏览器将确定应用程序是否显示弹出窗口、应用程序的外观以及“打开”或“保存”选项的措辞。）
Note: 必须具有相应的软件才能查看文档。
5. 选择要执行的作业，并完成任何使用特定应用程序打开文件的步骤或将文件保存到计算机的其他步骤。
