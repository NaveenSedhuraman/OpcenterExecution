定义查询

对调度管理建模的第一步是建立基于规则的查询。（记住，查询会确定向车间用户显示项目的顺序。）可以将规则基于用户查询或 Designer 查询。

定义用户查询时

定义用户查询时，必须在“查询文本”字段输入 SQL 语句。应用程序需要 SQL 语句从数据库中检索信息。或者，可以在“用户查询参数”表格中指定参数以限制搜索结果。应用程序会提示用户在执行查询时提供参数信息。

Note: 定义用户查询后，可以立即对生产执行此操作，而不必更新事务数据库。有关信息，请参考“定义用户查询”。

Designer 查询

此外，还可以通过 Designer 创建和维护自定义查询。有关创建自定义查询的信息，请参考《Opcenter Execution Medical Device and Diagnostics Designer 用户指南》或《Opcenter Execution Core Designer 用户指南》。
