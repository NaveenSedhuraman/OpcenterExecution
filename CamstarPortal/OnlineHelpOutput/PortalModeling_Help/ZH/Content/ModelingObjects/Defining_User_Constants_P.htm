定义用户常量

如果需要，可以在计算表达式中使用用户常量值。它们必须首先定义为用户常量。通过用户常量，可以创建和维护像“pi”一样不会更改或很少更改的常量值。可能有一些特定于业务流程的常量，这些常量用于多种计算。

在定义用户常量时

必须使用 Opcenter EX MDD 或 Opcenter EX CR 统一表达式来定义用户常量表达式。有关常量值和表达式的信息，请参考《Opcenter Execution Medical Device and Diagnostics Designer 用户指南》或《Opcenter Execution Core Designer 用户指南》中的统一表达式附录。

为用户常量输入文字时，用双引号 (“) 将该条目括起来。

用户常量函数支持以下运算符：

• 加法
• 减法
• 乘法
• 除法
• 求和（多个参数）
• 范围（多个参数）
• 一元减号/加号运算符
• 应用程序函数
• 基于时间戳的所有应用程序计算

如何定义用户常量

按照以下步骤定义用户常量：

1. 打开用户常量页。随即“建模”页中将显示“用户常量”页。
2. 单击新建。 显示用于定义新实例的空白字段。
3. 在用户常量字段中输入此实例的名称。
4. 从字段中选择数据类型。
5. 在值字段中输入要指派给数据类型的值。
6. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
7. 单击保存。 应用程序保存建模对象并显示成功消息。
