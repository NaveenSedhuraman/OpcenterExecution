Übersicht über die Änderungsverwaltung

Die Änderungsverwaltung ist ein Produktionsprozess, der Herstellern dabei hilft, den Lebenszyklus von Änderungen an ihren Produkt- und Prozessdefinitionen zu verwalten.

Die Änderungsverwaltung ermöglicht einem Unternehmen, Modellierungsobjekte aus einem System zu exportieren und diese Objekte dann in einem anderen System zu importieren. Z. B. könnte ein Unternehmen Modellierungsobjekte von seinem Testsystem exportieren und ins Produktionssystem importieren wollen.

Sie beginnen den Änderungsverwaltungsprozess, indem Sie ein Änderungsverwaltungspaket erstellen. Ein Änderungsverwaltungspaket ist eine Sammlung aller Modellierungsobjekt-Instanzen und Referenzen, die Sie von einem Quellsystem auf einem oder mehreren Zielsystemen bereitstellen möchten. Beim Erstellen eines Pakets müssen Sie einen Eigentümer für das Paket, den Grund für die Erstellung des Pakets und einen Arbeitsplan zur Änderungsverwaltung angeben.
