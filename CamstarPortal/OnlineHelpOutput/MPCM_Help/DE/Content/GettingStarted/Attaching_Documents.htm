Anhängen von Dokumenten

Sie können ein oder mehrere Dokumente an ein Paket anhängen. Dokumente enthalten relevante Informationen in Bezug auf das Paket, wie z. B. Abbildungen, Anweisungen und Prozeduren.

Beim Anhängen eines Dokuments an ein Paket

Sie rufen das Icon "Dokument anhängen" in der Paketinformationen-Kopfzeile am oberen Rand von Änderungsverwaltungspaket-Aufgabenseiten auf. Das Popup "Dokument anhängen" wird angezeigt, wenn Sie auf das Icon klicken.

Ziehen Sie beim Anhängen von Dokumenten folgende Überlegungen in Betracht:

• Sie müssen Dokumentinstanzen in der Modellierung mithilfe des Dokument-Modellierungsobjekts definieren, bevor Sie diese an ein Paket anhängen können. Sie können eine lokale Datei zum Hochladen in die Datenbank oder eine HTTP-Datei, die einen Hyperlink zu der Datei enthält, definieren. Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch oder im Opcenter Execution Core Modeling-Benutzerhandbuch.
• Sie können nur vorhandene Dokumente anhängen.
• Vor dem Anhängen eines vorhandenen Dokuments können Sie es anzeigen. Sie müssen über die entsprechende Software zum Anzeigen des Dokuments verfügen. Z. B. müssen Sie über Microsoft Word verfügen, um ein Word-Dokument anzuzeigen.

Vorgehensweise zum Anhängen eines Dokuments

Gehen Sie folgendermaßen vor, um ein Dokument anzuhängen:

1. Wählen Sie Paketsuche im Menü Änderungsverwaltung aus. Die Seite "Paketsuche" wird angezeigt.
2. Geben Sie ein oder mehrere Kriterien ein, um Ihre Suche einzugrenzen.
3. Klicken Sie auf Suchen. Die Anwendung zeigt die Pakete in der Tabelle "Suchergebnisse" an, die Ihren Suchkriterien entsprechen.
4. Wählen Sie das Paket aus, das Sie anzeigen möchten. Die Anwendung zeigt die relevanten Transaktionen für das Paket an.
5. Klicken Sie auf die gewünschte Transaktion. Die entsprechende Seite wird angezeigt.
6. Klicken Sie auf das Icon Dokument anhängen in der Paketinformationen-Kopfzeile am oberen Rand der Seite. Das Popup "Dokument anhängen" wird angezeigt.
7. Wählen Sie ein Dokument im Feld Dokumentinstanzen aus. Die Anwendung aktiviert die Schaltfläche "Dokument anzeigen".
8. Geben Sie die spezifische Revision in das Feld Revision ein.

Oder

Aktivieren Sie das Kontrollkästchen Standardrevision.

9. Klicken Sie auf Dokument anzeigen, wenn Sie das Dokument anzeigen möchten, bevor Sie es anhängen. Die Anwendung fordert Sie auf, die Datei zu öffnen oder zu speichern, oder sie lädt das Dokument automatisch herunter. (Ihr Internetbrowser bestimmt, ob die Anwendung ein Popup öffnet, wie es aussehen soll und was in den Optionen "Öffnen" und "Speichern" genau steht.)
10. Klicken Sie auf Anhängen. Daraufhin wird eine Meldung mit dem Hinweis angezeigt, dass das Dokument erfolgreich angehängt wurde.
