Anzeigen von Dokumenten

Über das Icon "Dokumentsatz" können Sie Dokumente anzeigen, die an das aktuelle Paket angehängt sind. Dokumente enthalten zusätzliche Informationen in Bezug auf das Paket, wie z. B. Abbildungen, Anweisungen und Verfahren.

Beim Anzeigen von Dokumenten

Sie rufen das Icon "Dokumentsatz" in der Paketinformationen-Kopfzeile am oberen Rand der Änderungsverwaltungspaket-Aufgabenseiten auf.

Wenn Sie auf das Icon "Dokumentsatz" klicken, wird eine Liste der Dokumente angezeigt, die mit dem Paket verbunden sind. Sie können jede Datei aus dem Dokumentsatz auswählen und anzeigen. Die Anwendung zeigt eine Nachricht an, die Sie fragt, ob Sie die Datei öffnen oder speichern möchten, wenn Sie ein Dokument auswählen.

Sie müssen über die entsprechende Software für den Dokumenttyp verfügen, den Sie öffnen möchten. Z. B. müssen Sie über Microsoft Word verfügen, um ein Word-Dokument anzuzeigen.

Die Standardeinstellungen für die Browsersicherheit verhindern, dass Dokumente auf unsicheren Websites geöffnet werden. Sie müssen Ihre Einstellungen ändern, um unsicheren Inhalt zuzulassen.

Vorgehensweise zum Anzeigen von Dokumenten

Gehen Sie folgendermaßen vor, um Dokumente anzuzeigen:

1. Wählen Sie Paketsuche im Menü Änderungsverwaltung aus. Die Seite "Paketsuche" wird angezeigt.
2. Geben Sie ein oder mehrere Kriterien ein, um Ihre Suche einzugrenzen.
3. Klicken Sie auf Suchen. Die Anwendung zeigt die Pakete in der Tabelle "Suchergebnisse" an, die Ihren Suchkriterien entsprechen.
4. Wählen Sie das Paket aus, das Sie anzeigen möchten. Die Anwendung zeigt die relevanten Transaktionen für das Paket an.
5. Klicken Sie auf die gewünschte Transaktion. Die entsprechende Seite wird angezeigt.
6. Klicken Sie auf das Icon Dokumentsatz in der Paketinformationen-Kopfzeile am oberen Rand der Seite. Das Menü "Dokumentsatz" wird angezeigt.
7. Wählen Sie das Dokument aus, das Sie anzeigen möchten. Daraufhin wird eine Meldung angezeigt, in der Sie gefragt werden, ob Sie das Dokument öffnen oder speichern oder die Anforderung abbrechen möchten.

Wenn Sie auf Folgendes klicken . . .

Dann . . .

Öffnen

Das Dokument wird in einem neuen Fenster angezeigt.

Speichern

Gibt die Anwendung die folgenden Optionen an:

• Speichern: Dokument in den Ordner C:\ herunterladen.
• Speichern unter: Dokument in den Ordner C:\ herunterladen und dem Dokument einen neuen Namen zuweisen.
• Speichern und öffnen: Dokument in den Ordner C:\ herunterladen und in einem neuen Fenster öffnen.

Abbrechen

Die Meldung wird geschlossen.
