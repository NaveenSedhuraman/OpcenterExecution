Paketinformationen

Informationen zu einem ausgewählten Paket werden am oberen Rand der Änderungsverwaltungspaket-Aufgabenseiten angezeigt. In der Regel umfassen diese Informationen die Kernattribute des Pakets und allgemeine Informationen über das Paket. Außerdem enthält dieser Bereich Icons, die den Zugriff auf weitere Funktionen ermöglichen oder Aufklapp-Menüs aktivieren, wenn Sie darauf klicken. Diese Icons enthalten Optionen zum:

• Ansehen von Dokumenten
• Anhängen von Dokumenten
• Anzeigen von Paketdetails
• Anzeigen der Auswirkungen der Aktivierung für das Paket
• Anzeigen der Inhaltsänderungshistorie

Die Anwendung erstellt die Paketinformationen, wenn Sie ein neues Paket speichern. Siehe "Erstellen eines Pakets" für weitere Informationen.

Beim Anzeigen von Kernattributen für ein Paket

Die Anwendung füllt automatisch die Kernattribute auf den Änderungsverwaltungsseiten. Das auf den Seiten "Aktivierungssuche" oder "Paketsuche" ausgewählte Paket ermittelt die angezeigten Attribute, wenn Sie die Transaktionsseite für das Paket laden.

Die Anwendung füllt die Paketinformationen mit Folgendem:

• Einem Icon, das den Status des Pakets angibt
• Einer beliebigen Beschreibung und Angaben zum Eigentümer, zur letzten Aktualisierung und zum Schritt

Hinweis: Ein Systemadministrator kann bei der Definition von Spezifikationen andere Icons angeben.

Durch Klicken auf das Paketstatus-Icon wird ein Popup angezeigt, das ein Diagramm des Arbeitsplans enthält, der dem angegebenen Paket zugeordnet ist. Das Popup zeigt die folgenden Informationen an:

• Den Namen des zugehörigen Arbeitsplans in der Titelleiste.
• Ein roter Rahmen um den aktuellen Paketschritt im Arbeitsplan.
• Eine Beschreibung des Arbeitsplans.
Hinweis: Beschreibungen werden nur dann für den Arbeitsplan und die Schritte (Spez.) angezeigt, wenn Beschreibungen in den Modellierungsobjekt-Instanzen angegeben wurden.

Sie können mit der Maus über einen Schritt im Arbeitsplan fahren, um eine Beschreibung dieses Schrittes anzuzeigen.

Beim Verwenden von Aufklapp-Menüs

Über einige der Icons in der Paketinformationen-Kopfzeile können Sie Aufklapp-Menüs anzeigen, die weitere Funktionen enthalten.

Sie können auf die Icons oder auf die Textbeschreibungen in diesen Menüs klicken, um die entsprechenden Seiten zu öffnen:

• Die Seite wird in einer neuen Registerkarte geladen, nachdem Sie auf das Icon oder die Textbeschreibung geklickt haben.
• Das Aufklapp-Menü wird beim Öffnen der Registerkarte geschlossen.
