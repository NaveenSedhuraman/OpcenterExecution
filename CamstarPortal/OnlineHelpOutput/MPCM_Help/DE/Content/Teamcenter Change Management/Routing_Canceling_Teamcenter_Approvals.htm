Weiterleiten und Abbrechen von Teamcenter-Genehmigungen

Der Teamcenter-Paketgenehmigungsprozess weicht geringfügig vom Camstar Change-Standardprozess ab. Der Paketgenehmigungsprozess erfolgt außerhalb von CEP, wenn die Genehmigungsanfrage an Teamcenter gesendet wird. Sie müssen sich an den Teamcenter-Ingenieur wenden, um eine ausstehende Genehmigungsanfrage abzubrechen.

Beim Weiterleiten eines Pakets an Teamcenter zur Genehmigung

Sie können ein Paket zur Genehmigung an Teamcenter weiterleiten, wenn sich ein Paket im Schritt "Pending Enrichment" (Anreicherung ausstehend) des standardmäßigen MOM PLM-Arbeitsplans befindet. Klicken Sie in der Seite "Route für Genehmigung" auf die Schaltfläche "Route für Genehmigung", um den Genehmigungsprozess zu beginnen. Die Anwendung ändert den Paketschritt in "Genehmigung ausstehend PLM", wenn Sie das Paket zur Genehmigung weiterleiten.

Zur Unterstützung des Genehmigungsentscheidungsprozesses sendet die Anwendung zwei verschlüsselte PDF-Dateien über Camstar Interoperability (CIO) an Teamcenter. Eine Datei enthält die Paketdetails, die andere die Inhaltsänderungshistorie. Der Genehmigerabschnitt der PDF mit den Paketdetails enthält keine Informationen, wenn der Genehmigungsprozess in Teamcenter durchgeführt wird. Weitere Informationen finden Sie unter "Anzeigen von Paketdetails" und "Anzeigen der Inhaltsänderungshistorie".

Nach dem Weiterleiten eines Pakets an Teamcenter zur Genehmigung

Nach der Weiterleitung zur Genehmigung gibt CIO eine Erfolgsmeldung zurück, wenn die beiden PDF-Dateien empfangen wurden. Wenn die Übertragung fehlschlägt, wird eine Fehlermeldung angezeigt.

Teamcenter verwendet den Paketstatus, um die Genehmigungsentscheidung zu bestimmen. An CEP wird eine der folgenden Genehmigungsentscheidungen zurückgesendet und der Spezifikations- oder Paketstatus wird aktualisiert.

Genehmigungsentscheidung

Spezifikation oder Status

Genehmigt

Das Paket wird in die Spezifikation "Pending Deployment" (Bereitstellung ausstehend) verschoben.

Abgelehnt (Rejected)

Das Paket wird in die Spezifikation "Rejected" (Abgelehnt) verschoben.

Ungültig gemacht

Der Paketstatus wird in "Ungültig gemacht" aktualisiert.
Hinweis: Sie können ein abgelehntes Paket manuell ungültig machen. Siehe "Ungültigmachen eines Pakets" für weitere Informationen.

Beim Abbrechen einer ausstehenden PLM-Genehmigung

Eine an Teamcenter gesendete PLM-Genehmigung kann abgebrochen werden, wenn sie sich im Schritt (Spezifikation) "Pending Approval PLM" (Genehmigung ausstehend PLM) des MOM PLM-Standardarbeitsplans befindet. Sie müssen sich an den für das Paket zuständigen Teamcenter-Ingenieur wenden, um einen Abbruch der ausstehenden Genehmigung anzufordern. Auf der Seite "Paketsuche" können Sie Informationen für ein Teamcenter-Änderungspaket suchen. Weitere Informationen finden Sie unter "Suchen nach Teamcenter-Paketen". Sie müssen das Paket dann manuell zurück in den Schritt "Pending Enrichment" (Anreicherung ausstehend) verschieben, indem Sie den Paketschritt ändern.

Ändern des Schritts eines Pakets

Sie können den Schritt eines Pakets im MOM PLM-Arbeitsplan manuell ändern, nachdem Sie die Bestätigung erhalten haben, dass die Genehmigungsanfrage abgebrochen wurde. Sie müssen ein Teamcenter-Paket auf der Seite "Paketsuche" suchen und das Paket auswählen, um die verfügbaren Transaktionen für das Paket anzuzeigen. Anschließend können Sie das Paket über die Schaltfläche "Schritt ändern" zurück in den Schritt "Pending Enrichment" (Anreicherung ausstehend) des MOM PLM-Standardarbeitsplans verschieben. Weitere Informationen finden Sie unter "Ändern des Schritts eines Pakets".

Hinweis: Siemens empfiehlt, den Schritt des Änderungspakets erst dann zu ändern, nachdem Sie vom zuständigen Teamcenter-Ingenieur für das Paket eine Bestätigung erhalten haben, dass die Genehmigungsanfrage abgebrochen wurde.

Ein Teamcenter-Ingenieur kann ein Paket auch ungültig machen, nachdem das Paket zur Genehmigung weitergeleitet wurde. Beim Empfang der Genehmigungsentscheidung ändert die Anwendung den Status des Pakets in "Ungültig gemacht". Darüber hinaus können Sie den Schritt eines Pakets manuell in "Ungültig gemacht" ändern, wenn der Teamcenter-Ingenieur den Genehmigungsstatus "Abgelehnt" zurückgibt.
