Suchen nach Teamcenter-Paketen

Auf der Seite "Paketsuche" können Sie nach MOM PLM-Paketen suchen. Sie können nach Teamcenter-Paketen und Paketen, die in CEP erstellt wurden, suchen. Weitere Informationen finden Sie unter "Suchen nach einem Paket".

Beim Suchen nach Teamcenter-Paketen

Bei der Suche nach Paketen, die von Teamcenter heruntergeladen wurden, sollten Sie Folgendes wissen:

• Es sind keine Transaktionen im Schritt "Download wird durchgeführt" verfügbar, sondern lediglich ein Status.
• Es gibt keine Transaktionsseite für eine PLM-Genehmigung. Der standardmäßige MOM PLM-Arbeitsplan ist nicht für manuelle Genehmigungen eingerichtet, kann aber entsprechend konfiguriert werden, wenn eine manuelle Genehmigung erforderlich ist. In Teamcenter gibt es Genehmigungstransaktionen. Diese werden an CEP gesendet und für das Paket aufgezeichnet. Das Paket wird dann in den Status "Rejected" (Abgelehnt) oder "Pending Deployment" (Bereitstellung ausstehend) versetzt. Weitere Informationen finden Sie unter "Weiterleiten und Abbrechen von Teamcenter-Genehmigungen".
Hinweis: In Teamcenter können auch ungültige Transaktionen auftreten. Diese werden an CEP gesendet und für das Paket aufgezeichnet.
• Es gibt keine Transaktion zum Abbrechen einer Genehmigung. Sie müssen den Teamcenter-Paketeigentümer über die Notwendigkeit eines Abbruchs benachrichtigen und den Paketstatus manuell auf "Pending Enrichment" (Anreicherung ausstehend) zurücksetzen. Weitere Informationen finden Sie unter "Weiterleiten und Abbrechen von Teamcenter-Genehmigungen".

Anzeigen eines CEP-Änderungspakets in Teamcenter

Unter "Verfügbare Transaktionen" wird die Transaktionsschaltfläche "Verbindung zu PLM" angezeigt und aktiviert, wenn ein Paket von Teamcenter heruntergeladen oder ein Teamcenter-Änderungspaket in den Suchergebnissen ausgewählt wird. Durch Klicken auf die Schaltfläche wird die Teamcenter-Anmeldeansicht geöffnet. Bei einer erfolgreichen Anmeldung wird die entsprechende ECN in Teamcenter angezeigt.

Die folgenden Bedingungen müssen erfüllt sein, um die entsprechende ECN anzuzeigen:

• Teamcenter muss installiert sein.
• Sie verfügen über gültige Teamcenter-Anmeldeinformationen sowie über entsprechende Rollen und Berechtigungen innerhalb Teamcenter und CEP.

Anzeigen einer Teamcenter-ECN in CEP

Sie können eine ECN in CEP anzeigen, indem Sie den Paketnamen in der URL-Vorlage eingeben und die betreffende URL mit dem Camstar-Referenzelement der ECN in Teamcenter verknüpfen. Durch Klicken auf das Camstar-Referenzelement der ECN wird die Camstar-Anmeldeseite für Benutzer geöffnet. Nach einer erfolgreichen Anmeldung bei CEP wird die Seite "Paketsuche" geöffnet. Das entsprechende Änderungspaket wird in der Tabelle "Suchergebnisse" angezeigt und ist standardmäßig ausgewählt.

Hinweis: Wenn der Name des Pakets in der URL-Vorlage falsch eingegeben wurde, öffnet die URL zwar Camstar und die Seite "Paketsuche", die Tabelle "Suchergebnisse" ist aber leer.
