Ändern des Schritts eines Pakets

Mit der Transaktion "Paketschritt ändern" können Sie ein Änderungsverwaltungspaket zu einem beliebigen Schritt in dem Arbeitsplan bewegen, der dem Paket zugewiesen wurde.

Beim Ändern des Schritts eines Pakets

Das Arbeitsplan-Diagramm wird dynamisch auf Grundlage des Arbeitsplans angezeigt, der dem Paket zugewiesen und im Feld "Ausgewähltes Paket" angegeben ist. Arbeitspläne werden Paketen während der Paketerstellung zugewiesen. Siehe "Erstellen eines Pakets" für weitere Informationen.

Das Arbeitsplan-Diagramm enthält alle Schritte im Arbeitsplan, darunter Schritte ohne eingehende oder ausgehende Pfade. Ein roter Rahmen wird um den aktuellen Paketschritt angezeigt. Wenn Sie mit der Maus über einen Schritt fahren, wird eine QuickInfo angezeigt, die eine Beschreibung des Schritts enthält, wenn während der Spez.-Erstellung eine Beschreibung eingegeben wurde.

Hinweis: Der Paketschritt kann nur in einen anderen Schritt im selben Arbeitsplan geändert werden.

Sie können von der Seite "Paketsuche" zur Seite "Paketschritt" gelangen, indem Sie ein Paket auswählen und auf die Schaltfläche "Schritt ändern" unter "Verfügbare Transaktionen" klicken. Die Anwendung füllt das Feld "Ausgewähltes Paket" automatisch mit dem Paket, das auf der Seite "Paketsuche" ausgewählt wurde.

Vorgehensweise zum Ändern des Schritts eines Pakets

Gehen Sie folgendermaßen vor, um den Schritt eines Pakets zu ändern:

1. Wählen Sie Paketsuche im Menü Änderungsverwaltung aus. Die Seite "Paketsuche" wird angezeigt.
2. Geben Sie ein oder mehrere Kriterien ein, um Ihre Suche einzugrenzen.
3. Klicken Sie auf Suchen. Die Anwendung zeigt die Pakete in der Tabelle "Suchergebnisse" an, die Ihren Suchkriterien entsprechen.
4. Wählen Sie das Paket aus, für das Sie den Schritt ändern möchten. Die Anwendung zeigt die relevanten Transaktionen für das Paket an.
5. Klicken Sie auf Schritt ändern. Die Seite "Paketschritt ändern" wird angezeigt.
6. Wählen Sie einen Paketschritt im Feld Wählen Sie einen neuen Paketschritt.
7. Geben Sie je nach den Anforderungen Ihres Betriebs optionale Informationen ein. In der Felddefinitionstabelle finden Sie Informationen zu den optionalen Feldern.
8. Klicken Sie auf Paketschritt ändern. Die Anwendung aktualisiert das Änderungsverwaltungspaket, das Arbeitsplan-Diagramm, das Paketschritt-Bild und zeigt eine Erfolgsmeldung an.
