Ungültigmachen eines Pakets

Sie können ein einzelnes Paket mit dem Status "Offen", "Bereitgestellt" oder "Abgelehnt" auf der Seite "Paketsuche" ungültig machen, wenn Sie über Berechtigungen als Paketeigentümer verfügen. Ein Paket wird normalerweise vor der Bereitstellung ungültig gemacht.

Nachdem ein Paket ungültig gemacht wurde, können Sie den Inhalt des Pakets nicht mehr ändern und das Paket nicht mehr bereitstellen. Sie können ein Paket mit dem Status "Ungültig gemacht" oder "Geschlossen" nicht ungültig machen.

Sie können auf der Seite "Aktivierungssuche" ein einzelnes Paket mit dem Paketstatus "Offen" und dem Aktivierungsstatus "Ausstehende Aktivierung" oder "Aktivierung unvollständig" ungültig machen.

Hinweis: Informationen über das Auswählen und gleichzeitige Ungültigmachen mehrerer Pakete finden Sie unter "Ungültigmachen mehrerer Pakete".

Beim Ungültigmachen eines Pakets

Das Popup "Paket ungültig machen" wird angezeigt, wenn Sie in der Tabelle "Suchergebnisse" auf der Seite "Paketsuche" oder "Aktivierungssuche" ein Paket und dann "Paket ungültig machen" auswählen. Sie können die entsprechenden Kommentare in das Feld "Kommentare" eingeben, wobei jeder Kommentar maximal 255 Zeichen umfassen darf, und dann auf "Ungültig machen" klicken.

Wenn Sie auf "Ungültig" klicken, führt die Anwendung folgende Aktionen durch:

• Das Paket wird ungültig gemacht und in den Status "Ungültig gemacht" versetzt.
• Das Popup "Paket ungültig machen" wird geschlossen.
• Es wird eine Erfolgsmeldung mit folgenden Informationen angezeigt:
• Name des Pakets
• Datum und Uhrzeit, wann das Paket ungültig gemacht wurde
• Name des Mitarbeiters, der das Paket ungültig gemacht hat
• Die Tabelle "Suchergebnisse" wird mit den aktuellen Suchkriterien neu geladen und Sie kehren zur gleichen Seite in der Tabelle zurück.
• Die folgenden Informationen werden in der Historie gespeichert:
• Name des Mitarbeiters, der das Paket ungültig gemacht hat
• Datum und Uhrzeit, wann die Anwendung das Paket ungültig gemacht hat
• Alle Kommentare, die Sie im Popup "Paket ungültig machen" eingegeben haben

Vorgehensweise zum Ungültigmachen eines Pakets

Gehen Sie folgendermaßen vor, um ein Paket ungültig zu machen:

1. Wird das Paket auf dem Quellsystem oder auf dem Zielsystem ungültig gemacht?

Antwort . . .

Dann . . .

Auf dem Quellsystem

Wählen Sie Paketsuche im Menü Änderungsverwaltung aus. Die Seite "Paketsuche" wird angezeigt.

Auf dem Zielsystem

Wählen Sie Aktivierungssuche im Menü Änderungsverwaltung aus. Die Seite "Aktivierungssuche" wird angezeigt.
2. Geben Sie ein oder mehrere Kriterien ein, um Ihre Suche einzugrenzen.
3. Klicken Sie auf Suchen. Die Anwendung zeigt die Pakete in der Tabelle "Suchergebnisse" an, die Ihren Suchkriterien entsprechen.
4. Wählen Sie das Paket aus, das Sie ungültig machen möchten. Die Anwendung zeigt die relevanten Transaktionen für das Paket an.
5. Klicken Sie auf Paket ungültig machen. Das Popup "Paket ungültig machen" wird angezeigt.
6. Geben Sie ggf. alle relevanten Kommentare in das Feld Kommentare ein.
7. Klicken Sie auf Ungültig. Die Anwendung macht das Paket ungültig, schließt das Popup "Paket ungültig machen", zeigt eine Erfolgsmeldung an, lädt die Tabelle "Suchergebnisse" neu und zeichnet die entsprechenden Informationen in der Historie auf.
