Öffnen eines Pakets

Sie können ein einzelnes Paket, das den Paketstatus "Geschlossen" oder "Ungültig gemacht" hat, auf der Seite "Paketsuche" öffnen, wenn Sie über Berechtigungen als Paketeigentümer verfügen.

Sie können außerdem ein einzelnes Paket, das den Paketstatus "Geschlossen" oder "Ungültig gemacht" hat, auf der Seite "Aktivierungssuche" öffnen, wenn Sie über Berechtigungen zur Paketaktivierung verfügen.

Hinweis: Informationen über das Auswählen und gleichzeitige Öffnen mehrerer Pakete finden Sie unter "Öffnen mehrerer Pakete".

Beim Öffnen eines Pakets

Das Popup "Paket öffnen" wird angezeigt, wenn Sie in der Tabelle "Suchergebnisse" auf der Seite "Paketsuche" oder "Aktivierungssuche" ein Paket und dann "Paket öffnen" auswählen. Sie können die entsprechenden Kommentare in das Feld "Kommentare" eingeben, wobei jeder Kommentar maximal 255 Zeichen umfassen darf, und dann auf "Öffnen" klicken.

Wenn Sie auf "Öffnen" klicken, führt die Anwendung folgende Aktionen durch:

• Das Paket wird zurück in den Arbeitsplan, Schritt und Status platziert, den es hatte, bevor es geschlossen oder ungültig gemacht wurde, als Sie das Paket auf der Seite "Paketsuche" geöffnet hatten. Wurde ein Paket z. B. geschlossen, während es sich in einem Arbeitsplan in einem Entwurfsschritt befunden hat, wird es in dem Arbeitsplan in dem Entwurfsschritt mit dem Paketstatus "Offen" platziert, wenn Sie es erneut öffnen.
Hinweis: Die Anwendung setzt außerdem die Modellierungsobjekt-Instanzen im Paket auf den Status "Gesperrt" zurück, wenn Sie es erneut öffnen und der Schritt angibt, dass der Inhalt gesperrt sein soll.
• Das Paket wird zurück in den Aktivierungsstatus platziert, den es hatte, bevor es geschlossen oder ungültig gemacht wurde, als Sie das Paket auf der Seite "Aktivierungssuche" geöffnet hatten. Wurde ein Paket z. B. geschlossen, während es den Status "Ausstehende Aktivierung" hatte, wird es in den Status "Ausstehende Aktivierung" mit dem Paketstatus "Offen" platziert, wenn Sie es erneut öffnen.
Hinweis: Der Aktivierungsstatus ist derselbe. Nur der Paketstatus ("Geschlossen", "Ungültig gemacht", "Offen") wird geändert.
• Das Popup "Paket öffnen" wird geschlossen.
• Es wird eine Erfolgsmeldung mit folgenden Informationen angezeigt:
• Name des Pakets
• Datum und Uhrzeit, wann das Paket geöffnet wurde
• Name des Mitarbeiters, der das Paket geöffnet hat
• Die Tabelle "Suchergebnisse" wird mit den aktuellen Suchkriterien neu geladen und Sie kehren zur gleichen Seite in der Tabelle zurück.
• Die folgenden Informationen werden in der Historie gespeichert:
• Name des Mitarbeiters, der das Paket geöffnet hat
• Datum und Uhrzeit, wann die Anwendung das Paket geöffnet hat
• Alle Kommentare, die Sie im Popup "Paket öffnen" eingegeben haben

Vorgehensweise zum Öffnen eines Pakets

Gehen Sie folgendermaßen vor, um ein Paket zu öffnen:

1. Wird das Paket auf dem Quellsystem oder auf dem Zielsystem geöffnet?

Antwort . . .

Dann . . .

Auf dem Quellsystem

Wählen Sie Paketsuche im Menü Änderungsverwaltung aus. Die Seite "Paketsuche" wird angezeigt.

Auf dem Zielsystem

Wählen Sie Aktivierungssuche im Menü Änderungsverwaltung aus. Die Seite "Aktivierungssuche" wird angezeigt.
2. Geben Sie ein oder mehrere Kriterien ein, um Ihre Suche einzugrenzen.
3. Klicken Sie auf Suchen. Die Anwendung zeigt die Pakete in der Tabelle "Suchergebnisse" an, die Ihren Suchkriterien entsprechen.
4. Wählen Sie das Paket aus, das Sie öffnen möchten. Die Anwendung zeigt die relevanten Transaktionen für das Paket an.
5. Klicken Sie auf Paket öffnen. Das Popup "Paket öffnen" wird angezeigt.
6. Geben Sie ggf. relevante Kommentare in das Feld Kommentare ein.
7. Klicken Sie auf Öffnen. Die Anwendung öffnet das Paket, schließt das Popup "Paket öffnen", zeigt eine Erfolgsmeldung an, lädt die Tabelle "Suchergebnisse" neu und zeichnet die entsprechenden Informationen in der Historie auf.
