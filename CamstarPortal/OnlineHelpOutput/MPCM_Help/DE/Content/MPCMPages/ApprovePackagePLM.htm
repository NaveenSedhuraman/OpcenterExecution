Genehmigen eines Pakets (PLM)

Sie können Pakete, die dem PLM-Arbeitsplan zur Änderungsverwaltung zugewiesen wurden, genehmigen, wenn sie den Schritt "Genehmigung ausstehend PLM" (Pending Approval PLM) haben.

Bei der Genehmigung eines Pakets im PLM-Arbeitsplan

Sie müssen die Rolle des Paketeigentümers haben.

"Genehmigt" und "Abgelehnt" sind die einzigen Optionen, die für eine PLM-Genehmigungsentscheidung verfügbar sind. Wenn Sie "Genehmigt" auswählen, wird das Paket zur Änderungsverwaltung in den Schritt "Bereitstellung ausstehend" bewegt. Wenn Sie "Abgelehnt" auswählen, wird das Paket zur Änderungsverwaltung in den Schritt "Abgelehnt" bewegt. Es ist nur eine Genehmigungsentscheidung des Typs "Ablehnen" erforderlich, damit die Anwendung das Paket in den Schritt "Abgelehnt" verschiebt. Allerdings muss die Anwendung alle Genehmigungsentscheidungen empfangen haben, bevor sie das Paket zum nächsten Schritt bewegt.

Felddefinitionen der Seite "Paket genehmigen (PLM)"

Diese Tabelle definiert die Felder auf der Seite "Paket genehmigen (PLM)".

Feld

Definition

Typ

Paketinformationen

Das ausgewählte Paket und dessen Kernattribute. Kernattribute enthalten Informationen wie Status, Beschreibung, Eigentümer, letzte Aktualisierung und Schritt. Außerdem enthält dieser Bereich Icons, die den Zugriff auf weitere Funktionen ermöglichen oder Aufklapp-Menüs aktivieren, wenn Sie darauf klicken. Siehe auch "Paketinformationen" für Informationen.

Erforderlich

PLM-Genehmigungsentscheidung

"Genehmigt" und "Abgelehnt" sind die Optionen.

Erforderlich

Kommentare

Optional

Vorgehensweise zum Aufzeichnen einer PLM-Genehmigungsentscheidung

Gehen Sie folgendermaßen vor, um eine PLM-Genehmigungsentscheidung aufzuzeichnen:

1. Wählen Sie Paketsuche im Menü Änderungsverwaltung aus. Die Seite "Paketsuche" wird angezeigt.
2. Geben Sie ein oder mehrere Kriterien ein, um Ihre Suche einzugrenzen.
3. Klicken Sie auf Suchen. Die Anwendung zeigt die Pakete in der Tabelle "Suchergebnisse" an, die Ihren Suchkriterien entsprechen.
4. Wählen Sie das Paket aus, das Sie genehmigen möchten. Die Anwendung zeigt die relevanten Transaktionen für das Paket an.
5. Klicken Sie auf Genehmigen (PLM). Die Seite "Paket genehmigen (PLM)" wird angezeigt.
6. Wählen Sie eine PLM-Genehmigungsentscheidung aus.
7. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. In der Felddefinitionstabelle finden Sie Informationen zu den optionalen Feldern.
8. Klicken Sie auf Entscheidung speichern. Die Anwendung zeigt eine Meldung an, dass Sie die Genehmigung abgeschlossen haben.
