Weiterleiten eines Pakets zur Genehmigung

Sie können ein Paket zur Genehmigung weiterleiten, wenn:

• Sie der Paketbesitzer oder ein Benutzer mit Eigentümerrechten sind,
• Das Paket im Entwurfsschritt ist und
• Dem Paket ein Arbeitsplan zur Änderungsverwaltung mit einem Genehmigungsschritt zugewiesen wurde.

Vor dem Weiterleiten eines Pakets zur Genehmigung

Gewährleisten Sie, dass die Genehmigungsbenachrichtigungen vor dem Weiterleiten eines Pakets eingerichtet sind. Es müssen Mitteilungen konfiguriert werden, damit Genehmiger eine E-Mail erhalten, die sie benachrichtigt, dass ein Paket ihre Zustimmung erfordert. Unter "Verstehen von Genehmigungsbenachrichtigungen" finden Sie Informationen zum Einrichten von Benachrichtigungen.

Hinweis: Derzeit erhalten nur Genehmiger für ein Paket auf dem Standardarbeitsplan für Änderungsverwaltung von Camstar Benachrichtigungen.

Die Anwendung erfordert, dass ein Änderungsverwaltungspaket die folgenden Kriterien erfüllt, damit Sie es zur Genehmigung weiterleiten können:

• Instanz-Inhalt wurde zugewiesen
• Ein oder mehrere Zielsysteme wurden ausgewählt
• Das Paket wurde dem standardmäßigen Camstar-Arbeitsplan zur Änderungsverwaltung zugewiesen:
• Ein oder mehrere Genehmiger wurden zugewiesen und
• Eine Genehmigungsentscheidungsliste wurde zugewiesen.
Hinweis: Sie müssen eine Genehmigungsvorlage auf der Seite "Paket aktualisieren" auswählen, um eine Genehmigungsentscheidungsliste zuzuweisen. Siehe "Aktualisieren eines Pakets" für weitere Informationen.

Beim Weiterleiten eines Pakets zur Genehmigung

Die Seite "Route für Genehmigung" zeigt die Kernattribute des ausgewählten Pakets und ein "Informationen"-Icon an, auf das Sie klicken können, um eine Zusammenfassung des Paketinhalts anzuzeigen.

Durch Klicken auf das "Informationen"-Icon wird eine Tabelle "Paketinhaltszusammenfassung" angezeigt. Die Tabelle "Paketinhaltszusammenfassung" zeigt die folgenden Informationen an:

• Anzahl der Modellierungsobjekte, die in der Anwendung definiert sind
• Gesamtzahl dieser definierten Modellierungsobjekt-Instanzen
• Die Anzahl der Modellierungsobjekte, die dem Paket zugewiesen sind
• Die Gesamtzahl der Modellierungsobjekt-Instanzen, die dem Paket zugewiesen sind
• Die Anzahl der Modellierungsobjekte, die von den ausgewählten Modellierungsobjekten referenziert werden
• Die Gesamtzahl der referenzierten Modellierungsobjekt-Instanzen, die dem Paket zugewiesen sind

Die Anwendung ändert den Paketschritt in "Genehmigung ausstehend Camstar" (Pending Approval Camstar) oder "Genehmigung ausstehend PLM" (Pending Approval PLM), wenn Sie das Paket zur Genehmigung weiterleiten.

Vorgehensweise zum Weiterleiten eines Pakets zur Genehmigung

Gehen Sie folgendermaßen vor, um ein Paket zur Genehmigung weiterzuleiten:

1. Wählen Sie Paketsuche im Menü Änderungsverwaltung aus. Die Seite "Paketsuche" wird angezeigt.
2. Geben Sie ein oder mehrere Kriterien ein, um Ihre Suche einzugrenzen.
3. Klicken Sie auf Suchen. Die Anwendung zeigt die Pakete in der Tabelle "Suchergebnisse" an, die Ihren Suchkriterien entsprechen.
4. Wählen Sie das Paket aus, das Sie zur Genehmigung weiterleiten möchten. Die Anwendung zeigt die relevanten Transaktionen für das Paket an.
5. Klicken Sie auf Route für Genehmigung. Die Seite "Route für Genehmigung" wird angezeigt.
6. Klicken Sie auf das "Informationen"-Icon, um bei Bedarf eine Zusammenfassung des Paketinhalts anzuzeigen.
7. Klicken Sie auf Route für Genehmigung. Die Anwendung zeigt eine Meldung an, dass das Paket weitergeleitet wurde.
