Abbrechen einer Genehmigung

Sie können den Genehmigungsprozess eines Pakets zur Änderungsverwaltung jederzeit abbrechen, unabhängig davon, ob Genehmiger bereits Entscheidungen erfasst haben. Sie können eine Paketgenehmigung abbrechen, wenn die folgenden Bedingungen erfüllt sind:

• Der Paketschritt ist "Genehmigung ausstehend Camstar" (Pending Approval Camstar) oder "Genehmigung ausstehend PLM" (Pending Approval PLM) und
• Sie sind der Paketeigentümer sind oder haben Eigentümerrechte.

Beim Abbrechen einer Paketgenehmigung

Die Seite "Genehmigung abbrechen" zeigt nur die Kernattribute des ausgewählten Pakets an. Die Anwendung aktiviert die Schaltfläche "Genehmigung abbrechen", wenn Sie ein Paket auswählen.

Das Abbrechen einer Genehmigung versetzt das Paket zurück in einen bearbeitbaren Schritt und setzt den Genehmigungsprozess zurück. Die Anwendung gibt z. B. Pakete, die sich im Camstar-Änderungsverwaltungs-Arbeitsplan beim Schritt "Genehmigung ausstehend Camstar" (Pending Approval Camstar) befinden, an den Schritt "Entwurf Camstar" (Draft Camstar) zurück. Sie können dann den Paketinhalt oder den Genehmigungsprozess ändern. Sie müssen dann das Paket zur Genehmigung weiterleiten, nachdem Sie Änderungen vorgenommen haben. Jeder Genehmiger, der das Paket bereits vor dem Abbruch genehmigt hatte, muss das Paket erneut genehmigen.

Einen Verlauf der abgebrochenen Genehmigungen und Genehmiger-Entscheidungen finden Sie auf der Seite "Paketdetails" und auf der Registerkarte "Genehmigungsstatus" der Seite "Paket genehmigen". Weitere Informationen finden Sie unter "Anzeigen von Paketdetails" und "Genehmigen eines Pakets (Camstar)".

Das Feld "Genehmigungskommentare abbrechen" ist für die betreffenden Kommentare über den Abbruch der Paketgenehmigung verfügbar. Die Anwendung verkürzt jeden Eintrag mit mehr als 255 Zeichen.

Vorgehensweise zum Abbrechen einer Genehmigung

Gehen Sie folgendermaßen vor, um eine Genehmigung abzubrechen:

1. Wählen Sie Paketsuche im Menü Änderungsverwaltung aus. Die Seite "Paketsuche" wird angezeigt.
2. Geben Sie ein oder mehrere Kriterien ein, um Ihre Suche einzugrenzen.
3. Klicken Sie auf Suchen. Die Anwendung zeigt die Pakete in der Tabelle "Suchergebnisse" an, die Ihren Suchkriterien entsprechen.
4. Wählen Sie das Paket aus, für das Sie den Genehmigungsprozess abbrechen möchten. Die Anwendung zeigt die relevanten Transaktionen für das Paket an.
5. Klicken Sie auf Genehmigung abbrechen. Die Seite "Genehmigung abbrechen" wird angezeigt.
6. Klicken Sie auf Genehmigung abbrechen. Die Anwendung zeigt eine Meldung an, dass der Genehmigungsprozess abgebrochen wurde.
