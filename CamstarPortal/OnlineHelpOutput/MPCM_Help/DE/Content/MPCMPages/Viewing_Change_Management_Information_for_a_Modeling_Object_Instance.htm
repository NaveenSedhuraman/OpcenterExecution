Anzeigen von Informationen zur Änderungsverwaltung für eine Modellierungsobjekt-Instanz

Wenn Sie auf der Seite des jeweiligen Objekts eine zusätzliche Registerkarte öffnen, können Sie Informationen über die Anzahl der einer Instanz zugeordneten Pakete anzeigen und Angaben dazu, ob die Instanz durch ein Paket gesperrt ist.

Beim Anzeigen von Informationen zur Änderungsverwaltung für eine Modellierungsobjekt-Instanz

Jede Modellierungsobjekt-Seite enthält grundlegende Informationen zu der Instanz, die Sie definieren. Diese Informationen werden im Kopfzeilenbereich der Seite angezeigt. Eine weitere Registerkarte mit der Beschriftung "Änderungsmanagement" wird in diesem Abschnitt angezeigt, wenn Sie eine Lizenz für die Änderungsverwaltung besitzen, mindestens ein Paket definiert haben und über eine Rolle mit der Berechtigung "Änderungspaket-Modellierungsabfrage" verfügen.

Felddefinitionen der Tabelle "Änderungsmanagement"

Diese Tabelle definiert die Felder auf der Registerkarte "Änderungsmanagement".

Feld

Definition

Typ

Zugeordnete Pakete

Anzahl der Pakete, denen die Instanz zugeordnet ist. Die Anzahl schließt keine geschlossenen oder ungültig gemachten Pakete ein, denen die Instanz zuvor zugeordnet war.

Nur Anzeige

Instanz gesperrt durch Änderungsmanagement

Gibt an, ob die Instanz durch ein Änderungsverwaltungspaket gesperrt ist. Wenn ja, können Sie die Instanz erst bearbeiten, wenn sich alle Pakete, denen sie zugeordnet ist, in einem Schritt befinden, in dem die Instanz entsperrt ist. Wenn Sie die Standardschritte verwenden, tritt dies ein, wenn die zugeordneten Pakete geschlossen, ungültig gemacht oder zurück in einen Entwurfsschritt versetzt wurden.

Nur Anzeige

Vorgehensweise zum Anzeigen von Informationen zur Änderungsverwaltung für eine Modellierungsobjekt-Instanz

Gehen Sie folgendermaßen vor, um Informationen zur Änderungsverwaltung für eine Modellierungsobjekt-Instanz anzuzeigen:

1. Wählen Sie Modellierung im Menü Modellierung aus. Die Seite "Modellierung" (Modeling) wird angezeigt.
2. Klicken Sie auf der Seite "Modellierung" auf den Pfeil Alle Wartung, um eine Liste der Modellierungsobjekte in dieser Gruppe anzuzeigen.

Oder

Wählen Sie auf der Seite "Modellierung" in der Liste Objekte anzeigen die Option Alle anzeigen (alphabetisch) aus, um eine Liste von Modellierungsobjekten in alphabetischer Reihenfolge anzuzeigen. Eine Liste aller Modellierungsobjekte wird in alphabetischer Reihenfolge angezeigt.

3. Wählen Sie ein Modellierungsobjekt aus. Die Seite des Objekts wird auf der Seite "Modellierung" angezeigt.
4. Wählen Sie eine Modellierungsobjekt-Instanz aus. Die Felder für die Instanz werden angezeigt. Die Registerkarte "Änderungsverwaltung" wird neben den Registerkarten "Beschreibung" und "Anmerkungen" angezeigt.
