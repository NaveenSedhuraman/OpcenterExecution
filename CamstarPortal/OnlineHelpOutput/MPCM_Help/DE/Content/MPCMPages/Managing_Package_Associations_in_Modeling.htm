Verwalten von Paketzuordnungen in der Modellierung

Sie können die einer Modellierungsobjekt-Instanz zugeordneten Pakete innerhalb der Modellierung verwalten. Sie können Zuordnungen anzeigen, entfernen oder hinzufügen, wenn Sie über die entsprechende Rolle und Berechtigungen verfügen.

Beim Verwalten von Paketzuordnungen in der Modellierung

Jede Modellierungsobjektseite enthält eine Symbolleiste in der Nähe des oberen Seitenrands. Eine weitere Schaltfläche mit der Beschriftung "Paketzuordn." wird in der Symbolleiste angezeigt, wenn Sie eine Lizenz für die Änderungsverwaltung besitzen, mindestens eine Instanz des Modellierungsobjekts gespeichert haben, das Sie einem Paket zuordnen möchten, und Ihre Rolle die Berechtigung "Änderungspaket-Modellierungsabfrage" einschließt. Wenn Sie auf diese Schaltfläche klicken, können Sie Paketzuordnungen in der Modellierung für die ausgewählte Modellierungsobjektinstanz verwalten. Sie können die der Instanz zugeordneten Pakete anzeigen oder Zuordnungen entfernen oder hinzufügen, wenn Sie über Paketeigentümer-Berechtigungen verfügen.

Das Popup "Paketzuordnungen verwalten" wird angezeigt, wenn Sie auf die Schaltfläche "Paketzuordn." klicken. Die Registerkarte "Zugeordnete Pakete" ist die Standardregisterkarte, wenn Sie über die Schaltfläche auf das Popup zugreifen.

Unter "Beim Zuweisen von Modellierungsobjekt-Instanzen zu einem Paket in der Modellierung" und "Felddefinitionen des Popups "Paketzuordnungen verwalten" finden Sie Informationen zu den Registerkarten und Feldern im Popup "Paketzuordnungen verwalten".

Vorgehensweise zum Verwalten von Paketzuordnungen in der Modellierung

Gehen Sie folgendermaßen vor, um Paketzuordnungen in der Modellierung zu verwalten:

1. Wählen Sie Modellierung im Menü Modellierung aus. Die Seite "Modellierung" (Modeling) wird angezeigt.
2. Klicken Sie auf der Seite "Modellierung" auf den Pfeil Alle Wartung, um eine Liste der Modellierungsobjekte in dieser Gruppe anzuzeigen.

Oder

Wählen Sie auf der Seite "Modellierung" in der Liste Objekte anzeigen die Option Alle anzeigen (alphabetisch) aus, um eine Liste von Modellierungsobjekten in alphabetischer Reihenfolge anzuzeigen.Eine Liste aller Modellierungsobjekte wird in alphabetischer Reihenfolge angezeigt.

3. Wählen Sie ein Modellierungsobjekt aus. Die Seite des Objekts wird auf der Seite "Modellierung" angezeigt.
4. Erstellen Sie eine neue Modellierungsobjekt-Instanz oder aktualisieren Sie eine vorhandene Instanz.
5. Klicken Sie auf Speichern. Die Anwendung speichert das Modellierungsobjekt und zeigt eine Erfolgsmeldung an. Die Schaltfläche "Paketzuordn." wird in der Symbolleiste der Modellierungsseite angezeigt.
6. Klicken Sie auf die Schaltfläche Paketzuordn. in der Symbolleiste der Modellierungsseite. Das Popup "Paketzuordnungen verwalten" mit ausgewählter Registerkarte "Zugeordnete Pakete" wird angezeigt.
7. Möchten Sie Paketzuordnungen anzeigen, entfernen oder hinzufügen?

Dann . . .

Paketzuordnungen anzeigen
a. Zeigen Sie die Liste der Pakete an, die der Instanz zugeordnet sind.
b. Klicken Sie auf Schließen. Die Anwendung schließt das Popup "Paketzuordnungen verwalten".

Paketzuordnungen entfernen
a. Klicken Sie auf die Registerkarte Aus Paket entfernen.
b. Wählen Sie die Pakete aus, aus denen Sie die Instanz entfernen möchten.

Hinweis: Sie können nur explizit hinzugefügte Instanzen entfernen, keine Abhängigkeiten.

c. Klicken Sie auf Entfernen. Die Anwendung entfernt die Modellierungsobjekt-Instanz aus den ausgewählten Paketen, zeigt eine Erfolgsmeldung an und schließt das Popup "Paketzuordnungen verwalten".

Paketzuordnungen hinzufügen
a. Klicken Sie auf die Registerkarte Zu Paket hinzufügen.
b. Wählen Sie das Paket aus, dem Sie die Instanz hinzufügen möchten.
c. Klicken Sie auf Hinzufügen. Die Anwendung fügt die Modellierungsobjekt-Instanz dem Paket hinzu, zeigt eine Erfolgsmeldung an und schließt das Popup "Paketzuordnungen verwalten".
