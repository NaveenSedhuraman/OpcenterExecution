Suchen nach einem Paket zum Aktivieren

Die Anwendung enthält eine Funktion, mit der Sie ein Änderungsverwaltungspaket zur Aktivierung auf dem Zielsystem anhand angegebener Kriterien suchen und abrufen können. Die Funktion "Aktivierungssuche" ist für Pakete verfügbar, die bereitgestellt wurden und einen für die Aktivierung zulässigen Status haben. Der Mitarbeiter, der nach einem Paket zur Aktivierung sucht, muss eine Rolle mit Aktivierungsberechtigungen zugewiesen bekommen haben oder über die standardmäßige Paketaktivierer-Rolle verfügen. Rollen werden Mitarbeitern in der Modellierung zugewiesen.

Hinweis: Informationen über das Auswählen und gleichzeitige Schließen, Ungültigmachen oder Öffnen mehrerer Pakete finden Sie unter "Suchen nach mehreren Paketen, die auf dem Zielsystem geschlossen, ungültig gemacht oder geöffnet werden sollen".

Beim Suchen nach einem Paket zum Aktivieren

Wenn Sie erstmals auf die Seite "Aktivierungssuche" zugreifen, zeigt die Anwendung die zum Filtern verfügbaren Felder und eine leere Tabelle mit Suchergebnissen an. Die Anwendung fügt ein Filtersymbol zur Registerkarte hinzu, um anzuzeigen, dass Filterkriterien angegeben wurden. Klicken Sie auf die Schaltfläche "Suchen", um die Tabelle "Suchergebnisse" zu füllen.

Beim Suchen nach Paketen:

• Sie können mithilfe von SQL-Platzhaltern nach Paketnamen suchen. Wenn Sie z. B. "%01%" eingeben, werden Pakete mit Namen ausgegeben, die die Zeichen "01" enthalten.
• Sie können nach Aktivierungsstatus suchen. Es gibt drei Standardpaketstatus. Die Anwendung wählt standardmäßig "Aktivierung unvollständig" und "Ausstehende Aktivierung" aus. Der Status "Aktiviert" ist nicht ausgewählt. Das Feld "Aktivierungsstatus" zeigt eine Zahl anstatt der ausgewählten Status an. Die Anwendung zeigt "2 von 3 ausgewählt" im Feld "Aktivierungsstatus" an, wenn die Seite geladen wird.
Hinweis: Bei der Suche nach einem Paket, das vor kurzem zu aktiviert wurde, können unerwartete Ergebnisse zurückgegeben werden. Der Aktivierungsstatus wird standardmäßig alle 5 Minuten aktualisiert. Sie können diese Einstellung in der Modellierung ändern, indem Sie die standardmäßige geplante Geschäftsregel SBR_ACTIVATION anpassen. Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch oder im Opcenter Execution Core Modeling-Benutzerhandbuch.
• Sie können nach Paketstatus suchen. Es sind drei Status verfügbar. Die Anwendung wählt standardmäßig "Offen" aus. "Geschlossen" und "Ungültig gemacht" werden nicht ausgewählt. Die Anwendung zeigt eine Zahl anstatt der ausgewählten Status an. Die Anwendung zeigt "1 von 3 ausgewählt" im Feld "Paketstatus" an, wenn die Seite geladen wird.
Hinweis: Sie können die Paketdetails für ein geschlossenes oder ungültig gemachtes Paket anzeigen, wenn Sie den Status "Geschlossen" oder "Ungültig gemacht" auswählen.
• Wenn Sie keine Filter angeben, wird jedes Änderungsverwaltungspaket, für das Sie eine Zugriffsberechtigung haben, in den Ergebnissen angezeigt.
• Sie können Ihre Suche mit bis zu sieben Filtern eingrenzen.

Wenn die Anwendung Änderungsverwaltungspakete ausgibt, die Ihren Kriterien entsprechen, können Sie das Paket auswählen, an dem Sie eine Transaktion durchführen möchten. Die Anwendung aktiviert die Transaktionen, die für das ausgewählte Paket relevant sind.

Durchführen einer Suche

Wenn Sie eine Suche ausführen, macht die Anwendung Folgendes:

• Sie zeigt die Suchergebnisse in einer Tabelle an
• Sie behält die Suchkriterien in den Filterfeldern bei

Die Anwendung fügt der Tabelle eine Umblätter-Funktion hinzu, wenn die Suche mehr Ergebnisse zurückgibt, als in den Zeilen der Tabelle angezeigt werden können. Wenn Sie ein Paket in der Tabelle "Suchergebnisse" auswählen, aktiviert die Anwendung Transaktionen, die für das Paket relevant sind. Siemens bietet auch die Möglichkeit, Suchergebnisse in eine Excel-Datei oder im CSV-Format zu exportieren. Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics Shop Floor-Benutzerhandbuch oder im Opcenter Execution Core Shop Floor-Benutzerhandbuch.

Löschen von Suchfiltern und Ergebnissen

Verwenden Sie die Schaltfläche "Alles löschen", um Filter und Ergebnisse zu löschen. Das Löschen der Seite "Aktivierungssuche" ermöglicht Ihnen den Start einer neuen Suche. Wenn Sie auf die Schaltfläche "Alles löschen" klicken, reagiert die Anwendung folgendermaßen:

• Löscht alle angegebenen Suchfilter
• Das Filtersymbol wird aus der Registerkarte "Allgemeine Suche" entfernt.
• Löscht alle Pakete aus der Tabelle "Suchergebnisse"
• Löscht alle Transaktionsschaltflächen
• Zeigt weiterhin die Seite "Aktivierungssuche" an

Verwenden der Transaktionsschaltflächen

Die Transaktionsschaltflächen ermöglichen es Ihnen, Transaktionen an in der Tabelle "Suchergebnisse" ausgewählten Paketen durchzuführen. Die Anwendung aktiviert gültige Transaktionen für das ausgewählte Paket.

Die Anwendung aktiviert die Transaktionsschaltflächen "Paket schließen" und "Paket ungültig machen" abhängig vom Paketstatus und Aktivierungsstatus des ausgewählten Pakets.

Wenn Sie ein Paket auswählen und auf eine Transaktionsschaltfläche klicken, zeigt die Anwendung abhängig vom Typ der ausgewählten Transaktion die entsprechende Transaktionsseite oder das entsprechende Popup an.

Wenn Sie ein Paket schließen, ungültig machen oder öffnen bzw. ein Paket mit Voraussetzungen aktivieren, wird ein Popup angezeigt. Sie können relevante Kommentare eingeben.

Hinweis: Sie können ein Paket öffnen, das zuvor auf der Seite "Aktivierungssuche" oder "Aktivierungssuche (mehrere)" geschlossen oder ungültig gemacht wurde.

Wenn Sie eine der anderen Transaktionen durchführen, wird eine Transaktionsseite angezeigt. Die Anwendung trägt den Namen des in der Tabelle ausgewählten Pakets in das Feld "Ausgewähltes Paket" (sofern verfügbar) ein. Wenn Sie z. B. ein Paket auswählen und auf die Schaltfläche "Paket aktivieren" klicken, wird die Seite "Paket aktivieren" angezeigt.

Vorgehensweise zum Suchen nach einem Paket zum Aktivieren

Gehen Sie folgendermaßen vor, um nach einem Paket zum Aktivieren zu suchen:

1. Wählen Sie Aktivierungssuche im Menü Änderungsverwaltung aus. Die Seite "Aktivierungssuche" wird angezeigt.
2. Geben Sie ein oder mehrere Kriterien ein, um Ihre Suche einzugrenzen.
3. Klicken Sie auf Suchen. Die Anwendung zeigt die Pakete in der Tabelle "Suchergebnisse" an, die Ihren Suchkriterien entsprechen.

Vorgehensweise zum Ausführen einer Transaktion an einem ausgewählten Paket

Gehen Sie folgendermaßen vor, um eine Transaktion an einem ausgewählten Paket auszuführen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Suchen nach einem Paket zum Aktivieren" aus.
2. Wählen Sie das Paket aus, an dem Sie eine Transaktion ausführen möchten. Die Anwendung aktiviert die relevanten Transaktionen für das Paket.
3. Klicken Sie auf eine Transaktionsschaltfläche. Die Anwendung zeigt die entsprechende Transaktionsseite oder das Popup für die ausgewählte Transaktion an. Die Anwendung trägt den Namen des in der Tabelle ausgewählten Pakets in das Feld "Ausgewähltes Paket" (sofern verfügbar) auf der Seite der entsprechenden Transaktion ein.
4. Vervollständigen Sie die Informationen auf der Transaktionsseite oder im Popup.
