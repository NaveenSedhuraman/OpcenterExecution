Anzeigen der Inhaltsänderungshistorie

Sie können über die Seite "Paketsuche" oder über das Aufklapp-Menü "Historie" in der Paketinformationen-Kopfzeile auf die Seite "Inhaltsänderungshistorie" zugreifen. Sie können die Änderungen anzeigen, die an den einem Änderungspaket zugewiesenen Instanzen vorgenommen wurden, und die Vorher- und Nachher-Werte der Modellierungsinstanzen anzeigen, um zu gewährleisten, dass alle Hinzufügungen oder Aktualisierungen für einen Herstellungsprozess ausgeführt wurden.

Beim Filtern der Inhaltsänderungshistorie

Die Anwendung zeigt standardmäßig die gesamte Inhaltsänderungshistorie an, wenn Sie auf die Schaltfläche "Suchen" klicken. Die Anwendung enthält außerdem Filter, mit denen Sie die Liste der Instanzen eingrenzen können, die bei einem Klick auf die Schaltfläche "Suchen" zurückgegeben werden.

Mit dem Optionsfeld "Datumsbereich auswählen" können Sie die Instanzen filtern, die bei Ihrer Suche zurückgegeben werden. So können Sie die Anzeige auf die Instanzen beschränken, die zwischen den Start- und Ende-Zeitstempelwerten geändert wurden. Der Start-Zeitstempel wird durch das Erstellungsdatum für das Paket definiert, und der Ende-Zeitstempel wird durch das letzte Bereitstellungsdatum für das Paket definiert. Der Ende-Zeitstempel wird auf die aktuelle Uhrzeit und das aktuelle Datum festgelegt, wenn das Paket nicht bereitgestellt wurde.

Sie können Ihre Suche außerdem nach Objekttyp, Instanzname oder Benutzer der letzten Änderung filtern. Mit der Schaltfläche "Alles löschen" entfernen Sie alle Filter, die Sie zuvor ausgewählt haben. Die von Ihrer Suche zurückgegebenen Instanzen werden gelöscht und der Standardzustand der Seite wird wiederhergestellt.

Beim Anzeigen der Inhaltsänderungshistorie

Die Ergebnisse auf der Seite "Inhaltsänderungshistorie" werden in der Tabelle "Inhaltsänderungshistorie nach Instanz" als Verknüpfungen zum Popup "Modellierungsprotokoll" angezeigt. Sie können die Inhaltsänderungshistorie jeweils nur für eine Instanz gleichzeitig anzeigen. Der Datumsbereich der Inhaltsänderungshistorie wird an das Popup "Modellierungsprotokoll" weitergegeben. Sie können die Ergebnisse des Modellierungsprotokolls nach Paketname und benutzerdefiniertem Datumsbereich filtern.

Schaltfläche "Dokument anzeigen"

Sie können auf die Schaltfläche "Dokument anzeigen" klicken, um die Informationen, die in der Tabelle "Inhaltsänderungshistorie nach Instanz" angezeigt werden, als PDF anzuzeigen oder zu speichern. Die Schaltfläche "Dokument anzeigen" ist aktiviert, nachdem Sie in der Tabelle "Inhaltsänderungshistorie nach Instanz" nach Informationen gesucht und sie in der Tabelle angezeigt haben. Wenn Sie auf die Schaltfläche "Dokument anzeigen" klicken, wird ein Dialogfeld mit den Optionen "Öffnen", "Speichern" und "Abbrechen" in Ihrem Browser angezeigt. Klicken Sie auf "Öffnen", um die PDF in der Standardanwendung für die PDF-Anzeige anzuzeigen. Sie müssen über eine Anwendung verfügen, die PDF-Dateien anzeigen kann, z. B. Adobe Acrobat Reader. Klicken Sie auf "Speichern", um das Popup "Speichern unter" anzuzeigen und die PDF auf dem lokalen Computer zu speichern.

Seite "Inhaltsänderungshistorie"

Sie haben folgende Möglichkeiten, auf die Seite "Inhaltsänderungshistorie" zuzugreifen:

• Durch Klicken auf "Inhaltshistorie", nachdem Sie auf der Seite "Paketsuche" in der Tabelle "Suchergebnisse" ein Paket ausgewählt haben.

Oder

• Durch Klicken auf die Schaltfläche "Historie" in der Paketinformationen-Kopfzeile und Klicken auf "Inhaltsänderungshistorie" im Aufklapp-Menü.

Vorgehensweise zum Anzeigen der Inhaltsänderungshistorie

Gehen Sie folgendermaßen vor, um die Inhaltsänderungshistorie anzuzeigen:

1. Zeigen Sie auf der Seite "Inhaltsänderungshistorie" mit einer der folgenden Vorgehensweisen die Inhaltsänderungshistorie an:
• Klicken Sie auf "Suchen", um den Standarddatumsbereich zu verwenden und die gesamte Inhaltsänderungshistorie anzuzeigen.

Oder

• Geben Sie einen Datumsbereich ein und klicken Sie dann auf "Suchen", um die Ergebnisse zu filtern.

Die Tabelle "Inhaltsänderungshistorie nach Instanz" wird gefüllt.

2. Klicken Sie auf die Verknüpfung unter der Instanz-Spalte für die Inhaltsänderungshistorie, die Sie anzeigen möchten. Das Popup "Modellierungsprotokoll" wird angezeigt.
Hinweis: Die Daten von der Seite "Inhaltsänderungshistorie" werden an das Popup "Modellierungsprotokoll" übergeben. Sie können den Paketnamen eingeben und die Daten im Popup "Modellierungsprotokoll" ändern, um die Ergebnisse weiter zu filtern.
