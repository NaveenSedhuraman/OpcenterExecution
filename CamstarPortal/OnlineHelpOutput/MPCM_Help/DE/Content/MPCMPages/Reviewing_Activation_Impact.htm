Überprüfen der Aktivierungsauswirkung

Auf der Seite "Aktivierungsauswirkung" können Sie die Auswirkungen der Bereitstellung eines Änderungsverwaltungspakets auf einem Zielsystem überprüfen.

Beim Überprüfen der Aktivierungsauswirkung

Sie können die Funktion "Aktivierungsauswirkung" nur auf dem Quellsystem ausführen.

Die Seite "Aktivierungsauswirkung" enthält eine Liste der Instanzen, die hinzugefügt oder aktualisiert werden, nachdem das Paket auf dem ausgewählten Ziel aktiviert wurde. Sie müssen auf "Aktivierungsauswirkung prüfen" klicken, um die Instanzen für das ausgewählte Ziel anzuzeigen. Die Anwendung zeigt standardmäßig ein Ziel im Feld "Ausgewähltes Ziel" an, wenn nur ein Ziel definiert wurde.

Verwenden Sie das Feld "Objekttyp" zum Filtern der Liste der Instanzen nach Modellierungsobjekt. Mit der Überprüfung der Aktivierungsauswirkung wird das Paket weder aktiviert noch bereitgestellt. Die Ergebnisse werden nach Objekt sortiert.

Das Feld "Auswirkung" zeigt an, ob der Aktivierungsprozess Instanzdaten hinzufügt oder aktiviert.

Hinweis: Die Einstellungen zur Überschreibung auf der Seite "Paket aktivieren" ermöglichen Ihnen anzugeben, ob und wie Modellierungsobjekt-Instanzen auf dem Zielsystem von Modellierungsobjekt-Instanzen im Änderungsverwaltungspaket überschrieben werden.

Rufen Sie die Seite "Aktivierungsauswirkung" über das Aufklapp-Menü "Inhalt" in der Paketinformationen-Kopfzeile oder über die Transaktionsschaltfläche "Aktivierungsauswirkung" auf.

Klicken Sie auf die Schaltfläche "Daten exportieren" in der Statusleiste der Tabelle, um das Popup "Daten exportieren" einzublenden und Aktivierungsauswirkungsdaten zu exportieren.

Vorgehensweise zum Überprüfen der Aktivierungsauswirkung für ein Paket

Gehen Sie folgendermaßen vor, um die Aktivierungsauswirkung für ein Paket anzuzeigen:

1. Wählen Sie Paketsuche im Menü Änderungsverwaltung aus. Die Seite "Paketsuche" wird angezeigt.
2. Geben Sie ein oder mehrere Kriterien ein, um Ihre Suche einzugrenzen.
3. Klicken Sie auf Suchen. Die Anwendung zeigt die Pakete in der Tabelle "Suchergebnisse" an, die Ihren Suchkriterien entsprechen.
4. Wählen Sie das Paket aus, das Sie überprüfen möchten. Die Anwendung zeigt die relevanten Transaktionen für das Paket an.
5. Klicken Sie auf Aktivierungsauswirkung.
6. Wählen Sie ein Ziel aus und klicken Sie auf Aktivierungsauswirkung prüfen. Die Anwendung füllt die Tabelle mit Daten.
