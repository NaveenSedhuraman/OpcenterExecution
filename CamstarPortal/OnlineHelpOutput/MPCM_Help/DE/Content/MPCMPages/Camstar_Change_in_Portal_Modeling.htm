Änderungsverwaltung in der Modellierung

Weitere Funktionen sind in der Modellierung verfügbar, wenn Sie eine Lizenz für die Änderungsverwaltung besitzen, mindestens ein Paket für die Änderungsverwaltung erstellt haben und über die entsprechende Rolle und die entsprechenden Berechtigungen verfügen. Sie können Modellierungsobjektinstanzen erstellen und aktualisieren und diese in der Modellierung zu vorhandenen, aktiven Paketen hinzufügen.

Auf einer weiteren Registerkarte auf jeder Objektseite können Sie Informationen über die Anzahl der einer Instanz zugeordneten Pakete anzeigen sowie darüber, ob die Instanz von einem Paket gesperrt ist. Über eine weitere Schaltfläche in der Symbolleiste auf der Seite "Modellierung" können Sie Paketzuordnungen aus der Modellierung anzeigen. Sie können auch Paketzuordnungen in der Modellierung hinzufügen oder aus dieser entfernen, wenn Sie über Paketeigentümerberechtigungen verfügen.

Die folgenden Themen enthalten Informationen zur Verwendung der Änderungsverwaltung in der Modellierung:

• Zuweisen von Modellierungsobjekt-Instanzen zu einem Paket in der Modellierung
• Anzeigen von Informationen zur Änderungsverwaltung für eine Modellierungsobjekt-Instanz
• Verwalten von Paketzuordnungen in der Modellierung
