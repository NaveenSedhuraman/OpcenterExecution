Genehmigen eines Pakets (Camstar)

Sie können Pakete, die dem Camstar-Arbeitsplan zur Änderungsverwaltung zugewiesen wurden, genehmigen, wenn sie den Schritt "Genehmigung ausstehend Camstar" (Pending Approval Camstar) haben.

Bei der Genehmigung eines Pakets auf dem Camstar-Arbeitsplan

Sie müssen die Rolle des Paketgenehmigers haben.

Die Seite "Paket genehmigen (Camstar)" hat zwei Registerkarten:

• Genehmigen

Die Registerkarte "Genehmigen" bietet Ihnen die Möglichkeit, ein Änderungsverwaltungspaket zu genehmigen. Die Anwendung speichert Ihre Entscheidung und bewegt das Paket zum nächsten entsprechenden Schritt, wenn Sie der letzte Genehmiger sind.

• Genehmigungsstatus

Die Registerkarte "Genehmigungsstatus" bietet Ihnen die Möglichkeit, den Status der aktuellen und vergangenen Genehmigungszyklen und die Entscheidungen aller Paketgenehmiger in jedem Zyklus anzuzeigen.

Beim Aufzeichnen von Genehmigungsentscheidungen

Die Anwendung verschiebt das Änderungsverwaltungspaket von "Genehmigung ausstehend Camstar" (Pending Approval Camstar) nach "Bereitstellung ausstehend" (Pending Deployment), wenn die Genehmigungsentscheidung "Genehmigen" lautet. Die Anwendung verschiebt das Änderungsverwaltungspaket von "Genehmigung ausstehend Camstar" (Pending Approval Camstar) nach "Abgelehnt" (Rejected), wenn die Genehmigungsentscheidung "Ablehnen" lautet. Es ist nur eine Genehmigungsentscheidung des Typs "Ablehnen" erforderlich, damit die Anwendung das Paket in den Schritt "Abgelehnt" verschiebt. Allerdings muss die Anwendung alle Genehmigungsentscheidungen empfangen haben, bevor sie das Paket zum nächsten Schritt bewegt.

Vorgehensweise zum Genehmigen eines Änderungsverwaltungspakets

Gehen Sie folgendermaßen vor, um ein Änderungsverwaltungspaket zu genehmigen:

1. Wählen Sie Paketsuche im Menü Änderungsverwaltung aus. Die Seite "Paketsuche" wird angezeigt.
2. Geben Sie ein oder mehrere Kriterien ein, um Ihre Suche einzugrenzen.
3. Klicken Sie auf Suchen. Die Anwendung zeigt die Pakete in der Tabelle "Suchergebnisse" an, die Ihren Suchkriterien entsprechen.
4. Wählen Sie das Paket aus, das Sie genehmigen möchten. Die Anwendung zeigt die relevanten Transaktionen für das Paket an.
5. Klicken Sie auf Genehmigen (Camstar). Die Seite "Paket genehmigen (Camstar)" wird angezeigt.
6. Wählen Sie Ihre Rolle aus.
7. Wählen Sie Ihre Benutzer-ID im Feld Genehmigung für aus.
8. Wählen Sie eine Genehmigungsentscheidung aus.
9. Geben Sie Kommentare oder Hinweise über die Genehmigungsentscheidung in das Feld Kommentare ein, falls notwendig.
10. Klicken Sie auf Entscheidung speichern. Die Anwendung zeigt eine Meldung an, dass Sie die Genehmigung abgeschlossen haben.

Vorgehensweise zum Anzeigen des Genehmigungsstatus oder der Genehmigungserfassungshistorie

Gehen Sie folgendermaßen vor, um den Genehmigungsstatus oder die Genehmigungserfassungshistorie anzuzeigen:

1. Wählen Sie Paketsuche im Menü Änderungsverwaltung aus. Die Seite "Paketsuche" wird angezeigt.
2. Geben Sie ein oder mehrere Kriterien ein, um Ihre Suche einzugrenzen.
3. Klicken Sie auf Suchen. Die Anwendung zeigt die Pakete in der Tabelle "Suchergebnisse" an, die Ihren Suchkriterien entsprechen.
4. Wählen Sie das Paket aus, für das Sie den Genehmigungsstatus ändern möchten. Die Anwendung zeigt die relevanten Transaktionen für das Paket an.
5. Klicken Sie auf Genehmigen (Camstar). Die Seite "Paket genehmigen (Camstar)" wird angezeigt.
6. Klicken Sie auf die Registerkarte Genehmigungsstatus. Die Anwendung zeigt aktuelle Genehmigungs-Informationen an.
7. Möchten Sie den Genehmigungsstatus oder die Genehmigungserfassungshistorie anzeigen?
