Suchen nach mehreren Paketen, die auf dem Zielsystem geschlossen, ungültig gemacht oder geöffnet werden sollen

Die Anwendung enthält eine Funktion, mit der Sie mehrere Änderungsverwaltungspakete, die auf dem Zielsystem geschlossen, ungültig gemacht oder geschlossen werden sollen, anhand angegebener Kriterien suchen und abrufen können. Sie können auch ein einzelnes Paket auf der Seite "Aktivierungssuche (mehrere)" auswählen, um es zu schließen, ungültig zu machen oder zu öffnen. Die Funktion "Aktivierungssuche (mehrere)" ist für Pakete verfügbar, die bereitgestellt wurden und einen für die Aktivierung zulässigen Status haben. Der Mitarbeiter, der auf dieser Seite nach Paketen sucht, muss über eine Rolle mit Aktivierungsberechtigungen oder über die Standardrolle "Paketaktivierer" verfügen. Rollen werden Mitarbeitern in der Modellierung zugewiesen.

Beim Suchen nach mehreren Paketen, die auf dem Zielsystem geschlossen, ungültig gemacht oder geöffnet werden sollen

Wenn Sie erstmals auf die Seite "Aktivierungssuche (mehrere)" zugreifen, zeigt die Anwendung die zum Filtern verfügbaren Felder und eine leere Tabelle mit Suchergebnissen an.

Beim Suchen nach Paketen:

• Sie können mithilfe von SQL-Platzhaltern nach Paketnamen suchen. Wenn Sie z. B. "%01%" eingeben, werden Pakete mit Namen ausgegeben, die die Zeichen "01" enthalten.
• Sie können nach Aktivierungsstatus suchen. Es gibt drei Standardpaketstatus. Die Anwendung wählt standardmäßig "Aktivierung unvollständig" und "Ausstehende Aktivierung" aus. Der Status "Aktiviert" ist nicht ausgewählt. Das Feld "Aktivierungsstatus" zeigt eine Zahl anstatt der ausgewählten Status an. Die Anwendung zeigt "2 von 3 ausgewählt" im Feld "Aktivierungsstatus" an, wenn die Seite geladen wird.
Hinweis: Bei der Suche nach einem Paket, das vor kurzem zu aktiviert wurde, können unerwartete Ergebnisse zurückgegeben werden. Der Aktivierungsstatus wird standardmäßig alle 5 Minuten aktualisiert. Sie können diese Einstellung in der Modellierung ändern, indem Sie die Standardgeschäftsregel "SBR_Activation" anpassen. Weitere Informationen dazu finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch oder im Opcenter Execution Core Modeling-Benutzerhandbuch.
• Sie können nach Paketstatus suchen. Es sind drei Status verfügbar. Die Anwendung wählt standardmäßig "Offen" aus. "Geschlossen" und "Ungültig gemacht" werden nicht ausgewählt. Die Anwendung zeigt eine Zahl anstatt der ausgewählten Status an. Die Anwendung zeigt "1 von 3 ausgewählt" im Feld "Paketstatus" an, wenn die Seite geladen wird.
Hinweis: Sie können die Paketdetails für ein geschlossenes oder ungültig gemachtes Paket anzeigen, wenn Sie den Status "Geschlossen" oder "Ungültig gemacht" auswählen.
• Wenn Sie keine Filter angeben, wird jedes Änderungsverwaltungspaket, für das Sie eine Zugriffsberechtigung haben, in den Ergebnissen angezeigt.
• Sie können Ihre Suche mit bis zu sieben Filtern eingrenzen.

Durchführen einer Suche

Wenn Sie eine Suche ausführen, macht die Anwendung Folgendes:

• Sie zeigt die Suchergebnisse in einer Tabelle an
• Sie behält die Suchkriterien in den Filterfeldern bei

Die Anwendung fügt der Tabelle eine Umblätter-Funktion hinzu, wenn die Suche mehr Ergebnisse zurückgibt, als in den Zeilen der Tabelle angezeigt werden können. Die Anwendung hebt alle Pakete hervor, die sich im Schritt "Bereitstellung unvollständig" (Deployment Incomplete) oder "Abgelehnt" (Rejected) befinden.

Wenn die Anwendung Pakete zurückgibt, die Ihren Kriterien entsprechen, können Sie die Pakete auswählen, für die Sie eine Transaktion durchführen möchten. Sie können mehrere Pakete in der Tabelle auswählen, indem Sie das Kontrollkästchen in der Kopfzeile der Tabelle oder die Kontrollkästchen für einzelne Pakete aktivieren.

Siemens bietet auch die Möglichkeit, Suchergebnisse in eine Excel-Datei oder im CSV-Format zu exportieren. Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch oder im Opcenter Execution Core Modeling-Benutzerhandbuch.

Löschen von Suchfiltern und Ergebnissen

Verwenden Sie die Schaltfläche "Alles löschen", um Filter und Ergebnisse zu löschen. Wenn Sie die Seite "Aktivierungssuche (mehrere)" löschen, können Sie eine neue Suche starten. Wenn Sie auf die Schaltfläche "Alles löschen" klicken, reagiert die Anwendung folgendermaßen:

• Löscht alle angegebenen Suchfilter
• Löscht alle Pakete aus der Tabelle "Suchergebnisse"
• Löscht alle Transaktionsschaltflächen
• Die Seite "Aktivierungssuche (mehrere)" wird weiterhin angezeigt.

Verwenden der Transaktionsschaltflächen

Sie müssen mindestens ein Paket auswählen und auf die Schaltfläche "Aktionen laden" klicken, um die Transaktionsschaltflächen zu aktivieren. Wenn Sie mehr als ein Paket ausgewählt haben, zeigt die Anwendung die Transaktionen an, die gleichzeitig für die ausgewählten Pakete durchgeführt werden können. Die einzigen Transaktionen, die Sie für mehrere Pakete durchführen können, sind "Schließen", "Ungültig machen" und "Öffnen". Es kann z. B. wünschenswert sein, alle Pakete mit dem Status "Aktivierung unvollständig" zu suchen und sie gleichzeitig zu schließen.

Die Anwendung aktiviert die Transaktionsschaltflächen "Paket schließen" und "Paket ungültig machen" abhängig vom Paketstatus und Aktivierungsstatus der ausgewählten Pakete.

Zusätzlich zu "Auswahl schließen", "Auswahl ungültig machen" und "Auswahl öffnen" wird die Transaktionsschaltfläche "Paketdetails" angezeigt, wenn Sie in der Tabelle "Suchergebnisse" nur ein Paket auswählen. Sie können auch die Paketdetails für ein geschlossenes oder ungültig gemachtes Paket anzeigen, wenn Sie den Status "Geschlossen" oder "Ungültig gemacht" auswählen.

Wenn Sie ein Paket auswählen und auf eine Transaktionsschaltfläche klicken, zeigt die Anwendung abhängig davon, ob Sie Pakete schließen, ungültig machen oder öffnen, das jeweils entsprechende Popup an. Sie können relevante Kommentare eingeben.

Die Anwendung aktualisiert den Paketstatus jedes Pakets, nachdem Sie es geschlossen, ungültig gemacht oder geöffnet haben. Pakete, die geschlossen oder ungültig gemacht wurden, werden in der Tabelle "Suchergebnisse" erst dann wieder angezeigt, wenn Sie "Geschlossen" oder "Ungültig gemacht" im Feld "Paketstatus" auswählen.

Vorgehensweise zum Suchen nach mehreren Paketen, die auf dem Zielsystem geschlossen, ungültig gemacht oder geöffnet werden sollen

Gehen Sie folgendermaßen vor, um nach mehreren Paketen zu suchen, die auf dem Zielsystem geschlossen, ungültig gemacht oder geöffnet werden sollen:

1. Wählen Sie Aktivierungssuche (mehrere) im Menü Änderungsverwaltung aus. Die Seite "Aktivierungssuche (mehrere)" wird angezeigt.
2. Geben Sie ein oder mehrere Kriterien ein, um Ihre Suche einzugrenzen.
3. Klicken Sie auf Suchen. Die Anwendung zeigt die Pakete in der Tabelle "Suchergebnisse" an, die Ihren Suchkriterien entsprechen.

Vorgehensweise zum Ausführen einer Transaktion für mehrere Pakete auf dem Zielsystem

Gehen Sie folgendermaßen vor, um eine Transaktion für mehrere Pakete auf dem Zielsystem auszuführen:

1. Führen Sie die Schritte unter "Vorgehensweise zum Suchen nach mehreren Paketen, die auf dem Zielsystem geschlossen, ungültig gemacht oder geöffnet werden sollen" durch.
2. Aktivieren Sie das Kontrollkästchen in der Kopfzeile der Tabelle "Suchergebnisse", um alle angezeigten Pakete auszuwählen.

Oder

Aktivieren Sie die Kontrollkästchen für einzelne Pakete.

3. Klicken Sie auf Aktionen laden. Die Anwendung aktiviert die relevanten Transaktionen für die ausgewählten Pakete.
4. Klicken Sie auf eine Transaktionsschaltfläche. Die Anwendung zeigt das entsprechende Popup für die ausgewählte Transaktion an.
5. Vervollständigen Sie die Informationen im Popup.
