Suchen nach einem Paket

Die Anwendung enthält eine Funktion, die es Ihnen ermöglicht, anhand angegebener Kriterien nach einem Änderungsverwaltungspaket zu suchen und es abzurufen. Sie können über die Mitarbeiterzuweisung oder über allgemeine Kriterien suchen. Die Funktion "Paketsuche" ist für Pakete in den folgenden Standardschritten verfügbar:

• Entwurf (standardmäßiger Arbeitsplan "Keine Genehmigung" zur Änderungsverwaltung)
• Entwurf Camstar (Draft Camstar) (standardmäßiger Arbeitsplan "Camstar" zur Änderungsverwaltung)
• Entwurf PLM (Draft PLM) (standardmäßiger Arbeitsplan "PLM" zur Änderungsverwaltung)
• Genehmigung ausstehend Camstar (Pending Approval Camstar) (standardmäßiger Camstar-Arbeitsplan zur Änderungsverwaltung)
• Pending Approval PLM (Genehmigung ausstehend PLM) (standardmäßiger PLM-Arbeitsplan für die Änderungsverwaltung und MOM PLM-Standardarbeitsplan)
• Abgelehnt (Rejected)
• Bereitstellung ausstehend (Pending Deployment)
• Bereitstellung beendet (Deployment Complete)
• Bereitstellung unvollständig (Deployment Incomplete)
Hinweis: Zusätzliche Schritte können durch einen Systemadministrator konfiguriert werden.

Die Anwendung zeigt zwei Registerkarten und die Tabelle "Suchergebnisse" an, wenn Sie auf die Seite "Paketsuche" zugreifen. Die beiden Registerkarten sind "Mitarbeiterzuordnungssuche" und "Allgemeine Suche". Klicken Sie auf die Schaltfläche "Suchen", um die Tabelle "Suchergebnisse" zu füllen. Zunächst listet die Tabelle alle Pakete auf, für die der aktuelle Benutzer der Paketeigentümer, Paketgenehmiger oder Paketprojektmitarbeiter ist. Standardmäßig enthält die Tabelle keine Pakete, für die der aktuelle Benutzer Genehmigungen erteilt hat.

Jede Registerkarte zeigt mehrere Felder an, nach denen Sie die Suche eingrenzen können. Die Anwendung fügt ein Filtersymbol zu jeder Registerkarte hinzu, um anzugeben, dass Filterkriterien festgelegt sind. Das Filter-Icon wird auf der Registerkarte "Mitarbeiterzuordnungssuche" automatisch angezeigt, da mehrere Felder auf dieser Registerkarte standardmäßig ausgewählt sind. Das Filter-Icon wird auf der Registerkarte "Allgemeine Suche" angezeigt, wenn Sie Kriterien auswählen. Die Anwendung zeigt die Ergebnisse der kombinierten Suchkriterien aus den zwei Registerkarten in der Tabelle "Suchergebnisse" an.

Wenn die Anwendung Änderungsverwaltungspakete ausgibt, die Ihren Kriterien entsprechen, können Sie das Paket auswählen, an dem Sie eine Transaktion durchführen möchten. Die Anwendung aktiviert nur jene Transaktionen, die für das ausgewählte Paket relevant sind.

Hinweis: Informationen zum Auswählen und gleichzeitigen Schließen, Öffnen oder Ungültigmachen mehrerer Pakete finden Sie unter "Suchen nach mehreren Paketen, die geschlossen, ungültig gemacht oder geöffnet werden sollen".

Beim Suchen nach einem Paket mithilfe von Mitarbeiter-Zuweisungskriterien

Die Registerkarte "Mitarbeiterzuordnungssuche" ist standardmäßig ausgewählt, wenn Sie erstmals auf die Seite "Paketsuche" zugreifen. Mehrere Felder ermöglichen Ihnen, Ihre Suche zu filtern.

Beim Suchen nach Paketen mithilfe der Felder auf dieser Registerkarte:

• Der Eintrag im Feld "Mitarbeiter" enthält standardmäßig den aktuellen Benutzer.
• Sie können nach Mitarbeiter-Paketzuweisungen suchen.
• Sie können nach Ersatzrollen suchen.
• Sie können angeben, ob fertiggestellte Zuweisungen ausgeblendet werden sollen.
• Die Felder "Ist Eigentümer", "Ist Genehmiger", "Ist Projektmitarbeiter", "Hat Eigentümerrolle", "Hat Genehmigerrolle" und "Hat Projektmitarbeiterrolle" sind standardmäßig ausgewählt.
• Die Kontrollkästchen unter "Paketzuweisungen" und "Ersetzungsrolle" fungieren als "oder"-Filter. Wenn irgendwelche der übereinstimmenden Kontrollkästchen-Attribute wahr sind, werden die übereinstimmenden Pakete angezeigt.
• Die Kontrollkästchen unter "Abgeschlossene Zuweisungen ausblenden" fungieren als "und"-Filter und können einige der Pakete ausblenden, die von den anderen Kontrollkästchen zurückgegeben werden.

Beim Suchen nach einem Paket mithilfe allgemeiner Kriterien

Sie können Ihre Suche weiter mithilfe von Filtern auf der Registerkarte "Allgemeine Suche" eingrenzen.

Beim Suchen nach Paketen mithilfe der Felder auf dieser Registerkarte:

• Sie können mithilfe von SQL-Platzhaltern nach Paketnamen suchen. Wenn Sie z. B. "%01%" eingeben, werden Pakete mit Namen ausgegeben, die die Zeichen "01" enthalten.
• Können Sie anhand eines Arbeitsplans zur Änderungsverwaltung suchen. Sie können immer nur einen Arbeitsplan gleichzeitig auswählen.
• Sie können anhand eines oder mehrerer Änderungsverwaltungsschritte suchen. Die verfügbaren Schritte hängen vom ausgewählten Arbeitsplan zur Änderungsverwaltung ab. Sie können mehrere Schritte auswählen. Das Feld "Änderungsmanagement Schritt" zeigt eine Zahl anstatt der ausgewählten Schritte an. Wenn Sie z. B. drei der fünf verfügbaren Schritte auswählen, zeigt die Anwendung "3 von 5 ausgewählt" im Feld "Änderungsmanagement-Schritt" an.
• Es gibt fünf Standardpaketstatus. "Offen", "Bereitgestellt" und "Abgelehnt" sind standardmäßig ausgewählt. "Ungültig gemacht" und "Geschlossen" sind nicht ausgewählt. Das Feld "Paketstatus" zeigt eine Zahl anstatt der ausgewählten Aktivierungsstatus an. Die Anwendung zeigt "3 von 5 ausgewählt" im Feld "Paketstatus" an, wenn die Seite geladen wird.
Hinweis: Mehrere in der Auswahlliste "Paketstatus" ausgewählte Werte fungieren als "oder"-Filter und erweitern die Liste der zurückgegebenen Pakete.
• Die übrigen "Allgemeine Suche"-Filter fungieren als "und"-Filter und können einige der zurückgegebenen Pakete ausblenden.

Durchführen einer Suche

Wenn Sie eine Suche ausführen, macht die Anwendung Folgendes:

• Sie zeigt die Suchergebnisse in einer Tabelle an
• Sie behält die Suchkriterien in den Filterfeldern bei

Die Anwendung fügt der Tabelle eine Umblätter-Funktion hinzu, wenn die Suche mehr Ergebnisse zurückgibt, als in den Zeilen der Tabelle angezeigt werden können. Die Anwendung hebt alle Pakete hervor, die sich im Schritt "Bereitstellung unvollständig" oder "Abgelehnt" befinden. Wenn Sie in der Tabelle "Suchergebnisse" ein Paket auswählen, erlaubt die Anwendung die Transaktionen, die für das Paket relevant sind. Siemens bietet auch die Möglichkeit, Suchergebnisse in eine Excel-Datei oder im CSV-Format zu exportieren. Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch oder im Opcenter Execution Core Modeling-Benutzerhandbuch.

Löschen von Suchfiltern und Ergebnissen

Verwenden Sie die Schaltfläche "Alles löschen", um Filter und Ergebnisse zu löschen. Das Löschen der Seite "Paketsuche" ermöglicht es Ihnen, eine neue Suche zu starten. Wenn Sie auf die Schaltfläche "Alles löschen" klicken, reagiert die Anwendung folgendermaßen:

• Löscht alle angegebenen Suchfilter auf beiden Registerkarten
• Löscht das Filter-Icon aus den Registerkarten der Suche
• Löscht alle Pakete aus der Tabelle "Suchergebnisse"
• Löscht alle Transaktionsschaltflächen
• Zeigt weiterhin die Seite "Paketsuche" an

Verwenden der Transaktionsschaltflächen

Die Transaktionsschaltflächen ermöglichen es Ihnen, Transaktionen an in der Tabelle "Suchergebnisse" ausgewählten Paketen durchzuführen. Die Anwendung aktiviert nur gültige Transaktionen für das Paket. Wenn Sie ein anderes Paket in der Tabelle auswählen, ändern sich die verfügbaren Transaktionen entsprechend.

Wenn Sie ein Paket auswählen und auf eine Transaktionsschaltfläche klicken, zeigt die Anwendung abhängig vom Typ der ausgewählten Transaktion die entsprechende Transaktionsseite oder das entsprechende Popup an.

Wenn Sie ein Paket schließen, ungültig machen oder öffnen, wird ein Popup angezeigt. Sie können relevante Kommentare eingeben.

Hinweis: Sie können ein Paket öffnen, das zuvor auf der Seite "Paketsuche" oder "Paketsuche (mehrere)" geschlossen oder ungültig gemacht wurde.

Wenn Sie eine der anderen Transaktionen durchführen, wird eine Transaktionsseite angezeigt. Die Anwendung trägt den Namen des in der Tabelle ausgewählten Pakets in das Feld "Ausgewähltes Paket" (sofern verfügbar) auf der Seite ein. Wenn Sie z. B. ein Paket auswählen und auf die Schaltfläche "Schritt ändern" klicken, wird die Seite "Paketschritt ändern" mit gefülltem Feld "Ausgewähltes Paket" angezeigt.
