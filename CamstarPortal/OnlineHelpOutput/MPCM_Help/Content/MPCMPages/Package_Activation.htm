Activating Packages

Change management enables you to perform these tasks:

• Search for activated packages using filters and then perform package activations on the target system
• Search for multiple packages to close, void, or open on the target system
• Activate a package on the target system

You must have roles with appropriate permissions assigned to use each of these options.

These topics provide information on managing packages on the target system:

• Searching for a Package to Activate
• Searching for Multiple Packages to Close, Void, or Open on the Target System
• Activating a Package
