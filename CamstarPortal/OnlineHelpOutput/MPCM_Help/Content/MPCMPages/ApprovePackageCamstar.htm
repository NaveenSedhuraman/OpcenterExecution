Approving a Package (Camstar)

You can approve packages assigned to the Camstar change management workflow when they are at the Pending Approval Camstar step.

When Approving a Package on the Camstar Workflow

You must have the package approver role.

The Approve Package (Camstar) page has two tabs:

• Approve

The Approve tab provides the ability for you to approve a change management package. The application records your decision and moves the package to the next appropriate step if you are the last approver.

• Approval Status

The Approval Status tab provides the ability for you to view the status of current and past approval cycles and to view the decisions of all package approvers in each cycle.

When Recording Approval Decisions

The application moves the change management package from Pending Approval Camstar to Pending Deployment when the approval decision is Approve. The application moves the package from Pending Approval Camstar to Rejected when the approval decision is Reject. Only one approval decision of Reject is necessary for the application to move the package to the Rejected step. However, all approval decisions must be received before the application moves the package to the next step.

How to Approve a Change Management Package

Follow these steps to approve a change management package:

1. Select Package Search from the Change Management menu. The Package Search page appears.
2. Enter or select one or more criteria to narrow your search.
3. Click Search. The application displays the packages matching your search criteria in the Search Results grid.
4. Select the package you want to approve. The application displays the relevant transactions for the package.
5. Click Approve (Camstar). The Approve Package (Camstar) page appears.
6. Select your Role.
7. Select your user ID from the Approval for field.
8. Select an Approval Decision.
9. Enter any comments or notes about the approval decision in the Comments field if required.
10. Click Save Decision. The application displays a message stating you have completed the approval.

How to View Approval Status or Approval Capture History

Follow these steps to view approval status or approval capture history:

1. Select Package Search from the Change Management menu. The Package Search page appears.
2. Enter or select one or more criteria to narrow your search.
3. Click Search. The application displays the packages matching your search criteria in the Search Results grid.
4. Select the package for which you want to view the approval status. The application displays the relevant transactions for the package.
5. Click Approve (Camstar). The Approve Package (Camstar) page appears.
6. Click the Approval Status tab. The application displays current approval information.
