Voiding Multiple Packages

You can void multiple packages that have a status of Open, Deployed, or Rejected on the Package Search (Multiple) page if you have package ownership permissions. You typically void packages before deployment occurs.

You cannot update the contents of packages or deploy them after you void them. You cannot void a package that is in a Voided or Closed status.

You can void multiple packages on the Activation Search (Multiple) page that have an Open package status, and an activation status of either Pending Activation or Activation Incomplete.

When Voiding Multiple Packages

The Void Packages pop-up appears when you select one or more packages in the Search Results grid on the Package Search (Multiple) or Activation Search (Multiple) page and select Void Selected. You can enter any relevant comments in the Comments field using a maximum of 255 characters and then click Void.

The application performs the following when you click Void:

• Voids packages and places them in Voided status
• Closes the Void Packages pop-up
• Displays a success message indicating the following:
• Name of the first selected package listed in the Search Results grid
• Date and time at which the package was voided
• Name of the employee who voided it
Note: The application voids all selected packages, but displays only the name of the first package in the success message.
• Reloads the Search Results grid with the current search criteria and returns you to the same page in the grid
• Records the following information in history:
• Name of the employee who voided the packages
• Date and time at which the application voided the packages
• Any comments you entered in the Void Packages pop-up

How to Void Multiple Packages

Follow these steps to void multiple packages:

1. Are you voiding packages on the source system or the target system?

If . . .

Then . . .

On the source system

Select Package Search (Multiple) from the Change Management menu. The Package Search (Multiple) page appears.

On the target system

Select Activation Search (Multiple) from the Change Management menu.The Activation Search (Multiple) page appears.
2. Enter or select one or more criteria to narrow your search.
3. Click Search. The application displays the packages matching your search criteria in the Search Results grid.
4. Select the check box in the header row in the Search Results grid to select all displayed packages.

Or

Select the check boxes for individual packages.

5. Click Load Actions. The application enables relevant transactions for the selected packages.
6. Click Void Selected. The Void Packages pop-up appears.
7. Enter any relevant comments in the Comments field.
8. Click Void. The application voids the packages, closes the Void Packages pop-up, displays a success message, reloads the Search Results grid, and records pertinent information in history.
