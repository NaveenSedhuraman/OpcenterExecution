Altering the Step of a Package

The Alter Package Step transaction enables you to move a change management package to any step in the package's assigned workflow.

When Altering the Step of a Package

The workflow diagram appears dynamically based on the workflow associated with the package specified in the Selected Package field. Workflows are associated with packages during package creation. Refer to "Creating a Package" for information.

The workflow diagram contains all the steps in the workflow, including steps with no incoming or outgoing paths. A red border appears around the package's current step. Hovering over a step displays a tooltip containing a description of the step if a description was entered during spec creation.

Note: A package's step can only be altered to another step within the same workflow.

You can navigate to the Alter Package Step page from the Package Search page by selecting a package and then clicking the Alter Step button under Available Transactions. The application populates the Selected Package field with the package selected on the Package Search page automatically.

How to Alter the Step of a Package

Follow these steps to alter the step of a package:

1. Select Package Search from the Change Management menu. The Package Search page appears.
2. Enter or select one or more criteria to narrow your search.
3. Click Search. The application displays the packages matching your search criteria in the Search Results grid.
4. Select the package for which you want to alter the step. The application displays the relevant transactions for the package.
5. Click Alter Step. The Alter Package Step page appears.
6. Select a package step from the Select New Package Step field.
7. Enter optional information according to your business requirements. Refer to the field definitions table for information about the optional fields.
8. Click Alter Package Step. The application updates the change management package, the workflow diagram, package step image, and displays a success message.
