Canceling an Approval

You can cancel the approval process for a change management package regardless of whether approvers have entered decisions. You can cancel a package approval when these conditions are met:

• The package step is Pending Approval Camstar or Pending Approval PLM, and
• You are the package owner or have ownership permissions.

When Canceling a Package Approval

The Cancel Approval page displays only the core attributes of the selected package. The application enables the Cancel Approval button when you select a package.

Canceling an approval returns the package to an editable step and resets the approval process. For example, the application will return packages at the Pending Approval Camstar step in the Camstar change management workflow to the Draft Camstar step . You can then alter package content or modify the approval process. You must then re-route the package for approval after making changes. Any approvers who approved the package before the cancellation will need to re-approve the package.

You can view a history of canceled approvals and approver decisions on the Package Details page and on the Approval Status tab of the Approve Package page. Refer to "Viewing Package Details" and "Approving a Package (Camstar)" for information.

The Cancel Approval Comments field is available for relevant comments about the package approval cancellation. The application truncates any entry exceeding 255 characters.

How to Cancel an Approval

Follow these steps to cancel an approval:

1. Select Package Search from the Change Management menu. The Package Search page appears.
2. Enter or select one or more criteria to narrow your search.
3. Click Search. The application displays the packages matching your search criteria in the Search Results grid.
4. Select the package for which you want to cancel the approval process. The application displays the relevant transactions for the package.
5. Click Cancel Approval. The Cancel Approval page appears.
6. Click Cancel Approval. The application displays a message stating the approval process has been canceled.
