Viewing Change Management Information for a Modeling Object Instance

You can view information about the number of packages associated with an instance and whether the instance is locked by a package by accessing an additional tab on each object's page.

When Viewing Change Management Information for a Modeling Object Instance

Every modeling object page contains basic information about the instance you are defining. This information appears in the header section of the page. An additional tab labeled Change Mgt appears in this section if you have licensed change management, have at least one package defined, and have a role with the Change Package Modeling Inquiry permission.

Change Mgt Tab Field Definitions

This table defines the fields on the Change Mgt tab.

Field

Definition

Type

Associated Packages

Number of packages with which the instance is associated. Count does not include any closed or voided packages with which the package was previously associated.

Display Only

Instance Locked by C/M

Indicates whether the instance is locked by a change management package. If Yes, you cannot edit the instance until all packages with which it is associated are at a step where the instance is unlocked. If you are using the default steps, this occurs when the associated packages are closed, voided, or returned to a draft step.

Display Only

How to View Change Management Information for a Modeling Object Instance

Follow these steps to view change management information for a modeling object instance:

1. Select Modeling from the Modeling menu. The Modeling page appears.
2. Click the All Maint arrow on the Modeling page to see a list of modeling objects under that group.

Or

Select View All (Alphabetical) from the View Objects list on the Modeling page to display a list of modeling objects in alphabetical order. A list of all modeling objects appears in alphabetical order.

3. Select a modeling object. The object's page appears within the Modeling page.
4. Select a modeling object instance. Fields for the instance appear. The Change Mgt tab appears beside the Description and Notes tabs.
