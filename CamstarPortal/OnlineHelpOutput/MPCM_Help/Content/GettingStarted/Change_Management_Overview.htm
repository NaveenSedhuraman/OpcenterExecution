Change Management Overview

Change management is a manufacturing process to help manufacturers manage the lifecycle of changes to their product and process definitions.

Change management enables an organization to export modeling objects from one system and to then import those objects on another system. For example, an organization might export modeling objects from their test system and import those objects on their production system.

You begin the change management process by creating a change management package. A change management package is a collection of all the modeling object instances and references you want to deploy from a source system to one or more target systems. You must specify an owner for the package, the reason for creating the package, and a change management workflow when creating a package.
