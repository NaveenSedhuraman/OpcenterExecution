Viewing Documents

The Document Set icon enables you to view any documents attached to the current package. Documents provide additional information related to the package such as images, instructions, and procedures.

When Viewing Documents

You access the Document Set icon from the package information header at the top of change management package task pages.

Clicking the Document Set icon displays a list of the documents associated with the package. You can select and view any file from the document set. The application displays a message asking whether you want to open or save the file when you select a document.

You must have the appropriate software for the type of document you are trying to open. For example, you must have Microsoft Word to view a Word document.

Default browser security settings prevent documents on unsecured websites from opening by default. You must change your settings to allow unsecured content.

How to View Documents

Follow these steps to view documents:

1. Select Package Search from the Change Management menu. The Package Search page appears.
2. Enter or select one or more criteria to narrow your search.
3. Click Search. The application displays the packages matching your search criteria in the Search Results grid.
4. Select the package you want to view. The application displays the relevant transactions for the package.
5. Click the transaction you need. The applicable page appears.
6. Click the Document Set icon in the package information header at the top of the page. The Document Set menu appears.
7. Click the document you want to view. A message appears asking whether you want to open the document, save it, or cancel the request.

If you click . . .

Then . . .

Open

The document appears in a new window.

Save

The application provides these options:

• Save: To download the document to the C:\ folder.
• Save as: To download the document to the C:\ folder and assign a new name.
• Save and open: To download the document to the C:\ folder and open it in a new window.

Cancel

The message closes.
