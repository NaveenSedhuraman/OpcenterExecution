Attaching Documents

You can attach one or more documents to a package. Documents provide relevant information related to the package such as images, instructions, and procedures.

When Attaching a Document to a Package

You access the Attach Document icon from the package information header at the top of change management package task pages. The Attach Document pop-up appears when you click the icon.

Consider the following when attaching documents:

• You must define document instances in Modeling using the Document modeling object before attaching them to a package. You can define a local file to upload to the database or an HTTP file, which provides a hyperlink to the file. Refer to the Opcenter Execution Medical Device and Diagnostics Modeling User Guide or the Opcenter Execution Core Modeling User Guide for information.
• You can only attach existing documents.
• You can view an existing document before attaching it. You must have the appropriate software to view the document. For example, you must have Microsoft Word to view a Word document.

How to Attach a Document

Follow these steps to attach a document:

1. Select Package Search from the Change Management menu. The Package Search page appears.
2. Enter or select one or more criteria to narrow your search.
3. Click Search. The application displays the packages matching your search criteria in the Search Results grid.
4. Select the package you want to view. The application displays the relevant transactions for the package.
5. Click the transaction you need. The applicable page appears.
6. Click the Attach Document icon in the package information header at the top of the page. The Attach Document pop-up appears.
7. Select a document in the Document Instances field. The application enables the View Document button.
8. Enter the specific revision in the Revision field.

Or

Select the Revision of Record check box.

9. Click View Document if you want to view the document before you attach it. The application prompts you to open or save the file, or it automatically downloads the document. (Your Internet browser determines whether the application displays a pop-up, its appearance, and the wording of the Open and Save options.)
10. Click Attach. A message appears indicating that the document has been attached successfully.
