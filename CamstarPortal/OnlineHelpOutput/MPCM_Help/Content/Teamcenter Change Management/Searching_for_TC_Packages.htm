Searching for Teamcenter Packages

You can use the Package Search page to search for MOM PLM packages. You can search for both Teamcenter packages and packages created in CEP. Refer to "Searching for a Package" for information.

When Searching for Teamcenter Packages

You should know the following when searching for packages downloaded from Teamcenter:

• There are no available transactions at the Download in Process step, only a status.
• There is no transaction page for Approve PLM. The default MOM PLM workflow is not set up for manual approvals, but it may be configured to require a manual approval. Approval transactions occur in Teamcenter and are sent to CEP and recorded against the package. The package is then moved to Rejected or Pending Deployment status. Refer to "Routing and Canceling Teamcenter Approvals" for information.
Note: Void transactions can also occur in Teamcenter and are sent to CEP and recorded against the package.
• There is no transaction to cancel an approval. You must notify the Teamcenter package owner of the need for a cancellation and manually move the package status back to Pending Enrichment. Refer to "Routing and Canceling Teamcenter Approvals" for information.

Viewing a CEP Change Package in Teamcenter

A Link to PLM transaction button is displayed and enabled under Available Transactions when a package downloaded from Teamcenter or Teamcenter change package is selected in the search results. Clicking the button opens the Teamcenter login view. The corresponding ECN displays in Teamcenter if the login is successful.

These conditions must be met to view the corresponding ECN:

• Teamcenter must be installed
• You have valid Teamcenter credentials and appropriate roles and permissions within Teamcenter and CEP

Viewing a Teamcenter ECN in CEP

You can view an ECN in CEP by entering the package name in the URL template and associating that URL to the ECN Camstar Reference Item in Teamcenter. Clicking the ECN Camstar Reference Item opens the Camstar user login page. The Package Search page opens after you successfully log in to CEP. The corresponding change package is displayed in the Search Results grid and is selected by default.

Note: The URL will open Camstar and the Package Search page if the package name was incorrectly entered in the URL template, but the Search Results grid will be blank.
