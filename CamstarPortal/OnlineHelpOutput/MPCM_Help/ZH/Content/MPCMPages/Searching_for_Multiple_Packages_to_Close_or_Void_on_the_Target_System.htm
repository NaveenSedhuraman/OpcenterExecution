在目标系统上搜索要关闭、作废或打开的多个包

应用程序提供了一个功能，用于根据指定的准则在目标系统上搜索和检索多个要关闭、作废或打开的更改管理包。也可以在“激活搜索（多个）”页上选择要关闭、作废或打开的单个包。已部署且状态符合激活条件的包可使用“激活搜索（多个）”功能。在此页上搜索包的员工必须具有指派了激活权限的角色，或者必须具有默认“包激活者”角色。角色将在建模过程中指派给员工。

在目标系统上搜索要关闭、作废或打开的多个包时

首次访问“激活搜索（多个）”页时，应用程序将显示可用于过滤的字段和空白的“搜索结果”表格。

在搜索包时：

• 可使用 SQL 通配符按包名称搜索。例如，输入 "%01%" 时，将返回名称中包含 "01" 字符的包。
• 可以按激活状态进行搜索。有三种默认包状态。默认情况下，应用程序将选择“激活未完成”和“等待激活”。不会选择“激活”状态。“激活状态”字段用于显示计数而不是所选状态。加载页面时，应用程序将在“激活状态”字段中显示“已选择 2 个，共 3 个”。
Note: 搜索最近激活的包可能会返回意外结果。默认情况下，激活状态每 5 分钟更新一次。要在 Modeling 中修改此设置，可通过调整默认计划的业务规则 SBR_Activation。有关信息，请参考《  Opcenter Execution Medical Device and Diagnostics Modeling 用户指南》或《  Opcenter Execution Core Modeling 用户指南》。
• 可以按包状态进行搜索。有三种可用状态。默认情况下，应用程序将选择“打开”。不会选择“关闭”和“已作废”。应用程序将显示计数而不是所选状态。加载页面时，应用程序将在“包状态”字段中显示“已选择 1 个，共 3 个”。
Note: 如果选择了“关闭”或“已作废”状态，则可以查看已关闭或已作废的包的包详细信息。
• 如果未指定任何过滤器，则授权访问的每个更改管理包都将显示在结果中。
• 最多可以使用 7 个过滤器来缩小搜索范围。

执行搜索

执行搜索时，应用程序将执行以下操作：

• 在表格中显示搜索结果
• 在过滤器字段中保留搜索准则

如果搜索返回的结果多于可在表格的行中显示的数量，则应用程序会向表格添加分页。应用程序将高亮显示处于“未完成部署”或“已拒绝”工步的任何包。

在应用程序返回符合准则的包后，可选择要对其执行事务的一个或多个包。要在表格中选择多个包，可通过选中表格的标题行中的复选框，也可以选中各个包对应的复选框。

Siemens 还能够将搜索结果导出为 Excel 文件或 CSV 格式。有关信息，请参考《  Opcenter Execution Medical Device and Diagnostics Modeling 用户指南》或《  Opcenter Execution Core Modeling 用户指南》。

清除搜索过滤器和结果

使用“全部清除”按钮可清除过滤器和结果。清除“激活搜索（多个）”页可以开始新的搜索。单击“全部清除”按钮时，应用程序将执行以下操作：

• 清除所有指定的搜索过滤器
• 清除“搜索结果”表格中的所有包
• 清除所有事务按钮
• 继续显示“激活搜索（多个）”页

使用事务按钮

必须至少选择一个包，然后单击“加载操作”按钮才能启用事务按钮。如果选择多个包，则应用程序将显示可同时对所选包执行的事务。可对多个包执行的事务只有关闭、作废和打开。例如，用户可能希望搜索状态为“激活未完成”的所有包，然后同时关闭所有这些包。

应用程序根据一个或多个所选包的包状态和激活状态启用“关闭包”和“作废包”事务按钮。

如果仅在“搜索结果”表格中选择一个包，除了“关闭所选（多个）”、“作废所选”和“打开所选”之外，还将显示“包详细信息”事务按钮。如果选择了“已关闭”或“已作废”状态，则还可以查看已关闭或已作废的包的包详细信息。

如果选择一个或多个包，然后单击事务按钮，则应用程序将根据用户是要关闭包、作废包还是打开包来显示相应的弹出窗口。可输入相关的注释。

应用程序将在关闭、作废或打开包后，更新每个包的包状态。已关闭或已作废的任何包将不再显示在“搜索结果”表格中，除非在“包状态”字段中选择“已关闭”或“已作废”。

如何在目标系统上搜索要关闭、作废或打开的多个包

按照以下步骤在目标系统上搜索要关闭、作废或打开的多个包：

1. 从更改管理菜单中选择激活搜索（多个）。将显示“激活搜索（多个）”页。
2. 输入或选择一个或多个条件以缩小搜索范围。
3. 单击搜索。 应用程序将在“搜索结果”表格中显示与搜索准则匹配的包。

如何在目标系统上对多个包执行事务

按照以下步骤在目标系统上对多个包执行事务：

1. 执行“如何在目标系统上搜索要关闭、作废或打开的多个包”过程。
2. 选中“搜索结果”表格的标题行中的复选框可全选显示的包。

或

选中各个包对应的复选框。

3. 单击加载操作。应用程序将启用所选包的相关事务。
4. 单击某事务按钮。应用程序将显示所选事务的相应弹出窗口。
5. 完成弹出窗口中的信息。
