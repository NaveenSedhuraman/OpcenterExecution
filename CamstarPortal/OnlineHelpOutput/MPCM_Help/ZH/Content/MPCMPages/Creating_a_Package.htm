创建包

在将建模数据和工艺流程的更新从源系统部署到一个或多个目标系统的过程中，创建更改管理包是第一步。

创建包之前

必须在建模中配置更改管理，然后才能创建及部署更改管理包。首先，必须定义以下建模对象：

• 包创建原因 - 创建包的原因
• 服务器目录系统 - 包的源和目标系统
Note: 如果未定义服务器目录系统，则在打开“创建包”页时，应用程序将显示一条错误消息。

在创建包之前，必须为员工指派具有更改管理权限的角色。创建更改管理包的员工必须具有已指派包创建权限的角色。默认情况下，Siemens 提供具有这些权限的“包创建者”角色。角色将在建模过程中指派给员工。

有关定义执行更改管理事务必需的建模对象的信息，请参考《  Opcenter Execution Medical Device and Diagnostics Modeling 用户指南》或《  Opcenter Execution Core Modeling 用户指南》。有关默认更改管理角色的信息，请参考《  Opcenter Execution Medical Device and Diagnostics 系统管理指南》或《  Opcenter Execution Core 系统管理指南》。

如何创建包

在创建包时，必须选择默认或自定义更改管理工作流程。单击“更改管理工作流程”字段旁的“信息”图标后，弹出窗口会显示所选工作流程的图示。此外，如果在定义实例时提供了更改管理工作流程的描述，则应用程序将显示相应的描述。

Note: 在创建包之后无法更新或更改已指派的工作流程。

默认情况下，Siemens 提供了三个工作流程用来创建和部署更改管理包：

• 无需审批 - 用于从源系统部署更改管理包，而无需对包进行审批。
• Camstar - 需要先对更改管理包进行审批，然后才能从源系统进行部署。选择此选项将启用“更新包”页上的“审批者”选项卡，用于查看审批模板和指派的审批者。
• PLM - 更改管理包必须有产品生命周期管理 (PLM) 审批决策，才能从源系统进行部署。

有关默认工作流程的信息，请参考“了解默认更改管理工作流程”。

可以在创建包时或在以后更新包时指定部署信息。以后可以在部署前将建模和工艺流程信息指派给包。在部署包之前，必须选择至少一个目标系统。

应用程序使用执行创建包事务时输入的信息，在后续更改管理包页上显示包的核心属性。应用程序在“更新包”页上显示包创建期间输入的核心属性。核心包属性包括包名称、工步和所有者等信息。有关核心属性及其显示在哪些页上的信息，请参考“包信息”。

使用包创建模板

可以使用创建包模板建模对象来定义包创建模板页。选择模板时，将以预定义的包和部署详细信息来填充“创建包”页。创建包后，适用的预定义审批者和协同模板将显示在“更新包”页上。每次创建包时不必手动输入所有信息。

可以覆盖页中的模板信息。选择其他包创建模板时，也会覆盖页中的信息。单击“创建”或“创建并更新”后，才会保存包信息。在创建包时做出的更改将仅应用于新创建的包。应用程序不会将对包做出的任何更新保存到包创建模板。

添加部署详细信息

创建包时部署详细信息是可选的，但在部署包之前是必需的。

如果选择的目标系统在关联的服务器目录系统中定义的 MDB 版本或应用程序版本与源系统的服务器目录系统中所定义的版本不同，则应用程序显示警告消息。如果目标系统和源系统上的 MDB 版本和应用程序版本都不相同，则警告消息会包括针对两个版本的警告。此警告消息仅对最接近表格底部的不匹配目标显示。可以关闭警告消息然后创建包。在源系统和目标系统之间的 MDB 版本或应用程序版本不同时，应用程序不会阻止创建包。

添加激活详细信息

可以指定在激活包之前激活必要条件包，以说明其他包中实例的相依项。在“必要条件更改包”表格中单击“添加”时，将打开“包搜索”弹出窗口。可以使用此弹出窗口搜索并选择要添加到包的必要条件包。

如何创建包

单击“创建并更新”按钮以创建包并显示“更新包”页。如果关联的工作流程需要审批，可以修改包的核心属性并指派审批者。有关信息，请参考“更新包”。

Note: 必须具有相应的角色才能更新包。

单击“创建”按钮时，将创建包并清除“创建包”页上的字段。可以开始输入新包的信息。

如何创建包

按照以下步骤创建包：

1. 从更改管理菜单中选择创建包。将显示“创建包”页。
2. 是否要使用包创建模板？

如果 ...

操作步骤 ...

是
a. 选择包创建模板。应用程序将更新“创建包”页，以显示与选定模板相关联的预定义信息。
b. 转到步骤 3。

否

转到步骤 3。
3. 在包名称字段中输入更改管理包的名称。
4. 选择包所有者。
5. 选择包创建原因。
6. 从更改管理工作流程字段选择用于创建和部署包的更改管理工作流程。
7. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
8. 执行下述中的一项操作：
• 单击创建。应用程序随即创建更改管理包并显示成功消息。
• 单击创建并更新。应用程序随即创建更改管理包，并显示填充了包信息的“更新包”页。
