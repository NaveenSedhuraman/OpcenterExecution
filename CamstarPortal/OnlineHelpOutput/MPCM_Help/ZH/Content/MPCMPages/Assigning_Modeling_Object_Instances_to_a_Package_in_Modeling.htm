在建模中将建模对象实例指派给包

可以创建和更新建模对象实例，然后在建模中将它们指派给现有的活动包。具有适当角色和权限的任何用户均可选择默认更改管理包，然后在会话期间将创建或更新的所有实例指派给包。

建模中将建模对象实例指派给包

如果使用的是已获许可的更改管理、已定义至少一个包而且拥有的角色具有“更改包建模查询”权限，则“建模”页上会显示“更改管理保存”选项。当满足以下条件时，可以在建模中将建模对象实例指派给包：

• 具有建模权限。
• 具有“指派更改包内容”权限。
• 具有“更改包建模查询”权限。
• 您是包所有者或具有所有者角色（例如，Siemens 提供的包所有者角色），或者是为包指定的内容协同者。
• 包处于允许指派内容的工步；例如，起草工步。

“更改管理保存”选项默认设为“关”。在“建模”中工作时行为与未集成更改管理时一样。

如果希望在“建模”中将建模对象实例指派给包，可将“更改管理保存”选项设为“开”。

关闭“建模”选项卡或退出应用程序时，会话结束。“更改管理保存”选项设为“关”。若要再次使用此功能，必须在下次加载页面时再将该选项设为“开”。

打开第二个“建模”选项卡即开启了另一个会话。

Note: 必须将“更改管理保存”选项设为“开”后按正确的顺序指派建模对象或者添加引用。有关将建模对象实例和引用指派给包的信息，请参考“指派和审核包内容”。有关“建模”页以及创建和升级建模对象实例的信息，请参考《  Opcenter Execution Medical Device and Diagnostics Modeling 用户指南》或《  Opcenter Execution Core Modeling 用户指南》。

如果未选择默认包，则可以在保存每个实例时将它指派给包。

如果设置默认包

如果要将在会话期间创建或更新的所有建模对象实例指派给默认包，将“更改管理保存”选项设为“开”并选择相应的包。

只能在“设置默认包”弹出窗口中选择一个包。选择包，然后单击“设置默认”按钮即可。

如果未设置默认包

如果将“更改管理保存”选项设为“开”，但未选择默认包，每次保存实例时将会显示“管理包关联”弹出窗口。这时便可以将实例指派给包。

“管理包关联”弹出窗口包含三个选项卡：

• 关联的包

“关联的包”选项卡列出与实例关联（无论是显式还是通过相依项）的所有活动包。无论角色如何，都将显示除已作废或关闭的包以外的所有其他包。

Note: 这是通过“建模”页工具栏上的“包关联”按钮访问“管理包关联”弹出窗口时显示的默认选项卡。
• 从包中移除

“从包中移除”选项卡列出与实例关联且满足角色和工步准则的所有活动包。可以将实例从多个包中移除。只能移除显式添加的实例。不能移除相依项。

Note: 即使实例是通过默认包指派的，也可以使用此选项卡将实例从一个或多个包中移除。
• 添加到包

“添加到包”选项卡列出符合角色和工步准则的所有活动包。应用程序不会列出已添加的包。这是默认选项卡。只能选择一个包。

如果使用的是已获许可的更改管理而且已创建至少一个包，则“建模”页上会显示“更改管理保存”选项。

如何在建模中将建模对象实例指派给默认包

按照以下步骤在建模中将建模对象实例指派给默认包：

1. 从建模菜单中选择建模。 将显示“建模”页。
2. 单击关将更改管理保存选项设为开。将显示“设置默认”链接。
3. 单击设置默认链接。将显示“设置默认包”弹出窗口。
4. 选择要设为默认的包。
5. 单击设置默认。将关闭“设置默认包”弹出窗口。
6. 单击“建模”页上的所有维护箭头以查看该组下建模对象的列表。

或

从“建模”页的查看对象列表中选择查看全部（按字母顺序），按字母顺序显示建模对象的列表。 随即将显示按字母顺序排列的所有建模对象的列表。

7. 选择一个建模对象。 随即“建模”页中将显示该对象的页面。
8. 新建一个建模对象实例或更新现有建模对象实例。
9. 单击保存。 应用程序保存建模对象并显示成功消息。它会将建模对象实例指派给默认包。
10. 为要在会话期间指派给默认包的任何其他建模对象实例重复步骤 7-9。

如何在建模中向包添加建模对象实例而不设置默认值

按照以下步骤在建模中向包添加建模对象实例而不设置默认包：

1. 从建模菜单中选择建模。 将显示“建模”页。
2. 单击关将更改管理保存选项设为开。将显示“设置默认”链接。
3. 单击“建模”页上的所有维护箭头以查看该组下建模对象的列表。

或

从“建模”页的查看对象列表中选择查看全部（按字母顺序），按字母顺序显示建模对象的列表。 随即将显示按字母顺序排列的所有建模对象的列表。

4. 选择一个建模对象。 随即“建模”页中将显示该对象的页面。
5. 新建一个建模对象实例或更新现有建模对象实例。
6. 单击保存。 应用程序保存建模对象并显示成功消息。将显示“管理包关联”弹出窗口，其中选择了“添加到包”选项卡。
7. 选择要向其添加实例的包。
8. 单击添加。应用程序会将建模对象实例添加到包，显示成功消息，然后关闭“管理包关联”弹出窗口。

如何在建模中从包中移除实例

按照以下步骤在建模中从包中移除实例：

1. 从建模菜单中选择建模。 将显示“建模”页。
2. 单击关将更改管理保存选项设为开。将显示“设置默认”链接。
3. 单击“建模”页上的所有维护箭头以查看该组下建模对象的列表。

或

从“建模”页的查看对象列表中选择查看全部（按字母顺序），按字母顺序显示建模对象的列表。 随即将显示按字母顺序排列的所有建模对象的列表。

4. 选择包含要从中移除包关联的实例的建模对象。 随即“建模”页中将显示该对象的页面。
5. 选择要从中移除包关联的实例。将显示该实例的字段。
6. 单击保存。 应用程序显示成功消息，指出建模对象已更新。将显示“管理包关联”弹出窗口，其中选择了“添加到包”选项卡。
7. 单击从包中移除选项卡。
8. 选择要从中移除实例的包。
Note: 只能移除显式添加的实例，而不能移除相依项。
9. 单击移除。应用程序将从包中移除建模对象实例，显示成功消息，然后关闭“管理包关联”弹出窗口。

如何在建模中查看与实例关联的包

按照以下步骤在建模中查看与实例关联的包：

1. 从建模菜单中选择建模。 将显示“建模”页。
2. 单击关将更改管理保存选项设为开。将显示“设置默认”链接。
3. 单击“建模”页上的所有维护箭头以查看该组下建模对象的列表。

或

从“建模”页的查看对象列表中选择查看全部（按字母顺序），按字母顺序显示建模对象的列表。 随即将显示按字母顺序排列的所有建模对象的列表。

4. 选择包含要查看其包关联的实例的建模对象。 随即“建模”页中将显示该对象的页面。
5. 选择要查看其包关联的实例。将显示该实例的字段。
6. 单击保存。 应用程序显示成功消息，指出建模对象已更新。将显示“管理包关联”弹出窗口，其中选择了“添加到包”选项卡。
7. 单击关联的包选项卡。
8. 查看与实例关联的包列表。
9. 单击关闭。应用程序将关闭“管理包关联”弹出窗口。
