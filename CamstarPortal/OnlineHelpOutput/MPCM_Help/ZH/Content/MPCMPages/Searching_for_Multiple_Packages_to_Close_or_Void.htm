搜索要关闭、作废或打开的多个包

应用程序提供了一个功能，用于根据指定的准则搜索和检索多个要关闭、作废或打开的更改管理包。“包搜索（多个）”功能适用于处于以下默认工步的包：

• 起草（默认“无需审批”更改管理工作流程）
• 起草 Camstar（默认 Camstar 更改管理工作流程）
• 起草 PLM（默认 PLM 更改管理工作流程）
• 待审批 Camstar（默认 Camstar 更改管理工作流程）
• 待审批 PLM（默认 PLM 更改管理工作流程）
• 已拒绝
• 待处理部署
• 已完成部署
• 未完成部署
Note: 其他工步可由系统管理员进行配置。

搜索要关闭、作废或打开的多个包

首次访问“包搜索（多个）”页时，应用程序将显示可用于过滤的字段和空白的“搜索结果”表格。应用程序会向该选项卡添加过滤器图标，以指示过滤器准则已指定。单击“搜索”按钮可填充“搜索结果”表格。

搜索要关闭、作废或打开的多个包时：

• 可使用 SQL 通配符按包名称搜索。例如，输入 "%01%" 时，将返回名称中包含 "01" 字符的包。
• 可按更改管理工作流程进行搜索。一次只能选择一个工作流程。
• 可按一个或多个更改管理工步进行搜索。可用工步取决于所选的更改管理工作流程。可选择多个工步。“更改管理工步”字段显示的是计数而不是所选的各工步。例如，如果在五个可用工步中选择了三个，则应用程序将在“更改管理工步”字段中显示“已选择 3 个，共 5 个”。
• 有五种默认包状态。默认情况下会选择“开放”、“已部署”和“已拒绝”。不会选择“已作废”和“已关闭”。“包状态”字段用于显示计数而不是所选包状态。加载页面时，应用程序将在“包状态”字段中显示“已选择 3 个，共 5 个”。
Note: 如果选择了“已关闭”或“已作废”包状态，则可以查看已关闭或已作废的包的包详细信息。“包状态”选择列表中的多个选定值相当于“或”过滤器，可添加到返回包的列表中。
• 其余“常规搜索”过滤器相当于“和”过滤器，可隐藏返回的一些包。

执行搜索

执行搜索时，应用程序将执行以下操作：

• 在表格中显示搜索结果
• 在过滤器字段中保留搜索准则

如果搜索返回的结果多于可在表格的行中显示的数量，则应用程序会向表格添加分页。应用程序将高亮显示处于“未完成部署”或“已拒绝”工步的任何包。

在应用程序返回符合准则的包后，可选择要对其执行事务的一个或多个包。要在表格中选择多个包，可通过选中表格的标题行中的复选框，也可以选中各个包对应的复选框。

Siemens 还能够将搜索结果导出为 Excel 文件或 CSV 格式。有关信息，请参考《  Opcenter Execution Medical Device and Diagnostics Modeling 用户指南》或《  Opcenter Execution Core Modeling 用户指南》。

清除搜索过滤器和结果

使用“全部清除”按钮可清除过滤器和结果。清除“包搜索（多个）”页可以开始新的搜索。单击“全部清除”按钮时，应用程序将执行以下操作：

• 清除所有指定的搜索过滤器
• 清除“常规搜索”选项卡中的过滤器图标
• 清除“搜索结果”表格中的所有包
• 清除所有事务按钮
• 继续显示“包搜索（多个）”页

使用事务按钮

必须至少选择一个包，然后单击“加载操作”按钮才能启用事务按钮。如果选择多个包，则应用程序将显示可同时对所选包执行的事务。可对多个包执行的事务只有关闭、作废和打开。例如，用户可能希望搜索状态为“已拒绝”的所有包，然后同时关闭所有这些包。

如果仅在“搜索结果”表格中选择一个包，除了“关闭所选”、“作废所选”和“打开所选”之外，还将显示“包详细信息”事务按钮。如果选择了“已关闭”或“已作废”状态，则还可以查看已关闭或已作废的包的包详细信息。

如果选择一个或多个包，然后单击事务按钮，则应用程序将根据用户是要关闭包、作废包还是打开包来显示相应的弹出窗口。可输入相关的注释。

应用程序将在关闭、作废或打开包后，更新每个包的包状态。已关闭或已作废的任何包将不再显示在“搜索结果”表格中，除非在“包状态”字段中选择“已关闭”或“已作废”。

如何搜索要关闭、作废或打开的多个包

按照以下步骤搜索要关闭、作废或打开的多个包：

1. 从更改管理菜单中选择包搜索（多个）。将显示“包搜索（多个）”页。
2. 输入或选择一个或多个条件以缩小搜索范围。
3. 单击搜索。 应用程序将在“搜索结果”表格中显示与搜索准则匹配的包。

如何对多个包执行事务

按照以下步骤对多个包执行事务：

1. 执行“如何搜索要关闭、作废或打开的多个包”过程。
2. 选中“搜索结果”表格的标题行中的复选框可全选显示的包。

或

选中各个包对应的复选框。

3. 单击加载操作。应用程序将启用所选包的相关事务。
4. 单击某事务按钮。应用程序将显示所选事务的相应弹出窗口。
5. 完成弹出窗口中的信息。
