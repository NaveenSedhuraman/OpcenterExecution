搜索包

应用程序提供了一个功能，用于根据指定的准则搜索和检索更改管理包。可以按员工指派或常规准则进行搜索。“包搜索”功能适用于处于以下默认工步的包：

• 起草（默认“无需审批”更改管理工作流程）
• 起草 Camstar（默认 Camstar 更改管理工作流程）
• 起草 PLM（默认 PLM 更改管理工作流程）
• 待审批 Camstar（默认 Camstar 更改管理工作流程）
• 待处理审批 PLM（默认 PLM 更改管理工作流程和默认 MOM PLM 工作流程）
• 已拒绝
• 待处理部署
• 已完成部署
• 未完成部署
Note: 其他工步可由系统管理员进行配置。

当访问“包搜索”页时，应用程序将显示两个选项卡和“搜索结果”表格。这两个选项卡分别为“员工指派搜索”和“常规搜索”。单击“搜索”按钮可填充“搜索结果”表格。起初，表格会列出角色为包所有者、包审批者或包协作者的当前用户的所有包。默认情况下，表格不含当前用户已完成审批的包。

每个选项卡都包含多个字段，用户可通过这些字段缩小搜索范围。应用程序向每个选项卡添加一个过滤器图标，以指示已指定过滤条件。过滤器图标会自动显示在“员工指派搜索”选项卡上，因为此选项卡上默认选择了多个字段。选择准则时，“常规搜索”选项卡上会显示过滤器图标。应用程序将在“搜索结果”表格的两个选项卡中显示组合的搜索准则的结果。

在应用程序返回符合准则的各更改管理包后，可选择要在其上执行事务的包。应用程序将仅启用与所选包相关的事务。

Note: 如果想同时选择要关闭、打开或作废的多个包，请参考“搜索要关闭、作废或打开的多个包”。

使用员工指派准则搜索包

首次访问“包搜索”页时，“员工指派搜索”选项卡默认处于选中状态。可以使用多个字段过滤搜索。

使用此选项卡上的字段搜索包时：

• “员工”字段中的条目默认为当前用户。
• 可以按员工包指派进行搜索。
• 可以按替代角色进行搜索。
• 可以指定是否隐藏已完成的指派。
• 字段“是所有者”、“是审批者”、“是协同者”、“具有所有者角色”、“具有审批者角色”和“具有协同者角色”默认处于选中状态。
• “包指派”和“替代角色”下的复选框充当“或”过滤器。如果有匹配的复选框属性为“是”，则显示匹配的包。
• “隐藏已完成的指派”下的复选框充当“和”过滤器，可以隐藏其他复选框返回的某些包。

使用常规准则搜索包

可通过“常规搜索”选项卡上的过滤器进一步缩小搜索范围。

使用此选项卡上的字段搜索包时：

• 可使用 SQL 通配符按包名称搜索。例如，输入 "%01%" 时，将返回名称中包含 "01" 字符的包。
• 可按更改管理工作流程进行搜索。一次只能选择一个工作流程。
• 可按一个或多个更改管理工步进行搜索。可用工步取决于所选的更改管理工作流程。可选择多个工步。“更改管理工步”字段显示的是计数而不是所选的各工步。例如，如果在五个可用工步中选择了三个，则应用程序将在“更改管理工步”字段中显示“已选择 3 个，共 5 个”。
• 有五种默认包状态。默认情况下会选择“开放”、“已部署”和“已拒绝”。不会选择“已作废”和“已关闭”。“包状态”字段用于显示计数而不是所选包状态。加载页面时，应用程序将在“包状态”字段中显示“已选择 3 个，共 5 个”。
Note: “包状态”选择列表中的多个选定值相当于“或”过滤器，可添加到返回包的列表中。
• 其余“常规搜索”过滤器相当于“和”过滤器，可隐藏返回的一些包。

执行搜索

执行搜索时，应用程序将执行以下操作：

• 在表格中显示搜索结果
• 在过滤器字段中保留搜索准则

如果搜索返回的结果多于可在表格的行中显示的数量，则应用程序会向表格添加分页。应用程序将高亮显示处于“未完成部署”或“已拒绝”工步的任何包。在“搜索结果”表格中选择包后，应用程序将启用与包相关的事务。Siemens 还能够将搜索结果导出为 Excel 文件或 CSV 格式。有关信息，请参考《  Opcenter Execution Medical Device and Diagnostics Modeling 用户指南》或《  Opcenter Execution Core Modeling 用户指南》。

清除搜索过滤器和结果

使用“全部清除”按钮可清除过滤器和结果。清除“包搜索”页可以开始新的搜索。单击“全部清除”按钮时，应用程序将执行以下操作：

• 清除两个选项卡上所有指定的搜索过滤器
• 清除“搜索”选项卡中的过滤器图标
• 清除“搜索结果”表格中的所有包
• 清除所有事务按钮
• 继续显示“包搜索”页

使用事务按钮

可使用事务按钮对“搜索结果”表格中的任何选定包执行事务。应用程序将仅启用对于包有效的事务。如果在表格中选择其他包，则可用事务将发生相应的变化。

选择包，然后单击事务按钮时，应用程序将根据所选事务类型显示相应的事务页或弹出窗口。

关闭、作废或打开包时，将显示一个弹出窗口。可输入相关的注释。

Note: 可打开之前在“包搜索”或“包搜索（多个）”页上已关闭或已作废的包。

执行任何其他事务时，都将显示一个事务页。应用程序将使用在表格中选择的包的名称填充页面上的“所选包”字段（如果可用）。例如，如果选择包，然后单击“更改工步”按钮，则将显示“更改包的工步”页，其中“所选包”字段已填充。
