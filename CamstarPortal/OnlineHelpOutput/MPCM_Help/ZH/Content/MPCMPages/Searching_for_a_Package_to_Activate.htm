搜索要激活的包

应用程序提供了一个功能，用于根据指定的准则在目标系统上搜索和检索要激活的更改管理包。已部署且状态符合激活条件的包可使用“激活搜索”功能。搜索要激活的包的员工必须具有指派了激活权限的角色，或者必须具有默认“包激活者”角色。角色将在建模过程中指派给员工。

Note: 如果想同时选择要关闭、作废或打开的多个包，请参考“在目标系统上搜索要关闭、作废或打开的多个包”。

搜索要激活的包

首次访问“激活搜索”页时，应用程序将显示可用于过滤的字段和空白的“搜索结果”表格。应用程序会向该选项卡添加过滤器图标，以指示过滤器准则已指定。单击“搜索”按钮可填充“搜索结果”表格。

在搜索包时：

• 可使用 SQL 通配符按包名称搜索。例如，输入 "%01%" 时，将返回名称中包含 "01" 字符的包。
• 可以按激活状态进行搜索。有三种默认包状态。默认情况下，应用程序将选择“激活未完成”和“等待激活”。不会选择“激活”状态。“激活状态”字段用于显示计数而不是所选状态。加载页面时，应用程序将在“激活状态”字段中显示“已选择 2 个，共 3 个”。
Note: 搜索最近激活的包可能会返回意外结果。默认情况下，激活状态每 5 分钟更新一次。可以通过调整默认计划的业务规则 SBR_ACTIVATION 在 Portal Modeling 中修改此设置。有关信息，请参考《  Opcenter Execution Medical Device and Diagnostics Modeling 用户指南》或《  Opcenter Execution Core Modeling 用户指南》。
• 可以按包状态进行搜索。有三种可用状态。默认情况下，应用程序将选择“打开”。不会选择“关闭”和“已作废”。应用程序将显示计数而不是所选状态。加载页面时，应用程序将在“包状态”字段中显示“已选择 1 个，共 3 个”。
Note: 如果选择了“关闭”或“已作废”状态，则可以查看已关闭或已作废的包的包详细信息。
• 如果未指定任何过滤器，则授权访问的每个更改管理包都将显示在结果中。
• 最多可以使用 7 个过滤器来缩小搜索范围。

在应用程序返回符合准则的各更改管理包后，可选择要在其上执行事务的包。应用程序将启用与所选包相关的事务。

执行搜索

执行搜索时，应用程序将执行以下操作：

• 在表格中显示搜索结果
• 在过滤器字段中保留搜索准则

如果搜索返回的结果多于可在表格的行中显示的数量，则应用程序会向表格添加分页。在“搜索结果”表格中选择一个包时，应用程序将启用与该包相关的事务。Siemens 还能够将搜索结果导出为 Excel 文件或 CSV 格式。有关信息，请参考《  Opcenter Execution Medical Device and Diagnostics Shop Floor 用户指南》或《  Opcenter Execution Core Shop Floor 用户指南》。

清除搜索过滤器和结果

使用“全部清除”按钮可清除过滤器和结果。清除“激活搜索”页可以开始新的搜索。单击“全部清除”按钮时，应用程序将执行以下操作：

• 清除所有指定的搜索过滤器
• 清除“常规搜索”选项卡中的过滤器图标
• 清除“搜索结果”表格中的所有包
• 清除所有事务按钮
• 继续显示“激活搜索”页

使用事务按钮

可使用事务按钮对“搜索结果”表格中的任何选定包执行事务。应用程序将启用对于所选包有效的事务。

应用程序根据所选包的包状态和激活状态启用“关闭包”和“作废包”事务按钮。

选择包，然后单击事务按钮时，应用程序将根据所选事务类型显示相应的事务页或弹出窗口。

关闭、作废或打开包或者在具备必要条件的情况下激活包时，将显示一个弹出窗口。可输入相关的注释。

Note: 可打开之前在“激活搜索”或“激活搜索（多个）”页上已关闭或已作废的包。

执行任何其他事务时，都将显示一个事务页。应用程序将使用在表格中选择的包的名称填充“所选包”字段（如果可用）。例如，如果选择一个包并单击“激活包”按钮，则将显示“激活包”页。

如何搜索要激活的包

按照以下步骤搜索要激活的包：

1. 从更改管理菜单中选择激活搜索。将显示“激活搜索”页。
2. 输入或选择一个或多个条件以缩小搜索范围。
3. 单击搜索。 应用程序将在“搜索结果”表格中显示与搜索准则匹配的包。

如何对所选包执行事务

按照以下步骤对所选包执行事务：

1. 执行“如何搜索要激活的包”过程。
2. 选择要执行事务的包。应用程序将启用该包的相关事务。
3. 单击某事务按钮。应用程序将显示所选事务的相应事务页或弹出窗口。应用程序将使用在表格中选择的包的名称填充相应事务页上的“所选包”字段（如果可用）。
4. 完成事务页或弹出窗口中的信息。
