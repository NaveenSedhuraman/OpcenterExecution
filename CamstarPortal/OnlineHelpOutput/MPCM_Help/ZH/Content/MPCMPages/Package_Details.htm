查看包详细信息

通过“包详细信息”页，可以查看现有软件包的以下内容：

• 核心属性
• 当前和以前的审批周期信息
• 当前内容协同信息
• 指派给包的建模对象实例
• 指派给包的必要条件包

查看包详细信息

可以通过以下方式导航至“包详细信息”页：

• 在单击包含“包信息”标题的任何页上的“内容”按钮后显示的浮出控件菜单中，单击“包详细信息”按钮。
• 在“包搜索”或“激活搜索”页的“可用事务”下单击“包详细信息”按钮。
• 如果在这些页面的任一页上只选择一个包，则在“包搜索（多个）”或“激活搜索（多个）”页上的“可用事务”下单击“包详细信息”按钮。
Note: 系统将会自动使用在单击“包详细信息”按钮的页面上选择的包填充“所选包”字段。

只有在“审批者”表格中选择了某个条目后，在“对审批者的指示”或“审批者注释”字段中才会显示信息。

在“内容协同者”表格中，协同者按“层次”以升序排序。如果包所有者未指定包的协同者，则“内容协同者”表格为空。

查看文档按钮

可以单击“查看文档”按钮，以 PDF 方式查看或保存“包详细信息”页上显示的包详细信息。单击“查看文档”按钮可在浏览器中显示一个对话框，其中包含“打开”、“保存”和“取消”选项。单击“打开”可在默认 PDF 查看应用程序中查看 PDF。必须已安装了能够查看 .pdf 文件的应用程序（如 Adobe Acrobat Reader）。单击“保存”可显示“另存为”弹出窗口，将 PDF 保存到本地计算机。

Note: “查看文档”按钮创建的 PDF 文件默认命名为 Document.pdf。

“查看文档”按钮创建的 PDF 包含“包详细信息”页上的所有信息，无论字段处于展开还是折叠状态。

Note: 显示的对话框可能包含其他选项，如“使用系统查看器打开”，具体取决于浏览器。

更改包内容 - 指派的内容协同者

内容协同者表格

此表格列出指定为更改管理包的内容协同者的所有员工。

仅显示

层次

与协同者关联的协同者级别。级别仅由数字值表示。表格按此列的升序排序。

级别定义任务必须完成的顺序。如果协作任务是按顺序设置的, 则应用程序在每个任务完成后发送电子邮件通知。如果协作任务是并行设置的, 则应用程序在最后一个协作者完成任务时发送电子邮件通知。

串行表示级别数字较低的协同者必须在级别数字较高的协同者开始之前完成指派内容。并行表示具有相同级别数字的协同者可以同时指派内容。

仅显示

名称

已被指派协同工作责任的员工。

仅显示

状态

协作工作的状态。单元格突出显示颜色会根据状态动态变化。状态设置为:

• 待处理 - 员工尚未完成任务。员工可能已经或可能没有启动任务。橙色突出显示单元格。
• 完成-员工已经完成了协作任务, 无论工作是否完成得很晚。当协作者单击内容协作者的 "完整" 按钮时, 工作就完成了-通过 "分配和审阅内容" 页访问完成的任务弹出, 分配和审阅内容选项卡。红色突出显示单元格。
• 逾期 - 员工尚未完成任务，指派的完成任务的时间已过期。当当前日期晚于截止日期时, 任务延迟。员工可能已经或可能没有启动任务。红色突出显示单元格。

仅显示

如何查看包详细信息

按照以下步骤查看包详细信息：

1. 在源系统还是目标系统上？

如果在 ...

操作步骤 ...

源系统

从更改管理菜单中选择包搜索。将显示“包搜索”页。

目标系统

从更改管理菜单中选择激活搜索。将显示“激活搜索”页。
2. 输入或选择一个或多个条件以缩小搜索范围。
3. 单击搜索。 应用程序将在“搜索结果”表格中显示与搜索准则匹配的包。
4. 选择要查看的包。 应用程序将显示该包的相关事务。
5. 单击包详细信息。将显示“包详细信息”页。
