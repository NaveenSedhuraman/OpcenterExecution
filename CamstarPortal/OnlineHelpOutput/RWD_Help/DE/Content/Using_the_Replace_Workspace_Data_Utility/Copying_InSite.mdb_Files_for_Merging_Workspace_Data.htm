Kopieren von InSite.mdb-Dateien zum Zusammenführen von Arbeitsbereichsdaten

Siemens empfiehlt, einen Ordner für die Dateien zu erstellen, die Sie beim Zusammenführen von Arbeitsbereichsdaten verwendet werden. Anschließend können Sie die InSite.mdb-Dateien für aktuelle Metadaten und Basismetadaten aus den jeweiligen Ordnern in diesen Ordner kopieren und umbenennen, damit sie vom Dienstprogramm zum Ersetzen von Arbeitsbereichsdaten (Replace Workspace Data) als Eingabedateien verwendet werden können.

Beim Kopieren von InSite.mdb-Dateien zum Zusammenführen von Arbeitsbereichsdaten

Erstellen Sie einen Ordner (z. B. "C:\Users\Administrator\Desktop\Merge"), in dem Sie die MDB-Dateien verwalten, die als Eingabedateien für das Dienstprogramm zum Ersetzen von Arbeitsbereichsdaten (Replace Workspace Data) verwendet werden.

Sie müssen die aktuelle InSite.mdb-Datei und die InSite.mdb-Basismetadatendateien in diesen Ordner kopieren.

Wichtig: Aktualisieren Sie nicht die Datei "InSite.mdb" im Ordner "C:\Programme (x86)\Camstar\InSite Administration\Base Metadata". Für Opcenter EX MDD und Opcenter EX CR ist diese Datei für den Prozess zur Metadatenaktualisierung erforderlich. Lassen Sie diese Datei im Ordner "Base Metadata" unverändert, um zu gewährleisten, dass die Installationsprogramme für Softwareaktualisierungen die Datei bei jeder Softwareaktualisierung ersetzen können.

Vorgehensweise zum Kopieren von InSite.mdb-Dateien zum Zusammenführen von Arbeitsbereichsdaten

Führen Sie die folgenden Schritte aus, um InSite.mdb-Dateien für das Zusammenführen von Arbeitsbereichsdaten zu kopieren:

1. Greifen Sie auf den Server zu, auf dem Opcenter EX MDD oder Opcenter EX CR installiert ist.
2. Erstellen Sie einen Ordner, in dem Sie die verschiedenen MDB-Dateien verwalten können, die Sie als Eingabedateien für das Dienstprogramm zum Ersetzen von Arbeitsbereichsdaten (Replace Workspace Data) verwenden möchten.
3. Kopieren Sie die InSite.mdb-Basismetadatendatei (standardmäßig "C:\Programme (x86)\Camstar\InSite Administration\Base Metadata\InSite.mdb") in den neuen Ordner.
4. Benennen Sie die Kopie um, z. B. in "InSite_new.mdb".
5. Entfernen Sie das Schreibschutzattribut dieser Datei:
a. Klicken Sie mit der rechten Maustaste auf den Dateinamen. Daraufhin wird ein Menü angezeigt.
b. Wählen Sie im Menü Eigenschaften (Properties) aus. Das Dialogfeld "Eigenschaften" (Properties) wird angezeigt.
c. Deaktivieren Sie das Kontrollkästchen Schreibgeschützt (Read-only).
d. Klicken Sie auf OK. Das Dialogfeld "Eigenschaften" (Properties) wird geschlossen und die Datei ist nicht mehr schreibgeschützt.
6. Erstellen Sie eine Kopie der Datei, die Sie soeben kopiert haben.
7. Benennen Sie die Kopie um, z. B. in "InSite_Base.mdb".
8. Kopieren Sie die aktuelle InSite.mdb-Datei (standardmäßig "C:\Programme (x86)\Camstar\InSite Administration\InSite.mdb") in den neuen Ordner.
9. Benennen Sie die Kopie um, z. B. in "InSite_Current.mdb".
