Starten der Dienste und Anwendungspools

Sie müssen die Dienste und Anwendungspools starten.

Hinweis: Dieses Thema behandelt das Starten aller produktbezogenen Anwendungspools in IIS und aller produktbezogenen Dienste in den Windows-Verwaltungsprogrammen. Die Opcenter EX MDD- und Opcenter EX CR-Server werden als Dienste registriert. Sie können den Status dieser Dienste (außer für CMSAdmin und den Siemens PLM-Lizenzserver) im Management Studio-Bereich "Dienste" (Services) anzeigen. Sie können diese Dienste im Bereich "Services" auch starten, beenden und neu starten. Weitere Informationen finden Sie in der Opcenter Execution Medical Device and Diagnostics-Anleitung zur Systemadministration oder in der Opcenter Execution Core-Anleitung zur Systemadministration.

Vorgehensweise zum Starten des Siemens PLM-Lizenzservers

Gehen Sie folgendermaßen vor, um den Siemens PLM-Lizenzserver zu installieren:

1. Klicken Sie auf Start > Windows-Verwaltungsprogramme > Dienste. Das Fenster "Dienste" (Services) wird angezeigt.
2. Wählen Sie "Siemens PLM-Lizenzserver" (Siemens PLM License Server) aus der Liste aus, klicken Sie mit der rechten Maustaste und wählen Sie Start aus.

Vorgehensweise zum Starten von Anwendungspools

Gehen Sie folgendermaßen vor, um die Anwendungspools in IIS zu starten:

1. Klicken Sie auf Start > Server-Manager. Das Fenster "Server-Manager" wird angezeigt.
2. Wählen Sie Internetinformationsdienste-Manager im Menü Extras aus. Das Fenster "Internetinformationsdienste-Manager" wird angezeigt.
3. Erweitern Sie den Server im Bereich "Verbindungen".
4. Klicken Sie auf Anwendungspools (Application Pools). Das Fenster "Anwendungspools" (Application Pools) wird angezeigt.
5. Wählen Sie jeden der folgenden Anwendungspools einzeln und anschließend Start aus:
• CamstarSecurityServices
• CMSAdmin
• CamstarAppServer
• Camstar WCF-Dienste
• Portal-Registerkarte
Hinweis: Sie können einen IIS-Reset (iisreset) ausführen, um alle diese Anwendungspools zu starten.

Vorgehensweise zum Starten der Dienste

Gehen Sie folgendermaßen vor, um die anderen Dienste zu starten:

1. Klicken Sie auf Start > Windows-Verwaltungsprogramme > Dienste. Das Fenster "Dienste" (Services) wird angezeigt.
2. Wählen Sie alle nachfolgend aufgeführten Dienste einzeln in der Liste aus, klicken Sie mit der rechten Maustaste darauf und wählen Sie Starten (Start) aus:
• Camstar-Bereitstellungsserver (Camstar Deployment Server)
• Camstar-Benachrichtigungsserver (Camstar Notification Server)
