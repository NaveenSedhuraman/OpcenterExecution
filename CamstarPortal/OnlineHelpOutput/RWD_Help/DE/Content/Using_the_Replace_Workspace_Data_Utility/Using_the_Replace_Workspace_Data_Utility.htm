Verwenden des Dienstprogramms zum Ersetzen von Arbeitsbereichsdaten (Replace Workspace Data)

Siemens stellt ein Dienstprogramm zum Ersetzen von Arbeitsbereichsdaten (Replace Workspace Data) zur Verfügung, mit dem Sie Arbeitsbereiche verwalten können. Das Dienstprogramm führt Arbeitsbereiche von mehreren Stellen an einer einzigen Position zusammen, die dann in Ihrer Anwendungsinstallation verwendet werden kann. Das Dienstprogramm wird z. B. nach der Konvertierung von einer Version 7-Softwareaktualisierung auf Release 8 ausgeführt.
