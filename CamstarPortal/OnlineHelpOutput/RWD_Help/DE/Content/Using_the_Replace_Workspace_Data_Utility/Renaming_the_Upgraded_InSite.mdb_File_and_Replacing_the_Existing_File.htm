Umbenennen der aktualisierten InSite.mdb-Datei und Ersetzen der bestehenden Datei

Siemens empfiehlt, die folgenden Schritte auszuführen, wenn das Kontrollkästchen "Aktuelle nach Abschluss ersetzen" (Replace Current When Done) im Fenster "Arbeitsbereichsdaten ersetzen" (Replace Workspace Data) nicht aktiviert wurde:

• Umbenennen der aktualisierten InSite.mdb-Datei, die mit dem Dienstprogramm erstellt wurde, in den Standardnamen: InSite.mdb
• Kopieren der aktualisierten, umbenannten InSite.mdb-Datei in den Ordner "InSite Administration". Der Standardspeicherort ist "C:\Programme (x86)\Camstar\InSite Administration".

Diese Schritte müssen nicht ausgeführt werden, wenn das Kontrollkästchen "Aktuelle nach Abschluss ersetzen" (Replace Current When Done) aktiviert wurde.

Vorgehensweise zum Umbenennen der aktualisierten InSite.mdb-Datei und Ersetzen der bestehenden Datei

Führen Sie die folgenden Schritte aus, um die aktualisierte InSite.mdb-Datei umzubenennen und die bestehende Datei zu ersetzen:

1. Navigieren Sie zum Ordner "C:\Programme (x86)\Camstar\InSite Administration" und erstellen Sie eine Sicherungskopie der Datei "InSite.mdb".
2. Navigieren Sie zum Ordner, der die Datei "UpgradedInSite.mdb" enthält.
3. Benennen Sie "UpgradedInSite.mdb" in "InSite.mdb" um.
4. Kopieren Sie die Datei "InSite.mdb" in den Ordner "C:\Programme (x86)\Camstar\InSite Administration" und ersetzen Sie die vorhandene Datei.
