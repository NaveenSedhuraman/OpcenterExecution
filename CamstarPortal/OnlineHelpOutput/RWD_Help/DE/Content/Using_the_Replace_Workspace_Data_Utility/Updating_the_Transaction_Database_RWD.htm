Aktualisieren der Transaktionsdatenbank

Die Transaktionsdatenbank muss aktualisiert werden, damit die aktualisierte InSite.mdb-Datei verwendet wird.

Weitere Informationen zu den Feldern im Dialogfeld "Datenbank aktualisieren" und zu sonstigen Überlegungen finden Sie in der Opcenter Execution Medical Device and Diagnostics-Anleitung zur Systemadministration und in der Opcenter Execution Core-Anleitung zur Systemadministration.

Beim Aktualisieren der Transaktionsdatenbank

Sie können die WCF-Dienste beim Aktualisieren der Datenbank generieren; allerdings dauert die Aktualisierung dann länger. Verwenden Sie den WCF-Dienstgenerator (WCF Services Generator), um diesen Schritt getrennt durchzuführen.

Weitere Informationen finden Sie unter "Neugenerieren der WCF-Dienste" in der Opcenter Execution Medical Device and Diagnostics-Installationsanleitung und in der Opcenter Execution Core-Installationsanleitung.

Vorgehensweise zum Aktualisieren der Transaktionsdatenbank

Führen Sie die folgenden Schritte aus, um die Transaktionsdatenbank zu aktualisieren:

1. Klicken Sie auf Start > Opcenter Execution MDD > Management Studio. Das Fenster "Management Studio" wird angezeigt.

Oder

Klicken Sie auf Start > Opcenter Execution Core > Management Studio. Das Fenster "Management Studio" wird angezeigt.

2. Erweitern Sie Datenbanken (Databases) und klicken Sie dann mit der rechten Maustaste auf die Transaktionsdatenbank (Transaction Database).
3. Wählen Sie Datenbank aktualisieren (Update Database) aus. Das Dialogfeld "Datenbank aktualisieren" (Update Database) wird angezeigt.
4. Aktivieren oder deaktivieren Sie Ihren Geschäftsanforderungen entsprechend die Aktualisierungsoptionen.
Wichtig: Die Kontrollkästchen "Metadaten und Schema aktualisieren" (Update Metadata and Schema), "Aktualisierung des Datenbankschemas erzwingen" (Force Database Schema Update), "Camstar Portal-Objekte laden" (Load Camstar Portal Objects) und "Alle neu laden" (Reload All) im Dialogfeld "Datenbank aktualisieren" (Update Database) müssen aktiviert sein.
5. Klicken Sie auf OK. Die Statusanzeige zeigt an, dass der Prozess ausgeführt wird. In einem SQL-Skriptpopupfenster wird der Ausführungsfortschritt der Skripts angezeigt. Am unteren Fensterrand wird eine Erfolgsmeldung angezeigt, wenn die Aktualisierung durchgeführt wurde.
