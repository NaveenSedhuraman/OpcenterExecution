Updating the Transaction Database

You must update the transaction database so that it uses the upgraded InSite.mdb file.

Refer to the Opcenter Execution Medical Device and Diagnostics System Administration Guide or the Opcenter Execution Core System Administration Guide for detailed information on the fields on the Update Database dialog box and other considerations.

When Updating the Transaction Database

You can generate the WCF services when updating the database; however, the update process takes longer. Use the WCF Services Generator to perform this step separately.

Refer to "Regenerating the WCF Services" in the Opcenter Execution Medical Device and Diagnostics Installation Guide or the Opcenter Execution Core Installation Guide for information.

How to Update the Transaction Database

Follow these steps to update the transaction database:

1. Click Start > Opcenter Execution MDD > Management Studio. The Management Studio window appears.

Or

Click Start > Opcenter Execution Core > Management Studio. The Management Studio window appears.

2. Expand Databases and then right-click Transaction Database.
3. Select Update Database. The Update Database dialog box appears.
4. Select or clear the update options according to your business requirements.
Important: Ensure that the Update Metadata and Schema, Force Database Schema Update, Load Camstar Portal Objects, and Reload All check boxes are selected in the Update Database dialog box.
5. Click OK. The progress line indicates that the process is running and a SQL Script pop-up appears, showing the progress of the scripts as they execute. A success message appears at the bottom of the window when the update completes.
