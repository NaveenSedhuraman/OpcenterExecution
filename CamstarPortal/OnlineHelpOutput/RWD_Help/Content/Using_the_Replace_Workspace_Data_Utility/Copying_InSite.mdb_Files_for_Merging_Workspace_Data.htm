Copying InSite.mdb Files for Merging Workspace Data

Siemens recommends that you create a folder for the files you will use when merging the workspace data. You can then copy the current and Base Metadata InSite.mdb files from their respective folders into this folder and rename them so that they can be used as input for the Replace Workspace Data utility.

When Copying InSite.mdb Files for Merging Workspace Data

Create a folder (for example, C:\Users\Administrator\Desktop\Merge) in which to maintain the mdb files that will be used as input for the Replace Workspace Data utility.

You must copy the current InSite.mdb file and Base Metadata InSite.mdb files into this folder.

Important: Do not update the InSite.mdb file in the C:\Program Files (x86)\Camstar\InSite Administration\Base Metadata folder. Opcenter EX MDD and Opcenter EX CR require this file for the metadata upgrade process. Keep this file unmodified in the Base Metadata folder to ensure Software Update (SU) installers can successfully replace the file with each SU.

How to Copy InSite.mdb Files for Merging Workspace Data

Follow these steps to copy InSite.mdb files for merging workspace data:

1. Access the server where Opcenter EX MDD or Opcenter EX CR is installed.
2. Create a folder in which to maintain the various mdb files to be used as input for the Replace Workspace Data utility.
3. Copy the Base Metadata InSite.mdb file (C:\Program Files (x86)\Camstar\InSite Administration\Base Metadata\InSite.mdb, by default) into the new folder.
4. Rename the copied file to InSite_New.mdb, for example.
5. Remove the read-only attribute from this file:
a. Right-click the file name. A menu appears.
b. Select Properties from the menu. The Properties dialog box appears.
c. Clear the check from the Read-only check box.
d. Click OK. The Properties dialog box closes and the file is no longer marked as read-only.
6. Make a copy of the file you just copied.
7. Rename the copied file to InSite_Base.mdb, for example.
8. Copy the current InSite.mdb file (C:\Program Files (x86)\Camstar\InSite Administration\InSite.mdb, by default) into the new folder.
9. Rename the copy to InSite_Current.mdb, for example.
