Regenerating WCF Services After Updating the Database

You must generate WCF services when you make changes to the InSite.mdb file. Run the WCF Services Generator if you did not select the Generate WCF Services check box when updating the transaction database.

Refer to the Opcenter Execution Medical Device and Diagnostics Installation Guide or the Opcenter Execution Core Installation Guide for complete information on regenerating the WCF services.
