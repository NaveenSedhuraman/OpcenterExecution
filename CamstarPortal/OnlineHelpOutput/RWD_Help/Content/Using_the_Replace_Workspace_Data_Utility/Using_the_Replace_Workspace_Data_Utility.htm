Using the Replace Workspace Data Utility

Siemens provides a Replace Workspace Data utility that enables you to manage workspaces. The utility takes workspaces from several places and merges them into a single location for use in your application installation. For example, you run the utility after converting from any Version 7 software update to Release 8.
