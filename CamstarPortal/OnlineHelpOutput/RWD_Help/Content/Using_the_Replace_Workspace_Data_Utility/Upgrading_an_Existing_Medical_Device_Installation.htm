Upgrading an Existing Medical Device Installation

You must take additional steps if upgrading an existing Medical Device installation.

Note: Refer to the Opcenter Execution Medical Device and Diagnostics System Administration Guide or the Opcenter Execution Core System Administration Guide as well as the Opcenter Execution MDD Medical Device User Guide or the Opcenter Execution Core Medical Device User Guide for information on configuring the Medical Device features.

If Upgrading from a Version Earlier than Version 6 SU6

You must remove these files if you are currently using a version earlier than Version 6 SU6 and you want to upgrade the Medical Device workspace:

• ManageTeamEmployees.cs
• TeamTrackingTxn.cs

If Upgrading from Version 6 SU8

Contact Product Support vis Support Center at https://support.sw.siemens.com/ if you are currently using Version 6 SU8 and you want to upgrade.

If Upgrading from Version 6 SU9 or SU10

You must uninstall Camstar Solutions Center if you are currently using Version 6 SU9 or Version 6 SU10 and you want to upgrade to Version SU12 or later.
