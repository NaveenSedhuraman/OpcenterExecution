Verwendung des Concierge

Der Concierge stellt eine konsolidierte Liste der Datensätze und Aufgaben zur Verfügung, die Ihnen zugewiesen wurden. Elemente, die als individuelle Aufgaben oder Qualitätsdatensätze erstellt werden, werden von der Anwendung initiiert und an die entsprechenden Eigentümer weitergeleitet. Diese Elemente werden im Concierge angezeigt und bieten die Möglichkeit, alle Aufgabenelemente an einer einzigen Stelle anzuzeigen.

Aufrufen des Concierge

Der Concierge ist auf jeder Seite der Anwendung verfügbar. Klicken Sie auf das Concierge-Symbol im rechten oberen Teil der Seite, um den Concierge anzuzeigen oder auszublenden.

Bei Verwendung des Concierge

Der Concierge ist ein einblendbares Steuerelement, in dem Ihre Aufgaben in vier Abschnitte (Kategorien) unterteilt sind. Jedes Element im Concierge ist ein Link. Wenn Sie auf den Link klicken, wird die jeweils erforderliche Seite angezeigt, um das Element anzuzeigen oder zu vervollständigen. Die folgende Liste beschreibt die Kategorien und die aufgerufenen Seiten, wenn Sie auf ein Element in einer Kategorie klicken:

• Meine Zuweisungen - Datensätze, die Ihnen zugewiesen wurden. Wenn Sie in diesem Abschnitt auf ein Element klicken, wird die Verwaltungsseite des Datensatzes angezeigt.
• Meine Genehmigungen - Datensätze, die Sie genehmigen müssen. Wenn Sie in diesem Abschnitt auf ein Element klicken, wird die Seite "Genehmigung unterschreiben" für den Datensatz angezeigt.
• Meine ausstehenden Elemente - Datensätze, die gespeichert aber nicht übermittelt wurden. Wenn Sie in diesem Abschnitt auf ein Element klicken, wird die Seitenfolge für die Ereigniserfassung angezeigt, sodass Sie den Vorgang fortsetzen können.
• Meine Warnhinweise - Benachrichtigungen über Aufgaben oder Datensätze, die Ihnen zugewiesen wurden. Wenn Sie in diesem Abschnitt auf ein Element klicken, wird die entsprechende Seite für das Element geöffnet. Wenn die Meldung z. B. anzeigt, dass ein Datensatz gelöst werden kann, wird durch Klicken auf das Element die Seite "Qualitätsobjektlösung" für den Datensatz aufgerufen.

Der Abschnitt "Meine Zuweisungen" ist standardmäßig erweitert, wenn Sie den Concierge erstmals anzeigen. Durch Klicken auf einen anderen Abschnitt wird dieser erweitert, sodass die Liste der zugehörigen Elemente angezeigt wird, und der Abschnitt "Meine Zuweisungen" wird reduziert. Die Anwendung zeigt standardmäßig höchstens 10 Elemente in jedem Abschnitt an.

Hinweis: Die Anzahl der angezeigten Elemente pro Abschnitt kann in Portal Studio konfiguriert werden. Weitere Informationen finden Sie im Opcenter EX MDD Portal Studio-Benutzerhandbuch bzw. im Opcenter EX CR Portal Studio-Benutzerhandbuch.

Am Ende jeder Elementliste gibt es den Link "Weitere Nachrichten anzeigen". Der Link wird nur angezeigt, wenn es mindestens ein Element im Abschnitt gibt. Durch Klicken auf diesen Link wird die Seite "Nachrichtenzentrale" angezeigt, in der Sie weitere Informationen zu den Elementen in dieser Kategorie anzeigen oder nach anderen Elementen suchen können. Weitere Informationen zur Nachrichtenzentrale finden Sie unter "Verwendung der Nachrichtenzentrale".

Die Anwendung zeigt ein QuickInfo an, wenn Sie die Maus über den Concierge bewegen. QuickInfos enthalten eine vollständige Beschreibung des Elements und das Fälligkeitsdatum (falls ein Fälligkeitsdatum angegeben wurde).

Die Anwendung entfernt ein Element aus dem Concierge, wenn die Aufgabe erledigt ist, der Eigentümer wechselt oder der Datensatz geschlossen wird. Klicken Sie auf die Schaltfläche "Aktualisieren" oben im Concierge, um die angezeigte Liste zu aktualisieren.

Fälligkeitsdatumsindikatoren

Fälligkeitsdatumssymbole zeigen an, wann eine Aufgabe fällig ist. Bewegen Sie die Maus über ein Element im Concierge, um dessen Fälligkeitsdatum (falls zutreffend) im QuickInfo anzuzeigen.

Fälligkeitsdatumsindikatoren werden von einem Systemadministrator in der Modellierung im Modellierungsobjekt "Portalnachrichtenkategorie" konfiguriert. Weitere Informationen finden Sie im Opcenter EX MDD Modeling-Benutzerhandbuch bzw. im Opcenter EX CR Modeling-Benutzerhandbuch.

Vorgehensweise zum Zugreifen auf die Elemente des Concierge

Führen Sie die folgenden Schritte aus, um im Concierge zu navigieren und die Elemente aufzurufen, die Ihre Aufmerksamkeit erfordern:

1. Melden Sie sich bei Opcenter Execution Medical Device and Diagnostics an. Die Homepage erscheint und die Concierge-Leiste wird in der oberen, rechten Ecke angezeigt.

Oder

Melden Sie sich bei Opcenter Execution Core an. Die Homepage erscheint und die Concierge-Leiste wird in der oberen, rechten Ecke angezeigt.

2. Klicken Sie auf das Concierge-Symbol, wenn der Concierge nicht angezeigt wird. Klicken Sie erneut auf dieses Symbol, um den Concierge auszublenden.

Oder

Klicken Sie auf Schließen unten im Bereich "Concierge".

3. Klicken Sie auf ein Element, um die entsprechende Seite aufzurufen und das Element anzuzeigen oder zu bearbeiten. Die Elemente, die Ihnen zugewiesen sind oder deren Eigentümer Sie sind, werden im entsprechenden Abschnitt des Concierge angezeigt. Die entsprechende Seite für den ausgewählten Link wird angezeigt.
4. Klicken Sie auf Weitere Nachrichten anzeigen, um alle Nachrichten einer bestimmten Kategorie anzuzeigen. Die Nachrichtenzentrale wird geöffnet und zeigt alle Nachrichten der betreffenden Kategorie an.
5. Klicken Sie auf den Elementnamen, um auf ein zugewiesenes oder eigenes Element zuzugreifen. Abhängig von Elementtyp und Status wird die entsprechende Seite angezeigt.
