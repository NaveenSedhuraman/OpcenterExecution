E-Mail-Benachrichtigungen

Siemens stellt eine optionale E-Mail-Benachrichtigungsfunktion zur Verfügung, mit der Benutzer benachrichtigt werden können, wenn bestimmte Ereignisse in der Anwendung auftreten oder der Benutzer eine Aktion durchführen muss. Nachstehend sind einige Beispiele aufgeführt, für die ein Administrator eine E-Mail-Benachrichtigung in der Anwendung konfigurieren kann:

• Ein Qualitätselement wurde erstellt und dem Benutzer zugewiesen.
• Die Kategorie eines Qualitätsdatensatzes wurde geändert.
• Der Eigentümer wurde neu zugewiesen.
• Eine Genehmigung ist erforderlich.

Konfigurieren von E-Mail-Benachrichtigungen

Die E-Mail-Nachrichten können so konfiguriert werden, dass abhängig vom Aufgabentyp eine generische Nachricht gesendet wird oder der Identifikator des Qualitätsdatensatzes und die Meldung des Concierge in der E-Mail enthalten sind. Darüber hinaus kann der Systemadministrator die E-Mail so konfigurieren, dass sie an einen bestimmten Benutzer oder eine bestimmte Rolle gesendet wird. Informationen zum Konfigurieren von E-Mail-Benachrichtigungen finden Sie in der Opcenter EX MDD-Anleitung zur Systemadministration bzw. der Opcenter EX CR-Anleitung zur Systemadministration.
