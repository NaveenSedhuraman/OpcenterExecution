Verwendung der Nachrichtenzentrale

Die Nachrichtenzentrale ermöglicht eine zentrale Anzeige aller Elemente (Genehmigungen, Bearbeitungen usw.), für die Sie verantwortlich sind. In der Nachrichtenzentrale werden sowohl Nachrichten des Concierge als auch Nachrichten von Qualitätsdatensätzen angezeigt, die Ihnen als Eigentümer oder aufgrund Ihrer Rolle zugewiesen wurden. Sie können die Suchfunktionen der Nachrichtenzentrale verwenden, um bestimmte Datensätze zu finden.

Beim Aufrufen der Nachrichtenzentrale

Sie können die Nachrichtenzentrale aufrufen, indem Sie im Suchmenü auf "Nachrichtenzentrale" oder im Concierge auf den Link "Weitere Nachrichten anzeigen" klicken. Die Anwendung verwendet standardmäßig die Option "Meine Nachrichten anzeigen" und zeigt Ihren Benutzernamen im Feld "Sachbearbeiter" unabhängig davon an, wie Sie die Seite aufgerufen haben.

Suchmenü

Die Tabelle "Ergebnisse" ist leer, wenn Sie die Nachrichtenzentrale über das Suchmenü aufrufen. Sie müssen eine Suche durchführen, um Nachrichten in der Tabelle "Ergebnisse" anzuzeigen.

Concierge

Die Tabelle "Ergebnisse" enthält eine Liste von Nachrichten, wenn Sie die Nachrichtenzentrale im Concierge aufrufen. Die Anwendung verwendet im Feld "Nachrichtentyp" standardmäßig die Concierge-Kategorie, in der Sie auf den Link "Weitere Nachrichten anzeigen" geklickt haben, und zeigt alle Nachrichten dieser Kategorie in der Tabelle "Ergebnisse" an. Wenn Sie z. B. in der Kategorie "Meine Zuweisungen" auf den Link "Weitere Nachrichten anzeigen" klicken, wird das Feld "Nachrichtentyp" auf "Meine Zuweisungen" gesetzt, und die Anwendung zeigt alle Nachrichten aus "Meine Zuweisungen" in der Tabelle "Ergebnisse" an.

Bei Verwendung der Nachrichtenzentrale

Sie können in der Nachrichtenzentrale die folgenden Aufgaben durchführen:

• Nachrichten anzeigen, die Ihnen zugewiesen wurden
• Nachrichten anzeigen, die Ihrer Rolle (und Ihnen nicht direkt) zugewiesen wurden
• Nachrichten suchen
• Inhalt der Tabelle "Ergebnisse" in eine Excel-Datei exportieren
• Verwaltungs- oder Transaktionsseite für den Qualitätsdatensatz aufrufen

Wenn Sie einen Datensatz in der Tabelle "Ergebnisse" auswählen, werden die relevanten Transaktionsschaltflächen aktiviert. Die nachfolgende Tabelle enthält eine Beschreibung der verfügbaren Transaktionsschaltflächen.

Transaktionsschaltfläche

Funktion der Schaltfläche . . .

Datensätze verwalten

Zeigt die Verwaltungsseite des Datensatzes im Ansichtsmodus an. Die Anwendung aktiviert diese Schaltfläche für alle Qualitätsdatensätze.

Einteilung

Zeigt die Seite "Einteilung" an. Die Anwendung aktiviert diese Schaltfläche, wenn keine manuelle Einteilung für den ausgewählten Datensatz durchgeführt wurde und der Status des Datensatzes nicht "In Klärung" oder "Geschlossen" ist.

Genehmigen

Zeigt die Seite "Genehmigung unterschreiben" an. Die Anwendung aktiviert diese Schaltfläche nur für "Meine Genehmigungen"-Nachrichten.

Hinweis: Ihr Systemadministrator kann weitere Transaktionen für diese Seite konfigurieren.

Wenn Sie auf die Schaltfläche "Alles löschen" klicken, werden die Werte aller Felder in der Nachrichtenzentrale gelöscht (außer Organisation).

Felddefinitionen der Seite "Nachrichtenzentrale"

Diese Tabelle definiert die Felder der Seite "Nachrichtenzentrale".

Feld

Definition

Typ

Meine Nachrichten anzeigen

Option, um nur die Nachrichten anzuzeigen, deren Eigentümer Sie sind. Die ist die Standardeinstellung. Bei Auswahl dieser Option wird das Feld "Sachbearbeiter" deaktiviert und die Anwendung zeigt Ihren Benutzernamen im deaktivierten Feld an.

Optional

Nachrichten für meine Rolle anzeigen

Option, um Nachrichten für die Rolle anzuzeigen, der Sie zugewiesen sind. Die Anwendung zeigt Nachrichten an, bei denen Sie nicht direkt als Eigentümer zugewiesen sind. Bei Auswahl dieser Option wird das Feld "Sachbearbeiter" aktiviert, damit Sie Ihre Suche auf Nachrichten eingrenzen können, die einer bestimmten Rolle zugewiesen wurden.

Optional

Organisation

Organisation, für die der Qualitätsdatensatz übermittelt oder gespeichert wurde. Die verfügbaren Optionen sind auf die Organisationen begrenzt, zu denen der angemeldete Benutzer gehört und in denen der angemeldete Benutzer mindestens eine Rolle hat (entweder direkt oder durch Rollenweitergabe).

Optional

Nachrichtentyp

Kategorie (oder Gruppe) von Nachrichten, die Sie anzeigen möchten. Die Kategorien umfassen:

• Meine Zuweisungen
• Meine Genehmigungen
• Meine offenen Punkte
• Meine Warnhinweise

Optional

Datum von

Anfangsdatum des Datumsbereichs, in dem der Qualitätsdatensatz bearbeitet wurde. Die Ergebnisse enthalten nur Datensätze, die nach diesem Datum bearbeitet wurden. Sie können ein Datum im MM/TT/JJJJ-Format eingeben oder das Datum über das Kalendersymbol auswählen.

Optional

Datum bis

Enddatum des Datumsbereichs, in dem der Qualitätsdatensatz bearbeitet wurde. Die Ergebnisse enthalten nur Datensätze, die vor diesem Datum bearbeitet wurden. Sie können ein Datum im MM/TT/JJJJ-Format eingeben oder das Datum über das Kalendersymbol auswählen.

Bei Angabe eines Datumsbereichs gelten die folgenden Felder für jedes ausgewählte Objekt:

• Qualitätsdatensätze: Berichtsdatum
• Genehmigungseintragsnachrichten: Fälligkeitsdatum

Optional

Rolle

Rolle, die der Nachricht zugeordnet ist. Die verfügbaren Auswahlmöglichkeiten sind auf die Rollen beschränkt, die dem angemeldeten Benutzer zugewiesen wurden.

Optional

Sachbearbeiter

Mitarbeiter (Eigentümer), der der Nachricht zugeordnet ist. Die Anwendung deaktiviert dieses Feld, wenn Sie die Option "Meine Nachrichten anzeigen" auswählen und zeigt den Benutzernamen des angemeldeten Benutzers in diesem Feld an. Die Anwendung aktiviert dieses Feld, wenn Sie die Option "Nachrichten für meine Rolle anzeigen" auswählen. Die verfügbaren Auswahlmöglichkeiten sind auf den im Feld "Rolle" ausgewählten Wert bzw. auf alle Benutzer mit denselben Rollen wie der angemeldete Benutzer beschränkt, falls keine Rolle festgelegt wurde.

Optional

Nachrichten sortieren

Optionen zum Sortieren der zurückgegebenen Nachrichten in der Tabelle "Ergebnisse". Die folgenden Optionen sind verfügbar:

• Fälligkeitsdatum (gilt für Genehmigungen; andere Datensätze werden an das Ende sortiert)
• Nachrichtenkategorie
• Berichtsdatum

Optional

Tabelle "Ergebnisse"

Tabelle mit einer Liste aller Nachrichten, die den angegebenen Suchkriterien entsprechen. Wenn Sie eine Nachricht auswählen, werden die für die Nachricht relevanten Transaktionen aktiviert. Die Anzeigereihenfolge der Nachrichten hängt von dem Wert ab, der ggf. im Feld "Nachrichten Sortieren" angegeben wurde.

Nur Anzeige

Prioritätsindikator

Symbole, die die Priorität des Elements abhängig vom Fälligkeitsdatum anzeigen. Weitere Informationen zu den Symbolen auf der Seite "Nachrichtenzentrale" finden Sie unter "Symbole der Nachrichtenzentrale".

Nur Anzeige

Nachrichtentyp

Kategorie oder Nachrichtengruppe, zu der diese Nachricht gehört.

Nur Anzeige

Kategorie

Art des Qualitätsdatensatzes (Ereignis oder Abweichung), für die diese Nachricht gilt.

Nur Anzeige

Identifikator

Identifikationsnummer, die von der Anwendung für den Qualitätsdatensatz erstellt wird. Die Anwendung verwendet die im Modellierungsobjekt "Organisation" angegebene Nummerierungsregel, um ggf. Präfix oder Suffix sowie die Sequenzlänge zu bestimmen.

Nur Anzeige

Status

Aktueller Status des Qualitätsdatensatzes (z. B. "Aktiv", "Ausstehend", "Weitergeleitet" usw.).

Nur Anzeige

Rolle

Rolle, die dem Qualitätsdatensatz zugewiesen wurde.

Nur Anzeige

Sachbearbeiter

Mitarbeiter, dem der Qualitätsdatensatz zugewiesen wurde.

Nur Anzeige

Nachricht

Nachricht, die dem Qualitätsdatensatz zugeordnet wurde. Bewegen Sie die Maus über die Nachricht, um die vollständige Nachricht in einem frei beweglichen QuickInfo anzuzeigen, oder klicken Sie auf das Erweiterungssymbol, um die vollständige Nachricht in einem erweiterten QuickInfo anzuzeigen.

Nur Anzeige

Symbole der Nachrichtenzentrale

Diese Tabelle beschreibt die Symbole auf der Seite "Nachrichtenzentrale".

Symbol

Beschreibung

Datumsauswahlfunktion, mit der Sie ein Datum (und ggf. eine Uhrzeit) auswählen können.

Das Element ist überfällig.

Das Element ist innerhalb der angegebenen Anzahl von Tagen fällig. Die Anwendung zeigt dieses Symbol standardmäßig für den Tag an, an dem die Elemente fällig sind. Ein Systemadministrator kann die Anwendung aber so konfigurieren, dass das Symbol vor dem Fälligkeitsdatum des Elements angezeigt wird. Ein Systemadministrator kann die Anwendung z. B. so konfigurieren, dass das Symbol 2 Tage vor Fälligkeit angezeigt wird.

Hinweis: Die Anwendung zeigt das Fälligkeitsdatum im QuickInfo an, das angezeigt wird, wenn Sie die Maus über ein Element mit diesem Symbol bewegen.

Das Element ist irgendwann in der Zukunft fällig oder es wurde kein bestimmtes Fälligkeitsdatum angegeben.

Vorgehensweise zum Suchen nach Nachrichten

Führen Sie die folgenden Schritte aus, um nach einer Nachricht zu suchen:

1. Rufen Sie die Seite "Nachrichtenzentrale" auf. Die Seite "Nachrichtenzentrale" wird angezeigt.
2. Enthält die Tabelle "Ergebnisse" Nachrichten?

Dann . . .

Ja

Sie haben die Nachrichtenzentrale vom Concierge aus aufgerufen. In der Tabelle "Ergebnisse" werden alle Nachrichten angezeigt, denen Sie in der betreffenden Concierge-Kategorie als Eigentümer zugewiesen wurden.

Führen Sie eine der folgenden Aktionen durch:

• Wählen Sie eine Nachricht in der Tabelle "Ergebnisse" aus, um eine Transaktion für den zugehörigen Qualitätsdatensatz durchzuführen.
• Klicken Sie auf die Schaltfläche Alles löschen, um eine neue Suche durchzuführen, und gehen Sie dann zu Schritt 3. Die Anwendung löscht die Suchkriterienfelder und die Ergebnistabelle.

Nein

Sie haben die Nachrichtenzentrale vom Suchmenü aus aufgerufen. Gehen Sie zu Schritt 3.

3. Geben Sie Parameter an, um Ihre Suche einzugrenzen.
4. Klicken Sie auf Suchen. Die Anwendung zeigt in der Tabelle "Ergebnisse" alle Nachrichten an, die Ihre Suchkriterien erfüllen.
