Löschen eines gespeicherten Ereignisses

Sie können Ereignisse löschen, die gespeichert wurden, aber keine Ereignisse, die übermittelt wurden. Über die Schaltfläche "Löschen" können Sie auf einer beliebigen Seite in der Seitenfolge für die Aufzeichnung von generischen Ereignissen gespeicherte Ereignisse löschen.

Hinweis: Übermittelte Ereignisse müssen geschlossen sein. Weitere Informationen zum Schließen (Lösen) von Ereignisdatensätzen finden Sie unter "Verwalten von Datensätzen".

Beim Löschen eines gespeicherten Ereignisses

Die Schaltfläche "Löschen" ist nur für gespeicherte Ereignisse aktiviert. Wenn Sie ein gespeichertes Ereignis löschen, entfernt die Anwendung das Ereignis auch aus dem Concierge.

Gelöschte Ereignisse können bei der Suche von Qualitätsdatensätzen angezeigt werden, sie weisen dann aber den Status "Gelöscht" auf. Gelöschte Ereignisse können nicht an andere Qualitätsdatensätze angehängt werden.

Vorgehensweise zum Löschen eines gespeicherten Ereignisses

Führen Sie die folgenden Schritte aus, um ein gespeichertes Ereignis zu löschen:

1. Klicken Sie ggf. auf das Symbol "Concierge", um den Concierge aufzurufen.
2. Klicken Sie in den Abschnitt Meine ausstehenden Elemente, um ausstehende Elemente anzuzeigen.
3. Klicken Sie auf das ausstehende Element. Die entsprechende Seite wird in der Seitenfolge für die Ereignisaufzeichnung angezeigt.
4. Klicken Sie auf die Schaltfläche Löschen. Daraufhin wird eine Warnmeldung mit der Aufforderung angezeigt, das Löschen des Datensatzes zu bestätigen.
5. Klicken Sie auf OK. Die Seite wird geschlossen. Der Datensatz wird gelöscht und nicht mehr im Concierge angezeigt.
