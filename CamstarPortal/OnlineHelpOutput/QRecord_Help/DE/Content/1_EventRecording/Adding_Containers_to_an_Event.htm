Hinzufügen von Containern (Losen) zu einem Ereignis

Die Seite "Betroffenes Material" bietet die Möglichkeit, ggf. Container (Lose) mit betroffenem Material zum Ereignis hinzuzufügen. Sie können auch Container hinzufügen, die nicht vom Manufacturing Execution System (MES) überwacht werden. Die Seite "Betroffenes Material" ist die dritte Seite in der Seitenfolge für die Aufzeichnung von generischen Ereignissen.

Beim Hinzufügen von betroffenem Material

Die Seite "Betroffenes Material" umfasst zwei Unterregisterkarten:

• Suchen – Die Unterregisterkarte "Suchen" stellt Filter für die Suche von Containern zur Verfügung, die Sie zum Ereignis hinzufügen möchten.
• Manuell hinzufügen – Die Unterregisterkarte "Manuell hinzufügen" bietet die Möglichkeit, Container zu scannen oder einzugeben, die Sie zum Ereignis hinzufügen möchten. Sie können diese Unterregisterkarte verwenden, um Container oder Lose hinzuzufügen, die nicht vom MES überwacht werden.

Ein Container kann zu mehr als einem Ereignis hinzugefügt werden. Beim Hinzufügen eines Eltern-Containers zu einem Ereignis wird der Eltern-Container im Ereignis aufgelistet, Kind-Container werden jedoch nicht angezeigt. Beim Hinzufügen eines Kind-Containers wird der Kind-Container im Ereignis aufgelistet, Eltern-Container und Geschwister-Container werden jedoch nicht angezeigt.

Ein Systemadministrator kann Transaktionen, z. B. "Verbinden", "Vereinigen" und "Aufteilen", in der Tabelle "Betroffene Container zu Ereignissen hinzufügen" im Modellierungsobjekt "Organisation" angeben. Die Anwendung fügt betroffene Container automatisch zu einem Ereignis hinzu, wenn eine der angegebenen Transaktionen für einen Container ausgeführt wird, der in der Tabelle "Ereignis > Betroffenes Material" aufgelistet ist. Weitere Informationen finden Sie unter "Hinzufügen von betroffenen Containern zu Ereignissen".

Beim Suchen nach Containern

Die Anwendung zeigt alle Container (Lose) an, die den eingegebenen Kriterien entsprechen, einschließlich der Container, die bereits zu anderen Ereignisdatensätzen hinzugefügt wurden. Sie können die zurückgegebenen Container auswählen, um sie zum Ereignis hinzuzufügen.

Die Anwendung stellt ein Kontrollkästchen zur Verfügung, mit dem Sie alle zurückgegebenen Container auf der Seite auswählen und die betreffenden Container gleichzeitig zum Ereignis hinzufügen können. Sie können Container auf der Seite einzeln auswählen. Wenn Sie auf die Schaltfläche "Ausgewählte Container hinzufügen" klicken, werden die ausgewählten Container zur Tabelle "Ereignis > Betroffenes Material" hinzugefügt.

Die Anwendung gibt für eine Suche maximal 1000 Container zurück. Die Anzahl der jeweils angezeigten Ergebnisse hängt jedoch davon ab, wie die Anwendung konfiguriert wurde. Falls die Suche mehr Lose zurückgibt als das für die Tabelle konfigurierte Zeilenlimit, zeigt die Anwendung Seitenwechselsymbole an. Informationen zum Konfigurieren der Anzahl der in einer Tabelle angezeigten Zeilen finden Sie im Opcenter EX MDD Portal Studio-Benutzerhandbuch bzw. im Opcenter EX CR Portal Studio-Benutzerhandbuch.

Löschen von Suchergebnissen

Mit der Schaltfläche "Alles löschen" können Sie Filter (Parameter) und Ergebnisse auf der Unterregisterkarte "Suchen" löschen. Wenn Sie den Inhalt der Unterregisterkarte "Suchen" gelöscht haben, können Sie eine neue Suche starten. Wenn Sie auf "Alles löschen" klicken, reagiert die Anwendung folgendermaßen:

• Es werden alle angegebenen Suchfilter gelöscht.
• Es werden alle Container aus der Suchergebnistabelle gelöscht.
• Es werden keine hinzugefügten Container aus der Tabelle "Betroffenes Material" entfernt.
• Die Unterregisterkarte "Suche" wird weiterhin angezeigt.

Beim manuellen Hinzufügen von Containern

Die Unterregisterkarte "Manuell hinzufügen" bietet die Möglichkeit, Container oder Lose hinzuzufügen, die von Ihrem MES überwacht werden, und solche, die nicht überwacht werden. Sie können den Containernamen scannen oder eingeben. Die Anwendung kann andere Felder füllen, wenn der eingegebene Container von Ihrem MES überwacht wird und die betreffenden Informationen bereits vorher für den Container eingegeben wurden. Wenn Sie auf die Schaltfläche "Container hinzufügen" klicken, werden der Container (das Los) zur Tabelle "Ereignis > Betroffenes Material" hinzugefügt und die Losfelder gelöscht.

Aktualisieren manuell hinzugefügter Container

Wenn Sie eine Zeile der Tabelle "Ereignis > Betroffenes Material" auswählen, trägt die Anwendung die Containerinformationen in die vorangehenden Felder ein und ersetzt die Schaltfläche "Container hinzufügen" durch die Schaltfläche "Ausgewählten Container aktualisieren". Wenn Sie auf "Ausgewählten Container aktualisieren" klicken, werden Ihre Änderungen gespeichert, die Containerfelder gelöscht und die Schaltfläche "Container hinzufügen" angezeigt.

Wenn Sie die Unterregisterkarte "Manuell hinzufügen" verwenden, um Container hinzuzufügen, die vom MES überwacht werden, werden die Container auf dieser Unterregisterkarte nicht aktualisiert. Verwenden Sie die Transaktion "Containerwartung", wenn Sie Attribute dieser Container aktualisieren müssen.

Seite "Betroffenes Material" mit Unterregisterkarte "Suchen"

Die Seite "Betroffenes Material" wird angezeigt, wenn Sie auf der Seite "Fehler" auf "Weiter" oder in der Seitenfolge für die Aufzeichnung eines generischen Ereignisses auf den dritten Schritt klicken. Die Unterregisterkarte "Suchen" ist die Standardregisterkarte. Sie können diese Seite auch über eine gespeicherte Seitenfolge aufrufen.

Vorgehensweise zum Hinzufügen eines Container zu einem Ereignis

Führen Sie die folgenden Schritte aus, um einen Container zum Ereignis hinzuzufügen:

1. Führen Sie die Schritte in "Vorgehensweise zum Eingeben von Fehlerinformationen" durch, um auf die Seite "Betroffenes Material" zuzugreifen.
2. Möchten Sie nach auszuwählenden Containern suchen oder Container manuell hinzufügen?

Sie möchten . . .

Dann . . .

Auszuwählende Container suchen
a. Geben Sie Parameter an, um Ihre Suche einzugrenzen.
b. Klicken Sie auf Suchen. Die Anwendung zeigt alle Container, die den Suchkriterien entsprechen, in der Tabelle "Suchergebnisse" an.
c. Wählen Sie mindestens einen Container in der Tabelle Suchergebnisse aus.
d. Klicken Sie auf Ausgewählte Container hinzufügen. Der Container wird der Tabelle "Ereignis > Betroffenes Material" hinzugefügt.
e. Klicken Sie auf die Schaltfläche Alles löschen, um den Inhalt der Parameterfelder und der Tabelle "Suchergebnisse" zu löschen, wenn Sie eine neue Suche durchführen möchten.

Oder

Gehen Sie zu Schritt 3.

Container manuell hinzufügen
a. Klicken Sie auf die Unterregisterkarte Manuell hinzufügen.
b. Geben Sie den Namen des Containers/Loses ein. Die Anwendung aktualisiert die Felder "Menge" und "Maßeinheit", wenn Sie die Felder verlassen oder ein anderes Feld auswählen.
c. Aktualisieren Sie ggf. die Menge des Containers oder Loses.
d. Aktualisieren Sie ggf. die Maßeinheit für die Mengenangabe.
e. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. Informationen zu den optionalen Feldern finden Sie in der Tabelle unter Felddefinitionen der Seite "Betroffenes Material", Unterregisterkarte "Manuell hinzufügen".
f. Klicken Sie auf Container hinzufügen. Der Container wird der Tabelle "Ereignis > Betroffenes Material" hinzugefügt.
g. Wiederholen Sie die Schritte b-f, um weitere Container hinzuzufügen.
h. Gehen Sie zu Schritt 3.
3. Klicken Sie auf Weiter (Next). Die Seite "Anhänge" wird angezeigt.
