Eingeben von Klassifikations- und allgemeinen Informationen

Die Seite "Klassifikation" ist die erste Seite in der Seitenfolge für die Aufzeichnung von generischen Ereignissen. Auf der Seite können Sie die Organisation, für die Sie das Ereignis aufzeichnen, sowie eine Klassifikation und Unterklassifikation innerhalb dieser Organisation angeben, um das Ereignis zu klassifizieren. Darüber hinaus können Sie allgemeine Informationen aufzeichnen, z. B.:

• Kurze und ausführliche Beschreibung des Ereignisses
• Prioritätsstufe
• Datum und Uhrzeit, zu der das Ereignis aufgetreten ist
• Entdeckungsbereich

Die Organisation ist standardmäßig auf Ihre primäre Organisation (die Organisation des angemeldeten Mitarbeiters) eingestellt. Es können nur die Organisationen ausgewählt werden, in denen Sie direkt oder durch Weitergabe eine Rolle haben.

Beim Eingeben von Klassifikations- und allgemeinen Informationen

Die Kombinationen von Klassifikation und Unterklassifikation werden in der Tabelle "Zuordnungen von Ereignisklassifizierungspez." des Modellierungsobjekts "Organisation" definiert. Wenn nur eine Kombination für die Organisation definiert wurde, verwendet die Anwendung diese Klassifikation und Unterklassifikation als Standardwert.

Sie können die Klassifikation und Unterklassifikation ändern, bevor Sie das Ereignis speichern. Dabei gehen aber die anderen Daten verloren, die Sie auf dieser Seite eingegebenen haben.

Die bei der Aufzeichnung eines Ereignisses verwendeten Prioritätsstufen werden in der Modellierung erstellt.

Das Feld "Entdeckungsbereich" stellt eine Liste von auswählbaren Bereichen zur Verfügung, die vorher in der Modellierung definiert wurden, z. B. Unternehmen, Fabrik und Arbeitsbereich. Diese Liste wird in einer hierarchischen Ansicht dargestellt. Sie können in diesem Feld auch einen neuen Bereich eingeben.

Die Felder "Urheber" und "Organisation" enthalten standardmäßig die Informationen für den bei der Anwendung angemeldeten Benutzer. Sie können nur die Organisationen auswählen, in denen Sie direkt oder durch Weitergabe eine Rolle haben.

Sie können Ihre Informationen an dieser Stelle speichern und später zurückkehren, um weitere Informationen zu erfassen. Sie können auch auf "Weiter" klicken, um zur nächsten Seite in der Seitenfolge zu gelangen. Sobald Sie jedoch das Ereignis gespeichert haben, können Sie die Klassifikation und Unterklassifikation nicht mehr ändern.

Texteditor

Siemens stellt einen Texteditor zur Verfügung, mit dem Sie Beschreibungen formatieren können, wenn Sie ein Ereignis aufzeichnen. Der Text-Editor unterstützt übliche Textformatierungen wie Schriftgröße, Schriftfarbe und Schriftart, Textausrichtung, gegliederte und nummerierte Listen, Markierungen sowie das Einfügen von Bildern. Informationen zum Einfügen von Bildern mit dem Texteditor finden Sie im Opcenter EX MDD Modeling-Benutzerhandbuch bzw. im Opcenter EX CR Modeling-Benutzerhandbuch.

Siemens empfiehlt, die vorhandene Beschreibung zu markieren und sie dann entsprechend zu formatieren. Wenn Sie keine Formatierungsattribute auswählen, verwendet die Anwendung Standardschriftgröße, -schriftstil und -farbe. Wenn Sie den Datensatz anzeigen, zeigt die Anwendung die verwendete Formatierung an.

Sie können den Text-Editor entweder im Design- oder im HTML-Modus benutzen. Wenn Sie Text im Design-Modus kopieren, werden die verborgenen Tags nicht kopiert. Das bedeutet, dass das Format unter Umständen nicht beibehalten wird, wenn Sie kopierten Text wieder einfügen. Allerdings können Sie die verborgenen Tags kopieren, indem Sie in die HTML-Ansicht wechseln.

Es gibt eine Begrenzung von 4.000 Zeichen im Feld "Anweisung", einschließlich des eingebetteten HTML-Codes. Wenn Sie Text aus einer anderen Anwendung (z. B. Microsoft Word) kopieren und ihn in dieses Feld einfügen, wird verborgener HTML-Code zusätzlich zum Text in das Feld eingefügt. Daher kann es passieren, dass Sie über der Grenze liegen, auch wenn der Text weniger als 4.000 Zeichen enthält.

Hinweis: Google Chrome unterstützt nicht die Schaltflächen "Kopieren" und "Einfügen" des Texteditors. Sie können die Tastenkombinationen zum Kopieren und Einfügen (Strg+C und Strg+V) verwenden, wenn Sie mit Google Chrome arbeiten.

Design-Modus

Der Design-Modus ist ein WYSIWYG-Texteditor (What You See Is What You Get), der die Anweisungen so anzeigt, wie sie später bei der Anzeige eines Datensatzes angezeigt werden.

HTML-Modus

Wenn Sie auf die Schaltfläche "HTML" klicken, wird der Quellcode angezeigt, der auf die Anweisungen angewandt wird. Da ein WYSIWYG-Editor Markup-Code (Tags) automatisch einfügen kann, können Sie den Code im HTML-Modus überprüfen und ggf. verändern.

Seite "Klassifikation"

Die Seite "Klassifikation" wird angezeigt, wenn Sie "Generisches Ereignis aufzeichnen" im Menü "Ereignis" auswählen oder auf den ersten Schritt in einer Ereignisseitenfolge klicken. Sie können diese Seite auch über eine gespeicherte Seitenfolge aufrufen.

Vorgehensweise zum Eingeben von Informationen für Klassifikation und Beschreibung

Führen Sie die folgenden Schritte aus, um Informationen für Klassifikation und Beschreibung einzugeben:

1. Wählen Sie Generisches Ereignis aufzeichnen im Menü Ereignis aus. Die Seite "Generisches Ereignis aufzeichnen: Klassifikation" wird angezeigt.
2. Wählen Sie eine Organisation in der Liste aus oder übernehmen Sie die Standardorganisation.
3. Wählen Sie eine Klassifikation aus. Die Anwendung aktualisiert die Liste "Unterklassifikation" entsprechend.
4. Wählen Sie eine Unterklassifikation aus.
5. Geben Sie eine Kurzbeschreibung und eine Beschreibung für das Ereignis ein.
6. Geben Sie eine Prioritätsstufe an.
7. Wählen Sie Datum und Zeit, zu der das Ereignis aufgetreten ist, im Feld Datum und Zeit des Auftretens aus, falls die Angaben vom Standardwert abweichen.
8. Geben Sie einen Entdeckungsbereich an.
9. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. In der Felddefinitionstabelle finden Sie Informationen zu den optionalen Feldern.
10. Klicken Sie auf Weiter, um zur nächsten Seite der Seitenfolge zu gelangen. Die Seite "Fehler" wird angezeigt.
Sie können auch auf Speichern klicken, um Ihre Änderungen zu speichern, bevor Sie zur nächsten Seite wechseln.
