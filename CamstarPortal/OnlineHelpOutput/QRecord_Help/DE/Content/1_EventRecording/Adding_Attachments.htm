Hinzufügen von Anhängen

Die Seite "Anhänge" ist die letzte Seite in der Seitenfolge für die Aufzeichnung von generischen Ereignissen. Auf der Seite "Anhänge" können Sie Anhänge für das Ereignis hochladen. Sie ist die einzige Seite, auf der Sie den Ereignisdatensatz an die Anwendung übermitteln oder ein gespeichertes Ereignis löschen können. Weitere Informationen finden Sie unter "Löschen eines gespeicherten Ereignisses".

Anhänge stellen zusätzliche Informationen zur Verfügung, die erklären können, warum das Ereignis erstellt wurde, welche Standardschritte für ein solches Ereignis durchgeführt werden müssen usw. Hochgeladene Anhänge werden in der Tabelle "Anhänge" angezeigt. Sie können in dieser Tabelle eine Datei auswählen, um sie anzuzeigen, zu bearbeiten oder zu löschen.

Beim Hinzufügen von Anhängen

Wenn Sie eine Datei hinzufügen, wird der Qualitätsdatensatz automatisch gespeichert.

Wenn Sie Anhänge hinzufügen, verhindert die Anwendung, dass Sie folgende Zeichen im Titel eingeben: +, &, #.

Die maximale Dateigröße für einen Anhang beträgt 30 MB.

Beim Anzeigen von Anhängen

Sie benötigen die richtige Software, um die Datei anzuzeigen. Falls der Anhang z. B. eine Excel-Tabelle ist, muss Microsoft Excel installiert sein, um die Datei anzuzeigen.

Wenn Sie remote auf die Datei zugreifen, müssen Sie u. U. im Browser zusätzliche Schritte durchführen, um die Datei herunterzuladen. Z. B. blockiert Internet Explorer möglicherweise das Herunterladen der Datei, bis Sie in der IE-Leiste auf "Datei herunterladen" geklickt haben.

Beim Bearbeiten von Anhangsinformationen

Das einzige Feld, das Sie beim Bearbeiten von Anhangsinformationen aktualisieren können, ist "Beschreibung".

Seite "Anhänge"

Die Seite "Anhänge" wird angezeigt, wenn Sie auf der Seite "Betroffenes Material" auf "Weiter" oder in der Seitenfolge auf den letzten Schritt klicken. Sie können diese Seite auch über eine gespeicherte Seitenfolge aufrufen.

Vorgehensweise zum Anhängen einer Datei

Führen Sie die folgenden Schritte aus, um eine Datei anzuhängen:

1. Führen Sie die Schritte in "Vorgehensweise zum Hinzufügen eines Container zu einem Ereignis" durch, um auf die Seite "Anhänge" zuzugreifen.
2. Klicken Sie auf Neue Zeile hinzufügen. Das Popup "Anhänge hinzufügen" wird angezeigt.
3. Geben Sie den Pfad zur Datei im Feld Originaler Dateipfad ein oder klicken Sie auf Durchsuchen, um die Datei auszuwählen.
4. Geben Sie den Titel der Datei ein.
5. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. In der Felddefinitionstabelle finden Sie Informationen zu den optionalen Feldern.
6. Klicken Sie auf Hochladen. Die Dateiinformationen werden zur Tabelle "Anhänge" hinzugefügt.
7. Wiederholen Sie die Schritte 2 bis 6, um weitere Dateien hinzuzufügen.
8. Klicken Sie auf Speichern.

Vorgehensweise zum Anzeigen der angehängten Datei

Führen Sie die folgenden Schritte aus, um die angehängte Datei anzuzeigen:

1. Wählen Sie den Anhang aus, den Sie anzeigen möchten.
2. Klicken Sie auf Anhang anzeigen. Daraufhin wird ein Popup mit der Aufforderung angezeigt, den Anhang zu speichern oder zu öffnen.
3. Klicken Sie auf Öffnen und dann auf OK. Die Datei wird zur Anzeige in einer separaten Seite geöffnet.

Vorgehensweise zum Ändern von Anhangsinformationen

Führen Sie die folgenden Schritte aus, um Anhangsinformationen zu bearbeiten:

1. Wählen Sie den Anhang aus, den Sie bearbeiten möchten.
2. Klicken Sie auf Ausgewählte Zeile bearbeiten.. Das Popup "Bearbeiten" wird angezeigt.
3. Aktualisieren Sie das Feld Beschreibung je nach Bedarf.
4. Klicken Sie auf Hochladen. Die Dateiinformationen werden in der Tabelle "Anhänge" aktualisiert.
