Durchführen einer Einteilung

Siemens bietet die Möglichkeit, Qualitätsdatensätze einzuteilen. Eine Einteilung kann automatisch oder manuell erfolgen und ist in der Regel die erste Aktion, die für einen Datensatz durchgeführt wird.

Die Einteilung ermöglicht die Durchführung folgender Aktionen für einen Datensatz:

• Kategorie ändern
• Rolle und Eigentümer ändern
• Prioritätsstufe ändern
• Checkliste zuweisen

Die Einteilung kann z. B. die Kategorie eines Qualitätsdatensatzes von "Ereignis" in "Abweichung" ändern.

Hinweis: Beim Ändern der Kategorie eines Qualitätsdatensatzes wird auch der Ereignistyp geändert. Wenn Sie z. B. die Kategorie eines Datensatzes von "Ereignis" in "Abweichung" ändern, wird das generische Ereignis zu einem Produktionsereignis.

Automatische Einteilung

Die automatische Einteilung beginnt, sobald Sie einen Qualitätsdatensatz erstellen, wenn ein Systemadministrator die Geschäftsregeln in einer Einteilungsspezifikation so eingerichtet hat, dass diese bei der Erstellung eines Ereignisses ausgeführt werden. Unabhängig davon, ob eine automatische Einteilung stattfindet, können Sie den Datensatz später manuell einteilen.

Die Anwendung ermittelt die auszuführenden Geschäftsregeln folgendermaßen:

• Verwendung der Kategoriezuordnung der Organisation, um die Einteilungsspezifikation für die angegebene Kategorie zu ermitteln.
• Ermittlung, ob an die Einteilungsspezifikation eine oder mehrere Geschäftsregeln angehängt sind. Die Anwendung führt diese nacheinander aus, sofern Geschäftsregeln vorhanden sind.

Manuelle Einteilung

Sie können eine Einteilung manuell durchführen, nachdem die Anwendung eine automatische Einteilung vorgenommen hat. Die Anwendung bietet die Möglichkeit, für einen Datensatz nur einmal eine Einteilung durchzuführen. Sie können eine Seite nicht manuell einteilen, wenn Sie auf der Seite "Kategorie ändern" Kategorien ändern.

Sie können eine manuelle Einteilung in der Nachrichtenzentrale, auf der Seite "Qualitätssuche" und auf der Verwaltungsseite des Qualitätsdatensatzes einleiten. Die Anwendung zeigt die Seite "Einteilung" für den betreffenden Ereignisdatensatz an. Auf der Seite "Einteilung" werden die Standardwerte angezeigt, die in der Einteilungsspezifikation festgelegt wurden, falls die Einteilung bei der Ereigniserstellung automatisch durchgeführt wurde. Benutzer mit Eigentümerrechten können die Standardwerte bei der manuellen Einteilung überschreiben.

Voraussetzungen für eine Einteilung

Sie müssen möglicherweise folgende Modellierungsobjekte einrichten, um automatische und manuelle Einteilung zu ermöglichen, damit Sie eine Einteilung durchführen können:

• Geschäftsregel
• Mitarbeiter
• Organisation
• Rolle
• Einteilungsspez.

Informationen zum Definieren dieser Modellierungsobjekte finden Sie im Opcenter EX MDD Modeling-Benutzerhandbuch bzw. im Opcenter EX CR Modeling-Benutzerhandbuch.

Beim Durchführen einer Einteilung

Die Anwendung sieht die Einteilung als beendet an, wenn Sie auf der Seite "Einteilung" eine der folgenden Aktionen durchführen:

• Eigentümer zuweisen
• Kategorie des Datensatzes ändern
Hinweis: Die Anwendung fordert Sie auf, eine Rolle und einen Eigentümer zuzuweisen, damit Sie eine andere Aktion für den Datensatz durchführen können, wenn bei der automatischen Einteilung kein Eigentümer festgelegt wurde. Daher sieht die Anwendung die Einteilung als beendet an, wenn Sie einen Eigentümer zuweisen.

Die Anwendung zeigt eine Warnmeldung an, wenn Sie versuchen, dem Qualitätsdatensatz eine Checkliste zuzuweisen, aber bereits eine Checkliste zugewiesen wurde. Die Warnmeldung wird angezeigt, wenn Sie auf "Abschicken" klicken. Wenn Sie auf "Ja" klicken, überschreibt die Anwendung die zugewiesene Checkliste und alle für die betreffende Checkliste gespeicherten Daten. Wenn Sie auf "Nein" klicken, können Sie die angegebene Checklistenvorlage entfernen, bevor Sie erneut auf "Abschicken" klicken.

Die Anwendung verwirft die bei einer manuellen Einteilung eingegebene Einteilung in folgenden Situationen:

• Abmelden bei der Anwendung
• Verlassen der Seite "Einteilung"
• Klicken auf die Schaltfläche "Abbrechen" auf der Seite "Einteilung"
• Zulassen eines Anwendungstimeouts

Rolle und Eigentümer

Die Auswahlliste "Rolle" ist nicht auf die Rollen beschränkt, die der Organisation des Qualitätsdatensatzes zugeordnet sind. Demgegenüber ist die Auswahlliste "Eigentümer" auf die Mitarbeiter beschränkt, denen die ausgewählte Rolle für die Organisation des Qualitätsdatensatzes zugewiesen wurde. Die entsprechenden Mitarbeiter können explizit für die Organisation angegeben oder von einer übergeordneten Organisation übernommen werden.

Hinweis: Die Anwendung zeigt eine Fehlermeldung mit dem Hinweis an, dass ein Benutzer nicht zum Ändern des Datensatzes berechtigt ist, wenn der ursprüngliche Eigentümer des Datensatzes nicht der aktuelle Benutzer oder ein Benutzer mit derselben Rolle wie der aktuelle Benutzer ist. Der Eigentümer kann nur dann neu zugewiesen werden, wenn das Ereignis dem aktuellen Benutzer zugewiesen wurde oder der neue Eigentümer die gleiche Rolle hat.

Beim Aufrufen der Seite "Einteilung"

Sie können die Seite "Einteilung" an folgenden Stellen aufrufen:

• Verwaltungsseite des Datensatzes
• Nachrichtenzentrale
• Seite "Qualitätssuche"

Die Aktion "Einteilung" ist nur für Ereignisdatensätze verfügbar, die noch nicht manuell eingeteilt wurden.

Wenn Sie auf "Abschicken" oder "Abbrechen" klicken, kehren Sie zu der Seite zurück, auf der Sie die Seite "Einteilung" aufgerufen haben.

Vorgehensweise zum Durchführen einer manuellen Einteilung für einen Qualitätsdatensatz

Führen Sie die folgenden Schritte aus, um eine manuelle Einteilung für einen Qualitätsdatensatz durchzuführen:

1. Führen Sie einen der folgenden Schritte aus, um den Qualitätsdatensatz auszuwählen, den Sie bearbeiten möchten:
• Zeigen Sie die Verwaltungsseite des Datensatzes an.
• Wählen Sie den Datensatz in der Tabelle "Suchergebnisse" auf der Seite der "Qualitätssuche" aus.
• Wählen Sie den Datensatz in der Tabelle "Ergebnisse" der Nachrichtenzentrale aus.
2. Klicken Sie auf Einteilung. Die Seite "Einteilung" wird angezeigt.
3. Wählen Sie eine Kategorie aus.
4. Wählen Sie eine Rolle aus.
5. Wählen Sie einen Eigentümer aus.
6. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. In der Felddefinitionstabelle finden Sie Informationen zu den optionalen Feldern.
7. Klicken Sie auf Einreichen (Submit). Die Anwendung zeigt eine Meldung mit dem Hinweis an, dass der Datensatz eingeteilt wurde. Anschließend wird die Seite angezeigt, auf der Sie die Seite "Einteilung" aufgerufen haben.
