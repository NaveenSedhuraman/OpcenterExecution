Genehmigen einer Qualitätsdatensatzlösung

Verwenden Sie die Aktion "Genehmigen", um eine Qualitätsdatensatzlösung zu genehmigen.

Sie können eine Qualitätsdatensatzlösung genehmigen, wenn folgende Bedingungen erfüllt sind:

• Der Datensatzsatzstatus ist "In Klärung".
• Sie sind als Genehmiger angegeben.

Die Anwendung deaktiviert die Schaltfläche "Genehmigung", wenn eine der beiden obigen Bedingungen nicht erfüllt ist.

Hinweis: Nachdem alle Genehmigungen der Qualitätsdatensatzlösung durchgeführt wurden, können Sie die aktuellen Genehmigungen auf der Seite "Qualitätsobjektlösung" oder der Registerkarte "Lösung" der Datensatzverwaltungsseite anzeigen.

Weitere Informationen zur Durchführung von Genehmigungen finden Sie unter "Weiterleiten und Erfassen von Genehmigungen".
