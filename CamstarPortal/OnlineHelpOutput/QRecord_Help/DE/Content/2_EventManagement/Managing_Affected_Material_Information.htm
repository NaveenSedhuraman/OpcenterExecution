Verwalten von Informationen zu betroffenem Material

Die Registerkarte "Betroffenes Material" bietet die Möglichkeit, Container mit betroffenem Material anzuzeigen und zu einem Qualitätsdatensatz hinzuzufügen. Sie können nach Containern suchen und als betroffenes Material zum Ereignisdatensatz hinzufügen. Sie können auch Container hinzufügen, die nicht von MES überwacht werden.

Beim Ändern von Informationen zu betroffenem Material

Die Registerkarte "Betroffenes Material" umfasst zwei Unterregisterkarten: "Suchen" und "Manuell hinzufügen".

Klicken Sie auf die Schaltfläche "Betroffenes Material aktualisieren", um alle Informationen zu speichern, die Sie auf der Registerkarte "Betroffenes Material" hinzugefügt oder geändert haben.

Unterregisterkarte "Suchen"

Die Unterregisterkarte "Suchen" stellt Filter für die Suche von Containern zur Verfügung, die Sie zum Ereignis hinzufügen möchten. Diese Unterregisterkarte umfasst drei Suchoptionen:

• Nur Container zurückgeben - Gibt alle Container zurück, die den Suchkriterien entsprechen. Dies ist die Standardoption.
• Container zurückgeben, die den Suchkriterien entsprechen, zusammen mit dem auf der ersten Ebene verbrauchten Material als Zusatzzeile - Gibt alle Container, die den Suchkriterien entsprechen, und alle in diesen Containern verbrauchten Materialien zurück. Die Anwendung zeigt die Container, die den Suchkriterien entsprechen, als übergeordnete Zeilen und alle verbrauchten Materialien der jeweiligen Container als untergeordnete Zeilen an. Die übergeordneten Zeilen sind standardmäßig reduziert. (Beachten Sie, dass verbrauchte Materialien, die in den verbrauchten Materialien verbraucht wurden, nicht angezeigt werden.)
• Container zurückgeben, für die das verbrauchte Material den Suchkriterien entspricht - Gibt die Container zurück, in denen der angegebene Container verbraucht wurde. Bei Auswahl dieser Option werden Felder wie folgt aktualisiert und hinzugefügt:
• Das Feld "Produkt" wird in "Verbrauchtes Produkt" umbenannt.
• Die Felder "Aus verbrauchtem Los" und "Aus Verbrauchslagerort" werden hinzugefügt.

Löschen von Suchergebnissen

Mit der Schaltfläche "Alles löschen" können Sie Filter (Parameter) und Ergebnisse auf der Unterregisterkarte "Suchen" löschen. Wenn Sie den Inhalt der Unterregisterkarte "Suchen" gelöscht haben, können Sie eine neue Suche starten. Wenn Sie auf "Alles löschen" klicken, reagiert die Anwendung folgendermaßen:

• Es werden alle angegebenen Suchfilter gelöscht.
• Es werden alle Container aus der Suchergebnistabelle gelöscht.
• Es werden keine hinzugefügten Container aus der Tabelle "Betroffenes Material" entfernt.
• Die Registerkarte "Suche" wird weiterhin angezeigt.

Unterregisterkarte "Manuell hinzufügen"

Die Unterregisterkarte "Manuell hinzufügen" bietet die Möglichkeit, Container zu scannen oder einzugeben, die Sie zum Ereignis hinzufügen möchten. Sie können diese Registerkarte verwenden, um Container oder Lose hinzuzufügen, die nicht vom MES überwacht werden. Wenn Sie die Unterregisterkarte "Manuell hinzufügen" verwenden, um Container hinzuzufügen, die vom MES überwacht werden, werden die Container auf dieser Unterregisterkarte nicht aktualisiert. Verwenden Sie die Transaktion "Containerwartung", wenn Sie Attribute dieser Container aktualisieren müssen.

Fertigungsaktionen für betroffenes Material

Wenn Sie die Registerkarte "Betroffenes Material" auswählen, aktiviert die Anwendung die Fertigungsaktionsschaltflächen "Anhalten (mehrere)"und "Freigeben (mehrere)". Durch Klicken auf die Fertigungsaktionsschaltflächen werden die entsprechenden Popups aufgerufen. Wenn Sie z. B. auf "Anhalten (mehrere)" klicken, zeigt die Anwendung das Popup "Anhalten (mehrere)" an.

Die Anwendung aktualisiert die Tabelle "Ereignis > Betroffenes Material", wenn Sie das Popup schließen, nachdem die Fertigungsaktion durchgeführt wurde. Die Schaltfläche "Aktualisieren" deaktiviert die Kontrollkästchen neben den ausgewählten Containern, für die Aktionen durchgeführt wurden, und aktualisiert ggf. die Details für die betreffenden Container. Bei Bedarf können Sie die nächsten Container auswählen, um Aktionen für diese Container durchzuführen.

Weitere Informationen finden Sie unter "Fertigungsaktionen".

Schaltflächendefinitionen der Registerkarte "Betroffenes Material"

Diese Tabelle definiert die Schaltflächen der Unterregisterkarten "Suche" und "Manuell hinzufügen" der Verwaltungsseite eines Datensatzes.

Schaltfläche

Funktion der Schaltfläche . . .

Alles löschen

Löscht alle Informationen, die auf dieser Registerkarte eingegeben oder geändert wurden. Diese Schaltfläche wird nicht auf der Unterregisterkarte "Manuell hinzufügen" oder im Ansichtsmodus der Registerkarte angezeigt.

Vorgehensweise zum Hinzufügen von Containern zu einem Qualitätsdatensatz auf der Unterregisterkarte "Suchen"

Führen Sie die folgenden Schritte aus, um Container mithilfe der Unterregisterkarte "Suchen" zu einem Qualitätsdatensatz hinzuzufügen:

1. Rufen Sie den Qualitätsdatensatz auf. Die Verwaltungsseite des Datensatzes wird angezeigt.
2. Klicken Sie auf die Registerkarte Betroffenes Material. Die Seite "Betroffenes Material" wird angezeigt.
3. Klicken Sie auf Bearbeitungsmodus aktivieren. Der Datensatz wird in den Bearbeitungsmodus versetzt.
4. Geben Sie Parameter an, um Ihre Suche einzugrenzen.
5. Klicken Sie auf Suchen. Die Anwendung zeigt alle Container, die den Suchkriterien entsprechen, in der Tabelle "Suchergebnisse" an.
6. Wählen Sie mindestens einen Container in der Tabelle "Suchergebnisse" aus.
7. Klicken Sie auf Ausgewählte Container hinzufügen. Der Container wird der Tabelle "Ereignis > Betroffenes Material" hinzugefügt.
8. Klicken Sie auf die Schaltfläche Alles löschen, um den Inhalt der Parameterfelder und der Tabelle "Suchergebnisse" zu löschen, wenn Sie eine neue Suche durchführen möchten.

Oder

Klicken Sie auf Betroffenes Material aktualisieren. Die Anwendung zeigt eine Meldung mit dem Hinweis an, dass das Ereignis aktualisiert wurde.

Vorgehensweise zum Hinzufügen von Containern zu einem Qualitätsdatensatz auf der Unterregisterkarte "Manuell hinzufügen"

Führen Sie die folgenden Schritte aus, um Container mithilfe der Unterregisterkarte "Manuell hinzufügen" zu einem Qualitätsdatensatz hinzuzufügen:

1. Rufen Sie den Qualitätsdatensatz auf. Die Verwaltungsseite des Datensatzes wird angezeigt.
2. Klicken Sie auf die Registerkarte Betroffenes Material. Die Seite "Betroffenes Material" wird angezeigt.
3. Klicken Sie auf Bearbeitungsmodus aktivieren. Der Datensatz wird in den Bearbeitungsmodus versetzt.
4. Klicken Sie auf die Unterregisterkarte Manuell hinzufügen.
5. Geben Sie den Namen des Containers/Loses ein. Die Anwendung aktualisiert die Felder "Produkt", "Menge" und "Maßeinheit", wenn Sie die Felder verlassen oder ein anderes Feld auswählen.
6. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen Informationen ein. Informationen zu den Suchfeldern finden Sie in der Tabelle mit den Felddefinitionen für die Registerkarte "Betroffenes Material", Unterregisterkarte "Manuell hinzufügen".
7. Klicken Sie auf Container hinzufügen. Der Container wird der Tabelle "Ereignis > Betroffenes Material" hinzugefügt.
8. Wiederholen Sie Schritte 5-7, um zusätzliche Container hinzuzufügen.

Oder

Klicken Sie auf Betroffenes Material aktualisieren. Die Anwendung zeigt eine Meldung mit dem Hinweis an, dass das Ereignis aktualisiert wurde.

Vorgehensweise zum Entfernen von Containern aus einem Qualitätsdatensatz

Führen Sie die folgenden Schritte aus, um Container aus einem Qualitätsdatensatz zu entfernen:

1. Rufen Sie den Qualitätsdatensatz auf. Die Verwaltungsseite des Datensatzes wird angezeigt.
2. Klicken Sie auf die Registerkarte Betroffenes Material. Die Seite "Betroffenes Material" wird angezeigt.
3. Klicken Sie auf Bearbeitungsmodus aktivieren. Der Datensatz wird in den Bearbeitungsmodus versetzt.
4. Wählen Sie den Container oder das Los, den/das Sie entfernen möchten, in der Tabelle "Ereignis > Betroffenes Material" aus.
5. Klicken Sie in der Tabelle auf Ausgewählte Zeile löschen. Der Container/das Los wird aus der Tabelle entfernt.
6. Klicken Sie auf Betroffenes Material aktualisieren. Auf der Registerkarte werden die aktualisierten Informationen angezeigt.
