Zuweisen einer Checkliste

Verwenden Sie die Aktion "Checkliste zuweisen", um dem Datensatz eine Checklistenvorlage zuzuweisen. Eine Checkliste gewährleistet, dass der Benutzer die erforderlichen Schritte bei der Verarbeitung des Ereignisdatensatzes durchführt. Checklistenvorlagen werden in der Modellierung definiert.

Beim Zuweisen einer Checkliste

Sie können eine Checkliste zuweisen, wenn die folgenden Bedingungen erfüllt sind:

• Der Status des Datensatzes ist nicht "Ausstehend", "In Klärung" oder "Geschlossen".
• Sie sind berechtigt, eine Checkliste zuzuweisen.
• Für die momentan zugewiesene Checkliste wurden keine Daten gespeichert.

Die Anwendung deaktiviert die Schaltfläche "Checkliste zuweisen", wenn eine der obigen Bedingungen nicht zutrefft.

Vorgehensweise zum Zuweisen einer Checkliste

Führen Sie die folgenden Schritte aus, um eine Checkliste zuzuweisen:

1. Rufen Sie den Qualitätsdatensatz auf. Die Verwaltungsseite des Datensatzes wird angezeigt.
2. Klicken Sie im Bereich "Qualitätsaktionen" auf Checkliste zuweisen. Die Seite "Checkliste zuweisen" wird angezeigt.
3. Wählen Sie eine Checklistenvorlage aus.
4. Geben Sie ggf. Kommentare ein.
5. Klicken Sie auf Abschicken. Die Anwendung zeigt eine Meldung mit dem Hinweis an, dass die Checkliste zugewiesen wurde, und ruft die Verwaltungsseite des Ereignisses auf.
