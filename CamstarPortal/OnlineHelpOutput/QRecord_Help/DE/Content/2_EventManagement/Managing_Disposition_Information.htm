Verwalten von Verteilungsinformationen

Die Registerkarte "Verteilungen" bietet die Möglichkeit, den Containern, die dem Ereignisdatensatz zugeordnet sind, Verteilungen zuzuweisen. Bei der Zuweisung einer Verteilung müssen Sie auch entscheiden, was mit dem betroffenen Material geschehen soll. Darüber hinaus können Sie den zugewiesenen Verteilungen Fehlermodi zuordnen, um anzugeben, wie die einzelnen Fehler gelöst wurden.

Beim Zuweisen von Verteilungen und Zuordnen von Fehlermodi

Bei der Zuweisung von Verteilungen zu Containern, die dem Ereignisdatensatz zugeordnet sind, können einem Container eine oder mehrere Verteilungen zuweisen. Sie können dann einer einzelnen Verteilung einen oder mehrere Fehlermodi zuordnen. Verteilungen und Fehlermodi werden von einem Systemadministrator in der Modellierung definiert.

Die Registerkarte "Verteilungen" enthält zwei Unterregisterkarten: "Verteilung zuordnen" und "Fehlermodus verbinden".

Unterregisterkarte "Verteilung zuordnen"

Sie können die Unterregisterkarte "Verteilung zuordnen" verwenden, um Verteilungen mit Containermengen zu verknüpfen. Alle mit dem Ereignisdatensatz verknüpften Container werden auf dieser Unterregisterkarte in einer Tabelle angezeigt. Wenn Sie einen Container auswählen und eine Verteilung zuweisen, fügt die Anwendung zur betreffenden Containerzeile eine Untertabelle mit Verteilungsinformationen hinzu. Sie können die Verteilungsinformationen ein- und ausblenden, indem Sie auf das Zeilenanzeigesymbol am Anfang der Zeile klicken.

Sie können Verteilungen mehreren Containern oder jedem Container einzeln zuweisen. Darüber hinaus können Sie eine Verteilung der Gesamtmenge eines Containers oder einer Teilmenge zuweisen. Wenn Sie eine Verteilung der Gesamtmenge zuweisen, blendet die Anwendung das Feld "Verteilungsmenge" aus. Wenn Sie Verteilungen einer Teilmenge zuweisen, aktualisiert die Anwendung den Wert "Zu verteilende Menge" des Containers, um die zu verteilende Restmenge anzuzeigen. Sie können nicht mehr verteilen als die derzeit im Container vorhandene Menge.

Wichtig: Verteilen Sie keine größere Menge als die Menge im Container.

Die Unterregisterkarte "Verteilung zuordnen" bietet die Möglichkeit, Dispositionsinformationen zu bearbeiten. Wenn Sie eine zugewiesene Verteilung in der Tabelle auswählen, werden die Verteilungsfelder oben auf der Unterregisterkarte gefüllt und die Bezeichnung der Schaltfläche "Auf ausgewählte Container anwenden" wird in "Ausgewählte Verteilung aktualisieren" geändert.

Wenn der aktuell ausgewählten Verteilung Fehlermodi zugeordnet sind, bleiben diese auch dann verknüpft, wenn Sie eine andere Verteilung auswählen. Sie müssen ggf. die Unterregisterkarte "Fehlermodus verbinden" verwenden, um die zugeordneten Fehlermodi zu ändern.

Sie können eine zugewiesene Verteilung aus einem Container löschen. Wählen Sie die Verteilungszeile in der Tabelle aus und klicken Sie auf die Schaltfläche "Ausgewählte Zeile löschen".

Unterregisterkarte "Fehlermodus verbinden"

Verwenden Sie die Unterregisterkarte "Fehlermodus verbinden", um den zugewiesenen Verteilungen Fehlermodi zuzuordnen. Die Verknüpfung von Fehlermodi mit einer bestimmten Verteilung gibt an, wie die einzelnen Fehler gelöst wurden. Sie können Fehlermodi mit mehreren Verteilungen oder einzeln mit Verteilungen verknüpfen.

Sie können nur die Fehlermodi auswählen, die dem Ereignisdatensatz zugewiesen wurden. Wenn Sie eine Verteilung auswählen und einen Fehlermodus zuordnen, fügt die Anwendung eine Untertabelle mit Informationen zum Fehlermodus der Verteilung hinzu. Sie können die Informationen für Verteilung und Fehlermodus ein- und ausblenden, indem Sie auf das Zeilenanzeigesymbol am Anfang der Zeilen klicken.

Sie können einen zugeordneten Fehlermodus aus einer Verteilung löschen. Wählen Sie die Fehlermoduszeile in der Tabelle aus und klicken Sie auf die Schaltfläche "Ausgewählte Zeile entfernen".

Fertigungsaktionen für Verteilungen

Wenn Sie die Registerkarte "Verteilung" auswählen, aktiviert die Anwendung alle Schaltflächen für Fertigungsaktionen. Durch Klicken auf eine Aktionsschaltfläche wird das entsprechende Popup aufgerufen. Wenn Sie z. B. auf "Anhalten (mehrere)" klicken, zeigt die Anwendung das Popup "Anhalten (mehrere)" an.

Weitere Informationen finden Sie unter "Fertigungsaktionen".

Vorgehensweise zum Zuweisen einer Verteilung

Führen Sie die folgenden Schritte aus, um dem Ereignis eine Verteilung zuzuweisen:

1. Rufen Sie den Qualitätsdatensatz auf. Die Verwaltungsseite des Datensatzes wird angezeigt.
2. Klicken Sie auf die Registerkarte Verteilung. Die Seite "Verteilung" wird angezeigt.
3. Klicken Sie auf Bearbeitungsmodus aktivieren. Der Datensatz wird in den Bearbeitungsmodus versetzt.
4. Wählen Sie eine Verteilung aus.
5. Möchten Sie eine Teilmenge oder die Gesamtmenge verteilen?

Wenn Sie möchten verteilen . . .

Dann . . .

Eine Teilmenge
a. Geben Sie im Feld Verteilungsmenge eine zu verteilende Menge ein, die kleiner ist als die Menge im Container.
b. Gehen Sie zu Schritt 6.

Die gesamte Menge
a. Wählen Sie Auf die gesamte Menge anwenden aus. Die Anwendung entfernt das Feld "Verteilungsmenge".
b. Gehen Sie zu Schritt 6.
6. Geben Sie ggf. Verteilungskommentare ein.
7. Wählen Sie in der Tabelle "Ereignis > Container > Verteilung" die Container aus, die Sie der angegebenen Verteilung zuweisen möchten.
8. Klicken Sie auf Auf ausgewählte Container anwenden. Die Anwendung fügt eine Untertabelle mit Verteilungsinformationen zu den ausgewählten Containern hinzu und verringert den Wert "Zu verteilende Menge" der einzelnen Container entsprechend.
9. Wiederholen Sie die Schritte 4 bis 8, um Containern weitere Verteilungen zuzuweisen.
10. Klicken Sie auf Verteilung aktualisieren. Die Anwendung zeigt eine Meldung mit dem Hinweis an, dass das Ereignis aktualisiert wurde.

Vorgehensweise zum Verbinden eines Fehlermodus mit einer Verteilung

Führen Sie die folgenden Schritte aus, um einen Fehlermodus mit einer Verteilung zu verbinden:

1. Rufen Sie den Qualitätsdatensatz auf. Die Verwaltungsseite des Datensatzes wird angezeigt.
2. Klicken Sie auf die Registerkarte Verteilung. Die Seite "Verteilung" wird angezeigt.
3. Klicken Sie auf Bearbeitungsmodus aktivieren. Der Datensatz wird in den Bearbeitungsmodus versetzt.
4. Klicken Sie auf die Registerkarte Fehlermodus verbinden. Die Fehlermodusfelder werden angezeigt.
5. Wählen Sie einen Fehlermodus aus. Die Anwendung zeigt die Fehlermodusbeschreibung an, wenn vom Benutzer, der den Fehlermodus definiert hat, eine Beschreibung angegeben wurde.
6. Erweitern Sie die Containerzeilen in der Tabelle "Ereignis > Container > Verteilung", um die zugeordneten Verteilungen anzuzeigen.
7. Wählen Sie die Verteilungen aus, mit denen Sie den angegebenen Fehlermodus verbinden möchten.
8. Klicken Sie auf Auf ausgewählte Verteilung(en) anwenden. Die Anwendung fügt eine untergeordnete Untertabelle mit Fehlermodusinformationen zu den ausgewählten Verteilungen hinzu.
9. Wiederholen Sie die Schritte 5-8, um zusätzliche Fehlermodi mit den zugewiesenen Dispositionen zu verbinden.
10. Klicken Sie auf Verteilung aktualisieren. Die Anwendung zeigt eine Meldung mit dem Hinweis an, dass das Ereignis aktualisiert wurde.
