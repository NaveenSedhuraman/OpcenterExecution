Verwalten von Querverweisen

Die Registerkarte "Querverweise" umfasst eine Liste von Ereignisdatensätzen, die als Querverweis zum Datensatz hinzugefügt wurden. Sie können diese Registerkarte verwenden, um Querverweise zu suchen, hinzuzufügen und zu entfernen. Die Anwendung aktualisiert alle Datensätze, die als Querverweis hinzugefügt wurden, um deutlich zu machen, dass sie mit diesem Datensatz als Querverweis verknüpft sind.

Beim Hinzufügen von Datensätzen als Querverweis

Sie können für die einzelnen Datensätze Informationen wie Name des Eigentümers, Klassifikation, Unterklassifikation anzeigen. In den Feldern Identifikator und Container können Sie SQL-Platzhalter verwenden, wenn Sie nach Datensätzen suchen, die Sie als Querverweise hinzufügen möchten. Das Prozentzeichen (%) ist ein SQL-Platzhalter. Weitere Informationen zu SQL-Platzhalten finden Sie auf der w3schools-Website unter: http://www.w3schools.com/sql/sql_wildcards.asp.

Sie können auf die Schaltfläche "Details des ausgewählten Ereignisses anzeigen" klicken, um einen Datensatz, der von der Suche zurückgegebenen wurde bzw. auf dem im Datensatz verwiesen wird, in einem separaten Popup anzuzeigen.

Die Anwendung entfernt Querverweise aus der Liste des Datensatzes, wenn Sie einen Querverweis entfernen.

Wenn Sie das Kontrollkästchen in der Kopfzeile der Tabelle "Suchergebnisse" verwenden, um alle von der Suche zurückgegebenen Datensätze auszuwählen, werden nicht alle, sondern nur die angezeigten Datensätze ausgewählt.

Klicken Sie auf "Querverweise aktualisieren", nachdem Querverweise hinzugefügt oder entfernt wurden.

Schaltflächendefinitionen der Registerkarte "Querverweise"

Diese Tabelle definiert die Schaltflächen auf der Registerkarte "Querverweise" der Verwaltungsseite eines Datensatzes.

Schaltfläche

Funktion der Schaltfläche . . .

Alles löschen

Löscht den Inhalt der Suchkriterienfelder und der Tabelle "Suchergebnisse". Nachdem Sie die Inhalte der Registerkarte "Querverweise" gelöscht haben, können Sie eine neue Suche starten.

Suchen

Sucht alle Datensätze, die den eingegebenen Suchkriterien entsprechen und als Querverweise hinzugefügt werden können.

Ausgewählte Referenz hinzufügen

Fügt die in der Tabelle "Suchergebnisse" ausgewählten Datensätze als Querverweis zum aktuellen Datensatz hinzu. Sie können einen oder mehrere Container gleichzeitig hinzufügen.

Hinweis: Die Anwendung aktiviert diese Schaltfläche für eine Tabelle, wenn Sie mindestens einen Datensatz in der Tabelle ausgewählt haben.

Details des ausgewählten Ereignisses anzeigen

Zeigt die Details des ausgewählten Datensatzes in einem separaten Popup an. Die Kategorie des ausgewählten Datensatzes bestimmt, welches Popup angezeigt wird.

Die Anwendung zeigt folgende Popups an:

• "Verwalten A/E" für generische Ereignisdatensätze
• "Verwalten P/E" für Produktionsereignisse (Abweichung)

Diese Schaltfläche ist in den Tabellen "Suchergebnissen" und "Aktuelle Referenzen" für Datensätze verfügbar.

Hinweis: Die Anwendung aktiviert diese Schaltfläche für eine Tabelle, wenn Sie einen Datensatz der betreffenden Tabelle auswählen. Wenn Sie mehr als einen Datensatz in der Tabelle auswählen, deaktiviert die Anwendung die Schaltfläche.

Vorgehensweise zum Hinzufügen eines Querverweises

Führen Sie die folgenden Schritte aus, um einen Querverweis hinzuzufügen:

1. Rufen Sie den Qualitätsdatensatz auf. Die Verwaltungsseite des Datensatzes wird angezeigt.
2. Klicken Sie auf die Registerkarte Querverweise. Die Seite "Querverweise" wird angezeigt.
3. Klicken Sie auf Bearbeitungsmodus aktivieren. Der Datensatz wird in den Bearbeitungsmodus versetzt.
4. Wählen Sie die gewünschten Suchkriterien in den verfügbaren Feldern oben auf der Registerkarte aus.
5. Klicken Sie auf Suchen. Die Anwendung zeigt alle Datensätze an, die den Suchkriterien entsprechen und in der Tabelle "Suchergebnisse" als Querverweis hinzugefügt werden können.
6. Führen Sie eine der folgenden Aktionen durch:
• Aktivieren Sie das Kontrollkästchen für die Ereignisdatensätze, die Sie als Querverweis hinzufügen möchten.
• Aktivieren Sie das Kontrollkästchen in der Kopfzeile, um alle angezeigten Ereignisdatensätze auszuwählen.
7. Klicken Sie auf Ausgewählte Referenz hinzufügen. Die Anwendung aktualisiert den Datensatz und zeigt die hinzugefügten Referenzen in der Tabelle "Aktuelle Referenzen" an.
8. Zeigen Sie die nachfolgenden Seiten der Suchergebnisse an und wiederholen Sie ggf. die Schritte 6 bis 7.
9. Klicken Sie auf Querverweise aktualisieren. Die Anwendung zeigt eine Meldung mit dem Hinweis an, dass das Ereignis aktualisiert wurde.

Vorgehensweise zum Entfernen eines Querverweises

Führen Sie die folgenden Schritte aus, um einen Querverweis zu entfernen:

1. Rufen Sie den Qualitätsdatensatz auf. Die Verwaltungsseite des Datensatzes wird angezeigt.
2. Klicken Sie auf die Registerkarte Querverweise. Die Seite "Querverweise" wird angezeigt.
3. Klicken Sie auf Bearbeitungsmodus aktivieren. Der Datensatz wird in den Bearbeitungsmodus versetzt.
4. Wählen Sie den Querverweis, der entfernt werden soll, in der Tabelle "Aktuelle Referenzen" aus.
5. Klicken Sie in der Tabelle auf Ausgewählte Zeile löschen. Das Protokoll wird aus der Tabelle entfernt.
6. Klicken Sie auf Querverweise aktualisieren. Die Anwendung zeigt eine Meldung mit dem Hinweis an, dass das Ereignis aktualisiert wurde.
