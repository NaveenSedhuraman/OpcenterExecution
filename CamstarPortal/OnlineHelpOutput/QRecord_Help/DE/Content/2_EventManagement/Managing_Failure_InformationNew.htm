Verwalten von Fehlerinformationen

Die Registerkarte "Fehler" bietet die Möglichkeit, die dem Ereignis zugewiesenen Fehlermodi anzuzeigen und bestehende Fehler zu aktualisieren. Darüber hinaus können Sie dem Ereignis weitere Fehlermodi hinzufügen und zugewiesene Fehlermodi ändern.

Beim Ändern von Fehlerinformationen

Durch Auswahl einer Fehlermodusgruppe wird die Liste der Fehlermodi auf die Modi beschränkt, die zur Gruppe gehören. Standardfehlerschweregrad und -fehlertyp für den ausgewählten Fehlermodus werden angezeigt und können geändert werden.

In der Tabelle werden alle bei der Ereignisaufzeichnung oder soeben hinzugefügten Fehler angezeigt. Bei Bedarf können Sie bestehende Fehler aus der Tabelle entfernen.

Die Schaltfläche "Fehler hinzufügen" wird angezeigt, wenn Sie sich auf der Registerkarte "Fehler" im Bearbeitungsmodus befinden. Über diese Schaltfläche können Sie einem Qualitätsdatensatz Fehler hinzufügen. Wenn Sie auf die Schaltfläche "Fehler hinzufügen" klicken, wird der Fehler zur Tabelle "Ereignis > Fehlermodi" hinzugefügt. Wenn Sie auf "Fehler aktualisieren" klicken, wird der Qualitätsdatensatz aktualisiert.

Felddefinitionen der Registerkarte "Fehler"

Diese Tabelle definiert die Felder der Registerkarte "Fehler".

Feld

Definition

Typ

Fehlermodusgruppe

Gruppe mit den Fehlermodi (Nutzercodes), die für das Ereignis relevant sind. Wird verwendet, um die im Feld "Fehlermodi" angezeigten Fehlermodi zu beschränken.

Optional

Fehlermodus

Fehlermodi, die dem Qualitätsdatensatz zugeordnet sind. Die angegebene Fehlermodusgruppe beschränkt die auswählbaren Fehlermodi.

Erforderlich

Beschreibung des Fehlermodus

Beschreibung des ausgewählten Fehlermodus.

Nur Anzeige

Fehlerschweregrad

Schweregrad des Fehlers während des Ereignisses. Standardmäßig wird der Wert verwendet, der in der Modellierung für den ausgewählten Fehlermodus festgelegt wurde. Sie können aber einen anderen Wert auswählen.

Optional

Fehlertyp

Fehlertyp, der dem Container oder dem Los zugeordnet ist. Beschreibt die Eigenschaften des Fehlers und bietet eine Möglichkeit, den Fehler zu kategorisieren (z. B. sporadisch oder ständig). Standardmäßig wird der Wert verwendet, der in der Modellierung für den ausgewählten Fehlermodus festgelegt wurde. Sie können aber einen anderen Wert auswählen.

Produkt

Name des Produkts, das einen Defekt oder einen Fehler aufweist.

Optional

Operation

Operation, in der das Ereignis entdeckt wurde.

Optional

Zusätzliche Informationen oder Anweisungen zu diesem Fehler. Das Feld akzeptiert maximal 255 Zeichen. Wenn die Kommentare 255 Zeichen überschreiten, verkürzt die Anwendung die Daten.

Optional

Tabelle "Ereignis > Fehlermodi"

Tabelle, in der die einzelnen Fehler für das Ereignis angezeigt werden. Die Anwendung fordert Sie auf, mindestens einen Fehler in der Tabelle einzugeben.

Vorgehensweise zum Hinzufügen eines Fehlers

Führen Sie die folgenden Schritte aus, um einen Fehler zu einem vorhandenen Datensatz hinzuzufügen:

1. Rufen Sie den Qualitätsdatensatz auf. Die Verwaltungsseite des Datensatzes wird angezeigt.
2. Klicken Sie auf die Registerkarte Fehler. Die Registerkarte "Fehler" wird angezeigt.
3. Klicken Sie auf Bearbeitungsmodus aktivieren. Der Datensatz wird in den Bearbeitungsmodus versetzt.
4. Wählen Sie einen Fehlermodus aus. Die Felder "Beschreibung des Fehlermodus", "Fehlerschweregrad" und "Fehlertyp" werden ausgefüllt, wenn sie in der Modellierung für den Fehlermodus definiert wurden.
5. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. In der Felddefinitionstabelle finden Sie Informationen zu den optionalen Feldern.
6. Klicken Sie auf Fehler hinzufügen. Die neuen Fehlerinformationen werden zur Tabelle "Ereignis > Fehlermodi" hinzugefügt.
7. Klicken Sie auf Fehler aktualisieren. Die Anwendung zeigt eine Meldung mit dem Hinweis an, dass das Ereignis aktualisiert wurde.

Vorgehensweise zum Ändern eines bestehenden Fehlers

Führen Sie die folgenden Schritte aus, um einen bestehenden Fehler eines vorhandenen Datensatzes zu ändern:

1. Rufen Sie den Qualitätsdatensatz auf. Die Verwaltungsseite des Datensatzes wird angezeigt.
2. Klicken Sie auf die Registerkarte Fehler. Die Seite "Fehler" wird angezeigt.
3. Klicken Sie auf Bearbeitungsmodus aktivieren. Die Anwendung versetzt den Datensatz in den Bearbeitungsmodus.
4. Suchen Sie den Fehler, den Sie ändern möchten, in der Tabelle "Ereignis > Fehlermodi" und wählen Sie die zugehörige Zeile aus. Die Anwendung zeigt die Werte des ausgewählten Ereignisses in den Feldern darüber an und ersetzt die Schaltfläche "Fehler hinzufügen" durch die Schaltfläche "Ausgewählten Fehler aktualisieren".
5. Nehmen Sie die erforderlichen Änderungen vor.
6. Klicken Sie auf Ausgewählten Fehler aktualisieren. Die Anwendung aktualisiert die Fehlerinformation in der Tabelle "Ereignis > Fehlermodi" und ersetzt die Schaltfläche "Ausgewählte Fehler aktualisieren" durch die Schaltfläche "Fehler hinzufügen".
7. Klicken Sie auf Fehler aktualisieren. Die Anwendung zeigt eine Meldung mit dem Hinweis an, dass das Ereignis aktualisiert wurde.

Vorgehensweise zum Entfernen eines Fehlers

Führen Sie die folgenden Schritte aus, um einen Fehler aus einem vorhandenen Datensatz zu entfernen:

1. Rufen Sie den Qualitätsdatensatz auf. Die Verwaltungsseite des Datensatzes wird angezeigt.
2. Klicken Sie auf die Registerkarte Fehler. Die Seite "Fehler" wird angezeigt.
3. Klicken Sie auf Bearbeitungsmodus aktivieren. Die Anwendung versetzt den Datensatz in den Bearbeitungsmodus.
4. Suchen Sie den Fehler, den Sie entfernen möchten, in der Tabelle "Ereignis > Fehlermodi" und wählen Sie die zugehörige Zeile aus.
5. Klicken Sie in der Tabelle auf Ausgewählte Zeile löschen. Der Fehler wird aus der Tabelle entfernt.
6. Klicken Sie auf Fehler aktualisieren. Die Anwendung zeigt eine Meldung mit dem Hinweis an, dass das Ereignis aktualisiert wurde.
