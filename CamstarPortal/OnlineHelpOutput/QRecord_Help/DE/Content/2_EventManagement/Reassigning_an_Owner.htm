Neuzuweisen eines Eigentümers

Verwenden Sie die Aktion "Neu zuweisen", um dem Datensatz einen neuen Eigentümer zuzuweisen. Wenn Sie dem Datensatz einen neuen Eigentümer zuweisen, fügt die Anwendung eine Aufgabe zum Concierge und zur Nachrichtenzentrale des neuen Eigentümers hinzu und entfernt die Aufgabe aus dem Concierge und der Nachrichtenzentrale des alten Eigentümers. Wenn ein Systemadministrator eine E-Mail für die Neuzuweisung von Eigentümern konfiguriert hat, sendet die Anwendung diese automatisch an den neuen und vorherigen Eigentümer, um diese über die Neuzuweisung zu informieren.

Beim Neuzuweisen eines Eigentümers

Wenn Sie einen Eigentümer neu zuweisen, hängen die auswählbaren Rollen von der Organisation ab, die mit dem Ereignis verbunden ist. Die ausgewählte Rolle legt dann fest, welche Eigentümer zur Auswahl stehen.

Alle in das Feld "Kommentare" eingegebenen Informationen sind als Teil der Datensatzhistorie verfügbar. Die hier eingegebenen Kommentare werden nicht auf der Verwaltungsseite des Datensatzes angezeigt.

Sie können dem Datensatz einen anderen Eigentümer zuweisen, wenn die folgenden Bedingungen erfüllt sind:

• Der Status des Datensatzes ist NICHT "Geschlossen".
• Sie sind berechtigt, einen Datensatz neu zuzuweisen.

Die Anwendung deaktiviert die Schaltfläche "Neu zuweisen", wenn eine der beiden obigen Bedingungen nicht erfüllt ist.

Die Zuweisung eines Datensatzes an einen anderen Eigentümer wird als manuelle Einteilung angesehen. Wenn Sie einen Datensatz einem anderen Eigentümer zuweisen, deaktiviert die Anwendung die Aktionsschaltfläche "Einteilung".

Optional

Vorgehensweise zum Neuzuweisen eines Eigentümers

Führen Sie die folgenden Schritte aus, um einem Datensatz einen neuen Eigentümer zuzuweisen:

1. Rufen Sie den Datensatz auf, der aktualisiert werden soll.
2. Klicken Sie im Bereich "Qualitätsaktionen" auf Neu zuweisen. Die Seite "Eigentümer neu zuweisen" wird angezeigt.
3. Wählen Sie die entsprechende Rolle aus.
4. Wählen Sie den entsprechenden Eigentümer aus.
5. Geben Sie relevante Kommentare oder Hinweise für die Neuzuweisung des Eigentümers im Feld Kommentare ein.
6. Klicken Sie auf Abschicken. Die Anwendung zeigt eine Meldung mit dem Hinweis an, dass der Eigentümer geändert wurde.
