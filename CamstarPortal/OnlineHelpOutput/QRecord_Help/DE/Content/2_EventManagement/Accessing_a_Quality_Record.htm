Aufrufen eines Qualitätsdatensatzes

Sie haben folgende Möglichkeiten, die Verwaltungsseite eines Qualitätsdatensatzes aufrufen:

• Auswahl des Datensatzes im Concierge
• Auswahl des Datensatzes in der Nachrichtenzentrale
• Verwendung der Seite "Qualitätssuche"

Automatischer Aufruf eines Qualitätsdatensatzes nach Übermittlung eines Ereignisses

Die Anwendung erstellt automatisch einen generischen Ereignisdatensatz und zeigt automatisch die Verwaltungsseite des Datensatzes an, wenn Sie ein Ereignis am Ende der Aufzeichnung übermitteln. Weitere Informationen zum Aufzeichnen eines generischen Ereignisses finden Sie unter "Aufzeichnen eines Ereignisses".

Aufrufen eines Qualitätsdatensatzes im Concierge

Im Abschnitt "Meine Zuweisungen" des Concierge zeigt die Anwendung Qualitätsdatensätze an, auf die Sie Zugriff haben. Dies sind Datensätze, für die Sie Eigentümerrechte besitzen. Wenn Sie einen Datensatz im Concierge auswählen, wird die Verwaltungsseite des betreffenden Datensatzes angezeigt.

Sie können auf den Concierge zugreifen, indem Sie auf das Concierge-Symbol in der rechten oberen Ecke der Seite klicken.

Aufrufen eines Qualitätsdatensatzes in der Nachrichtenzentrale

Die Nachrichtenzentrale ermöglicht eine zentrale Anzeige aller Qualitätsdatensätze, für die Sie verantwortlich sind, einschließlich Genehmigungen, ausstehenden Datensätzen und Eigentümerschaft. In der Nachrichtenzentrale werden sowohl Nachrichten des Concierge als auch Nachrichten von Qualitätsdatensätzen angezeigt, die Ihnen als Eigentümer oder aufgrund Ihrer Rolle zugewiesen wurden. Sie können die Suchfunktionen der Nachrichtenzentrale verwenden, um bestimmte Datensätze zu finden.

Die Auswahl eines Datensatzes in der Tabelle "Ergebnisse" der Nachrichtenzentrale aktiviert die Schaltfläche "Datensätze verwalten". Wenn Sie auf "Datensätze verwalten" klicken, wird die Seite "Datensätze verwalten" angezeigt.

Sie können die Nachrichtenzentrale aufrufen, indem Sie im Suchmenü auf "Nachrichtenzentrale" oder im Concierge auf den Link "Weitere Nachrichten anzeigen" klicken. Weitere Informationen zur Verwendung der Nachrichtenzentrale finden Sie unter "Verwendung der Nachrichtenzentrale".

Aufrufen eines Qualitätsdatensatzes auf der Seite "Qualitätssuche"

Sie können einen Qualitätsdatensatz aufrufen, indem Sie auf der Seite "Qualitätssuche" eine Suche durchführen. Wählen Sie im Suchmenü "Qualitätssuche" aus, um die Seite aufzurufen. Weitere Informationen finden Sie unter "Suchen nach einem Qualitätsdatensatz".
