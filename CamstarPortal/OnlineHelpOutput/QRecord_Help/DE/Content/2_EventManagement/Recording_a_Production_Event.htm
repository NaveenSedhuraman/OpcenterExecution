Aufzeichnen eines Produktionsereignisses

Die Transaktion "Produktionsereignis aufzeichnen" ermöglicht es Ihnen, die Details einer Container-bezogenen Abweichung einzugeben, darunter:

• Relevante Fehlermodi
• Container, die von dem Ereignis betroffen sind
• Betroffene Ressource (optional)
• Jeweiliger Schritt im Arbeitsplan, dem der Fehler zugeordnet werden kann (optional)

Sie können auch:

• Betroffene Materialien anhalten
• Zugehörige Dateien anhängen

Beim Aufzeichnen eines Produktionsereignisses

Sie müssen mindestens einen Fehlermodus für das Produkt in der Tabelle "Fehlermodi" eingeben, wenn Sie Informationen auf der Seite "Produktionsereignis aufzeichnen" eingeben. Sie müssen außerdem mindestens einen Container in der Tabelle "Betroffene Container" eingeben. Mindestens ein Fehlermodus und ein Container sind erforderlich.

Die Anwendung trägt die Container, die auf der ursprünglichen Seite ausgewählt wurden, automatisch in die Tabelle "Betroffene Container" ein, wenn Sie das Popup "Produktionsereignis aufzeichnen" über eine andere Seite (z. B. "Containersuche") aufrufen.

Die Anwendung umgeht Anforderungen für elektronische Unterschriften, die für die Standardtransaktion "Anhalten" definiert wurden, wenn Sie das Kontrollkästchen "Betroffene Container anhalten" aktivieren. Die angegebenen Container werden angehalten, ohne dass eine elektronische Unterschrift erforderlich ist.

Sie müssen dem Mitarbeiter eine primäre Organisation zuweisen, damit Sie ein Produktionsereignis aufzeichnen können. Außerdem müssen Sie in der Modellierung eine Nummerierungsregel erstellen und diese Regel im Objekt "Organisation" einer Abweichung zuweisen. Darüber hinaus muss die primäre Organisation eine Rolle und einen Eigentümer für den angemeldeten Benutzer haben, die bzw. der einer entsprechenden Klassifikation und Unterklassifikation in der Tabelle "Zuordnungen von Ereignisklassifizierungspez." zugewiesen ist.

Die Felder "Klassifikation" und "Unterklassifikation" im Bereich "Zusätzliche Felder anzeigen/ausblenden" auf der Seite "Produktionsereignis aufzeichnen" dürfen nicht leer sein. Sie enthalten standardmäßig die Werte, die in der als PE-Standard ausgewählten Zeile der Tabelle "Zuordnungen von Ereignisklassifizierungspez." für die Felder "Klassifikation" und "Unterklassifikation" angegeben sind. Sie können die Standardeinstellungen ändern. Vervollständigen Sie die Tabelle "Zuordnungen von Ereignisklassifizierungspez." mithilfe des Modellierungsobjekts "Organisation". Weitere Informationen finden Sie im Opcenter EX MDD Modeling-Benutzerhandbuch bzw. im Opcenter EX CR Modeling-Benutzerhandbuch.

Beim Laden der Seite "Produktionsereignis aufzeichnen" wird eine Meldung mit dem Hinweis angezeigt, dass mit dem Modellierungsobjekt "Organisation" eine Standardklassifikation und -unterklassifikation für ein Produktionsereignis festgelegt werden muss, wenn keine Standardwerte für "Klassifikation" und "Unterklassifikation" vorhanden sind.

Sie können auch festlegen, dass Etiketten nach einer Produktionsereignistransaktion manuell gedruckt werden sollen, wenn die Zuordnung für Produktionsereignisetiketten für die Fabrik nicht konfiguriert wurde. Weitere Informationen finden Sie im Opcenter EX MDD Modeling-Benutzerhandbuch bzw. im Opcenter EX CR Modeling-Benutzerhandbuch.

Die Anwendung erstellt eine Historie für das Ereignis, wenn ein Produktionsereignis erstellt wird.

Wenn Sie Anhänge hinzufügen, verhindert die Anwendung, dass Sie folgende Zeichen im Titel eingeben: +, &, #.

Die maximale Dateigröße für einen Anhang beträgt 30 MB.

Seite "Produktionsereignis aufzeichnen"

Sie können die Seite "Produktionsereignis aufzeichnen" über das Menü "Container" aufrufen. Auf der Seite "Produktionsereignis aufzeichnen" können Sie containerbezogene Abweichungen aufzeichnen.

Hinweis: Die Seite "Produktionsereignis aufzeichnen" ist u. U. auch als Popup verfügbar, wenn die Transaktion "Produktionsereignis aufzeichnen" als verfügbare Aktion auf einer anderen Seite konfiguriert wurde (z. B. Containersuche).

Vorgehensweise zum Aufzeichnen eines Produktionsereignisses

Führen Sie die folgenden Schritte aus, um ein Produktionsereignis aufzuzeichnen:

1. Wählen Sie Produktionsereignis aufzeichnen im Menü Container aus. Die Seite "Produktionsereignis aufzeichnen" wird angezeigt.
2. Geben Sie eine Beschreibung für das Ereignis ein.
3. Wählen Sie eine Fehlermodus-Gruppe. Die Tabelle Fehlermodus wird aktualisiert, um alle Fehlermodi in der angegebenen Fehlermodus-Gruppe anzuzeigen.
4. Wählen Sie einen oder mehrere Fehlermodi in der Tabelle Fehlermodi aus.
5. Wählen Sie einen Container im Container hinzufügen-Feld aus oder geben Sie ihn ein. Der Containername wird zur Tabelle "Betroffene Container" hinzugefügt.
Hinweis: Wenn Sie im Popup "Produktionsereignis aufzeichnen" arbeiten, werden die Container, die Sie auf der vorherigen Seite ausgewählt haben, automatisch in der Tabelle "Betroffene Container" angezeigt.
6. Wiederholen Sie Schritt 5, um weitere Container hinzuzufügen.
7. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. In der Felddefinitionstabelle finden Sie Informationen zu den optionalen Feldern.
8. Führen Sie die Schritte unter "Vorgehensweise zum Anhängen von Dateien an einen Ereignisdatensatz" aus, wenn Sie Dateien an den Ereignisdatensatz anhängen möchten.
9. Klicken Sie auf Produktionsereignis aufzeichnen. Die Anwendung zeigt eine Statusmeldung mit dem Hinweis an, dass der Produktionsereignisdatensatz erstellt wurde. Die Seite "Produktionsereignisetikett drucken" wird angezeigt, wenn Ihre Organisation das Modul "Etikettendruck" erworben hat und das Kontrollkästchen "Etiketten nach Aufzeichnung manuell drucken" aktiviert wurde.

Vorgehensweise zum Anhängen von Dateien an einen Ereignisdatensatz

Führen Sie die folgenden Schritte aus, um eine oder mehrere Dateien an eine Ereignis-Aufzeichnung auf der Seite "Produktionsereignis aufzeichnen" anzuhängen:

1. Führen Sie die Schritte 1-8 unter "Vorgehensweise zum Aufzeichnen eines Produktionsereignisses" aus.

Oder

Wählen Sie ein vorhandenes Produktionsereignis aus.

2. Klicken Sie auf das Steuerelement Zusätzliche Felder anzeigen (Dateien anhängen). Die Anwendung zeigt die Tabelle "Anhänge" an.
3. Klicken Sie auf Neue Zeile hinzufügen. Das Popup "Anhänge hinzufügen" wird angezeigt.
4. Geben Sie den Pfad und den Dateinamen in das Feld Originaler Dateipfad ein.

Oder

Klicken Sie auf Durchsuchen, um zu der Datei zu navigieren.

5. Geben Sie den Titel der Datei ein.
6. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. In der Felddefinitionstabelle finden Sie Informationen zu den optionalen Feldern.
7. Klicken Sie auf Hochladen. Die Dateiinformationen werden zur Tabelle "Anhänge" hinzugefügt.
8. Wiederholen Sie die Schritte 3 bis 7, um weitere Dateien hinzuzufügen.
