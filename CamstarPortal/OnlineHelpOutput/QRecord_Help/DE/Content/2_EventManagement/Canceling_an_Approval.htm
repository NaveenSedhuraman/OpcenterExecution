Abbrechen einer Genehmigung

Mit der Aktion "Genehmigung abbrechen" können Sie den Genehmigungsprozess für eine Qualitätsdatensatzlösung abbrechen. Sie können den Prozess jederzeit unabhängig davon abbrechen, ob Genehmiger Entscheidungen eingegeben haben.

Sie können eine Genehmigung abbrechen, wenn folgende Bedingungen erfüllt sind:

• Der Datensatzsatzstatus ist "In Klärung".
• Sie sind der Eigentümer des Datensatzes.

Die Anwendung deaktiviert die Schaltfläche "Genehmigung abbrechen", wenn eine der beiden obigen Bedingungen nicht erfüllt ist.

Hinweis: Sie können abgebrochene Genehmigungen auf der Registerkarte "Lösung" der Verwaltungsseite des Datensatzes und auf der Seite "Genehmigung unterschreiben" anzeigen, wenn die Genehmigungsanfrage anschließend weitergeleitet wurde.

Weitere Informationen zum Abbrechen von Genehmigungen finden Sie unter "Abbrechen von Genehmigungen".
