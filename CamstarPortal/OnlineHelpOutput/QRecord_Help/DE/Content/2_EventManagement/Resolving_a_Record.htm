Lösen eines Datensatzes

Verwenden Sie die Aktion "Lösen", um Datensätze zu schließen. Wenn Sie einen Datensatz lösen, wird dessen Status in "Geschlossen" geändert und Sie können keine weiteren Aktionen für den Datensatz durchführen. Sie können sowohl Ereignisse als auch Abweichungen lösen.

Hinweis: Sie können die Aktion "Erneut öffnen" durchführen, um einen geschlossenen Datensatz zu öffnen.

Die Auflösung ist ein zweiteiliger Prozess, wenn Genehmigungen für die Organisation konfiguriert wurden, die Eigentümer des Datensatzes ist. Qualitätsdatensatzlösungen müssen erst zur Genehmigung eingereicht (und genehmigt) werden, damit sie gelöst werden können.

Beim Lösen eines Qualitätsdatensatzes

Die Schaltfläche "Lösen" auf der Seite "Verwalten A/E" ist deaktiviert, wenn der Status des Datensatzes "Ausstehend", "In Klärung" oder "Geschlossen" ist. Sie können einen Datensatz mit dem Status "In Klärung" lösen, wenn alle Genehmigungen durchgeführt wurden. Die Schaltfläche "Lösen" auf der Seite "Verwalten A/E" ist in diesem Fall aktiviert.

Die Schaltfläche "Lösen" auf der Seite "Verwalten P/E" ist deaktiviert, wenn der Status des Datensatzes "Ausstehend", "In Klärung" oder "Geschlossen" ist. Sie können einen Datensatz mit dem Status "In Klärung" lösen, wenn alle Genehmigungen durchgeführt wurden. Die Schaltfläche "Lösen" auf der Seite "Verwalten P/E" ist in diesem Fall aktiviert.

Sie können einen Qualitätsdatensatz lösen, wenn folgende Bedingungen erfüllt sind:

• Alle erforderlichen Elemente in der Checkliste des Datensatzes wurden abgearbeitet.
• Sie sind berechtigt, Qualitätsdatensätze zu lösen.
• Die gesamte Menge in Containern, die mit dem Datensatz verbunden sind, wurde verteilt.

Die Anwendung zeigt eine Fehlermeldung an, wenn Sie versuchen, einen Datensatz zu lösen, ohne dass diese Bedingungen erfüllt sind.

Die Anwendung fordert Sie bei der Lösung eines Qualitätsdatensatzes auf, einen Lösungscode auszuwählen. Beim Lösen eines Datensatzes werden folgende Aktionen durchgeführt:

• Der Datensatz wird gesperrt, sodass keine Bearbeitung oder Durchführung von Aktionen möglich ist (mit Ausnahme von "Erneut öffnen").
• Der Datensatz wird aus dem Concierge entfernt. Sie müssen eine Suche durchführen, um geschlossene Datensätze zu finden und anzuzeigen.

Qualitätsdatensatzlösung, Genehmigungen und elektronische Unterschriften

Während des Lösungsprozesses kann die Anwendung Sie abhängig davon, wie die Eigentümerorganisation definiert wurde, u. U. auffordern, Lösungsgenehmigungen und/oder elektronische Unterschriften einzugeben:

• Die Anforderung, die Lösung von Qualitätsdatensätzen zu genehmigen, wird in der Kategorienzuordnung der Organisation festgelegt. Sie müssen die Lösung zur Genehmigung abschicken und die Bestätigung erhalten, bevor Sie den Lösungsprozess durchführen können, wenn entweder eine Genehmigungsvorlage festgelegt wurde oder das Kontrollkästchen "Genehmigung erforderlich“ für einen Datensatztyp (wie ein Ereignis) in der Kategorie-Zuordnung ausgewählt wurde.
• Die Anwendung fordert Sie zur Eingabe einer elektronischen Unterschrift auf, wenn die Qualitätsobjekttransaktion in der Organisation, der der Qualitätsdatensatz zugewiesen wurde, mit einer Anforderung für elektronische Unterschriften konfiguriert wurde (in der Transaktionszuordnung für elektronische Unterschriften der Organisation). In diesem Fall fordert die Anwendung eine Unterschrift an, wenn Sie die Lösung übermitteln.
• Die Anwendung fordert bei der Übermittlung der Lösung zur Eingabe der Unterschrift auf, wenn der Qualitätsservice "Qualitätsobjekt schließen" zur Transaktionszuordnung für elektronische Unterschriften hinzugefügt wurde.

Informationen zum Definieren von Genehmigungsvorlagen, zum Definieren von Organisationen und zum Konfigurieren von Anforderungen für elektronische Unterschriften für eine Organisation finden Sie im Opcenter EX MDD Modeling-Benutzerhandbuch bzw. im Opcenter EX CR Modeling-Benutzerhandbuch.

Ändern der angegebenen Genehmigungsvorlage

Die ursprüngliche, auf der Seite "Qualitätsobjektlösung" angegebene Genehmigungsvorlage ist die Vorlage, die auf der Organisationsebene für den betreffenden Datensatztyp festgelegt wurde. Sie können eine andere Genehmigungsvorlage angeben, bevor Sie die Genehmigung weiterleiten. Wenn keine Genehmigungsvorlage angegeben wurde, können Sie eine auswählen.

Hinweis: Die Anwendung fordert Sie zur Auswahl einer Genehmigungsvorlage auf, bevor Sie fortfahren, falls keine Genehmigungsvorlage auf Organisationsebene zugewiesen wurde, aber eine Genehmigung erforderlich ist.

Sie können die zugewiesene Genehmigungsvorlage ändern, indem Sie Genehmiger hinzufügen oder entfernen und Informationen für vorhandene Genehmiger aktualisieren. Durch Auswahl eines Genehmigers in der Tabelle wird die Zeile aktiviert, sodass Sie die Informationen für den jeweiligen Genehmiger bearbeiten können.

Wert, der die Art des Änderung angibt, die ein Benutzer während des Genehmigungsprozesses an einer Genehmigerzeile vornehmen kann:

Vorgehensweise zum Bearbeiten von Genehmigungsinformationen

Führen Sie die folgenden Schritte aus, um Genehmigungsinformationen zu bearbeiten:

1. Rufen Sie den Qualitätsdatensatz auf. Die Verwaltungsseite des Datensatzes wird angezeigt.
2. Klicken Sie im Bereich "Qualitätsaktionen" auf Lösen. Die Seite "Qualitätsobjektlösung" wird angezeigt.
3. Wählen Sie eine Genehmigungsvorlage aus der Liste aus.
4. Wählen Sie einen Genehmiger in der Liste "Genehmiger" aus. Die Zeile wird aktiviert.
5. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. In der Felddefinitionstabelle finden Sie Informationen zu den optionalen Feldern.
6. Klicken Sie auf eine Stelle außerhalb der Tabelle, um die Änderungen zu speichern.
7. Wiederholen Sie die Schritte 3 bis 6 für jeden Genehmiger, den Sie ändern möchten.

Vorgehensweise zum Hinzufügen von Genehmigern für die Genehmigung von Qualitätslösungen

Führen Sie die folgenden Schritte aus, um Genehmiger zur Qualitätslösungsgenehmigung hinzuzufügen:

1. Rufen Sie den Qualitätsdatensatz auf. Die Verwaltungsseite des Datensatzes wird angezeigt.
2. Klicken Sie im Bereich "Qualitätsaktionen" auf Lösen. Die Seite "Qualitätsobjektlösung" wird angezeigt.
3. Wählen Sie eine Genehmigungsvorlage aus der Liste aus.
4. Klicken Sie auf Neue Zeile hinzufügen in der Tabelle. Eine Zeile mit Anzeige der bearbeitbaren Felder wird aktiviert.
5. Geben Sie den Level des Genehmigers ein.
6. Wählen Sie die Rolle des Genehmigers in der Liste aus.
7. Wählen Sie den Namen des verantwortlichen Genehmigers in der Liste aus.
8. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. In der Felddefinitionstabelle finden Sie Informationen zu den optionalen Feldern.
9. Klicken Sie auf eine Stelle außerhalb der Tabelle, um die Änderungen zu speichern.
10. Wiederholen Sie die Schritte 3 bis 9, um weitere Genehmiger hinzuzufügen.
