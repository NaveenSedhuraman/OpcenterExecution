Verwalten von Checklisteninformationen

Eine Checkliste gewährleistet, dass ein Operator bei der Bearbeitung eines Qualitätsdatensatzes die erforderlichen Schritte durchführt. Checklistenfragen oder -elemente können je nach Definition der Checkliste erforderlich oder optional sein. Checklistenvorlagen können manuell zugewiesen werden, wenn Sie einen Datensatz verwalten.Checklisten können automatisch zugewiesen werden, wenn Sie ein Ereignis abschicken, oder manuell, wenn Sie einen Datensatz verwalten.

Beim Zuweisen von Checklisten

Checklisten sind für einen Qualitätsdatensatz optional. Wenn dem Datensatz eine Checkliste zugewiesen wurde und die Checkliste erforderliche Elemente enthält, kann der Datensatz jedoch erst gelöst werden, nachdem die Checkliste ausgefüllt wurde. Darüber hinaus fordert Sie die Anwendung auf, die erforderlichen Checklistenelemente abzuarbeiten, bevor Sie einen Qualitätsdatensatz speichern können. Klicken Sie auf die Schaltfläche "Checkliste aktualisieren", um die Informationen zu speichern, nachdem ein Checklistenelement abgearbeitet wurde.

Automatische Zuweisung einer Checkliste bei einer Ereignisübermittlung

Wenn ein Ereignis übermittelt wird, verwendet die Anwendung die Zuordnungen von Ereignisklassifizierungsspezifikationen und die Kategorieverarbeitungszuordnung für die primäre Organisation des Benutzers, der das Ereignis aufzeichnet, um die Checkliste zu bestimmen, die ggf. zugewiesen werden soll.

Checklisten können für jede Kombination von Ereignisklassifikation/-unterklassifikation direkt in der Zuordnung der Ereignisklassifizierungsspezifikation der Organisation angegeben werden. Checklisten können indirekt über die Einteilungsspezifikation einer Kategorie in der Kategorieverarbeitungszuordnung einer Organisation angegeben werden.

Hinweis: Wenn Checklisten in sowohl der Zuordnung der Ereignisklassifizierungsspezifikation als auch in der Einteilungsspezifikation der Kategorieverarbeitungszuordnung angegeben wurden, hat die Checkliste der Einteilungsspezifikation Vorrang.

Manuelle Zuweisung einer Checkliste

Checklisten werden folgendermaßen manuell zugewiesen:

• Durchführung der Aktion "Einteilung" bei Auswahl einer Checklistenvorlage. Die Aktion "Einteilung" kann auf mehreren Seiten durchgeführt werden, z. B. auf der Verwaltungsseite des Qualitätsdatensatzes und der Seite "Qualitätssuche". Weitere Informationen zur manuellen Durchführung der Einteilung finden Sie unter "Durchführen einer Einteilung".
Hinweis: Die Aktion "Einteilung" kann nur einmal für einen Datensatz durchgeführt werden. Die Anwendung deaktiviert die Aktion "Einteilung", nachdem diese verwendet wurde.
• Durchführung der Aktion "Checkliste zuweisen" auf der Verwaltungsseite eines Qualitätsdatensatzes.
