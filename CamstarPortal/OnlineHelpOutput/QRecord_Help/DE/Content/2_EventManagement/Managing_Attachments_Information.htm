Verwalten von Anhangsinformationen

Die Registerkarte "Anhänge" listet die Dateien auf, die mit dem Ereignis verbunden sind, und bietet die Möglichkeit, Dateien hinzuzufügen, zu bearbeiten und zu entfernen. Darüber hinaus können Sie die angehängten Dateien anzeigen.

Beim Verwalten von Anhängen

Unabhängig vom Modus des Ereignisses (Ansichtsmodus oder Bearbeitungsmodus) können Sie die Tabelle aktualisieren, die in der Tabelle angezeigten Informationen exportieren und angehängte Dateien anzeigen. Sie müssen den Bearbeitungsmodus für das Ereignis aktivieren, um Anhänge hinzuzufügen, zu bearbeiten oder zu entfernen. Im Bearbeitungsmodus sind in der Tabelle "Anhänge" zusätzliche Schaltflächen verfügbar, damit Sie diese Aufgaben durchführen können. Unter der Tabelle wird eine Erfolgsmeldung angezeigt, wenn Sie Anhänge hinzufügen, bearbeiten oder entfernen.

Wenn Sie Anhänge hinzufügen, verhindert die Anwendung, dass Sie folgende Zeichen im Titel eingeben: +, &, #.

Die maximale Dateigröße für einen Anhang beträgt 30 MB.

Beim Anzeigen von Anhängen

Wenn Sie remote auf die Datei zugreifen, müssen Sie u. U. im Browser zusätzliche Schritte durchführen, um die Datei herunterzuladen. Z. B. blockiert Internet Explorer möglicherweise das Herunterladen der Datei, bis Sie in der IE-Leiste auf "Datei herunterladen" geklickt haben.

Darüber hinaus müssen Sie die entsprechende Software installiert haben, um die Datei anzuzeigen. Falls der Anhang z. B. eine Excel-Tabelle ist, muss Microsoft Excel installiert sein, um die Datei anzuzeigen.

Beim Bearbeiten von Anhangsinformationen

Das einzige Feld, das Sie beim Bearbeiten von Anhangsinformationen aktualisieren können, ist "Beschreibung".

Vorgehensweise zum Anhängen einer Datei

Führen Sie die folgenden Schritte aus, um eine Datei anzuhängen:

1. Rufen Sie den Qualitätsdatensatz auf. Die Verwaltungsseite des Datensatzes wird angezeigt.
2. Klicken Sie auf die Registerkarte Anhänge. Die Seite "Anhänge" wird angezeigt.
3. Klicken Sie auf Bearbeitungsmodus aktivieren. Der Datensatz wird in den Bearbeitungsmodus versetzt.
4. Klicken Sie auf Neue Zeile hinzufügen. Das Popup "Anhänge hinzufügen" wird angezeigt.
5. Geben Sie den Dateipfad im Feld Originaler Dateipfad ein.

Oder

Klicken Sie auf Durchsuchen, um zu der Datei zu navigieren.

6. Geben Sie den Titel der Datei ein.
7. Geben Sie abhängig von den jeweiligen Geschäftsanforderungen optionale Informationen ein. In der Felddefinitionstabelle finden Sie Informationen zu den optionalen Feldern.
8. Klicken Sie auf Hochladen. Die Anwendung zeigt eine Meldung mit dem Hinweis an, dass der Anhang hochgeladen wurde. Die Dateiinformationen werden zur Tabelle "Anhänge" hinzugefügt.
9. Wiederholen Sie die Schritte 4 bis 8, um weitere Dateien hinzuzufügen.

Vorgehensweise zum Ändern von Anhangsinformationen

Führen Sie die folgenden Schritte aus, um Informationen für einen Anhang zu ändern:

1. Rufen Sie den Qualitätsdatensatz auf. Die Verwaltungsseite des Datensatzes wird angezeigt.
2. Klicken Sie auf die Registerkarte Anhänge. Die Seite "Anhänge" wird angezeigt.
3. Wählen Sie den Anhang aus, den Sie ändern möchten.
4. Klicken Sie in der Tabelle auf Ausgewählte Zeile bearbeiten. Das Popup "Bearbeiten" wird angezeigt.
5. Aktualisieren Sie das Feld Beschreibung je nach Bedarf.
6. Klicken Sie auf Hochladen. Die Anwendung zeigt eine Meldung mit dem Hinweis an, dass der Anhang aktualisiert wurde. Die Dateiinformationen werden in der Tabelle "Anhänge" aktualisiert.

Vorgehensweise zum Anzeigen einer angehängten Datei

Führen Sie die folgenden Schritte aus, um eine angehängte Datei anzuzeigen:

1. Rufen Sie den Qualitätsdatensatz auf. Die Seite für den Qualitätsdatensatz wird angezeigt.
2. Klicken Sie auf die Registerkarte Anhänge. Die Seite "Anhänge" wird angezeigt.
3. Wählen Sie den Anhang aus, den Sie anzeigen möchten.
4. Klicken Sie auf Anhang anzeigen. Daraufhin wird ein Popup mit der Aufforderung angezeigt, den Anhang zu speichern oder zu öffnen.
5. Klicken Sie auf Öffnen und dann auf OK. Die Anwendung öffnet die Datei zur Anzeige in einer separaten Seite.

Vorgehensweise zum Entfernen eines Anhangs

Führen Sie die folgenden Schritte aus, um einen Anhang zu entfernen:

1. Rufen Sie den Qualitätsdatensatz auf. Die Verwaltungsseite des Datensatzes wird angezeigt.
2. Klicken Sie auf die Registerkarte Anhänge. Die Seite "Anhänge" wird angezeigt.
3. Wählen Sie den Anhang aus, den Sie entfernen möchten.
4. Die ausgewählten Ausfallgründe werden aus der Tabelle gelöscht. Die Anwendung zeigt eine Meldung mit dem Hinweis an, dass der Anhang gelöscht wurde. Der Anhang wird aus der Tabelle entfernt.
