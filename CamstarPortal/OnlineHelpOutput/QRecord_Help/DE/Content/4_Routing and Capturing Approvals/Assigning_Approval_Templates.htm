Zuweisen von Genehmigungsvorlagen

Genehmigungsvorlagen enthalten eine Liste von Genehmigungsrollen, die entsprechenden Genehmiger und eine Liste von vordefinierten Genehmigungsentscheidungen. Wenn Sie einem Qualitätsdatensatz eine Genehmigungsvorlage zuweisen, wird gewährleistet, dass in der Anwendung den jeweiligen Genehmigern bei der Genehmigung die entsprechenden Entscheidungen angezeigt werden.

Beim Zuweisen von Genehmigungsvorlagen

Die Zuweisung von Genehmigungsvorlagen für Qualitätsdatensätze kann automatisch oder manuell erfolgen. Die Anwendung weist Genehmigungsvorlagen abhängig vom Datensatztyp automatisch Qualitätsdatensätzen zu. Genehmigungsvorlagen werden in der Kategorieverarbeitungszuordnung der Organisation Qualitätsdatensätzen zugewiesen.

Wenn einem Qualitätsdatensatz auf Organisationsebene keine Genehmigungsvorlagen zugewiesen werden, bietet die Anwendung dem Benutzer, der den Qualitätsdatensatz löst, die Möglichkeit, vor der Lösung des Datensatzes eine Genehmigungsvorlage auszuwählen und zu ändern. Informationen zum Zuweisen von Genehmigungsvorlagen beim Definieren von Kategoriezuordnungen finden Sie im Opcenter EX MDD Modeling-Benutzerhandbuch bzw. im Opcenter EX CR Modeling-Benutzerhandbuch.

Hinweis: Wenn Sie Kategoriezuordnungen für eine Organisation eingerichtet und festgelegt haben, dass eine Genehmigung für den Qualitätsdatensatztyp erforderlich ist, ohne eine Genehmigungsvorlage anzugeben, verlangt die Anwendung, dass ein Operator mit Eigentumsrechten bei der Lösungsgenehmigung eine Genehmigungsvorlage zuweist.

Wenn einem Qualitätsdatensatz eine Genehmigungsvorlage zugewiesen wird, wird diese zu einem Genehmigungsblatt. Alle Änderungen, die Sie an einem Genehmigungsblatt vornehmen, haben keinen Einfluss auf die zugrunde liegende Genehmigungsvorlage.

Beim Bearbeiten von Genehmigungsblättern, die Qualitätsdatensätzen zugeordnet sind

Sie können Genehmigungsblätter, die Qualitätsdatensätzen zugewiesen sind, auf der Seite "Qualitätsobjektlösung" bearbeiten (im Rahmen der Qualitätsdatensatzlösung). Weitere Informationen zur Bearbeitung von Genehmigungsblättern bei der Lösung von Qualitätsdatensätzen finden Sie unter "Vorgehensweise zum Bearbeiten von Genehmigungsinformationen" und "Vorgehensweise zum Hinzufügen von Genehmigern für die Genehmigung von Qualitätslösungen".
