Genehmigen von Qualitätsdatensatzlösungen

Der Genehmigungsprozess für einen Qualitätsdatensatz beginnt, wenn der Eigentümer des Datensatzes das Genehmigungsblatt an die vorrangigen Genehmiger weiterleitet. Die Anwendung führt folgende Aktionen durch:

• Im Concierge des auf dem Genehmigungsblatt angegebenen Genehmigers wird eine Genehmigungsaufgabe erstellt.
• In der Nachrichtenzentrale der angegebenen Genehmiger und der entsprechenden Rollenmitglieder werden eine Nachricht und eine Genehmigungsaktion erstellt.
• Es wird eine E-Mail-Benachrichtigung erstellt und an alle Genehmiger und die entsprechenden Rollenmitglieder gesendet (sofern vom Systemadministrator konfiguriert).
Hinweis: Die Anwendung ist in der Lage, die Einträge in der Verteilerliste aufzulösen, sodass der in der Rolle definierte Benutzer, der auch als Genehmiger angegeben wurde, nur eine E-Mail erhält.

Rollenmitglieder können die Genehmigungsseite auch über die Aktion "Genehmigen" auf der Seite "Qualitätssuche" und der Verwaltungsseite des Qualitätsdatensatzes aufrufen.

Beim Genehmigen von Qualitätsdatensatzlösungen

Bei der Genehmigung einer Qualitätsdatensatzlösung umfasst der Genehmigungsprozess folgende Aktionen:

• Anzeigen der Qualitätsdatensatzlösung und der Genehmigungsdetails
• Erfassen einer Genehmigungsentscheidung
• Erfassen einer elektronischen Unterschrift (sofern zutreffend)

Wenn Sie im Concierge eine Genehmigungsaufgabe oder in der Nachrichtenzentrale, der Qualitätssuche oder der Datensatzverwaltungsseite die Aktion "Genehmigen" auswählen, zeigt die Anwendung die Seite "Genehmigung unterschreiben" für den Qualitätsdatensatz an. Sie können Anweisungen oder Kommentare, die im Zusammenhang mit dem Genehmiger stehen anzeigen, indem Sie den Nutzer in der Genehmiger-Tabelle auf der Seite "Genehmigung Unterzeichnen“ auswählen.

Je nachdem, wie die Genehmigungsentscheidungsoptionen definiert wurden, kann die Anwendung Sie u. U. auffordern, Genehmigungskommentare einzugeben, bevor Sie Ihre Entscheidung übermitteln. Es ist z. B. möglich, eine Genehmigungsentscheidungsliste mit zwei Entscheidungen zu erstellen: "Akzeptiert" und "Abgelehnt". Für die Entscheidung "Abgelehnt" wird die Option "Kommentare einschließen" ausgewählt. Wenn der Genehmiger bei der Genehmigung des Qualitätsdatensatzes "Abgelehnt" auswählt, aktualisiert die Anwendung das Feld "Genehmigungskommentare", um darauf hinzuweisen, dass ein Kommentar erforderlich ist.

Beim Übermitteln einer Genehmigungsentscheidung

Die Anwendung zeigt das Popup "Elektronische Unterschrift" an, wenn eine gültige elektronische Unterschrift erforderlich ist, um die Genehmigungsentscheidung aufzuzeichnen. (Anforderungen für elektronische Unterschriften für Qualitätsdatensätze werden auf Organisationsebene festgelegt.) Das Element wird aus dem Concierge und der Nachrichtenzentrale des Genehmigers entfernt, nachdem die Genehmigungsentscheidung übermittelt wurde.

Seite "Genehmigung unterschreiben"

Die Seite "Genehmigung unterschreiben" wird angezeigt, wenn Sie im Concierge auf eine Genehmigungsaufgabe oder auf einer der folgenden Seiten auf die Aktion "Genehmigen" klicken:

• Nachrichtenzentrale
• Seite "Qualitätssuche"
• Verwaltungsseite des Datensatzes

Vorgehensweise zum Genehmigen einer Qualitätsdatensatzlösung

Führen Sie die folgenden Schritte aus, um eine Qualitätsdatensatzlösung zu genehmigen:

1. Rufen Sie die Seite "Genehmigung unterschreiben" auf. Die Seite "Genehmigung unterschreiben" wird angezeigt.
2. Wählen Sie Ihre Rolle aus. Die Liste "Genehmigung für" wird aktualisiert.
3. Wählen Sie im Feld Genehmigung für eine Benutzer-ID aus.
4. Wählen Sie eine Genehmigungsentscheidung aus.
5. Geben Sie ggf. Kommentare oder Hinweise für die Genehmigungsentscheidung in das Feld Genehmigungskommentare ein.
6. Klicken Sie auf Abschicken. Wenn die Anwendung so konfiguriert wurde, dass eine elektronische Unterschrift für die Genehmigung erforderlich ist, wird das Popup "Elektronische Unterschrift" angezeigt. Andernfalls zeigt die Anwendung eine Nachricht mit dem Hinweis an, dass die Genehmigung durchgeführt wurde. Anschließend wird die Verwaltungsseite für den Datensatz angezeigt.
