Grundlegende Informationen zur Weiterleitung und Erfassung von Genehmigungen

Genehmigungsweiterleitung und -erfassung basieren auf der Bewegung von Genehmigungsblättern durch die Anwendung. Ein Genehmigungsblatt ist der Hauptmechanismus des Genehmigungsprozesses. Es enthält eine Liste der Genehmiger und Entscheidungen, die bei der Genehmigung möglich sind.

Definieren von Genehmigungsentscheidungslisten und Genehmigungsvorlagen

Vor Beginn des Genehmigungsverfahrens müssen Sie die jeweiligen Genehmigungsentscheidungen und Genehmigungsvorlagen definieren, die Sie dem Qualitätsdatensatz zuweisen, um Genehmigungen zu erfassen. Genehmigungsentscheidungslisten und Genehmigungsvorlagen werden in der Modellierung definiert.

Zuweisen von Genehmigungsvorlagen

Sie können Genehmigungsvorlagen für die Lösung von Qualitätsdatensätzen zuweisen. Wenn Sie Genehmigungsvorlagen für Qualitätsdatensatzlösungen zuweisen, werden die Vorlagen einer Kategorie in einer Organisation zugewiesen. Die Anwendung weist die Genehmigungsvorlage dann abhängig von der ausgewählten Lösungsoption automatisch dem Qualitätsdatensatz zu. Sie können eine Genehmigungsvorlage auch einem bestimmten Qualitätsdatensatztyp zuweisen, sodass die Genehmigung nur für den betreffenden Datensatztyp verwendet wird.

Nachdem einem Qualitätsdatensatz eine Genehmigungsvorlage zugewiesen wurde, wird die Vorlage zu einem Genehmigungsblatt. Genehmigungsblätter werden dann durch die Anwendung weitergeleitet.

Genehmigungsweiterleitung

Die Anwendung leitet eine Genehmigung weiter, wenn der Eigentümer einen Qualitätsdatensatz löst und auf die Schaltfläche "Weiterleiten" klickt. Wenn eine Genehmigungsvorlage für den Datensatztyp auf Organisationsebene ausgewählt wurde, kann der Eigentümer Veränderungen am Genehmigungsblatt vornehmen, bevor es übermittelt wird. Darüber hinaus kann der Eigentümer eine andere Genehmigungsvorlage auswählen, die ebenfalls geändert werden kann.

Erstellen von Concierge-Aufgaben für Genehmiger, von Nachrichten in der Nachrichtenzentrale und von E-Mail-Benachrichtigungen

Wenn Genehmigungen weitergeleitet werden, generiert die Anwendung eine Genehmigungsaufgabe im Concierge und eine Genehmigungsnachricht in der Nachrichtenzentrale für alle vorrangigen Genehmiger in der Genehmigungsvorlage. Die Concierge Genehmigungsaufgabe ist ein Link für die Durchführung der Genehmigung. Die Genehmigungsnachricht der Nachrichtenzentrale enthält eine Aktion für die Durchführung der Genehmigung.

Wenn ein Systemadministrator darüber hinaus noch eine E-Mail-Genehmigungsnachricht konfiguriert hat, sendet die Anwendung an alle vorrangigen Genehmiger eine E-Mail-Nachricht. Die E-Mail soll Genehmigern darüber informieren, dass sie eine Genehmigungsaufgabe durchführen müssen.

Die Einstellungen in der Genehmigungsvorlage bestimmen, ob nur der angegebenen Genehmiger oder auch andere Benutzer mit der gleichen Rolle wie der Genehmiger den Übergang genehmigen können. Wenn alle Mitglieder einer Rolle eine Lösung genehmigen dürfen, führt die Anwendung folgende Schritte aus:

• Es wird eine Genehmigungsaufgabe im Concierge des angegebenen Genehmigers erstellt.
• In der Nachrichtenzentrale des angegebenen Genehmigers werden eine Nachricht und eine Genehmigungsaktion generiert.
• In der Nachrichtenzentrale der entsprechenden Rollenmitglieder wird eine Nachricht generiert.
• An alle Rollenmitglieder wird eine E-Mail-Nachricht gesendet, wenn eine Genehmigungs-E-Mail konfiguriert wurde.

Nachdem die vorrangigen Genehmiger ihre Genehmigungen durchgeführt haben, erstellt die Anwendung automatisch Concierge-Aufgaben, Nachrichten und E-Mail-Benachrichtigungen für die nächste Ebene von Genehmigern, die in der Genehmigungsvorlage festgelegt sind. Die Anwendung sendet außerdem E-Mail-Benachrichtigungen an alle Benutzer mit derselben Rolle wie die betreffenden Genehmiger, wenn die Genehmigungsblätter und E-Mails entsprechend eingerichtet wurden.

Hinweis: Sie können die E-Mail Verteilerliste der Genehmiger in der Modellierung definieren und die E-Mail-Nachricht in der Modellierung erstellen. Weitere Informationen finden Sie im Opcenter EX MDD Modeling-Benutzerhandbuch bzw. im Opcenter EX CR Modeling-Benutzerhandbuch.

Erfassen von Genehmigungsentscheidungen

Wenn Sie der angegebene Genehmiger sind, können Sie auf ein Element im Concierge oder in der Nachrichtenzentrale auf das Element für eine Genehmigungsaufgabe klicken, um die Qualitätsdatensatzlösung anzuzeigen, die Sie genehmigen müssen. Wenn Sie die gleiche Rolle wie der angegebene Genehmiger haben und berechtigt sind, die Genehmigung durchzuführen, müssen Sie zuerst den Datensatz aufrufen und dann im Bereich "Qualitätsaktionen" oder in der Nachrichtenzentrale auf die Aktion "Genehmigen" klicken, um die Genehmigungsseite anzuzeigen. Wenn Sie Ihre Genehmigungsentscheidung übermitteln, kann die Anwendung Sie u. U. auffordern, eine elektronische Unterschrift einzugeben, falls für den Genehmigungsvorgang elektronische Unterschriften konfiguriert wurden.

Abbrechen von Genehmigungen

Jeder Benutzer mit Eigentumsrechten kann den Genehmigungsprozess unabhängig davon abbrechen, ob ein Genehmiger bereits eine Entscheidung eingegeben hat. Der Status des Qualitätsdatensatzes muss "In Klärung" sein, damit der Genehmigungsprozess abgebrochen werden kann.

Anzeigen der Genehmigungshistorie

Sie können auf den folgenden Seiten jederzeit aktuelle und frühere Genehmigungszyklen (basierend auf dem Weiterleitungsdatum) einer bestimmten Genehmigungsvorlage anzeigen:

• Verwaltungsseite des Datensatzes, insbesondere die Registerkarte "Lösung"
• Seite "Genehmigung unterschreiben"

Die Anwendung erfasst für jeden Genehmiger die folgenden Informationen in der Zyklushistorie und zeigt diese an:

• "Level", "Rolle", "Name", "Genehmigt von" und "Entscheidung"
• Genehmigerkommentare (sofern vorhanden)
• Weiterleitungsdatum, Fälligkeitsdatum
• Fertigstellungsdatum (das Datum, an dem der Genehmiger den Qualitätsdatensatz genehmigt oder abgelehnt hat)
Hinweis: Die Anwendung ändert die Entscheidung in "Weiterleitung abgebrochen" und lässt das Fertigstellungsdatum leer, wenn der Eigentümer des Qualitätsdatensatzes eine Genehmigung abbricht.
