Erfassen einer elektronischen Unterschrift

Wenn Sie auf die Schaltfläche "Abschicken" einer Qualitätsdatensatztransaktion klicken, für die eine Unterschrift erforderlich ist, wird das Popup "Elektronische Unterschrift" angezeigt.

Beim Erfassen einer elektronischen Unterschrift

Wenn die Felder "Unterzeichner" und "Passwort" im Popup leer sind, müssen beide Felder ausgefüllt werden. Wenn das Feld "Unterzeichner" im Popup bereits Ihren Benutzernamen enthält, muss nur das Feld "Passwort" ausgefüllt werden.

Hinweis: Das Feld "Benutzer-ID-Eingabe erforderlich" im Modellierungsobjekt "Organisation" bestimmt, ob Sie sowohl Unterzeichner und Kennwort oder nur das Kennwort eingeben müssen.

Felddefinitionen des Popups "Elektronische Unterschrift"

Diese Tabelle definiert die Felder des Popups "Elektronische Unterschrift".

Feld

Definition

Typ

Unterzeichner

Benutzername der Person, die die elektronische Unterschrift übermittelt. Die Einstellungen im Modellierungsobjekt "Organisation" bestimmen, ob in diesem Feld standardmäßig der Name des angemeldeten Benutzers enthalten ist.

Erforderlich

Unterzeichner-Kennwort

Das Kennwort des Benutzers, der die elektronische Unterschrift übermittelt.

Erforderlich

Vorgehensweise zum Übermitteln einer elektronischen Unterschrift

Führen Sie die folgenden Schritte aus, um eine elektronische Unterschrift einzugeben und zu übermitteln:

1. Übermitteln Sie eine Transaktion, für die eine elektronische Unterschrift erforderlich ist. Das Popup "Elektronische Unterschrift" wird angezeigt.
2. Geben Sie den entsprechenden Benutzernamen in das Feld Unterzeichner ein, wenn dieses leer ist.
3. Geben Sie das entsprechende Kennwort in das Feld Passwort ein.
4. Klicken Sie auf Abschicken. Die Anwendung zeigt eine Erfolgsmeldung und dann wieder die Seite an, auf der Sie die Transaktion durchgeführt haben.
