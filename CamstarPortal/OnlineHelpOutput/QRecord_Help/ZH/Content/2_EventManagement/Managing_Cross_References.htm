管理交叉引用

“交叉引用”选项卡列出作为交叉引用添加到记录的事件记录。使用此选项卡可搜索和添加以及移除交叉引用。应用程序会更新添加为交叉引用的任何记录，以显示其与此记录交叉引用。

将记录添加为交叉引用

可以查看每个记录的信息，如所有者名称、分类、子分类等等。在搜索要添加为交叉引用的记录时，可以在“标识”和“容器”字段中使用 SQL 通配符。百分号 (%) 是 SQL 通配符。有关 SQL 通配符的信息，请参考 w3schools 网站：http://www.w3schools.com/sql/sql_wildcards.asp。

可以单击“查看所选事件详细信息”按钮来查看搜索返回的记录，或在单独的弹出窗口中查看交叉引用的记录。

在移除交叉引用时，应用程序将从记录列表中移除交叉引用。

在使用“搜索结果”表格的标题行中的复选框选择搜索返回的所有记录时，只选择显示的结果，而不是所有结果。

在添加或移除交叉引用后，单击“更新交叉引用”按钮。

如何添加交叉引用

按照以下步骤添加交叉引用：

1. 访问质量记录。将显示记录的管理页。
2. 单击交叉引用选项卡。将显示“交叉引用”页。
3. 单击设置编辑模式。记录切换到编辑模式。
4. 从选项卡顶部的一个或多个可用字段中选择搜索记录的准则。
5. 单击搜索。应用程序将显示符合搜索准则的所有记录，这些记录适合在“搜索结果”表格中添加为交叉引用。
6. 执行下述中的一项操作：
• 选中要添加为交叉引用的每个事件记录对应的复选框。
• 选中标题行中的复选框可全选显示的事件记录。
7. 单击添加所选引用。应用程序将更新记录并在“当前引用”表格中显示添加的引用。
8. 连续查看几页搜索结果并在必要时重复步骤 6-7。
9. 单击更新交叉引用。应用程序将显示一条消息，指出事件已更新。

如何移除交叉引用

按照以下步骤移除交叉引用：

1. 访问质量记录。将显示记录的管理页。
2. 单击交叉引用选项卡。将显示“交叉引用”页。
3. 单击设置编辑模式。记录切换到编辑模式。
4. 在“当前引用”表格中选择要移除的交叉引用。
5. 单击表格中的删除所选行。记录将从表格中删除。
6. 单击更新交叉引用。应用程序将显示一条消息，指出事件已更新。
