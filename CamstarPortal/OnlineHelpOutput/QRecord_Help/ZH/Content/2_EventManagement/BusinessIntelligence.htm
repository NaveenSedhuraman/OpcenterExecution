Opcenter Execution Core Intelligence

Opcenter Execution Core Intelligence (Opcenter EX CR Intelligence) 使用功能强大的 SAP® BusinessObjects™ 软件为企业提供实时业务智能 (BI) 信息。Opcenter EX CR Intelligence 是单独许可的模块。通过使用 Opcenter EX CR Intelligence，可以快速评估工厂生产流程的效率和质量。Opcenter EX CR Intelligence 是由 SAP® BusinessObjects™ 产品与 Opcenter EX CR 应用程序及 Opcenter Execution Medical Device and Diagnostics (Opcenter EX MDD) 应用程序的组件组合而成的综合产品。

Opcenter EX CR Intelligence 支持 Microsoft SQL Server 和 Oracle 数据库管理系统。基于门户的界面可用于访问多个报告和仪表板，确保能够快速应对业务环境中的变化。

报告

Opcenter EX CR Intelligence 包含 SAP® BusinessObjects™ Web Intelligence 报告和 SAP® Crystal Reports。这些报告用于监视生产环境的特定方面。有关可用报告的信息，请参考《Opcenter EX CR Intelligence 参考》中的“报告”一章。

仪表板

Opcenter EX MDD 或 Opcenter EX CR 仪表板在单个位置以可视化方式表示工厂生产的四个方面。快速分析仪表板之后，可以针对生产的特定区域进行改进。应用程序包含三个默认仪表板，其中一个用于管理质量。通过默认的“质量管理”仪表板，可以查看与工厂相关联的质量数据的各个方面。有关可用仪表板的信息，请参考《Opcenter EX CR Intelligence 参考》。

绩效指标

绩效指标提供了一种监视制造工艺关键方面的方法。绩效指标还通知用户维持最佳运营效率所需的适当操作。有关可用绩效指标的信息，请参考《Opcenter EX CR Intelligence 参考》中的“绩效指标”一章。
