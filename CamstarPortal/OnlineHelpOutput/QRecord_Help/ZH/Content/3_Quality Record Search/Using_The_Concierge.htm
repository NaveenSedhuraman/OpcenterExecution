使用任务中心

任务中心控件提供指派给您的记录和任务的合并列表。作为单个任务或质量记录创建的项将由应用程序启动并路由给相应的所有者。这些项显示在任务中心中，可供查看单一位置中的所有任务项。

访问任务中心

从应用程序的所有页面都可以访问任务中心。单击页面右上角的“任务中心”图标可以显示或隐藏任务中心。

使用任务中心

任务中心是一个可折叠控件，它将任务分类为四个部分（类别）。任务中心的每个项都是一个链接。单击链接将显示查看或完成该项所需的相应页面。以下列表描述各个类别和单击类别中的项时启动的页面：

• 我的指派 - 指派给您的记录。单击此部分中的项将显示该记录的管理页。
• 我的审批 - 需要您审批的记录。单击此部分中的项将显示该记录的“签署审批”页。
• 我的待处理项 - 已保存但未提交的记录。单击此部分中的项将显示事件记录页面流，以供恢复流程。
• 我的警报 - 有关指派给您的任务或记录的消息。单击此部分中的项将显示与该项相关的页面。例如，如果消息指示记录可以解决，单击该项将显示该记录的“质量对象解决”页。

首次查看任务中心时，默认展开“我的指派”部分。单击其他部分将展开相应部分以显示相关项的列表，并折叠“我的指派”部分。默认情况下，应用程序在每个部分中最多显示 10 个项。

注释: 每个部分显示的项数可在 Portal Studio 中进行配置。有关信息，请参考《Opcenter EX MDD Portal Studio 用户指南》或《Opcenter EX CR Portal Studio 用户指南》。

在每个项列表的底部，有一个“查看其他消息”链接。仅当该部分中有一个或多个项时，才会显示该链接。单击该链接可显示“消息中心”页，可以在其中查看该类别中项的信息或搜索其他项。有关消息中心的信息，请参考“使用消息中心”。

当将鼠标悬停在任务中心的某个项上时，应用程序将显示工具提示。工具提示包含项的完整描述和到期日期（如果已关联到期日期）。

当任务完成、更改了所有权或者记录关闭时，应用程序将从任务中心移除项。单击任务中心顶部的“刷新”按钮可刷新显示的列表。

到期日期指示器

到期日期图标提供了一个可视指示器，用于提醒任务何时到期。将鼠标悬停在任务中心的某个项上，可查看工具提示中的到期日期（如果适用）。

到期日期指示器是系统管理员在“建模”页的“门户消息类别”建模对象中配置的。有关信息，请参考《Opcenter EX MDD Modeling 用户指南》或《Opcenter EX CR Modeling 用户指南》。

如何访问任务中心项

按照以下步骤导航任务中心并访问需要注意的项：

1. 登录到 Opcenter Execution Medical Device and Diagnostics。将显示主页，任务中心窗格显示在页面右上角。

或

登录到 Opcenter Execution Core。将显示主页，任务中心窗格显示在页面右上角。

2. 单击任务中心图标（如果任务中心未显示）。再次单击此图标可隐藏任务中心。

或

单击“任务中心”窗格底部的关闭按钮。

3. 单击某个项可转到相应的页面并查看或处理该项。指派给您或您拥有的项显示在任务中心的相应部分。将显示所选链接的相应页面。
4. 单击查看其他消息链接可查看特定类别的所有消息。将显示消息中心，其中显示了该类别的所有邮件。
5. 单击项名称可访问已指派或拥有的项。将根据项类型和状态显示相应的页面。
