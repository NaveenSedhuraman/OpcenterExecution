使用消息中心

消息中心提供了一个集中的位置，可供查看您负责的所有项（审批、完成等）。消息中心同时显示任务中心消息和指派给所有者或所有者角色的质量记录的消息。可以使用消息中心的搜索功能查找特定记录。

访问消息中心

通过单击“搜索”菜单上的“消息中心”和单击“任务中心”中的“查看其他消息”链接，可以访问消息中心。无论以何种方式访问页面，应用程序都默认选中“显示我的消息”选项且在“受派者”字段中显示用户名。

搜索菜单

当从“搜索”菜单访问消息中心时，“结果”表格为空。必须执行搜索才能在“结果”表格中查看消息。

任务中心

当从任务中心访问消息中心时，“结果”表格包含消息列表。应用程序默认将“消息类型”字段设为“任务中心”类别（单击了其中的“查看其他消息”链接），并在“结果”表格中显示该类别的所有消息。例如，如果在“我的指派”类别中单击“查看其他消息”链接，则应用程序将默认把“消息类型”字段设为“我的指派”，并在“结果”表格中显示所有“我的指派”消息。

使用消息中心

可以从消息中心执行以下任务：

• 查看指派给您的消息
• 查看指派给您的角色（未直接指派给您）的消息
• 搜索消息
• 将“结果”表格的内容导出到 Excel 文件。
• 启动质量记录的管理或事务页

在“结果”表格中选择记录将启用相关的事务按钮。下表介绍可用的事务按钮。

管理记录

显示“归类”页。如果尚未对所选记录执行手动归类，并且记录的状态不是“审核中”或“关闭”，应用程序将启用此按钮。

如何搜索消息

按照以下步骤搜索消息：

1. 访问“消息中心”页。将显示“消息中心”页。
2. “结果”表格是否显示消息？

如果 ...

操作步骤 ...

是

您从任务中心访问消息中心，“结果”表格中将显示该任务中心类别中您被指派为所有者的所有消息。

执行下述中的一项操作：

• 在“结果”表格中选择一条消息，以对相关质量记录执行事务。
• 单击全部清除按钮执行新的搜索，然后转到步骤 3。应用程序清除搜索准则字段和“结果”表格。

否

您从“搜索”菜单访问消息中心。转到步骤 3。

3. 输入或选择一个或多个参数以缩小搜索范围。
4. 单击搜索。应用程序将在“结果”表格中显示所有符合搜索准则的消息。
