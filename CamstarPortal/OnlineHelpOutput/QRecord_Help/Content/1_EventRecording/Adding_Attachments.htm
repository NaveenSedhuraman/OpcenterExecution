Adding Attachments

The Attachment page is the last page in the generic event recording pageflow. The Attachments page provides you with the option to upload one or more attachments to the event. It is the only page where you can submit the event record to the application or delete a saved event. Refer to "Deleting a Saved Event" for information.

Attachments provide additional information that may explain why the event was created, standard operating procedure for such an event, and so on. Uploaded attachments appear in the Attachments grid. You can select a file in this grid to view, edit, or delete.

When Adding Attachments

Attaching a file automatically saves the quality record.

The application prevents you from entering these characters in the Title when adding attachments: +, &, #.

The maximum size for an attachment is 30 MB.

When Viewing Attachments

You must have the correct software installed to view the file. For example, if the attachment is an Excel spreadsheet, you must have Microsoft Excel installed to view the file.

If you are accessing the file remotely, your browser may block the file from being downloaded until you take additional steps. For example, Internet Explorer may block the file from being downloaded until you click Download File in the IE information bar.

When Editing Attachment Information

Description is the only field you can update when editing attachment information.

Attachments Page

The Attachments page appears when you click Next on the Affected Material page or when you click the last step in the pageflow. You can also access this page from a saved pageflow.

How to Attach a File

Follow these steps to attach a file:

1. Complete the "How to Add a Container to the Event" procedure to access the Attachments page.
2. Click Add new row. The Add Attachments pop-up appears.
3. Enter the path to the file in the Original File Path field or click Browse to navigate to the file.
4. Enter the Title of the file.
5. Enter optional information according to your business requirements. Refer to the field definitions table for information on optional fields.
6. Click Upload. The file information is added to the Attachments grid.
7. Repeat steps 2-6 to add more files.
8. Click Save.

How to View the Attached File

Follow these steps to view the attached file:

1. Select the attachment you want to view.
2. Click View Attachment. A pop-up appears prompting you to save or open the attachment.
3. Select Open and click OK. The file is opened in a separate page for viewing.

How to Edit Attachment Information

Follow these steps to edit attachment information:

1. Select the attachment you want to edit.
2. Click Edit selected row. The Edit pop-up appears.
3. Update the Description field as necessary.
4. Click Upload. The file information is updated in the Attachments grid.
