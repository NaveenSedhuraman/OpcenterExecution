Entering Failure Information

The Failures page provides a place for you to enter one or more failure sources for the event. Information for each failure source includes failure mode, type, severity, and description. This is the second page in the pageflow.

When Entering Failure Information

You can enter one or more failure modes for the event. If you select a failure mode for which a default failure type and failure severity were specified, the application populates those values automatically. The failure mode, failure type, and failure severity options are defined in Modeling.

Note: You can change the default Failure Type and Failure Severity values if necessary, or select those values if no defaults were specified.

Clicking the Add Failure button enters the failure mode information in the Event > Failure Modes grid. The Add Failure button is enabled only when you select or enter a failure mode.

Selecting a failure mode in the Event > Failure Modes grid populates the preceding fields with the failure mode information. The Add Failure button label changes to Update Selected Failure. Make your changes in the fields above and click Update Selected Failure to save your changes.

Failures Page

The Failures page appears when you click Next on the Classification page or when you click the second step in the Product Acceptance pageflow. You can also access this page from a saved pageflow.

How to Enter Failure Information

Follow these steps to enter failure information:

1. Complete the "How to Enter Classification and Description Information" procedure to access the Failures page.
2. Enter or select a Failure Mode Group, if applicable.
3. Enter or select a Failure Mode. Default information for the Failure Type, Failure Description, and Failure Severity fields appears if these associations were created in Modeling.
4. Enter optional information according to your business requirements. Refer to the field definitions table for information on optional fields.
5. Click Add Failure. The failure information is added to the Event > Failure Modes grid.
6. Repeat steps 2-5 to add additional failure modes.
7. Click Next. The Affected Material page appears.
