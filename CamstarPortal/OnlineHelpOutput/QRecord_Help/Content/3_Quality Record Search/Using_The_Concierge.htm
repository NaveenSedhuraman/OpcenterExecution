Using the Concierge

The Concierge control provides a consolidated list of the records and tasks assigned to you. Items that are created as individual tasks or quality records are initiated and routed by the application to the appropriate owners. These items appear on the Concierge and allow you to view all task items in a single location.

Accessing the Concierge

The Concierge is available on any page of the application. Click the Concierge icon in the upper right part of the page to display or hide the Concierge.

When Using the Concierge

The Concierge is an accordion control that sorts your tasks into four sections (categories). Each item in the Concierge is a link. Clicking on the link displays the appropriate page required to view or complete the item. The following list describes the categories and the pages launched when you click an item in a category:

• My Assignments - Records assigned to you. Clicking an item in this section displays that record's management page.
• My Approvals - Records that require your approval. Clicking an item in this section displays the Sign Approval page for that record.
• My Pending Items - Records that have been saved, but not submitted. Clicking an item in this section displays the event recording pageflow so you can resume the process.
• My Alerts - Messages about tasks or records assigned to you. Clicking an item in this section displays the page relevant to that item. For example, if the message indicates a record can be resolved, clicking the item displays the Quality Object Resolution page for that record.

The My Assignments section is expanded by default when you view the Concierge initially. Clicking a different section expands that section to display the list of related items and collapses the My Assignments section. The application displays a maximum of 10 items in each section by default.

Note: The number of items displayed per section is configurable in Portal Studio. Refer to the Opcenter EX MDD Portal Studio User Guide or the Opcenter EX CR Portal Studio User Guide for information.

At the bottom of each item list is a View Additional Messages link. The link appears only if there are one or more items in that section. Clicking this link displays the Message Center page where you can view information on the items in that category or search for other items. Refer to "Using the Message Center" for information on the Message Center.

The application displays a tool tip when you hover the mouse over an item in the Concierge. Tool tips include the complete description of the item and the due date (if a due date has been associated).

The application removes an item from the Concierge when the task is complete, ownership is changed, or the record is closed. Click the Refresh button at the top of the Concierge to refresh the displayed list.

Due Date Indicators

Due date icons provide a visual indicator to let you know when a task is due. Hover your mouse over an item in the Concierge to view its due date (if applicable) in the tool tip.

Due date indicators are configured in the Portal Message Category modeling object by a system administrator in Modeling. Refer to the Opcenter EX MDD Modeling User Guide or the Opcenter EX CR Modeling User Guide for information.

This table describes the indicator icons for items appearing in the Concierge.

For example, the system administrator may configure the application to display this icon 2 days before items are due.

Note: The application displays the due date in the tooltip that appears when you hover over an item with this icon.

How to Access Your Concierge Items

Follow these steps to navigate the Concierge and access the items requiring your attention:

1. Log on to Opcenter Execution Medical Device and Diagnostics. The home page appears with the Concierge pane displayed in the upper right corner.

Or

Log on to Opcenter Execution Core. The home page appears with the Concierge pane displayed in the upper right corner.

2. Click the Concierge icon if the Concierge does not appear. Click this icon again to hide the Concierge.

Or

Click the Close button at the bottom of the Concierge pane.

3. Click on an item to go to the appropriate page and view or process the item. Your assigned or owned items are displayed in the appropriate section of the Concierge. The appropriate page for the link you selected appears.
4. Click the View Additional Messages link to view all messages for a particular category. The Message Center displays with all the messages in that category.
5. Click on the item name to access an assigned or owned item. The appropriate page displays based on the item type and status.
