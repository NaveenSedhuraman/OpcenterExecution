E-mail Notifications

Siemens provides optional e-mail notification functionality to notify users when certain events occur in the application or action is required by the user. These are examples for which an administrator might configure the application to send an e-mail notification:

• A quality item has been created and assigned to the user
• The category of an quality record has changed
• Owner has been reassigned
• An approval is required

Configuring E-mail Notifications

The e-mail messages can be configured to send a generic message based on the task type, or include the quality record identifier and Concierge message in the e-mail. Additionally, the system administrator can configure the e-mail to go to a specific user or role. Refer to the Opcenter EX MDD System Administration Guide or the Opcenter EX CR System Administration Guide for information on configuring e-mail notifications.
