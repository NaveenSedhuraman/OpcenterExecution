Using the Message Center

The Message Center provides a centralized location for you to view all items (approvals, completions, and so on) for which you are responsible. The Message Center displays both Concierge messages and messages for quality records assigned to you as the owner or those assigned to your role. You can use the search features of the Message Center to locate specific records.

When Accessing the Message Center

You can access the Message Center by clicking Message Center on the Search menu and by clicking the View Additional Messages link in the Concierge. The application defaults to the Display My Messages option and displays your user name in the Assignee field regardless of how you access the page.

Search Menu

The Results grid is blank when you access the Message Center from the Search menu. You must perform a search to view messages in the Results grid.

Concierge

The Results grid contains a list of messages when you access the Message Center from the Concierge. The application defaults the Message Type field to the Concierge category in which you clicked the View Additional Messages link and displays all messages in that category in the Results grid. For example, if you click the View Additional Messages link in the My Assignments category, the application defaults the Message Type field to My Assignments and displays all My Assignment messages in the Results grid.

When Using the Message Center

You can perform the following tasks from the Message Center:

• View messages assigned to you
• View messages assigned to your role (that are not assigned to you directly)
• Search for messages
• Export the contents of the Results grid to an Excel file.
• Launch a management or transaction page for the quality record

Selecting a record in the Results grid enables the relevant transaction buttons. The table below describes the available transaction buttons.

Transaction Button

Click this button to . . .

Manage Records

Display the record's management page in view mode. The application enables this button for all quality records.

Triage

Display the Triage page. The application enables this button if manual triage has not been performed for the selected record and the status of the record is not In Review or Closed.

Approve

Display the Sign Approval page. The application enables this button only for My Approvals messages.

Note: Your system administrator can configure additional transactions for this page.

Clicking the Clear All button clears the values from all fields on the Message Center page, except for Organization.

Display Role Messages

Option to display the messages that belong to the roles to which you are assigned. The application will return messages that are not assigned to you directly as the owner. Selecting this option enables the Assignee field so that you can narrow your search to messages assigned to a specific role.

Organization

Organization for which the quality record was submitted or saved. Available selections are limited to the organizations to which the logged-in user belongs and within which the logged-in user is assigned one or more roles (either directly or through role propagation).

Category (or group) of messages you want to view. The categories include:

Date From

Beginning date of the date range within which a quality record was processed. Results include only records processed after this date. You can enter the date using the mm/dd/yyyy format or select the date by clicking the calendar icon to use the date picker control.

Optional

Date To

Ending date of the date range within which a quality record was processed. Results include only records processed before this date. You can enter the date using the mm/dd/yyyy format or select the date by clicking the calendar icon to use the date picker control.

When a date range is specified, the following fields apply for each entity selected:

• Quality records: Reporting Date
• Approval Entry Messages: Due Date

Optional

Role

Role associated with the message. Available selections are limited to roles to which the logged in user is assigned.

Optional

Assignee

Employee (owner) associated with the message. The application disables this field when you select the Display My Messages option and displays the username of the logged in user in this field. The application enables this field when you select the Display Role Messages option. Available selections are limited to the value selected in the Role field or, if no role is specified, all users in the same roles as the logged in user.

Optional

Sort Messages

Options for sorting the messages returned in the Results grid. Options include the following:

• Due Date (applies to approvals; other records sort to the end)
• Message Category
• Reporting Date

Optional

Results grid

Grid listing all messages that met the specified search criteria. Selecting a message enables transactions relevant to that message. The order the messages are displayed depends on the value, if any, specified in the Sort Messages field.

How to Search for Messages

Follow these steps to search for a message:

1. Access the Message Center page. The Message Center page appears.
2. Does the Results grid display messages?

If . . .

Then . . .

Yes

You accessed the Message Center from the Concierge and all messages assigned to you as the owner in that Concierge category appear in the Results grid.

Do one of the following:

• Select a message in the Results grid to perform a transaction on the associated quality record.
• Click the Clear All button to perform a new search, and then go to step 3. The application clears the search criteria fields and the Results grid.

No

You accessed the Message Center from the Search menu. Go to step 3.

3. Enter or select one or more parameters to narrow your search.
4. Click Search. The application displays all messages meeting your search criteria in the Results grid.
