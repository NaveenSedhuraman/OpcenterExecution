Managing Failure Information

The Failures tab allows you to view the failure modes assigned to the event and it also allows you to update existing failures. Additionally, you can add more failure modes to the event and modify assigned failure modes.

When Modifying Failure Information

Selection of a failure mode group restricts the list of failure modes to those that belong to the group. The default failure severity and failure type associated with the selected failure mode are displayed but can be changed.

Any failures added when the event was recorded, or that you just added, are displayed in the grid. You can remove an existing failure from the grid, if needed.

The Add Failure button appears when you are in edit mode on the Failures tab. You use this button to add failures to a quality record. Clicking the Add Failure button adds the failure to the Event > Failure Modes grid, and clicking the Update Failures button updates the quality record.

How to Add a Failure

Follow these steps to add a failure to an existing record:

1. Access the quality record. The record's management page appears.
2. Click the Failures tab. The Failures tab appears.
3. Click Set Edit Mode. The record is switched to edit mode.
4. Select a Failure Mode. Failure Mode Description, Failure Severity, and Failure Type are populated if those fields were defined with the Failure Mode in Modeling.
5. Enter optional information according to your business requirements. Refer to the field definitions table for information on optional fields.
6. Click Add Failure. The new failure information is added to the Event > Failure Modes grid.
7. Click Update Failures. The application displays a message stating the event has been updated.

How to Modify an Existing Failure

Follow these steps to modify an existing failure for an existing record:

1. Access the quality record. The record's management page appears.
2. Click the Failures tab. The Failures page appears.
3. Click Set Edit Mode. The application switches the record to edit mode.
4. Locate the failure you want to modify in the Event > Failure Modes grid, and select that row. The application displays the values for the selected event in the fields above and replaces the Add Failure button with the Update Selected Failure button.
5. Make any necessary changes.
6. Click Update Selected Failure. The application updates the failure information in the Event > Failure Modes grid and replaces the Update Selected Failure button with the Add Failure button.
7. Click Update Failures. The application displays a message stating the event has been updated.

How to Remove a Failure

Follow these steps to remove a failure from an existing record:

1. Access the quality record. The record's management page appears.
2. Click the Failures tab. The Failures page appears.
3. Click Set Edit Mode. The application switches the record to edit mode.
4. Locate the failure to be deleted in the Event > Failure Modes grid and select that row.
5. Click Delete selected row in the grid. The failure is removed from the grid.
6. Click Update Failures. The application displays a message stating the event has been updated.
