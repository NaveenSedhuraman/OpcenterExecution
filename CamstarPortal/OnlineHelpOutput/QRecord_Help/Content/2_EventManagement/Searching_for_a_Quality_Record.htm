Searching for a Quality Record

Opcenter EX MDD and Opcenter EX CR provide a search feature enabling you to search for and retrieve submitted quality records based on specified criteria. The Quality Search page, accessible from the Search menu, allows you to perform the following tasks:

• Specify criteria to locate a record
• Perform a transaction on a record selected in the search results
• Export the search results to Excel

When Searching for a Quality Record

You can view only quality records that exist in an organization in which you are assigned one or more roles, either directly, or through role propagation. The application returns only those records that have been submitted. The application does not return records with these statuses:

• Pending (Records that have been saved, but not submitted)
• Deleted (Records that were deleted before being submitted)

Clicking Search without specifying any search criteria returns every submitted quality record you are authorized to access. You can select only one record in the search results. The application enables only the transactions relevant to the selected record. The application displays paging icons if your search returns more records than the row limit configured for the Search Results grid.

Using SQL Wildcards

The Identifier, Container, and Description fields support SQL wildcards for searching.

Specifying Multiple Failure Modes

You can select one or more failure modes when specifying search criteria. The Failure Modes field will display a count rather than the selected failure modes. For example, if you select two of the six available failure modes, the application displays Selected 2 of 6 in the Failure Modes field.

When Viewing Search Results

The Search Results grid displays all quality records matching the specified search criteria. Selecting a record in the Search Results grid after the initial search displays buttons for the available, default actions you can perform. This table describes these actions.

Button

Click this button to . . .

Manage Records

Display the record’s management page. The application enables this button for all quality records.

Triage

Display the Triage page. The application enables this button if manual triage has not been performed for the selected record and the status of the record is not In Review or Closed.

Approve

Display the Sign Approval page. The application enables this button after you create the record and route it for approval.

How to Search for a Quality Record

Follow these steps to search for a quality record:

1. Select Quality Search from the Search menu. The Quality Search page appears.
2. Enter or select one or more criteria to narrow your search.
3. Click Search. The application displays the quality records matching your search criteria in the Search Results grid.
