Resolving a Record

Use the Resolve action to close records. When you resolve a record, you change its status to Closed and you can no longer perform any actions on it. You can resolve both events and nonconformances.

Note: You can perform the Reopen action to open a closed record.

Resolution is a two-part process if approvals have been configured for the organization that owns the record. Quality record resolutions must first be submitted for approval (and approved) before they can be resolved.

When Resolving a Quality Record

The Resolve button on the Manage G/E page is disabled if the status of the record is Pending, In Review, or Closed. You can resolve a record with a status of In Review if all approvals have been completed. The Resolve button is enabled on the Manage G/E page in this situation.

The Resolve button on the Manage P/E page is disabled if the status of the record is Pending, In Review, or Closed. You can resolve a record with a status of In Review if all approvals have been completed. The Resolve button is enabled on the Manage P/E page in this situation.

You can resolve a quality record when:

• All required items have been completed on the checklist assigned to the record.
• You have permission to resolve quality records.
• All quantity in containers associated with the record has been dispositioned.

The application displays an error message if you attempt to resolve a record without these conditions being met.

The application requires you to select a resolution code when resolving a quality record. Resolving a record:

• Locks the record against editing or performing any action except Reopen.
• Removes the record from the Concierge. You must perform a search to locate and view Closed records.

Quality Record Resolution, Approvals, and Electronic Signatures

During the resolution process, the application may require resolution approvals, electronic signatures, or both, depending on how the organization that owns the record was defined:

• The requirement for quality record resolution approval is set in the organization's Category Map. You must submit the resolution for approval and receive approval before you can complete the resolution process, if either an approval template has been specified or the Approval Required check box has been selected for a record type (such as Event) in the Category Map.
• The application requires electronic signatures if the quality object transaction in the organization to which the quality record is assigned has been configured with an electronic signature requirement (set in the organization’s Electronic Signature Txn Map). If so, the application will request the signature when you submit the resolution.
• The application will request the signature when you submit the resolution if the Close Quality Object quality service has been added to the Electronic Signature Txn Map.

Refer to the Opcenter EX MDD Modeling User Guide or the Opcenter EX CR Modeling User Guide for information on defining approval templates, defining organizations, and configuring electronic signature requirements for an organization.

Modifying the Specified Approval Template

The initial approval template specified on the Quality Object Resolution page is the template specified at the organization level for this record type. You can specify a different approval template before routing the approval. Additionally, if there is no approval template specified, you can select one.

Note: The application requires you to select an approval template before continuing if an approval template was not assigned at the organization level, but approval is required.

You can modify the assigned approval template by adding or removing approvers and updating information for existing approvers. Selecting an approver in the grid activates the row, enabling you to modify that approver’s information.

How to Resolve a Quality Record

Follow these steps to resolve a quality record:

1. Access the quality record. The record's management page appears.
2. Click Resolve in the Quality Actions pane. The Quality Object Resolution page appears.
3. Select a Resolution Code.
4. Enter an explanation or notes about the resolution in the Resolution Comments field, if necessary.
5. Is an Approval Template specified?

If . . .

Then . . .

Yes
a. Modify the template as necessary by completing "How to Edit Approval Information" and "How to Add Approvers for the Quality Resolution Approval" according to your business needs.

Note: You can also select a different approval template, if necessary. Go to step 6.

b. Click Route. The application displays a message indicating the record has been routed for approval. You must wait for the resolution to be approved before you can resolve the quality record.
c. Go to step 7.

No

Go to step 6.
6. Do you want to apply an approval template?

If . . .

Then . . .

Yes
a. Select a template from the Approval Template list. The application displays the template information on the page.
b. Modify the template as necessary by completing "How to Edit Approval Information" and "How to Add Approvers for the Quality Resolution Approval" according to your business needs.
c. Click Route. The application displays a message indicating the record has been routed for approval. You must wait for the resolution to be approved before you can resolve the quality record.
d. Go to step 7.

No

Go to step 7.
7. Click Resolve. The application displays a message indicating the record has been closed and displays the record's management page.

How to Edit Approval Information

Follow these steps to edit approval information:

1. Access the quality record. The record's management page appears.
2. Click Resolve in the Quality Actions pane. The Quality Object Resolution page appears.
3. Select an Approval Template from the list.
4. Select an approver in the Approvers grid. The row is activated.
5. Enter optional information according to your business requirements. Refer to the field definitions table for information on optional fields.
6. Click outside the grid to save your changes.
7. Repeat steps 3-6 for each approver you want to modify.

How to Add Approvers for the Quality Resolution Approval

Follow these steps to add approvers for the quality resolution approval:

1. Access the quality record. The record's management page appears.
2. Click Resolve in the Quality Actions pane. The Quality Object Resolution page appears.
3. Select an Approval Template from the list.
4. Click Add new row in the grid. A row is activated, displaying editable fields.
5. Enter the Level of the approver.
6. Select the approver’s Role from the list.
7. Select the responsible approver's Name from the list.
8. Enter optional information according to your business requirements. Refer to the field definitions table for information on optional fields.
9. Click outside the grid to save your changes.
10. Repeat steps 3-9 to add additional approvers.
