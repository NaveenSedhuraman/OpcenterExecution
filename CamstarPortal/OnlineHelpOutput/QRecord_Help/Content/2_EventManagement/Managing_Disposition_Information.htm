Managing Disposition Information

The Dispositions tab allows you to assign dispositions to the containers associated with the event record. Assigning a disposition involves making a decision on what to do with the affected material. Additionally, you can associate failure modes with the assigned dispositions to indicate how each failure was resolved.

When Assigning a Disposition and Associating Failure Modes

When assigning dispositions to containers associated with the event record, you can assign one or more dispositions to a single container. You can then associate one or more failure modes with a single disposition. Dispositions and failure modes are defined by a system administrator in Modeling.

The Dispositions tab contains two subtabs: Assign Disposition and Link Failure Mode.

Assign Disposition Subtab

Use the Assign Disposition subtab to associate dispositions with container quantities. All containers associated with the event record appear in a grid on this subtab. When you select a container and assign a disposition, the application adds a subgrid containing disposition information to that container row. You can show and hide disposition information by clicking the show row icon at the beginning of the row.

You can assign dispositions to multiple containers or to each container individually. Additionally, you can assign a disposition to the entire quantity of a container or a partial quantity. If you select to assign a disposition to the entire quantity, the application hides the Disposition Qty field. If you assign dispositions to a partial quantity, the application updates the container’s Qty Pending Disp value to indicate the remaining quantity to be dispositioned. You cannot disposition more than the amount currently in the container.

Important: Do not disposition an amount greater than the quantity in the container.

The Assign Disposition subtab also allows you to edit disposition information. Selecting an assigned disposition in the grid populates the disposition fields at the top of the subtab and changes the label on the Apply to Selected Container(s) button to Update Selected Disposition.

If failure modes are associated with the currently selected disposition, they remain linked even if you select a different disposition. You must use the Link Failure Mode subtab to change the associated failure modes, if necessary.

You can delete an assigned disposition from a container. Select the disposition row in the grid and click the Delete selected row button.

Link Failure Mode Subtab

Use the Link Failure Mode subtab to associate failure modes with the assigned dispositions. Linking failure modes to a specified disposition indicates how each failure was resolved. You can associate failure modes to multiple dispositions or to each disposition individually.

Only the failure modes assigned to the event record are available for selection. When you select a disposition and associate a failure mode, the application adds a subgrid containing failure mode information to disposition. You can show and hide disposition and failure mode information by clicking the show row icon at the beginning of the rows.

You can delete an associated failure mode from a disposition. Select the failure mode row in the grid and click the Delete selected row button.

Disposition Shop Floor Actions

When you select the Disposition tab, the application enables all of the shop floor action buttons. Clicking any of these action buttons launches the appropriate pop-up. For example, if you click Hold (Multiple), the application displays the Hold (Multiple) pop-up.

Refer to "Shop Floor Actions" for information.

How to Assign a Disposition

Follow these steps to assign a disposition to the event:

1. Access the quality record. The record's management page appears.
2. Click the Disposition tab. The Disposition page appears.
3. Click Set Edit Mode. The record is switched to edit mode.
4. Select a Disposition.
5. Do want to disposition a partial quantity or the entire quantity?

If you want to disposition . . .

Then . . .

A partial quantity
a. Enter a quantity less than the quantity in the container to disposition in the Disposition Qty field.
b. Go to step 6.

The entire quantity
a. Select Apply to Entire Qty. The application removes the Disposition Qty field.
b. Go to step 6.
6. Type any Disposition Comments, if necessary.
7. Select the containers in the Event > Container > Disposition grid to which you want to assign the specified disposition.
8. Click Apply to Selected Container(s). The application adds a subgrid with disposition information to the selected containers and decreases each container’s Qty Pending Disp value accordingly.
9. Repeat steps 4-8 to assign additional dispositions to containers.
10. Click Update Disposition. The application displays a message stating the event has been updated.

How to Link a Failure Mode to a Disposition

Follow these steps to link a failure mode to a disposition:

1. Access the quality record. The record's management page appears.
2. Click the Disposition tab. The Disposition page appears.
3. Click Set Edit Mode. The record is switched to edit mode.
4. Click the Link Failure Mode tab. The failure mode fields are displayed.
5. Select a Failure Mode. The application displays the Failure Mode Description if one was provided by the user who defined the failure mode.
6. Expand the container rows in the Event > Container > Disposition grid to view the assigned dispositions.
7. Select the dispositions to which you want to link the specified failure mode.
8. Click Apply to Selected Disposition(s). The application adds a child subgrid with failure mode information to the selected dispositions.
9. Repeat steps 5-8 to link additional failure modes with the assigned dispositions.
10. Click Update Disposition. The application displays a message stating the event has been updated.
