Managing Resolution Information

The Resolution tab displays information entered during the quality record resolution process by the record owner and any users approving the resolution. The Resolution tab is always read only regardless of mode.

Note: Refer to "Resolving a Record" for information on resolving a quality record.
When Viewing the Resolution Tab

The Resolution section remains blank until a user with ownership rights resolves the record. After quality record resolution, this section displays a resolution code and any comments entered by the owner during resolution.

The Approval Sheet Cycle section remains blank until an approval template is assigned during record resolution. After an approval template is assigned, this section displays information such as the instructions for all approvers, a list of approvers, and information and instructions specific to each approver.

Resolution Tab Field Definitions

This table describes the fields on the Resolution tab.

Field

Definition

Type

Resolution

Resolution code selected by the owner when resolving the record.

Resolution Comments

Explanation or additional information about the resolution entered by the owner when resolving the record.

View Current

Option that displays approval capture information for the current approval process cycle. The information displayed in the Instructions for All Approvers, Instructions for Approver, and Approver Comments fields and the Approvers grid is for the current cycle.

Display Only

View History

Option that displays approval capture information for the period selected in the Approval Cycle field. Selecting this option enables the Approval Cycle field.

Optional

Approval Cycle

Previous approval cycles, if any, displayed by date and time. When you select a previous cycle, the application updates the Instructions for All Approvers, Instructions for Approver, and Approver Comments fields and the Approvers grid to display information for that cycle.

Note: Selecting the View History option enables this field.
