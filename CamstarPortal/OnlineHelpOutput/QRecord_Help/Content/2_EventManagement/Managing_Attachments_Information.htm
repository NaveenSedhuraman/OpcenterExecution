Managing Attachments Information

The Attachments tab lists the files attached to the event and allows you to add, edit, and remove files. Additionally, you can view the attached files.

When Managing Attachments

You can refresh the grid, export the information displayed in the grid, and view attached files regardless of the event’s mode (view or edit). You must switch the event to edit mode to add, edit, or remove attachments. Additional buttons are available on the Attachments grid when in edit mode to enable you to perform these tasks. A success message appears below the grid when you add, edit, or remove attachments.

The application prevents you from entering these characters in the Title when adding attachments: +, &, #.

The maximum size for an attachment is 30 MB.

When Viewing Attachments

If you are accessing the file remotely, your browser may block the file from being downloaded until you take additional steps. For example, Internet Explorer may block the file from being downloaded until you click Download File in the IE information bar.

Additionally, you must have the correct software installed to view the file. For example, if the attachment is an Excel spreadsheet, you must have Microsoft Excel installed to view the file.

When Editing Attachment Information

Description is the only field you can update when editing attachment information.

How to Attach a File

Follow these steps to attach a file:

1. Access the quality record. The record's management page appears.
2. Click the Attachments tab. The Attachments page appears.
3. Click Set Edit Mode. The record is switched to edit mode.
4. Click Add new row. The Add Attachments pop-up appears.
5. Enter the path to the file in the Original File Path field.

Or

Click Browse to navigate to the file.

6. Enter the Title of the file.
7. Enter optional information according to your business requirements. Refer to the field definitions table for information on optional fields.
8. Click Upload. The application displays a message stating that the attachment was uploaded. The file information is added to the Attachments grid.
9. Repeat steps 4-8 to add more files.

How to Edit Attachment Information

Follow these steps to modify attachment information:

1. Access the quality record. The record's management page appears.
2. Click the Attachments tab. The Attachments page appears.
3. Select the attachment you want to modify.
4. Click Edit selected row in the grid. The Edit pop-up appears.
5. Update the Description field as necessary.
6. Click Upload. The application displays a message stating that the attachment was updated. The file's information is updated in the Attachments grid.

How to View an Attached File

Follow these steps to view an attached file:

1. Access the quality record. The quality record's page appears.
2. Click the Attachments tab. The Attachments page appears.
3. Select the attachment you want to view.
4. Click View Attachment. A pop-up appears prompting you to save or open the attachment.
5. Select Open and click OK. The application opens the file in a separate page for viewing.

How to Remove an Attachment

Follow these steps to remove an attachment:

1. Access the quality record. The record's management page appears.
2. Click the Attachments tab. The Attachments page appears.
3. Select the attachment you want to remove.
4. Click Delete selected row in the grid. The application displays a message stating that the attachment was deleted. The attachment is removed from the grid.
