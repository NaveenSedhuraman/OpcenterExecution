Managing Affected Material Information

The Affected Material tab allows you to view and add containers with affected material to a quality record. You can search for and select one or more containers to be added to the event record as affected material. You can also add containers that are not tracked by the MES.

When Modifying Affected Materials Information

The Affected Material tab has two subtabs: Search and Add Manually.

Click the Update Affected Material button to save any information you add or modify on the Affected Material tab.

Search Subtab

The Search subtab provides filters you can use to search for containers you want to add to the event. This subtab provides three search options:

• Return Containers Only - Returns all containers matching the search criteria. This is the default option.
• Return containers matching search criteria, along with first level issued material as child rows - Returns all containers matching the search criteria and any materials issued to these containers. The application displays the containers matching the search criteria as parent rows and any materials issued to those containers as child rows. The parent rows are collapsed by default. (Note that materials issued to the issued materials are not shown.)
• Return container where issued material matches search criteria - Returns containers to which the specified container was issued. Selecting this option updates and adds fields by:
• Renaming the Product field to Issued Product
• Adding a From Issued Lot field and a From Issued Stock Point field

Clearing Search Results

Use the Clear All button to clear filters (parameters) and results on the Search subtab. Clearing the Search subtab enables you to start a new search. When you click the Clear All button, the application:

• Clears all specified search filters
• Clears all containers from the search results grid
• Does not clear added containers from the Affected Materials grid
• Continues displaying the Search tab

Add Manually Subtab

The Add Manually subtab allows you to scan or enter containers that you want to add to the event. You can use this tab to add containers or lots not tracked by the MES. If you use the Add Manually subtab to add containers that are tracked by the MES, the application does not update these containers on this subtab. Use the Container Maintenance transaction if you need to update attributes for these containers.

Affected Material Shop Floor Actions

When you select the Affected Material tab, the application enables the Hold (Multiple) and Release (Multiple) shop floor action buttons. Clicking any of the shop floor action buttons launches the appropriate pop-up. For example, if you click Hold (Multiple), the application displays the Hold (Multiple) pop-up.

The application refreshes the Event > Affected Material grid when you close the pop-up after performing the shop floor action. The refresh button clears the check boxes next to the selected containers you performed actions against and updates the details for those containers, if necessary. You can select the next containers to perform actions against as needed.

Refer to "Shop Floor Actions" for information.

How to Add Containers to a Quality Record Using the Search Subtab

Follow these steps to add containers to a quality record with the Search subtab:

1. Access the quality record. The record's management page appears.
2. Click the Affected Material tab. The Affected Material page appears.
3. Click Set Edit Mode. The record is switched to edit mode.
4. Enter or select one or more parameters to narrow your search.
5. Click Search. The application displays all containers matching the search criteria in the Search Results grid.
6. Select one or more containers in the Search Results grid.
7. Click Add Selected Containers. The container is added to the Event > Affected Material grid.
8. Click the Clear All button to clear the parameter fields and Search Results grid if you want to perform a new search.

Or

Click Update Affected Material. The application displays a message stating the event has been updated.

How to Add Containers to a Quality Record Using the Add Manually Subtab

Follow these steps to add containers to a quality record using the add manually subtab:

1. Access the quality record. The record's management page appears.
2. Click the Affected Material tab. The Affected Material page appears.
3. Click Set Edit Mode. The record is switched to edit mode.
4. Click the Add Manually subtab.
5. Enter the name of the Container/Lot. The application updates the Product, Quantity, and UOM fields when you tab out of the field or place focus on a different field.
6. Enter information according to your business requirements. Refer to the Affected Material Tab with Add Manually Subtab Field Definitions table for information on the search fields.
7. Click Add Container. The container is added to the Event > Affected Material grid.
8. Repeat steps 5-7 to add additional containers.

Or

Click Update Affected Material. The application displays a message stating the event has been updated.

How to Remove Containers from a Quality Record

Follow these steps to remove containers from a quality record:

1. Access the quality record. The record's management page appears.
2. Click the Affected Material tab. The Affected Material page appears.
3. Click Set Edit Mode. The record is switched to edit mode.
4. Select the container or lot you want to remove in the Event > Affected Material grid.
5. Click Delete selected row in the grid. The container/lot is removed from the grid.
6. Click Update Affected Material. The tab displays the updated information.
