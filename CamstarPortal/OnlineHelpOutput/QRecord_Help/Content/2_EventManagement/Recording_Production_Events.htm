Recording Production Events

When a user, such as a quality control inspector or an operator working on a product, finds a possible nonconformance, the user records a production event. Production event records allow your organization to track quality issues and to improve product quality. The event can also be closed if it is determined the event is a non-issue that can be resolved without further processing.
