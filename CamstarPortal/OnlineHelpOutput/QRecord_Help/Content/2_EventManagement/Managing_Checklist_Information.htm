Managing Checklist Information

A checklist ensures an operator completes required steps when processing a quality record. A checklist question or item may be required or optional, depending on how the template was defined. Checklist templates can be assigned manually when managing the record.Checklist templates can be assigned automatically when submitting an event or manually when managing the record.

When Assigning Checklists

Checklists are optional for a quality record. However, if a checklist is assigned to the record, and the checklist contains required items, the record cannot be resolved until the checklist is satisfied. Additionally, the application requires you to complete the required checklist items before saving a quality record. Click the Update Checklist button to save information after completing a checklist item.

Assigning a Checklist Automatically at Event Submission

When an event is submitted, the application uses the event classification spec maps and the category processing map for the primary organization of the event recorder to determine which checklist, if any, to assign.

Checklists can be specified directly for each event classification and subclassification combination in the organization’s event classification spec map. Checklists can be specified indirectly through the triage spec specified for a category in the organization’s category processing map.

Note: When checklists have been specified in both the event classification spec map and the triage spec referenced in the category processing map, the checklist specified in the triage spec takes precedence.

Assigning a Checklist Manually

Checklists are assigned manually by:

• Performing the Triage action, when you select a checklist template. The Triage action can be performed from multiple pages including the quality record’s management page and the Quality Search page. Refer to "Performing Triage" for information on performing triage manually.
Note: The Triage action can only be performed against the record once. The application disables the Triage action after one use.
• Performing the Assign Checklist action on the quality record’s management page.
