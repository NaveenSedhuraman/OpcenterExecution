Pageflow Properties

Pageflow properties allow you to view or edit the properties of Portal Studio pageflows. Pageflow properties appear in the Properties pane when a pageflow is selected in the working area. Refer to "Properties Pane" for information on the Properties pane.

Individual page nodes that appear within pageflows contain properties that allow you to view or edit the functionality of pages appearing in the pageflow. Refer to "Pageflow Page Node Properties Descriptions" for information on the properties that appear in the Properties pane when a page node is selected within a pageflow.

A description window displaying information about pageflow and pageflow page node properties appears beneath the Properties pane when a property is selected.

Note: Some properties contain sub-properties allowing for further customization. Sub-properties appear in a separate pop-up when you click the More button for their parent property.
Pageflow Properties Descriptions

This table provides descriptions of pageflow properties.

Page Property

Property Description

Created

Creation date and time of the pageflow.

CreatedBy

Creator of the pageflow.

Description

Short description of the pageflow.

HowLong

Time period since creation of the pageflow.

LastModified

Last modified date of the pageflow.

LastModifiedBy

User who modified the pageflow most recently.

TypeName

Code behind type of the pageflow.
Pageflow Page Node Properties Descriptions

This table provides descriptions of page node properties that appear when selecting a page node within a pageflow.

Page Property

Property Description

IsStartPage

Sets the page as the start page in the pageflow when set to True.

Note: Only one page can be set as the start page within a single pageflow.

Location

Location of the page node within the pageflow diagram when viewing a pageflow in the working area. This property uses an "x,y" format when setting the location of a page node.

PageDescriptionLabelName

Selects a label as the description beneath the page title on the tabs of the pageflow. Refer to "Searching for Page Action Labels" for information on using the Label Lookup pop-up.

PageDescriptionLabelText

Text that appears as the description beneath the page title on the tabs of the pageflow. Text entered here takes priority over the PageDescriptionLabelName property when both text is entered and a label is selected.

PageTitleLabelName

Selects a label as the title of the page on the tabs of the pageflow. Refer to "Searching for Page Action Labels" for information on using the Label Lookup pop-up.

PageTitleLabelText

Text that appears as the title of the page on the tabs of the pageflow. Text entered here takes priority over the PageTitleLabelName property when both text is entered and a label is selected.
