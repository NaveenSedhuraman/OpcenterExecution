Adding SeriesSettings

You can customize the appearance of data points on scatterplot and wafer map charts by adding SeriesSettings to your chart control.

When Adding SeriesSettings

SeriesSettings are a chart control specific property that allows you to specify these properties for a series of data points:

• Color
• Shape
• Size

You associate the SeriesSettings to the data point series' key. You can add multiple SeriesSettings to set unique data point appearances for each series passed to a chart control.

You can add SeriesSettings without associating the settings to specific series of data points. The application will automatically assign SeriesSettings to data point series in the order they are passed in to the chart control if you leave the Key field blank.

Note: You must set your chart control's ChartMode property to Scatterplot when adding SeriesSettings.
SeriesSettings Pop-up Field Definitions

This table defines the fields on the SeriesSettings pop-up.

Field

Definition

Type

General

Color

Sets the color of the data points in the series. Colors must be entered as a hexadecimal value. Set to #FF0000 by default.

Optional

Key

Associates the related Color, Shape, and Size properties with a series of data points. Color, Shape, and Size properties are applied to series in the order they are passed in to the chart control when no key is entered.

Optional

Shape

Sets the shape of the data points in the series. Options include Circle, Cross, Diamond, Square, TriangleDown, and TriangleUp.

Optional

Size

Sets the size of the data points in the series. Sizes can range from 1 to 100.

Optional

How to Add SeriesSettings

Follow these steps to add SeriesSettings.

1. Open a page or web part containing a chart control. The page or web part appears in the working area.
2. Click Show/Hide Properties if the Properties pane is hidden. The Properties pane appears.
3. Select a chart control in the working area. The Properties pane displays properties associated with the chart control.
4. Click More for SeriesSettings. The <control> - SeriesSettings pop-up appears.
5. Click Add. A new SeriesSetting is added to the list and its general properties appear in the right pane.
6. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
7. Repeat steps 5-6 to add additional SeriesSettings.
8. Click OK. The <control> - SeriesSettings pop-up closes.
9. Click Save. The application saves your changes and displays a success message.
