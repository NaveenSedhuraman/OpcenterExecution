Control Properties

Control properties allow you to view or edit the properties of controls. Control properties appear in the Properties pane when a control is selected in the working area. Refer to "Properties Pane" for information on the Properties pane.

Control properties that are not required by the currently selected control are hidden on the Properties pane.

Properties used for grids are hidden based on the type of grid currently selected.

Controls that format date or time settings use Microsoftâ€™s standard date and time formats. Refer to https://msdn.microsoft.com/en-us/library/8kb3ddd4(v=vs.110).aspx?cs-save-lang=1&cs-lang=vb#code-snippet-2 for information on Microsoft date and time formats.

Note: Some properties contain sub-properties allowing for further customization. Sub-properties will continue to display their parent property's description in the Properties pane description window.
