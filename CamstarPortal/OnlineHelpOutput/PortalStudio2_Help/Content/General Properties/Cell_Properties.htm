Cell Properties

Cell properties allow you to view or edit the properties of cells. Cell properties appear in the Properties pane when a cell is selected within a page or web part in the working area. Refer to "Properties Pane" for information on the Properties pane.

Cell Properties Descriptions

This table provides descriptions of cell properties.

Cell Property

Property Description

General

Column

Displays the column index within the layout grid.

ColumnSpan

Displays the amount of columns spanned within the layout grid.

Items

Collection of items within the cell.

Row

Displays the row index within the layout grid.

RowSpan

Displays the amount of rows spanned within the layout grid.

Layout

CssClass

Stylesheet class associated with the cell.

HorizontalAlignment

Sets the horizontal alignment of cell elements.

Orientation

Orientation. Options include the following:

• Native
• Vertical
• Horizontal

Padding

Sets the amount of space between the cell border and cell elements.

VerticalAlignment

Sets the vertical alignment of cell elements.
