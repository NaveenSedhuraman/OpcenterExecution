Data Contracts

A Data Contract is the object stack that defines the classes that can be used to generate the input and output messages that conform to the service contract expectations. They allow you to expose data, which is encapsulated in the UIComponents, to external sources.

Data Contracts also provide a means of transferring data as inputs or outputs to Web Pages and Web Parts.

When Enabling the Command Bar for a Web Page or Web Part

You can use data contracts to enable or disable the Command bar on any portal pages. The Command bar is enabled for certain shop floor pages by default. You can see the command bar is enabled for a page by viewing the data contract settings the Data Contract in the page's Properties pane.

How to Enable the Command Bar for a Web Page or Web Part

Follow these steps to enable the command bar for a Web Page or Web Part:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Double-click a Web Page or Web Part. The page or part appears in the working area.
4. Click Show/Hide Properties if the Properties pane is hidden. The Properties pane appears.
5. Select the DataContractMembers property from the Properties pane. The DataContractMembers popup appears.
6. Click Add. The Data Contract fields appear.
7. Complete the Data Contract fields with this data:

How to Disable the Command Bar for a Web Page or Web Part

Follow these steps to disable the command bar for a Web Page or Web Part:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Double-click a Web Page or Web Part. The page or part appears in the working area.
4. Click Show/Hide Properties if the Properties pane is hidden. The Properties pane appears.
5. Select the DataContractMembers property from the Properties pane. The DataContractMembers popup appears.
6. Select the CommandBarSettingType data contract. The data contract properties appear.
7. Clear the value in the StaticValue field.
8. Click OK. The DataContractMembers popup closes. The change is saved. The command bar is disabled for the selected Web Page or Web Part.
