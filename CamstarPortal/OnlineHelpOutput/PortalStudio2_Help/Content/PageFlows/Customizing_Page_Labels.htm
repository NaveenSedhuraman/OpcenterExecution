Customizing Page Labels

You can customize the title and description labels that appear on pages within a pageflow from the Properties pane.

When Customizing Page Labels

Page labels are customized by modifying pageflow page node properties. These properties customize page labels:

• The PageTitleLabelText and PageTitleLabelName properties configure the title that appears for a page.
• The PageDescriptionLabelText and PageDescriptionLabelName properties configure the description beneath a page title.

Refer to "Pageflow Page Node Properties Descriptions" for information on pageflow page node properties.

Note: LabelText properties take priority over LabelName properties when both text is entered and a label is selected. Refer to "Searching for Page Action Labels" for information on using the Label Lookup pop-up to select labels for the PageTitleLabelName and PageDescriptionLabelName properties.

How to Customize Page Labels

Follow these steps to customize page labels:

1. Access Portal Studio.
2. Expand Pageflows within the Components tab of the UI Components catalog.
3. Double-click a pageflow. The pageflow appears in the working area.
4. Select a page node on the pageflow. The page node properties for the selected page node appear in the Properties pane.
5. Enter a title in the PageTitleLabelText property in the Properties pane.
6. Enter a description in the PageDescriptionLabelText property in the Properties pane.
7. Click Save. The application saves your changes and displays a success message.
