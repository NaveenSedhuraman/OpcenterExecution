Grid Columns

You can add default columns to grids from the <Grid Name> - Columns pop-up. The <Grid Name> - Columns pop-up appears when you click More for Columns in the Grid Settings Editor pop-up. Refer to "Grid Settings Editor" for information on the grid settings.

Column properties appear dynamically depending on the currently selected column type.

<Grid Name> - Columns pop-up Field Definitions

This table provides a list of column settings found in the <Grid Name> - Columns pop-up.

Column Settings Property

Property Description

BindPath

Relative parent path or field name where data is contained.

CellActions

Displays the <Column Name> - CellActions pop-up. You can define additional cell actions including buttons, fly-outs, and links from the <Column Name> - CellActions pop-up.

CellStyle

Sets the cell style of the column.

CustomScriptHandlers

Displays the <Column Name> - CustomScriptHandlers pop-up. You can designate a custom script handler action including click, change, and onAddCellAttributes from the <Column Name> - CellActions pop-up.

DateFormat

Sets the accepted format for date and time.

DateFormatExample

Displays an example of an accepted date format.

DefaultValue

Sets a default value for entries in the column.

Editable

Sets the column as editable.

EditorProperties

Series of sub-properties that determine how data is passed to and from the column.

EditorProperties - General

FieldExpressions

Sets a bind path to the Business Logic object model to retrieve and/or submit data.

RequestData

Sets the request for data from an event.

RetrieveList

Determines when values are retrieved.

SelectionData

Sets the request for the selection data from an event.

ValueFilter

Filter selection items based on value of another object.

Frozen

Freezes columns which stay visible as the rest of the grid continues to scroll. Refer to "Freezing Columns" for information.

GroupByColumnName

Refers to the Named Subentity control to group by column name.

HeaderStyle

Sets the header style of the column.

Hidden

Hides the column.

HiddenContent

Hides content on the column.

HTMLTagTransform

Determines how HTML tags are converted or preserved. This property contains these options for transforming HTML tags:

• Auto - Automatically searches the text for HTML tags, “<TAG>”, and preserves and transfers the HTML code, while allowing for content formatting. This option is selected by default.
• Convert - Replaces the characters “<” and “>” with “&lt;” and “&gt;”. Replaces the characters "\" and "/" with "&#x22;" and "&#x2F;". Replaces the character " ' " with "&#x27;". This option displays the content contained within the HTML tags and disables any HTML formatting. This option is selected by default.
• Preserve - Preserves the text exactly as it exists in the database and displays any HTML formatting.

IsRowID

Specifies the column as the row identifier. Only one column can be defined as the row ID.

LabelName

Determines the caption that displays on the column. Refer to "Searching for Page Action Labels" for information on searching for predefined labels.

LabelText

Determines the caption that displays on the column.

ListDisplayExpression

List of display expression selection values.

ListValueColumn

An object model expression which identifies where the column's value comes from.

Name

Name of the column.

Required

Requires a value to be entered in the column.

Resizable

Allows the column to be resized.

Sortable

Allows the column to be sorted.

TypeFormat

Sets the Type as either a Subentity or NamedSubentity.

TypeValidation

Determines whether or not the application validates the selected Type.

UseCurrentTime

Sets the column to use the current time format.

ValueExpression

Textbox with a link to the Field Expression builder.

ValueFormat

Sets the value format as either Name or ID.

Width

Width of the column in pixels.
