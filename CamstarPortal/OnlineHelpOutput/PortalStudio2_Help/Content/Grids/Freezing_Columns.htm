Freezing Columns

You can freeze columns that appear in grids in the Portal by modifying the columns' grid control in Portal Studio. Frozen columns stay visible as the rest of the grid continues to scroll.

When Freezing Columns

You must freeze columns in the order they appear from top to bottom and without gaps in the <Grid Name> - Columns pop-up within the Grid Settings Editor.

Grids that contain sub-grids, row grouping, or are set for tree view cannot contain frozen columns. Grids that are sub-grids can contain frozen columns. Frozen columns cannot be edited inline in the Portal.

How to Freeze Columns

Follow these steps to freeze columns:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the Components Catalog is hidden. The Components Catalog appears.
3. Double-click any Portal page that contains a grid. The page appears in the working area.
4. Click Show/Hide Properties if the Properties pane is hidden. The Properties pane appears.
5. Select a grid control on the page. The settings for the control appear in the Properties pane.
6. Click More for Settings in the Properties pane. The Grid Settings Editor pop-up appears.
7. Click More for Columns. The <Grid Name> - Columns pop-up appears.
8. Select a column.
9. Select True for Frozen.
10. Repeat steps 8-9 for each sequential column you want to freeze.
11. Click OK twice. The Grid Settings Editor closes.
12. Click Save. The application saves your changes and displays a success message.
