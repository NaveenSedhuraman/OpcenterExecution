Enabling Auto Refresh Rate for SPC Charts

Portal Studio provides a configurable option to choose and determine an auto refresh rate for the SPC chart viewer page.

Note: The rate is set in milliseconds.

How to Enable an Auto Refresh Rate for SPC Charts

Follow these steps to enable an auto refresh rate for SPC charts:

1. Access Portal Studio.
2. Select the Components tab.
3. Enter "SPC" in the Search field. A list of SPC objects appears.
4. Select SPCChartsViewerVP from the list.
5. Click the SPCChartControl button.
6. Click Show/Hide Properties if the Properties pane is hidden. The Properties pane appears.
7. Click the SPCChartParams Expand button under Report Settings.
8. Enter a numeric value in the AutoRefreshRate field. (The minimum polling interval is 10 seconds.)
9. Click Save. The application saves your changes and displays a success message.
