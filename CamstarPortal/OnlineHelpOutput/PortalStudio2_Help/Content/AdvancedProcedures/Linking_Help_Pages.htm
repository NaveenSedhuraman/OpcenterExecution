Adding Help to a Portal Page

You can add help to a Portal page by modifying the page's HelpFileURL property in Portal Studio.

When Adding Help to a Portal Page

You must enter the help file's URL in the format FilePath#FileName, where FilePath is the file path and FileName is the help file's name.

Important: Siemens recommends you do not modify the HelpFileURL property for default Portal pages. Modifying this property can remove existing help files and any supported translated help files from Portal pages.

You may only need to enter part of a file path depending on your Portal Studio settings. Refer to "Configuring Portal Settings" for information on Portal Studio settings.

How to Add Help to a Portal Page

Follow these steps to add help to a Portal page:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the Components Catalog is hidden. The Components Catalog appears.
3. Double-click a page. The page appears in the working area.
4. Click Show/Hide Properties if the Properties pane is hidden. The Properties pane appears.
5. Enter the help file's URL in the HelpFileURL field in the Properties pane.
6. Click Save. The application saves your changes and displays a success message.
