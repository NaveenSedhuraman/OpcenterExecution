Enabling Status Message Sounds

Portal Studio allows you to configure sounds for status messages appearing on Portal pages. Status messages appear following the execution of a function in Portal Studio and consist of three types:

• Success
• Error
• Warning
Note: Status message sounds are disabled in the Portal by default.

When Enabling Status Message Sounds

Enabling status message sounds enables sounds to play for every type of status message. However, status message sounds can be disabled by type. For example, you can disable success and warning sounds, but still have a sound play for error status messages. Status message sounds can be further customized with user uploaded WAV or MP3 files. Uploaded files are automatically set to the sound path upon upload.

Note: Some versions of Internet Explorer do not support WAV files. Siemens recommends you use the MP3 file format for status message sounds.

How to Enable Status Message Sounds

Follow these steps to enable status message sounds:

1. Access Portal Studio.
2. Click the Portal Settings button.The Portal Settings pop-up appears.
3. Select StatusMessageSettings from the list of settings. The properties for StatusMessageSettings appear.
4. Select True for PlaySound.
5. Click OK. The Portal Settings pop-up closes.

How to Disable Status Message Sounds by Type

Follow these steps to disable status message sounds by type:

1. Access Portal Studio.
2. Click the Portal Settings button.The Portal Settings pop-up appears.
3. Select StatusMessageSettings from the list of settings. The properties for StatusMessageSettings appear.
4. Delete the file name for the sound path.
5. Click OK. The Portal Settings pop-up closes.

How to Upload Status Message Sounds

Follow these steps to upload status message sounds:

1. Access Portal Studio.
2. Click the Portal Settings button. The Portal Settings pop-up appears.
3. Select StatusMessageSettings from the list of settings. The properties for StatusMessageSettings appear.
4. Click the button. The Open pop-up appears.
5. Select a sound file and click Open. The sound file is uploaded for the status message type.
6. Click OK. The Portal Settings pop-up closes.

How to Set a Status Message Sound

Follow these steps to set a status message sound:

1. Access Portal Studio.
2. Click the Portal Settings button.The Portal Settings pop-up appears.
3. Select StatusMessageSettings from the list of settings. The properties for StatusMessageSettings appear.
4. Select a sound path file.
5. Click OK. The Portal Settings pop-up closes.
