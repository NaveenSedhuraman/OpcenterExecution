Using 3D Visualization

The 3D Visualization control enables you to view 3D images from within the application. The images appear in a separate 3D File Viewer pop-up that allows you perform these actions on an image:

• Rotate
• Zoom in and out
• Mark up
• Save a snapshot
When Using 3D Visualization

The 3D Visualization control supports JT files. 3D image files open in the 3D File Viewer pop-up. You can view 3D images using the 3D Visualization control or anywhere that a document set can be viewed.

The 3D File Viewer pop-up contains two modes:

• 3D, which allows you to freely rotate and zoom in and out on a 3D image.
• Markup (2D), which allows you to markup and save a 2D still image.

The 3D File Viewer pop-up displays images in 3D mode upon first opening a 3D image file. You can click and drag an image in 3D mode to freely rotate it. Scrolling up on the mouse wheel zooms in on the image and scrolling down zooms out. Switching to Markup (2D) mode creates a 2D still of the 3D image from the current perspective seen in the 3D File Viewer pop-up.

Markup (2D) mode contains a toolbar that allows you to add text and shapes with customizable line weights and colors. Refer to "3D File Viewer Pop-up Toolbar Options" for information on toolbar options and the tools available for marking up your 2D image.

You can click the Save button to take snapshots of your 2D image. Clicking the Save button displays a Save dialog box in your browser that allows you to save your snapshot as an image file on your local machine. A red border appears around the image whenever unsaved changes are present.

You can remove any markings or annotations by clicking the Reset button. Clicking the Reset button displays a warning message when unsaved changes are present. Click Yes on the warning message to remove your changes. Click No on the warning message to retain your changes.

You can click Back to 3D to switch back to 3D mode. Switching back to 3D mode removes any markings or annotations you have made. A warning message appears when you click Back to 3D if unsaved changes are present on your 2D image. Click Yes on the warning message to display a Save dialog box in your browser that allows you to save a snapshot on your local machine and then switch back to 3D mode. Click No to discard your changes and switch back to 3D mode.

3D File Viewer Pop-up Toolbar Options

This table defines the options available on the 3D File Viewer pop-up.

Option

Select this option to . . .

3D Mode

Markup (2D)

Switch to Markup (2D) mode. Switching to 2D mode creates a 2D still of the 3D image from the current perspective seen in the 3D File Viewer pop-up.

Markup (2D) Mode

Back to 3D

Switch back to 3D  mode.

Annotate

Add text to the image. The mouse cursor changes to a text tool after clicking Annotate. Clicking on the image with the text tool allows you to set text over the image.

Shape

Select a tool and add a straight or free form line, square, or circle.

Color

Select or enter custom color values for shapes and text. Set to black by default. Refer to "Using the Color Picker" for information on using the color picker to set custom colors.

Line Weight

Set the line weight of the shape tools. The default value is 3.
