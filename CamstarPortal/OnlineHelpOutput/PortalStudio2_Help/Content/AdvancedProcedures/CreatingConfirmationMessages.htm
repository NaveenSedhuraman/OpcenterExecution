Creating Confirmation Messages

You can create custom confirmation messages in Portal Studio that display when an action button is clicked in the Portal. Confirmation messages include a customized text message and the option to cancel or confirm a page action.

When Creating Confirmation Messages

Confirmation messages include a title, a message, a Cancel and an OK button. The labels for these elements are blank by default and require that you enter custom text.

Confirmation messages require action buttons to be present on a Portal page. Refer to "Configuring Page Actions" for information on how to add actions to a Portal page.

How to Create a Confirmation Message

Follow these steps to create a confirmation message:

1. Access Portal Studio.
2. Click the Show/Hide Catalog button if the Components catalog is hidden. The Components catalog appears.
3. Double-click any Portal page. The page appears.
4. Click More for Actions field in the Properties pane. The <pagename>-Actions pop-up appears.
5. Create a new action or select an existing action from the list available.
6. Expand Confirmation.
7. Enter text for the cancel button in the Cancel_LabelText field.
8. Enter a unique message for the confirmation message body in the Message_LabelText field.
9. Enter text for the OK button in the OK_LabelText field.
10. Enter a title for the confirmation message in the Title_LabelText field.
11. Click OK. The <pagename>-Actions pop-up closes.
12. Click Save. The application saves your changes and displays a success message.
