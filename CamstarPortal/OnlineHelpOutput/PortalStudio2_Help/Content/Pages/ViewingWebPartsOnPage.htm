Viewing Web Parts on a Page

Portal Studio allows you to view which web parts are being used on a page. You can view both the dynamic web parts and static web parts used on a page.

When Viewing Web Parts on a Page

When you select a web part on the DynamicWebParts modal, its properties appear in the right pane. You can then use the right pane to drill down further.

Note: You cannot modify web part properties from the DynamicWebParts modal.

How to View Web Parts on a Page

Follow these steps to view web parts on a page:

1. Access Portal Studio.
2. Double-click a page. The page appears in the working area.
3. Click More for DynamicWebParts in the Properties pane. The DynamicWebParts modal appears and displays a list of the web parts currently on the page.
4. Select a web part from the list in the left pane. The web part's properties are displayed in the right pane.
