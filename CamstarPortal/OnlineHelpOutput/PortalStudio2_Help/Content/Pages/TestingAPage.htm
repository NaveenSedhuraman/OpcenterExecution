Testing a Page

Portal Studio allows you to test a page within the Portal framework.

When Testing a Page

You test a page using the Run button. The Run button is enabled when the page is saved or when no unsaved changes are present.

Tested pages open in the Portal framework in a separate browser tab as if it were accessed from a Portal menu .

Important: Any data you enter on a page while testing is saved to your environment's database.

How to Test a Page

Follow these steps to test a page:

1. Access Portal Studio.
2. Double-click a page. The page appears in the working area.
3. Click Run. The page is opened in the Portal in a separate browser tab.
