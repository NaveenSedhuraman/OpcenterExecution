Managing Page Groups

Portal Studio enables you to categorize the list of available pages into groups in the Components catalog. Pages can be assigned to multiple groups.

Note: Clear browser data or click “Restore Default Settings” to return the default group and page assignments.

When Creating New Page Groups

Right-clicking a page and selecting Add to a new group automatically adds the page to the newly-created page group.

You can edit the name of an existing page group from the shortcut menu used to create a page group.

When Adding Pages to Groups

Some default pages can appear in the Undefined group in the Components tab of the Components catalog when logging into Portal Studio for the first time when upgrading from an earlier version.

You can automatically sort the default pages in the Undefined group into other groups from the Components catalog. Refer to "How to Sort Default Pages Automatically" for information.

Note: Pages cannot be assigned to the All or Undefined groups.

When Deleting Page Groups

Page groups can be deleted from the Components catalog. Removing a group does not delete the pages contained within it.

Note: The All group cannot be deleted.

How to Create a New Page Group

Follow these steps to create a new page group:

1. Access Portal Studio.
2. Right-click a page and select Add to a new group. The Enter a group name dialog appears.

Or

Right-click an existing page group and select Add a new group. The Enter a group name dialog appears.

3. Enter a name for the group.
4. Click OK. The dialog closes and a new group appears in the Components catalog.

How to Edit a Page Group

Follow these steps to edit a page group:

1. Access Portal Studio.
2. Right-click a page group and select Edit. The Enter a group name dialog appears.
3. Enter a new name for the group.
4. Click OK. The dialog closes and the group appears in the Components catalog with the new name.

How to Add a Page to a Group

Follow these steps to add a page to a group:

1. Access Portal Studio.
2. Right-click a page. A list of existing groups appears.
3. Mark the check box for the group to which you want to add the page. The list closes and the page is added to the selected group.
4. Repeat step 4 to add the page to additional groups as needed.

How to Delete a Page from a Page Group

Follow these steps to delete a page from a page group:

1. Access Portal Studio.
2. Expand a page group.
3. Right-click a page and clear the page groups you want to remove the page from. The page is removed from the cleared groups.

How to Delete a Page Group

Follow these steps to delete a page group:

1. Access Portal Studio.
2. Right-click a group and select Remove. The group is removed from the Components catalog.

How to Sort Default Pages Automatically

Follow these steps to sort default pages automatically:

1. Access Portal Studio.
2. Select the All group.
3. Right-click the All group and select Restore default settings. The default pages in the Undefined group are sorted into other groups.
Note: Clearing browser settings also returns groups and pages to the default settings.
