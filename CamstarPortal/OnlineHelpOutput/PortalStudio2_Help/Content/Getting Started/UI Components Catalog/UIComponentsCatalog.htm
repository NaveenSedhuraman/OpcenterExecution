Components Catalog

The Components catalog contains several tabs that provide access to component, navigation, and mapping components.

The Components tab provides access to these components:

• Pages
• Web Parts
• Pageflows

The Navigation tab provides access to these navigation components:

• Menus
• Tabs

The Modeling tab provides access to these mapping components:

• Page Actions
• Page Action Categories
• Page Action Rules
• Page Maps
• Target Matrices

The Components catalog tabs are organized into drop-downs for each component type. Each drop-down contains a search field, Create New Item button, a set of view mode buttons, and a refresh button. Clicking the Create New Item button will display a new item for that item type in the working area.

The Catalog button on the Portal Studio toolbar displays the Components catalog if hidden, and hides the Components catalog if visible.

Note: Some default pages can appear in the Undefined group in the Components tab of the Components catalog when logging into Portal Studio for the first time after upgrading from an earlier version. You can sort the default pages in the Undefined group into other groups automatically. Refer to "How to Sort Default Pages Automatically" for information.
Opening Existing Components

The Components Catalog contains all default components and any components you have created. You can open any component from the Components Catalog by navigating to a component's tab and section and then double-clicking the components name. The component you open appears in the working area.
