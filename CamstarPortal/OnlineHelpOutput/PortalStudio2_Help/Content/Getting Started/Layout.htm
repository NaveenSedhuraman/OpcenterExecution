Managing Layouts

Portal Studio allows you to configure the layout of a page or web part using a row and column layout containing cells.

You can modify a single cell's properties in the Properties pane. Refer to "Cell Properties" for information on modifying cell properties.

When Managing Layouts

Every page or web part contains at least one cell as part of the component's layout. Cells allow you to specify where components will appear when placed on pages and web parts in relation to components in other cells.

You can modify a cell 's layout using the Actions button when viewing a component in the Design view. You can select multiple cells by first selecting one cell and then selecting another cell. All cells between the two cells are included in your selection. Selected cells display a blue background.

This table defines the options available from the Actions button for managing cell layouts.

Option

Select this option to . . .

Merge

Merge the selected cells into a single cell.

Split

Split a merged cell back into its component cells.

Remove Row

Delete the selected row.

Remove Column

Delete the selected column.

Insert Row Before

Insert a new row above the selected row.

Insert Column Before

Insert a new column to the left of the selected column.

Insert Row After

Insert a new row below the selected row.

Insert Column After

Insert a new column to the right of the selected column.

Reset

Deselects the currently selected cells.

Toggle Cell Position

Toggle [row, column] position indication of the cell.
