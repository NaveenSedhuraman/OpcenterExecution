Portal Studio Global Toolbar

The Portal Studio global toolbar provides easy access to the functions used in Portal Studio.

Portal Studio Global Toolbar Buttons

This table defines the buttons available on the Portal Studio global toolbar.

Button

Click this button to . . .

View Contents

Change the view of the component in the working area to another workspace. Set to Developer by default. Refer to "Developing in Workspaces" for information.

Note: This option only appears when a component is open and selected in the working area.

Portal Settings

Display the Portal Settings window. Refer to "Configuring Portal Settings" for information on modifying Portal settings.

Back to Portal

Return to the Portal.

Help

Display Portal Studio online help.

Logout

Logout of Portal Studio and the Portal and return to the login page.

New

Create a new component . Clicking this button displays a drop-down with options for creating new UI components.

Save

Save the pending changes for component selected in the working area. Refer to "Saving Changes" for information on saving your changes.

Note: Portal Studio enables this button when you make a change to the component currently in focus in the working area. You can set this button to always be enabled by setting the AlwaysEnableSave Studio setting to True. Refer to "Studio Settings" for information on modifying Portal Studio settings.

Save All

Save the pending changes for components in all open tabs in the working area. Refer to "Saving Changes" for information on saving your changes.

Note: Portal Studio enables this button when you make a change to the component currently in focus in the working area. You can set this button to always be enabled by setting the AlwaysEnableSave Studio setting to True. Refer to "Studio Settings" for information on modifying Portal Studio settings.

Undo

Undo your recent changes. The Undo button displays a counter showing the number of changes you can undo. Pressing the Shift key while clicking Undo displays a list of actions you can undo.

Redo

Reverse changes that were undone by the Undo button. The Redo button displays a counter showing the number of undone changes you can reverse. Pressing the Shift key while clicking Redo displays a list of actions you can reverse.

Copy

Copy the selected component .

Delete

Delete the selected component.

Import

Import a component from an XML file. Refer to "Importing a Component" for information on importing a component.

Export

Export a component from the current workspace or from all active workspaces. Refer to "Exporting a Component" for information on exporting a component.

Batch Import

Import multiple web parts, pages, and pageflows from a single XML file. Refer to "Performing Batch Imports" for information on performing batch imports.

Batch Export

Export multiple web parts, pages, and pageflows to a single XML file or a Zip file. Refer to "Performing Batch Exports" for information on performing batch exports.

Revert To

Return to the published content or system content version of the component. Refer to "Reverting Content" for information.

Run

Test the current page in your browser in the Portal framework. You must save a page before you can test it. Refer to "Testing a Page" for information.

Publish

Publish saved changes to the highest active workspace. Refer to "Publishing Content" for information. Refer to "Developing in Workspaces" for information on workspaces.

Generate

Generate the current page. Refer to "Generating a Page" for information.

Extract Web Part

Extract and open the selected web part currently on a page in a new tab. Refer to "Extracting a Web Part" for information on extracting web parts.

Reload Web Part

Reloads the selected web part currently on a page to the original state of the web part in the Components catalog. Refer to "Reloading a Web Part" for information on reloading web parts.

Catalog

Display or hide the Components catalog. Refer to "Components Catalog" for information on the Components catalog.

Toolbox

Display or hide the Toolbox pane. Refer to "Toolbox" for information on the Toolbox pane.

Properties

Display or hide the Properties pane. Refer to "Properties Pane" for information on the Properties pane.

Trace Viewer

Display or hide the Trace Viewer. Refer to "Running the Trace Log" for information on the Trace Viewer.

Reset Settings

Reset any changes made to the layout of the panes in the working area.
