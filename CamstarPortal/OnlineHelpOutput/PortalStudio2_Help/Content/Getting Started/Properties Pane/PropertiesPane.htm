Properties Pane

The Properties pane is used to view and edit the properties of Portal Studio components such as pages, web parts, and controls.

The Properties pane displays properties in the Sorted view by default. Refer to "Viewing Fields and Properties" for information on viewing properties.

Properties that contain a value display a blue check mark icon next to the property name.

A description window displaying information about the currently selected property appears beneath the Properties pane when a property is selected.

Showing the Properties Pane

You can click the Properties button on the global toolbar to display the Properties pane when the pane is hidden. Refer to "Portal Studio Global Toolbar" for information on toolbar buttons.
