Editing Modes

You can use the editing modes to add, update, or view pages and web parts. The modes appear in a drop-down list near the top of the working area.

This table defines the editing modes.

Mode

Select this mode to . . .

Controls

Add, move, or remove web parts and controls. This is the default mode.

Design

Modify the layout. Selecting this mode displays an Actions button containing options to merge or split cells, and insert or remove columns and rows. Refer to "Managing Layouts" for information.

View

Display the HTML rendering of the page.

Xml

Display or edit the XML code for the page.

Personalization Report

Display the personalization report which contains basic page information, data contracts, and information on the web parts and controls used on the page.

Where Used

Display all components that reference a web part.
