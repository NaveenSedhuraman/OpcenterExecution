Toolbox

The Toolbox provides access to the following tabs:

• Controls, which displays a list of controls that you can add to web parts. Refer to "Controls" for information.
• Fields Directory, which displays a list of services and parametric data fields that you can add to web parts. Refer to "Fields Directory" for information.

The Toolbox button on the Portal Studio toolbar displays the toolbox if hidden, and hides the toolbox if visible.

Relative Button

The Fields Directory tab contains the Relative button (uppercase R). The Relative button toggles between the relative and absolute settings for a field when adding a field to a web part. Refer to "Relative Field Expressions" for information.
