Controls

Controls enable you to customize web parts in Portal Studio and are available from the Controls tab of the Toolbox. Refer to "Toolbox" for information on viewing the Toolbox.

Controls can only be added to a web part and cannot be added directly to a page. Refer to "Adding a Control to a Web Part" for information on adding controls to a web part.

A pop-up warning message appears if you add a control that is planned for deprecation in an upcoming version of Portal Studio. The warning message includes when the control will be deprecated and alternate controls to use instead. A tool-tip with this information also appears on controls that will be deprecated.

This table describes the controls available from the Controls tab of the Toolbox.

Icon

Control

Description

3D Visualization

Displays 3D visualizations with the ability to rotate, zoom in and out, mark up the image, and save a snapshot.

Note: This control supports JT files.

A 3D File Viewer pop-up appears and displays your 3D image when using the 3D Visualization control and allows you to rotate and zoom in and out on your 3D image.

The 3D File Viewer pop-up contains a toolbar that enables you to manipulate your 3D image and save 2D snapshots. The toolbar enables you to:

• Add text
• Add shapes
• Draw free-form on the image
• Select text and shape colors from a color picker
• Set custom line weights for shape tools
• Reset your changes
• Save a 2D snapshot

You can also view a 3D image anywhere that a document set can be viewed. Refer to "Using 3D Visualization" for information on using the 3D Visualization control.

Boolean Drop-down

Boolean selector with configurable labels and options to clear the value.

Boolean Switch

Boolean radio button control with configurable labels.

Breadcrumb Control

Control that enables breadcrumb navigation on a page.

Button

Clickable control that can be presented as text or an image.

Chart Control

Places a customizable chart on a page or web part. Multiple chart controls can exist simultaneously on a single page.

You can modify the format and data of chart controls using the ChartMode, ShowLegend, SeriesSettings, SeriesValueColumn, TooltipColumn, ValueColumn, XAxis, and YAxis properties in the Properties pane. Chart controls update dynamically when you modify the data sources for these properties. Refer to "Control Properties" for information on chart control properties.

The SeriesSettings property enables you to further customize the appearance of data points on scatterplot and wafer map charts. Refer to "Adding SeriesSettings" for information on adding SeriesSettings properties.

You can create custom chart formats, including wafer maps, by modifying the d3.js file. Refer to http://d3js.org for information on the D3 document object model.

Note: You can zoom in and out on chart controls on Portal pages using the mouse wheel.

Check Box

Box that is selected or cleared to turn an option on or off.

Note: You must set the AutoPostBack property to "True" when adding check box controls to pages containing pop-up grids. Refer to "Control Properties" for information on modifying control properties.

Checklist

Control that displays as a checklist on a page.

Color Picker

Allows for the selection of a color pallete to set a color for a field. Refer to "Using the Color Picker" for information.

Container Drop-down

Container selector with container names only.

Container Picklist

Container selector with selection values displayed in a grid.

Data Collection

Control that allows user input of one or more values.

Date Chooser

Date selector with date entry option.

Discovery Area

Displays a custom Discovery Area control.

Drop-down List

Drop-down control that displays a list of selection values.

Duration

Control that specifies a period of time (DD.HH:MM:SS) and can work in conjunction with process time enforcement. Refer to the Opcenter Execution Medical Device and Diagnostics Portal Modeling User Guide or the Opcenter Execution Core Portal Modeling User Guide for information on defining process timers.

File Browse

Enables you to browse for and select a document to upload.

File Input

Enables you to browse for and select a file to use for data input.

Flyout Drop-down

Appears as a button that opens another page in a flyout panel.

Grid

Grid with a selectable data mode.

Note: Adding a grid control to a web part prompts you to either create a grid using the Grid Wizard or place a blank grid control that you can manually customize. Refer to "Defining Grids Using the Grid Wizard" for information.

HTML Panel

Enables the user to specify HTML content or a path to a template file.

Note: This control supports HTML content entered as text using the HTMLText property or a folder path to an HTML template file using the TemplateFile property. You must enter a value in either the HTMLText or TemplateFile properties when using this control. Refer to "Control Properties" for information on the HTMLText and TemplateFile properties.

IFrame

Places an inline HTML page inside another page.

Image Control

Displays an image on a page.

Inquiry Control

Displays a lightweight read only field on a page.

Instruction Text

Places a read only text box on a page.

Intelligence Object

Note: This control accesses and executes Business Objects reports using an HTTP connection by default. You can change the control setting in the Properties pane. Refer to "Control Properties" for information.

Intelligence Report Object

Opcenter Intelligence Report control. Refer to the Opcenter Intelligence Core Report Installation Guide for information.

Note: Refer to "Control Properties" for information.

Label

Link Button

ModelingObjectList

Named Object

Named Subentity

PI Metric

Radio Button

Radio Button List

Readonly Date

Readonly Enum

Revisioned Object

Ribbon Container

Scale

Tab (Client)

Task List Menu

Text Box

Control that appears as a single line text box without editing options.

Text Editor

Provides a rich text editor that allows you to enter and format text in a text box.

Tile Container

Control that enables you to add multiple tiles.

Title

Configures titles that appear on components on a page.

Toggle Container

Control that holds multiple web parts and preserves the layout of web parts placed on it. Optionally creates collapsible sections on Portal pages using the ToggleDisplayMode property. Refer to "Control Properties" for information.

User Fields

Control that displays user fields.

View Documents

Enables you to view uploaded documents on a page.

Web Part Container

Container control that holds and groups web parts on a page.

Web Parts Wrapper

Preserves the layout of web parts placed on it when merging content from one workspace to another.

Workflow Navigator

Enables you to transition along a workflow.

Workflow Viewer

Displays the overall workflow on a page.
