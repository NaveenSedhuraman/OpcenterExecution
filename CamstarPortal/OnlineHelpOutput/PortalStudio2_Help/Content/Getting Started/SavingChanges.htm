Saving Changes

Portal Studio enables you to save changes to an open component or multiple open components simultaneously.

When Saving Changes

An unsaved data indicator appears as an asterisk in the title of a component's tab in the working area when unsaved changes are present. A warning message appears if you attempt to close the tab without first saving your changes. You can close the component by clicking Yes on the warning message pop-up. Any unsaved changes are lost.

You can click the Save button on the toolbar to save the pending changes for the component selected in the working area. You can click the Save All button on the toolbar to save all open components' pending changes. Refer to "Portal Studio Global Toolbar Buttons" for information.

An error message appears if you attempt to save a component or collection property with the same name as an existing component or collection property. You must enter a unique name and then save your changes.

How to Save Your Changes

Follow these steps to save changes:

1. Access Portal Studio.
2. Create a new component or open an existing component.
3. Make any necessary changes.
4. Click Save. The application saves your changes and displays a success message.
