Adding Page Mapping Parameters to a Page Map

Page mapping parameters are used to configure conditions under which the application redirects you to a target page. Page mapping parameters serve as the mediator between source pages and target pages.

When Adding Page Mapping Parameters to a Page Map

A parameter should be unique to a page map. Parameters transfer data between pages and configure conditions when redirection occurs.

Parameters Modal Field Definitions

The Parameters modal contains only the Name field. The Name field is used to select an existing parameter and associate it with the page map.

How to Add Page Mapping Parameters to a Page Map

Follow these steps to add page mapping parameters to a map item:

1. Access Portal Studio.
2. Select the Modeling tab on the Components catalog and double-click a page map. The page map appears in the working area.
3. Click More for Parameters. The Parameters modal appears.
4. Click Add. A parameter is added to the list.
5. Select a parameter from the Name field.
6. Repeat steps 4-5 to add additional parameters.
7. Click OK. The Parameters modal closes.
8. Click Save. The application saves your changes and displays a success message.
