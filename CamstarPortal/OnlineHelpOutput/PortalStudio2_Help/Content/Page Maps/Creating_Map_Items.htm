Creating a Page Map

Page mapping uses fall-through pattern matching logic. Portal Studio evaluates the parameters you set to determine the best target page for an action.

How to Create a Page Map

Follow these steps to create a map item:

1. Access Portal Studio.
2. Click New and select Map Item. A blank page map appears in the working area.
3. Enter a name for the page map in the Name field.
4. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
5. Click Save. The application saves your changes and displays a success message.
