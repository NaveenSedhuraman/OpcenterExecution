Exporting a Component

Portal Studio allows you to export pages, web parts, and pageflows. Exporting a component enables you to save a copy of that component locally .

When Exporting a Component

You can export a component from the workspace currently selected in the View Contents list or from all active workspaces.

Exported components are saved as XML files. The XML file uses the name of the exported component by default. The application prompts you to open or save the XML file, or it automatically downloads the file. Your Internet browser determines whether the application displays a pop-up, its appearance, and the wording of the Open and Save options.

How to Export a Component

Follow these steps to export a component:

1. Access Portal Studio.
2. Open the component you want to export from the Components catalog.
3. Do you want to export the component data for the currently selected workspace or all workspaces?

If you want to . . .

Then . . .

Export only the currently selected workspace data
a. Click the Export button.
b. Select Selected Workspace.
c. Go to step 4.

Export the component data for all workspaces
a. Click the Export button.
b. Select All Workspaces.
c. Go to step 4.
4. Open or save the XML file. Your Internet browser determines whether the application displays a pop-up, its appearance, and the wording of the Open and Save options.
