Performing Batch Exports

Batch export allows you to export multiple pages, web parts, and pageflows as a single XML file or a Zip file containing multiple XML files.

When Performing Batch Exports

Saving a batch export as a Zip file creates a Zip file containing an XML file for each page, web part, and pageflow included in the batch export.

You must select at least one workspace from the Personalization Type to Export list.

An Export Progress... pop-up appears when you click the Start button. The pop-up disappears when the export is compete.

The default file name for the XML or Zip file is BatchExport.xml or BatchExport.zip. The application prompts you to open or save the file, or it automatically downloads the file. Your Internet browser determines whether the application displays a pop-up, its appearance, and the wording of the Open and Save options.

Important: Batch export does not automatically find and export dependent pages. For example, when you are exporting a page that has dependent pages referenced in a page redirect action or a flyout control, the dependent pages are not found and exported. You must manually select the dependent pages as part of the export for the page to function correctly when imported.

Batch Export Window

The Batch Export window's left pane shows all pages, web parts, and pageflows available for batch export divided into tabs. Components selected in one tab remain selected when another tab is brought into focus.

How to Perform Batch Exports

Follow these steps to perform a batch export:

1. Access Portal Studio.
2. Click the Batch Export button. The Batch Export window appears.
3. Select items to export from the Pages, Web Parts, and Pageflows tabs.
4. Select at least one workspace from the Personalization Type to Export list.
5. Select the Export To File check box.
6. Select XML or Zip for the file format.
7. Click Start. The Export Progress... pop-up appears. The Export Progress... pop-up closes, the application displays a success message when the export process is complete, and your browser prompts you to open or save the file. Your Internet browser determines whether the application displays a pop-up, its appearance, and the wording of the Open and Save options.
