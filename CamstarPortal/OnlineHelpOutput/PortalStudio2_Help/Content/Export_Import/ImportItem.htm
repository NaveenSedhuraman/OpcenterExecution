Importing a Component

You can import XML files in Portal Studio to update or modify existing components.

When Importing a Component

You can import an XML file only to pages, web parts, and pageflows that are in the highest active workspace and currently selected in the working area.

Once you import a component you can then save the component with an updated name using the Name field in the Properties pane.

Note: The size of an import is limited to 30 Megabytes (MB) by default. If your bundle is larger than this, you can increase the value for the MaxAttachmentSizeLimit setting in the Portal Settings. Refer to "Configuring Portal Settings" for information.

How to Import a Component

Follow these steps to import a component:

1. Access Portal Studio.
2. Create or open an existing page, web part, or pageflow.
3. Click the Import button. A Choose File to Upload pop-up appears.
4. Locate and select the XML file you want to import.
5. Click Open. The Choose File to Upload pop-up closes and the XML file is imported into the open component.
6. Change the name of the component in the Name field in the Properties pane, if necessary.
7. Click Save. The application saves your changes and displays a success message.
