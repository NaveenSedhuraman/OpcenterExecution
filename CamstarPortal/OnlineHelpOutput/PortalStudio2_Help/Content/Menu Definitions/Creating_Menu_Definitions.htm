Creating a Menu

You can create a menu structure consisting of multiple menus and submenus.

When Creating a Menu

Menus appear as a set of fields within the working area when creating a new menu.

Consider the following when creating a menu:

• The csiPortalMenuV8, csiMobileMenu,and csiPortalMenu default menus are assigned to the Administrator, CamstarAdmin, and InSite Admin users automatically in an initial implementation of Opcenter EX CR and Opcenter EX MDD, or when updating to a new version and upgrading the default instance data. You can use the Employee modeling object to assign Portal menus to other employees. Refer to the Opcenter Execution Medical Device and Diagnostics Portal Modeling User Guide or the Opcenter Execution Core Portal Modeling User Guide for information.
• Siemens recommends you create tabs before you create menus as menus reference tabs. Refer to "Creating a Tab" for information.

Tabs can be referenced three ways by menus. This table describes the ways tabs can be referenced.

Tab Reference

Description

Current Tab

Inherits the tab setting from the current tab.

New Tab

Always opens in a new blank tab in the Portal. Selecting this option allows you to have multiple copies of a tab open at the same time.

Specified Tab

Specifies a tab that was previously created. Only one version of a specified tab can be open at a time.

If you set a menu or a submenu to a specified tab, and leave all of the items below with the default values of inherit, they will all open in the same specified tab. The latest page opened will replace the page currently open in the specified tab.

Refer to "How to Add a Specified Tab to a Menu Definition" for information.

Opening Existing Menus

Existing menus can be opened from the Components catalog. Refer to “Opening Existing Components” for information on opening existing menus.

How to Create a Menu Definition

Follow these steps to create a menu definition:

1. Access Portal Studio.
2. Click New and select Menu Definition. A blank menu appears in the working area.
3. Enter a name for the menu in the Name field.
4. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
5. Click Save. The application saves your changes and displays a success message.

How to Add a Specified Tab to a Menu Definition

Follow these steps to add a specified tab to a menu definition:

1. Access Portal Studio.
2. Select the Navigation tab on the Components catalog and then double-click a menu. The menu appears in the working area.
3. Select SpecifiedTab from the PortalTabOption field.
4. Select a tab from the PortalTab field.
5. Click Save. The application saves your changes and displays a success message.
