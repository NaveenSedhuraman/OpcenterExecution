Adding Menu Items to a Menu

Menu items are the pages you can access from the menu. Menu items are defined in the MenuItems modal. You can also optionally add submenus and pageflows from the MenuItems modal.

When Adding Menu Items to a Menu

You can associate submenus with a menu after you have defined the Portal main menu. Any defined menu can be added as a submenu. Adding a submenu enables you to add a group of transactions rather than adding each transaction individually. Refer to "How to Add a Submenu Icon" for information.

You can modify submenu icons by entering an icon file name in the ApolloIcon field on the MenuItems modal. Icon files are located in the following directory: C:\Program Files (x86)\Camstar\Camstar Portal\assets\image. You must enter the file name as it appears in the folder.

How to Add Menu Items to a Menu Definition

Follow these steps to add menu items to a menu definition:

1. Access Portal Studio.
2. Select the Navigation tab on the Components catalog and then double-click a menu. The menu appears in the working area.
3. Click More for MenuItems. The MenuItems modal appears.
4. Select PortalMenuCommandChanges from the drop-down and click Add. A new menu item appears in the list and fields appear in the right pane.
5. Enter a caption for the menu item in the Caption field.
6. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
7. Repeat steps 4-6 for each menu item you want to add.
8. Click OK. The MenuItems modal closes.
9. Click Save. The application saves your changes and displays a success message.

How to Add Submenus to a Menu Definition

Follow these steps to add subs menu to a menu definition:

1. Access Portal Studio.
2. Select the Navigation tab on the Components catalog and then double-click a menu. The menu appears in the working area.
3. Click More for MenuItems. The MenuItems modal appears.
4. Select PortalMenuSubMenuChanges from the drop-down and click Add. A new submenu appears in the list and fields appear in the right pane.
5. Enter a caption for the submenu in the Caption field.
6. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
7. Repeat steps 4-6 for each submenu you want to add.
8. Click OK.The MenuItems modal closes.
9. Click Save. The application saves your changes and displays a success message.

How to Add a Submenu Icon

Follow these steps to add a submenu icon:

1. Access Portal Studio.
2. Select the Navigation tab on the Components catalog and then double-click a menu. The menu appears in the working area.
3. Click More for MenuItems. The MenuItems modal appears.
4. Select PortalMenuSubMenuChanges from the drop-down and select a submenu. The submenu's fields appear in the right pane.
5. Enter an icon file name in the ApolloIcon field that corresponds with the icon image file in the C:\Program Files (x86)\Camstar\Camstar Portal\assets\image directory.
6. Click OK.The MenuItems modal closes.
7. Click Save. The application saves your changes and displays a success message.

How to Add Pageflows to a Menu Definition

Follow these steps to add pageflows to a menu definition:

1. Access Portal Studio.
2. Select the Navigation tab on the Components catalog and then double-click a menu. The menu appears in the working area.
3. Click More for MenuItems. The MenuItems modal appears.
4. Select PortalMenuCommandChanges from the drop-down and click Add. A new menu item appears in the list and fields appear in the right pane.
5. Enter a caption for the menu item in the Caption field.
6. Select a pageflow from the Pageflow field.
7. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
8. Repeat steps 4-7 for each pageflow you want to add.
9. Click OK. The MenuItems modal closes.
10. Click Save. The application saves your changes and displays a success message.
