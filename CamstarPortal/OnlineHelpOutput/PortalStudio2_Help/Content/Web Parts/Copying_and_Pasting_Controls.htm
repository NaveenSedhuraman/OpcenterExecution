Copying Controls

You can copy controls in the working area of Portal Studio.

When Copying Controls

The copied control is assigned a unique name.

How to Copy Controls

Follow these steps to copy controls:

1. Access Portal Studio.
2. Double-click a page or web part. The page or web part appears in the working area.
3. Select a control in the working area.
4. Press the CTRL key and drag the selected control to a cell in a web part.
5. Click Save. The application saves your changes.
