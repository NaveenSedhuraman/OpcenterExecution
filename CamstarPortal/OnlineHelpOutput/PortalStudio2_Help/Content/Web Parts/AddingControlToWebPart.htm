Adding a Control to a Web Part

Controls incorporate the standard Web controls with Opcenter Execution Core properties added to the control.

Controls that display fields or data may require field expressions. Controls that are for layout purposes such as tabs, expanding sections, or fly-outs typically do not require field expressions.

When Adding a Control to a Web Part

Controls can only be added to a web part. Controls cannot be added directly to a page.

Some controls, such as the Tab or Flyout controls, do not interact with a service and do not require field expressions.

You must select a control that is compatible with the field expression you want to use. Refer to "Adding a Field Expression to a Control" for information.

How to Add a Control to a Web Part

Follow these steps to add a control to a web part:

1. Access Portal Studio.
2. Double-click a web part or a page containing a web part. The web part or page appears in the working area.
3. Click and drag a control from the Toolbox onto a cell on a web part. The control appears in the web part.
4. Enter a name for the control in the Name field in the Properties pane.
5. Click Save. The application saves your changes and displays a success message.
