Configuring Boolean Drop-down Labels

You can set custom labels for the true and false values in Boolean drop-down controls by modifying the control's properties in Portal Studio.

When Configuring Boolean Drop-down Labels

True and false labels must be entered in the format X_CustomLabel, where X is a Designer or User Label in the system and CustomLabel is your custom label name. Labels will read as True or False by default if they are not configured.

True and false labels are configured independently of one another. Configuring true and false labels is unique to the control being modified and does not affect other control labels.

Boolean Drop-down appears in the list of available controls in the Toolbox pane. Refer to "Adding a Control to a Web Part" for information on adding controls to a web part.

How to Configure Boolean Drop-down Labels

Follow these steps to configure Boolean drop-down labels:

1. Access Portal Studio.
2. Double-click a page or web part that contains a Boolean drop-down control. The page or web part appears in the working area.
3. Select a Boolean Drop-down control. The settings for the Boolean drop-down control appear in the Properties pane.
4. Enter the new false label in the FalseLabel field in the Properties pane.
5. Enter the new true label in the TrueLabel field in the Properties pane.
6. Click Save. The application saves your changes and displays a success message.
