Adding a Field to a Web Part

Fields are used to enter information and are contained in a web part. For example, you add a Container field to a web part to allow a user to enter a container name.

When Adding a Field to a Web Part

Field expression and control type are set automatically when adding a field to a web part.

Fields can only be added to a web part and cannot be added directly to a page.

Note: You can only add a field to a web part when you are in Controls view. You cannot drag a field onto a cell on the page if you are in Design view.

How to Add a Field to a Web Part

Follow these steps to add a field to a web part:

1. Access Portal Studio.
2. Double-click a web part or a page containing a web part. The web part or page appears in the working area.
3. Ensure that Controls is selected in the View list.
4. Click the Fields Directory tab on the Toolbox. A list of services appears in the Toolbox pane.
5. Expand the service that contains the field you want to add.
6. Select a field and drag it onto a cell on the web part.
7. Repeat steps 6-7 to add additional fields to the web part.
8. Click Save. The application saves your changes and displays a success message.
