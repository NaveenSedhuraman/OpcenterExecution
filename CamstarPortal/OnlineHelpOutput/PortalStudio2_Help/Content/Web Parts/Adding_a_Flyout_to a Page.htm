Adding a Flyout to a Page

You can add a Section Drop Down control to a page that acts as a container for web parts that appear as a flyout panel on a Portal page.

When Adding a Flyout to a Page

Controls can only be added to a web part. Controls cannot be added directly to a page. Refer to "Adding a Control to a Web Part" for information on adding controls to a web part.

The AltOpenLabelName and AltOpenLabelText properties determine the text that appears on the label of the flyout when open. The AltCloseLabelName and AltCloseLabelText properties determine the text that appears on the label of the flyout when closed.

The same text will appear on the label of the flyout when open or closed if only one property is defined. No text will appear on the label of the flyout if none of these properties are defined. Refer to "Control Properties" for information on control properties.

Text entered in the AltOpenLabelText field overrides the dictionary label selected in the AltOpenLabelName field when both text and labels are specified. The text entered in the AltCloseLabelText field overrides the dictionary label selected in the AlltCloseLabelName field when both text and labels are specified.

How to Add a Flyout to a Page

Follow these steps to add a flyout to a page:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the Components Catalog is hidden. The Components Catalog appears.
3. Double-click a page or web part. The page or web part appears in the working area.
4. Click Show/Hide Toolbox if the Toolbox is hidden. The Toolbox appears.
5. Click and drag a Section Drop Down control onto a cell on a web part. The Section Drop Down control appears in the web part.
6. Click Show/Hide Properties if the Properties pane is hidden. The Properties pane appears.
7. Enter a name for the control in the Name field in the Properties pane.
8. Enter text or select a dictionary label in the AltOpenLabelName, AltOpenLabelText, AltCloseLabelName, or AltCloseLabelText field.
9. Repeat step 8 to add additional label text as needed.
10. Click and drag web parts onto the Section Drop Down control as needed.
11. Click Save. The application saves your changes and displays a success message.
