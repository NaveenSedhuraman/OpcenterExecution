Relative Field Expressions

Relative field expressions can be picked up by different subclass services. Field expressions are relative by default and can be set to absolute in Portal Studio.

When Using Relative Field Expressions

Field expressions are set as either relative or absolute using the Relative button in the Fields Directory catalog in the Toolbox. Field expressions are set to relative when the Relative button has a red R.

Field expressions set to relative do not contain the service in the FieldExpressions property, allowing them to be picked up by other services. For example, the field expression Container for the service MoveIn has a FieldExpression property of MoveIn.Container when absolute. The same field expression has the property .Container when relative.

How to Set Field Expressions as Absolute

Follow these steps to set field expressions as absolute:

1. Access Portal Studio.
2. Click Show/Hide Toolbox if the Toolbox is hidden. The Toolbox appears.
3. Select the Fields Directory tab. The Fields Directory catalog appears in the Toolbox pane.
4. Click the Relative button. The Relative button is is red.
5. Add field expressions as needed. Refer to "Adding a Field Expression to a Control" for information on adding field expressions to web parts.
6. Click Save. The application saves your changes and displays a success message.
