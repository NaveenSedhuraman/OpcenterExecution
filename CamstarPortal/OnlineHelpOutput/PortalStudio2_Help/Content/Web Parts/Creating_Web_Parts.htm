Creating a Web Part

You can create web parts in Portal Studio which contain controls and fields on pages.

When Creating a Web Part

Fields and controls can be dragged directly onto a web part, however, they cannot be dragged directly onto a page.

You can test a web part after it is saved to a page by clicking the Run button. Refer to Testing a Page for information on testing web parts and pages.

Siemens recommends that you enter at least a name and title in the Properties pane when creating a web part. The Name field defines the name of the web part. The Title property defines the text that appears as the title of the web part. Refer to "Web Part Properties" for information on web part properties.

Opening Existing Web Parts

Existing web parts can be opened from the Components Catalog. Refer to "Opening Existing Components" for information on opening existing web parts.

Reloading a Web part

You can reload a web part by clicking the Reload Web Part button on the Portal Studio Global Toolbar. Reloading a web part reverts it to its previously saved state. You can only reload web parts when they are part of a page.

How to Create a Web Part

Follow these steps to create a web part:

1. Access Portal Studio.
2. Click New and select Web Part. A blank web part appears in the working area.
3. Enter a name for the web part in the Name field in the Properties pane.
4. Enter a label in the Title property in the Properties pane.
5. Click Save. The application saves your changes and displays a success message.
