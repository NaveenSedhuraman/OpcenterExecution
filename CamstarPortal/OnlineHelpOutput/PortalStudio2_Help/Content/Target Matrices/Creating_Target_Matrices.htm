Creating a Target Matrix

Objects can be added to a target matrix to set a best pattern match configuration matrix of Target and Benchmark values for a configurable list of object parameters.

When Creating a Target Matrix

Target matrices provide the option for an instance to resolve a target or benchmark from a standard method or through an optional target matrix where it would resolve the target and benchmark from the best match given the object fields that are resolved, such as an operation or resource.

Lines in the matrix can be added, edited, deleted, and viewed with values for any or all of the object parameters and also for the Target and Benchmark.

Opening Existing Target Matrices

Existing target matrices can be opened from the Modeling tab of the Components Catalog. Refer to “Opening Existing Components” for information on opening existing target matrices.

Optional

How to Create a Target Matrix

Follow these steps to create a target matrix:

1. Access Portal Studio.
2. Click New and select Target Matrix Item. A blank target matrix appears in the working area.
3. Enter a name for the target matrix in the Name field.
4. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
5. Click Save. The application saves your changes and displays a success message.
