Adding Rows and Columns to a Target Matrix

Portal Studio enables you to add rows and columns to a target matrix.

When Adding Rows and Columns to a Target Matrix

All target matrices share columns. Adding a column to one target matrix adds the column to all target matrices.

You can import data from an Excel spreadsheet when using the Rows modal. You can use a combination of imported data and added data.

How to Add Rows to a Target Matrix

Follow these steps to add rows to a target matrix:

1. Access Portal Studio.
2. Select the Modeling tab on the Components catalog and double-click a target matrix. The target matrix appears in the working area.
3. Click More for Rows. The Rows modal appears.
4. Are you adding rows or importing data from an Excel spreadsheet?

If you are . . .

Then . . .

Adding rows
a. Click Add Row. A new row appears in the Rows modal.
b. Enter a value in each cell.
c. Repeat steps a-b to add additional rows.
d. Go to step 5.

Importing data from an Excel spreadsheet
a. Click Import from Excel. The imported data appears in the Rows modal.
b. Go to step 5.
5. Click OK. The Rows modal closes.
6. Click Save. The application saves your changes and displays a success message.

How to Add Columns to a Target Matrix

Follow these steps to add columns to a target matrix:

1. Access Portal Studio.
2. Select the Modeling tab on the Components catalog and double-click a target matrix. The target matrix appears in the working area.
3. Click More for Columns. The Columns modal appears.
4. Click Add. A new column appears in the list.
5. Enter a name for the column in the Name field.
6. Enter a caption for the column in the Caption field.
7. Repeat steps 4-6 to add additional columns.
8. Click OK. The Columns modal closes.
9. Click Save. The application saves your changes and displays a success message.
