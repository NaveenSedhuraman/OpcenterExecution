导出组件

可通过 Portal Studio 导出页面、Web 部件和页面流。通过导出组件，可将该组件的副本保存到本地。

在导出组件时

可从“查看内容”列表中当前选择的工作区或所有活动工作区导出组件。

导出的组件将另存为 XML 文件。XML 文件默认以导出的组件的名称命名。应用程序会提示打开或保存该 XML 文件，或者它会自动下载该 XML 文件。应用程序是否显示弹出窗口、该弹出窗口的外观以及“打开”和“保存”选项的描述由 Internet 浏览器决定。

如何导出组件

按照以下步骤导出组件：

1. 访问 Portal Studio。
2. 从“组件”目录中打开要导出的组件。
3. 是否要导出当前所选工作区或所有工作区的组件数据？

如果您想完成以下操作 ...

则...

仅导出当前选定工作区的数据
a. 单击导出按钮。
b. 选择所选工作区。
c. 转到步骤 4。

导出所有工作区的组件数据
a. 单击导出按钮。
b. 选择所有工作区。
c. 转到步骤 4。
4. 打开或保存 XML 文件。应用程序是否显示弹出窗口、该弹出窗口的外观以及“打开”和“保存”选项的描述由 Internet 浏览器决定。
