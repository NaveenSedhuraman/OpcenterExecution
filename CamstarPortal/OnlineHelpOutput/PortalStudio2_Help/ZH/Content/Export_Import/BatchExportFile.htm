执行批量导出

通过批量导入，可以将多个页面、Web 部件和页面流导出为单个 XML 文件，或导出为包含多个 XML 文件的 Zip 文件。

在执行批量导出时

将批量导出另存为 Zip 文件可为批量导出中包含的每个页面、Web 部件和页面流创建一个包含 XML 文件的 Zip 文件。

必须至少从“要导出的个性化类型”列表中选择一个工作区。

单击“开始”按钮时，将显示“导出进度…”弹出窗口。导出完成后，该弹出窗口会消失。

XML 文件和 Zip 文件的默认文件名分别为 BatchExport.xml 和 BatchExport.zip。应用程序会提示打开或保存该文件，或者它会自动下载该文件。应用程序是否显示弹出窗口、该弹出窗口的外观以及“打开”和“保存”选项的描述由 Internet 浏览器决定。

重要提示: 批量导出不会自动查找和导出从属页面。例如，如果要导出的页面在页面重定向操作或浮出控件中引用了从属页面，则不会查找和导出从属页面。必须手动选择从属页面作为页面导出的一部分，才能在导入时正常运行。

批量导出窗口

“批量导出”窗口的左窗格显示可供批量导出的所有页面、Web 部件和页面流，并将其分为选项卡。在一个选项卡中选择的组件会在另一个选项卡位于焦点时保持选定状态。

如何执行批量导出

按照以下步骤执行批量导出：

1. 访问 Portal Studio。
2. 单击批量导出按钮。将显示“批量导出”窗口。
3. 从“页面”、“Web 部件”和“页面流”选项卡中选择要导出的项。
4. 从要导出的个性化类型列表中选择至少一个工作区。
5. 选中导出到文件复选框。
6. 为文件格式选择 XML 或 Zip。
7. 单击启动。将显示“导出进度…”弹出窗口。导出过程完成后，“导出进度…”弹出窗口会关闭，应用程序将显示成功消息，并且浏览器会提示您打开或保存文件。应用程序是否显示弹出窗口、该弹出窗口的外观以及“打开”和“保存”选项的描述由 Internet 浏览器决定。
