导入组件

可以将 XML 文件导入至 Portal Studio，以更新或修改现有组件。

在导入组件时

只能将 XML 文件导入至位于最高活动工作区中且当前在工作区域中已选定的页面、Web 部件和页面流。

导入组件后，可以在“属性”窗格的“名称”字段中使用已更新的名称来保存该组件。

注释: 默认情况下，导入项的大小限制为不超过 30 MB。如果捆绑包超出了该限制值，可以在“门户设置”中增加 MaxAttachmentSizeLimit 设置的值。有关信息，请参考“配置门户设置”。

如何导入组件

按照以下步骤导入组件：

1. 访问 Portal Studio。
2. 创建或打开页面、Web 部件或页面流。
3. 单击导入按钮。将显示“选择要上传的文件”弹出窗口。
4. 找到并选择要导入的 XML 文件。
5. 单击打开。“选择要上传的文件”弹出窗口将关闭，并且 XML 文件会导入至打开的组件。
6. 如果需要，在“属性”窗格的“名称”字段中更改该组件的名称。
7. 单击保存。应用程序将保存更改并显示成功消息。
