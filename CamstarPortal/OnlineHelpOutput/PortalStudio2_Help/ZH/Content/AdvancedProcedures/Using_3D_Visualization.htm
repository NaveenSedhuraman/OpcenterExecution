使用 3D 可视化

通过 3D 可视化控件，用户可以在应用程序中查看 3D 图像。图像将显示在单独的“3D 文件查看器”弹出窗口中，允许用户对图像执行以下操作：

• 旋转
• 放大和缩小
• 标记
• 保存快照
在使用 3D 可视化时

3D 可视化控件支持 JT 文件。3D 图像文件会在“3D 文件查看器”弹出窗口中打开。可以使用 3D 可视化控件或可以查看文档集的任何位置来查看 3D 图像。

“3D 文件查看器”弹出窗口包含两种模式：

• 3D，其允许在 3D 图像上自由旋转，以及放大和缩小。
• 标记 (2D)，其允许标记和保存 2D 静止图像。

首次打开 3D 图像文件后，“3D 文件查看器”弹出窗口将以 3D 模式显示图像。可以在 3D 模式下单击并拖动图像以将其自由旋转。向上滚动鼠标滚轮可放大图像，向下滚动可缩小图像。切换到标记 (2D) 模式可创建 3D 图像（从“3D 文件查看器”弹出窗口中的当前视角观察）的 2D 静止图。

标记 (2D) 模式包含工具栏，允许使用可自定义的线宽和颜色添加文本和形状。有关工具栏选项和可用于标记 2D 图像的工具的信息，请参考“3D 文件查看器弹出窗口工具栏选项”。

可以单击“保存”按钮来拍摄 2D 图像的快照。单击“保存”按钮将在浏览器中显示“保存”对话框，允许将快照保存为本地机器上的图像文件。只要存在未保存的更改，图像周围就会显示红色边框。

可以单击“重置”按钮移除任何标记或批注。当存在未保存的更改时，单击“重置”按钮将显示警告消息。在警告消息上单击“是”可移除更改。在警告消息上单击“否”可保留更改。

可以单击“返回 3D”以切换回 3D 模式。切换回 3D 模式将移除您添加的任何标记或批注。如果 2D 图像上存在未保存的更改，单击“返回 3D”时将显示警告消息。在警告消息上单击“是”可在浏览器中显示“保存”对话框，该对话框允许在本地机器上保存快照，然后切换回 3D 模式。单击“否”可放弃更改并切换回 3D 模式。

3D 模式

标记 (2D)

标记 (2D) 模式

返回 3D

切换回 3D 模式。

添加注释

向图像添加文本。单击“批注”后，鼠标光标将更改为文本工具。使用文本工具单击图像可以在图像上设置文本。

形状

选择工具并添加直线或自由形式线、正方形或圆形。

颜色

选择或输入形状和文本的自定义颜色值。默认情况下设为黑色。有关使用颜色选取器设置自定义颜色的信息，请参考“使用颜色选取器”。

线宽

设置形状工具的线宽。默认值为 3。
