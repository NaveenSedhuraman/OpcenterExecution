向页面添加 Web 部件

Web 部件可以作为本地副本或引用添加到页面中。

在向页面添加 Web 部件时

引用的 Web 部件是只读副本，它们引用 Web 部件的主副本。通过这些部件，可在一个位置维护一个 Web 部件，而不是维护多个单独的 Web 部件。

Siemens 建议作为以下项添加 Web 部件：

• 参考（当 Web 部件在所有页面上相同并在一个位置维护时）。有关查看引用 Web 部件的组件列表的信息，请参考“查看引用的 Web 部件位置”。
• 副本（当 Web 部件是模板或使用一个 Web 部件创建类似的 Web 部件时）。

测试带有参考 Web 部件的页面时，将始终显示参考 Web 部件的开发者内容。

发布带有参考 Web 部件的页面时，将始终显示参考 Web 部件的已发布内容。

注释: 只有在位于“控件”视图中时，才能向页面添加 Web 部件。如果位于“设计”视图中，则无法将 Web 部件拖动到页面上的单元格中。

如何向页面添加 Web 部件

按照以下步骤将 Web 部件添加到页面：

1. 访问 Portal Studio。
2. 双击页面。 随即工作区域中将显示该页面。
3. 确保在“视图”列表中选择控件。
4. 在“组件”目录中展开 Web 部件。
5. 选择一个 Web 部件，然后将其拖放到页面上的单元格中。将显示“添加 Web 部件”弹出窗口。
