查看页面上的 Web 部件

Portal Studio 允许查看页面上使用的 Web 部件。可以查看页面上使用的动态 Web 部件和静态 Web 部件。

在查看页面上的 Web 部件时

在 DynamicWebParts 模式窗口中选择 Web 部件时，其属性将显示在右窗格中。然后可以使用右窗格进一步向下查找。

注释: 无法从 DynamicWebParts 模式窗口中修改 Web 部件属性。

如何查看页面上的 Web 部件

按照以下步骤查看页面上的 Web 部件：

1. 访问 Portal Studio。
2. 双击页面。 随即工作区域中将显示该页面。
3. 在“属性”窗格中单击 DynamicWebParts 对应的更多。将显示 DynamicWebParts 模式窗口，其中显示了当前位于页面上的 Web 部件的列表。
4. 从左窗格中的列表中选择 Web 部件。Web 部件的属性显示在右窗格中。
