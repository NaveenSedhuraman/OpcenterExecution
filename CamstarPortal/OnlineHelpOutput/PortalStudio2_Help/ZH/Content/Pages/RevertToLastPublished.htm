恢复内容

“恢复为”按钮可用于将最高活动工作区中的页面和 Web 部件回滚到其上次发布的状态或在基础 Siemens PLM CEP (CSI) 工作区中找到的系统内容状态。

在恢复页面和 Web 部件内容时

“恢复为”按钮在基础 Siemens PLM CEP (CSI) 工作区中禁用，仅在更改了最高活动工作区的页面和 Web 部件上启用。有关在工作区中保存和发布更改的信息，请参考“在工作区中进行开发”。

单击“恢复为”并选择“已发布内容”会将页面或 Web 部件恢复为上次发布的状态。单击“恢复为”并选择“系统内容”时，会将页面或 Web 部件恢复为与在基础 Siemens PLM CEP (CSI) 工作区中找到的组件相匹配。

如何将页面和 Web 部件恢复为上次发布的状态

按照以下步骤将页面和 Web 部件恢复为上次发布的状态：

1. 访问 Portal Studio。
2. 打开具有已发布的更改的页面或 Web 部件。随即工作区域中将显示该页面或 Web 部件。
3. 单击还原到并选择已发布内容。将显示“还原到上次发布”对话框。
4. 单击确定。 “恢复为上次发布”对话框将关闭，页面或 Web 部件会刷新以显示上次发布的状态的内容。

如何将页面和 Web 部件恢复为系统内容

按照以下步骤将页面和 Web 部件恢复为系统内容：

1. 访问 Portal Studio。
2. 打开已修改的页面或 Web 部件。随即工作区域中将显示该页面或 Web 部件。
3. 单击还原到... 并选择系统内容。将显示“还原到系统内容”对话框。
4. 单击确定。 “恢复为系统内容”对话框将关闭，页面或 Web 部件会刷新以显示在基础 Siemens PLM CEP (CSI) 工作区中找到的内容。
