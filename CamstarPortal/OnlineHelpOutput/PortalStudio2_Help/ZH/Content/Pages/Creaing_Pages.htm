创建页面

通过 Portal Studio 可以创建在 Portal 中使用的页面。

在创建页面时

可以通过单击全局工具栏上的“新建”并选择“页面”来创建页面，也可以通过单击“组件”目录的“页面”选项卡中的“新建”按钮来创建。创建页面时，将显示“新建页面”弹出窗口，您可以在其中为页面选择模板。“新建页面”弹出窗口包含以下模板：

• “空白”- 为基本门户页面提供空白模板。
• “容器”- 提供基本组件用于开发容器页面。
• “建模”- 提供基本组件用于开发建模页面。

必须在“页面名称”字段中输入页面的名称，以在“新建页面”弹出窗口中启用“确定”按钮。

新建页面后，可以将 Web 部件拖到该页面上。不能将控件和字段直接拖动到页面上；它们必须放在 Web 部件上。有关向页面添加 Web 部件的信息，请参考“向页面添加 Web 部件”。

使用“运行”按钮保存页面后，可对其进行测试。有关信息，请参考“测试页面”。

默认情况下，“WIP 消息”在“门户设置”中设为 True。由于页面设置会替代全局设置，因此可以使用“属性”窗格中的 WIPMessages 字段必需 来禁用页面的“WIP 消息”。

打开现有页面

可以从“组件”目录中打开现有页面。有关打开现有页面的信息，请参考“打开现有组件”。

如何创建页面

按照以下步骤创建页面：

1. 访问 Portal Studio。
2. 单击新建按钮并选择页面。将显示“新建页面”弹出窗口。
3. 选择页面模板。
4. 在页面名称字段中输入页面的名称。
5. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
6. 单击确定。“新建页面”弹出窗口关闭，且工作区域中将显示新页面。
7. 单击保存。应用程序将保存更改并显示成功消息。
