提取 Web 部件

可通过 Portal Studio 在页面上选择 Web 部件，然后将该 Web 部件提取至工作区域中的新选项卡。

在提取 Web 部件时

在页面上选择 Web 部件后，“提取 Web 部件”按钮处于启用状态。

保存后，提取的 Web 部件将在“组件”目录中显示为新的 Web 部件。

如何提取 Web 部件

按照以下步骤提取 Web 部件：

1. 访问 Portal Studio。
2. 双击页面。 随即工作区域中将显示该页面。
3. 在页面上选择 Web 部件，然后单击提取 Web 部件。随即该 Web 部件将显示在工作区域中的新选项卡中。
4. 如果需要，在“属性”窗格的“名称”字段中更改该 Web 部件的名称。
5. 单击保存。应用程序将保存更改并显示成功消息。
