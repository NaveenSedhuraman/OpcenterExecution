创建页面映射

页面映射使用跌落模式匹配逻辑。Portal Studio 将计算所设置的参数，以确定某个操作的最佳目标页面。

页面映射字段定义

此表定义用于创建页面映射的字段。

字段

定义

类型

Name

页面映射的唯一名称。

必需

Parameters

可用于为页面映射定义参数。

注：单击“更多”按钮可显示 Parameters 模式窗口。有关此模式窗口中显示的属性的信息，请参考“Parameters 模式窗口字段定义”。

可选

SourcePages

可用于将源页面与页面映射相关联。

注：单击“更多”按钮可显示 SourcePages 模式窗口。有关此模式窗口中显示的属性的信息，请参考“SourcePages 模式窗口字段定义”。

可选

Targets

可用于为页面映射定义目标。

注：单击“更多”按钮可显示 Targets 模式窗口。有关此模式窗口中显示的属性的信息，请参考“Targets 模式窗口字段定义”。

可选
SourcePages 模式窗口字段定义

此表定义 SourcePages 模式窗口中的字段。

注释: 此模式窗口与 SourcePages 模式窗口不同，后者用于定义页面操作。在定义页面映射时显示的 SourcePages 模式窗口用于将源页面与页面映射相关联。有关在定义页面操作时显示的 SourcePages 模式窗口的信息，请参考“向页面操作添加源页面”。

定义

可选

如何创建页面映射

按照以下步骤创建映射项：

1. 访问 Portal Studio。
2. 单击新建，然后选择映射项。随即工作区域中将显示一个空白页面映射。
3. 在名称字段中输入页面映射的名称。
4. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
5. 单击保存。应用程序将保存更改并显示成功消息。
