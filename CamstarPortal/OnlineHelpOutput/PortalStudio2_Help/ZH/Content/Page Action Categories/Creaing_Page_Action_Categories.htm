创建页面操作类别

使用页面操作类别可以对操作进行分组，以在门户页面上显示。

页面操作类别字段定义

此表定义用于创建页面操作类别的字段。

字段

定义

类型

Name

页面操作类别的唯一名称。

必需

描述

页面操作类别的简短描述。

可选

FilterTags

指定一个过滤器标记，可用于限制可供查看和选择的项。

可选

LabelName

确定在页面操作类别上显示的标题。有关搜索标签的信息，请参考“搜索页面操作标签”。

可选

LabelText

确定在页面操作类别上显示的标题。此字段将取代在 LabelName 字段中输入的文本。

可选

注释

页面操作类别的相关注释。

可选

顺序

此页面操作类别的顺序。

可选

如何创建页面操作类别

按照以下步骤创建页面操作类别：

1. 访问 Portal Studio。
2. 单击新建并选择页面操作类别。随即工作区域中将显示一个空白页面操作类别。
3. 在名称字段中输入页面操作类别的名称。
4. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
5. 单击保存。应用程序将保存更改并显示成功消息。
