创建菜单

可以创建包含多个菜单和子菜单的菜单结构。

在创建菜单时

创建新菜单时，菜单显示为工作区域中的一组字段。

创建菜单时应考虑以下事项：

• 初次实施 Opcenter EX CR 和 Opcenter EX MDD 或者在更新到新版本以及升级默认实例数据时，系统会自动将 csiPortalMenuV8、csiMobileMenu 和 csiPortalMenu 默认菜单指派给 Administrator、CamstarAdmin 和 InSite Admin 用户。可使用员工建模对象将 Portal 菜单指派给其他员工。 有关信息，请参考《Opcenter Execution Medical Device and Diagnostics Portal Modeling 用户指南》或《Opcenter Execution Core Portal Modeling 用户指南》。
• Siemens 建议先创建选项卡以用作菜单引用选项卡，然后再创建菜单。有关信息，请参考“创建选项卡”。

菜单可通过三种方式引用选项卡。此表描述可引用选项卡的方式。

选项卡引用

描述

当前选项卡

从当前选项卡继承选项卡设置。

新建选项卡

始终在门户中新的空白选项卡中打开。选择此选项可同时打开一个选项卡的多个副本。

指定选项卡

指定之前创建的选项卡。一次仅能打开指定选项卡的一个版本。

如果将菜单或子菜单设置到指定选项卡，并保留了下面所有的项与继承的默认值，则它们都将在同一指定选项卡中打开。最后打开的页面将替换指定选项卡中当前打开的页面。

有关信息，请参考“如何向菜单定义添加指定选项卡”。

打开现有菜单

可以从“组件”目录中打开现有菜单。有关打开现有菜单的信息，请参考“打开现有组件”。

如何创建菜单定义

按照以下步骤创建菜单定义：

1. 访问 Portal Studio。
2. 单击新建，然后选择菜单定义。随即工作区域中将显示一个空白菜单。
3. 在名称字段中输入菜单的名称。
4. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
5. 单击保存。应用程序将保存更改并显示成功消息。

如何向菜单定义添加指定选项卡

按照以下步骤向菜单定义添加指定选项卡：

1. 访问 Portal Studio。
2. 在“组件”目录中选择导航选项卡，然后双击一个菜单。随即工作区域中将显示该菜单。
3. 从 PortalTabOption 字段中选择 SpecifiedTab。
4. 从 PortalTab 字段中选择选项卡。
5. 单击保存。应用程序将保存更改并显示成功消息。
