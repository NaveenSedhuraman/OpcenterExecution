重新加载 Web 部件

Portal Studio 允许从“组件”目录中重新加载 Web 部件。

在重新加载 Web 部件时

仅在 Web 部件处于以下状态时，才能重新加载 Web 部件：

• 是页面的一部分。
• 作为“组件”目录中列出的独立 Web 部件存在。
• 与“组件”目录中列出的 Web 部件共享其名称。
注释: 只有在位于“控件”视图中时，才能重新加载 Web 部件。如果位于“设计”视图中，则无法将 Web 部件重新加载到页面上的单元格中。

如何重新加载 Web 部件

按照以下步骤重新加载 Web 部件：

1. 访问 Portal Studio。
2. 双击包含 Web 部件的页面。 随即工作区域中将显示该页面。
3. 确保在“视图”列表中选择控件。
4. 在页面上选择 Web 部件并对其进行任何更改。
5. 单击重新加载 Web 部件。已移除上次保存后的所有更改。
