上传和指派图像

可以将图像上传到 Portal Studio 并将其指派给支持图像的控件，例如按钮。

在上传和指派图像时

ImageControl、Button、FlyoutDropdown 和 SectionDropdown 控件支持上传和指派图像。

使用“属性”窗格中的 DefaultImage 图标上传图像。如果图像已指派给控件，则该按钮显示图像缩略图和图像名称。上传的图像成为 Portal Studio 图像库的一部分，可供支持图像的所有控件使用。

可以在 DefaultImage 文本字段中输入图像文件名的一部分，以返回与下拉列表中的部分图像名称匹配的图像。

如何上传图像

按照以下步骤上传图像：

1. 访问 Portal Studio。
2. 如果组件目录处于隐藏状态，单击显示/隐藏目录。将显示组件目录。
3. 双击页面或 Web 部件。随即工作区域中将显示该页面或 Web 部件。
4. 如果“属性”窗格处于隐藏状态，单击显示/隐藏属性。将显示“属性”窗格。
5. 选择支持图像的控件。随即“属性”窗格中将显示该控件的属性。
6. 在“属性”窗格中单击 DefaultImage 图标。将显示“打开”弹出窗口。
7. 选择图像文件，然后单击打开。“打开”弹出窗口关闭，图像将上传并指派给控件。
8. 单击保存。应用程序将保存更改并显示成功消息。

如何指派图像

按照以下步骤指派图像：

1. 访问 Portal Studio。
2. 如果组件目录处于隐藏状态，单击显示/隐藏目录。将显示组件目录。
3. 双击页面或 Web 部件。随即工作区域中将显示该页面或 Web 部件。
4. 如果“属性”窗格处于隐藏状态，单击显示/隐藏属性。将显示“属性”窗格。
5. 选择支持图像的控件。随即“属性”窗格中将显示该控件的属性。
6. 从“属性”窗格的 DefaultImage 下拉列表中选择图像。图像将显示在图标中，其文件名将显示在文本字段中。
7. 单击保存。应用程序将保存更改并显示成功消息。
