向页面添加浮出控件

如果某页充当以门户页面上浮出控件面板形式显示的 Web 部件的容器，可以向该页添加“部分”下拉控件。

在向页面添加浮出控件时

只能向 Web 部件添加控件。不能直接向页面添加控件。有关向 Web 部件添加控件的信息，请参考“向 Web 部件添加控件”。

AltOpenLabelName 和 AltOpenLabelText 属性确定打开时浮出控件标签上显示的文本。AltCloseLabelName 和 AltCloseLabelText 属性确定关闭时浮出控件标签上显示的文本。

如果只定义一个属性，则在打开或关闭时，浮出控件的标签上将显示相同的文本。如果未定义任何这些属性，则浮出控件的标签上不会显示任何文本。有关控件属性的信息，请参考“控件属性”。

如果同时指定了文本和标签，在 AltOpenLabelText 字段中输入的文本将替代在 AltOpenLabelName 字段中选择的字典标签。如果同时指定了文本和标签，在 AltCloseLabelText 字段中输入的文本将替代在 AlltCloseLabelName 字段中选择的字典标签。

如何向页面添加浮出控件

按照以下步骤向页面添加浮出控件：

1. 访问 Portal Studio。
2. 如果组件目录处于隐藏状态，单击显示/隐藏目录。将显示组件目录。
3. 双击页面或 Web 部件。随即工作区域中将显示该页面或 Web 部件。
4. 如果工具箱处于隐藏状态，则单击显示/隐藏工具箱。将显示工具箱。
5. 单击“部分”下拉控件，并将其拖到 Web 部件上的单元格中。随即在 Web 部件中将显示“部分”下拉控件。
6. 如果“属性”窗格处于隐藏状态，单击显示/隐藏属性。将显示“属性”窗格。
7. 在“属性”窗格的名称字段中输入控件的名称。
8. 在 AltOpenLabelName、AltOpenLabelText、AltCloseLabelName 或 AltCloseLabelText 字段中输入文本或选择字典标签。
9. 重复步骤 8，根据需要添加其他标签文本。
10. 根据需要，单击 Web 部件并将其拖到“部分”下拉控件中。
11. 单击保存。应用程序将保存更改并显示成功消息。
