添加文件浏览控件

可以向门户页面添加文件浏览控件，以浏览文件并将文件上传到门户页面。

在添加文件浏览控件时

文件浏览控件包含两个字段：“浏览模式”和“查找文件”。

只能向 Web 部件添加控件。不能直接向页面添加控件。有关信息，请参考“向 Web 部件添加控件”。

重要提示: 对于具有文件上传功能的“建模”页，需要文件浏览控件。

如何添加文件浏览控件

按照以下步骤添加文件浏览控件：

1. 访问 Portal Studio。
2. 双击页面或 Web 部件。随即工作区域中将显示该页面或 Web 部件。
3. 单击工具箱中的文件浏览控件，并将其拖动到 Web 部件上的单元格中。随即 Web 部件中将显示文件浏览控件。
4. 在“属性”窗格的名称字段中输入控件的名称。
5. 单击保存。应用程序将保存更改并显示成功消息。
