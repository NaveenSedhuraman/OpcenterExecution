创建 Web 部件

可以在 Portal Studio 中创建 Web 部件，包括页面上的控件和字段。

在创建 Web 部件时

字段和控件可直接拖至 Web 部件，但是不能直接拖至页面。

将 Web 部件保存到页面后，可以通过单击“运行”按钮来测试该 Web 部件。有关测试 Web 部件和页面的信息，请参考测试页面。

Siemens 建议创建 Web 部件时在“属性”窗格中输入至少一个名称和标题。Name 字段用于定义 Web 部件的名称。Title 属性用于定义显示为 Web 部件标题的文本。有关 Web 部件属性的信息，请参考“Web 部件属性”。

打开现有 Web 部件

可以从“组件”目录中打开现有 Web 部件。有关打开现有 Web 部件的信息，请参考“打开现有组件”。

重新加载 Web 部件

可以通过单击 Portal Studio 全局工具栏上的“重新加载 Web 部件”按钮来重新加载 Web 部件。重新加载 Web 部件会将其恢复为先前保存的状态。只能重新加载属于页面一部分的 Web 部件。

如何创建 Web 部件

按照以下步骤创建 Web 部件：

1. 访问 Portal Studio。
2. 单击新建，然后选择 Web 部件。随即工作区域中将显示空白 Web 部件。
3. 在“属性”窗格的 Name 字段中输入 Web 部件的名称。
4. 在“属性”窗格的 Title 属性中输入标签。
5. 单击保存。应用程序将保存更改并显示成功消息。
