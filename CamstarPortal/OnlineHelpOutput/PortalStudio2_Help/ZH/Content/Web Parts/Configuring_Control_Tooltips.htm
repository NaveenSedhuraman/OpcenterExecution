配置控件工具提示

将鼠标光标悬停在页面中的控件上时，将显示控件工具提示。可以使用文本或标签在 Portal Studio 中配置工具提示。

在配置控件工具提示时

控件工具提示在“属性”窗格中配置。ToolTipText 属性使用在 ToolTipText 字段中输入的文本创建工具提示。ToolTipLabel 属性将打开“标签查找”弹出窗口，以搜索并选择要用作工具提示的标签。有关“标签查找”弹出窗口的信息，请参考“搜索页面操作标签”。

输入文本并选择标签时，ToolTipText 属性优先于 ToolTipLabel 属性。

如何配置控件工具提示

按照以下步骤配置控件工具提示：

1. 访问 Portal Studio。
2. 双击包含控件的页面或 Web 部件。随即工作区域中将显示该页面或 Web 部件。
3. 选择一个控件。随即“属性”窗格中将显示该控件的属性。
4. 要为工具提示输入文本还是选择标签？

如果您想完成以下操作 ...

则...

输入文本
a. 在“属性”窗格的 ToolTipText 字段中输入工具提示的文本。
b. 转到步骤 5。

选择标签
a. 在“属性”窗格中单击 ToolTipLabel 对应的更多。将显示“标签查找”弹出窗口。
b. 输入一个或多个搜索准则以缩小搜索范围，然后单击搜索。应用程序将在表格中显示与搜索准则匹配的标签。
c. 选择标签。随即“默认文本”和“本地文本”字段中将显示该标签的默认文本和本地文本。
d. 单击确定。 “标签查找”弹出窗口关闭，标签的信息会显示在 ToolTipLabel 字段中。
e. 转到步骤 5。
5. 单击保存。应用程序将保存更改并显示成功消息。
