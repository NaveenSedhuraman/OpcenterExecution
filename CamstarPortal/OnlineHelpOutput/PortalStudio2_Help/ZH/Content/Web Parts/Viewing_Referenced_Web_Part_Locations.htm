查看引用的 Web 部件位置

可以查看引用 Web 部件的组件列表，以验证哪些组件受到工作区域中“何处使用”视图对 Web 部件所做更新的影响。

在查看引用的 Web 部件位置时

引用的 Web 部件是只读副本，它们引用 Web 部件的主副本。通过这些部件，可在一个位置维护一个 Web 部件，而不是维护多个单独的 Web 部件。

打开 Web 部件时，可以从编辑模式下拉菜单中访问“何处使用”视图。选择“何处使用”视图将显示两个表，其中列出了引用当前 Web 部件的虚拟页面和其他 Web 部件，及其关联的工作区。

注释: 可以单击“名称”和“工作区”列标题对列出的组件进行排序。

如何查看引用的 Web 部件位置

按照以下步骤查看引用的 Web 部件位置：

1. 访问 Portal Studio。
2. 展开 Web 部件。
3. 双击一个 Web 部件。随即工作区域中将显示该 Web 部件。
4. 选择何处使用编辑模式。工作区域将刷新以显示引用 Web 部件的页面和其他 Web 部件。
