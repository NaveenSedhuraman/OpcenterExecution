配置布尔下拉标签

通过在 Portal Studio 中修改布尔下拉控件的属性，可以在这些控件中为 true 和 false 值设置自定义标签。

在配置布尔下拉标签时

true 和 false 值必须以 X_CustomLabel 格式输入，其中 X 是系统中的设计者或用户标签，CustomLabel 是自定义标签名称。如果未配置，标签将默认显示为 True 或 False。

true 标签和 false 标签彼此独立地进行配置。所配置的 true 标签和 false 标签仅应用于正在修改的控件，并且不会影响其他控件标签。

布尔下拉列表会显示在“工具箱”窗格的可用控件列表中。有关向 Web 部件添加控件的信息，请参考“向 Web 部件添加控件”。

如何配置布尔下拉标签

按照以下步骤配置布尔下拉标签：

1. 访问 Portal Studio。
2. 双击包含布尔下拉列表控件的页面或 Web 部件。随即工作区域中将显示该页面或 Web 部件。
3. 选择布尔下拉控件。随即“属性”窗格中将显示布尔下拉控件的设置。
4. 在“属性”窗格的 FalseLabel 字段中输入新的 false 标签。
5. 在“属性”窗格的 TrueLabel 字段中输入新的 true 标签。
6. 单击保存。应用程序将保存更改并显示成功消息。
