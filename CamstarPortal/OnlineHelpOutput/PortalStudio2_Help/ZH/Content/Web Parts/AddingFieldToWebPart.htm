向 Web 部件添加字段

字段用于输入信息，并且包含在 Web 部件中。例如，向 Web 部件添加“容器”字段，供用户用于输入容器名称。

在向 Web 部件添加字段时

向 Web 部件添加字段时，系统会自动设置字段表达式和控件类型。

只能向 Web 部件添加字段；不能直接向页面添加字段。

注释: 只有在位于“控件”视图中时，才能向 Web 部件添加字段。如果位于“设计”视图中，则无法将字段拖动到页面上的单元格中。

如何向 Web 部件添加字段

按照以下步骤将字段添加到 Web 部件：

1. 访问 Portal Studio。
2. 双击 Web 部件或包含 Web 部件的页面。随即工作区域中将显示该 Web 部件或页面。
3. 确保在“视图”列表中选择控件。
4. 单击工具箱上的字段目录选项卡。随即“工具箱”窗格中将显示服务列表。
5. 展开包含要添加的字段的服务。
6. 选择字段，然后将其拖放到 Web 部件上的单元格上。
7. 重复步骤 6-7 以将其他字段添加到 Web 部件。
8. 单击保存。应用程序将保存更改并显示成功消息。
