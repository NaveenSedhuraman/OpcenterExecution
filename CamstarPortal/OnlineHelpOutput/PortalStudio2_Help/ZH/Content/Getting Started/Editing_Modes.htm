编辑模式

可以使用编辑模式来添加、更新或查看页面和 Web 部件。模式显示在工作区域顶部附近的下拉列表中。

此表定义编辑模式。

Mode

选择此模式可 ...

控件

添加、移动或移除 Web 部件和控件。这是默认模式。

设计

修改布局。选择此模式可显示“操作”按钮，其中包含用于合并或拆分单元格以及插入或移除列和行的选项。有关信息，请参考“管理布局”。

查看

显示页面的 HTML 呈现。

Xml

显示或编辑页面的 XML 代码。

个性化报告

显示个性化报告，其中包含页面的基本信息、数据协定以及有关该页面上使用的 Web 部件和控件的信息。

何处使用

显示引用 Web 部件的所有组件。
