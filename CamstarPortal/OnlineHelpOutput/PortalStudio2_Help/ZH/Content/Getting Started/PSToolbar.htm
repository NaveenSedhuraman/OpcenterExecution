Portal Studio 全局工具栏

Portal Studio 全局工具栏提供轻松访问 Portal Studio 中所使用的功能的方式。

Portal Studio 全局工具栏按钮

此表定义 Portal Studio 全局工具栏上的可用按钮。

按钮

按钮功能 ...

查看内容

将工作区域中的组件视图更改为另一个工作区。默认情况下设为“开发者”。有关信息，请参考“在工作区中进行开发”。

注：只有某个组件已打开并在工作区域中选择了该组件时，才会显示此选项。

门户设置

显示“门户设置”窗口。有关修改门户设置的信息，请参考“配置门户设置”。

回到门户

返回 Portal。

帮助

显示 Portal Studio 联机帮助。

注销

从 Portal Studio 和门户中注销并返回到登录页面。

新建

创建新组件。单击此按钮可显示一个下拉列表，其中包含用于创建新的用户界面组件的选项。

保存

保存工作区域中已选组件的待处理更改。有关保存更改的信息，请参考“保存更改”。

注：对工作区域中当前处于焦点的组件进行更改时，Portal Studio 将启用此按钮。通过将 AlwaysEnableSave 工作室设置设为 True，可将此按钮设为始终启用。有关修改 Portal Studio 设置的信息，请参考“工作室设置”。

全部保存

保存工作区域内所有开放选项卡中组件的待处理更改。有关保存更改的信息，请参考“保存更改”。

注：对工作区域中当前处于焦点的组件进行更改时，Portal Studio 将启用此按钮。通过将 AlwaysEnableSave 工作室设置设为 True，可将此按钮设为始终启用。有关修改 Portal Studio 设置的信息，请参考“工作室设置”。

撤消

撤消最近的更改。“撤消”按钮会显示一个计数，指明可撤消的更改的数量。按住 Shift 键并单击“撤消”将显示可撤消的操作的列表。

重做

恢复使用“撤消”按钮撤消的更改。“重做”按钮会显示一个计数，指明可恢复的已撤消更改的数量。按住 Shift 键并单击“重做”将显示可恢复的操作的列表。

复制

复制选定的组件。

删除

删除选定的组件。

导入

从 XML 文件导入组件。有关导入组件的信息，请参考“导入组件”。

导出

从当前工作区或所有活动工作区导出组件。有关导出组件的信息，请参考“导出组件”。

批量导入

从单个 XML 文件导入多个 Web 部件、页面和页面流。有关执行批量导入的信息，请参考“执行批量导入”。

批导出

将多个 Web 部件、页面和页面流导出至单个 XML 文件或 Zip 文件。有关执行批量导出的信息，请参考“执行批量导出”。

恢复为

返回到组件的已发布内容或系统内容版本。有关信息，请参考“恢复内容”。

运行

使用浏览器在 Portal 框架中测试当前页面。必须先保存页面，然后才能测试该页面。有关信息，请参考“测试页面”。

发布

将保存的更改发布到最高活动工作区。有关信息，请参考“发布内容”。有关工作区的信息，请参考“在工作区中进行开发”。

生成

生成当前页。有关信息，请参考“生成页面”。

提取 Web 部件

提取当前在页面上选择的 Web 部件并在新选项卡中打开。有关提取 Web 部件的信息，请参考“提取 Web 部件”。

重新加载 Web 部件

将当前在页面上选择的 Web 部件恢复为该 Web 部件在“组件”目录中的原始状态。有关重新加载 Web 部件的信息，请参考“重新加载 Web 部件”。

目录

显示或隐藏“组件”目录。有关“组件”目录的信息，请参考“组件目录”。

工具箱

显示或隐藏“工具箱”窗格。有关“工具箱”窗格的信息，请参考“工具箱”。

属性

显示或隐藏“属性”窗格。 有关“属性”窗格的信息，请参考“属性窗格”。

跟踪查看器

显示或隐藏跟踪查看器。有关跟踪查看器的信息，请参考“运行跟踪日志”。

重置设置

重置对工作区域中窗格布局所做的任何更改。
