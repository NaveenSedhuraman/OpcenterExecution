Portal Studio 用户界面功能

Portal Studio 的用户界面在所有应用程序窗格和弹出窗口中都具有标准按钮和图标。

标准按钮

使用标准按钮是为了整个应用程序用户界面的一致性，以增强用户体验并简化数据输入任务。

此表定义 Portal Studio 中的常用按钮。

保存更改并关闭弹出窗口或模式窗口。

取消

关闭弹出窗口或模式窗口而不保存更改。

注：也可以单击右上角的 X 来关闭弹出窗口。

清除

清除在弹出窗口、窗格或模式窗口上的字段中输入的选择和值。

在窗格中搜索组件

Portal Studio 中的窗格提供了一个字段，可用于在相关编录或目录中查找特定组件、字段和属性。在搜索字段中输入前几个字符或完整名称，以搜索特定组件、字段或属性。

在用户键入内容的同时，搜索字段会过滤返回的组件的列表。搜索字段假定输入内容的每端都有通配符，从而可以找到名称中包含整个短语或短语的一部分的所有组件。

查看字段和属性

Portal Studio 以三种视图显示字段和属性：

• 收藏夹 - 显示您收藏的字段和属性的列表。
• 排序 - 按字母顺序显示所有字段和属性的列表。
• 分组 - 显示所有字段和属性，并按其功能（例如，行为、外观或布局）对它们进行分类。

随时可以通过选择视图选项卡来切换视图。窗格默认以排序视图显示字段和属性。

将字段和属性添加到收藏夹

通过按住 Ctrl 键并单击字段名称或属性名称，可将字段和属性添加到收藏夹视图。添加到收藏夹列表后，字段名称和属性名称以蓝色显示。

注释: 只有在设置了字段和属性的客户端机器上，相应字段和属性才会保留在收藏夹中。
导航模式窗口

Portal Studio 使用模式窗口来配置需要添加实例的设置。模式窗口包含两个窗格：

• 左窗格，列出以前创建的实例。
• 右窗格，提供所选实例的详细信息以及用于自定义所选实例的属性。

所有模式窗口都包含一个“添加”按钮，单击该按钮可将新实例添加到左窗格中的实例列表。单击“确定”可保存所有所做更改并关闭该模式窗口。单击“取消”可放弃所做更改并关闭该模式窗口。

布尔值

有些字段和属性显示为包含以下布尔值的下拉列表：

• 未设置
• True
• False

许多布尔值下拉列表都默认使用“未设置”。值“未设置”可触发与字段或属性设为 "False" 相同的功能。Siemens 建议将字段和属性保持设为“未设置”，除非业务需求要求将字段设为 "False"。
