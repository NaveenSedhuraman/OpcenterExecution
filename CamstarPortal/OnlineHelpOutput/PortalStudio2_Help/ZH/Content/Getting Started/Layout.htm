管理布局

Portal Studio 支持使用包含单元格的行和列布局来配置页面或 Web 部件的布局。

可以在“属性”窗格中修改单个单元格的属性。有关修改单元格属性的信息，请参考“单元格属性”。

在管理布局时

每个页面或 Web 部件都包含至少一个单元格作为组件布局的一部分。通过单元格可以指定组件显示在页面和 Web 部件上时相对于其他单元格中组件的位置。

在“设计”视图中查看组件时，可以使用“操作”按钮来修改单元格的布局。可以通过首先选择一个单元格然后选择另一个单元格，来选择多个单元格。这个两个单元格之间的所有单元格都将包含到选择内容中。选定的单元格带有蓝色背景。

此表定义“操作”按钮中提供用于管理单元格布局的选项。

选项

选择此选项可 ...

合并

将选定的单元格合并为一个单元格。

拆分

将合并的单元格重新拆分为两个或多个单元格。

移除行

删除所选行。

移除列

删除所选列。

在前面插入行

在所选行的上方插入新行。

在前面插入列

在所选列的左侧插入新列。

在后面插入行

在所选行的下方插入新行。

在后面插入列

在所选列的右侧插入新列。

重置

取消选择当前选择的单元格。

切换单元格位置

切换单元格的（行或列）位置指示。
