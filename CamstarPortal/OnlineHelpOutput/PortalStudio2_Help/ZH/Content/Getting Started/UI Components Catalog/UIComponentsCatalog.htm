组件目录

“组件”目录包含多个选项卡，其提供对组件、导航和映射组件的访问。

“组件”选项卡提供对以下组件的访问：

• 页面
• Web 部件
• 页面流

“导航”选项卡提供对以下导航组件的访问：

• 菜单
• 选项卡

“建模”选项卡提供对以下映射组件的访问：

• 页面操作
• 页面操作类别
• 页面操作规则
• 页面映射
• 目标矩阵

会将“组件”目录选项卡组织到各种组件类型的下拉列表中。各下拉列表都包含搜索字段、“新建项”按钮、一组视图模式按钮和刷新按钮。单击“新建项”按钮将在工作区域中显示该项类型的新项。

Portal Studio 工具栏上的“目录”按钮可显示隐藏的“组件”目录，以及隐藏可见的“组件”目录。

注释: 从早期版本升级后，首次登录到 Portal Studio 时，某些默认页面可能会显示在“组件”目录的“组件”选项卡中的“未定义”组中。可以自动将“未定义”组中的默认页面排序到其他组中。有关信息，请参考“如何自动对默认页面进行排序”。
打开现有组件

“组件”目录包含所有默认组件以及您创建的任何组件。可以通过导航至组件的选项卡和部分，然后双击组件名称，从“组件”目录中打开任何组件。随即工作区域中将显示打开的组件。
