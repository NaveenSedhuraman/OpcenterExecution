控件

控件可用于在 Portal Studio 中自定义 Web 部件，并显示在“工具箱”的“控件”选项卡中。有关查看工具箱的信息，请参考“工具箱”。

只能向 Web 部件添加控件；不能直接向页面添加控件。有关向 Web 部件添加控件的信息，请参考“向 Web 部件添加控件”。

如果添加的控件已计划在即将推出的 Portal Studio 版本中弃用，则会显示一条弹出式警告消息。该警告消息包括将弃用该控件的时间和要改用的备选控件。将弃用的控件上也会显示包含此信息的工具提示。

此表介绍“工具箱”的“控件”选项卡中提供的控件。

图标

控件

描述

3D 可视化

显示具有旋转、放大和缩小、标记图像以及保存快照等功能的 3D 可视化。

注：此控件支持 JT 文件。

使用“3D 可视化”控件时，将显示一个 3D 文件查看器弹出窗口，该弹出窗口会显示您的 3D 图像，您可以在其中旋转以及放大和缩小 3D 图像。

3D 文件查看器弹出窗口包含一个工具栏，可用于操作 3D 图像以及保存 2D 快照。 该工具栏可用于执行以下操作：

• 添加文本
• 添加形状
• 在图像上绘制自由窗体
• 从颜色选取器中选择文本颜色和形状颜色
• 为形状工具设置自定义线宽
• 重置所做更改
• 保存 2D 快照

还可以在可查看文档集的位置查看 3D 图像。有关使用“3D 可视化”控件的信息，请参考“使用 3D 可视化”。

布尔下拉列表

包含可配置标签和选项以清除值的布尔选择器。

布尔开关

带有可配置标签的布尔单选按钮控件。

痕迹导航控件

此控件使用户可以在页面上进行痕迹导航。

按钮

可以显示为文本或图像的可单击控件。

图表控件

在页面或 Web 部件上放置可自定义的图表。多个图表控件可以同时存在于一页上。

可以使用“属性”窗格中的 ChartMode、ShowLegend、SeriesSettings、SeriesValueColumn、TooltipColumn、ValueColumn、XAxis 和 YAxis 属性修改图表控件的格式和数据。在修改这些属性的数据源时，图表控件会动态更新。有关图表控件属性的信息，请参考“控件属性”。

SeriesSettings 属性可用于进一步自定义数据点在散点图和晶元图图表上的外观。有关添加 SeriesSettings 属性的信息，请参考“添加 SeriesSettings”。

可以通过修改 d3.js 文件来创建自定义图表格式，包括晶圆图。有关 D3 文档对象模型的信息，请参考 http://d3js.org。

注：可以使用鼠标滚轮在门户页面上放大和缩小图表控件。

复选框

选中或清除此框可打开或关闭选项。

注：向包含弹出式表格的页面添加复选框控件时，必须将 AutoPostBack 属性设为 "True"。有关修改控件属性的信息，请参考“控件属性”。

检查表

此控件在页面上显示为检查表。

颜色选取器

可用于选择调色板以设置字段的颜色。有关信息，请参考“使用颜色选取器”。

容器下拉列表

容器选择列表

容器选择器，包含表格中显示的选择值。

数据收集

此控件允许用户输入一个或多个值。

日期选择器

包含日期输入选项的日期选择器。

发现区域

显示自定义“发现区域”控件。

下拉列表

一个下拉控件，用于显示选择值列表。

持续时间

此控件用于指定一个时间段 (DD.HH:MM:SS)，并可与流程时间强制结合使用。有关定义流程计时器的信息，请参考《Opcenter Execution Medical Device and Diagnostics Portal Modeling 用户指南》或《Opcenter Execution Core Portal Modeling 用户指南》。

文件浏览

可用于浏览并选择要上传的文档。

文件输入

可用于浏览并选择要用作数据输入的文件。

浮出控件下拉列表

显示为一个按钮，单击该按钮可在浮出控件面板中打开另一个页面。

表格

具有可选数据模式的表格。

注：向 Web 部件添加表格控件时，应用程序会提示您使用表格向导来创建表格，或放置一个空白表格控件并手动对其进行自定义。有关信息，请参考“使用表格向导定义表格”。

HTML 面板

使用户可以指定 HTML 内容或模板文件的路径。

注：此控件支持使用 HTMLText 属性作为文本输入 HTML 内容，也支持使用 TemplateFile 属性指定 HTML 模板文件的文件夹路径。使用此控件时，必须在 HTMLText 或 TemplateFile 属性中输入值。有关 HTMLText 和 TemplateFile 属性的信息，请参考“控件属性”。

IFrame

将内联 HTML 页面放入另一个页面内。

图像控件

在页面上显示图像。

查询控件

在页面上显示轻型只读字段。

指示文本

在页面上放置只读文本框。

智能对象

Business Objects 控件。

注：默认情况下，此控件使用 HTTP 连接来访问和执行 Business Objects 报告。可以在“属性”窗格中更改控件设置。有关信息，请参考“控件属性”。

Intelligence Report 对象

Opcenter Intelligence Report 控件。有关信息，请参考《Opcenter Intelligence Core Report 安装指南》。

注：有关信息，请参考“控件属性”。

区段下拉列表

切换容器

此控件包含多个 Web 部件并保留放置在其上的 Web 部件的布局。可选择使用 ToggleDisplayMode 属性在门户页面上创建可折叠部分。有关信息，请参考“控件属性”。

用户字段
