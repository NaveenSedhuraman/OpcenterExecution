表格列

可以从“<表格名称> - 列”弹出窗口将默认列添加到表格。在“表格设置编辑器”弹出窗口中单击“列”的“更多”，将显示“<表格名称> - 列”弹出窗口。有关表格设置的信息，请参考“表格设置编辑器”。

列属性根据当前所选列的类型动态显示。

<表格名称> - 列弹出窗口字段定义

此表提供“<表格名称> - 列”弹出窗口中包含的列设置列表。

列设置属性

属性描述

BindPath

包含数据的相对父路径或字段名称。

CellActions

显示“<列名称> - CellActions”弹出窗口。可以从“<列名称> - CellActions”弹出窗口定义其他单元格操作，包括按钮、浮出控件和链接。

CellStyle

设置列的单元格样式。

CustomScriptHandlers

显示“<列名称> - CustomScriptHandlers”弹出窗口。可以从“<列名称> - CellActions”弹出窗口指定自定义脚本处理程序操作，包括单击、更改和 onAddCellAttributes。

DateFormat

为日期和时间设置可接受的格式。

DateFormatExample

显示可接受日期格式的示例。

DefaultValue

为列中的条目设置默认值。

Editable

将列设为可编辑。

EditorProperties

一系列子属性，用于确定数据如何传递到列以及如何从列中传递。

EditorProperties - 常规

FieldExpressions

设置“业务逻辑”对象模型的绑定路径以检索和/或提交数据。

RequestData

设置事件的数据请求。

RetrieveList

确定检索值的时间。

SelectionData

设置事件的选择数据的请求。

ValueFilter

根据另一个对象的值过滤选择项。

Frozen

冻结列，这些列将在表格的其余部分继续滚动时保持可见。有关信息，请参考“冻结列”。

GroupByColumnName

参考“已命名子实体”控件，按列名称分组。

HeaderStyle

设置列的标题样式。

Hidden

隐藏列。

HiddenContent

隐藏列中的内容。

HTMLTagTransform

确定如何转换或保留 HTML 标签。此属性包含以下用于转换 HTML 标签的选项：

• 自动 - 自动搜索 HTML 标签的文本，“<标签>”，并保存和传输 HTML 代码，同时允许内容格式调整。默认情况下选中此选项。
• 转换 - 用 "&lt;" 和 "&gt;" 替换字符 "<" 和 ">"。用 "&#x22;" 和 "&#x2F;" 替换字符 "\" 和 "/"。用 "&#x27;" 替换字符 " ' "。此选项显示 HTML 标签中包含的内容，并禁用任何 HTML 格式调整。默认情况下选中此选项。
• 保留 - 像数据库中文本存在的格式一样保留文本，并显示任何 HTML 格式。

IsRowID

将列指定为行标识。只能将一列定义为行 ID。

LabelName

确定列上显示的标题。有关搜索预定义标签的信息，请参考“搜索页面操作标签”。

LabelText

确定列上显示的标题。

ListDisplayExpression

显示表达式选择值的列表。

ListValueColumn

对象模型表达式，标识列的值来自何处。

Name

列的名称。

必需

需要在列中输入值。

Resizable

允许调整列的大小。

Sortable

允许对列进行排序。

TypeFormat

将 Type 设为 Subentity 或 NamedSubentity。

TypeValidation

确定应用程序是否验证所选“类型”。

UseCurrentTime

设置列以使用当前时间格式。

ValueExpression

带有“字段表达式”构建器链接的文本框。

ValueFormat

将值格式设为“名称”或 ID。

Width

列的宽度（以像素为单位）。
