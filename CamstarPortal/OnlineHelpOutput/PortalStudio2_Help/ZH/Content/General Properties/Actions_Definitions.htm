操作属性

通过操作属性，可以查看或编辑页面、Web 部件和其他对象的本地操作的属性。当在“属性”窗格中单击操作的“更多”按钮时，“<对象> - 操作”弹出窗口中将显示操作属性。有关“属性”窗格的信息，请参考“属性窗格”。

“<对象> - 操作”弹出窗口中隐藏当前所选操作类型不需要的操作属性。

此表提供操作属性的描述。

操作属性

属性描述

ActionCategory

指定要与操作关联的操作类别。

ButtonPosition

确定当“位置”字段设为“按钮”时，操作按钮是显示在页面的左侧还是右侧。

ClearValues

在提交操作时清除值。

ConditionHandler

选项：“DefaultConditionHandler”或“SpecificConditionHandler”。定义当提交操作时调用的代码中的处理程序方法。这样，便可以在操作提交时从应用程序代码执行其他功能。 

Confirmation

对在确认消息上显示的标签进行配置的属性集。

Confirmation - 常规

Cancel_LabelName

确定在确认消息的“取消”按钮上显示的文本。

Cancel_LabelText

确定在确认消息的“取消”按钮上显示的文本。此字段取代在 Cancel_LabelName 子属性中输入的文本。

Message_LabelName

确定显示为确认消息的消息的文本。

Message_LabelText

确定显示为确认消息的消息的文本。此字段取代在 Message_LabelName 子属性中输入的文本。

OK_LabelName

确定在确认消息的“确定”按钮上显示的文本。

OK_LabelText

确定在确认消息的“确定”按钮上显示的文本。此字段取代在 OK_LabelName 子属性中输入的文本。

Title_LabelName

确定显示为确认消息标题的文本。

Title_LabelText

确定显示为确认消息标题的文本。此字段取代在 Title_LabelName 子属性中输入的文本。

CssClass

指定操作的自定义 CSS 类。

CustomMethodHandler

选项：ExecuteMaintenanceTransaction、ExecuteNewTransaction、ShopfloorReset、TrueSubmit、VIewDocument、VIewInExcel 或 WebPartCustomAction。

描述

操作的描述。

DisableCondition

属性条件表达式，如果该表达式计算结果为 True，将导致禁用操作。

DisableCondition - 描述

Expression

计算的条件表达式。

DisableCondition - 对等

ColumnName

比较中使用的列的名称。

Comparison

用于计算条件的比较运算符。有效条目为 "Equal"、"NonEqual" 和 "Contains"。

DisableCondition - 对等 - LeftValue - 常规

ConnectionType

指定事件处理程序的连接类型。

注：将根据所选的连接类型显示其他字段。

Control

指定要在条件计算的给定端使用的控件。

注：当在 ConnectionType 字段中选择“控件”时，将显示此字段。

PageDataContract

指定要在条件计算的给定端使用的页面数据协定。

注：当在 ConnectionType 字段中选择 PageDataContract 时，将显示此字段。

SessionDataContract

指定要在条件计算的给定端使用的会话数据协定。

注：当在 ConnectionType 字段中选择 SessionDataContract 时，将显示此字段。

SessionVariable

指定要在条件计算的给定端使用的会话变量。

注：当在 ConnectionType 字段中选择 SessionVariable 时，将显示此字段。

FieldExpression

指定要在条件计算的给定端使用的字段表达式。

注：当在 ConnectionType 字段中选择 FieldExpression 时，将显示此字段。

StaticValue

指定要在条件计算的给定端使用的静态值。

注：当在 ConnectionType 字段中选择 StaticValue 时，将显示此字段。

GridItem

指定要在条件计算的给定端使用的表格项。

注：当在 ConnectionType 字段中选择 GridItem 时，将显示此字段。

Property

指定计算的控件属性值。

注：当在 ConnectionType 字段中选择“控件”时，将显示此字段。

RequestDataMode

指定用于从控件检索属性值的方法。

注：当在 ConnectionType 字段中选择“控件”时，将显示此字段。

AllowEmptyValue

确定值能否为空。

Initiators

单击“更多”按钮时，将显示 Initiators 弹出窗口。

DisableCondition - 对等 - RightValue

注：RightValue 字段与 LeftValue 字段相同。有关字段定义，请参考“DisableCondition - 对等 - LeftValue - 常规”字段。

DisableCondition - 常规

And

单击“更多”按钮时，将显示 And 弹出窗口。可为条件事件处理程序定义将使用 "And" 运算符添加的其他条件。

注：And 弹出窗口字段与定义条件事件处理程序时显示的字段相同。有关字段定义，请参考 "DisableCondition" 字段。

Or

单击“更多”按钮时，将显示 Or 弹出窗口。可为条件事件处理程序定义将使用 "Or" 运算符添加的其他条件。

注：Or 弹出窗口字段与定义条件事件处理程序时显示的字段相同。有关字段定义，请参考 "DisableCondition" 字段。

DynamicDisplayMode

对于具有不允许查看操作的角色的用户，确定操作的显示模式。

ESignatureDependencies(Collection)

提交操作所需的电子签名的集合。

ESignature必需

是否需要电子签名？

FormValidate

表示是否在表单提交之前打开表单验证（在 html5 中实现）。

GroupName

操作组的名称。

HiddenCondition

计算的条件，用于确定页面中是否隐藏页面操作。

注：当定义条件表达式时，HiddenCondition 子属性与 DisableCondition 的子属性相同。有关属性描述，请参考 "DisableCondition" 字段。

HorizontalShift

移位顺序：从左向右

HorizontalShift - 常规

HorizontalShiftDirection

指定在运行时调整操作的水平位置的方向。接受的值为 "Left"、"Right" 和 "None"。

HorizontalShiftValue

调整操作的水平位置的像素数。

索引

用于对操作的显示进行排序的整数。

IsControlDisabled

控制是否已禁用？

IsDisabled

操作是否已禁用？

IsHidden

操作是否已隐藏？

IsPrimary

操作是主要操作吗？

IsReturn

提交后关闭页面。

LabelName

操作的字典标签名称。

LabelText

操作的标签文本。此字段取代在 LabelName 属性中输入的文本。

位置

确定操作将在页面上显示的位置。如果选择“按钮”，则会将操作设为以按钮的形式显示在关联页面底部。如果选择 ActionPanel，则会将操作设为显示在关联页上的 ActionsControl Web 部件中。

Parameters

可用于输入可在页面的 App_Code 文件的 WebPartCustomAction 函数替代中使用的字符串值。参数通常用于确定在页面上有多个自定义操作的情况下单击了哪个按钮。

Permissions - 常规

DisplayMode

选项：Disable 或 Hide。权限不满足情况下的显示模式。

PagePermission

是否应用页面权限？

ServicePermission

是否应用服务权限？

ReloadValues

当设为 True 时，操作在提交后重新加载页面值。

ServiceName

单击操作时执行的服务。

SidebarCollectible

表示操作在水平主题中的命令栏上可见。

SidebarCSS

指定边栏的 CSS。

SPCEnabled

启用 SPC 操作。

WIPMessages必需

指定操作是否需要 WIP 消息。
