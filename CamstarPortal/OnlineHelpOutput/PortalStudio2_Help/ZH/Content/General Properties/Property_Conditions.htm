属性条件

属性条件允许使用一系列自定义条件和触发器来设置或更改控件上的属性。

在设置属性条件时

属性条件是称为条件集的左右值之间的比较。计算条件集，当比较计算为 True 时，在另一个元素上设置值。

属性条件可以是包含单个条件集的单个属性条件，也可以包含由 And 和 Or 运算符分隔的多个条件集组。属性条件必须至少包含一个条件集和一个 Value If True 条件。例如，一个简单条件集可以利用 Value If True 条件检查两个控件是否在页面上都包含相等值，Value If True 条件为：当条件集计算为 True 时，将另一个控件设为该值。

属性条件为树结构。每个顶级控件都包含属性条件，在“<控件> - PropertyConditions”弹出窗口的“属性条件”窗口中可见。

可以在每个属性条件中定义以下内容：

• 包含左右值的条件集。可以从该级别定义以下内容：
• 包含指定何时评估条件集中的“左侧值”或“右侧值”的事件的发起程序。
• 由 And 和 Or 运算符分隔以添加其他条件集的条件组。条件组可根据需要包含自己的发起程序和进一步的子条件组。
• 以及当条件集计算为 True 时设置的 Value If True 条件。可以从该级别定义以下内容：
• 包含指定何时应设置 Value If True 条件的事件的发起程序。

此结构可用于根据需要创建多个条件集、发起程序触发器和结果。

<控件> - PropertyConditions 弹出窗口

当在工作区域中选择控件时，在“属性”窗格上单击“属性条件”的“更多...”按钮，将显示“<控件> - PropertyConditions”弹出窗口。有关查看“属性”窗格的信息，请参考”属性窗格”。

可以在“<控件> - PropertyConditions”弹出窗口中创建新的属性条件或查看现有的属性条件。“属性条件”窗格显示当前在控件上设置的属性条件。可以单击箭头展开和折叠每个属性条件，以查看关联的条件集、条件组和“相关属性”字段。为了可读性，条件组缩进显示，每个子条件组进一步缩进显示。

“属性条件”窗格中的每个属性条件都具有“名称”和“相关属性”字段。添加新属性条件时，必须从“相关属性”字段中选择属性。“相关属性”字段确定控件上的哪个属性将接收来自 Value If True 条件的值。通过在“属性条件”窗格中选择属性条件名称并输入新名称，可编辑现有属性条件的名称。

“编辑条件...”和“编辑 Value if True...”按钮在“属性条件”窗格中的属性条件下方显示。单击：

• “编辑条件...”按钮将显示“条件编辑器”弹出窗口。有关信息，请参考“条件编辑器”。
• “编辑 Value if True...”按钮将显示“Value If True 编辑器”弹出窗口。有关信息，请参考“Value If True 编辑器”。

可以通过单击属性条件右侧的 X 来删除属性条件。

可以通过单击“取消”按钮或单击右上角的 X 来关闭“<控件> - PropertyConditions”弹出窗口。

条件编辑器

当单击“<控件> - PropertyConditions”弹出窗口中的“编辑条件...”按钮时将显示“条件编辑器”弹出窗口。可在“条件编辑器”弹出窗口中定义属性条件集以及多个条件组和子条件组。

通过从第一个下拉列表中选择比较运算符来确定条件集如何比较“左侧值”和“右侧值”，可创建条件集。然后选择引用对象（如控件、数据协定等）的“左侧值”和“右侧值”。

如何访问<控件> - PropertyConditions 弹出窗口

按照以下步骤访问“<控件> - PropertyConditions”弹出窗口：

1. 访问 Portal Studio。
2. 如果组件目录处于隐藏状态，单击显示/隐藏目录。将显示组件目录。
3. 双击 Web 部件或包含 Web 部件的页面。随即工作区域中将显示该 Web 部件或页面。
4. 如果“属性”窗格处于隐藏状态，单击显示/隐藏属性。将显示“属性”窗格。
5. 在工作区域中选择一个控件。
6. 单击属性条件的更多。将显示“<控件> - PropertyConditions”弹出窗口。

如何定义属性条件

按照以下步骤定义属性条件：

1. 执行“如何访问<控件> - PropertyConditions 弹出窗口”程序。
2. 在输入属性条件名称字段中输入属性条件的名称，然后单击添加属性条件。“属性条件”窗格中将显示属性条件。
3. 展开属性条件并从相关属性字段中选择一个属性。
4. 单击编辑条件...。将显示“条件编辑器”弹出窗口。
5. 从第一个下拉字段中选择比较运算符。
6. 从 ConnectionType 字段中选择“左侧值”的连接类型。左侧值的相关字段将根据所选的连接类型进行刷新。
7. 根据业务需求在显示的字段中输入信息。有关必填字段和可选字段的信息，请参考字段定义表。
8. 对“右侧值”字段重复步骤 5-7。
9. 单击确定。 “条件编辑器”弹出窗口关闭。
10. 单击编辑 Value If True...。将显示“Value If True 编辑器”弹出窗口。
11. 从 ConnectionType 字段中选择连接类型。字段将根据所选的连接类型进行刷新。
12. 根据业务需求在显示的字段中输入信息。有关必填字段和可选字段的信息，请参考字段定义表。
13. 单击确定。 “Value If True 编辑器”弹出窗口关闭。
14. 单击确定。 “<控件> - PropertyConditions”弹出窗口关闭并保存更改。
