在配置门户设置时

Portal Studio 设置保存在以下文件路径：

C:\Program Files (x86)\Camstar\Camstar Portal\User\Settings.xml

此表定义已选 DefaultSettings 的“门户设置”弹出窗口中的字段。

字段

定义

AutoPagePermission

在发布新页面时自动向“默认页面”角色添加页面权限。默认情况下设为 True。

CachePersonalization

是否使用缓存进行个性化设置？设为 True 以提高应用程序性能

CancelKeyPressList

浏览器将忽略的键控代码列表。

ComboRowsRange

当前未实施。

ConciergeItemsDisplayedPerSection

任务中心每个部分中显示的项目数。

ConciergeRefreshInterval

任务中心面板刷新数据之前的秒数。

CopyNDO

当前未实施。

CopyNDOFloating

当前未实施。

CopyRDO

当前未实施。

CopyRDOFloating

当前未实施。

CountersEnabled

在使用 Portal Studio 跟踪时记录耗时。

DataTransKeepSessionAlive

当设为 True 时，导出、导入、包部署和包激活会话将运行，直到会话完成。当设为 False 时，二十分钟后将发生默认会话超时。默认情况下设为 True。

注：通过在 web.config 文件中输入“会话状态”属性的新值，可以修改默认超时。

DefaultTheme

确定 Portal 的默认主题是使用经典样式表还是水平样式表。默认情况下设为“水平”。

如果使用默认或移动进入模式登录门户，则默认主题为水平。如果使用经典进入模式，则默认主题为经典。在使用这些进入模式中的其中一种登录门户后，不应更改默认主题。相应的默认主题旨在与每个进入模式配合使用。有关信息，请参考《Opcenter Execution Medical Device and Diagnostics 入门用户指南》或《Opcenter Execution Core 入门用户指南》。

注：默认情况下，如果 DefaultTheme 设为“水平”，并且正在计算机上查看页面，则所有车间页面上的文本将以较大的字体大小显示。默认情况下，所有“建模”、“更改管理”和“导出/导入”页均以标准字体大小显示。页面级别的字体大小使用 CssClass 页面属性进行设置。

DisplayDropdownIconWhenReadOnly

当字段为“只读”时，是否显示“下拉”图标？ True 或 False。

DisplayUserDCDTooltips

是否显示用户数据收集工具提示？ True 或 False。

EnforceCheckOutWhenEditing

在保存之前，需要在 Portal Studio 中签出页面或 Web 部件。通常用于多开发者环境中。

ESignature必需

全局电子签名默认设置。 页面设置将替代全局设置。

FullScreenMode

当设为 True 时，将以全屏模式显示应用程序的框架。默认情况下设为 False。

注：此设置将以下应用程序框架元素扩展到完整浏览器宽度：

• 全局页脚
• 全局页眉
• 选项卡行
• 菜单栏

可以在各个页面上设置全屏模式，并使用页面属性隐藏页面元素。有关页面属性的信息，请参考“页面属性描述”。

LogonAutoComplete

是否允许自动完成登录？ True 或 False。

MainPage

默认主页。

MaxAttachmentSizeLimit

文件附件的最大大小（单位：MB）。默认情况下设为 30。

注：可能需要增加以下设置中的值，才能留出足够的时间导入或导出较大的文件：

• Portal web.config 和 WCF 服务 web.config 文件中的 SendTimeout 和 ReceiveTimeout 值
• XMLClientConfig.xml 文件中的 HttpTimeout 和 HttpReadWriteTimeout

MaxDataTransferRefreshAttempts

指定数据传输刷新的最大尝试次数。

MobileBarcodeEnabled

可用于将移动设备中的数据扫描到车间页面上的可编辑字段中。默认情况下设为 True。此设置和页面级别设置 MobileBarcodeEnabled 必须全部设为 True，条形码扫描图标才会在车间页面的可编辑字段中显示。如果设为 False，所有车间页面上将禁用条形码扫描功能。

工作室设置

“工作室设置”为 Portal Studio 中的页面配置默认设置。

此表定义已选 StudioSettings 的“门户设置”弹出窗口中的字段。

字段

定义

AlwaysEnableSave

当设为 True 时，将设置“保存”和“全部保存”按钮为始终启用。默认情况下设为 False。有关 Portal Studio 工具栏按钮的信息，请参考“Portal Studio 全局工具栏”。

注：当在以下情况下将 AlwaysEnableSave 设为 True 时，“保存”和“全部保存”按钮都将保持禁用状态：

字段

定义

DateTimeFormat

可设置“门户”的日期和时间格式。这将影响日期和时间在页面和在成功消息中的显示方式，以及事务日期在“审核追踪”中的显示方式。

此设置将替代在浏览器语言中所做的任何设置。

DateTimeFormat - 常规

LongTimePattern

在显示时间时替代浏览器语言设置的时间模式。此设置使用 Microsoft 的标准日期和时间格式。有关 Microsoft 日期和时间格式的信息，请参考 https://msdn.microsoft.com/en-us/library/8kb3ddd4(v=vs.110).aspx?cs-save-lang=1&cs-lang=vb#code-snippet-2。

输入自定义格式将替代应用程序中的时间显示设置，例如，时间戳和服务器消息。对于在控件的“自定义格式”属性中设置了自定义时间格式的控件，此设置不会修改该控件的时间格式。有关“自定义格式”属性的信息，请参考“控件属性”。

ShortDatePattern

替代有关日期显示的浏览器语言设置的日期模式。此设置使用 Microsoft 的标准日期和时间格式。有关 Microsoft 日期和时间格式的信息，请参考 https://msdn.microsoft.com/en-us/library/8kb3ddd4(v=vs.110).aspx?cs-save-lang=1&cs-lang=vb#code-snippet-2。

注：对于在控件的“自定义格式”属性中设置了自定义日期格式的控件，此设置不会修改该控件的日期格式。有关“自定义格式”属性的信息，请参考“控件属性”。

如何配置门户设置

按照以下步骤配置“门户设置”：

1. 访问 Portal Studio。
2. 单击门户设置按钮。将显示“门户设置”弹出窗口。
3. 选择要查看或更改的设置。将显示该设置的属性。
4. 选择一个设置并进行必要的更改。
5. 单击确定。 此操作将关闭“门户设置”弹出窗口并保存更改。
