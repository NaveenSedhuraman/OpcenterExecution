数据协定

“数据协定”指定义用于定义类的对象堆栈，这些类可用于生成符合服务协定预期的输入和输出消息。可通过“数据协定”向外部源公开封装在 UIComponents 中的数据。

还可通过“数据协定”将数据作为输入或输出传输至 Web 页面和 Web 部件。

启用网页或 Web 部件的命令栏

可以使用数据协定在任何门户页面上启用或禁用命令栏。默认情况下，命令栏对某些车间页面是启用的。通过查看页面“属性”窗格中“数据协定”的数据协定设置，可以看到为页面启用的命令栏。

如何为网页或 Web 部件启用命令栏

按照以下步骤为网页或 Web 部件启用命令栏：

1. 访问 Portal Studio。
2. 如果“用户界面组件”目录处于隐藏状态，单击“显示/隐藏目录”。将显示“用户界面组件”目录。
3. 双击网页或 Web 部件。随即工作区域中将显示该页面或部件。
4. 如果“属性”窗格处于隐藏状态，单击显示/隐藏属性。将显示“属性”窗格。
5. 从“属性”窗格中选择 DataContractMembers 属性。将显示 DataContractMembers 弹出窗口。
6. 单击添加。将显示数据协定字段。

8. 单击确定。将关闭 DataContractMembers 弹出窗口。输入的数据协定即保存。系统将为选定网页或 Web 部件启用命令栏。

如何禁用网页或 Web 部件的命令栏

按照以下步骤禁用网页或 Web 部件的命令栏：

1. 访问 Portal Studio。
2. 如果“用户界面组件”目录处于隐藏状态，单击“显示/隐藏目录”。将显示“用户界面组件”目录。
3. 双击网页或 Web 部件。随即工作区域中将显示该页面或部件。
4. 如果“属性”窗格处于隐藏状态，单击显示/隐藏属性。将显示“属性”窗格。
5. 从“属性”窗格中选择 DataContractMembers 属性。将显示 DataContractMembers 弹出窗口。
6. 选择 CommandBarSettingType 数据协定。将显示数据协定属性。
7. 清除 StaticValue 字段中的值。
8. 单击确定。将关闭 DataContractMembers 弹出窗口。更改即保存。系统将禁用所选网页或 Web 部件的命令栏。
