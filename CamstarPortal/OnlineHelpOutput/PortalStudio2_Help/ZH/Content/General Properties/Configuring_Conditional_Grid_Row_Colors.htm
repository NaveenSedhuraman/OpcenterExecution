配置条件表格行颜色

可以对表格行进行配置，使其在满足自定义条件时显示 css 样式属性，例如，彩色背景或彩色文本。例如，如果容器处于“搁置”状态，则可以将这些行设为显示红色背景。

在配置条件表格行颜色时

可以选择为表格行设置条件，使其在满足条件并选择行时显示不同的背景颜色或彩色文本。所选行的背景颜色在 SelectedStyle 字段中定义。

如果在表格中进行内联编辑，则会在更新或通过 Tab 键远离行后计算表达式。

表达式可以使用 "and" 或 "or" 运算符包含多个条件。例如，以下是可接受的条件表达式：

"DataType = 'Integer' and AttributeValue > 10"

注释: Condition 字段需要 Microsoft DataColumn.Expression Property 语法。有关其他条件选项的信息，请参考 Microsoft Developer Network。

NormalStyle 和 SelectedStyle 字段需要样式表引用或样式表语法来定义行或文本颜色。正确的语法是 "class:<css-class-name>"，其中 <css-class-name> 是在样式表中定义的 css 类的名称。例如，可接受的值可能是 "background:red"。

注释: 对 NormalStyle 或 SelectedStyle 使用类时，必须包含背景颜色的 "!important" 指示器。

如何配置条件表格行颜色

按照以下步骤配置条件表格行颜色：

1. 访问 Portal Studio。
2. 如果组件目录处于隐藏状态，单击显示/隐藏目录。将显示组件目录。
3. 双击包含表格控件的页面。 随即工作区域中将显示该页面。
4. 如果“属性”窗格处于隐藏状态，单击显示/隐藏属性。将显示“属性”窗格。
5. 选择一个表格控件。随即“属性”窗格中将显示表格控件的属性。
6. 单击设置的更多。将显示“表格设置编辑器”弹出窗口。
7. 展开布局。
8. 单击 ConditionalRowStyles 的更多。将显示 ConditionalRowStyles 弹出窗口。
9. 单击添加。 StyleRule 将添加到列表，右窗格中会显示空白字段。
10. 在 Condition 字段中输入条件表达式。
11. 在 NormalStyle 字段中输入背景颜色。
12. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
13. 重复步骤 9-12 以添加其他表格行条件。
14. 单击确定。 将关闭 ConditionalRowHandler 弹出窗口。
15. 单击确定。 将关闭“表格设置编辑器”弹出窗口。
16. 单击保存。应用程序将保存更改并显示成功消息。
