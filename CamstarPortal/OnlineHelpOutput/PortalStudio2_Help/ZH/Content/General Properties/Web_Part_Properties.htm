Web 部件属性

Web 部件属性允许查看或编辑 Web 部件的属性。当在工作区域中选择 Web 部件时，Web 部件属性将显示在“属性”窗格中。有关“属性”窗格的信息，请参考“属性窗格”。

选择属性后，“属性”窗格下方将出现一个描述窗口，其中显示了有关 Web 部件属性的信息。

注释: 某些属性包含可进一步自定义的子属性。单击父属性对应的“更多”按钮时，相应的子属性将显示在单独弹出窗口中。

Web 部件属性描述

此表提供 Web 部件属性的描述。

Web 部件属性

属性描述

Actions

Web 部件的操作。可用于定义 Web 部件级别操作和按钮。

注：单击“更多”按钮可显示 Actions 模式窗口。有关此模式窗口中显示的属性的信息，请参考“Actions 属性描述”。

BroadcastInitiators

用于创建将触发 Web 部件之间的页面级连接数据协定的发起程序。

注：单击“更多”按钮可显示 BroadcastInitiators 模式窗口。有关此模式窗口中显示的属性的信息，请参考“BroadcastInitiators 属性描述”。

ClientPropertyConditions

可用于根据浏览器端事件的条件结果将数据设为属性。

CollapsibleGroups

将在即将发布的发行版中弃用。

注：必须使用“切换容器”控件的 ToggleDisplayMode 属性创建可折叠部分。有关信息，请参考“控件属性”。

Control

在页面或 Web 部件上引用控件。

注：单击“更多”按钮可显示 Control 模式窗口。Control 模式窗口与控件共享属性。有关此模式窗口中显示的属性的信息，请参考“控件属性”。

Created

Web 部件的创建日期和时间。

CreatedBy

Web 部件的创建者。

CSSClass

Web 部件的样式表引用。

DataContractMembers

用于在页面之间进行通信的数据协定。

注：单击“更多”按钮可显示 DataContractMembers 模式窗口。有关此模式窗口中显示的属性的信息，请参考“DataContractMembers 属性描述”。

DataSubmissionMode

当页面向服务器提交数据时字段如何回应。选项包括：

• NotSet：未设置模式。这是默认模式。
• Force：值始终包含在提交数据中。
• Changed：值仅在更改时才提交。
• Filled：值仅在字段不为空时才提交。
• ChangedAndFilled：值仅在字段不为空且已更改时才提交。
• Skip：值未提交或加载。
• ViewOnly：值未提交。
• Skip Request：值未加载。不加载控件的选择值。

重要事项：Siemens 建议对“建模”页上的可编辑表格使用 Changed 选项。

DirtyFlagTrigger

如果 Web 部件中的任何字段在设为 True 时更改值，将该页标记为已更新的布尔字段。

注：此属性当前只影响建模框架内的 Web 部件。

DisplayBorder

在 Web 部件上显示边框。

DisplayFooter

在 Web 部件上显示页脚。

DisplayHeader

在 Web 部件上显示标题。

EventHandlers

可用于从事件指定处理程序。

注：单击“更多”按钮可显示 EventHandlers 模式窗口。有关定义事件处理程序的信息，请参考“定义事件处理程序”。

Height

Web 部件显示的高度（以像素为单位）。

Hidden

如果为 True，则控件或 Web 部件将以 HTML 呈现，但对用户不可见。例如，在 HTML 标记中显示 "none"。

HorizontalAlignment

Web 部件元素的垂直对齐方式。

HowLong

创建 Web 部件以来的时间段。

LastModified

Web 部件的上次修改日期。

LastModifiedBy

上次修改 Web 部件的用户。

Layout

Web 部件的可视外观。Siemens 建议您不要直接更改此属性。

Margin

Web 部件的边距。

Padding

Web 部件的填充。

PrimaryServiceType

未指定服务类型的控件的服务类型。

PropertyConditions

允许将 Web 部件的属性值定义为计算表达式（条件）的结果。该属性在页面执行运行时计算。

注：单击“更多”按钮可显示“<Component> - 属性条件”窗口。有关定义属性条件的信息，请参考“属性条件”。

SessionVariables

用于配置页面级别会话变量。

有关定义会话变量的信息，请参考“定义会话变量”。

Title

显示为 Web 部件标题的文本。当输入了文本并选择了标签时，此处输入的文本将优先于 TitleLabel 字段。

TitleLabel

选择要用作 Web 部件标题的标签。有关使用“标签查找”弹出窗口的信息，请参考“搜索页面操作标签”。

TypeName

Web 部件类型后面的代码。

VerticalAlignment

Web 部件元素的垂直对齐方式。

Width

Web 部件显示的宽度（以像素为单位）。
