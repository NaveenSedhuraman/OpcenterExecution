Seiten

Seiten (Pages) sind die visuellen Darstellungen von Portalseiten, die in Portal Studio mit diversen Seitenteilen wie Webparts oder dynamischen Aktionen in erstellt oder geändert werden können.

Diese Themen enthalten Informationen zu Seiten:

• Erstellen einer Seite
• Verwalten von Layouts
• Anzeigen der Webparts auf einer Seite
• Testen einer Seite
• Generieren einer Seite
• Verwalten von Seitengruppen
• Löschen von Komponenten
• Veröffentlichen von Inhalten
• Zurücksetzen von Inhalten
