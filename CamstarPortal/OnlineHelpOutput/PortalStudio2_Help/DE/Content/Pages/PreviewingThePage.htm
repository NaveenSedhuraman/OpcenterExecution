Anzeigen einer Seitenvorschau

Portal Studio bietet Ihnen die Möglichkeit, eine Seitenvorschau in einem Browserfenster anzuzeigen.

Beim Anzeigen einer Seitenvorschau

Die Schaltfläche "Vorschau" (Preview) ist aktiviert, wenn die Seite gespeichert wird oder wenn keine nicht gespeicherten Änderungen vorhanden sind.

Seiten in der Vorschau werden in einem neuen Fenster geöffnet, das nicht Teil des Portal-Frameworks ist. Weitere Informationen zum Testen einer Seite innerhalb des Portal-Frameworks finden Sie unter "Testen einer Seite".

Vorgehensweise zum Anzeigen einer Seitenvorschau

Gehen Sie folgendermaßen vor, um eine Seitenvorschau anzuzeigen:

1. Greifen Sie auf Portal Studio zu.
2. Doppelklicken Sie auf eine Seite. Die Seite wird im Arbeitsbereich angezeigt.
3. Klicken Sie auf Vorschau (Preview). Die Seite wird in einem Webbrowser geöffnet.
