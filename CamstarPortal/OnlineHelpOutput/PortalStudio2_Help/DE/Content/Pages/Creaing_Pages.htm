Erstellen einer Seite

In Portal Studio können Sie Seiten erstellen, die im Portal verwendet werden.

Beim Erstellen einer Seite

Sie können Seiten erstellen, indem Sie in der globalen Symbolleiste auf die Schaltfläche "Neu" (New) klicken und die Seite auswählen oder indem Sie auf der Registerkarte "Seiten" (Pages) auf die Schaltfläche "Neu" (New) des Komponentenkatalogs klicken. Das Popup "Neue Seite erstellen" (Create New Page) wird angezeigt, wenn Sie eine Seite erstellen. Es ermöglicht Ihnen, eine Vorlage für Ihre Seite auszuwählen. Das Popup "Neue Seite erstellen" (Create New Page) enthält folgende Vorlagen:

• "Leer" (Blank) – Stellt eine leere Vorlage für eine grundlegende Portalseite bereit.
• "Container" – Stellt die Basiskomponenten bereit, die zum Entwickeln einer Containerseite verwendet werden.
• "Modellierung" (Modeling) – Stellt die grundlegenden Komponenten zum Entwickeln einer Modellierungsseite bereit.

Sie müssen einen Namen für Ihre Seite in das Feld "Seitenname" (Page Name) eingeben, um die Schaltfläche "OK" auf der Popup-Seite "Neue Seite erstellen" (Create New Page) zu aktivieren.

Nachdem Sie eine neue Seite erstellt haben, können Sie Webparts, Steuerelemente und Felder auf die Seite ziehen. Sie können Steuerelemente und Felder nicht direkt auf eine Seite ziehen, sondern müssen diese auf einem Webpart platzieren. Weitere Informationen zum Hinzufügen von Webparts zu Seiten finden Sie unter "Hinzufügen eines Webparts zu einer Seite".

Mit der Schaltfläche "Ausführen" (Run) können Sie eine Seite testen, nachdem Sie sie gespeichert haben. Weitere Informationen finden Sie unter "Testen einer Seite".

Umlaufbestandsnachrichten sind in den Portaleinstellungen standardmäßig auf "True" (Wahr) festgelegt. Da die globalen Einstellungen durch Seiteneinstellungen überschreiben werden, können Sie Umlaufbestandsnachrichten für eine Seite über das Feld "WIPMessages"Erforderlich im Bereich "Eigenschaften" (Properties) deaktivieren.

Vorhandene Seiten öffnen

Vorhandene Seiten können im Komponentenkatalog geöffnet werden. Weitere Informationen zum Öffnen vorhandener Seiten finden Sie unter "Öffnen vorhandener Komponenten".

Vorgehensweise zum Erstellen einer Seite

Gehen Sie folgendermaßen vor, um eine Seite zu erstellen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf die Schaltfläche Neu und wählen Sie Seite aus. Das Popup "Neue Seite erstellen" (Create New Page) wird angezeigt.
3. Wählen Sie eine Seitenvorlage aus.
4. Geben Sie im Feld Seitenname (Page Name) einen Namen für die Seite ein.
5. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
6. Klicken Sie auf OK. Das Popup "Neue Seite erstellen" (Create New Page) und die neue Seite werden im Arbeitsbereich angezeigt.
7. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
