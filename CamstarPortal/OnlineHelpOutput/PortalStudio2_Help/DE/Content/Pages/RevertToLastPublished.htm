Zurücksetzen von Inhalten

Mit der Schaltfläche "Zurücksetzen zu" (Revert To) können Sie Seiten und Webparts im höchsten aktiven Arbeitsbereich auf den zuletzt veröffentlichten Zustand oder auf den Zustand des Systeminhalts im Basisarbeitsbereich von Siemens PLM CEP (CSI) zurücksetzen.

Beim Zurücksetzen von Seiten- und Webpartinhalten

Die Schaltfläche "Zurücksetzen zu" (Revert To) ist im Basisarbeitsbereich von Siemens PLM CEP (CSI) deaktiviert. Sie ist nur auf Seiten und Webparts aktiviert, auf denen Änderungen im höchsten aktiven Arbeitsbereich vorgenommen wurden. Weitere Informationen zum Speichern und Veröffentlichen von Änderungen in Arbeitsbereichen finden Sie unter "Entwickeln in Arbeitsbereichen".

Wenn Sie auf "Zurücksetzen zu" (Revert To) klicken und "Veröffentlichter Inhalt" (Published Content) auswählen, wird die Seite oder das Webpart auf den zuletzt veröffentlichten Zustand zurückgesetzt. Wenn Sie auf "Zurücksetzen zu" (Revert To) klicken und "Systeminhalt" (System Content) auswählen, wird die Seite oder das Webpart zurückgesetzt, sodass sie mit der Komponente im Basisarbeitsbereich von Siemens PLM CEP (CSI) übereinstimmt.

Vorgehensweise zum Zurücksetzen von Seiten und Webparts auf den zuletzt veröffentlichten Zustand

Gehen Sie folgendermaßen vor, um Seiten und Webparts auf den zuletzt veröffentlichten Zustand zurückzusetzen:

1. Greifen Sie auf Portal Studio zu.
2. Öffnen Sie eine Seite oder ein Webpart mit veröffentlichten Änderungen. Die Seite bzw. das Webpart wird im Arbeitsbereich angezeigt.
3. Klicken Sie auf Zurücksetzen auf (Revert To) und wählen Sie Veröffentlichter Inhalt (Published Content) aus. Das Dialogfeld "Zurücksetzen auf letzte Veröffentlichung" (Revert to Last Published) wird angezeigt.
4. Klicken Sie auf OK. Das Dialogfeld "Zurücksetzen auf letzte Veröffentlichung" (Revert to Last Published) wird geschlossen und die Seite bzw. das Webpart wird aktualisiert. Der Inhalt des zuletzt veröffentlichten Zustands wird angezeigt.

Vorgehensweise zum Zurücksetzen von Seiten und Webparts auf den Systeminhalt

Gehen Sie folgendermaßen vor, um Seiten und Webparts auf den Systeminhalt zurückzusetzen:

1. Greifen Sie auf Portal Studio zu.
2. Öffnen Sie eine geänderte Seite oder ein geändertes Webpart. Die Seite bzw. das Webpart wird im Arbeitsbereich angezeigt.
3. Klicken Sie auf Zurücksetzen auf... (Revert to...) und wählen Sie Systeminhalt (System Content) aus. Das Dialogfeld "Zurücksetzen auf Systeminhalt" (Revert to System Content) wird angezeigt.
4. Klicken Sie auf OK. Das Dialogfeld "Zurücksetzen auf Systeminhalt" (Revert to System Content) wird geschlossen und die Seite oder das Webpart wird aktualisiert. Der im Basisarbeitsbereich von Siemens PLM CEP (CSI) enthaltene Inhalt wird angezeigt.
