Anzeigen der Webparts auf einer Seite

Portal Studio bietet Ihnen die Möglichkeit anzuzeigen, welche Webparts auf einer Seite verwendet werden. Dabei können Sie sowohl die dynamischen als auch die statischen Webparts anzeigen, die auf einer Seite verwendet werden.

Beim Anzeigen der Webparts auf einer Seite

Wenn Sie im modalen Fenster "DynamicWebParts" ein Webpart auswählen, werden seine Eigenschaften im rechten Fensterbereich angezeigt. Im rechten Bereich kann dann ein weiterer Drilldown durchgeführt werden.

Note: Sie können die Webpart-Eigenschaften nicht im modalen Fenster "DynamicWebParts" ändern.

Vorgehensweise zum Anzeigen der Webparts auf einer Seite

Gehen Sie folgendermaßen vor, um Webparts auf einer Seite anzuzeigen:

1. Greifen Sie auf Portal Studio zu.
2. Doppelklicken Sie auf eine Seite. Die Seite wird im Arbeitsbereich angezeigt.
3. Klicken Sie im Fensterbereich "Eigenschaften" (Properties) auf Mehr (More) für DynamicWebParts. Das modale Fenster "DynamicWebParts" wird angezeigt. Es enthält eine Liste der Webparts, die sich derzeit auf der Seite befinden.
4. Wählen Sie im linken Bereich ein Webpart aus der Liste aus. Die Eigenschaften des Webparts werden im rechten Bereich angezeigt.
