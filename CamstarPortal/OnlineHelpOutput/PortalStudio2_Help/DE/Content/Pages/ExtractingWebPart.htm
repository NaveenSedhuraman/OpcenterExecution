Extrahieren eines Webparts

In Portal Studio können Sie ein Webpart auf einer Seite auswählen und dieses Webpart auf eine neue Registerkarte im Arbeitsbereich extrahieren.

Beim Extrahieren eines Webparts

Die Schaltfläche "WebPart extrahieren" (Extract Web Part) ist aktiviert, wenn Sie ein Webpart auf einer Seite auswählen.

Nach dem Speichern wird das extrahierte Webpart im Komponentenkatalog als neues Webpart angezeigt.

Vorgehensweise zum Extrahieren eines Webparts

Gehen Sie folgendermaßen vor, um ein Webpart zu extrahieren:

1. Greifen Sie auf Portal Studio zu.
2. Doppelklicken Sie auf eine Seite. Die Seite wird im Arbeitsbereich angezeigt.
3. Wählen Sie ein Webpart auf der Seite aus und klicken Sie auf Webpart extrahieren (Extract Web Part). Das Webpart wird auf einer neuen Registerkarte im Arbeitsbereich angezeigt.
4. Ändern Sie ggf. den Namen des Webparts im Feld "Name" im Fenster "Eigenschaften" (Properties).
5. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
