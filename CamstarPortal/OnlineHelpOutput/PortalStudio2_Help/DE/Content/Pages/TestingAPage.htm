Testen einer Seite

Portal Studio ermöglicht Ihnen das Testen einer Seite im Portal-Framework.

Beim Testen einer Seite

Sie testen eine Seite mit der Schaltfläche "Ausführen" (Run). Die Schaltfläche "Ausführen" (Run) ist aktiviert, wenn die Seite gespeichert wird oder wenn keine nicht gespeicherten Änderungen vorhanden sind.

Getestete Seiten werden im Portal-Framework in einer separaten Browserregisterkarte geöffnet, als ob sie über ein Portalmenü aufgerufen wurden.

Important: Daten, die Sie beim Testen auf einer Seite eingeben, werden in der Datenbank Ihrer Umgebung gespeichert.

Vorgehensweise zum Testen einer Seite

Gehen Sie folgendermaßen vor, um eine Seite zu testen:

1. Greifen Sie auf Portal Studio zu.
2. Doppelklicken Sie auf eine Seite. Die Seite wird im Arbeitsbereich angezeigt.
3. Klicken Sie auf Ausführen (Run). Die Seite wird auf einer separaten Browserregisterkarte im Portal geöffnet.
