Verwalten von Seitengruppen

Portal Studio ermöglicht es Ihnen, die Liste der verfügbaren Seiten im Komponentenkatalog in Gruppen einzuteilen. Seiten können mehreren Gruppen zugewiesen werden.

Note: Löschen Sie die Browserdaten oder klicken Sie auf "Standardeinstellungen wiederherstellen" (Restore Default Settings), um die standardmäßigen Gruppen- und Seitenzuweisungen zurückzugeben.

Beim Erstellen von neuen Seitengruppen

Wenn Sie mit der rechten Maustaste auf eine Seite klicken und "Zu neuer Gruppe hinzufügen" (Add to a New Group) auswählen, wird die Seite automatisch zu der neu erstellten Seitengruppe hinzugefügt.

Sie können den Namen einer vorhandenen Seitengruppe aus dem Kontextmenü, das Sie zum Erstellen einer Seitengruppe verwendet haben, bearbeiten.

Beim Hinzufügen von Seiten zu Gruppen

Wenn Sie sich beim Upgrade von einer früheren Version erstmals bei Portal Studio anmelden, können einige Standardseiten in der Gruppe "Nicht definiert" (Undefined) auf der Registerkarte "Komponenten" (Components) im Komponentenkatalog angezeigt werden.

Die Standardseiten in der Gruppe "Nicht definiert" (Undefined) können aus dem Komponentenkatalog automatisch in andere Gruppen sortiert werden. Weitere Informationen finden Sie unter "Vorgehensweise zum automatischen Sortieren von Standardseiten".

Note: Seiten können nicht den Gruppen "Alle" (All) oder "Nicht definiert" (Undefined) zugewiesen werden.

Beim Löschen von Seitengruppen

Seitengruppen lassen sich aus dem Komponentenkatalog löschen. Beim Entfernen einer Gruppe werden die Seiten, auf denen die Gruppe enthalten ist, nicht gelöscht.

Note: Die Gruppe "Alle" (All) kann nicht gelöscht werden.

Vorgehensweise zum Erstellen einer neuen Seitengruppe

Gehen Sie folgendermaßen vor, um eine neue Seitengruppe zu erstellen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie mit der rechten Maustaste auf eine Seite und wählen Sie Zu neuer Gruppe hinzufügen (Add to a New Group) aus. Das Dialogfeld "Gruppennamen eingeben" (Enter a Group Name) wird angezeigt.

Oder

Klicken Sie mit der rechten Maustaste auf eine vorhandene Seitengruppe und wählen Sie Neue Gruppe hinzufügen (Add a new group) aus. Das Dialogfeld "Gruppennamen eingeben" (Enter a Group Name) wird angezeigt.

3. Geben Sie einen Namen für die Gruppe ein.
4. Klicken Sie auf OK. Das Dialogfeld wird geschlossen und im Komponentenkatalog wird eine neue Gruppe angezeigt.

Vorgehensweise zum Bearbeiten einer Seitengruppe

Gehen Sie folgendermaßen vor, um eine Seitengruppe zu bearbeiten:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie mit der rechten Maustaste auf eine Seitengruppe und wählen Sie Bearbeiten (Edit) aus. Das Dialogfeld "Gruppennamen eingeben" (Enter a Group Name) wird angezeigt.
3. Geben Sie einen neuen Namen für die Gruppe ein.
4. Klicken Sie auf OK. Das Dialogfeld wird geschlossen und die Gruppe wird im Komponentenkatalog mit dem neuen Namen angezeigt.

Vorgehensweise zum Hinzufügen einer Seite zu einer Gruppe

Gehen Sie folgendermaßen vor, um eine Seite zu einer Gruppe hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie mit der rechten Maustaste auf eine Seite. Eine Liste der vorhandenen Gruppen wird angezeigt.
3. Aktivieren Sie das Kontrollkästchen für die Gruppe, zu der Sie die Seite hinzufügen möchten. Die Liste wird geschlossen und die Seite wird zu der ausgewählten Gruppe hinzugefügt.
4. Wiederholen Sie Schritt 4, um die Seite nach Bedarf zu weiteren Gruppen hinzuzufügen.

Vorgehensweise zum Löschen einer Seite aus einer Seitengruppe

Gehen Sie folgendermaßen vor, um eine Seite aus einer Seitengruppe zu löschen:

1. Greifen Sie auf Portal Studio zu.
2. Erweitern Sie eine Seitengruppe.
3. Klicken Sie mit der rechten Maustaste auf eine Seite und deaktivieren Sie die Seitengruppen, aus denen Sie die Seite entfernen möchten. Die Seite wird aus den geleerten Gruppen entfernt.

Vorgehensweise zum Löschen einer Seitengruppe

Gehen Sie folgendermaßen vor, um eine Seitengruppe zu löschen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie mit der rechten Maustaste auf eine Gruppe und wählen Sie Entfernen (Remove) aus. Die Gruppe wird aus dem Komponentenkatalog entfernt.

Vorgehensweise zum automatischen Sortieren von Standardseiten

Gehen Sie folgendermaßen vor, um Standardseiten automatisch zu sortieren:

1. Greifen Sie auf Portal Studio zu.
2. Wählen Sie die Gruppe Alle (All) aus.
3. Klicken Sie mit der rechten Maustaste auf die Gruppe Alle (All) und wählen Sie Standardeinstellungen wiederherstellen (Restore Default Settings) aus. Die Standardseiten in der Gruppe "Nicht definiert" (Undefined) werden in andere Gruppen sortiert.
Note: Durch Löschen der Browsereinstellungen werden auch Gruppen und Seiten auf die Standardeinstellungen zurückgesetzt.
