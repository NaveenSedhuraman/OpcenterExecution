Löschen von Komponenten

Sie können Komponenten in Portal Studio dauerhaft löschen.

Note: In Portal Studio gelöschte Komponenten werden dauerhaft aus der Anwendung entfernt und können nicht wiederhergestellt werden.

Vorgehensweise zum Löschen von Komponenten

Gehen Sie folgendermaßen vor, um Komponenten zu löschen:

1. Greifen Sie auf Portal Studio zu.
2. Öffnen Sie eine Komponente aus dem Komponentenkatalog. Die Komponente erscheint im Arbeitsbereich.
3. Klicken Sie in der globalen Symbolleiste von Portal Studio auf Löschen (Delete). Die Bestätigungsmeldung "<Komponente> löschen" (Delete <Component>) wird angezeigt.
4. Klicken Sie auf OK. Die Bestätigung "<Komponente> löschen" (Delete <Component>) wird geschlossen und die Komponente wird gelöscht.
5. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
