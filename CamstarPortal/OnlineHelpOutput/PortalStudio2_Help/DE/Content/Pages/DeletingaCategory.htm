Löschen einer Seitengruppen

Seitengruppen (Page Groups) können aus dem Katalog der Benutzeroberflächenkomponenten entfernt werden. Beim Entfernen einer Gruppe werden die Seiten, auf denen die Gruppe enthalten ist, nicht gelöscht.

Note: Die Gruppe "Alle" (All) kann nicht entfernt werden.

Vorgehensweise zum Löschen einer Seite aus einer Seitengruppe

Gehen Sie folgendermaßen vor, um eine Seite aus einer Seitengruppe zu löschen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Komponentenkatalog ausgeblendet ist. Der Komponentenkatalog wird angezeigt.
3. Erweitern Sie eine Seitengruppe.
4. Klicken Sie mit der rechten Maustaste auf eine Seite und deaktivieren Sie die Seitengruppen, aus denen Sie die Seite entfernen möchten. Die Seite wird aus den geleerten Gruppen entfernt.

Vorgehensweise zum Löschen einer Seitengruppe

Gehen Sie folgendermaßen vor, um eine Seitengruppe aus Portal Studio zu löschen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Komponentenkatalog ausgeblendet ist. Der Komponentenkatalog wird angezeigt.
3. Klicken Sie mit der rechten Maustaste auf eine Gruppe und wählen Sie Entfernen (Remove) aus. Die Gruppe wird aus dem Katalog der Benutzeroberflächenkomponenten entfernt.
