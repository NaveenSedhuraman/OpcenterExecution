Verwenden der 3D-Visualisierung

Mit dem 3D-Visualisierungs-Steuerelement können Sie 3D-Bilder aus einer Anwendung anzeigen. Die Bilder werden in einem separaten Popup "3D-Datei-Viewer" (3D File Viewer) angezeigt, in dem Sie folgende Aktionen an Bildern ausführen können:

• Drehen
• Vergrößern und Verkleinern
• Markieren
• Momentaufnahme speichern
Beim Verwenden der 3D-Visualisierung

Das 3D-Visualisierungs-Steuerelement unterstützt JT-Dateien. 3D-Bilddateien werden im Popup "3D-Datei-Viewer" (3D File Viewer) angezeigt. Sie können 3D-Bilder mit dem 3D-Visualisierungs-Steuerelement oder in einer beliebigen Anwendung anzeigen, mit der Dokumentsätze angezeigt werden können.

Das Popup "3D-Datei-Viewer" (3D File Viewer) bietet zwei Modi:

• 3D, in dem Sie 3D-Bilder frei drehen und vergrößern bzw. verkleinern können.
• Markierung (2D), in dem Sie 2D-Standbilder markieren und speichern können.

Das Popup "3D-Datei-Viewer" (3D File Viewer) zeigt beim ersten Öffnen einer 3D-Bilddatei Bilder im 3D-Modus an. Sie können auf ein 3D-Bild klicken und es ziehen, um es frei zu drehen. Außerdem können Sie Bilder durch Drehen des Mausrads vergrößern bzw. verkleinern. Durch Wechseln in den Markierungsmodus (2D) wird ein 2D-Standbild des 3D-Bilds in der Perspektive erzeugt, die aktuell im Popup "3D-Datei-Viewer" (3D File Viewer) angezeigt wird.

Der Markierungsmodus (2D) enthält eine Symbolleiste, über die Sie Text und Formen mit anpassbaren Linienstärken und -farben hinzufügen können. Informationen zu den Symbolleistenoptionen und den Werkzeugen, die zum Markieren von 2D-Bildern verfügbar sind, finden Sie unter "Symbolleistenoptionen im Popup "3D-Datei-Viewer" (3D File Viewer)".

Sie können auf die Schaltfläche "Speichern" (Save) klicken, um Momentaufnahmen von 2D-Bildern zu erzeugen. Durch Klicken auf die Schaltfläche "Speichern" (Save) wird das Dialogfenster "Speichern" (Save) in Ihrem Browser angezeigt, über das Sie Momentaufnahmen als Bilddatei auf dem lokalen Computer speichern können. Ein roter Rahmen wird um Bilder angezeigt, wenn sie nicht gespeicherte Änderungen enthalten.

Durch Klicken auf die Schaltfläche "Zurücksetzen" (Reset) können Sie alle Markierungen und Anmerkungen entfernen. Wenn beim Klicken auf die Schaltfläche "Zurücksetzen" (Reset) nicht gespeicherte Änderungen vorhanden sind, wird eine Warnmeldung angezeigt. Klicken Sie in der Warnmeldung auf "Ja" (Yes), um die Änderungen zu entfernen. Klicken Sie in der Warnmeldung auf "Nein" (No), um die Änderungen beizubehalten.

Sie können auf "Zurück zu 3D" (Back to 3D) klicken, um wieder in den 3D-Modus zu wechseln. Durch das Wechseln zurück in den 3D-Modus werden alle Markierungen oder Anmerkungen entfernt. Wenn beim Klicken auf "Zurück zu 3D" (Back to 3D) nicht gespeicherte Änderungen in dem 2D-Bild vorhanden sind, wird eine Warnmeldung angezeigt. Klicken Sie in der Warnmeldung auf "Ja" (Yes), um das Dialogfeld "Speichern" (Save) in Ihrem Browser anzuzeigen, über das Sie Momentaufnahmen auf dem lokalen Computer speichern und dann wieder in den 3D-Modus wechseln können. Klicken Sie auf "Nein" (No), um Ihre Änderungen zu verwerfen und wieder in den 3D-Modus zu wechseln.

Anmerken (Annotate)

Fügen Sie Text zu dem Bild hinzu. Der Mauszeiger ändert sich nach dem Klicken auf "Anmerken" (Annotate) zu einem Textwerkzeug. Durch Klicken auf das Bild mit dem Textwerkzeug können Sie Text über dem Bild festlegen.

Form (Shape)

Wählen Sie ein Werkzeug aus und fügen Sie eine gerade oder Freiformlinie, ein Quadrat oder einen Kreis hinzu.

Farbe (Color)

Wählen Sie benutzerdefinierte Farbwerte für Formen und Text aus bzw. geben Sie die Werte ein. Der Standardwert ist "Schwarz" (Black). Informationen zum Verwenden der Farbauswahl zum Festlegen benutzerdefinierter Farben finden Sie unter "Verwenden der Farbauswahl".

Linienstärke (Line Weight)

Legen Sie die Linienstärke der Formwerkzeuge fest. Der Standardwert ist 3.
