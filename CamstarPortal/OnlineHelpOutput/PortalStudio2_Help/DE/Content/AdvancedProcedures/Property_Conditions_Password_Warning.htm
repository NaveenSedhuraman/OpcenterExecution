Aktivieren von Kennwort-Warnmeldungen

Sie können Warnmeldungen aktivieren, um Portalbenutzer auf ein leeres Kennwortfeld hinzuweisen. Ändern Sie dazu die Eigenschaftsbedingungen eines Kennwort-Steuerelements in Portal Studio.

Beim Aktivieren von Kennwort-Warnmeldungen

Warnmeldungen weisen einen Portalbenutzer auf fehlende Angaben in Pflichtfeldern hin. Das Kennwortfeld ist jedoch nicht standardmäßig Bestandteil einer Warnmeldung. Eine Fehlermeldung aufgrund eines leeren Kennwortfelds wird nur angezeigt, wenn alle anderen Pflichtfelder ausgefüllt wurden. Warnmeldungen können nur auf Seiten innerhalb des Modellierungsframeworks aktiviert werden.

Das Feld "Statischer Wert" (StaticValue) bei den Eigenschaften für "Rechter Wert" (RightValue) muss leer bleiben, um Kennwort-Warnmeldungen zu aktivieren.

Vorgehensweise zum Aktivieren von Kennwort-Warnmeldungen

Gehen Sie folgendermaßen vor, um Kennwort-Warnmeldungen zu aktivieren:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf die Schaltfläche Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Komponentenkatalog ausgeblendet ist. Der "Komponentenkatalog" (Components Catalog) wird angezeigt.
3. Doppelklicken Sie auf eine beliebige Portal Modeling-Seite, die ein Kennwort-Steuerelement aufweist. Die Seite wird angezeigt.
4. Klicken Sie auf Eigenschaften anzeigen/ausblenden (Show/Hide Properties), wenn der Bereich "Eigenschaften" (Properties) ausgeblendet ist. Der Bereich "Eigenschaften" (Properties) wird angezeigt.
5. Wählen Sie das Kennwort-Steuerelement auf der Seite aus, z. B. ObjectChanges_Password.
6. Klicken Sie im Bereich "Eigenschaften" (Properties) auf Mehr (More) für Eigenschaftsbedingungen (PropertyConditions). Das Popup "Eigenschaftsbedingungen" (PropertyConditions) wird angezeigt.
7. Klicken Sie auf Hinzufügen. Eine neue Eigenschaftsbedingung wird zu der Liste hinzugefügt und die zugehörigen allgemeinen Eigenschaften werden angezeigt.
8. Wählen Sie Erforderlich für Eigenschaft (Property) aus.
9. Klicken Sie unter Allgemein (General) auf die Schaltfläche Erweitern (Expand) für Bedingung (Condition). Die Eigenschaften für "Bedingung" (Condition) werden angezeigt.
10. Klicken Sie unter Gleichwertigkeit (Equivalency) auf Erweitern (Expand) für Linker Wert (LeftValue). Die Eigenschaften für "Linker Wert" (LeftValue) werden angezeigt.
11. Wählen Sie Seitendatenvertrag (PageDataContract) als Verbindungstyp (Connection Type) aus.
12. Geben Sie im Feld Vertragsdaten (PageDataContract) den Wert Ausgewählte Instanzreferenz (SelectedInstanceRef) ein.
13. Klicken Sie unter Gleichwertigkeit (Equivalency) auf Erweitern (Expand) für Rechter Wert (RightValue). Die Eigenschaften für "Rechter Wert" (RightValue) werden angezeigt.
14. Wählen Sie Statischer Wert (StaticValue) als Verbindungstyp (Connection Type) aus.
15. Klicken Sie auf OK. Das Popup "Eigenschaftsbedingungen" (PropertyConditions) wird geschlossen.
16. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
