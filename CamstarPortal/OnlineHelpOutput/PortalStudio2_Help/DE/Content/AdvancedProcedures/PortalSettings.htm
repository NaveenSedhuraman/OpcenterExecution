Erweiterte Prozeduren

Sie können das Portal über die erweiterten Eigenschaften und Prozeduren in Portal Studio weiter anpassen.

Die folgenden Themen enthalten Informationen zu erweiterten Einstellungen:

• Hinzufügen eines Flyouts zu einer Seite
• Aktivieren von Kennwort-Warnmeldungen
• Aktivieren von Transaktionen mit automatischem Ablauf
• Erweitern der Anweisungen für automatische Abläufe
• Aktivieren von akustischen Signalen für Statusnachrichten
• Aktivieren der automatischen Aktualisierung für SPC-Charts und Festlegen einer Aktualisierungsrate
• Erstellen von Bestätigungsnachrichten
• Konfigurieren von Lokalisierungseinstellungen
• Aktivieren von Haftfeldern
• Definieren von Sitzungsvariablen
• Hinzufügen von Hilfeinformationen zu einer Portalseite
• Ausführen des Ablaufverfolgungsprotokolls
• Aktivieren des Schalters zum Speichern für die Änderungsverwaltung (Change Mgt Save)
• Hochladen und Zuweisen von Bildern
• Verwenden der Farbauswahl
• Visuelle Gewichtsdarstellung auf Portalseiten
• Verwenden der 3D-Visualisierung
