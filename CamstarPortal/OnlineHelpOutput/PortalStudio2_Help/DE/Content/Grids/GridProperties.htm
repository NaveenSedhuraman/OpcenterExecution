Tabelleneigenschaften

Diese Tabelle enthält eine Liste der Tabelleneigenschaften (Grid Properties).

Tabellendatenmodus

Beschreibung

DataTable

Modus für eine generische Tabelle, die alle nichtstrukturellen Datenquellen in Datentabelle (DataTable) oder RecordSet akzeptiert. Wird hauptsächlich zur manuellen Programmierung von Code-Behind verwendet.

ItemList

Unterstützt jeden Array-Typ als Datenquelle. Kann an Feldausdruck (FieldExpression) gebunden sein und nutzt das reguläre Laden von Werten vom Server. Kann auch im ungebundenen Modus (z. B. ohne Feldausdruck) verwendet werden. Im ungebundenen Modus muss der Elementtyp definiert werden; das Laden und Einreichen von Daten muss im Code implementiert werden. Paging mit Server ist nicht erlaubt.

SubEntityList

Von Elementliste (ItemList) geerbt. Unterstützt einen besonderen Bindemodus für Elementtypen, die aus der SubEntity-Klasse abgeleitet wurden.

SelectionValues

Verwendet einen Feldausdruck, um die Spalten und Werte der Tabelle aufzulösen. Von Datentabelle geerbt. Arbeitet im gebundenen Modus. Lädt Daten per Auswahlwertanforderung. Unterstützt die Einreichung von Daten, die auf Basis ausgewählter Zeilen erstellt wurden. Zwei Feldausdrücke können verwendet werden – einer zum Laden von Daten und einer zum Einreichen.

PredefinedQuery

Referenziert eine Designer- oder Benutzerabfrage. Von Datentabelle geerbt. Erlaubt dynamischen Abfragenamen zur Laufzeit.

AdHocQuery

Die Abfrage wird direkt auf der virtuellen Seite gespeichert. Von Datentabelle geerbt. Erlaubt dynamischen Abfragetext zur Laufzeit.
