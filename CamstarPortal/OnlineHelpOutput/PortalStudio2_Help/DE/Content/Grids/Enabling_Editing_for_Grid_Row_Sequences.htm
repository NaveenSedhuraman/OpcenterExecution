Aktivieren von Änderungen der Zeilenreihenfolge

Sie haben die Möglichkeit, Änderungen an der Reihenfolge der Zeilen in "Elementlistentabellen" (ItemList Grids) oder "Unterobjektlistentabellen" (SubentityList Grids) im Portal zu aktivieren, indem Sie die Tabellen-Steuerelemente in Portal Studio ändern. Durch entsprechende Änderung der Tabellen können Sie das Ziehen und Ablegen von Zeilen, das Auswählen und Neuanordnen von Zeilen mit den Schaltflächen zum Verschieben von Zeilen (Move Row) oder auch beides aktivieren.

Beim Aktivieren von Änderungen der Zeilenreihenfolge

Achten Sie darauf, dass das Steuerelement "Dateneinreichungsmodus" (DataSubmissionMode) auf "Geändert" (Changed) festgelegt ist. Weitere Informationen finden Sie unter "Beschreibungen der Steuerelementeigenschaften".

Definieren Sie die Eigenschaften "Sequenzspalte" (SequenceColumn) und "Sequenzmodus" (SequenceMode), um die Änderung der Zeilenreihenfolge zu ermöglichen. Weitere Informationen finden Sie unter "Tabelleneinstellungseditor (Grid Settings Editor)".

Durch die Aktivierung von Änderungen der Zeilenreihenfolge ändert sich die Darstellung der Tabelle im Portal. Wenn Sie für eine Tabelle Ziehen und Ablegen festlegen, wird neben jeder verschiebbaren Zeile ein Doppelpfeilsymbol angezeigt. Wenn Sie für eine Tabelle das Verschieben mit Pfeilen festlegen, werden in der Symbolleiste der Tabelle Schaltflächen zum Verschieben von Zeilen (Move Row) angezeigt.

Diese Tabelle definiert die verfügbaren Einstellungen für die Änderungen der Reihenfolge von Tabellenzeilen.

Einstellung

Definition

Kein (None)

Deaktiviert die Änderung der Reihenfolge von Tabellenzeilen.

Dragging

Aktiviert das Neuanordnen der Tabellenzeilen durch Ziehen (Dragging) und Ablegen.

ArrowMoving

Aktiviert das Neuanordnen der Tabellenzeilen mit Schaltflächen zum Verschieben von Zeilen (Move Row).

DraggingAndMoving

Aktiviert sowohl Ziehen und Ablegen als auch die Verwendung der Schaltflächen zum Verschieben von Zeilen (Move Row).

Vorgehensweise zum Aktivieren von Änderungen der Zeilenreihenfolge

Gehen Sie folgendermaßen vor, um Änderungen der Zeilenreihenfolge zu aktivieren:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Komponentenkatalog ausgeblendet ist. Der Komponentenkatalog wird angezeigt.
3. Doppelklicken Sie auf eine beliebige Portalseite, die eine Elementlistentabelle (ItemList) oder Unterobjektlistentabelle (SubentityList) enthält. Die Seite wird im Arbeitsbereich angezeigt.
4. Klicken Sie auf Eigenschaften anzeigen/ausblenden (Show/Hide Properties), wenn der Bereich "Eigenschaften" (Properties) ausgeblendet ist. Der Bereich "Eigenschaften" (Properties) wird angezeigt.
5. Wählen Sie auf der Seite ein Tabellen-Steuerelement aus. Die Einstellungen für das Steuerelement werden im Bereich "Eigenschaften" (Properties) angezeigt.
Note: Gewährleisten Sie, dass der "Dateneinreichungsmodus" (DataSubmissionMode) für die ausgewählte Tabelle auf "Geändert" (Changed) festgelegt ist.
6. Klicken Sie im Bereich "Eigenschaften" (Properties) auf die Schaltfläche Mehr (More) für das Feld Einstellungen (Settings). Das Popup "Tabelleneinstellungseditor" (Grid Settings Editor) wird angezeigt.
7. Erweitern Sie Layout.
8. Wählen Sie im Feld Sequenzspalte (SequenceColumn) im Tabellen-Steuerelement die Spalte Index (--index--) aus.
9. Wählen Sie eine Einstellung für Sequenzmodus (SequenceMode) aus.
10. Klicken Sie auf OK.
11. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
