Einfrieren von Spalten

Sie können Spalten einfrieren, die im Portal in Tabellen angezeigt werden. Ändern Sie dazu das Tabellen-Steuerelement der Spalten in Portal Studio. Eingefrorene Spalten bleiben sichtbar, wenn ein Bildlauf für die übrige Tabelle durchgeführt wird.

Beim Einfrieren von Spalten

Verwenden Sie zum Einfrieren von Spalten das Popup "<Tabellenname> – Spalten" (Columns) im Tabelleneinstellungseditor. Sie müssen Spalten in der Reihenfolge, in der sie angezeigt werden, von oben nach unten und ohne Lücken einfrieren.

Tabellen, die Untertabellen oder Zeilengruppen enthalten oder für die Strukturansicht festgelegt ist, können keine eingefrorenen Spalten enthalten. Tabellen, die Untertabellen sind, können eingefrorene Spalten enthalten. Eingefroren Spalten können nicht inline im Portal bearbeitet werden.

Vorgehensweise zum Einfrieren von Spalten

Gehen Sie folgendermaßen vor, um Spalten einzufrieren:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Komponentenkatalog ausgeblendet ist. Der Komponentenkatalog wird angezeigt.
3. Doppelklicken Sie auf eine beliebige Portalseite, die eine Tabelle enthält. Die Seite wird im Arbeitsbereich angezeigt.
4. Klicken Sie auf Eigenschaften anzeigen/ausblenden (Show/Hide Properties), wenn der Bereich "Eigenschaften" (Properties) ausgeblendet ist. Der Bereich "Eigenschaften" (Properties) wird angezeigt.
5. Wählen Sie auf der Seite ein Tabellen-Steuerelement aus. Die Einstellungen für das Steuerelement werden im Bereich "Eigenschaften" (Properties) angezeigt.
6. Klicken Sie im Bereich "Eigenschaften" (Properties) auf Mehr (More) für Einstellungen (Settings). Das Popup "Tabelleneinstellungseditor" (Grid Settings Editor) wird angezeigt.
7. Klicken Sie auf Mehr (More) für Spalten (Columns). Das Popup "<Tabellenname> – Spalten" (Columns) wird angezeigt.
8. Wählen Sie eine Spalte aus.
9. Wählen Sie für Eingefroren (Frozen) den Eintrag Wahr (True) aus.
10. Wiederholen Sie die Schritte 8 bis 9 für jede sequenzielle Spalte, die Sie einfrieren möchten.
11. Klicken Sie zweimal auf OK. Das Popup "Tabelleneinstellungseditor" (Grid Settings Editor) wird geschlossen.
12. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
