Konfigurieren von Portaleinstellungen

In Portal Studio können Sie die folgenden Einstellungen anzeigen, ändern und speichern:

• Standardwert (Default)
• Studio
• Domäne (Domain)
• Thema
• Sprache
• Zeitzone (Time Zone)
• Aktuelle Kultur (Current Culture)
• Tabelle (Grid)
• Suchen (Search)
• Smart-Scanning (Smart Scanning)
• Zwischenspeicherung (Caching)
• Intelligenz (Intelligence)
• CDO-Formulare (CDO Forms)
• Gruppen exportieren (Export Groups)
• Statusnachricht (Status Message)
• Vorlage (Template)
• Darstellung (Appearance)

Beim Konfigurieren von Portaleinstellungen

Die Portal Studio-Einstellungen werden im folgenden Dateipfad gespeichert:

C:\Programme (x86)\Camstar\Camstar Portal\User\Settings.xml

Hinweis: Sie können die Standardzeitüberschreitung ändern. Geben Sie dazu in die web.config-Datei einen neuen Wert für die Eigenschaft "Sitzungszustand" (Session State) ein.

DefaultTheme

Bestimmt, ob das Standardthema für das Portal das Stylesheet "Klassisch" oder "Horizon" verwendet. Die Standardeinstellung ist "Horizon".

"Horizon" ist das Standardthema, wenn Sie sich beim Portal mithilfe des Standard- oder mobilen Einstiegsmodus anmelden. "Klassisch" ist das Standardthema, wenn Sie den klassischen Einstiegsmodus verwenden. Sie sollten das Standardthema nicht ändern, nachdem Sie sich beim Portal mit einem dieser Einstiegsmodi angemeldet haben. Das entsprechende Standardthema ist für die Verwendung mit dem jeweiligen Einstiegsmodus vorgesehen. Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics-Benutzerhandbuch – Erste Schritte oder im Opcenter Execution Core-Benutzerhandbuch – Erste Schritte.

Hinweis: Standardmäßig wird der Text auf allen Fertigungsseiten in einem größeren Schriftgrad angezeigt, wenn "DefaultTheme" auf "Horizon" festgelegt ist und Sie die Seiten auf einem Computer anzeigen. Die Seiten "Modellierung", "Änderungsverwaltung" und "Export/Import" werden standardmäßig in der Standardschriftgröße angezeigt. Die Schriftgröße auf Seitenebene wird über die CssClass-Seiteneigenschaft festgelegt.

Vorgehensweise zum Konfigurieren von Portaleinstellungen

Gehen Sie folgendermaßen vor, um Portaleinstellungen zu konfigurieren:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf die Schaltfläche Portaleinstellungen (Portal Settings).Das Popup "Portaleinstellungen" (Portal Settings) wird angezeigt.
3. Wählen Sie die Einstellung aus, die Sie anzeigen oder ändern möchten. Die Eigenschaften dieser Einstellung werden angezeigt.
4. Wählen Sie eine Einstellung und nehmen Sie die gewünschten Änderungen vor.
5. Klicken Sie auf OK. Das Popup "Portaleinstellungen" (Portal Settings) wird geschlossen, und Ihre Änderungen werden gespeichert.
