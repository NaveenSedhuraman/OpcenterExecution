Konfigurieren von bedingten Tabellenzeilenfarben

Sie können Tabellenzeilen so konfigurieren, dass CSS-Stilattribute wie ein farbiger Hintergrund oder farbiger Text angezeigt werden, wenn benutzerdefinierte Bedingungen erfüllt sind. Z. B. könnte für die Zeilen festgelegt werden, dass ein roter Hintergrund angezeigt wird, wenn ein Container den Status "Angehalten" (On Hold) hat.

Beim Konfigurieren von bedingten Tabellenzeilenfarben

Optional können Sie festlegen, dass in Tabellenzeilen eine andere Hintergrundfarbe oder Textfarbe angezeigt wird, wenn sowohl die Bedingung erfüllt als auch die Zeile ausgewählt ist. Die Hintergrundfarbe für die ausgewählten Zeilen wird im Feld "Ausgewählter Stil" (SelectedStyle) definiert.

Wenn Inlinebearbeitung für eine Tabelle erfolgt, werden Ausdrücke ausgewertet, nachdem Sie eine Zeile aktualisiert oder mit der Tabulatortaste verlassen haben.

Ein Ausdruck kann mehrere, mit den Operatoren "Und" (And) und "Oder" (Or) verbundene Bedingungen enthalten. Ein gültiger bedingter Ausdruck ist z. B.:

"DataType = 'Integer' and AttributeValue > 10"

Diese Tabelle definiert einige allgemeine Bedingungen für die Auswertung eines im Feld "Bedingung" (Condition) eingegebenen Ausdrucks:

Bedingung

Definition

>

Größer als (Greater than).

<

Kleiner als (Less than).

=

Ist gleich (Equals).

Like

Wenn das Feld eine Zeichenkette (String) enthält. Die erwartete Syntax ist: "Like 'STRING' "
Note: Das Feld "Bedingung" (Condition) erwartet die Syntax der Microsoft DataColumn.Expression. Weitere Informationen zu zusätzlichen Bedingungsoptionen finden Sie im Microsoft Developer Network.

Die Felder "Normaler Stil" (NormalStyle) und "Ausgewählter Stil" (SelectedStyle) erwarten Stylesheet-Referenzen oder Stylesheet-Syntax zum Definieren der Zeilen- oder Textfarbe. Die erwartete Syntax lautet: "class:<css-class-name>" wobei <css-class-name> der Name einer in einem Stylesheet definierten CSS-Klasse ist. Ein gültiger Wert ist z. B. "background: red".

Note: Sie müssen den Indikator "!important" für Hintergrundfarben einschließen, wenn Sie eine Klasse für NormalStyle oder SelectedStyle verwenden.

Felddefinitionen des Popups "Bedingte Zeilenstile" (ConditionalRowStyles)

Diese Tabelle definiert die Felder im Popup "Bedingte Zeilenstile" (ConditionalRowStyles).

Feld

Definition

Typ

Condition

Bedingter Ausdruck, der die Hintergrundfarbe der Zeile ändert, wenn der Ausdruck als "wahr" (true) ausgewertet wird.

Erforderlich

NormalStyle

Die Hintergrundfarbe der Zeile, wenn der bedingte Ausdruck "wahr" (true) ist. Z. B.: background:red;color:yellow wobei "color" (Farbe) die Angabe für den Text ist. Hinweis: Der Benutzer kann in der Zeile für jeden Stil entweder nur "background", nur "color" oder beides definieren.

Erforderlich

SelectedStyle

Die Hintergrundfarbe der Zeile, wenn der bedingte Ausdruck "wahr" (true) und die Zeile ausgewählt ist. Z. B.: background:red;color:yellow wobei "color" (Farbe) die Angabe für den Text ist. Hinweis: Der Benutzer kann in der Zeile für jeden Stil entweder nur "background", nur "color" oder beides definieren.

Vorgehensweise zum Konfigurieren von bedingten Tabellenzeilenfarben

Gehen Sie folgendermaßen vor, um bedingte Tabellenzeilenfarben zu konfigurieren:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Komponentenkatalog ausgeblendet ist. Der Komponentenkatalog wird angezeigt.
3. Doppelklicken Sie auf eine Seite mit einem Tabellen-Steuerelement. Die Seite wird im Arbeitsbereich angezeigt.
4. Klicken Sie auf Eigenschaften anzeigen/ausblenden (Show/Hide Properties), wenn der Bereich "Eigenschaften" (Properties) ausgeblendet ist. Der Bereich "Eigenschaften" (Properties) wird angezeigt.
5. Wählen Sie ein Tabellen-Steuerelement (Grid Control) aus. Die Eigenschaften für das Tabellen-Steuerelement werden im Bereich "Eigenschaften" (Properties) angezeigt.
6. Klicken Sie auf Mehr (More) für Einstellungen (Settings). Das Popup "Tabelleneinstellungseditor" (Grid Settings Editor) wird angezeigt.
7. Erweitern Sie Layout.
8. Klicken Sie auf Mehr (More) für ConditionalRowStyles. Das Popup "Bedingte Zeilenstile" (ConditionalRowStyles) wird angezeigt.
9. Klicken Sie auf Hinzufügen. Eine Stilregel (StyleRule) wird zur Liste hinzugefügt und im rechten Bereich werden leere Felder angezeigt.
10. Geben Sie einen bedingten Ausdruck in das Feld Bedingung (Condition) ein.
11. Geben Sie eine Hintergrundfarbe in das Feld NormalStyle ein.
12. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
13. Wiederholen Sie die Schritte 9-12, um zusätzliche Bedingungen für Tabellenzeilen hinzufügen.
14. Klicken Sie auf OK. Das Popup "Bedingter Zeilenhandler" (ConditionalRowHandler) wird geschlossen.
15. Klicken Sie auf OK. Das Popup "Tabelleneinstellungseditor" (Grid Settings Editor) wird geschlossen.
16. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
