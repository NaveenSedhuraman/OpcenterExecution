Webpart-Eigenschaften

Über Webpart-Eigenschaften können Sie die Eigenschaften von Webparts anzeigen oder bearbeiten. Webpart-Eigenschaften werden im Bereich "Eigenschaften" (Properties) angezeigt, wenn im Arbeitsbereich ein Webpart ausgewählt ist. Weitere Informationen zum Bereich "Eigenschaften" (Properties) finden Sie unter "Fensterbereich "Eigenschaften" (Properties)".

Wenn Sie eine Eigenschaft auswählen, wird unterhalb des Fensterbereichs "Eigenschaften" (Properties) ein Beschreibungsfenster mit Informationen zu Webpart-Eigenschaften angezeigt.

Note: Einige Eigenschaften enthalten Untereigenschaften, die die weitere Anpassung ermöglichen. Untergeordnete Eigenschaften werden in einem separaten Popup angezeigt, wenn Sie auf die Schaltfläche "Mehr" (More) für deren übergeordnete Eigenschaft klicken.

Beschreibungen der Webpart-Eigenschaften

Diese Tabelle enthält die Beschreibungen der Webpart-Eigenschaften.

Webpart-Eigenschaft

Beschreibung der Eigenschaft

Actions

Aktionen des Webparts. Ermöglicht das Definieren von Aktionen und Schaltfeldern auf der Webpart-Ebene.

Hinweis: Wenn Sie auf die Schaltfläche "Mehr" (More) klicken, wird das modale Fenster "Actions" angezeigt. Weitere Informationen zu den Eigenschaften, die in diesem modalen Fenster angezeigt werden, finden Sie unter "Beschreibungen der Eigenschaften des Typs "Actions"".

BroadcastInitiators

Wird zum Erstellen von Initiatoren verwendet, die Verbindungsdatenverträge zwischen Webparts auf der Seitenebene auslösen.

Hinweis: Wenn Sie auf die Schaltfläche "Mehr" (More) klicken, wird das modale Fenster "BroadcastInitiators" angezeigt. Weitere Informationen zu den Eigenschaften, die in diesem modalen Fenster angezeigt werden, finden Sie unter "Beschreibungen der Eigenschaften des Typs "BroadcastInitiators"".

ClientPropertyConditions

Hier können Sie Daten auf Basis eines Bedingungsergebnisses von einem browserseitigen Ereignis auf eine Eigenschaft festlegen.

CollapsibleGroups

Einstellung der Unterstützung in einem kommenden Release.

Hinweis: Zum Erstellen von reduzierbaren Abschnitten müssen Sie die Eigenschaft "Anzeigemodus umschalten" (ToggleDisplayMode) des Umschaltcontainer-Steuerelements (Toggle Container) verwenden. Weitere Informationen finden Sie unter "Steuerelementeigenschaften".

Control

Referenziert ein Steuerelement auf der Seite oder auf dem Webpart.

Hinweis: Wenn Sie auf die Schaltfläche "Mehr" (More) klicken, wird das modale Fenster "Control" angezeigt. Das modale Fenster "Control" verwendet dieselben Eigenschaften wie Steuerelemente. Weitere Informationen zu den Eigenschaften, die in diesem modalen Fenster angezeigt werden, finden Sie unter "Steuerelementeigenschaften".

Created

Datum und Uhrzeit der Erstellung des Webparts.

CreatedBy

Ersteller des Webparts.

CSSClass

Stylesheet-Referenz für das Webpart.

DataContractMembers

Datenverträge, die für die Kommunikation zwischen Seiten verwendet werden.

Hinweis: Wenn Sie auf die Schaltfläche "Mehr" (More) klicken, wird das modale Fenster "DataContractMembers" angezeigt. Weitere Informationen zu den Eigenschaften, die in diesem modalen Fenster angezeigt werden, finden Sie unter "Beschreibungen der Eigenschaften des Typs "DataContractMembers"".

DataSubmissionMode

Legt fest, wie ein Feld reagiert, wenn eine Seite Daten an den Server übermittelt. Operanden schließen ein:

• NotSet: Modus nicht festgelegt. Dies ist der Standardmodus.
• Force: Der Wert ist immer in den übermittelten Daten enthalten.
• Changed: Der Wert wird nur übermittelt, wenn er geändert wurde.
• Filled: Der Wert wird nur übermittelt, wenn das Feld nicht leer ist.
• ChangedAndFilled: Der Wert wird nur übermittelt, wenn das Feld nicht leer ist und der Wert geändert wurde.
• Skip: Der Wert wird nicht übermittelt oder geladen.
• ViewOnly: Der Wert wird nicht übermittelt.
• Skip Request: Der Wert wird nicht geladen. Die Auswahlwerte für das Steuerelement werden nicht geladen.

Wichtig: Siemens empfiehlt, für Modellierungsseiten die Option "Geändert" (Changed) zu verwenden.

DirtyFlagTrigger

Boolesches Feld, das die Seite als "Geändert" (Dirty) markiert, wenn ein Feld innerhalb des Webparts bei der Einstellung "Wahr" (True) seinen Wert ändert.

Hinweis: Diese Eigenschaft wirkt sich derzeit nur auf Webparts innerhalb des Modellierungs-Frameworks aus.

DisplayBorder

Zeigt einen Rahmen für das Webpart an.

DisplayFooter

Zeigt eine Fußzeile für das Webpart an.

DisplayHeader

Zeigt eine Kopfzeile für das Webpart an.

EventHandlers

Ermöglicht die Angabe eines Handlers von einem Ereignis.

Hinweis: Wenn Sie auf die Schaltfläche "Mehr" (More) klicken, wird das modale Fenster "EventHandler" angezeigt. Weitere Informationen zum Definieren von Ereignishandlern finden Sie unter "Definieren von Ereignishandlern".

Height

Höhe in Pixel für die Anzeige des Webparts.

Hidden

Wenn "Wahr" (True), wird das Steuerelement oder Webpart in der HTML gerendert, ist jedoch für den Benutzer nicht sichtbar. Z. B. Anzeige "Kein" (None) im HTML-Tag.

HorizontalAlignment

Vertikale Ausrichtung der Elemente des Webparts.

HowLong

Zeitraum seit der Erstellung des Webparts.

LastModified

Datum der letzten Änderung des Webparts.

LastModifiedBy

Benutzer, der das Webpart zuletzt geändert hat.

Layout

Visuelle Darstellung des Webparts. Siemens empfiehlt, diese Eigenschaft nicht direkt zu ändern.

Margin

Rand des Webparts.

Padding

Textabstand des Webparts.

PrimaryServiceType

Servicetyp für Steuerelemente, die keinen Servicetyp angeben.

PropertyConditions

Bietet die Möglichkeit, den Eigenschaftswert eines Webparts als Ergebnis eines ausgewerteten Ausdrucks (einer Bedingung) zu definieren. Die Eigenschaft wird ausgewertet, wenn die Seite zur Laufzeit ausgeführt wird.

Hinweis: Wenn Sie auf die Schaltfläche "Mehr" (More) klicken, wird das Fenster "<Komponente> – Eigenschaftsbedingungen" (<Komponente> – Property Conditions) angezeigt. Weitere Informationen zum Definieren von Eigenschaftsbedingungen finden Sie unter "Eigenschaftsbedingungen".

SessionVariables

Wird verwendet, um Sitzungsvariablen auf Seitenebene zu konfigurieren.

Weitere Informationen zum Definieren von Sitzungsvariablen finden Sie unter "Definieren von Sitzungsvariablen".

Title

Der Text, der als Titel des Webparts angezeigt wird. Der hier eingegebene Text hat Vorrang vor dem Feld "Titelbeschriftung" (TitleLabel), wenn sowohl Text eingegeben als auch eine Beschriftung ausgewählt wird.

TitleLabel

Wählt eine Beschriftung aus, die als Titel des Webparts verwendet werden soll. Weitere Informationen zur Verwendung des Popups "Beschriftungssuche" (Label Lookup) finden Sie unter "Suchen nach Seitenaktionsbeschriftungen".

TypeName

Code-Behind-Typ des Webparts.

VerticalAlignment

Vertikale Ausrichtung der Elemente des Webparts.

Width

Breite in Pixel für die Anzeige des Webparts.
