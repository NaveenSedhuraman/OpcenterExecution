Hinzufügen von SeriesSettings

Sie können die Darstellung von Datenpunkten in Streu- und Wafermap-Diagrammen durch Hinzufügen von SeriesSettings zu Ihrem Chart-Steuerelement anpassen.

Beim Hinzufügen von SeriesSettings

SeriesSettings sind für ein Chart-Steuerelement spezifische Eigenschaften, die Ihnen das Angeben dieser Eigenschaften für eine Serie von Datenpunkten ermöglichen:

• Farbe
• Form
• Größe

Sie können die SeriesSettings dem Schlüssel der Datenpunktserie zuweisen. Sie können mehrere SeriesSettings hinzufügen, um eine eindeutige Datenpunktdarstellung für jede Serie festzulegen, die an ein Chart-Steuerelement übergeben wird.

Sie können SeriesSettings hinzufügen, ohne die Einstellungen zu bestimmten Serien von Datenpunkten zuzuweisen. Die Anwendung weist die SeriesSettings automatisch in der Reihenfolge zu Datenpunktserien zu, in der sie an das Chart-Steuerelement übergeben wurden, wenn Sie das Feld "Schlüssel" (Key) leer lassen.

Note: Sie müssen die Eigenschaft "ChartMode" des Chart-Steuerelements auf "Streudiagramm" (Scatterplot) einstellen, wenn Sie SeriesSettings hinzufügen.
Felddefinitionen des Popups "SeriesSettings"

Diese Tabelle definiert die Felder im Popup "SeriesSettings".

Feld

Definition

Typ

Allgemein

Color

Legt die Farbe der Datenpunkte in der Serie fest. Die Farben müssen als Hexadezimalwert eingegeben werden. Die Standardeinstellung ist #FF0000.

Optional

Key

Weist die zugehörigen Eigenschaften für die Farbe, Form und Größe einer Serie von Datenpunkten zu. Die Eigenschaften für die Farbe, Form und Größe werden auf die Serie in der Reihenfolge angewendet, in der sie an das Chart-Steuerelement übergeben wurden, wenn kein Schlüssel eingegeben ist.

Optional

Shape

Legt die Form der Datenpunkte in der Serie fest. Mögliche Optionen sind "Kreis" (Circle), "Kreuz" (Cross), "Diamant" (Diamond), "Quadrat" (Square), "TriangleDown" und "TriangleUp".

Optional

Size

Legt die Größe der Datenpunkte in der Serie fest. Die Größe kann zwischen 1 und 100 liegen.

Optional

Vorgehensweise zum Hinzufügen von SeriesSettings

Gehen Sie folgendermaßen vor, um SeriesSettings hinzuzufügen.

1. Öffnen Sie eine Seite oder ein Webpart, die bzw. das ein Chart-Steuerelement enthält. Die Seite bzw. das Webpart wird im Arbeitsbereich angezeigt.
2. Klicken Sie auf Eigenschaften anzeigen/ausblenden (Show/Hide Properties), wenn der Bereich "Eigenschaften" (Properties) ausgeblendet ist. Der Bereich "Eigenschaften" (Properties) wird angezeigt.
3. Wählen Sie im Arbeitsbereich ein Chart-Steuerelement aus. Im Bereich "Eigenschaften" (Properties) werden die dem Chart-Steuerelement zugeordneten Eigenschaften angezeigt.
4. Klicken Sie auf Mehr (More) für SeriesSettings. Das Popup "<Steuerelement> – SeriesSettings" wird angezeigt.
5. Klicken Sie auf Hinzufügen. Eine neue SeriesSetting wird zur Liste hinzugefügt und ihre allgemeinen Einstellungen werden im rechten Bereich angezeigt.
6. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
7. Wiederholen Sie die Schritte 5-6, um zusätzliche SeriesSettings hinzuzufügen.
8. Klicken Sie auf OK. Das Popup "<Steuerelement> – SeriesSettings" wird geschlossen.
9. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
