Seitenfolge-Eigenschaften

Über die Seitenfolge-Eigenschaften können Sie die Eigenschaften von Portal Studio-Seitenfolgen anzeigen oder bearbeiten. Seitenfolge-Eigenschaften werden im Fensterbereich "Eigenschaften" (Properties) angezeigt, wenn im Arbeitsbereich eine Seitenfolge ausgewählt ist. Weitere Informationen zum Bereich "Eigenschaften" (Properties) finden Sie unter "Bereich 'Eigenschaften' (Properties)".

Einzelne Seitenknoten, die in Seitenfolgen angezeigt werden, enthalten Eigenschaften, mit denen Sie die Funktionalität von Seiten anzeigen oder bearbeiten können, die in der Seitenfolge angezeigt werden. Weitere Informationen zu den Eigenschaften, die im Fensterbereich "Eigenschaften" (Properties) angezeigt werden, wenn ein Seitenknoten in einer Seitenfolge ausgewählt wird, finden Sie unter "Beschreibungen der Eigenschaften von Seitenfolge-Seitenknoten".

Wenn Sie eine Eigenschaft auswählen, wird unterhalb des Fensterbereichs "Eigenschaften" (Properties) ein Beschreibungsfenster mit Informationen zu den Eigenschaften der Seitenfolge und des Seitenfolge-Seitenknotens angezeigt.

Note: Einige Eigenschaften enthalten Untereigenschaften, die die weitere Anpassung ermöglichen. Untergeordnete Eigenschaften werden in einem separaten Popup angezeigt, wenn Sie auf die Schaltfläche "Mehr" (More) für deren übergeordnete Eigenschaft klicken.
Beschreibungen der Seitenfolge-Eigenschaften

Diese Tabelle enthält Beschreibungen der Seitenfolge-Eigenschaften.

Seiteneigenschaft

Beschreibung der Eigenschaft

Created

Erstellungsdatum und -uhrzeit der Seitenfolge.

CreatedBy

Ersteller der Seitenfolge.

Description

Kurzbeschreibung der Seitenfolge.

HowLong

Zeitraum seit dem Erstellen der Seitenfolge.

LastModified

Datum der letzten Änderung der Seitenfolge.

LastModifiedBy

Benutzer, der die letzte Änderung an der Seitenfolge vorgenommen hat.

TypeName

Zugrunde liegender Code der Seitenfolge.
Beschreibungen der Eigenschaften von Seitenfolge-Seitenknoten

Diese Tabelle enthält Beschreibungen der Seitenknoteneigenschaften, die angezeigt werden, wenn Sie einen Seitenknoten innerhalb einer Seitenfolge auswählen.

Seiteneigenschaft

Beschreibung der Eigenschaft

IsStartPage

Legt die Seite als Startseite in der Seitenfolge fest, sofern auf "Wahr" (True) festgelegt.

Hinweis: Es kann nur eine Seite als Startseite innerhalb einer einzelnen Seitenfolge festgelegt werden.

Location

Position des Seitenknotens im Seitenfolgediagramm, wenn eine Seitenfolge im Arbeitsbereich angezeigt wird. Diese Eigenschaft verwendet das Format "x,y", wenn die Position eines Seitenknotens festgelegt wird.

PageDescriptionLabelName

Wählt eine Beschriftung als Beschreibung unter dem Seitentitel auf den Registerkarten der Seitenfolge aus. Weitere Informationen zur Verwendung des Popups "Beschriftungssuche" (Label Lookup) finden Sie unter "Suchen nach Seitenaktionsbeschriftungen".

PageDescriptionLabelText

Text, der als Beschreibung unter dem Seitentitel auf den Registerkarten der Seitenfolge angezeigt wird. Der hier eingegebene Text hat Vorrang vor der Eigenschaft "PageDescriptionLabelName", wenn sowohl Text eingegeben als auch eine Beschriftung ausgewählt wird.

PageTitleLabelName

Wählt eine Beschriftung als Titel der Seite auf den Registerkarten der Seitenfolge aus. Weitere Informationen zur Verwendung des Popups "Beschriftungssuche" (Label Lookup) finden Sie unter "Suchen nach Seitenaktionsbeschriftungen".

PageTitleLabelText

Text, der als Titel der Seite auf den Registerkarten der Seitenfolge angezeigt wird. Der hier eingegebene Text hat Vorrang vor der Eigenschaft "PageTitleLabelName", wenn sowohl Text eingegeben als auch eine Beschriftung ausgewählt wird.
