Hinzufügen von Zielen zu einer Seitenzuordnung

Ziele sind das Zielseiten-, Seitenfolge- oder das separate Seitenzuordnungselement, zu dem Sie nach dem Anwenden der Logik weitergeleitet werden.

Felddefinitionen des modalen Fensters "Targets"

Diese Tabelle definiert die Felder im modalen Fenster "Targets".

Feld

Definition

Typ

Conditions

Ermöglicht es Ihnen, das Ziel mit Seitenzuordnungs-Filterbedingungen zu verbinden.

Hinweis: Wenn Sie auf die Schaltfläche "Mehr" (More) klicken, wird das modale Fenster "Conditions" angezeigt. Weitere Informationen zu den Eigenschaften, die in diesem modalen Fenster angezeigt werden, finden Sie unter "Felddefinitionen des modalen Fensters "Conditions"".

Optional

Links

Ermöglicht es Ihnen, für das Ziel Datenvertragsverknüpfungen für UI-Komponenten zu definieren.

Hinweis: Wenn Sie auf die Schaltfläche "Mehr" (More) klicken, wird das modale Fenster "Links" angezeigt. Weitere Informationen zu den Eigenschaften, die in diesem modalen Fenster angezeigt werden, finden Sie unter "Felddefinitionen des modalen Fensters "Links"".

Optional

Target

Wählt ein Zielseiten-, Seitenfolge- oder Seitenzuordnungselement aus und legt es als Ziel fest.

Erforderlich

TargetType

Wählt einen Zieltyp aus. Die Standardeinstellung ist "Page".

Hinweis: Durch Auswahl eines neuen Zieltyps wird das Feld "Target" gelöscht. Die im Feld "Target" verfügbaren Optionen hängen vom ausgewählten Zieltyp ab.

Erforderlich
Felddefinitionen des modalen Fensters "Conditions"

Diese Tabelle definiert die Felder im modalen Fenster "Conditions".

Feld

Definition

Typ

ConditionType

Legt einen Bedingungstyp für den Bedingungswert (ConditionValue) fest, z. B. "Equal", "NonEqual" oder "Contains".

Optional

ConditionValue

Legt einen Bedingungswert fest, der zusammen mit dem Bedingungstyp für das Ziel ausgewertet wird.

Optional

Parameter

Legt einen Parameter fest, der bei der Auswertung mit der Bedingung verbunden werden soll.

Optional

Vorgehensweise zum Hinzufügen von Zielen zu einer Seitenzuordnung

Gehen Sie folgendermaßen vor, um Ziele zu einem Zuordnungselement hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Wählen Sie die Registerkarte Modellierung (Modeling) im Komponentenkatalog aus und doppelklicken Sie auf eine Seitenzuordnung. Die Seitenzuordnung wird im Arbeitsbereich angezeigt.
3. Klicken Sie auf Mehr (More) für Targets. Das modale Fenster "Targets" wird angezeigt.
4. Klicken Sie auf Hinzufügen. Ein Ziel wird zu der Liste hinzugefügt.
5. Wählen Sie im Feld Zieltyp (TargetType) einen Zieltyp aus.
6. Wählen Sie im Feld Ziel (Target) ein Ziel aus.
7. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
8. Wiederholen Sie die Schritte 4 bis 7, um weitere Ziele hinzuzufügen.
9. Klicken Sie auf OK. Das modale Fenster "Targets" wird geschlossen.
10. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
