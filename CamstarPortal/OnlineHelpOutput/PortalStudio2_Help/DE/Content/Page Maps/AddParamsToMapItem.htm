Hinzufügen von Seitenzuordnungsparametern zu einer Seitenzuordnung

Seitenzuordnungsparameter werden zum Konfigurieren von Bedingungen verwendet, unter denen Sie die Anwendung auf eine Zielseite umleitet. Seitenzuordnungsparameter dienen als Vermittler zwischen Quell- und Zielseiten.

Beim Hinzufügen von Seitenzuordnungsparametern zu einer Seitenzuordnung

Ein Parameter muss für eine Seitenzuordnung eindeutig sein. Parameter übertragen Daten zwischen Seiten und konfigurieren Bedingungen, wenn eine Umleitung erfolgt.

Felddefinitionen des modalen Fensters "Parameters"

Das modale Fenster "Parameters" enthält nur das Feld "Name". Das Feld "Name" wird verwendet, um einen vorhandenen Parameter auszuwählen und ihn mit der Seitenzuordnung zu verbinden.

Vorgehensweise zum Hinzufügen von Seitenzuordnungsparametern zu einer Seitenzuordnung

Gehen Sie folgendermaßen vor, um Seitenzuordnungsparameter zu einem Zuordnungselement hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Wählen Sie die Registerkarte Modellierung (Modeling) im Komponentenkatalog aus und doppelklicken Sie auf eine Seitenzuordnung. Die Seitenzuordnung wird im Arbeitsbereich angezeigt.
3. Klicken Sie auf Mehr (More) für Parameter. Das modale Fenster "Parameters" wird angezeigt.
4. Klicken Sie auf Hinzufügen. Ein Parameter wird zu der Liste hinzugefügt.
5. Wählen Sie im Feld Name einen Parameter aus.
6. Wiederholen Sie die Schritte 4 bis 5, um weitere Parameter hinzuzufügen.
7. Klicken Sie auf OK. Das modale Fenster "Parameters" wird geschlossen.
8. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
