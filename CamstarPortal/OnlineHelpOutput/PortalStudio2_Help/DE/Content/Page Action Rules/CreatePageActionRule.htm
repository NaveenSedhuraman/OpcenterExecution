Erstellen von Seitenaktionsregeln

Seitenaktionsregeln (Page Action Rules) legen anhand von benutzerdefinierten Ausdrücken fest, wann eine Aktion auf einer Portalseite aktiv ist.

Beim Erstellen von Seitenaktionsregeln

Um Seitenaktionsregeln zu erstellen, definieren Sie im Popup "Seitenaktionsregeln erstellen" (Page Action Rule Builder) mindestens einen Ausdruck. Ausdrücke sind Anweisungen, die angeben, dass eine Seitenaktion angezeigt oder aktiviert werden soll, wenn die Anweisung als wahr ausgewertet wird. Ausdrücke lassen sich problemlos anpassen und Sie können mit Operatoren wie "Und" (And) und "Oder" (Or) innerhalb einer einzigen Seitenaktionsregel mehrere Ausdrücke definieren. Folgende Ausdrucksoptionen werden unterstützt: "Container", "Ressource" (Resource), "Ereignis" (Event) oder "Änderungspaket" (Change Package).

Note: Die "Und"-Operation wird standardmäßig im Popup "Seitenaktionsregeln erstellen" (Page Action Rule Builder) in der ersten Dropdownliste angezeigt; sie wird jedoch nicht zum ersten für eine Seitenaktionsregel definierten Ausdruck hinzugefügt. Die "Und"- oder "Oder"-Operationen werden zu allen nachfolgenden Ausdrücken hinzugefügt, die zur Seitenaktionsregel hinzugefügt werden.

Sie müssen benutzerdefinierte Seitenaktionsregeln zu Seitenaktionen hinzufügen, um das Verhalten der Seitenaktionen anzupassen. Weitere Informationen finden Sie unter "Hinzufügen von Seitenaktionsregeln zu Seitenaktionen".

Felddefinitionen einer Seitenaktionsregel

Diese Tabelle definiert Felder, die zum Erstellen einer Seitenaktionsregel verwendet werden.

Feld

Definition

Typ

Name

Eindeutiger Name für diese Seitenaktionsregel.

Erforderlich

Description

Kurze Beschreibung der Seitenaktionsregel.

Optional

Expression

Für diese Seitenaktionsregel definierte Ausdrücke. Klicken Sie auf die Schaltfläche "More" (Mehr), um einen Ausdruck zu definieren, eine vollständige Liste der zuvor definierten Ausdrücke anzuzeigen oder vorhandene Ausdrücke zu ändern.

Erforderlich

Filter Tags

Gibt ein Filter-Tag an, mit dem Sie die für die Anzeige und Auswahl verfügbaren Elemente begrenzen können.

Notes

Relevante Kommentare zur Seitenaktionsregel.

Vorgehensweise zum Erstellen von Seitenaktionsregeln

Gehen Sie folgendermaßen vor, um Seitenaktionsregeln zu erstellen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Neu (New) und wählen Sie Seitenaktionsregel (Page Action Rule) aus. Im Arbeitsbereich wird eine leere Seitenaktionsregel angezeigt.
3. Geben Sie im Feld Name einen Namen ein.
4. Klicken Sie auf Mehr (More) für Ausdruck (Expression). Das Popup "Seitenaktionsregeln erstellen" (Page Action Rule Builder) wird angezeigt.
5. Wählen Sie Operationen und Werte aus den Dropdownlisten aus oder geben Sie manuell benutzerdefinierte Informationen in die Textfelder ein, um Ihren benutzerdefinierten Ausdruck zu definieren.
6. Klicken Sie auf Hinzufügen. Der Ausdruck wird im Textfeld über den Dropdownlisten und in der Liste der Ausdrücke unter den Dropdownlisten angezeigt.
7. Wiederholen Sie die Schritte 5 bis 6, um weitere Ausdrücke hinzuzufügen.
8. Klicken Sie auf OK. Das Popup "Seitenaktionsregeln erstellen" (Page Action Rule Builder) wird geschlossen.
9. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
