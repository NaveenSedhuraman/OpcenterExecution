Hinzufügen von Menüeinträgen zu einem Menü

Menüeinträge sind die Seiten, auf die Sie über das Menü zugreifen können. Menüeinträge werden im modalen Fenster "MenuItems" definiert. Sie können optional auch Untermenüs und Seitenfolgen aus dem modalen Fenster "MenuItems" hinzufügen.

Beim Hinzufügen von Menüeinträgen zu einem Menü

Sie können Untermenüs mit einem Menü verbinden, nachdem Sie das Portalhauptmenü definiert haben. Jedes definierte Menü kann als Untermenü hinzugefügt werden. Durch das Hinzufügen eines Untermenüs können Sie Transaktionen als Gruppe hinzufügen, sodass Sie nicht jede Transaktion einzeln hinzufügen müssen. Weitere Informationen finden Sie unter "Vorgehensweise zum Hinzufügen eines Untermenüsymbols".

Sie können Untermenüsymbole ändern, indem Sie im modalen Fenster "MenuItems" einen Namen für die Symboldatei in das Feld "ApolloIcon" eingeben. Symboldateien befinden sich im folgenden Verzeichnis: C:\Program Files (x86)\Camstar\Camstar Portal\assets\image. Sie müssen den Dateinamen so eingeben, wie er im Ordner angezeigt wird.

Vorgehensweise zum Hinzufügen von Menüelementen zu einer Menüdefinition

Gehen Sie folgendermaßen vor, um Menüeinträge zu einer Menüdefinition hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Wählen Sie die Registerkarte Navigation im Komponentenkatalog aus und doppelklicken Sie anschließend auf ein Menü. Das Menü wird im Arbeitsbereich angezeigt.
3. Klicken Sie auf Mehr (More) für Menü-Elemente. Das modale Fenster "MenuItems" wird angezeigt.
4. Wählen Sie aus der Dropdownliste Änderungen von Portal-Menübefehlen (PortalMenuCommandChanges) aus und klicken Sie auf Hinzufügen (Add). In der Liste wird ein neues Menüelement und im rechten Bereich werden Felder angezeigt.
5. Geben Sie im Feld Bezeichnung (Caption) eine Bezeichnung für das Menüelement ein.
6. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
7. Wiederholen Sie die Schritte 4-6 für jedes Menüelement, das Sie hinzufügen möchten.
8. Klicken Sie auf OK. Das modale Fenster "MenuItems" wird geschlossen.
9. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Hinzufügen von Untermenüs zu einer Menüdefinition

Gehen Sie folgendermaßen vor, um ein Untermenü zu einer Menüdefinition hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Wählen Sie die Registerkarte Navigation im Komponentenkatalog aus und doppelklicken Sie anschließend auf ein Menü. Das Menü wird im Arbeitsbereich angezeigt.
3. Klicken Sie auf Mehr (More) für Menü-Elemente. Das modale Fenster "MenuItems" wird angezeigt.
4. Wählen Sie Untermenüänderungen im Portalmenü (PortalMenuSubMenuChanges) aus der Dropdownliste aus und klicken Sie auf Hinzufügen (Add). In der Liste wird ein neues Untermenü und im rechten Fensterbereich werden Felder angezeigt.
5. Geben Sie in das Feld Überschrift (Caption) eine Bezeichnung für das Untermenü ein.
6. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
7. Wiederholen Sie die Schritte 4 bis 6 für jedes hinzuzufügende Untermenü.
8. Klicken Sie auf OK.Das modale Fenster "MenuItems" wird geschlossen.
9. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Hinzufügen eines Untermenüsymbols

Gehen Sie folgendermaßen vor, um ein Untermenüsymbol hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Wählen Sie die Registerkarte Navigation im Komponentenkatalog aus und doppelklicken Sie anschließend auf ein Menü. Das Menü wird im Arbeitsbereich angezeigt.
3. Klicken Sie auf Mehr (More) für Menü-Elemente. Das modale Fenster "MenuItems" wird angezeigt.
4. Wählen Sie PortalMenuSubMenuChanges aus der Dropdown-Liste und ein Untermenü aus. Die Felder des Untermenüs werden im rechten Bereich angezeigt.
5. Geben Sie einen Symboldateinamen in das Feld ApolloIcon ein, das der Symbolbilddatei im Verzeichnis C:\Program Files (x86)\Camstar\Camstar Portal\assets\image entspricht.
6. Klicken Sie auf OK.Das modale Fenster "MenuItems" wird geschlossen.
7. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
