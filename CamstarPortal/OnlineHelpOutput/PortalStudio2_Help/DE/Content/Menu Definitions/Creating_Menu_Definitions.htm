Erstellen eines Menüs

Sie können eine Menüstruktur erstellen, die aus mehreren Menüs und Untermenüs besteht.

Beim Erstellen eines Menüs

Menüs werden im Arbeitsbereich als eine Reihe von Feldern angezeigt, wenn Sie ein neues Menü erstellen.

Beachten Sie Folgendes, wenn Sie ein Menü erstellen:

• Die Standardmenüs "csiPortalMenuV8", "csiMobileMenu" und "csiPortalMenu" werden den Benutzern "Administrator", "CamstarAdmin" und "InSite-Admin" automatisch in einer anfänglichen Implementierung von Opcenter EX CR und Opcenter EX MDD oder beim Aktualisieren auf eine neue Version und Aktualisieren der Standardinstanzdaten zugewiesen. Verwenden Sie das Modellierungsobjekt "Mitarbeiter", um Portalmenüs anderen Mitarbeitern zuzuweisen. Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics Portal Modeling-Benutzerhandbuch oder im Opcenter Execution Core Portal Modeling-Benutzerhandbuch.
• Siemens empfiehlt, vor dem Erstellen von Menüs zunächst Registerkarten zu erstellen, da Menüs Registerkarten referenzieren. Weitere Informationen finden Sie unter "Erstellen einer Registerkarte".

Es gibt drei Möglichkeiten, wie Registerkarten von Menüs referenziert werden können. Diese Tabelle beschreibt die verschiedenen Möglichkeiten zum Referenzieren von Registerkarten.

Registerkartenreferenz

Beschreibung

Current Tab

Erbt die Registerkarteneinstellung von der aktuellen Registerkarte.

New Tab

Wird immer in einer neuen leeren Registerkarte im Portal geöffnet. Wenn Sie diese Option auswählen, können Sie mehrere Kopien einer Registerkarte gleichzeitig öffnen.

Specified Tab

Gibt eine bereits erstellte Registerkarte an. Es kann jeweils nur eine Version einer angegebenen Registerkarte gleichzeitig geöffnet sein.

Wenn Sie ein Menü oder ein Untermenü einer angegebenen Registerkarte zuweisen und für alle nachstehenden Elemente den Standardwert "Erben" (Inherit) beibehalten, werden diese alle auf derselben angegebenen Registerkarte geöffnet. Daher ersetzt die zuletzt geöffnete Seite die Seite, die derzeit auf der angegebenen Registerkarte geöffnet ist.

Weitere Informationen finden Sie unter "Vorgehensweise zum Hinzufügen einer angegebenen Registerkarte zu einer Menüdefinition".

Öffnen vorhandener Menüs

Vorhandene Menüs können im Komponentenkatalog geöffnet werden. Weitere Informationen zum Öffnen vorhandener Menüs finden Sie unter "Öffnen vorhandener Komponenten".

Vorgehensweise zum Erstellen einer Menüdefinition

Gehen Sie folgendermaßen vor, um eine Menüdefinition zu erstellen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Neu (New) und wählen Sie Menüdefinition (Menu Definition) aus. Im Arbeitsbereich wird ein leeres Menü angezeigt.
3. Geben Sie in das Feld Name einen Namen für das Menü ein.
4. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
5. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Hinzufügen einer angegebenen Registerkarte zu einer Menüdefinition

Gehen Sie folgendermaßen vor, um eine angegebene Registerkarte zu einer Menüdefinition hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Wählen Sie die Registerkarte Navigation im Komponentenkatalog aus und doppelklicken Sie anschließend auf ein Menü. Das Menü wird im Arbeitsbereich angezeigt.
3. Wählen Sie im Feld Portal-Registerkarten-Option (PortalTabOption) den Eintrag Angegebene Registerkarte (SpecifiedTab) aus.
4. Wählen Sie im Feld Portal-Registerkarte (Portal Tab) eine Registerkarte aus.
5. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
