Globale Symbolleiste von Portal Studio

Die globale Symbolleiste von Portal Studio ermöglicht den einfachen Zugriff auf die in Portal Studio verwendeten Funktionen.

Schaltflächen der globalen Symbolleiste von Portal Studio

Diese Tabelle beschreibt die Schaltflächen in der globalen Symbolleiste von Portal Studio.

Schaltfläche

Funktion der Schaltfläche . . .

Inhalte anzeigen (View Contents)

Ändert die Ansicht der Komponente im Arbeitsbereich in einen anderen Arbeitsbereich. Die Standardeinstellung ist "Entwickler" (Developer). Weitere Informationen finden Sie unter "Entwickeln in Arbeitsbereichen".

Hinweis: Diese Option wird nur angezeigt, wenn eine Komponente geöffnet und im Arbeitsbereich ausgewählt ist.

Portal-Einstellungen

Anzeigen des Fensters "Portaleinstellungen" (Portal Settings). Weitere Informationen zum Ändern der Portaleinstellungen finden Sie unter "Konfigurieren von Portaleinstellungen".

Zurück zu Portal (Back to Portal)

Zum Portal zurückzukehren.

Hilfe

Anzeigen der Onlinehilfe für Portal Studio.

Abmelden (Logout)

Hiermit können Sie sich von Portal Studio und vom Portal abmelden und zur Anmeldeseite zurückkehren.

Neu

Erzeugen einer neuen Komponente. Wenn Sie auf diese Schaltfläche klicken, wird eine Dropdown-Liste mit Optionen zum Erzeugen neuer Benutzeroberflächenkomponenten angezeigt.

Speichern (Save)

Speichern Sie die ausstehenden Änderungen für die Komponente, die im Arbeitsbereich ausgewählt ist. Weitere Informationen zum Speichern von Änderungen finden Sie unter "Speichern von Änderungen".

Hinweis: In Portal Studio wird diese Schaltfläche aktiviert, wenn Sie eine Änderung an der Komponente durchführen, die derzeit im Arbeitsbereich im Fokus ist. Sie können für diese Schaltfläche festlegen, dass sie immer aktiviert ist, indem Sie die Einstellung "AlwaysEnableSave" auf "Wahr" (True) festlegen. Weitere Informationen zum Ändern von Einstellungen für Portal Studio finden Sie unter "Studio-Einstellungen (Studio Settings)".

Alles Speichern (Save All)

Speichern Sie die ausstehenden Änderungen für Komponenten auf allen geöffneten Registerkarten im Arbeitsbereich. Informationen zum Speichern von Änderungen finden Sie unter "Speichern von Änderungen".

Hinweis: In Portal Studio wird diese Schaltfläche aktiviert, wenn Sie eine Änderung an der Komponente durchführen, die derzeit im Arbeitsbereich im Fokus ist. Sie können für diese Schaltfläche festlegen, dass sie immer aktiviert ist, indem Sie die Einstellung "AlwaysEnableSave" auf "Wahr" (True) festlegen. Weitere Informationen zum Ändern von Einstellungen für Portal Studio finden Sie unter "Studio-Einstellungen (Studio Settings)".

Rückgängig machen (Undo)

Hiermit können Sie Ihre letzten Änderungen rückgängig machen. Die Schaltfläche "Rückgängig machen" (Undo) zeigt einen Zähler mit der Anzahl der Änderungen an, die Sie rückgängig machen können. Wenn Sie die Umschalttaste gedrückt halten, während Sie auf "Rückgängig machen" (Undo) klicken, wird eine Liste der Aktionen angezeigt, die Sie rückgängig machen können.

Wiederholen (Redo)

Stellt Änderungen wieder her, die über die Schaltfläche "Rückgängig machen" (Undo) rückgängig gemacht wurden. Die Schaltfläche "Wiederholen" (Redo) zeigt einen Zähler mit der Anzahl der rückgängig gemachten Änderungen an, die Sie wiederherstellen können. Wenn Sie die Umschalttaste gedrückt halten, während Sie auf "Wiederholen" (Redo) klicken, wird eine Liste der Aktionen angezeigt, die Sie wiederherstellen können.

Kopieren

Kopiert die ausgewählte Komponente.

Delete

Löscht die ausgewählte Komponente.

Import

Importiert eine Komponente aus einer XML-Datei. Weitere Informationen zum Importieren einer Komponente finden Sie unter "Importieren einer Komponente".

Export

Exportieren Sie eine Komponente aus dem aktuellen Arbeitsbereich oder aus allen aktiven Arbeitsbereichen. Weitere Informationen zum Exportieren einer Komponente finden Sie unter "Exportieren einer Komponente".

Batch-Import (Batch Import)

Importiert mehrere Webparts, Seiten und Seitenfolgen aus einer einzelnen XML-Datei. Weitere Informationen zum Durchführen von Batch-Importen finden Sie unter "Durchführen von Batch-Importen".

Batch-Export (Batch Export)

Exportiert mehrere Webparts, Seiten und Seitenfolgen in eine einzelne XML-Datei oder eine ZIP-Datei. Weitere Informationen zum Ausführen von Batch-Exporten finden Sie unter "Durchführen von Batch-Exporten".

Zurücksetzen (Revert To)

Hiermit können Sie zur veröffentlichten Inhalts- oder Systeminhaltsversion der Komponente zurückkehren. Weitere Informationen finden Sie unter "Zurücksetzen von Inhalten".

Ausführen

Testen Sie die aktuelle Seite in Ihrem Browser im Portal-Framework. Sie müssen eine Seite speichern, bevor Sie sie testen können. Weitere Informationen finden Sie unter "Testen einer Seite".

Veröffentlichen

Veröffentlicht gespeicherte Änderungen im höchsten aktiven Arbeitsbereich. Weitere Informationen finden Sie unter "Veröffentlichen von Inhalten". Weitere Informationen finden Sie unter "Entwickeln in Arbeitsbereichen".

Erzeugen

Erstellt die aktuelle Seite. Weitere Informationen finden Sie unter "Generieren einer Seite".

Webpart extrahieren (Extract Web Part)

Öffnet das aktuell auf einer Seite ausgewählte Webpart in einer neuen Registerkarte. Weitere Informationen zum Extrahieren von Webparts finden Sie unter "Extrahieren eines Webparts".

Webpart neu laden (Reload Web Part)

Lädt das aktuell auf einer Seite ausgewählte Webpart neu und stellt dabei seinen ursprünglichen Zustand im Komponentenkatalog wieder her. Weitere Informationen zum Neuladen von Webparts finden Sie unter "Neuladen eines Webparts".

Katalog

Zeigt den Komponentenkatalog an oder blendet ihn aus. Weitere Informationen zum Komponentenkatalog finden Sie unter "Komponentenkatalog".

Toolbox

Zeigt den Toolboxbereich an oder blendet ihn aus. Weitere Informationen zum Fensterbereich "Toolbox" finden Sie unter "Toolbox".

Eigenschaften

Zeigt den Bereich "Eigenschaften" (Properties) an oder blendet ihn aus. Weitere Informationen zum Bereich "Eigenschaften" (Properties) finden Sie unter "Fensterbereich "Eigenschaften" (Properties)".

Trace Viewer

Zeigt den Trace Viewer an oder blendet ihn aus. Weitere Informationen zum Trace Viewer finden Sie unter "Ausführen des Ablaufverfolgungsprotokolls".

Einstellungen zurücksetzen (Reset Settings)

Setzt alle Änderungen zurück, die am Layout der Fensterbereiche im Arbeitsbereich vorgenommen wurden.
