Steuerelemente

Mit Steuerelementen können Sie Webparts in Portal Studio anpassen. Der Zugriff auf Steuerelemente erfolgt über die Registerkarte "Steuerelemente" (Controls) in der Toolbox. Weitere Informationen zum Anzeigen der Toolbox finden Sie unter "Toolbox".

Sie können Steuerelemente nur zu einem Webpart, nicht jedoch direkt zu einer Seite hinzufügen. Weitere Informationen zum Hinzufügen von Steuerelementen zu einem Webpart finden Sie unter "Hinzufügen eines Steuerelements zu einem Webpart".

Eine Popup-Warnmeldung wird angezeigt, wenn Sie ein Steuerelement hinzufügen, dessen Unterstützung in einer zukünftigen Version von Portal Studio eingestellt werden soll. Die Warnmeldung gibt an, wann die Unterstützung für das Steuerelement eingestellt wird und welche anderen Steuerelemente Sie stattdessen verwenden können. Auf Steuerelementen, deren Unterstützung eingestellt wird, wird außerdem eine QuickInfo mit diesen Informationen angezeigt.

Diese Tabelle beschreibt die Steuerelemente, die auf der Registerkarte "Steuerelemente" (Controls) in der Toolbox verfügbar sind.

Steuerelement

Beschreibung

3D-Visualisierung

Zeigt 3D-Visualisierungen mit der Möglichkeit zum Drehen, Vergrößern bzw. Verkleinern, Markieren von Bildern und Speichern von Momentaufnahmen an.

Hinweis: Dieses Steuerelement unterstützt JT-Dateien.

Wenn Sie das Steuerelement "3D-Visualisierung" (3D Visualization) verwenden, werden das Popup "3D-Datei-Viewer" (3D File Viewer) und Ihr 3D-Bild angezeigt. Sie haben die Möglichkeit, das 3D-Bild zu drehen, zu vergrößern und zu verkleinern.

Das Popup "3D-Datei-Viewer" (3D File Viewer) enthält eine Symbolleiste, mit der Sie Ihr 3D-Bild manipulieren und 2D-Momentaufnahmen speichern können. In der Symbolleiste haben Sie folgende Möglichkeiten:

• Hinzufügen von Text
• Hinzufügen von Formen
• Zeichnen von Freiformen auf dem Bild
• Auswählen von Text- und Formfarben aus einer Farbauswahl
• Festlegen benutzerdefinierter Linienstärken für Formwerkzeuge
• Zurücksetzen der Änderungen
• Speichern einer 2D-Momentaufnahme

Sie können ein 3D-Bild überall dort anzeigen, wo Sie auch einen Dokumentsatz anzeigen können. Weitere Informationen zum 3D-Visualisierungs-Steuerelement finden Sie unter "Verwenden der 3D-Visualisierung".

Boolean Drop-down

Boolesches Dropdown-Element. Boolescher Selektor mit konfigurierbaren Beschriftungen und Optionen zum Löschen des Werts.

Boolean Switch

Boolescher Schalter. Boolesches Optionsfeld-Steuerelement mit konfigurierbaren Beschriftungen.

Breadcrumb-Steuerelement

Steuerelement, das die Breadcrumb-Navigation auf einer Seite ermöglicht.

Schaltfläche

Klickbares Steuerelement, das als Text oder Bild angezeigt werden kann.

Chart Control

Chart-Steuerelement. Positioniert ein benutzerdefiniertes Diagramm auf einer Seite oder einem Webpart. Eine einzelne Seite kann gleichzeitig mehrere Chart-Steuerelemente enthalten.

Sie können das Format und die Daten der Chart-Steuerelemente über die Eigenschaften "ChartMode", "ShowLegend", "SeriesSettings", "SeriesValueColumn", "TooltipColumn", "ValueColumn", "XAxis" und "YAxis" im Bereich "Eigenschaften" (Properties) ändern. Chart-Steuerelemente werden dynamisch aktualisiert, wenn Sie die Datenquellen für diese Eigenschaften ändern. Weitere Informationen zu Eigenschaften von Chart-Steuerelementen finden Sie unter "Steuerelementeigenschaften".

Die Eigenschaft "SeriesSettings" ermöglicht es Ihnen, die Darstellung von Datenpunkten in Streu- und Wafermap-Diagrammen weiter anzupassen. Informationen zum Hinzufügen von SeriesSettings-Eigenschaften finden Sie unter "Hinzufügen von SeriesSettings".

Sie können benutzerdefinierte Diagrammformate einschließlich Wafer-Zuordnungen erstellen, indem Sie die Datei d3.js ändern. Weitere Informationen zum D3-Dokumentobjektmodell finden Sie unter http://d3js.org.

Hinweis: Sie können auf Portalseiten mit dem Mausrad in Chart-Steuerelemente hineinzoomen und daraus herauszoomen.

Check Box

Kontrollkästchen. Feld, das markiert wird oder dessen Markierung aufgehoben wird, um eine Option zu aktivieren oder zu deaktivieren.

Hinweis: Sie müssen die Eigenschaft "Autom. Postback" (AutoPostBack) auf "Wahr" (True) setzen, wenn Sie Kontrollkästchen-Steuerelemente zu Seiten hinzufügen, die Popup-Tabellen enthalten. Weitere Informationen zum Ändern der Eigenschaften von Steuerelementen finden Sie unter "Steuerelementeigenschaften".

Checkliste

Steuerelement, das als Prüfliste auf einer Seite angezeigt wird.

Color Picker

Ermöglicht die Auswahl einer Farbpalette, um eine Farbe für ein Feld festzulegen. Weitere Informationen finden Sie unter "Verwenden der Farbauswahl".

Container Drop-down

Drop-down List

Dauer

Informationen zum Definieren von Prozesstimern finden Sie im Opcenter Execution Medical Device and Diagnostics Portal Modeling-Benutzerhandbuch bzw. im Opcenter Execution Core Portal Modeling-Benutzerhandbuch.

File Browse

Flyout Drop-down

Wird als Schaltfläche angezeigt, die eine andere Seite in einem Flyout-Fenster öffnet.

Tabelle (Grid)

Hinweis: Beim Hinzufügen eines Tabellensteuerelements zu einem Webpart werden Sie aufgefordert, entweder eine Tabelle mit dem Tabellenassistent zu erstellen oder ein leeres Tabellensteuerelement zu platzieren, das Sie manuell anpassen können. Weitere Informationen finden Sie unter "Definieren von Tabellen mit dem Tabellenassistenten".

HTML-Fenster

Ermöglicht dem Benutzer die Angabe von HTML-Inhalt oder eines Pfads zu einer Vorlagendatei.

Hinweis: Dieses Steuerelement unterstützt HTML-Inhalt, der mithilfe der Eigenschaft "HTMLText" eingegeben wurde, oder einen Ordnerpfad zu einer HTML-Vorlagendatei mithilfe der Eigenschaft "TemplateFile". Bei Verwendung dieses Steuerelements müssen Sie einen Wert für die Eigenschaft "HTMLText" oder "TemplateFile" eingeben. Weitere Informationen zu den Eigenschaften "HTMLText" und "TemplateFile" finden Sie unter "Steuerelementeigenschaften".

Image Control

Anweisungstext
