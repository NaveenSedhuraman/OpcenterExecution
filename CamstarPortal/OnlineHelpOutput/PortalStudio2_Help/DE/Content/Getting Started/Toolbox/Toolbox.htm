Toolbox

Über die Toolbox können Sie auf die folgenden Registerkarten zugreifen:

• Steuerelemente (Controls): Zeigt eine Liste der Steuerelemente an, die Sie zu Webparts hinzufügen können. Weitere Informationen finden Sie unter "Steuerelemente".
• Feldverzeichnis (Fields Directory): Zeigt eine Liste der Services und parametrischen Datenfelder an, die Sie zu Webparts hinzufügen können. Weitere Informationen finden Sie unter "Feldverzeichnis".

Mit der Schaltfläche "Toolbox" in der Symbolleiste von Portal Studio kann die ausgeblendete Toolbox angezeigt bzw. die angezeigte Toolbox ausgeblendet werden.

Relative Schaltfläche

Die Registerkarte "Feldverzeichnis" (Fields Directory) enthält die Schaltfläche "Relativ" (Relative) – angezeigt durch den Großbuchstaben R. Die Schaltfläche "Relativ" (Relative) schaltet zwischen den relativen und absoluten Einstellungen für ein Feld um, wenn ein Feld zu einem Webpart hinzugefügt wird. Weitere Informationen finden Sie unter "Relative Feldausdrücke".
