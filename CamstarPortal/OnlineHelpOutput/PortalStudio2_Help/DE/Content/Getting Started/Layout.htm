Verwalten von Layouts

In Portal Studio können Sie das Layout einer Seite oder eines Webparts mit einem Zeilen- und Spaltenlayout konfigurieren, das Zellen enthält.

Im Bereich "Eigenschaften" (Properties) können Sie die Eigenschaften einer einzelnen Zelle ändern. Informationen zum Ändern von Zelleneigenschaften finden Sie unter "Zelleneigenschaften".

Beim Verwalten von Layouts

Jede Seite oder jedes Webpart enthält mindestens eine Zelle als Teil des Komponentenlayouts. Mithilfe von Zellen können Sie angeben, wo Komponenten angezeigt werden, wenn sie auf Seiten und in Webparts in Bezug auf Komponenten in anderen Zellen platziert werden.

Sie können das Layout einer Zelle mit der Schaltfläche "Aktionen" (Actions) ändern, wenn Sie eine Komponente in der Ansicht "Design" anzeigen. Sie können mehrere Zellen auswählen, indem Sie zuerst eine Zelle und dann eine andere Zelle auswählen. Alle Zellen zwischen den beiden Zellen sind in Ihrer Auswahl enthalten. Ausgewählte Zellen werden mit einem blauen Hintergrund angezeigt.

Diese Tabelle definiert die Optionen, die über die Schaltfläche "Aktionen" (Actions) für die Verwaltung von Zellenlayouts verfügbar sind.

Option

Aktion . . .

Verbinden (Merge)

Verbindet die ausgewählten Zellen zu einer einzelnen Zelle.

Aufteilen

Teilt eine verbundene Zelle wieder in ihre Komponentenzellen.

Zeile entfernen (Remove Row)

Löscht die ausgewählte Zeile.

Spalte entfernen (Remove Column)

Löscht die ausgewählte Spalte.

Zeile einfügen vor (Insert Row Before)

Fügt eine neue Zeile über der ausgewählten Zeile ein.

Spalte einfügen vor (Insert Column Before)

Fügt eine neue Spalte links von der ausgewählten Spalte ein.

Zeile einfügen nach (Insert Row After)

Fügt eine neue Zeile unter der ausgewählten Zeile ein.

Spalte einfügen nach (Insert Column After)

Fügt eine neue Spalte rechts von der ausgewählten Spalte ein.

Zurücksetzen

Hebt die aktuelle Auswahl der Zellen auf.

Zellenposition umschalten (Toggle Cell Position)

Schaltet die Positionsanzeige der Zelle [Zeile, Spalte] um.
