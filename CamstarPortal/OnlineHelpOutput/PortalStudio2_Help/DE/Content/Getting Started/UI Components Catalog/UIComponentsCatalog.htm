Komponentenkatalog

Der Komponentenkatalog enthält mehrere Registerkarten, die den Zugriff auf Komponenten sowie Navigations- und Zuordnungskomponenten ermöglichen.

Über die Registerkarte "Komponenten" (Components) können Sie auf folgende Komponenten zugreifen:

• Seiten
• Webparts
• Seitenfolgen

Über die Registerkarte "Navigation" können Sie auf folgende Navigationskomponenten zugreifen:

• Menüs
• Tabs

Über die Registerkarte "Modellierung" (Modeling) können Sie auf folgende Zuordnungskomponenten zugreifen:

• Seitenaktionen
• Seitenaktionskategorien
• Seitenaktionsregeln
• Seitenzuordnungen
• Zielmatrizen

Die Registerkarten des Komponentenkatalogs sind in Dropdownlisten für die verschiedenen Komponententypen zusammengefasst. Jede Dropdownliste enthält ein Suchfeld, die Schaltfläche "Neues Element erstellen" (Create New Item), eine Reihe von Anzeigemodus-Schaltfeldern und die Schaltfläche "Aktualisieren" (Refresh). Wenn Sie auf die Schaltfläche "Neues Element erstellen" (Create New Item) klicken, wird im Arbeitsbereich ein neues Element für den betreffenden Elementtyp angezeigt.

Mit der Schaltfläche "Katalog" (Catalog) in der Symbolleiste von Portal Studio wird der Komponentenkatalog angezeigt, falls er ausgeblendet ist, und ausgeblendet, falls er angezeigt wird.

Note: Wenn Sie sich beim Upgrade von einer früheren Version erstmals bei Portal Studio anmelden, werden u. U. einige Standardseiten im Komponentenkatalog auf der Registerkarte "Komponenten" (Components) in der Gruppe "Nicht definiert" (Undefined) angezeigt. Sie können die Standardseiten in der Gruppe "Nicht definiert" (Undefined) automatisch in andere Gruppen sortieren. Weitere Informationen finden Sie unter "Vorgehensweise zum automatischen Sortieren von Standardseiten".
Öffnen vorhandener Komponenten

Der Komponentenkatalog enthält alle Standardkomponenten und alle Komponenten, die Sie erstellt haben. Sie können eine beliebige Komponente im Komponentenkatalog öffnen, indem Sie zur Registerkarte und zum Abschnitt einer Komponente navigieren und dann auf den Komponentennamen doppelklicken. Die Komponente, die Sie öffnen, wird im Arbeitsbereich angezeigt.
