Anzeigen der Onlinehilfe

Sie können die Onlinehilfe für Portal Studio anzeigen, indem Sie in der globalen Symbolleiste von Portal Studio auf die Schaltfläche "Hilfe" (Help) klicken.

Beim Anzeigen der Onlinehilfe

Die Onlinehilfe wird als Popup in Ihrem Browser angezeigt. Die Hilfeinformationen werden in der Sprache angezeigt, die Sie im Browser als bevorzugte Sprache eingestellt haben, vorausgesetzt, die Informationen sind in dieser Sprache verfügbar. Standardmäßig wird die Hilfe in englischer Sprache angezeigt. Gewährleisten Sie, dass Sie die neuesten Sprachpakete für Ihre bevorzugte Sprache installiert haben.

Note: Möglicherweise müssen Sie blockierte Inhalte für Skripts und ActiveX-Steuerelemente zulassen, wenn Sie die Onlinehilfe in Internet Explorer anzeigen.

Vorgehensweise zum Anzeigen der Onlinehilfe

Gehen Sie folgendermaßen vor, um die Onlinehilfe anzuzeigen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Hilfe (Help). Ein Browser-Popup mit der Onlinehilfe für Portal Studio wird angezeigt.
