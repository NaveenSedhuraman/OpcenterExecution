Suchen nach Seitenaktionsbeschriftungen

Mit dem Popup "Beschriftungssuche" (Label Lookup) können Sie beim Definieren von Seitenaktionen nach vordefinierten Beschriftungen suchen.

Beim Suchen nach Seitenaktionsbeschriftungen

Siemens empfiehlt, die vordefinierten Beschriftungen in der Datenbank zu verwenden, anstatt jedes Mal neue Beschriftungen zu erstellen, wenn eine neue Seitenaktion erstellt wird.

Beschriftungen werden beim Erstellen von Seitenaktionen verwendet. Weitere Informationen zum Erstellen von Seitenaktionen finden Sie unter "Konfigurieren von Seitenaktionen".

Die Spalte "Kategorie" (Category) in der Tabelle mit Beschriftungssuchergebnissen wird nur angezeigt, wenn im Suchparameterfeld "Kategorie" (Category) die Option "Alle" (All) ausgewählt ist.

Vorgehensweise zum Suchen nach Seitenaktionsbeschriftungen

Gehen Sie folgendermaßen vor, um nach Seitenaktionsbeschriftungen zu suchen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Neu erstellen (Create New) und wählen Sie Seitenaktion (Page Action) aus. Im Arbeitsbereich wird eine neue Seitenaktion angezeigt.

Oder

Wählen Sie im Bereich Komponenten (Components) die Registerkarte Modellierung (Modeling) aus und doppelklicken Sie auf eine vorhandene Seitenaktion. Die Seitenaktion wird im Arbeitsbereich angezeigt.

3. Erweitern Sie den Abschnitt Benutzeroberfläche: Aktion (UIAction).
4. Klicken Sie auf die Schaltfläche Mehr (More) für Beschriftungsname (LabelName). Das Popup "Beschriftungssuche" (Label Lookup) wird angezeigt.
5. Geben Sie ein oder mehrere Kriterien ein, um Ihre Suche einzugrenzen.
6. Klicken Sie auf Suchen (Search). Die Anwendung zeigt die Beschriftungen, die den Suchkriterien entsprechen, in der Tabelle an.
7. Wählen Sie eine Beschriftung aus. Der Standardtext und der lokale Text für die Beschriftung werden in den Feldern "Standardtext" (Default Text) und "Lokaler Text" (Local Text) angezeigt.
8. Klicken Sie auf OK. Das Popup "Beschriftungssuche" (Label Lookup) wird geschlossen und die Informationen für die Beschriftung werden in die zugehörigen Felder im Aktionsfenster eingetragen.
