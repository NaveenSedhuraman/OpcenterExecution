Konfigurieren von Seitenaktionen

Sie können Seitenaktionen (Page Actions) so konfigurieren, dass sie auf den zugehörigen Quellseiten entweder als Verknüpfungen oder als Schaltflächen angezeigt werden. Sie können Seitenaktionen mit folgenden Elementen konfigurieren:

• Mit dem Webpart "Aktionssteuerung" (ActionsControl), um eine Seitenaktion direkt als Verknüpfung oder als Schaltfläche auf einer Seite zu platzieren.
• Mit dem Popup "Seitenaktionseigenschaften", um eine Seitenaktion als Schaltfläche darzustellen, die unten links oder unten rechts auf einer Seite angezeigt wird.

Beim Konfigurieren von Seitenaktionen

Damit Aktionen auf einer Portalseite angezeigt werden, müssen Sie Seitenaktionen aktivieren und Quellseiten zu Seitenaktionen hinzufügen. Weitere Informationen zum Aktivieren von Seitenaktionen finden Sie unter "Aktivieren von Seitenaktionen". Weitere Informationen zum Hinzufügen von Seiten zu Quellseitenlisten finden Sie unter "Hinzufügen von Quellseiten zu Seitenaktionen".

Durch Platzieren des Webparts "Aktionsteuerung" legen Sie fest, wo auf einer Portalseite Seitenaktionen angezeigt werden. Weitere Informationen zum Hinzufügen von Webparts zu Seiten finden Sie unter "Hinzufügen eines Webparts zu einer Seite".

Vorgehensweise zum Konfigurieren von Seitenaktionen als Schaltflächen

Gehen Sie folgendermaßen vor, um Seitenaktionen zu konfigurieren:

1. Greifen Sie auf Portal Studio zu.
2. Wählen Sie die Registerkarte Modellierung (Modeling) im Komponentenkatalog aus und doppelklicken Sie auf eine Seitenaktion. Die Seitenaktion wird im Arbeitsbereich angezeigt.
3. Klicken Sie auf Mehr (More) für UIAction. Das Popup "UIAction" wird angezeigt.
4. Wählen Sie im Feld Standort (Location) den Eintrag Schaltfläche aus.
5. Wählen Sie im Feld Schaltflächenposition (ButtonPosition) den Bereich aus, in dem die Schaltfläche angezeigt werden soll.
6. Klicken Sie auf OK.Das Popup "Page UIAction" wird geschlossen.
7. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Konfigurieren von Seitenaktionen mit dem Webpart "Aktionssteuerung" (ActionsControl)

Gehen Sie folgendermaßen vor, um Seitenaktionen mit dem Webpart "Aktionssteuerung" (ActionsControl) zu konfigurieren:

1. Greifen Sie auf Portal Studio zu.
2. Wählen Sie die Registerkarte Modellierung (Modeling) im Komponentenkatalog aus und doppelklicken Sie auf eine Seitenaktion. Die Seitenaktion wird im Arbeitsbereich angezeigt.
3. Klicken Sie auf Mehr (More) für UIAction. Das Popup "UIAction" wird angezeigt.
4. Wählen Sie im Feld Standort (Location) den Eintrag Bedientafel (ActionPanel) aus.
5. Klicken Sie auf OK. Das Popup "UIAction" wird geschlossen.
6. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
7. Wählen Sie die Registerkarte Komponenten im Komponentenkatalog aus und doppelklicken Sie auf eine Seite. Die Seite wird im Arbeitsbereich angezeigt.
8. Klicken Sie auf das Webpart ActionsControl und ziehen Sie es auf eine Zelle auf der Seite.
9. Soll die Seitenaktion als Verknüpfung oder als Schaltfläche angezeigt werden?

Seitenaktion anzeigen als . . .

Dann . . .

Verknüpfung.
a. Geben Sie im Bereich "Eigenschaften" (Properties) im Feld Name einen eindeutigen Namen für das Webpart "Aktionssteuerung" (ActionsControl) ein.
b. Gehen Sie zu Schritt 11.

Schaltfläche.
a. Geben Sie im Bereich "Eigenschaften" (Properties) im Feld Name einen eindeutigen Namen für das Webpart "Aktionssteuerung" (ActionsControl) ein.
b. Wählen Sie im Bereich "Eigenschaften" (Properties) im Feld Aktionsstil (ActionsStyle) den Eintrag Schaltfläche (Button) aus.
c. Gehen Sie zu Schritt 11.
10. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
