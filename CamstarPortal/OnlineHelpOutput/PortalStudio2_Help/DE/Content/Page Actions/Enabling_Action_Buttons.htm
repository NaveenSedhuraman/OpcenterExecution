Aktivieren von Seitenaktionen

Seitenaktionen (Page Actions) sind Verknüpfungen und Schaltflächen, die eine Aktion auslösen, wenn sie angeklickt werden. Sie können Seitenaktionen auf einer beliebigen Seite über den Bereich "Eigenschaften" (Properties) aktivieren.

Beim Aktivieren von Seitenaktionen

Sie müssen Seitenaktionen im Popup "DynamicActions" erst aktivieren, damit sie auf einer Seite angezeigt werden.

Seitenaktionen sind entweder deaktiviert oder ausgeblendet, wenn die Seitenaktionsregeln nicht erfüllt sind. Deaktivierte Seitenaktionen sind sichtbar, können aber nicht angeklickt werden. Ausgeblendete Seitenaktionen werden auf einer Seite nicht angezeigt. Seitenaktionen sind standardmäßig ausgeblendet. Sie können die Eigenschaft "DisabledDisplayOption" im Popup "DynamicActions" so ändern, dass Aktionen mit fehlgeschlagenen Regeln deaktiviert werden. Weitere Informationen zu den Feldern, die im Popup "DynamicActions" angezeigt werden, finden Sie unter "Beschreibungen der Seiteneigenschaften".

Vorgehensweise zum Aktivieren von Seitenaktionen

Gehen Sie folgendermaßen vor, um Seitenaktionen zu aktivieren:

1. Greifen Sie auf Portal Studio zu.
2. Doppelklicken Sie auf eine Seite im Komponentenkatalog. Die Seite wird im Arbeitsbereich angezeigt.
3. Klicken Sie im Fensterbereich "Eigenschaften" (Properties) auf Mehr (More) für DynamicActions. Das Popup "DynamicActions" wird angezeigt.
4. Wählen Sie im Feld Erforderliche Aktionen (ActionsRequired) den Eintrag Wahr (True) aus.
5. Klicken Sie auf OK. Das Popup "DynamicActions" wird geschlossen.
6. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Festlegen von Seitenaktionen als deaktiviert

Gehen Sie folgendermaßen vor, um Seitenaktionen als deaktiviert festzulegen:

1. Greifen Sie auf Portal Studio zu.
2. Doppelklicken Sie auf eine Seite im Komponentenkatalog. Die Seite wird im Arbeitsbereich angezeigt.
3. Klicken Sie im Fensterbereich "Eigenschaften" (Properties) auf Mehr (More) für DynamicActions. Das Popup "DynamicActions" wird angezeigt.
4. Wählen Sie im Feld DeaktiviertAnzeigeoptionen (DisabledDisplayOptions) die Option Deaktivieren (Disable) aus.
5. Klicken Sie auf OK. Das Popup "DynamicActions" wird geschlossen.
6. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
