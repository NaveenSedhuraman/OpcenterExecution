Erstellen von Seitenaktionen

Sie können Seitenaktionen in Portal Studio erstellen. Die Eigenschaften einer Seitenaktion und die Funktion, die nach Klicken auf die Seitenaktion ausgeführt wird, richten sich nach dem jeweiligen Typ der Benutzeroberflächenaktion (UIAction).

Beim Erstellen von Seitenaktionen

Seitenaktionen werden über die Seitenaktionseigenschaften konfiguriert. Die weitere Konfiguration erfolgt über Seitenaktionsregeln und Quellseiten.

Seitenaktionsregeln (Page Action Rules) legen über benutzerdefinierte Bedingungen fest, wann eine Seitenaktion auf einer Seite verfügbar sein soll. Weitere Informationen zum Konfigurieren von Seitenaktionsregeln finden Sie unter "Erstellen von Seitenaktionsregeln".

Quellseiten (Source Pages) sind eine Liste der Seiten, auf denen die Aktion angezeigt wird, wenn die Bedingungen der Seitenaktionsregel erfüllt sind. Damit eine Seitenaktion auf einer Seite angezeigt wird, müssen Quellseiten für die Seitenaktion definiert sein. Weitere Informationen zum Hinzufügen von Seiten zu Quellseitenlisten finden Sie unter "Hinzufügen von Quellseiten zu Seitenaktionen".

Seitenaktionen sind so konfiguriert, dass sie entweder als Verknüpfung oder als Schaltfläche dargestellt werden. Seitenaktionen werden standardmäßig als Verknüpfungen eingerichtet, wenn sie mit dem Webpart "Aktionssteuerung" (ActionsControl) zu einer Seite hinzugefügt werden. Sie können die Eigenschaft "Aktionsstil" (ActionStyle) des Webparts "Aktionssteuerung" so ändern, dass Aktionen als Schaltflächen angezeigt werden, oder Seitenaktionen direkt als Schaltflächen zu Seiten hinzufügen. Weitere Informationen finden Sie unter "Konfigurieren von Seitenaktionen".

Important: Sie können den Benutzeroberflächen-Aktionstyp nicht mehr ändern, nachdem Sie den Typ ausgewählt und die Seitenaktion gespeichert haben.

Öffnen vorhandener Seitenaktionen

Der Zugriff auf die Seitenaktionen erfolgt über die Registerkarte "Modellierung" (Modeling) im Katalog der Benutzeroberflächenkomponenten. Zahlreiche Seitenaktionen wurden bereits standardmäßig erstellt. Weitere Informationen zum Öffnen vorhandener Seitenaktionen finden Sie unter "Öffnen vorhandener Komponenten".

Vorgehensweise zum Erstellen von Seitenaktionen

Gehen Sie folgendermaßen vor, um Seitenaktionen zu erstellen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Neu (New) und wählen Sie Seitenaktion (Page Action) aus. Im Arbeitsbereich wird eine leere Seitenaktion angezeigt.
3. Geben Sie in das Feld Name einen Namen für die Seitenaktion ein.
4. Klicken Sie für UIAction auf "Mehr" (More). Das Popup "UIAction" wird angezeigt.
5. Wählen Sie aus der Dropdown-Liste am oberen Rand des Popups einen Benutzeroberflächen-Aktionstyp aus. Die Felder werden auf Basis des ausgewählten Benutzeroberflächen-Aktionstyps zurückgesetzt.
6. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
7. Klicken Sie auf OK. Das Popup "UIAction" wird geschlossen.
8. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
9. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
