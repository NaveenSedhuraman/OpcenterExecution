Hinzufügen von Seitenaktionsregeln zu Seitenaktionen

Seitenaktionsregeln bestimmen die Bedingungen, unter denen eine Seitenaktion auf einer Seite angezeigt wird. Sie können einer Seitenaktion mehrere Regeln hinzufügen, um anzugeben, wann die Seitenaktion angezeigt wird.

Beim Hinzufügen von Seitenaktionsregeln zu Seitenaktionen

Seitenaktionsregeln können auch aus einer Seitenaktion entfernt werden. Wenn die Seitenaktionsregel entfernt wurde, wird die Seitenaktion nicht mehr durch eine Seitenaktionsregel gesteuert.

Felddefinitionen des modalen Fensters "ActionRules"

Das modale Fenster "ActionRules" enthält nur das Feld "Name". Im Feld "Name" wird eine vorhandene Seitenaktionsregel ausgewählt und der Seitenaktion zugewiesen.

Vorgehensweise zum Hinzufügen von Seitenaktionsregeln zu Seitenaktionen

Gehen Sie folgendermaßen vor, um Seitenaktionsregeln zu Seitenaktionen hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Wählen Sie die Registerkarte Modellierung (Modeling) im Komponentenkatalog aus und doppelklicken Sie auf eine Seitenaktion. Die Seitenaktion wird im Arbeitsbereich angezeigt.
3. Klicken Sie für ActionRules auf "Mehr" (More). Das modale Fenster "ActionRules" wird angezeigt.
4. Klicken Sie auf Hinzufügen. In der Liste im linken Bereich wird eine neue Aktionsregel angezeigt.
5. Wählen Sie im Feld Name eine Seitenaktionsregel aus.
6. Wiederholen Sie ggf. die Schritte 4 bis 5, um weitere Seitenaktionsregeln hinzuzufügen.
7. Klicken Sie auf OK. Das modale Fenster "ActionRules" wird geschlossen.
8. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Löschen von Seitenaktionsregeln aus Seitenaktionen

Gehen Sie folgendermaßen vor, um Seitenaktionsregeln aus Seitenaktionen zu entfernen:

1. Greifen Sie auf Portal Studio zu.
2. Wählen Sie die Registerkarte Modellierung (Modeling) im Komponentenkatalog aus und doppelklicken Sie auf eine Seitenaktion. Die Seitenaktion wird im Arbeitsbereich angezeigt.
3. Klicken Sie für ActionRules auf "Mehr" (More). Das modale Fenster "ActionRules" wird angezeigt.
4. Wählen Sie aus der Liste im linken Bereich eine Seitenaktionsregel aus.
5. Klicken Sie auf das Symbol Löschen (Delete). Die Seitenaktionsregel wird aus der Liste entfernt.
6. Wiederholen Sie ggf. die Schritte 4 bis 5, um weitere Seitenaktionsregeln zu löschen.
7. Klicken Sie auf OK. Das modale Fenster "ActionRules" wird geschlossen.
8. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
