Erstellen einer Seitenaktionskategorie

Mithilfe von Seitenaktionskategorien können Sie Gruppenaktionen für die Anzeige auf einer Portalseite gruppieren.

Felddefinitionen für Seitenaktionskategorien

Diese Tabelle definiert Felder, die zum Erstellen einer Seitenaktionskategorie verwendet werden.

Feld

Definition

Typ

Name

Eindeutiger Name für die Seitenaktionskategorie.

Erforderlich

Description

Kurze Beschreibung der Seitenaktionskategorie.

Optional

FilterTags

Gibt ein Filter-Tag an, mit dem Sie die für die Anzeige und Auswahl verfügbaren Elemente begrenzen können.

Optional

LabelName

Bestimmt die Bezeichnung, die auf der Seitenaktionskategorie angezeigt wird. Weitere Informationen zum Suchen von Beschriftungen finden Sie unter "Suchen nach Seitenaktionsbeschriftungen".

Optional

LabelText

Bestimmt die Bezeichnung, die auf der Seitenaktionskategorie angezeigt wird. Dieses Feld ersetzt den im Feld "LabelName" eingegebenen Text.

Optional

Notes

Relevante Kommentare zur Seitenaktionskategorie.

Optional

Sequence

Reihenfolgeposition dieser Seitenaktionskategorie.

Optional

Vorgehensweise zum Erstellen einer Seitenaktionskategorie

Gehen Sie folgendermaßen vor, um eine Seitenaktionskategorie zu erstellen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Neu (New) und wählen Sie Seitenaktionskategorie (Page Action Category) aus. Im Arbeitsbereich wird eine leere Seitenaktionskategorie angezeigt.
3. Geben Sie im Feld Name einen Namen für die Seitenaktionskategorie ein.
4. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
5. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
