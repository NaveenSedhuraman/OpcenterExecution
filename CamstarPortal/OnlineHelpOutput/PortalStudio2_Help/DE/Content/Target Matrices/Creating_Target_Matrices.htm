Erstellen einer Zielmatrix

Sie können einer Zielmatrix Objekte hinzufügen, um eine Konfigurationsmatrix mit Ziel- und Benchmarkwerten auf Basis der besten Musterübereinstimmung festzulegen, und erhalten so eine konfigurierbare Liste von Objektparametern.

Beim Erstellen einer Zielmatrix

Zielmatrizen bieten die Möglichkeit, dass eine Instanz ein Ziel oder eine Benchmark mit einer Standardmethode oder mit einer optionalen Zielmatrix auflöst. Die Auflösung des Ziels und der Benchmark erfolgt anhand der besten Übereinstimmung basierend auf den aufgelösten Objektfeldern, z. B. einer Operation oder Ressource.

In der Matrix können Zeilen mit Werten für alle Objektparameter und auch für das Ziel und die Benchmark hinzugefügt, bearbeitet, gelöscht und angezeigt werden.

Öffnen vorhandener Zielmatrizen

Sie können vorhandene Zielmatrizen über die Registerkarte "Modellierung" (Modeling) im Komponentenkatalog öffnen. Weitere Informationen zum Öffnen vorhandener Zielmatrizen finden Sie unter "Öffnen vorhandener Komponenten".

Felddefinitionen für die Zielmatrix

Diese Tabelle definiert Felder, die zum Erstellen einer Zielmatrix verwendet werden.

Feld

Definition

Typ

Name

Eindeutiger Name für die Zielmatrix.

Erforderlich

Columns

Wenn Sie auf die Schaltfläche "Mehr" (More) klicken, wird das modale Fenster "Columns" angezeigt. Hier können Sie Spalten für die Zielmatrix definieren. Weitere Informationen finden Sie unter "Hinzufügen von Zeilen und Spalten zu einer Zielmatrix".

Optional

FileName

Zeigt einen der Zielmatrix zugewiesenen Dateinamen an.

Nur Anzeige

Rows

Wenn Sie auf die Schaltfläche "Mehr" (More) klicken, wird das modale Fenster "Rows" angezeigt. Hier können Sie Zeilen zur Zielmatrix hinzufügen. Weitere Informationen finden Sie unter "Hinzufügen von Zeilen und Spalten zu einer Zielmatrix".

Optional

Vorgehensweise zum Erstellen einer Zielmatrix

Gehen Sie folgendermaßen vor, um eine Zielmatrix zu erstellen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Neu (New) und wählen Sie Zielmatrixelement (Target Matrix Item) aus. Im Arbeitsbereich wird eine leere Zielmatrix angezeigt.
3. Geben Sie in das Feld Name einen Namen für die Zielmatrix ein.
4. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
5. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
