Hinzufügen von Zeilen und Spalten zu einer Zielmatrix

In Portal Studio können Sie Zeilen und Spalten zu einer Zielmatrix hinzufügen.

Beim Hinzufügen von Zeilen und Spalten zu einer Zielmatrix

Alle Zielmatrizen verwenden gemeinsame Spalten. Wenn Sie einer Zielmatrix eine Spalte hinzufügen, wird die Spalte allen Zielmatrizen hinzugefügt.

Mithilfe des modalen Fensters "Rows" können Sie Daten aus einer Excel-Tabellenkalkulation importieren. Sie können eine Kombination aus importierten Daten und hinzugefügten Daten verwenden.

Vorgehensweise zum Hinzufügen von Zeilen zu einer Zielmatrix

Gehen Sie folgendermaßen vor, um Zeilen zu einer Zielmatrix hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Wählen Sie die Registerkarte Modellierung (Modeling) im Komponentenkatalog aus und doppelklicken Sie auf eine Zielmatrix. Die Zielmatrix wird im Arbeitsbereich angezeigt.
3. Klicken Sie auf Mehr (More) für Zeilen (Rows). Das modale Fenster "Rows" wird angezeigt.
4. Fügen Sie Zeilen hinzu oder importieren Sie Daten aus einer Excel-Tabellenkalkulation?

Hinzufügen von Zeilen
a. Klicken Sie auf Zeile hinzufügen (Add Row). Eine neue Zeile wird im modalen Fenster "Rows" angezeigt.
b. Geben Sie in jede Zelle einen Wert ein.
c. Wiederholen Sie die Schritte a und b, um weitere Zeilen hinzuzufügen.
d. Gehen Sie zu Schritt 5.

Importieren von Daten aus einer Excel-Tabellenkalkulation
a. Klicken Sie auf Aus Excel importieren (Import from Excel). Die importierten Daten werden im modalen Fenster "Rows" angezeigt.
b. Gehen Sie zu Schritt 5.
5. Klicken Sie auf OK. Das modale Fenster "Zeilen" (Rows) wird geschlossen.
6. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Hinzufügen von Spalten zu einer Zielmatrix

Gehen Sie folgendermaßen vor, um Spalten zu einer Zielmatrix hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Wählen Sie die Registerkarte Modellierung (Modeling) im Komponentenkatalog aus und doppelklicken Sie auf eine Zielmatrix. Die Zielmatrix wird im Arbeitsbereich angezeigt.
3. Klicken Sie auf Mehr (More) für Spalten (Columns). Das modale Fenster "Spalten" (Columns) wird angezeigt.
4. Klicken Sie auf Hinzufügen. In der Liste wird eine neue Spalte angezeigt.
5. Geben Sie im Feld Name einen Namen für die Spalte ein.
6. Geben Sie in das Feld Überschrift (Caption) eine Überschrift für die Spalte ein.
7. Wiederholen Sie die Schritte 4 bis 6, um weitere Spalten hinzuzufügen.
8. Klicken Sie auf OK. Das modale Fenster "Spalten" (Columns) wird geschlossen.
9. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
