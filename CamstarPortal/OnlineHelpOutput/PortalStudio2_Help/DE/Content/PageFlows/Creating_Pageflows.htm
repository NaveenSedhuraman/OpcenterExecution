Erstellen einer Seitenfolge

Seitenfolgen (Pageflows) werden verwendet, um Seiten miteinander zu verknüpfen und so die gewünschte Seitenabfolge zu erstellen.

Beim Erstellen einer Seitenfolge

Wenn Sie eine neue Seite auf die Seitenfolge ziehen, wird automatisch eine Verknüpfung zu der Seite erstellt, die derzeit in der Seitenfolge hervorgehoben ist.

Sie können den Übergang zwischen zwei Seiten verwalten, indem Sie den Bedingungseditor (Conditions Editor) verwenden, um Bedingungsausdrücke zu erstellen. Klicken Sie auf den Pfeil zwischen den Seiten, um die Eigenschaft "RuleCondition" im Fenster "Eigenschaften" (Properties) anzuzeigen. Wenn Sie auf die Schaltfläche "Mehr" (More) für die Eigenschaft "RuleCondition" klicken, wird der Bedingungseditor (Conditions Editor) angezeigt. Weitere Informationen zur Verwendung des Bedingungseditors finden Sie unter "Bedingungseditor (Conditions Editor)".

Sie können die Eigenschaften von Seitenfolgen und die Seitenknoten innerhalb von Seitenfolgen im Fensterbereich "Eigenschaften" (Properties) anzeigen oder bearbeiten. Informationen zum Ändern der Eigenschaften von Seitenfolge finden Sie unter "Seitenfolge-Eigenschaften".

Öffnen vorhandener Seitenfolgen

Seitenfolgen, die Sie erstellt haben, können über den Katalog mit den Benutzeroberflächenkomponenten geöffnet werden. Weitere Informationen zum Öffnen vorhandener Seitenfolgen finden Sie unter "Öffnen vorhandener Komponenten".

Vorgehensweise zum Erstellen einer Seitenfolge

Gehen Sie folgendermaßen vor, um eine Seitenfolge zu erstellen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Neu (New) und wählen Sie Seitenfolge (Pageflow) aus. Im Arbeitsbereich wird eine leere Seitenfolge angezeigt.
3. Geben Sie einen Namen für die Seitenfolge in das Feld PageFlowContent Name im Fensterbereich Name ein.
4. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
