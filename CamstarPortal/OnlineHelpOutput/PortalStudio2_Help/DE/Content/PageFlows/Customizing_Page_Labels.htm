Anpassen von Seitenbeschriftungen

Sie können die Titel und Beschreibungen, die auf Seiten innerhalb einer Seitenfolge angezeigt werden, im Fensterbereich "Eigenschaften" (Properties) anpassen.

Beim Anpassen von Seitenbeschriftungen

Seitenbeschriftungen werden durch Ändern der Eigenschaften von Seitenfolge-Seitenknoten angepasst. Die folgenden Eigenschaften dienen zum Anpassen von Seitenbeschriftungen:

• Mit den Eigenschaften "PageTitleLabelText" und "PageTitleLabelName" wird der Titel konfiguriert, der für eine Seite angezeigt wird.
• Mit den Eigenschaften "PageDescriptionLabelText" und "PageDescriptionLabelName" wird die Beschreibung unterhalb eines Seitentitels konfiguriert.

Weitere Informationen zu den Eigenschaften von Seitenfolge-Seitenknoten finden Sie unter "Beschreibungen der Eigenschaften von Seitenfolge-Seitenknoten".

Note: Die Eigenschaften des Typs "LabelText" haben Vorrang vor den Eigenschaften des Typs "LabelName", wenn sowohl Text eingegeben als auch eine Beschriftung ausgewählt wurde. Weitere Informationen zur Verwendung des Popups "Beschriftungssuche" (Label Lookup) zum Auswählen von Beschriftungen für die Eigenschaften "PageTitleLabelName" und "PageDescriptionLabelName" finden Sie unter "Suchen nach Seitenaktionsbeschriftungen".

Vorgehensweise zum Anpassen von Seitenbeschriftungen

Gehen Sie folgendermaßen vor, um Seitenbeschriftungen anzupassen:

1. Greifen Sie auf Portal Studio zu.
2. Erweitern Sie Seitenfolgen (Pageflows) auf der Registerkarte "Komponenten" (Components) des Katalogs mit den Benutzeroberflächenkomponenten.
3. Doppelklicken Sie auf eine Seitenfolge. Die Seitenfolge wird im Arbeitsbereich angezeigt.
4. Wählen Sie einen Seitenknoten aus der Seitenfolge aus. Die Seitenknoteneigenschaften für den ausgewählten Seitenknoten werden im Fensterbereich "Eigenschaften" (Properties) angezeigt.
5. Geben Sie im Fensterbereich "Eigenschaften" (Properties) in das Feld PageTitleLabelText einen Titel ein.
6. Geben Sie im Fensterbereich "Eigenschaften" (Properties) in das Feld PageDescriptionLabelText eine Beschreibung ein.
7. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
