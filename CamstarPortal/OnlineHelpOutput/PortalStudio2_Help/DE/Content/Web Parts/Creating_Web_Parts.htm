Erstellen eines Webparts

Sie können Webparts in Portal Studio erstellen, die Steuerelemente und Felder auf Seiten einbinden.

Beim Erstellen eines Webparts

Sie können Felder und Steuerelemente direkt auf ein Webpart, nicht jedoch direkt auf eine Seite ziehen.

Nach dem Speichern eines Webparts auf einer Seite können Sie das Webpart durch Klicken auf die Schaltfläche "Ausführen" (Run) testen. Weitere Informationen zum Testen von Webparts und Seiten finden Sie unter "Testen einer Seite".

Siemens empfiehlt, im Fensterbereich "Eigenschaften" (Properties) einen Namen und einen Titel einzugeben, wenn Sie ein Webpart erstellen. Das Feld "Name" definiert den Namen des Webparts. Die Eigenschaft "Title" definiert den Text, der als Titel des Webparts angezeigt wird. Weitere Informationen zu bestimmten Webpart-Eigenschaften finden Sie unter "Webpart-Eigenschaften".

Öffnen vorhandener Webparts

Vorhandene Webparts können im Komponentenkatalog geöffnet werden. Weitere Informationen zum Öffnen vorhandener Webparts finden Sie unter "Öffnen vorhandener Komponenten".

Neuladen eines Webparts

Sie können ein Webpart neu laden, indem Sie in der globalen Symbolleiste von Portal Studio auf die Schaltfläche "Webpart neu laden" (Reload Web Part) klicken. Beim Neuladen eines Webparts wird es auf den zuvor gespeicherten Zustand zurückgesetzt. Sie können Webparts nur neu laden, wenn sie Teil einer Seite sind.

Vorgehensweise zum Erstellen eines Webparts

Gehen Sie folgendermaßen vor, um ein Webpart zu erstellen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Neu (New) und wählen Sie Webpart aus. Im Arbeitsbereich wird ein leeres Webpart angezeigt.
3. Geben Sie im Bereich "Eigenschaften" (Properties) im Feld Name einen Namen für das Webpart ein.
4. Geben Sie im Fensterbereich "Eigenschaften" (Properties) in das Feld Titel (Title) die neue Beschriftung ein.
5. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
