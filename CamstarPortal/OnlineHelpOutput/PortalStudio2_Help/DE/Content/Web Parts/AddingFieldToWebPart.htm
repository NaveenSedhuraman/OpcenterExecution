Hinzufügen eines Felds zu einem Webpart

Felder werden zum Eingeben von Informationen verwendet und sind in einem Webpart enthalten. Wenn Sie einem Webpart z. B. ein Feld namens "Container" hinzufügen, erlauben Sie einem Benutzer, einen Containernamen einzugeben.

Beim Hinzufügen eines Felds zu einem Webpart

Feldausdruck und Steuerungstyp werden automatisch festgelegt, wenn Sie einem Webpart ein Feld hinzufügen.

Felder können nur zu einem Webpart hinzugefügt werden, nicht jedoch direkt zu einer Seite.

Note: Sie können einem Webpart nur dann ein Feld hinzufügen, wenn Sie sich in der Ansicht "Steuerelemente" (Controls) befinden. Sie können ein Feld nicht auf eine Zelle auf der Seite ziehen, wenn Sie sich auf der Seite "Design" (Design) befinden.

Vorgehensweise zum Hinzufügen eines Felds zu einem Webpart

Gehen Sie folgendermaßen vor, um ein Feld zu einem Webpart hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Doppelklicken Sie auf ein Webpart oder auf eine Seite, die ein Webpart enthält. Das Webpart bzw. die Seite wird im Arbeitsbereich angezeigt.
3. Gewährleisten Sie, dass Steuerelemente (Controls) in der Liste "Ansicht" (View) ausgewählt ist.
4. Klicken Sie auf die Registerkarte Felderverzeichnis (Fields Directory) in der Toolbox. Im Fensterbereich "Toolbox" wird eine Liste mit Services angezeigt.
5. Erweitern Sie den Service, der das Feld enthält, das Sie hinzufügen möchten.
6. Wählen Sie ein Feld aus und ziehen Sie es auf eine Zelle des Webparts.
7. Wiederholen Sie die Schritte 6 bis 7, um weitere Felder zum Webpart hinzuzufügen.
8. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
