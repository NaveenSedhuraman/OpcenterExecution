Hinzufügen eines Steuerelements zum Suchen von Dateien

Sie können Portalseiten ein Steuerelement zum Suchen von Dateien hinzufügen, um nach Dateien auf Portalseiten zu suchen und diese hochzuladen.

Beim Hinzufügen eines Steuerelements zum Suchen von Dateien

Steuerelemente zum Suchen von Dateien (File Browse) enthalten zwei Felder: "Suchmodus" (Browse Mode) und "Datei suchen" (Locate File).

Steuerelemente können nur zu einem Webpart hinzugefügt werden. Es ist nicht möglich, Steuerelemente direkt zu einer Seite hinzuzufügen. Weitere Informationen finden Sie unter "Hinzufügen eines Steuerelements zu einem Webpart".

Important: Ein Steuerelement zum Suchen von Dateien (File Browse) ist für Modellierungsseiten erforderlich, auf denen Dateien hochgeladen werden können.

Vorgehensweise zum Hinzufügen eines Steuerelements zum Suchen von Dateien

Gehen Sie folgendermaßen vor, um ein Steuerelement zum Suchen von Dateien hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Doppelklicken Sie auf eine Seite oder ein Webpart. Die Seite bzw. das Webpart wird im Arbeitsbereich angezeigt.
3. Klicken Sie in der Toolbox auf ein Steuerelement zum Suchen von Dateien (File Browse) und ziehen Sie es auf eine Zelle eines Webparts. Das Steuerelement zum Suchen von Dateien (File Browse) wird auf dem Webpart angezeigt.
4. Geben Sie im Bereich "Eigenschaften" (Properties) im Feld Name einen Namen für das Steuerelement ein.
5. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
