Verwenden der Farbauswahl

Anhand des Steuerelements zur Farbauswahl (Color Picker) können Sie für Aufgaben wie die Farbkodierung von Elementen auf einer Portalseite ein Farbauswahlfeld zu einem Webpart hinzufügen.

Die Farbauswahl wird im Toolboxbereich in der Liste der verfügbaren Steuerelemente angezeigt. Weitere Informationen zum Hinzufügen der Farbauswahl zu einem Webpart finden Sie unter "Hinzufügen eines Steuerelements zu einem Webpart".

Beim Verwenden der Farbauswahl

Die Farbauswahl enthält sowohl ein Feld zur Eingabe von Hexadezimal-Farbwerten als auch ein Farbsymbol zur Auswahl einer Farbe aus einer Reihe von Farbmustern. Hexadezimalwerte müssen mit dem Nummernzeichen (#) beginnen, wenn Sie die Werte manuell eingeben. Ein gültiger Wert ist z. B. "#ffff00".

Wenn Sie auf das leere Symbol klicken, wird eine Farbpalette mit Thema- und Standard-Farbmustern angezeigt. Außerdem werden die derzeit ausgewählte Farbe und ihr Hexadezimalwert angezeigt. Sie können auf "Weitere Farben" (More Colors) klicken, um eine Farbe aus einem Farbenrad auszuwählen, oder auf "Verlauf" (History) klicken, um eine zuvor ausgewählte Farbe anzuzeigen und auszuwählen. In der Farbpalette werden bis zu 28 zuvor ausgewählte Farben gespeichert.
