Neuladen eines Webparts

In Portal Studio können Sie ein Webpart aus dem Komponentenkatalog neu laden.

Beim Neuladen eines Webparts

Das Neuladen eines Webparts funktioniert nur, wenn das Webpart:

• Teil einer Seite ist.
• als unabhängiges Webpart im Komponentenkatalog aufgeführt ist.
• denselben Namen wie das im Komponentenkatalog aufgeführte Webpart aufweist.
Note: Sie können ein Webpart nur neu laden, wenn Sie sich in der Ansicht "Steuerelemente" (Controls) befinden. Wenn Sie sich in der Ansicht "Design" befinden, können Sie ein Webpart nicht in eine Zelle auf der Seite laden.

Vorgehensweise zum Neuladen eines Webparts

Gehen Sie folgendermaßen vor, um ein Webpart neu zu laden:

1. Greifen Sie auf Portal Studio zu.
2. Doppelklicken Sie auf eine Seite mit einem Webpart. Die Seite wird im Arbeitsbereich angezeigt.
3. Gewährleisten Sie, dass Steuerelemente (Controls) in der Liste "Ansicht" (View) ausgewählt ist.
4. Wählen Sie ein Webpart auf der Seite aus und ändern Sie es.
5. Klicken Sie auf Webpart neu laden (Reload Web Part). Alle Änderungen seit dem letzten Speichern werden entfernt.
