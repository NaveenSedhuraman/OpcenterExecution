Hinzufügen eines Flyouts zu einer Seite

Sie können einer Seite das Dropdown-Steuerelement "Abschnitt" (Section) hinzufügen. Dieses dient als Container für Webparts, die auf einer Portalseite als Flyoutfenster angezeigt werden.

Beim Hinzufügen eines Flyouts zu einer Seite

Steuerelemente können nur zu einem Webpart hinzugefügt werden. Es ist nicht möglich, Steuerelemente direkt zu einer Seite hinzuzufügen. Weitere Informationen zum Hinzufügen von Steuerelementen zu einem Webpart finden Sie unter "Hinzufügen eines Steuerelements zu einem Webpart".

Die Eigenschaften "AltBeschriftungsnameOffen" (AltOpenLabelName) und "AltBeschriftungstextOffen" (AltOpenLabelText) bestimmen, welcher Text beim geöffneten Flyout als Beschriftung angezeigt wird. Die Eigenschaften "AltBeschriftungsnameGeschlossen" (AltCloseLabelName) und "AltBeschriftungstextGeschlossen" (AltCloseLabelText) bestimmen, welcher Text beim geschlossenen Flyout als Beschriftung angezeigt wird.

Wenn nur eine der Eigenschaften definiert wird, dann wird beim geöffneten und beim geschlossenen Flyout derselbe Text als Beschriftung angezeigt. Die Beschriftung des Flyouts enthält keinen Text, wenn keine dieser Eigenschaften definiert wird. Weitere Informationen zu Steuerelementeigenschaften finden Sie unter "Steuerelementeigenschaften".

Der im Feld "AltBeschriftungstextOffen" (AltOpenLabelText) eingegebene Text überschreibt die im Feld "AltBeschriftungsnameOffen" (AltOpenLabelName) ausgewählte Wörterbuchbeschriftung, wenn sowohl Text als auch Beschriftungen angegeben sind. Der im Feld "AltBeschriftungstextGeschlossen" (AltCloseLabelText) eingegebene Text überschreibt die im Feld "AltBeschriftungsnameGeschlossen" (AltCloseLabelName) ausgewählte Wörterbuchbeschriftung, wenn sowohl Text als auch Beschriftungen angegeben sind.

Vorgehensweise zum Hinzufügen eines Flyouts zu einer Seite

Gehen Sie folgendermaßen vor, um ein Flyout zu einer Seite hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Komponentenkatalog ausgeblendet ist. Der Komponentenkatalog wird angezeigt.
3. Doppelklicken Sie auf eine Seite oder ein Webpart. Die Seite bzw. das Webpart wird im Arbeitsbereich angezeigt.
4. Klicken Sie auf Toolbox anzeigen/ausblenden (Show/Hide Toolbox), wenn die Toolbox ausgeblendet ist. Die Toolbox wird angezeigt.
5. Klicken Sie auf ein Dropdown-Steuerelement "Abschnitt" (Section) und ziehen Sie es auf eine Zelle eines Webparts. Das Dropdown-Steuerelement "Abschnitt" (Section) wird im Webpart angezeigt.
6. Klicken Sie auf Eigenschaften anzeigen/ausblenden (Show/Hide Properties), wenn der Bereich "Eigenschaften" (Properties) ausgeblendet ist. Der Bereich "Eigenschaften" (Properties) wird angezeigt.
7. Geben Sie im Bereich "Eigenschaften" (Properties) im Feld Name einen Namen für das Steuerelement ein.
8. Geben Sie im Feld AltBeschriftungsnameOffen (AltOpenLabelName), AltBeschriftungstextOffen (AltOpenLabelText), AltBeschriftungsnameGeschlossen (AltCloseLabelName) oder AltBeschriftungstextGeschlossen (AltCloseLabelText) einen Text ein oder wählen Sie eine Wörterbuchbeschriftung aus.
9. Wiederholen Sie Schritt 8, um nach Bedarf weiteren Beschriftungstext hinzuzufügen.
10. Klicken Sie nach Bedarf auf Webparts und ziehen Sie sie auf das Dropdown-Steuerelement "Abschnitt" (Section).
11. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
