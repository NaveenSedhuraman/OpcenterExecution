Hinzufügen eines Feldausdrucks zu einem Steuerelement

In Portal Studio können Sie einem Webpart Steuerelemente hinzufügen. Einem Steuerelement im Webpart können Sie anschließend Feldausdrücke hinzufügen und es so mit bestimmten Servicefeldern verknüpfen.

Beim Hinzufügen eines Feldausdrucks zu einem Steuerelement

Datengesteuerte Steuerelemente weisen im Fensterbereich "Eigenschaften" (Properties) die Eigenschaft "FieldExpressions" auf, mit der das Steuerelement mit bestimmten Servicefeldern verknüpft wird.

Nicht alle Steuerelemente verwenden Feldausdrücke. Der Registerkartencontainer ist z. B. ein Steuerelement, das nur eine andere Seite enthält und keinen Service verarbeitet. Weitere Informationen finden Sie unter "Feldausdrücke".

Vorgehensweise zum Hinzufügen eines Feldausdrucks zu einem Steuerelement

Gehen Sie folgendermaßen vor, um einen Feldausdruck zu einem Steuerelement hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Doppelklicken Sie auf eine Seite oder ein Webpart mit einem Steuerelement. Die Seite bzw. das Webpart wird im Arbeitsbereich angezeigt.
3. Wählen Sie im Arbeitsbereich ein Steuerelement aus.
4. Klicken Sie im Bereich "Eigenschaften" (Properties) auf Mehr (More) für Feldausdrücke (FieldExpressions). Das Popup "Feldausdruckerstellung" (Field Expressions Builder) wird angezeigt.
5. Wählen Sie einen Feldausdruck (Field Expression) aus und klicken Sie auf OK. Das Popup "Feldausdruckerstellung" wird geschlossen und der Feldausdruck wird im Bereich "Eigenschaften" zum Feld "FieldExpressions" hinzugefügt.
6. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
