Hinzufügen eines Steuerelements zu einem Webpart

Die Steuerelemente umfassen die Standardsteuerelemente für das Web, die um die Eigenschaften von Opcenter Execution Core ergänzt wurden.

Steuerelemente, die Felder oder Daten anzeigen, erfordern möglicherweise Feldausdrücke. Steuerelemente, die zu Layoutzwecken vorgesehen sind, z. B. Registerkarten, erweiterbare Abschnitte oder Fly-Outs, erfordern normalerweise keine Feldausdrücke.

Beim Hinzufügen eines Steuerelements zu einem Webpart

Steuerelemente können nur zu einem Webpart hinzugefügt werden. Es ist nicht möglich, Steuerelemente direkt zu einer Seite hinzuzufügen.

Einige Steuerelemente, z. B. Registerkarten oder Fly-Outs, interagieren nicht mit einem Service und erfordern keine Feldausdrücke.

Sie müssen ein Steuerelement auswählen, das mit dem Feldausdruck kompatibel ist, den Sie verwenden möchten. Weitere Informationen finden Sie unter "Hinzufügen eines Feldausdrucks zu einem Steuerelement".

Vorgehensweise zum Hinzufügen eines Steuerelements zu einem Webpart

Gehen Sie folgendermaßen vor, um ein Steuerelement zu einem Webpart hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Doppelklicken Sie auf ein Webpart oder auf eine Seite, die ein Webpart enthält. Das Webpart bzw. die Seite wird im Arbeitsbereich angezeigt.
3. Klicken Sie in der Toolbox auf ein Steuerelement und ziehen Sie es auf eine Zelle eines Webparts. Das Steuerelement wird in dem Webpart angezeigt.
4. Geben Sie im Bereich "Eigenschaften" (Properties) im Feld Name einen Namen für das Steuerelement ein.
5. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
