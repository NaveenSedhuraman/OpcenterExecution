Anzeigen der Positionen referenzierter Webparts

Sie können eine Liste der Komponenten anzeigen, die ein Webpart referenzieren, um festzustellen, auf welche Komponenten sich Aktualisierungen auswirken, die in der Ansicht "Wo genutzt" (Where Used) im Arbeitsbereich am Webpart vorgenommen werden.

Beim Anzeigen der Positionen referenzierter Webparts

Referenzierte Webparts sind schreibgeschützte Kopien, die auf eine Masterkopie des Webparts verweisen. So können Sie statt mehrerer einzelner Webparts ein Webpart an einem zentralen Standort verwalten.

Die Ansicht "Wo genutzt" (Where Used) steht in der Dropdown-Liste für den Bearbeitungsmodus zur Verfügung, wenn Sie ein Webpart öffnen. Bei Auswahl der Registerkarte "Wo genutzt" (Where Used) werden zwei Tabellen angezeigt, in denen die virtuellen Seiten und die anderen Webparts, die das aktuelle Webpart referenzieren, mit den jeweils zugehörigen Arbeitsbereichen aufgelistet sind.

Note: Sie können auf die Spaltenköpfe für Name und Arbeitsbereich klicken, um die aufgelisteten Komponenten zu sortieren.

Vorgehensweise zum Anzeigen der Positionen referenzierter Webparts

Gehen Sie folgendermaßen vor, um Positionen referenzierter Webparts anzuzeigen:

1. Greifen Sie auf Portal Studio zu.
2. Erweitern Sie Webparts.
3. Doppelklicken Sie auf ein Webpart. Das Webpart wird im Arbeitsbereich angezeigt.
4. Wählen Sie den Bearbeitungsmodus Wo genutzt (Where Used) aus. Der Arbeitsbereich wird aktualisiert und zeigt die Seiten sowie die anderen Webparts an, die das Webpart referenzieren.
