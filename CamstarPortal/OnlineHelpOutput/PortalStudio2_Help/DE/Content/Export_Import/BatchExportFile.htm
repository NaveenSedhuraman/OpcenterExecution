Durchführen von Batch-Exporten

Mit dem Batch-Export können Sie mehrere Seiten, Webparts und Seitenfolgen als einzelne XML-Datei oder ZIP-Datei mit mehreren XML-Dateien exportieren.

Beim Durchführen von Batch-Exporten

Beim Speichern eines Batch-Exports als ZIP-Datei wird eine ZIP-Datei erstellt, die für jede Seite, jedes Webpart und jede Seitenfolge, die im Batch-Export enthalten ist, eine XML-Datei enthält.

Sie müssen in der Liste "Personalisierungstyp für Export" (Personalization Type to Export) mindestens einen Arbeitsbereich auswählen.

Das Popup "Exportfortschritt..." (Export Progress) wird angezeigt, wenn Sie auf die Schaltfläche "Start" klicken. Nach Abschluss des Exports wird das Popup ausgeblendet.

Der Standarddateiname für die XML- oder ZIP-Datei ist BatchExport.xml oder BatchExport.zip. Die Anwendung fordert Sie dazu auf, die Datei zu öffnen oder zu speichern, oder sie lädt die Datei automatisch herunter. Ihr Internetbrowser bestimmt, ob die Anwendung ein Popup öffnet, wie es aussehen soll und was in den Optionen "Öffnen" und "Speichern" genau steht.

Important: Beim Batch-Export werden abhängige Seiten nicht automatisch gefunden und exportiert. Wenn Sie z. B. eine Seite mit abhängigen Seiten exportieren, die in einer Seitenumleitungsaktion oder von einem Flyout-Steuerelement referenziert werden, werden die abhängigen Seiten nicht gefunden und exportiert. Sie müssen die abhängigen Seiten manuell als Bestandteil des Exports auswählen, damit die importierte Seite korrekt funktioniert.

Fenster "Batch-Export" (Batch Export)

Im linken Bereich des Fensters "Batch-Export" (Batch Export) werden alle für den Batch-Export verfügbaren Seiten, Webparts und Seitenfolgen auf Registerkarten angezeigt. Die auf einer Registerkarte ausgewählten Komponenten verbleiben in der Auswahl, wenn Sie auf eine andere Registerkarte wechseln.

Vorgehensweise zum Durchführen von Batch-Exporten

Gehen Sie folgendermaßen vor, um einen Batch-Export durchzuführen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf die Schaltfläche Batch-Export (Batch Export). Das Fenster "Batch-Export" (Batch Export) wird angezeigt.
3. Wählen Sie die zu exportierenden Elemente auf den Registerkarten "Seiten" (Pages), "Webparts" (Web Parts) und "Seitenfolgen" (Pageflows) aus.
4. Wählen Sie in der Liste Personalisierungstyp für Export (Personalization Type to Export) mindestens einen Arbeitsbereich aus.
5. Aktivieren Sie das Kontrollkästchen In Datei exportieren (Export to File).
6. Wählen Sie XML oder Zip für das Dateiformat aus.
7. Klicken Sie auf Start. Das Popup "Exportfortschritt" (Export Progress) wird angezeigt. Das Popup "Exportfortschritt..." (Export Progress) wird geschlossen und in der Anwendung wird eine Erfolgsmeldung angezeigt, wenn der Exportprozess abgeschlossen ist. Ihr Browser fordert Sie auf, die Datei zu öffnen oder zu speichern. Ihr Internetbrowser bestimmt, ob die Anwendung ein Popup öffnet, wie es aussehen soll und was in den Optionen "Öffnen" und "Speichern" genau steht.
