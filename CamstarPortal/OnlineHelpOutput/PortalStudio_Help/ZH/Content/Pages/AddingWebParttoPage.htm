向页面添加 Web 部件

Web 部件可以作为本地副本或作为参考添加到页面。

在向页面添加 Web 部件时

控件和字段无法直接拖放到页面上。必须将控件和字段放在 Web 部件上。

引用的 Web 部件是只读副本，它们引用 Web 部件的主副本。通过这些部件，可在一个位置维护一个 Web 部件，而不是维护多个单独的 Web 部件。

通常，应该将 Web 部件添加为：

• 参考（当 Web 部件在所有页面上相同并在一个位置维护时）。有关查看引用 Web 部件的组件列表的信息，请参考“查看引用的 Web 部件位置”。
• 副本（当 Web 部件是模板或使用一个 Web 部件创建类似的 Web 部件时）。

测试带有参考 Web 部件的页面时，将始终显示参考 Web 部件的开发者内容。

发布带有参考 Web 部件的页面时，将始终显示参考 Web 部件的已发布内容。

如何向页面添加 Web 部件

按照以下步骤将 Web 部件添加到页面：

1. 访问 Portal Studio。
2. 如果“用户界面组件”目录处于隐藏状态，单击显示/隐藏目录。将显示“用户界面组件”目录。
3. 双击页面。 随即工作区域中将显示该页面。
4. 在“用户界面组件”目录中展开 Web 部件。
5. 选择 Web 部件，然后将其拖放到单元上。随即单元中将显示 Web 部件。
6. 单击保存。应用程序将保存更改并显示成功消息。
