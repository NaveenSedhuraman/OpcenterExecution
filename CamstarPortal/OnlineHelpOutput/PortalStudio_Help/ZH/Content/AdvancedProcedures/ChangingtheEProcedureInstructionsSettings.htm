展开电子流程指示

通过 Portal Studio，可以配置在默认情况下展开还是折叠电子流程指示。默认情况下将折叠电子流程指示。

在展开电子流程指示时

配置电子流程指示需要修改电子流程 UI 虚拟页面。Portal Studio 中的虚拟页面包含页面的所有部分。配置电子流程指示的页面是 TaskInstructionsField，未设置标签。TaskInstructionsField 显示为不带边框的黄色矩形。

如何在默认情况下展开电子流程指示

默认情况下，按照以下步骤展开电子流程指示：

1. 访问 Portal Studio。
2. 如果“用户界面组件”目录处于隐藏状态，单击显示/隐藏目录。将显示“用户界面组件”目录。
3. 展开全部页面分组并双击 EProcedureVPR2。随即工作区域中将显示 EprocedureVPR2 页。
Note: 如果使用的是旧版本的页面，则必须更新 EProcedureVP。有关信息，请参考《Opcenter Execution Medical Device and Diagnostics Shop Floor 用户指南》或《Opcenter Execution Core Shop Floor 用户指南》。
4. 选择 TaskInstructionsField。随即“属性”窗格中将显示 TaskInstructionsField 的设置。
5. 为 DefaultState 选择已展开。
6. 为 ExpandFull 选择 True 。
7. 单击保存。应用程序将保存更改并显示成功消息。
