定义会话变量

可以在 Portal Studio 中为页面或 Web 部件创建会话变量。可以使用会话变量来收集数据并将收集的数据提供给另一组件。

在定义会话变量时

会话变量在 SessionVariables 弹出窗口中表示，当在工作区域中选择页面、Web 部件或弹出窗口时，可以从“属性”窗格访问该弹出窗口。

会话变量在其定义的页面中全局可用。Web 部件中定义的会话变量存储在包含该 Web 部件的页面中。

离开页面之前，会话变量将保留数据。重新加载页面会将会话变量重置为 InitialValue 字段中指定的值。默认情况下，会话变量的初始值为 "null"。

定义条件事件处理程序时设置会话变量。有关定义条件事件处理程序的信息，请参考“定义事件处理程序”。

SessionVariables 弹出窗口字段定义

此表定义 SessionVariables 弹出窗口中的字段。

字段

定义

类型

常规

ID

会话变量的唯一名称。

必需

InitialValue

确定加载页面时会话变量的起始值。每次重新加载页面时，会话变量都将重置为该值。默认情况下，会话变量的初始值为 "null"。

可选

IsArray

表明会话变量是否在数组中。

可选

类型

确定预期的对象类型，如 "NamedObjectRef" 或 "System.String"。默认情况下，会话变量的类型为 "System.Object"。

可选

如何定义会话变量

按照以下步骤定义会话变量：

1. 访问 Portal Studio。
2. 如果“用户界面组件”目录处于隐藏状态，单击显示/隐藏目录。将显示“用户界面组件”目录。
3. 双击页面。随即工作区域中将显示该页面。
4. 如果“属性”窗格处于隐藏状态，单击显示/隐藏属性。将显示“属性”窗格。
5. 在工作区域中选择页面、Web 部件或弹出窗口。所选对象的属性将显示在属性窗格中。
6. 在“属性”窗格中单击 SessionVariables 的更多。将显示 SessionVariables 弹出窗口。
7. 单击添加。 新会话变量将添加到列表中，其常规属性将显示在右窗格中。
8. 在 ID 字段中输入会话变量的名称。
9. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
10. 重复步骤 7-9 以添加其他会话变量。
11. 单击确定。 SessionVariables 弹出窗口关闭。
12. 单击保存。应用程序将保存更改并显示成功消息。
