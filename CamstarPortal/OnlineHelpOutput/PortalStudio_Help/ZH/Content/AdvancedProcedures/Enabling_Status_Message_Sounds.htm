启用状态消息声音

可通过 Portal Studio 为显示在门户页面上的状态消息配置声音。在 Portal Studio 中执行功能后出现的状态消息有三种类型：

• 成功
• 错误
• 警告
Note: 默认情况下，门户中禁用状态消息声音。

在启用状态消息声音时

启用状态消息声音后，系统将为每种类型的状态消息播放声音。当然，可以按类型禁用状态消息声音。例如，可以禁用成功和警告声音，但仍可以为错误状态消息播放声音。可通过用户上传的 WAV 或 MP3 文件进一步自定义状态消息声音。上传的文件在上传时将自动设为声音路径。

Note: 某些版本的 Internet Explorer 不支持 WAV 文件。Siemens 建议状态消息声音采用 MP3 文件格式。

如何启用状态消息声音

按照以下步骤启用状态消息声音：

1. 访问 Portal Studio。
2. 单击门户设置按钮。将显示“门户设置”弹出窗口。
3. 从设置列表选择 StatusMessageSettings。将显示 StatusMessageSettings 的属性。
4. 为 PlaySound 选择 True。
5. 单击确定。 “门户设置”弹出窗口关闭。

如何按类型禁用状态消息声音

按照以下步骤按类型禁用状态消息声音：

1. 访问 Portal Studio。
2. 单击门户设置按钮。将显示“门户设置”弹出窗口。
3. 从设置列表选择 StatusMessageSettings。将显示 StatusMessageSettings 的属性。
4. 删除声音路径的文件名。
5. 单击确定。 “门户设置”弹出窗口关闭。

如何上传状态消息声音

按照以下步骤上传状态消息声音：

1. 访问 Portal Studio。
2. 单击门户设置按钮。 将显示“门户设置”弹出窗口。
3. 从设置列表选择 StatusMessageSettings。将显示 StatusMessageSettings 的属性。
4. 单击按钮。将显示“打开”弹出窗口。
5. 选择声音文件，然后单击打开。状态消息类型的声音文件即会上传。
6. 单击确定。 “门户设置”弹出窗口关闭。

如何设置状态消息声音

按照以下步骤设置状态消息声音：

1. 访问 Portal Studio。
2. 单击门户设置按钮。将显示“门户设置”弹出窗口。
3. 从设置列表选择 StatusMessageSettings。将显示 StatusMessageSettings 的属性。
4. 选择声音路径文件。
5. 单击确定。 “门户设置”弹出窗口关闭。
