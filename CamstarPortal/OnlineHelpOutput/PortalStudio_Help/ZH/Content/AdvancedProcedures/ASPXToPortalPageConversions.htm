将 ASPX 文件转换为门户页面

Portal Studio 可用于将 ASPX 文件自动转换为门户页面并将其作为门户页面继续进行编辑和使用。

在将 ASPX 文件转换为门户页面时

以下注释适用于要转换为门户页面的 ASPX 文件：

• 转换实用程序将 Web 应用程序页面中的控件转换为 Portal Studio 中最接近的等效控件类型，并为逻辑部分创建 Web 部件。
• DisplayInFrame=True 的 Web 应用程序按钮将转换为“页面操作”按钮。如果 DisplayInFrame=False，则这些按钮将转换为页面上的按钮控件。
• 尽可能保留相依项，例如当“资源”字段在 Web 应用程序页面中将 DependsOn 属性设为“容器”字段时。
• 页面的隐藏代码不会转换，因为该实用程序仅转换 ASPX 文件。
• 不支持 ASPX 文件的批转换。

如何将 ASPX 文件转换为门户页面

按照以下步骤将 ASPX 文件转换为门户页面。

1. 访问 Portal Studio。
2. 单击转换 ASPX 页面按钮。将显示“打开”弹出窗口。
3. 导航至并选择要转换的 ASPX 文件，然后单击打开按钮。Portal Studio 将自动执行转换并保存该页面。将显示一条成功消息，指示文件已成功转换。
Note: 转换后的页面显示在前缀为 "ASPX_" 的页面列表中。可以在“属性”窗格中重命名该页面。
