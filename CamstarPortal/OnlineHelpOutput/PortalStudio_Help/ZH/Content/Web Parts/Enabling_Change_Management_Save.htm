启用更改管理保存开关

可在 Portal Studio 中向各个“建模”页添加“布尔开关”控件，该控件将在 Modeling 中显示为“更改管理保存”开关。Modeling 用户可使用“更改管理保存”开关打开和关闭选项，以将会话中创建的建模实例与更改管理包相关联。

如何启用更改管理保存开关

“布尔开关”控件默认显示为单选按钮。必须在“属性”窗格中将“布尔开关”控件的 BooleanSwitchDisplayMode 属性设为“滑动”，这些控件才会显示为“更改管理保存”开关。

创建更改管理包之前，“更改管理保存”开关在 Modeling 的“建模”页上一直处于隐藏状态。有关创建包的信息，请参考《Opcenter Execution Medical Device and Diagnostics Change Management 用户指南》或《Opcenter Execution Core Change Management 用户指南》。

只能向 Web 部件添加“布尔开关”控件。不能直接向页面添加控件，但可以向页面上的 Web 部件进行添加。有关信息，请参考“向 Web 部件添加控件”。

可自定义“更改管理开关”左侧和右侧单元中的显示文本，默认情况下，在左侧单元中显示为“关”，在右侧单元中显示为“开”。还可以使用 CSS 类自定义单元的颜色和字体。

Note: Modeling 用户必须具有将内容指派给更改管理包的权限，才能使用“更改管理保存”开关。 有关信息，请参考《Opcenter Execution Medical Device and Diagnostics Modeling 用户指南》或《Opcenter Execution Core Modeling 用户指南》。

如何启用更改管理保存开关

按照以下步骤启用“更改管理保存”开关：

1. 访问 Portal Studio。
2. 如果“用户界面组件”目录处于隐藏状态，单击显示/隐藏目录。将显示“用户界面组件”目录。
3. 双击“建模”页。随即工作区域中将显示“建模”页。
4. 如果工具箱处于隐藏状态，则单击显示/隐藏工具箱。将显示工具箱。
5. 单击“布尔开关”控件并将其拖至 Web 部件的单元上。随即“布尔开关”控件显示在 Web 部件中。
6. 如果“属性”窗格处于隐藏状态，单击显示/隐藏属性。将显示“属性”窗格。
7. 在“属性”窗格的名称字段中输入“布尔开关”控件的名称。
8. 在“属性”窗格的 BooleanSwitchDisplayMode 字段中选择滑动。
9. 单击保存。应用程序将保存更改并显示成功消息。
