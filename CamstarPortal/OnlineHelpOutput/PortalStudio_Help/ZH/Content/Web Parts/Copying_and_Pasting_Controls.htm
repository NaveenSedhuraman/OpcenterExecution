复制和粘贴控件

可以在 Portal Studio 的工作区域中复制和粘贴控件。

在复制和粘贴控件时

粘贴控件时，必须存在具有空白单元的 Web 部件。有关向门户页面添加 Web 部件的信息，请参考“创建 Web 部件”。

可以通过以下操作复制和粘贴控件：

• 使用键盘命令 Ctrl+C 和 Ctrl+V。
• 在工作区域中右键单击并从快捷菜单中选择“复制”或“粘贴”。
• 单击“编辑”，然后选择“复制控件”或“粘贴控件”。
Note: 可以通过单击“编辑”然后选择“剪切控件”来剪切控件。然后，可以将控件粘贴到空白单元中。

如何复制和粘贴控件

按照以下步骤复制和粘贴控件：

1. 访问 Portal Studio。
2. 如果“用户界面组件”目录处于隐藏状态，单击显示/隐藏目录。将显示“用户界面组件”目录。
3. 双击页面或 Web 部件。随即工作区域中将显示该页面或 Web 部件。
4. 在工作区域中选择一个控件。
5. 右键单击该控件并选择复制。
6. 如果需要，在 Web 部件中选择一个空白单元，打开另一个页面或 Web 部件。
7. 右键单击空白单元并选择粘贴。该控件将显示在该单元中。
8. 单击保存。应用程序将保存更改并显示成功消息。
