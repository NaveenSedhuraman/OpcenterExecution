向 Web 部件添加控件

控件将标准 Web 控件与添加到控件中的 Manufacturing 属性相结合。添加控件（而不是字段）的缺点是必须手动设置控件，而添加字段时则会自动设置字段表达式和控件类型。

显示字段或数据的控件通常需要字段表达式。用于布局目的的控件（例如选项卡、展开部分、浮出控件等）通常不需要字段表达式。

在向 Web 部件添加控件时

只能向 Web 部件添加控件。不能直接向页面添加控件。

某些控件（例如选项卡控件或浮出控件）不与服务交互，因此不需要字段表达式。

添加控件时，必须为该控件添加字段表达式。必须选择与要使用的字段表达式兼容的控件。

如何向 Web 部件添加控件

按照以下步骤将控件添加到 Web 部件：

1. 访问 Portal Studio。
2. 如果“用户界面组件”目录处于隐藏状态，单击显示/隐藏目录。将显示“用户界面组件”目录。
3. 双击 Web 部件或包含 Web 部件的页面。随即工作区域中将显示该 Web 部件或页面。
4. 如果工具箱处于隐藏状态，则单击显示/隐藏工具箱。将显示工具箱。
5. 单击并将控件拖放到 Web 部件上的单元上。随即 Web 部件中将显示该控件。
6. 在“属性”窗格的名称字段中输入控件的名称。
7. 重复步骤 5-6 以添加其他控件。
8. 单击保存。应用程序将保存更改并显示成功消息。
