定义参数数据收集

可使用 Web 部件定义参数数据收集。

数据收集定义参考参数数据定义，包含一组数据收集参数。数据收集参数将在事务处理期间对用户可见。将在与收集数据的工步关联的规范定义（在建模中）中，标识数据收集定义和使用它的事务。

在定义参数数据收集时

必须先在“建模”中定义“数据收集”，然后才能在 Portal Studio 中使用。有关信息，请参考《Opcenter Execution Medical Device and Diagnostics Modeling 用户指南》或《Opcenter Execution Core Modeling 用户指南》，并参考《Opcenter Execution Medical Device and Diagnostics Data Collection 用户指南》或《Opcenter Execution Core Data Collection 用户指南》。

如何定义 Web 部件的参数数据收集

按照以下步骤为 Web 部件定义参数数据收集：

1. 访问 Portal Studio。
2. 单击新建，然后选择 Web 部件。随即工作区域中将显示空白 Web 部件。
3. 如果“属性”窗格处于隐藏状态，单击显示/隐藏属性。将显示“属性”窗格。
4. 在属性窗格的 Name 字段中输入 Web 部件的名称。
5. 如果工具箱处于隐藏状态，则单击显示/隐藏工具箱。将显示工具箱。
6. 单击字段目录选项卡。随即工具箱窗格中将显示服务列表。
7. 选择要添加的字段，然后将其拖至 Web 部件中的单元上。
8. 为数据收集中的所有字段重复步骤 7。
9. 单击保存。应用程序将保存更改并显示成功消息。
