向控件添加字段表达式

Portal Studio 可用于向 Web 部件添加控件。随后，可以将字段表达式添加到 Web 部件上的控件，以将该控件链接到特定服务字段。

在向控件添加字段表达式时

数据驱动控件具有 FieldExpression 属性，该属性将控件与特定服务字段链接。

并非所有控件均使用字段表达式。例如，选项卡容器就是仅包含另一页面而不处理服务的控件。有关信息，请参考“字段表达式”。

如何向控件添加字段表达式

按照以下步骤将字段表达式添加到控件：

1. 访问 Portal Studio。
2. 如果“用户界面组件”目录处于隐藏状态，单击显示/隐藏目录。将显示“用户界面组件”目录。
3. 双击包含控件的页面或 Web 部件。随即工作区域中将显示该页面或 Web 部件。
4. 在工作区域中选择一个控件。
5. 在“属性”窗格中，单击 FieldExpressions 的更多。 将显示“字段表达式构建器”弹出窗口。
6. 搜索包含要使用的字段的服务。
7. 选择字段，然后单击确定。将关闭“字段表达式构建器”弹出窗口，字段表达式将添加到“属性”窗格中的 FieldExpressions 字段中。
8. 单击保存。应用程序将保存更改并显示成功消息。
