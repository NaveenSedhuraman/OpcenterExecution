使用颜色选取器

通过“颜色选取器”控件，可以向任务的 Web 部件添加颜色选择字段，例如，门户页面上的颜色编码项。

颜色选取器会显示在“工具箱”窗格的可用控件列表中。有关向 Web 部件添加颜色选取器的信息，请参考“向 Web 部件添加控件”。

在使用颜色选取器时

颜色选取器包含用于输入十六进制颜色值的字段和用于从一系列样本中选择颜色的颜色图标。在手动输入值时，十六进制值必须以英镑符号 (#) 开头，例如，"#ffff00" 是可接受的值。

单击空白图标将显示包含“主题”和“标准”色样的调色板，以及当前选定的颜色及其十六进制值。可以单击“更多颜色”来从颜色盘中选择颜色，或单击“历史”来查看和选择先前所选的颜色。调色板最多保留先前所选的 28 种颜色。
