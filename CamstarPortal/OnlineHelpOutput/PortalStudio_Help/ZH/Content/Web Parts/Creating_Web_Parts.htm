创建 Web 部件

可在 Portal Studio 中创建要在多个页面上使用的 Web 部件。

在创建 Web 部件时

字段和控件可直接拖至 Web 部件，但是不能直接拖至页面。

可通过单击“运行”按钮预览已保存的 Web 部件。

Siemens 建议当创建 Web 部件时在“属性”窗格中输入名称、标题和标题标签。Name 字段用于定义 Web 部件在页面中的实际名称。Title 字段用于定义为 Web 部件显示的标签。TitleLabel 字段用于定义在数据库级别引用的标签。

如何创建 Web 部件

按照以下步骤创建 Web 部件：

1. 访问 Portal Studio。
2. 单击新建，然后选择 Web 部件。随即工作区域中将显示空白 Web 部件。
3. 在“属性”窗格的 Name 字段中输入 Web 部件的名称。
4. 在“属性”窗格的 Title 字段中输入标签。
5. 在“属性”窗格的 TitleLabel 字段中输入标签。
6. 单击保存。应用程序将保存更改并显示成功消息。
