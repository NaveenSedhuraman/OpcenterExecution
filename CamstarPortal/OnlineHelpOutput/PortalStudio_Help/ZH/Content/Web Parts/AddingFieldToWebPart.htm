向 Web 部件添加字段

字段用于将信息输入到 Web 部件中。例如，将“容器”字段添加到 Web 部件，使用户输入容器。

可以将字段添加到新的 Web 部件、现有 Web 部件或已在 Web 页面上使用的 Web 部件。

添加字段（而不是控件）的优点是可以自动设置字段表达式和控件类型。

在向 Web 部件添加字段时

只能向 Web 部件添加字段；不能直接向页面添加字段。

带有快捷方式图标的字段是服务的推荐字段。

红色针表示该字段已在打开的对象中使用。

黄色针表示正在使用该部分中的字段。

如何向 Web 部件添加字段

按照以下步骤将字段添加到 Web 部件：

1. 访问 Portal Studio。
2. 如果“用户界面组件”目录处于隐藏状态，单击显示/隐藏目录。将显示“用户界面组件”目录。
3. 双击 Web 部件或包含 Web 部件的页面。随即工作区域中将显示该 Web 部件或页面。
4. 如果工具箱处于隐藏状态，则单击显示/隐藏工具箱。将显示工具箱。
5. 单击字段目录选项卡。随即工具箱窗格中将显示服务列表。
6. 展开包含要添加的字段的服务。
7. 选择字段，然后将其拖放到 Web 部件上的单元上。
8. 重复步骤 6-7 以将其他字段添加到 Web 部件。
9. 单击保存。应用程序将保存更改并显示成功消息。
