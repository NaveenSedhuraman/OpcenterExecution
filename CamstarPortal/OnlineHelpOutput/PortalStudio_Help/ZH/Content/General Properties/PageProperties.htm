页面属性

通过页面属性可查看或编辑 Portal Studio 页面的属性。在工作区域中选择页面后，“属性”窗格中将显示页面属性。有关“属性”窗格的信息，请参考“属性窗格”。

当选择页面属性时，“属性”窗格下方将出现显示页面属性相关信息的描述窗口。

Note: 某些属性包含可进一步自定义的子属性。子属性将继续在“属性”窗格的描述窗口中显示其父属性的描述。

页面属性描述

此表提供页面属性的描述。

页面属性

属性描述

常规

ActionGroups

与“操作控制 WP”结合使用，控制页面操作的分组。

Actions

页面的操作。可用于定义页面级别的操作和按钮。

AllowEditCondition

允许根据计算表达式（条件）编辑页面。

Behavior

页面的行为类型。

Connections

页面上 Web 部件之间的“数据协定”连接。

DataContractMembers

“数据协定”用于页面之间的通信。

DefaultButton

表示在按下 Enter 按钮时将自动单击的按钮。

Display

通过使页面全屏显示或隐藏页面的各种框架元素来自定义页面显示，例如：

• 全局页脚
• 全局页眉
• 选项卡行
• 菜单栏

重要事项：在 ModelingVP 页上设置的显示属性将适用并控制“CDOFormsSettings - CDO 窗体”弹出窗口中所列的所有页面。有关“CDOFormsSettings - CDO 窗体”弹出窗口的信息，请参考“CDO 窗体设置”。

FullScreen

通过移除应用程序两侧的灰色边栏以使用完全浏览器宽度来展开页面显示的布尔值。 默认情况下设为 False。

注：在查看全屏页面时，字段、按钮和其他页面对象的布局可能会发生变化。基本应用程序页面不是为全屏查看而设计的。可以在设计自己的自定义全屏页面时启用此功能。还可以最大化全屏页面上显示的任何弹出窗口，使其达到完全浏览器宽度。

可以使用 FullScreenMode 设置在这些应用程序框架元素上启用全屏模式：

• 全局页脚
• 全局页眉
• 选项卡行
• 菜单栏

有关门户设置，请参考“默认设置”。

HideFooter

确定页面是否隐藏应用程序的全局页脚的布尔值。 默认情况下设为 False。

HideHeader

确定页面是否隐藏应用程序的全局页眉的布尔值。 默认情况下设为 False。

HideTabs

确定页面是否隐藏应用程序的选项卡行的布尔值。 默认情况下设为 False。

注：如果页面不再显示选项卡行或菜单栏，Siemens 建议在该页面上添加“关闭”按钮。

HideTopMenu

确定页面是否隐藏应用程序的菜单栏的布尔值。 默认情况下设为 False。

注：如果页面不再显示选项卡行或菜单栏，Siemens 建议在该页面上添加“关闭”按钮。

DriverComponent

在建模框架中用于指定在数据更改时强制加载值调用的控件。

DynamicActions

与“操作控制 WP”结合使用，控制动态操作的显示和行为。

DynamicWebparts

包含当前在页面上的动态 Web 部件的列表以及这些 Web 部件的属性。

EProcEnabled

指示此页面是否可作为事务页在电子流程框架内使用。

ForceEditMode

对于 Behavior 设为 "ViewEdit" 的页面，强制该页面以“编辑”模式加载。

HelpFileUrl

页面帮助文件的 URL。

Layout

页面的视觉外观。建议不要直接更改此属性。

MobileBarcodeEnabled

可用于将移动设备中的数据扫描到车间页面上的可编辑字段中。默认情况下在所有车间页面上设为 True。此页面级别设置和全局设置 MobileBarcodeEnabled 必须全部设为 True，条形码扫描图标才会在页面的可编辑字段中显示。如果设为 False，则各个页面上将禁用条形码扫描功能。

Name

页面的名称。

PrimaryServiceType

未指定服务类型的控件的服务类型。

RenderMode

指示页面是否使用响应式输出。包括以下选项：

• NotSet
• 固定
• Responsive

所有车间页面的默认设置为 NotSet，等同于 Responsive。所有其他页面的默认设置为 Fixed。Fixed 页面包含固定宽度，且页面上的元素不会因浏览器大小调整而移动来进行调整。

注：可使用三种进入模式之一登录应用程序。如果在页面级别设置 RenderMode，则输入的设置将替代分别与三种进入模式关联的默认呈现模式设置。有关信息，请参考《Opcenter Execution Medical Device and Diagnostics 入门用户指南》或《Opcenter Execution Core 入门用户指南》。

设置水平主题后，将为所有这些页面设置 FullScreen 模式页面属性。有关信息，请参考 FullScreen。

SessionVariables

用于配置“页面”级别会话变量。

StaticWebParts

当前未实施。

TagCollection

标记名称及其控件 ID 或会话变量的集合。

Title

页面标题的文本。取代 TitleLabel 属性。

Titlelabel

标题字典标签的名称。

TypeName

页面的隐藏代码类型。

注：当前不支持在页面级别替代 TypeName。

Zones

当前未实施。

布局

CssClass

页面的样式表引用。

HorizontalAlignment

Web 部件的水平对齐方式。

Margin

页面的外边距。

Padding

页面的内边距。

历史

Created

页面的创建日期和时间。

CreatedBy

页面的创建者。

HowLong

页面的创建天数。

LastModified

上次修改页面的日期。

LastModifiedBy

上次修改页面的用户。
