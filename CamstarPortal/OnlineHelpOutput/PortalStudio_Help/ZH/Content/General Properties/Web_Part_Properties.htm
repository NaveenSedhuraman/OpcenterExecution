Web 部件属性

Web 部件属性允许查看或编辑 Web 部件的属性。当在工作区域中选择 Web 部件时，Web 部件属性将显示在“属性”窗格中。有关“属性”窗格的信息，请参考“属性窗格”。

Web 部件属性描述

此表提供 Web 部件属性的描述。

Web 部件属性

属性描述

常规

Actions

Web 部件的操作。可用于定义 Web 部件级别操作和按钮。有关信息，请参考“操作属性”。

BroadcastInitiators

用于创建将触发 Web 部件之间的页面级连接数据协定的发起程序。

CollapsibleGroups

将在即将发布的发行版中弃用。

注：必须使用“切换容器”控件的 ToggleDisplayMode 属性创建可折叠部分。有关信息，请参考“控件属性”。

Control

引用页面上的控件。

DataContractMembers

用于在页面之间进行通信的数据协定。

DataSubmissionMode

页面向服务器提交数据时字段所做的回应。选项包括：

• NotSet：未设置模式。这是默认模式。
• Force：值始终包含在提交数据中。
• Changed：值仅在更改时才提交。
• Filled：值仅在字段不为空时才提交。
• ChangedAndFilled：值仅在字段不为空且已更改时才提交。
• Skip：值未提交或加载。
• ViewOnly：值未提交。
• Skip Request：值未加载。不加载控件的选择值。

重要事项：Siemens 建议对“建模”页使用 Changed 选项。

DirtyFlagTrigger

如果 Web 部件中的任何字段在设为“是”时更改值，将该页标记为已更新的布尔字段。

注：当前只影响建模框架内的 Web 部件。

EventHandlers

可用于从事件指定处理程序。有关信息，请参考“定义事件处理程序”。

Layout

Web 部件的可视外观。Siemens 建议您不要直接更改此属性。

注：子属性基于当前选择的 Web 部件中的单元格和控件显示。

Name

Web 部件的名称。

PrimaryServiceType

未指定服务类型的控件的服务类型。

SessionVariables

用于配置会话变量。有关信息，请参考“定义会话变量”。

TypeName

Web 部件类型后面的代码。

WebProperties

Web 部件的 Web 属性的集合。

行为

ActionsStyle

确定将如何呈现操作，例如按钮或链接。

ClientPropertyConditions

可用于根据浏览器端事件的条件结果将数据设为属性。

IncludeHandlerActions

设为 True 时，则包含当前页面的操作、Web 部件操作和页面操作建模项。设为 False 时仅包含 ActionControl Web 部件的操作。

PropertyConditions

允许将 Web 部件的属性值定义为计算表达式（条件）的结果。该属性在页面执行运行时计算。有关信息，请参考“属性条件”。

外观

IsHidden

将 Web 部件设为不在页面上显示。

IsMinimized

将 Web 部件设为最小化显示。

Title

显示为 Web 部件标题的文本。当输入了文本并选择了标签时，此处输入的文本将优先于 TitleLabel 字段。

TitleLabel

选择要用作 Web 部件标题的标签。有关使用“标签查找”弹出窗口的信息，请参考“搜索页面操作标签”。

布局

CssClass

Web 部件的样式表引用。

DisplayBorder

在 Web 部件上显示边框。

DisplayFooter

在 Web 部件上显示页脚。

DisplayHeader

在 Web 部件上显示标题。

Height

Web 部件显示的高度（以像素为单位）。

HorizontalAlignment

Web 部件元素的水平对齐方式。

Margin

Web 部件的边距。

Padding

Web 部件的填充。

VerticalAlignment

Web 部件元素的垂直对齐方式。

Width

Web 部件显示的宽度（以像素为单位）。

历史

Created

Web 部件的创建日期和时间。

CreatedBy

Web 部件的创建者。

HowLong

创建 Web 部件以来的时间段。

LastModified

Web 部件的上次修改日期。

LastModifiedBy

上次修改 Web 部件的用户。
