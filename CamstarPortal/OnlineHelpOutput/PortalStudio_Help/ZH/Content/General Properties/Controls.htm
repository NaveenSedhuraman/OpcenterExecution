控件属性

控件属性可用于查看和编辑控件的属性。在工作区域中选择控件时，控件属性将显示在“属性”窗格中。有关“属性”窗格的信息，请参考“属性窗格”。

当前选定控件不需要的控件属性将隐藏在“属性”窗格中。

用于表格的属性基于当前选定的表格类型隐藏。

格式化日期或时间设置的控件使用 Microsoft 的标准日期和时间格式。有关 Microsoft 日期和时间格式的信息，请参考 https://msdn.microsoft.com/en-us/library/8kb3ddd4(v=vs.110).aspx?cs-save-lang=1&cs-lang=vb#code-snippet-2。

Note: 某些属性包含可进一步自定义的子属性。子属性将继续在“属性”窗格的描述窗口中显示其父属性的描述。

控件属性描述

此表提供控件属性的描述。

控件属性

属性描述

常规

BrowseMode

选项：File 或 FileAndFolder。File 用于在当前文件夹中浏览。FileAndFolder 用于在当前文件夹和子文件夹中浏览。

CustomData

应用自定义 Setter、Getter、Requester（如果已设置）。

CustomListValues

用户定义的列表值。

描述

控件的描述。

DirectoryMode

选项：Web 或 Local。Web 使用虚拟 IIS 文件夹。Local 使用 Web 服务器上的本地文件夹。

DirectoryPath

最初打开“文件浏览”控件时要使用的路径。

DisplayObjectsForExport

如果设为 True，则仅显示对导出有效的建模对象。如果设为 False，则显示所有建模对象。

FileMask

可用于选择性地过滤文件类型的掩码。示例: "*.*"、"*.rpt"、"*.txt"。

FilePathFormat

选项：FileName 或 FullPath。FileName 仅需提供要使用的文件的名称。FullPath 取决于所选的 DirectoryMode。选择 Local 将使用本地服务器路径加上文件名，选择 Web 将使用 IIS 路径加上文件名。

GroupName

单选按钮组的名称。

ModelDocumentRef

设置对在控件中查看的文档 CDO 的引用。此属性从 DataContract 或页面控件等元素检索文档实例。

注：此属性包含与 ScaleMin 属性相同的连接子属性。有关信息，请参考 ScaleMin 子属性描述。

ModelName

指定控件显示的 JT 文件的名称（不带扩展名）。JT 文件必须上传到 CamstarUploads/Models 文件夹。

注：此属性包含与 ScaleMin 属性相同的连接子属性。有关信息，请参考 ScaleMin 子属性描述。

Name

控件的名称。

ScaleMax

在测重器控件上传递并设置最大值。如果未传入任何内容，将根据 TargetMaximum 值计算 ScaleMax。

注：此属性支持负数。ScaleMax 属性包含与 ScaleMin 属性相同的子属性。有关信息，请参考 ScaleMin 子属性描述。

ScaleMin

在测重器控件上传递并设置最小值。如果未传入任何值，此属性将默认为零或设为 TargetMinimumWeight 值（当 TargetMinimumWeight 小于零时）。

注：此属性支持负数。

ConnectionType

指定测重器值最小的连接类型。

注：其他子属性根据所选的连接类型显示。

Control

指定一个控件，为测重器的最小值传递值。

注：在 ConnectionType 子属性中选择 Control 时，将显示此子属性。

PageDataContract

指定一个页面数据协定，为测重器的最小值传递值。

注：在 ConnectionType 子属性中选择 PageDataContract 时，将显示此子属性。

SessionDataContract

指定一个会话数据协定，为测重器的最小值传递值。

注：在 ConnectionType 子属性中选择 SessionDataContract 时，将显示此子属性。

SessionVariable

指定一个会话变量，为测重器的最小值传递值。

注：在 ConnectionType 子属性中选择 SessionVariable 时，将显示此子属性。

FieldExpression

指定一个字段表达式，为测重器的最小值传递值。

注：在 ConnectionType 属性中选择 FieldExpression 时，将显示此子属性。

StaticValue

指定一个静态值，为测重器的最小值传递值。

注：在 ConnectionType 子属性中选择 StaticValue 时，将显示此属性。

GridItem

指定一个表格项，为测重器的最小值传递值。

注：在 ConnectionType 子属性中选择 GridItem 时，将显示此子属性。

Property

指定计算的控件属性值。

注：在 ConnectionType 子属性中选择 Control 时，将显示此子属性。

RequestDataMode

指定用于从控件检索属性值的方法。

注：在 ConnectionType 子属性中选择 Control 时，将显示此子属性。

AllowEmptyValue

确定值能否为空。

Initiators

单击“更多”按钮时，将显示 Initiators 弹出窗口。有关信息，请参考“Initiators 弹出窗口字段定义”。

TabIndex

表示使用 Tab 键的优先顺序。

TargetMaximumWeight

为测重器的重量传递并设置允许的上限。如果没有任何值传入测重器控件，则此属性默认为针对 ScaleMax 属性设置的值。如果传入这两个值，则此属性必须小于或等于 ScaleMax 属性。

注：此属性支持负数。TargetMaximumWeight 属性包含与 ScaleMin 属性相同的子属性。有关信息，请参考 ScaleMin 子属性描述。

TargetMinimumWeight

为测重器的重量传递并设置允许的下限。

注：此属性支持负数。TargetMinimumWeight 属性包含与 ScaleMin 属性相同的子属性。有关信息，请参考 ScaleMin 子属性描述。

TargetWeight

为测重器传递并设置理想的目标重量。

注：此属性支持负数。TargetWeight 属性包含与 ScaleMin 属性相同的子属性。有关信息，请参考 ScaleMin 子属性描述。

ToolTipLabel

注：如果将页面的 UseHttps 设为 True，则将 InfoViewPort 设为 8443。有关信息，请参考“InfoViewPort”。
