管理布局

Portal Studio 支持使用包含单元格的行和列布局来配置页面或 Web 部件的布局。

可以在“属性”窗格中修改单个单元格的属性。有关修改单元格属性的信息，请参考“单元格属性”。

在管理布局时

每个页面或 Web 部件的布局中都包含至少一个单元格。通过单元格可以指定对象显示在页面和 Web 部件上时相对于其他单元格中对象的位置。

右键单击单元格可显示用于修改单元格布局的选项。可以选择单击并拖动以选定多个单元格，然后右键单击突出显示的单元格以同时修改多个布局。

此表定义用于管理单元格布局的选项。
