Portal Studio 用户界面功能

Portal Studio 的用户界面在所有应用程序窗格和弹出窗口中都具有标准按钮和图标。

标准按钮

使用标准按钮是为了整个应用程序用户界面的一致性，以增强用户体验并简化数据输入任务。

此表定义 Portal Studio 中的常用按钮。

按钮

按钮功能 ...

确定

保存更改并关闭弹出窗口。

取消

关闭弹出窗口而不保存更改。

注：也可以单击右上角的 X 来关闭弹出窗口。

...

显示弹出窗口，通过该弹出窗口可对相关字段或属性进行更多自定义设置。此按钮被称为“更多”按钮。

展开

展开部分以显示其他字段。

折叠

折叠展开的部分。

清除

清除在弹出窗口或窗格上的字段中输入的选择和值。

标准图标

使用标准图标是为了整个应用程序用户界面的一致性，以增强用户体验并简化数据输入任务。

此表提供标准图标的图示和描述。

图标

描述

单击此图标可以 ...

新建项

在窗格中创建新项。

列表模式

显示项的简单列表。

树模式

以层次结构显示项的列表。

平铺模式

在平铺视图中显示一系列项。

详细信息模式

显示具有名称和描述信息的项列表。

分层排序

以层次结构显示字段或属性列表。

按字母排序

按字母顺序显示字段或属性列表。

刷新

刷新窗格。
