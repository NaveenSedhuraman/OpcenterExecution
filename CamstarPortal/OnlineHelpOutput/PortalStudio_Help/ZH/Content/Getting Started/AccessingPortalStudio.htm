访问 Portal Studio

可以根据用于访问门户的主题，以不同方式访问 Portal Studio。如果使用默认模式登录门户，则默认主题为水平。如果使用经典进入模式，则默认主题为经典。有关进入模式的信息，请参考《 Opcenter Medical Device and Diagnostics 入门用户指南》或《 Execution Core 入门用户指南》。

Important: 必须安装 Microsoft Silverlight 才能访问 Portal Studio。Google Chrome 不支持 Silverlight 44 以上的版本或仅对其提供有限支持。有关使用 Chrome 访问 Portal Studio 的信息，请参考“在 Google Chrome 中启用 Silverlight”。

如何访问 Portal Studio

可以从 Portal 单击以下按钮访问 Portal Studio：

• 如果使用的是水平主题，则单击 Portal 中主导航栏上的 Portal Studio 按钮。
• 如果使用的是经典主题，则单击 Portal 全局页眉中的 Studio 按钮。

如果 CSI 是最活跃的工作区，则会出现一条错误消息，阻止您访问 Portal Studio。必须在 Management Studio 中激活另一个工作区，然后再访问 Portal Studio。有关激活工作区的信息，请参考《Opcenter Execution Medical Device and Diagnostics 系统管理指南》或《Opcenter Execution Core 系统管理指南》。

Note: 如果已经登录 Portal，CSI 然后才成为最活跃的工作区，则 Portal 不会显示错误消息。在注销并重新登录 Portal 后，将出现一条错误消息，阻止您访问 Portal Studio。

当访问 Portal Studio 时，工作区域中将显示“起始页面”仪表板。“起始页面”仪表板提供指向常用组件的链接以及有关使用 Portal Studio 的信息。

“起始页面”仪表板包含以下部分：

• 新建，提供用于创建新组件（如页面、Web 部件、页面流等）的链接。
• 新增功能，提供指向发行说明的链接，其中描述 Portal Studio 中的新增功能。
• 入门，提供有关使用 Portal Studio 的信息的链接。
• 最近打开，提供用于打开五个最近打开的页面、Web 部件和页面流的链接。

“起始页面”仪表板会记住十个最近打开的页面、Web 部件和页面流，并在打开组件时更新列表。如果在“最近打开”列表中单击自从访问 Portal Studio 后已删除的组件的链接，将显示一条错误消息。关闭错误消息将从列表中移除该组件，并将其替换为另一个最近打开的组件。如果所有十个最近打开的组件都已删除，且尚未打开其他页面、Web 部件或页面流，则下次访问 Portal Studio 时，“最近打开”列表中将不会显示任何组件。

必须安装 Microsoft Silverlight，并且必须在浏览器中启用 cookie 才能访问 Portal Studio。如果要使用 Chrome 版本 42、43 或 44 访问 Portal Studio，必须首先启用 Silverlight。有关启用 Silverlight 的信息，请参考“在 Google Chrome 中启用 Silverlight”。

通过单击“回到门户”按钮，可以在 Portal Studio 会话期间的任何时间返回 Portal。在返回 Portal 之前，Portal Studio 将提示您保存更改。

还可以在本地安装 Portal Studio，这样，无需通过 Portal 便可以访问。有关信息，请参考“在本地安装 Portal Studio”。

访问 Portal Studio 2.0

可以访问 Portal Studio 2.0（Opcenter EX MDD 和 Opcenter EX CR 的新版 Portal Studio）。Portal web.config 文件中的 PortalStudioVersion 设置控制使用水平主题或经典主题访问 Portal Studio 时的可用版本。选择 2 表示 Portal Studio 2.0。

Note: Siemens 建议使用 Microsoft Internet Explorer 浏览器来访问 Portal Studio 1.0。对于 Chrome 浏览器，默认版本是 Portal Studio 2.0。

如果使用 ...

操作步骤 ...

水平

如何返回 Portal

按照以下步骤返回 Portal：

1. 访问 Portal Studio。
2. 执行任何必要的工作。
3. 单击返回门户按钮。Portal Studio 关闭，您将返回到 Portal。
