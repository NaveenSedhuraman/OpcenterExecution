工具箱

工具箱提供对以下选项卡的访问：

• 控件，其显示可添加到 Web 部件的控件的列表。有关信息，请参考“控件”。
• 字段目录，其显示可添加到 Web 部件的服务和参数数据字段的列表。有关信息，请参考“字段目录”。

Portal Studio 工具箱上的“显示/隐藏工具箱”按钮可显示工具箱（如果已隐藏），以及隐藏工具箱（如果可见）。还可以通过单击“查看”，然后选择“显示/隐藏工具箱”来显示或隐藏工具箱。

工具栏显示在工具箱窗格的顶部。显示的按钮可能会有所不同，具体取决于所选的选项卡。

相对按钮

“字段目录”选项卡包含“相对”按钮。将字段添加到 Web 部件时，“相对”按钮会在字段的相对和绝对设置之间切换。默认情况下，该按钮具有青色背景，表示该字段将为相对。单击该按钮将使该按钮呈灰色显示，表示该字段将为绝对。有关信息，请参考“相对字段表达式”。

如何显示或隐藏工具箱

按照以下步骤显示或隐藏工具箱：

1. 访问 Portal Studio。
2. 如果工具箱处于隐藏状态，则单击显示/隐藏工具箱。将显示工具箱。
