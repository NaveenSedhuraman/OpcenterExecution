控件

控件可用于在 Portal Studio 中自定义 Web 部件，并显示在“工具箱”的“控件”选项卡中。有关查看工具箱的信息，请参考“工具箱”。

只能向 Web 部件添加控件；不能直接向页面添加控件。有关向 Web 部件添加控件的信息，请参考“向 Web 部件添加控件”。

如果添加的控件已计划在即将推出的 Portal Studio 发行版中弃用，则会显示一条弹出式警告消息。该警告消息包括将弃用该控件的时间和要改用的备选控件。将弃用的控件上也会显示包含此信息的工具提示。

此表介绍“工具箱”的“控件”选项卡中提供的控件。

3D 可视化

显示具有旋转、缩放、标记图像和保存快照等功能的 3D 可视化效果。

注：此控件支持 JT 文件。

有关使用“3D 可视化”控件的信息，请参考“使用 3D 可视化”。

布尔下拉列表

带有可配置标签的布尔单选按钮控件。

按钮

可以用文本或图像显示的可单击控件。

图表控件

在页面或 Web 部件上放置可自定义的图表。多个图表控件可以同时存在于一页上。

可以使用“属性”窗格中的 ChartMode、ShowLegend、SeriesSettings、SeriesValueColumn、TooltipColumn、ValueColumn、XAxis 和 YAxis 属性修改图表控件的格式和数据。在修改这些属性的数据源时，图表控件会动态更新。有关图表控件属性的信息，请参考“控件属性”。

SeriesSettings 属性可用于进一步自定义数据点在散点图和晶圆图图表上的外观。有关添加 SeriesSettings 属性的信息，请参考“添加 SeriesSettings”。

可以通过修改 d3.js 文件来创建自定义图表格式，包括晶圆图。有关 D3 文档对象模型的信息，请参考 http://d3js.org。

注：可以使用鼠标滚轮在门户页面上放大和缩小图表控件。

复选框

选中或清除此框可打开或关闭选项。

注：向包含弹出式表格的页面添加复选框控件时，必须将 AutoPostBack 属性设为 "True"。有关修改控件属性的信息，请参考“控件属性”。
