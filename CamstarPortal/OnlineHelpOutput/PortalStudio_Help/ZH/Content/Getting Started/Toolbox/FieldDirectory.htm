字段目录

借助工具箱中的“字段目录”目录，用户可通过查看可用服务，然后将字段拖放到 Web 部件上来构建 Web 部件。它是一种服务导向型体系结构样式显示。它具有提前键入过滤功能，能够让用户更轻松地找到要查找的字段。

最合适或最常用的控件与它们的字段相关联。例如，“级别（容器级别）”字段是“已命名对象”控件，其中某些字段属性将被设为“容器级别”的相应值。

在字段上提供了以下属性的视觉提示：

• 必需 - 加粗
• 只读 - 灰色显示
• 列表 - 括号
• 继承 - 图标上的关键词

在使用字段目录时

字段和服务元数据在“属性”窗格中可用。

切换相对或绝对表达式按钮以指定字段是相对的还是绝对的。按钮上的灰色背景表示字段是相对的。按钮默认设为相对表达式。没有灰色背景表示字段是绝对的。有关信息，请参考“相对字段表达式”。
