用户界面组件目录

用户界面组件目录包含多个选项卡，其提供对组件、导航和映射项的访问。

“组件”选项卡提供对以下组件的访问：

• 页面
• Web 部件
• 页面流

“导航”选项卡提供对以下导航项的访问：

• 菜单定义
• 选项卡

“建模”选项卡提供对以下映射项的访问：

• 页面操作
• 页面操作类别
• 创建页面操作规则
• 页面映射
• 目标矩阵

会将“用户界面组件”目录选项卡安排到各组件项类型的下拉列表中。各下拉列表都包含搜索字段、“新建项”按钮、一组视图模式按钮和刷新按钮。单击“新建项”按钮将在工作区域中显示该项类型的新项。

Portal Studio 工具栏上的“显示/隐藏目录”按钮可显示“用户界面组件”目录（如果已隐藏），以及隐藏“用户界面组件”目录（如果可见）。还可以通过单击“查看”，然后选择“显示/隐藏目录”来显示或隐藏“用户界面组件”目录。

Note: 从早期版本升级，首次登录 Portal Studio 时，某些默认页面可以显示在“用户界面组件”目录的“组件”选项卡中的“未定义”组中。可以从“用户界面组件”目录中自动将“未定义”组中的默认页面排序到其他组中。有关信息，请参考“如何自动对默认页面进行排序”。

如何导航用户界面组件目录

按照以下步骤导航“用户界面组件”目录：

1. 访问 Portal Studio。
2. 如果“用户界面组件”目录处于隐藏状态，单击显示/隐藏目录。将显示“用户界面组件”目录。
3. 单击目录底部的选项卡以显示该选项卡上的选项。
4. 使用某节旁边的箭头来展开或收缩该节。
