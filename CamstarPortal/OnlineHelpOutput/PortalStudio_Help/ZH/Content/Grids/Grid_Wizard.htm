使用表格向导定义表格

Portal Studio 提供表格向导，该向导提供用户导向的端到端表格创建过程，用于定义新的表格控件。

在使用表格向导定义表格时

“表格向导”显示为包含一系列屏幕的弹出窗口。屏幕和字段会动态显示，以响应您在初始屏幕上选择的表格数据源类型以及在定义表格时所输入或选择的值。

在 Web 部件上放置表格控件时，将显示“表格控件创建”弹出窗口。通过选择“表格向导”来显示“表格向导”弹出窗口，即可使用“表格向导”定义表格。通过选择“创建默认表格”来放置默认表格控件，即可在“表格设置编辑器”中手动设置表格属性。有关手动定义表格的信息，请参考“表格设置编辑器”。

单击“下一步”按钮，即可在“表格向导”中从一个屏幕导航至下一个屏幕。单击“返回”按钮可返回到上一个屏幕。单击“取消”将关闭“表格向导”，并且表格控件不会被添加到 Web 部件。

单击“表格向导”弹出窗口最后一个屏幕上的“完成”按钮后，Web 部件上会出现新的表格控件。“表格向导”中输入的信息可从“属性”窗格的“表格设置编辑器”弹出窗口中查看或编辑。有关信息，请参考“表格设置编辑器”。

表格数据源

在“表格向导”的初始屏幕上选择一个表格数据源，以确定所定义的表格类型。在整个“表格向导”过程中，屏幕和字段会根据所选的表格数据源动态显示。

此表定义表格数据源。

表格数据源

描述

项列表

支持任何类型数组作为数据源。可绑定到字段表达式，并使用服务器的常规值加载。“项列表”表格也可以在未绑定模式下工作（例如，没有字段表达式）。在未绑定模式下，必须定义 ItemType，并通过代码实现数据加载和提交。不允许在服务器中进行分页。

子实体列表

从“项列表”继承数据。支持从“子实体”类派生的项类型的特定绑定模式。

选择值

在绑定模式下工作，并从选择值请求加载数据。可提交从所选行构建的数据。“选择值”表格数据源可以使用两个字段表达式 - 一个用于数据加载，另一个用于提交。

预定义查询

参考“系统”或“用户”查询。“预定义查询”表格允许使用字段表达式向字段提交数据。

特别查询

直接在虚拟页面上存储自定义查询。“特别查询”表格允许使用字段表达式向字段提交数据。

表格向导中的列设置

可以在“表格向导”的“设置列”屏幕上创建和自定义表格中显示的列。

如果在定义“项列表”、“子实体列表”和“选择值”表格时，在“字段表达式设置”屏幕上的“字段表达式”或“请求字段表达式”字段中设置了字段表达式，则会显示“设置列”屏幕。在定义“预定义查询”和“特别查询”表格时，将始终显示此屏幕。

“设置列”屏幕显示列类型、ID、以像素为单位的列宽以及将列设为隐藏的复选框。在定义“项列表”和“子实体列表”表格时，还会出现将列设为可编辑或必需列的其他复选框。通过单击字段并输入或选择新值，可修改现有列的值。

根据在“字段表达式”或“请求字段表达式”字段中设置的字段表达式，“设置列”屏幕上将显示预定义的列。如果在“字段表达式”或“请求字段表达式”字段中都设置了字段表达式，则只有在“请求字段表达式”字段中设置的字段表达式会用于预定义列。

Note: 如果无法根据所选字段表达式创建预定义列，则“设置列”屏幕上将显示消息弹出窗口。可以单击“确定”关闭消息并手动定义列。

在定义“预定义查询”和“特别查询”表格时，预定义的列也会根据预定义查询或自定义查询以及为查询参数输入的值来显示。在定义“预定义查询”和“特别查询”表格时，预定义的列不会根据所设置的字段表达式显示。

Note: 如果在定义“预定义查询”和“特别查询”表格时，无法根据在“定义查询”屏幕上输入的值创建列，则“设置列”屏幕上会显示消息弹出窗口。可以单击“确定”关闭消息并手动定义列。

通过从现有列列表下方的字段中选择列类型，然后单击“创建”按钮，可以添加其他列。新列将显示为列表中的最后一列，并默认选中以包含在表格中。默认情况下，新添加的列设为 200 像素的宽度。

通过在“设置列”屏幕上选择列并单击“向上”和“向下”按钮，可重新对列进行排序。列在“设置列”屏幕最上方的列将在门户页面的表格中作为最左边的列来显示。

如何使用表格向导定义项列表和子实体列表表格

按照以下步骤，使用“表格向导”定义“项列表”和“子实体列表”表格：

1. 访问 Portal Studio。
2. 如果“用户界面组件”目录处于隐藏状态，单击显示/隐藏目录。将显示“用户界面组件”目录。
3. 双击页面或 Web 部件。随即工作区域中将显示该页面或 Web 部件。
4. 如果工具箱处于隐藏状态，则单击显示/隐藏工具箱。将显示工具箱。
5. 单击并将表格控件拖拽到单元格上。将显示“表格控件创建”弹出窗口。
6. 单击表格向导。“表格控件创建”弹出窗口关闭，并显示“表格向导”弹出窗口。
7. 选择项列表或子实体列表作为表格数据源，然后单击下一步。
8. 选择表格编辑模式，然后单击下一步。
9. 单击字段表达式的更多按钮。 将显示“字段表达式构建器”弹出窗口。
10. 选择一个字段表达式，然后单击确定。“字段表达式构建器”弹出窗口关闭，并在“字段表达式”字段中显示该字段表达式。
11. 是否要设置请求字段表达式？

如果 ...

操作步骤 ...

要设置请求字段表达式。
a. 单击请求字段表达式的更多按钮。 将显示“字段表达式构建器”弹出窗口。
b. 选择一个字段表达式，然后单击确定。“字段表达式构建器”弹出窗口关闭，并在“请求字段表达式”字段中显示该字段表达式。
c. 单击下一步。
d. 转到步骤 12。

不想设置请求字段表达式。
a. 单击下一步。
b. 转到步骤 12。
12. 选中或清除可见和已启用复选框，以根据需要显示或隐藏及启用或禁用表格导航栏操作按钮。 然后单击下一步。
13. 选择、修改或创建要添加到表格中的列。 然后单击下一步。 有关设置列的信息，请参考“表格向导中的列设置”。
14. 从行 ID 字段中选择一列。
15. 从行选择模式字段中选择行选择模式，然后单击下一步。
16. 在名称字段中输入表格控件的名称。
17. 在标题文本字段中输入将显示为表格标题的标题。

或

单击标题标签名称的更多按钮以浏览并选择将显示为表格标题的标题标签。有关搜索预定义标签的信息，请参考“搜索页面操作标签”。

18. 单击完成。“表格向导”弹出窗口关闭，并在单元格中显示该表格控件。
19. 单击保存。应用程序将保存更改并显示成功消息。

“预定义查询”表格。
a. 选择预定义查询作为表格数据源，然后单击下一步。
b. 选择一个查询。查询的任何关联查询参数都在下面的“查询参数”表格中显示。
c. 在值字段中为显示的查询参数输入值。
d. 单击下一步。
e. 转到步骤 8。

“特别查询”表格。
a. 选择特别查询作为表格数据源，然后单击下一步。
b. 在文本字段中输入自定义查询文本。查询的任何关联查询参数都在下面的“查询参数”表格中显示。
c. 在值字段中为显示的查询参数输入值。
d. 单击下一步。
e. 转到步骤 8。
8. 单击字段表达式的更多按钮。 将显示“字段表达式构建器”弹出窗口。
9. 选择一个字段表达式，然后单击确定。“字段表达式构建器”弹出窗口关闭，并在“字段表达式”字段中显示该字段表达式。
10. 选中或清除可见和已启用复选框，以根据需要显示或隐藏及启用或禁用表格导航栏操作按钮。 然后单击下一步。
11. 选择、修改或创建要添加到表格中的列。 然后单击下一步。 有关设置列的信息，请参考“表格向导中的列设置”。
12. 从行 ID 字段中选择行标识。
13. 从行选择模式字段中选择行选择模式。
14. 要在提交数据时提交所选行、全部行还是都不提交？

如果要在提交数据时提交 ...

操作步骤 ...

所选行。
a. 从提交数据模式字段中选择所选行。将显示“提交的列数据映射”表格。
b. 从提交的字段字段中为要提交数据的每一列选择字段表达式。
c. 单击下一步。
d. 转到步骤 15。

全部行。
a. 从提交数据模式字段中选择全部行。将显示“提交的列数据映射”表格。
b. 从提交的字段字段中为要提交数据的每一列选择字段表达式。
c. 单击下一步。
d. 转到步骤 15。

不提交。
a. 从提交数据模式字段中选择禁用。
b. 单击下一步。
c. 转到步骤 15。
15. 在名称字段中输入表格控件的名称。
16. 在标题文本字段中输入将在表格上显示的标题。

或

单击标题标签名称的更多按钮以浏览并选择将显示为表格标题的标题标签。有关搜索预定义标签的信息，请参考“搜索页面操作标签”。

17. 单击完成。“表格向导”弹出窗口关闭，并在单元格中显示该表格控件。
18. 单击保存。应用程序将保存更改并显示成功消息。
