执行批量导入

通过批量导入，可以从单个 XML 或 Zip 文件导入多个页面、Web 部件和页面流。

在执行批量导入时

必须至少从“要导入的个性化类型”列表中选择一个工作区。

单击“开始导入”按钮时，将显示带有两个进度条的“导入进度...”弹出窗口。第一个进度条显示每个导入项的进度，第二个进度条显示导入操作的总进度。

如何执行批量导入

按照以下步骤执行批量导入：

1. 访问 Portal Studio。
2. 单击批量导入按钮。将显示“批量导入”窗口。
3. 至少从要导入的个性化类型列表中选择一个工作区。
4. 单击从文件导入的更多。将显示“打开”弹出窗口。
5. 导航至并选择 XML 或 Zip 文件，然后单击打开。“打开”弹出窗口关闭，文件名会显示在“从文件导入”字段中。
6. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
7. 单击开始导入。将出现“导入进度...”弹出窗口，其中显示导入操作的进度。当导入流程完成时，“导入进度...”弹出窗口关闭，“批量导入”窗口会显示一条成功消息。
