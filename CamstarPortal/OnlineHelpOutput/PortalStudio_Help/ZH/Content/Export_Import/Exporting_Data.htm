导出项

可通过 Portal Studio 导出 Web 部件、页面和页面流。从而可以从开发服务器离线保存这些对象的副本。

在导出项时

可从“查看内容”菜单中当前选择的工作区或所有活动工作区导出项。

导出项将另存为 XML 文件。XML 文件将默认使用导出项的名称。无法从 Portal Studio 导出菜单。

如何导出项

按照以下步骤导出项：

1. 访问 Portal Studio。
2. 打开要导出的项。
3. 是否要导出当前所选工作区或所有工作区的项数据？

如果您想完成以下操作 ...

操作步骤 ...

仅导出当前选定工作区的数据
a. 单击导出按钮。
b. 选择所选工作区。将显示“文件下载 - 安全警告”对话框。
c. 转到步骤 4。

导出所有工作区的项数据
a. 单击导出按钮。
b. 选择所有工作区。将显示“文件下载 - 安全警告”对话框。
c. 转到步骤 4。
4. 单击确定。 将关闭对话框并显示“另存为”弹出窗口。
5. 导航至保存位置，然后单击保存。将关闭弹出窗口，项将作为 XML 文件导出至指定位置，Portal Studio 将显示一条成功消息。
