导入项

可以在 Portal Studio 中导入 XML 文件。可以导入新的 Web 部件和页面，也可以导入更改以更新现有的 Web 部件和页面。

在导入项时

只能将 XML 文件导入到最高活动工作区中当前打开的页面或 Web 部件。

导入页面或 Web 部件后，可以用更新的名称保存该页面或 Web 部件。

Note: 默认情况下，导入项的大小限制为不超过 30 MB。如果捆绑包超出了该限制值，可以在“门户设置”中增加 MaxAttachmentSizeLimit 设置的值。有关信息，请参考“配置门户设置”。

如何导入项

按照以下步骤导入项：

1. 访问 Portal Studio。
2. 创建或打开现有页面或 Web 部件。
3. 单击导入按钮。将显示“打开”对话框。
4. 找到并选择要导入的 XML 文件。
5. 单击打开。XML 文件将被导入到打开的页面或 Web 部件中。
6. 在“属性”窗格中更改页面或 Web 部件的名称。
7. 单击保存。应用程序将保存更改并显示成功消息。
