Adding Targets to a Map Item

Targets are the target page to which end users will be directed after the logic has been applied.

How to Add Targets to a Map Item

Follow these steps to add targets to a map item:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Select the Modeling tab.
4. Expand Page Maps.
5. Double-click a page map. The page map appears in the working area.
6. Click Show/Hide Properties if the Properties pane is hidden. The Properties pane appears.
7. Click More for Targets in the Properties pane. The Targets pop-up appears.
8. Click Add. A target is added to the list.
9. Select a target from the Target field.
10. Select a target type from the Target Type field.
11. Repeat steps 8-10 to add additional targets.
12. Click OK. The parameter is added.
13. Click Save. The application saves your changes and displays a success message.
