Adding Page Mapping Parameters to a Map Item

Page mapping parameters are used to configure conditions under which the system redirects a user to a target page. Parameters serve as the mediator between a source page Data Contract and a target page Data Contract.

When Adding Page Mapping Parameters to a Map Item

A parameter should be unique for a map item. Parameters are for involving them into transferring data between pages and configuring conditions of redirecting.

How to Add Page Mapping Parameters to a Map Item

Follow these steps to add page mapping parameters to a map item:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Select the Modeling tab.
4. Expand Page Maps.
5. Double-click a page map you want to open. The page map appears in the working area.
6. Click More for Parameters. The Parameters pop-up appears.
7. Click Add. A parameter is added to the list.
8. Enter or select a name from the Name field.
9. Repeat steps 7-8 to add additional parameters.
10. Click OK. The Parameters pop-up closes.
11. Click Save. The application saves your changes and displays a success message.
