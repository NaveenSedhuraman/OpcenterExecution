Adding a Sub Menu to a Menu Definition

You can associate sub menus with a menu after you have defined the Portal main menu. Any defined menu can be added as a sub menu. Adding a sub menu enables you to add a group of transactions rather than adding each transaction individually. This continues the process of defining a multi-level menu structure.

Sub menus are defined in the MenuItems pop-up.

You can modify submenu icons by entering an icon file name in the ApolloIcon field on the MenuItems pop-up. Icon files are located in the following directory: C:\Program Files (x86)\Camstar\Camstar Portal\assets\image.

You must use the following standards when adding an icon file in the ApolloIcon field:

• The file must be an .svg file located within the folder.
• The file must be 24-pixels wide which is indicated in the file name using '24'.
• The file name must be entered as case-sensitive.
• The file name must be entered without the image size and file extension.

For example, if the icon file name appears in the folder as "cmdAttach24.svg" you should enter "cmdAttach" in the ApolloIcon field.

Note: Siemens does not support adding custom submenu icons.

How to Add a Sub Menu to a Menu Definition

Follow these steps to add a sub menu to a menu definition:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Select the Navigation tab.
4. Expand Menus.
5. Double-click a menu. The menu appears in the working area.
6. Click More for MenuItems. The MenuItems pop-up appears.
7. Select PortalMenuSubMenuChanges from the drop-down and click Add. A new sub menu appears in the list and fields appear in the right pane.
8. Enter a caption for the sub menu in the Caption field.
9. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
10. Repeat steps 7-9 for each sub menu you want to add.
11. Click OK. The MenuItems pop-up closes.
12. Click Save. The application saves your changes and displays a success message.

How to Add a Submenu Icon

Follow these steps to add a submenu icon:

1. Access Portal Studio.
2. Select the Navigation tab on the Components catalog and then double-click a menu. The menu appears in the working area.
3. Click More for MenuItems. The MenuItems modal appears.
4. Select PortalMenuSubMenuChanges from the drop-down and select a submenu. The submenu's fields appear in the right pane.
5. Enter an icon file name in the ApolloIcon field that corresponds with the icon image file in the C:\Program Files (x86)\Camstar\Camstar Portal\assets\image directory.
6. Click OK. The MenuItems modal closes.
7. Click Save. The application saves your changes and displays a success message.
