Adding a Pageflow to a Menu Definition

Pageflows are used to link pages together to create the desired flow of pages. Refer to "Creating a Pageflow" for information.

When Adding a Pageflow to a Menu Definition

Pageflows are added to Menu Definitions in the MenuItems pop-up. Refer to "Adding Menu Items to a Menu Definition" for information on fields on the MenuItems pop-up.

How to Add a Pageflow to a Menu Definition

Follow these steps to add a pageflow to a menu definition:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Select the Navigation tab.
4. Expand Menus.
5. Double-click a menu. The menu appears in the working area.
6. Click More for MenuItems. The MenuItems pop-up appears.
7. Select PortalMenuCommandChanges from the drop-down and click Add. A new menu item appears in the list and fields appear in the right pane.
8. Enter a caption for the menu item in the Caption field.
9. Select a pageflow from the Pageflow field.
10. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
11. Click OK. The MenuItems pop-up closes.
12. Click Save. The application saves your changes and displays a success message.
