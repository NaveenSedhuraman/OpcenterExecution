Adding Menu Items to a Menu Definition

Menu items are the pages an end user will be able to access from the menu. Menu items are defined in the MenuItems pop-up.

MenuItems Pop-up Field Definitions

Field

Definition

Type

Caption

Unique name for the menu item.

Required

LabelName

Determines the text that displays on the menu item.

Optional

PageDisplay

Determines whether the menu item displays in the current tab, a new tab, or a new browser window.

Optional

PageFlow

Specifies a pageflow for the menu item.

Optional

PageURL

Specifies a page URL for the menu item.

Note: You can add static pages to a menu item by entering a relative path to a custom ASPX page file. Static pages render directly from the Web and do not support web parts. Static pages are stored within the StaticPages sub-folder in the CamstarPortal directory.

Optional

PortalTab

Specifies a portal tab.

Note: This field appears when SpecifiedTab is select in the PortalTabOption field.

Optional

PortalTabOption

Specifies the way the menu item references a tab. Refer to "Creating a Menu Definition" for information.

Optional

QueryString

Specifies a query for the menu item.

Optional

Sequence

Determines the sequence of the menu item.

Optional

ServiceName

Specifies a service for the menu item.

Optional

VirtualPage

Specifies a virtual page for the menu item.

Optional

How to Add Menu Items to a Menu Definition

Follow these steps to add menu items to a menu definition:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Select the Navigation tab.
4. Expand Menus.
5. Double-click a menu. The menu appears in the working area.
6. Click More for MenuItems. The MenuItems pop-up appears.
7. Select PortalMenuCommandChanges from the drop-down and click Add. A new menu item appears in the list and fields appear in the right pane.
8. Enter a caption for the menu item in the Caption field.
9. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
10. Repeat steps 7-9 for each menu item you want to add.
11. Click OK. The MenuItems pop-up closes.
12. Click Save. The application saves your changes and displays a success message.
