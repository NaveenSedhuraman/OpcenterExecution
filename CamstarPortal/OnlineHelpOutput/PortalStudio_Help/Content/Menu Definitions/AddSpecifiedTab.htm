Adding a Specified Tab to a Menu Definition

Portal Studio allows you to specify in which tab a page will open. Tabs are referenced by menu items and should be created before creating menus.

How to Add a Specified Tab to a Menu Definition

Follow these steps to add a specified tab to a menu definition:

1. Access Portal Studio.
2. Create a new menu definition or open an existing menu definition.
3. Select SpecifiedTab from the PortalTabOption field.
4. Select a tab from the PortalTab field.
5. Click Save. The application saves your changes and displays a success message.
