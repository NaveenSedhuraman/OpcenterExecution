Expanding EProcedure Instructions

Portal Studio allows you to configure whether instructions are expanded or collapsed in EProcedure by default. EProcedure instructions are collapsed by default.

When Expanding EProcedure Instructions

Configuring EProcedure instructions requires modifying the EProcedure UI virtual page. Virtual pages in Portal Studio contain all the parts of a page. The part that configures EProcedure instructions is the TaskInstructionsField and is not labeled. The TaskInstructionsField appears as a yellow rectangle without a border.

How to Expand EProcedure Instructions by Default

Follow these steps to expand EProcedure instructions by default:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Expand the All page grouping and double-click EProcedureVPR2. The EprocedureVPR2 page appears in the working area.
Note: You must update EProcedureVP if you are using the older version of the page. Refer to the Opcenter Execution Medical Device and Diagnostics Shop Floor User Guide or the Opcenter Execution Core Shop Floor User Guide for information.
4. Select the TaskInstructionsField. The settings for TaskInstructionsField appear in the Properties pane.
5. Select Expanded for DefaultState.
6. Select True for ExpandFull.
7. Click Save. The application saves your changes and displays a success message.
