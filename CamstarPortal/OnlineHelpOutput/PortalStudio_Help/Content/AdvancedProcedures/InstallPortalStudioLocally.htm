Installing Portal Studio Locally

Portal Studio can be installed on your local environment. This allows you to open Portal Studio directly, rather than logging into the Portal every time you want to access Portal Studio.

When Installing Portal Studio Locally

You must open Portal Studio from Portal the first time, where you can access the shortcut menu for installing locally.

You have the choice of installing shortcuts on the Start menu and/or the Desktop.

Once installed, clicking the shortcut opens a separate Sign In page that will open Portal Studio directly without first opening Portal.

Note: You may need to clear your browser cache prior to logging in with the shortcut.

How to Install Portal Studio Locally

Follow these steps to install Portal Studio locally:

1. Access Portal Studio.
2. Right-click in the work area. A shortcut menu appears.
3. Select Install Camstar Portal Studio application on this computer. The Install application dialog appears.
4. Select the appropriate checkbox to install Portal Studio as a Start Menu item and/or on the desktop and click OK. A Sign In page appears.
5. Enter the Password and click Log On. A new Portal Studio page appears.
