Enabling Sticky Fields

Sticky fields are not cleared of your entered or selected data after the page is submitted.

When Enabling Sticky Fields

Sticky fields are denoted by a pushpin icon.

Portal studio enables a field to have the option of being sticky. You must then activate sticky fields on Portal pages by clicking on the pushpin icon. Sticky fields are inactive by default.

Icon

Status

Inactive. The field will not retain data.

Active. The field will retain data.

Sticky fields will be cleared of data if the page action for submit redirects to another page or is set to reload values. Refer to "Configuring Page Actions" for information about configuring actions.

How to Enable Sticky Fields

Follow these steps to enable sticky fields:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Double-click a page. The page appears in the working area.
4. Select a field on the page. The properties for the field appear in the Properties pane.
5. Select True for StickyCapable in the Properties pane.
6. Repeat steps 4-5 to enable sticky fields on additional fields as needed.
7. Click Save. The application saves your changes and displays a success message.
