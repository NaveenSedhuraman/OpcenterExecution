Converting ASPX Files to Portal Pages

Portal Studio allows you to automatically convert ASPX files to Portal pages and continue editing and using them as Portal pages.

When Converting ASPX Files to Portal Pages

The following notes apply to ASPX files to Portal page conversions:

• The conversion utility converts controls from the Web app pages into the closest equivalent control types in Portal Studio with web parts created for logical sections.
• The Web app buttons with DisplayInFrame=True are converted into Page Action buttons. Those buttons will be converted into button controls on the page if DisplayInFrame=False.
• Dependencies are preserved where possible, such as when a Resource field has a DependsOn property set in the Web app page to the Container field.
• The code behind a page is not converted, as the utility only converts the ASPX file.
• Batch conversion of ASPX files is not supported.

How to Convert ASPX Files to Portal Pages

Follow these steps to convert an ASPX file to a Portal page.

1. Access Portal Studio.
2. Click the Convert ASPX Page button. An Open pop-up appears.
3. Navigate to and select an ASPX file to convert and click the Open button. Portal Studio automatically performs the conversion and saves the page. A success message appears indicating the file was converted successfully.
Note: The converted page is displayed in the list of pages with the prefix “ASPX_”. You can rename the page in the Properties pane.
