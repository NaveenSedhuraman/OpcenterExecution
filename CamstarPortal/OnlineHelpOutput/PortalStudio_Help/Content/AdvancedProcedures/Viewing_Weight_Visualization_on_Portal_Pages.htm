Viewing Weight Visualization on Portal Pages

Scale controls appear as graphical bars that visualize weight in real-time on Portal pages.

Note: Contact Product Support for assistance adding weight visualization to custom Portal pages.

When Viewing Weight Visualization on Portal Pages

The scale control appears as a color-coded bar on Portal pages. The scale control requires you to pass three target weight values to the control in the Properties pane:

• TargetMinimumWeight or ScaleMin
• TargetWeight
• TargetMaximumWeight or ScaleMax
Important: An error message appears stating the scale control cannot be rendered if any of the required values are not passed in to the control. Refer to "Control Properties" for information on the required properties.
Note: You can also pass alternate minimum and maximum values for the scale control to start from and end at using the ScaleMin and ScaleMax properties in the Properties pane. Scale controls start at zero by default.

Scale controls support negative numbers for minimum, target, and maximum values. For example, you could set your scale control's ScaleMin property to a negative number and then add weight to reach zero as your TargetWeight. Refer to "Control Properties" for information on properties that support negative numbers.

Modifying Colored Sections

The bar displays three, colored sections that reference the target weight values set on the scale control:

• The left section of the bar references the TargetMinimumWeight. This section is yellow and a yellow line appears at the default value for the TargetMinimumWeight.
• The middle section of the bar is the range between the TargetMinimumWeight and TargetMaximumWeight. This section is green.
• The right section of the bar references the TargetMaximumWeight. This section is red and a red line appears at the default value for the TargetMaximumWeight.
• The TargetWeight value appears on the bar as a vertical, solid black line with an arrow at the top.

You can modify the scale control's properties to display custom colors in the three colored sections. You can modify the:

• BelowMinZoneColor property to customize the color of the left section of the bar. The line that appears at the default value for the TargetMiniumWeight updates to match the custom color set for this section.
• TargetZoneColor property to customize the color of the middle section of the bar. The actual weight value's text color will change to the custom color set for this section when the actual weight value is within the range between the TargetMinimumWeight and TargetMaximumWeight.
• AboveMaxZoneColor property to customize the color of the right section of the bar. The line that appears at the default value for the TargetMaximumWeight updates to match the custom color set for this section. The actual weight value's text color will change to the custom color set for this section when the actual weight surpasses the TargetMaximumWeight.

Refer to "Control Properties" for information on modifying scale control properties.

Viewing Weight Values

The TargetWeight can include up to 5 decimal places. The ScaleMin, ScaleMax, TargetMinimumWeight and the TargetMaximumWeight can include up to 4 decimal places.

The actual weight appears on the bar as a vertical black line with an arrow at the top and bottom. The line changes position on the bar in real-time as weight is added or removed from the associated scale.

The application zooms in on the control when:

• The weight on the scale approaches the target weight,
• The weight is within 10% of the TargetMinimumWeight,
• Both a TargetMinimumWeight and TargetMaximumWeight are set.

For example, if the TargetMinimumWeight is set to 100 and the TargetMaximumWeight is set to 120, the scale control will zoom in when the actual weight is 90 or greater.

Note: You can set a custom percentage that specifies when the scale control zooms in by modifying the control's ZoomPercentage property. Refer to "Control Properties" for information.

The zoomed in view shows 10% of the TargetMinimumWeight section, 10% of the TargetMaximumWeight section, and 80% of the TargetWeight section. When zoomed in, the numerical value of the:

• TargetWeight appears above the actual weight line and displays up to five decimal places.
• TargetMinimumWeight appears below the yellow line and displays up to four decimal places.
• TargetMaximumWeight appears below the red line and displays up to four decimal places.

The bar will not return to the zoomed out view if the weight goes above 10% of the TargetMaximumWeight. The actual weight line will move to the far right of the TargetMaximumWeight section when the actual weight surpasses the ScaleMax value and remains there as more weight is added. The actual weight line will move to the far left of the TargetMinimumWeight section when the actual weight falls below the ScaleMin value and remains there as more weight is removed.

Note: You can hide the ScaleMin, TargetMinimumWeight, TargetMaximumWeight, and ScaleMax values by setting the RenderScaleSettings property to False in the Properties pane.

You can use the scale control with an actual weight value. The actual weight value displays the weight from the scale. The actual weight value appears at the top left of the scale control by default. You can change the position of the actual weight value or hide it by modifying the ActualWeightPosition property in the Properties pane. The actual weight value starts at zero and can display up to five decimal places.

The application appends a unit of measure to the actual weight when:

• The associated scale passes the unit of measure value. The unit of measure must be appended to the weight value with a space between the two values when passing the values, for example, "100.02 lbs."
• You set a unit of measure in the ActualWeightUOM property. The unit of measure passed in from the associated scale overrides the ActualWeightUOM property. Refer to "Control Properties" for information on control properties.

The application modifies the font color of the actual weight value based on the target weight values. The font of the actual weight value appears in:

• Bold black font when the actual weight is less than the TargetMinimumWeight.
• Bold green font when the actual weight is within the target weight range.
• Bold red font when the actual weight is more than the TargetMaximumWeight.
Note: The scale control can be used independent of an actual weight value.
