Customizing Page Labels

Portal Studio allows you to customize the labels that appear on pages.

When Customizing Page Labels

The PageTitleLabelText field in the Properties pane configures the title that appears for a page.

The PageDescriptionLabelText field in the Properties pane configures additional information, such as a description beneath a page title.

How to Customize Page Labels

Follow these steps to customize page labels:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Expand Pageflows.
4. Double-click a pageflow. The pageflow appears in the working area.
5. Click Show/Hide Properties if the Properties pane is hidden. The Properties pane appears.
6. Select a page on the pageflow. The general page properties for the selected page appear in the Properties pane.
7. Enter a new title in the PageTitleLabelText field in the Properties pane.
8. Enter a new page description in the PageDescriptionLabelText field in the Properties pane.
9. Click Save. The application saves your changes and displays a success message.
