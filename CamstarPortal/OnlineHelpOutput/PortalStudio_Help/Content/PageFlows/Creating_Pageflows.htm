Creating a Pageflow

Pageflows are used to link pages together to create the desired flow of pages.

When Creating a Pageflow

When you create a pageflow, the buttons are created dynamically by the pageflow.

Dragging a new page onto the pageflow automatically creates a link to the page that is currently highlighted in the pageflow.

You can manage the transition between two pages using the rule manager. Click on the arrow between pages to open the Properties pane with the Rule manager field.

How to Create a Pageflow

Follow these steps to create a pageflow:

1. Access Portal Studio.
2. Click New and select Pageflow. A blank pageflow appears in the working area.
3. Click Show/Hide Properties if the Properties pane is hidden. The Properties pane appears.
4. Enter a name for the pageflow in the Name field of the Properties pane.
5. Click Save. The application saves your changes and displays a success message.
