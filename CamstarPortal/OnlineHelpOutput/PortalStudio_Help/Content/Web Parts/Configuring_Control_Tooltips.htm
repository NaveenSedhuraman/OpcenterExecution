Configuring Control Tooltips

Control tooltips display when hovering the mouse cursor over controls on pages in the Portal. Tooltips can be configured in Portal Studio using text or labels.

When Configuring Control Tooltips

Control tooltips are configured in the Properties pane. The ToolTipText property creates a tooltip using text entered in the ToolTipText field. The ToolTipLabel property opens a Label Lookup pop-up to search and select a label to be used as the tooltip. Refer to "Searching for Page Action Labels" for information on the Label Lookup pop-up.

The ToolTipText property takes priority over the ToolTipLabel property when both text is entered and a label is selected.

How to Configure Control Tooltips

Follow these steps to configure control tooltips:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Double-click a page. The page appears in the working area.
4. Select a control on the page. The properties for the control appear in the Properties pane.
5. Do you want to enter text or select a label for the tooltip?

If you want to . . .

Then . . .

Enter text
a. Enter text for the tooltip in the ToolTipText field in the Properties pane.
b. Go to step 6.

Select a label
a. Click the More button for ToolTipLabel. The Label Lookup pop-up appears.
b. Enter one or more search criteria to narrow your search.
c. Click Search. The application displays the labels matching your search criteria in the grid.
d. Select a label. The Default Text and Local Text for the label appear in the Default Text and Local Text fields.
e. Click OK. The Label Lookup pop-up closes and the information for the label appears in the ToolTipLabel field.
f. Go to step 6.
6. Click Save. The application saves your changes and displays a success message.
