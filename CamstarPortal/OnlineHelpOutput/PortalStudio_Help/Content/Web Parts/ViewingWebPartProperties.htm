Viewing Web Part Properties

Portal Studio allows you to view the properties of a collection of web parts. Refer to "Web Part Properties" for information on specific web part properties.

How to View Web Part Properties

Follow these steps to view web part properties:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Double-click a page. The page appears in the working area.
4. Select a cell that contains a web part.
5. Click the WebProperties button in the Properties pane. The WebProperties window appears.
