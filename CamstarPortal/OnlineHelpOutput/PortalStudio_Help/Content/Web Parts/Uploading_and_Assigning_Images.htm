Uploading and Assigning Images

You can upload images to Portal Studio and assign them to controls that support images, such as buttons.

When Uploading and Assigning Images

The ImageControl, Button, FlyoutDropdown, and SectionDropdown controls support uploading and assigning images.

Images are uploaded using the DefaultImage icon in the Properties pane. The button shows an image thumbnail and image name if an image is already assigned to the control. Uploaded images become part of Portal Studio's image library and are available to all controls that support images.

You can enter part of an image's file name in the DefaultImage text field to return images matching the partial image name in the drop-down.

How to Upload Images

Follow these steps to upload images:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Double-click a page or web part. The page or web part appears in the working area.
4. Click Show/Hide Properties if the Properties pane is hidden. The Properties pane appears.
5. Select a control that supports images. The properties for the control appear in the Properties pane.
6. Click the icon for DefaultImage in the Properties pane. The Open pop-up appears.
7. Select an image file and click Open. The Open pop-up closes and the image is uploaded and assigned to the control.
8. Click Save. The application saves your changes and displays a success message.

How to Assign Images

Follow these steps to assign images:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Double-click a page or web part. The page or web part appears in the working area.
4. Click Show/Hide Properties if the Properties pane is hidden. The Properties pane appears.
5. Select a control that supports images. The properties for the control appear in the Properties pane.
6. Select an image from the DefaultImage drop-down in the Properties pane. The image appears in the icon and its filename appears in the text field.
7. Click Save. The application saves your changes and displays a success message.
