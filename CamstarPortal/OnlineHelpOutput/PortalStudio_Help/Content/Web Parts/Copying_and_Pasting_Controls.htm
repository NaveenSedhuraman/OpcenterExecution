Copying and Pasting Controls

You can copy and paste controls in the working area of Portal Studio.

When Copying and Pasting Controls

A web part with a blank cell must be present when pasting a control. Refer to "Creating a Web Part" for information on adding web parts to Portal pages.

You can copy and paste controls by:

• Using the keyboard commands Ctrl+C and Ctrl+V.
• Right-clicking in the working area and selecting Copy or Paste from the short-cut menu.
• Clicking Edit and then selecting Copy Control or Paste Control.
Note: You can cut controls by clicking Edit and then selecting Cut Control. You can then paste the control in a blank cell.

How to Copy and Paste Controls

Follow these steps to copy and paste controls:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Double-click a page or web part. The page or web part appears in the working area.
4. Select a control in the working area.
5. Right-click the control and select Copy.
6. Select a blank cell in a web part, opening another page or web part, if necessary.
7. Right-click the blank cell and select Paste. The control appears in the cell.
8. Click Save. The application saves your changes and displays a success message.
