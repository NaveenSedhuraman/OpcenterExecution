Enabling the Change Mgt Save Switch

You can add a Boolean Switch control to Modeling pages in Portal Studio that appears as a Change Mgt Save switch in Modeling. The Change Mgt Save switch allows Modeling users to turn on and off the option to associate modeling instances created during a session to change management packages.

When Enabling the Change Mgt Save Switch

Boolean Switch controls display as radio buttons by default. Boolean Switch controls must have the BooleanSwitchDisplayMode property set to Slide in the Properties pane to appear as the Change Mgt Save switch.

The Change Mgt Save switch is hidden on Modeling pages in Modeling until a change management package has been created. Refer to the Opcenter Execution Medical Device and Diagnostics Change Management User Guide or the Opcenter Execution Core Change Management User Guide for information on creating packages.

Boolean Switch controls can only be added to a web part. Controls cannot be added directly to a page, however, controls can be added to a web part that is on a page. Refer to "Adding a Control to a Web Part" for information.

The text that appears in the left and right cells of the Change Mgt Switch is customizable and appears as "Off" in the left cell and "On" in the right cell by default. You can also customize the color and font of the cells using the CSS class.

Note: A Modeling user must have permission to assign content to a change management package to use the Change Mgt Save switch. Refer to the Opcenter Execution Medical Device and Diagnostics Modeling User Guide or the Opcenter Execution Core Modeling User Guide for information.

How to Enable the Change Mgt Save Switch

Follow these steps to enable the Change Mgt Save switch:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Double-click a Modeling page. The Modeling page appears in the working area.
4. Click Show/Hide Toolbox if the Toolbox is hidden. The Toolbox appears.
5. Click and drag a Boolean Switch control onto a cell on a web part. The Boolean Switch control appears in the web part.
6. Click Show/Hide Properties if the Properties pane is hidden. The Properties pane appears.
7. Enter a name for the Boolean Switch control in the Name field in the Properties pane.
8. Select Slide from the BooleanSwitchDisplayMode field in the Properties pane.
9. Click Save. The application saves your changes and displays a success message.
