Reloading a Web Part

Portal Studio allows you to reload a web part from the Components catalog.

When Reloading a Web Part

Reloading a web part only works when the web part:

• Is part of a page.
• Exists as an independent web part listed in the Components catalog.
• Shares its name with the web part listed in the Components catalog.

 

How to Reload a Web Part

Follow these steps to reload a web part:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Double-click a page containing a web part. The page appears in the working area.
4. Select a web part on the page and make any changes to it.
5. Click Reload Web Part. All changes since your last save are removed.
