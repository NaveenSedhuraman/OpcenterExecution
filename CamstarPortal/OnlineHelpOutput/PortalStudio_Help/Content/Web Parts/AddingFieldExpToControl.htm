Adding a Field Expression to a Control

Portal Studio allows you to add controls to a web part. Field expressions can then be added to a control on the web part to link that control to specific service fields.

When Adding a Field Expression to a Control

Data-driven controls have a FieldExpression property that link the control with specific service field(s).

Not all controls use field expressions. For example, tab container is a control that only holds another page and does not process a service. Refer to "Field Expressions" for information.

How to Add a Field Expression to a Control

Follow these steps to add a Field Expression to a control:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Double-click a page or web part that contains a control. The page or web part appears in the working area.
4. Select a control in the working area.
5. Click More for FieldExpressions in the Properties pane. The Field Expressions Builder pop-up appears.
6. Search for the service that contains the field you want to use.
7. Select the field and click OK. The Field Expressions Builder pop-up closes and the field expression is added to the FieldExpressions field in the Properties pane.
8. Click Save. The application saves your changes and displays a success message.
