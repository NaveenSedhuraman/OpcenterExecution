Using the Color Picker

The Color Picker control allows you to add a color selection field to a web part for tasks such as color coding items on a Portal page.

The Color Picker appears in the list of available controls in the Toolbox pane. Refer to "Adding a Control to a Web Part" for information on adding the Color Picker to a web part.

When Using the Color Picker

The Color Picker contains both a field for entering hexadecimal color values and a color icon for selecting a color from a series of swatches. Hexadecimal values must begin with the pound symbol (#) when entering values manually, for example, "#ffff00" would be an accepted value.

Clicking the blank icon displays a color pallet containing Theme and Standard color swatches as well as the currently selected color and its hexadecimal value. You can click More Colors to select a color from a color wheel, or click History to view and select a previously selected color. The pallet will retain up to 28 previously selected colors.
