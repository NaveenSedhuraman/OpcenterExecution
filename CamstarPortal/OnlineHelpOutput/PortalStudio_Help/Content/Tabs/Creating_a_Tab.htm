Creating a Tab

You can create a tab and then specify on a menu that a page will open in that tab.

When Creating a Tab

You can select an icon to use as an indicator on the tab when you view it on the menu. Images are uploaded using the Image icon in the IconUrl field. The button shows an image thumbnail if an image is already assigned to the tab. Uploaded images become part of Portal Studio's image library and are available to all tabs.

You can enter part of an image's file name in the IconUrl text field to return images matching the partial image name in the drop-down.

The process for uploading and assigning images to tabs is similar to the process for uploading and assigning images to controls. Refer to "Uploading and Assigning Images" for procedures on uploading and assigning images.

How to Create a Tab

Follow these steps to create a tab:

1. Access Portal Studio.
2. Click New and select Tab. A blank tab appears in the working area.
3. Enter a name field for the tab in the Name field.
4. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
5. Click Save. The application saves your changes and displays a success message.
