Importing an Item

You can import XML files in Portal Studio. You can import new web parts and pages or import changes to update existing web parts and pages.

When Importing an Item

You can import an XML file only to an open page or web part in the highest active workspace.

Once you import a page or web part, you can then save the page or web part with an updated name.

Note: The size of an import is limited to 30 Megabytes (MB) by default. If your bundle is larger than this, you can increase the value for the MaxAttachmentSizeLimit setting in the Portal Settings. Refer to "Configuring Portal Settings" for information.

How to Import an Item

Follow these steps to import an item:

1. Access Portal Studio.
2. Create or open an existing page or web part.
3. Click the Import button. An Open dialog box appears.
4. Locate and select the XML file you want to import.
5. Click Open. The XML file is imported into the open page or web part.
6. Change the name of the page or web part in the Properties pane.
7. Click Save. The application saves your changes and displays a success message.
