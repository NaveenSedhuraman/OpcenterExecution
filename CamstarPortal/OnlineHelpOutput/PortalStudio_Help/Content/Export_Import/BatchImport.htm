Performing Batch Imports

Batch Import allows you to import multiple pages, web parts, and pageflows from a single XML or Zip file.

When Performing Batch Imports

You must select at least one workspace from the Personalization Type to Import list.

An Import Progress... pop-up displaying two progress bars appears when you click the Start Import button. The first bar displays the progress of each imported item and the second bar displays the total progress of the import.

How to Perform Batch Imports

Follow these steps to perform a batch import:

1. Access Portal Studio.
2. Click the Batch Import button. The Batch Import window appears.
3. Select at least one workspace from the Personalization Type to Import list.
4. Click More for Import from File. The Open pop-up appears.
5. Navigate to and select an XML or Zip file and click Open. The Open pop-up closes and the file name appears in the Import from File field.
6. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
7. Click Start Import. The Import Progress... pop-up appears and displays the progress of the import. The Import Progress... pop-up closes and the Batch Import window displays a success message when the import process is complete.
