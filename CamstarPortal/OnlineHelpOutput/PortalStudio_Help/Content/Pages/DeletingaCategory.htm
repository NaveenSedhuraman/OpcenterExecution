Deleting a Page Group

Page groups can be removed from the UI Components catalog. Removing a group does not delete the pages contained within it.

Note: The All group cannot be removed.

How to Delete a Page from a Page Group

Follow these steps to delete a page from a page group:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Expand a page group.
4. Right-click a page and clear the page groups you want to remove the page from. The page is removed from the cleared groups.

How to Delete a Page Group

Follow these steps to delete a page group from Portal Studio:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Right-click a group and select Remove. The group is removed from the UI Components catalog.
