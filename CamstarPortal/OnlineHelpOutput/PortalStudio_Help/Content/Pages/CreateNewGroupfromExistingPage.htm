Creating a New Page Group

New page groups can be created using the shortcut menu used to assign a page to a group.

When Creating New Page Groups

Right-clicking a page and selecting Add to a new group automatically adds the page to the newly created page group.

You can edit the name of an existing page group from the shortcut menu used to create a page group.

How to Create a New Page Group

Follow these steps to create a new page group:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Right-click a page and select Add to a new group. The Enter a group name dialog appears.

Or

Right-click an existing page group and select Add a new group. The Enter a group name dialog appears.

4. Enter a name for the group.
5. Click OK. The dialog closes and a new group appears in the UI Components catalog.

How to Edit a Page Group

Follow these steps to edit a page group:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Right-click a page group and select Edit. The Enter a group name dialog appears.
4. Enter a new name for the group.
5. Click OK. The dialog closes and the group appears in the UI Components catalog with the new name.
