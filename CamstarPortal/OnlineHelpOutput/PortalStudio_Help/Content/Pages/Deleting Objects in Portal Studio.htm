Deleting Objects and Pages

You can permanently delete controls, web parts, and pages in Portal Studio.

When Deleting Objects and Pages

Objects and pages deleted in Portal Studio are permanently removed from the application and cannot be recovered. Deleting an object or page in Portal Studio displays a confirmation message.

How to Delete Objects

Follow these steps to delete objects:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Double-click a page, web part, or control. The object appears in the working area.
4. Select the object to be deleted, if necessary.
5. Press the Delete key. A Delete confirmation message appears.
6. Click OK. The Delete confirmation closes and the object is deleted.
7. Click Save. The application saves your changes and displays a success message.

How to Delete Pages

Follow these steps to delete pages:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Double-click a page. The page appears in the working area.
4. Click Delete. The Delete WebPage dialog appears.
5. Click OK. The Delete WebPage dialog closes and the page is deleted.
