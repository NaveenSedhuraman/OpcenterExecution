Viewing Web Parts on a Page

Portal Studio allows you to view which web parts are being used on a page. You can view either the dynamic web parts or static web parts used on a page.

When Viewing Web Parts on a Page

When you select a web part in the DynamicWebParts pop-up, its properties are displayed in the right pane. You can then use the right pane to drill down further.

How to View Web Parts on a Page

Follow these steps to view web parts on a page:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Double-click a page. The page appears in the working area.
4. Click Show/Hide Properties if the Properties pane is hidden. The Properties pane appears.
5. Click the DynamicWebParts button in the Properties pane. The DynamicWebParts pop-up appears.
6. Select a web part from the list in the left pane. The web part's properties are displayed in the right pane.
