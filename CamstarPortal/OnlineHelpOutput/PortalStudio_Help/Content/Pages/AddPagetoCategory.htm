Adding a Page to a Group

Portal Studio allows you to categorize the list of available pages into groups. Pages can be assigned to multiple groups.

When Adding a Page to a Group

Some default pages can appear in the Undefined group in the Components tab of the UI Components catalog when logging into Portal Studio for the first time when upgrading from an earlier version. You can sort the default pages in the Undefined group into other groups automatically from the UI Components catalog. Refer to "How to Sort Default Pages Automatically" for information.

How to Add a Page to a Group

Follow these steps to add a page to a group:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Right-click a page. A list of existing groups appears.
4. Mark the checkbox for the group to which you want to add the page. The list closes and the page is added to the selected group.
5. Repeat step 4 to add the page to additional groups as needed.

How to Sort Default Pages Automatically

Follow these steps to sort default pages automatically:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Select the All group.
4. Right-click the All group and select Restore default settings. The default pages in the Undefined group are sorted into other groups.
