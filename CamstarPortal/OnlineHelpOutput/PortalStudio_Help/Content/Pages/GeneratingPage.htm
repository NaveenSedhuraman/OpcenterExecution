Generating a Page

The generate page function allows you to select a service from the Field Directory and automatically generate a page from that service (using the fields that have been pre-defined).

When Generating a Page

The service must have common fields already defined for it. These are the fields that will be added to the page when it is generated. Common fields are indicated by the shortcut icon.

An action appropriate to the selected service is automatically added to the page. Therefore, you could test this page as soon as you save it.

How to Generate a Page

Follow these steps to generate a page:

1. Access Portal Studio.
2. Click Show/Hide Toolbox if the Toolbox is hidden. The Toolbox appears.
3. Click the Fields Directory tab. A list of services appears.
4. Select the service from which you want to generate a page.
5. Click Generate Page. A new page is created.
6. Click Show/Hide Properties if the Properties pane is hidden. The Properties pane appears.
7. Enter a name for the page in the Name field of the Properties pane.
8. Click Save. The application saves your changes and displays a success message.
