Previewing a Page

Portal Studio allows you to preview a page in a browser window.

When Previewing a Page

The Preview button is enabled when the page is saved.

When you preview a page, the page is opened in a new window that is not part of the Portal framework. Refer to "Testing a Page" to test a page within the Portal framework.

How to Preview a Page

Follow these steps to preview a page:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Double-click a page. The page appears in the working area.
4. Click Save. The application saves your changes and displays a success message.
5. Click Preview. The page is opened in a web browser.
