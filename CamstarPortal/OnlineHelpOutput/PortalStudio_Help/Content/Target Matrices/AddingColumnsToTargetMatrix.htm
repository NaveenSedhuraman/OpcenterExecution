Adding Columns to a Target Matrix

Portal Studio allows you to add columns to a new or existing target matrix.

When Adding Columns to a Target Matrix

All target matrices have the same columns, so adding a column to one adds the column to all target matrices.

How to Add Columns to a Target Matrix

Follow these steps to add columns to a target matrix:

1. Access Portal Studio.
2. Perform the "How to Create a Target Matrix" procedure.

Or

Open an existing target matrix.

3. Click More for Columns. The TargetMatrix# - Columns pop-up appears.
4. Click Add. A new column appears in the list.
5. Enter a name for the column in the Name field.
6. Repeat steps 4-5 to add additional columns.
7. Click OK. The TargetMatrix# - Columns pop-up closes.
8. Click Save. The application saves your changes and displays a success message.
