Opening an Existing Target Matrix

Existing target matrices can be opened from the Modeling tab in the UI Components catalog.

How to Open an Existing Target Matrix

Follow these steps to open an existing target matrix:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Select the Modeling tab.
4. Expand Target Matrices.
5. Double-click a target matrix. The target matrix appears in the working area.
