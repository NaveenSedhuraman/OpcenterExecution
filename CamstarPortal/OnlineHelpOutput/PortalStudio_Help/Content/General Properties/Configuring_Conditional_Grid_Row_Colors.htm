Configuring Conditional Grid Row Colors

You can configure grid rows to display css style attributes, such as a colored background or colored text, when custom conditions are met. For example, if a container's status is "on hold" the rows could be set to display a red background.

When Configuring Conditional Grid Row Colors

You can optionally condition grid rows to display a different background color or colored text when both the condition is met and the row is selected. The background color for selected rows is defined in the SelectedStyle field.

Expressions are evaluated after you update or tab away from a row when inline editing occurs within a grid.

An expression can contain multiple conditions using the "and" and "or" operators. For example, the following is an acceptable conditional expression:

"DataType = 'Integer' and AttributeValue > 10"

This table defines some common conditions for evaluating an expression entered in the Condition field:

Condition

Definition

>

Greater than.

<

Less than.

=

Equals.

Like

If the field contains a string. The expected syntax is " Like 'STRING' "
Note: The Condition field expects Microsoft DataColumn.Expression Property syntax. Refer to the Microsoft Developer Network for information on additional condition options.

The NormalStyle and SelectedStyle fields expect stylesheet references or stylesheet syntax to define the row or text color. The expected syntax is "class:<css-class-name>" where <css-class-name> is a name of a css class defined in a stylesheet. For example, an accepted value might be "background:red".

Note: You must include the "!important" indicator for background colors when using a class for the NormalStyle or SelectedStyle.

ConditionalRowStyles Pop-up Field Definitions

This table defines the fields on the ConditionalRowStyle pop-up.

Field

Definition

Type

Condition

Conditional expression that changes the background row color when the expression evaluates as true.

Required

NormalStyle

The background color of the row when the conditional expression is true. For example: background:red;color:yellow, where ‘color’ is the text indication.

Note: You can set ‘background’ only or ‘color’ only, or both, on a row for each Style.

How to Configure Conditional Grid Row Colors

Follow these steps to configure conditional grid row colors:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Double-click a page containing a grid control. The page appears in the working area.
4. Click Show/Hide Properties if the Properties pane is hidden. The Properties pane appears.
5. Select a grid control. The properties for the grid control appear in the Properties pane.
6. Click More for Settings. The Grid Settings Editor pop-up appears.
7. Expand Layout.
8. Click More for ConditionalRowStyles. The ConditionalRowStyles pop-up appears.
9. Click Add. A StyleRule is added to the list and blank fields appear in the right pane.
10. Enter a conditional expression in the Condition field.
11. Enter a background color in the NormalStyle field.
12. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
13. Repeat steps 9-12 to add additional grid row conditions.
14. Click OK. The ConditionalRowHandler pop-up closes.
15. Click OK. The Grid Settings Editor pop-up closes.
16. Click Save. The application saves your changes and displays a success message.
