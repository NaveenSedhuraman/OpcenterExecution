Selection Value Expressions

Fields can be configured to offer a list of values. Selection value expressions have the same default behavior they had in the Portal web application. The default behavior can be overwritten through the control properties in the Field Expression Builder.

Selection value expressions can be set to one of the following:

• Static Value List
• Set Data
• Query

Static selection values allow you to display all the instances objects that have the same field type.

A query allows you to define a field’s selection values based on the query.

When Using Selection Value Expressions

The Static option does not allow you to change the Selection Values Query.

ValueExpression can be used to combine Product Name and Revision into a single column. You must use the proper syntax such as:

(Product) : (Rev)
