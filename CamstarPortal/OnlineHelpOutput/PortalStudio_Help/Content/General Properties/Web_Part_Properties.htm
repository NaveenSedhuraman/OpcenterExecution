Web Part Properties

Web part properties allow you to view or edit the properties of web parts. Web part properties appear in the Properties pane when a web part is selected in the working area. Refer to "Properties Pane" for information on the Properties pane.

Web Part Properties Descriptions

This table provides the descriptions of web part properties.

Web Part Property

Property Description

General

Actions

Actions of the web part. Allows for definition of web part level actions and buttons. Refer to "Action Properties" for information.

BroadcastInitiators

Used to create initiators that will trigger page level connection data contracts between web parts.

CollapsibleGroups

Will be deprecated in an upcoming release.

Note: You must create collapsible sections using the ToggleDisplayMode property of the Toggle Container control. Refer to "Control Properties" for information.

Control

References a control on the page.

DataContractMembers

Data Contracts used for communication between pages.

DataSubmissionMode

How a field reacts when a page submits data to the server. Options include:

• NotSet: Mode is not set. This is the default mode.
• Force: Value is always included in submission data.
• Changed: Value is only submitted if it was changed.
• Filled: Value is only submitted if the field is not empty.
• ChangedAndFilled: Value is only submitted if the field is not empty and was changed.
• Skip: Value is not submitted or loaded.
• ViewOnly: Value is not submitted.
• Skip Request: Value is not loaded. The selection values for the control are not loaded.

Important: Siemens recommends the Changed option for Modeling pages.

DirtyFlagTrigger

Boolean field that marks the page as dirty if any field within the web part changes value when set to true.

Note: Currently only affects web parts within the modeling framework.

EventHandlers

Allows for specification of a handler from an event. Refer to "Defining Event Handlers" for information.

Layout

Visual appearance of the web part. Siemens recommends that you do not change this property directly.

Note: Sub-properties appear based on the cells and controls in the currently selected web part.

Name

Name of the web part.

PrimaryServiceType

Service type for controls that do not specify a service type.

SessionVariables

Used to configure session variables. Refer to "Defining Session Variables" for information.

TypeName

Code behind the type of the web part.

WebProperties

Collection of web properties for the web part.

Behavior

ActionsStyle

Determines how actions will be rendered, such as buttons or links.

ClientPropertyConditions

Allows for setting data to a property based on a condition result from an event on browser side.

IncludeHandlerActions

Includes the current page’s actions, web part actions, and page action modeling items when set to True. Includes only the ActionControl web part’s actions when set to False.

PropertyConditions

Allows definition of a web part's property value as a result of an evaluated expression (condition). The property is evaluated at Page execution run-time. Refer to "Property Conditions" for information.

Appearance

IsHidden

Sets the web part to not appear on pages.

IsMinimized

Sets the web part to appear minimized.

Title

Text that appears as the title of the web part. Text entered here takes priority over the TitleLabel field when both text is entered and a label is selected.

TitleLabel

Selects a label to be used as the title of the web part. Refer to "Searching for Page Action Labels" for information on using the Label Lookup pop-up.

Layout

CssClass

Stylesheet reference for the web part.

DisplayBorder

Displays a border on the web part.

DisplayFooter

Displays a footer on the web part.

DisplayHeader

Displays a header on the web part.

Height

Height in pixels for the display of the web part.

HorizontalAlignment

Horizontal alignment web part elements.

Margin

The margin of the web part.

Padding

The padding of the web part.

VerticalAlignment

Vertical alignment of web part elements.

Width

Width in pixels for the display of the web part.

History

Created

Creation date and time of the web part.

CreatedBy

Creator of the web part.

HowLong

Time period since creation of the web part.

LastModified

Last modified date of the web part.

LastModifiedBy

User who modified the web part most recently.
