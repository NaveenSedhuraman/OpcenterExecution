Field Expressions

Field expressions are set automatically when you drop a field onto a grid from the Field Directory. They can be changed in the Properties pane once the field has been added to the grid.

There are two types of field expressions: absolute and relative. Relative field expressions append to the page service and are particularly useful for a shared web part (used on multiple pages) so you do not have to modify them on every page.

When Using the Field Expressions

Field expressions can use relative expressions (by removing the first entry) to make them reusable.

The value of the control never gets submitted when a Web Service method is called, if the value of the FieldExpressions property is blank.
