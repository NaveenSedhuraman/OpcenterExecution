Grid Settings Editor

Grid properties can be modified from the Grid Settings Editor pop-up in Portal Studio. The fields that appear in the Grid Settings Editor pop-up are dependent on the type of grid data mode selected in the Properties pane.

Note: Selecting Generic from the GridDataMode field in the Properties pane will set all fields on the Grid Settings Editor. Siemens recommends you select a grid data mode other than Generic based on the function the grid will perform.

How to Access the Grid Settings Editor

Follow these steps to access the Grid Settings Editor:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Double-click any Portal page or web part containing a grid control. The page or web part appears in the working area.
4. Click Show/Hide Properties if the Properties pane is hidden. The Properties pane appears.
5. Select a grid control. The settings for the grid appear in the Properties pane.
6. Click More for Settings in the Properties pane. The Grid Settings Editor pop-up appears.
Note: You can also select a grid and press the g key to show the Grid Settings Editor pop-up.
