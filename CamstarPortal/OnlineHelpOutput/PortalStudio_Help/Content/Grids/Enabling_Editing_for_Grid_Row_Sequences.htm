Enabling Row Sequence Changes

You can enable changes to the sequence of rows in ItemList or SubentityList grids in the Portal by modifying grid controls in Portal Studio. You can modify grids to enable dragging and dropping rows, selecting and then rearranging rows using the Move Row buttons, or both.

When Enabling Row Sequence Changes

Ensure that the grid’s DataSubmissionMode control is set to Changed. Refer to "Control Properties Descriptions" for information.

You enable row sequencing by setting the SequenceColumn and SequenceMode properties. Refer to "Grid Settings Editor" for information.

Enabling grid row sequence changes will alter the appearance of the grid in the Portal. Setting a grid for dragging and dropping displays a double-arrow icon next to each moveable row. Setting a grid for arrow moving displays Move Row buttons in the grid's toolbar.

This table defines the settings available for grid row sequences.

Setting

Definition

None

Disables changing the sequence of grid rows.

Dragging

Enables dragging and dropping to rearrange grid rows.

ArrowMoving

Enables use of the Move Row buttons to rearrange grid rows.

DraggingAndMoving

Enables both dragging and dropping and use of the Move Row buttons.

How to Enable Row Sequence Changes

Follow these steps to enable row sequence changes:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Double-click any Portal page that contains an ItemList or SubentityList grid. The page appears in the working area.
4. Click Show/Hide Properties if the Properties pane is hidden. The Properties pane appears.
5. Select a grid control on the page. The settings for the control appear in the Properties pane.
Note: Ensure that DataSubmissionMode is set to Changed for the selected grid.
6. Click the More button for the Settings field in the Properties pane. The Grid Settings Editor pop-up appears.
7. Expand Layout.
8. Select the --index-- column in the grid control in the SequenceColumn field.
9. Select a setting for SequenceMode.
10. Click OK.
11. Click Save. The application saves your changes and displays a success message.
