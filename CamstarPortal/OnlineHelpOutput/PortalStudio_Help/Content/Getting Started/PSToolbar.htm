Portal Studio Global Toolbar

The Portal Studio global toolbar provides easy access to the functions used in Portal Studio.

Note: The functions on the Portal Studio global toolbar are also available through the drop-down menus on the Menu bar.

Portal Studio Global Toolbar Buttons

This table defines the buttons available on the Portal Studio global toolbar.

Button

Click this button to . . .

Change the working area to another workspace. The color that appears next to the workspace name is determined by the color specified for the workspace in Management Studio. Refer to "Developing in Workspaces" for information.

Open the Portal Settings window.

Return to the Portal.

Logout of the Portal and return to the login page.

Create a new virtual object. The Create New Page pop-up appears when the plus sign portion of the button is clicked, allowing you to configure a new Portal page. Clicking the "New" text portion of the button displays a drop-down with options for creating a new web part, pageflow, menu definition, tab, map item, target matrix item, page action, page action rule, or page action category.

Save the pending changes for component selected in the working area. Refer to "Saving Changes" for information on saving your changes.

Note: Portal Studio enables this button when you make a change to the component currently in focus in the working area. You can set this button to always be enabled by setting the AlwaysEnableSave Studio setting to True. Refer to "Studio Settings" for information on modifying Portal Studio settings.

Save the pending changes for components in all open tabs in the working area. Refer to "Saving Changes" for information on saving your changes.

Note: Portal Studio enables this button when you make a change to the component currently in focus in the working area. You can set this button to always be enabled by setting the AlwaysEnableSave Studio setting to True. Refer to "Studio Settings" for information on modifying Portal Studio settings.

Publish changes to the developer content.

Reverse your last change.

Copy the selected work item.

Delete the open work item.

Import an item from an XML file.

Export the entire work item, the changes between contents, or the system content.

Import multiple web parts, pages, and pageflows from a single XML file.

Export multiple web parts, pages, and pageflows to a single XML file.

Check out the current work item from your source control system.

Check the current work item into your source control system.

Preview the current page in a browser.

Preview or test a saved item.

Generate the current page.

Convert an .aspx page to a Portal page.

Return to the published content or system content for the work item.

Open the selected web part (on a page) in a new tab.

Refresh the web part to its previously saved state.

Display or hide the Page catalog.

Display or hide the Toolbox pane.

Display or hide the Properties pane.

Display or hide the Trace pane.

Reset any changes made to the layout of panes in the working area.
