Viewing Online Help

You can view online help for Portal Studio from the Help menu.

When Viewing Online Help

Online help appears as a pop-up in your browser. Help appears in the preferred language set in your browser if translated help is available. Help appears in English by default. Ensure you have the latest language packages installed for your preferred language.

Note: You may need to allow blocked content for scripts and Active X controls when viewing online help in Internet Explorer.

How to View Online Help

Follow these steps to view online help:

1. Access Portal Studio.
2. Click Help and then select Portal Studio Help.... A browser pop-up displaying Portal Studio online help appears.
