Enabling Silverlight in Google Chrome

Support for the Netscape Plugin Application Programming Interface (NPAPI) in Google Chrome versions 42, 43, and 44 is disabled by default and disables certain plugins, such as Microsoft Silverlight. You must enable Silverlight before accessing Portal Studio when using Chrome as your browser.

Important: Chrome has limited or no support for Silverlight beyond version 44. Siemens recommends you access Portal Studio using a supported version of Internet Explorer.

When Enabling Silverlight in Google Chrome

You must also set the win32k renderer lockdown setting to disabled when accessing Portal Studio with versions 43 and 44 of Chrome. You must close and then reopen Chrome before accessing Portal Studio after modifying this setting.

How to Enable Silverlight in Google Chrome Version 42

Follow these steps to enable Silverlight in Google Chrome version 42:

1. Run Google Chrome.
2. Navigate to the address "chrome://flags/#enable-npapi". The Experimental Features page appears.
3. Click Enable for Enable NPAPI. The setting's background changes to white and NPAPI plugins are enabled.
4. Log on to the Portal.
5. Click Studio. A blank page appears.
6. Right-click the plugin icon and select Run this Plugin. The Portal Studio home page and Start Page dashboard appear.

How to Enable Silverlight in Google Chrome Versions 43 and 44

Follow these steps to enable Silverlight in Google Chrome versions 43 and 44:

1. Run Google Chrome.
2. Navigate to the address "chrome://flags/#enable-npapi". The Experimental Features page appears.
3. Click Enable for Enable NPAPI. The setting's background changes to white and NPAPI plugins are enabled.
4. Select Disabled for Enable win32k renderer lockdown Windows. The setting's background changes to yellow and the win32k renderer lockdown setting is disabled.
5. Close and reopen your browser.
6. Log on to the Portal.
7. Click Studio. A blank page appears.
8. Right-click the plugin icon and select Run this Plugin. The Portal Studio home page and Start Page dashboard appear.
