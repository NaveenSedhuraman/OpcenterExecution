UI Components Catalog

The UI Components catalog contains several tabs that provide access to component, navigation, and mapping items.

The Components tab provides access to these components:

• Pages
• Web Parts
• Pageflows

The Navigation tab provides access to these navigation items:

• Menu Definitions
• Tabs

The Modeling tab provides access to these mapping items:

• Page Actions
• Page Action Categories
• Creating Page Action Rules
• Page Maps
• Target Matrices

The UI Components catalog tabs are organized into drop-downs for each component item type. Each drop-down contains a search field, Create New Item button, a set of view mode buttons, and a refresh button. Clicking the Create New Item button will display a new item for that item type in the working area.

The Show/Hide Catalog button on the Portal Studio toolbar displays the UI Components catalog if hidden, and hides the UI Components catalog if visible. You can also display or hide the UI Components catalog by clicking View and then selecting Show/Hide Catalog.

Note: Some default pages can appear in the Undefined group in the Components tab of the UI Components catalog when logging into Portal Studio for the first time when upgrading from an earlier version. You can sort the default pages in the Undefined group into other groups automatically from the UI Components catalog. Refer to "How to Sort Default Pages Automatically" for information.

How to Navigate the UI Components Catalog

Follow these steps to navigate the UI Components catalog:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Click a tab on the bottom of the catalog to display the options on that tab.
4. Use the arrow next to a section to expand or contract that section.
