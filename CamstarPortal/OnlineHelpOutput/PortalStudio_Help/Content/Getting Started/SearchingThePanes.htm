Searching for Objects in a Pane

Each pane in Portal Studio provides a search field that allows you to easily find a specific object in that catalog or directory.

Portal Studio filters the list as you type in the Search field, helping you to find the desired object. The search works on both flat lists and trees.

When Searching for an Object in a Pane

You can enter a name or partial name in the search field at the top of the pane. The search assumes a wildcard on each end of the entry, allowing you to find all objects that contain the phrase or partial phrase in their name.

Clicking the X in the search field clears the field.

How to Search for an Object in a Pane

Follow these steps to search for an object in a pane:

1. Access Portal Studio.
2. Use the Show/Hide buttons to display the desired pane.
3. Start typing in the search field at the top of the pane. The pane filters as you type.
