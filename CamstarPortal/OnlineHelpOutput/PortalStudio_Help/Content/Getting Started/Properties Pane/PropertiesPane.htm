Properties Pane

The Properties pane is used to view and edit the properties of Portal Studio virtual objects such as pages, web parts, and controls.

A toolbar is displayed at the top of the Properties pane and sorts the list of properties alphabetically or by group.

A description window displaying information about control and page properties appears beneath the Properties pane when a property is selected. Refer to "Control Properties" for a list of control properties and their property descriptions. Refer to "Page Properties" for a list of page properties and their property descriptions.

How to Show the Properties Pane

Follow these steps to show the Properties pane:

1. Access Portal Studio.
2. Click Show/Hide Properties if the Properties pane is hidden. The Properties pane appears.
