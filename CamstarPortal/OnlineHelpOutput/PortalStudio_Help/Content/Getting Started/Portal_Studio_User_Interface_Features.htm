Portal Studio User Interface Features

Portal Studio's user interface features standard buttons and icons throughout the application's panes and pop-ups.

Standard Buttons

Standard buttons are used for consistency throughout the application's user interface to enhance your user experience and streamline your data entry tasks.

This table defines the common buttons in Portal Studio.

Button

Click this button to . . .

OK

Save your changes and close a pop-up.

Cancel

Close a pop-up without saving your changes.

Note: You can optionally click the X in the right corner to close a pop-up.

...

Display a pop-up allowing for greater customization for the related field or property. This button is referred to as the "More" button.

Expand

Expand a section to show additional fields.

Collapse

Collapse an expanded section.

Clear

Clear selections and values you have entered in fields on a pop-up or pane.

Standard Icons

Standard icons are used for consistency throughout the application's user interface to enhance your user experience and streamline your data entry tasks.

The standard icons are illustrated and described in this table.

Icon

Description

Click this icon to . . .

Create New Item

Create a new item in the pane.

List Mode

Display a simple list of items.

Tree Mode

Display a list of items in a hierarchy.

Tiles Mode

Display a series of items in a tiled view.

Details Mode

Display a list of items with name and description information.

Hierarchical Sort

Display a list of fields or properties in a hierarchy.

Alphabetical Sort

Display an alphabetized list of fields or properties.

Refresh

Refresh the pane.
