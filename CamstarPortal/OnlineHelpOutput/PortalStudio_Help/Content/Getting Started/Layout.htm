Managing Layouts

Portal Studio allows you to configure the layout of a page or web part using a row and column layout containing cells.

You can modify a single cell's properties in the Properties pane. Refer to "Cell Properties" for information on modifying cell properties.

When Managing Layouts

Every page or web part contains at least one cell as part of the object's layout. Cells allow you to specify where objects will appear when placed on pages and web parts in relation to objects in other cells.

You can right-click a cell to display options that allow you to modify the cell's layout. You can optionally click and drag to select multiple cells and then right-click the highlighted cells to modify multiple layouts at once.

This table defines the options for managing cell layouts.

Insert Row After

Insert Column After
