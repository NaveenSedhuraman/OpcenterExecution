Adding Page Action Rules to Page Actions

Page action rules determine the conditions for when a dynamic action displays on a Portal page. You can add multiple rules to an action to control when an action displays.

When Adding Page Action Rules to Page Actions

Page action rules are available from the Modeling tab of the UI Components catalog, with many already created by default. Refer to "Creating Page Action Rules" for information on creating additional page action rules.

Page action rules can also be removed from a page action. A page action is no longer controlled by a page action rule after the rule has been removed.

ActionRules Pop-up Field Definitions

The ActionRules pop-up contains only the Name field. The Name field is used to select an existing page action rule and associate it to the page action.

How to Add Page Action Rules to Page Actions

Follow these steps to add page action rules to page actions:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Select the Modeling tab.
4. Double-click a page action. The page action properties pop-up appears in the working area.
5. Click More for ActionRules. The ActionRules pop-up appears.
6. Click Add. A new action rule appears in the list in the left pane.
7. Select a page action rule from the Name field.
8. Repeat steps 5-7 to add additional page action rules as necessary.
9. Click OK. The ActionRules pop-up closes.
10. Click Save. The application saves your changes and displays a success message.

How to Delete Page Action Rules from Page Actions

Follow these steps to delete page action rules from page actions:

1. Access Portal Studio.
2. Click Show/Hide Catalog if the UI Components catalog is hidden. The UI Components catalog appears.
3. Select the Modeling tab.
4. Double-click a page action. The page action properties pop-up appears in the working area.
5. Click More for ActionRules. The ActionRules pop-up appears.
6. Select a page action rule from the list in the left pane.
7. Click Delete. The page action rule is removed from the list.
8. Repeat steps 6-7 to delete additional page action rules as necessary.
9. Click OK. The ActionRules pop-up closes.
10. Click Save. The application saves your changes and displays a success message.
