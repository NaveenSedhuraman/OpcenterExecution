Searching for Page Action Labels

You can use the Label Lookup pop-up to search for predefined labels when defining page actions.

When Searching for Page Action Labels

Siemens recommends using the predefined labels in the database, rather than creating new labels each time a new page action is created.

Labels are used during the creation of page actions. Refer to "Configuring Page Actions" for information on creating page actions.

The Category column in the label search results grid appears only when All is selected in the Category search parameter field.

How to Search for Page Action Labels

Follow these steps to search for page action labels:

1. Access Portal Studio.
2. Click Create New and select Page Action. A new page action appears in your working area.

Or

Select the Modeling Tab in the Components pane and double-click an existing page action. The page action appears in your working area.

3. Expand the UIAction section.
4. Click the More button for LabelName. The Label Lookup Pop-up appears.
5. Enter or select one or more criteria to narrow your search.
6. Click Search. The application displays the labels matching your search criteria in the grid.
7. Select a label. The Default Text and Local Text for the label appear in the Default Text and Local Text fields.
8. Click OK. The Label Lookup pop-up closes and the information for the label populates the associated fields on the action window.
