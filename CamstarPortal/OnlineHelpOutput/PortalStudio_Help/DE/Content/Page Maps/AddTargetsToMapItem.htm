Hinzufügen von Zielen zu einem Zuordnungselement

Ziele (Targets) sind die Zielseite, zu denen Endbenutzer nach dem Anwenden der Logik umgeleitet werden.

Vorgehensweise zum Hinzufügen von Zielen zu einem Zuordnungselement

Gehen Sie folgendermaßen vor, um Ziele zu einem Zuordnungselement hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Wählen Sie die Registerkarte Modellierung (Modeling) aus.
4. Erweitern Sie Seitenzuordnungen (Page Maps).
5. Doppelklicken Sie auf eine Seitenzuordnung. Die Seitenzuordnung wird im Arbeitsbereich angezeigt.
6. Klicken Sie auf Eigenschaften anzeigen/ausblenden (Show/Hide Properties), wenn der Bereich "Eigenschaften" (Properties) ausgeblendet ist. Der Bereich "Eigenschaften" (Properties) wird angezeigt.
7. Klicken Sie im Eigenschaftenpanel für Ziele (Targets) auf Mehr (More). Das Popup "Ziele" (Targets) wird angezeigt.
8. Klicken Sie auf Hinzufügen (Add). Ein Ziel wird zu der Liste hinzugefügt.
9. Wählen Sie im Feld Ziel (Target) ein Ziel aus.
10. Wählen Sie im Feld Zieltyp (TargetType) einen Zieltyp aus.
11. Wiederholen Sie die Schritte 8 bis 10, um weitere Ziele hinzuzufügen.
12. Klicken Sie auf OK. Der Parameter wird hinzugefügt.
13. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
