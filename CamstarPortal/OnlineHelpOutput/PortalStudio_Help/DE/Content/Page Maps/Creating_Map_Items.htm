Erstellen eines Zuordnungselements

Bei der Seitenzuordnung wird eine Mustervergleichslogik mit Fallthrough verwendet. Sie wertet die Parameter aus, die Sie konfiguriert haben, um die beste Zielseite für die Aktion zu ermitteln.

Wenn Sie ein neues Zuordnungselement (Map Item) erstellen, werden im Arbeitsbereich in einem Popup Zuordnungselemente als Gruppe von Feldern angezeigt.

Felddefinitionen des Popups "Zuordnungselement" (Map Item)

Diese Tabelle definiert die Felder im Popup "Zuordnungselement" (Map Item).

Feld

Definition

Typ

Name

Eindeutiger Name für das Zuordnungselement.

Erforderlich

Parameters

Wenn Sie auf die Schaltfläche "Mehr" (More) klicken, wird das Popup "Parameter" (Parameters) angezeigt. Hier können Sie Parameter für das Zuordnungselement definieren.

Optional

SourcePages

Wenn Sie auf die Schaltfläche "Mehr" (More) klicken, wird das Popup "Quellseiten" (SourcePages) angezeigt. Hier können Sie Quellseiten dem Zuordnungselement zuweisen.

Optional

Targets

Wenn Sie auf die Schaltfläche "Mehr" (More) klicken, wird das Popup "Ziele" (Targets) angezeigt. Hier können Sie Ziele für das Zuordnungselement definieren.

Optional

Vorgehensweise zum Erstellen eines Zuordnungselements

Gehen Sie folgendermaßen vor, um ein Zuordnungselement zu erstellen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Neu (New) und wählen Sie Zuordnungselement (Map Item) aus. Im Arbeitsbereich wird ein leeres Zuordnungselement angezeigt.
3. Geben Sie im Feld Name einen Namen für das Zuordnungselement ein oder wählen Sie einen Namen aus.
4. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
5. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
