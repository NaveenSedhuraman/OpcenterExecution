Hinzufügen von Seitenzuordnungsparametern zu einem Zuordnungselement Element

Seitenzuordnungsparameter werden zum Konfigurieren von Bedingungen verwendet, unter denen das System einen Benutzer zu einer Zielseite umleitet. Parameter dienen als Vermittler zwischen einem Quellseiten-Datenvertrag und einem Zielseiten-Datenvertrag.

Beim Hinzufügen von Seitenzuordnungsparametern zu einem Zuordnungselement

Ein Parameter muss für ein Zuordnungselement eindeutig sein. Parameter werden beim Übertragen von Daten zwischen Seiten und zum Konfigurieren der Umleitungsbedingungen verwendet.

Vorgehensweise zum Hinzufügen von Seitenzuordnungsparametern zu einem Zuordnungselement

Gehen Sie folgendermaßen vor, um Seitenzuordnungsparameter zu einem Zuordnungselement hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Wählen Sie die Registerkarte Modellierung (Modeling) aus.
4. Erweitern Sie Seitenzuordnungen (Page Maps).
5. Doppelklicken Sie auf eine Seitenzuordnung, die Sie öffnen möchten. Die Seitenzuordnung wird im Arbeitsbereich angezeigt.
6. Klicken Sie auf Mehr (More) für Parameter. Das Popup "Parameter" (Parameters) wird angezeigt.
7. Klicken Sie auf Hinzufügen (Add). Ein Parameter wird zu der Liste hinzugefügt.
8. Geben Sie im Feld Name einen Namen ein oder wählen Sie einen Namen aus.
9. Wiederholen Sie die Schritte 7 bis 8, um weitere Parameter hinzuzufügen.
10. Klicken Sie auf OK. Das Popup "Parameter" (Parameters) wird geschlossen.
11. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
