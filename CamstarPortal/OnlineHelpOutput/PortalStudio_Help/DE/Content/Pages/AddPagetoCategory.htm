Hinzufügen einer Seite zu einer Gruppe

In Portal Studio können Sie die Liste der verfügbaren Seiten in Gruppen kategorisieren. Seiten können mehreren Gruppen zugewiesen werden.

Beim Hinzufügen einer Seite zu einer Gruppe

Wenn Sie sich beim Upgrade von einer früheren Versions erstmals bei Portal Studio anmelden, werden u. U. einige Standardseiten im Katalog der Benutzeroberflächenkomponenten auf der Registerkarte "Components" (Komponenten) in der Gruppe "Nicht definiert" (Undefined) angezeigt. Die Standardseiten in der Gruppe "Nicht definiert" (Undefined) können aus dem Katalog der Benutzeroberflächenkomponenten automatisch in andere Gruppen sortiert werden. Weitere Informationen finden Sie unter "Vorgehensweise zum automatisches Sortieren von Standardseiten".

Vorgehensweise zum Hinzufügen einer Seite zu einer Gruppe

Gehen Sie folgendermaßen vor, um eine Seite zu einer Gruppe hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Klicken Sie mit der rechten Maustaste auf eine Seite. Eine Liste der vorhandenen Gruppen wird angezeigt.
4. Aktivieren Sie das Kontrollkästchen für die Gruppe, zu der Sie die Seite hinzufügen möchten. Die Liste wird geschlossen und die Seite wird zu der ausgewählten Gruppe hinzugefügt.
5. Wiederholen Sie Schritt 4, um die Seite nach Bedarf zu weiteren Gruppen hinzuzufügen.

Vorgehensweise zum automatisches Sortieren von Standardseiten

Gehen Sie folgendermaßen vor, um Standardseiten automatisch zu sortieren:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Wählen Sie die Gruppe Alle (All) aus.
4. Klicken Sie mit der rechten Maustaste auf die Gruppe Alle (All) und wählen Sie Standardeinstellungen wiederherstellen (Restore Default Settings) aus. Die Standardseiten in der Gruppe "Nicht definiert" (Undefined) werden in andere Gruppen sortiert.
