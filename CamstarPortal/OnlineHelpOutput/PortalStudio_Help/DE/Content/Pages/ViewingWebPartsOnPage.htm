Anzeigen der Webparts auf einer Seite

Portal Studio bietet Ihnen die Möglichkeit anzuzeigen, welche Webparts auf einer Seite verwendet werden. Dabei können entweder die dynamischen oder die statischen auf einer Seite verwendeten Webparts angezeigt werden.

Beim Anzeigen der Webparts auf einer Seite

Wenn Sie im Popup "Dynamische Webparts" (DynamicWebParts) ein Webpart auswählen, werden seine Eigenschaften im rechten Bereich angezeigt. Im rechten Bereich kann dann ein weiterer Drilldown durchgeführt werden.

Vorgehensweise zum Anzeigen der Webparts auf einer Seite

Gehen Sie folgendermaßen vor, um Webparts auf einer Seite anzuzeigen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Doppelklicken Sie auf eine Seite. Die Seite wird im Arbeitsbereich angezeigt.
4. Klicken Sie auf Eigenschaften anzeigen/ausblenden (Show/Hide Properties), wenn der Bereich "Eigenschaften" (Properties) ausgeblendet ist. Der Bereich "Eigenschaften" (Properties) wird angezeigt.
5. Klicken Sie im Bereich "Eigenschaften" (Properties) auf die Schaltfläche DynamischeWebparts (DynamicWebParts). Das Popup "Dynamische Webparts" (DynamicWebParts) wird angezeigt.
6. Wählen Sie im linken Bereich ein Webpart aus der Liste aus. Die Eigenschaften des Webparts werden im rechten Bereich angezeigt.
