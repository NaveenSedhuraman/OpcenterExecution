Zurücksetzen auf den zuletzt veröffentlichten Zustand

Mit der Schaltfläche "Zurücksetzen zu" können Sie Seiten im höchsten aktiven Arbeitsbereich auf den zuletzt veröffentlichten Zustand oder auf den Zustand des Systeminhalts im Siemens PLM CEP-Basisarbeitsbereich zurücksetzen.

Beim Zurücksetzen auf den zuletzt veröffentlichten Zustand

Die Schaltfläche "Zurücksetzen zu" ist im Siemens PLM CEP-Basisarbeitsbereich deaktiviert. Sie ist nur auf Seiten aktiviert, auf denen Änderungen im höchsten aktiven Arbeitsbereich vorgenommen wurden. Weitere Informationen zum Speichern und Veröffentlichen von Änderungen in Arbeitsbereichen finden Sie unter "Entwickeln in Arbeitsbereichen".

Wenn Sie auf "Zurücksetzen auf" (Revert to) klicken und "Veröffentlichter Inhalt" (Published Content) auswählen, wird die Seite auf den zuletzt veröffentlichten Zustand zurückgesetzt. Wenn Sie auf "Zurücksetzen zu" klicken und "Systeminhalt" auswählen, wird die Seite auf den Stand der Informationen im Siemens PLM CEP-Basisarbeitsbereich zurückgesetzt.

Vorgehensweise zum Zurücksetzen auf den zuletzt veröffentlichten Zustand

Gehen Sie folgendermaßen vor, um den zuletzt veröffentlichten Zustand wiederherzustellen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Doppelklicken Sie auf eine Seite mit veröffentlichten Änderungen. Die Seite wird im Arbeitsbereich angezeigt.
4. Klicken Sie auf Zurücksetzen auf (Revert To) und wählen Sie Veröffentlichter Inhalt (Published Content) aus. Das Dialogfeld "Zurücksetzen auf letzte Veröffentlichung" (Revert to Last Published) wird angezeigt.
5. Klicken Sie auf OK. Das Dialogfeld "Zurücksetzen auf letzte Veröffentlichung" (Revert to Last Published) wird geschlossen und die Seite wird aktualisiert. Der Inhalt des zuletzt veröffentlichten Zustands wird angezeigt.

Vorgehensweise zum Zurücksetzen von Seiten auf den Systeminhalt

Gehen Sie folgendermaßen vor, um Seiten auf den Systeminhalt zurückzusetzen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Doppelklicken Sie auf eine Seite mit gespeicherten oder veröffentlichten Änderungen. Die Seite wird im Arbeitsbereich angezeigt.
4. Klicken Sie auf Zurücksetzen zu und wählen Sie Systeminhalt aus. Das Dialogfeld "Zurücksetzen auf Systeminhalt" (Revert to System Content) wird angezeigt.
5. Klicken Sie auf OK. Das Dialogfeld "Zurücksetzen auf Systeminhalt" wird geschlossen und die Seite wird aktualisiert. Der im Siemens PLM CEP-Basisarbeitsbereich gefundene Inhalt wird angezeigt.
