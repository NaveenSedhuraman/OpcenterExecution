Generieren einer Seite

Mit der Seitengenerierungsfunktion können Sie einen Service aus dem Feldverzeichnis auswählen und auf Basis dieses Service automatisch eine Seite generieren (mit den vordefinierten Feldern).

Beim Generieren einer Seite

Der Service muss allgemeine Felder, die bereits für den Service definiert sind, enthalten. Dies sind die Felder, die zur Seite hinzugefügt werden, wenn diese generiert wird. Allgemeine Felder sind durch Verknüpfungssymbol gekennzeichnet.

Eine dem ausgewählten Service entsprechende Aktion wird automatisch zu der Seite hinzugefügt. Daher könnten Sie diese Seite testen, sobald Sie sie gespeichert haben.

Vorgehensweise zum Generieren einer Seite

Gehen Sie folgendermaßen vor, um eine Seite zu generieren:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Toolbox anzeigen/ausblenden (Show/Hide Toolbox), wenn die Toolbox ausgeblendet ist. Die Toolbox wird angezeigt.
3. Klicken Sie auf die Registerkarte Feldverzeichnis (Fields Directory). Eine Liste von Services wird angezeigt.
4. Wählen Sie den Service für die Generierung der Seite aus.
5. Klicken Sie auf Seite generieren (Generate Page). Eine neue Seite wird erstellt.
6. Klicken Sie auf Eigenschaften anzeigen/ausblenden (Show/Hide Properties), wenn der Bereich "Eigenschaften" (Properties) ausgeblendet ist. Der Bereich "Eigenschaften" (Properties) wird angezeigt.
7. Geben Sie im Bereich "Eigenschaften" (Properties) im Feld Name einen Namen für die Seite ein.
8. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
