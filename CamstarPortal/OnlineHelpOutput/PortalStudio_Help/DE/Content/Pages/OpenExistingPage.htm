Öffnen einer vorhandenen Seite

Vorhandene Webseiten können Sie im Katalog der Benutzeroberflächenkomponenten über die Registerkarte "Components" (Komponenten) öffnen.

Beim Öffnen einer vorhandenen Seite

Webparts, Steuerelemente und Felder können auf eine Seite gezogen werden. Steuerelemente und Felder können Sie nicht direkt auf eine Seite ziehen. Steuerelemente und Felder müssen auf einem Webpart platziert werden.

Mit der Schaltfläche "Ausführen" (Run) können Sie eine Seite testen, nachdem Sie sie gespeichert haben. Weitere Informationen finden Sie unter "Testen einer Seite".

Umlaufbestandsnachrichten sind in den Portaleinstellungen standardmäßig auf "True" (Wahr) festgelegt. Sie können Umlaufbestandsnachrichten für eine Seite über das Feld "WIPMessages"Erforderlich unter "Aktionen" (Actions) deaktivieren, da Seiteneinstellungen die globalen Einstellungen überschreiben.

Beim Öffnen einer vorhandenen Seite wird eine Statusanzeige geöffnet, die anzeigt, dass die Seite geladen wird.

Vorgehensweise zum Öffnen einer vorhandenen Seite

Gehen Sie folgendermaßen vor, um eine vorhandene Seite zu öffnen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Doppelklicken Sie auf eine Seite. Die Seite wird im Arbeitsbereich angezeigt.
