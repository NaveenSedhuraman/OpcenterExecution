Löschen von Objekten und Seiten

Sie können Steuerelemente, Webparts und Seiten in Portal Studio dauerhaft löschen.

Beim Löschen von Objekten und Seiten

In Portal Studio gelöschte Objekte und Seiten werden dauerhaft aus der Anwendung entfernt und können nicht wiederhergestellt werden. Wenn Sie ein Objekt oder eine Seite in Portal Studio löschen, wird eine Bestätigungsnachricht angezeigt.

Vorgehensweise zum Löschen von Objekten

Gehen Sie folgendermaßen vor, um Objekte zu löschen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Doppelklicken Sie auf eine Seite (Page), ein Webpart oder ein Steuerelement (Control). Das Objekt wird im Arbeitsbereich angezeigt.
4. Wählen Sie ggf. das zu löschende Objekt aus.
5. Klicken Sie auf Löschen (Delete). Eine Löschbestätigungsnachricht wird angezeigt.
6. Klicken Sie auf OK. Das Bestätigungsnachricht wird geschlossen und das Objekt gelöscht.
7. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Löschen von Seiten

Gehen Sie folgendermaßen vor, um Seiten zu löschen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Doppelklicken Sie auf eine Seite. Die Seite wird im Arbeitsbereich angezeigt.
4. Klicken Sie auf Löschen (Delete). Das Dialogfeld "Webseite löschen" (Delete WebPage) wird angezeigt.
5. Klicken Sie auf OK. Das Dialogfeld "Webseite löschen" (Delete WebPage) wird geschlossen und die Seite gelöscht.
