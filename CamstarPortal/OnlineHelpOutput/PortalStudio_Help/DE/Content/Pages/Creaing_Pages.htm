Erstellen einer Seite

In Portal Studio können Sie Seiten erstellen, die im Portal verwendet werden.

Beim Erstellen einer Seite

Nachdem Sie eine neue Seite erstellt haben, können Sie Webparts, Steuerelemente und Felder auf die Seite ziehen. Sie können Steuerelemente und Felder nicht direkt auf eine Seite ziehen, sondern müssen diese auf einem Webpart platzieren.

Mit der Schaltfläche "Ausführen" (Run) können Sie eine Seite testen, nachdem Sie sie gespeichert haben. Weitere Informationen finden Sie unter "Testen einer Seite".

Umlaufbestandsnachrichten sind in den Portaleinstellungen standardmäßig auf "True" (Wahr) festgelegt. Da die globalen Einstellungen durch Seiteneinstellungen überschreiben werden, können Sie Umlaufbestandsnachrichten für eine Seite über das Feld "WIPMessages"Erforderlich im Bereich "Eigenschaften" (Properties) deaktivieren.

Felddefinitionen des Popups "Neue Seite erstellen" (Create New Page)

Diese Tabelle definiert die Felder im Popup "Neue Seite erstellen" (Create New Page).

Feld

Definition

Typ

Select Page Template

Gibt die Seitenvorlage an, die bei der Seitenerstellung zu verwenden ist. Mögliche Optionen sind "Leer" (Blank), Container und "Modellierung" (Modeling).

Erforderlich

Eproc

Schließt, wenn ausgewählt, grundlegende Komponenten automatischer Abläufe in die Seitenvorlage ein. Weitere Informationen finden Sie unter "Aktivieren von Transaktionen mit automatischem Ablauf".

Hinweis: Diese Option ist nur verfügbar, wenn "Container" als Seitenvorlage ausgewählt wird.

Optional

Page Name

Eindeutiger Name für die Seite.

Erforderlich

Primary Service Type

Weist der Seite einen primären Servicetyp zu.

Optional

Vorgehensweise zum Erstellen einer Seite

Gehen Sie folgendermaßen vor, um eine Seite zu erstellen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf das Pluszeichen auf der Schaltfläche Neu (New). Das Popup "Neue Seite erstellen" (Create New Page) wird angezeigt.
3. Wählen Sie eine Seitenvorlage aus.
4. Geben Sie im Feld Seitenname (Page Name) einen Namen für die Seite ein.
5. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
6. Klicken Sie auf OK. Das Popup "Neue Seite erstellen" (Create New Page) und die neue Seite werden im Arbeitsbereich angezeigt.
7. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
