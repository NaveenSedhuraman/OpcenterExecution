Erstellen einer neuen Seitengruppe

Sie können neue Seitengruppen mit dem Kontextmenü zum Zuweisen einer Seite zu einer Gruppe erstellen.

Beim Erstellen von neuen Seitengruppen

Wenn Sie mit der rechten Maustaste auf eine Seite klicken und "Zu neuer Gruppe hinzufügen" (Add to a New Group) auswählen, wird die Seite automatisch zu der neu erstellten Seitengruppe hinzugefügt.

Sie können den Namen einer vorhandenen Seitengruppe aus dem Kontextmenü, das Sie zum Erstellen einer Seitengruppe verwendet haben, bearbeiten.

Vorgehensweise zum Erstellen einer neuen Seitengruppe

Gehen Sie folgendermaßen vor, um eine neue Seitengruppe zu erstellen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Klicken Sie mit der rechten Maustaste auf eine Seite und wählen Sie Zu neuer Gruppe hinzufügen (Add to a New Group) aus. Das Dialogfeld "Gruppennamen eingeben" (Enter a Group Name) wird angezeigt.

Oder

Klicken Sie mit der rechten Maustaste auf eine vorhandene Seitengruppe und wählen Sie Neue Gruppe hinzufügen (Add a new group) aus. Das Dialogfeld "Gruppennamen eingeben" (Enter a Group Name) wird angezeigt.

4. Geben Sie einen Namen für die Gruppe ein.
5. Klicken Sie auf OK. Das Dialogfeld wird geschlossen und im Katalog der Benutzeroberflächenkomponenten (UI Components) wird eine neue Gruppe angezeigt.

Vorgehensweise zum Bearbeiten einer Seitengruppe

Gehen Sie folgendermaßen vor, um eine Seitengruppe zu bearbeiten:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Klicken Sie mit der rechten Maustaste auf eine Seitengruppe und wählen Sie Bearbeiten (Edit) aus. Das Dialogfeld "Gruppennamen eingeben" (Enter a Group Name) wird angezeigt.
4. Geben Sie einen neuen Namen für die Gruppe ein.
5. Klicken Sie auf OK. Das Dialogfeld wird geschlossen und die Gruppe wird im Katalog der Benutzeroberflächenkomponenten (UI Components) mit dem neuen Namen angezeigt.
