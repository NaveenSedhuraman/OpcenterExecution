Hinzufügen von Zeilen zu einer Zielmatrix

In Portal Studio können Sie über das Fenster "Zielmatrixdaten" (Target Matrix Data) Zeilen zu einer neuen oder vorhandenen Zielmatrix hinzufügen.

Beim Hinzufügen von Zeilen zu einer Zielmatrix

Sie können Daten aus einer Excel-Tabelle importieren, wenn Sie das Fenster "Zielmatrixdaten" (Target Matrix Data) verwenden.

Sie können nur dann Zeilen hinzufügen, wenn Sie Spalten angegeben oder Daten importiert haben.

Vorgehensweise zum Hinzufügen von Zeilen zu einer Zielmatrix

Gehen Sie folgendermaßen vor, um Zeilen zu einer Zielmatrix hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Führen Sie die Prozedur unter "Vorgehensweise zum Erstellen einer Zielmatrix" aus.

Oder

Öffnen Sie eine vorhandene Zielmatrix.

3. Klicken Sie auf "Mehr" (More) für Zeilen (Rows). Das Popup "Zielmatrix – Daten" (TargetMatrix - Data) wird angezeigt.
4. Klicken Sie auf Zeile hinzufügen (Add Row). Eine neue Zeile wird zu den angegebenen Spalten oder importierten Daten hinzugefügt.
5. Wiederholen Sie Schritt 4 für jede weitere Zeile, die Sie hinzufügen möchten.
6. Klicken Sie auf OK. Das Popup "Zielmatrix – Daten" (TargetMatrix – Data) wird geschlossen.
7. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
