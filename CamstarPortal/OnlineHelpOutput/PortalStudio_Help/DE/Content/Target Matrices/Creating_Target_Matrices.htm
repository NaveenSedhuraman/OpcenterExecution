Erstellen einer Zielmatrix

Objekte können in Portal Studio zu dieser Liste hinzugefügt werden.

Beim Erstellen einer Zielmatrix

Bietet die Möglichkeit, dass eine Instanz ein Ziel oder eine Benchmark mit einer Standardmethode oder mit einer optionalen Zielmatrix (Target Matrix) auflöst; die Auflösung des Ziels und der Benchmark erfolgt anhand der besten Übereinstimmung basieren auf den aufgelösten Objektfeldern (Operation, Ressource usw.).

In der Matrix können Zeilen mit Werten für alle Objektparameter und auch für das Ziel und die Benchmark hinzugefügt, bearbeitet, gelöscht und angezeigt werden.

Wenn Sie eine neue Zielmatrix erstellen, werden im Arbeitsbereich in einem Popup Zielmatrizen als Gruppe von Feldern angezeigt.

Felddefinitionen des Popups "Zielmatrix" (Target Matrix)

Diese Tabelle definiert die Felder im Popup "Zielmatrix" (Target Matrix).

Feld

Definition

Typ

Columns

Wenn Sie auf die Schaltfläche "Mehr" (More) klicken, wird das Popup "Spalten" (Columns) angezeigt. Hier können Sie Spalten für die Zielmatrix definieren. Weitere Informationen finden Sie unter "Hinzufügen von Spalten zu einer Zielmatrix".

Optional

FileName

Zeigt einen der Zielmatrix zugewiesenen Dateinamen an.

Nur Anzeige

Name

Eindeutiger Namen für die Zielmatrix.

Erforderlich

Rows

Wenn Sie auf die Schaltfläche "Mehr" (More) klicken, wird das Popup "Zeilen" (Rows) angezeigt. Hier können Sie Zeilen zur Zielmatrix hinzufügen. Weitere Informationen finden Sie unter "Hinzufügen von Zeilen zu einer Zielmatrix".

Optional

Vorgehensweise zum Erstellen einer Zielmatrix

Gehen Sie folgendermaßen vor, um eine Zielmatrix zu erstellen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Neu (New) und wählen Sie Zielmatrix (Target Matrix) aus. Im Arbeitsbereich wird eine leere Zielmatrix angezeigt.
3. Geben Sie im Feld Name einen Name für die Zielmatrix ein oder wählen Sie einen Name aus.
4. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
5. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
