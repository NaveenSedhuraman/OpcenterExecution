Hinzufügen von Spalten zu einer Zielmatrix

Portal Studio ermöglicht es Ihnen, einer neuen oder vorhandenen Zielmatrix (Target Matrix) Spalten (Columns) hinzuzufügen.

Beim Hinzufügen von Spalten zu einer Zielmatrix

Alle Zielmatrizen haben die gleichen Spalten. Wenn Sie also eine Spalte zu einer Zielmatrix hinzufügen, wird diese Spalte allen Zielmatrizen hinzugefügt.

Vorgehensweise zum Hinzufügen von Spalten zu einer Zielmatrix

Gehen Sie folgendermaßen vor, um Spalten zu einer Zielmatrix hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Führen Sie die Prozedur unter "Vorgehensweise zum Erstellen einer Zielmatrix" aus.

Oder

Öffnen Sie eine vorhandene Zielmatrix.

3. Klicken Sie auf "Mehr" (More) für Spalten (Columns). Das Popup "Zielmatrix Nr. – Spalten" (TargetMatrix# - Columns) wird angezeigt.
4. Klicken Sie auf Hinzufügen (Add). In der Liste wird eine neue Spalte angezeigt.
5. Geben Sie im Feld Name einen Namen für die Spalte ein.
6. Wiederholen Sie die Schritte 4 bis 5, um weitere Spalten hinzuzufügen.
7. Klicken Sie auf OK. Das Popup "Zielmatrix Nr. – Spalten" (TargetMatrix# - Columns) wird geschlossen.
8. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
