Konfigurieren von Lokalisierungseinstellungen

Sie können die Einstellungen für Datum, Uhrzeit und Dezimaltrennzeichen für Portalseiten in Portal Studio konfigurieren.

Beim Konfigurieren von Lokalisierungseinstellungen

Die Einstellungen für Datum, Uhrzeit und Dezimaltrennzeichen können Sie in den bevorzugten Spracheinstellungen in Ihrem Internetbrowser oder über die Einstellungen von Portal Studio konfigurieren.

Wenn Sie eine bevorzugte Sprache in Ihrem Internetbrowser festlegen, werden die Portalseiten so formatiert, dass Datum, Uhrzeit und Dezimaltrennzeichen im Format der bevorzugten Sprache angezeigt werden. Diese Einstellung betrifft nur Ihre Instanz des Portals. Weitere Informationen dazu, wie Sie eine bevorzugte Sprache einstellen, finden Sie in der Hilfe zu Ihrem Internetbrowser.

Die aktuellen Kultureinstellungen in Portal Studio steuern die Einstellungen für Datum, Uhrzeit und Dezimaltrennzeichen auf Portalseiten. Die aktuellen Kultureinstellungen sind anpassbar und überschreiben Ihre Browsereinstellungen.

Important: Wenn Sie die aktuellen Kultureinstellungen in Portal Studio ändern, bewirkt dies eine globale Änderung. Alle einzelnen browserbasierten Einstellungen für sämtliche Portalbenutzer werden überschrieben.

Die akzeptierten Werte für die aktuellen Kultureinstellungen basieren auf den Werten von Microsoft für benutzerdefinierte Datums- und Uhrzeitformat-Zeichenketten.

Vorgehensweise zum Konfigurieren von Lokalisierungseinstellungen

Gehen Sie folgendermaßen vor, um Lokalisierungseinstellungen zu konfigurieren:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf die Schaltfläche Portaleinstellungen (Portal Settings). Das Popup "Portaleinstellungen" (Portal Settings) wird angezeigt.
3. Klicken Sie in der Liste der Einstellungen auf Aktuelle Kultureinstellungen (CurrentCultureSettings). Die Eigenschaften für "Aktuelle Kultureinstellungen" (CurrentCultureSettings) werden angezeigt.
4. Erweitern Sie Datum- und Uhrzeitformat (DateTimeFormat).
5. Geben Sie in den Feldern Langes Uhrzeitmuster (LongTimePattern) und Kurzes Datumsmuster (ShortDatePattern) Werte ein.
6. Erweitern Sie Zahlenformat (NumberFormat).
7. Geben Sie im Feld Dezimaltrennzeichen (NumberDecimalSeparator) einen Wert ein.
8. Klicken Sie auf OK. Das Popup "Portaleinstellungen" (Portal Settings) wird geschlossen.
