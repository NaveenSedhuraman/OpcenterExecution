Aktivieren von akustischen Signalen für Statusnachrichten

In Portal Studio können Sie akustische Signale für auf Portalseiten angezeigte Statusnachrichten aktivieren. Statusnachrichten werden im Anschluss an die Ausführung einer Funktion in Portal Studio ausgegeben. Drei verschiedene Typen werden unterschieden:

• Erfolg (Success)
• Fehler (Error)
• Warnung (Warning)
Note: Signale für Statusnachrichten sind im Portal standardmäßig deaktiviert.

Beim Aktivieren von akustischen Signalen für Statusnachrichten

Wenn Sie akustische Signale für Statusnachrichten aktivieren, werden damit akustische Signale aktiviert, die für den jeweiligen Typ der Statusnachricht wiedergegeben werden. Akustische Signale für Statusnachrichten können jedoch nach Typ deaktiviert werden. Z. B. können Sie akustische Signale für Erfolgs- und Warnmeldungen deaktivieren, aber dennoch ein akustisches Signal für Fehlerstatusnachrichten wiedergeben lassen. Akustische Signale für Statusnachrichten können mit vom Benutzer hochgeladenen WAV- oder MP3-Dateien weiter angepasst werden. Hochgeladene Dateien werden beim Hochladen automatisch dem Pfad für akustische Signale zugewiesen.

Note: Einige Versionen von Internet Explorer unterstützen keine WAV-Dateien. Siemens empfiehlt Ihnen die Verwendung des Dateiformats MP3 für akustische Signale für Statusnachrichten.

Vorgehensweise zum Aktivieren von akustischen Signalen für Statusnachrichten

Gehen Sie folgendermaßen vor, um akustische Signale für Statusnachrichten zu aktivieren:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf die Schaltfläche Portaleinstellungen (Portal Settings).Das Popup "Portaleinstellungen" (Portal Settings) wird angezeigt.
3. Wählen Sie Statusnachrichteneinstellungen (StatusMessageSettings) aus der Liste der Einstellungen aus. Die Eigenschaften für Statusnachrichteneinstellungen (StatusMessageSettings) werden angezeigt.
4. Wählen Sie Wahr (True) für Akustisches Signal ausgeben (PlaySound) aus.
5. Klicken Sie auf OK. Das Popup "Portaleinstellungen" (Portal Settings) wird geschlossen.

Vorgehensweise zum Deaktivieren von akustischen Signalen für Statusnachrichten nach Typ

Gehen Sie folgendermaßen vor, um akustische Signale für Statusnachrichten nach Typ zu deaktivieren:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf die Schaltfläche Portaleinstellungen (Portal Settings).Das Popup "Portaleinstellungen" (Portal Settings) wird angezeigt.
3. Wählen Sie Statusnachrichteneinstellungen (StatusMessageSettings) aus der Liste der Einstellungen aus. Die Eigenschaften für Statusnachrichteneinstellungen (StatusMessageSettings) werden angezeigt.
4. Löschen Sie den Dateinamen für den Pfad für akustische Signale.
5. Klicken Sie auf OK. Das Popup "Portaleinstellungen" (Portal Settings) wird geschlossen.

Vorgehensweise zum Hochladen von akustischen Signalen für Statusnachrichten

Gehen Sie folgendermaßen vor, um akustische Signale für Statusnachrichten hochzuladen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf die Schaltfläche Portaleinstellungen (Portal Settings). Das Popup "Portaleinstellungen" (Portal Settings) wird angezeigt.
3. Wählen Sie Statusnachrichteneinstellungen (StatusMessageSettings) aus der Liste der Einstellungen aus. Die Eigenschaften für Statusnachrichteneinstellungen (StatusMessageSettings) werden angezeigt.
4. Klicken Sie auf die Schaltfläche . Das Popup "Öffnen" (Open) wird angezeigt.
5. Wählen Sie eine Audiodatei aus und klicken Sie auf Öffnen (Open). Die Audiodatei wird für den Statusnachrichtentyp hochgeladen.
6. Klicken Sie auf OK. Das Popup "Portaleinstellungen" (Portal Settings) wird geschlossen.

Vorgehensweise zum Festlegen eines akustischen Signals für Statusnachrichten

Gehen Sie folgendermaßen vor, um ein akustisches Signal für Statusnachrichten festzulegen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf die Schaltfläche Portaleinstellungen (Portal Settings).Das Popup "Portaleinstellungen" (Portal Settings) wird angezeigt.
3. Wählen Sie Statusnachrichteneinstellungen (StatusMessageSettings) aus der Liste der Einstellungen aus. Die Eigenschaften für Statusnachrichteneinstellungen (StatusMessageSettings) werden angezeigt.
4. Wählen Sie eine Audiopfad-Datei aus.
5. Klicken Sie auf OK. Das Popup "Portaleinstellungen" (Portal Settings) wird geschlossen.
