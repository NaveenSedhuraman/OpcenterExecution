Aktivieren von Haftfeldern

Eingegebene oder ausgewählte Daten in Haftfeldern werden nicht gelöscht, nachdem Sie die betreffende Seite übergeben haben.

Beim Aktivieren von Haftfeldern

Haftfelder sind durch ein Pinnnadelsymbol gekennzeichnet.

In Portal Studio können Felder optional als Haftfelder definiert werden. Anschließend müssen Sie die Haftfelder durch Klicken auf das Pinnnadelsymbol aktivieren. Haftfelder sind standardmäßig inaktiv.

Symbol

Status

Inaktiv. In diesem Feld werden Daten nicht beibehalten.

Aktiv. In diesem Feld werden Daten beibehalten.

Die Daten in Haftfeldern werden gelöscht, wenn die Seitenaktion zum Einreichen auf eine andere Seite umleitet oder wenn das Neuladen von Werten eingestellt ist. Weitere Informationen zum Konfigurieren von Aktionen finden Sie unter "Konfigurieren von Seitenaktionen".

Vorgehensweise zum Aktivieren von Haftfeldern

Gehen Sie folgendermaßen vor, um Haftfelder zu aktivieren:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Doppelklicken Sie auf eine Seite. Die Seite wird im Arbeitsbereich angezeigt.
4. Wählen Sie ein Feld auf der Seite aus. Die Eigenschaften für das Feld werden im Bereich "Eigenschaften" (Properties) angezeigt.
5. Wahlen Sie im Bereich "Eigenschaften" (Properties) für Haftfähig (StickyCapable) den Eintrag Wahr (True) aus.
6. Wiederholen Sie ggf. die Schritte 4 bis 5, um Haftfelder für weitere Felder zu aktivieren.
7. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
