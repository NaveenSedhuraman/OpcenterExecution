Erweitern der Anweisungen für automatische Abläufe

In Portal Studio können Sie konfigurieren, ob Anweisungen in automatischen Abläufen standardmäßig erweitert oder reduziert sind. Anweisungen für automatische Abläufe sind standardmäßig reduziert.

Beim Erweitern der Anweisungen für automatische Abläufe

Zum Konfigurieren der Anweisungen für automatische Abläufe müssen Sie die virtuelle UI-Seite "Automatischer Ablauf" ändern. Virtuelle Seiten in Portal Studio enthalten alle Teile einer Seite. Der Teil, der die Anweisungen für automatische Abläufe konfiguriert, ist das Feld für Aufgabenanweisungen (TaskInstructionsField), das nicht beschriftet ist. Dieses Feld wird als gelbes Rechteck ohne Rahmen angezeigt.

Vorgehensweise zum standardmäßigen Erweitern der Anweisungen für automatische Abläufe

Gehen Sie folgendermaßen vor, um die Anweisungen für automatische Abläufe standardmäßig zu erweitern:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Erweitern Sie die Seitengruppierung Alle und doppelklicken Sie auf EprocedureVPR2. Die Seite "EprocedureVPR2" wird im Arbeitsbereich angezeigt.
Note: Sie müssen "EProcedureVP" aktualisieren, wenn Sie die ältere Version der Seite verwenden. Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics Shop Floor-Benutzerhandbuch oder im Opcenter Execution Core Shop Floor-Benutzerhandbuch.
4. Wählen Sie das Feld für Aufgabenanweisungen (TaskInstructionsField) aus. Die Einstellungen für das Feld "Aufgabenanweisungen" (TaskInstructions) werden im Bereich "Eigenschaften" (Properties) angezeigt.
5. Wählen Sie Erweitert (Expanded) als Standardzustand (DefaultState) aus.
6. Wählen Sie Wahr (True) für Vollständig erweitern (ExpandFull) aus.
7. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
