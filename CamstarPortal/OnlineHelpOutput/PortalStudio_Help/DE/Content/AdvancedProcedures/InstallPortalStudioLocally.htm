Lokale Installation von Portal Studio

Sie können Portal Studio in Ihrer lokalen Umgebung installieren. So können Sie Portal Studio direkt öffnen, anstatt sich bei jedem Zugriff auf Portal Studio erst beim Portal anzumelden.

Bei der lokalen Installation von Portal Studio

Sie müssen Portal Studio beim ersten Mal über das Portal öffnen. Dort können Sie auf das Kontextmenü für die lokale Installation zugreifen.

Sie haben die Möglichkeit, Verknüpfungen im Startmenü und/oder auf dem Desktop zu erstellen.

Wenn Sie nach der Installation auf die Verknüpfung klicken, wird eine separate Anmeldeseite geöffnet. Über diese Seite können Sie Portal Studio direkt öffnen, ohne zuvor Portal öffnen zu müssen.

Note: Möglicherweise müssen Sie Ihren Browsercache löschen, damit Sie sich über die Verknüpfung anmelden können.

Vorgehensweise zur lokalen Installation von Portal Studio

Gehen Sie folgendermaßen vor, um Portal Studio lokal zu installieren:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie mit der rechten Maustaste in den Arbeitsbereich. Ein Kontextmenü wird angezeigt.
3. Wählen Sie Die Anwendung Camstar Portal Studio auf diesem Computer installieren (Install Camstar Portal Studio application on this computer) aus. Das Dialogfeld "Anwendung installieren" (Install Application) wird angezeigt.
4. Aktivieren Sie das entsprechende Kontrollkästchen, um Portal Studio im Startmenü und/oder auf dem Desktop zu installieren, und klicken Sie auf OK. Die Seite "Anmelden" (Sign In) wird angezeigt.
5. Geben Sie das Kennwort (Password) ein und klicken Sie auf Anmelden (Log On). Eine neue Portal Studio-Seite wird angezeigt.
