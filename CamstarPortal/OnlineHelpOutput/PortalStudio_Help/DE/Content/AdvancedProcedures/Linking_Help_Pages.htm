Hinzufügen von Hilfeinformationen zu einer Portalseite

Sie können Hilfeinformationen zu einer Portalseite hinzufügen. Ändern Sie dazu in Portal Studio die Eigenschaft "URL der Hilfedatei" (HelpFileURL).

Beim Hinzufügen von Hilfeinformationen zu einer Portalseite

Sie müssen die URL der Hilfedatei im Format "FilePath#FileName" eingeben, wobei "FilePath" der Dateipfad und FileName der Name der Hilfedatei ist.

Important: Siemens empfiehlt, die Eigenschaft "HelpFileURL" für standardmäßige Portalseiten nicht zu ändern. Wenn Sie diese Eigenschaft ändern, könnte dies dazu führen, dass vorhandene Hilfedateien und unterstützte, übersetzte Hilfedateien von Portalseiten entfernt werden.

Unter Umständen müssen Sie nur einen Teil eines Dateipfads eingeben. Dies hängt von Ihren Portal Studio-Einstellungen ab. Weitere Informationen zu Einstellungen für Portal Studio finden Sie unter "Konfigurieren von Portaleinstellungen".

Vorgehensweise zum Hinzufügen von Hilfeinformationen zu einer Portalseite

Gehen Sie folgendermaßen vor, um Hilfeinformationen zu einer Portalseite hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Doppelklicken Sie auf eine Seite. Die Seite wird im Arbeitsbereich angezeigt.
4. Klicken Sie auf Eigenschaften anzeigen/ausblenden (Show/Hide Properties), wenn der Bereich "Eigenschaften" (Properties) ausgeblendet ist. Der Bereich "Eigenschaften" (Properties) wird angezeigt.
5. Geben Sie im Bereich "Eigenschaften" (Properties) im Feld URL der Hilfedatei (HelpFileURL) die URL der Hilfedatei ein.
6. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
