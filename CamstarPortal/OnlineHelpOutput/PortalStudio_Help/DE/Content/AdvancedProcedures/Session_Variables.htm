Definieren von Sitzungsvariablen

Sie können Sitzungsvariablen (Session Variables) für Seiten oder Webparts in Portal Studio erstellen. Mit Sitzungsvariablen können Sie Daten sammeln und die gesammelten Daten für eine andere Komponente zur Verfügung stellen.

Beim Definieren von Sitzungsvariablen

Sitzungsvariablen werden im Popup "Sitzungsvariablen" (SessionVariables) deklariert. Dieses wird im Bereich "Eigenschaften" (Properties) aufgerufen, wenn im Arbeitsbereich eine Seite, ein Webpart oder ein Popup ausgewählt wird.

Sitzungsvariablen sind innerhalb der Seite, auf der sie definiert wurden, global verfügbar. In einem Webpart definierte Sitzungsvariablen werden innerhalb der Seite gespeichert, die das betreffende Webpart enthält.

Sitzungsvariablen speichern Daten, bis Sie die Seite bei der Navigation verlassen. Wenn Sie die Seite neu laden, werden die Sitzungsvariablen auf den im Feld "InitialValue" angegebenen Wert zurückgesetzt. Sitzungsvariablen haben standardmäßig einen Anfangswert von "null".

Sitzungsvariablen werden beim Definieren von bedingten Ereignishandlern festgelegt. Weitere Informationen zum Definieren von bedingten Ereignishandlern finden Sie unter "Definieren von Ereignishandlern".

Felddefinitionen des Popups "Sitzungsvariablen" (SessionVariables)

Diese Tabelle definiert die Felder im Popup "Sitzungsvariablen" (SessionVariables).

Feld

Definition

Typ

Allgemein (General)

ID

Eindeutiger Namen für die Sitzungsvariable.

Erforderlich

InitialValue

Bestimmt den Startwert der Sitzungsvariablen, wenn die Seite geladen wird. Die Sitzungsvariable wird auf diesen Wert zurückgesetzt, wenn die Seite neu geladen wird. Sitzungsvariablen haben standardmäßig einen Anfangswert von "null".

Optional

IsArray

Gibt an, ob die Sitzungsvariable zu einem Array gehört.

Optional

Type

Bestimmt den erwarteten Objekttyp, z. B. "RefBenanntesObjekt" (NamedObjectRef) oder "System.Zeichenkette" (System.String). Sitzungsvariablen haben standardmäßig den Typ "System.Object".

Optional

Vorgehensweise zum Definieren einer Sitzungsvariablen

Gehen Sie folgendermaßen vor, um Sitzungsvariablen zu definieren:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Doppelklicken Sie auf eine Seite. Die Seite wird im Arbeitsbereich angezeigt.
4. Klicken Sie auf Eigenschaften anzeigen/ausblenden (Show/Hide Properties), wenn der Bereich "Eigenschaften" (Properties) ausgeblendet ist. Der Bereich "Eigenschaften" (Properties) wird angezeigt.
5. Wählen Sie im Arbeitsbereich eine Seite, ein Webpart oder ein Popup aus. Die Eigenschaften für das ausgewählte Objekt werden im Bereich "Eigenschaften" (Properties) angezeigt.
6. Klicken Sie im Bereich "Eigenschaften" (Properties) auf Mehr (More) für Sitzungsvariablen (SessionVariables). Das Popup "Sitzungsvariablen" (SessionVariables) wird angezeigt.
7. Klicken Sie auf Hinzufügen (Add). Eine neue Sitzungsvariable wird zur Liste hinzugefügt; die allgemeinen Eigenschaften der Liste werden im rechten Bereich angezeigt.
8. Geben Sie im Feld ID einen Namen für die Sitzungsvariable ein.
9. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
10. Wiederholen Sie die Schritte 7-9, um zusätzliche Sitzungsvariablen hinzuzufügen.
11. Klicken Sie auf OK. Das Popup "Sitzungsvariablen" (SessionVariables) wird geschlossen.
12. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
