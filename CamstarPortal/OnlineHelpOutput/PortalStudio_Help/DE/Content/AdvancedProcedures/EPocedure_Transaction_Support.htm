Aktivieren von Transaktionen mit automatischem Ablauf

Sie können Transaktionen mit automatischem Ablauf (EProcedure Transactions) auf einer Seite konfigurieren. Ändern Sie dazu die Eigenschaft "Automatischer Ablauf aktiviert" (EprocEnabled) der Seite in Portal Studio.

Beim Aktivieren von Transaktionen mit automatischem Ablauf

Transaktionen mit automatischem Ablauf sind auf bestimmten Transaktionsseiten standardmäßig aktiviert. Die Auswahlliste "Transaktionsseite" (Transaction Page) enthält eine Liste der Seiten, für die der automatische Ablauf aktuell aktiviert ist.

Bei neu erstellten Seiten sind Transaktionen mit automatischem Ablauf standardmäßig nicht aktiviert.

Sie können überprüfen, ob eine Seite für die Unterstützung automatischer Abläufe aktiviert wurde. Zeigen Sie dazu im Bereich "Eigenschaften" (Properties) der Seite das EProcEnabled-Flag an.

Diese Liste enthält die Seiten, die standardmäßig zur Ausführung einer Transaktion mit automatischem Ablauf in der Lage sind. Weitere Seiten, die nicht in dieser Liste aufgeführt sind, erfordern unter Umständen zusätzliche Programmierschritte entsprechend den benutzerdefinierten Funktionen.

Vorgehensweise zum Aktivieren von Transaktionen mit automatischem Ablauf

Gehen Sie folgendermaßen vor, um Transaktionen mit automatischem Ablauf zu aktivieren:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Doppelklicken Sie auf eine Seite, die automatische Abläufe (EProcedure) unterstützt. Die Seite wird im Arbeitsbereich angezeigt.
4. Klicken Sie auf Eigenschaften anzeigen/ausblenden (Show/Hide Properties), wenn der Bereich "Eigenschaften" (Properties) ausgeblendet ist. Der Bereich "Eigenschaften" (Properties) wird angezeigt.
5. Wählen Sie im Bereich "Eigenschaften" (Properties) für EProcEnabled den Wert Wahr (True) aus.
6. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
