Konfigurieren von Seitenaktionen

Sie können Seitenaktionen (Page Actions) so konfigurieren, dass sie auf den zugehörigen Quellseiten entweder als Verknüpfungen oder als Schaltflächen angezeigt werden. Sie können Seitenaktionen mit folgenden Elementen konfigurieren:

• Mit dem Webpart "Aktionssteuerung" (ActionsControl), um eine Seitenaktion direkt als Verknüpfung oder als Schaltfläche auf einer Seite zu platzieren.
• Mit dem Popup "Seitenaktionseigenschaften", um eine Seitenaktion als Schaltfläche darzustellen, die unten links oder unten rechts auf einer Seite angezeigt wird.

Beim Konfigurieren von Seitenaktionen

Damit Aktionen auf einer Portalseite angezeigt werden, müssen Sie Seitenaktionen aktivieren und Quellseiten zu Seitenaktionen hinzufügen. Weitere Informationen zum Aktivieren von dynamischen Aktionen finden Sie unter "Aktivieren von Seitenaktionen". Weitere Informationen zum Hinzufügen von Seiten zu Quellseitenlisten finden Sie unter "Hinzufügen von Quellseiten zu Seitenaktionen".

Durch Platzieren des Webparts "Aktionsteuerung" legen Sie fest, wo auf einer Portalseite Seitenaktionen angezeigt werden. Das Webpart "Aktionssteuerung" (ActionsControl) wird als weißes Rechteck mit grauem Rand angezeigt und ist nicht beschriftet. Weitere Informationen zum Hinzufügen von Webparts zu Seiten finden Sie unter "Hinzufügen eines Webparts zu einer Seite".

Vorgehensweise zum Konfigurieren von Seitenaktionen als Schaltflächen

Gehen Sie folgendermaßen vor, um Seitenaktionen zu konfigurieren:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Wählen Sie im Katalog der Benutzeroberflächenkomponenten (UI Components) die Registerkarte Modellierung (Modeling) aus.
4. Doppelklicken Sie auf eine Seitenaktion. Im Arbeitsbereich wird das Popup "Seitenaktionseigenschaften" angezeigt.
5. Wählen Sie Erweitert (Enhanced) als Bearbeitungsmodus (Edit Mode) aus. Im Popup "Seitenaktionseigenschaften" werden weitere Felder angezeigt.
6. Erweitern Sie Benutzeroberfläche: Aktion (UIAction).
7. Wählen Sie im Feld Standort (Location) den Eintrag Schaltfläche aus.
8. Wählen Sie im Feld Schaltflächenposition (ButtonPosition) den Bereich aus, in dem die Schaltfläche angezeigt werden soll.
9. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Konfigurieren von Seitenaktionen mit dem Webpart "Aktionssteuerung" (ActionsControl)

Gehen Sie folgendermaßen vor, um Seitenaktionen mit dem Webpart "Aktionssteuerung" (ActionsControl) zu konfigurieren:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Wählen Sie im Katalog der Benutzeroberflächenkomponenten (UI Components) die Registerkarte Modellierung (Modeling) aus.
4. Doppelklicken Sie auf eine Seitenaktion. Das Popup "Seitenaktionseigenschaften" erscheint im Arbeitsbereich.
5. Wählen Sie Erweitert (Enhanced) als Bearbeitungsmodus (Edit Mode) aus. Im Popup "Seitenaktionseigenschaften" werden weitere Felder angezeigt.
6. Erweitern Sie Benutzeroberfläche: Aktion (UIAction).
7. Wählen Sie im Feld Standort (Location) den Eintrag Bedientafel (ActionPanel) aus.
8. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
9. Wählen Sie im Katalog der Benutzeroberflächenkomponenten (UI Components) die Registerkarte Komponenten (Components) aus.
10. Doppelklicken Sie auf eine Seite. Die Seite wird im Arbeitsbereich angezeigt.
11. Erweitern Sie Webparts.
12. Klicken Sie auf das Webpart Aktionssteuerung (ActionsControl) und ziehen Sie es auf eine Zelle auf der Seite.
13. Klicken Sie auf Eigenschaften anzeigen/ausblenden (Show/Hide Properties), wenn der Bereich "Eigenschaften" (Properties) ausgeblendet ist. Der Bereich "Eigenschaften" (Properties) wird angezeigt.
14. Soll die Seitenaktion als Verknüpfung oder als Schaltfläche angezeigt werden?

Seitenaktion anzeigen als . . .

Dann . . .

Verknüpfung.
a. Geben Sie im Bereich "Eigenschaften" (Properties) im Feld Name einen eindeutigen Namen für das Webpart "Aktionssteuerung" (ActionsControl) ein.
b. Gehen Sie zu Schritt 15.

Schaltfläche.
a. Geben Sie im Bereich "Eigenschaften" (Properties) im Feld Name einen eindeutigen Namen für das Webpart "Aktionssteuerung" (ActionsControl) ein.
b. Wählen Sie im Bereich "Eigenschaften" (Properties) im Feld Aktionsstil (ActionsStyle) den Eintrag Schaltfläche (Button) aus.
c. Gehen Sie zu Schritt 15.
15. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
