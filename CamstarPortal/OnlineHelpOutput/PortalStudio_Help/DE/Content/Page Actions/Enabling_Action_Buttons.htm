Aktivieren von Seitenaktionen

Seitenaktionen (Page Actions) sind Verknüpfungen und Schaltflächen, die eine Aktion auslösen, wenn sie angeklickt werden. Sie können Seitenaktionen auf einer beliebigen Seite über den Bereich "Eigenschaften" (Properties) aktivieren.

Beim Aktivieren von Seitenaktionen

Sie müssen Seitenaktionen erst aktivieren, damit diese auf einer Seite angezeigt werden.

Seitenaktionen sind entweder deaktiviert oder ausgeblendet, wenn die Seitenaktionsregeln nicht erfüllt sind. Deaktivierte Seitenaktionen werden angezeigt, können aber nicht angeklickt werden. Ausgeblendete Aktionen werden auf der Seite nicht angezeigt. Seitenaktionen sind standardmäßig ausgeblendet. Sie können die Eigenschaft "AnzeigeoptionDeaktiviert" (DisabledDisplayOption) im Bereich "Eigenschaften" (Properties) so ändern, dass Aktionen mit nicht erfüllten Regeln deaktiviert werden.

Vorgehensweise zum Aktivieren von Seitenaktionen

Gehen Sie folgendermaßen vor, um Seitenaktionen zu aktivieren:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Doppelklicken Sie auf eine Seite. Die Seite wird im Arbeitsbereich angezeigt.
4. Klicken Sie auf Eigenschaften anzeigen/ausblenden (Show/Hide Properties), wenn der Bereich "Eigenschaften" (Properties) ausgeblendet ist. Der Bereich "Eigenschaften" (Properties) wird angezeigt.
5. Erweitern Sie im Bereich "Eigenschaften" (Properties) Dynamische Aktionen (Dynamic Actions).
6. Wählen Sie Wahr (True) im Feld Aktionen (Action)Erforderlich aus.
7. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Festlegen von Seitenaktionen als deaktiviert

Gehen Sie folgendermaßen vor, um Seitenaktionen als deaktiviert festzulegen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Doppelklicken Sie auf eine Seite. Die Seite wird im Arbeitsbereich angezeigt.
4. Klicken Sie auf Eigenschaften anzeigen/ausblenden (Show/Hide Properties), wenn der Bereich "Eigenschaften" (Properties) ausgeblendet ist. Der Bereich "Eigenschaften" (Properties) wird angezeigt.
5. Erweitern Sie im Bereich "Eigenschaften" (Properties) Dynamische Aktionen (Dynamic Actions).
6. Wählen Sie im Feld DeaktiviertAnzeigeoptionen (DisabledDisplayOptions) die Option Deaktivieren (Disable) aus.
7. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
