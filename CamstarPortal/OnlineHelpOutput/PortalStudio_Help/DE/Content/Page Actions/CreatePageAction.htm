Seitenaktionen

Seitenaktionen (Page Actions), oder dynamische Aktionen, sind Aktionen, die auf einer Portalseite ausgeführt werden können. "Container schließen" (Container Close) z. B. ist eine Aktion, mit der Sie einen auf der Seite ausgewählten Container schließen können. Seitenaktionen können als Verknüpfungen oder als Schaltflächen angezeigt und so angepasst werden, dass sie unter bestimmten Bedingungen angezeigt werden.

Die folgenden Themen enthalten Informationen zu dynamischen Aktionen:

• Aktivieren von Seitenaktionen
• Erstellen von Seitenaktionen
• Öffnen vorhandener Seitenaktionen
• Konfigurieren von Seitenaktionen
• Suchen nach Seitenaktionsbeschriftungen
• Hinzufügen von Seitenaktionsregeln zu Seitenaktionen
• Hinzufügen von Quellseiten zu Seitenaktionen
