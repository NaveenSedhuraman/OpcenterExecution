Hinzufügen von Quellseiten zu Seitenaktionen

Quellseiten sind die Portalseiten, auf denen die Seitenaktion angezeigt wird. Aktionen werden nicht auf einer Seite angezeigt, die nicht in der Liste der Quellseiten aufgeführt ist. Dies gilt unabhängig davon, ob die Bedingungen für die Seitenaktionsregeln erfüllt sind. Aktionen können mehrere als Quellseiten aufgeführte Seiten aufweisen.

Beim Hinzufügen von Quellseiten zu Seitenaktionen

Quellseiten können auch aus einer Seitenaktion entfernt werden. Eine Seitenaktion wird nicht mehr auf einer Seite angezeigt, wenn diese Seite aus der Liste der Quellseiten gelöscht wurde.

Felddefinitionen des Popups "Quellseiten" (SourcePages)

Diese Tabelle definiert die Felder im Popup "Quellseiten" (SourcePages).

Feld

Definition

Typ

Allgemein (General)

Sequence

Bestimmt die Reihenfolgeposition, an der die Seitenaktionen auf der Seite angezeigt werden, wenn mehrere Seitenaktionen vorhanden sind; dabei entspricht 0 der obersten Position, 1 der zweiten Position usw.

Optional

UIVirtualPage

Gibt eine Seite an, auf der die Seitenaktion angezeigt wird.

Erforderlich

Vorgehensweise zum Hinzufügen von Quellseiten zu Seitenaktionen

Gehen Sie folgendermaßen vor, um Quellseiten zu Seitenaktionen hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Wählen Sie die Registerkarte Modellierung (Modeling) aus.
4. Doppelklicken Sie auf eine Seitenaktion. Im Arbeitsbereich wird das Popup "Seitenaktionseigenschaften" angezeigt.
5. Klicken Sie auf "Mehr" (More) für Quellseiten (SourcePages). Das Popup "Quellseiten" (SourcePages) wird angezeigt.
6. Wählen Sie im Feld UI-Virtuelle Seite (UIVirtualPage) eine Seite aus.
7. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
8. Wiederholen Sie die Schritte 5-7, um nach Bedarf weitere Seiten hinzuzufügen.
9. Klicken Sie auf OK. Das Popup "Quellseiten" (SourcePages) wird geschlossen.
10. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Löschen von Quellseiten aus Seitenaktionen

Gehen Sie folgendermaßen vor, um Quellseiten aus Seitenaktionen zu löschen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Wählen Sie die Registerkarte Modellierung (Modeling) aus.
4. Doppelklicken Sie auf eine Seitenaktion. Im Arbeitsbereich wird das Popup "Seitenaktionseigenschaften" angezeigt.
5. Klicken Sie auf "Mehr" (More) für Quellseiten (SourcePages). Das Popup "Quellseiten" (SourcePages) wird angezeigt.
6. Wählen Sie aus der Liste im linken Bereich eine Seite aus.
7. Klicken Sie auf Löschen. Die Seite wird aus der Liste entfernt.
8. Wiederholen Sie die Schritte 6-7, um nach Bedarf weitere Seiten hinzuzufügen.
9. Klicken Sie auf OK. Das Popup "Quellseiten" (SourcePages) wird geschlossen.
10. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
