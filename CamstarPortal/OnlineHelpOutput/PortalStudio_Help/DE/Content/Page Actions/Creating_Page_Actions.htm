Erstellen von Seitenaktionen

Sie können Seitenaktionen (Page Actions) über das Popup "Seitenaktionseigenschaften" in Portal Studio erstellen. Die Eigenschaften einer Seitenaktion und die Funktion, die nach Klicken auf die Seitenaktion ausgeführt wird, richten sich nach dem jeweiligen Typ der Benutzeroberflächenaktion (UIAction).

Beim Erstellen von Seitenaktionen

Der Zugriff auf die Seitenaktionen erfolgt über die Registerkarte "Modellierung" (Modeling) im Katalog der Benutzeroberflächenkomponenten. Zahlreiche Seitenaktionen wurden bereits standardmäßig erstellt. Seitenaktionen werden über die Seitenaktionseigenschaften konfiguriert. Die weitere Konfiguration erfolgt über Seitenaktionsregeln und Quellseiten.

Important: Sie können den Benutzeroberflächen-Aktionstyp nicht mehr ändern, nachdem Sie den Typ ausgewählt und die Seitenaktion gespeichert haben.

Seitenaktionsregeln (Page Action Rules) legen über benutzerdefinierte Bedingungen fest, wann eine Seitenaktion auf einer Seite verfügbar sein soll. Weitere Informationen zum Konfigurieren von Seitenaktionsregeln finden Sie unter "Seitenaktionsregeln".

Quellseiten (Source Pages) sind eine Liste der Seiten, auf denen die Aktion angezeigt wird, wenn die Bedingungen der Seitenaktionsregel erfüllt sind. Damit eine Seitenaktion auf einer Seite angezeigt wird, müssen Quellseiten für die Seitenaktion definiert sein. Weitere Informationen zum Hinzufügen von Seiten zu Quellseitenlisten finden Sie unter "Hinzufügen von Quellseiten zu Seitenaktionen".

Seitenaktionen sind so konfiguriert, dass sie entweder als Verknüpfung oder als Schaltfläche dargestellt werden. Seitenaktionen werden standardmäßig als Verknüpfungen eingerichtet, wenn sie mit dem Webpart "Aktionssteuerung" (ActionsControl) zu einer Seite hinzugefügt werden. Sie können die Eigenschaft "Aktionsstil" (ActionStyle) des Webparts "Aktionssteuerung" so ändern, dass Aktionen als Schaltflächen angezeigt werden, oder Seitenaktionen direkt als Schaltflächen zu Seiten hinzufügen. Weitere Informationen finden Sie unter "Konfigurieren von Seitenaktionen".

Vorgehensweise zum Erstellen von Seitenaktionen

Gehen Sie folgendermaßen vor, um Seitenaktionen zu erstellen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf der Schaltfläche Neu (New) auf den Text "Neu" (New) und wählen Sie Seitenaktion (Page Action) aus. Im Arbeitsbereich wird das Popup "Seitenaktionseigenschaften" angezeigt.
3. Geben Sie im Feld Name einen Namen für die Aktion ein. 
4. Erweitern Sie den Abschnitt Benutzeroberfläche: Aktion (UIAction) und wählen Sie einen Benutzeroberflächen-Aktionstyp (UIAction Type) aus. Die allgemeinen Eigenschaften (General) für "UIAction" werden je nach dem ausgewählten Benutzeroberflächen-Aktionstyp zurückgesetzt.
5. Klicken Sie auf "Mehr" (More) für Aktionsregeln (ActionRules) und fügen Sie je nach Bedarf weitere Aktionsregeln hinzu. Eine Anleitung für das Hinzufügen von Aktionsregeln zu Aktionen finden Sie unter "Hinzufügen von Seitenaktionsregeln zu Seitenaktionen".
6. Klicken Sie auf "Mehr" und fügen Sie je nach Bedarf Quellseiten (SourcePages) hinzu. Eine Anleitung für das Hinzufügen von Quellseiten zu Aktionen finden Sie unter "Hinzufügen von Quellseiten zu Seitenaktionen".
7. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
8. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
