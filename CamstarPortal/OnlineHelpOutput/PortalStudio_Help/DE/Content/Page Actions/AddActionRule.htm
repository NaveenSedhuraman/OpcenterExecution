Hinzufügen von Seitenaktionsregeln zu Seitenaktionen

Seitenaktionsregeln (Page Action Rules) bestimmen die Bedingungen, unter denen eine dynamische Aktion auf einer Portalseite angezeigt wird. Sie können einer Aktion mehrere Regeln hinzufügen, um zu steuern, wann eine Aktion angezeigt wird.

Beim Hinzufügen von Seitenaktionsregeln zu Seitenaktionen

Der Zugriff auf die Seitenaktionsregeln erfolgt über die Registerkarte "Modellierung" (Modeling) im Katalog der Benutzeroberflächenkomponenten. Zahlreiche Seitenaktionsregeln wurden bereits standardmäßig erstellt. Weitere Informationen zum Erstellen zusätzlicher Seitenaktionsregeln finden Sie unter "Erstellen von Seitenaktionsregeln".

Seitenaktionsregeln können auch aus einer Seitenaktion entfernt werden. Nachdem eine Seitenaktionsregel entfernt wurde, wird die Seitenaktion nicht mehr durch die Regel gesteuert.

Felddefinitionen des Popups "Aktionsregeln" (ActionRules)

Das Popup "Aktionsregeln" (ActionRules) enthält nur das Feld "Name" (Name). Im Feld "Name" wird eine vorhandene Seitenaktionsregel ausgewählt und der Seitenaktion zugewiesen.

Vorgehensweise zum Hinzufügen von Seitenaktionsregeln zu Seitenaktionen

Gehen Sie folgendermaßen vor, um Seitenaktionsregeln zu Seitenaktionen hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Wählen Sie die Registerkarte Modellierung (Modeling) aus.
4. Doppelklicken Sie auf eine Seitenaktion. Im Arbeitsbereich wird das Popup "Seitenaktionseigenschaften" angezeigt.
5. Klicken Sie auf "Mehr" (More) für Aktionsregeln (ActionRules). Das Popup "Aktionsregeln" (ActionRules) wird angezeigt.
6. Klicken Sie auf Hinzufügen (Add). In der Liste im linken Bereich wird eine neue Aktionsregel angezeigt.
7. Wählen Sie im Feld Name eine Seitenaktionsregel aus.
8. Wiederholen Sie ggf. die Schritte 5 bis 7, um weitere Seitenaktionsregeln hinzuzufügen.
9. Klicken Sie auf OK. Das Popup "Aktionsregeln" (ActionRules) wird geschlossen.
10. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Löschen von Seitenaktionsregeln aus Seitenaktionen

Gehen Sie folgendermaßen vor, um Seitenaktionsregeln aus Seitenaktionen zu entfernen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Wählen Sie die Registerkarte Modellierung (Modeling) aus.
4. Doppelklicken Sie auf eine Seitenaktion. Im Arbeitsbereich wird das Popup "Seitenaktionseigenschaften" angezeigt.
5. Klicken Sie auf "Mehr" (More) für Aktionsregeln (ActionRules). Das Popup "Aktionsregeln" (ActionRules) wird angezeigt.
6. Wählen Sie aus der Liste im linken Bereich eine Seitenaktionsregel aus.
7. Klicken Sie auf Löschen. Die Seitenaktionsregel wird aus der Liste entfernt.
8. Wiederholen Sie ggf. die Schritte 6 bis 7, um weitere Seitenaktionsregeln zu löschen.
9. Klicken Sie auf OK. Das Popup "Aktionsregeln" (ActionRules) wird geschlossen.
10. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
