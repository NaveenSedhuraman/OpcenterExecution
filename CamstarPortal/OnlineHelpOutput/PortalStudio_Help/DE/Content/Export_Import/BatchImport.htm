Durchführen von Batch-Importen

Der Batch-Import bietet die Möglichkeit, mehrere Seiten, Webparts und Seitenfolgen aus einer einzelnen XML-Datei oder einer ZIP-Datei zu importieren.

Beim Durchführen von Batch-Importen

Sie müssen in der Liste "Personalisierungstyp für Import" (Personalization Type to Import) mindestens einen Arbeitsbereich auswählen.

Wenn Sie auf die Schaltfläche "Import starten" (Start Import) klicken, wird das Popup-Fenster "Importfortschritt..." (Import Progress) mit zwei Fortschrittsbalken angezeigt. Der erste Balken zeigt den Fortschritt des jeweils importierten Elements an, der zweite Balken den Gesamtfortschritt des Importvorgangs.

Felddefinitionen des Fensters "Batch-Import" (Batch Import)

Diese Tabelle definiert die Felder im Fenster "Batch-Import" (Batch Import).

Feld

Definition

Typ

Exportoptionen

Personalization Type to Import

Gibt die Arbeitsbereiche an, in die die Batch-Importdaten importiert werden. Mindestens ein Arbeitsbereich muss ausgewählt werden.

Erforderlich

Aus Datei importieren (Import from File)

Wenn Sie auf die Schaltfläche "Mehr" (More) klicken, wird das Popup-Fenster "Öffnen" (Open) angezeigt, in dem Sie zu einer XML-Datei oder einer ZIP-Datei mit XML-Dateien navigieren können. Bei Auswahl einer Datei und Klicken auf "Öffnen" (Open) wird das Popup-Fenster "Öffnen" (Open) geschlossen und der ausgewählte Dateiname wird in dieses Feld eingetragen.

Erforderlich

Webpart/Seite/Seitenfolge überschreiben, wenn bereits vorhanden (Override Web Part/Page/Pageflow if it already exists)

Wenn diese Option ausgewählt ist, werden vorhandene Elemente in den ausgewählten Arbeitsbereichen beim Import überschrieben. Diese Option ist standardmäßig aktiviert.

Hinweis: Seiten, Webparts und Seitenfolgen in der Importdatei, die bereits in den ausgewählten Arbeitsbereichen vorhanden sind, werden nicht importiert, wenn diese Option nicht ausgewählt ist.

Vorgehensweise zum Durchführen von Batch-Importen

Gehen Sie folgendermaßen vor, um einen Batch-Import durchzuführen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf die Schaltfläche Batch-Import (Batch Import). Das Fenster "Batch-Import" wird angezeigt.
3. Wählen Sie in der Liste Personalisierungstyp für Import (Personalization Type to Import) mindestens einen Arbeitsbereich aus.
4. Klicken Sie auf Mehr (More) für Aus Datei Importieren (Import from File). Das Popup "Öffnen" (Open) wird angezeigt.
5. Navigieren Sie zu einer XML-Datei oder ZIP-Datei. Wählen Sie die Datei aus und klicken Sie auf Öffnen (Open). Das Popup "Öffnen" (Open) wird geschlossen und im Feld "Aus Datei importieren" (Import from File) wird der Dateiname angezeigt.
6. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
7. Klicken Sie auf Import starten. Das Popup-Fenster "Importfortschritt..." (Import Progress) wird angezeigt und zeigt den Fortschritt des Imports an. Popup-Fenster "Importfortschritt..." (Import Progress) wird geschlossen und im Fenster "Batch-Import" (Batch Import) wird eine Erfolgsmeldung angezeigt, wenn der Importprozess abgeschlossen wurde.
