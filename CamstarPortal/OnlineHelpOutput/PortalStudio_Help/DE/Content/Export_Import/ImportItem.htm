Importieren eines Elements

Sie können XML-Dateien in Portal Studio importieren. Sie können neue Webparts und Seiten oder Änderungen zur Aktualisierung vorhandener Webparts und Seiten importieren.

Beim Importieren eines Elements

Eine XML-Datei können Sie nur in offene Seite oder ein offenes Webpart im höchsten aktiven Arbeitsbereich importieren.

Nachdem Sie eine Seite oder ein Webpart importiert haben, können Sie die Webseite bzw. das Webpart mit einem aktualisierten Namen speichern.

Note: Die Größe eines Imports ist standardmäßig auf 30 Megabyte (MB) beschränkt. Wenn Ihr Bundle größer ist, können Sie den Wert für die Einstellung "Maximale Anhangsgröße" (MaxAttachmentSizeLimit) in den Portaleinstellungen erhöhen. Weitere Informationen finden Sie unter "Konfigurieren von Portaleinstellungen".

Vorgehensweise zum Importieren eines Elements

Gehen Sie folgendermaßen vor, um ein Element zu importieren:

1. Greifen Sie auf Portal Studio zu.
2. Erstellen oder öffnen Sie eine vorhandene Seite oder ein vorhandenes Webpart.
3. Klicken Sie auf die Schaltfläche Import. Das Dialogfeld "Öffnen" (Open) wird angezeigt.
4. Suchen und wählen Sie die XML-Datei aus, die Sie importieren möchten.
5. Klicken Sie auf Öffnen (Open). Die XML-Datei wird in die geöffnete Seite oder das geöffnete Webpart importiert.
6. Ändern Sie den Namen der Seite oder des Webparts im Bereich "Eigenschaften" (Properties).
7. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
