Seitenaktionsregeln

Seitenaktionsregeln (Page Action Rules) legen fest, wann eine dynamische Aktion auf einer Portalseite im Webpart "Aktionssteuerung" (ActionsControl) angezeigt wird. Aktionen können mehrere Seitenaktionsregeln aufweisen. Wenn eine Aktion die Bedingungen einer der mit ihr zugewiesenen Regeln nicht erfüllt, wird die Aktion weder aktiviert noch angezeigt.

Seitenaktionsregeln erfordern Regelausdrücke, um festzulegen, wann eine dynamische Aktionsschaltfläche angezeigt wird.
