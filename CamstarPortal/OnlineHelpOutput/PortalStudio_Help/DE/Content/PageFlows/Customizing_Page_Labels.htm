Anpassen von Seitenbeschriftungen

In Portal Studio können Sie die Beschriftungen anpassen, die auf Seiten angezeigt werden.

Beim Anpassen von Seitenbeschriftungen

Im Bereich "Eigenschaften" (Properties) im Feld "SeitentitelBeschriftungstext" (PageTitleLabelText) wird der Titel konfiguriert, der für eine Seite angezeigt wird.

Im Bereich "Eigenschaften" (Properties) im Feld "SeitenbeschreibungBeschriftungstext" (PageDescriptionLabelText) werden weitere Informationen konfiguriert, z. B. eine Beschreibung unter einem Seitentitel.

Vorgehensweise zum Anpassen von Seitenbeschriftungen

Gehen Sie folgendermaßen vor, um Seitenbeschriftungen anzupassen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Erweitern Sie Seitenfolgen (Pageflows).
4. Doppelklicken Sie auf eine Seitenfolge. Die Seitenfolge wird im Arbeitsbereich angezeigt.
5. Klicken Sie auf Eigenschaften anzeigen/ausblenden (Show/Hide Properties), wenn der Bereich "Eigenschaften" (Properties) ausgeblendet ist. Der Bereich "Eigenschaften" (Properties) wird angezeigt.
6. Wählen Sie eine Seite aus der Seitenfolge aus. Die allgemeinen Seiteneigenschaften für die ausgewählt Seite werden im Bereich "Eigenschaften" (Properties) angezeigt.
7. Geben Sie im Bereich "Eigenschaften" (Properties) im Feld SeitentitelBeschriftungstext (PageTitleLabelText) einen neuen Titel ein.
8. Geben Sie im Bereich "Eigenschaften" (Properties) im Feld SeitenbeschreibungBeschriftungstext (PageDescriptionLabelText) eine neue Seitenbeschreibung ein.
9. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
