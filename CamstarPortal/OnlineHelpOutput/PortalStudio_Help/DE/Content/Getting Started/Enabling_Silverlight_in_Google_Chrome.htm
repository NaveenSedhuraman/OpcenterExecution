Aktivieren von Silverlight in Google Chrome

Die Unterstützung für die Netscape Plugin Application Programming Interface (NPAPI) in Google Chrome Version 42, 43 und 44 ist standardmäßig deaktiviert. Darum sind bestimmte Plug-Ins wie Microsoft Silverlight ebenfalls deaktiviert. Wenn Sie Chrome als Browser verwenden, müssen Sie Silverlight aktivieren, damit Sie auf Portal Studio zugreifen können.

Important: Silverlight wird in Chrome nach Version 44 nur begrenzt oder nicht unterstützt. Siemens empfiehlt, dass Sie auf Portal Studio in einer unterstützten Version von Internet Explorer zugreifen.

Beim Aktivieren von Silverlight in Google Chrome

Sie müssen außerdem die Einstellung "win32k renderer lockdown" auf "Deaktiviert" (Disabled) festlegen, wenn Sie mit Chrome, Version 43 und 44, auf Portal Studio zugreifen. Nachdem Sie diese Einstellung geändert haben, müssen Sie Chrome schließen und erneut öffnen, bevor Sie auf Portal Studio zugreifen.

Vorgehensweise zum Aktivieren von Silverlight in Google Chrome Version 42

Gehen Sie folgendermaßen vor, um Silverlight in Google Chrome, Version 42, zu aktivieren:

1. Starten Sie Google Chrome.
2. Navigieren Sie zu der Adresse "chrome://flags/#enable-npapi". Die Seite "Experimentelle Funktionen" (Experimental Features) wird angezeigt.
3. Klicken Sie auf Aktivieren (Enable) für NPAPI aktivieren (Enable NPAPI). Die Hintergrundfarbe der Einstellung ändert sich in Weiß und die NPAPI-Plug-Ins werden aktiviert.
4. Anmelden im Portal.
5. Klicken Sie auf Studio. Eine leere Seite wird angezeigt.
6. Klicken Sie mit der rechten Maustaste auf das Plug-In-Symbol und wählen Sie Dieses Plug-In ausführen (Run this Plugin) aus. Die Portal Studio-Homepage und das Startseiten-Dashboard (Start Page) werden angezeigt.

Vorgehensweise zum Aktivieren von Silverlight in Google Chrome, Version 43 und 44

Gehen Sie folgendermaßen vor, um Silverlight in Google Chrome, Version 43 und 44, zu aktivieren:

1. Starten Sie Google Chrome.
2. Navigieren Sie zu der Adresse "chrome://flags/#enable-npapi". Die Seite "Experimentelle Funktionen" (Experimental Features) wird angezeigt.
3. Klicken Sie auf Aktivieren (Enable) für NPAPI aktivieren (Enable NPAPI). Die Hintergrundfarbe der Einstellung ändert sich in Weiß und die NPAPI-Plug-Ins werden aktiviert.
4. Wählen Sie für win32k renderer lockdown-Fenster aktivieren (Enable win32k renderer lockdown Windows) den Eintrag Deaktiviert (Disabled) aus. Die Hintergrundfarbe der Einstellung ändert sich in Gelb und die Einstellung "win32k renderer lockdown" wird deaktiviert.
5. Schließen Sie den Browser und öffnen Sie ihn erneut.
6. Anmelden im Portal.
7. Klicken Sie auf Studio. Eine leere Seite wird angezeigt.
8. Klicken Sie mit der rechten Maustaste auf das Plug-In-Symbol und wählen Sie Dieses Plug-In ausführen (Run this Plugin) aus. Die Portal Studio-Homepage und das Startseiten-Dashboard (Start Page) werden angezeigt.
