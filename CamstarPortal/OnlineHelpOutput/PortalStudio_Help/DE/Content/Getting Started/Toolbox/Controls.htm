Steuerelemente

Mit Steuerelementen (Controls) können Sie Webparts in Portal Studio anpassen. Der Zugriff auf Steuerelemente erfolgt über die Registerkarte "Steuerelemente" (Controls) in der Toolbox. Weitere Informationen zum Anzeigen der Toolbox finden Sie unter "Toolbox".

Sie können Steuerelemente nur zu einem Webpart, nicht jedoch direkt zu einer Seite hinzufügen. Weitere Informationen zum Hinzufügen von Steuerelementen zu einem Webpart finden Sie unter "Hinzufügen eines Steuerelements zu einem Webpart".

Eine Popup-Warnmeldung wird angezeigt, wenn Sie ein Steuerelement hinzufügen, dessen Unterstützung in einem zukünftigen Release von Portal Studio eingestellt werden soll. Die Warnmeldung gibt an, wann die Unterstützung für das Steuerelement eingestellt wird und welche anderen Steuerelemente Sie stattdessen verwenden können. Auf Steuerelementen, deren Unterstützung eingestellt wird, wird außerdem eine QuickInfo mit diesen Informationen angezeigt.

Diese Tabelle beschreibt die Steuerelemente, die auf der Registerkarte "Steuerelemente" in der Toolbox verfügbar sind.

Symbol

Steuerelement

Beschreibung

3D-Visualisierung

Zeigt 3D-Visualisierungen mit der Möglichkeit zum Drehen, Vergrößern bzw. Verkleinern, Markieren von Bildern und Speichern von Momentaufnahmen an.

Hinweis: Dieses Steuerelement unterstützt JT-Dateien.

Weitere Informationen zum 3D-Visualisierungs-Steuerelement finden Sie unter "Verwenden der 3D-Visualisierung".

Boolean Drop-down

Boolesches Dropdown-Element. Boolescher Selektor mit konfigurierbaren Beschriftungen und Optionen zum Löschen des Werts.

Boolean Switch

Boolescher Schalter. Boolesches Optionsfeld-Steuerelement mit konfigurierbaren Beschriftungen.

Button

Schaltfläche. Klickbares Steuerelement, das mit Text oder als Bild angezeigt werden kann.

Chart Control

Chart-Steuerelement. Positioniert ein benutzerdefiniertes Diagramm auf einer Seite oder einem Webpart. Eine einzelne Seite kann gleichzeitig mehrere Chart-Steuerelemente enthalten.

Sie können das Format und die Daten der Chart-Steuerelemente über die Eigenschaften "ChartMode", "ShowLegend", "SeriesSettings", "SeriesValueColumn", "TooltipColumn", "ValueColumn", "XAxis" und "YAxis" im Bereich "Eigenschaften" (Properties) ändern. Chart-Steuerelemente werden dynamisch aktualisiert, wenn Sie die Datenquellen für diese Eigenschaften ändern. Weitere Informationen zu Eigenschaften von Chart-Steuerelementen finden Sie unter "Steuerelementeigenschaften".

Die Eigenschaft "SeriesSettings" ermöglicht es Ihnen, die Darstellung von Datenpunkten in Streu- und Wafermap-Diagrammen weiter anzupassen. Informationen zum Hinzufügen von SeriesSettings-Eigenschaften finden Sie unter "Hinzufügen von SeriesSettings".

Sie können benutzerdefinierte Diagrammformate einschließlich Wafer-Zuordnungen erstellen, indem Sie die Datei d3.js ändern. Weitere Informationen zum D3-Dokumentobjektmodell finden Sie unter http://d3js.org.

Hinweis: Sie können auf Portalseiten mit dem Mausrad in Chart-Steuerelemente hineinzoomen und daraus herauszoomen.

Check Box

Kontrollkästchen. Feld, das markiert wird oder dessen Markierung aufgehoben wird, um eine Option zu aktivieren oder zu deaktivieren.

Hinweis: Sie müssen die Eigenschaft "Autom. Postback" (AutoPostBack) auf "Wahr" (True) setzen, wenn Sie Kontrollkästchen-Steuerelemente zu Seiten hinzufügen, die Popup-Tabellen enthalten. Weitere Informationen zum Ändern der Eigenschaften von Steuerelementen finden Sie unter "Steuerelementeigenschaften".

Checklist

Prüfliste. Prüflisten-Steuerelement.

Color Picker

Farbauswahl. Ermöglicht die Auswahl einer Farbpalette, um eine Farbe für ein Feld festzulegen. Weitere Informationen finden Sie unter "Verwenden der Farbauswahl".

Container Drop-down

Container Picklist

Data Collection

Discovery Area

Drop-down List

Duration

Dauer. Steuerelement, das eine Zeitspanne angibt. (TT.HH:MM:SS) Es kann in Verbindung mit der Prozesszeiterzwingung verwendet werden. Weitere Informationen zum Definieren von Prozesstimern finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch oder im Opcenter Execution Core Modeling-Benutzerhandbuch.

HTML-Fenster

Ermöglicht die Angabe von Folgendem:

• HTML-Inhalte, die mit der Eigenschaft "HTMLText" als Text eingegeben werden, oder
• Ordnerpfad zu einer HTML-Vorlagendatei mit der Eigenschaft "TemplateFile".

Hinweis: Bei Verwendung dieses Steuerelements müssen Sie einen Wert für die Eigenschaft "HTMLText" oder "TemplateFile" eingeben. Weitere Informationen zu den Eigenschaften "HTMLText" und "TemplateFile" finden Sie unter "Steuerelementeigenschaften".

IFrame

Image Control

Intelligence Object

Hinweis: Dieses Steuerelement greift standardmäßig mit einer HTTP-Verbindung auf Business Objects-Berichte zu und führt sie aus. Sie können die Einstellung des Steuerelements im Bereich "Eigenschaften" ändern. Weitere Informationen finden Sie unter "Steuerelementeigenschaften".
