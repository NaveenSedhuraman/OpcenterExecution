Toolbox

Über die Toolbox können Sie auf die folgenden Registerkarten zugreifen:

• Steuerelemente (Controls): Zeigt eine Liste der Steuerelemente an, die Sie zu Webparts hinzufügen können. Weitere Informationen finden Sie unter "Steuerelemente".
• Feldverzeichnis (Fields Directory): Zeigt eine Liste der Services und parametrischen Datenfelder an, die Sie zu Webparts hinzufügen können. Weitere Informationen finden Sie unter "Feldverzeichnis".

Mit der Schaltfläche "Toolbox anzeigen/ausblenden" (Show/Hide Toolbox) in der Toolbox von Portal Studio kann die ausgeblendete Toolbox angezeigt bzw. die angezeigte Toolbox ausgeblendet werden. Zum Anzeigen oder Ausblenden der Toolbox können Sie auch auf "Anzeigen" (View) klicken und "Toolbox anzeigen/ausblenden" (Show/Hide Toolbox) auswählen.

Die Symbolleiste wird oben im Toolboxbereich angezeigt. Je nachdem, welche Registerkarte Sie auswählen, werden unterschiedliche Schaltflächen angezeigt.

Relative Schaltfläche

Die Registerkarte "Feldverzeichnis" (Fields Directory) enthält die Schaltfläche "Relativ" (Relative). Die Schaltfläche "Relativ" (Relative) wechselt zwischen den relativen und absoluten Einstellungen für ein Feld, wenn das Feld zu einem Webpart hinzugefügt wird. Die Schaltfläche hat standardmäßig einen blaugrünen Hintergrund, der das Feld als relativ kennzeichnet. Wenn Sie auf die Schaltfläche klicken, wird sie abgeblendet und zeigt an, dass es sich um ein absolutes Feld handelt. Weitere Informationen finden Sie unter "Relative Feldausdrücke".

Vorgehensweise zum Anzeigen oder Ausblenden der Toolbox

Gehen Sie folgendermaßen vor, um die Toolbox anzuzeigen oder auszublenden:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Toolbox anzeigen/ausblenden (Show/Hide Toolbox), wenn die Toolbox ausgeblendet ist. Die Toolbox wird angezeigt.
