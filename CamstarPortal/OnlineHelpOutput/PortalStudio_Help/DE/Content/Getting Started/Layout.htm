Verwalten von Layouts

In Portal Studio können Sie das Layout einer Seite oder eines Webparts mit einem Zeilen- und Spaltenlayout konfigurieren, das Zellen enthält.

Im Bereich "Eigenschaften" (Properties) können Sie die Eigenschaften einer einzelnen Zelle ändern. Informationen zum Ändern von Zelleneigenschaften finden Sie unter "Zelleneigenschaften".

Beim Verwalten von Layouts

Jede Seite oder jedes Webpart enthält mindestens eine Zelle als Teil des Objektlayouts. Mithilfe von Zellen können Sie angeben, wo Objekte angezeigt werden, wenn sie auf Seiten und in Webparts in Bezug auf Objekte in anderen Zellen platziert werden.

Sie können mit der rechten Maustaste in eine Zelle klicken, um Optionen anzuzeigen, mit denen Sie das Layout einer Zelle ändern können. Optional können Sie auch klicken und die Maus ziehen, um mehrere Zellen auszuwählen, und anschließend mit der rechten Maustaste in die hervorgehobenen Zellen klicken, um das Layout von mehreren Zellen gleichzeitig zu ändern.

In dieser Tabelle werden die zum Verwalten von Zellenlayouts verfügbaren Optionen definiert.

Option

Aktion . . .

Aufteilen

Teilt eine verbundene Zelle wieder in ihre Komponentenzellen.

Verbinden (Merge)

Verbindet die ausgewählten Zellen zu einer einzelnen Zelle.

Zeile entfernen (Remove Row)

Löscht die ausgewählte Zeile.

Spalte entfernen (Remove Column)

Löscht die ausgewählte Spalte.

Zeile einfügen vor (Insert Row Before)

Fügt eine neue Zeile über der ausgewählten Zeile ein.

Spalte einfügen vor (Insert Column Before)

Fügt eine neue Spalte links von der ausgewählten Spalte ein.

Zeile einfügen nach (Insert Row After)

Fügt eine neue Zeile unter der ausgewählten Zeile ein.

Spalte einfügen nach (Insert Column After)

Fügt eine neue Spalte rechts von der ausgewählten Spalte ein.

Horizontal

Links ausrichten (Align Left)

Richtet den Zelleninhalt am linken Zellenrand aus.

Rechts ausrichten (Align Right)

Richtet den Zelleninhalt am rechten Zellenrand aus.

Zentrieren (Align Center)

Richtet den Zelleninhalt mit der horizontalen Zellenmitte aus.

Nicht festgelegt (Not Set)

Entfernt die gesamte horizontale Ausrichtung.

Vertikal (Vertical)

 

Oben ausrichten (Align Top)

Richtet den Zelleninhalt am oberen Zellenrand aus.

Unten ausrichten (Align Bottom)

Richtet den Zelleninhalt am unteren Zellenrand aus.

Zentrieren (Align Center)

Richtet den Zelleninhalt mit der vertikalen Zellenmitte aus.

Nicht festgelegt (Not Set)

Entfernt die gesamte vertikale Ausrichtung.
