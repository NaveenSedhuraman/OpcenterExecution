Globale Symbolleiste von Portal Studio

Die globale Symbolleiste von Portal Studio ermöglicht den einfachen Zugriff auf die in Portal Studio verwendeten Funktionen.

Note: Die Funktionen in der globalen Symbolleiste von Portal Studio sind auch über die Dropdownmenüs in der Menüleiste verfügbar.

Schaltflächen der globalen Symbolleiste von Portal Studio

Diese Tabelle beschreibt die Schaltflächen in der globalen Symbolleiste von Portal Studio.

Ein neues virtuelles Objekt zu erstellen. Das Popup "Neue Seite erstellen" (Create New Page) wird angezeigt, wenn Sie auf den Teil der Schaltfläche mit dem Pluszeichen klicken. Hier können Sie eine neue Portalseite konfigurieren. Wenn Sie auf den Text "Neu" (New) der Schaltfläche klicken, wird eine Dropdownliste angezeigt. Die Optionen in dieser Liste können Sie verwenden, wenn Sie ein Webpart, eine Seitenfolge, eine Menüdefinition, eine Registerkarte, ein Zuordnungselement, ein Zielmatrixelement, eine Seitenaktion, eine Seitenaktionsregel oder eine Seitenaktionskategorie neu erstellen möchten.

Speichern Sie die ausstehenden Änderungen für die Komponente, die im Arbeitsbereich ausgewählt ist. Weitere Informationen zum Speichern von Änderungen finden Sie unter "Speichern von Änderungen".

Hinweis: In Portal Studio wird diese Schaltfläche aktiviert, wenn Sie eine Änderung an der Komponente durchführen, die derzeit im Arbeitsbereich im Fokus ist. Sie können für diese Schaltfläche festlegen, dass sie immer aktiviert ist, indem Sie die Studio-Einstellung "AlwaysEnableSave" auf "Wahr" (True) festlegen. Weitere Informationen zum Ändern von Einstellungen für Portal Studio finden Sie unter "Studio-Einstellungen (Studio Settings)".
