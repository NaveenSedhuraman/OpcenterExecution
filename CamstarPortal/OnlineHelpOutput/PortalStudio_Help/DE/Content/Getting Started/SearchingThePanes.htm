Suchen nach Objekten in einem Bereich

Jeder Bereich in Portal Studio enthält ein Suchfeld, über das Sie auf einfache Weise ein bestimmtes Objekt in dem betreffenden Katalog oder Verzeichnis suchen können.

Portal Studio filtert die Liste noch während Ihrer Eingabe in das Suchfeld, damit Sie das gesuchte Objekt schneller finden. Die Suche kann für flache Listen und für Baumstrukturen durchgeführt werden.

Beim Suchen nach Objekten in einem Bereich

Sie können oben im Bereich einen Namen oder einen Teil eines Namens in das Suchfeld eingeben. Bei der Suche wird davon ausgegangen, dass am Anfang und am Ende der Eingabe ein Platzhalterzeichen steht. So finden Sie alle Objekte, deren Name den Ausdruck oder Teilausdruck enthält.

Mit einem Klick auf das X im Suchfeld löschen Sie das Feld.

Vorgehensweise zum Suchen nach Objekten in einem Bereich

Gehen Sie folgendermaßen vor, um nach Objekten in einem Bereich zu suchen:

1. Greifen Sie auf Portal Studio zu.
2. Mit den Schaltflächen "Anzeigen/Ausblenden" (Show/Hide) können Sie den gewünschten Bereich anzeigen.
3. Beginnen Sie mit der Eingabe in das Suchfeld oben im Bereich. Der Bereich filtert die Suchergebnisse während Ihrer Eingabe.
