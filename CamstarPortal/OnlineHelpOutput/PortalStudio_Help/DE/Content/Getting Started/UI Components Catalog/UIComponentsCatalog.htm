Katalog der Benutzeroberflächenkomponenten

Der Katalog der Benutzeroberflächenkomponenten (UI Components) enthält mehrere Registerkarten, die den Zugriff auf Komponenten sowie Navigations- und Zuordnungselemente ermöglichen.

Über die Registerkarte "Komponenten" (Components) können Sie auf folgende Komponenten zugreifen:

• Seiten
• Webparts
• Seitenfolgen

Über die Registerkarte "Navigation" können Sie auf folgende Navigationselemente zugreifen:

• Menüdefinitionen
• Registerkarten

Über die Registerkarte "Modellierung" (Modeling) können Sie auf folgende Zuordnungselemente zugreifen:

• Seitenaktionen
• Seitenaktionskategorien
• Erstellen von Seitenaktionsregeln
• Seitenzuordnungen
• Zielmatrizen

Die Registerkarten des Katalogs der Benutzeroberflächenkomponenten sind in Dropdownlisten für die verschiedenen Komponentenelementtypen zusammengefasst. Jede Dropdownliste enthält ein Suchfeld, die Schaltfläche "Neues Element erstellen" (Create New Item), eine Reihe von Anzeigemodus-Schaltfeldern und die Schaltfläche "Aktualisieren" (Refresh). Wenn Sie auf die Schaltfläche "Neues Element erstellen" (Create New Item) klicken, wird im Arbeitsbereich ein neues Element für den betreffenden Elementtyp angezeigt.

Mit der Schaltfläche "Katalog anzeigen/ausblenden" (Show/Hide Catalog) in der Symbolleiste von Portal Studio wird der Katalog angezeigt, falls er ausgeblendet ist, und ausgeblendet, falls er angezeigt wird. Sie können den Katalog der Benutzeroberflächenkomponenten auch anzeigen bzw. ausblenden, indem Sie auf "Anzeigen" (View) klicken und dann "Katalog anzeigen/ausblenden" (Show/Hide Catalog) auswählen.

Note: Wenn Sie sich beim Upgrade von einer früheren Versions erstmals bei Portal Studio anmelden, werden u. U. einige Standardseiten im Katalog der Benutzeroberflächenkomponenten auf der Registerkarte "Components" (Komponenten) in der Gruppe "Nicht definiert" (Undefined) angezeigt. Die Standardseiten in der Gruppe "Nicht definiert" (Undefined) können aus dem Katalog der Benutzeroberflächenkomponenten automatisch in andere Gruppen sortiert werden. Weitere Informationen finden Sie unter "Vorgehensweise zum automatisches Sortieren von Standardseiten".

Vorgehensweise zum Navigieren im Katalog der Benutzeroberflächenkomponenten

Gehen Sie folgendermaßen vor, um im Katalog der Benutzeroberflächenkomponenten zu navigieren:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Klicken Sie unten im Katalog auf eine Registerkarte, um die Optionen dieser Registerkarte anzuzeigen.
4. Mit dem Pfeil neben einem Abschnitt können Sie den Abschnitt erweitern bzw. reduzieren.
