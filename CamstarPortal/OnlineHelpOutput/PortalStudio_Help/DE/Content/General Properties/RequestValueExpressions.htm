Anforderungswertausdrücke

Anforderungswertausdrücke (Request Value Expressions) können in derselben Weise festgelegt werden wie ein Feldausdruck.

In Portal Studio kann ein Wert vom Server angefordert und als Wert in dem Feld verwendet werden, das an die Transaktion eingereicht werden soll.

Beim Verwenden von Anforderungswertausdrücken

Anforderungswerte erfordern einen Roundtrip zum Anwendungsserver, um den Wert zu generieren, bevor er eingereicht werden kann.
