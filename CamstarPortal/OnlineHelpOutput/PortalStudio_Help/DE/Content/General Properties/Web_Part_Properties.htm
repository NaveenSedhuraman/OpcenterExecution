Webpart-Eigenschaften

Über Webpart-Eigenschaften können Sie die Eigenschaften von Webparts anzeigen oder bearbeiten. Webpart-Eigenschaften werden im Bereich "Eigenschaften" (Properties) angezeigt, wenn im Arbeitsbereich ein Webpart ausgewählt ist. Weitere Informationen zum Bereich "Eigenschaften" (Properties) finden Sie unter "Bereich "Eigenschaften" (Properties)".

Beschreibungen der Webpart-Eigenschaften

Diese Tabelle enthält die Beschreibungen der Webpart-Eigenschaften.

Webpart-Eigenschaft

Beschreibung der Eigenschaft

Allgemein (General)

Aktionen

Aktionen des Webparts. Ermöglicht das Definieren von Aktionen und Schaltfeldern auf der Webpart-Ebene. Weitere Informationen finden Sie unter "Aktionseigenschaften".

BroadcastInitiators

Wird zum Erstellen von Initiatoren verwendet, die Verbindungsdatenverträge zwischen Webparts auf der Seitenebene auslösen.

CollapsibleGroups

Einstellung der Unterstützung in einem kommenden Release.

Hinweis: Zum Erstellen von reduzierbaren Abschnitten müssen Sie die Eigenschaft "Anzeigemodus umschalten" (ToggleDisplayMode) des Umschaltcontainer-Steuerelements (Toggle Container) verwenden. Weitere Informationen finden Sie unter "Steuerelementeigenschaften".

Control

Referenziert ein Steuerelement auf der Seite.

DataContractMembers

Datenverträge, die für die Kommunikation zwischen Seiten verwendet werden.

DataSubmissionMode

Legt fest, wie ein Feld reagiert, wenn eine Seite Daten an den Server übermittelt. Operanden schließen ein:

• NotSet: Modus nicht festgelegt. Dies ist der Standardmodus.
• Force: Der Wert ist immer in den übermittelten Daten enthalten.
• Changed: Der Wert wird nur übermittelt, wenn er geändert wurde.
• Filled: Der Wert wird nur übermittelt, wenn das Feld nicht leer ist.
• ChangedAndFilled: Der Wert wird nur übermittelt, wenn das Feld nicht leer ist und der Wert geändert wurde.
• Skip: Der Wert wird nicht übermittelt oder geladen.
• ViewOnly: Der Wert wird nicht übermittelt.
• Skip Request: Der Wert wird nicht geladen. Die Auswahlwerte für das Steuerelement werden nicht geladen.

Wichtig: Siemens empfiehlt, für Modellierungsseiten die Option "Changed" zu verwenden.

DirtyFlagTrigger

Boolesches Feld, das die Seite als "Geändert" (Dirty) markiert, wenn ein Feld innerhalb des Webparts bei der Einstellung "Wahr" (True) seinen Wert ändert.

Hinweis: Dies betrifft derzeit nur Webparts innerhalb des Modellierungsframeworks.

EventHandlers

Ermöglicht die Angabe eines Handlers von einem Ereignis. Weitere Informationen finden Sie unter "Definieren von Ereignishandlern".

Layout

Visuelle Darstellung des Webparts. Siemens empfiehlt, diese Eigenschaft nicht direkt zu ändern.

Hinweis: Untergeordnete Eigenschaften werden basierend auf den Zellen und Steuerelementen im aktuell ausgewählten Webpart angezeigt.

Name

Name des Webparts.

PrimaryServiceType

Servicetyp für Steuerelemente, die keinen Servicetyp angeben.

SessionVariables

Wird zum Konfigurieren von Sitzungsvariablen verwendet. Weitere Informationen finden Sie unter "Definieren von Sitzungsvariablen".

TypeName

Code-Behind-Typ des Webparts.

WebProperties

Sammlung von Web-Eigenschaften für das Webpart.

Verhalten (Behavior)

ActionsStyle

Legt fest, wie Aktionen (z. B. Schaltflächen oder Links) gerendert werden.

ClientPropertyConditions

Hier können Sie Daten auf Basis eines Bedingungsergebnisses von einem browserseitigen Ereignis auf eine Eigenschaft festlegen.

IncludeHandlerActions

Schließt die Aktionen, Webpart-Aktionen und Modellierungselemente für Seitenaktionen der aktuellen Seite ein, wenn auf "Wahr" (True) festgelegt. Schließt nur die Aktionen des ActionControl-Webparts ein, wenn auf "Falsch" (False) festgelegt.

PropertyConditions

Bietet die Möglichkeit, den Eigenschaftswert eines Webparts als Ergebnis eines ausgewerteten Ausdrucks (einer Bedingung) zu definieren. Die Eigenschaft wird ausgewertet, wenn die Seite zur Laufzeit ausgeführt wird. Weitere Informationen finden Sie unter "Eigenschaftsbedingungen".

Darstellung (Appearance)

IsHidden

Legt fest, dass das Webpart nicht auf Seiten angezeigt wird.

IsMinimized

Legt fest, dass das Webpart minimiert angezeigt wird.

Title

Der Text, der als Titel des Webparts angezeigt wird. Der hier eingegebene Text hat Vorrang vor dem Feld "Titelbeschriftung" (TitleLabel), wenn sowohl Text eingegeben als auch eine Beschriftung ausgewählt wird.

TitleLabel

Wählt eine Beschriftung aus, die als Titel des Webparts verwendet werden soll. Weitere Informationen zur Verwendung des Popups "Beschriftungssuche" (Label Lookup) finden Sie unter "Suchen nach Seitenaktionsbeschriftungen".

Layout

CssClass

Stylesheet-Referenz für das Webpart.

DisplayBorder

Zeigt einen Rahmen für das Webpart an.

DisplayFooter

Zeigt eine Fußzeile für das Webpart an.

DisplayHeader

Zeigt eine Kopfzeile für das Webpart an.

Height

Höhe in Pixel für die Anzeige des Webparts.

HorizontalAlignment

Horizontale Ausrichtung der Elemente des Webparts.

Margin

Rand des Webparts.

Padding

Textabstand des Webparts.

VerticalAlignment

Vertikale Ausrichtung der Elemente des Webparts.

Width

Breite in Pixel für die Anzeige des Webparts.

Verlauf (History)

Created

Datum und Uhrzeit der Erstellung des Webparts.

CreatedBy

Ersteller des Webparts.

HowLong

Zeitraum seit der Erstellung des Webparts.

LastModified

Datum der letzten Änderung des Webparts.

LastModifiedBy

Benutzer, der das Webpart zuletzt geändert hat.
