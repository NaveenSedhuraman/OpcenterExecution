Hinzufügen eines Untermenüs zu einer Menüdefinition

Sie können Untermenüs (Sub Menus) einem Menü zuweisen, nachdem Sie das Portal-Hauptmenü definiert haben. Jedes definierte Menü kann als Untermenü hinzugefügt werden. Mit dem Hinzufügen eines Untermenüs haben Sie die Möglichkeit, Transaktionen als Gruppe hinzuzufügen, sodass Sie nicht jede Transaktion einzeln hinzufügen müssen. Dies ist ein weiterer Schritt der Definition eine Menüstruktur mit mehreren Ebenen.

Untermenüs werden im Popup "Menüelemente" (MenuItems) definiert.

Sie können Untermenüsymbole ändern, indem Sie im Popup "MenuItems" einen Namen für die Symboldatei in das Feld "ApolloIcon" eingeben. Symboldateien befinden sich im folgenden Verzeichnis: C:\Program Files (x86)\Camstar\Camstar Portal\assets\image.

Sie müssen die folgenden Standards verwenden, wenn Sie eine Symboldatei im Feld "ApolloIcon" hinzufügen:

• Die Datei muss eine SVG-Datei sein, die sich im Ordner befindet.
• Die Datei muss eine Breite von 24 Pixel aufweisen, was sich im Dateinamen durch die Zahl 24 widerspiegelt.
• Beim Eingeben des Dateinamens muss die Groß-/Kleinschreibung beachtet werden.
• Der Dateiname muss ohne Bildgröße und Dateierweiterung eingegeben werden.

Wenn z. B. der Name der Symboldatei im Ordner "cmdAttach24.svg" lautet, müssen Sie "cmdAttach" in das Feld "ApolloIcon" eingeben.

Note: Das Hinzufügen benutzerdefinierter Untermenüsymbole wird von Siemens nicht unterstützt.

Vorgehensweise zum Hinzufügen eines Untermenüs zu einer Menüdefinition

Gehen Sie folgendermaßen vor, um ein Untermenü zu einer Menüdefinition hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Wählen Sie die Registerkarte Navigation aus.
4. Erweitern Sie Menüs (Menus).
5. Doppelklicken Sie auf ein Menü. Das Menü wird im Arbeitsbereich angezeigt.
6. Klicken Sie auf Mehr (More) für Menü-Elemente. Das Popup "Menüelemente" (MenuItems) wird angezeigt.
7. Wählen Sie Untermenüänderungen im Portalmenü (PortalMenuSubMenuChanges) aus der Dropdownliste aus und klicken Sie auf Hinzufügen (Add). In der Liste wird ein neues Untermenü und im rechten Bereich werden Felder angezeigt.
8. Geben Sie im Feld Bezeichnung (Caption) eine Bezeichnung für das Untermenü ein.
9. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
10. Wiederholen Sie die Schritte 7 bis 9 für jedes Untermenü, das Sie hinzufügen möchten.
11. Klicken Sie auf OK. Das Popup "Menüelemente" (MenuItems) wird geschlossen.
12. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Hinzufügen eines Untermenüsymbols

Gehen Sie folgendermaßen vor, um ein Untermenüsymbol hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Wählen Sie die Registerkarte Navigation im Komponentenkatalog aus und doppelklicken Sie anschließend auf ein Menü. Das Menü wird im Arbeitsbereich angezeigt.
3. Klicken Sie auf Mehr (More) für Menü-Elemente. Das modale Fenster "MenuItems" wird angezeigt.
4. Wählen Sie PortalMenuSubMenuChanges aus der Dropdown-Liste und ein Untermenü aus. Die Felder des Untermenüs werden im rechten Bereich angezeigt.
5. Geben Sie einen Symboldateinamen in das Feld ApolloIcon ein, das der Symbolbilddatei im Verzeichnis C:\Program Files (x86)\Camstar\Camstar Portal\assets\image entspricht.
6. Klicken Sie auf OK. Das modale Fenster "MenuItems" wird geschlossen.
7. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
