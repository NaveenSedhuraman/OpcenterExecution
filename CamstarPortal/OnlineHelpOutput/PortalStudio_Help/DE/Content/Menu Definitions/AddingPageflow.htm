Hinzufügen einer Seitenfolge zu einer Menüdefinition

Seitenfolgen (Pageflows) werden verwendet, um Seiten miteinander zu verknüpfen und so die gewünschte Seitenabfolge zu erstellen. Weitere Informationen finden Sie unter "Erstellen einer Seitenfolge".

Beim Hinzufügen einer Seitenfolge zu einer Menüdefinition

Im Popup "Menüelemente" (MenuItems) können Seitenfolgen zu Menüdefinitionen hinzugefügt werden. Weitere Informationen zu den Feldern im Popup "Menüelemente" (MenuItems) finden Sie unter "Hinzufügen von Menüelementen zu einer Menüdefinition".

Vorgehensweise zum Hinzufügen einer Seitenfolge zu einer Menüdefinition

Gehen Sie folgendermaßen vor, um eine Seitenfolge zu einer Menüdefinition hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Wählen Sie die Registerkarte Navigation aus.
4. Erweitern Sie Menüs (Menus).
5. Doppelklicken Sie auf ein Menü. Das Menü wird im Arbeitsbereich angezeigt.
6. Klicken Sie auf Mehr (More) für Menü-Elemente. Das Popup "Menüelemente" (MenuItems) wird angezeigt.
7. Wählen Sie aus der Dropdownliste Änderungen von Portal-Menübefehlen (PortalMenuCommandChanges) aus und klicken Sie auf Hinzufügen (Add). In der Liste wird ein neues Menüelement und im rechten Bereich werden Felder angezeigt.
8. Geben Sie im Feld Bezeichnung (Caption) eine Bezeichnung für das Menüelement ein.
9. Wählen Sie im Feld Seitenfolge (Pageflow) eine Seitenfolge aus.
10. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
11. Klicken Sie auf OK. Das Popup "Menüelemente" (MenuItems) wird geschlossen.
12. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
