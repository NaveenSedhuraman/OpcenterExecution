Hinzufügen einer angegebenen Registerkarte zu einer Menüdefinition

In Portal Studio können Sie angeben, auf welcher Registerkarte eine Seite geöffnet wird. Registerkarten werden von Menüelementen referenziert. Erstellen Sie daher zuerst die Menüs, bevor Sie Registerkarten erstellen.

Vorgehensweise zum Hinzufügen einer angegebenen Registerkarte zu einer Menüdefinition

Gehen Sie folgendermaßen vor, um eine angegebene Registerkarte zu einer Menüdefinition hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Erstellen Sie eine neue oder öffnen Sie eine vorhandene Menüdefinition.
3. Wählen Sie im Feld Portal-Registerkarten-Option (PortalTabOption) den Eintrag Angegebene Registerkarte (SpecifiedTab) aus.
4. Wählen Sie im Feld Portal-Registerkarte (Portal Tab) eine Registerkarte aus.
5. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
