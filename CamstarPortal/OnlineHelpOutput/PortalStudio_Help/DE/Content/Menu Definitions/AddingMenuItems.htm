Hinzufügen von Menüelementen zu einer Menüdefinition

Menüelemente sind die Seiten, auf die ein Endbenutzer über das Menü zugreifen kann. Menüelemente werden im Popup "Menüelemente" (MenuItems) definiert.

Felddefinitionen des Popups "Menüelemente" (MenuItems)

Feld

Definition

Typ

Caption

Eindeutiger Name für das Menüelement.

Erforderlich

LabelName

Bestimmt den Text, der auf dem Menüelement angezeigt wird.

Optional

PageDisplay

Bestimmt, ob das Menüelement auf der aktuellen Registerkarte, auf einer neuen Registerkarte oder in einem neuen Browserfenster angezeigt wird.

Optional

PageFlow

Gibt eine Seitenfolge für das Menüelement an.

Optional

PageURL

Gibt eine Seiten-URL für das Menüelement an.

Hinweis: Sie können statische Seiten zu einem Menüelement hinzufügen, indem Sie einen relativen Pfad zu einer benutzerdefinierten ASPX-Seitendatei eingeben. Statische Seiten werden direkt aus dem Web gerendert und unterstützen keine Webparts. Statische Seiten werden im Unterordner "Statische Seiten" (StaticPages) im Camstar Portal-Verzeichnis gespeichert.

Optional

PortalTab

Gibt eine Portal-Registerkarte an.

Hinweis: Dieses Feld wird angezeigt, wenn im Feld "Portal-Registerkarten-Option" (PortalTabOption) der Eintrag "Angegebene Registerkarte" (SpecifiedTab) ausgewählt ist.

Optional

PortalTabOption

Gibt an, wie das Menüelement eine Registerkarte referenziert. Weitere Informationen finden Sie unter "Erstellen einer Menüdefinition".

Optional

QueryString

Gibt eine Abfrage für das Menüelement an.

Optional

Sequence

Bestimmt die Reihenfolgeposition des Menüelements.

Optional

ServiceName

Gibt einen Service für das Menüelement an.

Optional

VirtualPage

Gibt eine virtuelle Seite für das Menüelement an.

Optional

Vorgehensweise zum Hinzufügen von Menüelementen zu einer Menüdefinition

Gehen Sie folgendermaßen vor, um Menüeinträge zu einer Menüdefinition hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Wählen Sie die Registerkarte Navigation aus.
4. Erweitern Sie Menüs (Menus).
5. Doppelklicken Sie auf ein Menü. Das Menü wird im Arbeitsbereich angezeigt.
6. Klicken Sie auf Mehr (More) für Menü-Elemente. Das Popup "Menüelemente" (MenuItems) wird angezeigt.
7. Wählen Sie aus der Dropdownliste Änderungen von Portal-Menübefehlen (PortalMenuCommandChanges) aus und klicken Sie auf Hinzufügen (Add). In der Liste wird ein neues Menüelement und im rechten Bereich werden Felder angezeigt.
8. Geben Sie im Feld Bezeichnung (Caption) eine Bezeichnung für das Menüelement ein.
9. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
10. Wiederholen Sie die Schritte 7-9 für jedes Menüelement, das Sie hinzufügen möchten.
11. Klicken Sie auf OK. Das Popup "Menüelemente" (MenuItems) wird geschlossen.
12. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
