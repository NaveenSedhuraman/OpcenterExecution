Erstellen einer Menüdefinition

Sie können eine Struktur aus mehreren Menüs und Untermenüs erstellen, die auf die Anforderungen Ihrer Organisation abgestimmt ist.

Beim Erstellen einer Menüdefinition

Wenn Sie eine neue Menüdefinition erstellen, werden im Arbeitsbereich in einem Popup Menüdefinitionen als Gruppe von Feldern angezeigt.

Beachten Sie beim Erstellen einer Menüdefinition Folgendes:

• Die Standardmenüs "csiPortalMenuV8", "csiMobileMenu" und "csiPortalMenu" werden den Benutzern "Administrator", "CamstarAdmin" und "InSite-Admin" automatisch in einer anfänglichen Implementierung von CEP oder beim Aktualisieren auf eine neue Version und Aktualisieren der Standardinstanzdaten zugewiesen. Verwenden Sie das Modellierungsobjekt "Mitarbeiter", um Portalmenüs anderen Mitarbeitern zuzuweisen. Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch oder im Opcenter Execution Core Modeling-Benutzerhandbuch.
• Sie sollten erst Registerkarten und anschließend Menüs erstellen, weil die Menüs auf Registerkarten verweisen. Weitere Informationen finden Sie unter "Erstellen einer Registerkarte".

Sie haben drei verschiedene Möglichkeiten, um Registerkarten in Menüs zu referenzieren. Diese Tabelle beschreibt die verschiedenen Möglichkeiten zum Referenzieren von Registerkarten.

Registerkartenreferenz

Beschreibung

Inherit

Erbt die Registerkarteneinstellung von den Eltern. Die ist die Standardeinstellung.

New Tab

Es gibt kein definiertes Registerkartenobjekt. Daher wird es immer in einer neuen, leeren Registerkarte im Portal geöffnet. Bei Auswahl dieser Option können Sie mehrere Kopien gleichzeitig öffnen.

Specified Tab

Geben Sie eine bereits erstellte Registerkarte an. Es kann jeweils nur eine Version einer angegebenen Registerkarte gleichzeitig geöffnet sein.

Wenn Sie ein Menü oder ein Untermenü einer angegebenen Registerkarte zuweisen und für alle nachstehenden Elemente den Standardwert "Erben" (Inherit) beibehalten, werden diese alle auf derselben angegebenen Registerkarte geöffnet. Daher ersetzt die zuletzt geöffnete Seite die Seite, die derzeit auf der angegebenen Registerkarte offen ist.

Vorgehensweise zum Erstellen einer Menüdefinition

Gehen Sie folgendermaßen vor, um eine Menüdefinition zu erstellen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Neu (New) und wählen Sie Menüdefinition (Menu Definition) aus. Im Arbeitsbereich wird eine leere Menüdefinition angezeigt.
3. Geben Sie im Feld Name einen Namen für die Menüdefinition ein.
4. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
5. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
