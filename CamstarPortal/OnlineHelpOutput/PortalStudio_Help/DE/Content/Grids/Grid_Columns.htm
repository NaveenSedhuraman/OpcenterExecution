Tabellenspalten

Im Popup "<Tabellenname> – Spalten" (Columns) können Sie Standardspalten zu Tabellen hinzufügen. Das Popup "<Tabellenname> – Spalten" (Columns) wird angezeigt, wenn Sie im Popup "Tabelleneinstellungseditor" (Grid Settings Editor) auf "Mehr" (More) klicken. Informationen zu den Tabelleneinstellungen finden Sie unter "Tabelleneinstellungseditor (Grid Settings Editor)".

Spalteneigenschaften werden je nach dem derzeit ausgewählten Spaltentyp dynamisch angezeigt.

Felddefinitionen des Popups "<Tabellenname> – Spalten" (Columns)

Diese Tabelle enthält eine Liste der Tabelleneinstellungen im Popup "<Tabellenname> – Spalten" (Columns).

Eigenschaft Spalteneinstellungen

Beschreibung der Eigenschaft

BindPath

Relativer Eltern-Pfad oder -Feldname, der die Daten enthält.

CellActions

Zeigt das Popup "<Spaltenname> – Zellenaktionen" (CellActions) an. Im Popup "<Spaltenname> – Zellenaktionen" (CellActions) können Sie weitere Zellenaktionen definieren, z. B. Schaltflächen, Flyouts und Verknüpfungen.

CellStyle

Legt den Zellenstil der Spalte fest.

CustomScriptHandlers

Zeigt das Popup "<Spaltenname> – Benutzerdefinierte Skripthandler" (CustomScriptHandlers) an. Im Popup "<Spaltenname> – Zellenaktionen" (CellActions) können Sie eine benutzerdefinierte Skripthandler-Aktion, z. B. Klicken, Ändern und "Beim Hinzufügen von Zellenattributen" (onAddCellAttributes) festlegen.

DateFormat

Legt das akzeptierte Format für Datum und Uhrzeit fest.

DateFormatExample

Zeigt ein Beispiel eines akzeptierten Datumsformats an.

DefaultValue

Legt einen Standardwert für Einträge in der Spalte fest.

Editable

Legt die Spalte als bearbeitbar fest.

EditorProperties

Reihe von Untereigenschaften, die festlegen, wie Daten an die und von der Spalte übergeben werden.

Editoreigenschaften - Allgemein (EditorProperties – General)

FieldExpressions

Legt einen Bind-Pfad zum Geschäftslogik-Objektmodell fest, um Daten abzurufen und/oder einzureichen.

RequestData

Legt die Anforderung für Daten eines Ereignisses fest.

RetrieveList

Legt fest, wann Werte abgerufen werden.

SelectionData

Legt die Anfrage für die Auswahldaten eines Ereignisses fest.

ValueFilter

Filterauswahlelemente auf Basis des Werts eines anderen Objekts.

Frozen

Friert Spalten ein, die sichtbar bleiben, während ein Bildlauf für die übrige Tabelle durchgeführt wird. Weitere Informationen finden Sie unter "Einfrieren von Spalten".

GroupByColumnName

Verweist auf das Steuerelement "Benanntes Unterobjekt" (Named Subentity), um eine Gruppierung nach Spaltenname durchzuführen.

HeaderStyle

Legt den Kopfzeilenstil der Spalte fest.

Hidden

Blendet die Spalte aus.

HiddenContent

Blendet Inhalte der Spalte aus.

HTMLTagTransform

Legt fest, wie HTML-Tags konvertiert oder beibehalten werden. Diese Eigenschaft enthält die folgenden Optionen zum Transformieren von HTML-Tags:

• Auto – Durchsucht Text automatisch nach HTML-Tags ("<TAG>"), behält den HTML-Code bei und überträgt ihn, während gleichzeitig die Inhalte formatiert werden können. Diese Option ist standardmäßig aktiviert.
• Konvertieren (Convert) – Ersetzt die Zeichen "<" und ">" durch "&lt;" und "&gt;". Ersetzt die Zeichen "\" und "/" durch "&#x22;" und "&#x2F;". Ersetzt das Zeichen " ' " durch "&#x27;". Diese Option zeigt die in den HTML-Tags enthaltenen Inhalte an und deaktiviert die HTML-Formatierung. Diese Option ist standardmäßig aktiviert.
• Beibehalten (Preserve) – Behält den Text genau wie in der Datenbank vorhanden bei und zeigt alle HTML-Formatierungen an.

IsRowID

Gibt die Spalte als Zeilenidentifikator an. Es kann nur eine Spalte als Zeilen-ID definiert werden.

LabelName

Bestimmt die Bezeichnung, die auf der Spalte angezeigt wird. Weitere Informationen zum Suchen vordefinierter Beschriftungen finden Sie unter "Suchen nach Seitenaktionsbeschriftungen".

LabelText

Bestimmt die Bezeichnung, die auf der Spalte angezeigt wird.

ListDisplayExpression

Liste der Auswahlwerte für Anzeigeausdrücke.

ListValueColumn

Ein Objektmodellausdruck, der den Ursprung des Spaltenwerts angibt.

Name

Name der Spalte.

Erforderlich

Es muss ein Wert in die Spalte eingegeben werden.

Resizable

Ermöglicht die Anpassung der Größe der Spalte.

Sortable

Ermöglicht die Sortierung der Spalte.

TypeFormat

Legt den Typ entweder als "Unterobjekt" (Subentity) oder als "Benanntes Unterobjekt" (Named Subentity) fest.

TypeValidation

Legt fest, ob die Anwendung den ausgewählten Typ validiert.

UseCurrentTime

Legt fest, dass die Spalte das aktuelle Zeitformat verwendet.

ValueExpression

Textfeld mit Verknüpfung zur Feldausdruckerstellung.

ValueFormat

Legt das Werteformat entweder als Name oder als ID fest.

Width

Breite der Spalte in Pixel.
