Tabelleneinstellungseditor (Grid Settings Editor)

Im Popup "Tabelleneinstellungseditor" (Grid Settings Editor) in Portal Studio haben Sie die Möglichkeit, die Tabelleneigenschaften zu ändern. Welche Felder im Popup "Tabelleneinstellungseditor" angezeigt werden, hängt davon ab, welchen Typ von Tabellendatenmodus Sie im Bereich "Eigenschaften" (Properties) auswählen.

Note: Wenn Sie im Bereich "Eigenschaften" im Feld "Tabellendatenmodus" (GridDataMode) die Option "Generisch" (Generic) auswählen, werden alle Felder im Tabelleneinstellungseditor eingestellt. Siemens empfiehlt, je nach der Funktion der Tabelle einen anderen Tabellendatenmodus als "Generisch" auszuwählen.

AddRowMode

Bestimmt, wie neue Zeilen zu einer Tabelle hinzugefügt werden können:

• Schaltfläche hinzufügen (AddButton) – Zeilen können nur durch Klicken auf die Schaltfläche "Neue Zeile hinzufügen" (Add new row) hinzugefügt werden.
• Auswahl einer leeren Zeile (EmptyRowSelection) – Zeilen können nur durch Klicken in eine leere Zeile hinzugefügt werden. Bei Auswahl dieser Option wird die Schaltfläche "Neue Zeile hinzufügen" (Add new row) in der Tabelle ausgeblendet.
• Beides (Both) – Zeilen können durch Klicken auf die Schaltfläche "Neue Zeile hinzufügen" (Add new row) oder durch Klicken in eine leere Zeile hinzugefügt werden.

Die Standardeinstellung ist "Beides" (Both).

Hinweis: Diese Eigenschaft wirkt sich nur im Inlinebearbeitungsmodus auf Tabellen aus.

Vorgehensweise zum Zugreifen auf den Tabelleneinstellungseditor

Gehen Sie folgendermaßen vor, um den Tabelleneinstellungseditor (Grid Settings Editor) zu öffnen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Doppelklicken Sie auf eine beliebige Portalseite oder ein Webpart, die ein Tabellen-Steuerelement enthalten. Die Seite bzw. das Webpart wird im Arbeitsbereich angezeigt.
4. Klicken Sie auf Eigenschaften anzeigen/ausblenden (Show/Hide Properties), wenn der Bereich "Eigenschaften" (Properties) ausgeblendet ist. Der Bereich "Eigenschaften" (Properties) wird angezeigt.
5. Wählen Sie ein Tabellensteuerelement (Grid Control) aus. Die Einstellungen für die Tabelle werden im Bereich "Eigenschaften" (Properties) angezeigt.
6. Klicken Sie im Bereich "Eigenschaften" (Properties) auf Mehr (More) für Einstellungen (Settings). Das Popup "Tabelleneinstellungseditor" (Grid Settings Editor) wird angezeigt.
Note: Sie können auch eine Tabelle auswählen und die Taste "g" drücken, um das Popup "Tabelleneinstellungseditor" (Grid Settings Editor) anzuzeigen.
