Erstellen einer Registerkarte

Sie können eine Registerkarte (Tab) erstellen und dann in einem Menü angeben, dass eine Seite auf dieser Registerkarte geöffnet wird.

Beim Erstellen einer Registerkarte

Sie können ein Symbol auswählen, um es als Indikator auf der Registerkarte zu verwenden, wenn Sie diese im Menü anzeigen. Verwenden Sie zum Hochladen von Bildern das Symbol "Bild" (Image) im Feld "Symbol-URL" (IconUrl). Die Schaltfläche zeigt ein Miniaturbild an, wenn der Registerkarte bereits ein Bild zugewiesen wurde. Hochgeladene Bilder werden Teil der Bildbibliothek von Portal Studio und sind für alle Registerkarten verfügbar.

Sie können einen Teil eines Bilddateinamens in das Textfeld "IconUrl" eingeben, um Bilder auszugeben, deren Name mit einem Teil des Bildnamens in der Dropdownliste übereinstimmt.

Der Prozess zum Hochladen und Zuweisen von Bildern zu Registerkarten ähnelt dem Prozess zum Hochladen und Zuweisen von Bildern zu Steuerelementen. Weitere Informationen zu Prozeduren zum Hochladen und Zuweisen von Bildern finden Sie unter "Hochladen und Zuweisen von Bildern".

Felddefinitionen des Popups "Portal-Registerkarte" (Portal Tab)

Diese Tabelle definiert die Felder im Popup "Portal-Registerkarte" (Portal Tab).

Feld

Definition

Typ

IconUrl

Legt ein Symbol fest, das als Indikator auf der Registerkarte verwendet werden soll. Symbole können entweder in dieses Feld hochgeladen oder aus der Bildbibliothek von Portal Studio ausgewählt werden.

Optional

Name

Eindeutiger Name für die Registerkarte.

Erforderlich

Vorgehensweise zum Erstellen einer Registerkarte

Gehen Sie folgendermaßen vor, um eine Registerkarte zu erstellen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Neu (New) und wählen Sie Registerkarte (Tab) aus. Im Arbeitsbereich wird eine leere Registerkarte angezeigt.
3. Geben Sie im Feld Name einen Namen für die Registerkarte ein.
4. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
5. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
