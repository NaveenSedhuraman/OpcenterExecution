Hinzufügen von Steuerelementen zum Suchen von Dateien

Mit Portal Studio können Sie Portalseiten Steuerelemente zum Suchen von Dateien (File Browse) hinzufügen. Mit Steuerelementen zum Suchen von Dateien können Sie nach einer Datei suchen und diese auf eine Portalseite hochladen.

Beim Hinzufügen von Steuerelementen zum Suchen von Dateien

Steuerelemente zum Suchen von Dateien (File Browse) enthalten zwei Felder: "Suchmodus" (Browse Mode) und "Datei suchen" (Locate File).

Ein Webpart ist erforderlich, um einer Seite ein Steuerelement hinzuzufügen. Weitere Informationen zum Erstellen von Webparts finden Sie unter "Erstellen eines Webparts".

Important: Ein Steuerelement zum Suchen von Dateien (File Browse) ist für Modellierungsseiten erforderlich, auf denen Dateien hochgeladen werden können.

Vorgehensweise zum Hinzufügen von Steuerelementen zum Suchen von Dateien

Gehen Sie folgendermaßen vor, um ein Steuerelement zum Suchen von Dateien hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Doppelklicken Sie auf eine Seite oder ein Webpart. Die Seite bzw. das Webpart wird im Arbeitsbereich angezeigt.
4. Klicken Sie auf Toolbox anzeigen/ausblenden (Show/Hide Toolbox), wenn die Toolbox ausgeblendet ist. Die Toolbox wird angezeigt.
5. Klicken Sie in der Toolbox auf ein Steuerelement zum Suchen von Dateien (File Browse) und ziehen Sie es auf eine Zelle eines Webparts. Das Steuerelement zum Suchen von Dateien (File Browse) wird im Webpart angezeigt.
6. Klicken Sie auf Eigenschaften anzeigen/ausblenden (Show/Hide Properties), wenn der Bereich "Eigenschaften" (Properties) ausgeblendet ist. Der Bereich "Eigenschaften" (Properties) wird angezeigt.
7. Geben Sie im Bereich "Eigenschaften" (Properties) im Feld Name einen Namen für das Steuerelement zum Suchen von Dateien (File Browse) ein.
8. Wiederholen Sie die Schritte 5 bis 7, um weitere Steuerelemente zum Suchen von Dateien hinzuzufügen.
9. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
