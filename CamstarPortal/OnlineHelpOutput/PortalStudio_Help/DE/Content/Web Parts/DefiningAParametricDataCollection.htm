Definieren einer Sammlung parametrischer Daten

Sie können eine Sammlung parametrischer Daten (Parametric Data Collection) mithilfe eines Webparts definieren.

Eine Datensammlungsdefinition referenziert eine parametrische Datendefinition, die eine Gruppe von Datensammlungsparametern enthält. Die Datensammlungsparameter werden Benutzern während der Bearbeitung von Transaktionen angezeigt. Die Datensammlungsdefinition und die Transaktion, mit der diese verwendet wird, werden in der Spezifikationsdefinition (in der Modellierung) angegeben, die mit dem Schritt verbunden ist, in dem die Daten erfasst werden.

Beim Definieren einer Sammlung parametrischer Daten

Datensammlungen müssen in der Modellierung definiert werden, bevor sie in Portal Studio verwendet werden. Weitere Informationen hierzu finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch bzw. im Opcenter Execution Core Modeling-Benutzerhandbuch und im Opcenter Execution Medical Device and Diagnostics Data Collection-Benutzerhandbuch bzw. im Opcenter Execution Core Data Collection-Benutzerhandbuch.

Vorgehensweise zum Definieren einer Sammlung parametrischer Daten für ein Webpart

Gehen Sie folgendermaßen vor, um eine Sammlung parametrischer Daten für ein Webpart zu definieren:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Neu (New) und wählen Sie Webpart aus. Im Arbeitsbereich wird ein leeres Webpart angezeigt.
3. Klicken Sie auf Eigenschaften anzeigen/ausblenden (Show/Hide Properties), wenn der Bereich "Eigenschaften" (Properties) ausgeblendet ist. Der Bereich "Eigenschaften" (Properties) wird angezeigt.
4. Geben Sie im Bereich Eigenschaften (Properties) im Feld Name einen Namen für das Webpart ein.
5. Klicken Sie auf Toolbox anzeigen/ausblenden (Show/Hide Toolbox), wenn die Toolbox ausgeblendet ist. Die Toolbox wird angezeigt.
6. Klicken Sie auf die Registerkarte Feldverzeichnis (Fields Directory). Im Toolboxbereich wird eine Liste von Services angezeigt.
7. Wählen Sie das Feld aus, das Sie hinzufügen möchten, und ziehen Sie es auf eine Zelle auf dem Webpart.
8. Wiederholen Sie Schritt 7 für alle Felder in der Datensammlung.
9. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
