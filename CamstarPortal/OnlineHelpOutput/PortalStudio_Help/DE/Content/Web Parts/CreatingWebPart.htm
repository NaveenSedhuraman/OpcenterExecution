Webparts

Webparts werden verwendet, um eine Seite zu erstellen. Sie können Webparts unabhängig von einer Seite speichern, sodass sie auf unterschiedlichen Seiten wiederverwendet werden können. Der Containerkopf ist ein gutes Beispiel für ein Webpart, das auf mehreren Seiten verwendet wird.

• Kopieren und Einfügen von Steuerelementen
• Hinzufügen von Steuerelementen zum Suchen von Dateien
