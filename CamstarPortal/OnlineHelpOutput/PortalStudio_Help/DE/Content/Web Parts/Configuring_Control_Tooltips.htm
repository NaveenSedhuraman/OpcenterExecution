Konfigurieren von QuickInfos für Steuerelemente

Wenn Sie mit dem Mauszeiger über Steuerelemente auf Seiten im Portal fahren, werden QuickInfos zu den Steuerelementen angezeigt. QuickInfos können Sie unter Verwendung von Text und Beschriftungen in Portal Studio konfigurieren.

Beim Konfigurieren von QuickInfos für Steuerelemente

QuickInfos für Steuerelemente werden im Bereich "Eigenschaften" (Properties) konfiguriert. Die Eigenschaft "QuickInfoText" (ToolTipText) erstellt eine QuickInfo mit dem Text, den Sie in das Feld "ToolTipText" eingeben. Die Eigenschaft "QuickInfoBeschriftung" (ToolTipLabel) öffnet das Popup "Beschriftungssuche" (Label Lookup) zum Suchen und Auswählen einer Beschriftung, die als QuickInfo verwendet werden soll. Weitere Informationen zum Popup "Beschriftungssuche" (Label Lookup) finden Sie unter "Suchen nach Seitenaktionsbeschriftungen".

Die Eigenschaft "QuickInfoText" (ToolTipText) hat Vorrang vor der Eigenschaft "QuickInfoBeschriftung" (ToolTipLabel), wenn sowohl Text eingegeben als auch eine Beschriftung ausgewählt wird.

Vorgehensweise zum Konfigurieren von QuickInfos für Steuerelemente

Gehen Sie folgendermaßen vor, um QuickInfos für Steuerelemente zu konfigurieren:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Doppelklicken Sie auf eine Seite. Die Seite wird im Arbeitsbereich angezeigt.
4. Wählen Sie auf der Seite ein Steuerelement aus. Die Eigenschaften für das Steuerelement werden im Bereich "Eigenschaften" (Properties) angezeigt.
5. Möchten Sie für die QuickInfo einen Text eingeben oder eine Beschriftung auswählen?

Sie möchten . . .

Dann . . .

Einen Text eingeben
a. Geben Sie den Text für die QuickInfo im Bereich "Eigenschaften" (Properties) in das Feld QuickInfoText (ToolTipText) ein.
b. Gehen Sie zu Schritt 6.

Eine Beschriftung auswählen
a. Klicken Sie auf die Schaltfläche Mehr (More) für QuickInfoBeschriftung (ToolTipLabel). Das Popup "Beschriftungssuche" (Label Lookup) wird angezeigt.
b. Geben Sie ein oder mehrere Suchkriterien ein, um Ihre Suche einzugrenzen.
c. Klicken Sie auf Suchen (Search). Die Anwendung zeigt die Beschriftungen, die den Suchkriterien entsprechen, in der Tabelle an.
d. Wählen Sie eine Beschriftung aus. Der Standardtext und der lokale Text für die Beschriftung werden in den Feldern "Standardtext" (Default Text) und "Lokaler Text" (Local Text) angezeigt.
e. Klicken Sie auf OK. Das Popup "Beschriftungssuche" (Label Lookup) wird geschlossen und die Informationen für die Beschriftung werden im Feld "QuickInfoBeschriftung" (ToolTipLabel) angezeigt.
f. Gehen Sie zu Schritt 6.
6. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
