Aktivieren des Schalters zum Speichern für die Änderungsverwaltung (Change Mgt Save)

In Portal Studio können Sie Modellierungsseiten ein Steuerelement des Typs "Boolescher Schalter" hinzufügen, das in der Modellierung als Schalter "Änderungsmanagement-Speicherung" angezeigt wird. Mit dem Schalter "Änderungsmanagement-Speicherung" können Modellierungsbenutzer die Option ein- und ausschalten, um Änderungsverwaltungspaketen in einer Sitzung erstellte Modellierungsinstanzen zuzuweisen.

Beim Aktivieren des Schalters zum Speichern für die Änderungsverwaltung (Change Mgt Save)

Steuerelemente des Typs boolescher Schalter werden standardmäßig als Optionsfelder angezeigt. Damit Steuerelemente des Typs boolescher Schalter als Schalter zum Speichern für die Änderungsverwaltung (Change Mgt Save) angezeigt werden, muss im Bereich "Eigenschaften" (Properties) die Eigenschaft "Anzeigemodus boolescher Schalter" (BooleanSwitchDisplayMode) auf "Schieben" (Slide) festgelegt sein.

Der Schalter "Änderungsmanagement-Speicherung" ist in der Modellierung auf Modellierungsseiten ausgeblendet, bis ein Änderungsverwaltungspaket erstellt wurde. Weitere Informationen zum Erstellen von Paketen finden Sie im Opcenter Execution Medical Device and Diagnostics Change Management-Benutzerhandbuch oder im Opcenter Execution Core Change Management-Benutzerhandbuch.

Sie können Steuerelemente des Typs boolescher Schalter nur zu einem Webpart hinzufügen. Sie können die Steuerelemente nicht direkt zu einer Seite hinzufügen, stattdessen aber zu einem Webpart, das sich auf einer Seite befindet. Weitere Informationen finden Sie unter "Hinzufügen eines Steuerelements zu einem Webpart".

Sie können den Text in der linken und rechten Zelle des Schalters für die Änderungsverwaltung (Change Mgt) anpassen. Standardmäßig wird in der linken Zelle der Text "Aus" und in der rechten Zelle der Text "Ein" angezeigt. Mit der CSS-Klasse können Sie auch die Farbe und den Schrifttyp der Zellen anpassen.

Note: Benutzer der Modellierung müssen über eine Berechtigung zum Zuweisen von Inhalten zu einem Änderungsverwaltungspaket verfügen, damit sie den Schalter "Änderungsmanagement-Speicherung" verwenden können. Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch oder im Opcenter Execution Core Modeling-Benutzerhandbuch.

Vorgehensweise zum Aktivieren des Schalters zum Speichern für die Änderungsverwaltung (Change Mgt Save)

Gehen Sie folgendermaßen vor, um den Schalter zum Speichern für die Änderungsverwaltung (Change Mgt Save) zu aktivieren:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Doppelklicken Sie auf eine Modellierungsseite (Modeling). Die Modellierungsseite (Modeling) wird im Arbeitsbereich angezeigt.
4. Klicken Sie auf Toolbox anzeigen/ausblenden (Show/Hide Toolbox), wenn die Toolbox ausgeblendet ist. Die Toolbox wird angezeigt.
5. Klicken Sie auf ein Steuerelement des Typs boolescher Schalter (Boolean Switch) und ziehen Sie es auf eine Zelle auf einem Webpart. Das Steuerelement des Typs boolescher Schalter (Boolean Switch) wird in dem Webpart angezeigt.
6. Klicken Sie auf Eigenschaften anzeigen/ausblenden (Show/Hide Properties), wenn der Bereich "Eigenschaften" (Properties) ausgeblendet ist. Der Bereich "Eigenschaften" (Properties) wird angezeigt.
7. Geben Sie im Bereich "Eigenschaften" (Properties) im Feld Name einen Namen für das Steuerelement des Typs boolescher Schalter ein.
8. Wählen Sie im Bereich "Eigenschaften" (Properties) im Feld Anzeigemodus boolescher Schalter (BooleanSwitchDisplayMode) den Eintrag Schieben (Slide) aus.
9. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
