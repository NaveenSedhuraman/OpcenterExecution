Erstellen eines Webparts

In Portal Studio können Sie Webparts zur Verwendung auf mehreren Seiten erstellen.

Beim Erstellen eines Webparts

Sie können Felder und Steuerelemente direkt auf ein Webpart, nicht jedoch direkt auf eine Seite ziehen.

Nach dem Speichern eines Webparts können Sie mit einem Klick auf die Schaltfläche "Ausführen" (Run) eine Vorschau des Webparts anzeigen.

Siemens empfiehlt, im Bereich "Eigenschaften" (Properties) einen Namen, einen Titel und eine Titelbeschriftung einzugeben, wenn Sie ein Webpart erstellen. Das Feld Name definiert den aktuellen Namen des Webparts auf der Seite. Das Feld "Titel" (Title) definiert die Beschriftung, die für das Webpart angezeigt wird. Das Feld "Titelbeschriftung" (TitleLabel) definiert die Beschriftung, die auf Datenbankebene referenziert wird.

Vorgehensweise zum Erstellen eines Webparts

Gehen Sie folgendermaßen vor, um ein Webpart zu erstellen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Neu (New) und wählen Sie Webpart aus. Im Arbeitsbereich wird ein leeres Webpart angezeigt.
3. Geben Sie im Bereich "Eigenschaften" (Properties) im Feld Name einen Namen für das Webpart ein.
4. Geben Sie im Bereich "Eigenschaften" (Properties) im Feld Titel die neue Beschriftung ein.
5. Geben Sie im Bereich "Eigenschaften" (Properties) im Feld Titelbeschriftung (TitleLabel) eine Beschriftung ein.
6. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
