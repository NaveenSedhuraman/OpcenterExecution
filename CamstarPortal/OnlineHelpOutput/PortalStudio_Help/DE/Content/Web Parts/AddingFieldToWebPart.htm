Hinzufügen eines Felds zu einem Webpart

Felder (Fields) werden verwendet, um Informationen in ein Webpart einzugeben. Wenn Sie dem Webpart z. B. ein Feld namens "Container" hinzufügen, geben Sie dem Benutzer damit die Möglichkeit, einen Container einzugeben.

Sie können ein Feld zu einem neuen oder vorhandenen Webpart oder zu einem Webpart, das bereits auf einer Webseite verwendet wird, hinzufügen.

Wenn Sie ein Feld anstelle eines Steuerelements hinzufügen, hat dies den Vorteil, dass der Feldausdruck und der Steuerelementtyp automatisch für Sie festgelegt werden.

Beim Hinzufügen eines Felds zu einem Webpart

Sie können Felder nur zu einem Webpart, nicht jedoch direkt zu einer Seite hinzufügen.

Felder mit dem Verknüpfungssymbol sind empfohlene Felder für den Service.

Eine rote Pinnnadel bedeutet, dass das betreffende Feld bereits in dem geöffneten Objekt verwendet wird.

Eine gelbe Pinnnadel bedeutet, dass ein Feld im betreffenden Abschnitt verwendet wird.

Vorgehensweise zum Hinzufügen eines Felds zu einem Webpart

Gehen Sie folgendermaßen vor, um ein Feld zu einem Webpart hinzuzufügen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Doppelklicken Sie auf ein Webpart oder auf eine Seite, die ein Webpart enthält. Das Webpart bzw. die Seite wird im Arbeitsbereich angezeigt.
4. Klicken Sie auf Toolbox anzeigen/ausblenden (Show/Hide Toolbox), wenn die Toolbox ausgeblendet ist. Die Toolbox wird angezeigt.
5. Klicken Sie auf die Registerkarte Feldverzeichnis (Fields Directory). Im Toolboxbereich wird eine Liste von Services angezeigt.
6. Erweitern Sie den Service, der das Feld enthält, das Sie hinzufügen möchten.
7. Wählen Sie ein Feld aus und ziehen Sie es auf eine Zelle des Webparts.
8. Wiederholen Sie die Schritte 6 bis 7, um weitere Felder zum Webpart hinzuzufügen.
9. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
