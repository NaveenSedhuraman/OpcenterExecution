Öffnen eines vorhandenen Webparts

Sie können vorhandene Webparts im Katalog der Benutzeroberflächenkomponenten auf der Registerkarte "Komponenten" (Components) über die Dropdownliste "Webparts" öffnen.

Beim Öffnen eines vorhandenen Webparts wird eine Statusanzeige geöffnet, die anzeigt, dass das Webpart geladen wird.

Vorgehensweise zum Öffnen eines vorhandenen Webparts

Gehen Sie folgendermaßen vor, um ein vorhandenes Webpart zu öffnen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Erweitern Sie Webparts.
4. Doppelklicken Sie auf ein Webpart. Das Webpart wird im Arbeitsbereich angezeigt.
