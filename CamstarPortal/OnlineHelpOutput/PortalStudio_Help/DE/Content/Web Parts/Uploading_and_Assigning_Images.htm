Hochladen und Zuweisen von Bildern

Sie können Bilder in Portal Studio hochladen und sie Steuerelementen zuweisen, die Bilder unterstützen, z. B. Schaltflächen.

Beim Hochladen und Zuweisen von Bildern

Die Steuerelemente "Bildsteuerelement" (ImageControl), "Schaltfläche" (Button), "FlyoutDropdown" und "Abschnitt" (SectionDropdown) unterstützen das Hochladen und Zuweisen von Bildern.

Verwenden Sie zum Hochladen von Bildern die Schaltfläche "Standardbild" (DefaultImage) im Bereich "Eigenschaften" (Properties). Auf der Schaltfläche werden ein Miniaturbild und ein Bildname angezeigt, falls dem Steuerelement bereits ein Bild zugewiesen wurde. Hochgeladene Bilder werden in die Bildbibliothek von Portal Studio übernommen und sind für alle Steuerelemente verfügbar, die Bilder unterstützen.

Wenn Sie einen Teil eines Bilddateinamens in das Textfeld "Standardbild" (DefaultImage) eingeben, werden in der Dropdownliste Bildnamen ausgegeben, die mit dem Teilbildnamen übereinstimmen.

Vorgehensweise zum Hochladen von Bildern

Gehen Sie folgendermaßen vor, um Bilder hochzuladen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Doppelklicken Sie auf eine Seite oder ein Webpart. Die Seite bzw. das Webpart wird im Arbeitsbereich angezeigt.
4. Klicken Sie auf Eigenschaften anzeigen/ausblenden (Show/Hide Properties), wenn der Bereich "Eigenschaften" (Properties) ausgeblendet ist. Der Bereich "Eigenschaften" (Properties) wird angezeigt.
5. Wählen Sie ein Steuerelement aus, das Bilder unterstützt. Die Eigenschaften für das Steuerelement werden im Bereich "Eigenschaften" (Properties) angezeigt.
6. Klicken Sie im Bereich "Eigenschaften" (Properties) auf das Symbol für Standardbild (DefaultImage). Das Popup "Öffnen" (Open) wird angezeigt.
7. Wählen Sie eine Bilddatei aus und klicken Sie auf Öffnen (Open). Das Popup "Öffnen" (Open) wird geschlossen; das Bild wird hochgeladen und dem Steuerelement zugewiesen.
8. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.

Vorgehensweise zum Zuweisen von Bildern

Gehen Sie folgendermaßen vor, um Bilder zuzuweisen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Doppelklicken Sie auf eine Seite oder ein Webpart. Die Seite bzw. das Webpart wird im Arbeitsbereich angezeigt.
4. Klicken Sie auf Eigenschaften anzeigen/ausblenden (Show/Hide Properties), wenn der Bereich "Eigenschaften" (Properties) ausgeblendet ist. Der Bereich "Eigenschaften" (Properties) wird angezeigt.
5. Wählen Sie ein Steuerelement aus, das Bilder unterstützt. Die Eigenschaften für das Steuerelement werden im Bereich "Eigenschaften" (Properties) angezeigt.
6. Wählen Sie im Bereich "Eigenschaften" (Properties) ein Bild aus der Dropdownliste Standardbild (DefaultImage) aus. Das Bild wird im Symbol, der Dateiname im Textfeld angezeigt.
7. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
