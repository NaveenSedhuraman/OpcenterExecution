Anzeigen von Webpart-Eigenschaften

In Portal Studio können Sie die Eigenschaften einer Sammlung von Webparts anzeigen. Weitere Informationen zu bestimmten Webpart-Eigenschaften finden Sie unter "Webpart-Eigenschaften".

Vorgehensweise zum Anzeigen von Webpart-Eigenschaften

Gehen Sie folgendermaßen vor, um Webpart-Eigenschaften anzuzeigen:

1. Greifen Sie auf Portal Studio zu.
2. Klicken Sie auf Katalog anzeigen/ausblenden (Show/Hide Catalog), wenn der Katalog der Benutzeroberflächenkomponenten ausgeblendet ist. Der Katalog der Benutzeroberflächenkomponenten wird angezeigt.
3. Doppelklicken Sie auf eine Seite. Die Seite wird im Arbeitsbereich angezeigt.
4. Wählen Sie eine Zelle aus, die ein Webpart enthält.
5. Klicken Sie im Bereich "Eigenschaften" (Properties) auf Webeigenschaften (WebProperties). Das Fenster "Webeigenschaften" (WebProperties) wird angezeigt.
