Container verbinden

Die "Verbinden"-Transaktion erstellt Eltern-Kind-Beziehungen zwischen Containern. Die Transaktion ermöglicht Ihnen, vorhandene Container unter einem einzelnen Elternteil zu gruppieren. Daraufhin gilt eine einmal für den Eltern-Container durchgeführte Aktion automatisch auch für den Kind-Container.

Beim Verbinden von Containern

Sie können einen Container (oder eine Gruppe von Containern) nur mit einem Eltern-Container verbinden. Wenn Sie einen Container verbinden möchten, der zur Zeit mit einem anderen Eltern-Container verbunden ist, müssen Sie diesen Container zunächst vom seinem aktuellen Elternteil trennen.

Der Eltern-Container muss bereits Kind-Container oder die Menge Null haben. Die angezeigte Anzahl (Menge) im Containerinformationen-Kopf wird aktualisiert, um die Anzahl an Kind-Containern anzugeben, die dem Eltern-Container hinzugefügt werden.

Es gibt zwei Tabellen auf der Verbinden-Seite:

• Tabelle "Ausstehende erlaubte Container" - Wenn Sie den Eltern-Container angeben, wird diese Tabelle automatisch mit allen Containern gefüllt, die für die Zuweisung an diesen Eltern-Container geeignet sind.
• Tabelle "Aktuell verbundene Container" - diese Tabelle bietet eine Liste aller Container, die aktuell mit dem Eltern-Container verbunden sind.

Alle Anzeigefelder im Containerinformationen-Kopf auf der Seite "Verbinden" sind leer, bis Sie einen Eltern-Container auswählen. Anzeigefelder werden im "Verbinden"-Popup automatisch gefüllt.

Die Schaltfläche "Zurücksetzen" löscht die Tabelle "Aktuell verbundene Container" und das Feld "Kommentare" auf der Seite und dem Popup "Verbinden". Sie löscht außerdem den Containerinformationen-Kopf auf der Seite "Verbinden".

Gültige Eltern- und Kind-Level sind in der Funktion "Container-Level" der Modellierung definiert.

Weitere Informationen zu Container-Leveln finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch oder im Opcenter Execution Core Modeling-Benutzerhandbuch.

Containerauswahlregeln

Container müssen diese Auswahlregeln erfüllen, damit die Eltern-/Kind-Zuweisung erfolgen kann:

• Kind- und Eltern-Container müssen zur selben Fabrik gehören.
• Kind- und Eltern-Container müssen an derselben Operation, demselben Arbeitsplan und Schritt sein.
• Kind- und Eltern-Container müssen beide geöffnet und dürfen nicht angehalten sein.
• Kind-Container dürfen nicht mit einem anderen Elternteil verbunden sein.
• Kind-Container dürfen nicht abgelaufen sein.

Hinzufügen von betroffenen Containern zu Ereignissen

Die Anwendung kann Container automatisch zu Qualitätsereignissen hinzufügen, wenn Sie die Transaktion "Verbinden" für einen Container durchführen, der mit mindestens einem Qualitätsereignis verknüpft ist. Wenn z. B. ein ausgewählter erlaubter Container mit einem Ereignis verknüpft ist, fügt die Anwendung den Eltern-Container, vorhandene Kind-Container und ausgewählte erlaubte Container automatisch zum Qualitätsereignis hinzu. Ein Systemadministrator konfiguriert die Transaktion "Verbinden" in der Modellierung, um Container zu Ereignissen hinzuzufügen. Informationen zum automatischen Hinzufügen von betroffenen Containern zu Qualitätsereignissen finden Sie im Opcenter Execution Medical Device and Diagnostics Event Management-Benutzerhandbuch und im Opcenter Execution Core Event Management-Benutzerhandbuch sowie im Opcenter Execution Medical Device and Diagnostics Nonconformance Management-Benutzerhandbuch und im Opcenter Execution Core Nonconformance Management-Benutzerhandbuch.

Seite "Verbinden"

Verwenden Sie die "Verbinden"-Seite, um einen Eltern-Container auszuwählen und weisen Sie dann dem Eltern-Container geeignete Kind-Container zu. Der Eltern-Container ist bereits im "Verbinden"-Popup ausgewählt.

Vorgehensweise zum Verbinden eines oder mehrerer Container

Gehen Sie folgendermaßen vor, um einen oder mehrere Container zu verbinden:

1. Wählen Sie Verbinden aus dem Container-Menü. Die Seite "Verbinden" wird angezeigt.
2. Wählen Sie einen Container im Container-Feld aus oder geben Sie ihn ein. Die Anwendung aktualisiert den Container-Kopf und die Auswahlfelder, sodass diese Informationen für den angegebenen Container angezeigt werden. Werte werden nur angezeigt, wenn sie für den Container eingegeben wurden.
Hinweis: Die Anwendung füllt das Feld "Container" und den Container-Kopf automatisch, wenn Sie von einer anderen Fertigungsseite auf das Popup "Verbinden" zugreifen.
3. Wählen Sie mit einer der folgenden Methoden die Container aus, die verbunden werden sollen:
• Benutzen Sie das Feld Auszuwählenden Container eingeben, um einen Container einzugeben, entweder durch Scannen oder manuelle Eingabe des Containernamens und anschließendes Drücken der Tab-Taste, um das Feld zu verlassen. Die Anwendung wählt die Zeile mit diesem Container in der Tabelle "Ausstehende erlaubte Container" aus.
• Aktivieren Sie das Kontrollkästchen neben einem oder mehreren Containern in der Tabelle.
• Aktivieren Sie das Kontrollkästchen in der Kopfzeile der Tabelle, um alle Container auf der aktuellen Tabellenseite auszuwählen.
4. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
5. Klicken Sie auf Alle ausgewählten verbinden oder in der Befehlsleiste auf Abschicken. Die Anwendung löscht das "Container"-Feld und beide Tabellen und zeigt eine Statusmeldung an, die mitteilt, dass die ausgewählten Container mit dem Eltern-Container verbunden wurden.
