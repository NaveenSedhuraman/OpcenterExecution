Wartungsklassenaktivierung

Die Transaktion "Wartungsklassenaktivierung" ermöglicht es Ihnen, eine Wartungsanforderung für alle Ressourcen oder Behälter zu aktivieren, die zu einer bestimmten Wartungsklasse gehören. Sie können die Anforderung für mehr als eine Wartungsklasse gleichzeitig aktivieren.

Beim Aktivieren einer Wartungsanforderung

Die Untertabelle "Ressourcen" auf der Seite "Wartungsklasse" zeigt Ressourcen und Behälter für die ausgewählten Wartungsklassen an.

Ein Häkchen neben einer Wartungsklasse gibt an, dass diese Anforderung für die Wartungsklasse aktiv ist. Sie können das Häkchen im Ankreuzfeld entfernen, um eine Anforderung für eine Wartungsklasse zu deaktivieren.

Seite "Wartungsklassenaktivierung"

Sie können über das Ressource-Menü auf die Seite "Wartungsklassenaktivierung" zugreifen. Verwenden Sie die Seite "Wartungsklassenaktivierung", um Wartungsklassen und Ressourcen für eine Wartungsanforderung auszuwählen.

Vorgehensweise zum Aktivieren einer Wartungsanforderung für Wartungsklassen

Gehen Sie folgendermaßen vor, um eine Wartungsanforderung für Wartungsklassen zu aktivieren:

1. Öffnen Sie die Seite Wartungsklassenaktivierung im Menü Ressource. Die Seite "Wartungsklassenaktivierung" wird angezeigt.
2. Versehen Sie das Ankreuzfeld Aktiviert mit einem Häkchen, um diese Anforderung für eine Wartungsklasse zu aktivieren.
3. Geben Sie eine Wartungsanforderung ein oder wählen Sie sie aus der Liste mit Wartungsanforderungen aus. Die Tabelle "Wartungsklassen" zeigt alle Wartungsklassen an, für die diese Anforderung bereits aktiviert wurde, sowie die Wartungsklassen, die für die Zuweisung an diese Anforderung verfügbar sind.
4. Klicken Sie auf das "Pfeil"-Icon in der Zeile, die den Wartungsklassennamen enthält, um die Ressourcen oder Behälter anzuzeigen, die dieser Wartungsklasse zugeordnet sind. Die zugeordneten Ressourcen werden in der Tabelle "Ressourcen" angezeigt.
5. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
6. Klicken Sie auf Abschicken. Die Wartungsanforderung wird für die ausgewählten Wartungsklassen aktiviert.
