Container vereinigen

Die Transaktion "Container vereinigen" überträgt einen oder mehrere Kind-Container von einem oder mehreren Eltern-Containern auf einen vorhandenen Eltern-Container. Eine "Vereinigen"-Transaktion bewegt weder die Quell- noch die Zielcontainer. Es impliziert auch nicht, dass eine Prozess-Aktivität (Durchsatz) an einem der Container durchgeführt wurde.

Beim Vereinigen von Containern

Alle Anzeigefelder sind auf der Seite "Container vereinigen" leer, bis ein Eltern-Container ausgewählt wird. Die Anwendung füllt die Anzeigefelder im Popup "Container vereinigen" automatisch. Informationen über das Ausführen einer Transaktion über eine Seite oder ein Popup finden Sie unter "Verwenden von Transaktionsseiten und Popups".

Das Auswählen eines Ziel-Containers im Kopf führt dazu, dass die Anwendung alle Gruppen-Container (Eltern-Container mit Kind-Containern) und alle Container mit einer Menge von null (0) in der Tabelle "Erlaubte Container zum Vereinigen" anzeigt, die für die Vereinigung mit dem Ziel-Container geeignet sind. Kind-Container werden in einer Untertabelle unterhalb der Zeile des Eltern-Containers angezeigt.

Darüber hinaus können Sie den Quell-Container im Feld "Auszuwählenden Container eingeben" einscannen oder manuell eingeben und das Feld durch Drücken der Tab-Taste verlassen. Die Anwendung wählt diesen Container in der Tabelle "Erlaubte Container zum Vereinigen" aus. Die Anwendung stellt sicher, dass ihre Kind-Container zurückgegeben und die Eltern-Zeile beim Scannen oder Eingeben eines Eltern-Containers erweitert wird. Die Anwendung legt den Fokus auf die Eltern-Zeile und belässt die Eltern-Zeile erweitert, falls sie erweitert ist, um die Untertabelle "Kind-Container" anzuzeigen. Wenn die Eltern-Zeile reduziert ist, erweitert die Anwendung die Eltern-Zeile, um die Kind-Container anzuzeigen.

Wenn Sie einen Kind-Container scannen oder eingeben, stellt die Anwendung sicher, dass das Elternteil dieses Containers und alle sonstigen Kind-Container in der Tabelle angezeigt werden. Wenn alle in der Tabelle angezeigt werden, wählt die Anwendung den Kind-Container. Ist dies nicht der Fall, füllt die Anwendung die Tabelle mit den Eltern- und Kind-Containern und wählt den gescannten Kind-Container aus.

Hinweis: Die Anwendung zeigt eine Fehlermeldung an, wenn Sie den Namen eines nicht erlaubten oder nicht vorhandenen Containers im Feld "Auszuwählenden Container eingeben" eintragen.

Die gesamte Menge des Kind-Containers (Quelle) wird an den vorhandenen (Ziel-)Container übertragen, wenn eine Vereinigung durchgeführt wird.

Wenn Sie Container vereinigen, wird das Ablaufdatum des Ziel-Containers auf das frühere Ablaufdatum von beiden Containern gesetzt. Wenn jedoch nur ein Quell-Container ein Ablaufdatum hat, wird das Ablaufdatum des Ziel-Containers auf das Ablaufdatum des Quell-Containers gesetzt.

Sie haben die Möglichkeit, den Eltern-Container schließen zu lassen, wenn dieser die Menge null erreicht.

Die Schaltfläche "Ausgewählte Container vereinigen" führt die Transaktion "Container vereinigen" für alle in der Tabelle ausgewählten Kind-Container durch. Die Anwendung löscht die Suchtabelle vollständig, wenn das "Vereinigen" erfolgreich war.

Ausführungsregeln für "Container vereinigen" und "Mengen vereinigen"

Die Container in einer "Vereinigen"-Transaktion müssen folgende Kriterien erfüllen:

• Keiner der zu vereinigenden Container darf geschlossen oder angehalten sein.
• Ziel- und Quell-Container müssen das gleiche Produkt enthalten und auf der gleichen Container-Ebene sein.
• Ziel- und Quell-Container müssen an derselben Operation in derselben Fabrik sein.
• Die Transaktion muss vor den Ablaufdaten erfolgen, wenn Ablaufdaten für den Quell (Von)-Container oder den Ziel (Zu)-Container angegeben wurden. Die "Vereinigen"-Transaktion scheitert, wenn das Ablaufdatum für einen der Container überschritten wurde, bevor Sie die Transaktion ausführen. Falls nur einer der Container ein Ablaufdatum hat, muss die Transaktion vor diesem Datum erfolgen.

Darüber hinaus sollten bei der Ausführung der Transaktion "Container vereinigen" alle Quell-Eltern-Container Kind-Container haben.

Hinzufügen von betroffenen Containern zu Ereignissen

Die Anwendung kann Container automatisch zu Qualitätsereignissen hinzufügen, wenn Sie die Transaktion "Container vereinigen" für einen Container durchführen, der mit mindestens einem Qualitätsereignis verknüpft ist. Wenn z. B. ein neuer Kind-Container mit einem Ereignis verknüpft ist, fügt die Anwendung den Eltern-Container, vorhandene Kind-Container und neue Kind-Container automatisch zum Qualitätsereignis hinzu. Ein Systemadministrator konfiguriert die Transaktion "Vereinigen" in der Modellierung, um Container zu Ereignissen hinzuzufügen. Informationen zum automatischen Hinzufügen von betroffenen Containern zu Ereignissen finden Sie im Opcenter Execution Medical Device and Diagnostics Event Management-Benutzerhandbuch und im Opcenter Execution Core Event Management-Benutzerhandbuch sowie im Opcenter Execution Medical Device and Diagnostics Nonconformance Management-Benutzerhandbuch und im Opcenter Execution Core Nonconformance Management-Benutzerhandbuch.

Seite "Container vereinigen"

Sie können über das Menü "Container" auf die Seite "Container vereinigen" zugreifen. Die Seite "Container vereinigen" ermöglicht es Ihnen, den Eltern-Container einzugeben oder auszuwählen, in dem Sie die gesamte Menge aus einem oder mehreren Kind-Containern vereinigen möchten.

Hinweis: Die Seite "Container vereinigen" ist eventuell auch als Popup verfügbar, wenn die Transaktion "Container vereinigen" als verfügbare Aktion auf einer anderen Seite konfiguriert wurde (zum Beispiel "Containersuche").

Vorgehensweise zum Vereinigen von einem oder mehreren Containern

Führen Sie die folgenden Schritte durch, um einen oder mehrere Container zu vereinigen:

1. Öffnen Sie Container vereinigen im Container-Menü. Die Seite "Container vereinigen" wird angezeigt.
2. Wählen Sie einen Container im Container-Feld aus oder geben Sie ihn ein. Die Anwendung aktualisiert den Container-Kopf und die Auswahlfelder, sodass diese Informationen für den angegebenen Container angezeigt werden. Werte werden nur angezeigt, wenn sie für den Container eingegeben wurden.
Hinweis: Die Anwendung füllt das Feld "Container" und den Container-Kopf automatisch, wenn Sie von einer anderen Fertigungsseite auf das Popup "Container vereinigen" zugreifen.
3. Wählen Sie die Quell-Container, die mithilfe einer der folgenden Methoden mit dem Ziel-Container vereinigt werden sollen:
• Aktivieren Sie das Kontrollkästchen Alle Kinder einschließen in der Eltern-Zeile, um alle Kind-Container auszuwählen.
• Aktivieren Sie das Kontrollkästchen für jeden einzelnen Kind-Container.
• Scannen oder geben Sie einen Container in das Feld Auszuwählenden Container eingeben ein und verlassen Sie das Feld durch Drücken der Tab-Taste. Die Anwendung hebt die Kind-Container-Zeile hervor und aktiviert das entsprechende Kontrollkästchen des Kind-Containers.
4. Wiederholen Sie Schritt 3, bis Sie jeden Quell-Container ausgewählt haben, den Sie mit dem Ziel-Container vereinigen möchten.
5. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
6. Klicken Sie auf Ausgewählte Container vereinigen oder in der Befehlsleiste auf Abschicken. Die Anwendung überträgt die ausgewählten Kind-Container an den Ziel-Container und zeigt eine Erfolgsmeldung an.
