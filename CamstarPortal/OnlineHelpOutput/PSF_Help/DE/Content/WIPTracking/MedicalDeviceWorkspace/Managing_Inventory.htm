Verwalten von Bestand

Die Seite "Lager verwalten" bietet folgende Möglichkeiten:

• Zuweisen von Containern, Losen oder Produkten zu einem Lagerort
• Aktualisieren von Produkt- oder Mengeninformationen für ein Los
• Aktualisieren von Mengeninformationen für ein Produkt
• Auswählen einer Entfernungsstrategie für ein Produkt
• Transferieren von Containern, Losen oder Produkten an einen anderen Lagerort
• Entfernen von Containern, Losen oder Produkten aus einem Lagerort

Informationen zum Definieren von Lagerorten finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch und im Opcenter Execution Core Modeling-Benutzerhandbuch.

Felddefinitionen der Seite "Lager verwalten"

Diese Tabelle definiert die Felder auf der Seite "Lager verwalten".

Feld

Definition

Typ

Lagerort

Lagerort für diese Transaktion.

Erforderlich

Container

Liste der verfügbaren Container mit einer Menge größer als 0, die keinem Lagerort zugeordnet sind.

Optional

Los

Los für diese Transaktion.

Optional

Produkt

Produkt für den ausgewählten Container oder das ausgewählte Los oder das Produkt, das zum Bestand hinzugefügt wird.

Erforderlich

Revision

Revision des Produkts.

Nur Anzeige

Standardrevision

Kontrollkästchen, das angibt, ob dieses Produkt die Standardrevision ist.

Nur Anzeige

Menge

Aktuelle Menge des Containers, Loses bzw. Produkts.

Hinweis: Dieses Feld enthält standardmäßig die Containermenge, wenn ein Container ausgewählt ist.

Erforderlich

Entfernungsstrategie

Entfernungsstrategie für das Produkt.

• FIFO (First In – First Out): Die zuerst gelagerten Materialien werden zuerst verbraucht.
• FEFO (First Expiry – First Out): Die Materialien werden entsprechend dem Ablaufdatum verbraucht.
• LIFO (Last In – First Out): Die zuletzt gelagerten Materialien werden zuerst verbraucht.

Optional

Ablaufdatum

Ablaufdatum für Entfernungsstrategie.

Hinweis: Dieses Feld ist aktiv, wenn FEFO als Entfernungsstrategie ausgewählt ist.

Optional

Maßeinheit

Maßeinheit des Containers, Loses bzw. Produkts.

Optional

Zu Standort

Lagerort, zu dem der Container, das Los oder das Produkt verschoben wird.

Hinweis: Dieses Feld wird nur angezeigt, wenn ein Container, Los oder Produkt in der Tabelle "Bestandsdetails" ausgewählt ist.

Optional

Tabelle "Bestandsdetails"

Tabelle, in der die Container, Lose oder Produkte angezeigt werden, die dem ausgewählten Lagerort zugewiesen sind.

Nur Anzeige

<Kontrollkästchen>

Kontrollkästchen, das angibt, ob der Container, das Los oder das Produkt übertragen oder entfernt wird.

Optional

Container/Los

Der Container bzw. das Los, der/das diesem Lagerort zugewiesen ist.

Nur Anzeige

Produkt

Produkt des Containers oder Loses oder das Produkt, das dem Lagerort zugewiesen ist.

Nur Anzeige

Menge

Menge im Container, Los oder Produkt.

Nur Anzeige

Maßeinheit

Maßeinheit für den Container, das Los oder das Produkt.

Nur Anzeige

Ablaufdatum

Das Ablaufdatum für die Entfernungsstrategie für Produkte, für die die FEFO-Entfernungsstrategie gilt.

Nur Anzeige

Entfernungsstrategie

Entfernungsstrategie für jedes hinzugefügte Material.

• FIFO: Die zuerst gelagerten Materialien werden zuerst verbraucht.
• FEFO: Die Materialien werden entsprechend dem Ablaufdatum verbraucht.
• LIFO: Die zuletzt gelagerten Materialien werden zuerst verbraucht.

Nur Anzeige

Schaltflächendefinitionen der Seite "Lager verwalten"

Diese Tabelle definiert die Schaltflächen auf der Seite "Lager verwalten".

Schaltfläche

Funktion der Schaltfläche . . .

Zurücksetzen

Löscht die Felder auf der Seite.

Abschicken

Weist einem Lagerort einen Container, ein Los oder ein Produkt zu oder aktualisiert die Produkt- bzw. Mengeninformationen für ein Los.

Transferieren

Transferiert einen Container, ein Los oder ein Produkt an einen anderen Lagerort.

Entfernen

Entfernt einen Container, ein Los oder ein Produkt aus einem Lagerort.

Vorgehensweise zum Zuweisen von Containern, Losen oder Produkten zu einem Lagerort

Gehen Sie folgendermaßen vor, um einen Container, ein Los oder ein Produkt einem Lagerort zuzuweisen:

1. Öffnen Sie die Seite Lager verwalten. Die Seite "Lager verwalten" wird angezeigt.
2. Wählen Sie einen Lagerort in der Liste aus.
3. Möchten Sie einen Container, ein Los oder ein Produkt einem Lagerort zuweisen?

Wenn Sie Folgendes zuweisen möchten . . .

Dann . . .

Container
a. Wählen Sie einen Container in der Liste aus. Die Anwendung deaktiviert das Feld "Los". Die Anwendung zeigt das Produkt, die aktuelle Menge und die Maßeinheit des Containers an. Die Schaltfläche "Abschicken" wird angezeigt.
b. Gehen Sie zu Schritt 4.

Los
a. Geben Sie ein Los in das Feld Los ein. Die Anwendung füllt das Feld "Produkt" mit einer Liste der verfügbaren Produkte. Die Anwendung macht das Feld "Menge" beschreibbar. Die Anwendung füllt das Feld "Maßeinheit" mit einer Liste der verfügbaren Maßeinheiten. Die Schaltfläche "Abschicken" wird angezeigt.
b. Wählen Sie ein Produkt in der Liste aus. Die Schaltfläche "Abschicken" wird angezeigt.
c. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
d. Gehen Sie zu Schritt 4.

Produkt
a. Wählen Sie ein Produkt in der Liste aus. Die Schaltfläche "Abschicken" wird angezeigt.
b. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
c. Gehen Sie zu Schritt 4.
4. Klicken Sie auf Einreichen (Submit). Die Anwendung fügt die Container-, Los- oder Produktinformationen zur Tabelle "Bestandsdetails" hinzu. Die Anwendung fügt die Informationen auch zur Tabelle "Bestand" der Modellierungsinstanz für den Lagerort hinzu.

Vorgehensweise zum Aktualisieren von Produkt- oder Mengeninformationen für ein Los

Gehen Sie folgendermaßen vor, um Produkt- oder Mengeninformationen für ein Los zu aktualisieren:

1. Öffnen Sie die Seite Lager verwalten. Die Seite "Lager verwalten" wird angezeigt.
2. Wählen Sie einen Lagerort in der Liste aus.
3. Geben Sie den Namen des Loses in das Feld Los ein. Die Schaltfläche "Abschicken" wird angezeigt.
4. Möchten Sie das Produkt oder die Menge aktualisieren?

Antwort . . .

Dann . . .

Produkt
a. Wählen Sie ein Produkt in der Liste aus.
b. Gehen Sie zu Schritt 5.

Menge
a. Geben Sie ein Los in das Feld Los ein. Die Anwendung füllt das Feld "Produkt" mit einer Liste der verfügbaren Produkte. Die Anwendung macht das Feld "Menge" beschreibbar. Die Anwendung füllt das Feld "Maßeinheit" mit einer Liste der verfügbaren Maßeinheiten. Die Schaltfläche "Abschicken" wird angezeigt.
b. Wählen Sie ein Produkt in der Liste aus.
c. Geben Sie die Menge in das Feld Menge ein.
d. Gehen Sie zu Schritt 5.
5. Klicken Sie auf Abschicken. Die Anwendung aktualisiert die Container- oder Losinformationen in der Tabelle "Bestandsdetails". Die Anwendung aktualisiert die Informationen auch in der Tabelle "Bestand" der Modellierungsinstanz für den Lagerort.

Vorgehensweise zum Transferieren von Containern, Losen oder Produkten an einen anderen Lagerort

Gehen Sie folgendermaßen vor, um einen Container, ein Los oder ein Produkt an einen anderen Lagerort zu transferieren:

1. Öffnen Sie die Seite Lager verwalten. Die Seite "Lager verwalten" wird angezeigt.
2. Wählen Sie einen Lagerort in der Liste aus. Die Anwendung füllt die Tabelle "Bestandsdetails".
3. Aktivieren Sie das Kontrollkästchen für einen Container, ein Los oder ein Produkt in der Tabelle Bestandsdetails. Das Feld "An Lagerort", die Schaltfläche "Transferieren" und die Schaltfläche "Entfernen" werden angezeigt.
4. Wählen Sie einen neuen Lagerort im Feld An Lagerort aus.
5. Klicken Sie auf Transferieren. Die Anwendung entfernt den Container, das Los oder das Produkt aus der Tabelle "Bestandsdetails". Darüber hinaus transferiert die Anwendung den Container, das Los oder das Produkt in die neue Modellierungsinstanz für den Lagerort.

Vorgehensweise zum Entfernen von Containern, Losen oder Produkten aus einem Lagerort

Gehen Sie folgendermaßen vor, um einen Container, ein Los oder ein Produkt aus einem Lagerort zu entfernen:

1. Öffnen Sie die Seite Lager verwalten. Die Seite "Lager verwalten" wird angezeigt.
2. Wählen Sie einen Lagerort in der Liste aus. Die Anwendung füllt die Tabelle "Bestandsdetails".
3. Aktivieren Sie das Kontrollkästchen für einen Container, ein Los oder ein Produkt in der Tabelle Bestandsdetails. Das Feld "An Lagerort", die Schaltfläche "Transferieren" und die Schaltfläche "Entfernen" werden angezeigt.
4. Klicken Sie auf Entfernen. Die Anwendung entfernt den Container, das Los oder das Produkt aus der Tabelle "Bestandsdetails".
