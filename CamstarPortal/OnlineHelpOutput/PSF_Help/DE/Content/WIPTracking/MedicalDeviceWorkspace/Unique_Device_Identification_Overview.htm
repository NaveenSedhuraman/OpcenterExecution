Unique Device Identification (UDI) - Übersicht

Hinweis: Diese Funktionalität ist nur verfügbar, wenn Medical Device installiert ist. Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics Medical Device-Benutzerhandbuch und im Opcenter Execution Core Medical Device-Benutzerhandbuch.

Unique Device Identification (UDI) ist eine Anforderung der Food and Drug Administration (FDA) zur eindeutigen Identifizierung von Medizinprodukten, kombinierten Produkten, die Medizinprodukte enthalten, und Geräten, die unter dem Public Health Service Act lizenziert werden. Die UDI unterstützt die FDA bei der Verfolgung der Produktion von Medizinprodukten und erleichtert Produktrückrufe, wodurch die Produktqualität und die Patientensicherheit verbessert werden.

Die UDI-Funktion bietet einen Mechanismus zum Definieren und Zuweisen einer UDI zu einem Container. Auf den Spez.- und Produkt-Modellierseiten können Sie UDI-Informationen definieren, die an einen Container übertragen werden. Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch und im Opcenter Execution Core Modeling-Benutzerhandbuch.

Sie können UDI-Informationen auf den folgenden Transaktionsseiten auch direkt einem Container hinzufügen:

• Start
• Auftragserledigung
• Containerwartung

Mit den Transaktionen "Abmelden" und "Anmelden" kann eine UDI einem Container automatisch zugewiesen werden, wenn spezifische Informationen für die Spezifikation und das Produkt definiert sind.
