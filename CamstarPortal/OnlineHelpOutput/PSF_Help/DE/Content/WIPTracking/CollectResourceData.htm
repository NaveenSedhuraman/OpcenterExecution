Resourcendaten sammeln

Verwenden Sie die Transaktion "Ressourcendaten sammeln", um parametrische und Benutzerdaten für eine Fertigungs-Ressource aufzuzeichnen. Eine Ressource ist gewöhnlich eine Maschine, an der ein Material verarbeitet wird. Die aufzuzeichnenden Daten werden von den Datensammlungs- oder Benutzerdatensammlungs-Definitionen bestimmt, die für die Ressource angegeben wurden.

Vor dem Sammeln von Daten

Ein Systemadministrator muss mindestens eine Datensammlungs- oder BenutzerDatensammlungs-Definition in der Modellierung definieren, bevor Sie Daten für eine Ressource sammeln können. Datensammlung-Definitionen beziehen sich auf ein Formular zur parametrischen Datenkontrolle, das von einem Designer erstellt wurde. Benutzerdatensammlungs-Definitionen beziehen sich auf Datenpunkte, welche Punkte in einem Geschäftsprozess sind, an denen Daten gesammelt werden können oder müssen.

Die Anforderungen für parametrische und Benutzer-basierte Daten werden von Ihrem Unternehmen definiert, und Ihr Systemadministrator fügt auf Grundlage dieser Anforderungen spezifische Felder hinzu. Darüber hinaus muss ein Systemadministrator die Anwendung so konfigurieren, dass sie Datensammlungs-Definitionen auf parametrischer Grundlage verwendet.

Weitere Informationen finden Sie in folgenden Handbüchern:

• Opcenter Execution Medical Device and Diagnostics Data Collection-Benutzerhandbuch oder Opcenter Execution Core Data Collection-Benutzerhandbuch für Informationen zum Definieren einer parametrischen Datenkontrolle
• Opcenter Execution Medical Device and Diagnostics Data Modeling-Benutzerhandbuch oder Opcenter Execution Core Modeling-Benutzerhandbuch für Informationen zum Definieren von Datensammlungsdefinitionen und Benutzerdatensammlungsdefinitionen
• Opcenter Execution Medical Device and Diagnostics Portal Studio-Benutzerhandbuch oder Opcenter Execution Core Portal Studio-Benutzerhandbuch für Informationen zum Einrichten von parametrischen Datenkontrollen

Beim Sammeln von Ressourcendaten

Alle verfügbaren Ressourcen und Behälter werden in der "Ressourcenliste" angezeigt, es sei denn, Sie wählen eine "Ressourcengruppe", um die Anzahl der angezeigten Ressourcen zu begrenzen. Wenn Sie eine Ressource auswählen, aktualisiert die Anwendung den Kopf, um Informationen in Bezug auf die Ressource, einschließlich der aktuellen Verfügbarkeit der Ressource, anzuzeigen.

Die Liste "Datensammlungsdefinition" enthält sowohl Datensammlungs- als auch Benutzer-Datensammlungsdefinitionen. Nach der Auswahl einer Datensammlungs- oder Benutzer-Datensammlungsdefinition zeigt die Anwendung entweder eine Tabelle oder Datenpunkte an, je nachdem, wie die Datensammlungsdefinition konfiguriert wurde.

Wenn Grenzen für benutzerdefinierte Datensammlungen definiert wurden

Die Anwendung kann Ober- und Untergrenzen für Datenpunkte anzeigen, in Abhängigkeit davon, wie die Benutzerdatensammlung definiert wurde. Wenn die Datensammlungsdefinition das Zeilen-Spalten-Format verwendet, beträgt der maximale Wert, der für die Ober- und Untergrenze angezeigt werden kann, 7 Zeichen. Wenn der Wert außerhalb einer definierten Grenze liegt, markiert die Anwendung die Zelle und die verletzte Grenze entweder in gelb oder in rot. Die Farbe hängt davon ab, wie die Datensammlung definiert wurde.

Die Anwendung markiert die Zelle und die verletzte Grenze in folgender Farbe:

• Gelb - Der Datenpunkt erlaubt Überschreitungen
• Rot - Der Datenpunkt verbietet Überschreitungen

Die Anwendung ermöglicht es Ihnen, eine Datensammlung-Transaktion unabhängig von den Einstellungen zur Überschreitung von Grenzwerten auszuführen. Wenn die Datensammlung einem automatischen Ablauf zugeordnet ist, bestimmen Einstellungen zur Überschreitung von Grenzwerten, ob die Aufgabe als Bestandteil des automatische Ablaufs als bestanden oder durchgefallen markiert wird.

Hinweis: Die Anwendung zeigt negative Grenzen in Klammern an.

Seite "Ressourcendatensammlung"

Sie können die über das Ressourcen-Menü auf die Seite "Ressourcendatensammlung" zugreifen. Die Seite "Ressourcendatensammlung" ermöglicht es Ihnen, Ressourcendaten unter Verwendung der Datensammlungs- oder Benutzerdatensammlungs-Definitionen aufzuzeichnen.

Wenn die Datensammlungsdefinition das Zeilen-Spalten-Format verwendet, beträgt der maximale Wert, der für die Ober- und Untergrenze angezeigt werden kann, 7 Zeichen.

Vorgehensweise zum Sammeln von Ressourcendaten

Gehen Sie folgendermaßen vor, um Ressourcendaten zu sammeln:

1. Öffnen Sie im Menü Ressource die Seite Ressourcendatensammlung. Die Seite "Ressourcendatensammlung" wird angezeigt.
2. Wählen Sie erst eine Ressourcengruppe, falls Sie die anzuzeigende Ressourcen-Liste filtern möchten.
3. Wählen Sie die Ressource, für die Sie Daten sammeln möchten. Die Anwendung aktualisiert den Kopf, um Informationen für die ausgewählte Ressource anzuzeigen.
4. Wählen Sie eine Datensammlungs-Definition aus dem Feld Datensammlungs-Definition. Die Datensammlungsfelder werden angezeigt.
5. Füllen Sie die Datensammlungs-Felder aus.
6. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
7. Klicken Sie auf Ressourcendaten sammeln. Eine Statusnachricht wird angezeigt, die angibt, dass Daten erfasst wurden.
