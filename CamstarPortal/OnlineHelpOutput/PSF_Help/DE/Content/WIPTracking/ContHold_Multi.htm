Mehrere Container anhalten

Mit der Transaktion "Anhalten (mehrere)" können Sie mehrere Container gleichzeitig aus einem bestimmten Grund (z. B. eine geplante Inspektion oder ein Protokoll) auf Halt setzen. Wenn Sie Container auf Halt setzen, verhindern Sie damit, das Benutzer Transaktionen an den Containern durchführen können. Der Container kann wieder für den normalen Betrieb freigegeben werden, sobald die Gründe für den Halt beseitigt wurden.

Beim Anhalten mehrerer Container

Sie müssen einen Anhaltegrund auswählen, um einen Container auf Angehalten zu setzen. Anhaltegründe werden in der Modellierung von einem Systemadministrator festgelegt. Die Anwendung zeigt folgende Fehlermeldung an, wenn Sie auf die Schaltfläche "Ausgewählte anhalten" klicken, ohne vorher einen Container in der Tabelle auszuwählen. Die Anwendung zeigt den Grund automatisch an, wenn nur der Anhaltegrund definiert wurde.

Die Anwendung überprüft, ob die angegebenen Container aktiv sind und nicht bereits auf Halt stehen. Sie können einen geschlossenen Container nicht auf Angehalten setzen.

Wenn Sie einen Container auf Angehalten setzen, werden dessen Kind-Container nicht auf Angehalten gesetzt. Wenn Sie jedoch einen Kind-Container auf Angehalten setzen, dann wird damit auch der Eltern-Container auf Angehalten gesetzt. Sie können die Bearbeitung des Eltern-Containers nur dann wieder fortsetzen, wenn Sie den angehaltenen Kind-Container wieder freigeben.

Unter "Multi-Container-Transaktionsübersicht" finden Sie Informationen darüber, wie die Anwendung beim Ausführen von Transaktionen an mehreren Containern funktioniert.

Seite "Anhalten (mehrere)"

Sie können über das Menü "Container" auf die Seite "Anhalten (mehrere)" zugreifen. Sie können auf das Popup "Anhalten (mehrere)" von einer anderen Fertigungsseite aus zugreifen (z. B. "Containersuche"), wenn die Aktion "Anhalten (mehrere)" für diese Seite konfiguriert wurde. Sowohl die Seite "Anhalten (mehrere)" als auch das Popup ermöglichen es Ihnen, einen oder mehrere Container zu scannen oder einzufügen und diese auf Angehalten zu setzen.

Die Seite und das Popup sind identisch in Aussehen und Funktionalität - mit einer Ausnahme. Die Anwendung füllt das Popup "Anhalten (mehrere)" automatisch mit den Containern, die Sie auf der Fertigungsseite ausgewählt haben, von der aus Sie auf das Popup zugegriffen haben.

Zusätzliche Felder anzeigen/ausblenden

Optionale Felder sind standardmäßig ausgeblendet, wenn die Anwendung eine Seite oder ein Popup anzeigt. Durch Klicken auf die Steuerung "Anzeigen" werden die Felder angezeigt. Durch Klicken auf die Steuerung "Verstecken" werden die Felder ausgeblendet.

  

Kommentare

Zusätzliche Informationen oder Anweisungen für diese Transaktion oder für diesen Container. Das Feld akzeptiert maximal 255 Zeichen. Wenn die Kommentare 255 Zeichen überschreiten, verkürzt die Anwendung die Daten.

Optional

Vorgehensweise zum Setzen mehrerer Container auf Angehalten

Gehen Sie folgendermaßen vor, um mehrere Container auf Anhalten zu setzen:

1. Öffnen Sie auf Anhalten (mehrere) im Container-Menü. Die Seite "Anhalten (mehrere)" wird angezeigt.
2. Können Sie die Seite "Anhalten (mehrere)" oder das Popup "Anhalten (mehrere)" sehen?

Wird Folgendes angezeigt . . .

Dann . . .

Seite "Anhalten (mehrfach)"

Die Anzuhaltende Container Tabelle ist leer. Gehen Sie zu Schritt 3.

Popup "Anhalten (mehrfach)"

Die "Anzuhaltende Container" Tabelle wird mit den Containern gefüllt, die sie auf der Seite ausgewählt haben, von der aus Sie das Popup gestartet haben. Wenn Sie zusätzliche zu bewegende Container hinzufügen möchten, gehen Sie zu Schritt 3. Fahren Sie andernfalls mit Schritt 6 fort.
3. Scannen Sie oder geben Sie einen Container in das Feld Auszuwählenden Container eingeben ein.
4. Drücken Sie die Eingabetaste oder verlassen Sie das Feld durch Drücken der Tab-Taste. Der Container wird zur Tabelle "Anzuhaltende Container" hinzugefügt und automatisch ausgewählt.
5. Wiederholen Sie Schritte 3-4, um zusätzliche Container hinzuzufügen:
6. Stellen Sie sicher, dass die ausgewählten Container diejenigen sind, die Sie anhalten möchten.
7. Klicken Sie auf einen Haltegrund.
8. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
9. Klicken Sie auf Ausgewählte Container anhalten oder in der Befehlsleiste auf Abschicken. Die Anwendung zeigt ein Status-Icon für jeden Container an, das angibt, ob die Transaktion erfolgreich ausgeführt wurde.
10. Schließen Sie das Popup wenn es offen ist, und kehren Sie zur vorherigen Seite zurück.
