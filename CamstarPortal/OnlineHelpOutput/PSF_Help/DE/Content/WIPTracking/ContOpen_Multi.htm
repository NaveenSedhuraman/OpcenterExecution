Mehrere Container öffnen

Die Transaktion "Öffnen (mehrere)" ermöglicht es Ihnen, mehrere geschlossene Container gleichzeitig zu öffnen. Zusätzlich können Sie Kind-Container öffnen, die mit dem ausgewählten Container verbunden sind. Nach dem Öffnen von Containern, können Sie Prozess-Transaktionen an ihnen fortsetzen.

Beim Öffnen mehrerer Container

Sie haben beim Öffnen mehrerer Container die Möglichkeit, einen "Grund für Statusänderung" anzugeben. Gründe für Statusänderungen werden in der Modellierung von einem Systemadministrator definiert. Wenn nur ein Grund für Statusänderung festgelegt wurde, zeigt die Anwendung diesen Grund automatisch an. Weitere Informationen zu Gründen für Statusänderungen finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch und im Opcenter Execution Core Modeling-Benutzerhandbuch.

Die Anwendung prüft dass die angegebenen Container geschlossen oder unterwegs sind.

Prozesstimer, die bei einer Transaktion des Typs "Schließen (mehrere)" gestoppt wurden, werden nicht fortgeführt, wenn die Transaktion "Öffnen (mehrere)" ausgeführt wird.

Unter "Multi-Container-Transaktionsübersicht" finden Sie Informationen darüber, wie die Anwendung beim Ausführen von Transaktionen an mehreren Containern funktioniert.

Seite "Öffnen (mehrere)"

Sie können die Seite "Öffnen (mehrere)" im Containermenü aufrufen. Sie können auf das Popup "Öffnen (mehrere)" von einer anderen Fertigungsseite aus zugreifen (zum Beispiel Containersuche), wenn die Aktion "Öffnen (mehrere)" für diese Seite konfiguriert wurde. Sowohl die Seite "Öffnen (mehrere)" als auch das Popup ermöglichen es Ihnen, einen oder mehrere Container zu scannen oder einzugeben.

Die Seite und das Popup sind identisch in Aussehen und Funktionalität - mit einer Ausnahme. Die Anwendung füllt das Popup "Öffnen (mehrere)" automatisch mit den Containern, die Sie auf der Fertigungsseite ausgewählt haben, von der aus Sie auf das Popup zugegriffen haben.

Zusätzliche Felder anzeigen/ausblenden

Optionale Felder sind standardmäßig ausgeblendet, wenn die Anwendung eine Seite oder ein Popup anzeigt. Durch Klicken auf die Steuerung "Anzeigen" werden die Felder angezeigt. Durch Klicken auf die Steuerung "Verstecken" werden die Felder ausgeblendet.

Optional

Vorgehensweise zum Öffnen mehrerer Container

Gehen Sie folgendermaßen vor, um mehrere geschlossene Container zu öffnen:

1. Öffnen Sie auf "Öffnen (mehrere)" im Menü Container. Die Seite "Öffnen (mehrere)" wird angezeigt.
2. Wird die Seite "Öffnen (mehrere)" oder das Popup "Öffnen (mehrere)" angezeigt?

Wird Folgendes angezeigt . . .

Dann . . .

Seite "Öffnen (mehrere)"

Die "Zu öffnende Container" Tabelle ist leer. Gehen Sie zu Schritt 3.

Popup "Öffnen (mehrere)"

Die Tabelle "Zu öffnende Container" wird mit den Containern gefüllt, die Sie auf der Seite, von der aus Sie das Popup-Fenster gestartet haben, ausgewählt haben. Wenn Sie zusätzliche zu öffnende Container hinzufügen möchten, gehen Sie zu Schritt 3. Fahren Sie andernfalls mit Schritt 6 fort.
3. Scannen Sie oder geben Sie einen Container in das Feld Auszuwählenden Container eingeben ein.
4. Drücken Sie die Eingabetaste oder verlassen Sie das Feld durch Drücken der Tab-Taste. Der Container wird zur "Zu öffnende Container" hinzugefügt und automatisch ausgewählt.
5. Wiederholen Sie Schritte 3-4, um zusätzliche Container hinzuzufügen.
6. Gewährleisten Sie, dass die ausgewählten Container diejenigen sind, die Sie öffnen möchten.
7. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
8. Klicken Sie auf Auswahl öffnen oder in der Befehlsleiste auf Abschicken. Die Anwendung zeigt ein Status-Icon für jeden Container an, das angibt, ob die Transaktion erfolgreich ausgeführt wurde.
9. Schließen Sie das Popup wenn es offen ist, und kehren Sie zur vorherigen Seite zurück.
