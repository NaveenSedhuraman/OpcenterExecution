Aktualisieren von Trainingseinträgen

Mit der Schaltfläche "Eintrag aktualisieren" auf der Seite "Trainingseintragsverwaltung" können Sie Trainingseinträge für einen Mitarbeiter aktualisieren. Aktualisieren Sie einen Trainingseintrag, wenn Sie die zum Eintrag gehörige Trainingsanforderung, den Status oder das Ablaufdatum ändern müssen.

Beim Aktualisieren von Trainingseinträgen

Sie müssen nach dem Eintrag suchen, den Sie aktualisieren möchten, und ihn dann in der Tabelle "Trainingseinträge" auswählen.

Nur Trainingsanforderungen, für die Sie als Trainer angegeben sind, sind in der Liste "Trainingsanforderung" enthalten. Sie können eine bestimmte Revision der Trainingsanforderung auswählen.

Popup "Trainingseintrag"

Das Popup "Trainingseintrag" wird angezeigt, wenn Sie einen Eintrag auswählen und dann auf der Seite "Trainingseintragsverwaltung" auf "Eintrag aktualisieren" klicken.

Vorgehensweise zum Aktualisieren eines Trainingseintrags

Gehen Sie folgendermaßen vor, um einen Trainingseintrag zu aktualisieren:

1. Öffnen Sie die Seite Trainingseintragsverwaltung im Menü Training. Die Seite "Trainingseintragsverwaltung" wird angezeigt.
2. Suchen Sie nach dem Eintrag, den Sie aktualisieren möchten, indem Sie eines oder mehrere dieser Suchkriterien zum Abrufen von Trainingseintragsinformationen verwenden:
• Geben Sie einen Mitarbeiter ein oder wählen Sie ihn aus.
• Geben Sie eine Trainingsanforderung ein oder wählen Sie sie aus.
• Geben Sie einen Status ein oder wählen Sie ihn aus.
3. Klicken Sie auf Suchen. Die Tabelle "Trainingseinträge" wird mit den Einträgen gefüllt, die Ihren Kriterien entsprechen.
4. Wählen Sie den Eintrag, den Sie aktualisieren möchten, in der Tabelle "Trainingseinträge" aus.
5. Klicken Sie auf Eintrag aktualisieren. Das Popup "Trainingseintrag" wird angezeigt.
6. Geben Sie eine Trainingsanforderung im Feld Trainingsanforderung ein oder wählen Sie sie aus und wählen Sie falls erforderlich eine Revision aus.
7. Geben Sie den Status des Protokolls in das Feld Status ein.
8. Geben Sie eine erforderliche elektronische Unterschrift direkt oder durch Auswählen in das Feld Anforderung für elektr. Unterschrift ein (sofern zutreffend).
9. Geben Sie ein Datum im Feld Ablaufdatum ein oder wählen Sie es aus, falls nötig.
10. Klicken Sie auf Abschicken. Das Popup "Trainingseintrag" wird geschlossen und der aktualisierte Eintrag wird in der Tabelle "Trainingseinträge" auf der Seite "Trainingseintragsverwaltung" angezeigt.
