Hinzufügen von Trainingseinträgen

Mit der Schaltfläche "Eintrag hinzufügen" auf der Seite "Trainingseintragsverwaltung" können Sie Trainingseinträge für einen Mitarbeiter hinzufügen.

Sie fügen Trainingseinträge hinzu, wenn Sie gewährleisten wollen, dass ein Mitarbeiter einen bestimmten Ausbildungsplan absolviert. Z. B. müssen alle Mitarbeiter einen grundlegenden Sicherheitskurs absolvieren, während ein Monteur außerdem in einem bestimmten Montageschritt unterwiesen werden muss.

Beim Hinzufügen von Trainingseinträgen

Sie müssen Folgendes in Ihrem Fabrikmodell angeben, bevor Ihre Trainingseinträge hinzugefügt werden können:

• Trainingsanforderungen
• Trainingseintrag-Statuscodes
• Mitarbeiter

Sie müssen ein zugewiesener Trainer sein, um Trainingseinträge für einen Mitarbeiter hinzufügen zu können. Es stehen keine Trainingsanforderungen zur Auswahl, wenn Sie nicht der zugewiesene Trainer sind.

Wenn Sie eine Trainingsanforderung auswählen, können Sie die Anforderung erweitern, um eine bestimmte Revision dieser Anforderung auszuwählen, wenn Sie eine andere Revision als die Standardrevision verwenden möchten.

Der Eintrag verweist immer auf eine bestimmte Revision. Wenn Sie die Standardrevision auswählen und die Revision hinzufügen, ermittelt die Anwendung die aktuelle Standardrevision, wenn Sie die Trainingsanforderung zu einem späteren Zeitpunkt aktualisieren.

Popup "Trainingseintrag hinzufügen"

Das Popup "Trainingseintrag hinzufügen" wird angezeigt, wenn Sie "Eintrag hinzufügen" auf der Seite "Trainingseintrag" auswählen.

Felddefinitionen für das Popup "Trainingseintrag hinzufügen"

Diese Tabelle definiert die Felder des Popups "Trainingseintrag hinzufügen".

Feld

Definition

Typ

Trainer

Mitarbeiter-ID des Benutzers, der aktuell angemeldet ist.

Nur Anzeige

Mitarbeiter

Name des Mitarbeiters, dessen Datensatz Sie hinzufügen möchten. Wenn ein Mitarbeiter auf der Seite "Trainingseintragsverwaltung" ausgewählt wurde, wird dieses Feld mit dessen Namen gefüllt. Sie können einen anderen Namen auswählen.

Erforderlich

Trainingsanforderung

Liste der Trainingsanforderungen, die Sie für den Mitarbeiter hinzufügen können. Nur Trainingsanforderungen, für die Sie als Trainer angegeben sind, sind in der Liste enthalten. Sie können eine bestimmte Revision der Trainingsanforderung auswählen.

Erforderlich

Status

Aktueller Status der Trainingsanforderung, die Sie hinzufügen. Z. B. ausgebildet oder nicht ausgebildet.

Erforderlich

Elektr. Unterschrift: Anforderung

Erforderliche elektronische Unterschrift. Diese ist erforderlich, wenn der Mitarbeiter eine Transaktion für die angegebene Trainingsanforderung ausführt. Der Status des Trainingseintrags muss mit der Berechtigung "Mit elektr. Unterschrift erlauben" konfiguriert sein, damit dieses Feld aktiviert werden kann.

Erforderlich

Vorgehensweise zum Hinzufügen eines Trainingseintrags

Gehen Sie folgendermaßen vor, um einen Trainingseintrag hinzuzufügen:

1. Öffnen Sie die Seite Trainingseintragsverwaltung im Menü Training. Die Seite "Trainingseintragsverwaltung" wird angezeigt.
2. Klicken Sie auf Eintrag hinzufügen. Das Popup "Trainingseintrag hinzufügen" wird angezeigt.
3. Geben Sie den Namen des Mitarbeiters im Feld Mitarbeiter ein oder wählen Sie ihn aus.
4. Geben Sie eine Trainingsanforderung im Feld Trainingsanforderung ein oder wählen Sie sie aus und wählen Sie falls erforderlich eine Revision aus.
5. Geben Sie den Status des Protokolls in das Feld Status ein.
6. Geben Sie eine erforderliche elektronische Unterschrift direkt oder durch Auswählen in das Feld Anforderung für elektr. Unterschrift ein (sofern zutreffend).
7. Klicken Sie auf Abschicken. Das Popup "Trainingseintrag hinzufügen" wird geschlossen und der neue Eintrag wird in der Tabelle "Trainingseinträge" auf der Seite "Trainingseintragsverwaltung" angezeigt.
