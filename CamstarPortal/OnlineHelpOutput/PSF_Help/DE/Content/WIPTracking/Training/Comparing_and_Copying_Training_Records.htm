Vergleichen und Kopieren von Trainingseinträgen

Die Transaktion "Trainingseintragsvergleich" erleichtert die Erstellung von Trainingseinträgen durch den Vergleich der Trainingseinträge von zwei Mitarbeitern und das Kopieren eines Eintrags von einem zum anderen. Dies ist nützlich, wenn Sie einen neuen Mitarbeiter haben, der den gleichen oder einen ähnlichen Trainingsplan durchlaufen muss, wie ein anderer Mitarbeiter, dessen Einträge bereits definiert sind. Durch Kopieren eines vorhandenen Protokolls können Sie dieses Ihren Anforderungen entsprechend anpassen, anstatt das Protokoll komplett neu zu erstellen.

Beim Kopieren von Mitarbeitereinträgen

Verwenden Sie die Schaltfläche "Eintrag kopieren", um den in der Trainingseintragstabelle "Von Mitarbeiter" ausgewählten Eintrag in die Trainingseintragstabelle "Zu Mitarbeiter" zu kopieren. Die Anwendung aktiviert diese Schaltfläche, wenn Sie einen Mitarbeiter im Feld "Von Mitarbeiter" und einen Mitarbeiter im Feld "Zu Mitarbeiter" auswählen.

Sie müssen nicht alle Trainingseinträge auf einmal oder von einem bestimmten Mitarbeiter kopieren. Sie können ein oder mehrere Trainingseinträge von einem Mitarbeiter kopieren und anschließend zu einem späteren Zeitpunkt zusätzliche Einträge von einem anderen Mitarbeiter kopieren.

Beispielsweise können Sie einen grundlegende Sicherheitskurs und Trainingsanforderungen für eine Montage-Operation von Mitarbeiter A zu Mitarbeiter B kopieren. Später, wenn Sie sich entscheiden, Training für weitere Operationen hinzuzufügen, können Sie dieses von Mitarbeiter C zu Mitarbeiter B kopieren.

Sie können die Trainingseinträge eines Mitarbeiters verwenden, um die Trainingseinträge mehrerer anderer Mitarbeiter zu aktualisieren. Lassen Sie denselben Mitarbeiter im Feld "Von Mitarbeiter" wählen Sie jeweils neue Mitarbeiter im Feld "Zu Mitarbeiter" aus. Vergessen Sie nicht, auf "Eintrag kopieren" zu klicken und den Eintrag zu speichern, bevor Sie einen neuen "Zu Mitarbeiter" auswählen.

Beim Anzeigen der Mitarbeiter-Trainingseintragstabellen

Die Tabellen, die die Trainingseinträge für die angegebenen "Zu"- und "Von"-Mitarbeiter anzeigen, sind schreibgeschützt. Trainingsanforderungen und Trainingsanforderungs-Status werden in der Modellierung definiert. Weitere Informationen hierzu finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch oder im Opcenter Execution Core Modeling-Benutzerhandbuch.

Hinweis: Ein Ablaufdatum wird nur dann für eine Trainingsanforderung angezeigt, wenn ein Datum angegeben wurde, als die Trainingsanforderung definiert wurde.

Vorgehensweise zum Vergleichen von Mitarbeiter-Trainingseinträgen

Gehen Sie folgendermaßen vor, um Mitarbeiter-Trainingseinträge zu vergleichen:

1. Öffnen Sie die Seite Trainingseintragsvergleich im Menü Training. Die Seite "Trainingseintragsvergleich" wird angezeigt.
2. Wählen Sie im Feld Von Mitarbeiter einen Mitarbeiter aus. Die Trainingseinträge, die diesem Mitarbeiter zugewiesen wurden, werden in der Tabelle unterhalb des Feldes "Von Mitarbeiter" angezeigt.
3. Wählen Sie im Feld Zu Mitarbeiter einen Mitarbeiter aus. Die Trainingseinträge, die diesem Mitarbeiter zugewiesen wurden, werden in der Tabelle unterhalb des Feldes "Zu Mitarbeiter" angezeigt und die Anwendung aktiviert die Schaltfläche "Eintrag kopieren".
4. Vergleichen Sie die Trainingseinträge, die in den zwei Tabellen angezeigt werden.

Vorgehensweise zum Kopieren eines Trainingseintrags von einem Mitarbeiter zu einem anderen

Gehen Sie folgendermaßen vor, um einen Trainingseintrag von einem Mitarbeiter zu einem anderen zu kopieren:

1. Öffnen Sie die Seite Trainingseintragsvergleich im Menü Training. Die Seite "Trainingseintragsvergleich" wird angezeigt.
2. Wählen Sie den Mitarbeiter, dessen Trainingseinträge Sie kopieren möchten, im Feld Von Mitarbeiter aus. Die Trainingseinträge, die diesem Mitarbeiter zugewiesen wurden, werden in der Tabelle unterhalb des Feldes "Von Mitarbeiter" angezeigt.
3. Wählen Sie den Mitarbeiter, zu dem Sie die Trainingseinträge kopieren möchten, im Feld Zu Mitarbeiter aus. Die Trainingseinträge, die diesem Mitarbeiter zugewiesen wurden, werden in der Tabelle unterhalb des Feldes "Zu Mitarbeiter" angezeigt und die Anwendung aktiviert die Schaltfläche "Eintrag kopieren".
4. Wählen Sie den Trainingseintrag, den Sie kopieren möchten, in der Trainingseintrags-Tabelle von "Von Mitarbeiter" aus.
5. Klicken Sie auf Eintrag kopieren. Eine Kopie des Eintrags, der dem Mitarbeiter zugeordnet ist, wird der Trainingseintrags-Tabelle von "Zu Mitarbeiter" hinzugefügt.
6. Wiederholen Sie die Schritte 2 bis 5, um weitere Protokolle zu kopieren.
7. Klicken Sie auf Speichern. Die Anwendung zeigt eine Statusmeldung an, die bestätigt, dass die kopierten Datensätze für den Mitarbeiter hinzugefügt wurden.
