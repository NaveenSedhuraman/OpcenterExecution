Ressourceneinrichtung und -verfügbarkeit

Verwenden Sie die Transaktion "Ressourceneinrichtung", um Änderungen an einem Ressource-Setup aufzuzeichnen oder die Verfügbarkeit einer Ressource oder eines Behälters anzugeben.

Ressourceneinrichtungsinformationen enthalten Verweise auf eine bestimmte Einrichtungsdefinition und das Produkt, das bearbeitet wird.

Die Informationen zur Verfügbarkeit enthalten folgende Informationen über die Ressource:

• Verfügbarkeit (verfügbar, nicht verfügbar oder nicht eingestellt)
• Status (z. B. in Produktion, Wartezeit oder Reparatur)
• Ursachencode (z. B. defektes Messgerät)

Eine Historie der Ressourcennutzung und Fehlerinformationen wird anhand der Verfügbarkeitsinformationen erstellt, die in Manufacturing eingegeben wurden. Wenn eine Ressource z. B. durchfällt, können Sie die Transaktion "Ressourceneinrichtung" durchführen, um die Verfügbarkeit auf "Nicht verfügbar", den Status auf "Reparatur" und den Grund auf "Defektes Messgerät" zu ändern.

Wenn die Reparatur abgeschlossen ist, können Sie eine weitere Transaktion "Ressourceneinrichtung" durchführen, um die Verfügbarkeit wieder auf "Verfügbar" und den Status wieder auf "In Produktion" zu setzen. Die Verfügbarkeit wird durch den Ressourcenstatuscode gesteuert, der wiederum in der Modellierung definiert wird. Sie haben nun einen Datensatz darüber, wann der Fehler auftrat, was die Ursache des Fehlers war und wie lange die Reparatur dauerte.

Beim Verwenden von "Ressourceneinrichtung"

Das Ressourcenfeld zeigt alle Ressourcen und Behälter an, es sei denn, Sie wählen eine Ressourcengruppe aus. Dann wird die Anzeige von Ressourcen und Behältern nach dieser Ressourcengruppe gefiltert.

Wenn Sie eine Ressource auf der Seite "Ressourceneinrichtung" auswählen, wird die Verfügbarkeit der Ressource angezeigt. Diese Bedingungen bestimmen, was auf der Seite im Ressource-Kopfbereich unter "Verfügbarkeit" angezeigt wird.

Seite "Ressourceneinrichtung"

Sie können über das Menü "Ressourcen" auf die Seite "Ressourceneinrichtung" zugreifen. Verwenden Sie die Seite "Ressourceneinrichtung", um Änderungen an einer Ressourceneinrichtung aufzuzeichnen oder die Verfügbarkeit einer Ressource oder eines Behälters anzugeben.

Felddefinitionen der Seite "Ressourceneinrichtung"

Diese Tabelle definiert die Felder auf der Seite "Ressourceneinrichtung".

Feld

Definition

Typ

Informationen zu Ressourcen

Kopf mit dem Ressource-Namen und Informationen zu dieser Ressource. Sie müssen eine Ressource auswählen oder eingeben, um die Kopf-Felder zu füllen. (Sie können eine Ressourcengruppe auswählen, um die zur Auswahl stehenden Ressourcen zu beschränken.) Weitere Informationen finden Sie unter "Informationen zu Ressourcen".

Erforderlich

Ressourcengruppe

Auswahl beschränkt die Liste der angezeigten "Ressourcen" oder Behälter auf die "Ressourcengruppe" oder Behältergruppe, zu der sie gehören.

Optional

Ressource

Maschine, Arbeitszelle, Vorrichtung, Behälter usw. für die Datensammlung. Die Liste beinhaltet alle Ressource und Behälter, es sei denn, sie ist nach Ressourcengruppen gefiltert.

Erforderlich

Ressource-Statuscode

Beschreibt den Status, der zunächst bei der Einstellung und später bei jeder Statusänderung einer Ressource zugewiesen werden kann. Einige Beispiele sind "Verfügbar", "Im Einsatz" und "Nicht verfügbar".

Erforderlich

Ressourcenstatusgrund

Höheres Maß an Details zur Beschreibung der Ursache für den aktuellen Status einer Ressource, wie zum Beispiel Reparatur. "Zur Reparatur geschickt" könnte beispielsweise den Grund für den "Reparatur"-Status einer Ressource beschreiben. Dieses Detail wird in die Transaktions-Historie aufgenommen, wodurch es für Analyseberichte zur Verfügung steht.

Optional

Verfügbarkeit

Verfügbarkeitsstatus dieser Ressource basierend auf dem Ressource-Statuscode. Dies sind die drei Zustände:

• Verfügbar (die Ressource ist verfügbar)
• Nicht verfügbar (die Ressource ist nicht verfügbar)
• Leer (die Ressource wurde nicht eingestellt)

Nur Anzeige

Zusätzliche Felder anzeigen/ausblenden

Optionale Felder sind standardmäßig ausgeblendet, wenn die Anwendung eine Seite oder ein Popup anzeigt. Durch Klicken auf die Steuerung "Anzeigen" werden die Felder angezeigt. Durch Klicken auf die Steuerung "Verstecken" werden die Felder ausgeblendet.

  

Einstellung

Konfiguration der Ressource (Hardware und andere Optionen, deren Änderung erhebliche Zeit in Anspruch nimmt), die vorliegen wird, wenn die Transaktion abgeschlossen ist.

Optional

Produkt

Dieser Ressource zugeordnetes Produkt.

Optional

Ausführen

Name, der sämtliche auf Grundlage der Einstellung durchgeführte Arbeit identifiziert. Allen Containern, die an dieser Ressource mit dieser Einstellungs-Konfiguration bearbeitet werden, wird der Name zugeordnet, der dieser Ausführung zugewiesen wurde.

Optional

Rezept

Einrichtungsanforderungen, die auf eine Produktionsressource angewendet werden können.

Hinweis: Ein Wert, der über diese Seite eingegeben und gespeichert wurde, wird im Feld "Rezept" der Seiten "Ressource" gespeichert.

Hinweis: Dieses Feld wird nur angezeigt, wenn Opcenter EX EL oder der Industry Solutions-Arbeitsbereich installiert ist.

Optional

Kommentare

Relevante Kommentare zu dieser Transaktion. Das Feld akzeptiert maximal 255 Zeichen. Wenn die Kommentare 255 Zeichen überschreiten, verkürzt die Anwendung die Daten.

Optional

Vorgehensweise zum Einrichten einer Ressource

Gehen Sie folgendermaßen vor, um eine Ressource einzurichten:

1. Öffnen Sie die Seite Ressourceneinrichtung im Menü Ressource. Die Seite "Ressourceneinrichtung" wird angezeigt.
2. Wählen Sie erst eine Ressourcengruppe aus, falls Sie die angezeigte Liste von Ressourcen begrenzen möchten.
3. Wählen Sie die Ressource, die Sie einstellen möchten. Die Seite wird aktualisiert und zeigt Folgendes an, falls die Ressource bereits eingestellt wurde: die "Zeit im Zustand" (Zeit im aktuellen Zustand), "Statuscode" (aktueller "Ressourcenstatus") und "Statusgrund" (aktueller "Ressourcenstatusgrund"). Das "Verfügbarkeit"-Feld zeigt die aktuelle Verfügbarkeit der Ressource an.
4. Wählen Sie einen Ressourcenstatuscode.
5. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
6. Klicken Sie auf Einrichtung. Eine Erfolgsmeldung wird angezeigt.
