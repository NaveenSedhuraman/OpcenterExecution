Mengen aufteilen

Die Transaktion "Menge aufteilen" erzeugt eine oder mehrere Ziel-Container aus einem vorhandenen Quell-Container und überträgt dann eine Teilmenge des Produkts aus dem Quell-Container in die neuen Ziel-Container.

Hinweis: Wenn Sie einen oder mehrere Kind-Container von einem Eltern-Container in einen neuen Eltern-Container übertragen möchten, verwenden Sie die Transaktion "Container aufteilen". Weitere Informationen finden Sie unter "Container aufteilen".

Beispiele für das Aufteilen von Mengen und Containern

Dies sind einige Beispiele für die Verwendung der "Aufteilen"-Transaktionen:

• Eine kleine Menge eines Containers wird aufgeteilt, damit er schneller verarbeitet werden kann, um die Kunden- oder Inventar-Anforderungen zu erfüllen.
• Ein Teil eines Containers mit Qualitätsproblemen wird geteilt, bevor eine Nacharbeits- oder Anhalten-Transaktion durchgeführt wird. Der Rest des Containers wird normal bearbeitet.
• Ein Teil des Containers wird geteilt, sodass er zu einem anderen Arbeitsplan zur Bearbeitung eines Spezial-Auftrags bewegt werden kann.

Beim Aufteilen einer Menge oder eines Containers

Das Aufteilen eines Containers oder einer Menge bewegt weder die Quell- noch die Zielcontainer. Es impliziert auch nicht, dass eine Prozess-Aktivität (Durchsatz) an einem der Container durchgeführt wurde.

Sie müssen einen eindeutigen Containernamen angeben, wenn Sie einen neuen Container erstellen. Sie können einen Containernamen entweder manuell eingeben oder ihn automatisch von der Anwendung erstellen lassen. Es werden geschlossene Container von der Anwendung einbezogen, wenn ein eindeutiger Name geprüft wird.

Aktivieren Sie das Kontrollkästchen "Namen automatisch erstellen", um den Containernamen unter Verwendung der definierten Nummerierungsregeln von der Anwendung erstellen zu lassen. Wenn Sie dieses Kontrollkästchen aktivieren, löscht und deaktiviert die Anwendung das Feld "Zielcontainer".

Wenn die Option "Namen automatisch erstellen" ausgewählt wird, ohne vorher eine automatische Nummerierungsregel festzulegen, wird eine Fehlermeldung angezeigt.

Die Anwendung verwendet zum Erstellen des Zielcontainernamens die Nummerierungsregeln, die dem Eltern-Container zugeordnet sind, von dem aus der Kind-Container aufgeteilt wird. Wenn Sie auf die Schaltfläche "Abschicken" klicken, erstellt die Anwendung den Namen des neuen Eltern-Containers.

Nummerierungsregeln können für mehrere, dem Container zugeordnete Modellierungsobjektinstanzen festgelegt werden. Nummerierungsregeln folgen dieser Standardrangfolge:

1. Containerlevel
2. Produktionsauftrag
3. Produkt
4. Produktfamilie
5. Fabrik

Hinzufügen von betroffenen Containern zu Ereignissen

Die Anwendung kann Container automatisch zu Qualitätsereignissen hinzufügen, wenn Sie die Transaktion "Menge aufteilen" für einen Container durchführen, der mit mindestens einem Qualitätsereignis verknüpft ist. Wenn z. B. ein Quell-Container mit einem Ereignis verknüpft ist, fügt die Anwendung alle neuen Container, die durch die Aufteilung des betreffenden Quell-Containers entstehen, automatisch zum Qualitätsereignis hinzu. Ein Systemadministrator konfiguriert die Transaktion "Aufteilen" in der Modellierung, um Container zu Ereignissen hinzuzufügen. Informationen zum automatischen Hinzufügen von betroffenen Containern zu Qualitätsereignissen finden Sie im Opcenter Execution Medical Device and Diagnostics Event Management-Benutzerhandbuch und im Opcenter Execution Core Event Management-Benutzerhandbuch sowie im Opcenter Execution Medical Device and Diagnostics Nonconformance Management-Benutzerhandbuch und im Opcenter Execution Core Nonconformance Management-Benutzerhandbuch.

Seite "Menge aufteilen"

Die Seite "Menge teilen" ermöglicht es Ihnen:

• Einen erlaubten Container auszuwählen.
• Einen oder mehrere neue Ziel-Container zu definieren, die die aufzuteilende Menge aus dem Quell-Container aufnehmen sollen.
• Die Menge, die aus dem Quell-Container in die neuen Ziel-Container aufgeteilt werden soll.
• Wählen Sie eine Option, um den Quell-Container zu schließen, wenn die gesamte Container-Menge in die Zielcontainer verschoben wurde.

Vorgehensweise zum Aufteilen der Menge eines Containers

Gehen Sie folgendermaßen vor, um die Seite oder das Popup "Menge aufteilen" zu verwenden, um den gesamten Inhalt eines Quell-Containers in einen oder mehrere neue Ziel-Container aufzuteilen.

1. Öffnen Sie die Seite Menge teilen im Menü Container. Die Seite "Menge aufteilen" wird angezeigt.
2. Wählen Sie einen Container im Container-Feld aus oder geben Sie ihn ein. Die Anwendung aktualisiert den Container-Kopf und die Auswahlfelder, sodass diese Informationen für den angegebenen Container angezeigt werden. Werte werden nur angezeigt, wenn sie für den Container eingegeben wurden.
Hinweis: Die Anwendung füllt das Feld "Container" und den Container-Kopf automatisch, wenn Sie von einer anderen Fertigungsseite auf das Popup "Menge aufteilen" zugreifen.
3. Klicken Sie auf Neue Zeile hinzufügen oder wählen Sie eine Zeile, um einen neuen Container zu starten.
4. Geben Sie einen Namen für den neuen Container im Feld Zu Container-Name in der ersten Zeile der Tabelle "Container hinzufügen" und "Aufzuteilende Menge" ein.

Oder

Aktivieren Sie das Kontrollkästchen Namen automatisch erstellen.

5. Geben Sie die Menge für den neuen Container in das zugehörige Feld Menge ein.
6. Wiederholen Sie die Schritte 3 bis 5, wenn Sie einen anderen neuen Container und Menge hinzufügen möchten.
7. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
8. Klicken Sie auf Teilen oder in der Befehlsleiste auf Abschicken. Die angegebene Menge oder Mengen werden in den Ziel-Container aufgeteilt, die Anwendung löscht die Felder und zeigt dann eine Bestätigungsmeldung an.
