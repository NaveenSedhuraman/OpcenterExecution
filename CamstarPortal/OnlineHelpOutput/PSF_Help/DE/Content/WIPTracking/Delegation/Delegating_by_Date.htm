Delegieren nach Datum

Die Anwendung bietet eine Suchfunktion, mit der Sie nach Delegierungsdefinitionen, die einen Datumsbereich angeben, suchen und diese abrufen können. Sie fügen Delegierungsdefinitionen in der Tabelle "Suchergebnisse" hinzu.

Beim Suchen nach Datum

Mit Delegierungsdefinitionen können Sie Aufgaben für einen angegebenen Datumsbereich von einem Mitarbeiter an einen anderen delegieren. Die Anwendung zeigt mehrere Felder an, nach denen Sie die Suche eingrenzen können. Beachten Sie Folgendes:

• Sie können nach Anwendung suchen. Das Änderungsmanagement ist die einzige Anwendung, die derzeit verfügbar ist.
• Die Anwendung wählt standardmäßig Delegierungsdefinitionen mit einem Status "Aktiv" oder "Künftig" aus.

Es werden erst Delegierungsdefinitionen in der Tabelle "Suchergebnisse" angezeigt, wenn Sie mindestens eine hinzufügen, die den Suchkriterien entspricht. Informationen zu Delegierungsdefinitionen finden Sie unter "Beim Delegieren nach Datum".

Durchführen einer Suche nach Datum

Die Anwendung führt folgende Vorgänge aus, wenn Sie auf der Registerkarte "Nach Datum" auf die Schaltfläche "Suchen" klicken:

• Sie zeigt die Ergebnisse in der Tabelle "Suchergebnisse" an
• Sie behält die Suchkriterien in den Filterfeldern bei

Die Anwendung zeigt Suchergebnisse in aufsteigender Reihenfolge sortiert nach folgenden Feldern an:

• Anfangsdatum
• Enddatum
• Zugewiesener Mitarbeiter
• Zielmitarbeiter

Die Anwendung fügt der Tabelle eine Umblätter-Funktion hinzu, wenn die Suche mehr Ergebnisse zurückgibt als in den Zeilen der Tabelle angezeigt werden können. Siemens bietet auch die Möglichkeit, Suchergebnisse im Excel- oder CSV-Format zu exportieren.

Beim Delegieren nach Datum

Sie müssen eine Delegierungsdefinition hinzufügen, um Aufgaben für einen angegebenen Datumsbereich von einem Mitarbeiter an einen anderen neu zuzuweisen. Sie können eine Delegierungsdefinition hinzufügen, indem Sie in der Tabelle "Suchergebnisse" auf die Schaltfläche "Neue Zeile hinzufügen" klicken.

Hinzufügen von Delegierungsdefinitionen

Das Popup "Aufgaben nach Datum delegieren" wird geöffnet, wenn Sie auf "Neue Zeile hinzufügen" in der Tabelle "Suchergebnisse" auf der Registerkarte "Nach Aufgabe" klicken. Über das Popup "Aufgaben nach Datum delegieren" können Sie für einen angegebenen Datumsbereich eine oder mehrere Aufgaben für die ausgewählte Anwendung vom zugewiesenen Mitarbeiter an einen Zielmitarbeiter delegieren. Sie müssen bei der Übertragung von Aufgaben auch einen Delegierungsgrund auswählen. Informationen zum Definieren von Delegierungsgründen finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch und im Opcenter Execution Core Modeling-Benutzerhandbuch.

Folgendes gilt beim Hinzufügen einer Delegierungsdefinition:

• Der zugewiesene Mitarbeiter und der Zielmitarbeiter dürfen nicht dieselbe Person sein.
• Das Startdatum muss in der Zukunft liegen.
• Das Enddatum muss nach dem Startdatum liegen.
• Delegierungsdefinitionen dürfen keine Duplikate sein. Eine Duplikatdelegierungsdefinition ist eine Definition, die denselben zugewiesenen Mitarbeiter für dieselbe Anwendung innerhalb desselben Datumsbereichs angibt, selbst wenn die Datumsbereiche sich nur um eine Minute überlappen.

Mit dem Kontrollkästchen "Delegierungsweiterleitung aktivieren" können Sie angeben, ob Aufgaben, die an den Zielmitarbeiter delegiert wurden, an einen anderen Mitarbeiter weitergeleitet werden dürfen. Die Anwendung leitet die Übertragung an einen anderen Mitarbeiter weiter, wenn der Zielmitarbeiter in der Definition, die Sie hinzufügen, ein zugewiesener Mitarbeiter in einer anderen Definition ist. Die Anwendung leitet die Übertragung an den Zielmitarbeiter weiter, der in der zweiten Definition angegeben ist.

Die Anwendung führt folgende Vorgänge aus, wenn Sie im Popup "Aufgaben nach Datum delegieren" auf die Schaltfläche "OK" klicken:

• Sie aktualisiert die Tabelle "Suchergebnisse"

Bearbeiten von Delegierungsdefinitionen

Sie können eine zukünftige oder aktive Übertragung durch Auswählen einer Delegierungsdefinition in der Tabelle "Suchergebnisse" und Klicken auf die Schaltfläche "Ausgewählte Zeile bearbeiten" bearbeiten. Die Felder, die Sie in dem Popup "Aufgaben nach Datum delegieren" bearbeiten können, hängen vom Status der Delegierung ab. Es gilt Folgendes:

Ist der Status . . .

Dann können Sie . . .

Künftig

alle Felder bearbeiten.

Aktiv

nur das Enddatum bearbeiten. Das Enddatum muss in der Zukunft liegen. Die Anwendung setzt alle anderen Felder auf "Nur anzeigen".

Abgelaufen

keine Felder bearbeiten. Die Anwendung setzt alle Felder auf "Nur anzeigen".

Löschen von Delegierungsdefinitionen

Sie können eine zukünftige Delegierung durch Auswählen einer Delegierungsdefinition in der Tabelle "Suchergebnisse" und Klicken auf die Schaltfläche "Ausgewählte Zeile löschen" löschen. Ein Bestätigungsdialogfeld wird geöffnet, in dem Sie zum Bestätigen des Löschvorgangs aufgefordert werden. Sie können keine aktiven oder abgelaufenen Übertragungen löschen.

Vorgehensweise zum Suchen nach Delegierungsdefinitionen

Gehen Sie folgendermaßen vor, um nach Delegierungsdefinitionen zu suchen:

1. Öffnen Sie die Seite Delegierungssuche im Menü Suche. Die Seite "Delegierungssuche" wird mit ausgewählter Registerkarte "Nach Aufgabe" angezeigt. Ein Filtersymbol gibt an, dass Kriterien angewendet wurden.
2. Klicken Sie auf die Registerkarte Nach Datum. Die Registerkarte "Nach Datum" wird angezeigt. Ein Filtersymbol gibt an, dass Kriterien angewendet wurden.
3. Hier können Sie einen oder mehrere Filter eingeben oder auswählen, um Ihre Suche einzugrenzen.
4. Klicken Sie auf Suchen. Die Anwendung zeigt die Delegierungsdefinitionen, die mit Ihren Suchkriterien übereinstimmen, in der Tabelle "Suchergebnisse" an.

Vorgehensweise zum Hinzufügen einer Delegierungsdefinition

Gehen Sie folgendermaßen vor, um eine Delegierungsdefinition hinzuzufügen:

1. Befolgen Sie die Schritte unter "Vorgehensweise zum Suchen nach Delegierungsdefinitionen".
2. Klicken Sie in der Tabelle "Suchergebnisse" auf Neue Zeile hinzufügen. Das Popup "Aufgaben nach Datum delegieren" wird angezeigt.
3. Wählen Sie in der Liste Zugewiesener Mitarbeiter einen Mitarbeiter aus.
4. Wählen Sie in der Liste Zielmitarbeiter einen Mitarbeiter aus.
5. Wählen Sie in der Liste Delegierungsgrund einen Delegierungsgrund aus.
6. Wählen Sie eine Anwendung aus der Liste Anwendung aus.
7. Geben Sie das Startdatum für diese Delegierungsdefinition im Feld Startdatum ein oder wählen Sie es aus dem Kalendersteuerelement aus.
8. Geben Sie das Enddatum für diese Delegierungsdefinition im Feld Enddatum ein oder wählen Sie es aus dem Kalendersteuerelement aus.
9. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
10. Klicken Sie auf OK. Die Anwendung speichert die Delegierungssdefinition, zeigt eine Erfolgsnachricht an, schließt das Popup "Aufgaben nach Datum delegieren" und aktualisiert die Tabelle "Suchergebnisse".

Vorgehensweise zum Bearbeiten einer Delegierungsdefinition

Gehen Sie folgendermaßen vor, um eine Delegierungsdefinition zu bearbeiten:

1. Befolgen Sie die Schritte unter "Vorgehensweise zum Suchen nach Delegierungsdefinitionen".
2. Wählen Sie eine zukünftige oder aktive Delegierungsdefinition in der Tabelle "Suchergebnisse" aus.
3. Klicken Sie auf Ausgewählte Zeile bearbeiten.. Das Popup "Aufgaben nach Datum delegieren" wird angezeigt.
4. Wie lautet der Status der Delegierungsdefinition?

Ist der Status . . .

Dann können Sie . . .

Künftig

alle Felder nach Belieben bearbeiten.

Aktiv

das Enddatum nach Belieben bearbeiten.
5. Klicken Sie auf OK. Die Anwendung aktualisiert die Delegierungsdefinition, zeigt eine Erfolgsmeldung an, schließt das Popup "Aufgaben nach Datum delegieren" und aktualisiert die Tabelle "Suchergebnisse".

Vorgehensweise zum Löschen einer Delegierungsdefinition

Gehen Sie folgendermaßen vor, um eine Delegierungsdefinition zu löschen:

1. Befolgen Sie die Schritte unter "Vorgehensweise zum Suchen nach Delegierungsdefinitionen".
2. Wählen Sie eine zukünftige Delegierungsdefinition in der Tabelle "Suchergebnisse" aus.
3. Klicken Sie auf Ausgewählte Zeile löschen.. Ein Bestätigungsdialogfeld wird geöffnet, in dem Sie zum Bestätigen des Löschvorgangs aufgefordert werden.
4. Klicken Sie auf Ja. Die Anwendung löscht die Delegierungsdefinition, zeigt eine Erfolgsnachricht an, schließt das Bestätigungsdialogfeld und aktualisiert die Tabelle "Suchergebnisse".
