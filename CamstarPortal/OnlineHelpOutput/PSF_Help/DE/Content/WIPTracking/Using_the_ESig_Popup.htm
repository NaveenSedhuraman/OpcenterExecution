Verwenden des Popups "Elektr. Unterschrift"

Das Popup "Elektronische Unterschrift" wird in folgenden Fällen automatisch angezeigt:

• Die Anwendung wurde so konfiguriert, dass elektronische Unterschriften erfasst werden, wenn ein Benutzer eine Transaktion ausführt.

Oder

• Eine Transaktion versucht, einen Prozesstimer anzuhalten, der eine Anforderung zur elektronischen Unterschrift hat und außerhalb der Grenzen liegt.

Die Darstellung des Popups hängt von der Fertigungsseite oder dem Popup ab, von dem darauf zugegriffen wurde.

Ist dieselbe erforderliche elektronische Unterschrift mehrmals für dieselbe Transaktion erforderlich, werden diese Unterschriften kombiniert und als eine einzige Anforderung im Popup "Elektr. Unterschrift" angezeigt

Wenn Sie auf das Popup "Elektr. Unterschrift" zugreifen von:

• Einer Mehr-Container-Transaktionsseite oder einem Mehr-Container-Popup – Die Anwendung fügt zusätzliche Felder hinzu, mit denen Sie mehreren Containern Unterschriften hinzufügen können.
• Eine einzelne Transaktionsseite oder ein Popup - Das Popup hat weniger Felder und Sie können nur Unterschriften für den auf der Transaktionsseite oder dem Popup ausgewählten Container eingeben.

Beim Hinzufügen von Unterschriften zum Popup "Elektr. Unterschrift"

Wenn Sie auf die Schaltfläche "Unterschrift hinzufügen" klicken, bestätigen Sie, dass die eingegebenen Unterzeichner- und Mitunterzeichnerinformationen die elektronische Unterschrifts-Anforderung der Transaktion erfüllen.

Die Anwendung überprüft die Unterschriften, um festzustellen, ob diese Bedingungen erfüllt werden:

• Der Unterzeichner und Mitunterzeichner sind berechtigte Mitarbeiter und haben gültige Passwörter eingegeben.
• Die Rolle oder Mitunterzeichnerrolle, die in der Anforderung zur elektronischen Unterschrift definiert ist, ist gültig. Daraus folgt, dass die Rolle:
• Mit Unterzeichner oder Mitunterzeichner verknüpft ist
• Teil der elektronischen Unterschrift Rollengruppe, die dem Unterzeichner oder Mitunterzeichner zugewiesen ist.

Wenn die Unterschriften gültig sind, fügt die Anwendung die Informationen der ausgewählten Zeile in der Kind-Tabelle hinzu, wählt die nächste Kind-Zeile automatisch aus und stellt den Cursor in das Unterzeichnerfeld der nächsten Unterschrift.

Wenn alle Unterschriften für die ausgewählte Unterschriftsbedeutung erfasst werden, reagiert die Anwendung wie folgt:

• Die Hintergrundfarbe der fertiggestellten Bedeutung wechselt zu grün
• Die nächste Bedeutung in der Tabelle wird ausgewählt, falls zutreffend
Wichtig: Wenn Sie eine Unterschrift verändern müssen, so müssen Sie die Unterschrift aus der Bedeutung löschen und sie mit den neuen Informationen neu hinzufügen.

Ungültiger Informationseintrag

Wenn Sie eine falsche ID oder ein falsches Kennwort eingegeben haben, zeigt die Anwendung eine Fehlermeldung im elektronische Unterschriften Popup an. Sie können die Informationen entweder korrigieren und die Unterschrift hinzufügen oder das elektronische Unterschriften-Popup schließen. Wenn Sie das Popup-Fenster schließen, ohne Änderungen vorzunehmen, wird die Transaktion nicht abgeschlossen werden.

Beim Löschen von Unterschriften

Die Anwendung bietet zwei Möglichkeiten, um eine Unterschrift aus dem elektronischen Unterschriften Feld zu löschen. Durch Klicken auf:

• "Ausgewählte Unterschriften löschen" werden alle ausgewählten Unterschriften aus der momentan ausgewählten Unterschriftsanforderung entfernt.
• Durch Klicken auf die Schaltfläche "Zurücksetzen" werden alle Unterschriften aus der Tabelle entfernt.

Beim Abschicken elektronischer Unterschriften

Wenn Sie erfasste Unterschriften abschicken, überprüft die Anwendung Folgendes:

• Ob alle Unterschriften für jede Anforderung hinzugefügt wurden.
• Ob derselbe Benutzer nicht mehrmals für dieselbe Anforderung unterschrieben hat. Allerdings können Mitunterzeichner mehrere Unterschriften erfüllen, solange dieser Mitunterzeichner die entsprechenden Rollen hat.

Wenn die Voraussetzungen erfüllt sind, schließt sich das Popup "Elektr. Unterschrift" und die Transaktionsverarbeitung geht weiter. Wenn nicht, lässt die Anwendung das Popup offen und zeigt eine Fehlermeldung an.

Felddefinitionen des elektronische Unterschriften-Popups

Diese Tabelle definiert die Felder im elektronische Unterschriften-Popup.

Feld

Definition

Typ

Eingabe der elektronischen Unterschrift

Zeigt alle Unterschrifts-Anforderungen für elektronische Unterschriften der Transaktion an. Die Anwendung erweitert standardmäßig die erste Unterschriften-Anforderungszeile, wählt die erste leere Zeile der Kind-Tabelle aus und platziert den Cursor im Unterzeichnerfeld.

Jede Unterschriften-Anforderungszeile zeigt folgende Informationen an:

• Bedeutung: Name oder Beschreibung, die es Ihnen erlaubt, den Grund für die Unterschrift festzustellen.
• Anzahl: Anzahl der notwendigen Unterschriften, um die Bedeutung zu erfüllen. Dieser Wert bestimmt die Anzahl der Kind-Zeilen, die für die Bedeutung angezeigt werden.
• Rolle: Benutzerrolle, die für die Unterzeichnung der Bedeutung erforderlich ist.
• Mitunterzeichnerrolle: Nutzerrolle, die eine zusätzliche Unterschrift zur Verfügung stellt, sollte der Hauptunterzeichner nicht verfügbar sein oder unzureichend autorisiert sein, um allein zu unterzeichnen. Die Mitunterzeichnerrollen-Information wird nur angezeigt, wenn die Mitunterzeichnerrolle mit den Unterschrifts-Anforderungen im Modellierungsobjekt mit der Anforderung zur elektronischen Unterschrift verknüpft wurde.

Nur Anzeige

Eingabe der elektronischen Unterschrift
• Container: <#> (Zum Anzeigen anklicken): Ein Link, der ein Popup anzeigt, das die Container auflistet, auf die die Unterschriften zutreffen. Dieser Link wird nur angezeigt, wenn Sie das Popup "Elektr. Unterschrift" von einer Mehr-Container-Transaktionsseite oder einem Popup aus aufgerufen haben.

Hinweis: Elektronische Unterschrifts-Bedeutungen und Rollen werden in der Modellierung definiert und mit dem Elektronischen Unterschrifts-Anforderung-Modellierungsobjekt verknüpft. Der Zähler wird im Elektronische Unterschrift Anforderung-Modellierungsobjekt festgelegt.

Nur Anzeige

Unterzeichner

Name des Nutzers, der eine Unterschrift für die Transaktion geleistet hat.

Nur Anzeige

Mitunterzeichner

Name des Nutzers, der eine zusätzliche oder alternative elektronische Unterschrift geleistet hat.

Nur Anzeige

Mitunterzeichnungsgrund

Grund für die zusätzliche Unterschrift.

Nur Anzeige

Anmerkungen

Zusätzliche Informationen, die vom Unterzeichner oder vom Mitunterzeichner für deren elektronische Unterschrift eingetragen wurden. Das Feld akzeptiert maximal 255 Zeichen. Wenn die Kommentare 255 Zeichen überschreiten, verkürzt die Anwendung die Daten.

Nur Anzeige

Jede Bedeutung extra abzeichnen (Zeilenauswahl erforderlich)

Wenn ausgewählt, kann ein Nutzer eine Unterschrift für nur eine Bedeutung gleichzeitig hinzufügen (Allerdings kann diese Unterschrift auf mehrere Container zutreffen). Die Auswahl dieser Option deaktiviert das Unterzeichner Rollenfeld.

Hinweis: Dieses Feld wird nur angezeigt, wenn Sie das Popup "Elektr. Unterschrift" von einer Mehr-Container-Transaktionsseite oder einem Popup aus aufgerufen haben.

Optional

Gruppenunterzeichnung aller erfüllter Anforderungen (keine Zeilenauswahl notwendig)

Wenn ausgewählt, kann ein Benutzer mehreren Bedeutungen eine Unterschrift hinzufügen. Die Anwendung fügt die Unterschrift jeder Bedeutung hinzu, die von der Benutzerrolle, dem Benutzernamen und dem Kennwort erfüllt werden. Die ist die Standardeinstellung. Die Auswahl dieser Option aktiviert das Unterzeichner Rollenfeld.

Hinweis: Dieses Feld wird nur angezeigt, wenn Sie das Popup "Elektr. Unterschrift" von einer Mehr-Container-Transaktionsseite oder einem Popup aus aufgerufen haben.

Optional

Unterzeichner-Rolle

Die Rolle des Benutzers liefert eine elektronische Unterschrift. Dieses Feld ist ein erforderliches Feld im Batchmodus (wenn die Option "Gruppenabzeichnung aller erfüllten Bedeutungen (keine Zeilenauswahl erforderlich)" ausgewählt ist). Die Anwendung aktualisiert die Auswahl im Unterzeichner-Rollenfeld anhand der Rollen der verschiedenen Bedeutungen in der elektronische Unterzeichner-Tabelle. Andernfalls deaktiviert die Anwendung dieses Feld und setzt den Wert auf die Nutzerrolle zurück, die in der Unterschrift-Anforderung der ausgewählten elektronischen Unterschrift-Tabelle festgelegt ist.

Hinweis: Dieses Feld wird nur angezeigt, wenn Sie das Popup "Elektr. Unterschrift" von einer Mehr-Container-Transaktionsseite oder einem Popup aus aufgerufen haben.

Erforderlich

Unterzeichner

Der Name des Benutzers, der der zugehörigen Fertigungstransaktion eine elektronische Unterschrift hinzufügt.

Dieses Feld ist ein erforderliches Feld im Batchmodus (wenn die Option "Gruppenabzeichnung aller erfüllten Bedeutungen (keine Zeilenauswahl erforderlich)" ausgewählt ist).

Optional/ Erforderlich

Unterzeichner-Kennwort

Das Kennwort, das den Unterzeichner identifiziert.

Optional

Zusätzliche Felder anzeigen/ausblenden

Optionale Felder sind standardmäßig ausgeblendet, wenn die Anwendung eine Seite oder ein Popup anzeigt. Durch Klicken auf die Steuerung "Anzeigen" werden die Felder angezeigt. Durch Klicken auf die Steuerung "Verstecken" werden die Felder ausgeblendet.

  

Mitunterzeichner

Der Name des Benutzers, der der zugehörigen Fertigungstransaktion eine Mitunterschrift hinzufügt.

Optional

Mitunterzeichner-Kennwort

Das Kennwort, dass den Nutzer identifiziert, der eine Mitunterschrift der zugehörigen Fertigungstransaktion hinzufügt. Die Anwendung verlangt ein Mitunterzeichner-Kennwort, wenn Sie eine Nutzer-ID in das Mitunterzeichner-Feld eingeben.

Optional/ Erforderlich

Mitunterzeichnungsgrund

Falls vorhanden, wählen Sie einen Grund aus, warum ein Mitunterzeichner für die verbunden Transaktion verwendet wird.

Optional

Anmerkungen

Zusätzliche Informationen für die elektronische Unterschrift. Das Feld akzeptiert maximal 255 Zeichen. Wenn die Kommentare 255 Zeichen überschreiten, verkürzt die Anwendung die Daten.

Optional

Schaltflächendefinitionen des Popups "Elektr. Unterschrift"

Diese Tabelle definiert die Schaltflächen im Popup "Elektr. Unterschrift".

Schaltfläche

Funktion der Schaltfläche . . .

Unterschrift hinzufügen

Fügen Sie die Unterzeichner- und/oder Mitunterzeichnerunterschrift in der Kind-Zeile hinzu, die Sie in der Tabelle "Eintrag der elektronischen Unterschrift" ausgewählt haben. Wenn Sie eine falsche ID oder ein falsches Passwort eingegeben haben, zeigt die Anwendung sofort eine Fehlermeldung an.

Schließen

Schließen Sie das Popup "Elektr. Unterschrift", ohne eine Unterschrift abzuschicken. Die Anwendung löscht alle Unterschriften von der Anforderung zur elektronischen Unterschrift.

Ausgewählte Unterschriften löschen

Löscht die ausgewählten Unterschriften aus der Tabelle "Eintrag der elektronischen Unterschrift". Die Anwendung entfernt die Unterschriften aus den ausgewählten Kind-Zeilen.

Zurücksetzen

Löscht alle Unterschriften aus der Tabelle "Eintrag der elektronischen Unterschrift". Die Anwendung entfernt die Unterschriften aus den Kind-Zeilen.

Unterschriften abschicken

Schicken Sie die Unterschriften ab, schließen Sie das Popup "Elektr. Unterschrift" und beenden Sie die Transaktion. Die Anwendung ermöglicht es Ihnen nur dann, Unterschriften abzuschicken und die Transaktion zu beenden, wenn alle Unterschriftsanforderungen erfüllt wurden. Wenn Sie auf "Unterschriften abschicken" klicken, ohne alle Unterschriftsanforderungen erfüllt zu haben, zeigt die Anwendung eine Fehlermeldung an und lässt das Popup "Elektr. Unterschrift" weiterhin geöffnet.

Vorgehensweise zum Erfassen elektronischer Unterschriften für eine Einzelcontainer-Transaktion

Gehen Sie folgendermaßen vor, um eine elektronische Unterschrift für eine Einzelcontainer-Transaktion zu Sammeln:

1. Führen Sie eine Fertigungs-Transaktion aus, die eine zugehörige elektronische Unterschriften-Anforderung hat. Das Popup "Elektr. Unterschrift" wird angezeigt. Die Anwendung wählt die erste Unterschriften (Kind-)Zeile der ersten nicht erfüllten Unterschriften-Anforderung standardmäßig aus.
2. Wählen Sie eine andere Unterschriftsanforderung aus, falls erforderlich.
3. Füllen Sie diese Felder aus:
• Unterzeichner
• Unterzeichner-Kennwort
4. Füllen Sie diese Felder aus, wenn ein Mitunterzeichner erforderlich ist:
• Mitunterzeichner
• Mitunterzeichner-Kennwort
• Mitunterzeichnungsgrund
5. Lassen Sie den Unterzeichner oder Mitunterzeichner einen Kommentar eintragen, falls erforderlich.
6. Klicken Sie auf Unterschrift hinzufügen. Die Anwendung überprüft und fügt die Unterschrifteninformationen der Anforderung hinzu, die in der Tabelle "Eingabe der elektronischen Unterschrift" ausgewählt wurde. Wenn zusätzliche Unterschriften benötigt werden, fügt die Anwendung die nächste Kind-Zeile automatisch hinzu und stellt den Cursor in das Unterzeichner-Feld der nächsten Unterschrift.
7. Wiederholen Sie die Schritte 2-6, um weitere Unterschriften für die ausgewählte Anforderung hinzuzufügen. Sobald alle Unterschriften für eine Anforderung erfasst wurden, färbt die Anwendung die Hintergrundfarbe der Elternzeile grün und wählt die nächste leere Unterschriftenzeile aus.
8. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
9. Klicken Sie auf Unterschriften Abschicken. Die Anwendung schließt das Popup "Elektr. Unterschrift" und zeigt eine Statusmeldung an, die besagt, dass die Transaktion beendet wurde.

Vorgehensweise zum Erfassen elektronischer Unterschriften für mehrere Bedeutungen gleichzeitig

Gehen Sie folgendermaßen vor, um elektronische Unterschriften für mehrere Bedeutungen auf einmal zu erfassen:

1. Greifen Sie auf eine Multi-Container-Transaktionsseite oder ein -Popup zu, die bzw. das eine zugewiesene erforderliche elektronische Unterschrift hat.
2. Wählen sie einen oder mehrere Container aus.
3. Füllen Sie die erforderlichen Felder auf der Seite oder dem Popup aus.
4. Klicken Sie auf die Transaktionsschaltfläche, um die Transaktion auszuführen. Das Popup "Elektr. Unterschrift" wird angezeigt, die Option "Gruppenabzeichnung aller erfüllten Bedeutungen (keine Zeilenauswahl erforderlich)" ist ausgewählt und das Feld "Unterzeichner-Rolle" ist aktiviert.
5. Wählen Sie eine Rolle im Unterzeichnerrolle-Feld aus.
6. Füllen Sie diese Felder aus:
• Unterzeichner
• Unterzeichner-Kennwort
7. Füllen Sie diese Felder aus, wenn ein Mitunterzeichner erforderlich ist:
• Mitunterzeichner
• Mitunterzeichner-Kennwort
• Mitunterzeichnungsgrund
8. Lassen Sie den Unterzeichner oder Mitunterzeichner einen Kommentar eintragen, falls erforderlich.
9. Klicken Sie auf Unterschrift hinzufügen. Die Anwendung überprüft und fügt die Unterschriftsinformationen für alle Unterschriftsanforderungen hinzu, die der Nutzerrolle, dem Nutzernamen und dem Kennwort gerecht werden. Danach wird die hinzugefügte Unterschrift unter jeder Eltern-Zeile markiert.
10. Wiederholen Sie Schritte 5-9, um zusätzliche Unterschriften hinzuzufügen. Sobald alle Unterschriften für eine Anforderung erfasst wurden, färbt die Anwendung die Hintergrundfarbe der Elternzeile grün und wählt die nächste leere Unterschriftenzeile aus.
11. Klicken Sie auf Unterschriften Abschicken. Die Anwendung schließt das Popup "Elektr. Unterschrift" und zeigt eine Statusmeldung an, die besagt, dass die Transaktion beendet wurde.

Vorgehensweise zum Erfassen elektronischer Unterschriften für eine einzelne Bedeutung

Gehen Sie folgendermaßen vor, um elektronische Unterschriften für eine einzelne Bedeutung zu erfassen:

1. Greifen Sie auf eine Multi-Container-Transaktionsseite oder ein -Popup zu, die bzw. das eine zugewiesene erforderliche elektronische Unterschrift hat.
2. Wählen sie einen oder mehrere Container aus.
3. Füllen Sie die erforderlichen Felder auf der Seite oder dem Popup aus.
4. Klicken Sie auf die Transaktionsschaltfläche, um die Transaktion auszuführen. Das Popup "Elektr. Unterschrift" wird angezeigt, die Option "Gruppenabzeichnung aller erfüllten Bedeutungen (keine Zeilenauswahl erforderlich)" ist ausgewählt und das Feld "Unterzeichner-Rolle" ist aktiviert.
5. Wählen Sie die Option: Individuelle Abmeldung für eine Bedeutung durchführen (Zeilenauswahl erforderlich). Die Anwendung wählt die erste Unterschriften (Kind-)Zeile der ersten nicht erfüllten Unterschriften-Anforderung standardmäßig aus.
6. Wählen Sie eine andere Unterschriftsanforderung aus, falls erforderlich.
7. Füllen Sie diese Felder aus:
• Unterzeichner
• Unterzeichner-Kennwort
8. Füllen Sie diese Felder aus, wenn ein Mitunterzeichner erforderlich ist:
• Mitunterzeichner
• Mitunterzeichner-Kennwort
• Mitunterzeichnungsgrund
9. Lassen Sie den Unterzeichner oder Mitunterzeichner einen Kommentar eintragen, falls erforderlich.
10. Klicken Sie auf Unterschrift Hinzufügen. Die Anwendung überprüft und fügt die Unterschrifteninformationen der Anforderung hinzu, die in der Tabelle Eintrag der elektronischen Unterschrift ausgewählt wurde. Sollten weitere Unterschriften erforderlich sein, so wählt die Anwendung die nächsten Kind-Zeilen automatisch aus und setzt den Cursor in das Unterzeichner-Feld der nächsten Unterschrift.
11. Wiederholen Sie Schritte 6-10, um zusätzliche Unterschriften hinzuzufügen. Sobald alle Unterschriften für eine Anforderung erfasst wurden, färbt die Anwendung die Hintergrundfarbe der Elternzeile grün und wählt die nächste leere Unterschriftenzeile aus.
12. Klicken Sie auf Unterschriften Abschicken. Die Anwendung schließt das Popup "Elektr. Unterschrift" und zeigt eine Statusmeldung an, die besagt, dass die Transaktion beendet wurde.
