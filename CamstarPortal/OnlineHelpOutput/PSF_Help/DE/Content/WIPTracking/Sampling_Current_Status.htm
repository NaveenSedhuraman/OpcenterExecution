Aktualisierung aktueller Stichprobenstatus

Die Seite "Aktualisierung aktueller Stichprobenstatus" erlaubt es Ihnen, sowohl den "Stichprobenratenzähler" als auch die "Stichprobenplan-Inspektionsebene" entweder für einen bestimmten "Aktueller Stichprobenstatus"-Datensatz oder für alle aktuellen "Aktueller Stichprobenstatus"-Datensätze zu aktualisieren, die mit einem Produkt verknüpft sind. Sie können einen Wert im Feld "Neuer Zählerwert für Stichprobenrate" eingeben, um die Häufigkeit zu ändern, mit der Stichproben aus Containern entnommen werden. Sie können auch eine Prüfebene im Feld "Neue Prüfebene" auswählen, um die Prüfebene für den Stichprobenplan zu ändern. Informationen zum Definieren von Stichprobenplänen finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch und im Opcenter Execution Core Modeling-Benutzerhandbuch.

Beim Verwenden der Seite "Aktualisierung aktueller Stichprobenstatus"

Beim Verwenden der Seite "Aktualisierung aktueller Stichprobenstatus" gelten folgende Regeln:

• Nur Produkte mit containerbasierten Stichprobenplänen werden im Feld "Produkt" angezeigt.
• Änderungen, die auf dieser Seite vorgenommen werden, treten beim nächsten Container in Kraft, der die Stichprobenanforderung erfüllt. Container, die bereits in Bearbeitung sind, sind nicht betroffen.
• Für Losstichproben gilt diese Seite nicht.
Seite "Aktualisierung aktueller Stichprobenstatus"

Vorgehensweise zum Aktualisieren des Stichprobenratenzählers

Gehen Sie folgendermaßen vor, um den Stichprobenratenzähler zu aktualisieren:

1. Öffnen Sie die Seite Aktualisierung aktueller Stichprobenstatus im Menü Container. Die Seite "Aktualisierung aktueller Stichprobenstatus" wird angezeigt.
2. Wählen Sie ein Produkt im Produkt-Feld aus. Die Anwendung zeigt eine Liste von "Aktueller Stichprobenstatus"-Datensätzen in der Tabelle "Details des aktuellen Stichprobenstatus" an.
3. Aktivieren Sie das Kontrollkästchen des "Aktueller Stichprobenstatus"-Datensatzes, für den Sie den Stichprobenratenzähler aktualisieren möchten.

Oder

Aktivieren Sie das Kontrollkästchen in der Kopfzeile der Tabelle "Details des aktuellen Stichprobenstatus", um alle Stichprobenratenzähler zu aktualisieren.

4. Geben Sie einen Wert in das Feld Neuer Zählerwert für Stichprobenrate ein.
5. Klicken Sie auf Aktualisieren oder in der Befehlsleiste auf Abschicken. Die Anwendung aktualisiert die Tabelle "Details des aktuellen Stichprobenstatus".

Vorgehensweise zum Aktualisieren der Prüfebene

Gehen Sie folgendermaßen vor, um die Prüfebene zu aktualisieren:

1. Öffnen Sie die Seite Aktualisierung aktueller Stichprobenstatus im Menü Container. Die Seite "Aktualisierung aktueller Stichprobenstatus" wird angezeigt.
2. Wählen Sie ein Produkt im Produkt-Feld aus. Die Anwendung zeigt eine Liste von "Aktueller Stichprobenstatus"-Datensätzen in der Tabelle "Details des aktuellen Stichprobenstatus" an.
3. Aktivieren Sie das Kontrollkästchen des "Aktueller Stichprobenstatus"-Datensatzes, für den Sie die Prüfebene aktualisieren möchten.
4. Aktivieren Sie optional das Kontrollkästchen in der Kopfzeile der Tabelle "Details des aktuellen Stichprobenstatus", um alle Prüfebenen zu aktualisieren.
5. Wählen Sie eine Prüfebene im Feld Neue Prüfebene.
6. Klicken Sie auf Aktualisieren oder in der Befehlsleiste auf Abschicken. Die Anwendung aktualisiert die Tabelle "Details des aktuellen Stichprobenstatus".
