Containerwartung

Nutzen Sie die "Containerwartung"-Transaktion um die im Designer voreingestellten statischen Container-Attribute eines erlaubten Containers auszuwählen und zu aktualisieren. Im Falle von Eltern- oder Kind-Containern können Sie die Eigenschaften eines Eltern-Containers und dessen Kind-Containern gleichzeitig ändern.

Hinweis: Nutzen Sie die "Containerattribut Wartungsseite", um dynamische, vom Nutzer festgelegte Attribute für einzelne Container festzulegen und zu verwalten.

Beim Verwenden von "Containerwartung"

Die einzigen erforderlichen Felder der "Containerwartungsseite" sind "Level, Eigentümer und Startgrund".

Falls der Wert eines bestimmten Attributs für einen Kind-Container von dem des Eltern-Containers abweicht, wird der Wert für die Kind-Container nicht im Verlauf protokolliert. Allerdings wird die Historie des Eltern-Containers angezeigt, wenn das Kontrollkästchen "Gesamte Historie anzeigen" im "Produktionsprotokoll" aktiviert ist.

Sie können diese Seite nicht verwenden, um den aktuellen Schritt des Containers im Arbeitsplan (der Ort kann nur angezeigt werden) oder dessen Menge zu ändern. Sie können die mit dem Container verbundene Maßeinheit "UOM" ändern.

Zuweisen einer eindeutigen "Gerätekennung (UDI)" auf der Seite "Containerwartung"
Hinweis: Diese Funktionalität ist nur verfügbar, wenn Medical Device installiert ist. Weitere Informationen finden Sie unter "Arbeitsbereich "Medical Device"".

Auf der Seite "Containerwartung" können Sie eine Gerätekennung, eine Produktionskennung oder eine UDI einem vorhandenen Container zuweisen oder diese ändern. Der letzte eingegebene oder automatisch berechnete Wert wird für den Container aufgezeichnet. Weitere Informationen zum Modellieren von Gerätekennungen, Produktionskennungen und UDI finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch und im Opcenter Execution Core Modeling-Benutzerhandbuch.

Seite "Containerwartung"

Die Seite "Containerwartung" ermöglicht Ihnen Folgendes:

• Wählen Sie einen Container aus oder geben Sie ihn ein.
• Zeigen Sie Informationen des Containers an.
• Greifen Sie auf den Verlauf des Containers zu.
• Ändern Sie die Informationen in bestimmten Feldern wie "Level, Eigentümer und Startgrund".

Optional

Vorgehensweise zum Ändern von Containerattributen

Gehen Sie folgendermaßen vor, um Containerattribute zu ändern:

1. Öffnen Sie die Seite Containerwartung im Menü Container.  Die Seite "Containerwartung" wird angezeigt.
2. Wählen Sie einen Container im Container-Feld aus oder geben Sie ihn ein. Die Anwendung aktualisiert den Container-Kopf und die Auswahlfelder, sodass diese Informationen für den angegebenen Container angezeigt werden. Werte werden nur angezeigt, wenn sie für den Container eingegeben wurden.
Hinweis: Die Anwendung füllt das Feld "Container" und den Container-Kopf automatisch, wenn Sie von einer anderen Fertigungsseite auf das Popup "Containerwartung" zugreifen.
3. Ändern Sie die Container-bezogenen Werte nach belieben. Sehen Sie in der Felddefinitionen-Tabelle nach weiteren Information zu diesen Feldern nach.
4. Klicken Sie auf Container aktualisieren oder in der Befehlsleiste auf Abschicken. Alle Felder werden gelöscht und eine Statusmeldung wird angezeigt, die besagt, dass die Transaktion erfolgreich war. Wenn Sie diese Transaktion vom Popup "Containerwartung" gestartet haben, wird die Seite "Operative Ansicht" mit aktualisierten Informationen angezeigt. Der Containerverlauf wurde aktualisiert.
