Container abtrennen

Die Transaktion "Abtrennen" entfernt Kind-Container von Eltern-Containern. Kind-Container werden in der Regel von einem Elternteil abgetrennt, wenn eine getrennte Verarbeitung erforderlich ist, wie z. B.:

• Ein Kind-Container erfordert Nacharbeit im Gegensatz zu den verbleibenden Kind- und Eltern-Containern.
• Ein Kind-Container erfordert einen Bearbeitung über einen alternativen Pfad.
• Ein Kind-Container muss an einem Schritt auf Halt gesetzt werden.
• Mehrere Kind-Container erfordern beschleunigte Bearbeitung, um die Nachfrage seitens der Kunden zu erfüllen.

Beim Trennen von Containern

Wenn Sie mit einer Linienzuordnung angemeldet sind, so wird die Eltern-Container-Liste durch die Werte der Linienzuordnung gefiltert.

Sie können einen oder mehrere einzelne Kind-Container oder alle aufgelisteten Kind-Container auswählen, um sie vom Eltern-Container abzutrennen.

Wenn Sie einen oder mehrere Kind-Container abtrennen, wird die Menge der abgetrennten Kind-Container von der Menge der Eltern-Container abgezogen.

Seite "Abtrennen"

Mit der Trennen-Seite können Sie:

• Einen Eltern-Container auswählen.
• Die Liste der mit einem entsprechenden Eltern-Container verbundenen Kind-Container ansehen.
• Wählen Sie einen oder mehrere Kind-Container vom Eltern-Container zu trennen.

Vorgehensweise zum Abtrennen von Containern

Gehen Sie folgendermaßen vor, um Kind-Container zu bearbeiten:

1. Öffnen Sie auf Trennen im Container-Menü. Die Seite "Abtrennen" wird angezeigt.
2. Wählen Sie einen Container im Container-Feld aus oder geben Sie ihn ein. Die Anwendung aktualisiert den Container-Kopf und die Auswahlfelder, sodass diese Informationen für den angegebenen Container angezeigt werden. Werte werden nur angezeigt, wenn sie für den Container eingegeben wurden.
Hinweis: Die Anwendung füllt das Feld "Container" und den Container-Kopf automatisch, wenn Sie von einer anderen Fertigungsseite auf das Popup "Abtrennen" zugreifen.
3. Wählen Sie mit einer der folgenden Methoden die Container aus, die getrennt werden sollen:
• Benutzen Sie das Feld Auszuwählenden Container eingeben, um einen Kind-Container einzugeben, entweder durch Scannen oder manuelle Eingabe des Containernamens und anschließendes Drücken der Tab-Taste, um das Feld zu verlassen. Wenn der eingegebene Container in der Tabelle "Momentan Verbundene Container" enthalten ist, markiert die Anwendung die Zeile, in der der Container steht und wählt das Kontrollkästchen der Zeile aus.
• Aktivieren Sie das Kontrollkästchen in der Tabelle neben jedem Kind-Container, der von seinem Eltern-Container abgetrennt werden soll.
• Aktivieren Sie das oberste Kontrollkästchen, um alle Container in der Tabelle auszuwählen.
4. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
5. Klicken Sie auf Alle ausgewählten abtrennen oder in der Befehlsleiste auf Abschicken. Eine Statusmeldung wird angezeigt, die besagt, dass der oder die Kind-Container vom Eltern-Container abgetrennt wurden.
