Ressourcendurchsatz

Die "Ressource Durchsatz"-Transaktion zeichnet Durchsatz-Informationen - tatsächlich verarbeitete Mengen - für eine Ressource auf. Dies dient dazu, dass der Produktionsprozess in Situationen aufgezeichnet werden kann, in denen das Material, das produziert wird, nicht auf nachvollziehbare Weise nachverfolgt wird.

Es kann auch zutreffen, wenn Sie Durchsatz-Informationen aufzeichnen müssen, die keinem Container zugeordnet sind. Dies kann auftreten, wenn eine Ressource durch Bearbeitung von Testmaterial getestet wird, das nicht Teil eines normalen Arbeitsplans ist. Befindet sich das Ressourcen-Testmaterial außerhalb des normalen Arbeitsplans, ist es nicht praktisch, Container-Transaktionen zum Ressourcen-Test zu verwenden.

Beim Verwenden von "Ressourcendurchsatz"

Die "Ressource Durchsatz"-Transaktion kann nur den Durchsatz für eine einzelne Ressource pro Transaktion aufzeichnen. Allerdings können mehrere "Ressource Durchsatz"-Transaktionen an einer einzelnen Ressource durchgeführt werden.

Die Ressourcenverfügbarkeit muss "Verfügbar" sein, damit Durchsatz akzeptiert werden kann. Sie können die Verfügbarkeit für die Ressource auf der Seite "Ressourceneinrichtung" ändern.

Das Ressourcenfeld zeigt alle Ressourcen und Behälter an, es sei denn, Sie wählen eine Ressourcengruppe aus. Dann wird die Anzeige von Ressourcen und Behältern nach dieser Ressourcengruppe gefiltert.

Das "Maßeinheit"-Feld kann leer gelassen werden, aber es darf nicht auf einen falschen Wert gesetzt werden.

Wenn an Ihrem Standort Maintenance Management implementiert ist, können Durchsatzmengen hinzugefügt werden, die das Mengenlimit überschreiten. Allerdings kann kein Durchsatz hinzugefügt werden, wenn dieses Limit bereits überschritten wurde.

Weitere Methoden zum Hinzufügen von Durchsatz

"Ressourcendurchsatz" ist eine der Transaktionen, die verwendet werden, um Durchsatz zu einer Ressource mit einer Durchsatzanforderung (mithilfe von Maintenance Management) hinzuzufügen. Darüber hinaus können Sie auch diese Transaktionen verwenden, um Durchsatz hinzuzufügen:

• Seite "Container Durchsatz"
• "Anmelden"-Transaktion
• "Abmelden"-Transaktion (bei einer Operation, bei der "Warteschlange verwenden" mit einem Häkchen versehen ist)

Seite "Ressourcendurchsatz"

Sie können die über das Menü "Ressource" auf die Seite "Ressourcendurchsatz" zugreifen. Verwenden Sie die Seite "Ressourcendurchsatz", um Durchsatz-Informationen (tatsächlich verarbeitete Mengen) für eine Ressource aufzuzeichnen. Dies gilt, wenn Sie Material nachverfolgen müssen, das keinem Container zugeordnet ist; beispielsweise ein Qualifikations-Los, das noch von der Ressource bearbeitet werden muss.

Vorgehensweise zum Aufzeichnen des Ressourcendurchsatzes

Gehen Sie folgendermaßen vor, um den Ressourcendurchsatz aufzuzeichnen:

1. Öffnen Sie die Seite Ressource Durchsatz im Menü Ressource. Die Seite "Ressourcendurchsatz" wird angezeigt.
2. Wählen Sie eine Ressourcengruppe aus, falls Sie die Liste angezeigter Ressourcen begrenzen möchten.
3. Wählen Sie die Ressource, für die Sie eine verarbeitete Menge aufzeichnen möchten. Die Seite wird aktualisiert und zeigt ein Symbol für die Verfügbarkeit der Ressource, die "Zeit im Zustand" (Zeit im aktuellen Zustand), den "Statuscode" (aktueller "Ressourcenstatus") und den Statusgrund (aktueller "Ressourcenstatusgrund") an.
4. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
5. Klicken Sie auf Durchsatz. Eine Erfolgsmeldung wird angezeigt und der Zähler für den verbleibenden Durchsatz der Ressource wird um die Menge erhöht, die für die Bearbeitung geplant war, aber nicht bearbeitet wurde.
