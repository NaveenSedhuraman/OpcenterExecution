Bearbeitung von Umlaufbestand-Nachrichten auf Portal Fertigungsseiten

Siemens bietet die Möglichkeit, vordefinierte Umlaufbestandsnachrichten zu einem der folgenden Kontexte an eine Portal Shop Floor-Transaktion zu liefern:

• Kontext der Umlaufbestandsnachricht (Produkt, Operation, Eigentümer, Startgrund usw.)
• Definition der Umlaufbestandsnachricht (Status, relevanter Diensttyp, Gültigkeitsdaten)
• Aktueller Kontext/Sitzungswerte des Benutzers
• Container, der bearbeitet wird

Automatische Umlaufbestandsnachrichten können verwendet werden, um:

• Eine Erinnerung oder eine besondere Bearbeitungs-Anweisung für die Person anzuzeigen, die Arbeit durchführt
• Einen Container anzuhalten, wenn er einen bestimmten Punkt erreicht
• Zu verlangen, dass ein Operator oder Supervisor bestätigt, dass die besonderen Anweisungen, die in Form der Umlaufbestandsnachricht erteilt wurden, gelesen wurden
• Einen Ingenieur zu benachrichtigen, dass ein Container einen bestimmten Standort erreicht hat

Weitere Informationen zum Konfigurieren von Umlaufbestandsnachrichten finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch und im Opcenter Execution Core Modeling-Benutzerhandbuch.

Wenn Sie beim Ausführen einer Fertigungs-Transaktion eine Umlaufbestandsnachricht erhalten, wurde eine der oben beschriebenen Bedingungen vordefiniert und im Prozess automatisch getriggert. Sie sollten die Nachricht lesen. Die Nachricht kann zusätzliche Informationen enthalten oder Sie anweisen, eine Aufgabe durchzuführen. Darüber hinaus kann die Nachricht eine Bestätigung und ein Kennwort erfordern.

Beim Bearbeiten von Umlaufbestandsnachrichten

Bei der Bearbeitung mehrerer Umlaufbestandsnachrichten für eine Transaktion müssen Sie alle Nachrichten bestätigen, für die eine Bestätigung erforderlich ist, bevor Sie die Nachrichten übermitteln und die Transaktion ausführen können. Die Anwendung zeigt eine Fehlermeldung an, wenn Sie auf "Abschicken" klicken, ohne alle Nachrichten mit erforderlicher Empfangsbestätigung bestätigt zu haben. Sie können Umlaufbestandsnachrichten nicht anzeigen, nachdem sie erfolgreich übermittelt wurden.

Wenn Sie eine Nachricht im Popup "Umlaufbestandsnachricht Bearbeitung" auswählen, zeigt die Anwendung die Felder, die für diese Nachricht konfiguriert wurden, unterhalb der Tabelle "Umlaufbestandsnachrichten" an. Welche Felder angezeigt werden, hängt davon ab, wie Ihr Systemadministrator die ausgewählte Umlaufbestandsnachricht in der Modellierung konfiguriert hat.

Wenn außerdem elektronische Unterschriften für diese Transaktion konfiguriert wurden, werden die Nachrichten mit elektronischer Unterschrift angezeigt, nachdem alle Umlaufbestandsnachrichten bearbeitet wurden.

Hinweis: Umlaufbestand-Nachrichten werden nur für Einzelcontainer-Transaktionen unterstützt.

Popup "Verarbeitung von Umlaufbestandsnachrichten"

Das Popup "Verarbeitung von Umlaufbestandsnachrichten" kann auf einer Fertigungsseite angezeigt werden, wenn Sie eine Transaktion durchführen.

Felddefinitionen des Popups "Verarbeitung von Umlaufbestandsnachrichten"

Diese Tabelle definiert die Felder im Popup "Verarbeitung von Umlaufbestandsnachrichten".

Feld

Definition

Typ

Umlaufbestandsnachrichten

Tabelle, die die zugehörigen Umlaufbestandsnachrichten auflistet.

Nur Anzeige

Nachrichten-Text

Zeigt den Text der aktuellen Umlaufbestandsnachricht an. Wenn eine Nachricht den vorhandenen Platz überschreitet, zeigt die Anwendung eine "Erweitern"-Schaltfläche an. Wenn Sie auf die Schaltfläche "Erweitern" klicken, wird der gesamte Text der Nachricht angezeigt. Wenn Sie erneut auf die Schaltfläche "Erweitern" klicken, wird die Nachricht wieder reduziert.

Nur Anzeige

Bestätigung erforderlich

Wert, der angibt, ob Sie die Nachricht bestätigen müssen, bevor Sie fortfahren. Gültige Werte sind entweder Ja oder Nein. Wenn der Wert "Ja" ist, wird das Kontrollkästchen "Nachricht bestätigt" unterhalb der Tabelle angezeigt.

Nur Anzeige

Nachricht bestätigt

Wert, der angibt, ob die Nachricht bestätigt wurde. Gültige Werte sind entweder Ja oder Nein. Die Anwendung aktualisiert diesen Wert, wenn Sie ein Häkchen in das Ankreuzfeld "Nachricht bestätigt" setzen oder dieses entfernen.

Nur Anzeige

Bearbeitung anhalten

Wert der angibt, ob der Container angehalten wird, wenn die Transaktion erfolgreich abgeschickt wurde, um weitere Bearbeitung zu verhindern. Gültige Werte sind entweder Ja oder Nein.

Nur Anzeige

Nachricht bestätigt

Wenn Sie dieses Kontrollkästchen aktivieren, bestätigen Sie die Umlaufbestandsnachricht. Die Anwendung ändert den Wert in der Spalte "Nachricht bestätigt" in "Ja". Wenn Sie das Häkchen im Ankreuzfeld entfernen, aktualisiert die Anwendung den Wert im Feld "Nachricht bestätigt" auf "Nein".

Hinweis: Dieses Kontrollkästchen wird nur angezeigt, wenn ein Systemadministrator beim Konfigurieren der Umlaufbestandsnachricht angegeben hat, dass eine Bestätigung oder ein Kennwort erforderlich ist.

Erforderlich

Kennwort für Umlaufbestandsnachricht

Kennwort zum Bestätigen der Umlaufbestandsnachricht. Dies ist das Kennwort, das von einem Systemadministrator für die Umlaufbestandsnachricht festgelegt wurde – nicht Ihre Benutzer-ID. Im Kennwortfeld wird Groß-/Kleinschreibung beachtet.

Hinweis: Dieses Feld wird nur angezeigt, wenn ein Systemadministrator beim Konfigurieren dieser Umlaufbestandsnachricht angegeben hat, dass ein Kennwort erforderlich ist. Darüber hinaus wird das Kontrollkästchen "Nachricht bestätigt" angezeigt, wenn ein Kennwort erforderlich ist.

Erforderlich

Dokument

Zeigt das Dokument an, das der Umlaufbestandsnachricht zugeordnet ist, sowie ein "Dokument anzeigen"-Symbol, das es Ihnen erlaubt, das Dokument in einem separaten Popup anzuzeigen.

Hinweis: Dieses Feld wird nur angezeigt, wenn der Systemadministrator beim Konfigurieren der Umlaufbestandsnachricht ein Dokument zugeordnet hat.

Nur Anzeige

Kontaktinformation

Kontaktinformationen, die der Umlaufbestandsnachricht zugeordnet sind.

Hinweis: Dieses Feld wird nur angezeigt, wenn der Systemadministrator beim Konfigurieren der Umlaufbestandsnachricht Kontaktinformationen hinzugefügt hat.

Nur Anzeige

Vorgehensweise zum Bestätigen einer Umlaufbestandsnachricht

Führen Sie die folgenden Schritte aus, um eine Umlaufbestandsnachricht zu bestätigen:

1. Wählen Sie eine Umlaufbestandsnachricht in der Tabelle aus, wenn das Popup "Verarbeitung von Umlaufbestandsnachricht" während einer Transaktion angezeigt wird. Die Anwendung aktualisiert die unterhalb der Tabelle "Umlaufbestandsnachrichten" angezeigten Felder.
2. Lesen Sie alle Dokumente, die der Umlaufbestandsnachricht zugeordnet sind, durch Klicken auf das Symbol Dokument anzeigen. Das Dokument wird in einem separaten Popup angezeigt.
3. Wählen Sie das Ankreuzfeld Nachricht bestätigt, um die Nachricht abzuzeichnen, wenn das Ankreuzfeld angezeigt wird.
4. Geben Sie Ihr Kennwort in das Feld Kennwort für Umlaufbestandsnachricht ein, wenn das Feld angezeigt wird.
5. Wiederholen Sie die Schritte 1 bis 4, um alle zusätzlichen Umlaufbestandsnachrichten zu bearbeiten.
6. Klicken Sie auf Abschicken. Die Anwendung verarbeitet die Umlaufbestandsnachrichten und zeigt eine Statusmeldung an.
