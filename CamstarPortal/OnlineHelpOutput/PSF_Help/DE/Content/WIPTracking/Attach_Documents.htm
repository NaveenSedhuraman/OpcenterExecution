Anhängen von Dokumenten

Sie können Dokumente über die Seite "Dokument anhängen", die Seite "Anhänge verwalten" und in der Befehlsleiste über die Schaltfläche "Dokument anhängen" anhängen. Die Seiten "Dokument anhängen" und "Anhänge verwalten" sind über das Menü "Anhänge" verfügbar. Sie können Dokumente auch über die Seiten "Start", "Auftragserledigung" und "Produktionsauftrag" anhängen. Informationen zum Anhängen von Dokumenten über die Seite "Start" finden Sie unter "Container starten". Informationen zum Anhängen von Dokumenten über die Seite "Auftragserledigung" finden Sie unter "Verwenden der Auftragserledigung". Informationen zum Anhängen von Dokumenten über die Seite "Produktionsauftrag" finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch und im Opcenter Execution Core Modeling-Benutzerhandbuch.

Zum Anhängen von Dokumenten über die Seite "Anhänge verwalten" können Sie bei Bedarf eine Dokumentsuche ausführen.

Wenn Sie auf der Seite "Anhänge verwalten" auf "Anhängen" oder auf einer Fertigungsseite in der Befehlsleiste auf die Schaltfläche "Dokument anhängen" klicken, wird das Popup "Dokument anhängen" angezeigt. Das Popup "Dokument anhängen" enthält dieselben Felder und Funktionen wie die Seite "Dokument anhängen".

Sie können drei Arten von Dokumenten anhängen:

• Ein neues Dokument, das nicht wiederverwendet wird
• Ein neues Dokument, das wiederverwendet wird
• Ein vorhandenes Dokument

Die Anwendung erlaubt Sonderzeichen im Dokumentnamen. Die maximale Dateigröße für einen Anhang beträgt 30 MB.

Wenn Sie auf der Seite "Dokument anhängen" oder im Popup "Dokument anhängen" auf "Anhängen" klicken, wird das Dokument an das Objekt angehängt. Sie können das angehängte Dokument suchen, entfernen und ansehen. Weitere Informationen finden Sie unter "Anhänge verwalten".

Anhängen eines neuen Dokuments, das nicht wiederverwendet wird

Die Option Neues Dokument anhängen, das NICHT wiederverwendet wird (keine Revision benötigt) hängt ein neues Dokument an, das spezifisch für eine Objektinstanz ist. Diese Art von Dokument wird als Anhang betrachtet und kann nicht zu einem späteren Zeitpunkt anderen Objekten zugewiesen werden. Beispiele sind Testergebnisse und Fehleraufzeichnungen.

Sie müssen keine Revision für diesen Anhangstyp angeben. Die Anwendung weist eine Revision zu, wenn das Dokument angehängt wird.

Anhängen eines neuen Dokuments, das wiederverwendet wird

Die Option Neues Dokument anhängen, das wiederverwendet wird (Revision muss angegeben werden) hängt ein neues Dokument an, das nicht spezifisch für eine Objektinstanz ist. Diese Option erstellt eine Dokumentinstanz, die in der Anwendung verwaltet werden und zu einem späteren Zeitpunkt anderen Objekten zugewiesen werden kann.

Dieser Anhangstyp wird in der Modellierung angezeigt und kann mithilfe der Seite "Dokument" verändert werden. Beispiele sind kundenspezifische Anweisungen oder Verarbeitungsanweisungen für Forschung und Entwicklung.

Für diesen Anhangstyp müssen Sie eine Revision angeben.

Das Modellierungsobjekt verweist immer auf die Revision des Dokuments, die als Standardrevision (ROR) markiert wurde, wenn das Kontrollkästchen "ROR" aktiviert ist. Dies bedeutet, dass alle Objekte, bei denen angegeben wurde, dass sie die "ROR" dieses Dokuments verwenden, betroffen sind, wenn sich die "ROR"-Revision für ein aktuell angehängtes Dokument ändert. Das Objekt verweist immer auf die spezifische Revision, wenn das Ankreuzfeld "ROR" nicht ausgewählt ist. Eine Änderung an der Standardrevision des Dokuments wirkt sich nicht auf dieses Objekt aus.

Eine neue Revision für das Dokument kann bei dieser Option nicht hinzugefügt und als die "ROR" markiert werden, wenn die "ROR" eines Dokuments aktuell an ein Objekt angehängt ist. Sie müssen dies direkt über die Seite "Dokument" vornehmen. Sie können diese Option verwenden, um das "ROR"-Dokument anzuhängen, sobald die neue "ROR"-Dokumentrevision über die Modellierung erstellt wurde. Unter "Anhängen eines vorhandenen Dokuments" finden Sie Informationen zum Anhängen von vorhandenen Dokumenten.

Anhängen eines vorhandenen Dokuments

Die Option Existierendes Dokument anhängen (unten auswählen) hängt ein Dokument an, das als wiederverwendbares Dokument erstellt wurde. Unter "Anhängen eines neuen Dokuments, das wiederverwendet wird" finden Sie Informationen zum Erstellen von nachnutzbaren Dokumenten.

Sie können die Standard-Revision des Dokuments oder eine spezifische Revision anhängen, wenn Sie vorhandene Dokumente anhängen.

Das Objekt verwendet immer die Revision des Dokuments, die als Standardrevision (ROR) markiert wurde, wenn das "ROR"-Dokument ausgewählt ist. Das bedeutet, wenn sich die "ROR"-Revision für ein Dokument ändert, verweist das Objekt auf die neue "ROR"-Revision. Das Objekt verweist immer auf die spezifische Revision, wenn eine spezifische Revision ausgewählt ist; daher wirkt sich eine Änderung an der "ROR" des Dokuments nicht auf dieses Objekt aus.

Sie können ein vorhandenes Dokument vor dem Anhängen an ein Objekt anzeigen. Sie müssen die entsprechende Software zum Anzeigen des Dokuments haben. Beispielsweise müssen Sie Microsoft Word haben, um ein Word-Dokument anzuzeigen.

Vorgehensweise zum Anhängen eines Dokuments

Gehen Sie folgendermaßen vor, um ein Dokument anzuhängen:

1. Welche Methode möchten Sie zum Anhängen von Dokumenten verwenden?

Wenn Sie ein Dokument anhängen möchten . . .

Dann . . .

Verwenden der Seite "Dokument anhängen"
a. Öffnen Sie die Seite Dokument anhängen im Menü Anhänge. Die Seite "Dokument anhängen" wird angezeigt.
b. Wählen Sie im Feld Objekttyp einen Objekttyp aus. Die Anwendung zeigt das Containerfeld an, wenn der ausgewählte Objekttyp "Container" ist. Andernfalls zeigt die Anwendung das Feld "Instanzname" an. Die Anwendung aktiviert den Knopf "Anhängen“.
c. Wählen Sie eine Instanz aus der Liste im Feld Instanzname.

Oder

Wählen Sie einen Container im Feld Container aus.

d. Gehen Sie zu Schritt 2.

Verwenden der Seite "Dokumente verwalten" mit einer Dokumentsuche
a. Öffnen Sie die Seite Anhänge verwalten im Menü Anhänge. Die Seite "Anhänge verwalten" wird angezeigt.
b. Treffen Sie in mindestens einem Feld eine Auswahl. Die Anwendung aktiviert die Schaltfläche "Suchen". Unter "Felddefinitionen der Seite "Anhänge verwalten"'" finden Sie Informationen zu den Anforderungen zur Feldauswahl.
c. Klicken Sie auf Suchen. Die Anwendung füllt die Tabelle "Angehängte Dokumente" automatisch.
d. Wählen Sie eine Zeile in der Tabelle "Angehängte Dokumente" aus.
e. Klicken Sie auf Anhängen. Das Popup "Dokument anhängen" wird angezeigt.
f. Gehen Sie zu Schritt 2.

Verwenden der Seite "Dokumente verwalten" ohne Dokumentsuche
a. Öffnen Sie die Seite Anhänge verwalten im Menü Anhänge. Die Seite "Anhänge verwalten" wird angezeigt.
b. Klicken Sie auf Anhängen. Das Popup "Dokument anhängen" wird angezeigt.
c. Gehen Sie zu Schritt 2.

Verwenden der Schaltfläche "Dokument anhängen" in der Befehlsleiste
a. Öffnen Sie eine Container-Transaktionsseite. Die Container-Transaktionsseite wird angezeigt.
b. Wählen Sie einen Container im Container-Feld aus oder geben Sie ihn ein. Die Anwendung aktualisiert den Container-Kopf und die Auswahlfelder, sodass diese Informationen für den angegebenen Container angezeigt werden.Werte werden nur angezeigt, wenn sie für den Container eingegeben wurden.
c. Klicken Sie in der Befehlsleiste auf die Schaltfläche Dokument anhängen. Das Popup "Dokument anhängen" wird angezeigt, wobei das Feld "Container" automatisch gefüllt wurde.
d. Gehen Sie zu Schritt 2.
2. Welche Art von Dokument möchten Sie anhängen?

Wenn Sie Folgendes anhängen möchten . . .

Dann . . .

Ein neues Dokument, das nicht wiederverwendet wird
a. Wählen Sie Neues Dokument anhängen, das NICHT wiederverwendet wird (keine Revision benötigt) aus. Die Felder "Dokumentpfad", "Dokumentname" und "Dokumentbeschreibung" werden angezeigt.
b. Klicken Sie auf Dokumentpfad, um eine Datei auszuwählen.
c. Geben Sie einen Namen für das Dokument im Feld Dokumentname ein.
d. Gehen Sie zu Schritt 3.

Ein neues Dokument, das wiederverwendet wird
a. Wählen Sie Neues Dokument anhängen, das wiederverwendet wird (Revision muss angegeben werden) aus. Die Felder "Dokumentpfad", "Dokumentname", "Revision", "Standardrevision" und "Dokumentbeschreibung" werden angezeigt.
b. Klicken Sie auf Dokumentpfad, um eine Datei auszuwählen.
c. Geben Sie einen Namen für das Dokument im Feld Dokumentname ein.
d. Geben Sie die Revision dieses Dokuments in das Feld Revision ein.
e. Aktivieren Sie optional das Kontrollkästchen neben der Revision, um anzugeben, dass dieses Dokument die Standardrevision ist.
f. Gehen Sie zu Schritt 3.

Ein vorhandenes Dokument
a. Wählen Sie Anhängen eines existierenden Dokumentes (unten auswählen). Die Felder "Dokumentinstanzen", "Revision", "Standardrevision" sowie die Schaltfläche "Dokument anzeigen" werden angezeigt.
b. Wählen Sie ein Dokument im Feld Dokumentinstanzen aus. Die Anwendung aktiviert die Schaltfläche "Dokument anzeigen".
c. Geben Sie die jeweilige Revision in das Feld Revision ein oder aktivieren Sie das Kontrollkästchen Standardrevision.
d. Klicken Sie wahlweise auf Dokument anzeigen, wenn Sie das Dokument ansehen möchten, bevor Sie es anhängen. Die Anwendung zeigt eine Meldung an, die Sie auffordert, das Dokument zu öffnen bzw. zu speichern oder die Anfrage abzubrechen.
e. Gehen Sie zu Schritt 3.
3. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
4. Klicken Sie auf Anhängen. Die Anwendung meldet, dass das Dokument erfolgreich angehängt wurde.
