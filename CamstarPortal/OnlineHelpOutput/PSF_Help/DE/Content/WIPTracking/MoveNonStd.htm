Container abmelden (nicht standardmäßig)

Die Transaktion "Abmelden - nicht standardmäßig" erlaubt es Ihnen, einen Container zu einem beliebigen Schritt in einem beliebigen Arbeitsplan abzumelden. Diese Transaktion bietet Ihnen mehr Flexibilität als die "Abmelden"-Transaktion, die es Ihnen lediglich erlaubt, einen Container zum nächsten Schritt innerhalb desselben Arbeitsplans zu abzumelden. (Die Transaktion "Abmelden - nicht standardmäßig" ähnelt der Transaktion "Transferieren", aber "Abmelden - nicht standardmäßig" unterliegt jeglichen "Anmelden"-Anforderungen für eine Operation.)

Hinweis: Informationen über das gleichzeitige Abmelden mehrerer Container finden Sie unter "Mehrere Container abmelden (nicht standardmäßig)".

Beim Verwenden von "Abmelden - nicht standardmäßig"

Alle Anzeigefelder sind auf der Seite "Abmelden - nicht standardmäßig" leer, bis ein Container ausgewählt wird. Anzeigefelder werden im "Abmelden - nicht standardmäßig"-Popup automatisch gefüllt. Informationen über das Ausführen einer Transaktion über eine Seite oder ein Popup finden Sie unter "Verwenden von Transaktionsseiten und Popups".

Die Anwendung zeigt einen Standardwert im Feld "Ressource" an, wenn eine dieser Bedingungen zutrifft:

• Für den aktuellen Schritt des Containers war ein Anmelden erforderlich, und der Benutzer, der den Container angemeldet hat, hat eine Ressource angegeben.

Oder

• Eine Ressource wurde angegeben, als der Container gestartet wurde, und "Abmelden - nicht standardmäßig" ist die erste Transaktion (mit Angabe einer Ressource), die Sie nach dem Start des Containers durchführen.

Sie können, falls erforderlich, eine andere Ressource auswählen.

Seite "Abmelden - nicht standardmäßig"

Sie können über das Container-Menü auf die Seite "Abmelden - nicht standardmäßig" zugreifen. Die Seite "Abmelden - nicht standardmäßig" ermöglicht es Ihnen, den Container einzugeben oder auszuwählen, den Sie zu einem Schritt in einem Arbeitsplan abmelden möchten.

Hinweis: Die Seite "Abmelden - nicht standardmäßig" ist eventuell auch als Popup verfügbar, wenn die Transaktion "Anmelden" als verfügbare Aktion auf einer anderen Seite konfiguriert wurde (z. B. "Containersuche").

Felddefinitionen der Seite und des Popups "Abmelden - nicht standardmäßig"

Diese Tabelle definiert die Felder auf der Seite und im Popup "Abmelden - nicht standardmäßig".

Feld

Definition

Typ

Containerinformationen

Kopf mit dem Containernamen und Informationen zu diesem Container. Sie müssen einen Container auf einer Transaktionsseite auswählen oder eingeben. Alle Informationen in einem Transaktionen-Popup sind schreibgeschützt. Weitere Informationen finden Sie unter "Containerinformationen".

Erforderlich/ Nur Anzeige

Ressource

Ressource, die den Container im aktuellen Schritt bearbeitet hat. Sie können eine andere Ressource auswählen, wenn die Anwendung bereits standardmäßig eine angezeigt. Wenn Sie die Ressource in diesem Schritt angeben, bevor Sie den Container abmelden, sorgt das für Nachverfolgbarkeit.

Welche Ressourcenwerte zur Auswahl stehen, hängt von der Ressourcengruppe ab, die dem Schritt zugewiesen wurde (Spez.). Wenn der Spezifikation keine Ressourcengruppe zugewiesen wurde, stehen alle definierten Ressourcen zur Auswahl.

Optional

Zu Arbeitsplan

Ziel-Arbeitsplan und Schritte. (Die Schritte werden angezeigt, wenn Sie den Arbeitsplan erweitern.) Die Anwendung verschiebt den Container in den nächsten Schritt im aktuellen Arbeitsplan, es sei denn, Sie geben hier einen anderen Arbeitsplan oder Schritt an.

Optional

Zu Schritt

Schritt im Ziel-Arbeitsplan. Die Anwendung füllt dieses Feld, wenn Sie einen Schritt (Spez.) aus dem erweiterten Arbeitsplan im Feld "Zu Arbeitsplan" auswählen.

Optional

Zusätzliche Felder anzeigen/ausblenden

Optionale Felder sind standardmäßig ausgeblendet, wenn die Anwendung eine Seite oder ein Popup anzeigt. Durch Klicken auf die Steuerung "Anzeigen" werden die Felder angezeigt. Durch Klicken auf die Steuerung "Verstecken" werden die Felder ausgeblendet.

  

Zu Standort

Standort, an den der Container im nächsten Schritt zur Bearbeitung bewegt wird. Ein Standort kann eine ganze Fabrik, ein Bereich einer Fabrik oder so spezifisch wie ein Regal-Identifikator sein. (Standorte werden innerhalb des Fabrik-Objekts in der Modellierung definiert.)

Hinweis: Dieses Feld ist standardmäßig ausgeblendet.

Optional

Kommentare

Zusätzliche Informationen oder Anweisungen für diese Transaktion oder für diesen Container. Das Feld akzeptiert maximal 255 Zeichen. Wenn die Kommentare 255 Zeichen überschreiten, verkürzt die Anwendung die Daten.

Optional

Vorgehensweise zum Abmelden eines Containers zu einem beliebigen Schritt in einem Arbeitsplan

Gehen Sie folgendermaßen vor, um einen Container zu einem beliebigen Schritt in einem Arbeitsplan abzumelden:

1. Öffnen Sie die Seite Abmelden - nicht standardmäßig im Menü Container. Die Seite "Abmelden - nicht standardmäßig" wird angezeigt.
2. Wählen Sie einen Container im Container-Feld aus oder geben Sie ihn ein. Die Anwendung aktualisiert den Container-Kopf und die Auswahlfelder, sodass diese Informationen für den angegebenen Container angezeigt werden. Werte werden nur angezeigt, wenn sie für den Container eingegeben wurden.
Hinweis: Die Anwendung füllt das Feld "Container" und den Container-Kopf automatisch, wenn Sie von einer anderen Fertigungsseite auf das Popup "Abmelden - nicht standardmäßig" zugreifen.
3. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
4. Klicken Sie auf Abmelden oder in der Befehlsleiste auf Abschicken. Eine Statusmeldung wird angezeigt, die bestätigt, dass der Container abgemeldet wurde.
