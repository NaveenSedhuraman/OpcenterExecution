Anhänge verwalten

Über die Seite "Anhänge verwalten" können Sie Dokumente suchen, abtrennen, anhängen und anzeigen. Die Seite "Anhänge verwalten" ist über das Menü "Anhänge" verfügbar. Informationen zum Anhängen von Dokumenten über die Seite "Anhänge verwalten" finden Sie unter "Anhängen von Dokumenten".

Sie können an einen Container angehängte Dokumente auch über das Symbol "Dokumentsatz" im Bereich "Containerinformationen" der Fertigungstransaktionen anzeigen. Weitere Informationen finden Sie unter "Containerinformationen".

Beim Suchen nach Dokumenten

Sie müssen eines dieser Felder ausfüllen, um die Schaltfläche "Suchen" zu aktivieren, obwohl alle Felder optional sind.

• Objekttyp
• Von Zeitstempel
• Bis Zeitstempel
• Dokumenttyp
• Dateierweiterung
• Mitarbeiter

Beim Anhängen von Dokumenten

Das Popup "Dokument anhängen" wird angezeigt, wenn Sie auf der Seite "Anhänge verwalten" auf "Anhängen" klicken. Informationen zum Anhängen von Dokumenten über das Popup "Dokument anhängen" finden Sie unter "Anhängen von Dokumenten".

Beim Abtrennen von Dokumenten

Die Anwendung füllt das Feld "Instanzname" im Popup "Dokument abtrennen" automatisch mit dem Instanznamen. Die Anwendung füllt außerdem den Namen und die Revisionsinformationen des abzutrennenden Dokuments aus.

Beim Anzeigen von Dokumenten

Sie müssen die entsprechende Software zum Anzeigen des angehängten Dokuments haben. Beispielsweise müssen Sie Microsoft Word haben, um ein Word-Dokument anzuzeigen.

Unterstützte 3D-Bilddateien werden in einem separaten Popup "3D-Datei-Viewer" (3D File Viewer) angezeigt. Im Popup "3D-Datei-Viewer" (3D File Viewer) können Sie Bilder drehen, vergrößern bzw. verkleinern und markieren sowie als eine 2D-Momentaufnahme speichern. Weitere Informationen zum Anzeigen von 3D-Bildern finden Sie im Opcenter Execution Medical Device and Diagnostics Portal Studio-Benutzerhandbuch und im Opcenter Execution Core Portal Studio-Benutzerhandbuch.

EWI-Dateien (Electronic Work Instructions) werden in einem separaten Popup geöffnet, in dem Sie zur Eingabe Ihres Teamcenter-Benutzernamens und -Kennworts aufgefordert werden. Sie müssen über Zugriff auf Teamcenter verfügen, um EWI-Dateien anzeigen zu können. Die Teamcenter-Sitzung bleibt geöffnet, bis Ihr Browser geschlossen wird.

Standardeinstellungen für die Browsersicherheit verhindern das Öffnen von Dokumenten auf unsicheren Websites. Sie müssen Ihre Einstellungen ändern, um unsicheren Inhalt zuzulassen.

Hinweis: Wenn Sie diese Einstellung im Browser ändern, werden Sie von der Anwendung abgemeldet.

Vorgehensweise zum Suchen nach einem Dokument

Gehen Sie folgendermaßen vor, um ein Dokument zu suchen:

1. Öffnen Sie die Seite Anhänge verwalten im Menü Anhänge. Die Seite "Anhänge verwalten" wird angezeigt.
2. Treffen Sie in mindestens einem Feld eine Auswahl. Unter "Felddefinitionen der Seite "Anhänge verwalten"" finden Sie Informationen zu den Anforderungen zur Feldauswahl. Die Anwendung aktiviert die Schaltfläche "Suchen".
3. Klicken Sie auf Suchen. Die Anwendung zeigt die ausgegebenen Elemente in der Tabelle "Angehängte Dokumente" an.

Vorgehensweise zum Abtrennen eines Dokuments

Gehen Sie folgendermaßen vor, um ein Dokument abzutrennen:

1. Befolgen Sie die Schritte unter "Vorgehensweise zum Suchen nach einem Dokument".
2. Wählen Sie eine einzelne Zeile in der Tabelle Angehängte Dokumente.
3. Klicken Sie auf Abtrennen. Das Popup "Dokument abtrennen" wird angezeigt, wobei die Felder "Instanzname" und "Angehängtes Dokument" ausgefüllt sind.
4. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
5. Klicken Sie auf Abtrennen. Die Anwendung meldet, dass das Dokument erfolgreich abgetrennt wurde.

Vorgehensweise zum Anzeigen eines Dokuments mit der Seite "Anhänge verwalten"

Gehen Sie folgendermaßen vor, um ein Dokument über die Seite "Anhänge verwalten" anzusehen:

1. Befolgen Sie die Schritte unter "Vorgehensweise zum Suchen nach einem Dokument".
2. Wählen Sie eine einzelne Zeile in der Tabelle Angehängte Dokumente.
3. Klicken Sie auf Anzeigen. Die Anwendung fordert Sie dazu auf, die Datei zu öffnen oder zu speichern, oder sie lädt das Dokument automatisch herunter. (Ihr Internet Browser bestimmt, ob die Anwendung ein Popup öffnet, wie es aussehen soll und was in den Öffnen- und Speichern-Optionen genau steht.)

Vorgehensweise zum Anzeigen eines Dokuments über das Icon "Dokumentsatz"

Gehen Sie folgendermaßen vor, um ein Dokument über das Symbol "Dokumentsatz" anzuzeigen:

1. Öffnen Sie eine Container-Transaktionsseite. Die Container-Transaktionsseite wird angezeigt.
2. Wählen Sie einen Container im Container-Feld aus oder geben Sie ihn ein. Die Anwendung aktualisiert den Container-Kopf und die Auswahlfelder, sodass diese Informationen für den angegebenen Container angezeigt werden. Werte werden nur angezeigt, wenn sie für den Container eingegeben wurden.
3. Klicken Sie auf das Symbol Dokumentsatz. Die Anwendung zeigt eine Liste der Dokumente an, die mit dem Container und dem Auftrag des Containers verknüpft sind.
4. Klicken Sie auf ein Dokument, um es anzusehen. Die Anwendung fordert Sie dazu auf, die Datei zu öffnen oder zu speichern, oder sie lädt das Dokument automatisch herunter. (Ihr Internet Browser bestimmt, ob die Anwendung ein Popup öffnet, wie es aussehen soll und was in den Öffnen- und Speichern-Optionen genau steht.)
