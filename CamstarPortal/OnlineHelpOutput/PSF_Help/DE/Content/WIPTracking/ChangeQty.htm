Mengenänderung

Die "Mengenänderung"-Transaktion ermöglicht Ihnen, die Menge des Produkts in einem Container zu ändern. Diese Transaktion ist ideal, wenn Sie die Menge aus einem einzigen Grund ändern.

Hinweis: Weitere Informationen über mehrere Gründe für das Ändern der Menge in einem Container finden Sie unter "Mengenänderung: mehrere Gründe".

Arten von Mengenänderungen

Diese Tabelle beschreibt die fünf Arten von Mengenänderungen, die Sie angeben können, wenn Sie die Menge in einem Container ändern.

Anpassen

Anpassen gibt entweder eine Zunahme oder Abnahme der Menge aus einem unbekannten Grund an. Verwenden Sie diesen Typ, wenn das Ergebnis nach dem Zählen der Menge eines Containers nicht die erwartete Menge ist.

Bonus

Ein Bonus gibt eine Zunahme der Menge des Containers infolge eines Produktionsprozesses an. Beispielsweise hat sich das Gewicht eines Produkts infolge der Verarbeitung erhöht.

Kaufen

Ein Kaufen gibt eine Zunahme der Menge eines Containers an, die nicht das Resultat eines Produktionsprozesses ist. Beispielsweise wurden bestimmte Einheiten von der Ingenieur-Abteilung zurückgegeben.

Verlust

Ein Verlust gibt eine Abnahme der Menge eines Containers infolge eines Produktionsprozesses an.

Verkaufen

Ein Verkaufen gibt eine Abnahme der Menge eines Containers an, die nicht das Resultat eines Produktionsprozesses ist. Beispielsweise sind einige Container zu Testzwecken abgegeben oder zu Demonstrationszwecken an die Verkaufsabteilung abgegeben.

Die Arten von Mengenänderungen stehen unter diesen Bedingungen zur Wahl:

• Ein Grund wurde definiert
• Eine Grund-Gruppe, die den Grund enthält, wurde definiert
• Die Grundgruppe wurde der aktuellen Operation des Containers zugewiesen
Hinweis: "Gründe", "Grundgruppen" und "Operationen" werden in der Modellierung definiert. Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch und im Opcenter Execution Core Modeling-Benutzerhandbuch.

Beim Ändern einer Menge

Alle Anzeigefelder sind auf der Seite "Mengenänderung" leer, bis ein Container ausgewählt wird. Anzeigefelder werden im Popup "Mengenänderung" automatisch gefüllt. Informationen über das Ausführen einer Transaktion über eine Seite oder ein Popup finden Sie unter "Verwenden von Transaktionsseiten und Popups".

Die Anwendung erfordert, dass einer Operation eine Grund-Gruppe zugewiesen wird, bevor Sie die "Mengenänderungs"-Transaktion durchführen können. Wenn Sie auf der Seite oder im Popup "Mengenänderung" einen Container angeben, der sich in einer Operation befindet, für die keine Grundgruppe definiert wurde, zeigt die Anwendung eine Warnmeldung an. Die Anwendung aktiviert die Schaltfläche "Mengenänderung" nur dann, wenn ein Container ausgewählt und eine Grund-Gruppe für die aktuelle Operation definiert wurde.

Sie können die Menge eines Containers unter Verwendung mehrerer Typen und Gründe ändern, aber Sie können nur einen Änderungstyp und -grund pro Transaktion verwenden. Die Anwendung zeigt den Grund automatisch an, wenn Sie einen Änderungstyp wählen, für den nur ein Grund definiert wurde.

Die Anwendung zeigt die Arten von Mengenänderungen in dieser Reihenfolge an:

• Verlust, die Standardauswahl
• Bonus
• Verkaufen
• Kaufen
• Anpassen

Die Liste kann variieren, je nachdem, ob der aktuellen Operation des Containers Grund-Gruppen zugewiesen wurden. Wenn der aktuellen Operation nur wenige Grund-Gruppen zugewiesen wurden, verkürzt die Anwendung die Liste, während sie die Listen-Reihenfolge beibehält. Der erste Änderungstyp ist die Standardauswahl. Wenn beispielsweise der aktuellen Operation keine "Verlustgrundgruppe" zugewiesen wurde, ist Bonus der erste Änderungstyp in der Liste und damit die Standardauswahl.

Die Art der Mengenänderung, die Sie auswählen, betrifft Feld-Beschriftungsnamen. Die Namen der Felder "Verlustgrund", "Ausfallmenge" und "Ausfallmenge2" ändern sich je nach Auswahl. Wenn Sie beispielsweise "Bonus" auswählen, benennt die Anwendung die Felder in "Bonusgrund" und "Bonusmenge" und "Bonusmenge2" um.

In dem seltenen Fall, dass Sie nur die sekundäre Menge ändern müssen, geben Sie eine Null (0) in das primäre Mengenfeld ein und geben Sie die sekundäre Menge ein, um die Transaktion abzuschließen.

Die Anwendung akzeptiert negative Zahlen im Feld <Änderungstyp>-Menge". Negative Zahlen bieten die Möglichkeit, eine Mengenänderungs-Transaktion zurückzurollen. Wenn Sie beispielsweise fälschlicherweise einen Verlust eingeben, können Sie den Verlust zurückrollen, indem Sie eine negative Zahl eingeben. Die Eingabe negativer Zahlen für "Verlust"- und "Verkaufen"-Gründe erhöht die Container-Menge, während die Eingabe negativer Zahlen für "Bonus", "Kaufen" und "Anpassen" die Container-Menge verringert.

Seite "Mengenänderung"

Sie können über das Container-Menü auf die Seite "Mengenänderung" zugreifen. Die Seite "Mengenänderung" ermöglicht es Ihnen, einen Container einzugeben oder auszuwählen und dann die Menge innerhalb des Containers zu ändern.

Hinweis: Die Seite "Mengenänderung" ist eventuell auch als Popup verfügbar, wenn die Transaktion "Mengenänderung (mehrere Gründe)" als verfügbare Aktion auf einer anderen Seite konfiguriert wurde (zum Beispiel "Containersuche").

Höhe der Zunahme oder Abnahme von Material, je nach Art der Änderung Menge. Der Name dieses Feldes ändert sich je nach der ausgewählten Art der Mengenänderung.

Vorgehensweise zum Ändern der Menge eines Containers

Gehen Sie folgendermaßen vor, um die Menge eines Containers zu ändern:

1. Wählen Sie im Container-Menü Mengenänderung. Die Seite "Mengenänderung" wird angezeigt.
2. Wählen Sie einen Container im Container-Feld aus oder geben Sie ihn ein. Die Anwendung aktualisiert den Container-Kopf und die Auswahlfelder, sodass diese Informationen für den angegebenen Container angezeigt werden. Werte werden nur angezeigt, wenn sie für den Container eingegeben wurden.
Hinweis: Die Anwendung füllt das Feld "Container" und den Container-Kopf automatisch, wenn Sie von einer anderen Fertigungsseite auf das Popup "Mengenänderung" zugreifen.
3. Wählen Sie einen Mengenänderungstyp. Die Feld-Etiketten werden entsprechend aktualisiert.
4. Wählen Sie einen Grund aus dem Feld <type>-Grund aus.
5. Geben Sie die Menge in das Feld <Typ>-Menge ein.

Oder

Aktivieren Sie das Kontrollkästchen Aktuelle Menge verwenden, um die gesamte aktuelle Menge des Containers zu verwenden.

6. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
7. Klicken Sie auf Mengenänderung oder in der Befehlsleiste auf Abschicken, um die Menge des Containers zu ändern. Die Anwendung zeigt eine Statusmeldung an, die bestätigt, dass die Menge für den angegebenen Container geändert wurde, und aktualisiert den Wert der "Menge" im Kopf und auf der Seite oder im Popup.
