Komponenten entfernen

Die Transaktion "Komponente entfernen" ermöglicht es Ihnen, Komponenten (Produkte und Ersatzprodukte) zu entfernen, die in einem Container verbraucht wurden. Sie verwenden diese Transaktion außerdem, um eine Anordnung für die entfernten Komponenten anzugeben.

Beim Entfernen von Komponenten

Alle Anzeigefelder sind auf der Seite "Komponente entfernen" leer, bis ein Container ausgewählt wird. Wenn Sie einen Container auswählen, aktualisiert die Anwendung den Kopf und füllt die Tabelle "Materialverbräuche" mit allen Komponenten, die im Container verbraucht wurden.

Sie können jeweils nur eine Komponente entfernen. Die Anwendung aktualisiert die Tabelle "Materialverbräuche", um die Komponente zu entfernen, wenn die "Entfernen"-Transaktion erfolgreich ist. Wenn die Transaktion fehlschlägt, zeigt die Anwendung entsprechend eine Fehlermeldung an.

Wenn Sie einen Container, ein Produkt oder ein Ersatzprodukt im Feld "Zur Auswahl Container/Produkt scannen" scannen oder eingeben, identifiziert und wählt die Anwendung den Container oder das Produkt in der Tabelle "Materialverbräuche". Wenn die Tabelle "Materialverbräuche" keinen Container oder kein Produkt enthält, das dem von Ihnen eingegebenen entspricht, zeigt die Anwendung eine Fehlermeldung an.

Die Anwendung erfordert, dass Sie einen gültigen Container für die Verbrauchstypen "Container (Los)" und "Container (Seriell)" scannen oder eingeben. Darüber hinaus müssen Sie zu einer Fabrik gehören, die einen Standort spezifiziert, wenn Sie sich entscheiden, "Zielort/Lagerort" zu verwenden. Der Standort, den Sie angeben, wenn Sie die Komponente entfernen, muss nicht der Ort sein, von dem sie stammt.

Scannen vs. manuelle Eingabe

Die Anwendung füllt die Nutzereingabefelder und überprüft das Material automatisch, wenn Sie einen Container oder ein Produkt scannen. Sie können auf "Entfernen" klicken, wenn Sie keine Änderungen vornehmen müssen, oder zusätzliche Felder ausfüllen.

Hinweis: Die Anwendung füllt Felder und überprüft das Material nur dann automatisch beim Scannen, wenn ein Terminator für das Gerät und in Portal Studio konfiguriert wurde. Andernfalls müssen Sie das Feld "Zur Auswahl Container/Produkt scannen" durch Drücken der Tab-Taste verlassen, bevor die Anwendung die Felder füllt und das Material überprüft. Weitere Informationen zur Konfiguration eines Terminators für einen Scanner finden Sie in der mit dem Gerät mitgelieferten Dokumentation. Weitere Informationen zum Konfigurieren von Terminatoren in Portal Studio finden Sie in der Opcenter Execution Medical Device and Diagnostics-Anleitung zur Systemadministration und in der Opcenter Execution Core-Anleitung zur Systemadministration.

Sie müssen das Feld "Zur Auswahl Container/Produkt scannen" nach manueller Eingabe verlassen, bevor die Anwendung das Feld überprüft, weitere Dateneingabefelder anzeigt und den Container oder das Produkt überprüft.

Beim Verwenden der Tabelle "Materialverbräuche"

Ihre Auswahl in der Tabelle "Materialverbräuche" bestimmt die Nutzerdateneingabefelder, die unterhalb der Tabelle "Materialverbräuche" angezeigt werden, sowie deren Verfügbarkeit. Wenn Sie z. B. eine Komponente mit dem Verbrauchssteuerungstyp "Container (Seriell)" auswählen, aktiviert die Anwendung das Kontrollkästchen "Gesamte Menge entfernen", füllt das Feld "Entfernte Menge" mit dem Wert "Verbleibende zu verbrauchende Menge" und deaktiviert das Feld "Menge entfernt".

Hinweis: Das Verhalten der Anwendung ist dasselbe, wenn Sie für jede zweite ausgewählte Komponente manuell das Kontrollkästchen "Gesamte Menge entfernen" aktivieren.

Hinzufügen von betroffenen Containern zu Ereignissen

Die Anwendung kann Container automatisch zu Qualitätsereignissen hinzufügen, wenn Sie die Transaktion "Komponente entfernen" für einen Container durchführen, der mit mindestens einem Qualitätsereignis verknüpft ist. Wenn z. B. ein Quell-Container mit einem Ereignis verknüpft ist, fügt die Anwendung den Komponentencontainer, in den die Komponenten entfernt werden, automatisch zum Qualitätsereignis hinzu. Ein Systemadministrator konfiguriert die Transaktion "Komponente entfernen" in der Modellierung, um Container zu Ereignissen hinzuzufügen. Informationen zum automatischen Hinzufügen von betroffenen Containern zu Qualitätsereignissen finden Sie im Opcenter Execution Medical Device and Diagnostics Event Management-Benutzerhandbuch und im Opcenter Execution Core Event Management-Benutzerhandbuch sowie im Opcenter Execution Medical Device and Diagnostics Nonconformance Management-Benutzerhandbuch und im Opcenter Execution Core Nonconformance Management-Benutzerhandbuch.

Seite "Komponente entfernen"

Sie können über das Menü "Container" auf die Seite "Komponente entfernen" zugreifen. Die Seite "Komponente entfernen" ermöglicht es Ihnen, die verbrauchten Komponenten in einem Container anzuzeigen und diese Komponenten aus dem Container zu entfernen.

Hinweis: Die Seite "Komponente entfernen" ist eventuell auch als Popup verfügbar, wenn die Transaktion "Komponente entfernen" als verfügbare Aktion auf einer anderen Seite konfiguriert wurde (zum Beispiel "Containersuche").

Vorgehensweise zum Entfernen verbrauchter Komponenten

Gehen Sie folgendermaßen vor, um verbrauchte Komponenten zu entfernen:

1. Öffnen Sie Komponente entfernen im Container-Menü. Die Seite "Komponente entfernen" wird angezeigt.
2. Wählen Sie einen Container im Container-Feld aus oder geben Sie ihn ein. Die Anwendung aktualisiert den Container-Kopf und die Auswahlfelder, um Informationen für den angegebenen Container anzuzeigen, und zeigt alle verbrauchten Komponenten im Container an.
Hinweis: Die Anwendung füllt das Feld "Container" und den Container-Kopf automatisch, wenn Sie von einer anderen Fertigungsseite auf das Popup "Komponente entfernen" zugreifen.
3. Scannen oder geben Sie den Container, das Produkt oder Ersatzprodukt ein, das Sie aus dem ausgewählten Container entfernen möchten. Die Anwendung wählt die Komponente in der Tabelle "Materialverbräuche" und aktualisiert die Nutzerdateneingabefelder entsprechend.

Oder

Wählen Sie die Komponente, die Sie entfernen möchten, in der Tabelle "Materialverbräuche" aus. Die Anwendung wählt die Komponente in der Tabelle "Materialverbräuche" und aktualisiert die Nutzerdateneingabefelder entsprechend.

4. Wählen Sie einen Entnahmegrund.
5. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
6. Klicken Sie auf Entfernen oder in der Befehlsleiste auf Abschicken. Die Anwendung entfernt die Komponente aus dem Container, zeigt eine Erfolgsmeldung an und entfernt die Komponente aus der Tabelle "Materialverbräuche".
7. Wiederholen Sie Schritte 3-6, um zusätzlichen Komponenten zu entfernen.
