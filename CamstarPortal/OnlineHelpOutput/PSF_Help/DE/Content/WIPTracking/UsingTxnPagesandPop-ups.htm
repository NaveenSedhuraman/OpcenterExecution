Verwenden von Transaktionsseiten und Popups

Container-bezogene Transaktionen können über Seiten und Popups ausgeführt werden. Auf Transaktionsseiten wird über das Container-Menü und auf Transaktions-Popups über die Seiten "Operative Ansicht" und "Containersuche" zugegriffen. Wählen Sie einen Container aus, klicken Sie in der Befehlsleiste auf die Schaltfläche "Fertigungsaktionen" und wählen Sie eine verfügbare Transaktion aus, um ein Transaktions-Popup zu starten.

Hinweis: Die Seiten "Operative Ansicht" und "Containersuche" sind die einzigen Seiten, die für Transaktions-Popups standardmäßig konfiguriert sind. Ihr Systemadministrator kann je nach den Geschäftsbedürfnissen Ihres Unternehmens weitere Seiten zum Anzeigen von Transaktions-Popups konfigurieren.

Darüber hinaus bietet Siemens Transaktionsseiten, die es Ihnen ermöglichen, Transaktionen für mehrere Container auszuführen. Die Seite "Containersuche" beinhaltet die Option, ein Multi-Container-Transaktions-Popup anzuzeigen, wenn mehrere Container ausgewählt sind. Weitere Informationen zu Transaktionen für mehrere Container finden Sie unter "Multi-Container-Transaktionsübersicht".

Ressourcen-bezogene Transaktionen können nur über Seiten ausgeführt werden. Die Ressourcen-Seiten sind über das Ressourcen-Menü zu erreichen.

Hinweis: Sie können Daten in Felder auf den Fertigungsseiten eingeben, indem Sie Barcodes von einem Mobilgerät aus scannen, sofern Sie eine Lizenz zum Scannen von Barcodes besitzen. Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics-Benutzerhandbuch – Erste Schritte oder im Opcenter Execution Core-Benutzerhandbuch – Erste Schritte.

Beim Verwenden einer Transaktionsseite

Wenn die Anwendung eine Transaktionsseite anzeigt, müssen Sie einen Container oder eine Ressource auswählen, bevor die Anwendung vordefinierte Informationen füllt und die Befehlsleiste aktiviert. Wenn Sie beispielsweise "Abmelden" aus dem Containermenü auswählen, zeigt die Anwendung eine leere "Abmelden"-Seite an. Wenn Sie einen Container auswählen, füllt die Anwendung den Bereich "Containerinformationen" mit den Werten, die zuvor für den Container angegeben wurden, und aktiviert die Befehlsleiste.

Hinweis: Multi-Container-Transaktionsseiten enthalten keinen Bereich mit Containerinformationen.

Die Anwendung löscht die Felder auf der Seite und zeigt eine Bestätigungsmeldung an, wenn die Transaktion erfolgreich ist. Wenn die Transaktion fehlschlägt, zeigt die Anwendung eine Fehlermeldung auf der Seite an. Sie können entweder die Werte korrigieren oder die Seite verlassen.

Beim Verwenden eines Transaktions-Popups

Transaktions-Popups gelten nur für Container. Ein Transaktions-Popup zeigt den Containernamen und Informationen über den Container an, der auf der Seite, über die Sie das Popup geöffnet haben, ausgewählt ist. Das Container-Feld ist schreibgeschützt. Wenn Sie die Transaktion an einem anderen Container durchführen möchten, müssen Sie das Popup schließen und diesen Container auswählen.

Hinweis: Multi-Container-Transaktions-Popups enthalten keinen Bereich mit Containerinformationen.

Wenn die Transaktion erfolgreich ist, schließt die Anwendung das Popup und zeigt eine Bestätigungsmeldung auf der Seite an, über die das Popup geöffnet wurde. Wenn die Transaktion fehlschlägt, zeigt die Anwendung eine Fehlermeldung im Popup an. Sie können entweder die Werte korrigieren oder das Popup schließen.

Beim Initiieren einer Transaktion von der Seite "Operative Ansicht"

Die Transaktionen, die Sie auf der Seite "Operative Ansicht" (mit Tabellen) für einen ausgewählten Container initiieren können, hängen von Folgendem ab:

• Ob der Container "In Warteschlange" oder "In Bearbeitung" ist
• Den für diese Seiten standardmäßig und von einem Systemadministrator konfigurierten Aktionen
• Dem Status des Containers

Die "Anmelden"-Transaktion beispielsweise ist nur für einen Container "In Warteschlange" verfügbar, während die "Abmelden"-Transaktion nur für einen Container "In Bearbeitung" verfügbar ist. Manche Transaktionen wie z. B. "Menge ändern" können sowohl für Container "In Warteschlange" als auch "In Bearbeitung" verfügbar sein.

Sie müssen einen Container auswählen, um in der Befehlsleiste die Schaltfläche "Fertigungsaktionen" zu aktivieren. Durch Klicken auf eine Aktion wird das Transaktions-Popup für diesen Container angezeigt.

Beim Initiieren einer Transaktion von der Seite "Containersuche"

Sie müssen eine Suche auf der Seite "Containersuche" durchführen, bevor Sie in der Tabelle "Suchergebnisse" Container auswählen können. Die Transaktionen, die Sie an den ausgewählten Containern initiieren können, hängen von Folgendem ab:

• Den für diese Seiten standardmäßig und von einem Systemadministrator konfigurierten Aktionen
• Dem Status des Containers
• Ob Sie einen oder mehrere Container in der Tabelle "Suchergebnisse" ausgewählt haben

Sie müssen mindestens einen Container auswählen, um in der Befehlsleiste die Schaltfläche "Fertigungsaktionen" zu aktivieren. Wenn Sie mehrere Container ausgewählt haben, zeigt die Befehlsleistenschaltfläche "Fertigungsaktionen" Aktionen an, die für die ausgewählten Container gleichzeitig ausgeführt werden können. Durch Klicken auf eine Aktion wird das Transaktions-Popup für die ausgewählten Container angezeigt.
