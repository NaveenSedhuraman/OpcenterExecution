Mehrere Container abmelden (nicht standardmäßig)

Die Transaktion "Abmelden - nicht standardmäßig (mehrere)" ermöglicht es Ihnen, mehrere Container gleichzeitig zu einem beliebigen Schritt in einem beliebigen Arbeitsplan abzumelden.

Beim Verwenden von "Abmelden - nicht standardmäßig" für mehrere Container

Die Anwendung verschiebt den Container in den nächsten Schritt im aktuellen Arbeitsplan, es sei denn, Sie geben einen anderen Arbeitsplan oder Schritt an. Arbeitspläne werden in der Modellierung von einem Systemadministrator festgelegt.

Die Anwendung überprüft, ob die ausgewählten Container aktiv, "In Bearbeitung" und nicht "Angehalten" sind. Darüber hinaus kann ein Container kein Kind-Container sein.

Unter "Multi-Container-Transaktionsübersicht" finden Sie Informationen darüber, wie die Anwendung beim Ausführen von Transaktionen an mehreren Containern funktioniert.

Seite "Abmelden - nicht standardmäßig (mehrere)"

Sie können über das Container-Menü auf die Seite "Abmelden - nicht standardmäßig (mehrere)" zugreifen. Sie können auf das Popup "Abmelden - nicht standardmäßig (mehrere)" von einer anderen Fertigungsseite aus zugreifen (z. B. von der "Containersuche"), wenn die "Abmelden - nicht standardmäßig (mehrere)"-Aktion für diese Seite konfiguriert wurde. Sowohl die Seite "Abmelden - nicht standardmäßig (mehrere)" als auch das Popup ermöglichen es Ihnen, einen oder mehrere Container zum Abmelden zu scannen oder einzugeben.

Die Seite und das Popup sind identisch in Aussehen und Funktionalität - mit einer Ausnahme. Die Anwendung füllt das Popup "Abmelden - nicht standardmäßig (mehrere)" automatisch mit den Containern, die Sie auf der Fertigungsseite ausgewählt haben, von der aus Sie auf das Popup zugegriffen haben.

Falls die Ressource für einen oder mehrere Container unterschiedlich ist, versteckt die Anwendung das "Ressource"-Feld und verschiebt die Felder "Zu Arbeitsplan" und "Zu Schritt" nach links.

Nur Anzeige

Zu Arbeitsplan

Ziel-Arbeitsplan und Schritte. Die Schritte werden angezeigt, wenn Sie den Arbeitsplan erweitern. Die Anwendung verschiebt die Container in den nächsten Schritt im aktuellen Arbeitsplan, es sei denn, Sie geben hier einen anderen Arbeitsplan oder Schritt an.

Optional

Zu Schritt

Schritt im Ziel-Arbeitsplan. Die Anwendung füllt dieses Feld, wenn Sie einen Schritt (Spez.) aus dem erweiterten Arbeitsplan im Feld "Zu Arbeitsplan" auswählen.

Optional

Zusätzliche Felder anzeigen/ausblenden

Optionale Felder sind standardmäßig ausgeblendet, wenn die Anwendung eine Seite oder ein Popup anzeigt. Durch Klicken auf die Steuerung "Anzeigen" werden die Felder angezeigt. Durch Klicken auf die Steuerung "Verstecken" werden die Felder ausgeblendet.

  

Zu Standort

Standort, an den die Container im nächsten Schritt zur Bearbeitung bewegt werden. Ein Standort kann eine ganze Fabrik, ein Bereich einer Fabrik oder so spezifisch wie ein Regal-Identifikator sein. Standorte werden innerhalb des Fabrik-Objekts in der Modellierung definiert.

Optional

Kommentare

Zusätzliche Informationen oder Anweisungen für diese Transaktion oder für diesen Container. Das Feld akzeptiert maximal 255 Zeichen. Wenn die Kommentare 255 Zeichen überschreiten, verkürzt die Anwendung die Daten.

Optional

Vorgehensweise zum Abmelden mehrerer Container zu einem beliebigen Schritt in einem Arbeitsplan

Gehen Sie folgendermaßen vor, um mehrere Container zu einem beliebigen Schritt in einem Arbeitsplan abzumelden:

1. Öffnen Sie Abmelden - nicht standardmäßig (mehrere) im Menü Container. Die Seite "Abmelden - nicht standardmäßig (mehrere)" wird angezeigt.
2. Wird die Seite "Abmelden - nicht standardmäßig (mehrere)" oder das Popup "Abmelden - nicht standardmäßig (mehrere)" angezeigt?

Wird Folgendes angezeigt . . .

Dann . . .

Seite "Abmelden - nicht standardmäßig (mehrere)"

Die Tabelle "Abzumeldende Container" ist leer. Gehen Sie zu Schritt 3.

Popup "Abmelden - nicht standardmäßig (mehrere)"

Die Tabelle "Abzumeldende Container" wird mit den Containern gefüllt, die Sie auf der Seite ausgewählt haben, von der aus Sie das Popup gestartet haben. Wenn Sie zusätzliche abzumeldende Container hinzufügen möchten, gehen Sie zu Schritt 3. Fahren Sie andernfalls mit Schritt 6 fort.
3. Scannen Sie oder geben Sie einen Container in das Feld Auszuwählenden Container eingeben ein.
4. Drücken Sie die Eingabetaste oder verlassen Sie das Feld durch Drücken der Tab-Taste. Der Container wird zur Tabelle "Abzumeldende Container" hinzugefügt und automatisch ausgewählt.
5. Wiederholen Sie Schritte 3-4, um zusätzliche Container hinzuzufügen.
6. Gewährleisten Sie, dass die ausgewählten Container diejenigen sind, die Sie abmelden möchten.
7. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
8. Klicken Sie auf Ausgewählte abmelden oder in der Befehlsleiste auf Abschicken. Die Anwendung zeigt ein Status-Icon für jeden Container an, das angibt, ob die Transaktion erfolgreich ausgeführt wurde.
9. Schließen Sie das Popup, falls es angezeigt wird, indem Sie auf das Symbol "Schließen" klicken, um zur vorherigen Seite zurückzukehren.
