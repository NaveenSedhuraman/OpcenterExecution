Containeretiketten drucken

Die Transaktion "Containeretikett drucken" ermöglicht es Ihnen, ein oder mehrere Etiketten manuell für einen Container zu drucken. Ein Systemadministrator kann das Portal so einstellen, dass Etiketten für jegliche Transaktionen automatisch gedruckt werden, aber es kann Situationen geben, in denen Sie das Etikett manuell drucken müssen (wenn z. B. keine Transaktionen an einem Container durchgeführt werden oder kein automatischer Druck mit der entsprechenden Transaktion verbunden ist). Wenn Sie ein Containeretikett manuell ausdrucken, so druckt die Anwendung das Etikett der zuletzt durchgeführten Transaktion.

Vor dem manuellen Drucken von Containeretiketten

Ein Systemadministrator muss zumindest eine Druckeretiketten-Definition und eine Druckerwarteschlange in der Modellierung definiert haben, bevor Sie Etiketten drucken können. Der Systemadministrator kann eine Druckeretiketten-Definition mit einer Spezifikation und Druckerwarteschlangen-Definitionen mit den Ressourcen- oder Operations-Modellierungsobjekten verbinden. Weitere Informationen zum Definieren und Zuordnen von Druckeretiketten- und Druckerwarteschlangendefinitionen finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch und im Opcenter Execution Core Modeling-Benutzerhandbuch.

Beim manuellen Drucken von Containeretiketten

Die Anwendung füllt das Feld "Drucketikettendefinition" automatisch aus, wenn nur eine Drucketikettendefinition mit der momentanen Spezifikation des Containers verknüpft ist. Wenn keine Druckeretiketten Definitionen zugeordnet wurden, stehen alle Definitionen zur Auswahl. Wenn die Anwendung standardmäßig auf eine bestimmte Druckeretiketten Definition eingestellt ist, können Sie diese ändern.

Die Anwendung füllt das Feld "Druckerwarteschlange" automatisch, wenn die Druckerwarteschlange nur mit der momentanen Operation des Containers oder mit der Ressource, die momentan den Container bearbeitet verknüpft ist. Wenn eine Druckwarteschlange sowohl für Operation und Ressource angegeben wurde, dann legt die Common Value Expression (CVE), die im Designer festgelegt ist, die Rangfolge fest.

Seite "Containeretiketten drucken"

Sie können über das Menü "Container" auf die Seite "Containeretiketten Drucken" zugreifen. Die Seite "Containeretiketten Drucken" ermöglicht es Ihnen, ein oder mehrere Etiketten für die ausgewählten Container oder Behälter zu drucken.

Hinweis: Das Menüelement "Containeretikett drucken" ist nur dann verfügbar, wenn Ihr Unternehmen das Modul "Label Printing" erworben hat und ein Systemadministrator das Menü "Container" um die Option "Containeretikett drucken" erweitert hat. Wenn das Menüelement "Containeretiketten drucken" Menüelement nicht angezeigt wird, benachrichtigen Sie Ihren Systemadministrator.

Vorgehensweise zum manuellen Drucken eines Containeretiketts

Gehen Sie folgendermaßen vor, um ein Containeretikett manuell zu drucken:

1. Öffnen Sie auf Containeretikett Drucken im Container-Menü. Die Seite "Containeretikett drucken" wird angezeigt.
2. Wählen Sie einen Container im Container-Feld aus oder geben Sie ihn ein. Die Anwendung aktualisiert den Container-Kopf und die Auswahlfelder, sodass diese Informationen für den angegebenen Container angezeigt werden. Werte werden nur angezeigt, wenn sie für den Container eingegeben wurden. Außerdem kann die Anwendung Standardwerte in den Feldern "Drucketikettendefinition" und "Druckwarteschlange" anzeigen.
3. Wählen Sie eine Druckeretiketten Definition aus.
4. Wählen Sie eine Druckerwarteschlange aus.
5. Geben Sie die Anzahl der Etiketten, die Sie drucken wollen, in das Etikettenzahl-Feld ein.
6. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
7. Klicken Sie auf Etikett drucken oder in der Befehlsleiste auf Abschicken. Eine Bestätigungsmeldung wird angezeigt und das Containeretikett wird am entsprechenden Etikettendrucker ausgedruckt. Oder, sollte die entsprechende Druckerwarteschlange die Option "Als Datei ausgeben" verwenden, so wird die Information des Etiketts in die entsprechende Textdatei der Warteschlange geschrieben.

Vorgehensweise zum manuellen Drucken eines Behälteretiketts

Gehen Sie folgendermaßen vor, um ein Behälteretikett manuell zu drucken:

1. Öffnen Sie auf Containeretikett Drucken im Container-Menü. Die Seite "Containeretikett drucken" wird angezeigt.
2. Wählen Sie einen Behälter im Behälter-Feld aus oder geben Sie ihn ein. Die Anwendung aktualisiert Überschrift und Auswahlfelder, um Informationen für den angegebenen Behälter anzuzeigen. Werte werden nur angezeigt, wenn sie für den Behälter eingegeben wurden. Außerdem kann die Anwendung Standardwerte in den Feldern "Drucketikettendefinition" und "Druckwarteschlange" anzeigen.
3. Wählen Sie eine Drucketikettendefinition aus.
4. Wählen Sie eine Druckerwarteschlange aus.
5. Geben Sie die Anzahl der Etiketten, die Sie drucken wollen, in das Etikettenzahl-Feld ein.
6. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
7. Klicken Sie auf Etikett drucken oder in der Befehlsleiste auf Abschicken. Eine Bestätigungsmeldung wird angezeigt und das Behälteretikett wird am entsprechenden Etikettendrucker ausgedruckt. Oder, sollte die entsprechende Druckerwarteschlange die Option "Als Datei ausgeben" verwenden, so wird die Information des Etiketts in die entsprechende Textdatei der Warteschlange geschrieben.
