Arbeiten mit einer Binärdatei

Wenn Ihre Drucksoftware für Barcode-Etiketten, wie etwa BarTender® eine binäre Datei erstellt, können Sie eine Vorlagendatei in ZPL (Zebra Print Language) erzeugen.

Die Software ermöglicht es Ihnen, eine Ausgabedatei durch einen druckerspezifischen Treiber zu verfolgen. Dazu werden die Eigenschaften des Zebra-Druckers so geändert, dass der Druckercode über eine Protokollierungsoption aufgezeichnet wird. Das Ergebnis ist eine .prn-Datei, die bei Verwendung des Zebra-Druckertreibers das erforderliche ZPL-Dateiformat liefert.

Sehen Sie in den Anleitungen des Herstellers nach, um genauere Informationen zur Barcode-Druckersoftware zu finden.
