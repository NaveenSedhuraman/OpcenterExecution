Anzeigen von Detailinformationen für gedruckte Etiketten

Die Historie eines jeden Etiketts, gedruckt oder neu gedruckt, wird aufgezeichnet. Wenn konfiguriert, nutzt das Herstellungsprotokoll den Verlauf, um Details für jedes gedruckte und erneut gedruckte Etikett zu liefern. Diese Informationen umfassen die Etikettenzahl, die Drucketiketten Definition, die Druckerwarteschlange, Tag-Namen und deren Werte.

Die Vorgehensweise zum Konfigurieren eines Produktionsprotokolls finden Sie im Opcenter Execution Medical Device and Diagnostics Audit Trail-Benutzerhandbuch und im Opcenter Execution Core Audit Trail-Benutzerhandbuch.

Vorgehensweise zum Anzeigen von Informationen über ein gedrucktes oder erneut gedrucktes Etikett

Gehen Sie folgendermaßen vor, um Informationen über ein gedrucktes oder erneut gedrucktes Etikett anzuzeigen:

1. Öffnen Sie die Seite Produktionsprotokoll über das Menü Suchen. Die Seite "Produktionsprotokoll" wird angezeigt.
Hinweis: Der Name der Produktionsprotokollseite hängt davon ab, wie sie vom Systemadministrator konfiguriert wurde.
2. Wählen Sie falls nötig den Aufzeichnungstyp.
3. Geben Sie den Container Aufzeichnungs-Namen ein, für den ein oder mehrere Etiketten gedruckt wurden.
4. Klicken Sie auf Suchen. Die Verlaufs-Hauptlinie-Tabelle listet den Verlauf der Transaktionen dieses Containers auf.
5. Klicken Sie auf die Zeile in der Tabelle, welche die Transaktion enthält, deren Informationen Sie anzeigen möchten. Eine "Etikettendruck"-Schaltfläche wird angezeigt, wenn Etiketten für diese Transaktion ausgedruckt wurden.
6. Klicken Sie auf Etikettendruck oder in der Befehlsleiste auf Abschicken. Die Anwendung zeigt Informationen über jegliches Etikett, dass für die ausgewählte Transaktion gedruckt oder erneut gedruckt wurde.
