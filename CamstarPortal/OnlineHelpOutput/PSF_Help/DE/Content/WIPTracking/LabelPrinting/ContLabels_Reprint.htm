Containeretiketten erneut drucken

Die Transaktion "Containeretikett erneut drucken" ermöglicht es Ihnen, ein Containeretikett nach Bedarf erneut zu drucken. Z. B. können Sie ein Etikett erneut drucken, wenn es beschädigt ist, oder wegen eines Druckerproblems nicht ordentlich ausgedruckt wurde.

Beim erneuten Drucken von Containeretiketten

Die Anwendung zeigt alle Transaktionen in absteigender Reihenfolge an, die an dem Container durchgeführt wurden, der in der "Transaktionstabelle" ausgewählt ist. Die neueste Transaktion wird am Anfang der Liste angezeigt. Falls die Anzahl der Transaktionen den Platz in der Tabelle übersteigt, fügt die Anwendung eine Umblätter-Funktion hinzu.

Sie können ein Containeretikett nur für eine Transaktion gleichzeitig ausdrucken. Wenn Sie die Transaktion auswählen, so setzt die Anwendung die Werte in der "Druckeretiketten Definition", der "Druckerwarteschlange" und des "Etikettenanzahl"-Feldes auf die Werte die das letzte Mal beim Drucken verwendet wurden zurück. Sie haben die Möglichkeit, andere Werte für die Druckerwarteschlange und die Etikettenanzahl-Felder einzugeben oder auszuwählen.

Beim erneuten Drucken von Behälteretiketten

Sie müssen das Behälter-Control über das Portal-Studio hinzufügen, bevor Sie Etiketten für einen Behälter erneut drucken können.

Die Anwendung zeigt den Container im Containerfeld an, wenn Sie einen Behälter ausgewählt haben. Wenn Sie einen Container ausgewählt haben, der in einem Behälter ist, so zeigt die Anwendung den entsprechenden Behälter im Behälterfeld an.

Seite "Containeretikett erneut drucken"

Sie können auf die Seite "Containeretiketten erneut drucken" vom "Containermenü" aus zugreifen. Die Seite "Containeretiketten erneut drucken" ermöglicht es Ihnen, ein oder mehrere Etiketten für einen bestimmten Container oder Behälter neu zu drucken.

Hinweis: Das Menüelement "Containeretiketten erneut drucken" ist nur dann verfügbar, wenn Ihr Unternehmen das Modul "Label Printing" erworben hat und ein Systemadministrator das Menü "Container" um die Option "Containeretikett erneut drucken" erweitert hat. Wenn das Menüelement "Containeretiketten erneut drucken" nicht angezeigt wird, benachrichtigen Sie Ihren Systemadministrator.

Vorgehensweise zum erneuten Drucken eines Containeretiketts

Gehen Sie wie folgt vor, um ein Containeretikett erneut zu drucken:

1. Öffnen Sie auf Containeretikett erneut drucken im Container-Menü. Die Seite "Containeretikett erneut drucken" wird angezeigt.
2. Wählen Sie einen Container im Container-Feld aus oder geben Sie ihn ein. Die Anwendung aktualisiert die Containerüberschrift und füllt die Transaktionsdetails-Tabelle.
3. Wählen Sie eine Transaktion in der Transaktionsdetails-Tabelle aus. Die Anwendung zeigt die Druckeretiketten Definition, die Druckerwarteschlange und die Etikettenanzahl an, die beim letzten Druck verwendet wurde.
4. Sie können diese Werte verändern, wenn dies notwendig ist.
• Druck-Warteschlange
• Etikettenzahl
5. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
6. Klicken Sie auf Etikett erneut ausdrucken oder in der Befehlsleiste auf Abschicken. Eine Bestätigungsmeldung wird angezeigt und das Containeretikett wird am entsprechenden Etikettendrucker ausgedruckt. Die Anwendung schreibt die Etiketteninformationen in die entsprechende Textdatei, die für die Druckerwarteschlange angegeben wurde, wenn die "Als Datei ausgeben" Option ausgewählt wurde.
