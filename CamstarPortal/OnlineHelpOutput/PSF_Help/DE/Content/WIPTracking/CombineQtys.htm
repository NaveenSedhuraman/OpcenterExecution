Mengen vereinigen

Die Transaktion "Mengen vereinigen" überträgt Mengen von einem oder mehreren bestehenden Containern zu einem anderen. Eine typische Anwendungsmöglichkeit für die Transaktion "Menge vereinigen" ist die erneute Vereinigung von Containern, die zuvor mithilfe der Transaktion "Menge aufteilen" aufgeteilt wurden.

Eine "Vereinigen"-Transaktion bewegt weder die Quell- noch die Zielcontainer. Es impliziert auch nicht, dass eine Prozess-Aktivität (Durchsatz) an einem der Container durchgeführt wurde. Jedoch ermöglicht Ihnen die Transaktion "Mengen vereinigen", die Container zu schließen, aus denen Sie die Menge übertragen.

Ausführungsregeln für "Container vereinigen" und "Mengen vereinigen"

Die Container in einer "Vereinigen"-Transaktion müssen folgende Kriterien erfüllen:

• Keiner der zu vereinigenden Container darf geschlossen oder angehalten sein.
• Ziel- und Quell-Container müssen das gleiche Produkt enthalten und auf der gleichen Container-Ebene sein.
• Ziel- und Quell-Container müssen an derselben Operation in derselben Fabrik sein.
• Die Transaktion muss vor den Ablaufdaten erfolgen, wenn Ablaufdaten für den Quell (Von)-Container oder den Ziel (Zu)-Container angegeben wurden. Die "Mengen vereinigen"-Transaktion scheitert, wenn das Ablaufdatum für einen der Container überschritten wurde, bevor Sie die Transaktion ausführen. Falls nur einer der Container ein Ablaufdatum hat, muss die Transaktion vor diesem Datum erfolgen.

Darüber hinaus sollten bei der Ausführung der Transaktion "Container vereinigen" alle Quell-Eltern-Container Kind-Container haben.

Beim Vereinigen von Mengen

Alle Anzeigefelder sind auf der Seite "Mengen vereinigen" leer, bis ein Container ausgewählt wird. Anzeigefelder werden im "Mengen vereinigen"-Popup automatisch gefüllt. Informationen über das Ausführen einer Transaktion über eine Seite oder ein Popup finden Sie unter "Verwenden von Transaktionsseiten und Popups".

Die Anwendung zeigt alle Quell-Container für die Vereinigung mit dem Ziel-Container in der Tabelle "Container zum Zusammenführen", wenn Sie einen Ziel-Container im Kopf auswählen. Sie können das Kontrollkästchen in der Kopfzeile aktivieren, um alle Quell-Container automatisch auszuwählen, oder Sie aktivieren die Kontrollkästchen für einzelne Container. Darüber hinaus können Sie den Quell-Container ins Feld "Auszuwählenden Container eingeben" einscannen oder manuell eingeben und das Feld durch Drücken der Tab-Taste verlassen. Die Anwendung wählt diesen Container in der Tabelle "Container zum Zusammenführen" aus.

Hinweis: Die Anwendung zeigt eine Fehlermeldung an, wenn Sie den Namen eines nicht erlaubten oder nicht vorhandenen Containers im Feld "Auszuwählenden Container eingeben" eintragen.

Sie müssen mindestens einen Container in der Tabelle "Container zum Zusammenführen" auswählen, um die Transaktion "Mengen vereinigen" auszuführen. Die Anwendung zeigt folgende Fehlermeldung an, wenn Sie auf die Schaltfläche "Ausgewählte Container vereinigen" klicken, ohne vorher einen Container in der Tabelle auszuwählen. Die Anwendung vereinigt nur die Quell-Container, die in der Tabelle ausgewählt wurden.

Wenn Sie eine Menge zum Vereinigen angeben:

• Sie können die Menge eines Containers nicht mit sich selbst vereinigen.
• Die zu übertragende Menge kann nicht größer sein als die Menge im Quell-Container.
• Durch Aktivieren des Kontrollkästchens "Schließen, wenn leer" wird der Quell-Container automatisch geschlossen, wenn seine Menge null erreicht.
• Der Ziel-Container wird auf das frühere Ablaufdatum gesetzt, das von den beiden Containern verwendet wird. Wenn jedoch nur der Quell-Container ein Ablaufdatum hat, wird das Ablaufdatum des Ziel-Containers auf das Ablaufdatum des Quell-Containers gesetzt.

Wenn ein Vereinigen durchgeführt wird, finden folgende Ereignisse statt:

• Die angegebene Menge wird von einem Container (Quelle) auf einen anderen bestehenden Container (Ziel) übertragen.
• Die aus dem Quell-Container hinzugefügte Menge übernimmt die Attribute des Ziel-Containers.

Hinzufügen von betroffenen Containern zu Ereignissen

Die Anwendung kann Container automatisch zu Qualitätsereignissen hinzufügen, wenn Sie die Transaktion "Mengen vereinigen" für einen Container durchführen, der mit mindestens einem Qualitätsereignis verknüpft ist. Wenn z. B. ein Quell-Container mit einem Ereignis verknüpft ist, fügt die Anwendung den Ziel-Container automatisch zum Qualitätsereignis hinzu. Ein Systemadministrator konfiguriert die Transaktion "Mengen vereinigen" in der Modellierung, um Container zu Ereignissen hinzuzufügen. Informationen zum automatischen Hinzufügen von betroffenen Containern zu Ereignissen finden Sie im Opcenter Execution Medical Device and Diagnostics Event Management-Benutzerhandbuch und im Opcenter Execution Core Event Management-Benutzerhandbuch sowie im Opcenter Execution Medical Device and Diagnostics Nonconformance Management-Benutzerhandbuch und im Opcenter Execution Core Nonconformance Management-Benutzerhandbuch.

Seite "Mengen vereinigen"

Sie können über das Container-Menü auf die Seite "Mengen vereinigen" zugreifen. Die Seite "Mengen vereinigen" erlaubt es Ihnen, die Produktmengen von einem oder mehreren Quell-Containern auf einen anderen vorhandenen Ziel-Container zu übertragen.

Hinweis: Die Seite "Mengen vereinigen" ist eventuell auch als Popup verfügbar, wenn die Transaktion "Mengen vereinigen" als verfügbare Aktion auf einer anderen Seite konfiguriert wurde (z. B. "Containersuche").

Vorgehensweise zum Vereinigen von Mengen

Gehen Sie folgendermaßen vor, um Mengen zu vereinigen:

1. Wählen Sie Mengen vereinigen im Container-Menü. Die Seite "Mengen vereinigen" wird angezeigt.
2. Wählen Sie einen Container im Container-Feld aus oder geben Sie ihn ein. Die Anwendung aktualisiert den Container-Kopf und die Auswahlfelder, sodass diese Informationen für den angegebenen Container angezeigt werden. Werte werden nur angezeigt, wenn sie für den Container eingegeben wurden. Darüber hinaus zeigt die Anwendung alle Container, die für die Vereinigung mit dem Ziel-Container zugelassen sind, in der Tabelle "Container zum Vereinigen" an.
Hinweis: Die Anwendung füllt das Feld "Container" und den Container-Kopf automatisch, wenn Sie von einer anderen Fertigungsseite auf das Popup "Mengen vereinigen" zugreifen.
3. Wählen Sie die Quell-Container, die mithilfe einer der folgenden Methoden mit dem Ziel-Container vereinigt werden sollen:
• Setzen Sie ein Häkchen in das Ankreuzfeld in der Kopfzeile der Tabelle Container vereinigen, um alle Container auszuwählen.
• Setzen Sie ein Häkchen in das Ankreuzfeld für jeden einzelnen Container.
• Scannen oder geben Sie einen Container in das Feld Auszuwählenden Container eingeben ein und verlassen Sie das Feld durch Drücken der Tab-Taste. Die Anwendung hebt die Container-Zeile hervor und wählt das entsprechende Ankreuzfeld des Containers aus.
4. Klicken Sie in die Spalte Anzahl zum Zusammenführen für den ausgewählten Container, um die Menge bei Bedarf zu ändern. Die Anwendung aktiviert die Zelle, wenn Sie in die Spalte klicken.
5. Wiederholen Sie Schritt 3-4, bis Sie jeden Quell-Container ausgewählt haben, den Sie mit den Ziel-Container vereinigen möchten.
6. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
7. Klicken Sie auf Ausgewählte Container vereinigen oder in der Befehlsleiste auf Abschicken. Die Anwendung überträgt die angegebenen Mengen von den Quell-Containern zum Ziel-Container und zeigt eine Erfolgsmeldung an.
