Operative Ansicht

Die Seite "Operative Ansicht" ist für die Containerbearbeitung in einer Produktionslinie, z. B. einem Montageschritt, ausgelegt. Sie können einen Container scannen oder auswählen, um dieselben Transaktionen für einen Container auszuführen, basierend darauf, wie Ihre Produktionslinie funktioniert.

Die Seite "Operative Ansicht " zeigt eine "Container in Bearbeitung"-Tabelle an. Sie listet die Container auf, die zur Bearbeitung an der Operation zur Verfügung stehen.

Die Seite kann außerdem eine Tabelle "Container in Warteschlange" für Container anzeigen, die gerade an der Operation angekommen sind und auf Bearbeitung warten.

Sie scannen einen Container in einer der Tabellen oder wählen ihn aus und wählen anschließend die entsprechende Aktion zur Bearbeitung aus.

Beispiel

Ein Montageoperator kann einen Container bearbeiten, der "In Warteschlange" ist, indem er eine "Anmelden"- oder "Sofort Anmelden"-Transaktion durchführt (um den Status von "In Warteschlange" zu "In Bearbeitung" zu ändern). Der Operator kann dann entweder weitere Container "In Warteschlange" bearbeiten oder einen Container "In Bearbeitung" für die weitere Bearbeitung mit Aktionen wie "Abmelden", "Nacharbeit", "Öffnen", "Schließen" usw. festlegen.

Folgendes tritt auf, wenn Sie einen Container auswählen oder scannen oder auf einen Container in einer der Tabellen klicken:

• Der ausgewählte Container wird in der Tabelle hervorgehoben und im Container-Feld angezeigt.
• Das Symbol "Containerstatus" zeigt den aktuellen Zustand des ausgewählten Containers an.
• Die Schaltfläche "Fertigungsaktionen" wird in der Befehlsleiste angezeigt.

Wenn Sie in der Befehlsleiste auf die Schaltfläche "Fertigungsaktionen" klicken, werden die Transaktionen angezeigt, die zur Bearbeitung für diesen Container zur Verfügung stehen und für die Anzeige konfiguriert wurden. Einige der verfügbaren Transaktionen weisen hinter dem Namen Auslassungspunkte auf, z. B. "Abmelden...". Die Auslassungspunkte zeigen an, dass die Anwendung Folgendes durchführt, wenn Sie auf die Transaktion klicken:

• Zeigt das entsprechende Transaktions-Popup an. In einigen Fällen wie bei "Automatischer Ablauf", die über kein entsprechendes Popup verfügen, wird die Transaktionsseite auf einer eigenen Registerkarte angezeigt, und der Fokus wird auf diese Seite verschoben, während die Seite "Operative Ansicht" in ihrer eigenen Registerkarte geöffnet bleibt.
• Zeigt den ausgewählten Container-Kopf im Transaktions-Popup (oder auf der -Seite) an.

Sie können dann die erforderlichen Informationen im Popup eingeben und es vervollständigen.

Transaktionen ohne Auslassungspunkte nach dem Namen, z. B. "Sofort anmelden" und "Sofort abmelden", geben an, dass die entsprechende Transaktion sofort unter automatischer Verwendung der Standardwerte für die Transaktion ausgeführt wird.

Hinweis: Wenn eine Umlaufbestandsnachricht entweder für die "Anmelden"- oder die "Abmelden"-Transaktion konfiguriert wurde, so verlangt die Anwendung von Ihnen, diese zu bestätigen, bevor die "Sofort anmelden"- oder "Sofort abmelden"-Transaktion ausgeführt werden kann.

Durch Klicken in der Befehlsleiste auf die Schaltfläche "Dokumentsatz" wird eine Liste der Dokumente/SOPs im Dokumentsatz angezeigt, falls dem Container Dokumente zugeordnet sind. Sie können ein Dokument in der Liste auswählen und es öffnen.

Containerpriorität mit Erledigungsregeln

Wenn Erledigungsregeln durchgesetzt werden und abhängig davon, wie sie konfiguriert wurden, sind Sie eventuell darauf beschränkt, nur den obersten Container in der jeweiligen Liste abzumelden. Container werden in einer standardmäßigen Manufacturing-Einstellung nach Priorität, Fälligkeitsdatum und Container sortiert.

Vorgehensweise zum Bearbeiten eines Containers durch Verwendung von Tabellen

Gehen Sie folgendermaßen vor, um einen Container mithilfe der Seite "Operative Ansicht" zu bearbeiten:

1. Öffnen Sie die Seite Operative Ansicht im Menü Container. Die Seite "Operative Ansicht" wird angezeigt. Standardmäßig sind die Optionen "In Warteschlange" und "In Bearbeitung" ausgewählt und beide Tabellen werden angezeigt.
2. Klicken Sie Optional auf die Option In Warteschlange oder In Bearbeitung, um nur Container anzuzeigen, die bereits in der Warteschlange oder in Bearbeitung sind.
3. Führen Sie eine der folgenden Aktionen für einen Container durch:
• Wählen Sie einen Container im Feld Container aus. Der ausgewählte Containername wird im Feld "Container" angezeigt, die Zeile mit diesem Container wird in der entsprechenden Tabelle hervorgehoben und Schaltflächen für die verfügbaren Transaktionen werden angezeigt.
• Scannen Sie einen Container. Der gescannte Containername wird im Feld "Container" angezeigt, die Zeile mit diesem Container wird in der entsprechenden Tabelle hervorgehoben und Schaltflächen für die verfügbaren Transaktionen werden angezeigt.
• Klicken Sie auf die Zeile in der entsprechenden Tabelle, die die zu verarbeitenden Container auflistet. Die Zeile wird hervorgehoben, der Containername wird im Feld "Container" angezeigt und die Schaltfläche "Fertigungsaktionen" wird in der Befehlsleiste angezeigt.
4. Klicken Sie in der Befehlsleiste auf die Schaltfläche "Fertigungsaktionen" und wählen Sie die entsprechende Transaktion aus.Wenn Sie Folgendes ausführen:
• Transaktion "Sofort anmelden": Eine Erfolgsmeldung wird angezeigt, der Container wird bearbeitet, die Container-Zeile wird in der Tabelle "Container in Warteschlange" gelöscht und in der Tabelle "Container in Bearbeitung" angezeigt und ausgewählt. Der Container bleibt für die eventuelle weitere Bearbeitung ausgewählt.
• Eine "Sofort anmelden"-Transaktion: Eine Erfolgsmeldung wird angezeigt, der Container wird bearbeitet und in der Tabelle "Container in Bearbeitung" angezeigt und ausgewählt. Der Container bleibt für die eventuelle weitere Bearbeitung ausgewählt.
• Beliebige andere Transaktion: Das Transaktions-Popup oder die Transaktions-Seite wird angezeigt.
5. Schließen Sie die Transaktion im Popup oder auf der Seite ab. Der Container wird entsprechend bearbeitet, eine Erfolgsmeldung wird angezeigt und der Container bleibt für die eventuelle weitere Bearbeitung ausgewählt.
6. Wiederholen Sie Schritte 2 bis 5, um bei Bedarf einen weiteren Container zu bearbeiten.
