Container versenden

Die Seite und das Popup "Versenden" erlauben es Ihnen, einen Container an einen entfernten Standort zu versenden, z. B. an einen Kunden oder Händler oder an eine andere Fabrik, einen anderen Standort oder Arbeitsplan-Schritt.

Beim Versenden von Containern

Alle Anzeigefelder der Seite "Versenden" sind leer, bis Sie einen Container auswählen. Anzeigefelder werden automatisch im Popup "Versenden" gefüllt. Informationen über das Ausführen einer Transaktion über eine Seite oder ein Popup finden Sie unter "Verwenden von Transaktionsseiten und Popups".

Die Anwendung kann diese Felder füllen, wenn Sie ein Versandziel auswählen, falls die Informationen in der "Versandziel"-Modellierungsdefinition angegeben wurden.

• An Kunde
• Zu Standort
• An Fabrik

Sie können die Informationen, die von der Anwendung gefüllt wurden, bei Bedarf ändern. Darüber hinaus können Sie den Container an einen Arbeitsplan versenden. Sie müssen einen Schritt im Arbeitsplan angeben.

Hinweis: Standorte müssen einer Fabrik zugewiesen sein. Daher müssen Sie Standorte für die Fabrik in der Modellierung definiert haben oder sie sind nicht in der Liste "Zu Standort" verfügbar. Informationen zu Standorten und Fabriken finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch und im Opcenter Execution Core Modeling-Benutzerhandbuch.

Versand-Einschränkungen

Jede dieser Bedingungen verhindert, dass der Container versandt wird:

• Der Container ist "In Warteschlange". Ein Container muss im Prozess sein, bevor Sie ihn versenden können.
• Der Status des Containers ist "Angehalten" oder "Geschlossen".
• Der Container ist ein zugeordneter Kind-Container. Sie müssen den Kind-Container trennen, bevor Sie ihn unabhängig vom Eltern-Container versenden können.
• Es wurden nicht alle Materialien verbraucht. Wenn das Häkchen "Materialverbrauch prüfen" für die Spezifikation gesetzt ist, überprüft die Anwendung, ob alle Materialien und Mengen für den aktuellen Schritt verbraucht wurden, bevor sie den Container zum nächsten Schritt im Arbeitsplan bewegt.
• Der Container ist am letzten Schritt im Arbeitsplan. Der Container wird an den nächsten Schritt im Arbeitsplan bewegt, wenn Sie ihn versenden. Daher erhalten Sie eine Fehlermeldung, wenn Sie am letzten Schritt im Arbeitsplan sind.

Seite "Versenden"

Sie können über das Container-Menü auf die "Versenden"-Seite zugreifen. Die "Versenden"-Seite erlaubt es Ihnen, einen Container einzugeben oder auszuwählen und den Kunden, Standort, die Fabrik oder den Arbeitsplan-Schritt auszuwählen, an den/die Sie den Container versenden möchten.

Zu Standort

Zielort innerhalb der Fabrik für den Container. Ein Standort kann eine ganze Fabrik, ein Bereich einer Fabrik oder so spezifisch wie ein Regal-Identifikator sein. Standorte werden innerhalb des Fabrik-Objekts in der Modellierung definiert.

Ziel-Arbeitsplan und Schritte. Die Schritte werden angezeigt, wenn Sie den Arbeitsplan erweitern. Die Anwendung verschiebt den Container in den nächsten Schritt im aktuellen Arbeitsplan, es sei denn, Sie geben hier einen anderen Arbeitsplan oder Schritt an.

Optional

Schritt im Arbeitsplan, an den Sie den Container versenden möchten.

Vorgehensweise zum Versenden eines Containers

Gehen Sie folgendermaßen vor, um einen Container zu versenden:

1. Öffnen Sie die Seite Versenden im Menü Container. Die Seite "Versenden" wird angezeigt.
2. Wählen Sie einen Container im Container-Feld aus oder geben Sie ihn ein. Die Anwendung aktualisiert den Container-Kopf und die Auswahlfelder, sodass diese Informationen für den angegebenen Container angezeigt werden. Werte werden nur angezeigt, wenn sie für den Container eingegeben wurden.
Hinweis: Die Anwendung füllt das Feld "Container" und den Container-Kopf automatisch, wenn Sie von einer anderen Fertigungsseite auf das Popup "Versenden" zugreifen.
3. Wählen Sie ein Versandziel für den Container aus. Die Anwendung aktualisiert die optionalen Felder, um alle Informationen anzuzeigen, die für das Versandziel-Objekt in der Modellierung angegeben wurden.
4. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
5. Klicken Sie auf Versenden oder in der Befehlsleiste auf Abschicken. Die Anwendung zeigt eine Statusmeldung an, die bestätigt, dass der Container versandt wurde.
