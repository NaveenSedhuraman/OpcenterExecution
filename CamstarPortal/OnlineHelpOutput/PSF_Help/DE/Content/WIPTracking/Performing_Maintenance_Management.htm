Durchführen von Wartungsverwaltung

Die Transaktion "Wartungsverwaltung" bietet Suchfilter-Funktionalität, die es Ihnen ermöglicht, Suchkriterien für Ressourcen mit Wartungsanforderungen anzugeben, die bereit zur Beendigung sind.

Die Tabelle "Suchergebnisse" zeigt die Ergebnisse Ihrer Ressourcen-Suche an. Neben den erwarteten Suchergebnissen gibt die Suchfunktion auch:

• Alle Durchsatzanforderungen aus, wenn Sie "Verbleibende Tage"-Filterkriterien für ein Datums- oder eine Wiederholungsterminanforderung angeben.
• Alle Datums- und Wiederholungsterminanforderungen aus, wenn Sie für eine Durchsatzanforderung die Filterkriterien "Verbleibende Menge in Prozent", "Verbleibende Menge2 in Prozent", "Genutzte Menge", "Genutzte Menge2" angeben.

Wenn Sie eine Zeile in der Tabelle "Suchergebnisse" auswählen, werden die entsprechenden Felder im Bereich "Wartungsdetails" automatisch gefüllt. Alle "Wartungsdetails"-Felder sind schreibgeschützt, abgesehen von den Kommentar-Feldern.

Das Ressourcenprotokoll enthält die komplette Transaktionshistorie für eine Ressource. Weitere Informationen finden Sie unter "Anzeigen des Ressourcenprotokolls".

Anzeigen zugeordneter Dokumente

Das Symbol "Dokumente anzeigen" wird nur auf der Seite angezeigt, wenn der ausgewählten Ressource Dokumente zugeordnet sind. Durch Klicken auf dieses Symbol wird das Popup "Dokumente anzeigen" angezeigt, das Links zu den zugeordneten Dokumenten auflistet. Jeder Link enthält eine Dokumentbeschreibung, Revision und die Angabe, ob es sich bei der Revision um die aktuelle Standardrevision handelt.

Das Symbol "Dokumente anzeigen" ist deaktiviert, wenn keine Dokumente zugeordnet sind.

Daten sammeln

Datensammlung-Felder werden am Ende der Seite "Wartungsverwaltung" angezeigt, wenn eine Datensammlungsdefinition (benutzerdefiniert oder parametrisch) an der Wartungsanforderung definiert wurde. Scrollen Sie bis zum Ende der Seite, um Datensammlung-Felder anzuzeigen.

Vorgehensweise zum Durchführen einer erforderlichen Wartung für eine Ressource

Gehen Sie folgendermaßen vor, um die erforderliche Wartung für eine Ressource durchzuführen:

1. Öffnen Sie die Seite Wartungsverwaltung im Menü Ressource. Die Seite "Wartungsverwaltung" wird angezeigt.
2. Geben Sie Werte in die Felder Suchkriterien eingeben ein, um die Anzeige von Ressourcen zu beschränken.
3. Klicken Sie auf Suchen. Die Tabelle "Suchergebnisse" zeigt die Wartungsstatus-Informationen der Ressourcen an, die den eingegebenen Suchkriterien entsprechen. Falls keine Suchkriterien eingegeben werden, werden alle möglichen Ergebnisse angezeigt.
4. Klicken Sie auf die Zeile für die Ressource, die eine Wartungsanforderung hat, die entweder in der Warnungsfrist ("Ausstehend"), "Fällig" oder "Überfällig" ist. Details der Wartungsanforderung werden in den schreibgeschützten "Wartungsdetails"-Feldern angezeigt.
5. Klicken Sie auf das Symbol "Dokumente anzeigen", falls dieses aktiviert ist, und falls Sie eine Liste der Dokumente anzeigen möchten, die der ausgewählten Ressource zugeordnet sind. Das Popup "Dokument anzeigen" wird angezeigt.
6. Klicken Sie auf den zugehörigen Link, um ein Dokument anzuzeigen. Wenn Sie fertig sind, klicken Sie auf Schließen, um das Popup "Dokumente anzeigen" zu schließen.
7. Klicken Sie auf das Steuerelement "Zusätzliche Felder anzeigen", wenn Sie Kommentare speziell für diese Wartungsanforderung eingeben möchten.
8. Klicken Sie auf Beendet.
