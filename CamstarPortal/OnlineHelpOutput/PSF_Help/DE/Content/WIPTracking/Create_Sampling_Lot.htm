Stichprobenlos erstellen

Sie können die Seite "Stichprobenlos erstellen" verwenden, um ein Stichprobenlos zu erstellen und Sampling basierend auf einer Gruppe von Containern anstatt auf einzelnen Containern zu erlauben. Die Menge des Los wird verwendet, um die Anzahl der notwendigen Stichproben festzulegen. Die Container-Stichprobendaten werden für das Stichprobenlos verwendet, um festzulegen, ob das Los die Überprüfung bestanden hat. Los-basierte Stichproben sind nützlich, um festzulegen, ob ein kompletter Produktionsauftrag die Überprüfung besteht. Es ist auch bei Eingangskontrollen nützlich, wenn die Stichproben mehrere Container umfassen.

Beim Verwenden der Seite "Stichprobenlos erstellen"

Wenn eine elektronische Unterschrift in der Modellierung für die Transaktion "Stichprobenlos erstellen" als erforderlich definiert wurde, wird bei der Ausführung der Transaktion für die Instanz ein "Elektr. Unterschrift"-Popup angezeigt. Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch und im Opcenter Execution Core Modeling-Benutzerhandbuch.

Seite "Stichprobenlos erstellen"

Sie können über das Menü "Container" auf die Seite "Erstellen von Stichprobenlosen" zugreifen.

Felddefinitionen der Seite "Stichprobenlos erstellen"

Diese Tabelle definiert die Felder der Seite "Stichprobenlos erstellen".

Feld

Definition

Typ

Losname

Name des zu erstellenden Stichprobenloses.

Erforderlich

Menge

Menge des Stichprobenloses.

Erforderlich

Maßeinheit

Maßeinheit für die Menge.

Optional

Produkt

Das Produkt, das durch das Stichprobenlos dargestellt wird.

Erforderlich

Stichprobenrate

Wenn dieser Wert eingetragen wird, werden alle Stichprobenraten des Stichprobenplans des Produkts übergangen.

Optional

Anpassung der Stichprobengröße, wenn die Losmenge überschritten wird

Ein Kontrollkästchen, dass anzeigt, das die für das Los benötigte Stichprobengröße an die Gesamtmenge aller in diesem Los gestarteten Container angepasst wird.

Optional

Zusätzliche Felder anzeigen/ausblenden

Optionale Felder sind standardmäßig ausgeblendet, wenn die Anwendung eine Seite oder ein Popup anzeigt. Durch Klicken auf die Steuerung "Anzeigen" werden die Felder angezeigt. Durch Klicken auf die Steuerung "Verstecken" werden die Felder ausgeblendet.

  

Kommentare

Zusätzliche Informationen oder Anweisungen für diese Transaktion. Das Feld akzeptiert maximal 255 Zeichen. Wenn die Kommentare 255 Zeichen überschreiten, verkürzt die Anwendung die Daten.

Optional

Vorgehensweise zum Erstellen eines Stichprobenloses

Gehen Sie folgendermaßen vor, um ein Stichprobenlos zu erstellen:

1. Öffnen Sie auf Stichprobenlos erstellen im Menü Container. Die Seite "Stichprobenlos erstellen" wird angezeigt.
2. Geben Sie einen Namen im Losname-Feld ein.
3. Geben Sie einen Wert im Mengen-Feld ein.
4. Wählen Sie ein Produkt im Produkt-Feld aus.
5. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
6. Klicken Sie auf Los erstellen oder in der Befehlsleiste auf Abschicken. Eine Statusmeldung wird angezeigt, die besagt, dass das Stichprobenlos erstellt wurde.
