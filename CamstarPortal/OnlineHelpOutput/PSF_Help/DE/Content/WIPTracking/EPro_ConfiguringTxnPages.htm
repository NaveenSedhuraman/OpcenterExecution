Konfigurieren von Fertigungsseiten für automatische Abläufe

Siemens bietet Ihnen die Möglichkeit, Fertigungstransaktionen in automatischen Abläufen auszuführen. Fertigungstransaktionsseiten müssen zur Verwendung in automatischen Abläufen konfiguriert werden. Siemens bietet bereits einige Transaktionsseiten für automatische Abläufe. Ein Systemadministrator muss alle weiteren Seiten mit Hilfe des Portal-Studios konfigurieren.

Folgende Fertigungstransaktionsseiten sind standardmäßig für automatische Abläufe konfiguriert:

• Verbinden
• Dokument anhängen*
• Mengenänderung (Einfach und mehrere Gründe)
• Schließen*
• Container vereinigen
• Menge vereinigen
• Komponentendefekt
• Komponentenverbrauch (Einfach und Erweitert)
• Komponente entfernen
• Komponente ersetzen
• Containerattributwartung
• Containerdefekt
• Container anhalten
• Containerwartung
• Container freigeben
• Containerdurchsatz
• Datensammlung
• Abtrennen
• Abmelden
• Anmelden
• Abmelden - nicht standardmäßig
• Nacharbeit
• Versenden
• Transferieren

* Die Prozesstimer-Funktionalität ist für diese Transaktionen nicht verfügbar.

Beim Konfigurieren von Fertigungsseiten

Sie müssen folgende Aufgaben durchführen, wenn Sie Fertigungsseiten konfigurieren:

• Fügen Sie das "EProcHiddenFields_WP"-Webpart zur Seite hinzu

• EProcHiddenTaskDM
• EProcHiddenTaskContainerDM
• EProcHiddenTaskList DM

Vorgehensweise zum Konfigurieren einer Fertigungstransaktionsseite für automatische Abläufe

Gehen Sie wie folgendermaßen vor, um eine Fertigungsseite für automatische Abläufe zu konfigurieren:

1. Öffnen Sie Portal Studio.
2. Wählen Sie die Seite, die Sie konfigurieren möchten in der Komponentenleiste.
3. Doppelklicken Sie auf die Seite. Die Seite wird angezeigt.
4. Ziehen Sie EProcHiddenFields_WP aus der Webparts-Leiste auf die Seite. Die Anwendung fragt Sie, ob Sie das Webpart als lokale Kopie oder als Referenz auf den Inhalt hinzufügen wollen.
5. Wahlen Sie auf Kopieren. Das Webpart wird auf der Seite angezeigt.
6. Klicken Sie auf Speichern.
7. Fügen Sie diese drei Data-Contracts der Seite hinzu:
• EProcHiddenTaskDM
• EProcHiddenTaskContainerDM
• EProcHiddenTaskListDM
8. Klicken Sie auf OK um das "Data-Contract"-Fenster zu schließen.
9. Klicken Sie auf Speichern. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
10. Wiederholen Sie Schritte 2-9 für jede Seite, die Sie konfigurieren möchten.
Hinweis: Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics Portal Studio-Benutzerhandbuch oder im Opcenter Execution Core Portal Studio-Benutzerhandbuch.
