Container freigeben

Mit der Transaktion "Freigeben" können Sie einen angehaltenen Container freigeben, so dass dieser normal weiterbearbeitet werden kann. Sie können an den freigegebenen Containern Transaktionen durchführen.

Hinweis: Informationen über das gleichzeitige Freigeben mehrerer Container finden Sie unter "Mehrere Container freigeben".

Beim Freigeben eines Containers

Alle Anzeigefelder auf der Freigeben-Seite sind leer, bis ein Container ausgewählt wird. Anzeigefelder werden im Popup "Freigeben" automatisch gefüllt. Informationen über das Ausführen einer Transaktion über eine Seite oder ein Popup finden Sie unter "Verwenden von Transaktionsseiten und Popups".

Es kann nur ein Container pro Transaktion freigegeben werden. Wenn Sie jedoch einen Kind-Container freigeben, wird auch dessen Eltern-Container zur Bearbeitung freigegeben, wenn dieser keinen anderen Kind-Container hat, der auf Angehalten gesetzt wurde.

Sie können als Teil der Freigabe einen Freigabegrund angeben. Freigabegründe werden in der Modellierung definiert. Weitere Informationen zu Freigabegründen finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch und im Opcenter Execution Core Modeling-Benutzerhandbuch.

Seite "Freigeben"

Sie können über das Menü "Container" auf die Seite "Freigeben" zugreifen. Die Freigeben-Seite ermöglicht es Ihnen, einen angehaltenen Container einzugeben oder auszuwählen und diesen freizugeben.

Hinweis: Die Seite "Freigeben" kann auch als Popup verfügbar sein, wenn die Container-Freigabe-Transaktion auf einer anderen Seite (z. B. "Containersuche") als verfügbare Aktion konfiguriert wurde.

Diese Tabelle definiert die Felder auf der Seite und im Popup "Freigeben".

Zusätzliche Felder anzeigen/ausblenden

Optionale Felder sind standardmäßig ausgeblendet, wenn die Anwendung eine Seite oder ein Popup anzeigt. Durch Klicken auf die Steuerung "Anzeigen" werden die Felder angezeigt. Durch Klicken auf die Steuerung "Verstecken" werden die Felder ausgeblendet.

  

Kommentare

Zusätzliche Informationen oder Anweisungen für diese Transaktion oder für diesen Container. Das Feld akzeptiert maximal 255 Zeichen. Wenn die Kommentare 255 Zeichen überschreiten, verkürzt die Anwendung die Daten.

Optional

Vorgehensweise zum Freigeben eines Containers

Gehen Sie folgendermaßen vor, um einen Container freizugeben:

1. Sie können über das Container-Menü auf die Seite Freigeben zugreifen. Die Seite "Freigeben" wird angezeigt.
2. Wählen Sie einen Container im Container-Feld aus oder geben Sie ihn ein. Die Anwendung aktualisiert den Container-Kopf und die Auswahlfelder, sodass diese Informationen für den angegebenen Container angezeigt werden. Werte werden nur angezeigt, wenn sie für den Container eingegeben wurden.
Hinweis: Die Anwendung füllt das Feld "Container" und den Container-Kopf automatisch, wenn Sie von einer anderen Fertigungsseite auf das Popup "Freigeben" zugreifen.
3. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
4. Klicken Sie auf Freigeben oder in der Befehlsleiste auf Abschicken. Die Anwendung zeigt eine Statusmeldung an, die bestätigt, dass der Container freigegeben wurde.
