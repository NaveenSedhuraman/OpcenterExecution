Aktivieren der automatischen Auffüllung für Materialien

Sie können die automatische Auffüllung von Produkten aktivieren, die mit Materialwarteschlangen verknüpft sind, um Materialien aufzufüllen, wenn die Materialmenge auf ein benutzerdefiniertes Niveau sinkt. An das entsprechende Bestandsverwaltungssystem wird eine Auffüllanforderung gesendet.

Beim Aktivieren der automatischen Auffüllung für Materialien in der Materialwarteschlange

Die automatische Auffüllung kann für Materialien aktiviert werden, die in eine ausgewählte Materialwarteschlange geladen werden. Produkte, die für die Auffüllung konfiguriert werden können, werden in der Tabelle "Materialwarteschlange: Auffülldetails" auf den Seiten "Materialwarteschlange verwalten" oder "Materialwarteschlange verwalten - Einfach" angezeigt, nachdem Sie eine Materialwarteschlange ausgewählt haben.

Die automatische Auffüllung ist für Produkte, für die eine manuelle Auffüllung in der Tabelle festgelegt wurde, nicht aktiviert.

Vorgehensweise zum Aktivieren der automatischen Auffüllung für Materialien

1. Öffnen Sie die Seite Materialwarteschlange verwalten - Einfach. Die Seite "Materialwarteschlange verwalten - Einfach" wird angezeigt.
2. Wählen Sie eine Materialwarteschlange in der Liste aus. Die Anwendung trägt die entsprechende Ressource in das Feld "Ressource" ein (sofern zutreffend). Die Anwendung aktiviert das Kontrollkästchen "Ist aktiv", wenn die Materialwarteschlange aktiv ist. Die Anwendung füllt die Tabelle "Materialwarteschlange" mit Materialien, die derzeit in der Materialwarteschlange geladen sind. Die Anwendung aktiviert die Schaltflächen "Laden" und "Zurücksetzen".
3. Wählen Sie ein Produkt in der Tabelle Tabelle Materialwarteschlange: Auffülldetails aus.
4. Geben Sie einen Wert im Feld Grenzwertmenge ein.
5. Wählen Sie einen Wert im Feld Grenzwertmaßeinheit aus.
6. Geben Sie einen Wert im Feld Menge auffüllen ein.
7. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie unter Felddefinitionen der Seiten "Materialwarteschlange verwalten" und "Materialwarteschlange verwalten - Einfach".
8. Klicken Sie auf Abschicken. Die Anwendung speichert Ihre Änderungen und zeigt eine Erfolgsmeldung an.
