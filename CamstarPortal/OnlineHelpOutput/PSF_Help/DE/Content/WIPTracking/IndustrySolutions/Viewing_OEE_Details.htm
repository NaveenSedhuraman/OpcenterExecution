Anzeigen von GAE-Details

Mithilfe der Seite "GAE-Details" können Sie Diagramme für detaillierte GAE-Metriken entweder für eine ausgewählte Ressource oder für alle Ressourcen in einer angegebenen Ressourcenfamilie anzeigen. Diese Metriken umfassen Folgendes:

• Qualität
• Verfügbarkeit
• Leistung
• Hauptgründe für Ausfallzeiten
• Hauptgründe für Ausschuss/Nacharbeit

Felddefinitionen der Seite "GAE-Dtails"

Diese Tabelle definiert die Felder auf der Seite "GAE-Details".

Feld

Definition

Typ

Ressourcenfamilie

Liste der verfügbaren Ressourcenfamilien.

Optional

Ressource (Ressource)

Liste der Ressourcen, die mit der ausgewählten Ressourcenfamilie verknüpft sind.

Optional

Hauptgründe für Ausfallzeiten

Anzahl der anzuzeigenden Ausfallzeitengründe.

Optional

Hauptgründe für Ausschuss/Nacharbeit

Anzahl der anzuzeigenden Gründe für Ausschuss/Nacharbeit.

Optional

TimeSpan

Zeitraum von berechneten GAE-Daten. Operanden schließen ein:

• Schicht – Zeigt GAE-Daten an, die für die aktuelle Schicht berechnet sind.
• Tag – Zeigt GAE-Daten an, deren Berechnung mit der ersten Schicht des Tages beginnt, wie für die Ressource, die Fabrik oder den Produktionskalender definiert.

Optional

Benutzerdefiniert

Kontrollkästchen, das angibt, ob die Anwendung GAE-Daten zwischen den Zeiten anzeigt, die in den Feldern "Startzeit" und "Endzeit" angegeben sind.

Optional

Startzeit

Angegebene Startzeit für die Berechnung der GAE-Daten.

Optional

Endzeit

Angegebene Endzeit für die Berechnung der GAE-Daten.

Optional

Vorgehensweise beim Anzeigen von GAE-Details für eine Ressourcenfamilie

Gehen Sie folgendermaßen vor, um GAE-Details für eine Ressourcenfamilie anzuzeigen:

1. Öffnen Sie die Seite GAE-Details über das Menü Ressource. Die Seite "GAE" wird angezeigt.
2. Wählen Sie in der Liste eine Ressourcenfamilie aus.
3. Geben Sie einen Wert in das Feld Hauptgründe für Ausfallzeiten ein.
4. Geben Sie einen Wert in das Feld Hauptgründe für Ausschuss/Nacharbeit ein.
5. Möchten Sie die Daten basierend auf einer Zeitspanne oder auf benutzerdefinierten Parametern anzeigen?

Antwort . . .

Dann . . .

Zeitspanne
a. Wählen Sie in der Liste eine Zeitspanne aus.
b. Gehen Sie zu Schritt 6.

Benutzerdefinierte Parameter
a. Aktivieren Sie das Kontrollkästchen Benutzerdefiniert.
b. Wählen Sie im Feld Startzeit eine Startzeit aus.
c. Wählen Sie im Feld Endzeit eine Endzeit aus.
d. Gehen Sie zu Schritt 6.
6. Klicken Sie auf Daten laden. Die Anwendung zeigt Diagramme mit GAE-Details für alle Ressourcen an, die mit der ausgewählten Ressourcenfamilie verknüpft sind.

Vorgehensweise beim Anzeigen von GAE-Details für eine Ressource

Gehen Sie folgendermaßen vor, um GAE-Details für eine Ressource anzuzeigen:

1. Öffnen Sie die Seite GAE-Details über das Menü Ressource. Die Seite "GAE-Details" wird angezeigt.
2. Wählen Sie in der Liste eine Ressourcenfamilie aus. Die Anwendung füllt das Feld "Ressource" mit einer Liste von Ressourcen, die mit der ausgewählten Ressourcenfamilie verknüpft sind.
3. Wählen Sie eine Ressource aus der Liste aus.
4. Geben Sie einen Wert in das Feld Hauptgründe für Ausfallzeiten ein.
5. Geben Sie einen Wert in das Feld Hauptgründe für Ausschuss/Nacharbeit ein.
6. Möchten Sie die Daten basierend auf einer Zeitspanne oder auf benutzerdefinierten Parametern anzeigen?

Antwort . . .

Dann . . .

Zeitspanne
a. Wählen Sie in der Liste eine Zeitspanne aus.
b. Gehen Sie zu Schritt 7.

Benutzerdefinierte Parameter
a. Aktivieren Sie das Kontrollkästchen Benutzerdefiniert.
b. Wählen Sie im Feld Startzeit eine Startzeit aus.
c. Wählen Sie im Feld Endzeit eine Endzeit aus.
d. Gehen Sie zu Schritt 7.
7. Klicken Sie auf Daten laden. Die Anwendung zeigt Diagramme mit GAE-Details für die Ressource an.
