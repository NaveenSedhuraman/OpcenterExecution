Anzeigen von GAE-Daten

Mithilfe der Seite "GAE" können Sie GAE-Daten entweder für eine ausgewählte Ressource oder für alle Ressourcen in einer angegebenen Ressourcenfamilie anzeigen. Durch Klicken auf die Schaltfläche "Daten laden" werden GAE-Daten für die ausgewählte Ressourcenfamilie oder Ressource angezeigt.

Felddefinitionen der Seite "GAE"

Diese Tabelle definiert die Felder auf der Seite "GAE".

Feld

Definition

Typ

Ressourcenfamilie

Liste der verfügbaren Ressourcenfamilien.

Optional

Ressource

Liste der Ressourcen, die mit der ausgewählten Ressourcenfamilie verknüpft ist.

Optional

Zeitspanne

Zeitspanne der berechneten GAE-Daten. Operanden schließen ein:

• Schicht – zeigt die für die aktuelle Schicht berechneten GAE-Daten an.
• Tag – zeigt GAE-Daten an, deren Berechnung mit der ersten Schicht des Tages beginnen, wie für die Ressource, die Fabrik oder den Produktionskalender definiert.

Optional

Benutzerdefiniert

Kontrollkästchen, das angibt, ob die Anwendung GAE-Daten zwischen den Zeiten anzeigt, die in den Feldern "Startzeit" und "Endzeit" angegeben sind.

Optional

Startzeit

Angegebene Startzeit für die Berechnung der GAE-Daten.

Optional

Endzeit

Angegebene Endzeit für die Berechnung der GAE-Daten.

Optional

Vorgehensweise zum Anzeigen von GAE-Daten für eine Ressourcenfamilie

Gehen Sie folgendermaßen vor, um GAE-Daten für eine Ressourcenfamilie anzuzeigen:

1. Öffnen Sie die Seite GAE im Menü Ressource. Die Seite "GAE" wird angezeigt.
2. Wählen Sie in der Liste eine Ressourcenfamilie aus.
3. Möchten Sie die Daten basierend auf einer Zeitspanne oder auf benutzerdefinierten Parametern anzeigen?

Antwort . . .

Dann . . .

Zeitspanne
a. Wählen Sie in der Liste eine Zeitspanne aus.
b. Gehen Sie zu Schritt 4.

Benutzerdefinierte Parameter
a. Wählen Sie das Kontrollkästchen Benutzerdefiniert aus.
b. Geben Sie im Feld Startzeit die Startzeit ein.
c. Wählen Sie im Feld Endzeit die Endzeit aus.
d. Gehen Sie zu Schritt 4.
4. Klicken Sie auf Daten laden. Die Anwendung zeigt ein horizontales Balkendiagramm mit GAE-Daten für alle Ressourcen an, die mit der ausgewählten Ressourcenfamilie verknüpft sind.

Vorgehensweise zum Anzeigen von GAE-Daten für eine Ressource

Gehen Sie folgendermaßen vor, um GAE-Daten für eine Ressource anzuzeigen:

1. Öffnen Sie die Seite GAE im Menü Ressource. Die Seite "GAE" wird angezeigt.
2. Wählen Sie in der Liste eine Ressourcenfamilie aus. Die Anwendung füllt das Feld "Ressource" mit einer Liste von Ressourcen, die mit der ausgewählten Ressourcenfamilie verknüpft sind.
3. Wählen Sie eine Ressource aus der Liste aus.
4. Möchten Sie die Daten basierend auf einer Zeitspanne oder auf benutzerdefinierten Parametern anzeigen?

Antwort . . .

Dann . . .

Zeitspanne
a. Wählen Sie in der Liste eine Zeitspanne aus.
b. Gehen Sie zu Schritt 5.

Benutzerdefinierte Parameter
a. Wählen Sie das Kontrollkästchen Benutzerdefiniert aus.
b. Geben Sie im Feld Startzeit die Startzeit ein.
c. Wählen Sie im Feld Endzeit die Endzeit aus.
d. Gehen Sie zu Schritt 5.
5. Klicken Sie auf Daten laden. Die Anwendung zeigt ein horizontales Balkendiagramm mit GAE-Daten für die Ressource an.
