Materialbestätigung

Auf der Seite "Materialanforderungsbestätigung" können Sie Bestätigungen und Stornierungen offener Materialauffüllanforderungen senden. Bestätigungen geben an, dass Material empfangen wurde. Stornierungen geben an, dass Materialanforderungen abgebrochen wurden.

Beim Durchführen einer Materialbestätigung

Sie müssen einen Standardlagerort definieren, bevor Sie eine Materialbestätigung durchführen. Weitere Informationen finden Sie im Abschnitt zum Zuweisen von Standardlagerorten.

Die Anwendung aktualisiert die Materialmengen in der zugehörigen Materialwarteschlange, wenn Sie bestätigen, dass das Material empfangen wurde. Die Zeile wird aus der Tabelle "Materialanforderungs-Transaktionsservicedetail" entfernt, wenn die verbleibende Menge Null erreicht.

Vorgehensweise zum Bestätigen einer Materialanforderung

Führen Sie die folgenden Schritte aus, um eine Materialanforderung zu bestätigen:

1. Öffnen Sie die Seite Material bestätigen.
2. Wählen Sie eine Materialwarteschlange aus oder übernehmen Sie die Standardauswahl. Die Anwendung füllt die Tabelle basierend auf Ihrer Auswahl aus, wenn Sie die Standardauswahl geändert haben.
3. Wählen Sie die Tabellenzeile aus, in der der zu bestätigende Bestand aufgeführt ist.
4. Geben Sie einen Wert in das Feld Empfangene Menge ein. Die Anwendung füllt das Feld "Verbleibende Menge" mit dem Unterschied zwischen "Angeforderte Menge" und "Empfangene Menge".
5. Führen Sie einen der folgenden Schritte durch:

Wenn das empfangene Material . . .

Dann . . .

Weist einen Containeridentifikator auf
a. Scannen Sie den Identifikator in das Feld Materialcontainer/Los bzw. geben Sie ihn ein.
b. Gehen Sie zu Schritt 6.

Weist keinen Containeridentifikator auf
a. Aktivieren Sie das Kontrollkästchen Akzeptieren (Kein Container).
b. Gehen Sie zu Schritt 6.
6. Wiederholen Sie die Schritte 3-5 für jede zusätzliche Materialanforderung in der Tabelle, die Sie bestätigen müssen.
7. Klicken Sie auf Abschicken. An das Bestandsverwaltungssystem wird eine Nachricht mit der Bestätigungen bzw. den Bestätigungen gesendet. Die empfangene Materialmenge wird in die entsprechende Materialwarteschlange geladen. Das Feld "Verbleibende Menge" wird mit der Differenz zwischen der angeforderten Menge und der empfangenen Menge aktualisiert.

Vorgehensweise zum Abbrechen einer Materialanforderung

Gehen Sie folgendermaßen vor, um eine Materialanforderung abzubrechen:

1. Öffnen Sie die Seite Material bestätigen.
2. Wählen Sie eine Materialwarteschlange aus oder übernehmen Sie die Standardauswahl. Die Anwendung füllt die Tabelle basierend auf Ihrer Auswahl aus, wenn Sie die Standardauswahl geändert haben.
3. Wählen Sie die Tabellenzeile aus, in der die abzubrechende Materialanforderung aufgeführt ist.
4. Aktivieren Sie das Kontrollkästchen Abbrechen.
5. Wiederholen Sie die Schritte 3-4 für jede zusätzliche Materialanforderung in der Tabelle, die Sie abbrechen müssen.
6. Klicken Sie auf Abschicken. An das Bestandsverwaltungssystem wird eine Nachricht mit der Stornierung bzw. den Stornierungen gesendet.
