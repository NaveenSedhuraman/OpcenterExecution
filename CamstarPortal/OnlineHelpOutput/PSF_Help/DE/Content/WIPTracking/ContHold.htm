Container anhalten

Die Transaktion "Anhalten" ermöglicht es Ihnen, einen Container wegen eines bestimmten Grundes auf Angehalten zu setzen (z. B. für eine geplante Überprüfung oder ein Protokoll). Wenn Sie einen Container auf Angehalten setzen, verhindern Sie damit, dass Benutzer Transaktionen an dem Container durchführen können. Der Container kann wieder für den normalen Betrieb freigegeben werden, sobald die Gründe für den Halt beseitigt wurden.

Hinweis: Informationen über das gleichzeitige Anhalten mehrerer Container finden Sie unter "Mehrere Container anhalten".

Beim Setzen eines Containers auf Angehalten

Alle Anzeigefelder sind auf der Seite "Anhalten" leer, bis ein Container ausgewählt wird. Anzeigefelder werden im Anhalten-Popup automatisch gefüllt. Informationen über das Ausführen einer Transaktion über eine Seite oder ein Popup finden Sie unter "Verwenden von Transaktionsseiten und Popups".

Es kann nur ein Container pro Transaktion auf Angehalten gesetzt werden. Sie müssen einen Anhaltegrund auswählen, um einen Container auf Angehalten zu setzen. Anhaltegründe werden in der Modellierung von einem Systemadministrator festgelegt. Die Anwendung zeigt eine Fehlermeldung an, wenn Sie auf die Schaltfläche "Anhalten" klicken, ohne vorher einen Anhaltegrund auszuwählen. Informationen zum Definieren von Anhaltegründen finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch oder im Opcenter Execution Core Modeling-Benutzerhandbuch.

Die Anwendung überprüft, ob der angegebene Container aktiv ist und nicht bereits auf Angehalten gesetzt wurde. Sie können einen geschlossenen Container nicht auf Angehalten setzen.

Wenn Sie einen Container auf Angehalten setzen, werden dessen Kind-Container nicht auf Angehalten gesetzt. Wenn Sie jedoch einen Kind-Container auf Angehalten setzen, dann wird damit auch der Eltern-Container auf Angehalten gesetzt. Sie können die Bearbeitung des Eltern-Containers nur dann wieder fortsetzen, wenn Sie den angehaltenen Kind-Container wieder freigeben.

Seite "Anhalten"

Sie können über das Menü "Container" auf die "Anhalten"-Seite zugreifen. Die "Anhalten"-Seite ermöglicht es Ihnen, einen Container zu scannen oder einzutragen und diesen dann auf Angehalten zu setzen.

Hinweis: Die Seite "Anhalten" ist eventuell auch als Popup verfügbar, wenn die Transaktion "Container anhalten" als verfügbare Aktion auf einer anderen Seite konfiguriert wurde (z. B. "Containersuche").

Vorgehensweise zum Setzen eines Containers auf Angehalten

Gehen Sie folgendermaßen vor, um einen Container auf Angehalten zu setzen:

1. Öffnen Sie im Menü im Container die Seite Anhalten. Die Seite "Anhalten" wird angezeigt.
2. Wählen Sie einen Container im Container-Feld aus oder geben Sie ihn ein. Die Anwendung aktualisiert den Container-Kopf und die Auswahlfelder, sodass diese Informationen für den angegebenen Container angezeigt werden.Werte werden nur angezeigt, wenn sie für den Container eingegeben wurden.
Hinweis: Die Anwendung füllt das Feld "Container" und den Container-Kopf automatisch, wenn Sie von einer anderen Fertigungsseite auf das Popup "Anhalten" zugreifen.
3. Klicken Sie auf einen Haltegrund.
4. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
5. Klicken Sie auf Anhalten oder in der Befehlsleiste auf Abschicken, um den Container auf "Angehalten" zu setzen. Die Anwendung zeigt eine Statusmeldung an, die bestätigt, dass der Container angehalten wurde.
