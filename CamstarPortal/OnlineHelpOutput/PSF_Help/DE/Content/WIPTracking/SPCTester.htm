SPC-Tester

Siemens stellt einen SPC-Tester bereit, mit dem Sie die in der Anwendung enthaltenen Muster-Charts und -daten testen können. Er erlaubt Ihnen außerdem, Ihre Einstellungen zu prüfen, nachdem Sie Charts und erforderliche Komponenten konfiguriert haben.

Siemens bietet Muster-Charts und Daten, die über das Management Studio geladen werden können.

Beim Verwenden der Seite "SPC-Tester"

Sie müssen entweder ein SPC-Chart, eine SPC-Chart-Gruppe oder eine Datensammlung auswählen. Alle anderen Felder sind Optional.

Testdaten können über das DB-Update im Management Studio geladen werden. Siemens empfiehlt, dass Sie diese Daten nur in einer Entwicklungsumgebung laden und nicht in der Produktionsumgebung.

Muster-Makros und Muster-Charts verweisen auf die aus dem DB-Update geladenen Daten. Pfade in den Muster-Charts verweisen auf standardmäßige Installationsverzeichnisse. Sie müssen die Pfade aktualisieren, wenn Sie nicht die standardmäßigen Installationsverzeichnisse verwendet haben.

Auch müssen Sie vor der Verwendung der Tester-Seite die Muster-Makros mit Authentifizierungs-Infos für Ihr ODBC aktualisieren.

UI-Parameter, die Sie auf dieser Seite eingeben, überschreiben die im "SPC-Chart"-Objekt festgelegten.

Versuche OCAPs an außer-Kontrolle-Charts

Mit diesem Ankreuzfeld kann das Chart die Out-of-Control-Aktionen auslösen, wenn eine Verletzung auftritt. Wenn es nicht mit einem Häkchen versehen ist, werden die Out-des-Control-Aktionen nicht ausgelöst, auch wenn eine Verletzung auftritt.

Liste von Containern, die angehalten werden können, wenn eine Verletzung auftritt.

Optional

Ressource im Zusammenhang mit einer Verletzung, die aus dem Verkehr gezogen oder auf einen anderen Status geändert wird.

Optional

Wenn Sie dieses Ankreuzfeld mit einem Häkchen versehen, werden die Charts automatisch gelöscht, nachdem Sie sie angesehen haben, anstatt dass eine manuelle Löschung zu einem anderen Zeitpunkt erforderlich wäre.

Optional

UI-Parameter

Es können Zeilen hinzugefügt werden, um Parameter an das Makro weiterzugeben, das Chart erzeugt.

Optional

Vorgehensweise zum Verwenden der Seite "SPC-Tester"

Gehen Sie folgendermaßen vor, um die Seite "SPC-Tester" verwenden:

1. Wählen Sie SPC-Tester im Menü SPC. Die Seite "SPC-Tester" wird angezeigt.
2. Wählen Sie ein SPC-Chart, eine SPC-Chart-Gruppe oder eine Datensammlung. Nur eines ist erforderlich.
3. Wählen Sie mindestens ein optionales Feld aus, um genauer zu definieren, welche Charts generiert werden sollen.
4. Klicken Sie auf Charts erzeugen. Die ausgewählten Charts wird/werden angezeigt.
