Beim Erledigen von Containern

Sie können über das Menü "Container" auf die Seite "Auftragserledigung" zugreifen. Verwenden Sie die Seite "Auftragserledigung", um einen Container zu einem ausgewählten Produktionsauftrag zu starten.

Sie müssen einen eindeutigen Containernamen angeben, wenn Sie einen neuen Container erstellen. Sie können einen Containernamen entweder manuell eingeben oder ihn automatisch von der Anwendung erstellen lassen. Es werden geschlossene Container von der Anwendung einbezogen, wenn ein eindeutiger Name geprüft wird.

Aktivieren Sie das Kontrollkästchen "Namen automatisch erstellen", um den Containernamen unter Verwendung der definierten Nummerierungsregeln von der Anwendung erstellen zu lassen. Wenn Sie dieses Kontrollkästchen aktivieren, löscht und deaktiviert die Anwendung das Feld "Container".

Wenn die Option "Namen automatisch erstellen" ausgewählt wird, ohne vorher eine automatische Nummerierungsregel festzulegen, wird eine Fehlermeldung angezeigt.

Wenn Sie auf die Schaltfläche "Abschicken" klicken, erstellt die Anwendung den Namen des neuen Containers.

Nummerierungsregeln können für mehrere, dem Container zugeordnete Modellierungsobjektinstanzen festgelegt werden. Nummerierungsregeln folgen dieser Standardrangfolge:

1. Containerlevel
2. Produktionsauftrag
3. Produkt
4. Produktfamilie
5. Fabrik

Sie können über die Seite "Auftragserledigung" Dokumente an einen Produktionsauftrag oder an den Container anhängen, der für einen Produktionsauftrag gestartet wird, und diese dann über die Seite "Anhänge verwalten" verwalten. Unter "Anhängen von Dokumenten" finden Sie Informationen zu den Optionen zum Anhängen von Dokumenten, und unter "Anhänge verwalten" finden Sie Informationen zum Verwalten von Dokumenten. Anhänge sind optional. Sie müssen beim Anhängen von Dateien jedoch bestimmte Felder in der Anwendung ausfüllen, bevor Sie den Container starten.

Hinweis: Die Seite "Auftragserledigung" ist eventuell auch als Popup verfügbar, wenn die Transaktion "Auftragserledigung" als verfügbare Aktion auf einer anderen Seite konfiguriert wurde (z. B. "Containersuche").
