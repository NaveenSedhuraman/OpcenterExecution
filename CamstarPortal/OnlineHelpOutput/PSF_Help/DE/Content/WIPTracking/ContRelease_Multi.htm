Mehrere Container freigeben

Die "Mehrere Container freigeben" Transaktion ermöglicht es Ihnen, mehrere angehaltene Container gleichzeitig freizugeben, so dass die normale Bearbeitung fortgesetzt werden kann. Sie können an den freigegebenen Containern Transaktionen durchführen.

Beim Freigeben mehrerer angehaltener Container

Sie müssen einen Freigabegrund auswählen um einen Container freizugeben. Freigabegründe werden in der Modellierung von einem Systemadministrator definiert. Die Anwendung zeigt folgende Fehlermeldung an, wenn Sie auf die Schaltfläche "Ausgewählte freigeben" klicken, ohne vorher einen Grund für die Freigabe auszuwählen. Die Anwendung zeigt den Grund automatisch an, wenn nur ein Freigabegrund definiert wurde. Weitere Informationen zu Freigabegründen finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch und im Opcenter Execution Core Modeling-Benutzerhandbuch.

Die Anwendung überprüft, ob die angegebenen Container aktiv sind und auf Halt stehen.

Informationen über das Ausführen einer Transaktion über eine Seite oder ein Popup finden Sie unter "Verwenden von Transaktionsseiten und Popups".

Seite "Freigeben (mehrere)"

Sie können über das Container-Menü auf die Seite "Freigeben (mehrere)" zugreifen. Sie können auf das Popup "Freigeben (mehrere)" von einer anderen Fertigungsseite aus zugreifen (z. B. "Containersuche"), wenn die Aktion "Freigeben (mehrere)" für diese Seite konfiguriert wurde. Sowohl die Seite "Freigeben (mehrere)" als auch das Popup ermöglichen es Ihnen, einen oder mehrere Container zur Freigabe zu scannen oder einzugeben.

Die Seite und das Popup sind identisch in Aussehen und Funktionalität - mit einer Ausnahme. Die Anwendung füllt das Popup "Freigeben (mehrere)" automatisch mit den Containern, die Sie auf der Fertigungsseite ausgewählt haben, von der aus Sie auf das Popup zugegriffen haben.

Zusätzliche Felder anzeigen/ausblenden

Kommentare

Optional

Vorgehensweise zum Freigeben mehrerer angehaltener Container

Gehen Sie folgendermaßen vor, um mehrere angehaltene Container freizugeben:

1. Öffnen Sie die Seite Freigeben (mehrere) im Menü Container. Die Seite "Freigeben (mehrere)" wird angezeigt.
2. Können Sie die Seite "Freigeben (mehrere)" oder das Popup "Freigeben (mehrere)" sehen?

Wird Folgendes angezeigt . . .

Dann . . .

Seite "Freigeben (mehrere)"

Die Tabelle "Freizugebende Container" ist leer. Gehen Sie zu Schritt 3.

Popup "Freigeben (mehrere)"

Die Tabelle "Freizugebende Container" wird mit den Containern gefüllt, die Sie auf der Seite ausgewählt haben, von der aus Sie das Popup gestartet haben. Wenn Sie weitere Container zur Freigabe hinzufügen möchten, gehen Sie zu Schritt 3. Fahren Sie andernfalls mit Schritt 6 fort.
3. Scannen Sie oder geben Sie einen Container in das Feld Auszuwählenden Container eingeben ein.
4. Drücken Sie die Eingabetaste oder verlassen Sie das Feld durch Drücken der Tab-Taste. Der Container wird zur "Tabelle Freizugebende Container" hinzugefügt und automatisch ausgewählt.
5. Wiederholen Sie Schritte 3-4, um zusätzliche Container hinzuzufügen.
6. Gewährleisten Sie, dass die ausgewählten Container diejenigen sind, die Sie freigeben möchten.
7. Wählen Sie einen Freigabegrund aus.
8. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
9. Klicken Sie auf Ausgewählte freigeben oder in der Befehlsleiste auf Abschicken. Die Anwendung zeigt ein Status-Icon für jeden Container an, das angibt, ob die Transaktion erfolgreich ausgeführt wurde.
10. Schließen Sie das Popup wenn es offen ist, und kehren Sie zur vorherigen Seite zurück.
