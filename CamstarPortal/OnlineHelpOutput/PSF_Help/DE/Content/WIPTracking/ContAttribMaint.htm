Containerattributwartung

Die Seite "Containerattributwartung" erlaubt es Ihnen, benutzerdefinierte, dynamische Attribute für einzelne Container hinzuzufügen und zu verwalten. Benutzerdefinierte Containerattribute sind ein dynamischer Satz an Eigenschaften, die ein Nutzer jederzeit ändern kann. Diese Attribute können für jeden Container spezifisch sein und können dynamisch zur Laufzeit verwaltet werden.

Dank der Möglichkeit, benutzerdefinierte Attribute zuzuordnen, können Sie Informationen über den Container auf leicht zugängliche Weise speichern. Sie könnten beispielsweise Übungsergebnisse auf dem Container selbst speichern, um nicht die Container-Historie durchsuchen zu müssen. Ein Attribut könnte z.B. etwas so Simples wie die Container-Farbe sein.

Hinweis: Verwenden Sie die Seite "Containerwartung", um vordefinierte, statische Container-Eigenschaften (z. B. Container-Level, Besitzer usw.) zu verwalten.

Beim Hinzufügen von Attributen zu einem Container

Die Anwendung kann wenn Sie einen Container starten und während BenutzerDatensammlungen benutzerdefinierte Attribute automatisch zuweisen. Sie können Attribute manuell auf der Seite oder im Popup "Containerattribut Wartung" zuweisen.

In folgenden Dokumenten finden Sie weitere Informationen zum automatischen Zuweisen definierter Attribute:

• Informationen zum Zuweisen von benutzerdefinierten Attributen zu einem Containerstart finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch und im Opcenter Execution Core Modeling-Benutzerhandbuch.
• Im Thema "Automatischer Ablauf" finden Sie Informationen über die Zuweisung von benutzerdefinierten Attributen im Rahmen der Benutzerdatensammlung.

Die Seite oder das Popup "Containerattributwartung" zeigt die Attribute an, die dem ausgewählten Container zu diesem Zeitpunkt zugewiesen sind. Die Attributliste enthält Attribute, die beim Containerstart, als Teil einer BenutzerDatensammlung oder manuell zugewiesen wurden.

Alle Anzeigefelder sind auf der Seite "Containerattribute" leer, bis ein Container ausgewählt wird. Anzeigefelder werden im Popup "Containerattributwartung" automatisch gefüllt. Informationen über das Ausführen einer Transaktion über eine Seite oder ein Popup finden Sie unter "Verwenden von Transaktionsseiten und Popups".

Der Name jedes Attributs muss eindeutig sein, wenn Sie einem Container ein Attribut hinzufügen. Der Attributwert kann statischer Text oder ein Ausdruck sein. Beispielsweise könnte ein Attribut für Farbe statischer Text sein, während ein Attribut für Containervolumen ein Ausdruck sein könnte. Die Anwendung erfordert, dass Sie einen Attributnamen angeben, wenn Sie einen Attributtyp auswählen oder einen Wert eingeben, obwohl Attribute nicht erforderlich sind.

Doppelklicken auf eine beliebigen Punkt einer Zeile aktiviert diese, sodass Sie Attribute aktualisieren können. Durch Klicken auf die Schaltfläche "Ausgewählte Zeile löschen" werden die ausgewählten Attribute aus der Tabelle entfernt. Sie können das Kontrollkästchen in der Kopfzeile aktivieren, um alle angezeigten Attribute automatisch auszuwählen, oder Sie aktivieren die Kontrollkästchen für einzelne Attribute.

Durch Löschen des "Container"-Feld werden sämtliche Attribute aus der Tabelle gelöscht.

Sie können Attribute für geschlossene und angehaltene Container hinzufügen oder aktualisieren.

Sie können einen Kommentar zu einem Container mit oder ohne zusätzliche Attribute hinzufügen.

Seite "Containerattributwartung"

Sie können über das Menü "Container" auf die Seite "Containerattributwartung" zugreifen. Die Seite "Containerattributwartung" ermöglicht Ihnen das Hinzufügen von Attributen zu einem Container oder das Aktualisieren zuvor hinzugefügter Attribute.

Hinweis: Die Seite "Containerattributwartung" ist eventuell auch als Popup verfügbar, wenn die Transaktion "Containerattributwartung" als verfügbare Aktion auf einer anderen Seite konfiguriert wurde (z. B. "Containersuche").

Felddefinitionen der Seite und des Popups "Containerattributwartung"

Diese Tabelle definiert die Felder auf der Seite und im Popup "Containerattributwartung".

Feld

Definition

Typ

Containerinformationen

Kopf mit dem Containernamen und Informationen zu diesem Container. Sie müssen einen Container auf einer Transaktionsseite auswählen oder eingeben. Alle Informationen in einem Transaktionen-Popup sind schreibgeschützt. Weitere Informationen finden Sie unter "Containerinformationen".

Erforderlich/ Nur Anzeige

Liste von Benutzer-Attributen

Tabelle, die Attribute für den Container angezeigt.

[Kontrollkästchen]

Kontrollkästchen, die es Ihnen ermöglichen, die Attribute auszuwählen. Sie können das Kontrollkästchen in der Kopfzeile aktivieren, um alle angezeigten Attribute automatisch auszuwählen, oder Sie aktivieren die Kontrollkästchen für einzelne Attribute.

Optional

Attribut-Name

Name des Attributs (bis zu 255 Zeichen). Jeder Attributname muss eindeutig sein.

Hinweis: Die Anwendung erfordert, dass Sie einen Attributnamen eingeben, wenn Sie im Feld "Typ" einen Typ auswählen oder einen Wert in das Feld "Wert" eingeben.

Optional

Typ

Typ des Attributs (Integer, Dezimal und so weiter).

Optional

Wert

Standartwert für das Attribut. Wenn Sie ein Attribut hinzufügen, geben Sie statischen Text oder einen Ausdruck zur Berechnung des Werts ein. Sie können maximal 2000 Zeichen eingeben.

Hinweis: Wenn der Wert ein Ausdruck ist, empfiehlt Siemens, dass Sie das Kontrollkästchen "Ist Ausdruck" aktivieren, um zu gewährleisten, dass der Wert richtig interpretiert wird.

Optional

Ist Ausdruck

Wenn dieses Kontrollkästchen aktiviert ist, handelt es sich bei dem Wert im Feld "Wert" um einen Ausdruck. Durch Aktivieren dieses Kontrollkästchens wird gewährleistet, dass die Anwendung den Ausdruck richtig interpretiert.

Hinweis: Dieses Kontrollkästchen gilt nur dann, wenn Sie als Typ "Zeichenkette" angeben.

Optional

Zusätzliche Felder anzeigen/ausblenden

Optionale Felder sind standardmäßig ausgeblendet, wenn die Anwendung eine Seite oder ein Popup anzeigt. Durch Klicken auf die Steuerung "Anzeigen" werden die Felder angezeigt. Durch Klicken auf die Steuerung "Verstecken" werden die Felder ausgeblendet.

  

Kommentare

Zusätzliche Informationen oder Anweisungen für diese Transaktion oder für diesen Container. Das Feld akzeptiert maximal 255 Zeichen. Wenn die Kommentare 255 Zeichen überschreiten, verkürzt die Anwendung die Daten.

Optional

Vorgehensweise zum Hinzufügen von Attributen zu einem Container

Gehen Sie folgendermaßen vor, um Attribute zu einem Container hinzuzufügen:

1. Öffnen Sie Containerattribut Wartung im Container-Menü. Die Seite "Containerattributwartung" wird angezeigt.
2. Wählen Sie einen Container im Container-Feld aus oder geben Sie ihn ein. Die Anwendung aktualisiert den Container-Kopf und die Auswahlfelder, sodass diese Informationen für den angegebenen Container angezeigt werden. Werte werden nur angezeigt, wenn sie für den Container eingegeben wurden.
Hinweis: Die Anwendung füllt das Feld "Container" und den Container-Kopf automatisch, wenn Sie von einer anderen Fertigungsseite auf das Popup "Containerattributwartung" zugreifen.
3. Klicken Sie auf die Schaltfläche Neue Zeile hinzufügen in der Tabelle Liste von Benutzer-Attributen. Eine Zeile wird in der Tabelle aktiviert.
4. Geben Sie einen Namen für das Attribut im Feld Attributname ein.
5. Wählen Sie den Attributtyp aus dem Listenfeld Typ aus.
6. Geben Sie einen statischen Wert oder einen Ausdruck in das Feld Wert ein.
7. Aktivieren Sie das Kontrollkästchen Ist Ausdruck, wenn der Wert im Feld Wert ein Ausdruck ist.
8. Wiederholen Sie die Schritte 3 bis 7, um weitere Attribute hinzuzufügen.
9. Klicken Sie auf Container aktualisieren oder in der Befehlsleiste auf Abschicken. Die Anwendung speichert Ihre Änderungen und zeigt eine Meldung an, dass Ihre Updates erfolgreich waren.
