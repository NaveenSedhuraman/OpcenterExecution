Aktivieren einer Ressource

Die Transaktion "Ressourcenaktivierung" ermöglicht Ihnen, eine Wartungsanforderung für eine oder mehrere Ressourcen oder Behälter zu aktivieren, egal ob diese zu einer Wartungsklasse gehören oder nicht.

Beim Aktivieren und Deaktivieren einer Ressource

Der "Ressourcengruppen-Filter" ermöglicht Ihnen, die Anzahl an Ressourcen und Behältern zu begrenzen, die in der "Ressourcen"-Tabelle angezeigt werden.

Ein Häkchen neben einer Ressource in der Tabelle gibt an, dass die angegebene Anforderung für die Ressource aktiv ist. Sie können das Kontrollkästchen deaktivieren, um eine Anforderung für eine Ressource zu deaktivieren.

Die Aktivierungstransaktion schlägt bei Ressourcen fehl, die bereits als Teil einer Wartungsklasse aktiviert wurden. Die Spalte "Klasse aktiviert" in der Ressourcen-Tabelle gibt an, ob die Ressource als Teil einer Wartungsklasse aktiviert wurde. Siehe "Wartungsklassenaktivierung" für weitere Informationen bezüglich der Aktivierung und Deaktivierung von Wartungsklassen.

Vorgehensweise zum Aktivieren einer Wartungsanforderung für Ressourcen oder Behälter

Befolgen Sie diese Schritte, um eine Wartungsanforderung für Ressourcen oder Behälter zu aktivieren:

1. Öffnen Sie die Seite Ressourcenaktivierung im Menü Ressource. Die Ressource-Aktivierungsseite wird angezeigt.
2. Wählen Sie eine Wartungsanforderung aus dem Feld Wartungsanforderung aus. Die Ressourcen-Tabelle zeigt alle definierten Ressourcen und Behälter an.
3. Wählen Sie eine Ressourcengruppe aus dem Feld Ressourcengruppen-Filter, um die Anzahl der angezeigten Ressourcen zu begrenzen. De Anwendung aktualisiert die Tabelle, um nur die Ressourcen oder Behälter anzuzeigen, die zur angegebenen Ressourcengruppe gehören.
4. Aktivieren Sie das Kontrollkästchen Aktivieren für jede Ressource, die Sie aktivieren möchten.

Oder

Deaktivieren Sie das Kontrollkästchen Aktivieren für jede Ressource, die Sie deaktivieren möchten.

5. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
6. Klicken Sie auf Abschicken. Die Wartungsanforderung wird für die ausgewählten Wartungsklassen aktiviert.
