Container suchen

Siemens bietet eine Suchfunktion, mit der Sie anhand bestimmter Kriterien nach Containern suchen und diese finden können. Sie können mithilfe bereitgestellter Filter oder benutzerspezifischer Abfragen nach einem Container suchen. Siemens stellt Filter zur Suche nach Containern basierend auf ihrem momentanen Status und ihrer Historie zur Verfügung.

Sobald das System die Container anzeigt, welche die Auswahlkriterien erfüllen, können Sie einen oder mehrere Container für das Ausführen von Transaktionen auswählen. Die Befehlsleiste enthält Transaktionsschaltflächen, die für die ausgewählten Container relevant sind.

Beim Suchen nach Containern

Sie können mehrere Parameter verwenden, um Ihre Suchergebnisse einzugrenzen. Die Anwendung stellt die folgenden Suchvorgänge bereit:

• Aktueller Status – Ermöglicht Ihnen die Suche nach Containern basierend auf ihrem aktuellen Status. Dazu zählen die folgenden:
• Allgemein
• Arbeitsplan
• Status
• Verlauf (History)
• Benutzerabfrage – Ermöglicht Ihnen das Erstellen individueller Abfragen für Container

Die Anwendung zeigt zwei Registerkarten und eine leere Tabelle mit Suchergebnissen an, wenn Sie zum ersten Mal auf die Seite "Containersuche" zugreifen. Die Registerkarten sind "Aktueller Status" und "Benutzerabfrage".

Die Anwendung erfordert, dass Sie mindestens einen Parameter (Filter oder Benutzerabfrage) festlegen, bevor die Schaltfläche "Suchen" aktiviert wird. Wenn Sie einen Parameter auf einer Registerkarte auswählen, reagiert die Anwendung wie folgt:

• Sie fügt einer Registerkarte ein Symbol hinzu, das anzeigt, dass ein Filter oder eine Benutzerabfrage für die Registerkarte ausgewählt wurde.
• Sie aktiviert die Schaltfläche "Suchen".
Hinweis: Sie müssen eine Benutzerabfrage in der Registerkarte "Benutzerabfrage" auswählen, um die Suchfunktion zu aktivieren.

Beim Verwenden der Linienzuordnungseinstellung als Suchfilter

Sie können die Einstellungen der Linienzuordnung als Filter verwenden, wenn Sie Container anhand von aktuellen oder historischen Kriterien suchen. Aktivieren Sie das Kontrollkästchen "Linienzuordnung anwenden", wenn Sie die Einstellungen der Linienzuordnung als Filter verwenden möchten. Das Kontrollkästchen "Linienzuordnung anwenden" wird auf der Registerkarte "Aktueller Status" in den Abschnitten "Status" und "Historie" angezeigt.

Zu den Einstellungen der Linienzuordnung zählen:

• Ressource / Arbeitszelle
• Arbeitsbereich
• Operation
• Arbeitsplatz

Weitere Informationen zu den Einstellungen der Linienzuordnung finden Sie unter "Einstellungen der Linienzuordnung".

Beispiel:

"Operation" steht in den Abschnitten "Arbeitsplan" und "Historie" als separater Filter zur Verfügung. "Ressource" steht im Abschnitt "Historie" als separater Filter zur Verfügung. Die Suchergebnisse umfassen alle Container mit einem der beiden Filterwerte, wenn der Filterwert für "Aktueller Status" und der Filterwert der Linienzuordnung nicht übereinstimmen.

Beim Suchen nach Containern mithilfe von Benutzerabfragen

Benutzerabfragen können nicht mit anderen Suchfiltern kombiniert werden und Sie müssen eine Benutzerabfrage-Suche von der Registerkarte "Benutzerabfrage" durchführen. Die Anwendung ignoriert sämtliche Filter, die auf der Registerkarte "Aktueller Status" ausgewählt sind, wenn Sie eine Suche von der Registerkarte "Benutzerabfrage" aus ausführen.

Sie müssen eine oder mehrere Benutzerabfragen zur Verwendung in der Modellierung definieren, um nach Containern zu suchen, bevor Sie eine Benutzerabfrage-Suche durchführen können. Die Abfrage muss eine Container-ID in den Auswahlkriterien enthalten. Andernfalls liefert die Abfrage keine Ergebnisse, wenn Sie von der Registerkarte "Benutzerabfrage" aus gestartet wird. Zusätzlich müssen Sie Werte für alle Parameter eingeben, damit die Suche Ergebnisse liefert.

Wichtig: Benutzerabfragen sind in der Regel datenbankspezifisch.

Optional können Sie eine oder mehrere Benutzerabfrage-Gruppen in der Modellierung definieren. Benutzerabfrage-Gruppen ermöglichen es Ihnen, die zur Auswahl vorhandenen Benutzerabfragen auf der Registerkarte "Benutzerabfrage" einzuschränken. Wenn Sie z. B. Benutzerabfragen für verschiedene Geschäftsanforderungen definiert haben, können Sie eine Containersuch-Benutzerabfragegruppe erstellen, so dass Sie lediglich relevante Benutzerabfragen für die Containersuche sehen.

Durch das Auswählen einer Benutzerabfrage reagiert die Anwendung wie folgt:

• Sie zeigt die Parameter an, die Sie zum Durchführen der Suche festlegen müssen.
• Sie zeigt eine Abfragebeschreibung an (wenn sie bei der Definition der Benutzerabfrage angegeben wurde).
• Sie aktiviert die Schaltfläche "Suchen".

Die Anwendung löscht das Feld "Benutzerabfrage" und alle damit verbundenen Daten, wenn Sie eine Benutzerabfragegruppe auswählen, nachdem Sie eine Benutzerabfrage ausgewählt haben.

Weitere Informationen zu Benutzerabfragen finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch und im Opcenter Execution Core Modeling-Benutzerhandbuch.

Beispiel-Abfragen

Es folgen zwei Beispiele für Abfragen:

Abfrage 1:

select ContainerId from Container c

join Product p on c.ProductId = p.ProductId

join ProductBase pb on p.ProductBaseId = pb.ProductBaseId

join CurrentStatus cs on c.CurrentStatusId = cs.CurrentStatusId

left join ResourceDef r on cs.ResourceId = r.ResourceId

Where pb.ProductName = ?ProductName -- 'Widget'

and r.ResourceName = ?ResourceName --'Workstation 1'

Order By ContainerName

Abfrage 2:

select ContainerId from Container c

join Product p on c.ProductId = p.ProductId

join ProductBase pb on p.ProductBaseId = pb.ProductBaseId

<{?ResourceName}>?<{ join CurrentStatus cs on c.CurrentStatusId =

cs.CurrentStatusId

left join ResourceDef r on cs.ResourceId = r.ResourceId }>

Where pb.ProductName = ?ProductName

<{?ResourceName}>?<{ and r.ResourceName = ?ResourceName}>

Order By ContainerName

Verwenden der Befehlsleiste

Sie müssen einen oder mehrere Container in der Tabelle auswählen, um die Aktionen der Befehlsleiste anzuzeigen, die für die ausgewählten Container verfügbar sind.

Beim Auswählen eines einzelnen Containers

Wenn Sie einen einzelnen Container auswählen, zeigt die Anwendung die Aktionen der Befehlsleiste an, die für einzelne Container verfügbar sind. Dazu zählen die folgenden:

• Containerstatus – der Status und die Attribute des ausgewählten Containers
• Arbeitsplan – der Arbeitsplan des ausgewählten Containers
• Dokument anhängen – Anhängen eines neuen oder vorhandenen Dokuments an den Container
• Dokument – die mit dem Container verknüpften Dokumente. Sie können auf ein Dokument klicken, um dieses anzuzeigen.
• Produktionsprotokoll – das Produktionsprotokoll des ausgewählten Containers

Beim Auswählen mehrerer Container

Die Anwendung zeigt die Aktionen der Befehlsleiste an, die auf alle Container angewendet werden können, wenn Sie mehrere Container ausgewählt haben. Wenn Sie in der Befehlsleiste auf eine Schaltfläche klicken, können Sie die Transaktion an allen ausgewählten Containern gleichzeitig ausführen. Wenn die ausgewählten Container keine Transaktionen gemeinsam haben, zeigt die Anwendung die Nachricht "Es sind keine Aktionen für die ausgewählten Container verfügbar." an.

Darüber hinaus können Sie Container von mehr als einer Seite in der Suchergebnis-Tabelle auswählen. Die Anwendung zeigt die Anzahl der Container an, die Sie unter der Tabelle ausgewählt haben.

Beim Auswählen von Fertigungstransaktionen

Wählen Sie in der Befehlsleiste "Fertigungsaktionen" aus, um die verfügbaren Transaktionen für die ausgewählten Container anzuzeigen. Standardmäßig macht die Anwendung verschiedene Aktionen verfügbar, je nachdem, welchen Status die von Ihnen ausgewählten Container haben, Ihren Berechtigungen und der Anzahl der ausgewählten Container. Beim Ausführen einer Aktion an einem ausgewählten Container

Die Anwendung zeigt ein Popup an, welches Ihnen ermöglicht, die Transaktion durch einen Klick auf eine Aktionsschaltfläche auszuführen. Die Anwendung füllt die ausgewählten Container im Popup automatisch.

Hinweis: Die Aktion "Sofort anmelden" führt die Transaktion "Anmelden" für den Container aus, ohne das Transaktions-Popup "Anmelden" zu öffnen. Sollte jedoch eine Umlaufbestandsnachricht für die Transaktion "Anmelden" konfiguriert worden sein, verlangt die Anwendung von Ihnen, diese zu bestätigen, bevor die Transaktion "Sofort anmelden" ausgeführt wird.

Zurücksetzen von Suchergebnissen

Löschen Sie mithilfe der Schaltfläche "Zurücksetzen" die Parameter (Filter und Benutzerabfragen) und Ergebnisse. Durch Zurücksetzen der Seite "Containersuche" können Sie eine neue Suche starten. Wenn Sie auf die Schaltfläche "Zurücksetzen" klicken, reagiert die Anwendung wie folgt:

• Alle angegebenen Suchfilter oder Abfragen aus allen Registerkarten werden gelöscht.
• Alle Registerkartensymbole werden gelöscht, sofern festgelegt.
• Alle Container aus der Suchergebnis-Tabelle werden gelöscht.
• Alle Befehlsleistenschaltflächen werden gelöscht.
• Die Registerkarte "Benutzerabfrage" wird gelöscht, wenn diese verwendet wird.

Konfigurieren von Spalten für die Ergebnistabelle

Sie können die Suchergebnis-Tabelle anpassen, indem Sie die Spalten angeben, die Sie sehen möchten. Wenn Sie auf das Symbol "Spalten zeigen" klicken, so wird das Popup "Spalten auswählen" aufgerufen, das Ihnen ermöglicht, Suchergebnisspalten hinzuzufügen und zu entfernen. Popup "Spalten Auswählen"

Das Spalten Auswählen Popup enthält zwei Listen. "Sichtbare Spalten" listet die Spalten auf, die in den Suchergebnissen angezeigt werden. "Verborgene Spalten" listet die Spalten auf, die Sie den Suchergebnissen hinzufügen können.

• Fügt die bewegten Spalten dem Ende der Liste hinzu.
• Aktualisiert die Seite, um Ihre Änderungen anzuzeigen, wenn Sie auf die Schaltfläche "Anwenden" klicken. Sie müssen die Seite nicht aktualisieren, um die Änderungen zu sehen.

Sie können Spalten durch Drag&Drop hinzufügen, entfernen oder neu ordnen, oder die Schaltflächen verwenden. Die Anwendung fügt, falls notwendig, eine vertikale Scrollleiste hinzu.

Hinweis: Die Anwendung schließt den Revisions-Wert in der Spalte in den Suchergebnissen ein, wenn Sie eine Spalte anzeigen, die Revisions-Werte enthält. Wenn Sie z. B. die Spalte "Spez." auswählen, zeigt die Anwendung den Spez.-Namen und die Revision in der Spalte "Spez." an.

Vorgehensweise zum Konfigurieren von Spalten für die Ergebnistabelle

Gehen Sie folgendermaßen vor, um die Spalten für die Ergebnistabelle zu konfigurieren:

1. Öffnen Sie die Seite Containersuche im Menü Suchen. Die Seite "Containersuche" wird angezeigt.
2. Führen Sie eine Suche durch. Die Suchergebnistabelle wird angezeigt. Sie zeigt die Container an, auf die Ihre Suchkriterien zutreffen.
3. Klicken Sie auf Spalten anzeigen. Die Anwendung zeigt ein Popup mit einer Liste von sichtbaren Spalten und einer Liste von Spalten an, die Sie zur Ansicht auswählen können.
4. Wählen Sie mit einer der folgenden Möglichkeiten die Spalten aus, die Sie in der Tabelle anzeigen möchten.
• Verschieben Sie die Spalten durch Drag&Drop zwischen den Listen Sichtbare Spalten und Ausgeblendete Spalten hin und her und ändern Sie die Reihenfolge der Spalten in diesen Listen.
• Klicken Sie auf Alle anzeigen um alle Spalten in die Liste Sichtbare Spalten zu bewegen. Die Anwendung verschiebt alle Spalten aus der Liste "Ausgeblendete Spalten" in die Liste "Sichtbare Spalten" und fügt diese unten an.
• Klicken Sie auf Alle ausblenden um alle Spalten in die Liste Ausgeblendete Spalten zu verschieben. Die Anwendung verschiebt alle Spalten aus der Liste "Sichtbare Spalten" in die Liste "Ausgeblendete Spalten" und fügt diese unten an.
5. Klicken Sie auf Anwenden, um die Spaltenkonfiguration umzusetzen und schließen Sie das Popup Spalten anzeigen. Die Anwendung wendet die Konfiguration auf die Suchergebnisse an.

Vorgehensweise zum Suchen nach einem Container durch Verwendung aktueller Statuskriterien

Gehen Sie wie folgt vor, um nach einem Container mit den aktuellen Statuskriterien zu suchen:

1. Öffnen Sie die Seite Containersuche im Menü Suchen. Die Seite "Containersuche" wird angezeigt. Die Registerkarte "Aktueller Status" wird ausgewählt und der Abschnitt "Allgemein" wird im Panel "Filter" erweitert.
2. Geben Sie einen oder mehrere Parameter ein oder wählen Sie sie aus, um Ihre Suche im Panel "Filter" einzugrenzen. Klicken Sie auf die Abschnittstitel, um die Abschnitte zu erweitern oder zu reduzieren. Blättern Sie mithilfe der Bildlaufleiste des Panels durch die verfügbaren Abschnitte. Die Anwendung fügt dem Abschnittstitel ein Symbol hinzu und aktiviert die Schaltfläche "Suchen".
3. Klicken Sie auf Suchen. Die Anwendung zeigt Container, die den Suchkriterien entsprechen, in einer Tabelle an.

Vorgehensweise zum Suchen nach einem Container mithilfe von Benutzerabfragen

Gehen Sie folgendermaßen vor, um nach einem Container mit Benutzerabfragen zu suchen:

1. Öffnen Sie die Seite Containersuche im Menü Suchen. Die Seite "Containersuche" wird angezeigt.
2. Klicken Sie auf die Registerkarte Benutzerabfrage. Das Panel "Benutzerabfrage" wird angezeigt.
3. Wählen Sie eine Benutzerabfragegruppe aus.
4. Wählen Sie eine Benutzerabfrage aus. Die Anwendung aktualisiert die Registerkarte mit der Abfrageinformation, fügt der Registerkarte ein Symbol hinzu, zeigt eine Nachricht über die Art des zurückkommenden Ergebnisses an und aktiviert die Schaltfläche "Suchen".
5. Geben Sie die erforderlichen Werte in die Tabelle Abfrageparameter ein.
6. Klicken Sie auf Suchen. Die Anwendung zeigt die Container, die Ihren Suchkriterien entsprechen, in der Tabelle "Suchergebnisse" an.

Vorgehensweise zum Durchführen einer Transaktion an einem ausgewählten Container

Gehen Sie wie folgt vor, um eine Transaktion an einem ausgewählten Container durchzuführen:

1. Führen Sie eine Suche mit einer der oben genannten Suchmethoden durch.
2. Wählen Sie die Container aus, für die Sie eine Transaktion durchführen möchten. Die Anwendung zeigt Befehlsleistenschaltflächen für die Aktionen an, die für die ausgewählten Container zur Verfügung stehen.
3. Klicken Sie auf die Befehlsleistenschaltfläche "Fertigungsaktionen". Die Anwendung zeigt ein Panel mit allen verfügbaren Fertigungstransaktionen für die ausgewählten Container an.
4. Klicken Sie auf eine Transaktion. Die Anwendung zeigt ein Popup für die ausgewählte Transaktion an.
5. Vervollständigen Sie das Popup und klicken Sie auf die Schaltfläche zum Ausführen der Transaktion. Die Anwendung schließt das Popup und zeigt eine Erfolgsnachricht an.
