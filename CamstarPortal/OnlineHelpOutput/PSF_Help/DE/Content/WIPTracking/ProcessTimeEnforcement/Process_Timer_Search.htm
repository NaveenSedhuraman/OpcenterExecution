Prozesstimersuche

Auf der Seite "Prozesstimersuche" können Sie alle aktuellen und im Verlauf gespeicherten Prozesstimer ansehen. Außerdem können Sie hier Prozesstimer anhalten.

Beim Suchen nach Prozesstimern

Sie können mehrere Parameter verwenden, um Ihre Suchergebnisse einzugrenzen. So können Sie beispielsweise in das Feld "Timer gestartet – Beginn" den Wert "01/01/2016 08:00 AM" und in das Feld "Timer gestartet – Ende" den Wert "01/01/2016 05:00 PM" eingeben, um nur Timer zurückzugeben, die zwischen diesen beiden Zeiten gestartet wurden.

Die Anwendung zeigt eine leere Suchergebnistabelle an, wenn Sie die Seite "Prozesstimersuche" zum ersten Mal öffnen. Sie müssen mindestens einen Parameter angeben, bevor die Schaltfläche "Suchen" aktiviert wird.

Eine Filterung nach dem Abschlussstatus kann nur für abgeschlossene Timer durchgeführt werden. "Aktueller Status" wird automatisch auf "Abgeschlossen" gesetzt, wenn Abschlussstatuskriterien ausgewählt sind. "Abschlussstatus" wird deaktiviert, wenn "Aktueller Status" auf "In Bearbeitung" gesetzt wird.

Wenn Sie eine Suche ausführen, reagiert die Anwendung wie folgt:

• Die Suchergebnisse werden in der Tabelle "Suchergebnisse" angezeigt.
• Die Suchparameter werden beibehalten.

Alle Spaltenüberschriften der Tabelle "Suchergebnisse" sind sortierbar. Die Standardsortierreihenfolge für Timer lautet wie folgt:

• Zeit bis Max. – Timer mit der längsten Zeit über "Max. Zeit " bis zur kürzesten Zeit unter "Max. Zeit"
• Zeit bis Min. – Timer mit der längsten Zeit über "Min. Zeit" bis zur kürzesten Zeit unter "Min. Zeit"
• Timer beendet – Zuletzt beendete Timer.

Die Tabellendaten werden automatisch alle fünf Sekunden aktualisiert. Sie können festlegen, wie oft die Tabellendaten aktualisiert werden, indem Sie die Tabelle in Portal Studio anpassen. Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics Portal Studio-Benutzerhandbuch oder im Opcenter Execution Core Portal Studio-Benutzerhandbuch.

Siemens bietet auch die Möglichkeit, Suchergebnisse im Excel- oder CSV-Format zu exportieren.

Verwenden Sie die Schaltfläche "Alles löschen", um Parameter und Ergebnisse zu löschen. Wenn Sie die Seite "Prozesstimersuche" löschen, können Sie eine neue Suche starten. Wenn Sie auf die Schaltfläche "Alles Löschen" klicken, reagiert die Anwendung wie folgt:

• Alle angegebenen Suchfilter werden gelöscht.
• Die Timer aus der Tabelle "Suchergebnisse" werden gelöscht.
• Die Aktionsschaltfläche "Timer stoppen" wird gelöscht
• Die Schaltfläche "Suchen" wird deaktiviert.
Beim Anhalten von Prozesstimern

Die Anwendung zeigt die Schaltfläche "Timer stoppen" an, wenn Sie in der Tabelle "Suchergebnisse" einen Timer auswählen, dessen "Aktueller Status" "In Bearbeitung" lautet. Sie müssen Berechtigungen zum Stoppen von Timern haben.

Wenn Sie einen Timer mit dem Status "In Bearbeitung" auswählen und auf die Schaltfläche "Timer stoppen" klicken, wird das Popup "Timer stoppen" angezeigt. Die Eingabe eines Kommentars ist optional.

Wenn Sie auf "Timer stoppen" klicken, wird das Popup geschlossen und der ausgewählte Timer wird gestoppt. Nachdem ein Timer angehalten wurde, zeigt die Anwendung dieselbe Seite der Tabelle "Suchergebnisse" an. Ein angehaltener Timer, der nicht mehr den Filterkriterien entspricht, wird nicht mehr in der Tabelle "Suchergebnisse" angezeigt.

Timer, die über das Popup "Timer stoppen" gestoppt werden, können mithilfe von "Letzte Transaktion rückgängig machen" fortgeführt werden. Weitere Informationen finden Sie unter "Letzte Transaktion rückgängig machen".

Vorgehensweise zum Suchen nach Prozesstimern

Gehen Sie folgendermaßen vor, um nach Prozesstimern zu suchen:

1. Öffnen Sie die Seite Prozesstimersuche im Menü Suchen. Die Seite "Prozesstimersuche" wird angezeigt.
2. Wählen Sie Suchkriterien aus, oder geben Sie welche ein.
3. Klicken Sie auf Suchen. Die Anwendung zeigt die Prozesstimer, die mit Ihren Suchkriterien übereinstimmen, in der Tabelle "Suchergebnisse" an.

Vorgehensweise zum Anhalten eines Prozesstimers

Gehen Sie folgendermaßen vor, um einen Prozesstimer anzuhalten:

1. Befolgen Sie die Schritte in "Vorgehensweise zum Suchen nach Prozesstimern". Die Anwendung aktualisiert die Tabelle "Suchergebnisse" mit einer Liste der Prozesstimer, die den eingegebenen Suchkriterien entsprechen.
2. Wählen Sie einen Timer in der Tabelle "Suchergebnisse" aus. Die Anwendung zeigt die Aktionsschaltfäche "Timer stoppen" für Timer mit dem Status "In Bearbeitung" an. Sie müssen Berechtigungen zum Stoppen von Timern haben.
3. Klicken Sie auf Timer stoppen. Das Popup "Timer stoppen" wird angezeigt.
4. Geben Sie je nach Ihren Geschäftsanforderungen optionale Kommentare ein und klicken Sie auf Anhalten. Die Anwendung stoppt den ausgewählten Timer, die Endzeit des Timers wird auf die aktuelle Systemzeit gesetzt, "Aktueller Status" wird auf "Beendet" gesetzt und das Popup "Timer anhalten" wird geschlossen. Die Anwendung aktualisiert die Tabelle "Suchergebnisse" und das Produktionsprotokoll.
