Container schließen

Die "Schließen"-Transaktion ermöglicht es Ihnen, erlaubte Eltern- oder Kind-Container zu schließen. Durch das Schließen eines Eltern-Containers werden auch automatisch alle zugehörigen Kind-Container geschlossen. Sie können jedoch auch einen Kind-Container unabhängig vom Eltern-Container schließen. Alle aktiven Container können geschlossen werden. Das Schließen eines Containers hält alle aktiven Prozesstimer, die mit diesem Container verknüpft sind, an.

Hinweis: Wenn Sie mehrere Container gleichzeitig schließen möchten, finden Sie Informationen unter "Mehrere Container schließen".

Beim Schließen von Containern

Alle Anzeigefelder sind auf der Seite "Schließen" leer, bis ein Container ausgewählt wird. Anzeigefelder werden im "Schließen"-Popup automatisch gefüllt. Informationen über das Ausführen einer Transaktion über eine Seite oder ein Popup finden Sie unter "Verwenden von Transaktionsseiten und Popups".

Container mit dem Status "Geschlossen" stehen nicht zur Auswahl, wenn Sie einen Container auf der "Schließen"-Seite auswählen möchten. Wenn Sie einen Container mit dem Status "Geschlossen" scannen oder eingeben oder einen Container eingeben, der nicht vorhanden ist, zeigt die Anwendung eine Fehlermeldung an.

Optional können Sie einen Grund für Statusänderung angeben und Kommentare eingeben, wenn Sie einen Container schließen. Gründe für Statusänderungen werden in der Modellierung von einem Systemadministrator definiert. Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch und im Opcenter Execution Core Modeling-Benutzerhandbuch.

Geschlossene Container sind zur Archivierung geeignet, aber nicht für zusätzliche Transaktionen außer Kommentaren, Datensammlung und Zurückrollen einer Transaktion. Standardmäßig stehen geschlossene Container normalerweise nicht in anderen Feldern und Tabellen auf anderen Seiten zur Auswahl.

Die Anwendung sucht nach aktiven Prozesstimern, wenn die Transaktion "Schließen" zum Schließen eines Containers verwendet wird. Alle vorhandenen Timer werden angehalten und die Transaktion "Schließen" im Produktionsprotokoll für den Container zeigt die Prozesstimer-Historiendetails für alle Timer an, die angehalten werden.

Seite "Schließen"

Sie können über das Menü "Container" auf die Seite "Schließen" zugreifen. Die Seite "Schließen" ermöglicht es Ihnen, einen zu schließenden Container zu scannen oder einzugeben.

Hinweis: Die Seite "Schließen" ist eventuell auch als Popup verfügbar, wenn die Transaktion "Container schließen" als verfügbare Aktion auf einer anderen Seite konfiguriert wurde (z. B. "Containersuche").

Felddefinitionen der Seite und des Popups "Schließen"

Diese Tabelle definiert die Felder auf der Seite und im Popup "Schließen".

Definition

Typ

Containerinformationen

Erforderlich/ Nur Anzeige

Zusätzliche Felder anzeigen/ausblenden

Optionale Felder sind standardmäßig ausgeblendet, wenn die Anwendung eine Seite oder ein Popup anzeigt. Durch Klicken auf die Steuerung "Anzeigen" werden die Felder angezeigt. Durch Klicken auf die Steuerung "Verstecken" werden die Felder ausgeblendet.

  

Kommentare

Zusätzliche Informationen oder Anweisungen für diese Transaktion oder für diesen Container. Das Feld akzeptiert maximal 255 Zeichen. Wenn die Kommentare 255 Zeichen überschreiten, verkürzt die Anwendung die Daten.

Optional

Vorgehensweise zum Schließen eines Containers

Gehen Sie folgendermaßen vor, um einen Container zu schließen:

1. Wählen Sie Schließen im Container-Menü. Die Seite "Schließen" wird angezeigt.
2. Wählen Sie einen Container im Container-Feld aus oder geben Sie ihn ein. Die Anwendung aktualisiert den Container-Kopf und die Auswahlfelder, sodass diese Informationen für den angegebenen Container angezeigt werden. Werte werden nur angezeigt, wenn sie für den Container eingegeben wurden.
Hinweis: Die Anwendung füllt das Feld "Container" und den Container-Kopf automatisch, wenn Sie von einer anderen Fertigungsseite auf das Popup "Schließen" zugreifen.
3. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
4. Klicken Sie auf Schließen oder in der Befehlsleiste auf Abschicken. Die Anwendung zeigt eine Statusmeldung an, die bestätigt, dass der Container geschlossen wurde.
