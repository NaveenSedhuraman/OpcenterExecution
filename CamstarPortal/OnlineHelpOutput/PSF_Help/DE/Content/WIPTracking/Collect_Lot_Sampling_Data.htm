Sammeln von Los-Stichprobendaten

Sie können die Seite "Los-Stichprobendaten sammeln" verwenden, um Stichprobendaten für ein Los zu sammeln. Die Seite gibt alle Stichprobentests, die derzeit für das Los eines bestimmten Containers erforderlich sind, sowie die entsprechenden Anweisungen dazu aus. Wenn ein Stichprobentest ausgewählt wird, werden aktuelle Informationen einschließlich der erforderlichen Stichproben, getesteten Stichproben, bestandenen Stichproben, durchgefallenen Stichproben, AQL-Level und Inspektionsebene angezeigt.

Sobald ein Stichprobentest ausgewählt wurde, können Sie die Stichprobentestdaten für den Container des Loses sammeln.

Ist der Stichprobentest ein:

• Gemessener Stichprobentyp - geben Sie den numerischen Wert für jeden Datenpunkt ein
• Gezählter Stichprobentyp - verzeichnen Sie die Anzahl der abgelehnten Proben für jeden Grundcode
Beim Sammeln von Los-Stichprobendaten

Diese Regeln gelten beim Sammeln von Los Stichproben-Daten:

• Nur Container, die Stichprobentests enthalten und einem Los zugeordnet sind, werden in der "Container"-Auswahlliste angezeigt. Container in Warteschlange müssen verarbeitet werden, indem Sie eine Anmelden-Transaktion durchführen. Linienzuordnungs-Werte dienen zum Filtern der Container-Liste anhand von Operation, Arbeitsplatz, Ressource und Arbeitsbereich.
• Wenn an der aktuellen Spec des Containers Stichproben erforderlich sind, können Container nicht weiterbewegt werden, bis mindestens eine Stichprobe entnommen wurde. Das Kontrollkästchen "Stichproben erforderlich" in der Containerliste wird für die Container aktiviert, für die Stichproben erforderlich sind. Diese Container werden am Anfang der Liste angezeigt. Container, für die keine Stichproben erforderlich sind, sind für Zufallsstichprobenuntersuchungen verfügbar.
• Die Stichprobendaten für ein Los müssen nicht auf einmal erfasst werden. Die Anwendung sammelt Prüfdaten für das Los, während die Stichprobendaten für einen Loscontainer für jeden Stichprobentest im Stichprobenplan gesammelt werden.
• Die Seite "Los-Stichprobendaten sammeln" kann über einen automatischen Ablauf oder über ein eigenständiges Fertigungs-Formular gestartet werden.

Hinzufügen von Los Stichproben-Daten zu einem automatischen Ablauf

Ein automatischer Ablauf ist ein revisioniertes Objekt, das es Ihnen ermöglicht, einer Spezifikation (Spez.) eine Aufgabenliste mit einer "Los-Stichprobendaten sammeln"-Aufgabe zuzuweisen. Sie müssen eine Aufgabenliste definieren, die eine Transaktions-Aufgabe enthält, die das CollectLotSamplingDataVP-Formular referenziert, bevor Sie die Aufgabenliste dem automatischen Ablauf zuweisen. Informationen zum Definieren von Aufgabenlisten, automatischen Abläufen und Spezifikationen finden Sie im Opcenter Execution Medical Device and Diagnostics Modeling-Benutzerhandbuch oder im Opcenter Execution Core Modeling-Benutzerhandbuch.

Seite "Los-Stichprobendaten sammeln"

Vorgehensweise zum Durchführen einer gemessenen Stichproben-Datensammlung für ein Los

Gehen Sie folgendermaßen vor, um eine gemessene Stichproben-Datensammlung für ein Los durchzuführen:

1. Öffnen Sie die Seite Los-Stichprobendaten sammeln im Menü Container. Die Seite "Los-Stichprobendaten sammeln" wird angezeigt.
2. Wählen Sie einen Container im Container-Feld aus oder geben Sie ihn ein. Die Anwendung aktualisiert den Container-Kopf und die Auswahlfelder, sodass diese Informationen für den angegebenen Container angezeigt werden. Werte werden nur angezeigt, wenn sie für den Container eingegeben wurden.
3. Wählen Sie einen gemessenen Stichprobentest aus der Liste in der Tabelle Los-Stichprobendaten.
4. Geben Sie die Anzahl der Stichproben, die Sie für diesen Stichprobentest sammeln möchten, im Feld Erzeugen von Stichproben für den obigen Stichprobentest ein, um das Layout der Datensammlungs-Tabelle für gemessene Stichprobentests festzulegen.
5. Klicken Sie auf Erzeugen. Die Anwendung verbirgt das Feld "Generieren von Stichproben für den obigen Stichprobentest" und die Schaltfläche "Generieren" und zeigt die Tabelle "Gemessen" an.
6. Geben Sie den Datenpunkt-Wert für jeden Stichprobentest in die Tabelle "Gemessen" ein, falls erforderlich.
7. Klicken Sie auf Abschicken oder in der Befehlsleiste auf Abschicken, um die Daten für den Stichprobentest aufzuzeichnen. Die Anwendung hält den aktuellen Container ausgewählt und aktualisiert den Status in der Tabelle "Los-Stichprobendaten".

Vorgehensweise zum Durchführen einer gezählten Stichproben-Datensammlung für ein Los

Gehen Sie folgendermaßen vor, um eine gezählte Stichproben-Datensammlung für ein Los durchzuführen:

1. Öffnen Sie die Seite Los-Stichprobendaten sammeln im Menü Container. Die Seite "Los-Stichprobendaten sammeln" wird angezeigt.
2. Wählen Sie einen Container im Container-Feld aus oder geben Sie ihn ein. Die Anwendung aktualisiert den Container-Kopf und die Auswahlfelder, sodass diese Informationen für den angegebenen Container angezeigt werden. Werte werden nur angezeigt, wenn sie für den Container eingegeben wurden.
3. Wählen Sie einen gezählten Stichprobentest aus der Liste in der Tabelle Los-Stichprobendaten.
4. Geben Sie die Anzahl von Containern, die überprüft werden sollen, ins Feld Gesammelte Stichproben ein.
5. Geben Sie für jeden Rückweiscode die Anzahl der Stichproben ins Feld Zähler ein, die Inspektion nicht bestanden haben.
6. Klicken Sie auf Abschicken oder in der Befehlsleiste auf Abschicken. Die Anwendung hält den aktuellen Container ausgewählt und aktualisiert den Status in der Tabelle "Los-Stichprobendaten".
