Containerinformationen

Informationen über den ausgewählten Container werden oben auf Container-bezogenen Fertigungstransaktions-Seiten und -Popups angezeigt. Die Informationen für den ausgewählten Container können auch über die Befehlsleiste aufgerufen werden. In der Regel umfassen diese Informationen das relevante Produkt, Menge, Operation und Produktionsauftrag. Dieser Bereich beinhaltet auch alle aktiven Prozesstimer.

Hinweis: Containerinformationen werden nur für einzelne Container angezeigt. Multi-Container-Transaktionsseiten und -Popups enthalten keinen Bereich mit Containerinformationen.

Beim Anzeigen von Containerinformationen auf einer Transaktionsseite

Die Anwendung erfordert, dass Sie einen Container scannen oder ins "Container"-Feld auf Transaktionsseiten eingeben. Alle anderen Felder sind schreibgeschützt. Die Anwendung füllt die schreibgeschützten Containerinformationsfelder, wenn Sie das "Container"-Feld durch Drücken der Tab-Taste verlassen.

Das Angeben eines Containers auf einer Transaktionsseite führt dazu, dass die Anwendung:

• Ein Symbol anzeigt, das den Status des Containers angibt
• Informationen zu Produkt, Menge, Operation und Produktionsauftrag anzeigt, falls vorhanden
• Alle aktiven Prozesstimer anzeigt

Methoden zur Angabe eines Containers

Siemens bietet drei Möglichkeiten, einen Container auf einer Transaktion-Seite anzugeben:

• Den Barcode des Containers scannen
• Den Containernamen in das Feld "Container" eingeben
• Den Container im Feld "Container" auswählen

Beim Anzeigen von Containerinformationen in einem Transaktionsseiten-Popup

Die Anwendung füllt den Containernamen und die Containerinformationen automatisch, wenn Sie ein Transaktions-Popup starten. Alle Felder sind schreibgeschützt. Die Anwendung füllt die Containerinformationen basierend auf dem Container, der auf der Seite, von der aus Sie das Popup-Fenster gestartet haben, ausgewählt wurde.

Container-Informationsfeld und -Symbol-Definitionen

Diese Tabelle definiert die Container-Informationsfelder, die auf einer Seite Transaktionsseite oder in einem Transaktions-Popup angezeigt werden.

Feld/Symbol

Definition

Typ

Container

Name des Containers, an dem Sie eine Aktion durchführen möchten. In einem Popup ist dieses Feld schreibgeschützt.

Erforderlich/ Nur Anzeige

<Statussymbol>

Symbol, das den Status des Containers angibt. Dieses Symbol wird nur auf Transaktionsseiten angezeigt.

Hinweis: Das Aussehen des Symbols hängt vom Status des Containers ab.

Nur Anzeige

Produkt

Zum Container gehöriges Produkt, gefolgt von der Produktbeschreibung in Klammern.

Nur Anzeige

Menge

Mit dem Container verknüpfte Menge. Wenn die Maßeinheit für die Menge vorgegeben wurde, zeigt die Anwendung die "Maßeinheit" zusammen mit der Menge an.

Nur Anzeige

Operation

Operation, an der Container im Arbeitsplan ist.

Hinweis: Dieses Feld ist ausgeblendet, wenn Sie die Anwendung auf einem mobilen Gerät anzeigen.

Nur Anzeige

Produktionsauftrag

Produktionsauftrag, für den der Container gestartet wurde.

Hinweis: Dieses Feld ist ausgeblendet, wenn Sie die Anwendung auf einem mobilen Gerät anzeigen.

Nur Anzeige

Timer

Zeigt die aktiven Timer an, sofern dem ausgewählten Container welche zugeordnet sind. Der wichtigste Timer wird im Feld angezeigt. Wenn Sie auf das Dreieck neben der Feldbezeichnung "Timer" klicken, wird das Flyout "Aktive Timer" geöffnet und zeigt alle aktiven Timer an. Weitere Informationen finden Sie unter "Anzeigen von Prozesstimern".

Nur Anzeige
Hinweis: Auslassungspunkte werden neben dem Feldnamen angezeigt und eine QuickInfo zeigt den verbleibenden Text an, wenn ein Feld Informationen enthält, die die Feldlänge überschreiten.

Containerstatus (Container Status)

Das Symbol "Containerstatus" wird nur auf Transaktionsseiten angezeigt. Sein Aussehen gibt den Status des Containers an. Zu den möglichen Status gehören:

Container ist aktiv

Container ist angehalten

Container ist geschlossen

Hinweis: Zusätzliche Zustände können durch einen Systemadministrator konfiguriert werden.

Feld "Timer"

Das Feld "Timer" wird sowohl auf Transaktionsseiten als auch in Popups angezeigt, wenn dem Container mindestens ein aktiver Timer zugeordnet ist. Der wichtigste Timer wird im Feld angezeigt. Wenn Sie auf das Dreieck neben der Feldbezeichnung "Timer" klicken, wird das Flyout "Aktive Timer" geöffnet und zeigt alle aktiven Timer an. Weitere Informationen finden Sie unter "Anzeigen von Prozesstimern".
