Anzeigen des Ressourcenprotokolls

Im Ressourcenprotokoll können Sie die Transaktionshistorie für Ressourcen anzeigen. Die Anwendung beginnt sofort mit der Aufzeichnung von Transaktionen, wenn Sie eine Ressource einrichten.

Beim Durchsuchen des Ressourcenprotokolls

Wenn das Ressourcenprotokoll zum ersten Mal geladen wird, ist nur die Registerkarte "Suchen" sichtbar. Sie können Informationen in die Felder eingeben, um die Suche zu definieren.

Sie müssen eine Ressource in der Liste "Ressource" im Abschnitt "Filter" auswählen. Die anderen Felder im Abschnitt "Filter" sind optional. Bei Auswahl eines Elements in einem der optionalen Felder werden die im Feld "Ressource" aufgelisteten Ressourcen gefiltert. Die Auswahl eines Elements in den optionalen Feldern hat keinen Einfluss auf die in den anderen optionalen Feldern aufgelisteten Elemente.

Wenn Sie z. B. "Microscopes" in der Liste "Ressourcentyp" auswählen, listet die Anwendung nur "Microscope[01-05]" in der Liste "Ressource" auf. In der Ressourcengruppe "Inspection Lab" ist nur "Microscope01" enthalten. Wenn Sie dann "Inspection Lab" in der Liste "Ressourcengruppe" auswählen, wird in "Ressource" nur "Microscope01" aufgelistet. Die Auswahl von "Microscopes" in der Liste "Ressourcentyp" hat keinen Einfluss auf die in den Feldern "Ressourcenfamilie" oder "Ressourcengruppe" aufgelisteten Elemente.

Filter im Abschnitt "Zeitrahmen" bieten die Möglichkeit, die Suchergebnisse nach Datum und Uhrzeit einzugrenzen.

Filter im Abschnitt "Transaktionen" bieten die Möglichkeit, die Suchergebnisse nach Transaktionstyp einzugrenzen. Das Feld "Transaktionsfilter" bietet folgende Möglichkeiten:

• Begrenzen der Transaktionen auf ressourcenbasierte Transaktionen und deren Unterklassen
• Beispiel: Ressourcendatensammlung, Ressourceneinrichtung, Ressourcendurchsatz und Wartungsverwaltung
• Begrenzen der Transaktionen auf containerbasierte Transaktionen (und deren Unterklassen), die auf Ressourcen verweisen
• Beispiel: Transaktion "Anmelden", die eine Ressource angibt, oder eine Aufgabe "Automatischer Ablauf", in der eine Ressource ausgewählt wird

Durchführen einer Suche

Wenn Sie eine Suche durchführen, wird die Tabelle "Historienhauptlinie" angezeigt. Darüber hinaus können u. U. weitere Registerkarten angezeigt werden, z. B. "Attribute", "Elektronische Unterschrift", "Etikettendruck", "Datensammlung" und "Umlaufbestandsnachricht".

Nachrangige Filterfelder bietet die Möglichkeit, Ihre Suche weiter zu verfeinern. Diese Felder werden auf der Registerkarte "Suchen" angezeigt, nachdem die erste Suche durchgeführt wurde. Auswahlen in diesen Feldern hängen von den Transaktionsinformationen ab, die von der ersten Suche ausgegeben wurden. Die Anwendung schränkt die Ergebnisse, die in der Tabelle "Historienhauptlinie" angezeigt werden, anhand dieser Kriterien ein.

Beim Auswählen einer anzuzeigenden Transaktion

Die Anwendung zeigt eine Tabelle mit zusätzlichen Transaktionsinformationen unter der Tabelle "Historienhauptlinie" an, wenn Sie in der Tabelle "Historienhauptlinie" eine Transaktion auswählen. Die Tabelle kann abhängig von Ressourcentyp, Transaktionstyp und Funktionalität, die für die Transaktion konfiguriert wurde, eine oder mehrere Registerkarten enthalten.

Hinweis: Systemadministratoren können die Informationen, die auf den Registerkarten angezeigt werden, mit dem Protokoll-Konfigurationswerkzeug konfigurieren. Weitere Informationen finden Sie im Opcenter Execution Medical Device and Diagnostics Audit Trail-Benutzerhandbuch und im Opcenter Execution Core Audit Trail-Benutzerhandbuch.

Seite "Ressourcenprotokoll"

Auf der Seite "Ressourcenprotokoll" können Sie die Historie für eine Ressource anzeigen. Die Seite besteht aus folgenden Teilen:

• Registerkarte "Suchen" - wird beim Laden der Seite angezeigt
• Registerkarte "Status" - wird nur angezeigt, nachdem erste Suchergebnisse zurückgegeben wurden, wenn ein Systemadministrator in der Modellierung eine UI-Einstellung für die Ressource definiert hat.
Hinweis: Das Panel mit den Registerkarten kann zusammengeklappt werden. Durch das Zusammenklappen des Registerkartenbereichs werden die Tabellen horizontal erweitert.
• Breadcrumb-Navigation - wird angezeigt, nachdem Sie eine Transaktion in der Tabelle "Historienhauptlinie" ausgewählt haben, und zeigt eine Historie Ihrer ausgewählten Transaktionen mit Navigationsmöglichkeiten an.
• Tabelle "Historienhauptlinie" - wird nach Ausgabe der ersten Suchergebnisse angezeigt und zeigt alle Transaktionen für die angegebene Ressource an, die den Suchkriterien entsprechen.
• Registerkarten mit Transaktionsinformationen - werden angezeigt, wenn Sie eine Transaktion in der Tabelle "Historienhauptlinie" auswählen

Registerkarte "Status"

Auf der Registerkarte "Status" werden Informationen zur angegebenen Ressource angezeigt. Die Anwendung zeigt die Registerkarte "Status" nicht an, wenn von einem Systemadministrator keine UI-Einstellung definiert wurde.

Tabelle "Historienhauptlinie"

Die Tabelle "Historienhauptlinie" listet alle Transaktionen auf, die für die angegebene Ressource ausgeführt wurden. Wenn Sie eine Transaktion in dieser Tabelle auswählen, zeigt die Anwendung eine Untertabelle mit einer oder mehr Registerkarten mit zusätzlichen Transaktionsinformationen an.

Durch Schließen des Panels mit der Registerkarte "Suchen" können Sie die Tabelle "Historienhauptlinie" horizontal erweitern. Das Erweitern der Tabelle "Historienhauptlinie" ermöglicht es Ihnen, mehr vom Transaktionsverlauf zu sehen, ohne horizontal scrollen zu müssen. Die Tabelle "Historienhauptlinie" nimmt die ganze Breite der Seite "Protokoll" ein.

Vorgehensweise zum Anzeigen der Transaktionshistorie für eine Ressource

Gehen Sie folgendermaßen vor, um die Transaktionshistorie für eine Ressource anzuzeigen:

1. Öffnen Sie die Seite Ressourcenprotokoll im Menü "Ressource". Die Seite "Ressourcenprotokoll" wird angezeigt.
2. Wählen Sie ggf. Ressourcentyp, Ressourcenfamilie und Ressourcengruppe aus, um die Ressourcen zu beschränken, die in der Liste Ressource ausgewählt werden können.
3. Wählen Sie die gesuchte Ressource in der Liste Ressource aus.
4. Geben Sie entsprechend Ihren Geschäftsanforderungen optionale Informationen ein. Weitere Informationen zu den optionalen Feldern finden Sie in der Tabelle der Felddefinitionen.
5. Klicken Sie auf Suchen. Die Anwendung zeigt die Registerkarte "Status" (sofern zutreffend) und die Tabelle "Historienhauptlinie" für die angegebene Ressource an.
6. Wählen Sie eine Transaktion in der Tabelle "Historienhauptlinie". Die Anwendung aktualisiert den Breadcrumb-Pfad und zeigt die entsprechende Transaktionstabelle unter der Tabelle "Historienhauptlinie" an.
