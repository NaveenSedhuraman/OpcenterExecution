SPC 测试器

Siemens 提供了 SPC 测试器，用于测试应用程序中包含的样本图表和数据。还可以在配置了图表和必需组件后测试安装程序。

Siemens 提供了可以通过 Management Studio 加载的样本图表和数据。

使用 SPC 测试器页

必须选择“SPC 图表”、“SPC 图表组”或“数据收集”。所有其他字段都是可选字段。

测试数据可通过 Management Studio 中的 DB 更新加载。Siemens 建议只在开发环境中而不要在生产环境中加载该数据。

样本宏和样本图表指向从 DB 更新加载的数据。样本图表中的路径指向默认安装位置。如果没有使用默认安装位置，则需要更新路径。

在使用测试器页之前，还必须更新具有 ODBC 身份验证信息的样本宏。

在此页上输入的 UI 参数将替代在“SPC 图表”对象中设置的参数。

如何使用 SPC 测试器页

按照以下步骤使用“SPC 测试器”页：

1. 从 SPC 菜单选择 SPC 测试器。将显示“SPC 测试器”页。
2. 选择“SPC 图表”、“SPC 图表组”或“数据收集”。只有一个是必需的。
3. 选择一个或多个字段，以进一步定义要生成的图表。
4. 单击生成图表。将显示所选图表。
