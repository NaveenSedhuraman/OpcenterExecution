管理团队

通过团队管理功能可使用“管理团队员工”页将员工添加到团队中或从中移除员工。如果在“员工”页上选中了“允许多个团队”复选框，则可以将员工指派给多个团队。有关创建团队的信息，请参考《  Opcenter Execution Medical Device and Diagnostics Modeling 用户指南》或《  Opcenter Execution Core Modeling 用户指南》。

还可以使用“管理团队员工”页使团队从工作区域签到和签退。

团队从工作区域签到

必须选择一个团队，然后单击“指派团队”按钮，以显示“工作区域签到签退”弹出窗口。应用程序根据在“签到类型”字段中进行的选择显示可用工作区域列表。选择“签到类型”时，列表名称将动态更改。必须从列表中选择一个实例，才能使团队从该工作区域签到。

如何为团队添加员工

按照以下步骤为团队添加员工：

1. 打开管理团队员工页。将显示“管理团队员工”页。
2. 从列表中选择团队。应用程序将填充“可添加的员工”和“团队员工”表格。应用程序还会启用“添加员工”、“移除员工”和“指派团队”按钮。
3. 选中要添加到团队的一个或多个员工对应的复选框。
4. 单击添加员工。应用程序会从“可添加的员工”表格中移除所选员工的名称，并将其移动到“团队员工”表格。

如何从团队中移除员工

按照以下步骤从团队中移除员工：

1. 打开管理团队员工页。将显示“管理团队员工”页。
2. 从列表中选择团队。应用程序将填充“可添加的员工”和“团队员工”表格。应用程序还会启用“添加员工”、“移除员工”和“指派团队”按钮。
3. 选中要从团队中移除的一个或多个员工对应的复选框。
4. 单击移除员工。应用程序会从“团队员工”表格中移除所选员工的名称，并将其移动到“可添加的员工”表格。

如何使团队从工作区域签到

按照以下步骤使团队从工作区域签到：

1. 打开管理团队员工页。将显示“管理团队员工”页。
2. 从列表中选择团队。应用程序将填充“可添加的员工”和“团队员工”表格。应用程序还会启用“添加员工”、“移除员工”和“指派团队”按钮。
3. 单击指派团队。将显示“工作区域签到签退”弹出窗口，其中团队名称已填充。
4. 在签到类型字段中选择一个工作区域。应用程序会显示一个包含可用于所选工作区域的实例列表的字段。
5. 选择一个实例。
6. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
7. 单击签到。“工作区域签到签退”弹出窗口将关闭。应用程序会显示一条消息，表明团队已成功签到该工作区域。随即“签到工作区域”表格中将显示该工作区域

如何使团队从一个或多个工作区域签退

按照以下步骤使团队从一个或多个工作区域签退：

1. 打开管理团队员工页。将显示“管理团队员工”页。
2. 从列表中选择团队。应用程序将填充“可添加的员工”和“团队员工”表格。应用程序还会启用“添加员工”、“移除员工”和“指派团队”按钮。
3. 单击指派团队。将显示“工作区域签到签退”弹出窗口，其中团队名称和“从工作区域签到”表格已填充。
4. 在“从工作区域签到”表格中选中一个或多个工作区的复选框。
5. 单击签退。“工作区域签到签退”弹出窗口将关闭。应用程序会显示一条消息，表明团队已从这些工作区域成功签退。应用程序会将这些工作区域从“签到工作区域”表格中移除。
