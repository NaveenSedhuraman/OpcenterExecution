了解智能扫描

智能扫描在所有门户页面上都可用。通过该功能，在扫描条形码时，应用程序可填充一个或多个字段。例如，扫描容器上的条形码不仅可以填充“容器”字段，还可以填充显示有关该容器信息的任何字段。条形码包含以下一项或多项内容:

• 前导符，表示条形码是否为智能条形码的单个字符。当应用程序检测到前导符时，会将条形码标识为智能条形码，并在读取智能标记之前去除前导符。
• 智能标记，附加到扫描字符串的前缀，表示实体类型；例如，容器。应用程序使用智能标记定位要填充实体名称的字段。
• 实体名称。

智能条形码始终包含实体名称、前导符和智能标记。前导符和智能标记在 Portal Studio SmartScanningSettings 部分定义。有关 Portal Studio 中的设置以及配置前导符和智能标记的信息，请参考《  Opcenter Execution Medical Device and Diagnostics Portal Studio 用户指南》或《  Opcenter Execution Core Portal Studio 用户指南》。

在扫描条形码时

当车间操作员扫描条形码时，应用程序将从 Portal Studio SmartScanningSettings 加载实体名称、前导符和智能标记。如果当前页已启用智能扫描并且扫描的条形码包含智能标记，则应用程序不仅会填充实体名称，还会填充先前已定义了该实体的值的任何从属字段。如果条形码包含智能标记，则条形码被视为智能条码。

注释: 应用程序如何填充页面上的字段取决于该页面是否已启用“智能扫描”以及条形码是否为智能条形码。有关启用智能扫描功能的信息，请参考《  Opcenter Execution Medical Device and Diagnostics Portal Studio 用户指南》或《  Opcenter Execution Core Portal Studio 用户指南》。

光标放置

当在条形码中扫描到终端器（例如，由扫描设备设置的水平制表符）时，应用程序会在填充扫描字段后将光标移动到下一个逻辑字段，无论条形码是否为智能（包含前导符）条形码。手动输入值的从属字段的解析方式与之相同。如果在条形码中没有扫描到终端器，则应用程序不会将光标移动到下一个字段，而条形码类型（智能或非智能）将决定应用程序是否填充该字段。

如果条形码为非智能条形码，则应用程序将在光标所在的字段填充实体名称，并且光标停留在最后一个字符的末尾。如果条形码为智能条形码，当光标焦点从目标字段中移除时，应用程序将填充该字段。无论条形码为什么类型，可以通过扫描设备自动移除焦点（如果已为设备配置了终端器），或者通过输入目标字段手动移除。

注释: 来自扫描设备的终端器必须与 Portal Studio SmartScanningSettings 部分中定义的终端器值相匹配。有关智能扫描设置和定义终端器的信息，请参考《  Opcenter Execution Medical Device and Diagnostics Portal Studio 用户指南》或《  Opcenter Execution Core Portal Studio 用户指南》。有关为扫描设备配置终端器的信息，请参考随设备提供的文档。
