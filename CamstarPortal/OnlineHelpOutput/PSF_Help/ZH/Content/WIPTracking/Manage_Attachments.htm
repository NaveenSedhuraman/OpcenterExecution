管理附件

可以使用“管理附件”页搜索、拆离、附加和查看文档。“管理附件”页可从“附件”菜单中获得。有关从“管理附件”页附加文档的信息，请参考“附加文档”。

还可以使用车间事务的容器信息部分中的“文档集”图标查看附加到容器的文档。有关信息，请参考“容器信息”。

搜索文档

即使所有字段都是可选字段，必须填充其中一个字段以激活“搜索”按钮。

• 对象类型
• 开始时间戳记
• 结束时间戳记
• 文档类型
• 文件扩展名
• 员工

定义文档

单击“管理附件”页上的“附加”时，将显示“附加文档”弹出窗口。有关使用“附加文档”弹出窗口的附加文档信息，请参考“附加文档”。

拆离文档

应用程序会在“拆离文档”弹出窗口的“实例名称”字段中自动填充实例名称。应用程序还会填充正在拆离的文档名称和修订版本信息。

查看文档

必须具有相应的软件才能查看附加的文档。例如，必须具有 Microsoft Word 才能查看 Word 文档。

支持的 3D 图像文件会在单独的“3D 文件查看器”弹出窗口中打开。通过 3D 文件查看器弹出窗口可以旋转、放大和缩小、标记图像和保存 2D 快照。有关查看 3D 图像的信息，请参考《  Opcenter Execution Medical Device and Diagnostics Portal Studio 用户指南》或《  Opcenter Execution Core Portal Studio 用户指南》。

电子工作指令 (EWI) 文件会在单独的弹出窗口中打开，并提示输入 Teamcenter 用户名和密码。要查看 EWI 文件，必须具有 Teamcenter 的访问权限。Teamcenter 会话将保持打开状态，直至浏览器关闭。

默认浏览器安全设置禁止打开不安全网站上的文档。必须更改设置才能打开不安全的内容。

注释: 在浏览器中更改此设置时，您会从应用程序中注销。

如何搜索文档

按照以下步骤搜索文档：

1. 从附件菜单打开管理附件页。将显示“管理附件”页。
2. 至少在一个字段中进行选择。有关字段选择要求的信息，请参考“管理附件页字段定义”。应用程序将激活“搜索”按钮。
3. 单击搜索。应用程序将在“附加的文档”表格中显示返回的项目。

如何拆离文档

按照以下步骤拆离文档：

1. 执行“如何搜索文档”过程。
2. 在“附加的文档”表格中选择行。
3. 单击“拆离”。将显示“拆离文档”弹出窗口，其中填充了“实例名称”和“附加的文档”字段。
4. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
5. 单击“拆离”。应用程序会显示一条消息，表示文档已成功拆离。

如何使用管理附件页查看文档

按照以下步骤使用“管理附件”页查看文档：

1. 执行“如何搜索文档”过程。
2. 在“附加的文档”表格中选择行。
3. 单击查看。 应用程序会提示您打开或保存文件，或者应用程序自动下载文档。（Internet 浏览器决定应用程序是否显示弹出窗口、其外观以及“打开”和“保存”选项。）

如何使用“文档集”图标查看文档

按照以下步骤使用“文档集”图标查看文档：

1. 打开容器事务页。将显示“容器事务”页。
2. 在容器字段中选择或输入容器。 应用程序将更新容器标头和选择字段以显示指定容器的信息。 只有为容器输入了这些值，才会显示相应的值。
3. 单击“文档集”图标。应用程序会显示与容器以及容器制造订单关联的文档列表。
4. 单击要查看的文档。 应用程序会提示您打开或保存文件，或者应用程序自动下载文档。（Internet 浏览器决定应用程序是否显示弹出窗口、其外观以及“打开”和“保存”选项。）
