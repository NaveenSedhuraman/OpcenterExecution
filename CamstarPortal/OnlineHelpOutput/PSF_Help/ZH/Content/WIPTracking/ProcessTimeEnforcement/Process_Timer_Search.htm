流程计时器搜索

可以在“流程计时器搜索”页中查看所有当前流程计时器和历史流程计时器。还可以在该页面中停止流程计时器。

搜索流程计时器

搜索流程计时器时，可以使用多个参数来缩小搜索结果的范围。例如，在“开始的计时器 - 起始时间”字段中输入 01/01/2016 08:00 AM 并在“开始的计时器 - 终止时间”字段中输入 01/01/2016 05:00 PM 可以仅返回已在这两个时间之间开始的计时器。

最初访问“流程计时器搜索”页时，应用程序将显示空的搜索结果表格。必须先指定至少一个参数，“搜索”按钮才会启用。

按“完成状态”过滤只能对已完成的计时器执行。如果已选择“完成状态”准则，则“当前状态”会自动设为已完成。如果当前状态设为“处理中”，则会禁用“完成状态”。

执行搜索会导致应用程序：

• 在搜索结果表格中显示搜索结果。
• 保留搜索参数。

所有搜索结果表格列标题都可排序。默认计时器排序顺序为：

• 距离最大时间的时间 - 从超过最大时间最长时间的计时器到距离最大时间最长时间的计时器。
• 距离最小时间的时间 - 从超过最小时间最长时间的计时器到距离最小时间最长时间的计时器。
• 计时器结束时间 - 最近结束的计时器。

表格数据每五秒自动刷新一次。可以通过在 Portal Studio 中修改表格来指定表格数据的刷新频率。有关信息，请参考《  Opcenter Execution Medical Device and Diagnostics Portal Studio 用户指南》或《  Opcenter Execution Core Portal Studio 用户指南》。

Siemens 还提供了将搜索结果导出为 Excel 或 CSV 格式的能力。

可以使用“全部清除”按钮清除参数和结果。通过清除“流程计时器搜索”页，可以启动新搜索。单击“全部清除”按钮将导致应用程序：

• 清除所有已指定的搜索过滤器。
• 从搜索结果表格中清除计时器。
• 清除“停止计时器”操作按钮。
• 禁用“搜索”按钮。
停止流程计时器

在搜索结果表格中选择当前状态为“处理中”的计时器时，应用程序将显示“停止计时器”按钮。您必须具有停止计时器的权限。

选择一个“处理中”计时器并单击“停止计时器”按钮时，将显示“停止计时器”按钮。输入注释为可选操作。

单击“停止计时器”将关闭弹出窗口，并停止选定的计时器。停止计时器之后，应用程序将显示搜索结果表格的同一个页面。不再满足过滤器准则的已停止计时器不会再显示在搜索结果表格中。

可以使用“撤消上个事务”恢复在“停止计时器”弹出窗口中停止的计时器。有关信息，请参考“撤消上个事务”。

如何搜索流程计时器

按照以下步骤搜索流程计时器：

1. 从搜索菜单打开流程计时器搜索页。将显示“流程计时器搜索”页。
2. 选择或输入搜索准则。
3. 单击搜索。应用程序将在搜索结果表格中显示与搜索准则匹配的流程计时器。

如何停止流程计时器

按照以下步骤停止流程计时器：

1. 执行“如何搜索流程计时器”中所述的过程。应用程序会将满足所输入搜索准则的流程计时器的列表更新到搜索结果表格中。
2. 在搜索结果表格中选择一个计时器。对于处理中计时器，应用程序将显示“停止计时器”操作按钮。您必须具有停止计时器的权限。
3. 单击停止计时器。将显示“停止计时器”弹出窗口。
4. 根据业务要求输入可选注释，然后单击停止。应用程序将停止选定的计时器，计时器结束时间会设为当前系统时间，当前状态会设为“完成”，并且“停止计时器”弹出窗口将关闭。应用程序将更新搜索结果表格和制造审核追踪。
