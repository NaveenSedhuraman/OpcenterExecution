打印容器标签

“打印容器标签”事务允许您手动打印容器的一个或多个标签。系统管理员可以将应用程序设为为任何事务自动打印标签，但可能有需要手动打印标签的情况（例如，在容器上没有要执行的事务，或者没有与所执行的事务关联的自动标签打印）。手动打印容器标签时，应用程序将打印最近执行的事务的标签。

在手动打印容器标签之前

系统管理员必须在建模中定义至少一个打印机标签定义和至少一个打印队列定义，然后才能打印标签。系统管理员可以将打印机标签定义与规范关联，将打印队列定义与资源或作业建模对象关联。有关定义和关联打印机标签和打印队列定义的信息，请参考《  Opcenter Execution Medical Device and Diagnostics Modeling 用户指南》或《  Opcenter Execution Core Modeling 用户指南》。

手动打印容器标签

当只有一个打印机标签定义与容器的当前规范相关联时，应用程序将自动填充“打印机标签定义”字段。如果没有关联任何打印机标签定义，则所有定义都可供选择。如果应用程序默认为打印机标签定义，则可以选择其他定义。

当只有一个打印队列与容器的当前作业或当前处理容器的资源相关联时，应用程序将自动填充“打印队列”字段。如果为作业和资源都指定了打印队列，则 Designer 中指定的公共值表达式 (CVE) 将确定优先级。

打印容器标签页

可以从“容器”菜单中访问“打印容器标签”页。通过“打印容器标签”页，可以打印指定容器或物料箱的一个或多个标签。

注释: 仅当您的组织购买了 Label Printing 模块，并且系统管理员将“容器”菜单配置为包含“打印容器标签”选项时，“打印容器标签”菜单项才可用。如果没有看到“打印容器标签”菜单项，请与系统管理员联系。

如何手动打印容器标签

按照以下步骤手动打印容器标签：

1. 从容器菜单中打开打印容器标签页。将显示“打印容器标签”页。
2. 在容器字段中选择或输入容器。 应用程序将更新容器标头和选择字段以显示指定容器的信息。只有为容器输入了这些值，才会显示相应的值。 此外，应用程序可能在“打印机标签定义”和“打印队列”字段中显示默认值。
3. 选择打印机标签定义。
4. 选择打印队列。
5. 在标签计数字段中输入要打印的标签数。
6. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
7. 单击打印标签或命令栏上的提交。将显示一条确认消息，并且容器标签将在指定的标签打印机上打印。或者，如果指定的打印队列使用“输出到文件”选项，则标签信息写入特定于该打印队列的文本文件。

如何手动打印物料箱标签

按照以下步骤手动打印物料箱标签：

1. 从容器菜单中打开打印容器标签页。将显示“打印容器标签”页。
2. 在物料箱字段中选择或输入物料箱。应用程序更新标题和选择字段以显示指定物料箱的信息。值只有在输入物料箱时才会显示。此外，应用程序可能在“打印机标签定义”和“打印队列”字段中显示默认值。
3. 选择打印机标签定义。
4. 选择打印队列。
5. 在标签计数字段中输入要打印的标签数。
6. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
7. 单击打印标签或命令栏上的提交。将显示一条确认消息，并且物料箱标签将在指定的标签打印机上打印。或者，如果指定的打印队列使用“输出到文件”选项，则标签信息写入特定于该打印队列的文本文件。
