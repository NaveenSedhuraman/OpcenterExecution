当前采样状态更新

使用“当前采样状态更新”页可同时更新特定“采样当前状态”记录或与产品关联的所有“采样当前状态”记录的采样率计数器和采样计划检验级别。可以在“新建采样率计数器值”字段中输入一个值，以更改容器采样的频率。您还可以在“新建检验级别”字段中选择检验级别，以更改采样计划的检验级别。有关定义采样计划的信息，请参考《  Opcenter Execution Medical Device and Diagnostics Modeling 用户指南》或《  Opcenter Execution Core Modeling 用户指南》。

使用当前采样状态更新页

使用“当前采样状态更新”页时，以下规则适用：

• 只有具有基于容器的采样计划的产品才会在“产品”字段中显示。
• 此页上所做的更改将对满足采样要求的下一个容器生效。已在进行中的容器不受影响。
• 此页不适用于批次采样。
当前采样状态更新页

您可以从“容器”菜单访问“当前采样状态更新”页。

如何更新采样率计数器

按照以下步骤更新采样率计数器：

1. 从容器菜单打开当前采样状态更新页。将显示“当前采样状态更新”页。
2. 在产品字段中选择产品。应用程序在“采样当前状态详细信息”表格中显示“采样当前状态”记录的列表。
3. 选中要更新采样率计数器的“采样当前状态”记录的复选框。

或

选中“采样当前状态详细信息”表格的标题行中的复选框，以更新全部采样率计数器。

4. 在“新建采样率计数器值”字段中输入值。
5. 单击更新或命令栏上的提交。应用程序将刷新“采样当前状态详细信息”表格。

如何更新检验级别

按照以下步骤更新检验级别：

1. 从容器菜单打开当前采样状态更新页。将显示“当前采样状态更新”页。
2. 在产品字段中选择产品。应用程序在“采样当前状态详细信息”表格中显示“采样当前状态”记录的列表。
3. 选中要更新检验级别的“采样当前状态”记录的复选框。
4. 或者，选中“采样当前状态详细信息”表格的标题行中的复选框，以更新全部检验级别。
5. 在“新建检验级别”字段中选择检验级别。
6. 单击更新或命令栏上的提交。应用程序将刷新“采样当前状态详细信息”表格。
