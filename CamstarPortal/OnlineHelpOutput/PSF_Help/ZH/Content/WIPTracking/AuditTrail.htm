查看制造审核追踪

审核追踪可用于查看容器、物料箱和事件的事务历史。有关配置制造审核追踪的信息，请参考《  Opcenter Execution Medical Device and Diagnostics Audit Trail 用户指南》或《  Opcenter Execution Core Audit Trail 用户指南》。

访问制造审核追踪

可以通过以下途径访问制造审核追踪：

• 从 Portal 的“搜索”菜单选择制造审核追踪。应用程序即会显示“审核追踪”页。
• 单击“制造审核追踪”命令栏按钮。“制造审核追踪”命令栏按钮位于每个具有容器信息 Web 部件的车间事务页上。应用程序即会在弹出菜单中显示制造审核追踪。

查看事务历史

访问“制造审核追踪”页时，“搜索”选项卡是唯一可见的选项卡。必须在该页的“类型”部分为选定记录类型指定视图和名称才能执行搜索。还可以在“时间范围”和“事务”部分输入信息，进一步细化搜索。

当下列某项条件为 "true" 时，应用程序会自动填充“视图”字段：

• 仅配置了一个视图。
• 为登录的员工指定了默认历史视图。在“员工”建模对象中指定了默认历史视图。有关信息，请参考《  Opcenter Execution Medical Device and Diagnostics Modeling 用户指南》或《  Opcenter Execution Core Modeling 用户指南》。
注释: 如果已配置多个视图，则可更改默认视图。

在“车间”页或“容器搜索”页单击“制造审核追踪”命令栏按钮转到“制造审核追踪”页后，应用程序会自动填充“记录类型”和“名称”字段。

执行搜索后可能会出现其他选项卡，具体取决于选定的记录类型。仅会显示容器和物料箱的“状态”和“属性”选项卡。如果定义了用户界面首选项，则会始终显示“状态”选项卡，但仅当为指定的容器或物料箱提供了用户定义的属性数据时才会显示“属性”选项卡。仅当提供了事件日志数据时，才会显示事件的“事件日志”选项卡。

对容器或物料箱执行初始搜索后，应用程序将在“搜索”选项卡上显示次要过滤器。这些字段中的选择内容取决于初始搜索返回的事务信息。可以使用次要过滤器细化搜索。

注释: 在“车间”页单击“制造审核追踪”命令栏按钮后，应用程序将转到“制造审核追踪”页，此时会自动显示次要过滤器。

有关制造审核追踪和事务的信息，请参考《  Opcenter Execution Medical Device and Diagnostics Audit Trail 用户指南》或《  Opcenter Execution Core Audit Trail 用户指南》。

注释: 包含选项卡的面板可折叠。折叠选项卡面板会在页面上水平展开表格。

注释: 系统管理员可以使用 Audit Trail Configurator 配置选项卡上显示的信息。

制造审核追踪页

通过“制造审核追踪”页，可以查看容器、物料箱或事件的历史。该页面包含以下部分：

• “搜索”选项卡 - 加载页时显示
• “状态”选项卡 - 仅当存在状态数据的情况下，才会在返回初始搜索结果后显示
• “属性”选项卡 - 仅在为容器和物料箱返回初始搜索结果后显示此选项卡，且仅在已为物料箱或容器指派了用户定义的属性时才显示
• “事件日志”选项卡 - 仅在为事件返回初始搜索结果后显示此选项卡，且仅在管理生产和通用事件时在“日志”选项卡上输入了注释后才显示
• 痕迹导航 - 在“历史主线”表格中选择事务后显示，并显示所选事务的可导航历史
• “历史主线”表格 - 在初始搜索后显示，并显示满足搜索准则的指定实体的所有事务
• “事务信息”选项卡 - 在“历史主线”表格中选择事务时显示

如何查看容器、物料箱或事件的事务历史

按照以下步骤查看容器、物料箱或事件的事务历史：

1. 从搜索菜单打开制造审核追踪页。将显示“制造审核追踪”页。
2. 在记录类型字段中选择要搜索的实体类型。
3. 如果需要，从视图字段选择其他视图。
4. 输入实体的名称。
5. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
6. 单击搜索。应用程序将显示指定实体的“状态”选项卡（如果适用）、“属性”选项卡（如果适用）以及“历史主线”表格。
7. 在“历史主线”表格中选择一个事务。应用程序将填充痕迹导航追踪，并在“历史主线”表格下方显示适用的事务表格。
8. 在痕迹导航栏中选择事务，可返回到以前选择的事务（如果适用）。
9. 如果要查看与容器或事件关联的文档的详细信息，则执行“如何查看容器或事件的文档”程序。
