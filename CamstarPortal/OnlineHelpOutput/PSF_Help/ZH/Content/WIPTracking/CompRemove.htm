移除组件

“组件移除”事务使您能够移除已向容器分发的组件（产品和替代产品）。同时使用该事务对移除组件另行部署。

移除组件

在选定容器之前，“组件移除”页上的所有显示字段都为空。选择容器时，应用程序将更新标头，并使用向容器分发的所有组件填充“物料分发”表格。

一次只能移除一个组件。移除事务成功时，应用程序将更新“物料分发”表格以移除组件。如果事务失败，应用程序将相应地显示错误消息。

在“要选择的扫描容器/产品”字段中扫描或输入容器、产品或替代产品时，应用程序将标识并选择“物料分发”表格中的容器或产品。如果“物料分发”表格中没有与您输入的容器或产品匹配的容器或产品，则应用程序将显示一条错误消息。

应用程序要求您扫描或输入容器（批次）和容器（序列）的分发类型的有效容器。此外，如果选择使用“目标位置/库存点”，则必须与指定位置的工厂关联。移除组件时指定的位置不必是它所源自的位置。

扫描与手动输入

应用程序会填充用户输入字段，并在扫描容器或产品时自动验证物料。如果不需要进行任何更改或完成其他字段，则可以单击“移除”。

注释: 只有为设备和在 Portal Studio 中配置了终端器，应用程序才会在扫描时自动填充字段并验证物料。否则，在应用程序填充字段并验证物料之前，必须按 Tab 离开“要选择的扫描容器/产品”字段。有关为扫描设备配置终端器的信息，请参考随设备提供的文档。有关在 Portal Studio 中配置终端器的信息，请参考《  Opcenter Execution Medical Device and Diagnostics 系统管理指南》或《  Opcenter Execution Core 系统管理指南》。

在应用程序验证字段并显示其他数据输入字段以及验证容器或产品之前，必须在手动输入后离开“要选择的扫描容器/产品”字段。

使用物料分发表格

“物料分发”表格中的选择决定了在物料分发表格下显示的用户数据输入字段及其可用性。例如，如果选择分发控制类型为容器（序列）的组件，应用程序将选择“移除所有数量”复选框，使用“要分发的剩余数量”值填充“移除数量”字段，然后禁用“移除数量”字段。

注释: 如果为任何其他所选组件手动选中“移除所有数量”复选框，则应用程序行为是相同的。

将受影响容器添加到事件中

在与一个或多个质量事件关联的容器上执行“组件移除”事务时，应用程序可以将容器自动添加到质量事件中。例如，如果源容器与事件关联，则应用程序会将组件容器自动添加到将组件移除到的质量事件中。系统管理员配置“组件移除”事务，以便将容器添加到建模中的事件。有关将受影响容器自动添加到质量事件的信息，请参考《  Opcenter Execution Medical Device and Diagnostics Event Management 用户指南》或《  Opcenter Execution Core Event Management 用户指南》，以及《  Opcenter Execution Medical Device and Diagnostics Nonconformance Management 用户指南》或《  Opcenter Execution Core Nonconformance Management 用户指南》。

组件移除页

可以从“容器”菜单中访问“组件移除”页。通过“组件移除”页，可以查看向容器分发的组件，并从容器中移除这些组件。

注释: 如果组件移除事务已配置为另一页（例如，容器搜索）上的可用操作，则组件移除页也可用作弹出窗口。

如何移除已分发组件

按照以下步骤移除已发出零部件：

1. 从容器菜单打开组件移除页。将显示“组件移除”页。
2. 在容器字段中选择或输入容器。 应用程序将更新容器标头和选择字段以显示指定容器的信息，并显示向容器分发的所有组件。
注释: 从另一个车间页访问“组件移除”弹出窗口时，应用程序将自动填充容器字段和容器标头。
3. 扫描或输入要从选定容器中移除的容器、产品或替代产品。应用程序在“物料分发”表格中选择组件，并相应地更新用户数据输入字段。

或

选择想要在“物料分发”表格中移除的组件。应用程序在“物料分发”表格中选择组件，并相应地更新用户数据输入字段。

4. 选择移除原因。
5. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
6. 单击移除或命令栏上的提交。应用程序会从容器中移除组件，显示成功消息，并从“物料分发”表格中移除该组件。
7. 重复步骤 3-6，移除其他组件。
