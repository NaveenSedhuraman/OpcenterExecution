如何为选定的制造订单启动容器

按照以下步骤为所选制造订单启动容器：

1. 从容器菜单打开订单调度页。将显示“订单调度”页，其中的表格显示了当前制造订单的列表。
2. 选择要处理的制造订单。表格下的“制造订单”、“产品”和“产品描述”字段默认为与选定制造订单关联的信息。
3. 单击“将文档附加到此制造订单实例”图标，以将文档附加到制造订单。将显示“附加文档”弹出窗口。
4. 如果要将文档附加到制造订单，请参考相应的文档附加程序：
• “如何将不会重用的新文档附加到制造订单”
• “如何将会重用的新文档附加到制造订单”
• “如何将现有文档附加到制造订单”
5. 输入要在容器字段中启动的容器名称。

或

单击自动生成名称复选框。

6. 选择级别。
7. 选择所有者。
8. 在“启动原因”字段中选择启动此容器的原因。
9. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
10. 如果该部分未展开，单击“附加文档”控件以显示容器附件字段。
11. 如果要将文档附加到新容器，请参考相应的文档附加程序：
• “如何将不会重用的新文档附加到新容器”
• “如何将会重用的新文档附加到新容器”
• “如何将现有文档附加到新容器”
12. 单击启动或命令栏上的提交。应用程序显示一条确认容器的状态消息，并使用正在处理的数量更新“处理中数量”列。如果已履行订单数量，则应用程序从列表中移除制造订单。
13. 如果需要针对制造订单启动其他容器以履行数量，则重复步骤 5-8 直到数量已履行为止。完成此操作后，制造订单将从列表中移除。

如何将会重用的新文档附加到新容器

按照以下步骤将会重用的新文档附加到新容器：

1. 执行“如何为选定的制造订单启动容器”程序。
2. 在附加文档部分选择附加将会重用的新文档（必须指定修订版本）。将显示“文档路径”、“文档名称”、“修订版本”、“记录修订版本”和“文档描述”字段。
3. 单击文件上传按钮以选择文件。
4. 在文档名称字段中输入文档名称。
5. 在修订版本字段中输入此文档的修订版本。
6. 也可选中修订版本旁边的框以表示此文档为记录修订版本。
7. 根据您的业务需求输入可选信息。有关可选字段的信息，请参考字段定义表。
8. 单击启动或命令栏上的提交。应用程序将显示一条关于已成功启动容器的消息。

如何将现有文档附加到新容器

按照以下步骤将现有文档附加到新容器：

1. 执行“如何为选定的制造订单启动容器”程序。
2. 在附加文档部分选择附加现有文档（从下面选择）。将显示“文档实例”、“修订版本”和“记录修订版本”字段以及“查看文档”按钮。
3. 在文档实例字段中选择文档。
4. 输入具体修订版本。

或

选中记录修订版本复选框。

5. 如果要在附加之前查看文档，单击查看文档。 应用程序会提示您打开或保存文件，或者应用程序自动下载文档。（Internet 浏览器决定应用程序是否显示弹出窗口、其外观以及“打开”和“保存”选项。）
6. 单击启动或命令栏上的提交。应用程序将显示一条关于已成功启动容器的消息。
