SPC Tester

Siemens provides an SPC Tester that allows you to test the sample charts and data included with the application. It also allows you to test your setup after you have configured charts and required components.

Siemens provides sample charts and data that can be loaded through the Management Studio.

When Using the SPC Tester Page

You must select either an SPC Chart, an SPC Chart Group, or a Data Collection. All other fields are optional.

Test data can be loaded through the DB update in Management Studio. Siemens recommends you only load this data in a development environment, and not in your production environment.

Sample macros and sample charts point to the data loaded from the DB update. Paths in the sample charts point to default install locations. There you will need to update the paths if you did not use the default installation locations.

You must also update sample macros with authentication info for your ODBC before you use tester page.

UI Parameters you enter on this page will override those set in the SPC Chart object.

Attempt OCAPs on Out-of-Control Charts

Rows can be added to enter parameters to pass through to the macro creating the chart.

Optional

How to Use the SPC Tester Page

Follow these steps to use the SPC Tester page:

1. Select SPC Tester from the SPC menu. The SPC Tester page appears.
2. Select an SPC chart, an SPC chart group, or a Data Collection. Only one is required.
3. Optionally select one or more fields to further define what charts you want to generate.
4. Click Generate Chart(s). The selected chart(s) are displayed.
