Close Containers

The Close transaction allows you to close eligible parent or child containers. Closing a parent container closes all associated child containers automatically. However, you can close a child container without closing the parent. All active containers can be closed. Closing a container stops all active process timers associated with that container.

Note: Refer to "Close Multiple Containers" if you want to close multiple containers simultaneously.

When Closing Containers

All display fields are empty on the Close page until a container is selected. Display fields are populated automatically on the Close pop-up. Refer to "Using Transaction Pages and Pop-ups" for information on performing a transaction from a page or pop-up.

Containers with a status of Closed are not available for selection when selecting a container on the Close page. If you scan or enter a container with a status of Closed, or enter a container that does not exist, the application displays an error message.

Optionally, you can specify a change status reason and enter comments when closing a container. Change status reasons are defined in Modeling by a system administrator. Refer to the Opcenter Execution Medical Device and Diagnostics Modeling User Guide or the Opcenter Execution Core Modeling User Guide for information.

Closed containers are eligible for archiving, but are not eligible for additional transactions except comments, data collection, and transaction reversal. By default, closed containers usually are not available for selection in other fields and grids on other pages.

The application checks for active process timers when the Close transaction is used to close a container. Any timers present are stopped and the Close transaction in the manufacturing audit trail for the container displays the Process Timer History detail for any timers that are stopped.

Close Page

You can access the Close page from the Container menu. The Close page allows you to scan or enter a container to close.

Note: The Close page may be also available as a pop-up if the container close transaction has been configured as an available action on another page (for example, Container Search).

How to Close a Container

Follow these steps to close a container:

1. Open the Close page from the Container menu. The Close page appears.
2. Select or enter a container in the Container field. The application updates the container header and selection fields to display information for the specified container. Values appear only if they were entered for the container.
Note: The application populates the Container field and the container header automatically when you access the Close pop-up from another shop floor page.
3. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
4. Click Close or Submit on the command bar. The application displays a status message confirming the container has been closed.
