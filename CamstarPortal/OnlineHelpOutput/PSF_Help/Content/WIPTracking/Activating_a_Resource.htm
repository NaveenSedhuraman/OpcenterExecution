Activating a Resource

The Resource Activation transaction enables you to activate a maintenance requirement for one or more resources or carriers, regardless of whether they belong to a maintenance class.

When Activating and Deactivating a Resource

The Resource Group Filter enables you to limit the number of resources and carriers displayed in the Resources grid.

A check next to a resource in the grid indicates the specified requirement is active for the resource. You can clear the check box to deactivate a requirement for a resource.

The activation transaction will fail for resources that have been activated already as part of a maintenance class. The Class Activated column in the Resources grid indicates whether the resource has been activated as part of a maintenance class. Refer to "Maintenance Class Activation" for information on activating and deactivating maintenance classes.

How to Activate a Maintenance Requirement for Resources or Carriers

Follow these steps to activate a maintenance requirement for resources or carriers:

1. Open the Resource Activation page from the Resource menu. The Resource Activation page appears.
2. Select a maintenance requirement from the Maintenance Req field. The Resources grid displays all defined resources and carriers.
3. Select a resource group from the Resource Group Filter field to limit the number of resources displayed. The grid is updated to display only the resources or carriers belonging to the specified resource group.
4. Select the Activate check box for each resource you want to activate.

Or

Clear the Activate check box for each resource you want to deactivate.

5. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
6. Click Submit. The maintenance requirement is activated for the selected resources.
