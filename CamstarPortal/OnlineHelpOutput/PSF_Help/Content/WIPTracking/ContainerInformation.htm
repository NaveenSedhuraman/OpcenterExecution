Container Information

Information about the selected container appears at the top of container-related shop floor transaction pages and pop-ups. Information for the selected container is also accessed from the command bar. Typically, this information includes the relevant product, quantity, operation, and manufacturing order. This area also includes any active process timers.

Note: Container information appears for single containers only. Multi-container transaction pages and pop-ups do not include the container information section.

When Viewing Container Information on a Transaction Page

The application requires you to scan or enter a container name in the Container field on transaction pages. All other fields are display-only. The application populates the display-only container information fields when you tab out of the Container field.

Specifying a container on a transaction page causes the application to:

• Display an icon indicating the status of the container
• Display any product, quantity, operation, and manufacturing order information, if provided
• Display any active process timers

Methods for Specifying a Container

Siemens provides three ways to specify a container on a transaction page:

• Scan the container's barcode
• Enter the container name in the Container field
• Select the container from the Container field

When Viewing Container Information on a Transaction Pop-up

The application populates the container name and container information automatically when you launch a transaction pop-up. All fields are display-only. The application populates the container information based on the container selected on the page from which you launched the pop-up.

Note: An ellipsis appears next to the field name and a tooltip shows the remaining text if a field contains information that exceeds the field's length.

Container Status

The container Status icon appears on transaction pages only and its appearance indicates the status of the container. Possible statuses include:

Container is active

Container is on hold

Container is closed

Note: Additional statuses are configurable by a system administrator.

Timers Field

The Timers field appears on both transaction pages and pop-ups if there are one or more active timers associated with the container. The most critical timer appears in the field. Clicking the triangle next to the Timers field label opens the Active Timers flyout and shows all active timers. Refer to "Viewing Process Timers" for information.
