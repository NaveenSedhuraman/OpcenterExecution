Electronic Signatures

Electronic Signatures is a separately licensed module for Opcenter Execution Medical Device and Diagnostics or Opcenter Execution Core. This functionality is available only if your organization has purchased the module.

An electronic signature (ESignature) is an electronically stored record that serves as the equivalent of a handwritten signature. An ESignature:

• May be required to complete some shop floor transactions (A transaction will fail without the required electronic signature.)
• Denotes accountability for a transaction
• Is tracked through an audit trail

Using this module ensures that signature requirements are met for each prescribed shop floor transaction by comparing the captured electronic signatures with a predefined electronic signature requirement. The application confirms that the user signatures, number of signatures, and level of authority are met before the shop floor transaction is processed.
