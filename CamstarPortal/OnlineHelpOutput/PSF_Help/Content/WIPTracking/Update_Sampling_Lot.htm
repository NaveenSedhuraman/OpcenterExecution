Update Sampling Lot

You can use the Update Sampling Lot page to update or view an existing sampling lot. You can change the sampling lot quantity or sample rate, view the containers associated with the sampling lot, or view lot sampling details.

When Updating a Sampling Lot

If an electronic signature requirement was defined in Modeling for a sampling lot on a factory, an ESignature pop-up appears when executing the transaction for the instance. Refer to the Opcenter Execution Medical Device and Diagnostics Modeling User Guide or the Opcenter Execution Core Modeling User Guide for information.

Update Sampling Lot Page

You can access the Update Sampling Lot page from the Container menu.

How to Update a Sampling Lot

Follow these steps to update a sampling lot:

1. Open the Update Sampling Lot page from the Container menu. The Update Sampling Lot page appears.
2. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
3. Click Update or Submit on the command bar. A status message appears, confirming that the sampling lot has been updated.
