Accessing Transaction Pages and Pop-ups

Single container, multiple container, and resource-related transaction pages can be accessed using the Portal menu. Single container and multiple container transaction pop-ups can be accessed from the Operational View and Container Search pages.

Note: The Operational View and Container Search pages are the only pages configured for transaction pop-ups by default. Your Portal administrator can configure additional pages to display transaction pop-ups based on your organization's business needs.

The procedures below describe accessing transaction pages and pop-ups. Refer to the Opcenter Execution Medical Device and Diagnostics Getting Started User Guide or the Opcenter Execution Core Getting Started User Guide for information on common features and functionality in the application.

How to Access a Transaction Page

Follow these steps to access a transaction page:

1. Log on to the Opcenter Execution Medical Device and Diagnostics.

Or

Log on to the Opcenter Execution Core.

2. What type of transaction do you want to perform?

If you want to perform a . . .

Then . . .

Single container transaction
a. Click Container. The Container menu appears.
b. Select a single container transaction. The transaction page appears. You must select a container to have the application populate the container information section and enable the transaction button.

Multiple container transaction
a. Click Container. The Container menu appears.
b. Select a multiple container transaction. The transaction page appears. You must enter and select at least one container to perform the transaction.

Resource transaction
a. Click Resource. The Resource menu appears.
b. Select a resource transaction. The transaction page appears. You must select a resource to have the application populate the resource information and enable the transaction button.

How to Access a Container Transaction Pop-up

Follow these steps to access a container transaction pop-up:

1. Access a transaction page (Operational View or Container Search) for which actions have been configured.
2. Which page are you viewing?

If you are viewing the . . .

Then . . .

Operational View page
a. Select an In Queue or In Process container. The application enables the Shop Floor Txns command bar button.
b. Go to step 3.

Container Search page
a. Enter or select one or more parameters to narrow your search. The application enables the Search button.
b. Click Search. The application displays containers matching your search criteria.
c. Select the containers for which you want to perform a transaction. The application enables the Shop Floor Txns command bar button.
d. Go to step 3.
3. Click Shop Floor Txns on the command bar. The application displays a panel with all available shop floor transactions for the selected containers.
4. Click the transaction you want to perform. The transaction pop-up appears with the container information fields populated automatically.
Note: The container information section appears for single-container transactions only.
