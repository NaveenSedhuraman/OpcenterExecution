Performing Maintenance Management

The Maintenance Management transaction provides search-filtering capabilities, enabling you to specify search criteria for resources with maintenance requirements that are ready for completion.

The Search Results grid displays the results of your resource search. In addition to the expected search results, the search function also:

• Returns all Thruput Requirements when you specify Remaining Days filter criteria for a Date or Recurring Date Requirement.
• Returns all Date and Recurring Date Requirements when you specify Remaining Qty Percent, Remaining Qty2 Percent, Qty Used, or Qty2 Used filter criteria for a Thruput Requirement.

When you select a row in the Search Results grid, the corresponding fields in the Maintenance Details section are populated automatically. All Maintenance Details fields are read-only except the Comments field.

The resource audit trail provides a complete history of all transactions for a resource. Refer to "Viewing the Resource Audit Trail" for information.

Viewing Associated Documents

The View Documents icon appears on the page only when there are documents associated with the selected resource. Clicking this icon displays the View Documents pop-up, which lists links to the associated documents. Each link includes a document description, revision, and whether the revision is the current revision of record.

The View Documents icon is disabled when there are no associated documents.

Collecting Data

Data collection fields appear at the bottom of the Maintenance Management page if a data collection definition (user-defined or parametric) was specified on the maintenance requirement. Scroll to the bottom of the page to view data collection fields.

Use the Maintenance Management page to record as Complete the required maintenance for a resource. All fields are either optional or display-only. There are no required fields.

How to Complete Required Maintenance for a Resource

Follow these steps to complete required maintenance for a resource:

1. Open the Maintenance Management page from the Resource menu. The Maintenance Management page appears.
2. Enter values in the Enter Search Criteria fields to limit the display of resources.
3. Click Search. The Search Results grid displays maintenance status information for the resources that match the search criteria entered. If no search criteria is entered, then all possible results are displayed.
4. Click the row for the resource that has a maintenance requirement that is either in the Warning period (Pending), is Due, or is Past Due. Details of the maintenance requirement appear in the display only Maintenance Details fields.
5. Click the View Documents icon if enabled and if you want to view a list of documents associated with the selected resource. The View Document pop-up appears.
6. Click the associated link to view a document. When finished, click Close to close the View Documents pop-up.
7. Click the Show additional fields control if you want to enter comments specific to this maintenance requirement.
8. Click Complete.
