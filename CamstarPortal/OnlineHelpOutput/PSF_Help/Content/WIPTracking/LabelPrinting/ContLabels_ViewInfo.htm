Viewing Detail Information for Printed Labels

History is recorded for every label, printed or reprinted. If configured, the manufacturing audit trail uses this history to provide detailed information on each printed and reprinted label. The information includes the label count, printer label definition, print queue, tag name, and tag value.

The procedures to configure a manufacturing audit trail are in the Opcenter Execution Medical Device and Diagnostics Audit Trail User Guide or the Opcenter Execution Core Audit Trail User Guide.

How to View Information about a Printed or Reprinted Label

Follow these steps to view information about a printed or reprinted label:

1. Open the Mfg Audit Trail page from the Search menu. The Mfg Audit Trail page appears.
Note: The name of the manufacturing audit trail page depends on how it was configured by a system administrator.
2. Select the Record Type, if necessary.
3. Enter the Container Record name for which one or more labels have been printed.
4. Click Search. The History Mainline grid lists a history of the transactions executed against this container.
5. Click the row in the grid that contains the transaction whose details you want to display. A Label Printing button appears if labels have been printed for that transaction.
6. Click Label Printing or Submit on the command bar. The information appears for any labels that were printed or reprinted for the selected transaction.
