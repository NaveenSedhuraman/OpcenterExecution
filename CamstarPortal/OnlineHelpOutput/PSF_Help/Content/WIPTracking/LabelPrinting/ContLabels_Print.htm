Print Container Labels

The Print Container Label transaction allows you to manually print one or more labels for a container. A system administrator can set up the application to print labels automatically for any transaction, but there may be situations where you need to print labels manually (for example, there are no transactions to be performed on the container or there is no automatic label printing associated with the transaction you performed). When printing a container label manually, the application prints a label for the most recent transaction performed.

Before Printing Container Labels Manually

A system administrator must define at least one printer label definition and at least one print queue definition in Modeling before you can print labels. The system administrator can associate a printer label definition with a spec and print queue definitions with the resource or operation modeling objects. Refer to the Opcenter Execution Medical Device and Diagnostics Modeling User Guide or the Opcenter Execution Core Modeling User Guide for information on defining and associating printer label and print queue definitions.

When Printing Container Labels Manually

The application populates the Printer Label Definition field automatically when only one printer label definition has been associated with the container's current spec. If no printer label definitions have been associated, all definitions are available for selection. If the application defaults to a printer label definition, you have the option to select a different definition.

The application populates the Print Queue field automatically when only one print queue has been associated with the container's current operation or the resource processing the container currently. If a print queue is specified for both operation and resource, the common value expression (CVE) specified in Designer determines precedence.

Print Container Label Page

You can access the Print Container Label page from the Container menu. The Print Container Label page allows you to print one or more labels for the specified container or carrier.

Note: The Print Container Label menu item is available only if your organization purchased the Label Printing module and a system administrator configured the Container menu to include the Print Container Label option. If you do not see the Print Container Label menu item, contact your system administrator.

How to Print a Container Label Manually

Follow these steps to print a container label manually:

1. Open the Print Container Label page from the Container menu. The Print Container Label page appears.
2. Select or enter a container in the Container field. The application updates the container header and selection fields to display information for the specified container.Values appear only if they were entered for the container. Additionally, the application may display default values in the Printer Label Definition and Print Queue fields.
3. Select a Printer Label Definition.
4. Select a Print Queue.
5. Enter the number of labels you want to print in the Label Count field.
6. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
7. Click Print Label or Submit on the command bar. A confirmation message appears and the container label prints at the designated label printer. Or, if the designated print queue uses the Output to File option, the label information is written to the text file specified for that print queue.

How to Print a Carrier Label Manually

Follow these steps to print a carrier label manually:

1. Open the Print Container Label page from the Container menu. The Print Container Label page appears.
2. Select or enter a carrier in the Carrier field. The application updates the header and selection fields to display information for the specified carrier. Values appear only if they were entered for the carrier. Additionally, the application may display default values in the Printer Label Definition and Print Queue fields.
3. Select a Printer Label Definition.
4. Select a Print Queue.
5. Enter the number of labels you want to print in the Label Count field.
6. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
7. Click Print Labelor Submit on the command bar. A confirmation message appears and the carrier label prints at the designated label printer. Or, if the designated print queue uses the Output to File option, the label information is written to the text file specified for that print queue.
