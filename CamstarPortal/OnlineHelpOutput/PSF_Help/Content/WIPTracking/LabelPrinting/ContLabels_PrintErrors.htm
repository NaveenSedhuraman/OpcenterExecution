Handling Print Errors

There are two types of printer errors:

• Errors that occur during automatic processing
• Errors that occur during manual printing and reprinting

The application logs printer errors to the event log. The administrator monitoring the event log will take the appropriate action, such as printer repair.

Errors That Occur During Automatic Printing

When you submit a transaction that will result in automatic label printing, the success or failure of the label printing does not impact the ability of the transaction to execute successfully. If the transaction is executed successfully, but label printing fails, the application displays a printing error message in addition to the transaction success message.

Errors That Occur During Manual Printing and Reprinting

The application displays an error message when you print or reprint a label manually and label printing fails. Depending on your organization’s business practices, you should either place the container on hold or perform the Print or Reprint transaction again, specifying a different print queue.
