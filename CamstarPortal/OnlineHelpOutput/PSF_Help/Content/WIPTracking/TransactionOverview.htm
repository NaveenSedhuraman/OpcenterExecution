Transaction Overview

You perform Manufacturing Shop Floor transactions to record container and resource activity on the shop floor. Each transaction performs a unique function and records specific information in the Manufacturing database. This transaction information is then available for status tracking and performance analysis. You execute transactions using the application or custom client applications.

This diagram shows some common transactions used on the shop floor.

Container Scanning

Siemens facilitates shop floor transactions by providing the ability for shop floor operators to scan in container information (for example, container name, named object name, and so on). Refer to "Smart Scanning" for information on scanning containers. Refer to the Opcenter Execution Medical Device and Diagnostics System Administration Guide or Opcenter Execution Core System Administration Guide for information on enabling container scanning.
