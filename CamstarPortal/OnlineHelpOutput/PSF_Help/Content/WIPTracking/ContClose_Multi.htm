Close Multiple Containers

The Close (Multiple) transaction enables you to close multiple eligible parent or child containers simultaneously. Closing a parent container closes all associated child containers automatically; however, you can close a child container without closing the parent. All active containers can be closed. Closing a container stops all active process timers associated with that container.

When Closing Multiple Containers

You have the option to specify a change status reason for closing the containers when closing multiple containers. Change status reasons are defined in Modeling by a system administrator. Refer to the Opcenter Execution Medical Device and Diagnostics Modeling User Guide or the Opcenter Execution Core Modeling User Guide for information on status reasons.

The application validates that the specified containers are open and have an active or in transit status.

The application checks for active process timers on all child containers when the Close Multiple transaction is used to close a parent container. Any timers present are stopped and the Close transaction in the manufacturing audit trail for the child container displays the Process Timer History detail for any timers that are stopped.

Refer to "Multi-Container Transactions Overview" for information on how the application functions when executing transactions on multiple containers.

Close (Multiple) Page

You can access the Close (Multiple) page from the Container menu. You can access the Close (Multiple) pop-up from another shop floor page (for example, Container Search) if the Close (Multiple) action has been configured for that page. Both the Close (Multiple) page and pop-up allow you to scan or enter one or more containers to close.

The page and pop-up are identical in appearance and functionality with one exception. The application populates the Close (Multiple) pop-up automatically with the containers you selected on the shop floor page from which you accessed the pop-up.

How to Close Multiple Containers

Follow these steps to close multiple containers:

1. Open the Close (Multiple) page from the Container menu. The Close (Multiple) page appears.
2. Are you viewing the Close (Multiple) page or the Close (Multiple) pop-up?

If you are viewing the . . .

Then . . .

Close (Multiple) page

The Containers to Close grid is empty. Go to step 3.

Close (Multiple) pop-up

The Containers to Close grid is populated with the containers you selected on the page from which you launched the pop-up. If you want to add additional containers to close, go to step 3. Otherwise, go to step 6.
3. Scan or enter a container in the Enter Container to Select field.
4. Press Enter or tab out of the field. The container is added to the Containers to Close grid and selects it automatically.
5. Repeat steps 3-4 to add additional containers.
6. Ensure the selected containers are the ones you want to close.
7. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
8. Click Close Selected or Submit on the command bar. The application displays a status icon for each container indicating whether the transaction was executed successfully.
9. Close the pop-up, if you are viewing it, and return to the previous page.
