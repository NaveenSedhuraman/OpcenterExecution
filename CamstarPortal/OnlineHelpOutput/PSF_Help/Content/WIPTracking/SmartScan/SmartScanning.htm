Smart Scanning

The Smart Scanning feature allows a shop floor operator to enter information on certain shop floor pages by scanning a barcode. For example, when an operator scans a container barcode on the Move In page, the application can populate the Container field and the header fields for which information was entered during earlier transactions.

The application parses the barcode and then populates the appropriate field on the page with that information. Scanning a barcode produces the same results as typing a value in a field; however, the scanning feature minimizes data entry time and the chance for typing errors.

Smart scanning is available on all Portal pages, but is disabled by default. Refer to the Opcenter Execution Medical Device and Diagnostics Portal Studio User Guide or the Opcenter Execution Core Portal Studio User Guide for information on enabling the smart scanning feature.

Additionally, you can customize smart scanning functionality on the Portal pages by:

• Adding additional scannable entities (fields and text boxes) to the pages
Note: Scanning is possible for grids only if externally bound controls are used.
• Enabling smart scanning on other pages
• Disabling smart scanning for individual pages
