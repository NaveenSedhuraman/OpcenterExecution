Managing Attachments

You can search for, detach, attach, and view documents using the Manage Attachments page. The Manage Attachments page is available from the Attachments menu. Refer to "Attaching Documents" for information on attaching documents from the Manage Attachments page.

You can also view documents attached to a container using the Document Set icon in the container information section of the shop floor transactions. Refer to "Container Information" for information.

When Searching for Documents

You must populate one of these fields to activate the Search button even though all the fields are optional.

• Object Type
• From Timestamp
• To Timestamp
• Document Type
• File Extension
• Employee

When Attaching a Document

The Attach Document pop-up appears when you click Attach on the Manage Attachments page. Refer to "Attaching Documents" for information on attaching documents using the Attach Document pop-up.

When Detaching a Document

The application automatically populates the instance name in the Instance Name field on the Detach Document pop-up. The application also populates the name and revision information of the document being detached.

When Viewing a Document

You must have the appropriate software to view an attached document. For example, you must have Microsoft Word to view a Word document.

Supported 3D image files open in a separate 3D File Viewer pop-up. The 3D File Viewer pop-up allows you to rotate, zoom in and out, markup the image, and save a 2D snapshot. Refer to the Opcenter Execution Medical Device and Diagnostics Portal Studio User Guide or the Opcenter Execution Core Portal Studio User Guide for information on viewing 3D images.

Electronic Work Instruction (EWI) files open in a separate pop-up that prompts you for your Teamcenter user name and password. You must have access to Teamcenter to view EWI files. Your Teamcenter session will remain open until your browser is closed.

Default browser security settings prevent documents on unsecured websites from opening. You must change your settings to allow unsecured content.

Note: You are logged out of the application when you change this setting in the browser.

How to Search for a Document

Follow these steps to search for a document:

1. Open the Manage Attachments page from the Attachments menu. The Manage Attachments page appears.
2. Make a selection in at least one field. Refer to "Manage Attachments Page Field Definitions" for information on field selection requirements. The application activates the Search button.
3. Click Search. The application displays the returned items in the Attached Documents grid.

How to Detach a Document

Follow these steps to detach a document:

1. Perform the "How to Search for a Document" procedure.
2. Select a single row in the Attached Documents grid.
3. Click Detach. The Detach Document pop-up appears with the Instance Name and Attached Document fields populated.
4. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
5. Click Detach. The application displays a message indicating that the document has been detached successfully.

How to View a Document Using the Manage Attachments Page

Follow these steps to view a document using the Manage Attachments page:

1. Perform the "How to Search for a Document" procedure.
2. Select a single row in the Attached Documents grid.
3. Click View. The application prompts you to open or save the file, or it automatically downloads the document. (Your Internet browser determines whether the application displays a pop-up, its appearance, and the wording of the Open and Save options.)

How to View a Document Using the Document Set Icon

Follow these steps to view a document using the Document Set icon:

1. Open a container transaction page. The container transaction page appears.
2. Select or enter a container in the Container field. The application updates the container header and selection fields to display information for the specified container. Values appear only if they were entered for the container.
3. Click the Document Set icon. The application displays a list of the documents associated with the container and with the container's mfg order.
4. Click a document to view. The application prompts you to open or save the file, or it automatically downloads the document. (Your Internet browser determines whether the application displays a pop-up, its appearance, and the wording of the Open and Save options.)
