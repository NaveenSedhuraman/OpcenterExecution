Create Sampling Lot

You can use the Create Sampling Lot page to set up a sampling lot to allow sampling based on a group of containers rather than individual ones. The lot's quantity is used to determine the number of samples required for sampling. Container sampling data is rolled up to the sampling lot to determine if the lot passes inspection. Lot-based sampling is useful in determining if an entire manufacturing order passes inspection. It is also useful at incoming inspection when sampling spans multiple containers.

When Using the Create Sampling Lot Page

If an electronic signature requirement was defined in Modeling for the Create Sampling Lot transaction, an ESignature pop-up appears when executing the transaction for the instance. Refer to the Opcenter Execution Medical Device and Diagnostics Modeling User Guide or the Opcenter Execution Core Modeling User Guide for information.

How to Create a Sampling Lot

Follow these steps to create a sampling lot:

1. Open the Create Sampling Lot page from the Container menu. The Create Sampling Lot page appears.
2. Enter a name in the Lot Name field.
3. Enter a value in the Qty field.
4. Select a product in the Product field.
5. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
6. Click Create Lot or Submit on the command bar. A status message appears, confirming that the sampling lot has been created.
