Resource Information

Information about the selected resource appears at the top of every resource-related shop floor transaction page. You can select a resource group to limit the number of resources available for selection. The application displays this information when you select a resource that was setup previously:

• Icon indicating the availability (Up or Down) of the resource
• Time at that status
• Status Reason, if provided
• Status Code, if provided

The application populates the Status Reason and Status Code fields only if the user that set up the resource provided that information. The Time at Status as well as the Status Reason and Status Code fields are blank if the specified resource has not been set up.

Resource Information Field and Icon Definitions

This table defines the resource information fields and icons on a resource page.

Field

Definition

Type

Resource Group

Restricts the list of displayed resources or carriers by the Resource Group or Carrier Group to which they belong. Resource groups are defined in Modeling.

Optional

Resource

Machine, workcell, fixture, carrier, etc. (being tracked and currently updated) to be configured. Displays all resources and carriers unless filtered by the Resource Group.

Required

<availability icon>

Icon indicating the status of the resource. The appearance of the icon depends on the resource's status.

Possible statuses include:

• Up, the resource is available
• Down, the resource is not available
• Requires setup, the resource has not been set up and is not available

Display Only

Time at Status

Length of time the resource has been at the current status (Up or Down).

Display Only

Status Reason

Additional detail describing the reason for the current status of the resource. Status reasons are defined in Modeling.

Display Only

Status Code

Description of the current availability for a resource. Status codes are defined in Modeling.

Display Only
Note: An ellipsis appears next to the field name and a tooltip shows the remaining text if a resource information field contains information that exceeds the field's length.
