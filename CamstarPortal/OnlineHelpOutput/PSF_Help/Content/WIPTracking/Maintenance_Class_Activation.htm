Maintenance Class Activation

The Maintenance Class Activation transaction enables you to activate a maintenance requirement for all resources or carriers that belong to a maintenance class. You can activate the requirement for more than one maintenance class at a time.

When Activating a Maintenance Requirement

The Resources subgrid on the Maintenance Class page displays resources and carriers for the selected maintenance classes.

A check next to a maintenance class indicates this requirement is active for the maintenance class. You can clear the check box to deactivate a requirement for a maintenance class.

Maintenance Class Activation Page

You can access the Maintenance Class Activation page from the Resource menu. Use the Maintenance Class Activation page to select maintenance classes and resources for a maintenance requirement.

How to Activate a Maintenance Requirement for Maintenance Classes

Follow these steps to activate a maintenance requirement for maintenance classes:

1. Open the Maintenance Class Activation page on the Resource menu. The Maintenance Class Activation page appears.
2. Select the Activated check box to activate this requirement for a maintenance class.
3. Enter or select a maintenance requirement from the Maintenance Req list. The Maintenance Classes grid displays all maintenance classes for which this requirement has already been activated as well as the maintenance classes that are available for assignment to this requirement.
4. Click the arrow icon in the row containing the maintenance class name to view the resources or carriers associated with that maintenance class. The associated resources appear in the Resources grid.
5. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
6. Click Submit. The maintenance requirement is activated for the selected maintenance classes.
