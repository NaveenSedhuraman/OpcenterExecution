Task Delegation

A system administrator can reassign a task from one employee to another using Delegation. For example, the administrator can delegate a package approval or collaboration task to another employee if the employee originally assigned to the task will be on vacation when the task is due. The administrator can reassign specific tasks or delegate all tasks from one employee to another for a specified date range.
