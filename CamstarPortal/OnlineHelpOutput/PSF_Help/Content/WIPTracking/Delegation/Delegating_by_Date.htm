Delegating by Date

The application provides a search feature that enables you to search for and retrieve delegation definitions that specify a date range. You add delegation definitions in the Search Results grid.

When Searching by Date

Delegation definitions enable you to delegate tasks from one employee to another for a specified date range. The application displays multiple fields by which you can narrow your search. Note the following:

• You can search by application. Change Management is the only application currently available.
• The application selects delegation definitions with a status of Active or Future by default.

No delegation definitions appear in the Search Results grid unless you add at least one and it meets the search criteria. Refer to "When Delegating by Date" for information on delegation definitions.

Executing a Search by Date

The application performs the following when you click the Search button on the By Date tab:

• Displays the results in the Search Results grid
• Retains the search criteria in the filter fields

The application displays search results sorted in ascending order by these fields:

• Start Date
• End Date
• Assigned Employee
• Target Employee

The application adds paging to the grid if the search returns more results than it can display in the rows on the grid. Siemens also provides the ability to export your search results to an Excel or CSV format.

When Delegating by Date

You must add a delegation definition to reassign tasks from one employee to another for a specified date range. You can add a delegation definition by clicking the Add new row button in the Search Results grid.

Adding Delegation Definitions

The Delegate Tasks By Date pop-up appears when you click Add new row in the Search Results grid on the By Task tab. The Delegate Tasks By Date pop-up enables you to reassign a task or tasks for the selected application from the assigned employee to a target employee for a specified date range. You must also select a delegation reason when delegating tasks. Refer to the Opcenter Execution Medical Device and Diagnostics Modeling User Guide or Opcenter Execution Core Modeling User Guide for information on defining delegation reasons.

The following apply when you add a delegation definition:

• The assigned employee and the target employee cannot be the same.
• The start date must be in the future.
• The end date must be later than the start date.
• Delegation definitions cannot be duplicates. A duplicate delegation definition is a definition that specifies the same assigned employee for the same application within the same date range, even if the date range overlaps by only one minute.

The Enable delegation forwarding check box allows you to indicate whether tasks delegated to the target employee are allowed to be forwarded to another employee. The application forwards delegation to another employee if the target employee in the definition you are adding is an assigned employee in another definition. The application forwards the delegation to the target employee specified in the second definition.

The application enables the OK button when you successfully complete all required fields.

The application performs the following when you click OK in the Delegate Tasks By Date pop-up:

• Displays a success message indicating that the delegation definition was saved
• Closes the Delegate Tasks By Date pop-up
• Refreshes the Search Results grid

Editing Delegation Definitions

You can edit a future or active delegation by selecting a delegation definition in the Search Results grid and clicking the Edit selected row button. The fields you can edit in the Delegate Tasks By Date pop-up are dependent on the status of the delegation. The following applies.

If status is . . .

Then you . . .

Future

Can edit all fields.

Active

Can edit End Date only. The end date must be in the future. The application sets all other fields to display only.

Expired

Cannot edit any fields. The application sets all fields to display only.

Deleting Delegation Definitions

You can delete a future delegation by selecting a delegation definition in the Search Results grid and clicking the Delete selected row button. A confirmation dialog appears asking you to confirm the deletion. You cannot delete active and expired delegations.

How to Search for Delegation Definitions

Follow these steps to search for delegation definitions:

1. Open the Delegation Search page from the Search menu. The Delegation Search page appears with the By Task tab selected. A filter icon indicates that criteria is specified.
2. Click the By Date tab. The By Date tab appears. A filter icon indicates that criteria is specified.
3. Enter or select one or more filters if you want to narrow your search.
4. Click Search. The application displays the delegation definitions matching your search criteria in the Search Results grid.

How to Add a Delegation Definition

Follow these steps to add a delegation definition:

1. Perform the "How to Search for Delegation Definitions" procedure.
2. Click Add new row in the Search Results grid. The Delegate Tasks By Date pop-up appears.
3. Select an employee from the Assigned Employee list.
4. Select an employee from the Target Employee list.
5. Select a delegation reason from the Delegation Reason list.
6. Select an application from the Application list.
7. Enter (or select from the calendar control) the start date for this delegation definition in the Start Date field.
8. Enter (or select from the calendar control) the end date for this delegation definition in the End Date field.
9. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
10. Click OK. The application saves the delegation definition, displays a success message, closes the Delegate Tasks By Date pop-up, and refreshes the Search Results grid.

How to Edit a Delegation Definition

Follow these steps to edit a delegation definition:

1. Perform the "How to Search for Delegation Definitions" procedure.
2. Select a future or active delegation definition in the Search Results grid.
3. Click Edit selected row. The Delegate Tasks By Date pop-up appears.
4. What is the status of the delegation definition?

If status is . . .

Then you can . . .

Future

Edit any field as necessary.

Active

Edit the End Date as necessary.
5. Click OK. The application updates the delegation definition, displays a success message, closes the Delegate Tasks By Date pop-up, and refreshes the Search Results grid.

How to Delete a Delegation Definition

Follow these steps to delete a delegation definition:

1. Perform the "How to Search for Delegation Definitions" procedure.
2. Select a future delegation definition in the Search Results grid.
3. Click Delete selected row. A confirmation dialog appears asking you to confirm the deletion.
4. Click Yes. The application deletes the delegation definition, displays a success message, closes the confirmation dialog, and refreshes the Search Results grid.
