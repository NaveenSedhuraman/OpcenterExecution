Delegating by Task

The application provides a search feature that enables you to search for and retrieve tasks based on specified criteria. You can select one or more tasks to delegate when the application returns the tasks meeting your criteria.

When Searching by Task

The application displays multiple fields by which you can narrow your search. Note the following:

• You can search by application. The tasks available in the Task list are dependent on the application you select. The default setting for Application is currently set to Change Management.
• The application clears and disables the Due Date Begin Timestamp and Due Date End Timestamp fields if you select Task Due Today or Late.

Executing a Search by Task

The application performs the following when you click the Search button on the By Task tab:

• Displays the results in the Search Results grid
• Retains the search criteria in the filter fields

The application displays only tasks that are currently in process. It does not display completed tasks.

The application displays search results sorted in ascending order by these fields:

• Assignment Date
• Due Date
• Assigned Employee

The value in the Status column for change management tasks can be either Pending or Late. Cell highlight color is dynamic based on status. The application highlights the cell in orange for pending tasks and in red for late tasks.

The application adds paging to the grid if the search returns more results than it can display in the rows on the grid. Siemens also provides the ability to export your search results to an Excel or CSV format.

When Delegating by Task

The application enables the Delegate Task button when you select one or more tasks in the Search Results grid on the By Task tab. The Delegate Task pop-up appears when you click Delegate Task. The Delegate Task pop-up enables you to reassign the task or tasks from the assigned employee to a target employee. You must select a target employee and delegation reason when delegating a task. The application enables the Delegate button when you complete these two required fields. Refer to the Opcenter Execution Medical Device and Diagnostics Modeling User Guide or the Opcenter Execution Core Modeling User Guide for information on defining delegation reasons.

The application performs the following when you click Delegate in the Delegate Task pop-up:

• Replaces the assigned employee with the target employee for the selected tasks
• Closes the Delegate Task pop-up
• Displays a success message indicating the application delegated the selected tasks
• Refreshes the Search Results grid

How to Search by Task

Follow these steps to search by task:

1. Open the Delegation Search page from the Search menu. The Delegation Search page appears with the By Task tab selected. A filter icon indicates that criteria is specified.
2. Enter or select one or more filters if you want to narrow your search.
3. Click Search. The application displays the tasks matching your search criteria in the Search Results grid.

How to Delegate by Task

Follow these steps to delegate by task:

1. Perform the "How to Search by Task" procedure.
2. Select one or more tasks in the Search Results grid.
3. Click Delegate Task. The Delegate Task pop-up appears.
4. Select an employee from the Target Employee list.
5. Select a delegation reason from the Delegation Reason list.
6. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
7. Click Delegate. The application replaces the assigned employee with the target employee for the selected tasks, closes the Delegate Task pop-up, displays a success message, and refreshes the Search Results grid.
