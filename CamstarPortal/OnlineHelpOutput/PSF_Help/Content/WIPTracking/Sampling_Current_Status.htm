Current Sampling Status Update

The Current Sampling Status Update page allows you to update both the Sample Rate Counter and the Sampling Plan Inspection Level for either a specific Sampling Current Status record or for all Sampling Current Status records associated with a product. You can enter a value in the New Sample Rate Counter Value field to change the frequency containers are sampled. You can also select an inspection level in the New Inspection Level field to change the inspection level for the sampling plan. Refer to the Opcenter Execution Medical Device and Diagnostics Modeling User Guide or the Opcenter Execution Core Modeling User Guide for information on defining sampling plans.

When Using the Current Sampling Status Update Page

These rules apply when using the Current Sampling Status Update page:

• Only products with container-based sampling plans appear in the Product field.
• Changes made on this page take effect on the next container that meets the sampling requirement. Containers that are already in progress are not affected.
• This page is not applicable for lot sampling.
Current Sampling Status Update Page

You can access the Current Sampling Status Update page from the Container menu.

How to Update the Sample Rate Counter

Follow these steps to update the sample rate counter:

1. Open the Current Sampling Status Update page from the Container menu. The Current Sampling Status Update page appears.
2. Select a product in the Product field. The application displays a list of Sampling Current Status records in the Sampling Current Status Details grid.
3. Select the check box of the Sampling Current Status record for which you want to update the sample rate counter.

Or

Select the check box in the header row of the Sampling Current Status Details grid to update all sample rate counters.

4. Enter a value in the New Sample Rate Counter Value field.
5. Click Update or Submit on the command bar. The application refreshes the Sampling Current Status Details grid.

How to Update the Inspection Level

Follow these steps to update the inspection level:

1. Open the Current Sampling Status Update page from the Container menu. The Current Sampling Status Update page appears.
2. Select a product in the Product field. The application displays a list of Sampling Current Status records in the Sampling Current Status Details grid.
3. Select the check box of the Sampling Current Status record for which you want to update the inspection level.
4. Optionally, select the check box in the header row of the Sampling Current Status Details grid to update all inspection levels.
5. Select an inspection level in the New Inspection Level field.
6. Click Update or Submit on the command bar. The application refreshes the Sampling Current Status Details grid.
