Configuring Shop Floor Pages for Electronic Procedures

Siemens provides the ability for you to execute a shop floor transaction as an electronic procedure. Shop floor transaction pages must be configured for use in electronic procedures. Siemens provides some transaction pages configured for electronic procedures already. A system administrator must configure any other pages using Portal Studio.

These shop floor transaction pages are configured for electronic procedures by default:

• Associate
• Attach Document*
• Change Qty (Basic and Multi-Reason)
• Close*
• Combine Container
• Combine Qty
• Component Defect
• Component Issue (Basic and Advanced)
• Component Remove
• Component Replace
• Container Attribute Maintenance
• Container Defect
• Container Hold
• Container Maintenance
• Container Release
• Container Thruput
• Data Collection
• Disassociate
• Move
• Move In
• Move Non-Std
• Rework
• Ship
• Transfer

* Process timer functionality is not available for these transactions.

When Configuring Shop Floor Pages

You must perform these tasks when configuring shop floor pages:

• Add the EProcHiddenFields_WP web part to the page
• Add the following three data contracts at the page level:
• EProcHiddenTaskDM
• EProcHiddenTaskContainerDM
• EProcHiddenTaskList DM

How to Configure a Shop Floor Transaction Page for Electronic Procedures

Follow these steps to configure a shop floor transaction page for electronic procedures:

1. Open Portal Studio.
2. Select the page you want to configure in the Components pane.
3. Double-click page. The page appears.
4. Drag EProcHiddenFields_WP from the Web Parts pane to the page. The application displays a message asking whether you want to add the web part as a local copy or a reference to its content.
5. Select Copy. The web part appears on the page.
6. Click Save.
7. Add these three data contracts to the page:
• EProcHiddenTaskDM
• EProcHiddenTaskContainerDM
• EProcHiddenTaskListDM
8. Click OK to close the Data Contract window.
9. Click Save. The application saves your changes and displays a success message.
10. Repeat steps 2-9 for each page you want to configure.
Note: Refer to the Opcenter Execution Medical Device and Diagnostics Portal Studio User Guide or the Opcenter Execution Core Portal Studio User Guide for information.
