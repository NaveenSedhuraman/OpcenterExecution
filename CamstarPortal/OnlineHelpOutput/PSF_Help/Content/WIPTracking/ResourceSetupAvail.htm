Resource Setup and Availability

Use the Resource Setup transaction to record changes to a resource setup or to indicate the availability of a resource or carrier.

Resource setup information includes references to a specific Setup Definition and the Product being processed.

Availability information includes the resource’s:

• Availability (up, down, or not set up)
• Status (such as in production, idle, or repair)
• Reason code (such as broken gauge)

A history of resource utilization and failure details is developed from the availability information entered in Manufacturing. For example, when a resource fails, you can perform the Resource Setup transaction to change the availability to Down, the status to Repair, and the reason to Broken Gauge.

When the repair is completed, you can perform another Resource Setup transaction to return the availability to Up and the status to In Production. Availability is controlled by the Resource Status Code, defined in Modeling. You now have a record of when the failure occurred, the reason for the failure, and how long it took to repair.

When Using Resource Setup

The Resource field displays all resources and carriers unless you select a Resource Group. Then the display of resources and carriers is filtered by that Resource Group.

When you select a resource on the Resource Setup page, the resource's availability appears. These conditions determine what appears on the page in the resource header area for availability.

Resource Setup Page

You can access the Resource Setup page from the Resource menu. Use the Resource Setup page to record changes to a resource setup or to indicate the availability of a resource or carrier.

How to Set Up a Resource

Follow these steps to set up a resource:

1. Open the Resource Setup page from the Resource menu. The Resource Setup page appears.
2. Select a Resource Group if you want to limit the list of resources displayed.
3. Select the Resource you want to set up. The page updates, displaying the following if the resource has already been set up: the Time at Status (time at current status), Status Code (current Resource Status), and Status Reason (current Resource Status Reason). The Availability field displays the selected resource's current availability.
4. Select a Resource Status Code.
5. Enter optional information according to your business requirements. Refer to the field definitions table for information on the optional fields.
6. Click Setup. A message indicating success appears.
