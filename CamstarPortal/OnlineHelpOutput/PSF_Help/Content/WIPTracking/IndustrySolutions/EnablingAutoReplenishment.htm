Enabling Auto Replenishment of Materials

Configuring automatic replenishment is part of the eKanban process. Refer to the Opcenter Execution Medical Device and Diagnostics Industry Solutions User Guide or the Opcenter Execution Core Industry Solutions User Guide for information on material management using the eKanban process.

You can enable material replenishment requests to be automatically created when material quantities decrease below a user-defined level. Replenishment requests are then sent to the Valor Material Management system, if configured. Refer to the Opcenter Execution Core Industry Solutions User Guide or Opcenter Execution Medical Device and Diagnostics Industry Solutions User Guide for information.

When Enabling Auto Replenishment for Materials in the Material Queue

Automatic replenishment can be enabled for materials that are loaded on to a selected material queue. Products that can be configured for replenishment appear in the Material Queue Replenish Details grid on the Manage Material Queue or Manage Material Queue Simple pages after selecting a material queue.

Automatic replenishment is not enabled for products that are set for manual replenishment in the grid.

How to Enable Auto Replenishment for Materials

Follow these steps to enable auto replenishment for materials on a material queue:

1. Open the Manage Material Queue page. The Manage Material Queue page appears.

Or

Open the Manage Material Queue Simple page. The Manage Material Queue Simple page appears.

2. Select a Material Queue from the list. The application populates the Resource field with the associated resource if applicable. The application selects the Is Active check box if the material queue is active. The application populates the Material Queue grid with materials currently loaded on the material queue. The application enables the Load and Reset buttons.
3. Select a product in the Material Queue Replenish Details grid.
4. Enter a value in the Threshold Qty field.
5. Select a value in the Threshold UOM field.
6. Enter a value in the Replenish Qty field.
7. Enter optional information according to your business requirements. Refer to the "Managing Material Queues" for information about the optional fields.
8. Click Submit. The application saves your changes and displays a success message.
