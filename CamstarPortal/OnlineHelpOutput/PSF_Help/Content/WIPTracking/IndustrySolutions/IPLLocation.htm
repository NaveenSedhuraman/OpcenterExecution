Managing Valor Material Management Locations

The Valor Location page enables you to create and configure Material Queues and Inventory Locations directly in Valor Material Management. These locations are required to interface with Valor MSS Advanced Material Management. Specifically, the material queue is required when requesting material from Valor to fulfill an eKanban replenishment or a manual material request.

Configuring the Valor Material Management Location Page

You must configure the Valor Location page to appear in the application using Portal Studio. You must add the ValorMMS_Equip page to an existing portal menu using the navigation option. Siemens recommends that you add the page to the isMaterialQueues menu using “Valor Location” as the label. You must also enter the location of your Valor installation in the DataContractMembers properties for the page. Refer to the Opcenter Execution Medical Device and Diagnostics Portal Studio User Guide or Opcenter Execution Core Portal Studio User Guide for information.

Valor Location Page

You must select Inventory Location or Material Queue as the object type when using the Valor Location page. The object type selection determines whether Material Queues or Inventory Locations are displayed in the Instance Name field. You must then select the instance that you want to manage.

Clicking Load populates the grid with configurable fields and any values previously entered if the selected instance exists in Valor Material Management. If the instance does not exist in Valor Material Management, it is automatically created and a grid displaying configurable fields with their default values is returned.

How to View a Valor Location

Follow these steps to view a Valor Location.

1. Open the Valor Location page. The Valor Location page appears.
2. Select Material Queue from the Object Type list. The application populates the Instance Name list.

Or

Select Inventory Location from the Object Type list. The application populates the Instance Name list.

3. Select an object instance from the Instance Name list. The application populates the Instance Name field.
4. Click Load. The application displays configurable fields in the grid associated with the selected object instance in Valor Material Management.

How to Configure a Valor Location

Follow these steps to configure a Valor Location.

1. Open the Valor Location page. The Valor Location page appears.
2. Perform the "How to View a Valor Location" procedure.
3. Update the setting in the grid and click Save. A success message is displayed, and the setting is configured in Valor Material Management.
