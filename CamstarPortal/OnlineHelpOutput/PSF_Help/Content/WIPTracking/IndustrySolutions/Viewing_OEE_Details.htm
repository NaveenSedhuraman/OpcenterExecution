Viewing OEE Details

You can use the OEE Details page to view charts for detailed OEE metrics for either a selected resource or for all resources in a specified resource family. These metrics include:

• Quality
• Availability
• Performance
• Top Reasons for Downtime
• Top Reasons for Scrap/Rework

OEE Details Page Field Definitions

This table defines the fields on the OEE Details page.

Field

Definition

Type

Resource Family

List of available resource families.

Optional

Resource

List of resources associated with the selected resource family.

Optional

Top Downtime Reasons

Number of downtime reasons to display.

Optional

Top Scrap/Rework

Number of scrap/rework reasons to display.

Optional

Timespan

Time span of calculated OEE data. Options include:

• Shift - Displays OEE data calculated for the current shift.
• Day - Displays OEE data calculated beginning with the first shift of the day as defined on the resource, factory, or manufacturing calendar.

Optional

Custom

Check box indicating the application will display OEE data between times specified in the Start Time and End Time fields.

Optional

Start Time

Specified start time for the calculation of OEE data.

Optional

End Time

Specified end time for the calculation of OEE data.

Optional

How to View OEE Details for a Resource Family

Follow these steps to view OEE details for a resource family:

1. Open the OEE Details page from the Resource menu. The OEE page appears.
2. Select a Resource Family from the list.
3. Enter a value in the Top Downtime Reasons field.
4. Enter a value in the Top Scrap/Rework field.
5. Do you want to display the data based on a time span or on custom parameters?

If . . .

Then . . .

Time span
a. Select a Timespan from the list.
b. Go to step 6.

Custom parameters
a. Select the Custom check box.
b. Select a start time in the Start Time field.
c. Select an end time in the End Time field.
d. Go to step 6.
6. Click Load Data. The application displays charts with OEE details for all resources associated with the selected resource family.

How to View OEE Details for a Resource

Follow these steps to view OEE details for a resource:

1. Open the OEE Details page from the Resource menu. The OEE Details page appears.
2. Select a Resource Family from the list. The application populates the Resource field with a list of resources associated with the selected resource family.
3. Select a Resource from the list.
4. Enter a value in the Top Downtime Reasons field.
5. Enter a value in the Top Scrap/Rework field.
6. Do you want to display the data based on a time span or on custom parameters?

If . . .

Then . . .

Time span
a. Select a Timespan from the list.
b. Go to step 7.

Custom parameters
a. Select the Custom check box.
b. Select a start time in the Start Time field.
c. Select an end time in the End Time field.
d. Go to step 7.
7. Click Load Data. The application displays charts with OEE details for the resource.
