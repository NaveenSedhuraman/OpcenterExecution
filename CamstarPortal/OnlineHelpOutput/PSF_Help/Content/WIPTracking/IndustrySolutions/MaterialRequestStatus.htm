Viewing Material Request Statuses

You can view the status of material requests (automated or manual) using the Material Request Status Board page. You can access the Material Request Status Board page using the Resource Material Queue menu.

The Material Request Status Board functionality is part of the eKanban process. Refer to the Opcenter Execution Medical Device and Diagnostics Industry Solutions User Guide or the Opcenter Execution Core Industry Solutions User Guide for information.

Configuring the Material Request Status Board

Siemens provides auto-refresh functionality for the Material Request Status Board so that you can view active material requests in real time.

You must install and configure Opcenter Connect MOM to enable the auto-refresh functionality. Refer to the Opcenter Connect MOM Installation Guide for information.

Siemens provides the following Transaction Tester scripts to assist with configuring Opcenter Connect MOM:

• OPEXCON_MaterialRequest.cfg
• OPEXCR_MaterialRequest.cfg

The scripts are located at the following location by default:

C:\\Program Files (x86)\Camstar\Industry Solutions\Industry\Scripts\Config

Note: The application requires you to use the same Channel Source Name and Control Address in both Opcenter Connect MOM and in the isMaterialRequestStatusBoard VP tile SignalRURL property in Portal Studio. You must update both locations if changes are made.

When Viewing Material Request Statuses

The Material Request Status Board page displays all unfulfilled material request statuses. Each tile on the page represents a Material Queue Replenishment request. Tiles are added to the board when a material request has been created.

The tiles on the board are ordered so that the oldest statuses by date/time are displayed at the top-left of the column and progress in ascending order from left to right. The tiles wrap the screen and start a new row when space is not available for a new tile.

Each tile contains:

• Material request status
• Material Queue name
• Product name that is requested
• Product Description
• Qty Requested / UOM
• Date Requested
Material Request Status Board Page

The page is refreshed automatically to display current requests and statuses if you have installed Opcenter Connect MOM and configured the Material Request Status Board. Refer to the Opcenter Execution Medical Device and Diagnostics Industry Solutions User Guide or the Opcenter Execution Core Industry Solutions User Guide for information.

The application removes tiles from the board when a Material Acknowledgment occurs on a delivered Material Request.
