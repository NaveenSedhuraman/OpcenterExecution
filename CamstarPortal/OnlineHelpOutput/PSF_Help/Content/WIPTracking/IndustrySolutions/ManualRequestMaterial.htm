Manually Requesting Material

You can manually request material for a material queue or inventory location using the Material Request page.

The Product field in the Material Request grid only displays the products associated with the selected Material Queue or Inventory Location by default. Selecting the Show All Products checkbox enables the application to display all products defined in the application.

Required Setup for Electronics Suite

The Material Management Enabled check box must be selected on the Factory modeling page to use the Material Request page if you have Electronics Suite installed. Additionally, at least one inventory location must be created.

Note: Opcenter Execution Electronics users can access a Material Request pop-up from the Production Client when an open defect is selected.

Material Request Message Example

The Material Request transaction results in the generation of a message that is converted and sent to the Material Management REST API if you have installed Opcenter Connect MOM and configured an outbound message. This interface is intended to work in conjunction with Valor Material Management System. Refer to the Opcenter Execution Medical Device and Diagnostics Industry Solutions User Guide or the Opcenter Execution Core Industry Solutions User Guide for information.

This code shows an example of a converted Material Request message:

<ManualPicklistEntries xmlns="http://www.valor.com/MSS/Schemas/ManualPicklist/2011">

<ManualPicklistEntry ResourceName="Valor-Storage-1" CompName="0000_component_1" Amount="50" Units="" OperatorID="Operator" DisplayName="Valor-Storage-1" ExternalOrderID="49e073000000001c"/>ManualPicklistEntry ResourceName="Valor-Storage-1" CompName="0000_CompanyName_1" Amount="50" Units="" OperatorID="Operator" DisplayName="Valor-Storage-1" ExternalOrderID="49e073000000001c"/>

<ManualPicklistEntry ResourceName="Valor-Storage-1" CompName="000000589366" Amount="300" Units="" OperatorID="Operator" DisplayName="Valor-Storage-1" ExternalOrderID="49e073000000001d"/>

</ManualPicklistEntries>

How to Manually Request Material

Follow these steps to manually request material.

1. Open the Material Request page. The Material Request page appears.
2. Select an Inventory Location or Material Queue from the list. The application populates the Resource field with the associated resource if applicable.
3. Select the Show All Products checkbox, if applicable.
4. Select a product in the Material Request grid. The application populates the Description field.
5. Enter a value in the Requested Qty field.
6. Select a value in the UOM field.
7. Repeat steps 3-5 for each additional material request.
8. Click Submit. The application saves your changes and displays a success message.
