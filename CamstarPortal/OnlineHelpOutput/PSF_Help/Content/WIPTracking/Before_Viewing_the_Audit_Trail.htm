Before Viewing the Manufacturing Audit Trail or Resource Audit Trail

Before viewing container, carrier, event information in the manufacturing audit trail or resource information in the resource audit trail, a system administrator must perform these tasks:

• Define one or more UI preferences in Modeling. A UI preference sets the order of the fields displayed on the Status tab of the Mfg Audit Trail page for a container, carrier, or event. It also sets the order of the fields displayed on the Status tab of the Resource Audit Trail page for a resource. Your organization may consider defining three UI preferences: one for containers and carriers, one for events, and one for resources.
• Associate a UI Preference to:
• A container level for containers and carriers.
• An organization for events.
• A resource type or resource family for a resource
• Configure any custom views for the History Mainline grid using the Audit Trail Configurator. Siemens provides a default view.
