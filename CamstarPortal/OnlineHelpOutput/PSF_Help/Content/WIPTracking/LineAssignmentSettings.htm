Line Assignment Settings

The application displays your current line assignment settings in the global header of the Portal. Line assignment settings allow you to set an assignment based on specific attributes such as:

• Resource/Work Cell
• Work Center
• Operation
• Workstation

These settings limit the total number of objects displayed during query activities. For example, if you access the Move transaction page without having any line assignments set, you may have hundreds of objects (containers and resources) to select from. By contrast, if you set a few line assignments like Resource and Operation, the number of containers and resources to select from will be considerably fewer.

Setting Your Line Assignment

A system administrator can set a default work center and operation for your user ID so that these assignments are applied each time you log on. While you have the ability to customize your line assignment settings during each session, you will need to contact your system administrator for a permanent change. You can change your settings for the session at log on if the Line Assignment page appears or any time during the session using the Set Line Assignment pop-up.

Additionally, a system administrator may set the Set Line Assignment page to display each time you log onto the Portal. You do have the ability to set the application to skip the Set Line Assignment page for future logons. Again, you can use the Set Line Assignment page or pop-up to change your settings.

Note: You can restore your default settings (those defined by your system administrator) at any time by clicking the Reset button on the Set Line Assignment page or pop-up.

Terminal-Based Line Assignment

The application provides the option to configure terminal-based line assignment settings. These settings apply to a specific terminal rather than to a specific employee. Terminal-based line assignment settings override employee-based line assignment settings while the employee is using that terminal. A terminal can be configured with the following line-assignment settings:

• Home Page
• Mobile Home Page
• Work Center
• Operation
• Resource
• Workstation

Default Factory configuration does not permit employees to override terminal-based line assignments. System administrators can change this default by selecting the Factory page check box labeled Allow Terminal Line Assignment Override. Employee overrides to terminal-based line assignment settings remain in effect only during the employee’s current session on the terminal.

Note: Refer to the Opcenter Execution Medical Device and Diagnostics Modeling User Guide or the Opcenter Execution Core Modeling User Guide for information on configuring a terminal with line assignment settings.

Set Line Assignment Pop-up

The Set Line Assignment pop-up allows you to change your line assignment for your current session and to enable or disable the Set Line Assignment page for future logons. Clicking the Settings link in the global header displays the pop-up.

This table defines the fields on the Set Line Assignment page and pop-up.

When selected, the application does not display the Set Line Assignment page when you log on. If you clear the check box, the application displays the Set Line Assignment page at log on, but only if your system administrator set the page to be visible to you on logon.
